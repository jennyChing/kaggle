        <p>
         Hey
        </p>
        <p>
         If you are a into web development at all you should be familiar with
         <a href="http://macrabbit.com/cssedit/">
          CSSEdit
         </a>
         . I have reviewed it
         <a href="http://www.mactricksandtips.com/2008/04/cssedit-properly-edit-css-files.html">
          before
         </a>
         , its the one app that takes the pain away from hand writing CSS code, I love it and use it all of the time. This post is going to show you a couple of tricks and tips, to get your CSS formatting look neat and tidy, as well as efficient. It doesnt take much, just a bit of know how. If you want to learn more about CSS, I recommend
         <a href="http://www.amazon.com/gp/product/0596802447/ref=as_li_ss_tl?ie=UTF8&amp;tag=mact06-20&amp;linkCode=as2&amp;camp=1789&amp;creative=390957&amp;creativeASIN=0596802447">
          CSS: The Missing Manual
         </a>
         .
        </p>
        <p>
         Normal CSS code, with syntax highlighting will look like the image below. Pretty standard.
        </p>
        <p>
         <a href="http://www.mactricksandtips.com/wp-content/uploads/2008/05/css-text-mac.png">
          <img alt="" class="aligncenter size-medium wp-image-2253" height="300" src="http://www.mactricksandtips.com/wp-content/uploads/2008/05/css-text-mac-125x300.png" title="css text mac" width="125"/>
         </a>
         <br/>
         The first step is to get all of you code lined up nice an neatly. If you are like me, I prefer code to be tabbed correctly, have the required spacing underneath each section etc. Usually you would have to go through this by hand. But CSSEdit has a good little command called Re-Indent. If you hit this button, it will indent all of the code so it looks perfect. No need to select the text as I found out after using this tool about a hundred time. It turns code into something that looks similar to the image above. You can, by the way, just apply this to small sections.
        </p>
        <p>
         <a href="http://www.mactricksandtips.com/wp-content/uploads/2008/05/css-edit-menu.png">
          <img alt="" class="aligncenter size-full wp-image-2254" height="127" src="http://www.mactricksandtips.com/wp-content/uploads/2008/05/css-edit-menu.png" title="css edit menu" width="229"/>
         </a>
         <br/>
         The next tip is a way to change the formatting to what you want. Did you know that:
        </p>
        <p>
         <code>
          #style{padding: 0 5px; margin: 10px 5px;}
         </code>
        </p>
        <p>
         takes up disk space than:
        </p>
        <p>
         <code>
          #style{
          <br/>
          padding: 0 5px;
          <br/>
          margin: 10px 5px;
          <br/>
          }
         </code>
        </p>
        <p>
         its not by much, but it can add up. As well as this I think the first is a bit easier to work with. To change the formatting to this style (or any style you want). Go to Format &gt; Re-Indent With Spacing here you will see a screen similar to the one below. That is the formatting that is actually applied to the example above, but this box is change able, so you can change the spacing between the words.
        </p>
        <p>
         <a href="http://www.mactricksandtips.com/wp-content/uploads/2008/05/css-edit-text-spacing.png">
          <img alt="" class="aligncenter size-medium wp-image-2255" height="293" src="http://www.mactricksandtips.com/wp-content/uploads/2008/05/css-edit-text-spacing-300x293.png" title="css edit text spacing" width="300"/>
         </a>
         <br/>
         For example you can remove all of the spacing entirely this takes up a very small amount of room, and on a big style sheet it can really make a difference.
        </p>
        <p>
         <a href="http://www.mactricksandtips.com/wp-content/uploads/2008/05/custom-css-spacing.png">
          <img alt="" class="aligncenter size-medium wp-image-2256" height="296" src="http://www.mactricksandtips.com/wp-content/uploads/2008/05/custom-css-spacing-300x296.png" title="custom css spacing" width="300"/>
         </a>
         <br/>
         If you click through to the image below you can see the difference below. Everything is more compact. For example before with full, very easy to read formatting, the style sheet was 1558 lines. Once as the Re-Indent was applied it took up 263 lines. This is a massive difference. As well as this it also makes it a lot easier to find selectors you want.
        </p>
        <p>
         <a href="http://www.mactricksandtips.com/wp-content/uploads/2008/05/compressed-css-text.png">
          <img alt="" class="aligncenter size-medium wp-image-2257" height="106" src="http://www.mactricksandtips.com/wp-content/uploads/2008/05/compressed-css-text-300x106.png" title="compressed css text" width="300"/>
         </a>
        </p>
        <p>
         It is fun to play with this tool. You can easily change the way the formatting works to your liking. Although the second options doesnt have a whole lot of disk space, 4Kb, it will eventually add up. If you have a million visitors its around 4Gb of save bandwidth (if my math is correct). It would be even bigger of a deficit if you had a massive style sheet for a really large website.
        </p>
        <p>
         If you want to find out more on saving space with style sheets, if found a cool article as I was writing this post. Head over to
         <a href="http://www.thewebsqueeze.com/articles/css-shorthand-step-by-step-how-to-40.htm">
          The Web Squeeze
         </a>
         , which gives a lot more information. Hopefully you have found this article informative, and gives you insight into some of the cool tricks you can do with CSSEdit. Now I just wished there was a way to put the code in the selectors in alphabetical order.
        </p>
        <p>
         To learn more about CSS in all its glory I recommend
         <a href="http://www.amazon.com/gp/product/0596802447/ref=as_li_ss_tl?ie=UTF8&amp;tag=mact06-20&amp;linkCode=as2&amp;camp=1789&amp;creative=390957&amp;creativeASIN=0596802447">
          CSS: The Missing Manual
         </a>
         from Amazon.
        </p>
