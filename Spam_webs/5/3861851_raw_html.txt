      <p>
       Gif Scratcher is a jQuery plugin which allows you to animate image sprites with custom interactions. Helps you manually play &amp; stop an animated GIF like a 'GIF Player'.
      </p>

      <h2>
       How to use it:
      </h2>
      <p>
       1. Use
       <a href="http://www.imagemagick.org/" rel="nofollow" target="_blank">
        ImageMagick
       </a>
       to extract every single frame from an animated GIF as individual JPEGs and then create a sprite sheet.
      </p>
      <p>
       2. Include the required
       <code>
        gifscratcher.css
       </code>
       in the head section of the web page.
      </p>
      <pre class="brush:xml;">
       &lt;link rel="stylesheet" href="stylesheets/gifscratcher.css"&gt;
      </pre>
      <p>
       3. Insert the first frame of your image sprite sheet into a container.
      </p>
      <pre class="brush:xml;">
       &lt;div id="demo"&gt;
  &lt;img src="images/00.jpg"&gt;
&lt;/div&gt;
      </pre>
      <p>
       4. Call the plugin on the container and set the path of your image sprite sheet.
      </p>
      <pre class="brush:js;">
       $("#demo").gifscratcher({
  sprite : '1.jpg',
  frames : 26
});
      </pre>
      <p>
       5. All the default settings.
      </p>
      <pre class="brush:js;">
       // image sprite sheet path
sprite      : '',

// the number of columns the sprite sheet has
frames      : 0,

// 'hover', 'drag', 'auto', 
// 'autoWithHover', 'autoWithDrag'
interaction : 'hover',

// Add custom cursor
cursor      : true,

// define animation speed
speed       : 5
      </pre>
