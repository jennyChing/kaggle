      <h1 itemprop="name">
       Gain Administrator Access Without a Password
      </h1>
      <div class="article-meta">
       <div class="article-byline">
        By
        <a href="/author/Lincoln-Spector/" itemprop="url" rel="author">
         <span itemprop="name">
          Lincoln Spector
         </span>
        </a>
        , PCWorld
       </div>
       <ul class="fmenu">
        <li itemprop="datePublished">
         Oct 11, 2010 7:36 AM
        </li>
        <li>
         <a class="icn-link icn-email ir" href="#email" id="email-icon">
          e-mail
         </a>
        </li>
        <li>
         <a class="icn-link icn-print ir" href="javascript://" onclick="window.print()">
          print
         </a>
        </li>
       </ul>
      </div>
      <span itemprop="articleBody text">
       <section class="page">
        <p>
         <em>
          Alexleonardme acquired a used PC but doesn't have an administrator password. He asked the
          <a href="http://forums.pcworld.com/index.php?/forum/2000-windows/">
           Windows
          </a>
          forum for help.
         </em>
        </p>
        <p>
         There are a number of ways to gain access to a password-protected administrator account. I'll offer two.
        </p>
        <p>
         The first one is quick and easy, but only works in XP. If the used PC runs Vista or Windows 7, skip the next three paragraphs.
        </p>
        <p>
         <figure class="image small">
          <img alt="" height="119" src="http://images.pcworld.com/news/graphics/148186-hp_090403_passwords.jpg" width="180"/>
         </figure>
         XP contains a backdoor entrance to administrator control. The following will only work if the previous owner didn't know enough to close the backdoor.
        </p>
        <p>
         First, boot into Safe Mode: Start your PC, and press
         <strong>
          F5
         </strong>
         before the Windows logo appears. (It may take a few tries to get the timing right.) From the resulting Boot menu, select
         <em>
          Safe Mode
         </em>
         . When the logon screen appears, it will probably contain a user you've never seen before, Administrator, that does not have a password. Thus, you can enter Windows as an administrator-level user.
        </p>
        <p>
         Once inside Windows, you can use Control Panel's User Accounts applet to change the password on the other administrator account or create a new one for yourself. And while you're there, you might also want to add a password to the Administrator account in order to close the backdoor. But that's probably the case.
        </p>
        <p>
         If you're using Vista or Windows 7, or if someone closed that backdoor already, you can remove the administrator password with Ubuntu Linux. This gets a little complicated, so follow it closely.
        </p>
        <p>
         If you don't already have a live Ubuntu Linux CD, download the
         <a href="http://www.ubuntu.com/desktop/get-ubuntu/download">
          current version
         </a>
         . Double-click the just-downloaded .iso file. Windows might start a program that burns your Ubuntu CD. If it doesn't, download and install
         <a href="http://www.pcworld.com/downloads/file/fid,64310-page,1-c,images/description.html">
          ISO Recorder
         </a>
         , then try again. Simply copying the .iso file to a CD won't work.
        </p>
        <p>
         The instructions below are based on Ubuntu version 10.04.1 LTS--the most current version as I write this.
        </p>
        <p>
         Once you've created the disc, boot off the CD. When asked, click the
         <em>
          Try Ubuntu
         </em>
         button.
        </p>
        <p>
         <figure class="image left small">
          <a class="zoomLink" href="http://images.pcworld.com/howto/graphics/206809-1011-menu_original.jpg" target="_blank" title="">
           <img alt="" src="http://images.pcworld.com/howto/graphics/206809-1011-menu_original.jpg"/>
          </a>
         </figure>
         You'll first need to install the software: From the menus in the top-right corner of the screen, select
         <em>
          System, Administration, Software Sources.
         </em>
        </p>
        <p>
         In the resulting window, check the two options not yet checked:
         <em>
          Community-maintained Open Source software (universe)
         </em>
         and
         <em>
          Softrware restricted by copyright or legal issues (multiverse)
         </em>
         . Click
         <em>
          Close,
         </em>
         then
         <em>
          Reload.
         </em>
        </p>
        <p>
         After the program is done processing, select
         <em>
          System, Administration,
         </em>
         and
         <em>
          Synaptic Package Manager
         </em>
         from the top-right corner menus.
        </p>
        <p>
         In the resulting window's Search field, enter
         <strong>
          chntpw
         </strong>
         . A program with that name should appear in the list of packages below. Click it and select
         <em>
          Mark for installation.
         </em>
         Click the
         <em>
          Apply
         </em>
         toolbar icon. Click
         <em>
          Apply,
         </em>
         and wait until the installation is complete.
        </p>
        <p>
         <figure class="image medium">
          <img alt="" height="263" src="http://images.pcworld.com/howto/graphics/206809-1011-chntpw_original.jpg" width="350"/>
         </figure>
        </p>
        <p>
         Now mount your drive: From the top menus, select
         <em>
          Places,
         </em>
         then your internal hard drive or Windows partition. When it's mounted, a File Browser window (much like Windows Explorer) will appear.
        </p>
        <p>
         By double-clicking the folder icons, go to the Windows drive's Windows\system32\config folder. Leave this window up.
        </p>
        <p>
         Now you can remove the password.
        </p>
        <p>
         From the top menus, select
         <em>
          Applications, Accessories, Terminal.
         </em>
         This brings up a black window similar to Windows' command line (or the DOS box for you old-timers).
        </p>
        <p>
         <figure class="image left small">
          <a class="zoomLink" href="http://images.pcworld.com/howto/graphics/206809-1011-2window_original.jpg" target="_blank" title="">
           <img alt="" src="http://images.pcworld.com/howto/graphics/206809-1011-2window_original.jpg"/>
          </a>
         </figure>
         Arrange these two windows so that you can see the top section of the File Browser window behind the Terminal window.
        </p>
        <p>
         In the Terminal command line window, type
         <strong>
          cd /media/
         </strong>
         (note the SPACE after cd) and the path to your config folder, using the folder names at the top as a guide. I wish I could simply tell you to type "cd /media/windows/system32/config", but unlike the Windows command line, Ubuntu's Terminal is case sensitive. Depending on what version of Windows you're trying to reclaim, it may be windows/system32 or Windows/System32, and Ubuntu won't see it if you get the case wrong.
        </p>
        <p>
         When it's typed, press
         <strong>
          ENTER
         </strong>
         .
        </p>
        <p>
         Now type
         <strong>
          sudo chntpw -u
         </strong>
         <em>
          logon
         </em>
         <strong>
          SAM
         </strong>
         , where
         <em>
          logon
         </em>
         is the logon name for the administrator account. For instance, if you're trying to take over my account, you'd enter
         <strong>
          sudo chntpw -u Lincoln SAM
         </strong>
         . Remember, everything here is case sensitive, including the logon name.
        </p>
        <p>
         After you hit
         <strong>
          ENTER
         </strong>
         , a lot of text will scroll up very quickly. If you entered the text correctly, the last section will begin ". . . . User Edit Menu:"
        </p>
        <p>
         <figure class="image medium">
          <img alt="" height="122" src="http://images.pcworld.com/howto/graphics/206809-1011-end_original.jpg" width="350"/>
         </figure>
        </p>
        <p>
         Now press
         <strong>
          1
         </strong>
         (the number, not lower-case L), then
         <strong>
          ENTER
         </strong>
         .
        </p>
        <p>
         At the resulting question, press
         <strong>
          y
         </strong>
         .
        </p>
        <p>
         When the program is done, click the power button logo in the upper-right corner and reboot into Windows. The account in question will no longer require a password.
        </p>
        <p>
         Read the original
         <a href="http://forums.pcworld.com/index.php?showtopic=96486">
          forum discussion
         </a>
         .
        </p>

          PCW Answer Line forum

        </p>

function selectTSOAds(ads,max) {
	if (ads.length &lt;= max) {
		return ads;
	} else {
		var uniq = {};
		var found = 0;
		var selectedAds = [];
		while (found &lt; max) {
			var ad = ads[Math.floor(Math.random()*ads.length)];
			if (uniq[ad.id] != null) {
				continue;
			} else {
				uniq[ad.id] = true;
				found++;
				selectedAds.push(ad);
			}
		}
		return selectedAds;
	}
	return null;
