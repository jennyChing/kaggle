					&lt;div class="entry-content"&gt;
												
														
															&lt;img border="0" src="http://oubliettemagazine.com/wp-content/uploads/et_temp/padre-figlio2-18261_140x140.jpg" class='thumb alignleft'  alt='&#8220;Mo te lo spiego a pap&#8221; ed altri blogger: quando il pap si racconta in rete' /&gt;																											&lt;p class="date"&gt;&lt;span&gt;gen 4, 2012&lt;/span&gt;&lt;/p&gt;	
																																
										
						&lt;div style="float: left; margin-right: 10px;"&gt;&lt;/div&gt;&lt;blockquote&gt;&lt;p&gt;&lt;strong&gt;Daddy blogger, presenti e digitali quando pap si racconta in rete &#8211; di &lt;span style="color: #000000;"&gt;Alessia Manfredi&lt;/span&gt;&lt;/strong&gt;&lt;/p&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;&lt;/blockquote&gt;
&lt;p&gt;&lt;img class="alignright size-full wp-image-13289" title="padre-figlio" src="http://oubliettemagazine.com/wp-content/uploads/2012/01/padre-figlio2.jpg" alt="" width="261" height="299" /&gt;&lt;strong&gt;Dopo le mamme sul web,&lt;/strong&gt; arrivano i diari paterni in cui si racconta l&#8217;esperienza di essere genitori rigorosamente dal punto di vista maschile. Tra avventure tragicomiche, frustrazioni, consigli e molta dolcezza.&lt;/p&gt;
&lt;p&gt;&lt;strong&gt;Sanno tutto di cartoni animati e merendine.&lt;/strong&gt; Cambiano pannolini, dicono la loro su carrozzine e ovetti. Si giostrano con maestria tra phon e ciabattine negli spogliatoi della piscina, gestiscono nanne e pappe. Sono presenti, nel weekend come nel quotidiano, e rivendicando il loro posto al sole. O meglio, in rete: ecco i pap blogger, sbarcati sul web con il loro carico di &lt;strong&gt;racconti tragicomici&lt;/strong&gt;, riflessioni, frustrazioni e consigli. Certo, sono ancora una nicchia rispetto alle mamme in rete, fenomeno ormai planetario, che dalla cameretta hanno lanciato via pc i loro sfoghi sulla maternit al tempo del 2.0 fino a &lt;strong&gt;diventare una potente e corteggiata lobby.&lt;/strong&gt;&lt;/p&gt;
&lt;p&gt;Da tempo negli Usa anche i &lt;strong&gt;daddy blogger&lt;/strong&gt; hanno guadagnato stuoli di fedeli seguaci. Ora spuntano anche da noi. Qualche nome?Pap 2.0,Babbi Digitali,Babbo canguro,Daddy Supermaxieroe,&lt;span style="color: #000000;"&gt;&lt;strong&gt;Mo te lo spiego a pap&lt;/strong&gt;&lt;/span&gt;,Anche i pap hanno il pancione, per citarne alcuni.&lt;/p&gt;
&lt;p&gt;&lt;a href="http://oubliettemagazine.files.wordpress.com/2012/01/padre-e-figlio.jpg"&gt;&lt;img class="alignright size-medium wp-image-8177" title="padre-e-figlio" src="http://oubliettemagazine.files.wordpress.com/2012/01/padre-e-figlio.jpg?w=300" alt="" width="300" height="197" /&gt;&lt;/a&gt;&lt;strong&gt;Pionieri, che alla rete affidano pensieri ed opinioni su come si  genitori oggi,&lt;/strong&gt; rivelando molto di s. Tra aspirazioni, disavventure e momenti di tenerezza, con un punto di vista dichiaratamente da pap raccontano giornate che iniziano con risvegli convulsi, continuanocon gimkane fra ufficio, &lt;strong&gt;scuola e supermercato&lt;/strong&gt;, e si concludono con bimbi duracell che di andare a dormire proprio non ne vogliono sapere. Perch accanto alle mamme, ci sono anche loro, sempre pi spesso.&lt;/p&gt;
&lt;p&gt;&#8220;&lt;em&gt;Ho deciso di aprire un blog perch mi andava di raccontarmi, perch mi piace scrivere e perch volevo fissare da qualche parte alcuni momenti vissuti con i miei figli&lt;/em&gt;&#8220;:&lt;strong&gt; cos si presenta Pap 2.0.&lt;/strong&gt; Nella realt si chiama Andrea, ha 35 anni, lavora nell&#8217;online advertising e sta da anni incollato a Mamma 2.0, con cui ha due figli. Adolescente negli anni &#8217;90, maturo (?) &#8211; il punto interrogativo  tutto suo &#8211; nel nuovo millennio.&lt;/p&gt;
&lt;p&gt;&lt;a href="http://oubliettemagazine.files.wordpress.com/2012/01/padre-e-figlio-1.jpg"&gt;&lt;img class="alignright size-medium wp-image-8178" title="Padre-e-Figlio " src="http://oubliettemagazine.files.wordpress.com/2012/01/padre-e-figlio-1.jpg?w=300" alt="" width="300" height="186" /&gt;&lt;/a&gt;E in questi post c&#8217; un po&#8217; di tutto. &lt;strong&gt;Divagazioni da spunti di cronaca&lt;/strong&gt;, informazioni pratiche &#8211; da come donare il cordone ombelicale ai consultori di zona &#8211; a riflessioni che ci si aspetterebbero pi da una madre: la caterva di compiti di cui  investito un bambino di terza elementare, le ansie da broncospasmi e le fughe improvvise al pronto soccorso che, guarda un po&#8217;, provano emotivamente anche i pap. E molta dolcezza, cifra comune a queste &lt;strong&gt;confessioni paterne online&lt;/strong&gt;: come quel &#8220;sapore di famiglia&#8221; che rende felice l&#8217;autore diPensieri di un papquando i piccoli gli arrivano incontro la sera, al ritorno dal lavoro, o Daddysupermaxieroe &#8211; in omaggio a Ralph, supereroe molto umano della serie tv anni &#8217;80 &#8211; che si mette la tutina da&lt;em&gt;daddy&lt;/em&gt;&lt;em&gt;&lt;/em&gt;e cerca di &#8220;districarsi nei nodi della vita per sembrare agli occhi di mio figlio Scritch sempre un supermaxieroe&#8221;.&lt;/p&gt;
&lt;p&gt;&lt;a href="http://oubliettemagazine.files.wordpress.com/2012/01/padre_figlio_parigi_gr.jpg"&gt;&lt;img class="alignright size-medium wp-image-8179" title="padre_figlio" src="http://oubliettemagazine.files.wordpress.com/2012/01/padre_figlio_parigi_gr.jpg?w=300" alt="" width="300" height="202" /&gt;&lt;/a&gt;Un &lt;strong&gt;tratto rivendicato&lt;/strong&gt; con chiarezza: &#8220;Mentre le mamme hanno l&#8217;esigenza di farsi vedere nonsolomamme, i pap hanno quella di farsi vedere nonsololavoro&#8221;, rivelaPap express. ODesian, che quando va a prendere l&#8217;aperitivo con il suo &#8220;uomo piccolo&#8221;, se lo guarda, nota la zazzera che gli va da tutte le parti, che lo fa sembrare pi grande della sua et, e pensa che  davvero simpatico.&lt;/p&gt;
&lt;p&gt;Nuovi pap rivoluzionari? Negli Usa, in un recente articolo ripreso anche dall&#8217;&lt;em&gt;Huffington Pos&lt;/em&gt;tNathan Hegedus, lui stesso autore di dispacci in rete da &#8220;daddyland&#8221;, si dice convinto che i&lt;strong&gt;&lt;em&gt;daddy blogger&lt;/em&gt;&lt;/strong&gt;siano i pi importanti femministi online, quelli che stanno creando un nuovo paradigma di paternit, che non rovesceranno mai sulle donne il carico di cura dei figli e della casa.&lt;/p&gt;
&lt;p&gt;E da noi? &#8220;Si tratta ancora di un fenomeno che riguarda una fascia elitaria della popolazione, un&#8217;avanguardia illuminata&#8221;, spiega Paolo Maria Ferri, esperto di cultura del web e autore di &#8220;Nativi Digitali&#8221;, che all&#8217;universit di Milano Bicocca insegna teoria e tecnica dei nuovi media. La mamma detta ancora legge, anche online, ma i pap stanno recuperando terreno.&lt;/p&gt;
&lt;p&gt;&lt;a href="http://oubliettemagazine.files.wordpress.com/2012/01/padre-gioca-con-i-figli-bambini_650x447.jpg"&gt;&lt;img class="alignright size-medium wp-image-8180" title="padre-gioca-con-i-figli" src="http://oubliettemagazine.files.wordpress.com/2012/01/padre-gioca-con-i-figli-bambini_650x447.jpg?w=300" alt="" width="300" height="206" /&gt;&lt;/a&gt;&#8220;In giro, si trovano ovunque opinioni e punti di vista al femminile sull&#8217;essere genitori&#8221;, sbotta &lt;span style="color: #000000;"&gt;&lt;strong&gt;Francesco Uccello&lt;/strong&gt;&lt;/span&gt;, di&lt;span style="color: #000000;"&gt;&lt;strong&gt;&lt;em&gt;Mo te lo spiego a pap&lt;/em&gt;&lt;/strong&gt;&lt;/span&gt;. Campano, due figli di 4 anni e mezzo e 3, vive con Mps (non la banca, ma &#8220;mia principessa Stefania&#8221;) e si occupa di comunicazione per ragazzi. E&#8217; in rete col suo blog perch &#8220;tra stereotipi di maschi tutto muscoli, veline e lavoro,  difficile trovare un canale adatto per ospitare il punto di vista di un pap&#8221;, racconta.&lt;/p&gt;
&lt;p&gt;I suoi post da una&lt;strong&gt; famiglia mammocentrica&lt;/strong&gt; della periferia est di una grande citt &#8211; dove &#8220;quando usciamo tutti vogliono tenere la mano alla mamma&#8221; e &#8220;l&#8217;essere un bravo pap  direttamente proporzionale a quante cose riesci a portare&#8221; &#8211; raccontano di fatiche quotidiane, intoppi continui e momenti indimenticabili con i figli.&lt;br /&gt;
Per lui tutto  nato un po&#8217; per gioco, dalla passione per la scrittura alle situazioni limite quotidiane, per riderci sopra. &lt;strong&gt;E per rispondere alle domande scomode che fanno i figli,&lt;/strong&gt; da &#8220;cos&#8217; il ciclo&#8221; a &#8220;come si fanno i bambini&#8221;. &#8220;Certi modelli sono duri a morire, ammette, ma le cose stanno cambiando: si sta cominciando a credere che anche un pap sia capace di amore &#8216;materno&#8217;&#8221;, dice.&lt;/p&gt;
&lt;p&gt;Di sicuro, il mezzo aiuta ad eliminare i filtri: &#8220;&lt;em&gt;Internet  ormai vissuto come uno strumento strutturante della propria personalit. E&#8217; un modo sempre pi normale di manifestare s stessi, compresa la sfera intima dei sentimenti, specialmente per i pi giovani&lt;/em&gt;&#8220;. Pi si scende con l&#8217;et &#8211; e qualcuno si sente ancora bambino,Pater Puerdocet &#8211; &#8220;&lt;em&gt;pi Internet diventa una protesi usuale per raccontare anche aspetti molto personali&lt;/em&gt;&#8220;, chiarisce il professor Ferri.&lt;/p&gt;
&lt;p&gt;&lt;strong&gt;Attenzione, per, a parlare di rivoluzione:&lt;/strong&gt; &#8220;&lt;em&gt;Se si escludono i pap separati, la cura dei figli in Italia  ancora in carico alle madri per l&#8217;80 per cento&lt;/em&gt;&#8220;, aggiunge il professore. Detto questo,  vero che una rivendicazione del ruolo svolto nei giovani padri  evidente, ed  una tendenza destinata a rafforzarsi: &#8220;&lt;em&gt;l&#8217;Italia, come altri paesi latini,  ancora agli albori&lt;/em&gt;&#8220;. Ma il tempo gioca a favore: &#8220;&lt;em&gt;Quando diventeranno padri gli attuali venticinquenni, il cambiamento sar ancora pi evidente&lt;/em&gt;&#8220;, conclude Ferri.&lt;/p&gt;

								event: 'injectStyles'
						};

							fontFamily: jQuery( '.sd-text-color' ).css( 'font-family' ),
							fontSize: jQuery( '.sd-text-color' ).css( 'font-size' ),

							textDecoration: jQuery( '.sd-text-color' ).css('text-decoration')

							fontSize: jQuery( '.sd-link-color' ).css('font-size'),
							textDecoration: jQuery( '.sd-link-color' ).css('text-decoration'),
							fontWeight: jQuery( '.sd-link-color' ).css( 'font-weight' ),
							fontStyle: jQuery( '.sd-link-color' ).css( 'font-style' )
						};

						JetpackLikespostMessage( stylesData, window.frames[ 'likes-master' ] );

						var requests = [];
						jQuery( '.jetpack-likes-widget-wrapper' ).each( function( i ) {
							var regex = /like-(post|comment)-wrapper-(\d+)-(\d+)-(\w+)/;
							var match = regex.exec( this.id );
							if ( ! match || match.length != 5 )
								return;

							var info = {
								blog_id: match[2],
								width:   this.width
							};

							if ( 'post' == match[1] ) {
								info.post_id = match[3];
							} else if ( 'comment' == match[1] ) {
								info.comment_id = match[3];
							}

							info.obj_id = match[4];

							requests.push( info );
						});

						JetpackLikespostMessage( { event: 'initialBatch', requests: requests }, window.frames['likes-master'] );

						jQuery( document ).on( 'inview', 'div.jetpack-likes-widget-unloaded', function() {
							jetpackLikesWidgetQueue.push( this.id );

			function JetpackLikesWidgetQueueHandler() {
				var wrapperID;
				if ( ! jetpackLikesMasterReady ) {
					setTimeout( JetpackLikesWidgetQueueHandler, 500 );
					return;
				}

				if ( jetpackLikesWidgetQueue.length &gt; 0 ) {
					// We may have a widget that needs creating now
					var found = false;
					while( jetpackLikesWidgetQueue.length &gt; 0 ) {
						// Grab the first member of the queue that isn't already loading.
						wrapperID = jetpackLikesWidgetQueue.splice( 0, 1 )[0];
						if ( jQuery( '#' + wrapperID ).hasClass( 'jetpack-likes-widget-unloaded' ) ) {
							found = true;
							break;
						}
					}
					if ( ! found ) {
						setTimeout( JetpackLikesWidgetQueueHandler, 500 );
						return;
					}
				} else if ( jQuery( 'div.jetpack-likes-widget-unloaded' ).length &gt; 0 ) {
					// Get the next unloaded widget
					wrapperID = jQuery( 'div.jetpack-likes-widget-unloaded' ).first()[0].id;
					if ( ! wrapperID ) {
						// Everything is currently loaded
						setTimeout( JetpackLikesWidgetQueueHandler, 500 );
						return;
					}
				}

				var $wrapper = jQuery( '#' + wrapperID );
				$wrapper.find( 'iframe' ).remove();

				if ( $wrapper.hasClass( 'slim-likes-widget' ) ) {
					$wrapper.find( '.post-likes-widget-placeholder' ).after( "&lt;iframe class='post-likes-widget jetpack-likes-widget' name='" + $wrapper.data( 'name' ) + "' height='22px' width='68px' frameBorder='0' scrolling='no' src='" + $wrapper.data( 'src' ) + "'&gt;&lt;/iframe&gt;" );
				} else {
					$wrapper.find( '.post-likes-widget-placeholder' ).after( "&lt;iframe class='post-likes-widget jetpack-likes-widget' name='" + $wrapper.data( 'name' ) + "' height='55px' width='100%' frameBorder='0' src='" + $wrapper.data( 'src' ) + "'&gt;&lt;/iframe&gt;" );
				}

				$wrapper.removeClass( 'jetpack-likes-widget-unloaded' ).addClass( 'jetpack-likes-widget-loading' );

				$wrapper.find( 'iframe' ).load( function( e ) {
					var $iframe = jQuery( e.target );
					$wrapper.removeClass( 'jetpack-likes-widget-loading' ).addClass( 'jetpack-likes-widget-loaded' );

					JetpackLikespostMessage( { event: 'loadLikeWidget', name: $iframe.attr( 'name' ), width: $iframe.width() }, window.frames[ 'likes-master' ] );

					if ( $wrapper.hasClass( 'slim-likes-widget' ) ) {
						$wrapper.find( 'iframe' ).Jetpack( 'resizeable' );
					}
				});
				setTimeout( JetpackLikesWidgetQueueHandler, 250 );
			}
