              <div class="post-meta">

              <p style="text-align: right;">
               <em>
                We attended a press event as a guest of Disney
               </em>
              </p>
              <div class="wp-caption aligncenter" id="attachment_33333" style="width: 946px">
               <img alt="Photo courtesy of Disney" class="size-full wp-image-33333" height="624" src="http://ocmomblog.com/wp-content/uploads/2014/09/bigHero6537e1f98d488d.jpg?30ce8b" width="936"/>
               <p class="wp-caption-text">
                Photo courtesy of Disney
               </p>
              </div>
              <p>
               Who is Baymax? Just like Anna and Elsa from Frozen, Baymax will soon to become a household name in homes across America when the all-new Disney Film Big Hero 6 comes out in theaters on November 7th. Last month, we got an opportunity to be one of the first to learn about Baymax and see a short clip from the Disney film Big Hero 6.
              </p>
              <div class="wp-caption aligncenter" id="attachment_33334" style="width: 946px">
               <img alt="Photo courtesy of Disney" class="size-full wp-image-33334" height="624" src="http://ocmomblog.com/wp-content/uploads/2014/09/bigHero653f63c079b93e.jpg?30ce8b" width="936"/>
               <p class="wp-caption-text">
                Photo courtesy of Disney
               </p>
              </div>
              <p>
               Baymax is an inflatable robot who is programmed to be a Personal Healthcare Companion that was built by one of the boys in the film, Tadashi Hamada. Baymax loves to help people and sees Tadashis brother, Hiro as his patient.
              </p>
              <div class="wp-caption aligncenter" id="attachment_33332" style="width: 946px">
               <img alt="Photo courtesy of Disney" class="size-full wp-image-33332" height="624" src="http://ocmomblog.com/wp-content/uploads/2014/09/bigHero6537e1f9ae3757.jpg?30ce8b" width="936"/>
               <p class="wp-caption-text">
                Photo courtesy of Disney
               </p>
              </div>
              <p>
               Hiro transforms Baymax into Big Hero 6 complete with a rocket fist, super strength, and rocket thrusters so he can fly. That is when their adventure together begins in the heartwarming action packed superhero animated film Big Hero 6.
              </p>
              <div class="wp-caption aligncenter" id="attachment_33328" style="width: 970px">
               <img alt="Photo courtesy of Disney" class="size-full wp-image-33328" height="402" src="http://ocmomblog.com/wp-content/uploads/2014/09/Meet-Baymax-1.jpg?30ce8b" width="960"/>
               <p class="wp-caption-text">
                Photo courtesy of Disney
               </p>
              </div>
              <p>
               The film is inspired by the Marvel comic series and is filled with a lot of heart and humor. Your children will go on an epic adventure that unfolds in the futuristic town of San Fransokyo.
              </p>
              <div class="wp-caption aligncenter" id="attachment_33331" style="width: 946px">
               <img alt="Photo courtesy of Disney" class="size-full wp-image-33331" height="624" src="http://ocmomblog.com/wp-content/uploads/2014/09/bigHero651e697072eafb.jpg?30ce8b" width="936"/>
               <p class="wp-caption-text">

	submit.disabled = !isValid;
	submit.className = isValid ? "submit" : "disabled";

	return isValid;
	}

	function emailErrorMessage(input, allDivs) {
	if(input.getAttribute("data-webform-type") == "iframe") {

	} else {

	}

	function fieldErrorMessage(input, allDivs) {

	input.className = "required invalid";
	input.placeholder = input.getAttribute("data-required-message") || "This field is required";
	} else {
	for(var i = 0; i &lt; allDivs.length; ++i) {
	var element = allDivs[i];
	if(element.className.indexOf("mimi_field_feedback") &gt;= 0) {
	return element.innerHTML = input.getAttribute("data-required-message") || "This field is required";
	}
	}
	}
	}

	function removeErrorMessage(input, allDivs) {
	if(input.getAttribute("data-webform-type") == "iframe") {
	input.className = "required";
	}

	for(var i = 0; i &lt; allDivs.length; ++i) {
	var element = allDivs[i];
	if(element.className.indexOf("mimi_field_feedback") &gt;= 0) {
	return element.innerHTML = "";
	}
	}
	}
