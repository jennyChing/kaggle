    Black Perl
   </h1>

    <div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en">
     <p>
      <b>
       Black Perl
      </b>
      is a famous piece of
      <a href="/wiki/Perl#Perl_pastimes" title="Perl">
       Perl poetry
      </a>
      . It was posted to
      <a href="/wiki/Usenet" title="Usenet">
       Usenet
      </a>
      on April 1, 1990. It is written in
      <a href="/wiki/Perl" title="Perl">
       Perl
      </a>
      3 and will not parse under Perl 5. Multiple independent updates to Black Perl to make it parsable in Perl 5 have been published.
      <sup class="reference" id="cite_ref-1">
       <a href="#cite_note-1">
        <span>
         [
        </span>
        1
        <span>
         ]
        </span>
       </a>
      </sup>
      <sup class="reference" id="cite_ref-2">
       <a href="#cite_note-2">
        <span>
         [
        </span>
        2
        <span>
         ]
        </span>
       </a>
      </sup>
      The full text of the poem is reproduced below.
     </p>

     </p>

     <h2>
      <span class="mw-headline" id="Attribution">
       Attribution
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Black_Perl&amp;action=edit&amp;section=1" title="Edit section: Attribution">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      While the poem itself is signed
      <i>
       Larry Wall
      </i>
      , the original message was posted with forged message headers, causing uncertainty of authorship. Sharon Rauenzahn, born Hopkins, has been suspected, but has denied authorship.
      <sup class="reference" id="cite_ref-3">
       <a href="#cite_note-3">
        <span>
         [
        </span>
        3
        <span>
         ]
        </span>
       </a>
      </sup>
      <a class="mw-redirect" href="/wiki/Randal_Schwartz" title="Randal Schwartz">
       Randal Schwartz
      </a>
      has claimed that
      <a href="/wiki/Larry_Wall" title="Larry Wall">
       Larry Wall
      </a>
      is in fact the author,
      <sup class="reference" id="cite_ref-4">
       <a href="#cite_note-4">
        <span>
         [
        </span>
        4
        <span>
         ]
        </span>
       </a>
      </sup>
      and later on Larry Wall confirmed
      <sup class="reference" id="cite_ref-5">
       <a href="#cite_note-5">
        <span>
         [
        </span>
        5
        <span>
         ]
        </span>
       </a>
      </sup>
      that claim.
     </p>
     <h2>
      <span class="mw-headline" id="Result_of_program_execution">
       Result of program execution
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Black_Perl&amp;action=edit&amp;section=2" title="Edit section: Result of program execution">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      When executed, Black Perl exits on line one, upon reaching the function
      <a class="external text" href="http://perldoc.perl.org/functions/exit.html" rel="nofollow">
       exit
      </a>
      . The remaining lines are parsed by the Perl interpreter but never actually executed. The program produces no output.
     </p>
     <h2>
      <span class="mw-headline" id=".22Black_Perl.22">
       "Black Perl"
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Black_Perl&amp;action=edit&amp;section=3" title='Edit section: "Black Perl"'>
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <div class="mw-highlight">
      <pre>
       BEFOREHAND: close door, each window &amp; exit; wait until time.
    open spellbook, study, read (scan, select, tell us);
write it, print the hex while each watches,
    reverse its length, write again;
    kill spiders, pop them, chop, split, kill them.
        unlink arms, shift, wait &amp; listen (listening, wait),
sort the flock (then, warn the "goats" &amp; kill the "sheep");
    kill them, dump qualms, shift moralities,
    values aside, each one;
        die sheep! die to reverse the system
        you accept (reject, respect);
next step,
    kill the next sacrifice, each sacrifice,
    wait, redo ritual until "all the spirits are pleased";
    do it ("as they say").
do it(*everyone***must***participate***in***forbidden**s*e*x*).
return last victim; package body;
    exit crypt (time, times &amp; "half a time") &amp; close it,
    select (quickly) &amp; warn your next victim;
AFTERWARDS: tell nobody.
    wait, wait until time;
    wait until next year, next decade;
        sleep, sleep, die yourself,
        die at last
# Larry Wall
      </pre>
     </div>
