        <h2>
         Diabeto System
        </h2>
        <p>
         Diabeto is a combination of,
        </p>
        <p>
         1. Diabeto hardware which transmits readings from the Glucometer into smartphone.*
         <br/>
        </p>
        <p>
         2. A smartphone application which gives detailed trends and insights about your Diabetes. These readings are securely pushed onto a secure cloud server.
         <br/>
        </p>
        <p>
         3. A web application/interface which pulls the readings from the cloud server and give detailed insights similar to the Diabeto application. The web application is useful for remote monitoring by parents and healthcare practitioners.
         <br/>
        </p>
        <p>
         4. A pebble watch display for remote monitoring The readings are pulled from the Diabeto cloud server thus making your data easily accessible from anywhere.
        </p>
        <p>
         *
         <i>
          In case of Accuchek Glucometer users, one needs to buy another hardware Infra red jacket called Diabeto IR Jacket

         Easy
        </h2>

         <br/>
        </p>
