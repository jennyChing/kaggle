         <p>
          Mozilla Firefox 7 has been released. The new release fixes some stability issues and several bug fixes. Mozilla Firefox 7 available to Download for Linux, MAC OS X, and Microsoft windows. Also available to install on Ubuntu Natty, Oneiric using PPA.
          <span id="more-3716">
          </span>
         </p>
         <p>
          According to Mozilla blog post that firefox 7 and upcoming releases willsignificantlyreduce the memory usage by 50% less than earlier versions of firefox, better performance for handling HTML 5 animations, and it will become much better environment for developers, Firefox 7 supports W3C navigation timing spec API will help developerto easily measure the page load time, optimize websites, and web apps.
         </p>
         <p>
          <a href="http://www.linuxnov.com/wp-content/uploads/2011/09/average2-memory-usage-firefox-7-vs-firefox-8.png">
           <img alt="" class="alignnone size-full wp-image-3722" height="166" src="http://www.linuxnov.com/wp-content/uploads/2011/09/average2-memory-usage-firefox-7-vs-firefox-8.png" title="average2-memory-usage-firefox-7-vs-firefox-8" width="236"/>
          </a>
          <a href="http://www.linuxnov.com/wp-content/uploads/2011/09/peak-memory-usage-firefox-7-vs-firefox-8.png">
           <img alt="" class="alignnone size-full wp-image-3723" height="165" src="http://www.linuxnov.com/wp-content/uploads/2011/09/peak-memory-usage-firefox-7-vs-firefox-8.png" title="peak-memory-usage-firefox-7-vs-firefox-8" width="236"/>
          </a>
         </p>
         <p>
          Heres a Benchmark 
          <strong>
           <a href="https://blog.mozilla.com/nnethercote/2011/08/09/firefox-7-is-lean-and-fast-2/" target="_blank">
            Firefox 7 is lean and fast
           </a>
          </strong>
          for Firefox 7 vs older version Firefox 6 and development version Firefox 8.
         </p>
         <h2>
          <span style="text-decoration: underline;">
           Whats New In Mozilla Firefox 7 Changelog
          </span>
         </h2>
         <ul>
          <li>
           Drastically improved
           <strong>
            <a href="http://hacks.mozilla.org/2011/09/firefox-7-is-lean-and-fast/">
             memory handling
            </a>
           </strong>
           for certain use cases
          </li>
          <li>
           Added a new
           <strong>
            <a href="http://www.basschouten.com/blog1.php">
             rendering backend
            </a>
           </strong>
           to speed up Canvas operations on Windows systems
          </li>
          <li>
           Bookmark and password changes now sync almost instantly when using Firefox Sync
          </li>
          <li>
           The http:// URL prefix is now hidden by default
          </li>
          <li>
           Added support for
           <strong>
            <a href="https://developer.mozilla.org/En/CSS/Text-overflow">
             text-overflow: ellipsis
            </a>
           </strong>
          </li>
          <li>
           Added support for the
           <strong>
            <a href="https://dvcs.w3.org/hg/webperf/raw-file/tip/specs/NavigationTiming/Overview.html">
             Web Timing specification
            </a>
           </strong>
          </li>
          <li>
           Enhanced support for
           <strong>
            <a href="https://developer.mozilla.org/en/Firefox_7_for_developers#MathML">
             MathML
            </a>
           </strong>
          </li>
          <li>
           The WebSocket protocol has been updated from version 7 to version 8
          </li>
          <li>
           Added an
           <strong>
            <a href="http://weblogs.mozillazine.org/asa/archives/2011/07/telemetry_in_firefox.html">
             opt-in system
            </a>
           </strong>
           for users to send performance data back to Mozilla to improve future versions of Firefox
          </li>
          <li>
           Fixed several stability issues
          </li>
          <li>
           Fixed several
           <strong>
            <a href="http://www.mozilla.org/security/known-vulnerabilities/firefox.html#firefox7">
             security issues
            </a>
           </strong>
          </li>
         </ul>

         <h2>
          <span style="text-decoration: underline;">
           Install Mozilla Firefox 7 On Ubuntu 11.04 Natty Narwhal and Ubuntu 11.10 Oneiric Ocelot
          </span>
