               <div class="entry-title">
                Wat is embedden en hoe doe je dat?
               </div>
               <div class="entry-body">
                <div class="entry-description">
                 Sinds april 2014 weten we dat embedden in de Europese Unie is toegestaan.
Embedden deden we al langer, maar lang leek dat niet te mogen en hebben veel website-eigenaren schadevergoedingen moeten betalen aan onder meer fotografen, voor het embedden van materiaal.  
Het eerste artikel in een serie van drie: wat is embedden en hoe doe je dat?
                </div>
                <p>
                 <strong>
                  Wat is embedden?
                 </strong>
                 <br/>
                 Embedden is een techniek waarmee content van website A ook te zien zal zijn op website B, zonder dat de content op website B geupload hoeft te worden. De content staat dus alleen maar op website A. Toch lijkt het, alsof de content ook op website B te zien is.
                </p>
                <p>
                 De bezoeker van de website, hoeft de website dus niet te verlaten om de content te kunnen zien. Hiermee kan je gratis content van een ander gebruiken, waarbij je ook geen extra ruimte op je website nodig hebt. Het kan daarom erg interessant zijn om deze techniek in te zetten om content van een ander te gebruiken.
                 <br/>
                 <br/>
                 <strong>
                  Hoe moet je embedden?
                 </strong>
                 <br/>
                 Embedden doe je door via html-code content van een andere website in te sluiten. Dat gaat het makkelijkst met video, afbeeldingen en muziek. Veel sociale media maken het bijvoorbeeld mogelijk om te embedden en zorgen zelf al voor de juiste code.
                 <br/>
                 <br/>
                 <strong>
                  Social Media
                 </strong>
                 <br/>
                 <a href="https://www.youtube.com/?hl=nl&amp;gl=NL" target="_blank">
                  YouTube
                 </a>
                 geeft onder het woord insluiten de link die alleen maar gekopieerd en weer in je eigen website geplakt hoeft te worden, om filmpjes op de website te kunnen tonen.
                 <br/>
                 <br/>
                 <a href="http://www.slideshare.net">
                  SlideShare
                 </a>
                 geeft, wanneer je op het
                 <strong>
                  &lt;&gt;
                 </strong>
                 knopje klikt, ook een embedcode, te herkennen aan het woord 'iframe', waarmee een presentatie op de website getoond kan worden.
                 <br/>
                 <br/>
                 <a href="https://flickr.com" target="_blank">
                  Flickr
                 </a>
                 , de fotosite, maakt ook mogelijk dat fotos embed kunnen worden. Onder het pijltje dat naar rechts wijst, kan je weer de embedcode vinden.
                 <br/>
                 <br/>
                 <strong>
                  Zelf embedden
                 </strong>
                 <br/>
                 Embedden kan echter ook, zonder dat de website of het sociale medium daar zelf de juiste code voor geeft. Zo kan je bijvoorbeeld fotos van krantenwebsites embedden. Klik op de foto, zodat deze in een eigen venster opent. In de adresbalk van de browser verschijnt de url van de betreffende foto. Kopieer de url.
                 <br/>
                 Zet er
                 <strong>
                  &lt;embed src=
                 </strong>
                 voor en plaats er
                 <strong>
                  &gt;
                 </strong>
                 achter. Nu verschijnt de foto op je website.
                 <br/>
                 <br/>
                 <strong>
                  Naamsvermelding
                 </strong>
                 <br/>
                 Bij embedden is naamsvermelding niet meer nodig. Je mag echter niet om die toch al voor iedereen toegankelijk was. Is de content alleen toegankelijk met, bijvoorbeeld, een toegangscode, mag je hier niet omheen werken.
                 <br/>
                 <br/>
                 <em>
                  In het volgende artikel over embedden: Hoe mag u embedden voor uw onderneming
                  <br/>
                  inzetten?
                 </em>
                </p>
