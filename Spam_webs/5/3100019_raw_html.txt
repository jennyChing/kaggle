       <p class="google_elide">
        The Federal Communications Commission chairman's first foray into social media bombed today as Julius Genachowskiset to take questions via a Twitter chat at 1:30 p.m. for a half hourfailed to show for the first 20 minutes.
       </p>
       <p class="google_elide">
        Apparently, Genachowski wasn't tagging his answers with the #askjulius hashtag that participants were using to send him questions.
       </p>
       <p class="google_elide">
        As a result, the discussion, at least for those following the #askjulius hashtag, devolved into a lot of snark.
       </p>
       <script charset="utf-8" src="//platform.twitter.com/widgets.js">
       </script>
       <p class="google_elide">
        As one tweeter noted, it was a rookie mistake. Genachowski can be forgiven, but it's hard to believe there aren't any social media geniuses at the FCC who could have made sure the chairman of the agency in charge of the nation's airwaves and telecommunications didn't blow it.
       </p>
       <p class="google_elide">
        Bottom line? For an agency emphasizing policies to expand Internet broadband and find more wireless spectrum to meet rising demand, this just didn't look good.
       </p>
       <p class="google_elide">
        Not only did the FCC appear Twitter illiterate, but the agency made it worse by trying to convince other reporters not to write about the great Twitter chat debacle. Instead of a simple "mea culpa" or "we learned our lesson and we'll do better next time," the FCC tried to cover it up. And in Washington, cover-ups rarely work.
       </p>
       <p class="google_elide">
        Even better, sources said Genachowski was actually at Twitter headquarters during his Twitter fail.
       </p>
       <p class="google_elide">
        Here's a sampling of the Twitter conversation, which never really got off the ground.

    function _formatCounter(value) {
      value = (isNaN(value)) ? 0 : parseInt(value);

      if (value &gt; 1000 &amp;&amp; value &lt; 10000) {
        count = parseFloat(value/1000);
        count = (count).toFixed(1);
        return count + 'k';
      } else if (value &gt;= 10000) {
        count = parseInt(value/1000);
        return count + 'k';
      }
      else {
        return value;
      }
    }

    // determine the top and left for the share bar based on calculations involving different divs and the screen
    function reposition() {
      var element = $("#shell #main .left-share-bar-wrapper");
      var offset = element.offset();

      if($("#shell #main .node-headerimage").length &gt; 0) {
        var headerImage = $("#shell #main .node-headerimage");
      } else if($("#shell #main #adfreakad").length &gt; 0) {
        var headerImage = $("#shell #main #adfreakad");
      } else if($("#shell #main .views-field-field-sa-section-header-fid").length &gt; 0) {
        var headerImage = $("#shell #main .views-field-field-sa-section-header-fid");
      }

      if($("#shell #main .subheader .headings").length &gt; 0) {
        var header = $("#shell #main .subheader .headings");
      } else if($("#shell #main .headline").length &gt; 0) {
        var header = $("#shell #main .headline");
      }

      var headerOffset = header.offset();

      if($("#shell #main .article-footer").length &gt; 0) {
        var footer = $("#shell #main .article-footer");
      } else if($("#shell #main .cols.colsAB .mod.latest-news-popular-now").length &gt; 0) {
        var footer = $("#shell #main .mod.latest-news-popular-now");
      } else if($("#shell #main .cols.colsAB #comments").length &gt; 0) {
        var footer = $("#shell #main #comments");
      } else if($(".node-type-sa-article").length &gt; 0) {
        var footer = $("#shell #pre-footer");
      }

      footerOffset = footer.offset();

      if($(window).scrollTop() &gt;= headerOffset.top + header.outerHeight(true) + 3 &amp;&amp; $(window).scrollTop() &lt; footerOffset.top - element.outerHeight(true) + 2) {
        // we are below the header or above the footer; put the social icons on top
        // the positioning should be left: -82px from the header offset
        // = headerOffset.left - 82px;
        element.css({
            position: "fixed",
            top: 0,
            left: headerOffset.left - 82 + "px"
        });
      } 
      else if($(window).scrollTop() &gt;= footerOffset.top - element.outerHeight(true) + 2) {
        if($("#shell #main #adfreakad").length &gt; 0) {
          element.css({
            position: "absolute",
            top: headerImage.outerHeight(true) + $("#shell #main .inner #post").outerHeight(true) + $("#shell #main .mod.mod-related").outerHeight(true) - element.outerHeight(true) + 20,
            left: "-82px"
          });
        } 
        else if($("#shell #main .views-field-field-sa-section-header-fid").length &gt; 0) {
          element.css({
            position: "absolute",
            top: $("#shell #main .inner").outerHeight(true) - element.outerHeight(true) + 2,
            left: "-82px"
          });
        } 
        else {
          element.css({
            position: "absolute",
            top: $("#shell #main .inner").outerHeight(true) - footer.outerHeight(true) - element.outerHeight(true) + 2,
            left: "-82px"
          });
        }
      } 
      else {
        // the "reset" position is absolute -82px
        if(typeof headerImage != 'undefined') {
          if($("#shell #main .views-field-field-sa-section-header-fid").length &gt; 0) {
            element.css({
              position: "absolute",
              top: headerImage.outerHeight(true) + header.outerHeight(true) + 25 + "px",

            element.css({

              top: header.outerHeight(true) + 13 + "px",
              left: "-82px"
            });
          }
        }
      }

      <p>
       The Federal Communications Commission chairman's first foray into social media bombed today as Julius Genachowskiset to take questions via a Twitter chat at 1:30 p.m. for a half hourfailed to show for the first 20 minutes.
      </p>
      <p>
       Apparently, Genachowski wasn't tagging his answers with the #askjulius hashtag that participants were using to send him questions.
      </p>
      <p>
       As a result, the discussion, at least for those following the #askjulius hashtag, devolved into a lot of snark.
      </p>
      <script charset="utf-8" src="//platform.twitter.com/widgets.js">
      </script>
      <p>
       As one tweeter noted, it was a rookie mistake. Genachowski can be forgiven, but it's hard to believe there aren't any social media geniuses at the FCC who could have made sure the chairman of the agency in charge of the nation's airwaves and telecommunications didn't blow it.
      </p>
      <p>
       Bottom line? For an agency emphasizing policies to expand Internet broadband and find more wireless spectrum to meet rising demand, this just didn't look good.
      </p>
      <p>
       Not only did the FCC appear Twitter illiterate, but the agency made it worse by trying to convince other reporters not to write about the great Twitter chat debacle. Instead of a simple "mea culpa" or "we learned our lesson and we'll do better next time," the FCC tried to cover it up. And in Washington, cover-ups rarely work.
      </p>
      <p>
       Even better, sources said Genachowski was actually at Twitter headquarters during his Twitter fail.
      </p>
