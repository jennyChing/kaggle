    <h2>
     PHP Cross ReferenceLibrary
    </h2>
    <p>
     <a href="/home/use-the-source" title="Use the Source">
      PHPCrossRef
     </a>
     features a
     <a href="/home/php-open-source-code-library" title="PHP Open Source Code Library">
      library
     </a>
     of cross-referenced open source
     <a href="/home/php-hypertext-preprocessor" title="PHP: Hypertext Preprocessor">
      <span class="caps">
       PHP
      </span>
     </a>
     projects. The library is a look inside the
     <span class="caps">
      PHP
     </span>
     code that powers many a website, just like
     <a href="/xref/textpattern" title="Textpattern">
      this one
     </a>
     . The cross-referenced code is an excellent learning tool for the budding
     <span class="caps">
      PHP
     </span>
     programmer, as well as a convenient resource for the professional web developer.
    </p>

    <p>
     <strong>
      <em>
       Cross-Reference Features
      </em>
     </strong>
    </p>
    <ul>
     <li>
      Mouse-over information for classes and functions in the source view.
     </li>
     <li>
      Hot-jump to the source of any class/function definition.
     </li>
     <li>
      Instant lookup of classes, functions, variables and constants by name.
     </li>
     <li>
      Link to native
      <span class="caps">
       PHP
      </span>
      function documentation at the main
      <a href="http://www.php.net/docs.php">
       <span class="caps">
        PHP
       </span>
       .net
      </a>
      site.
     </li>
     <li>
      Search/lookup history, cookies must be enabled.
     </li>
     <li>
      Project directory explorer, can be toggled on/off.
     </li>
     <li>
      Text view option on source files, easier for copy/pasting.
     </li>
     <li>
      Project statistics, files/lines/variables/functions/classes/constants.
     </li>
    </ul>
