             <section class="article-content">
              <p>
               If you're a programmer, these are good times. Jobs in the segment are projected to grow
               <a href="http://www.bls.gov/ooh/computer-and-information-technology/computer-programmers.htm" target="_blank">
                8%
               </a>
               over the next seven years, according to the U.S. Bureau of Labor Statistics. If you're a hotshot coder, you can make up to $300 an hour or more.
              </p>
              <p>
               Those at the high end of the pay scale have mastered the languages that are most in demand. Which are those? We asked Doug Winnie, director of content for online learning platform
               <a href="http://www.lynda.com/" target="_blank">
                Lynda
               </a>
               . Here's his assessment:
              </p>
              <div class="see-also">
               <p>
                See also:
                <a data-crackerjax="#post-slider" href="http://mashable.com/2014/04/30/programming-sucks/">
                 Programming Isn't Manual Labor, But It Still Sucks
                </a>
               </p>
              </div>
              <h2>
               1. Java
              </h2>
              <figure class="image">
               <p>
                <img alt="java2" class="" data-caption="" data-credit-label="" data-credit-link="" data-credit-name="Mashable Composite/Wikimedia" data-credit-provider="Custom type" data-fragment="m!0062" data-image="http://rack.2.mshcdn.com/media/ZgkyMDE1LzAxLzE2LzE1L2phdmEyLmU3MTk3LmpwZwpwCXRodW1iCTEyMDB4OTYwMD4/4a96eae5/53c/java2.jpg" data-micro="1" data-width="640" src="http://rack.2.mshcdn.com/media/ZgkyMDE1LzAxLzE2LzE1L2phdmEyLmU3MTk3LmpwZwpwCXRodW1iCTEyMDB4OTYwMD4/4a96eae5/53c/java2.jpg" title="java2"/>
               </p>
               <div class="image-credit">
                <p>
                 Image:  Mashable Composite/Wikimedia
                </p>
               </div>
              </figure>
              <p>
               Java is one of the most popular languages for building back-ends for modern enterprise-web applications. With Java and frameworks based on it, web developers can build scalable web apps for a variety of users. Java is also the main language used to develop native Android apps for smartphones and tablets.
              </p>
              <h2>
               2. JavaScript
              </h2>
              <p>
               Every modern website uses JavaScript. Its the go-to language if you want to create interactivity for your site, or build user interfaces with one of the dozens of popular JavaScript frameworks.
              </p>
              <h2>
               3.C#
              </h2>
              <p>
               C# is the primary language for developing on Microsoft platforms and services. Whether youre building modern web applications using Azure and .NET, apps for Windows devices or powerful desktop apps for your business, C# is the quickest way to harness all that Microsoft has to offer. Want to play, as well? The popular Unity game development engine also uses C# as one of its primary languages.
              </p>
              <h2>
               4. PHP
              </h2>
              <p>
               Building a web app that needs to work with data? PHP, along with databases like MySQL, are essential tools for building modern web applications. PHP powers a majority of todays data-driven websites, and is the foundation technology for powerful content management systems, like WordPress, which you can extend to make your site more powerful.
              </p>
              <h2>
               5. C++
              </h2>
              <p>
               Want to get a little lower level with your programming? When you need to connect directly to hardware to get the most out of your processing power, C++ is the perfect choice for developing powerful desktop software, hardware-accelerated games and memory-intensive apps on desktops, consoles and mobile devices.
              </p>
              <h2>
               6. Python
              </h2>
              <p>
               Python can almost do it all. Web apps, user interfaces, data analysis, statistics  whatever your problem, theres likely a framework for it in Python. Most recently, Python has been used as a key tool for data scientists to sift through giant data sets for any industry.
              </p>
              <h2>
               7. C
              </h2>
              <figure class="image">
               <p>
                <img alt="c2" class="" data-caption="" data-credit-label="" data-credit-link="" data-credit-name="Mashable Composite/Wikimedia Commons" data-credit-provider="Custom type" data-fragment="m!054b" data-image="http://rack.0.mshcdn.com/media/ZgkyMDE1LzAxLzE2LzlhL2MyLjg4OGFlLmpwZwpwCXRodW1iCTEyMDB4OTYwMD4/87833c65/f45/c2.jpg" data-micro="1" data-width="640" src="http://rack.0.mshcdn.com/media/ZgkyMDE1LzAxLzE2LzlhL2MyLjg4OGFlLmpwZwpwCXRodW1iCTEyMDB4OTYwMD4/87833c65/f45/c2.jpg" title="c2"/>
               </p>
               <div class="image-credit">
                <p>
                 Image:  Mashable Composite/Wikimedia Commons
                </p>
               </div>
              </figure>
              <p>
               Why is the C language still popular? Size. C is small, fast and powerful. If youre building software for embedded systems, working with system kernels or just want to squeeze every last drop of the resources you have at hand, C is lean, mean and ready to scream.
              </p>
              <h2>
               8. SQL
              </h2>
              <p>
               Data is massive, its everywhere and its complex. SQL gives you the ability to find the exact information you want in a fast, repeatable and reliable way. Using SQL, you can easily query and extract meaningful data from large, complex databases.
              </p>
              <h2>
               9.Ruby
              </h2>
              <p>
               Want to kickstart your project in record time, or prototype a new idea for your next big web app? Ruby (and Ruby on Rails) can get you there quickly. The Ruby language is straightforward to learn and incredibly powerful, plus it powers tons of popular web apps around the globe.
              </p>
              <h2>
               10. Objective-C
              </h2>
              <figure class="image">
               <p>
                <img alt="objectivec" class="" data-caption="" data-credit-label="" data-credit-link="" data-credit-name="Mashable Composite/Wikmedia Commons" data-credit-provider="Custom type" data-fragment="m!2985" data-image="http://rack.1.mshcdn.com/media/ZgkyMDE1LzAxLzE2LzFmL29iamVjdGl2ZWMuZTJhNGMuanBnCnAJdGh1bWIJMTIwMHg5NjAwPg/24625b19/089/objectivec.jpg" data-micro="1" data-width="640" src="http://rack.1.mshcdn.com/media/ZgkyMDE1LzAxLzE2LzFmL29iamVjdGl2ZWMuZTJhNGMuanBnCnAJdGh1bWIJMTIwMHg5NjAwPg/24625b19/089/objectivec.jpg" title="objectivec"/>
               </p>
               <div class="image-credit">
                <p>
                 Image:  Mashable Composite/Wikmedia Commons
                </p>
               </div>
              </figure>
              <p>
               If you're interested in making an app for iOS, youll need to know Objective-C. While last years hype centered on Apples new language Swift, Objective-C is still the foundational language if you want to build apps for the Apple ecosystem. With Objective-C and XCode, the official software development tool from Apple, youll be in the App Store in no time.
              </p>
              <h2>
               11. Perl
              </h2>
              <p>
               Is Perl esoteric? Yes. Is it confusing? Yes. Is it a super powerful language, and a key component of anyones cyber security arsenal? Also true. Perl has powered the web since its early beginnings, and is still considered a key tool for any IT professional.
              </p>
              <h2>
               12 .NET
              </h2>
              <p>
               Although not a language in itself, .NET is a key Microsoft platform for cloud, service and app development that gets more advanced and valuable with each release. Due to the recent open-sourcing efforts of Microsoft, .NET is now coming to Google and Apple platforms. As a result, you can use .NET today with a variety of programming languages to build apps that easily support multiple platforms.
              </p>
              <h2>
               13. Visual Basic
              </h2>
              <p>
               Visual Basic is the language that gets business done. A key language of the .NET platform, it enables you to build applications to support your business, and automate powerful Office applications like Excel to accomplish super-human feats of computation, as well as streamline your most common tasks.
              </p>
              <h2>
               14. R
              </h2>
              <p>
               R is powering the revolution of big data, and is a must-know language in 2015 for anyone in need of serious data analysis. From science and business to entertainment and social media, R is the language to learn for statistical analysis across nearly every field of interest.
              </p>
              <h2>
               15. Swift
              </h2>
              <p>
               Not even a year old, the Swift programming language has captured the eyes and keyboards of developers worldwide as a new, fast and easy way to develop for Apples Mac and iOS operating systems. Swifts broad power and friendly syntax makes it possible for anyone with a Mac to build the next killer app for iOS or Mac OS X.
              </p>
