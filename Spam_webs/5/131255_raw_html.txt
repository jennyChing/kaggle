        <p style="text-align: justify;">
         Often after installing your favorite WordPress theme, you might want to customize it to make it look more appealing. Youll probably be thinking about writing code to change the look and feel of your theme. But, what if youre just a novice developer new to the working of WordPress and arent a programming expert? How hard can it be to customize a WP theme? In reality, it can be a pretty easy job and requires you to learn some basics of CSS.
        </p>
        <p style="text-align: justify;">
         Though this post, well discuss about how you can customize your WordPress installation by making simple changes to the style sheets.
        </p>
        <p style="text-align: justify;">
         <strong>
          Understanding the Basics of CSS
         </strong>
        </p>
        <p style="text-align: justify;">
         CSS (or Cascading Style Sheets) may seem complicated to you, if you dont have much knowledge about it. But, learning CSS isnt difficult and you can become familiar with its basics in no time. You can apply different style to your theme which includes changing colors, fonts, etc. in your HTML.
        </p>
        <p style="text-align: justify;">
         HTML is simply a web page (in our case WordPress web page) whose style can be changed using a CSS style sheet which specify selectors and apply properties, enabling you to make certain changes to different sections of your site. For example:
        </p>
        <p style="text-align: justify;">
         .intro {
        </p>
        <p style="text-align: justify;">
         background-color: green;
        </p>
        <p style="text-align: justify;">
         text-decoration: underline;
        </p>
        <p style="text-align: justify;">
         }
        </p>
        <p style="text-align: justify;">
         .intro is the CSS class selector, in our example, it contains two different properties: color and text-decoration. Using this code will change the background color to green and the text will be underlined. In CSS, you will find many different type of selectors that are patterns that helps to style the selected element(s).
        </p>
        <p style="text-align: justify;">
         <strong>
          Customizing Your WordPress Theme
         </strong>
        </p>
        <p style="text-align: justify;">
         Youll be wondering by now, where you can practice working on your CSS? How can you know if its working or not? Fortunately, there are some great tools available in major browsers such as Google Chrome and Safari, using which you can make some very basic yet important WordPress theme customizations.
        </p>
        <p style="text-align: justify;">
         Google Chrome tools are referred to as DevTools, while Safari calls their tools as Web Inspector. These tool let you test how your web page will look like, when youre making some CSS changes to it. For instance, you can test changing the text font to Sans Serif or change the text color to red and so on.
        </p>
        <p style="text-align: justify;">
         Wondering how to make use of these tool?
        </p>
        <p style="text-align: justify;">
         In order to make use of DevTools in Google Chrome, you dont need to do anything, as it is available by default. In order to use the tool, simply toggle the window. A better option is to use keyboard shortcuts that requires you to Press Ctrl + Shift + I, and it will open up the DevTools.
        </p>
        <p style="text-align: justify;">
         In case, youre using Safari as your web browser, then to open up the Web Inspector tool follow the below given steps:
        </p>
        <p style="text-align: justify;">
         1. Go to Safari  Preferences  Advanced, and check Show Develop menu in menu bar box.
        </p>
        <p>
         <a href="http://www.neodesigngeeks.com/wp-content/uploads/2015/04/Safari.jpg">
          <img alt="Safari" class="aligncenter size-full wp-image-1184" height="386" src="http://www.neodesigngeeks.com/wp-content/uploads/2015/04/Safari.jpg" width="600"/>
         </a>
        </p>
        <p style="text-align: justify;">
         2. Once the tool has turned on, you just need to use the same keyboard shortcuts as discussed above, so as to toggle the window.
        </p>
        <p style="text-align: justify;">
         <strong>
          Example: Demonstrating Making CSS to Customize WP Theme
         </strong>
        </p>
        <p style="text-align: justify;">
         Now lets consider an example to see how you can use the Google Chrome inspector tools. Lets see how to change the background color of the main title of your web page. First off, you will have to turn on the tool using the keyboard shortcut: Press Ctrl + Shift + I. Next, find a small spyglass icon given at the bottom in the left hand corner. Click on that spyglass icon.
        </p>
        <p style="text-align: justify;">
         Now, start hovering your mouse cursor around the actual webpage. As you hover, the cursor is selecting different elements on the page. Think of this like targeting. Target the element you want to change. If you hover carefully you can even drill down a bit inside some of the elements. Whenever you click, the inspector locks on to that element. To change the element youre locked on to, you need to click the spyglass icon again. Go ahead and click on the top navigation bar to lock on.
        </p>
        <p>
         <a href="http://www.neodesigngeeks.com/wp-content/uploads/2015/04/Safari2.jpg">
          <img alt="Safari2" class="aligncenter size-full wp-image-1185" height="355" src="http://www.neodesigngeeks.com/wp-content/uploads/2015/04/Safari2.jpg" width="600"/>
         </a>
        </p>
        <p style="text-align: justify;">
         When the inspector tool is turned on, youll find some code on the left (i.e. the HTML code) and some on the right (i.e. CSS code). We will make the changes in the right hand side code as follows:
        </p>
        <p style="text-align: justify;">
         media=all
        </p>
        <p style="text-align: justify;">
         .hire-section {
        </p>
        <p style="text-align: justify;">
         background:#fff;
        </p>
        <p style="text-align: justify;">
         padding:21px 99px 0 18px;
        </p>
        <p style="text-align: justify;">
         }
        </p>
        <p style="text-align: justify;">
         Since we only need to change the background color, replace the line of code background:#fff with background:#cccccc;
        </p>
        <p>
         <strong>
          Output:
         </strong>
        </p>
        <p>
         <a href="http://www.neodesigngeeks.com/wp-content/uploads/2015/04/Safari3.jpg">
          <img alt="Safari3" class="aligncenter size-full wp-image-1186" height="355" src="http://www.neodesigngeeks.com/wp-content/uploads/2015/04/Safari3.jpg" width="600"/>
         </a>
        </p>
        <p style="text-align: justify;">
         You can see that the background color of the main heading has changed to grey.
        </p>
        <p style="text-align: justify;">
         <strong>
          Applying Changes to WordPress Theme
         </strong>
        </p>
        <p style="text-align: justify;">
         Keep in mind that the tool only tells you about how your web page will look after making the changes, but the changes dont actually show up in your live site. And so, lets talk about how you can apply the changes to your theme. For this purpose, youll need to edit your WordPress theme CSS file.
        </p>
        <p style="text-align: justify;">
         You can easily edit your CSS file from your WordPress website dashboard. Navigate to Appearance  Editor menu in your WordPress dashboard, and then open the style.css file. Look for the style rule you would like to change in your live website.
        </p>
        <p style="text-align: justify;">
         Once youve made the desired changes, simply click on the Update File button. Open up your website, and youll be able to see the changes.
        </p>
        <p style="text-align: justify;">
         <strong>
          Make Changes in a Child Theme
         </strong>
        </p>
        <p style="text-align: justify;">
         Avoid making direct changes to your live site, as it can derail the site. There are two main problems youll face when changing your themes stylesheet:
        </p>
        <ol style="text-align: justify;">
         <li>
          If the theme author rolls out some updates, then updating your theme will make you lose all your changes. And, you will have to spend time in making the changes again.
         </li>
        </ol>
        <ol start="2" style="text-align: justify;">
         <li>
          You can end up messing your original code.
         </li>
        </ol>
        <p style="text-align: justify;">
         But, what if you can make the changes show up in real without affecting the current theme? Thanks to child themes, you can make changes to the code without having to alter your parent theme (i.e. main website theme). So, prior to the customization process, you must learn about using a WordPress child theme.
        </p>
        <p style="text-align: justify;">
         <strong>
          Wrapping it up!
         </strong>
        </p>
        <p style="text-align: justify;">
         Hopefully, reading this post will help you understand the basics of customizing a WordPress theme, by making simple changes in your CSS file. But, remember theres a lot more you need to know about WordPress theme customization. So, keep exploring!
        </p>
