         <h1 class="entry-title single-title" itemprop="headline">
          Ten Tips to Make Your Computer Servers More Secure
         </h1>
         <p class="byline vcard">
          Posted
          <time class="updated" datetime="2015-05-14" pubdate="">
           May 14, 2015
          </time>
          <span class="amp">
           &amp;
          </span>
          filed under
          <a href="http://www.s1mn.com/category/computer-servers/" rel="category tag">
           Computer Servers
          </a>
          .
         </p>
        </header>
        <section class="entry-content clearfix" itemprop="articleBody">
         <p>
          Computer security is in the news a lot lately. Between celebrities being hacked and big corporations having financial records and other confidential information leaked to the public, it might seem like data cannot be secured no matter how hard you try. But there are several steps you can take to ensure your information stays safe from the threat of a data breach. We specialize in
          <a href="http://www.s1mn.com/computer-servers-st-paul/">
           computer servers in St Paul
          </a>
          and the surrounding areas, and we have a few suggestions for you on how to keep computer servers safe and secure.
         </p>
         <ul>
          <li>
           Make sure that the room where your computer servers are housed are locked. Keep the switches and hubs locked too, and make sure you monitor and limit who has access. If the door is secured by a key code of some sort, make sure you change this regularly.
          </li>
          <li>
           Be sure to have some type of fire suppression system installed if possible. There are dry chemical fire suppression agents that can be used to make sure your computer servers do not sustain any water damage if the sprinklers go off.
          </li>
          <li>
           Keep your computer servers cool. Make sure the room has plenty of ventilation. Also, make sure the room has redundant power supplies in case there is a power failure.
          </li>
          <li>
           Restrict the number of admin passwords to your computer servers, and try to find some type of tool or software that will regularly remind you to change or update your passwords.
          </li>
          <li>
           Make sure there is a hardware firewall set up for all of your computer servers. Also be sure to regularly upgrade the software and the anti-virus software.
          </li>
          <li>
           Never let a computer server double as an employee workstation.
          </li>
          <li>
           Monitor the server logs and be sure to note any changes or irregularities.
          </li>
          <li>
           Make sure you regularly back up all of your data and keep that backed up data secure too.
          </li>
          <li>
           Your data protection should be tiered in that it offers a multiple lines of defense against multiple types of threats. These layers of defense can help isolate and protect your critical data if one of the layers is compromised in some way.
          </li>
          <li>
           Consider encrypting your data too. There are a variety of tools and software on the market that can do this.
          </li>
         </ul>
