  <script type="text/javascript">
   jQuery( function($) {
	var cbs = $('.compatibility-bar'),
	    worksBar = $('#compatibility-bar-works'),
	    brokeBar = $('#compatibility-bar-broken'),
	    dataBar = $('#compatibility-bar-data'),
	    buttons = $('#compatibility button'),
	    wpv = $('#compatibility-version'),
	    tv = $('#compatibility-topic-version'),
	    works,
	    cell,
	    ratio,
	    worksText = $('#compatibility-works'),
	    brokeText = $('#compatibility-broken'),
	    worksTextN = [ '1 person says it works.', '%d people say it works.' ],
	    brokeTextN = [ '1 person says it\'s broken.', '%d people say it\'s broken.' ];

	$('.compatibility select').change( function() {
		cell = $( '#compatibility-version-' + wpv.val().replace( /[.]/g, '_' ) ).find( '.compatibility-topic-version-' + tv.val().replace( /[.]/g, '_' ) );
		works = $.map( cell.text().split( ',' ), function(a) { return parseInt( a ); } );
		buttons.removeClass( 'active' );
		cbs.hide();
		if ( wpv.val() == null || tv.val() == null || isNaN( works[0] ) ) {
			dataBar.show();
			worksText.text( worksTextN[1].replace( '%d', 0 ) );
			brokeText.text( brokeTextN[1].replace( '%d', 0 ) );
		} else {
			if ( 4 &gt; works[1] ) {
				dataBar.show();
			} else {
				ratio = works[2] / works[1];
				if ( 0.7 &lt;= ratio ) {
					worksBar.show();
				} else if ( 0.3 &gt;= ratio ) {
					brokeBar.show();
				} else {
					dataBar.show();
				}
			}

			worksText.text( worksTextN[ 1 == works[2] ? 0 : 1 ].replace( '%d', works[2].toString() ) );
			brokeText.text( brokeTextN[ 1 == works[1] - works[2] ? 0 : 1 ].replace( '%d', ( works[1] - works[2] ).toString() ) );

			if ( cell.hasClass( 'user-works' ) ) {
				buttons.filter( '.works' ).addClass( 'active' );
			} else if ( cell.hasClass( 'user-broken' ) ) {
				buttons.filter( '.broken' ).addClass( 'active' );
			}
		}
	} ).filter( ':first' ).change();
} );
  </script>

        <div class="block-content">
         <p>
          Create exams and quizzes and display the result immediately after the user takes the exam. You can assign grades and point levels for every grade in the exam / quiz. Then assign points to every answer to a question and Watu will figure out the grade based on the total number of points collected.
         </p>
         <p>
          Watu for WordPress is a light version of
          <a href="http://calendarscripts.info/watupro/" rel="nofollow">
           Watu PRO
          </a>
          . Check it if you want to run fully featured exams with data exports, student logins, categories etc.
         </p>
         <p>
          <strong>
           This plugin is mobile / touch - friendly.
          </strong>
          The quizzes will work on mobile devices and phones.
         </p>
         <p>
          <strong>
           Please go to Tools -&gt; Watu Quizzes to start creating quizzes and exams.
          </strong>
         </p>

          <li>
           Creates quizzes and exams
          </li>
          <li>
           Use shortcodes to embed quizzes in posts or pages
          </li>
          <li>
           Single-choice questions
          </li>
          <li>
           Multiple-choice questions
          </li>
          <li>
           Open-end questions (essays)
          </li>
          <li>
           Required questions
          </li>
          <li>
           Grades
          </li>
          <li>
           Shows answers at the end of the quiz or immediately after selection
          </li>

          </li>
          <li>
           Ajax-based loading of the quiz results.
          </li>
          <li>
           Facebook sharing
          </li>
          <li>
           Mobile / touch - friendly
          </li>
          <li>
           Notify admin when someone takes a quiz
          </li>
         </ul>

           live demo here

If you have more doubts just download the plugin and check out if it works for you. It's free and takes a few seconds to install and activate.

         <h4>
          Troubleshooting
         </h4>
         <p>
          <strong>
           When opening a support thread please provide URL (link) where we can see your problem.
          </strong>
         </p>
         <p>
          A very common problem is not being able to submit the quiz, or the quiz does not displays at all. This is usually a fatal javascript error caused by other plugins or your them. If you are technical you can easily find the error yourself by checking the JavaScript error console in Chrome or Firefox. Disable the offending plugin and everything will start working normally.
         </p>

          Developers API
         </h4>
         <p>
          In order to allow other plugins to integrate better to Watu we have started working on developers API.
The following action calls are currently available:
         </p>
         <p>
          = do_action('watu_exam_submitted', $taking_id)
          <br/>
          Called when exam is submitted, passes the taken exam ID
         </p>
         <p>
          = do_action('watu_exam_saved', $exam_id)
Called when you add or edit exam (after submitting the changes). Passes the changed exam ID.
