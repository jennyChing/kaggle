        <div itemprop="articleBody">
         In our beginner classes we teach that module are small blocks of content around the outside of a Joomla site ... top, right, left, bottom.
         <br/>
         <br/>
         Now that you're a little more familiar with Joomla, let's see an exception to that rule ... we can place modules inside regular text articles.
         <br/>
         <br/>
         <ul>
          <li>
           <strong>
            Step 1
           </strong>
           : In the administrator area of your Joomla site, go to Extensions &gt;&gt; Plugin Manager
          </li>
          <li>
           <strong>
            Step 2
           </strong>
           : Find the plugin called "Content - Load Module" and make sure that it's enabled. On Joomla 1.0 its name is "Load Module Positions"
          </li>
          <li>
           <strong>
            Step 3
           </strong>
           : Go to Extensions &gt;&gt; Module Manager and find the module you'd like to place inside an article.
          </li>
          <li>
           <strong>
            Step 4
           </strong>
           : Open the module and click on the "position" dropdown.
          </li>
          <li>
           <strong>
            Step 5
           </strong>
           : Choose a position that doesn't appear on your template. Remember that you can check this by adding /?tp=1 to your site.
          </li>
          <li>
           <strong>
            Step 6
           </strong>
           : Save that module in its new position. In this example we've saved it in "User1"
          </li>
          <li>
           <strong>
            Step 7
           </strong>
           : Go to Content &gt;&gt; Article Manager and open the article you're using.
          </li>
          <li>
           <strong>
            Step 8
           </strong>
           : Place this code into the article: { loadposition user1 } ... remove the spaces between the brackets and the text. If you're using Joomla 1.0, then the code is {mosloadposition user1} .
          </li>
          <li>
           <strong>
            Step 9
           </strong>
           : Save the article and check your handiwork on the front of your Joomla site.
          </li>
         </ul>
