    <a href="http://www.blogger.com/">

    <a href="http://computgozez.blogspot.com.es/2013/05/google-to-acquiring-whatsapp-for-close.html">
     Microsoft attempted to
    </a>
    make typing appear smoother in Office 2013 by adding a typing animation feature. If youre a slow typist, you probably wont notice it. However, if you type upwards of 80+ words per minute, this feature can be distracting and annoying.
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    The typing animation feature can also be disruptive if you have a slower computer. This feature is on by default, but it can be disabled. To do so, you must edit the registry.
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    NOTE: Before making changes to the registry, be sure youback it up. We also recommendcreating a restore pointyou can use torestore your systemif something goes wrong.
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    To open the Registry Editor, press the Windows key + R to open the Run dialog box. Type regedit (without the quotes) in the Open edit box and click OK.
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    <img alt="" border="" class="lazyLoad" height="334" src="http://cdn.howtogeek.com/wp-content/uploads/2013/04/01_running_regedit.png" style="background-image: none; border: 0px; display: inline; margin-left: -2px; padding-left: 0px; padding-right: 0px; padding-top: 0px;" title="01_running_regedit" width="650"/>
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    If the User Account Control dialog box displays, click Yes to continue.
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    NOTE: You may not see this dialog box, depending on yourUser Account Control settings.
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    <img alt="" border="" class="lazyLoad" height="389" src="http://cdn.howtogeek.com/wp-content/uploads/2013/04/02_uac_dialog_for_registry_editor.png" style="background-image: none; border: 0px; display: inline; margin-left: -2px; padding-left: 0px; padding-right: 0px; padding-top: 0px;" title="02_uac_dialog_for_registry_editor" width="650"/>
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    Navigate to the following key:
   </div>
   <blockquote style="background-color: #f9f9f9; border: 1px dashed rgb(47, 111, 171); clear: both; font-family: monospace; font-size: 15px; line-height: 1.1em; margin: 0px; max-width: 638px; padding: 4px 4px 4px 6px;">
    <div style="color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 10px; padding: 0px;">
     HKEY_CURRENT_USER\Software\Microsoft\Office\15.0\Common
    </div>
   </blockquote>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    If there is no Graphics key under the Common key, right-click on the Common key and select New | Key from the popup menu.
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    <img alt="" border="" class="lazyLoad" height="458" src="http://cdn.howtogeek.com/wp-content/uploads/2013/04/03_selecting_new_key.png" style="background-image: none; border: 0px; display: inline; margin-left: -2px; padding-left: 0px; padding-right: 0px; padding-top: 0px;" title="03_selecting_new_key" width="650"/>
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    A new key is added to the bottom of the list under the Common key and the name is highlighted. Type Graphics for the name of the key.
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    <img alt="" border="" class="lazyLoad" height="584" src="http://cdn.howtogeek.com/wp-content/uploads/2013/04/04_new_key_name_highlighted.png" style="background-image: none; border: 0px; display: inline; margin-left: -2px; padding-left: 0px; padding-right: 0px; padding-top: 0px;" title="04_new_key_name_highlighted" width="650"/>
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    Right-click in an empty area on the right side of the editor and select New DWORD (32-bit) Value from the popup menu.
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    <img alt="" border="" class="lazyLoad" height="373" src="http://cdn.howtogeek.com/wp-content/uploads/2013/04/05_creating_new_dword.png" style="background-image: none; border: 0px; display: inline; margin-left: -2px; padding-left: 0px; padding-right: 0px; padding-top: 0px;" title="05_creating_new_dword" width="650"/>
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    Name that new value DisableAnimation, as shown below, and double-click on the new value.
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">

   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    Change the number in the Value data edit box to 1 and click OK.
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    <img alt="" border="" class="lazyLoad" height="384" src="http://cdn.howtogeek.com/wp-content/uploads/2013/04/07_setting_new_dword_value.png" style="background-image: none; border: 0px; display: inline; margin-left: -2px; padding-left: 0px; padding-right: 0px; padding-top: 0px;" title="07_setting_new_dword_value" width="650"/>
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    To close the Registry Editor, select Exit from the File menu. Restart Windows for the change to take effect.
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    <img alt="" border="" class="lazyLoad" height="315" src="http://cdn.howtogeek.com/wp-content/uploads/2013/04/08_closing_registry_editor.png" style="background-image: none; border: 0px; display: inline; margin-left: -2px; padding-left: 0px; padding-right: 0px; padding-top: 0px;" title="08_closing_registry_editor" width="650"/>
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    <a href="http://computgozez.blogspot.com.es/2013/05/ps4-call-of-duty.html">
     To turn the
    </a>
    typing animation back on, go back into the Registry Editor, and either delete the DisableAnimation value and the Graphics key or change the Value data for the DisableAnimation value to 0.
   </div>
   <div style="background-color: white; color: #333333; font-family: Verdana; font-size: 14px; line-height: 17px; margin-bottom: 20px; margin-top: 20px; padding: 0px;">
    NOTE: Disabling the typing animation feature in Office 2013 turns it off for all Office programs. You cannot leave the feature on for certain programs.
