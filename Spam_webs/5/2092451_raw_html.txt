										&lt;h1 class="title"&gt;Cerchi subacquei sul fondo delloceano: sono opera del pesce palla&lt;/h1&gt;
					
	
		&lt;div class="post-meta clearfix"&gt;

		&lt;span class="ae_body"&gt;&lt;b&gt;Name&lt;/b&gt;: A.M. &lt;br /&gt;&lt;b&gt;Email:&lt;/b&gt; oubliettemagazine@hotmail.it&lt;br /&gt;&lt;b&gt;Site:&lt;/b&gt; &lt;a href="http://oubliettemagazine.com/"&gt;http://oubliettemagazine.com/&lt;/a&gt;&lt;br /&gt;&lt;/span&gt;&lt;span class="ae_about"&gt;&lt;b&gt;About:&lt;/b&gt; &lt;/span&gt;&lt;span class="ae_body"&gt;&lt;a href="http://oubliettemagazine.com/author/oubliettemagazine/"&gt;See Authors Posts&lt;/a&gt; (4439)&lt;/span&gt;&lt;/span&gt; in &lt;a href="http://oubliettemagazine.com/category/attualita/" title="Visualizza tutti gli articoli in ATTUALITA&#039;" rel="category tag"&gt;ATTUALITA'&lt;/a&gt;, &lt;a href="http://oubliettemagazine.com/category/attualita/luoghi/" title="Visualizza tutti gli articoli in Luoghi" rel="category tag"&gt;Luoghi&lt;/a&gt;, &lt;a href="http://oubliettemagazine.com/category/attualita/natura/" title="Visualizza tutti gli articoli in Natura" rel="category tag"&gt;Natura&lt;/a&gt;, &lt;a href="http://oubliettemagazine.com/category/attualita/nius/" title="Visualizza tutti gli articoli in Nius" rel="category tag"&gt;Nius&lt;/a&gt; | &lt;a href="http://oubliettemagazine.com/2013/07/10/cerchi-subacquei-sul-fondo-delloceano-sono-opera-del-pesce-palla/#respond" title="Commenti a Cerchi subacquei sul fondo delloceano: sono opera del pesce palla"&gt;0 comments&lt;/a&gt;&lt;/p&gt;

					&lt;div class="entry-content"&gt;
												
														
															&lt;img border="0" src="http://oubliettemagazine.com/wp-content/uploads/et_temp/watercircle-47478_140x140.jpg" class='thumb alignleft'  alt='Cerchi subacquei sul fondo delloceano: sono opera del pesce palla' /&gt;																											&lt;p class="date"&gt;&lt;span&gt;lug 10, 2013&lt;/span&gt;&lt;/p&gt;	
																																
										
						&lt;div style="float: left; margin-right: 10px;"&gt;&lt;/div&gt;&lt;blockquote&gt;&lt;p&gt;&lt;strong&gt;Oggi vi narriamo la storia di un piccolo &lt;a href="http://oubliettemagazine.com/tag/pesce-palla/" target="_blank"&gt;pesce palla&lt;/a&gt;&lt;/strong&gt;, specie della quale, fino ad ora, non erano note le abilit artistiche. Lanimale, famoso invece in tutto il &lt;a href="http://oubliettemagazine.com/tag/giappone/" target="_blank"&gt;Giappone&lt;/a&gt; per essere &lt;strong&gt;una prelibatezza culinaria&lt;/strong&gt; che pu causare intossicazioni lievi o addirittura la morte se tagliato e servito nel modo sbagliato, pare essere&lt;strong&gt; lartefice di misteriosi cerchi subacquei.&lt;/strong&gt;&lt;/p&gt;&lt;/blockquote&gt;
&lt;p&gt;&lt;a href="http://oubliettemagazine.com/wp-content/uploads/cerchi-subacqui-pesce-palla.jpg"&gt;&lt;img class="alignright size-medium wp-image-37390" title="cerchi subacqui pesce palla" src="http://oubliettemagazine.com/wp-content/uploads/cerchi-subacqui-pesce-palla-300x180.jpg" alt="" width="300" height="180" /&gt;&lt;/a&gt;Durante unimmersione &lt;strong&gt;nelle acque di Amami Oshima, il fotografo giapponese &lt;a href="http://oubliettemagazine.com/tag/yoji-ookata/" target="_blank"&gt;Yoji Ookata&lt;/a&gt;,&lt;/strong&gt; esperto sommozzatore e fotografo subacqueo, si  imbattuto in &lt;strong&gt;bizzarre formazioni circolari&lt;/strong&gt; disegnate sul fondale sabbioso,&lt;strong&gt; a circa 25 metri&lt;/strong&gt; di profondit dalla superficie.&lt;/p&gt;
&lt;p&gt;Questi enormi segni circolari tracciati sulla sabbia, che &lt;strong&gt;tanto ricordano i famosi cerchi nel grano&lt;/strong&gt; celebrati dagli appassionati di &lt;a href="http://oubliettemagazine.com/tag/alieni/" target="_blank"&gt;Ufo&lt;/a&gt;, hanno &lt;strong&gt;un diametro di 60-70 cm&lt;/strong&gt; e sembrano essere stati meticolosamente disegnati in modo tale da creare creste simmetriche disposte in cerchi concentrici.&lt;/p&gt;
&lt;p&gt;&lt;strong&gt;Lorigine di questi disegni di straordinaria bellezza,&lt;/strong&gt; per qualche tempo  rimasta un mistero, fino a quando Ookata ha deciso di scoprire il colpevole di questa vera e propria opera darte subacquea.&lt;/p&gt;
&lt;p&gt;&lt;a href="http://oubliettemagazine.com/wp-content/uploads/pesce-palla-cerchi-sulla-sabbia.jpg"&gt;&lt;img class="alignright size-medium wp-image-37391" title="pesce palla cerchi sulla sabbia" src="http://oubliettemagazine.com/wp-content/uploads/pesce-palla-cerchi-sulla-sabbia-300x210.jpg" alt="" width="300" height="210" /&gt;&lt;/a&gt;Come rivelato da uno speciale televisivo giapponese, basato sui filmati ripresi dalla telecamere di Ookata, &lt;strong&gt;gli strani segni non sono stati causati da alieni o da correnti sottomarine,&lt;/strong&gt; ma, pi semplicemente, da un piccolo pesce palla. Con laiuto di una sola delle sue pinne,&lt;strong&gt; il piccolo artista lavora giorno e notte per creare queste strutture circolari.&lt;/strong&gt; E non solo. Questi pesci sono stati osservati mentre spaccano alcune conchiglie con il loro becco per ottenere piccoli frammenti da utilizzare, si pensa, come &lt;strong&gt;decorazioni per abbellire ulteriormente i loro disegni.&lt;/strong&gt;&lt;/p&gt;
&lt;p&gt;Durante le analisi, il fotografo e la sua equipe hanno osservato un esemplare maschio trascorrere giornate intere a creare queste creste circolari sul fondo delloceano, facendo sbattere solo una pinna. Oltre ad essere opere darte, questi cerchi hanno anche uno scopo ben preciso:&lt;strong&gt; attirare le femmine per laccoppiamento e deporre poi le uova al centro del disegno.&lt;/strong&gt; Secondo gli scienziati che hanno condotto la missione, le uova, cos facendo, sono effettivamente pi protette. &lt;strong&gt;I cerchi infatti neutralizzano le correnti&lt;/strong&gt; e le rendono meno sensibili ai predatori.&lt;/p&gt;
&lt;p&gt;&lt;strong&gt;&lt;a href="http://oubliettemagazine.com/wp-content/uploads/watercircle.jpg"&gt;&lt;img class="alignright size-medium wp-image-37392" title="watercircle" src="http://oubliettemagazine.com/wp-content/uploads/watercircle-300x180.jpg" alt="" width="300" height="180" /&gt;&lt;/a&gt;I maschi pi creativi&lt;/strong&gt; inoltre, realizzando ambienti complessi ed articolati, pare attirino un maggior numero di femmine.&lt;/p&gt;
&lt;p&gt;La localit dei disegni nella sabbia  Amami Oshima,&lt;strong&gt; unisola semi-tropicale&lt;/strong&gt; che fa parte del gruppo delle isole Amami, &lt;strong&gt;nellarcipelago giapponese di Ryukyu.&lt;/strong&gt; Tra le specie che popolano questa regione del Pacifico ci sono anche quelle appartenenti alla famiglia dei Tetraodontidae, comunemente noti come pesci palla. Questa famiglia comprende &lt;strong&gt;185 specie di pesci dacqua dolce e salata,&lt;/strong&gt; diffusi per la maggior parte nelle acque tropicali del globo.&lt;/p&gt;
&lt;p&gt;Il nome scientifico  di &lt;strong&gt;derivazione greca, e significa letteralmente quattro denti&lt;/strong&gt;con riferimento alle quattro placche dentarie caratteristiche di questi pesci. Pur non essendo un ottimo nuotatore per via della rigidit del suo corpo, il pesce palla non risulta essere preda ideale, poich dotato di due particolari sistemi di difesa. &lt;strong&gt;Egli  in grado di ingurgitare rapidamente grandi quantit di acqua,&lt;/strong&gt; diventando molto grande e quindi difficile da inghiottire anche per predatori di grosse dimensioni. Inoltre la sua carne contiene un veleno molto potente, &lt;strong&gt;la Tetradotossina,&lt;/strong&gt; una neurotossina che inibisce la funzione respiratoria, portando rapidamente alla morte chi ne ingerisce le carni.&lt;/p&gt;
&lt;p&gt;&lt;a href="http://oubliettemagazine.com/wp-content/uploads/cerchi_nella_sabbia_pesce_palla.jpg"&gt;&lt;img class="alignright size-medium wp-image-37394" title="cerchi_nella_sabbia_pesce_palla" src="http://oubliettemagazine.com/wp-content/uploads/cerchi_nella_sabbia_pesce_palla-300x225.jpg" alt="" width="300" height="225" /&gt;&lt;/a&gt;In Italia, proprio per questo motivo, &lt;strong&gt;dal 1992  proibito il commercio a scopo alimentare.&lt;/strong&gt; In Giappone, paese dedito al consumo di questa pietanza, si riscontra &lt;strong&gt;ogni anno un elevato numero di casi dintossicazione e di morte&lt;/strong&gt; dovuto al consumo di questi pesci. Ci accade perch, nonostante le istituzioni locali rilascino il nulla osta alla sua preparazione solo a cuochi specializzati che abbiano seguito appositi corsi di apprendimento delle tecniche di separazione delle carni dalle parti velenose, alcuni locali propongono la portata ai clienti, mettendo seriamente in pericolo la loro salute.&lt;/p&gt;
&lt;p&gt;&lt;strong&gt;La specie di pesce palla osservata da Ookata&lt;/strong&gt; sembra aver elaborato una strategia del tutto differente durante i suoi ultimi 80-100 milioni di anni di evoluzione. La serie di creste che decorano i cerchi non hanno solo una funzione visiva: servono soprattutto per mitigare le correnti oceaniche, proteggendo le uova ed evitando di esporle a potenziali pericoli. Anche le conchiglie frantumate sembrano avere un altro scopo oltre a quello decorativo, visto che potrebbero essere &lt;strong&gt;la prima fonte di nutrienti a disposizione dei nascituri.&lt;/strong&gt;&lt;/p&gt;
&lt;p&gt;La storia del piccolo pesce palla, abile decoratore di fondali oceanici, si conclude suscitando tutta la nostra ammirazione per &lt;strong&gt;questa figura di novello Romeo&lt;/strong&gt;. A lui giungano le nostre congratulazioni pi sincere per avere pensato proprio a tutto, e mai come in questo caso, vale il concetto: &lt;em&gt;&lt;strong&gt;guardate che cosa bisogna fare, pur di conquistare una donna!&lt;/strong&gt;&lt;/em&gt;.&lt;/p&gt;

			function JetpackLikesWidgetQueueHandler() {
				var wrapperID;
				if ( ! jetpackLikesMasterReady ) {
					setTimeout( JetpackLikesWidgetQueueHandler, 500 );
					return;
				}

				if ( jetpackLikesWidgetQueue.length &gt; 0 ) {
					// We may have a widget that needs creating now
					var found = false;
					while( jetpackLikesWidgetQueue.length &gt; 0 ) {
						// Grab the first member of the queue that isn't already loading.
						wrapperID = jetpackLikesWidgetQueue.splice( 0, 1 )[0];
						if ( jQuery( '#' + wrapperID ).hasClass( 'jetpack-likes-widget-unloaded' ) ) {
							found = true;
							break;
						}
					}
					if ( ! found ) {
						setTimeout( JetpackLikesWidgetQueueHandler, 500 );
						return;
					}
				} else if ( jQuery( 'div.jetpack-likes-widget-unloaded' ).length &gt; 0 ) {
					// Get the next unloaded widget
					wrapperID = jQuery( 'div.jetpack-likes-widget-unloaded' ).first()[0].id;
					if ( ! wrapperID ) {
						// Everything is currently loaded
						setTimeout( JetpackLikesWidgetQueueHandler, 500 );
						return;
					}
				}

				var $wrapper = jQuery( '#' + wrapperID );
				$wrapper.find( 'iframe' ).remove();

				if ( $wrapper.hasClass( 'slim-likes-widget' ) ) {
					$wrapper.find( '.post-likes-widget-placeholder' ).after( "&lt;iframe class='post-likes-widget jetpack-likes-widget' name='" + $wrapper.data( 'name' ) + "' height='22px' width='68px' frameBorder='0' scrolling='no' src='" + $wrapper.data( 'src' ) + "'&gt;&lt;/iframe&gt;" );
				} else {
					$wrapper.find( '.post-likes-widget-placeholder' ).after( "&lt;iframe class='post-likes-widget jetpack-likes-widget' name='" + $wrapper.data( 'name' ) + "' height='55px' width='100%' frameBorder='0' src='" + $wrapper.data( 'src' ) + "'&gt;&lt;/iframe&gt;" );
				}

				$wrapper.removeClass( 'jetpack-likes-widget-unloaded' ).addClass( 'jetpack-likes-widget-loading' );

				$wrapper.find( 'iframe' ).load( function( e ) {
					var $iframe = jQuery( e.target );
					$wrapper.removeClass( 'jetpack-likes-widget-loading' ).addClass( 'jetpack-likes-widget-loaded' );

					JetpackLikespostMessage( { event: 'loadLikeWidget', name: $iframe.attr( 'name' ), width: $iframe.width() }, window.frames[ 'likes-master' ] );

					if ( $wrapper.hasClass( 'slim-likes-widget' ) ) {
						$wrapper.find( 'iframe' ).Jetpack( 'resizeable' );
					}
				});
				setTimeout( JetpackLikesWidgetQueueHandler, 250 );
			}
