        <h1>
         <a href="2015/07/11/cakephp_2_7_0_released.html">
          CakePHP 2.7.0 Released
         </a>
        </h1>
        <p>
         The CakePHP core team is happy to announce the immediate availability of CakePHP
2.7.0. 2.7.0 is an API compatible feature release for the 2.x release series.
The last 2.6 release will be 2.6.9, which will be released later this week.
        </p>
        <p>
         2.7.0 should be a drop in replacement for your existing 2.5 or 2.6 libraries.
In 2.7 the following new features are available.
        </p>
        <div class="section" id="deprecations">
         <h2>
          Deprecations
         </h2>
         <ul class="simple">
          <li>
           The class
           <span class="docutils literal">
            <span class="pre">
             String
            </span>
           </span>
           has been renamed to
           <span class="docutils literal">
            <span class="pre">
             CakeText
            </span>
           </span>
           . This resolves some
conflicts around HHVM compatibility as well as PHP7. The
           <span class="docutils literal">
            <span class="pre">
             String
            </span>
           </span>
           class is still provided for compatibility reasons.
          </li>
          <li>
           <span class="docutils literal">
            <span class="pre">
             Validation::notEmpty()
            </span>
           </span>
           has been renamed to
           <span class="docutils literal">
            <span class="pre">
             Validation::notBlank()
            </span>
           </span>
           .
This aims to avoid confusion around the PHP
           <span class="docutils literal">
            <span class="pre">
             notEmpty()
            </span>
           </span>
           function and that
the validation rule accepts
           <span class="docutils literal">
            <span class="pre">
             0
            </span>
           </span>
           as valid input.
          </li>
          <li>
           <span class="docutils literal">
            <span class="pre">
             SessionComponent::setFlash()
            </span>
           </span>
           has been deprecated. You should use
           <span class="docutils literal">
            <span class="pre">
             FlashComponent
            </span>
           </span>
           instead.
          </li>
          <li>
           <span class="docutils literal">
            <span class="pre">
             SessionHelper::flash()
            </span>
           </span>
           has been deprecated. You should use
           <span class="docutils literal">
            <span class="pre">
             FlashHelper
            </span>
           </span>
           instead.
          </li>
         </ul>
        </div>
        <div class="section" id="backported-features-from-3-0">
         <h2>
          Backported Features From 3.0
         </h2>
         <ul class="simple">
          <li>
           Plugin shells that share a name with their plugin can now be called without
the plugin prefix. For example
           <span class="docutils literal">
            <span class="pre">
             Console/cake
            </span>
            <span class="pre">
             MyPlugin.my_plugin
            </span>
           </span>
           can now
be called with
           <span class="docutils literal">
            <span class="pre">
             Console/cake
            </span>
            <span class="pre">
             my_plugin
            </span>
           </span>
           .
          </li>
          <li>
           <span class="docutils literal">
            <span class="pre">
             Configure::consume()
            </span>
           </span>
           has been added to read and delete from
Configure in a single step.
          </li>
          <li>
           <span class="docutils literal">
            <span class="pre">
             CakeSession::consume()
            </span>
           </span>
           has been added to read and delete from
session in a single step.
          </li>
          <li>
           New setting
           <span class="docutils literal">
            <span class="pre">
             level
            </span>
           </span>
           is now available in
           <span class="docutils literal">
            <span class="pre">
             TreeBehavior
            </span>
           </span>
           . You can use it to
specify field name in which the depth of tree nodes will be stored.
          </li>
          <li>
           New method
           <span class="docutils literal">
            <span class="pre">
             TreeBehavior::getLevel()
            </span>
           </span>
           has been added which fetches depth of
a node.
          </li>
          <li>
           <span class="docutils literal">
            <span class="pre">
             SessionComponent::consume()
            </span>
           </span>
           has been added to read and delete
from session in a single step.
          </li>
          <li>
           <span class="docutils literal">
            <span class="pre">
             SessionHelper::consume()
            </span>
           </span>
           has been added to read and delete from
session in a single step.
          </li>
          <li>
           <span class="docutils literal">
            <span class="pre">
             Shell::param()
            </span>
           </span>
           has been added. This method allows you to read CLI options
without notice errors.
          </li>
          <li>
           The
           <span class="docutils literal">
            <span class="pre">
             text/plain
            </span>
           </span>
           content type is no longer mapped to the csv response type
by RequestHandlerComponent. This helps fix issues when applications used
jQuery and csv responses.
          </li>
         </ul>
        </div>
        <div class="section" id="other-improvements">
         <h2>
          Other Improvements
         </h2>
         <ul class="simple">
          <li>
           SQL datasources will now cast
           <span class="docutils literal">
            <span class="pre">
             ''
            </span>
           </span>
           and
           <span class="docutils literal">
            <span class="pre">
             null
            </span>
           </span>
           into
           <span class="docutils literal">
            <span class="pre">
             ''
            </span>
           </span>
           when columns are
not nullable and rows are being created or updated.
          </li>
          <li>
           Argument
           <span class="docutils literal">
            <span class="pre">
             $renew
            </span>
           </span>
           has been added to
           <span class="docutils literal">
            <span class="pre">
             CakeSession::clear()
            </span>
           </span>
           to allow
emptying the session without forcing a new id and renewing the session. It
defaults to
           <span class="docutils literal">
            <span class="pre">
             true
            </span>
           </span>
           .
          </li>
          <li>
           CakeEmail will now use the default config set when creating instances that
do not specify a configuration set to use. For example
           <span class="docutils literal">
            <span class="pre">
             $email
            </span>
            <span class="pre">
             =
            </span>
            <span class="pre">
             new
            </span>
            <span class="pre">
             CakeEmail();
            </span>
           </span>
           will now use the default config set.
          </li>
          <li>
           <span class="docutils literal">
            <span class="pre">
             ControllerTestCase::testAction()
            </span>
           </span>
           now supports an array as URL.
          </li>
         </ul>
         <p>
          As always, a huge thanks to all the community members that helped make this
release happen by reporting issues, writing documentation and sending pull requests.
         </p>
         <p>
          Download a
          <a class="reference external" href="https://github.com/cakephp/cakephp/releases">
           packaged release on github
          </a>
          .
         </p>
        </div>
