             <h3 class="post-title entry-title" itemprop="name">
              Condylactis
             </h3>
             <div class="post-header">
              <div class="post-header-line-1">
              </div>

             <div class="post-body entry-content" id="post-body-7913714254801784256" itemprop="description articleBody">
              I've wanted to knit a scarf with a curly fringe edging for a while now, but just couldn't decide on the body.  I finally decided on a tube scarf as the curly fringe edging would create a sort of  sea anemone-
              <span class="blsp-spelling-error" id="SPELLING_ERROR_0">
               ish
              </span>
              look.  This scarf is knit using the cross stitch method - a method usually used for creating mock cables and other texture patterns.  When an entire row is knit using crossed stitches it creates and interesting twisted texture, plus it's a heck of a lot more fun that plain
              <span class="blsp-spelling-error" id="SPELLING_ERROR_1">
               ol
              </span>
              ' stockinette.  This scarf could easily be knit in the round - just knit the purl rows and add one more Cr instead of the two knit stitches on the texture rows - but if you are like me and hate, Hate, HATE working on double-pointed needles, it's easier just to knit flat and sew up the seam.
              <br/>
              <br/>
              <a href="http://4.bp.blogspot.com/_AkxhOmGiYmA/R0DToivvIOI/AAAAAAAAAGg/0wnuw5JH5Gw/s1600-h/IM001626.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
               <img alt="" border="0" id="BLOGGER_PHOTO_ID_5134336268677619938" src="http://4.bp.blogspot.com/_AkxhOmGiYmA/R0DToivvIOI/AAAAAAAAAGg/0wnuw5JH5Gw/s320/IM001626.jpg" style="cursor: pointer;"/>
              </a>
              <br/>
              <br/>
              Materials:
              <br/>
              US size 3 knitting needles
              <br/>
              2 skeins Melody by
              <span class="blsp-spelling-error" id="SPELLING_ERROR_2">
               Jojoland
              </span>
              , color number Y46, or other fingering weight yarn
              <br/>
              <br/>
              Gauge:
              <br/>
              <br/>
              Terms:
              <br/>
              Cr: Cross stitch.  Knit into the
              <span style="font-weight: bold;">
               second
              </span>
              stitch on the
              <span class="blsp-spelling-error" id="SPELLING_ERROR_3">
               LH
              </span>
              needle, but do not slip the stitch off the needle.  Then, knit into the first stitch on the
              <span class="blsp-spelling-error" id="SPELLING_ERROR_4">
               LH
              </span>
              needle.  Slip both of the stitches off the needle.
              <br/>
              <br/>
              Pattern:
              <br/>
              <br/>
              <a href="http://1.bp.blogspot.com/_AkxhOmGiYmA/R0DT2yvvIQI/AAAAAAAAAGw/j349S1sgHFs/s1600-h/IM001623.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
               <img alt="" border="0" id="BLOGGER_PHOTO_ID_5134336513490755842" src="http://1.bp.blogspot.com/_AkxhOmGiYmA/R0DT2yvvIQI/AAAAAAAAAGw/j349S1sgHFs/s320/IM001623.jpg" style="cursor: pointer; width: 208px; height: 140px;"/>
              </a>
              <br/>
              <span style="color: rgb(204, 0, 0);">
               Close up of the cross stitch texture.
              </span>
              <br/>
              Body:
              <br/>
              <br/>
              Cast on 40  stitches.  Knit one row, purl one row.
              <br/>
              <br/>
              Row 1:  k1, Cr to last stitch, k1
              <br/>
              Row 2:  Purl
              <br/>
              <br/>
              Repeat rows 1 and 2 until the scarf is as long as you want.
              <br/>
              <br/>
              Bind off.
              <br/>
              <br/>
              Sew up the sides of the scarf creating a long tube.
              <br/>
              <br/>
              Curly fringe edging:
              <br/>
              <br/>
              <a href="http://2.bp.blogspot.com/_AkxhOmGiYmA/R0DTuCvvIPI/AAAAAAAAAGo/oah4UKMOFRk/s1600-h/IM001620.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
               <img alt="" border="0" id="BLOGGER_PHOTO_ID_5134336363166900466" src="http://2.bp.blogspot.com/_AkxhOmGiYmA/R0DTuCvvIPI/AAAAAAAAAGo/oah4UKMOFRk/s320/IM001620.jpg" style="cursor: pointer;"/>
              </a>
              <br/>
              <br/>
              Pick up one stitch at the end of the scarf.  Cast on 14 more stitches (I prefer the Knit On method as it gives a little more stability)  Bind off the 14 stitches you just cast on.
              <br/>
              Pick up another stitch.  Slip your one remaining stitch over the newly picked up stitch (so you again only have one stitch on the needle).  Cast on 14 more stitches.  Bind off the 14 stitches you just cast on.
              <br/>
              Continue in this manner until you have created 40 little "tentacles" around the entire edge of the scarf.
              <br/>
              Repeat at the other end.
              <div style="clear: both;">
              </div>
