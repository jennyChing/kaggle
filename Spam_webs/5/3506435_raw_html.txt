                    <h3 class="post-title entry-title" itemprop="name">
                     Jan 17, 2010 - My Mountain
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-7316359771703343333" itemprop="description articleBody">
                     <b style="color: #783f04;">
                      <span style="font-size: small;">
                      </span>
                     </b>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://2.bp.blogspot.com/_DLiSVt3iVg8/S1Mb8nmKhCI/AAAAAAAAANM/iUEjVS6N_rQ/s1600-h/my+mountain.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" height="359" src="http://2.bp.blogspot.com/_DLiSVt3iVg8/S1Mb8nmKhCI/AAAAAAAAANM/iUEjVS6N_rQ/s320/my+mountain.jpg" width="557"/>
                      </a>
                      <br/>
                     </div>
                     <div style="color: #783f04; font-family: Verdana,sans-serif; text-align: center;">
                      <br/>
                     </div>
                     <div style="color: #783f04; font-family: Verdana,sans-serif; text-align: center;">
                      <b>
                       <span style="font-size: small;">
                        My Mountain
                        <br/>
                        <br/>
                        All of my life
                        <br/>
                        I have been climbing a mountain
                        <br/>
                        And been beset often, by vast days of strife
                        <br/>
                        I never quite reach the top
                        <br/>
                        Yet, I keep climbing and I'll never stop
                        <br/>
                        <br/>
                        I have been fighting a battle
                        <br/>
                        Sometimes with each step that I take
                        <br/>
                        Fighting off demons with angels
                        <br/>
                        While deciding what to keep or forsake
                        <br/>
                        <br/>
                        The vision of truth in my destination
                        <br/>
                        Some days is so very unclear
                        <br/>
                        Of which paths I should be taking
                        <br/>
                        And of which paths
                        <br/>
                        I should steer clear
                        <br/>
                        <br/>
                        No matter how hard this endeavor
                        <br/>
                        Ever it shall become
                        <br/>
                        To get to the top of this mountain
                        <br/>
                        I shall persistently climb
                        <br/>
                        <br/>
                        Learning always
                        <br/>
                        From all my mistakes
                        <br/>
                        Until, my journey
                       </span>
                      </b>
                      <br/>
                      <b>
                       <span style="font-size: small;">
                        Is done
                       </span>
                      </b>
                      <br/>
                      <b>
                       <span style="font-size: small;">
                       </span>
                      </b>
                      <br/>
                     </div>

                      <b>
                       <span style="font-size: small;">
                       </span>
                      </b>
                      <br/>
                     </div>
                     <div style="color: #783f04; font-family: Verdana,sans-serif; text-align: center;">

                       <span style="font-size: small;">
                        <br/>
                        Neva Flores 2009
                       </span>
                      </b>
