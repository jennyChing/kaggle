     <section class="primary island c">
      <h2>
       About this Add-on
      </h2>
      <div class="prose">
       <div class="prose" id="addon-description">
        In Google Search, There was a feature that exclude from the search results to block certain websites.
        <br/>
        But, "site block feature" is discontinued.
        <br/>
        Then I developed this addon instead of "site block feature".
        <br/>
        <br/>
        How to specify the blocked sites are as follows.
        <br/>
        <ol>
         <li>
          Open Addon manager
         </li>
         <li>
          Press "Option" button on "Hide Unwanted Results of Google Search"
         </li>
         <li>
          Press "Settings" button to open panel.
         </li>
         <li>
          Type a URL that you want to hide, then press "Add" button.
         </li>
         <li>
          Press "OK" to Apply and close panel.
         </li>
        </ol>
        or
        <ol>
         <li>
          Use Google Search as normal
         </li>
         <li>
          Right click on a link that you don't want to see.
         </li>
         <li>
          Choose "Add this site to Hide Results"  &lt;---
          <strong>
           new in version 1.2
          </strong>
         </li>
        </ol>
        <br/>
        Note:
        <br/>
        URL accepts the regular expressions. (In regular expressions, combination "period" and "asterisk"
        <code>
         .*
        </code>
        matches anything (aka wildcard). sample:
        <code>
         <a href="http://outgoing.mozilla.org/v1/225130741e94df48c651b37f9814754cf162e70d00f587f7d8a8a2d612fdd605/http%3A//www.foobar.com/.%2A" rel="nofollow">
          www.foobar.com/.*
         </a>
        </code>
        )
        <br/>
        You can get more details about regular expressions as follows.
        <br/>
        <a href="http://outgoing.mozilla.org/v1/0a71e73e4f8d271cf761b14d5386141f76fd5ae9df9f624097de435be36c47b5/https%3A//developer.mozilla.org/en-US/docs/JavaScript/Reference/Global_Objects/RegExp" rel="nofollow">
         https://developer.mozilla.org/en-US/docs/JavaScript/Reference/Global_Objects/RegExp
        </a>
        <br/>
        <br/>
        <br/>
        Since version 1.2:
        <br/>
        Context menu is supported.
        <br/>
        From the context menu that appears when you right-click a link in the search result, you can add a site to hide from the search results.
        <br/>
        Added site that will become effective from the next search.
        <br/>
        <br/>
        Since version 1.4:
        <br/>
        Microsoft Bing Search is supported.
        <br/>
        <br/>
        Since version 1.5:
        <br/>
        Export and Import site data is supported.
        <br/>
        Indicator on results page is supported.
        <br/>
        Disable/Enable temporary on results page is supported.
        <br/>
        <br/>
        <br/>
        <strong>
         [What's next]
        </strong>
        <br/>
        <ul>
         <li>
          Support Bing search --&gt;
          <strong>
           supported in version 1.4
          </strong>
         </li>
         <li>
          Support Firefox Sync to share sites data with other PC
         </li>
         <li>
          Support Import/Export sites data  --&gt;
          <strong>
           supported in version 1.5
          </strong>
         </li>
         <li>
          Support Google Image Search
         </li>
         <li>
          Support Yahoo Search
         </li>
         <li>
          Support multiple language UI
         </li>
         <li>
          Indicator for hidden sites --&gt;
          <strong>
           supported in version 1.5
          </strong>
         </li>
         <li>
          Switch Enable/Disable on results page  --&gt;
          <strong>
           supported in version 1.5
          </strong>
         </li>
        </ul>
        <br/>
        <strong>
         [Tips for add lots of sites]
        </strong>
        <br/>
        Currently, this addon do not support Import feature.
        <br/>
        But you can add lots of sites by hand as follows.
        <br/>
        <ol>
         <li>
          Prepare sites data
          <br/>
          <ol>
           <li>
            maybe you can export blocked sites from Google
           </li>
           <li>
            remove protocol(http://, https://) from url
           </li>
           <li>
            append ".*" (wildcard) to url
           </li>
           <li>
            concatinate all sites into one line with space separator
           </li>
          </ol>
         </li>
         <li>
          Open about:config
         </li>
         <li>
          locate config "extensions.jid0-TpZJ4wPImlT1zIqfw58bD9vOeWQ@jetpack.unwantedSites"
         </li>
         <li>
          paste or append sites data
         </li>
        </ol>
