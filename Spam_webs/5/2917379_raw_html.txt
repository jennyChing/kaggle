    <div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en">
     <p>
      <b>
       Paraskevi
      </b>
      (
      <a href="/wiki/Greek_language" title="Greek language">
       Greek
      </a>
      :
      <span lang="el" xml:lang="el">
       
      </span>
      ,
      <i>
       Paraskeu
      </i>
      , literally "Preparation", and also the name of the "day of preparation" for the
      <a class="mw-redirect" href="/wiki/Seventh-day_Sabbath" title="Seventh-day Sabbath">
       Sabbath
      </a>
      , i.e.
      <a href="/wiki/Friday#Parasceve" title="Friday">
       Friday
      </a>
      ) is a female given name. Greek-language vriations include
      <b>
       Paraskevoula
      </b>
      (),
      <b>
       Voula
      </b>
      (),
      <b>
       Vivi
      </b>
      (),
      <b>
       Evi
      </b>
      (), etc. The male equivalent is
      <a href="/wiki/Paraskevas_(given_name)" title="Paraskevas (given name)">
       Paraskevas
      </a>
      ().
     </p>
     <p>
      Notable people with the name include:
     </p>
     <ul>
      <li>
       <a href="/wiki/Voula_Patoulidou" title="Voula Patoulidou">
        Voula Patoulidou
       </a>
       , Greek hurdler and long jumper
      </li>
      <li>
       <a href="/wiki/Paraskevi_Tsiamita" title="Paraskevi Tsiamita">
        Paraskevi Tsiamita
       </a>
       , Greek jumper
      </li>
      <li>
       <a href="/wiki/Voula_Zouboulaki" title="Voula Zouboulaki">
        Voula Zouboulaki
       </a>
       , Greek actress
      </li>
      <li>
       <a href="/wiki/Paraskevi_Papachristou" title="Paraskevi Papachristou">
        Paraskevi Papachristou
       </a>
       , Greek athlete
      </li>
     </ul>
     <h2>
      <span class="mw-headline" id="Placename">
       Placename
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Paraskevi&amp;action=edit&amp;section=1" title="Edit section: Placename">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      It is also the name of two villages in Greece:
     </p>
     <ul>
      <li>
       <a href="/wiki/Paraskevi,_Achaea" title="Paraskevi, Achaea">
        Paraskevi, Achaea
       </a>
       , a village in
       <a href="/wiki/Achaea" title="Achaea">
        Achaea
       </a>
      </li>
      <li>
       <a href="/wiki/Paraskevi,_Grevena" title="Paraskevi, Grevena">
        Paraskevi, Grevena
       </a>
       , a village in
       <a href="/wiki/Grevena_(regional_unit)" title="Grevena (regional unit)">
        Grevena
       </a>
      </li>
     </ul>
