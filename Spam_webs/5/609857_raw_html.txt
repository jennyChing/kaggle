         WiFi Radar is a Python/PyGTK2 utility for managing WiFi profiles.It enables you to scan for available networks and create profiles for your preferred networks. At boot time, running WiFi Radar will automatically scan for an available preferred network and connect to it. You can drag and drop your preferred networks to arrange the profile priority.
         <p>
          <span id="more-392">
          </span>
          <strong>
           Install WiFi Radar in ubuntu
          </strong>
         </p>
         <p>
          sudo aptitude install wifi-radar
         </p>
         <p>
          This will complete the installation.
         </p>
         <p>
          <strong>
           Using WiFi-Radar
          </strong>
         </p>
         <p>
          If you want to open goto Applications---&gt;Internet---&gt;Wifi-radar
         </p>
         <p>
          <img align="middle" src="//www.ubuntugeek.com/images/radar/1.png"/>
         </p>
         <p>
          This will prompt for root password enter and click ok
         </p>
         <p>
          <img align="middle" src="//www.ubuntugeek.com/images/radar/2.png"/>
         </p>
         <p>
          This should detect available wireless networks check your wireless and connect
         </p>
         <p>
          <img align="middle" src="//www.ubuntugeek.com/images/radar/5.png"/>
         </p>
         <p>
          If you want to create new profile open the application and click on new
         </p>
         <p>
          <img align="middle" src="//www.ubuntugeek.com/images/radar/3.png"/>
         </p>
         <p>
          You can see similar to the following scree enter all the details and you are ready for wireless
         </p>
         <p>
          <img align="middle" src="//www.ubuntugeek.com/images/radar/4.png"/>
         </p>

               Wifi Radar has a bug. Change the command line in the menu to:
              </p>
              <p>
               gksu /usr/sbin/wifi-radar
              </p>
              <p>
               and it will work fine  have fun setting it up tho!
              </p>
              <p>
               Atheros AR5007
               <img alt=":)" class="wp-smiley" src="//www.ubuntugeek.com/wp-includes/images/smilies/simple-smile.png" style="height: 1em; max-height: 1em;"/>
               This took ages and I finally got the solution at:
               <br/>
               <a href="http://ubuntuforums.org/showthread.php?p=6201697#post6201697" rel="nofollow">
                http://ubuntuforums.org/showthread.php?p=6201697#post6201697
               </a>
              </p>
              <p>
               which is:
              </p>
              <p>
               sudo -s
               <br/>
               echo blacklist ath_pci &gt;&gt; /etc/modprobe.d/blacklist
               <br/>
               apt-get install ndiswrapper*
              </p>
              <p>
               wget
               <a href="ftp://ftp.work.acer-euro.com/notebook/aspire_4710/driver/Wireless_Atheros.zip" rel="nofollow">
                ftp://ftp.work.acer-euro.com/notebook/aspire_4710/driver/Wireless_Atheros.zip
               </a>
              </p>
              <p>
               unzip Wireless*
               <br/>
               cd Atheros
               <br/>
               unzip HR*
               <br/>
               ndiswrapper -i net5211.inf
               <br/>
               echo ndiswrapper &gt;&gt; /etc/modules
              </p>
              <p>
               sudo reboot
              </p>
              <p>
               Sets the AR5007 up perfectly just remember that if youre using knetworkmanager that it doesnt handle WPA easily  the only way Ive got it to work is by using WEP encryption (not as good but less hassle)
              </p>

               And as far Wifi Radar.. well I guess that life is just sometimes too short :p
