          <strong>
           Spannender Science Fiction-Film mit faszinierender Idee:
           <img alt="4.5 Stars" class="usr" src="http://www.leselink.de/wp-content/plugins/universal-star-rating/includes/stars.php?img=01.png&amp;px=15&amp;max=5&amp;rat=4.5"/>

           <img alt="Surrogates-Hauptdarsteller Bruce Willis - der Echte" class="size-medium wp-image-16178" height="300" src="http://www.leselink.de/wp-content/uploads/BruceWillisLFDHpremiere07-263x300.jpg" width="263"/>

           Surrogates-Hauptdarsteller Bruce Willis  der Echte

         <p>
          Im Jahr 2054 hat sich einer der groen Trume der Menschheit erfllt: Man muss sich die Hnde nicht mehr selbst schmutzig machen. Mehr als 90% aller Menschen besitzen sogenannte Surrogates, menschen-hnliche Roboter, die sie im Alltag bei allen Aufgaben ersetzen. Das Ganze funktioniert mittels neurologischer Fernsteuerung: Der Operator liegt bequem im Stuhl zu Hause, empfindet durch die Sinne seines Surrogates und verlsst die Wohnung nicht mehr. Entsprechend ist die Kriminalittsrate drastisch gesunken, denn auf den Straen der groen Stdte laufen nur noch Roboter herum, denen Verletzungen keinen Schaden zufgen. Nur ein paar Aussteiger haben sich in Roboter-freien Reservaten zusammengetan, um das echte Leben nicht aus den Hnden zu geben.
         </p>
         <p>
          In dieser nahezu risikofreien Welt lst ein Doppel-Mord tiefes Entsetzen bei der Polizei aus: Ausgerechnet Jared Canter, Sohn des Surrogates-Erfinders Lionel Canter, wird mit einer Zufallsbekanntschaft vor einem Club ermordert. Wie das Ganze berhaupt mglich ist, stellt die Ermittler vor ein groes Rtsel, denn Canter und seine Begleitung waren selbstverstndlich in Form ihrer Surrogates unterwegs. Doch irgendetwas muss sie derart getroffen haben, dass auch die friedlich zu Hause im Liegestuhl verharrenden Operator davon gettet wurden.
         </p>

         <h2>
          Surrogates: Das bessere Ich
         </h2>
         <table style="color: #0d304b; float: right; border: 1px solid #888888; margin-right: 10px; margin-bottom: 6px; padding: 5px;">
          <colgroup>
           <col width="120%">
           </col>
          </colgroup>
          <tbody>
           <tr>
            <td>
             <strong>
              Surrogates  Mein zweites Ich
              <br/>
              (Surrogates)
             </strong>
             <br/>
             USA, 2009
             <br/>
             88 Minuten
             <br/>
             Filmverleih: Disney / Touchstone
             <br/>
             Regie: Jonathan Mostow
             <br/>
             Drehbuch: Michael Ferris
             <br/>
             mit Bruce Willis, Radha Mitchell
             <br/>
             FSK: frei ab 12
            </td>
           </tr>
           <tr>
            <td align="center">
             <iframe frameborder="0" height="150" marginheight="0" marginwidth="0" scrolling="no" src="http://rcm-eu.amazon-adsystem.com/e/cm?t=leselink-21&amp;o=3&amp;p=8&amp;l=as1&amp;asins=B00377IRTY&amp;ref=qf_sp_asin_til&amp;fc1=23272E&amp;IS2=1&amp;lt1=_blank&amp;m=amazon&amp;lc1=1743C5&amp;bc1=FFFFFF&amp;bg1=FFFFFF&amp;npa=1&amp;f=ifr" style="width: 120px; height: 240px;" width="300">
             </iframe>
            </td>
           </tr>
          </tbody>
         </table>
         <p>
          Um eine Massen-Panik zu verhindern, wird zunchst eine Nachrichten-Sperre verhngt. Auf den Plan gerufen werden die beiden Polizisten Tom Greer (
          <a href="http://www.leselink.de/tag/bruce-willis" title="Bruce Willis">
           <em>
            Bruce Willis
           </em>
          </a>
          ) und seine Partnerin Jennifer Peters (
          <em>
           Radha Mitchell
          </em>
          ), die zunchst im Umfeld von VSI ermitteln, dem Unternehmen, das die Surrogates herstellt und das den Mit-Grnder Canter ein paar Jahre zuvor aus der Geschftsfhrung geworfen hat. Eine Waffe, die vom Surrogate auf den Operator durchschlagen kann, kann sich angeblich niemand vorstellen, und doch ist Greer sich sicher, dass man sich schon ziemlich gut mit den Ersatz-Menschen auskennen muss, um ein Verbrechen wie das Geschehene ermglichen zu knnen.
         </p>
         <p>
          <em>
           Surrogates  Mein zweites Ich
          </em>
          scheint auf den ersten Blick aufs DVD-Cover  nicht zuletzt dank des action-liebenden Hauptdarstellers  eines dieser halbgaren Science Fiction-Action-Spektakel zu sein, die man sich getrost sparen kann. Dieses (Vor-)Urteil wird dem Film jedoch keinesfalls gerecht, denn nicht nur die Idee ist faszinierend, sondern auch die Details der Umsetzung. Mit den Surrogates erschaffen sich die Menschen einen Doppelgnger, der so ist, wie sie es gerne sein wollen: jnger, schner, beweglicher  und doch im Kern sie selbst. Das ursprngliche Ich bleibt zu Hause hinter verschlossenen Tren und wird nicht mal mehr  wie im Fall von Tom Greers Frau  dem Ehepartner zugemutet. So viel Oberflche ldt natrlich ein zu religisen und philosophischen Betrachtungen, und dass die Keine Roboter-Gruppierung um den Propheten keine friedliebende Natur-Religion, sondern gewaltbereite Sekte, ist zeigt deutlich,
          <em>
           was
          </em>
          die anderen Menschen da zu Hause lassen.
         </p>
         <p>
          Die Frage, die der Film sich stellt, lautet, ob man lieber ein unzulngliches, mit Fehlern und Mngeln behaftetes Leben fhren mchte, oder eine perfektionierte Kopie davon. Auch wenn
          <em>
           Surrogates  Mein zweites Ich
          </em>
          die Antwort nicht ganz eindeutig vorgibt, so deutet schon die Tatsache, dass auch die Emotionen, die Nhe zu anderen, das Erleben in der zweiten Variante eine Kopie sind, darauf hin, dass es eigentlich nur eine sinnvolle Entscheidung gibt.
         </p>

String.prototype.trim = function() {
  return this.replace(/^\s+|\s+$/g,"");
}

function createCookie(name,value,days) {
  if (days) {
    var date = new Date();
    date.setTime(date.getTime()+(days*24*60*60*1000));
    var expires = "; expires="+date.toGMTString();
  } else {
    var expires = "";
  }
  document.cookie = name+"="+value+expires+"; path=/";
}

function readCookie(name) {
  var nameEQ = name + "=";
  var ca = document.cookie.split(';');
  for(var i=0;i &lt; ca.length;i++) {
    var c = ca[i];
    while (c.charAt(0)==' ') {
      c = c.substring(1,c.length);
    }
    if (c.indexOf(nameEQ) == 0) {
      return c.substring(nameEQ.length,c.length);
    }
  }
  return null;
}

function eraseCookie(name) {
  createCookie(name,"",-1);
}

      var theDiv = jQuery(this).parent().parent().find('span.collapse').parent().find('div');

      jQuery(this).parent().parent().find('span.collapse').removeClass('collapse').addClass('expand');
        createCookie(theDiv.attr('id'), 0, 7);

    jQuery('#' + id + ' .expand .sym').html(expandSym);

    return false;
  });
  jQuery('#' + id + ' .collapse').live('click', function() {
    collapseCat(this, expandSym, collapseSym);
    return false;
  });
}
