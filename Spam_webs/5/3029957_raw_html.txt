            It wasn't until I'd saved most of these that I realised that the author/illustrator of this book was none other than the dinosaur sculptor,
            <a href="http://bibliodyssey.blogspot.com/2006/02/sketchbook-of-dinosaur-builder.html">
             Benjamin Waterhouse Hawkins - from a couple of days ago
            </a>
            . Vincent Brooks completed the 10 lithographic plates.
            <br/>
            <br/>
            Hawkins intended with his publication "to give a comparative view of the variation in form of the bony skeleton or framework of those animals most frequently required by the artist, designer, or ornamentist."
            <br/>
            <br/>
            The accompanying text didn't incorporate Darwinian evolutionary theory and I get the strong feeling that either Hawkins himself or at the behest of Sir Richard Owens, held contrary views. Nevertheless the work is regarded as a positive contribution to the understanding of mammalian anatomy.
            <br/>
            <br/>
            <a href="http://digital.library.wisc.edu/1711.dl/HistSciTech.CompAnat">
             <span style="font-style: italic;">
              A Comparative View of the Human and Animal Frame
             </span>
             by Benjamin Waterhouse Hawkins 1860 is online among the History of Science website at the fabulous University of Wisconsin
            </a>
            - my favourite repository.
            <br/>
            <br/>
            <small>
             [I'm away in the countryside at the moment so posting is a tad intermittent. Sunstroke from sailing and a dodgy internet connection hasn't helped. Things should be back to normal in a few days.]
            </small>
           </p>

function randomize (a_items, n_count) {
    var n_index, s_html = '&lt;ul&gt;';
    while (a_items.length &amp;&amp; n_count) {
        n_index = Math.ceil(Math.random() * a_items.length) - 1;
        s_html += '&lt;li&gt;' + a_items[n_index] + '&lt;/li&gt;';
        a_items[n_index] = a_items[a_items.length - 1];
        a_items.length = a_items.length - 1;
        n_count--;
    }
    return s_html + '&lt;/ul&gt;';
}

function randomize (a_items, n_count) {
    var n_index, s_html = '&lt;ul&gt;';
    while (a_items.length &amp;&amp; n_count) {
        n_index = Math.ceil(Math.random() * a_items.length) - 1;
        s_html += '&lt;li&gt;' + a_items[n_index] + '&lt;/li&gt;';
        a_items[n_index] = a_items[a_items.length - 1];
        a_items.length = a_items.length - 1;
        n_count--;
    }
    return s_html + '&lt;/ul&gt;';
}

function randomize (a_items, n_count) {
    var n_index, s_html = '&lt;ul&gt;';
    while (a_items.length &amp;&amp; n_count) {
        n_index = Math.ceil(Math.random() * a_items.length) - 1;
        s_html += '&lt;li&gt;' + a_items[n_index] + '&lt;/li&gt;';
        a_items[n_index] = a_items[a_items.length - 1];
        a_items.length = a_items.length - 1;
        n_count--;
    }
    return s_html + '&lt;/ul&gt;';
}
