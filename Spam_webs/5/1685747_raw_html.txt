     <p>
      <b>
       Semacode
      </b>
      is a software company based in
      <a href="/wiki/Waterloo,_Ontario" title="Waterloo, Ontario">
       Waterloo, Ontario
      </a>
      ,
      <a href="/wiki/Canada" title="Canada">
       Canada
      </a>
      .
      <sup class="reference" id="cite_ref-1">
       <a href="#cite_note-1">
        <span>
         [
        </span>
        1
        <span>
         ]
        </span>
       </a>
      </sup>
      It is also this company's trade name for machine-readable
      <a href="/wiki/International_Organization_for_Standardization" title="International Organization for Standardization">
       ISO
      </a>
      /IEC 16022
      <a class="mw-redirect" href="/wiki/Data_matrix_(computer)" title="Data matrix (computer)">
       data matrix
      </a>
      <a href="/wiki/Symbol" title="Symbol">
       symbols
      </a>
      , a type of
      <a href="/wiki/Barcode" title="Barcode">
       barcode
      </a>
      resembling a
      <a class="mw-redirect" href="/wiki/Crossword_puzzle" title="Crossword puzzle">
       crossword puzzle
      </a>
      , which encode
      <a href="/wiki/Internet" title="Internet">
       Internet
      </a>
      <a class="mw-redirect" href="/wiki/Uniform_Resource_Locator" title="Uniform Resource Locator">
       URLs
      </a>
      .
     </p>
     <p>
      Semacodes are primarily aimed at being used with
      <a class="mw-redirect" href="/wiki/Cellular_phone" title="Cellular phone">
       cellular phones
      </a>
      which have built-in
      <a href="/wiki/Camera" title="Camera">
       cameras
      </a>
      , to quickly capture a
      <a class="mw-redirect" href="/wiki/Web_site" title="Web site">
       Web site
      </a>
      address for use in the phone's
      <a href="/wiki/Web_browser" title="Web browser">
       web browser
      </a>
      . Semacodes are in fact
      <a class="mw-redirect" href="/wiki/DataMatrix" title="DataMatrix">
       DataMatrix
      </a>
      encoded URLs.
     </p>
     <p>
      The Semacode website states that Semacode tags are an "open system" and that tag creation is "completely unrestricted," with the SDK software tools being free of charge for non-commercial use.
      <sup class="reference" id="cite_ref-2">
       <a href="#cite_note-2">
        <span>
         [
        </span>
        2
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <p>
      Potential uses for Semacode tags are still being explored, and will complement development of the concept of using mobile phones as devices for information gathering and exchange. Suggestions from the Semacode.org website
      <sup class="reference" id="cite_ref-3">
       <a href="#cite_note-3">
        <span>
         [
        </span>
        3
        <span>
         ]
        </span>
       </a>
      </sup>
      included:
     </p>
     <ul>
      <li>
       placing Semacode tags on posters, such as those for concerts and public performances. Those interested could use their mobile phone to take a photo of the tag, which could link them directly to the web page where they could order tickets.
      </li>
      <li>
       using Semacode tags and mobile phones to enable multilingual museum exhibits - a tag photographed at the exhibition entrance could set a language
       <a href="/wiki/HTTP_cookie" title="HTTP cookie">
        cookie
       </a>
       in the phone's web browser, and subsequent Semacode tags displayed at each exhibit could then link the phone's browser directly to a web page about the item, displayed in the user's language of choice.
      </li>
      <li>
       placing Semacode tags on
       <a href="/wiki/Name_tag" title="Name tag">
        name tags
       </a>
       given to conference attendees. These tags could provide the corporate web address of each attendee's company, or their biography and contact details.
      </li>
     </ul>
