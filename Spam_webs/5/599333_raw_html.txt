        <div class="entry-content">
         <p>
          <strong>
           Mango Skin Pack
          </strong>
          is a Windows 7 transformation pack for giving your desktop the Windows Phone 7 Mango look. This Skin Pack is part of transformation packs developed by Hamed, who previously brought us the
          <a href="http://www.addictivetips.com/windows-tips/transform-windows-7-into-ios-5-with-ios-skin-pack/">
           iOS
          </a>
          ,
          <a href="http://www.addictivetips.com/windows-tips/how-to-transform-windows-7-into-windows-8-skin-pack/">
           Windows 8
          </a>
          ,
          <a href="http://www.addictivetips.com/windows-tips/how-to-transform-windows-7-into-mac-os-x-10-7-lion/">
           Mac OS X
          </a>
          ,
          <a href="http://www.addictivetips.com/windows-tips/transform-windows-7-into-ubuntu-11-04-natty-narwhal-ubuntu-skin-pack/">
           Ubuntu
          </a>
          and
          <a href="http://www.addictivetips.com/windows-tips/transform-windows-7-into-android-like-look-with-android-skin-pack/">
           Android
          </a>
          skin packs.
         </p>
         <p>
          <span id="more-79285">
          </span>
         </p>
         <div id="single-paragraph-add">
         </div>
         <blockquote>
          <p>
           <strong>
            Note:
           </strong>
           Please bear in mind that such transformation packs are difficult to uninstall and a system restore point will not revert the changes. Before installing the Mango Skin Pack, make sure that you backup your computer to revert the changes (if required). Moreover, make sure that UAC (User Account Control) is disabled.
          </p>
         </blockquote>
         <p>
          To get started, launch the EXE file and accept the terms and conditions. You can either go for the quick installation process or choose Custom Install option to uncheck unwanted items. It must be noted that this skin pack is ad-supported as it attempts to set the developers Skin Page as your default home page. You can uncheck this option during the custom installation.
         </p>
         <p>
          <img alt="Custom" border="0" height="426" src="http://cloud.addictivetips.com/wp-content/uploads/2011/10/Custom.jpg" style="background-image: none; margin: 0px 0px 18px; padding-left: 0px; padding-right: 0px; display: inline; padding-top: 0px; border: 0px;" title="Custom" width="541"/>
         </p>
         <p>
          When installing this transformation pack, you will be presented with the option to install the UXTheme Patch, DLL files and other extra items (e.g. wallpaper, visual styles, etc). You may select all or some of these options to continue. Click Install to begin the installation process.
         </p>
         <p>
          <img alt="Extras" border="0" height="426" src="http://cloud.addictivetips.com/wp-content/uploads/2011/10/Extras.jpg" style="background-image: none; margin: 0px 0px 18px; padding-left: 0px; padding-right: 0px; display: inline; padding-top: 0px; border: 0px;" title="Extras" width="541"/>
         </p>
         <p>
          This will install all required components of the skin pack. Once installation is complete, reboot the system to get the Windows Phone 7 Mango look for your Windows 7 desktop.
          <img alt="Installing" border="0" height="426" src="http://cloud.addictivetips.com/wp-content/uploads/2011/10/Installing1.jpg" style="background-image: none; margin: 0px 0px 18px; padding-left: 0px; padding-right: 0px; display: inline; padding-top: 0px; border: 0px;" title="Installing" width="541"/>
         </p>
