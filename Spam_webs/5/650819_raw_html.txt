         <h3 class="solution-subheader">
          Issue
         </h3>
         <div class="message-important">
          <h3 class="message-title">
           Remove previous Antivirus software!
          </h3>
          <p>
           It is important that any previously installed antivirus software is    uninstalled from your system before installing your ESET product.   Typical uninstallation* using the Windows removal tool  (Add/Remove   Programs) is often insufficient. The links below are for removing  antivirus products installed on your  system prior to the installation  of ESET. If you experience issues  using any of the tools below, contact  the vendor for assistance.
          </p>
         </div>
         <p>
          Looking for malware removal tools?
          <a href="/esetkb/index?page=content&amp;id=SOLN2372&amp;_ref=SOLN146" target="_self">
           Visit our list of available malware removal tools
          </a>
          .
         </p>
         <p>
          <a href="/esetkb/index?page=content&amp;id=SOLN2549" target="_blank">
           <strong>
            <img align="absMiddle" alt="" height="23" src="/library/ESET/KB%20Team%20Only/mac%20icons/finder.png" width="23"/>
            Are you a Mac user?
           </strong>
          </a>
         </p>
        </div>
       </div>

         <h3 class="solution-subheader">
          Solution
         </h3>
         <h3>
          <u>
           ESET AV Remover tool
          </u>
         </h3>
         <p>
          <br/>
          The ESET AV Remover tool will remove almost any antivirus software  previously installed on your system. Click the appropriate link below to download ESET AV Remover, we recommend that you save the file to your Desktop. If you don't know which version to use,
          <a href="/esetkb/index?page=content&amp;id=SOLN523" target="_blank">
           click here for instructions to find your system specifications
          </a>
          :
         </p>
         <p>
          <a class="btn-b-1" href="http://download.eset.com/special/avremover/avremover_nt64_enu.exe" target="_blank">
           <span>
            Download 64-bit AV Remover
           </span>
          </a>
          <a class="btn-b-1" href="http://download.eset.com/special/avremover/avremover_nt32_enu.exe" target="_blank">
           <span>
            Download 32-bit AV Remover
           </span>
          </a>
         </p>
         <ul>
          <li>
           <a href="/esetkb/index?page=content&amp;id=SOLN3527" target="_blank">
            View step-by-step instructions to run the ESET AV Remover tool
           </a>
           <br/>
          </li>
          <li>
           <a href="http://kb.eset.com/esetkb/index?page=content&amp;id=SOLN3527#removable" target="_blank">
            View a list of applications that can be removed using the ESET AV Remover tool
            <br/>
           </a>
          </li>
         </ul>
         <p>
          If unsuccessful, or unable to  identify the program installed, the tool  will direct you back to the  alphabetical list of uninstallers below.
         </p>
         <hr/>
         <h3>
          <br/>
          <u>
           List of manual antivirus uninstallers
          </u>
         </h3>
         <ol>
          <li>
           Click the link associated with the antivirus software currently  installed on your system and follow the prompts or instructions on the  website.
           <br/>
          </li>
          <li>
           Once uninstallation is complete, you will need to restart your  computer. You  may then proceed with installing your ESET product.
          </li>
         </ol>
