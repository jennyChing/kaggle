        <a href="/wiki/Nexus_S" title="Nexus S">
         Nexus S
        </a>
        , allow installation of applications from the
        <a class="mw-redirect" href="/wiki/Play_Store" title="Play Store">
         Play Store
        </a>
        ,
        <a href="/wiki/F-Droid" title="F-Droid">
         F-Droid
        </a>

        <a class="mw-redirect" href="/wiki/.apk" title=".apk">
         APK files
        </a>
       </div>
      </div>

     <p>
      This is an incomplete list of notable applications (
      <i>
       apps
      </i>
      ) that run on the
      <a class="mw-redirect" href="/wiki/Android_platform" title="Android platform">
       Android platform
      </a>
      that meet the guidelines for
      <a href="/wiki/Free_software" title="Free software">
       free software
      </a>
      and
      <a href="/wiki/Open-source_software" title="Open-source software">
       open-source software
      </a>
      . For a more extensive list of apps, see the
      <a href="#External_links">
       External links
      </a>
      section below.
     </p>
