     <h2 class="g">
      <a class="blogTitle" href="http://brian.shaler.name/pages/blog/hey-ya-typography-animation/" rel="bookmark">
       'Hey Ya' Flash Typography Animation Music Video
      </a>
     </h2>
     <span class="blogDate">
      May 30, 2007
     </span>
     <p>
     </p>
     <div style="float: left; width: 60px;">
      <script>
       digg_url = 'http://digg.com/design/Typography_Music_Video_Created_in_Flash_by_Brian_Shaler';
      </script>
      <script src="http://digg.com/api/diggthis.js">
      </script>
     </div>
     Using Obadiah Parker's cover of Outkast's "Hey Ya", I made this Adobe Flash typography animation.
     <br/>
     <br/>
     It was inspired by animations like
     <a class="darkBlueLink" href="http://youtube.com/watch?v=WXxouXt5dz4" target="_blank">
      The Hush Sound's "Lion's Roar"
     </a>
     , and movie scene animations like those for
     <a class="darkBlueLink" href="http://youtube.com/watch?v=syf8olcM0z4" target="_blank">
      Pulp Fiction
     </a>
     and
     <a class="darkBlueLink" href="http://youtube.com/watch?v=rIDdx7NPJgo" target="_blank">
      Ocean's Eleven
     </a>
     .
     <br/>
     <br/>
     It was a very fun exercise in freestyle Flash animation and linear timeline tweening (which was a nice change from my specialty: programmatic animation (though, I used some programming in the later scenes of this video).
     <br/>
     <br/>
     I rendered the output to a video to be easier for people to view, and to save some bandwidth.
     <br/>
     <br/>
