       <h1 class="entry-title" itemprop="headline">
        AVZ Antiviral Toolkit  advanced scanner and manager
       </h1>
      </header>
      <div class="entry-content" itemprop="articleBody">
       <p>
        <img align="right" alt="avz_icon" border="0" height="48" src="http://www.rarst.net/images/AVZAntiviralToolkitadvancedscannerandman_130BD/avz_icon.png" style="" title="avz_icon" width="48"/>
        <a href="http://www.rarst.net/software/combofix/">
         ComboFix
        </a>
        may be last resort that to blow away stubborn
        <a href="http://www.rarst.net/tag/malware/">
         malware
        </a>
        (with parts of Windows while at it :) but when even that fails there is no choice left except down and dirty hunt after pesky malware.
       </p>
       <p>
        <strong>
         AVZ is set of utilities honed to detect all flavors of hard to detect malware
        </strong>
        (such as rootkits). And to get rid of it.
       </p>
       <p>
        <span id="more-465">
        </span>
       </p>
       <h3>
        What it does
       </h3>
       <p>
        AVZ combines few related functions and capabilities for viable chance to detect malware that hides really well. Basically it:
       </p>
       <ul>
        <li>
         detects
         <strong>
          things that interfere or integrate too deep
         </strong>
         with system;
        </li>
        <li>
         checks them for signs of malware behavior;
        </li>
        <li>
         runs against white list of known safe components;
        </li>
        <li>
         provides tools to remove malware, while protecting itself and other tools from it.
        </li>
       </ul>
       <img alt="avz_inerface" border="0" height="474" src="http://www.rarst.net/images/AVZAntiviralToolkitadvancedscannerandman_130BD/avz_inerface.png" style="" title="avz_inerface" width="540"/>
       <p>
        It has regular scan process with option to scan files, but there is extensive set of
        <strong>
         tools to manage autorun, processes and pretty much every other place
        </strong>
        that might get unwanted malware hooked to it.
       </p>
       <h3>
        Case study
       </h3>
       <p>
        I had
        <span style="text-decoration: line-through;">
         freaking ugly
        </span>
        very educational encounter today with computer that was complete mess of driver conflicts (cracked Daemon Tools Pro had showdown with cracked
        <a href="http://www.rarst.net/tag/drweb/">
         DrWeb
        </a>
        , ouch).
       </p>
       <p>
        After few hours of untangling and
        <a href="http://www.rarst.net/software/sfc/">
         restoring system files
        </a>
        I had system perfectly clean and working Except that
        <strong>
         explorer.exe process completely trashed it with maxed out CPU usage
        </strong>
        and I had no idea where it came from. And 3.4GHz Pentium D spiking CPU makes computing experience really miserable.
       </p>
       <p>
        <a href="http://www.rarst.net/software/process-explorer/">
         Process Explorer
        </a>
        and
        <a href="http://www.rarst.net/software/process-monitor/">
         Process Monitor
        </a>
        couldnt detect a thing  all that came up was
        <strong>
         thread that originated out of non-descript memory space and spiked CPU, without any file activity
        </strong>
        . System was earlier through every
        <a href="http://www.rarst.net/software/choosing-portable-antivirus-clamwin-vs-cureit/">
         portable antivirus
        </a>
        scanner I use, so as clean as I could get it. It was complete
        <abbr title="sooo, who watches Defying Gravity? :)">
         H2IK sequence
        </abbr>
        , worst I saw lately.
       </p>
       <p>
        Quick (literally) scan with AVZ lighted up some
        <strong>
         obscure piece of driver
        </strong>
        (let it rest in peace)
        <strong>
         that interfered with numerous system APIs
        </strong>
        . And all was good without it.
       </p>
       <h3>
        Strong features
       </h3>
       <ul>
        <li>
         every sub-tool in AVZ makes use of main engine, so sees more and better than generic managers;
        </li>
        <li>
         AVZPM  optional driver for extensive process checks;
        </li>
        <li>
         AVZGuard  tool that restricts process and
         <a href="http://www.rarst.net/tag/registry/">
          registry
         </a>
         activity, except from trusted executables to guard them against aggressive malware;
        </li>
        <li>
         saves the day.
        </li>
       </ul>
       <h3>
        Overall
       </h3>
       <p>
        AVZ is semi-official product of Kaspersky Labs (seems they got it by hiring developer). Parts of AVZ are reused in other Kaspersky products such as
        <a href="http://www.rarst.net/software/kaspersky-virus-removal-tool/">
         AVPTool
        </a>
        , but as for manual diagnostics it still has edge over rest.
       </p>
       <p>
        <a href="http://www.rarst.net/software/portable-definition/">
         Works without installation
        </a>
        as any decent anti-malware tool should. And if you are down in trouble deep to this one then you wont care about traces left.
       </p>
       <p>

         Home (in Russian)

       </p>
