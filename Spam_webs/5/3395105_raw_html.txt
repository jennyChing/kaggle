              <a href="http://www.toolwar.com/2013/12/turbo-assembler-tools.html" itemprop="url">
               Turbo Assembler :: Tools
              </a>
             </h1>
             <div class="post-body entry-content" id="post-body-4863468114781327237" itemprop="description articleBody">
              <div style="text-align: center;">
               <img alt="Turbo Assembler" border="0" height="110" src="http://3.bp.blogspot.com/-8QGRrt7QE94/UrPVFUHLm1I/AAAAAAAABR8/B5_YrRBCB7M/s400/turbo+assembler.gif" title="Turbo Assembler" width="400"/>
               <b>
               </b>
              </div>
              <div style="text-align: justify;">
               <b>
                Turbo Assembler 5.0
               </b>
               is a full featured
               <b>
                stand-alone assembler.
               </b>
               This product includes all the
               <a href="http://www.toolwar.com/search/label/Tools" target="_blank">
                tools
               </a>
               needed to create and debug assembly programs for 16 and 32 bit DOS and
               <a href="http://www.toolwar.com/search/label/Windows" target="_blank">
                Windows
               </a>
               platforms, including Windows 3.X, Win95, Win98, and NT. Some of the tools included are assemblers, linkers, console style
               <a href="http://www.toolwar.com/search/label/Debugger" target="_blank">
                debuggers,
               </a>
               and resource compilers. Each of these tools comes in a 16 bit and a 32 bit version.
              </div>
              <div style="text-align: justify;">
               <b>
                C++Builder 5
               </b>
               Professional and Enterprise both include and integrate
               <b>
                TASM32
               </b>
               , the 32 bit assembler, with the IDE, allowing you to write or add inline assembly code or assembly sources to your 32bit Windows C/C++ projects. For building and debugging stand alone assembly programs for DOS and Windows, Turbo Assembler 5.0 is recommended.
              </div>
              <div style="text-align: justify;">
               Create Fast Applications, Faster Order the world-standard
               <b>
                Borland Turbo Assembler
               </b>
               and see how lightning-fast assembly speeds can turbocharge your turnaround times. In tests, the Borland Turbo Assembler proved itself up to 70% faster than other popular macro assemblers, with assembly speeds of up to 48,000 lines per minute! And it's versatile, with full Intel chip set support-from 8086 to Pentium.
              </div>
              <div style="text-align: justify;">
               IDEAL for Beginners and Pros . If you're just starting out in assembly
               <a href="http://www.toolwar.com/search/label/Language" target="_blank">
                language
               </a>
               programming,
               <b>
                Turbo Assembler's
               </b>
               innovative IDEAL assembly mode will help you get productive faster and catch more errors sooner. And if you're a C++ professional, Turbo Assembler's advanced coding instructions will help you get the most from your assembly language programs. You'll especially like how Turbo Assembler's MASM compatibility mode lets you add new life to legacy code.
              </div>
              <div style="text-align: justify;">
               <h3>
                <b>
                 Specifications ::
                </b>
               </h3>
              </div>
              <ul style="text-align: justify;" type="disc">
               <li>
                Up to 48,000 lines-per-minute assembly
               </li>
               <li>
                Full 8088, 8086, 80286, 80386, i486, and Pentium support
               </li>
               <li>
                IDEAL and MASM assembly modes
               </li>
               <li>
                Interface support for C, C++, Pascal, FORTRAN, and COBOL
               </li>
               <li>
                Multi-pass assembler with forward-reference resolution
               </li>
               <li>
                Fast 16- and 32-bit Turbo Linker
               </li>
               <li>
                Turbo Debugger for DOS and Windows
               </li>
              </ul>
