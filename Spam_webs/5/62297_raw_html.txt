           <title>
            Dual Booting Slax Linux and BartPE (Windows) from a USB Thumbdrive (UFD)
           </title>
           <p align="center">
            <b>
             <font size="5">
              Dual Booting Slax Linux and BartPE (Windows) 
from a USB Thumbdrive (UFD)
             </font>
            </b>
           </p>
           <p>
            <font size="2">
             Update 11/30/2006: Check out the video version of this 
article:
            </font>
            <a href="http://www.irongeek.com/i.php?page=videos/backtrack-bartpe-dual-boot">
             <font size="2">
              http://www.irongeek.com/i.php?page=videos/backtrack-bartpe-dual-boot
             </font>
            </a>
           </p>
           <p>
            While I was researching for my 
PhreakNIC X presentation on Bart's PE Builder I began to play around with 
booting it from a USB Thumbdrive (Also know as a USB Flash Drive - UFD). In the 
process, I began to wonder if you could dual boot it with Slax Linux to make a 
very handy recovery system you could carry around in your pocket. I found Thibeact's 
notes on the 911 CD forums (
            <a href="http://www.911cd.net/forums//index.php?showtopic=10806&amp;st=220&amp;p=98227">
             http://www.911cd.net/forums//index.php?showtopic=10806&amp;st=220&amp;p=98227entry98227
            </a>
            ) and modified them to fit my needs.
           </p>
           <p>
            What follows are the basic steps 
for dual booting Slax Linux and BartPE from a simple USB flash drive. I find 
this very useful for changing password, removing spyware and doing various other 
security related tasks. Check out my PhreakNIC presentation before you begin so 
understand the basics of using Pe Builder (
            <a href="http://www.irongeek.com/i.php?page=videos/barts-pe-builder-intro">
             Creating 
a Windows Live CD for System Recovery and Pen-Testing with Bart's PE Builder
            </a>
            ) 
then follow these steps to dual boot it with Slax using a Windows XP system to 
install everything:
           </p>
           <p>
            1. First, download, install and run MySlax on your Windows PC. You can obtain 
MySlax from:
           </p>
           <p>
            <a href="http://myslax.bonsonno.org/">
             http://myslax.bonsonno.org/
            </a>
           </p>
           <p>
            2. Once you have MySlax running choose the ISO-Default radio button, choose 
the Slax based ISO you wish to use, click the "Mount" button then hit the "USB 
Stick" button. You should be able to use any Slax based ISO as your source, 
including normal
            <a href="http://www.slax.org/download.php">
             Slax
            </a>
            ,
            <a href="http://www.remote-exploit.org/index.php/BackTrack">
             BackTrack
            </a>
            or
            <a href="http://slast.org/">
             Slast
            </a>
            .
           </p>
           <p align="center">
            <img border="0" height="490" src="images/dualbootusb/dualbootusb1.png" width="699"/>
           </p>
           <p>
            3. Click the "Create USB Stick" button, then when the format dialog comes up 
make sure you choose FAT (not FAT32) and click start. The UFD must be formatted 
as Fat16 for BartPE to work right in the later steps.
           </p>
           <p align="center">
            <img border="0" height="490" src="images/dualbootusb/dualbootusb2.png" width="699"/>
           </p>
           <p>
            4. Click "Next" and MySlax will being putting the need files to boot Linux onto the UFD 
(USB Flash Disk). When it finishes confirm its completion by click the "Success" 
button.
           </p>
           <p>
            5. Now would be a good time to make sure you have a valid Pe Build to 
use. Start Pe Builder, set the media output to none since we won't be building a 
normal CD ISO, then click the "Build" button.
           </p>
           <p align="center">
            <img border="0" height="402" src="images/dualbootusb/dualbootusb3.png" width="488"/>
           </p>
           <p>
            6. Once PE Builder is done building your system, drop out to a command line, 
go into your Pe Builder install directory and issue the command "pe2usb.cmd 
&lt;USB-Drive-Letter&gt;" where &lt;USB-Drive-Letter&gt; is whatever drive letter your UFD 
is mount as. In my case that's F:.
           </p>
           <p align="center">
            <img border="0" height="415" src="images/dualbootusb/dualbootusb4.png" width="668"/>
           </p>
           <p>
            7. Next, edit syslinux.cfg in the root of your UFD and paste in the following 
contents:
           </p>
           <table border="1" id="table1" width="100%">
            <tr>
             <td>
              <font color="#009933" face="Courier New">
               default vesamenu.c32
               <br/>
               prompt 0
               <br/>
               timeout 0
               <br/>
               F1 splash.txt
               <br/>
               F2 splash.cfg
               <br/>
               SAY Type "bartpe" for Bart's Pe, or type "slax" for Slax.
               <br/>
               DISPLAY syslogo.lss
               <br/>
               MENU TITLE Irongeek's Boot UFD
               <br/>
               MENU BACKGROUND mascot.png
               <br/>
               label slax
               <br/>
               menu label Slax
               <br/>
               kernel vmlinuz
               <br/>
               append max_loop=255 initrd=initrd.gz vga=0x317 maxcpus=1 init=linuxrc 
		load_ramdisk=1 prompt_ramdisk=0 ramdisk_size=4444 root=/dev/ram0 rw
               <br/>
               <br/>
               label bartpe
               <br/>
               menu label Bart's PE
               <br/>
               kernel pe2usb.bss
               <br/>
               <br/>
               #label cluster
               <br/>
               # kernel vmlinuz
               <br/>
               # append max_loop=255 initrd=initrd.gz vga=0x317 maxcpus=1 init=linuxrc 
		load_ramdisk=1 prompt_ramdisk=0 ramdisk_size=4444 root=/dev/ram0 rw
               <br/>
               <br/>
               #label btdc
               <br/>
               # kernel vmlinuz
               <br/>
               # append max_loop=255 initrd=initrd.gz init=linuxrc load_ramdisk=1 
		prompt_ramdisk=0 ramdisk_size=4444 root=/dev/ram0 rw
               <br/>
               <br/>
               #label memtest
               <br/>
               # kernel memtest
              </font>
             </td>
            </tr>
           </table>
           <p>
            The syslinux.cfg file tells SysLinux (the boot loader) how you want things 
set up. I based mine on the BackTrack syslinux.cfg file. Take a look at the
            <a href="http://syslinux.zytor.com/">
             SysLinux documentation
            </a>
            if you want to 
change things around. The line "MENU BACKGROUND mascot.png" lets me use the 
following PNG as my background at the boot prompt:
           </p>
           <p align="center">
            <img border="0" height="480" src="images/dualbootusb/mascot.png" width="640"/>
           </p>
           <p>
            You may want to copy the above image to the root of your USB drive, or make a 
PNG of your own to use. As long as it's a 640x480 PNG named mascot.png it should 
work fine with my syslinux.cfg.
           </p>
           <p>
            8. Download Bart's mkbt.exe from:
            <br/>
            <a href="http://www.nu2.nu/mkbt/">
             http://www.nu2.nu/mkbt/
            </a>
           </p>
           <p>
            and place it at the root of your UFD. We will be using this tool later to fix 
the boot sector of the UFD.
           </p>
           <p>
            9. Copy pe2usb.bin from the root of your PE Builder install 
(c:\pebuilder3110a in my case) to the root of your UFD and renamed it to pe2usb.bss. 
This step is very important, the bss extention must be there so SysLinux 
knows what sort of kernel/boot sector it is.
           </p>
           <p align="center">
            <img border="0" height="389" src="images/dualbootusb/dualbootusb5.png" width="466"/>
           </p>
           <p>
            10. While Slax comes with a version of SysLinux, I prefer to download it 
myself and use a few extra tools to make things prettier. You can download 
SysLinux from:
           </p>
           <p>
            <a href="http://www.kernel.org/pub/linux/utils/boot/syslinux/">
             http://www.kernel.org/pub/linux/utils/boot/syslinux/
            </a>
           </p>
           <p>
            I chose the syslinux-3.31.zip archive. Once you have the archive downloaded 
extract the files \win32\syslinux.exe and \com32\modules\vesamenu.c32 then place 
them in the root of your Thumbdrive.
           </p>
           <p>
            11. Go to a command prompt, change to your UFD's drive letter and then issue the commands:
            <br/>
            mkbt -x pe2usb.bss &lt;USB-Drive-Letter&gt;
            <br/>
            syslinux 
&lt;USB-Drive-Letter&gt;
           </p>
           <p>
            as seen in the screen shot below:
           </p>
           <p align="center">
            <img border="0" height="323" src="images/dualbootusb/dualbootusb6.png" width="548"/>
           </p>
           <p>
            12. Now you can reboot the system and try to boot from your thumbdrive. Your 
BIOS must support booting from USB, so it's won't work on all systems.
           </p>
           <p>
           </p>
           <p>
            A quick side note, one of the 
great things about Slax is how easy it it to add extra software. Just 
download the modules you want from
