        <p>
         <em>
          This easy dinner recipe is made possible with product samples from Salba Chia but all opinions expressed are my own.
         </em>
        </p>
        <p>
         I try very hard to eat healthy on a semi regular basis. I provide healthy snacks like fruit and yogurt for my kids, we eat veggies at least once a day and we limit fast food to the occasional busy lunchtime treat. However, my kids are just like everyone elses in that they LOVE junkfood. The want pop tarts and candy and kids meals just like every other kid in America. I have been working to find a balance that we are both happy with. They want classic junk food and I want healthier options. Sometimes I hit upon an idea for a recipe that we can both be happy with and these Mexican Chicken Tenders are an easy dinner recipe that is also much healthier than the fast food your kids are probably begging for.
        </p>
        <p>
         <a href="http://www.suburbia-unwrapped.com/wp-content/uploads/2013/06/salba-chia-easy-dinner-recipes.jpg">
          <img alt="salba chia easy dinner recipes" class="aligncenter size-full wp-image-3063" height="720" src="http://www.suburbia-unwrapped.com/wp-content/uploads/2013/06/salba-chia-easy-dinner-recipes.jpg" width="540"/>
         </a>
        </p>
        <p>
         For this easy dinner recipe I used organic chicken breasts that were coated with a crumb mixture of
         <a href="http://www.salbasmart.com/" onclick="_gaq.push(['_trackEvent', 'outbound-article', 'http://www.salbasmart.com/', 'Salba Chia ']);" target="_blank">
          Salba Chia
         </a>
         tortilla chips, fresh cilantro and fajita seasoning mix. Processed in the Cuisinart food processor it only took a few seconds to make. Salba Chia tortilla chips are organic and Non GMO project verified. The chia seeds make them high in Omega fatty acids and they are packed full of nutrients. They recently won the 2013 SELF Healthy Food Award and were featured in the June edition of Self Magazine. I paired the Mexican Chicken Tenders with Salba Chia Salsa for an additional kick of nutrition. My kids were happy with my easy dinner recipe and *I* was happy to offer them a nutritious meal that they didnt complain about!
        </p>
        <p>
         <a href="http://www.suburbia-unwrapped.com/wp-content/uploads/2013/06/easy-mexican-chicken-tenders-with-banner.jpg">
          <img alt="easy mexican chicken tenders " class="aligncenter size-full wp-image-3064" height="405" src="http://www.suburbia-unwrapped.com/wp-content/uploads/2013/06/easy-mexican-chicken-tenders-with-banner.jpg" width="540"/>

           Ingredients

          <ul>
           <li class="ingredient" itemprop="ingredients">
            1 pound boneless chicken breast, cut into strips
           </li>
           <li class="ingredient" itemprop="ingredients">
            1 bag (7 ounces) Salba Chia Tortilla Chips
           </li>
           <li class="ingredient" itemprop="ingredients">
             cup fresh cilantro

           <li class="ingredient" itemprop="ingredients">
            2 tsp fajita seasoning

           <li class="ingredient" itemprop="ingredients">
            1 egg

           <li class="ingredient" itemprop="ingredients">
            2 TBSP milk
           </li>
           <li class="ingredient" itemprop="ingredients">

          <div class="ERSInstructionsHeader ERSHeading">
           Instructions
          </div>
          <ol>
           <li class="instruction" itemprop="recipeInstructions">
            In a food processor, combine the Salba Chia tortilla chips, cilantro and fajita seasoning. Pulse until completely mixed and resembles course crumbs. Put onto a shallow plate.
           </li>
           <li class="instruction" itemprop="recipeInstructions">
            In a small bowl, whisk the egg and the milk
           </li>
           <li class="instruction" itemprop="recipeInstructions">
            Dip each chicken strip in the egg mixture, then coat with crumbs and place on a cookie sheet coated with olive oil
           </li>
           <li class="instruction" itemprop="recipeInstructions">
            Bake chicken at 350 degrees for approximately 20 minutes or until done, turning over half way through.
           </li>
           <li class="instruction" itemprop="recipeInstructions">
            Serve with salsa for dipping.
           </li>

    validate: function() {
      this.isValid = true;
      this.emailValidation();
      this.fieldAndListValidation();
      this.updateFormAfterValidation();
    },

    emailValidation: function() {
      var email = document.getElementById('signup_email');

      if (this.validEmail.test(email.value)) {
        this.removeTextFieldError(email);
      } else {
        this.textFieldError(email);
        this.isValid = false;
      }

    fieldAndListValidation: function() {
      var fields = this.form.querySelectorAll('.mimi_field.required');

      for (var i = 0; i &lt; fields.length; ++i) {
        var field = fields[i],
            type  = this.fieldType(field);
        if (type === 'checkboxes' || type === 'radio_buttons') {
          this.checkboxAndRadioValidation(field);
        } else {
          this.textAndDropdownValidation(field, type);
        }
      }
    },

    fieldType: function(field) {
      var type = field.querySelectorAll('.field_type');

      if (type.length) {
        return type[0].getAttribute('data-field-type');
      } else if (field.className.indexOf('checkgroup') &gt;= 0) {
        return 'checkboxes';
      } else {
        return 'text_field';
      }
    },

    checkboxAndRadioValidation: function(field) {
      var inputs   = field.getElementsByTagName('input'),
          selected = false;

      for (var i = 0; i &lt; inputs.length; ++i) {
        var input = inputs[i];
        if((input.type === 'checkbox' || input.type === 'radio') &amp;&amp; input.checked) {
          selected = true;
        }
      }

      if (selected) {
        field.className = field.className.replace(/ invalid/g, '');
      } else {
        if (field.className.indexOf('invalid') === -1) {
          field.className += ' invalid';
        }

        this.isValid = false;
      }
    },

    textAndDropdownValidation: function(field, type) {

      for (var i = 0; i &lt; inputs.length; ++i) {
        var input = inputs[i];
        if (input.name.indexOf('signup') &gt;= 0) {
          if (type === 'text_field') {
            this.textValidation(input);
          } else {
            this.dropdownValidation(field, input);
          }
        }
      }
      this.htmlEmbedDropdownValidation(field);
    },

    textValidation: function(input) {
      if (input.id === 'signup_email') return;

      if (input.value) {

      } else {
        this.textFieldError(input);
        this.isValid = false;
      }
    },
