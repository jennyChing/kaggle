         <p>
          The Linux 
          <strong>
           du
          </strong>
           (
          <strong>
           Disk Usage
          </strong>
          ) is a standard
          <strong>
           Unix/Linux
          </strong>
          command, used to check the information of disk usage of files and directories on a machine. The
          <strong>
           du
          </strong>
          command has many parameter options that can be used to get the results in many formats. The
          <strong>
           du
          </strong>
          command also displays the files and directory sizes in a recursively manner.
         </p>
         <div class="wp-caption aligncenter" id="attachment_2016" style="width: 310px">
          <a href="http://www.tecmint.com/10-useful-du-disk-usage-commands-to-find-disk-usage-of-files-and-directories/12-useful-du-commands/" rel="attachment wp-att-2016">
           <img alt="Check Disk Usage In Linux" class="size-medium wp-image-2016" data-lazy-src="http://www.tecmint.com/wp-content/uploads/2013/01/12-Useful-du-commands-300x215.png" height="215" src="http://www.tecmint.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" width="300"/>
           <noscript>

           </noscript>
          </a>
          <p class="wp-caption-text">
           Check Disk Usage of Files and Folders In Linux
          </p>
         </div>
         <p>
          This article explains
          <strong>
           10 useful du commands
          </strong>
          with their examples, that might helps you to find out the sizes of files and directories in Linux. The information provided in this article are taken from the man pages of
          <strong>
           du
          </strong>
          command.
         </p>
         <p>
          <strong>
           Read Also:
          </strong>
         </p>
         <ol>
          <li>
           <a href="http://www.tecmint.com/how-to-check-disk-space-in-linux/" target="_blank">
            12 df Command to Check Linux System Disk Space
           </a>
          </li>
         </ol>
         <p>
          1. To find out the disk usage summary of a
          <strong>
           /home/tecmint
          </strong>
          directory tree and each of its sub directories. Enter the command as:
         </p>
         <pre>
          [root@tecmint]# du  /home/tecmint

40      /home/tecmint/downloads
4       /home/tecmint/.mozilla/plugins
4       /home/tecmint/.mozilla/extensions
12      /home/tecmint/.mozilla
12      /home/tecmint/.ssh
689112  /home/tecmint/Ubuntu-12.10
689360  /home/tecmint
         </pre>
         <p>
          The output of the above command displays the number of disk blocks in the
          <strong>
           /home/tecmint
          </strong>
          directory along with its sub-directories.
         </p>
         <p>
          2. Using 
          <strong>
           -h
          </strong>
           option with 
          <strong>
           du
          </strong>
           command provides results in 
          <strong>
           Human Readable Format
          </strong>
          . Means you can see sizes in
          <strong>
           Bytes
          </strong>
          ,
          <strong>
           Kilobyte
          </strong>
          s,
          <strong>
           Megabytes
          </strong>
          ,
          <strong>
           Gigabytes
          </strong>
          etc.
         </p>
         <pre>
          [root@tecmint]# du -h /home/tecmint

40K     /home/tecmint/downloads
4.0K    /home/tecmint/.mozilla/plugins
4.0K    /home/tecmint/.mozilla/extensions
12K     /home/tecmint/.mozilla
12K     /home/tecmint/.ssh
673M    /home/tecmint/Ubuntu-12.10
674M    /home/tecmint
         </pre>
         <p>
          3. To get the summary of a grand total disk usage size of an directory use the option 
          <strong>
           -s
          </strong>
           as follows.
         </p>
         <pre>
          [root@tecmint]# du -sh /home/tecmint

674M    /home/tecmint
         </pre>
         <p>
          4. Using 
          <strong>
           -a
          </strong>
           flag with 
          <strong>
           du
          </strong>
           command displays the disk usage of all the files and directories.
         </p>
         <pre>
          [root@tecmint]# du -a /home/tecmint

4       /home/tecmint/.bash_logout
12      /home/tecmint/downloads/uploadprogress-1.0.3.1.tgz
24      /home/tecmint/downloads/Phpfiles-org.tar.bz2
40      /home/tecmint/downloads
12      /home/tecmint/uploadprogress-1.0.3.1.tgz
4       /home/tecmint/.mozilla/plugins
4       /home/tecmint/.mozilla/extensions
12      /home/tecmint/.mozilla
4       /home/tecmint/.bashrc
689108  /home/tecmint/Ubuntu-12.10/ubuntu-12.10-server-i386.iso
689112  /home/tecmint/Ubuntu-12.10
689360  /home/tecmint
         </pre>
         <p>
          5. Using 
          <strong>
           -a
          </strong>
           flag along with 
          <strong>
           -h
          </strong>
           displays disk usage of all files and folders in human readeable format. The below output is more easy to understand as it shows the files in
          <strong>
           Kilobytes
          </strong>
          ,
          <strong>
           Megabytes
          </strong>
          etc.
         </p>
         <pre>
          [root@tecmint]# du -ah /home/tecmint

4.0K    /home/tecmint/.bash_logout
12K     /home/tecmint/downloads/uploadprogress-1.0.3.1.tgz
24K     /home/tecmint/downloads/Phpfiles-org.tar.bz2
40K     /home/tecmint/downloads
12K     /home/tecmint/uploadprogress-1.0.3.1.tgz
4.0K    /home/tecmint/.mozilla/plugins
4.0K    /home/tecmint/.mozilla/extensions
12K     /home/tecmint/.mozilla
4.0K    /home/tecmint/.bashrc
673M    /home/tecmint/Ubuntu-12.10/ubuntu-12.10-server-i386.iso
673M    /home/tecmint/Ubuntu-12.10
674M    /home/tecmint
         </pre>
         <p>
          6. Find out the disk usage of a directory tree with its subtress in
          <strong>
           Kilobyte
          </strong>
          blcoks. Use the 
          <strong>
           -k
          </strong>
           (displays size in
          <strong>
           1024
          </strong>
          bytes units).
         </p>
         <pre>
          [root@tecmint]# du -k /home/tecmint
40      /home/tecmint/downloads
4       /home/tecmint/.mozilla/plugins
4       /home/tecmint/.mozilla/extensions
12      /home/tecmint/.mozilla
12      /home/tecmint/.ssh
689112  /home/tecmint/Ubuntu-12.10
689360  /home/tecmint
         </pre>
         <p>
          7. To get the summary of disk usage of directory tree along with its subtrees in
          <strong>
           Megabytes
          </strong>
          (
          <strong>
           MB
          </strong>
          ) only. Use the option 
          <strong>
           -mh
          </strong>
           as follows. The 
          <strong>
           -m
          </strong>
           flag counts the blocks in
          <strong>
           MB
          </strong>
          units and 
          <strong>
           -h
          </strong>
           stands for human readable format.
         </p>
         <pre>
          [root@tecmint]# du -mh /home/tecmint

40K     /home/tecmint/downloads
4.0K    /home/tecmint/.mozilla/plugins
4.0K    /home/tecmint/.mozilla/extensions
12K     /home/tecmint/.mozilla
12K     /home/tecmint/.ssh
673M    /home/tecmint/Ubuntu-12.10
674M    /home/tecmint
         </pre>
         <p>
          8. The 
          <strong>
           -c
          </strong>
           flag provides a grand total usage disk space at the last line. If your directory taken
          <strong>
           674MB
          </strong>
          space, then the last last two line of the output would be.
         </p>
         <pre>
          [root@tecmint]# du -ch /home/tecmint

40K     /home/tecmint/downloads
4.0K    /home/tecmint/.mozilla/plugins
4.0K    /home/tecmint/.mozilla/extensions
12K     /home/tecmint/.mozilla
12K     /home/tecmint/.ssh
673M    /home/tecmint/Ubuntu-12.10
          <strong>
           674M    /home/tecmint
          </strong>
          <strong>
           674M    total
          </strong>
         </pre>
         <p>
          9. The below command calculates and displays the disk usage of all files and directories, but excludes the files that matches given pattern. The below command excludes the 
          <strong>
           .txt
          </strong>
           files while calculating the total size of diretory. So, this way you can exclude any file formats by using flag 
          <strong>
           -exclude
          </strong>
          . See the output there is no
          <strong>
           txt
          </strong>
          files entry.
         </p>
         <pre>
          [root@tecmint]# du -ah --exclude="*.txt" /home/tecmint

4.0K    /home/tecmint/.bash_logout
12K     /home/tecmint/downloads/uploadprogress-1.0.3.1.tgz
24K     /home/tecmint/downloads/Phpfiles-org.tar.bz2
40K     /home/tecmint/downloads
12K     /home/tecmint/uploadprogress-1.0.3.1.tgz
4.0K    /home/tecmint/.bash_history
4.0K    /home/tecmint/.bash_profile
4.0K    /home/tecmint/.mozilla/plugins
4.0K    /home/tecmint/.mozilla/extensions
12K     /home/tecmint/.mozilla
4.0K    /home/tecmint/.bashrc
24K     /home/tecmint/Phpfiles-org.tar.bz2
4.0K    /home/tecmint/geoipupdate.sh
4.0K    /home/tecmint/.zshrc
120K    /home/tecmint/goaccess-0.4.2.tar.gz.1
673M    /home/tecmint/Ubuntu-12.10/ubuntu-12.10-server-i386.iso
673M    /home/tecmint/Ubuntu-12.10
674M    /home/tecmint
         </pre>
         <p>
          10. Display the disk usage based on modification of time, use the flag 
          <strong>
           time
          </strong>
           as shown below.
         </p>
         <pre>
          [root@tecmint]# du -ha --time /home/tecmint

4.0K    2012-10-12 22:32        /home/tecmint/.bash_logout
12K     2013-01-19 18:48        /home/tecmint/downloads/uploadprogress-1.0.3.1.tgz
24K     2013-01-19 18:48        /home/tecmint/downloads/Phpfiles-org.tar.bz2
40K     2013-01-19 18:48        /home/tecmint/downloads
12K     2013-01-19 18:32        /home/tecmint/uploadprogress-1.0.3.1.tgz
4.0K    2012-10-13 00:11        /home/tecmint/.bash_history
4.0K    2012-10-12 22:32        /home/tecmint/.bash_profile
0       2013-01-19 18:32        /home/tecmint/xyz.txt
0       2013-01-19 18:32        /home/tecmint/abc.txt
4.0K    2012-10-12 22:32        /home/tecmint/.mozilla/plugins
4.0K    2012-10-12 22:32        /home/tecmint/.mozilla/extensions
