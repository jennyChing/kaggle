         <h4>
          You can download
          <a href="https://build.openmodelica.org/omc/builds/windows/releases/1.9.2/" target="_blank">
           HERE
           <img alt="" border="0" height="52" src="/images/down5.png" style="vertical-align: middle;" width="60"/>
          </a>
          the
          <strong>
           last OpenModelica release
          </strong>
          : 1.9.2.
         </h4>
         <p>
          <strong>
           OpenModelica includes SystemDynamics library with World modeling.
           <a href="/images/docs/systemdynamics-world3-simulation-with-openmodelica-131023.pdf" target="_blank">
            Instructions for systemdynamics World3 simulations
           </a>
           .
          </strong>
         </p>
         <p>
          <strong>
           <strong>
            Previous OpenModelica releases
           </strong>
           are available on our repository,
           <a href="https://build.openmodelica.org/omc/builds/windows/releases/" target="_blank">
            HERE
           </a>
           .
          </strong>
         </p>
         <h4>
          <strong>
           <span style="color: #ff0000;">
            <span style="color: #333333; font-weight: normal;">
             Nightly builds are also available, please try the
             <a href="https://build.openmodelica.org/omc/builds/windows/nightly-builds/" target="_blank">
              last nightly build
             </a>
             as
            </span>
            <strong>
             <span style="color: #333333; font-weight: normal;">
              it usually contains many bug fixes, updates and improvements.
             </span>
            </strong>
           </span>
          </strong>
         </h4>
         <h4>
          <strong>
           <span style="color: #ff0000;">
            <strong>
             Please choose an installation path that has no spaces in it or use the path suggested by the installer, otherwise OpenModelica installation is not guaranteed to work properly.
            </strong>
           </span>
          </strong>
