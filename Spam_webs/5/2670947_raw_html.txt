     <p>
      <b>
       FreeMind
      </b>
      is a
      <a href="/wiki/Free_software" title="Free software">
       free
      </a>
      <a class="mw-redirect" href="/wiki/Mind_mapping" title="Mind mapping">
       mind mapping
      </a>
      application written in
      <a href="/wiki/Java_(programming_language)" title="Java (programming language)">
       Java
      </a>
      . FreeMind is licensed under the
      <a href="/wiki/GNU_General_Public_License" title="GNU General Public License">
       GNU General Public License
      </a>
      . It provides extensive export capabilities. It runs on
      <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">
       Microsoft Windows
      </a>
      ,
      <a href="/wiki/Linux" title="Linux">
       Linux
      </a>
      and
      <a class="mw-redirect" href="/wiki/Mac_OS_X" title="Mac OS X">
       Mac OS X
      </a>
      via the
      <a class="mw-redirect" href="/wiki/Java_Runtime_Environment" title="Java Runtime Environment">
       Java Runtime Environment
      </a>
      .
      <sup class="reference" id="cite_ref-LinuxWorld_2-0">
       <a href="#cite_note-LinuxWorld-2">
        <span>
         [
        </span>
        2
        <span>
         ]
        </span>
       </a>
      </sup>
      <sup class="reference" id="cite_ref-LinuxLinks_3-0">
       <a href="#cite_note-LinuxLinks-3">
        <span>
         [
        </span>
        3
        <span>
         ]
        </span>
       </a>
      </sup>
      <sup class="reference" id="cite_ref-InnovationTools_4-0">
       <a href="#cite_note-InnovationTools-4">
        <span>
         [
        </span>
        4
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <p>
      As with other mind mapping software packages, FreeMind allows the user to edit a hierarchical set of ideas around a central concept. The non-linear approach assists in brainstorming new outlines and projects as ideas are added around the mind map.
      <sup class="reference" id="cite_ref-LinuxWorld_2-1">
       <a href="#cite_note-LinuxWorld-2">
        <span>
         [
        </span>
        2
        <span>
         ]
        </span>
       </a>
      </sup>
      As a Java application, FreeMind is portable across
      <a href="/wiki/Cross-platform" title="Cross-platform">
       multiple platforms
      </a>
      and retains the same user interface, causing some amount of variation from the common interface on each platform.
      <a href="/wiki/Apple_Inc." title="Apple Inc.">
       Mac
      </a>
      users may notice the most difference from their traditional user interface, but a
      <a class="mw-redirect" href="/wiki/MacWorld" title="MacWorld">
       MacWorld
      </a>
      reviewer says the software's features should still appeal to the segment of users who accept function over form.
      <sup class="reference" id="cite_ref-MacWorld_5-0">
       <a href="#cite_note-MacWorld-5">
        <span>
         [
        </span>
        5
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <p>
      FreeMind was a finalist for Best Project in
      <a class="mw-redirect" href="/wiki/SourceForge.net" title="SourceForge.net">
       SourceForge.net
      </a>
      's Community Choice Awards for 2008, which featured
      <a class="mw-redirect" href="/wiki/Open_Source" title="Open Source">
       Open Source
      </a>
      software projects.
      <sup class="reference" id="cite_ref-foxbiz_6-0">
       <a href="#cite_note-foxbiz-6">
        <span>
         [
        </span>
        6
        <span>
         ]
        </span>
       </a>
      </sup>
      <sup class="reference" id="cite_ref-sfnet2008_7-0">
       <a href="#cite_note-sfnet2008-7">
        <span>
         [
        </span>
        7
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>

     <h2>
      <span class="mw-headline" id="Features">
       Features
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=FreeMind&amp;action=edit&amp;section=1" title="Edit section: Features">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      FreeMind's most significant features are as follows:
      <sup class="reference" id="cite_ref-InnovationTools_4-1">
       <a href="#cite_note-InnovationTools-4">
        <span>
         [
        </span>
        4
        <span>
         ]
        </span>
       </a>
      </sup>
      <sup class="reference" id="cite_ref-8">
       <a href="#cite_note-8">
        <span>
         [
        </span>
        8
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <ul>
      <li>
       Folding branches
      </li>
      <li>
       Save files as
       <a href="/wiki/XML" title="XML">
        XML
       </a>
       in
       <a class="new" href="/w/index.php?title=Mm-file&amp;action=edit&amp;redlink=1" title="Mm-file (page does not exist)">
        mm file format
       </a>
      </li>
      <li>
       Export hypertext to
       <a href="/wiki/HTML" title="HTML">
        HTML
       </a>
       and
       <a href="/wiki/XHTML" title="XHTML">
        XHTML
       </a>
      </li>
      <li>
       Export document to
       <a href="/wiki/Portable_Document_Format" title="Portable Document Format">
        PDF
       </a>
       and
       <a href="/wiki/OpenDocument" title="OpenDocument">
        OpenDocument
       </a>
      </li>
      <li>
       Exports image to
       <a href="/wiki/Portable_Network_Graphics" title="Portable Network Graphics">
        PNG
       </a>
       ,
       <a href="/wiki/JPEG" title="JPEG">
        JPEG
       </a>
       and
       <a href="/wiki/Scalable_Vector_Graphics" title="Scalable Vector Graphics">
        SVG
       </a>
      </li>
      <li>
       Icons on nodes
      </li>
      <li>
       Clouds around branches
      </li>
      <li>
       Graphical links connecting nodes
      </li>
      <li>
       Search restricted to single branches
      </li>
      <li>
       Web and file hyperlinks from nodes
      </li>
      <li>
       FreeMind browser/player for web in Java or
       <a href="/wiki/Adobe_Shockwave" title="Adobe Shockwave">
        Flash
       </a>
      </li>
      <li>
       Transform maps using
       <a href="/wiki/XSLT" title="XSLT">
        XSLT
       </a>
      </li>
     </ul>
     <p>
      FreeMind uses the
      <a href="/wiki/Swing_(Java)" title="Swing (Java)">
       Swing
      </a>
      <a href="/wiki/Widget_toolkit" title="Widget toolkit">
       GUI toolkit
      </a>
      for Java.
     </p>
     <p>
      FreeMind developers or developers of other projects have made
      <a href="/wiki/Plug-in_(computing)" title="Plug-in (computing)">
       plugins
      </a>
      for various
      <a href="/wiki/Wiki" title="Wiki">
       wiki
      </a>
      and
      <a class="mw-redirect" href="/wiki/Content_management_system" title="Content management system">
       content management system
      </a>
      software so that Freemind files can be viewed and in some cases created via the web interface.
     </p>
     <h2>
      <span class="mw-headline" id="Forks">
       Forks
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=FreeMind&amp;action=edit&amp;section=2" title="Edit section: Forks">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      There are (at least) two active
      <a href="/wiki/Fork_(software_development)" title="Fork (software development)">
       forks
      </a>
      of the FreeMind project.
      <a href="/wiki/Freeplane" title="Freeplane">
       Freeplane
      </a>
      is mainly developed by FreeMind's former developer Dimitri Polivaev and focuses on enhanced usability. SciPlore MindMapping, now
      <a href="/wiki/Docear" title="Docear">
       Docear
      </a>
      , focuses on enhanced PDF support (bookmark import) and integration of
      <a href="/wiki/Reference_management_software" title="Reference management software">
       reference management
      </a>
      including
      <a href="/wiki/BibTeX" title="BibTeX">
       BibTeX
      </a>
      support.
     </p>
