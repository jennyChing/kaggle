         <p>
          Here is a list of
          <strong>
           3search engines you dont know about
          </strong>
          . We all use search engines in our daily life for different purposes. No matter what we have to search on Internet, search engines are our starting point. And whenever we talk about search engines, we only think about the popular ones like Google, Bing, Ask.com, Yahoo! and some not so popular ones like
          <a href="https://duckduckgo.com/" target="_blank" title="DuckDuckGo">
           DuckDuckGo
          </a>
          , but there are many more search engines out there which can serve the purpose.
         </p>
         <p>
          Out of those many search engines, Ive shortlisted these 3lesser known search engines whichyou can try out if you are fed up of using Google a few thousand times a day :)
         </p>
         <p>
          Note: There are many other search engines as well, but most of them just get the search results from Google or Bing, so I have not included them in this list. These are the ones that do their own indexing anduse their own data to provide search results.
         </p>
         <h2>
          1.
          <a href="https://www.yandex.com/" target="_blank" title="Yandex Search Engine Landing Page">
           Yandex:
          </a>
         </h2>
         <p>
          <img alt="Yandex" class=" size-full wp-image-270614 aligncenter" height="417" src="http://cdn.ilovefreesoftware.com/wp-content/uploads/2015/01/Yandex.png" width="550"/>
          <br/>
          Sponsored Links
          <br/>
          <script async="" src="//pagead2.googlesyndication.com/pagead/js/adsbygoogle.js">
          </script>
          <!-- ILFS Below Image Ad Custom Size -->
          <ins class="adsbygoogle" data-ad-client="ca-pub-2954429995859667" data-ad-slot="6225615564" style="display:inline-block;width:600px;height:90px">
          </ins>
          <script>
           (adsbygoogle = window.adsbygoogle || []).push({});
          </script>
         </p>
         <p>
          The firstsearch engine in the list is
          <strong>
           Yandex
          </strong>
          . Although it is the most popular search engine in Russia, but it is still unknown for most of the Internet users who reside outside Russia. This search engine has one of the cleanestuser interface, with just a few icons and a big search bar. Apart from regular web search, it also lets you search for images, maps, videos, and translations. There are two more icons which takeyou to
          <a href="http://www.ilovefreesoftware.com/13/webware/yandex-mail.html" title="Yandex Mail: Another Great Email Service from Russia">
           Yandex Mail
          </a>
          and
          <a href="http://www.ilovefreesoftware.com/03/windows/internet/browser/free-web-browser-from-yandex-ru.html" title="Free Web Browser From Yandex.ru">
           Yandex Browser
          </a>
          .
         </p>
         <p>
          The search results page is also pretty clean with search box atthe top displaying the query entered by you, along with the number of search results found. For each individual search result, the Yandex displays that particular websites favicon. For search results, you can also set settings like search mode, results filter by time, language, and file format. It also has a unique feature using which you can specify aparticular website and region name to get limited and specific results.
         </p>
         <h2>
          2.
          <a href="http://blekko.com/" target="_blank" title="Blekko Search Engine Landing Page">
           Blekko:
          </a>
         </h2>
         <p>
          <img alt="Blekko" class=" size-full wp-image-270611 aligncenter" height="275" src="http://cdn.ilovefreesoftware.com/wp-content/uploads/2015/01/Blekko.png" width="550"/>
         </p>
         <p>
          The secondsearch engine in the list is
          <strong>
           Blekko
          </strong>
          . The main feature of Blekko is that it provides slashtags. So, if you are trying to do a search on a topic, you can try to do that using a slashtag and get results from a curated list. This potentially reduces the spam results that comes in Blekkos search results. Also, it is the search engine with one ofthe best user experience. It is also the only search engine in the list which displays Trending Topics on the homepage. Each trending topic is accompanied by a related image and you can search for that topic by clicking on the image.All the search results are divided into following categories: Quick Answer, Top Results, Shopping, Social, Latest Results, and some other sections which dependupon the query entered by you. It also shows images related to the query on the right side.
         </p>
         <h2>
          3.
          <a href="http://www.gigablast.com/" title="Gigablast Search Engine Landing Page">
           Gigablast:
          </a>
         </h2>
         <p>
          <img alt="Gigablast" class=" size-full wp-image-270721 aligncenter" height="337" src="http://cdn.ilovefreesoftware.com/wp-content/uploads/2015/01/Gigablast.png" width="550"/>
         </p>
         <p>
          The thirdlesser known search engine in the list is
          <strong>
           Gigablast
          </strong>
          . It has actually been around since the year 2000! It is an open-sourcesearch engine which also claims about being one of the 4 remaining United States search engines which maintains their own searchable index of more than 1 billion pages. It also claims itself to be the transparent search engine by showing why a particular search result has been ranked at the position which it holds.
         </p>
         <p>
          On search results page, it shows an option
          <strong>
           More from this website
          </strong>
          . This option lets you see results from that particular website. At the bottom of the search result page, it provides links to Google and Bing for performing your search on them.Apart from normal web search, it also lets you perform directory search and event search (powered by EventGuru.com).
         </p>
         <p>
          I hope you like these search engines that I unearthed for you. My personal favorite in this list is Blekko, as I really like its concept of curated search results. If you know some more of them, feel free to share them with me.
         </p>
