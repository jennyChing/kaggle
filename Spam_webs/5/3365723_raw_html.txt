     <section class="article_body">
      <p>
       <strong>
        <em>
         Client-side Code (jQuery)
        </em>
       </strong>
      </p>
      <p>
       Now, on to the jQuery front end. First, we need to declare the timestamp of the current message as 0, and call the function that loads the messages from the server:
      </p>
      <pre>
       <code>
        timestamp = 0;  
updateMsg();
       </code>
      </pre>
      <p>
       Next, well write the code for the form submission. jQuery allows us to add an event hook for the forms submit event, as though we were adding an
       <code>
        onSubmit
       </code>
       event within the HTML itself, except that we dont have to touch the HTML. Heres the submit event:
      </p>
      <pre>
       <code>
        $("form#chatform").submit(function(){ /* Code */ });
       </code>
      </pre>
      <p>
       Here were using CSS selector syntax to refer to the form element with an
       <code>
        id
       </code>
       of
       <code>
        'chatform'
       </code>
       . As soon as were into the form submission code, we can fire off a POST request to the server using jQuerys
       <code>
        $.post
       </code>
       . Within the call to
       <code>
        $.post
       </code>
       , we can select the values of the form elements on the page on the basis of their
       <code>
        id
       </code>
       s, as we determined earlier. With this in mind, we can dive into our Ajax call:
      </p>
      <pre>
       <code>
        $.post("backend.php",{ message: $("#msg").val(),  
name: $("#author").val(), action: "postmsg", time: timestamp }, function(xml) {
       </code>
      </pre>
      <p>
       Notice that the array of parameters to be passed in the request is enclosed with curly braces. If you have more parameters, simply separate them with commas and use the format presented here,
       <a class="sublink" href="http://www.json.org/">
        JSON-style
       </a>
       . You can also use jQuerys Ajax functions to send a GET request that expects a JSON-style response, and have jQuery convert the response text into an easy-to-use format. Bear in mind, though, that this capability is only available for the GET request type, not the POST request type we use for our chat system. As such, well stick with plain XML for the moment.
      </p>
      <p>
       Now, lets look at handling the XML response. Since were all for code reuse, well create a function that handles the XML and call it now:
      </p>
      <pre>
       <code>
        addMessages(xml);
       </code>
      </pre>
      <p>
       Well write this function later so that we can finish off the form submission event code. The code weve written so far is all we need for our
       <code>
        $.post
       </code>
       callback function, so we can close it off and add a return false; line. This line fails using the standard browser form submission error code. The browser doesnt send the user to another page to submit the form  weve already handled the form submission, so the browser doesnt need to. Heres the event code in its entirety:
      </p>
      <pre>
       <code>
        $("form#chatform").submit(function(){  
 $.post("backend.php",{  
    message: $("#msg").val(),  
    name: $("#author").val(),  
    action: "postmsg",  
    time: timestamp  
   }, function(xml) {  
  addMessages(xml);  
 });  
 return false;  
});
       </code>
      </pre>
      <p>
       Now, lets get back to the
       <code>
        addMessages()
       </code>
       function, which handles the response xml. Its quite simple, making use of jQuerys DOM manipulation and traversing functions. Remember that I mentioned the status code before? Nows the time to handle it:
      </p>
      <pre>
       <code>
        if($("status",xml).text() == "2") return;
       </code>
      </pre>
      <p>
       I havent mentioned context in jQuery yet. The XML in this function call tells jQuery to look not inside the document HTML, but in the XML that the server sent us.
      </p>
      <p>
       That line of code checks for a status code of 2, representing a successful request that resulted in us having no new messages to add to the window. The
       <code>
        'return'
       </code>
       keyword terminates the function call. Next, we set the timestamp to the timestamp in the XML:
      </p>
      <pre>
       <code>
        timestamp = $("time",xml).text();
       </code>
      </pre>
      <p>
       Again, this fetches the text value of the
       <code>
        &lt;time&gt;
       </code>
       tag in the XML.
      </p>
      <p>
       Now we can move on to jQuerys array iteration function,
       <code>
        each()
       </code>
       . jQuery has an interesting way of handling array iteration. We use a standard selector statement, and the
       <code>
        each()
       </code>
       function is passed one parameter  a function to handle each instance of the matched elements. In this case, the elements are the instances of the
       <code>
        &lt;message&gt;
       </code>
       tag in the server response, each instance representing a message to be displayed. One parameter  the
       <code>
        id
       </code>
       of the instance  is passed to the function. We can use this, with jQuerys
       <code>
        get()
       </code>
       function, to grab a new context  the actual XML of the
       <code>
        &lt;message&gt;
       </code>
       tag. This is how we select it:
      </p>
      <pre>
       <code>
        $("message",xml).each(function(id) {  
 message = $("message",xml).get(id);
       </code>
      </pre>
      <p>
       We can then select elements by passing the context
       <code>
        'message'
       </code>
       to the jQuery
       <code>
        / $
       </code>
       function. Now that we have all the data we need, we must add it to the top of the message window on the page. The message window has the id
       <code>
        'messagewindow'
       </code>
       , so we select it using
       <code>
        $("#messagewindow")
       </code>
       and use the
       <code>
        prepend()
       </code>
       function to add our data:
      </p>
      <pre>
       <code>
        $("#messagewindow").prepend("&lt;b&gt;"+$("author",message).text()+  
     "&lt;/b&gt;: "+$("text",message).text()+  
     "&lt;br /&gt;");
       </code>
      </pre>
      <p>
       Thats all there is to it! Putting it all together, heres the code for the function:
      </p>
      <pre>
       <code>
        function addMessages(xml) {  
 if($("status",xml).text() == "2") return;  
 timestamp = $("time",xml).text();  
 $("message",xml).each(function(id) {  
  message = $("message",xml).get(id);  
  $("#messagewindow").prepend("&lt;b&gt;"+$("author",message).text()+  
       "&lt;/b&gt;: "+$("text",message).text()+  
       "&lt;br /&gt;");  
 });  
}
       </code>
      </pre>
      <p>
       Finally, we need the
       <code>
        updateMsg
       </code>
       function we called at the very beginning of our code. This function has to query the server for new messages, and call the above
       <code>
        addMessages
       </code>
       function with the response. It also has to set a timeout to call itself after a set period of time, which makes the chat window update automatically. To begin, we dont need to submit anything to the server besides a timestamp, so this is our
       <code>
        $.post
       </code>
       call:
      </p>
      <pre>
       <code>
        $.post("backend.php",{ time: timestamp }, function(xml) {
       </code>
      </pre>
      <p>
       As I noted before, we also need to remove the loading message at this point, so we call jQuerys remove function on the span:
      </p>
      <pre>
       <code>
        $("#loading").remove();
       </code>
      </pre>
      <p>
       Then, weve received our xml response in the object
       <code>
        'xml'
       </code>
       , so we pass it to our
       <code>
        addMessages
       </code>
       function:
      </p>
      <pre>
       <code>
        addMessages(xml);
       </code>
      </pre>
      <p>
       We round it off with a call to the JavaScript
       <code>
        setTimeout()
       </code>
       function, which executes specified code after a specified interval. Heres the whole function put together:
      </p>
      <pre>
       <code>
        function updateMsg() {  
 $.post("backend.php",{ time: timestamp }, function(xml) {  
  $("#loading").remove();  
  addMessages(xml);  
 });  
 setTimeout('updateMsg()', 4000);  
}
       </code>
      </pre>
      <p>
       <strong>
        <em>
         Putting it All Together
        </em>
       </strong>
      </p>
      <p>
       Now we can put all of the pieces of the puzzle together. The code is, as I mentioned, available in
       <a class="sublink" href="http://www.sitepoint.com/examples/jquery/ajax-jquery-demo.zip">
        this downloadable zip file
       </a>
       . However, you can peruse it here, where Ive added a bit of HTML and CSS for layout:
      </p>
      <pre>
       <code>
        &lt;html&gt;  
&lt;head&gt;  
 &lt;title&gt;Ajax with jQuery Example&lt;/title&gt;  
 &lt;script type="text/JavaScript" src="jquery.js"&gt;&lt;/script&gt;  
 &lt;script type="text/JavaScript"&gt;  
  $(document).ready(function(){  
   timestamp = 0;  
   updateMsg();  
   $("form#chatform").submit(function(){  
    $.post("backend.php",{  
       message: $("#msg").val(),  
       name: $("#author").val(),  
       action: "postmsg",  
       time: timestamp  
      }, function(xml) {  
     $("#msg").empty();  
     addMessages(xml);  
    });  
    return false;  
   });  
  });  
  function addMessages(xml) {  
   if($("status",xml).text() == "2") return;  
   timestamp = $("time",xml).text();  
   $("message",xml).each(function(id) {  
    message = $("message",xml).get(id);  
    $("#messagewindow").prepend("&lt;b&gt;"+$("author",message).text()+  
           "&lt;/b&gt;: "+$("text",message).text()+  
           "&lt;br /&gt;");  
   });  
  }  
  function updateMsg() {  
   $.post("backend.php",{ time: timestamp }, function(xml) {  
    $("#loading").remove();  
    addMessages(xml);  
   });  
   setTimeout('updateMsg()', 4000);  
  }  
 &lt;/script&gt;
