           <h1 class="entry-title" itemprop="name headline">
            Guest Blogger: Pros and Cons of Hiring a Plumber for Bathroom Remodel
           </h1>
           <!-- post content -->
           <div class="entry-content clearfix" itemprop="articleBody">
            <div class="shareaholic-canvas" data-app="share_buttons" data-app-id="7384792" data-link="http://stagetecture.com/2011/08/guest-blogger-pros-and-cons-of-hiring-a-plumber-for-bathroom-remodel/" data-summary="" data-title="Guest Blogger: Pros and Cons of Hiring a Plumber for Bathroom Remodel">
            </div>
            <p style="text-align: right">
             Guest Blog #150, Entry #622, August 18, 201
            </p>
            <p style="text-align: left">
             Bathroom renovations are one of the most popular renovations homeowners choose for their homes. When it comes to the physical remodel, deciding to do it yourself (DIY) or hiring a plumber can be an essential part to the execution of your bathroom remodel.
            </p>
            <div class="wp-caption aligncenter" id="attachment_17367" style="width: 510px">
             <img alt="mp_bathroom-natural" class="size-full wp-image-17367" height="500" src="http://stagetecture.com/wp-content/uploads/2011/08/mp_bathroom-natural.jpg" title="mp_bathroom-natural" width="500"/>
             <p class="wp-caption-text">
              Before you start your remodel, do you need a professional plumber?
             </p>
            </div>
            <p style="text-align: center">
             <em>
              Image courtesy of Calfinder
             </em>
            </p>
            <p style="text-align: left">
             Today, our Guest Blogger , Magnolia Plumbing gives you the pros and cons of hiring a plumber for your bathroom model,
             <em>
              before
             </em>
             you get started!
            </p>
            <h3 style="text-align: center">
             <strong>
              __________________________
             </strong>
            </h3>
            <p>
             <img alt="magnolia plumbing logo" class="aligncenter size-full wp-image-17368" height="84" src="http://stagetecture.com/wp-content/uploads/2011/08/magnoliaplumbinglogo.bmp" title="magnolia plumbing logo" width="276"/>
            </p>
            <p>
             So youre all set to go for your bathroom remodeling project  youve got your new wallpaper all picked out, a brand new massaging showerhead and even a fancy new toilet! Theres only one decision left to make  should you hire a plumbing company to finish the remodel, or should you attempt it yourself?
            </p>
            <div class="wp-caption aligncenter" id="attachment_17369" style="width: 510px">
             <img alt="mp_blue-bathroom" class="size-full wp-image-17369" height="500" src="http://stagetecture.com/wp-content/uploads/2011/08/mp_blue-bathroom-remodeling-1-500x500.jpg" title="mp_blue-bathroom-remodeling-1-500x500" width="500"/>
             <p class="wp-caption-text">
              The pros and cons of hiring a plumber
             </p>
            </div>
            <p style="text-align: center">
             <em>
              Image courtesy of House Maintain
             </em>
            </p>
            <h3>
             <strong>
              Hiring a Plumber  Pros
             </strong>
            </h3>
            <p>
             <strong>
              You know the job will turn out right 
             </strong>
             when you hire a plumber to remodel your bathroom, you know the job will turn out how you want it (provided you hire a good plumber!). Professional plumbers will know exactly what needs to be done to ensure that your new bathroom matches your expectations as closely as possible.
            </p>
            <p>
             <strong>
              The project will be finished on time
             </strong>
              barring any last minute changes, a professional plumber will be able to budget enough time to make sure your bathroom remodel gets finished on time. Lets face it  if you want to remodel your bathroom yourself, you probably have at most two days to work on it  Saturday and Sunday. On the other hand, a good plumber will be at your house at least 4 days a week getting your bathroom done as quickly as possible.
            </p>
            <p>
             <strong>
              A plumber has more experience 
             </strong>
             bathroom remodeling involves a lot more than just popping in new fixtures and repainting the walls. Unfortunately, many DIYers dont realize this. The result? Repeated trips to the hardware store and mounting frustration as the job gets more and more complicated.
            </p>
            <div class="wp-caption aligncenter" id="attachment_17371" style="width: 510px">
             <img alt="mp_Modern_Bathroom_" class="size-full wp-image-17371" height="500" src="http://stagetecture.com/wp-content/uploads/2011/08/mp_Modern_Bathroom_Remodeling_003.jpeg" title="mp_Modern_Bathroom_Remodeling_003" width="500"/>
             <p class="wp-caption-text">
              Hiring a plumber can help avoid costly mistakes
             </p>
            </div>
            <p style="text-align: center">
             <em>
              Image courtesy ofLuxe Home
             </em>
            </p>
            <h3>
             <strong>
              Hiring a Plumber  Cons
             </strong>
            </h3>
            <p>
             <strong>
              Plumbers cost more up front 
             </strong>
             if all you want to do is change the faucets or showerheads, you can probably do that yourself and eliminate the cost of hiring a plumber. The best plumbers do cost a lot more. That said, you usually get what you pay for.
            </p>
            <p>
             <strong>
              You might lose some creative leeway
             </strong>
              one of the biggest problems that comes with hiring a plumber is that while you have a lot of creative freedom in the beginning, that tends to dry up quickly once the job gets started. Any changes you want to make will have to go through change orders, the costs of which will start to add up pretty rapidly.
            </p>
            <p>
             <em>
              Bio: Magnolia Plumbing is a plumbing company in that has been providing residential and commercial plumbing service Maryland, Washington, DC and Virginia since 1950.
             </em>
            </p>

             <script type="text/javascript">
              (function() {
  var form = document.getElementById('mad_mimi_signup_form'),
      submit = document.getElementById('webform_submit_button'),
      validEmail = /.+@.+\..+/,
      isValid;

  form.onsubmit = function(event) {
    validate();
    if(!isValid) {
      revalidateOnChange();
      return false;
    }
  };

  function validate() {
    isValid = true;
    emailValidation();
    fieldAndListValidation();
    updateFormAfterValidation();
  }

  function emailValidation() {
    var email = document.getElementById('signup_email');
    if(!validEmail.test(email.value)) {
      textFieldError(email);
      isValid = false;
    } else {
      removeTextFieldError(email);
    }
  }

  function fieldAndListValidation() {
    var fields = form.querySelectorAll('.mimi_field.required');
    for(var i = 0; i &lt; fields.length; ++i) {
      var field = fields[i],
          type  = fieldType(field);
      if(type == 'checkboxes' || type == 'radio_buttons') {
        checkboxAndRadioValidation(field);
      } else {
        textAndDropdownValidation(field, type);
      }
    }
  }

  function fieldType(field) {
    var type = field.querySelectorAll('.field_type');
    if(type.length &gt; 0) {
      return type[0].getAttribute('data-field-type');
    } else if(field.className.indexOf('checkgroup') &gt;= 0) {
      return 'checkboxes';
    } else {
      return 'text_field';
    }
  }

  function checkboxAndRadioValidation(field) {
    var inputs   = field.getElementsByTagName('input'),
        selected = false;
    for(var i = 0; i &lt; inputs.length; ++i) {
      var input = inputs[i];
      if((input.type == 'checkbox' || input.type == 'radio') &amp;&amp; input.checked) selected = true;
    }
    if(selected) {
      field.className = field.className.replace(/ invalid/g, '');
    } else {
      if(field.className.indexOf('invalid') == -1) field.className += ' invalid';
      isValid = false;
    }
  }

  function textAndDropdownValidation(field, type) {
    var inputs = field.getElementsByTagName('input');
    for(var i = 0; i &lt; inputs.length; ++i) {
      var input = inputs[i];
      if(input.name.indexOf('signup') &gt;= 0) {
        if(type == 'text_field') {
          textValidation(input);
        } else {
          dropdownValidation(field, input);
        }
      }
    }
    htmlEmbedDropdownValidation(field);
  }

  function textValidation(input) {
    if(input.id == 'signup_email') return;
    var val = input.value;
    if(val == '') {
      textFieldError(input);
      isValid = false;
      return;
    } else {
      removeTextFieldError(input)
    }
  }

  function dropdownValidation(field, input) {
    var val = input.value;
    if(val == '') {
      if(field.className.indexOf('invalid') == -1) field.className += ' invalid';
      onSelectCallback(input);
      isValid = false;
      return;
    } else {
      field.className = field.className.replace(/ invalid/g, '');
    }
  }

  function htmlEmbedDropdownValidation(field) {
    var dropdowns = field.querySelectorAll('.mimi_html_dropdown');
    for(var i = 0; i &lt; dropdowns.length; ++i) {
      var dropdown = dropdowns[i],
          val      = dropdown.value;
      if(val == '') {
        if(field.className.indexOf('invalid') == -1) field.className += ' invalid';
        isValid = false;
        dropdown.onchange = validate;
        return;
      } else {
        field.className = field.className.replace(/ invalid/g, '');
      }
    }
  }

  function textFieldError(input) {
    input.className   = 'required invalid';
    input.placeholder = input.getAttribute('data-required-field');
  }

  function removeTextFieldError(input) {
    input.className   = 'required';
    input.placeholder = '';
  }

  function onSelectCallback(input) {
    if(typeof Widget != 'undefined' &amp;&amp; Widget.BasicDropdown != undefined) {
      var dropdownEl = input.parentNode,
          instances  = Widget.BasicDropdown.instances;
      for(var i = 0; i &lt; instances.length; ++i) {
        var instance = instances[i];
        if(instance.wrapperEl == dropdownEl) {
          instance.onSelect = validate;
        }
      }
    }
  }

  function updateFormAfterValidation() {
    form.className   = setFormClassName();
    submit.value     = submitButtonText();
    submit.disabled  = !isValid;
    submit.className = isValid ? 'submit' : 'disabled';
  }

  function setFormClassName() {
    var name = form.className;
    if(isValid) {
      return name.replace(/\s?mimi_invalid/, '');
    } else {
      if(name.indexOf('mimi_invalid') == -1) {
        return name += ' mimi_invalid';
      } else {
        return name;
      }
    }
  }

  function submitButtonText() {
    var invalidFields = document.querySelectorAll('.invalid'),
        text;
    if(isValid || invalidFields == undefined) {
      text = submit.getAttribute('data-default-text');
    } else {
      if(invalidFields.length &gt; 1 || invalidFields[0].className.indexOf('checkgroup') == -1) {
        text = submit.getAttribute('data-invalid-text');
      } else {
        text = submit.getAttribute('data-choose-list');
      }
    }
    return text;
  }

  function revalidateOnChange() {
    var fields = form.querySelectorAll(".mimi_field.required");
    for(var i = 0; i &lt; fields.length; ++i) {
      var inputs = fields[i].getElementsByTagName('input');
      for(var j = 0; j &lt; inputs.length; ++j) {
        inputs[j].onchange = validate;
      }
    }
  }
})();
