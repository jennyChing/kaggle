              CSS Tutorial Series | Part 1 - Introduction to CSS
             </a>
            </h1>
            <div class="bwh-title">
             <div class="isibwh">
              <div class="d-header">
               <span class="timestamp-link">
                <abbr class="published updated" title="2013-12-18T00:31:00+05:30">

                </abbr>
               </span>
              </div>
              <div class="post-author vcard">
               <span class="fn">
                <a href="https://plus.google.com/103014711913669583836" itemprop="author" rel="author" title="author profile">
                 Biman Ghosh
                </a>
               </span>
              </div>
             </div>
            </div>
            <div class="post-header" style="position:absolute;left:-9999px">
             <div class="post-header-line-1">
             </div>
            </div>
            <div class="post-body entry-content" id="post-body-3043754032069070423">
             <div dir="ltr" style="text-align: left;" trbidi="on">
              <div style="text-align: justify;">
               <a href="http://3.bp.blogspot.com/-Yi98OVP0JpQ/Up8puJleS-I/AAAAAAAAAeQ/9izkHzBVd-s/s1600/css3.png" imageanchor="1" style="clear: right; float: right; margin-bottom: 1em; margin-left: 1em;">
                <img border="0" height="200" src="http://3.bp.blogspot.com/-Yi98OVP0JpQ/Up8puJleS-I/AAAAAAAAAeQ/9izkHzBVd-s/s1600/css3.png" width="142"/>
               </a>
               <span class="firstcharacter">
                B
               </span>
               efore we start this series we suppose that you have some understanding of HTML/XHTML. We will have tutorials about HTML/XHTML also soon for those who might need it. The power of CSS are shown in some of the articles that are published in our website. If you go through them then you will notice that CSS is really very useful and a bit tricky sometimes.
               <br/>
               <br/>
               See this article to check some wonderful CSS Creations:
               <a href="http://www.alljobindia.in/2013/12/top-5-most-wonderful-css-creations.html" target="_blank">
                CSS Magic
               </a>
               <br/>
               <br/>
               Well,now that the excitement has been build up, let's start the tutorial series.
              </div>
              <h4 style="text-align: left;">
               What is CSS?
              </h4>
              <div style="text-align: justify;">
               <br/>
               The acronym CSS means Cascading Style Sheets. You should know that we can use different methods (styles) to display HTML elements. Well, Style Sheets were introduced to HTML 4.0 as a solution to a problem regarding the styling of the HTML elements. Styles can be stored in External Style Sheets and they are called CSS Files. These files have the extension
               <span style="color: red;">
                (*.css).
               </span>
               <br/>
               <br/>
               Here is a CSS DEMO :
               <br/>
               <br/>
               <iframe allowfullscreen="allowfullscreen" frameborder="0" height="300" src="http://jsfiddle.net/biman7777/C6rSM/2/embedded/css,html,result/presentation/" width="100%">
               </iframe>
               <br/>
               <h4>
                Why is CSS important?
               </h4>
               <br/>
               <ul>
                <li>
                 CSS is used to define how the HTML elements will be displayed.
                </li>
                <li>
                 They save a lot of work. As the a single file can be used by all the pages in a website.
                </li>
                <li>
                 This code more easy to edit and make alterations. Editing one file will effect the all pages that have included the CSS file, thereby not requiring the web designer to make changes to all the pages.
                </li>
               </ul>
               <h4>
                Some key things to remember:
               </h4>
              </div>
              <div style="text-align: justify;">
              </div>
              <ul>
               <li>
                CSS was created by World Wide Web Consortium (W3C).
               </li>
               <li>
                CSS is now supported by all the browsers.
               </li>
               <li>
                From HTML 4.0 it was made possible to remove all formatting from the HTML in order to store it in a separate CSS file.
               </li>
               <li>
                CSS files are saved with the .css extension.
               </li>
              </ul>
              <div style="text-align: justify;">
               <br/>
              </div>
              <div style="text-align: justify;">
               That concludes our first tutorial. We will upload all the parts soon. In the meantime do share with us what you think about the tutorial that is here. If you have any suggestions or request then post your comments below. We are also accepting requests for any tutorials that you want in our website.
              </div>
