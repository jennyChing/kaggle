     <section class="primary island c">
      <h2>
       About this Add-on
      </h2>
      <div class="prose">
       <div class="prose" id="addon-description">
        <b>
         Download and save videos playing on a website to your hard disk.
        </b>
        <br/>
        <br/>
        <b>
         It's so easy:
        </b>
        <br/>
        <ul>
         <li>
          play a video e.g. on
          <a href="http://outgoing.mozilla.org/v1/577585700e711c66b2f847e9c2ba66f07706960fe129e88008d23cb2a93a014c/http%3A//youtube.com" rel="nofollow">
           youtube.com
          </a>
         </li>
         <li>
          click the "Downloader" icon in the toolbar. A menu comes up containing the available video formats
         </li>
         <li>
          click the green download-button of the video format you want to download
         </li>
         <li>
          a "file save"-dialog comes up where you can select the destination.
         </li>
        </ul>
        <br/>
        <b>
         That's it!
        </b>
        <br/>
        <br/>
        If you have disabled your toolbar you can also download the video via the context menu. Just right click to an empty area while the video is playing.
        <br/>
        <br/>
        Caution:
        <br/>
        The download of protected videos (RTMP protocol) is currently not possible.
        <br/>
        <br/>
        If you still want to download these videos, we recommend our free browser plugin at
        <a href="http://outgoing.mozilla.org/v1/8ba95ad2cbc89e471db440124484255604cfc7c8cb1928ebf8674786125d3d30/http%3A//www.startpage24.com" rel="nofollow">
         www.startpage24.com
        </a>
        for Chrome, Firefox and Internet Explorer.
       </div>
      </div>

     <div class="prose">
      Privacy Policy for Mozilla-Add-Ons
      <br/>
      <br/>
      When you use this Add-On with your Firefox web browser, we will place one or more data in a local storage of your browser. For example, we will store the path of your local destination folder there. Another example is, that we will store there the country of the a web store you are using (e.g. if you are using a certain store in the USA, or in another country).
      <br/>
      <br/>
      From time to time, we will display an advertising in your Add-On (e.g. in the Firefox UI). This will link to our server. So when you click on our advertising, Link64 collects the following:
      <br/>
      * Search term and time and date of your search
      <br/>
      * IP Address, browser configuration and approximate location
      <br/>
      * Country of the web store you are using, and language
      <br/>
      <br/>
      Then, we will re-direct you to our affiliate partner network. This might include companies like amazon, ebay, and others. With this re-direction we will forward the following information to our partner companies:
      <br/>
      * Search term and time and date of your search
      <br/>
      * IP Address, browser configuration and approximate location
      <br/>
      * Country of the web store you are using, and language
      <br/>
      <br/>
      Furthermore, our servers will from time to time get information from your web browser:
      <br/>
      * Time and date of this request
      <br/>
      * IP Address, browser configuration and approximate location
      <br/>
      This data will be stored for a limited time on our server and then be deleted.
      <br/>
      <br/>
      Unless described here, no data will be provided to any third party, and just used by us for internal statistics, which cannot be used to identify you.
      <br/>
      <br/>
      The above advertising is necessary for us, in order to maintain our business model of free Add-Ons.
