        <div class="articlebody">
         The conspicuous
         <a href="http://www.pedestrian.tv/news/arts-and-culture/simon-gittanys-girlfriend-rachelle-louise-defends-/2fa30caf-3d01-4b30-b6f3-f1e9be575a26.htm" target="_blank">
          absence of current girlfriend
         </a>
         <b>
          Rachelle Louise
         </b>
         cast a darkly ironic pall over an already bleak scene at the
         <b>
          NSW Supreme Court
         </b>
         this morning, where stoic
         <a href="http://www.pedestrian.tv/news/arts-and-culture/simon-gittany-found-guilty-of-murdering-girlfriend/bd50a5af-c61c-45bc-9243-b62ce388379b.htm" target="_blank">
          convicted criminal
         </a>
         <b>
          Simon Gittany
         </b>
         was sentenced to 26 years with an 18 year non parole period for the murder of his girlfriend
         <b>
          Lisa Harnum
         </b>
         in July 2011 by
         <a href="http://www.pedestrian.tv/news/arts-and-culture/simon-gittany-found-guilty-of-murdering-girlfriend/bd50a5af-c61c-45bc-9243-b62ce388379b.htm" target="_blank">
          throwing her off a fifteenth floor balcony
         </a>
         in what presiding
         <b>
          Justice Lucy McCallum
         </b>
         deemed a "state of rage".
         <div>
          <br/>
         </div>
         <div>
          Justice McCallum began her verdict by announcing that a sentence for the term of his natural life "would be excessive" for Gittany, a man she deemed "controlling, dominant and abusive" toward Harnum, whom she deduced must have been "in a state of complete terror" in the moments before her death when Gittany "unloaded [Harnum] with the intention of killing".
          <span style="font-size: 14px;">
           Senior Crown Prosector
          </span>
          <span style="font-size: 14px;">
          </span>
          <b style="font-size: 14px;">
           Mark Tedeschi
          </b>
          <span style="font-size: 14px;">
          </span>
          <span style="font-size: 14px;">
           had previously called for a sentence of at least 20 years; however Gittany's barrister argued for a more lenient term on the basis that the 40-year-old is "unlikely to reoffend."
          </span>
         </div>
         <div>
          <br/>
         </div>
         <div>
          Though she expressed no doubts as to its integrity, Justice McCallum also noted that the surprise witness testimony submitted last week from a friend of Harnum - who said Gittany "frequently threatened Lisa Harnum with death if she ever left him" - could not be taken into consideration as it undermined his right to a fair trial due to the laws of evidence and "unfair prejudice."
         </div>
         <div>
          <br/>
         </div>
         <div>
          McCallum determined that should could also not be satisfied that Harnum's murder was premeditated until a few minutes before her death, and was instead "facilitated by a sense of ownership over a woman [Gittany] claimed to love, an extraordinary degree of manipulative behaviour, arrogance and a sense of entitlement."
         </div>
         <div>
          <br/>
         </div>
         <div>
          The court also heard of Gittany's childhood as one of six immigrant children, whose mother provided "a secure and stable" upbringing likened to that of a hotel; how Gittany's previous offence of biting a police officer in the ear some twenty years prior has "troubling resonance with the present offence"; and that Gittany's total lack of remorse, maintaining his innocence, failure to accept responsibility and embracing of media attention demonstrates that his prospects of rehabilitation are "arid".
         </div>
         <div>
          <br/>
         </div>
         <div>
          Following the handing down of the verdict, Justice McCallum had to ask that screaming onlookers be removed from court, including Gittany's sister, who cried "In the name of Jesus Christ, you won't be doing any of that time". Gittany, meanwhile, demonstrated no emotion. His solicitor has stated that Gittany maintains his innocence and will lodge an appeal.
         </div>
         <div>
          <br/>
         </div>
         <div>
          He'll be eligible for parole in May 2031. Good riddance.
         </div>
         <div>

        $(function () {

    function addFloatingAdvert() {
        var adBox = $("#floating-ad-div")
        var floatingAd = $('.adspot.bottom')
        var bottomLimitElement = $("#page")
                
        if (isAdAllowed(floatingAd, bottomLimitElement.last(),adBox)) { 

            if (isScrolledTo(adBox)) { // floats the ad when scrolled to
                floatingAd.css({
                    position:'fixed',
                    top:'55px'
                });
                if (isElemTopScrolledToAnotherElem(floatingAd, adBox)) { // goes back into position if scrolling up
                    floatingAd.css({
                        position:'relative',
                        top:'auto'
                    });
                }
            } else { // the original position
                floatingAd.css({
                    position:'relative',
                    top:'auto'
                });
            }

            if (isElemBottomScrolledToBottomOf(floatingAd, bottomLimitElement.last())) {
                var topOfTheBottomElement = bottomLimitElement.last().offset().top;
                var bottomOfTheBottomElement = topOfTheBottomElement + bottomLimitElement.last().height();
                // topPosition should be the bottom of the bottomLimitElement + height of the ad - the mysterious 15
                var top = bottomOfTheBottomElement - floatingAd.height() - 15;
                floatingAd.css({
                    position:'absolute',
                    top:top
                });
            }
        }
    }

    function isScrolledTo(elem) {
        if (elem === undefined) {
            return false;
        }

        var docViewTop = $(window).scrollTop(); //num of pixels from the top of the viewable screen to the top of the page
        var elemTop = $(elem).offset().top - 80; //num of pixels from the top of the element to the top of the page
        return (elemTop &lt;= docViewTop);
    }

    function isElemBottomScrolledToBottomOf(elem, bottomLimit) {
        if (elem === undefined || bottomLimit === undefined) {
            return false;
        }

        var elemTop = $(elem).offset().top; //num of pixels from the top of the element to the top of the page
        var elemBottom = elemTop + $(elem).height();

        var bottomLimitTop = $(bottomLimit).offset().top;
        var bottomLimitBottom = bottomLimitTop + $(bottomLimit).height() -15;

        return (elemBottom &gt;= bottomLimitBottom);
    }
    
    function isAdAllowed(elem, bottomLimit, adBox) {
    	if (elem === undefined || bottomLimit === undefined) {
            return false;
        }

        var bottomLimitTop = $(bottomLimit).offset().top;
        var bottomLimitBottom = bottomLimitTop + $(bottomLimit).height() -15;

		var adBoxTop = $(adBox).offset().top
        
        return (bottomLimitBottom &gt;= adBoxTop + $(elem).height())

    }

    function isElemTopScrolledToAnotherElem(elem, otherElem) {
        var elemTop = $(elem).offset().top; //num of pixels from the top of the element to the top of the page
        var otherElemTop = $(otherElem).offset().top; //num of pixels from the top of the element to the top of the page
        var otherElemBottom = otherElemTop + $(elem).height();
        return (elemTop + 10 &lt;= otherElemBottom)
    }
