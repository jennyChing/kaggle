      <p>
       flipTimer is a jQuery plugin which enables you to count down towards or count up from a certain time like a vintage flip clock, depended on CSS3 animations, transitions, transforms and @keyframes.
      </p>
      <h2>
       See also:
      </h2>
      <ul>
       <li>
        <a href="http://www.jqueryscript.net/time-clock/jQuery-Flip-Clock-Countdown-Timer-Plugin-FlipClock.html">
         jQuery Retro Flip Clock &amp; Countdown Timer Plugin - FlipClock
        </a>
       </li>
       <li>
        <a href="http://www.jqueryscript.net/time-clock/Vintage-Flip-Clock-Style-Countdown-Timer-Plugin-For-jQuery-flipcountdown.html">
         Vintage Flip Clock Style Countdown &amp; Timer Plugin For jQuery - flipcountdown
        </a>
       </li>
       <li>
        <a href="http://www.jqueryscript.net/time-clock/Cool-Mechanical-Scoreboard-Style-Countdown-Plugin-For-jQuery-Countdown.html">
         Cool Mechanical Scoreboard Style Countdown Plugin For jQuery - Countdown
        </a>
       </li>
      </ul>
      <h2>
       How to use it:
      </h2>
      <p>
       1. Load jQuery library together with
       <code>
        flipTimer.css
       </code>
       and
       <code>
        jquery.flipTimer.js
       </code>
       .
      </p>
      <pre class="brush:xml;">
       &lt;link href="css/flipTimer.css" rel="stylesheet"&gt;

&lt;script src="//ajax.googleapis.com/ajax/libs/jquery/1.11.2/jquery.min.js"&gt;&lt;/script&gt;
&lt;script src="js/jquery.flipTimer.js"&gt;&lt;/script&gt;
      </pre>
      <p>
       2. Build the Html structure for the flip clock-style timer.
      </p>
      <pre class="brush:xml;">
       &lt;div class="flipTimer"&gt;
  &lt;div class="days"&gt;&lt;/div&gt;
  &lt;div class="hours"&gt;&lt;/div&gt;
  &lt;div class="minutes"&gt;&lt;/div&gt;
  &lt;div class="seconds"&gt;&lt;/div&gt;
&lt;/div&gt;
      </pre>
      <p>
       3. The sample JavaScript to start a countdown timer on page load.
      </p>
      <pre class="brush:js;">
       $(document).ready(function() {
$('.flipTimer').flipTimer({ 

// count up or countdown
direction: 'down', 

// the target
       <a href="http://www.jqueryscript.net/time-clock/">
        date
       </a>
       date: 'January 20, 2015 08:30:30', 

// callback works only with direction = "down"
callback: function() { alert('times up!'); }

});
});
      </pre>
      <p>
       4. All the configurations.
      </p>
      <pre class="brush:js;">
       $(document).ready(function() {
$('.flipTimer').flipTimer({ 

// count up or countdown
direction: 'down', 

// the target date
date: 'January 20, 2015 08:30:30', 

// callback works only with direction = "down"
callback: function() { alert('times up!'); }

// custom date
seconds: false,
minutes: false,
hours: false,
days: false,

});
});
      </pre>
