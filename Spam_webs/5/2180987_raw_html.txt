         <h1 class="title single-title">
          Why HTML5 Is Important?
         </h1>
         <!-- End Title -->
         <!-- Start Post Meta -->
         <div class="post-info">
          <span class="theauthor">
           <a href="http://www.weblinkindia.net/blog/author/palak-jain" rel="nofollow" title="Posts by Palak Jain">
            Palak Jain
           </a>
          </span>
          |
          <span class="thetime">
           December 8, 2011
          </span>
          |
          <span class="thecategory">

            HTML
           </a>
           ,
           <a href="http://www.weblinkindia.net/blog/category/html-5" rel="nofollow">
            HTML 5
           </a>
          </span>
          |
          <span class="thecomment">
           <a href="http://www.weblinkindia.net/blog/why-html5-is-important#comments">

        <!-- Start Content -->
        <div class="post-single-content box mark-links">
         <p>
          HTML5 is the fifth version of Hyper Text Mark-up Language (HTML). It is an improved version of HTML, where
          <a href="http://blogs.weblinkindia.net/2011/12/08/why-html5-is-important/why-html5-is-important/" rel="attachment wp-att-3283">
           <img alt="" class="alignleft wp-image-3283" height="288" src="http://weblinkindia.files.wordpress.com/2011/12/why-html5-is-important.jpg" title="Why HTML5 Is Important" width="300"/>
          </a>
          all the tools and techniques of the basic language are retained and presented in a simpler format. Since it supports all the data that has to be presented on the World Wide Web server, HTML5 is extremely significant. It is very helpful for the purpose of web designing and formatting. The language is used to structure &amp; present content on the websites and this latest version comes with advanced coding for providing more support to multimedia.
         </p>
         <p>
          <span style="color:#333399;">
           <strong>
            Why HTML5?
           </strong>
          </span>
          <br/>
          It is an easy to learn computer language, which helps in uploading content on the internet. It is a boon provided by the renowned IT company- Microsoft, for the web developers. This improved edition of Hyper Text Mark-up Language takes the support of various multimedia elements to improve the functions of the basic version of HTML and ahs many new syntactical features.
         </p>
         <p>
          <span style="color:#333399;">
           <strong>
            Why Using HTML5 Is A Must?
           </strong>
          </span>
         </p>
         <ul>
          <li>
           With HTML5, one can expect improvement in almost all the spheres of internet. It has triggered improvements in web development software as well. However, the main objective of HTML is to display multiple media without use of plugins and APIs on the internet. HTML5 is a successful User Interface platform.
          </li>
          <li>
           Applications are designed keeping in mind not only the computer but other gadgets as well. HTML is already popular among app developers. Now, Microsoft is promoting even HTML5 to them. Moreover, it is a fact that apps based on HTML5 will run conveniently on windows.
          </li>
          <li>
           HTML5 effectively acts as a base for a number of new computer applications. The latest version of HTML is well known for its following features:
           <ul>
            <li>
             Simple processing
            </li>
            <li>
             Data aggregation
            </li>
           </ul>
          </li>
         </ul>
         <p>
          HTML5 is surely something to look forward to. Numerous popular applications like PhoneGap and Appcelerator are now based on HTML5. Thus, it has a lot of scope and it makes perfect sense for the software developers to get acquainted with this version of the tried and tested HTML.
         </p>
