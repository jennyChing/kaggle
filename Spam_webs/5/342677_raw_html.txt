           <h1 class="post-title">
            Run a single Test Case in QA Complete
           </h1>
           <p class="post-byline">
            by
            <a href="http://www.rogeriodasilva.com/author/rdasilva/" rel="author" title="Posts by Rogerio da Silva">
             Rogerio da Silva
            </a>
             03/11/2015
           </p>
           <div class="clear">
           </div>
           <div class="entry">
            <div class="entry-inner">
             <p>
              To run a single test case in QA complete, just hit the
              <strong>
               blue play arrow
              </strong>
              indicated on this image under the test library of the test cases created earlier.
             </p>
             <div class="wp-caption aligncenter" id="attachment_549" style="width: 481px">
              <a href="http://i1.wp.com/www.rogeriodasilva.com/wp-content/uploads/2015/03/run_a_single_test_case1.png" rel="lightbox-0">
               <img alt="Run a single Test Case in QA Complete" class="wp-image-549 size-full" data-recalc-dims="1" src="http://i1.wp.com/www.rogeriodasilva.com/wp-content/uploads/2015/03/run_a_single_test_case1.png?resize=471%2C244"/>
              </a>
              <p class="wp-caption-text">
               Run a single Test Case in QA Complete
              </p>
             </div>
             <p>
              Wait for a pop up window with the test case in stand by ready to start.
             </p>
             <p>
              Hit the blue arrow named
              <strong>
               Run
              </strong>
              to start the test.
             </p>
             <div class="wp-caption aligncenter" id="attachment_550" style="width: 610px">
              <a href="http://i2.wp.com/www.rogeriodasilva.com/wp-content/uploads/2015/03/run_a_single_test_case-002.png" rel="lightbox-1">
               <img alt="Run a single Test Case in QA Complete" class="wp-image-550" data-recalc-dims="1" src="http://i2.wp.com/www.rogeriodasilva.com/wp-content/uploads/2015/03/run_a_single_test_case-002.png?resize=600%2C405"/>
              </a>
              <p class="wp-caption-text">
               Run a single Test Case in QA Complete
              </p>
             </div>
             <p>
              Here is the test screen where while you run your tests you can:
             </p>
             <ul>
              <li>
               Amend the line you are testing
              </li>
              <li>
               Add a new line
              </li>
              <li>
               Exclude the line
              </li>
              <li>
               Move up and down the line
              </li>
              <li>
               Make notes specially for failed tests
              </li>
              <li>
               You can pass the test and will close the test
              </li>
              <li>
               You can fail a test
              </li>
              <li>
               You can pause in case you need more time to complete the test
              </li>
              <li>
               You can end the run without complete the test
              </li>
             </ul>
             <div class="wp-caption aligncenter" id="attachment_551" style="width: 610px">
              <a href="http://i0.wp.com/www.rogeriodasilva.com/wp-content/uploads/2015/03/run_a_single_test_case-003.png" rel="lightbox-2">
               <img alt="Run a single Test Case in QA Complete" class="wp-image-551" data-recalc-dims="1" src="http://i0.wp.com/www.rogeriodasilva.com/wp-content/uploads/2015/03/run_a_single_test_case-003.png?resize=600%2C436"/>
              </a>
              <p class="wp-caption-text">
               Run a single Test Case in QA Complete
              </p>
             </div>
             <div class="alert alert-info ">
              if it fails and saved as fail you will be asked if you want to
              <a href="http://www.rogeriodasilva.com/raise-a-defect-in-qa-complete" target="_blank">
               raise a defect
              </a>
              .
             </div>
             <p>
              Once test is complete you can:
             </p>
             <ul>
              <li>
               Review Test
              </li>
              <li>
               Send Email with the test results
              </li>
             </ul>
             <div class="wp-caption aligncenter" id="attachment_552" style="width: 610px">
              <a href="http://i0.wp.com/www.rogeriodasilva.com/wp-content/uploads/2015/03/run_a_single_test_case-004.png" rel="lightbox-3">
               <img alt="Run a single Test Case in QA Complete" class="wp-image-552" data-recalc-dims="1" src="http://i0.wp.com/www.rogeriodasilva.com/wp-content/uploads/2015/03/run_a_single_test_case-004.png?resize=600%2C436"/>
              </a>
              <p class="wp-caption-text">
               Run a single Test Case in QA Complete
              </p>
             </div>
             <p>
              The print screen with friendly view.
             </p>
             <div class="wp-caption aligncenter" id="attachment_553" style="width: 610px">
              <a href="http://i1.wp.com/www.rogeriodasilva.com/wp-content/uploads/2015/03/run_a_single_test_case-005.png" rel="lightbox-4">
               <img alt="Run a single Test Case in QA Complete" class="wp-image-553" data-recalc-dims="1" src="http://i1.wp.com/www.rogeriodasilva.com/wp-content/uploads/2015/03/run_a_single_test_case-005.png?resize=600%2C403"/>
              </a>
              <p class="wp-caption-text">
               Run a single Test Case in QA Complete
              </p>
             </div>
             <p>
              Send email from the application it self.
             </p>
             <p>
              It attaches the run details.
             </p>
             <div class="wp-caption aligncenter" id="attachment_554" style="width: 610px">
              <a href="http://i2.wp.com/www.rogeriodasilva.com/wp-content/uploads/2015/03/run_a_single_test_case-006.png" rel="lightbox-5">
