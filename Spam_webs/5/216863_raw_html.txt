         <p class="google_elide">
          Glamour magazine isn't known for it's Internet-savvy marketing. But when you put Anna Kendrick on the cover, you have to do something special to celebrate the occasion.
         </p>
         <!--break-->
         <p class="google_elide">
          So, they had the Pitch Perfect 2 star recite some of Reddit's "Shower Thoughts," which are pretty much the same as SNL's old "Deep Thoughts" by Jack Handeybut on the Internet. Anna even throws a few of her own in there. Simple, smart and effective.
          <br/>
          <br/>
          <iframe allowfullscreen="" frameborder="0" height="367" src="//www.youtube.com/embed/O0wUwHIIg2A?rel=0" width="652">
          </iframe>
          <br/>
          <br/>
          If you're familiar with "Shower Thoughts," you will have heard most of these before, but somehow the magic of Kendrick, an advertising darling who hasn't had a miss with a campaign, elevates those old lines into true stoner mind-blowing territory. (Not that it's exactly groundbreakingyou can, after all, also watch
          <a href="https://www.youtube.com/watch?v=wBNJ0BH3Dgs" target="_blank">
           Cookie Monster's "Shower Thoughts"
          </a>
          for The Watercooler or
          <a href="https://www.youtube.com/watch?v=OK34L4-qaDQ" target="_blank">
           Nick Offerman's "Shower Thoughts"
          </a>
          for Mashable.)
         </p>
         <p class="google_elide">
          After just one day, it's already one of the most popular videos on Glamour's YouTube. Peep at all the other videos on the channel, and you'll see this is really out of character for them. But it's perfect for the Internet.
         </p>

    function _formatCounter(value) {
      value = (isNaN(value)) ? 0 : parseInt(value);

      if (value &gt; 1000 &amp;&amp; value &lt; 10000) {
        count = parseFloat(value/1000);
        count = (count).toFixed(1);
        return count + 'k';
      } else if (value &gt;= 10000) {
        count = parseInt(value/1000);
        return count + 'k';
      }
      else {
        return value;
      }

    // determine the top and left for the share bar based on calculations involving different divs and the screen
    function reposition() {
      var element = $("#shell #main .left-share-bar-wrapper");
      var offset = element.offset();

      if($("#shell #main .node-headerimage").length &gt; 0) {
        var headerImage = $("#shell #main .node-headerimage");
      } else if($("#shell #main #adfreakad").length &gt; 0) {
        var headerImage = $("#shell #main #adfreakad");
      } else if($("#shell #main .views-field-field-sa-section-header-fid").length &gt; 0) {
        var headerImage = $("#shell #main .views-field-field-sa-section-header-fid");
      }

      if($("#shell #main .subheader .headings").length &gt; 0) {
        var header = $("#shell #main .subheader .headings");
      } else if($("#shell #main .headline").length &gt; 0) {
        var header = $("#shell #main .headline");
      }

      var headerOffset = header.offset();

      if($("#shell #main .article-footer").length &gt; 0) {
        var footer = $("#shell #main .article-footer");
      } else if($("#shell #main .cols.colsAB .mod.latest-news-popular-now").length &gt; 0) {
        var footer = $("#shell #main .mod.latest-news-popular-now");
      } else if($("#shell #main .cols.colsAB #comments").length &gt; 0) {
        var footer = $("#shell #main #comments");
      } else if($(".node-type-sa-article").length &gt; 0) {
        var footer = $("#shell #pre-footer");
      }

      footerOffset = footer.offset();

      if($(window).scrollTop() &gt;= headerOffset.top + header.outerHeight(true) + 3 &amp;&amp; $(window).scrollTop() &lt; footerOffset.top - element.outerHeight(true) + 2) {
        // we are below the header or above the footer; put the social icons on top

        element.css({
            position: "fixed",
            top: 0,
            left: headerOffset.left - 82 + "px"
        });

            position: "absolute",
            top: headerImage.outerHeight(true) + $("#shell #main .inner #post").outerHeight(true) + $("#shell #main .mod.mod-related").outerHeight(true) - element.outerHeight(true) + 20,

            position: "absolute",
            top: $("#shell #main .inner").outerHeight(true) - element.outerHeight(true) + 2,

    <div id="print-content" style="display: none;">
     <p>
      Glamour magazine isn't known for it's Internet-savvy marketing. But when you put Anna Kendrick on the cover, you have to do something special to celebrate the occasion.
     </p>
     <!--break-->
     <p>
      So, they had the Pitch Perfect 2 star recite some of Reddit's "Shower Thoughts," which are pretty much the same as SNL's old "Deep Thoughts" by Jack Handeybut on the Internet. Anna even throws a few of her own in there. Simple, smart and effective.
      <br/>
      <br/>
