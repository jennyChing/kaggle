       <a class="postLink" href="post.php?id=8045">
        How a Gun Gets Made
       </a>
       <h2>
        June 21, 2012 - Community Creator and TF Team
       </h2>
       <p>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image001.jpg" width="546"/>
        <br/>
        <br/>
        A lot of you have been asking how all those nifty weapons get made for TF2. Well, youre in luck. TF2 workshop contributor
        <a href="http://steamcommunity.com/id/jalcober
">
         Jalcober
        </a>
        volunteered to talk you through exactly how he does it in his own words, tracking the progress of a pretty sweet Pyro gun he made called the Scorch Shot. But why not let him explain it himself:
        <br/>
        <br/>
        "I started out with this excellent concept sketch from my fellow Steam Workshop contributor
        <a href="http://steamcommunity.com/id/GRND_CTRL
">
         [Square]
        </a>
        ."
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image002.jpg" width="546"/>
        <br/>
        <br/>
        "Next, I created an image plane with the concept art on it and placed a cylinder primitive over it."
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image003.jpg" width="546"/>
        <br/>
        <br/>
        "Since I was hoping the gun would get accepted into TF2, I wanted it to match the size, style and polycount of the existing items for the Pyro. Luckily, I found an
        <a href=" https://developer.valvesoftware.com/wiki/Blender_SMD_Tools">
         importer
        </a>
        that brings Valve game assets into
        <a href="http://www.blender.org/">
         Blender
        </a>
        (an open source 3D modeling tool), so I could load the Pyros existing flare gun up next to mine."
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image004.jpg" width="546"/>
        <br/>
        <br/>
        "Since I wanted the gun to feel like it was assembled from parts in the TF2niverse, I decided to make each piece separately. Each 'part' started out as a flat polygon that I traced over the concept art on an image plane."
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image006.jpg" width="546"/>
        <br/>
        <br/>
        "Next, each part got extruded and adjusted for thickness."
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image007.jpg" width="546"/>
        <br/>
        <br/>
        "I repeated this process to complete the rest of the trigger assembly."
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image008.jpg" width="546"/>
        <br/>
        <br/>
        "It's easier to build half a model and then mirror across the centerline than it is to build the whole thing."
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image009.jpg" width="546"/>
        <br/>
        <br/>
        "I did a little prep work on the facet of the grip padding before extruding it, so I wouldn't have to tweak all the new extruded vertices. (Doing a little edge work before extruding can save a lot of time.)"
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image010.jpg" width="546"/>
        <br/>
        <br/>
        "I decided to add some smaller details the same way."
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image011.jpg" width="546"/>
        <br/>
        <br/>
        "Next, I added a cylinder to create the barrel grip and the mount."
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image012.jpg" width="546"/>
        <br/>
        <br/>
        "With all the pieces complete, I created a flattened UV layout so that I could paint directly onto the model in Photoshop."
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image013.jpg" width="546"/>
        <br/>
        <br/>
        "Before moving over to Photoshop, I baked the ambient occlusion lighting into the texture so I'd be able to tell which parts were which."
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image014.jpg" width="546"/>
        <br/>
        <br/>
        "Now I was ready to start painting the color texture map."
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image015.jpg" width="546"/>
        <br/>
        <br/>
        "Using [Square]'s concept art as a guide, I selected the regions of the 3D model that I wanted to be darkened."
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image016.jpg" width="546"/>
        <br/>
        <br/>
        "I then lasso-selected those same regions in Photoshop and created a layer underneath the ambient occlusion layer. I then set the ambient occlusion layer to "Multiply" so that it only darkened my color layer underneath. Any part of the ambient occlusion layer that was white then became transparent."
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image017.jpg" width="546"/>
        <br/>
        <br/>
        "Back in Blender, I turned off all the lights in the 3D viewport so I could see how my unlit texture was looking on the model while simultaneously painting it in Photoshop."
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image018.jpg" width="546"/>
        <br/>
        <br/>
        At this point, I took a screenshot of the gun in the 3D viewport and did a Photoshop paintover. This let me ignore the polygons and make broad changes with a Brush Tool without having to worry about the 3D scene complexity. When I was happy, I then merged these changes back onto the model in 3D. (I had to rebake the ambient occlusion layer because I'd moved the screws.)
        <br/>
        <br/>
        <img height="369" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image019.jpg" width="546"/>
        <br/>
        <br/>
        "Next, I compiled the model and tested it out in-game by using the item test map"
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image023.jpg" width="546"/>
        <br/>
        <br/>
        "Lastly, I published my item to the
        <a href="http://steamcommunity.com/workshop/browse/?appid=440&amp;p=1&amp;browsesort=toprated&amp;browsefilter=toprated&amp;section=items
">
         Steam Workshop
        </a>
        and crossed my fingers!"
        <br/>
        <br/>
        <img height="415" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image021.jpg" width="546"/>
        <br/>
        <br/>
        Thanks Jalcober! Now back at Valve, those of us in the TF Team are always reviewing items in the Workshop, and we thought Jalcober's flare gun model would be a great fit for the universe. We got in touch with Jalcober and asked if he could make some lower levels of detail (
        <a href="https://developer.valvesoftware.com/wiki/Level_of_detail">
         LODs
        </a>
        ) to help keep the game running smoothly on low-end computers. We also asked that he resubmit the item using the
        <a href=" https://developer.valvesoftware.com/wiki/Itemtest">
         Itemtest Wizard
        </a>
        file structure to make it easier for us to review the submission.
        <br/>
        <br/>
        <img height="340" src="http://media.steampowered.com/apps/tf2/blog/make_a_gun/gun_image022.jpg" width="546"/>
        <br/>
        <br/>
        From here, the team brainstormed some ideas for gameplay that wouldn't unbalance the game, but would hopefully still be a fun new addition for players. We created a new particle system and began playtesting it in-game.
        <br/>
        <br/>
        And that's how a concept becomes a working gun in Team Fortress 2. In fact, look for it in an upcoming update!
