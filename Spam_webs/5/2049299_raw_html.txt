     <p>
      <b>
       Molding
      </b>
      or
      <b>
       moulding
      </b>
      (see
      <a href="/wiki/American_and_British_English_spelling_differences" title="American and British English spelling differences">
       spelling differences
      </a>
      ) is the process of
      <a href="/wiki/Manufacturing" title="Manufacturing">
       manufacturing
      </a>
      by shaping liquid or pliable raw material using a rigid frame called a mold or matrix.
      <sup class="reference" id="cite_ref-1">
       <a href="#cite_note-1">
        <span>
         [
        </span>
        1
        <span>
         ]
        </span>
       </a>
      </sup>
      This itself may have been made using a pattern or model of the final object.
     </p>
     <p>
      A
      <b>
       mold
      </b>
      or
      <b>
       mould
      </b>
      is a hollowed-out block that is filled with a liquid or pliable material like
      <a href="/wiki/Plastic" title="Plastic">
       plastic
      </a>
      ,
      <a href="/wiki/Glass" title="Glass">
       glass
      </a>
      ,
      <a href="/wiki/Metal" title="Metal">
       metal
      </a>
      , or
      <a href="/wiki/Ceramic" title="Ceramic">
       ceramic raw materials
      </a>
      .
      <sup class="reference" id="cite_ref-2">
       <a href="#cite_note-2">
        <span>
         [
        </span>
        2
        <span>
         ]
        </span>
       </a>
      </sup>
      The liquid hardens or sets inside the mold, adopting its shape. A mold is the counterpart to a
      <a href="/wiki/Casting" title="Casting">
       cast
      </a>
      . The very common bi-valve molding process uses two molds, one for each half of the object.
      <b>
       Piece-molding
      </b>
      uses a number of different molds, each creating a section of a complicated object. This is generally only used for larger and more valuable objects.
     </p>
     <p>
      The manufacturer who makes the molds is called the
      <a href="/wiki/Moldmaker" title="Moldmaker">
       moldmaker
      </a>
      . A
      <a href="/wiki/Release_agent" title="Release agent">
       release agent
      </a>
      is typically used to make removal of the hardened/set substance from the mold easier. Typical uses for molded plastics include molded
      <a href="/wiki/Furniture" title="Furniture">
       furniture
      </a>
      , molded
      <a href="/wiki/Household_goods" title="Household goods">
       household goods
      </a>
      , molded
      <a href="/wiki/Case_(goods)" title="Case (goods)">
       cases
      </a>
      , and structural materials.
     </p>
     <p>
      Types of molding include:
     </p>
     <ul>
      <li>
       <a href="/wiki/Blow_molding" title="Blow molding">
        Blow molding
       </a>
      </li>
      <li>
       <a href="/wiki/Powder_metallurgy" title="Powder metallurgy">
        Powder metallurgy
       </a>
       plus
       <a href="/wiki/Sintering" title="Sintering">
        sintering
       </a>
      </li>
      <li>
       <a href="/wiki/Compression_molding" title="Compression molding">
        Compression molding
       </a>
      </li>
      <li>
       <a class="mw-redirect" href="/wiki/Extrusion_molding" title="Extrusion molding">
        Extrusion molding
       </a>
      </li>
      <li>
       <a class="mw-redirect" href="/wiki/Injection_molding" title="Injection molding">
        Injection molding
       </a>
      </li>
      <li>
       <a class="mw-redirect" href="/wiki/Laminating" title="Laminating">
        Laminating
       </a>
       <ul>
        <li>
         <a href="/wiki/Reaction_injection_molding" title="Reaction injection molding">
          Reaction injection molding
         </a>
        </li>
       </ul>
      </li>
      <li>
       <a href="/wiki/Matrix_molding" title="Matrix molding">
        Matrix molding
       </a>
      </li>
      <li>
       <a href="/wiki/Rotational_molding" title="Rotational molding">

       </a>
       (or Rotomolding)
      </li>
      <li>
       <a href="/wiki/Spin_casting" title="Spin casting">
        Spin casting
       </a>
      </li>
      <li>
       <a href="/wiki/Transfer_molding" title="Transfer molding">
        Transfer molding
       </a>
      </li>
      <li>
       <a href="/wiki/Thermoforming" title="Thermoforming">

        <li>

         </a>
         , a simplified version of thermoforming
        </li>
       </ul>
      </li>
