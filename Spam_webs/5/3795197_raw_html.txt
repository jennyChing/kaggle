        <p>
         As reported earlier that
         <a href="http://www.iphoneheat.com/2010/05/how-to-install-android-os-on-iphone-2g/" target="_blank">
          Android OS
         </a>
         has successfully been ported to
         <a href="http://www.iphoneheat.com/2010/05/install-android-on-iphone-3g-2g-iphodroid/" target="_blank">
          iPhone 3G
         </a>
         and
         <a href="http://www.iphoneheat.com/2010/05/how-to-install-android-os-on-iphone-2g/" target="_blank">
          iPhone 2G
         </a>
         by Planetbeing the member of the Dev-Team. Then we told you about
         <a href="http://www.iphoneheat.com/2010/05/install-android-on-iphone-3g-2g-iphodroid/" target="_blank">
          iPhoDroid
         </a>
         , an automated installation of
         <a href="http://www.iphoneheat.com/tag/android-on-iphone/" target="_blank">
          Android on iPhone
         </a>
         . Now, the developer has updated
         <strong>
          iPhoDroid
         </strong>
         to v0.6 which brings the following new features:
         <br/>
         <span id="more-12989">
         </span>
         <img alt="iphodoid" class="alignright size-full wp-image-13001" height="100" src="http://www.iphoneheat.com/wp-content/uploads/2010/06/iphodroid.jpg" title="iphodroid" width="100"/>
        </p>
        <ul>
         <li>
          Now supports iPhone 2G/3G
         </li>
         <li>
          Install/Uninstall
         </li>
         <li>
          OpeniBoot standalone
         </li>
         <li>
          Choose the image you want
         </li>
         <li>
          Choose the kernel
         </li>
         <li>
          MD5 checker
         </li>
         <li>
          NOR backup
         </li>
        </ul>
        <h2>
         Download iPhoDroid v0.6
        </h2>
        <p>
         You can
         <strong>
          download iPhoDroid v0.6
         </strong>
         from
         <a href="http://code.google.com/p/iphodroid/updates/list" target="_blank">
          here
         </a>
         .
        </p>
        <p>
         The developer has promised a video tutorial on
         <strong>
          how to install
          <a href="http://www.iphoneheat.com/tag/android-on-iphone/" target="_blank">
           android on iPhone
          </a>
          using iPhoDroid
         </strong>
         . Well keep you posted as the tutorial comes out.
        </p>
