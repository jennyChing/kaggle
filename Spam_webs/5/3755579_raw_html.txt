        <h4>
         Instructions:
        </h4>
        <p style="text-align: justify;">
         Building upon the concepts of
         <a href="http://raspberrypi.powersbrewery.com/project-4-sequential-led/" title="Project 4">
          Project 4
         </a>
         , my son and I decided we wanted to attempt to simulate a standard American traffic light. This is a common Raspberry Pi project but this seemed like the best time in our learning process to insert this type of experiment. It builds upon our previous LED projects and also introduces a bit more logic as well as more complex programming concepts.
        </p>
        <p style="text-align: justify;">
         To begin, we wired 3 LEDs (
         <span style="color: #ff0000;">
          Red
         </span>
         ,
         <span style="color: #ffcc00;">
          Yellow
         </span>
         , and
         <span style="color: #008000;">
          Green
         </span>
         ) in a similar manner to our previous projects. We utilized GPIO 18, GPIO 23, and GPIO 24 for the simulated traffic light. We utilized the Geany IDE within the Raspbian desktop to create thePython script shown below:
        </p>
        <pre class="brush: python; title: ; notranslate" title="">
         #!/usr/bin/env python
# -*- coding: utf-8 -*-
#
#  traffic_light_LED.py
#
# Make 3 LEDs simulate an American street light
#
#  Copyright 2015  Ken Powers
#

# Import the modules used in the script
import time
import RPi.GPIO as GPIO

# Assign constants for the light GPIO pins
red_led = 18
yellow_led = 23
green_led = 24
RUNNING = True

# Configure the GPIO to BCM and set the pins to output mode
GPIO.setmode(GPIO.BCM)
GPIO.setup(red_led, GPIO.OUT)
GPIO.setup(yellow_led, GPIO.OUT)
GPIO.setup(green_led, GPIO.OUT)

# Define a function to control the traffic light
def trafficState(red, yellow, green):
	GPIO.output(red_led, red)
	GPIO.output(yellow_led, yellow)
	GPIO.output(green_led, green)

print "Traffic Light Simulation. Press CTRL + C to quit"

# Main loop
try:
    while RUNNING:
		# Green for 13 seconds
		trafficState(0,0,1)
		time.sleep(13)
		# Yellow for 3 seconds
		trafficState(0,1,0)
		time.sleep(3)
		# Red for 10 seconds
		trafficState(1,0,0)
		time.sleep(10)

# If CTRL+C is pressed the main loop is broken
except KeyboardInterrupt:
    RUNNING = False
    print "\Quitting"

# Actions under 'finally' will always be called
finally:
    # Stop and finish cleanly so the pins
    # are available to be used again
    GPIO.cleanup()
        </pre>
        <h4>
         Code Analysis:
        </h4>
        <p style="text-align: justify;">
         Once again, we begin by making sure the interpreter knows we are using the Python language. We also make a few comments so code analysts know what our Python script is going to accomplish.
        </p>
        <div class="advads-in-post-content" id="advads-2057817525" style="text-align: center; ">
         <ins class="adsbygoogle" data-ad-client="ca-pub-4991912708243329" data-ad-format="auto" data-ad-slot="5625767649" style="display:block;">
         </ins>
         <script>
          (adsbygoogle = window.adsbygoogle || []).push({});
         </script>
        </div>
        <pre class="brush: python; title: ; notranslate" title="">
         #!/usr/bin/env python
# -*- coding: utf-8 -*-
#
#  traffic_light_LED.py
#
# Make 3 LEDs simulate an American street light
#
#  Copyright 2015  Ken Powers
#
        </pre>
        <p>
         Next, we import some of the external modules we plan to incorporate into our code.
        </p>
        <pre class="brush: python; title: ; notranslate" title="">
         # Import the modules used in the script
import time
import RPi.GPIO as GPIO
        </pre>
        <p>
         We define the constants used for this particular project.
        </p>
        <pre class="brush: python; title: ; notranslate" title="">
         # Assign constants for the light GPIO pins
red_led = 18
yellow_led = 23
green_led = 24
RUNNING = True
        </pre>
        <p>
         We then configure the GPIO to use the Broadcom numbering system and set our LED pins to output mode.
        </p>
        <pre class="brush: python; title: ; notranslate" title="">
         # Configure the GPIO to BCM and set the pins to output mode
GPIO.setmode(GPIO.BCM)
GPIO.setup(red_led, GPIO.OUT)
GPIO.setup(yellow_led, GPIO.OUT)
GPIO.setup(green_led, GPIO.OUT)
        </pre>
        <p style="text-align: justify;">
         For the first time in our series of projects, we introduce the usage of a function. In this case, we define a simple function that controls our traffic light. We are able to call it by name from our main program loop and pass either a zero or a one to indicate whether the
         <span style="color: #ff0000;">
          Red
         </span>
         ,
         <span style="color: #ffcc00;">
          Yellow
         </span>
         , or
         <span style="color: #008000;">
          Green
         </span>
         LEDs are to be illuminated.
        </p>
        <pre class="brush: python; title: ; notranslate" title="">
         # Define a function to control the traffic light
def trafficState(red, yellow, green):
	GPIO.output(red_led, red)
	GPIO.output(yellow_led, yellow)
	GPIO.output(green_led, green)
        </pre>
        <p style="text-align: justify;">
         Our main loop is very similar to previous projects except this time it calls the function
         <strong>
          trafficState
         </strong>
         that we defined earlier. We are able to call this function and pass a 0 or 1 to indicate whether we want the
         <span style="color: #ff0000;">
          Red
         </span>
         ,
         <span style="color: #ffcc00;">
          Yellow
         </span>
         , or
         <span style="color: #008000;">
          Green
         </span>
         LEDs to illuminate. In our case, the
         <span style="color: #008000;">
          Green
         </span>
         LED illuminates for 13 seconds, the
         <span style="color: #ffcc00;">
          yellow
         </span>
         LED illuminates for 3 seconds, and then the
         <span style="color: #ff0000;">
          Red
         </span>
         LED illuminates for 10 seconds to simulate the cycling that an American traffic light performs as it directs cars at an intersection.
        </p>
        <pre class="brush: python; title: ; notranslate" title="">
         # Main loop
try:
    while RUNNING:
		# Green for 13 seconds
		trafficState(0,0,1)
		time.sleep(13)
		# Yellow for 3 seconds
		trafficState(0,1,0)
		time.sleep(3)
		# Red for 10 seconds
		trafficState(1,0,0)
		time.sleep(10)

# If CTRL+C is pressed the main loop is broken
except KeyboardInterrupt:
    RUNNING = False
    print "\Quitting"

# Actions under 'finally' will always be called
finally:
    # Stop and finish cleanly so the pins
    # are available to be used again
    GPIO.cleanup()
        </pre>
        <p style="text-align: justify;">
         As can be seen from the above Python script, simply modifying our existing code from the previous project allows us to use a similar breadboard layout in order to accomplish a different task. Try modifying the script to have different cycling times for the LEDs. Above all, have fun!
        </p>
