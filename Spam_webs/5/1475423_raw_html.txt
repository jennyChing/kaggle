      <p>
       A simple implementation of responsive, mobile-friendly, off-canvas push menu using Html5, CSS3 and a little jQuery magic.
      </p>
      <h2>
       How to use it:
      </h2>
      <p>
       1. Create a regular nav menu from an unordered list.
      </p>
      <pre class="brush:xml;">
       &lt;header&gt;
  &lt;nav class="nav"&gt;
    &lt;ul class="nav-list"&gt;
      &lt;li class="nav-item"&gt;&lt;a href="#"&gt;Home&lt;/a&gt;&lt;/li&gt;
      &lt;li class="nav-item"&gt;&lt;a href="#"&gt;
       <a href="http://www.jqueryscript.net/tags.php?/Portfolio/">
        Portfolio
       </a>
       &lt;/a&gt;&lt;/li&gt;
      &lt;li class="nav-item"&gt;&lt;a href="#"&gt;Settings&lt;/a&gt;&lt;/li&gt;
      &lt;li class="nav-item"&gt;&lt;a href="#"&gt;Services&lt;/a&gt;&lt;/li&gt;
      &lt;li class="nav-item"&gt;&lt;a href="#"&gt;Contact&lt;/a&gt;&lt;/li&gt;
    &lt;/ul&gt;
  &lt;/nav&gt;
&lt;/header&gt;
      </pre>
      <p>
       2. Insert a trigger button into the nav menu.
      </p>
      <pre class="brush:xml;">
       &lt;button type="button" class="nav-trigger js-nav-trigger"&gt;&lt;span&gt;&lt;/span&gt;&lt;span&gt;&lt;/span&gt;&lt;span&gt;&lt;/span&gt;&lt;/button&gt;
      </pre>
      <p>
       3. Hide the trigger button &amp; mobile push menu on the large screens.
      </p>
      <pre class="brush:css;">
       .nav-trigger { display: none; }
.mobile-nav { display: none; }
      </pre>
      <p>
       4. Set the breakpoint using CSS3 media queries and style the mobile push menu whatever you want.
      </p>
      <pre class="brush:css;">
       @media (max-width: 640px) {

header {
  right: 0;
  transition: right .5s;
}

.nav-trigger {
  display: block;
  position: absolute;
  right: 0;
  top: 8px;
  background: transparent;
  border: none;
}

.nav-trigger:focus { outline: none; }

.show-nav .nav-trigger { margin-top: 10px; }

.nav-trigger span {
  position: relative;
  display: block;
  width: 25px;
  height: 4px;
  margin: 4px 0;
  border-radius: 5px;
  background-color: #fff;
  transition: all .2s;
}

.show-nav .nav-trigger span:nth-child(1) { transform: rotate(405deg); }

.show-nav .nav-trigger span:nth-child(2) { opacity: 0; }

.show-nav .nav-trigger span:nth-child(3) {
  transform: rotate(-405deg);
  bottom: 16px;
}

.nav {
  position: absolute;
  right: 0;
  top: 0;
  margin-top: 0;
  width: 50px;
  height: 50px;
}

.nav-list { display: none; }

.mobile-nav .nav-list { display: block; }

.mobile-nav {
  display: block;
  background: #2a2a2a;
  position: absolute;
  left: 100%;
  bottom: 0;
  top: 0;
  width: 80%;
  transition: left .5s;
}

.show-nav .mobile-nav { left: 20%; }

.nav-item {
  width: 100%;
  border-bottom: 1px solid #333;
}

.nav-item a .fa {
  position: absolute;
  left: 10px;
  color: #fff;
  font-size: 18px;
}

.nav-item a {
  display: block;
  padding: 15px 20px 15px 40px;
  border: none;
}

.nav-list a { color: #fff !important; }

.show-nav header { right: 80%; }

}
      </pre>
      <p>
       5. The JQuery script to active the push menu on mobile devices (screen width &lt; 640px).
      </p>
      <pre class="brush:js;">
       jQuery(function($){
  
  
  $('body').append($('&lt;div class="mobile-nav"&gt;&lt;/div&gt;').html($('.nav--list').clone()));
  

  $('.js-nav-trigger').on('click',function(event){
    event.preventDefault();
    $('body').toggleClass('show-nav');
  });
       
});
      </pre>
