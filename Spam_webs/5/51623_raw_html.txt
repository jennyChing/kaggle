      <div class="section" id="literally-canvas">
       <h1>
        Literally Canvas
        <a class="headerlink" href="#literally-canvas" title="Permalink to this headline">
         
        </a>
       </h1>
       <p>
        Literally Canvas is an extensible, open source (BSD-licensed), HTML5 drawing
widget. Its only dependency is
        <a class="reference external" href="http://facebook.github.io/react/">
         React.js
        </a>
        .
       </p>
       <div class="literally index">
       </div>
       <script>
        $(document).ready(function() {
    $('.literally.index').literallycanvas({
      backgroundColor: 'whiteSmoke'
    });
  });
       </script>
       <p>
        Literally Canvass major features include:
       </p>
       <ul class="simple">
        <li>
         Basic drawing tools
        </li>
        <li>
         An API to add more drawing tools
        </li>
        <li>
         Panning and zooming
        </li>
        <li>
         Constant-size or infinite canvases
        </li>
        <li>
         Background images/shapes and watermarks
        </li>
        <li>
         Retina support
        </li>
        <li>
         Saving and loading JSON
        </li>
        <li>
         Events
        </li>
        <li>
         Exporting drawings to PNG
        </li>
       </ul>
       <p>
        If you would like to support Literally Canvass development or provide extra
incentive to implement your favorite feature request, consider donating at
        <a class="reference external" href="https://www.gittip.com/irskep/">
         gittip.com/irskep
        </a>
        .
       </p>

      <div class="section" id="why">
       <h1>
        Why?
        <a class="headerlink" href="#why" title="Permalink to this headline">
         
        </a>
       </h1>
       <p>
        If you look at
        <a class="reference internal" href="#related-projects">
         <em>
          Related projects
         </em>
        </a>
        at the bottom of the page, you can see
that this isnt the only HTML5 drawing tool on the block. However, it is unique
in several ways:
       </p>
       <ul class="simple">
        <li>
         <strong>
          Its open source.
         </strong>
         Its developed by the community for many purposes and
is free to use and extend.
        </li>
        <li>
         <strong>
          Its a widget, not an application.
         </strong>
         Literally Canvas is intended to live
inside your application in the way that works best for you.
        </li>
        <li>
         <strong>
          It wants to be extended.
         </strong>
         There is a public API for adding new shapes and
tools.
        </li>
        <li>
         <strong>
          Its fast and looks good.
         </strong>
         Literally Canvas is optimized to work with
complex drawings, and it smooths lines naturally to avoid the unsightly line
angles often seen in simplistic drawing programs.
        </li>
       </ul>
       <p>
        Literally Canvas was created in part to replace the object drawing tool in
        <a class="reference external" href="http://playbuildy.com">
         Buildy
        </a>
        . We hope youre inspired to use it to make new things of your own!
       </p>
      </div>

      <div class="section" id="contributing">
       <h1>
        Contributing
        <a class="headerlink" href="#contributing" title="Permalink to this headline">
         
        </a>
       </h1>
       <p>
        We are excited to see contributions of any form. New brushes, optimizations,
feature requests, use cases, designs, bug reports  you name it, we want it!
Just head on over to
        <a class="reference external" href="http://github.com/literallycanvas/literallycanvas">
         GitHub
        </a>
        and get started.
       </p>
       <p>
        Literally Canvas is developed by volunteers for fun. If missing features or
compatibility are serious issues for you, consider that the solution to your
problem might be to help us write the code.
       </p>
      </div>

      <div class="section" id="related-projects">
       <span id="id3">

       <h1>
        Related projects
        <a class="headerlink" href="#related-projects" title="Permalink to this headline">
         
        </a>
       </h1>
       <p>
        The web has only recently arrived at the point where this sort of project is
feasible to implement without Flash, but there are several other projects with
similar features. If Literally Canvas isnt quite what youre looking for, we
encourage you to check them out and give them some love. All of them use
       </p>
       <ul class="simple">
        <li>
         <a class="reference external" href="http://leimi.github.io/drawingboard.js/">
          drawingboard.js
         </a>
         : Very similar
project, slightly different feature set.
        </li>
        <li>
         <a class="reference external" href="http://sta.sh/muro/">
          DeviantArt Muro
         </a>
         : Sophisticated, comprehensive
drawing
        </li>
        <li>
         <a class="reference external" href="http://bomomo.com/">
          Bomomo
         </a>
         : Whimsical doodling
        </li>
        <li>
         <a class="reference external" href="http://mudcu.be/sketchpad/">
          SketchPad
         </a>
         : Simple drawing
        </li>
        <li>
         <a class="reference external" href="http://sketchpad.io/">
          sketchpad.io
         </a>
         : Non-destructive, 2.0 version of
SketchPad
        </li>
        <li>
         <a class="reference external" href="https://github.com/mudcube/Sketch.js">
          Sketch.js
         </a>
         : Simplistic, open source
version of
         <a class="reference external" href="http://sketchpad.io/">
          sketchpad.io
         </a>
        </li>
        <li>
         <a class="reference external" href="http://hakim.se/experiments/html5/sketch/">
          Sketch
         </a>
         : Cartoon-style sketches
with gallery
        </li>
        <li>
         <a class="reference external" href="http://zwibbler.com/">
          Zwibbler
         </a>
         : Vector graphics
        </li>
        <li>
         <a class="reference external" href="http://figurepool.com/editor">
          FigurePool
         </a>
         : SVG diagramming
        </li>
        <li>
         <a class="reference external" href="http://svg-edit.googlecode.com/svn/trunk/editor/svg-editor.html">
          SVG-edit
         </a>
         :
Open source SVG non-destructive drawing
        </li>
        <li>
         <a class="reference external" href="http://editor.method.ac/">
          Method Draw
         </a>
         : SVG non-destructive drawing, based
on SVG-edit
        </li>
        <li>
         <a class="reference external" href="http://www.williammalone.com/articles/create-html5-canvas-javascript-drawing-app/">
          Creating a Drawing App with HTML5 Canvas and Javascript
         </a>
         :
Learn a bit about how to do these things yourself!
        </li>
       </ul>
