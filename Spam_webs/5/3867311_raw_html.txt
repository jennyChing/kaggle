    <div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en">
     <p>
      Freehackers once had some pages about gentoo flags: how to set them, which one to choose, and so on. By the time, this information was not provided by gentoo, although lot of people were googling for this.
      <a class="external text" href="http://www.freehackers.org/thomas/free-software/" rel="nofollow">
       I
      </a>
      was one of them.
     </p>
     <p>
      Nowadays, this kind of information, up-to-date, accurate, and maintained, can be found on the official site
     </p>
     <ul>
      <li>
       <a class="external free" href="https://wiki.gentoo.org/wiki/GCC_optimization" rel="nofollow">
        https://wiki.gentoo.org/wiki/GCC_optimization
       </a>
      </li>
     </ul>
     <p>
      Some more information is provided by gentoo-wiki.com, which is an unofficial site with interesting stuff about gentoo:
     </p>
     <ul>
      <li>
       <a class="external free" href="https://wiki.gentoo.org/wiki/Safe_Cflags" rel="nofollow">
        https://wiki.gentoo.org/wiki/Safe_Cflags
       </a>
      </li>
      <li>
       <a class="external free" href="https://wiki.gentoo.org/wiki/CFLAGS" rel="nofollow">
        https://wiki.gentoo.org/wiki/CFLAGS
       </a>
      </li>
     </ul>
