                     <p class="MsoNormal" style="margin-left: 0.25in; line-height: 150%;">
                      <b>
                       <span style="line-height: 150%;font-family:Calibri;font-size:10;">
                        Change your Limited account to Administrator account on XP
                        <o:p>
                        </o:p>
                       </span>
                      </b>
                     </p>
                     <p class="MsoNormal" style="margin-left: 0.25in; line-height: 150%;">
                      <span style="font-weight: bold;">
                       Step One:  Identify  Who is Administrator
                      </span>
                      <span style="line-height: 150%;font-family:Calibri;font-size:10;">
                       <o:p>
                       </o:p>
                      </span>
                     </p>
                     <ol start="1" type="1">
                      <li class="MsoNormal" style="line-height: 150%;">
                       <span style="line-height: 150%;font-family:Calibri;font-size:10;">
                        Go to Start      &gt; Run...
                        <o:p>
                        </o:p>
                       </span>
                      </li>
                      <li class="MsoNormal" style="line-height: 150%;">
                       <span style="line-height: 150%;font-family:Calibri;font-size:10;">
                        Type
                        <span style="">
                        </span>
                        <b>
                         cmd
                        </b>
                        and then press Enter.
                        <o:p>
                        </o:p>
                       </span>
                      </li>
                      <li class="MsoNormal" style="line-height: 150%;">
                       <span style="line-height: 150%;font-family:Calibri;font-size:10;">
                        Type
                       </span>
                       <span style="">
                       </span>
                       net user
                       <span style="line-height: 150%;font-family:Calibri;font-size:10;">
                        to identify the account name of the      local administrator.
                        <o:p>
                        </o:p>
                       </span>
                      </li>
                      <li class="MsoNormal" style="line-height: 150%;">
                       <span style="line-height: 150%;font-family:Calibri;font-size:10;">
                        Once you      have identified the admin account (example here :
                        <b>
                         admin
                        </b>
                        ), type in
                        <o:p>
                        </o:p>
                       </span>
                      </li>
                     </ol>
                     <p class="MsoNormal" style="margin-left: 0.25in; line-height: 150%; font-weight: bold;">
                      Step Two:
                      <span style="">
                      </span>
                      Change Administrator Password
                     </p>
                     <ol start="1" type="1">
                      <li class="MsoNormal" style="line-height: 150%;">
                       <span style="line-height: 150%;font-family:Calibri;font-size:10;">
                        Type in      command
                        <b>
                         <span style="">
                         </span>
                         net user
                         <span style="">
                         </span>
                         {admin-account-name}
                         <span style="">
                         </span>
                         <span style="">
                         </span>
                         *
                        </b>
                        and press Enter.
                        <span style="">
                        </span>
                        ,replace admin      account name with the name of the admin account
                        <o:p>
                        </o:p>
                       </span>
                      </li>
                      <li class="MsoNormal" style="line-height: 150%;">
                       <span style="line-height: 150%;font-family:Calibri;font-size:10;">
                        When it says      enter a password enter whatever password u want to use then press      enter(you dont see the password being typed but it IS being changed)
                        <o:p>
                        </o:p>
                       </span>
                      </li>
                      <li class="MsoNormal" style="line-height: 150%;">
                       <span style="line-height: 150%;font-family:Calibri;font-size:10;">
                        type your      password again then press enter
                        <o:p>
                        </o:p>
                       </span>
                      </li>
                      <li class="MsoNormal" style="line-height: 150%;">
                       <span style="line-height: 150%;font-family:Calibri;font-size:10;">
                        Type in      "exit" and press Enter.
                        <o:p>
                        </o:p>
                       </span>
                      </li>
                     </ol>
                     <p class="MsoNormal" style="margin-left: 0.25in; line-height: 150%;">
                      <span style="font-weight: bold;">
                       Step Three:  Login as Administrator
                      </span>
                      <span style="line-height: 150%;font-family:Calibri;font-size:10;">
                       <o:p>
                       </o:p>
                      </span>
                     </p>
                     <ol start="1" type="1">
                      <li class="MsoNormal" style="line-height: 150%;">
                       <span style="line-height: 150%;font-family:Calibri;font-size:10;">
                        Log out, and      then login into the admin account with your password.
                        <o:p>
                        </o:p>
                       </span>
                      </li>
                      <li class="MsoNormal" style="line-height: 150%;">
                       <span style="line-height: 150%;font-family:Calibri;font-size:10;">
                        Open User      Accounts from Control Panel.
                        <o:p>
                        </o:p>
                       </span>
                      </li>
                      <li class="MsoNormal" style="line-height: 150%;">
                       <span style="line-height: 150%;font-family:Calibri;font-size:10;">
                        Change your      account type as an administrator , or add a new User as Administrator , See      if your limited account is in the list (which contains the admin account      name, etc) and if it isnt, add it (Click Add...).
                        <o:p>
                        </o:p>
                       </span>
                      </li>
                      <li class="MsoNormal" style="line-height: 150%;">
                       <span style="line-height: 150%;font-family:Calibri;font-size:10;">
                        you can      change numerous file settings. Isnt this sound Bad , dont try and make      any systems get hacked..
                        <o:p>
                        </o:p>
                       </span>
                      </li>
                      <li class="MsoNormal" style="line-height: 150%;">
