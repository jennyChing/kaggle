                <p>
                </p>
                <h2 style="text-align: justify;">
                 <strong>
                  Brown Base Color  Tri-Color Representations
                 </strong>
                </h2>
                <p style="text-align: justify;">
                 Brown based color  Tri-Color Representations is a package of color palettes using brown as basic color. The color palette files are given in Adobe Swatch Exchange format (.ase) which can be loaded in every Adobe software such as (Photoshop, illustrator, in Design etc.). The package also includes a catalogue in PDF format of the tri-color representations in order to see and choose the desirable color palette before you load it.
                </p>
                <p style="text-align: justify;">
                 Every Tri-color representation has its own color palette and both of them are titled with the same name. The size of each square represents the amount of color that we can use when coloring surfaces. This amount of color isnt an absolute percentage, which means that we can choose the surfaces (drawings, designs, walls, clothes, product parts etc.) in order to fit better to our needs. Every color inside the tri-color matching representation has at least 10% contrast value from the other colors.
                </p>
                <p style="text-align: justify;">
                 Tri-Color Representations is a series of color palette packages that can be useful to graphic designers, web designers, architects, photographers, decorators, fashion designers, painters, digital artists, animators and everyone who has the need of color matching.
                </p>
