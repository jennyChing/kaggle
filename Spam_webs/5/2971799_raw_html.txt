           By
           <span rel="author">
            Adrian Tierney-Jones
           </span>
          </p>

          2:58PM GMT 03 Jan 2012

          <img alt="Comments" src="/template/ver1-0/i/share/comments.gif"/>
          <a dsqid="8989904" href="http://www.telegraph.co.uk/foodanddrink/pubs/8989904/Top-ten-drinks-trends-for-2012.html#disqus_thread">

        <div id="mainBodyArea" itemprop="articleBody">
         <div class="firstPar">
          <p>
           <strong>
            1.
           </strong>
           Real ale fans might blanch at the thought (and taste) of keg beer, but craft keg is another beast altogether featuring boldly flavoured lagers, IPAs and stouts from a new wave of technically savvy breweries such as Camden, BrewDog and Lovibonds.
          </p>
         </div>
         <div class="secondPar">
          <p>
           <strong>
            2.
           </strong>
           Wine cocktails are nothing new even if Sangria is these days a bit Benidorm  but ambitious mixologists looking for new frontiers of flavour and with an eye on US bar trends would be mad not to carry out such experiments as bourbon and Rioja or vodka and Chardonnay.
          </p>
         </div>
         <div class="thirdPar">
          <p>
           <strong>
            3.
           </strong>
           Here today, gone tomorrow is the philosophy behind the pop-up restaurant so expect a wave of pop-up bars  North Bar in Leeds leads the way already with Mr Frothy, an old ice cream van for hire whose chimes usher in the promise of a pint rather than a choc ice.
          </p>
         </div>
         <div class="fourthPar">
          <p>
           <strong>
            4.
           </strong>
           The governments increased tax on strong beer is a misguided attempt to deal with binge drinking, but the accompanying introduction of lower taxes on beers below 2.8% has led to flavoursome and weaker real ales from the likes of Adnams, Greene King and Fullers.
          </p>
         </div>
         <div class="fifthPar">
          <p>
           <strong>
            5.
           </strong>
           A pint of Martini might shake and stir the purists, but be ready to hear these words more often when the American fondness for ready-made cocktails served straight from the barrel lands in London.
          </p>
         </div>

            6.
           </strong>
           Even though cider topped with handfuls of ice cubes is not such the cool thing it was several years back, expect it to continue its upmarket growth especially with an emphasis on single varietal and single orchard ciders.
          </p>
          <p>
           <strong>
            7.
           </strong>
           The sweet and fizzy Moscato dAsti will keep defying the wine snobs as drinkers seek something lighter that actually has some taste and also goes very well with desserts such as lemon tart.
          </p>
          <p>
           <strong>
            8.
           </strong>
           As hard times continue, home brew remains a popular pastime, but its not just beer several ground-breaking books such as Home Brew (Pavilion) extend the idea of DIY tipple-making to wines, spirits and ciders.
          </p>
          <p>
           <strong>
            9
           </strong>
           . Even though the rate is slowing, too many pubs are still closing  however, so-called craft beer bars such as Cask in London and Port Street Beer House in Manchester will continue to open and thrive, as their young hip clientele seek real ale and craft keg beers with the emphasis on artisanal producers
          </p>
          <p>
           <strong>
            10.
           </strong>
           Expect gin to keep shaking off its mothers ruin image, especially as a new wave of micro distilleries such as London-based Sipsmith continue to push boundaries and add interesting botanicals.

                        if(isScrolledIntoView("#shareBottom") || isScrolledIntoView("#newGigyaShareBottom")) {
                            if(settings.autoOpen) {
                                openArticleContainer();
                            }
                        }

                        if($(window).scrollTop() + $(window).height() == $(document).height()) {
                            if(settings.autoOpen) {
                                openArticleContainer();
                            }
                        }
                    });
                }
            };

            var display = function () {
                $(settings.widgetContainer).fadeIn(2000, function () {
                    //animation complete
                    _gaq.push(['_trackEvent', 'wayfarer_read_next', 'display_closed', 'version_1']);
                });
            };

            var bindUIActions = function () {
                $(settings.openButton).mouseenter(function (event) {
                    event.preventDefault();
                    _gaq.push(['_trackEvent', 'wayfarer_read_next', 'hover_closed', 'version_1']);
                    openArticleContainer();
                });
                $(settings.widgetContainer).mouseleave(function (event) {
                    event.preventDefault();
                    closeArticleContainer();
                });
                $(settings.closeButton).click(function (event) {
                    event.preventDefault();
                    _gaq.push(['_trackEvent', 'wayfarer_read_next', 'click_close', 'version_1']);
                    updateCookie(settings.articleId, settings.articleListToSkip);
                    closeArticleContainer();
                    settings.autoOpen = false;
                });
            };

            var openArticleContainer = function () {
                if (!settings.panelOpen) {
                	$("#readNextArticleContainer").show();//show/hide because height is calculated incorrectly for elements with display:none
                    $("#readNextArticleContainer").hide();
                    $(settings.articleContainer).toggle('slide', {direction: 'left'}, function () {
                        //animation complete
                        _gaq.push(['_trackEvent', 'wayfarer_read_next', 'display_open', 'version_1']);
                    });
                    settings.panelOpen = !settings.panelOpen;
                    settings.autoOpen = false; //set auto open to false once user has viewed this once
                }
                return false;
            };

            var closeArticleContainer = function () {
                if (settings.panelOpen) {
                    $(settings.articleContainer).toggle('slide', {direction: 'left'}, function () {
                        //animation complete
                    });
                    settings.panelOpen = !settings.panelOpen;
                }
                return false;
            };


            var isScrolledIntoView = function(elem) {
                if($(elem).length) { //check if element exists
                    var docViewTop = $(window).scrollTop();
                    var docViewBottom = docViewTop + $(window).height();

                    var elemTop = $(elem).offset().top;
                    var elemBottom = elemTop + $(elem).height();

                    return ((elemBottom &lt;= docViewBottom) &amp;&amp; (elemTop &gt;= docViewTop));
                } else {
                    return false;
                }
            };

            var unique = function(array) {
                return array.filter(function (el, index, arr) {
                    return index == arr.indexOf(el);
                });
            }

            var createCookieBasedOnArticleList = function(currentArticleId, articleList, articleListToSkip) {
                articleList.some(function (article) {
                    if (article.id == currentArticleId) {
                        articleListToSkip[0] = currentArticleId;
                        $.cookie(settings.readNextCookieName, escape(articleListToSkip.join(',')), {path: '/', expires: 1, domain: 'telegraph.co.uk'});
                        return true;
                    }
                });
            }

            var updateCookieBasedOnArticleList = function(currentArticleId, articleList, articleListToSkip) {
                articleList.some(function (article) {
                    if (article.id == currentArticleId) {
                        updateCookie(currentArticleId, articleListToSkip);
                        return true;
                    }
                });
            }

            var updateCookie = function(articleIdToSkip, articleListToSkip) {
                articleListToSkip.push(articleIdToSkip);
                articleListToSkip = unique(articleListToSkip);
                $.cookie(settings.readNextCookieName, escape(articleListToSkip.join(',')), {path: '/', expires: 1, domain: 'telegraph.co.uk'});
            }

            var chooseNextArticle = function (currentArticleId, articleList, articleListToSkip) {
                articleList.some(function (article) {
                    if ((article.id != currentArticleId) &amp;&amp; ($.inArray(article.id, articleListToSkip) === -1)) {

                        //set article
                        settings.articleTitle = article.title;
                        settings.articleUrl = article.url;
                        settings.articleId = article.id;
						settings.articleImage = article.imgUrl;
                        return true;
                    }
                });
            }

            //public API
            return {
                settings: settings,
                init: init,
                display: display,
                openArticleContainer: openArticleContainer,
                closeArticleContainer: closeArticleContainer
            }
        })();

    };
  </script>
