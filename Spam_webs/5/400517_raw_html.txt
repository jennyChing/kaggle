       <h1 itemprop="headline name">
        How to be a more popular driver
       </h1>
       <h2 itemprop="alternativeHeadline description">
        Avoid enraging fellow motorists with your bad driving habits. Become a happy, 
  calm and, most importantly, safe driver with these handy hints and tips
       </h2>
       <style media="print">
        .newGigyaShare { display: none !important; }

            <div class="artImageExtras">
             <div class="ingCaptionCredit">

         <div>
          <p class="bylineBody">
           <span itemprop="author">
            By Tim Gibson
           </span>
          </p>
         </div>
         <p class="publishedDate">
          5:04PM BST 25 Oct 2013
         </p>
         <meta content="2013-10-25" itemprop="datePublished"/>
         <div class="cl">

        <div id="mainBodyArea" itemprop="articleBody">
         <div class="firstPar">
          <p>
           <strong>
            <img align="left" alttext="Step 1" caption="Step 1" declared-caption="" height="69" name="Number 1" photographer="" refid="2695675" src="http://i.telegraph.co.uk/multimedia/archive/02695/Number_1_2695675a.png" width="37"/>
            Read the signs
           </strong>
          </p>
         </div>
         <div class="secondPar">
          <p>
           Theres nothing more annoying to other road users than motorists who dont adhere to road signs. Whether youre driving illegitimately down a one-way street or occupying a lane intended for other traffic, youll quickly attract the ire of others if you dont know whats what. So read the Highway Code to mug up on traffic signs, and keep alert for them as you go about your business. Everyone will thank you for it.
          </p>
         </div>
         <div class="thirdPar">
          <p>
           <strong>
            <img align="left" alttext="Step 2" caption="Step 2" declared-caption="" height="67" name="Number 2" photographer="" refid="2695676" src="http://i.telegraph.co.uk/multimedia/archive/02695/Number_2_2695676a.png" width="52"/>
            Dither ye not
           </strong>
          </p>
         </div>
         <div class="fourthPar">
          <p>
           If you dont know where youre going, park up and look at a map. Dont slow to a crawl while you try to identify street names, or spot local landmarks that will help you find your destination. This isnt just the preserve of city drivers  rural lanes are full of people randomly stopping and starting while they look for a hidden entrance or concealed country pub. And its very annoying for the people behind. So dont do it. Just dont.
          </p>
         </div>
         <div class="fifthPar">
          <p>
           <strong>
            <img align="left" alttext="Step 3" caption="Step 3" declared-caption="" height="68" name="Number 3" photographer="" refid="2695680" src="http://i.telegraph.co.uk/multimedia/archive/02695/Number_3_2695680a.png" width="49"/>
           </strong>
           <strong>
            Keep smiling
           </strong>
          </p>
         </div>
         <div class="body">
          <p>
           Try this next time youre behind the wheel: smile at your fellow motorists, giving a cheery wave whenever someone lets you pass. Youll be amazed at the positive reaction it provokes, enabling you to bask in the glow of others adoration. This will make you a calmer driver, much more aware of whats going on around you, and much less likely to dither while you work out which way to go. And we all know how beneficial that can be from a popularity point of view.
          </p>
         </div>

                        if($(window).scrollTop() + $(window).height() == $(document).height()) {
                            if(settings.autoOpen) {
                                openArticleContainer();
                            }
                        }
                    });
                }
            };

            var display = function () {
                $(settings.widgetContainer).fadeIn(2000, function () {
                    //animation complete
                    _gaq.push(['_trackEvent', 'wayfarer_read_next', 'display_closed', 'version_1']);
                });
            };

            var bindUIActions = function () {
                $(settings.openButton).mouseenter(function (event) {
                    event.preventDefault();
                    _gaq.push(['_trackEvent', 'wayfarer_read_next', 'hover_closed', 'version_1']);
                    openArticleContainer();
                });
                $(settings.widgetContainer).mouseleave(function (event) {
                    event.preventDefault();
                    closeArticleContainer();
                });
                $(settings.closeButton).click(function (event) {
                    event.preventDefault();
                    _gaq.push(['_trackEvent', 'wayfarer_read_next', 'click_close', 'version_1']);
                    updateCookie(settings.articleId, settings.articleListToSkip);
                    closeArticleContainer();
                    settings.autoOpen = false;
                });
            };

            var openArticleContainer = function () {
                if (!settings.panelOpen) {
                	$("#readNextArticleContainer").show();//show/hide because height is calculated incorrectly for elements with display:none
                    $("#readNextArticleContainer").hide();
                    $(settings.articleContainer).toggle('slide', {direction: 'left'}, function () {
                        //animation complete
                        _gaq.push(['_trackEvent', 'wayfarer_read_next', 'display_open', 'version_1']);
                    });
                    settings.panelOpen = !settings.panelOpen;
                    settings.autoOpen = false; //set auto open to false once user has viewed this once
                }
                return false;
            };

            var closeArticleContainer = function () {
                if (settings.panelOpen) {
                    $(settings.articleContainer).toggle('slide', {direction: 'left'}, function () {
                        //animation complete
                    });
                    settings.panelOpen = !settings.panelOpen;
                }
                return false;
            };


            var isScrolledIntoView = function(elem) {
                if($(elem).length) { //check if element exists
                    var docViewTop = $(window).scrollTop();
                    var docViewBottom = docViewTop + $(window).height();

                    var elemTop = $(elem).offset().top;
                    var elemBottom = elemTop + $(elem).height();

                    return ((elemBottom &lt;= docViewBottom) &amp;&amp; (elemTop &gt;= docViewTop));
                } else {
                    return false;
                }
            };

            var unique = function(array) {
                return array.filter(function (el, index, arr) {
                    return index == arr.indexOf(el);
                });
            }

            var createCookieBasedOnArticleList = function(currentArticleId, articleList, articleListToSkip) {
                articleList.some(function (article) {
                    if (article.id == currentArticleId) {
                        articleListToSkip[0] = currentArticleId;
                        $.cookie(settings.readNextCookieName, escape(articleListToSkip.join(',')), {path: '/', expires: 1, domain: 'telegraph.co.uk'});
                        return true;
                    }
                });
            }

            var updateCookieBasedOnArticleList = function(currentArticleId, articleList, articleListToSkip) {
                articleList.some(function (article) {
                    if (article.id == currentArticleId) {
                        updateCookie(currentArticleId, articleListToSkip);
                        return true;
                    }
                });
            }

            var updateCookie = function(articleIdToSkip, articleListToSkip) {
                articleListToSkip.push(articleIdToSkip);
                articleListToSkip = unique(articleListToSkip);
                $.cookie(settings.readNextCookieName, escape(articleListToSkip.join(',')), {path: '/', expires: 1, domain: 'telegraph.co.uk'});
            }

            var chooseNextArticle = function (currentArticleId, articleList, articleListToSkip) {
                articleList.some(function (article) {
                    if ((article.id != currentArticleId) &amp;&amp; ($.inArray(article.id, articleListToSkip) === -1)) {

                        //set article
                        settings.articleTitle = article.title;
                        settings.articleUrl = article.url;
                        settings.articleId = article.id;
						settings.articleImage = article.imgUrl;
                        return true;
                    }
                });
            }

            //public API
            return {
                settings: settings,
                init: init,
                display: display,
                openArticleContainer: openArticleContainer,
                closeArticleContainer: closeArticleContainer
            }
        })();

    };
  </script>
