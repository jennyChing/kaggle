         <p>
          <a href="http://www.funkyspacemonkey.com/change-default-folder-grid-layout-ios-7/folderenhancer-folder-grid-layout-1-fsmdotcom" rel="attachment wp-att-44116">
           <img alt="folderenhancer-folder-grid-layout-1-FSMdotCOM" class="aligncenter size-full wp-image-44116" height="2000" src="http://cdn.funkyspacemonkey.com/wp-content/uploads/2014/02/folderenhancer-folder-grid-layout-1-FSMdotCOM.jpg" width="2000"/>
          </a>
         </p>
         <p>
          Notice anything different in the screenshots above? The folder grid layout its not set to 33 anymore. The folder preview layout its now set to 11 and the folder icon layout its set to 35. If youre looking to change iOS 7s default folder gird layout, theres an easy way to do it using
          <a href="http://www.funkyspacemonkey.com/folderenhancer-updated-ios-7-support-adds-new-functionality-ios-7s-folders" target="_blank">
           FolderEnhancer
          </a>
          .
         </p>
         <p>
          Jump over the break to learn how to change the default iOS 7s folder grid layout
         </p>
         <p>
          <span id="more-44115">
          </span>
         </p>
         <p>
          1. Load Cydia and install FolderEnhancer ( $0.99 via BigBoss repo )
         </p>
         <p>
          2. Once installed, load Settings.app and scroll down until you see FolderEnhancers dedicated preference pane.
         </p>
         <p>
          <a href="http://www.funkyspacemonkey.com/change-default-folder-grid-layout-ios-7/folderenhancer-folder-grid-layout-2-fsmdotcom" rel="attachment wp-att-44118">
           <img alt="folderenhancer-folder-grid-layout-2-FSMdotCOM" class="aligncenter size-full wp-image-44118" height="500" src="http://cdn.funkyspacemonkey.com/wp-content/uploads/2014/02/folderenhancer-folder-grid-layout-2-FSMdotCOM.jpg" width="295"/>
          </a>
          3. Inside the FolderEnhancer preference pane, scroll down until you see Folder Preview Layout and Icon Layout. From there you can customize the folder grid in any way you want. For Folder Preview Layout you can set up to 5 columns and rows and for the Icon Layout you can set 3 or 4 columns and 3 to 5 rows.
         </p>
         <p>
          <a href="http://www.funkyspacemonkey.com/change-default-folder-grid-layout-ios-7/folderenhancer-folder-grid-layout-3-fsmdotcom" rel="attachment wp-att-44119">

          </a>
