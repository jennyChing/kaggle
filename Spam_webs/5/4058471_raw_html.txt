     <a class="no-link" href="http://www.pathikshah.com/blog/install-ubuntu-linux-on-samsung-galaxy-tab/" rel="bookmark" title="Permalink to How to Install Ubuntu Linux on Samsung Galaxy Tab">
      How to Install Ubuntu Linux on Samsung Galaxy Tab
     </a>
    </h2>
    <div class="entry-content">
     <p>
      <strong>
       How to Install Ubuntu Linux on Samsung Galaxy Tab
      </strong>
     </p>
     <p>
      If you own a Samsung Galaxy Tab, heres some good news for you  you can now install Ubuntu on your Galaxy Tab. This tutorial will explain how you can install Ubuntu Linux on your Samsung Galaxy Tab.
     </p>
     <p>
      What you need:
     </p>
     <p>
      1. Busybox
      <br/>
      2. Superuser
      <br/>
      3. Android SDK
      <br/>
      4. AndroidVNC
      <br/>
      5. Terminal Emulator
     </p>
     <p>
      <strong>
       <a href="http://db.tt/YnJZuf1">
        Download Ubuntu for Samsung Galaxy Tab
       </a>
      </strong>
     </p>
     <p>
      <strong>
       <a href="http://db.tt/PLZTmH9">
        Alternate Scripts for Ubuntu
       </a>
      </strong>
      (If the default ones dont work)
      <strong>
      </strong>
     </p>
     <p>
      <strong>
       Steps to Install Ubuntu on Galaxy Tab
      </strong>
     </p>
     <p>
      1. Download the Ubuntu for Galaxy Tab ZIP and extract it on your computer. Copy all the files inside like bootlinux, fsrw, mountonly, ubuntu.img, ubuntu.sh and unionfs to a folder on your SD card named Ubuntu.
     </p>
     <p>
      2. Open the command prompt, CD to
      <strong>
       C:\SDK\tools
      </strong>
       where C:\SDK is the folder where the SDK is installed.
     </p>
     <p>
      Type these commands
     </p>
     <p>
      <strong>
       adb shell
       <br/>
       su
       <br/>
       sh ./ubuntu.sh
      </strong>
     </p>
     <p>
      <strong>
       <a href="http://www.pathikshah.com/blog/wp-content/uploads/2010/11/ubuntusetup1.jpg">
        <img alt="" class="aligncenter size-full wp-image-1480" height="107" src="http://www.pathikshah.com/blog/wp-content/uploads/2010/11/ubuntusetup1.jpg" title="Ubuntu on Galaxy Tab" width="512"/>
       </a>
      </strong>
     </p>
     <p>
      3. After this, just run this command.
     </p>
     <p>
      <strong>
       bootlinux
      </strong>
     </p>
     <p>
      If you get the root@localhost prompt, you have installed Ubuntu on your Galaxy Tab.
     </p>
     <p>
      4. Now, if you have
      <strong>
       AndroidVNC
      </strong>
      set up, type these commands
     </p>
     <p>
      <strong>
       rm -rf /tmp/.X*
       <br/>
       export USER=root
       <br/>
       vncserver -geometry 1280800
      </strong>
     </p>
     <p>
      <span id="more-1478">
      </span>
     </p>
     <p>
      <script type="text/javascript">
       // &lt;![CDATA[
 google_ad_client = "pub-3213448740337726"; /* patlargerect */ google_ad_slot = "9053374845"; google_ad_width = 336; google_ad_height = 280;
// ]]&gt;
      </script>
      <br/>
      <script src="http://pagead2.googlesyndication.com/pagead/show_ads.js" type="text/javascript">
      </script>
     </p>
     <p>
      <strong>
       <a href="http://www.pathikshah.com/blog/wp-content/uploads/2010/11/ubuntusetup4.jpg">
        <img alt="" class="aligncenter size-full wp-image-1481" height="376" src="http://www.pathikshah.com/blog/wp-content/uploads/2010/11/ubuntusetup4.jpg" title="Ubuntu on Galaxy Tab" width="512"/>
       </a>
      </strong>
     </p>
     <p>
      5. In
      <strong>
       Android VNC
      </strong>
      , your login details will be
     </p>
     <p>
      <strong>
       Nickname: "root"
       <br/>
       Password: "ubuntu"
       <br/>
       Address: "localhost"
       <br/>
       Port: "5901"
      </strong>
     </p>
     <p>
      <strong>
       <a href="http://www.pathikshah.com/blog/wp-content/uploads/2010/11/ubuntusetup5.jpg">
        <img alt="" class="aligncenter size-full wp-image-1482" height="307" src="http://www.pathikshah.com/blog/wp-content/uploads/2010/11/ubuntusetup5.jpg" title="Ubuntu on Galaxy Tab" width="512"/>
       </a>
      </strong>
     </p>
     <p>
      Your Samsung Galaxy Tab should now be running Ubuntu Linux. Congratulations.
     </p>
     <p>
      Now, to run Ubuntu again, you don't need to install it every time.
     </p>
     <p>
      Just open the
      <strong>
       Terminal Emulator
      </strong>
      on your Galaxy Tab. Type these commands again to start Ubuntu.
     </p>
     <p>
      <strong>
       su
       <br/>
       bootlinux
      </strong>
     </p>
     <p>
      <strong>
       rm -rf /tmp/.X*
       <br/>
       export USER=root
       <br/>
       vncserver -geometry 1280x800
      </strong>
     </p>
     <p>
      Start VNC and enter your login details.
     </p>

       <strong>
        XDA
       </strong>
      </a>
     </p>
