       <h1 itemprop="headline name">
        In defence of snowboarders
       </h1>
       <h2 itemprop="alternativeHeadline description">
        As the Winter Olympics continues in Sochi, a snowboarder speaks out about the 
  prejudice of "ski snobs"
       </h2>

          <p class="bylineBody">
           By
           <span rel="author">
            Tom Metcalf
           </span>
          </p>
         </div>
         <p class="publishedDate">
          1:47PM GMT 12 Feb 2014
         </p>
         <p class="comments">
          <img alt="Comments" src="/template/ver1-0/i/share/comments.gif"/>
          <a dsqid="10631655" href="http://www.telegraph.co.uk/travel/snowandski/features/10631655/In-defence-of-snowboarders.html#disqus_thread">
           Comments

         <div class="cl">

        <div id="mainBodyArea" itemprop="articleBody">
         <div class="firstPar">
          <p>
           Im disappointed to report that there still exists among certain skiers a belief that snowboarders are in some way inferior and unworthy of sharing the mountain.
          </p>
         </div>
         <div class="secondPar">
          <p>
           These we were here first types tend to view all boarders as unruly, scruffy yobs who charge down the slopes like maniacs, putting their lives (and everyone elses) at risk.
          </p>
         </div>
         <div class="thirdPar">
          <p>
           Let me put my cards on the table  Im a snowboarder. Always have been, always will be. From the moment I strapped on a board for the first time and felt the thrill of cruising down a slope, admittedly after several days of catching edges and face-planting, I knew this was the sport for me.
          </p>
         </div>
         <div class="fourthPar">
          <p>
           But Im not ideologically opposed to skiing. I even gave it a go recently at the Ski and Snowboard Show in London. But, just like most skiers, Ive never been seriously inclined to switch and leave the comfort of my board to learn a new discipline from scratch.
          </p>
         </div>
         <div class="fifthPar">
          <p>
           Im not here to start one of those futile debates about which of the two sports is better; such arguments are tiresome and unnecessary. Each one has its merits, and people are always going to have their preference. But I am here to stick up for snowboarders in the face of skier snobbery.
          </p>
         </div>

         <div class="body">
          <p>
           Over the years Ive encountered many people who, upon discovering I was a snowboarder, tutted disapprovingly or made a derisory comment about ruining the mountain for skiers. It happened around a chalet dinner table only last year. And there are even resorts, like Deer Valley and Alta in Utah, which still prohibit snowboarding. Come on, get over it!
          </p>
          <p>
           This idea that we ruin the mountain is complete nonsense. Yes, some boarders behave irresponsibly, but so do some skiers. Its about the person, not what theyre riding. Ive seen plenty of skiers careering down the slope, completely out of control. A reckless boarder is no worse than a reckless skier  and unfortunately there are too many of both.
          </p>
          <p>
           I urge ski snobs to tune in to the snowboarding events at the Winter Olympics [snowboard cross is on Sunday, February 16]. Watching a world-class snowboarder is just as exhilarating as watching a top skier.
          </p>
          <p>
           And at the end of the day, skiers and boarders all share the same goal  to enjoy the mountain to the full.
          </p>

                $(settings.widgetContainer).fadeIn(2000, function () {
                    //animation complete
                    _gaq.push(['_trackEvent', 'wayfarer_read_next', 'display_closed', 'version_1']);
                });
            };

            var bindUIActions = function () {
                $(settings.openButton).mouseenter(function (event) {
                    event.preventDefault();
                    _gaq.push(['_trackEvent', 'wayfarer_read_next', 'hover_closed', 'version_1']);
                    openArticleContainer();
                });
                $(settings.widgetContainer).mouseleave(function (event) {
                    event.preventDefault();
                    closeArticleContainer();
                });
                $(settings.closeButton).click(function (event) {
                    event.preventDefault();
                    _gaq.push(['_trackEvent', 'wayfarer_read_next', 'click_close', 'version_1']);
                    updateCookie(settings.articleId, settings.articleListToSkip);
                    closeArticleContainer();
                    settings.autoOpen = false;
                });
            };

            var openArticleContainer = function () {
                if (!settings.panelOpen) {
                	$("#readNextArticleContainer").show();//show/hide because height is calculated incorrectly for elements with display:none
                    $("#readNextArticleContainer").hide();
                    $(settings.articleContainer).toggle('slide', {direction: 'left'}, function () {
                        //animation complete
                        _gaq.push(['_trackEvent', 'wayfarer_read_next', 'display_open', 'version_1']);
                    });
                    settings.panelOpen = !settings.panelOpen;
                    settings.autoOpen = false; //set auto open to false once user has viewed this once
                }
                return false;
            };

            var closeArticleContainer = function () {
                if (settings.panelOpen) {
                    $(settings.articleContainer).toggle('slide', {direction: 'left'}, function () {
                        //animation complete
                    });
                    settings.panelOpen = !settings.panelOpen;
                }
                return false;
            };


            var isScrolledIntoView = function(elem) {
                if($(elem).length) { //check if element exists
                    var docViewTop = $(window).scrollTop();
                    var docViewBottom = docViewTop + $(window).height();

                    var elemTop = $(elem).offset().top;
                    var elemBottom = elemTop + $(elem).height();

                    return ((elemBottom &lt;= docViewBottom) &amp;&amp; (elemTop &gt;= docViewTop));
                } else {
                    return false;
                }
            };

            var unique = function(array) {
                return array.filter(function (el, index, arr) {
                    return index == arr.indexOf(el);
                });
            }

            var createCookieBasedOnArticleList = function(currentArticleId, articleList, articleListToSkip) {
                articleList.some(function (article) {
                    if (article.id == currentArticleId) {
                        articleListToSkip[0] = currentArticleId;
                        $.cookie(settings.readNextCookieName, escape(articleListToSkip.join(',')), {path: '/', expires: 1, domain: 'telegraph.co.uk'});
                        return true;
                    }
                });
            }

            var updateCookieBasedOnArticleList = function(currentArticleId, articleList, articleListToSkip) {
                articleList.some(function (article) {
                    if (article.id == currentArticleId) {
                        updateCookie(currentArticleId, articleListToSkip);
                        return true;
                    }
                });
            }

            var updateCookie = function(articleIdToSkip, articleListToSkip) {
                articleListToSkip.push(articleIdToSkip);
                articleListToSkip = unique(articleListToSkip);
                $.cookie(settings.readNextCookieName, escape(articleListToSkip.join(',')), {path: '/', expires: 1, domain: 'telegraph.co.uk'});
            }

            var chooseNextArticle = function (currentArticleId, articleList, articleListToSkip) {
                articleList.some(function (article) {
                    if ((article.id != currentArticleId) &amp;&amp; ($.inArray(article.id, articleListToSkip) === -1)) {

                        //set article
                        settings.articleTitle = article.title;
                        settings.articleUrl = article.url;
                        settings.articleId = article.id;
						settings.articleImage = article.imgUrl;
                        return true;
                    }
                });
            }

            //public API
            return {
                settings: settings,
                init: init,
                display: display,
                openArticleContainer: openArticleContainer,
                closeArticleContainer: closeArticleContainer
            }
        })();

    };
