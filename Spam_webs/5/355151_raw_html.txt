      <p>
       jslide is a fresh new jQuery slideshow plugin used to create a basic auto fading/sliding carousel slideshow with minimal html markups.
      </p>
      <h2>
       Features:
      </h2>
      <ul>
       <li>
        Simple and easy to implement.
       </li>
       <li>
        Easing effects. Requires jQuery easing plugin.
       </li>
       <li>
        Multiple instance in one page.
       </li>
       <li>
        Fully custom navigation and pagination.
       </li>
       <li>
        Auto play on page loaded.
       </li>
       <li>
        Endless loop like a carousel.
       </li>
       <li>
        Cross browser. Supports all modern browsers and even IE 7/8.
       </li>
      </ul>
      <h2>
       Basic Usage:
      </h2>
      <p>
       1. Include jQuery library and the jQuery jslide plugin at the bottom so the pages load faster.
      </p>
      <pre class="brush:xml;">
       &lt;script src="//code.jquery.com/jquery-1.11.1.min.js"&gt;&lt;/script&gt;
&lt;script src="jslide.js"&gt;&lt;/script&gt;
      </pre>
      <p>
       2. Include jQuery easing plugin for easing options (Optional).
      </p>
      <pre class="brush:xml;">
       &lt;script src="jquery.easing.min.js"&gt;&lt;/script&gt;
      </pre>
      <p>
       3. The Html to create a slideshow with next/prev navigation.
      </p>
      <pre class="brush:xml;">
       &lt;div id="slides-demo"&gt;
  &lt;ul class="slides"&gt;
    &lt;li class="slide"&gt;&lt;img src="images/1.png"&gt;&lt;/li&gt;
    &lt;li class="slide"&gt;&lt;img src="images/2.png"&gt;&lt;/li&gt;
    &lt;li class="slide"&gt;&lt;img src="images/3.png"&gt;&lt;/li&gt;
  &lt;/ul&gt;
  &lt;nav class="slide-pagination"&gt;
   &lt;a class="prev-slide" href="#"&gt;&laquo;&lt;/a&gt;
   &lt;a class="next-slide" href="#"&gt;&raquo;&lt;/a&gt;
  &lt;/nav&gt;
&lt;/div&gt;
      </pre>
      <p>
       4. Initialize the plugin with one JS call.
       <a href="http://www.jqueryscript.net/tags.php?/Navigation/">
        Navigation
       </a>
       is actually enabled by inserting the Html. The plugin looks for elements inside the outer element (the one you call the plugin on) that match the selectors for a Next and Previous link and binds the corresponding click events to them.
      </p>
      <pre class="brush:js;">
       $('#slides-demo').jslide()
      </pre>
      <p>
       5. Available options.
      </p>
      <pre class="brush:js;">
       // debug mode
debug: false,

// auto play
play: true,

// transition delay
timer: 1000,

// animation speed
speed: 750,

// easing options. 
easing: 'easeOutQuint',

// slide or fade
mode: 'slide',

// enable pagination
pagination: false,

// slideshow width
width: 600,

// slideshow height
height: 350,

// selectors
wrapper: 'ul.slides',
animationWrapper: 'ul.slides-wrapper',
slides: 'li.slide',
activeClass: 'active',
paginationClass: 'slide-pagination',
slidetoClass: 'goto-slide',
next: 'a.next-slide',
prev: 'a.prev-slide'
      </pre>
