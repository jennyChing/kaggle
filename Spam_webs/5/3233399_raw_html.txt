             I play with a few friends, maybe up to 10 or 12 players and I was wondering what I would need for a cheap but sufficient server. I know the basic requirements of my level of server is:
             <br/>
             <br/>
             3.0GHz + CPU, doesn't really matter about Dual Core because Minecraft Server doesn't utilise multi-core, but would be good to have for other game servers.
             <br/>
             <br/>
             2GB + RAM, I'm being safe with this one, giving me space for System Resources and enough for almost 200MB RAM each player.
             <br/>
             <br/>
             I have a selection of 10GB - 40GB drives which I can use to get a 40GB RAID0 set up, doesn't really matter about this anyway, Minecraft files will total on the downside of 200MB after even a few weeks.
             <br/>
             <br/>
             I will use my Phone as the monitor for this computer, after being setup on my current monitor. I won't need to check it much, just put it to sleep, stop the server and so on.
             <br/>
             <br/>
             The motherboard is my biggest concern... at first my choice was immediately to go with a Pentium D 3.0GHz setup, but then I realised that the motherboard I would need for this can be up to $95 minimum - which is more than 3 times the $30 price point of the Pentium D. I'm sure if I look more I can find the motherboard for cheaper second hand.
             <br/>
             <br/>
             I have a spare few cases that the server can go into.
             <br/>
             <br/>
             What else am I missing? I will leave the server tucked away in my lounge room with Ethernet plugged in. I will be using Windows XP Professional, because it uses less resources and I can server some of the older games I have (Hidden and Dangerous 2) a lot better on XP.
             <br/>
             <br/>
             As a side note, I'm just getting some cheap PQI RAM for this, as I'm not overclocking and may eventually even underclock for some temperature drops.
