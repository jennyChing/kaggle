     <h2>
      Windows backup software helps you backup system partition to image
     </h2>
     <h3>
      Why to backup system partition?
     </h3>
     <p>
      Most PC users have ever experienced such kind of problems:
     </p>
     <ul>
      <li>
       Operating system crash
      </li>
      <li>
       System attacked by virus
      </li>
      <li>
       Cannot boot the computer due to other reasons
      </li>
     </ul>
     <p>
      The best way to survive system crash due to various reasons is to backup system to image. Several clicks to restore image to get healthy system back.
     </p>
     <p>
      EaseUS Todo Backup is
      <a href="/products/">
       Windows backup image software
      </a>
      , it can help you image/backup system partition fast and safely. It can also backup system via full backup,
      <a href="/products/features/incremental-backup.htm">
       incremental backup
      </a>
      and
      <a href="/products/features/differential-backup.htm">
       differential backup
      </a>
      .
     </p>
     <h3>
      How to backup system partition to image?
     </h3>
     <p class="Bold">
      You can backup system with one-click system backup.
     </p>
     <p>
      1. Launch EaseUS Todo Backup and then click
      <span class="Bold">
       System backup
      </span>
      .
     </p>
     <p>
      2. Select the backup destination, the software will choose the most suitable location by default.
     </p>
     <p class="AlignCenter">
      <img alt="Select system backup location" src="/images/feature-screen/business/backup-system-state-1.png"/>
     </p>
     <p>
      3. Set schedule to backup files later, daily, weekly, monthly or upon event.
     </p>
     <p>
      <span class="Bold">
       Note
      </span>
      : enter account and password with Administrator privilege to make the schedule take effect.
     </p>
     <p class="AlignCenter">
      <img alt="Set Schedule for System Backup" src="/images/feature-screen/business/backup-system-state-2.png"/>
     </p>
     <p>
      4. Click
      <span class="Bold">
       Backup options
      </span>
      and pop up the following window, set backup compression, password, splitting etc.
     </p>
     <p class="AlignCenter">
      <img alt="Set Backup Options" src="/images/feature-screen/business/backup-system-state-3.png"/>
     </p>
     <p>
      5. Click
      <span class="Bold">
       Proceed
      </span>
      to execute system backup.
     </p>
