     <p id="swipeout-rulesP">
      Swipe as many colored tiles off the board as you can before it fills up.
      <br/>
      <br/>
      Tiles swipe off when you match the color of the tile with the color of a side panel.
      <br/>
      <br/>
      More tiles appear after the "moves" counter reaches zero.
      <br/>
      <br/>
      Use your arrow keys or W, A, S, D character keys to move the tiles.
