     <p>
      <b>
       App Inventor for Android
      </b>
      is an open-source web application originally provided by
      <a href="/wiki/Google" title="Google">
       Google
      </a>
      , and now maintained by the
      <a href="/wiki/Massachusetts_Institute_of_Technology" title="Massachusetts Institute of Technology">
       Massachusetts Institute of Technology
      </a>
      (MIT).
     </p>
     <p>
      It allows newcomers to
      <a href="/wiki/Computer_programming" title="Computer programming">
       computer programming
      </a>
      to create
      <a href="/wiki/Application_software" title="Application software">
       software applications
      </a>
      for the
      <a href="/wiki/Android_(operating_system)" title="Android (operating system)">
       Android
      </a>
      operating system (OS). It uses a graphical interface, very similar to
      <a href="/wiki/Scratch_(programming_language)" title="Scratch (programming language)">
       Scratch
      </a>
      and the
      <a class="mw-redirect" href="/wiki/StarLogo_TNG" title="StarLogo TNG">
       StarLogo TNG
      </a>
      <a href="/wiki/User_interface" title="User interface">
       user interface
      </a>
      , which allows users to
      <a class="mw-redirect" href="/wiki/Drag-and-drop" title="Drag-and-drop">
       drag-and-drop
      </a>
      visual objects to create an application that can run on Android devices. In creating App Inventor, Google drew upon significant prior research in educational computing, as well as work done within Google on online development environments.
      <sup class="reference" id="cite_ref-MIT_roots_1-0">
       <a href="#cite_note-MIT_roots-1">
        <span>
         [
        </span>
        1
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <p>
      App Inventor and the projects on which it is based are informed by
      <a class="mw-redirect" href="/wiki/Constructionist_learning" title="Constructionist learning">
       constructionist learning
      </a>
      theories, which emphasizes that programming can be a vehicle for engaging powerful ideas through active learning. As such, it is part of an ongoing movement in computers and education that began with the work of
      <a href="/wiki/Seymour_Papert" title="Seymour Papert">
       Seymour Papert
      </a>
      and the MIT Logo Group in the 1960s and has also manifested itself with
      <a href="/wiki/Mitchel_Resnick" title="Mitchel Resnick">
       Mitchel Resnick
      </a>
      's work on
      <a href="/wiki/Lego_Mindstorms" title="Lego Mindstorms">
       Lego Mindstorms
      </a>
      and
      <a href="/wiki/StarLogo" title="StarLogo">
       StarLogo
      </a>
      .
      <sup class="reference" id="cite_ref-MIT_roots_1-1">
       <a href="#cite_note-MIT_roots-1">
        <span>
         [
        </span>
        1
        <span>
         ]
        </span>
       </a>
      </sup>
      <sup class="reference" id="cite_ref-Giants_2-0">
       <a href="#cite_note-Giants-2">
        <span>
         [
        </span>
        2
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>

     <h2>
      <span class="mw-headline" id="Implementation">
       Implementation
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=App_Inventor_for_Android&amp;action=edit&amp;section=1" title="Edit section: Implementation">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      App Inventor includes:
     </p>
     <ul>
      <li>
       A
       <i>
        designer
       </i>
       , in which a program's
       <i>
        components
       </i>
       are specified. This includes visible components, such as buttons and images, which are placed on a simulated screen, and non-visible components, such as sensors and web connections.
      </li>
      <li>
       A
       <i>
        blocks editor
       </i>
       , in which the program's logic is created.
      </li>
      <li>
       A compiler based on the
       <a href="/wiki/Kawa_(Scheme_implementation)" title="Kawa (Scheme implementation)">
        Kawa language framework
       </a>
       and Kawa's dialect of the
       <a href="/wiki/Scheme_(programming_language)" title="Scheme (programming language)">
        Scheme
       </a>
       programming language, developed by Per Bothner
       <sup class="reference" id="cite_ref-kawa_3-0">
        <a href="#cite_note-kawa-3">
         <span>
          [
         </span>
         3
         <span>
          ]
         </span>
        </a>
       </sup>
       and distributed as part of the
       <a href="/wiki/GNU" title="GNU">
        GNU
       </a>
       operating system by the Free Software Foundation.
       <sup class="reference" id="cite_ref-gnu_4-0">
        <a href="#cite_note-gnu-4">
         <span>
          [
         </span>
         4
         <span>
          ]
         </span>
        </a>
       </sup>
      </li>
      <li>
       An app for real-time debugging on a connected Android device.
      </li>
     </ul>
     <h2>
      <span class="mw-headline" id="History">
       History
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=App_Inventor_for_Android&amp;action=edit&amp;section=2" title="Edit section: History">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      The application was made available through request on July 12, 2010, and released publicly on December 15, 2010. The App Inventor team was led by
      <a href="/wiki/Hal_Abelson" title="Hal Abelson">
       Hal Abelson
      </a>
      <sup class="reference" id="cite_ref-MIT_roots_1-2">
       <a href="#cite_note-MIT_roots-1">
        <span>
         [
        </span>
        1
        <span>
         ]
        </span>
       </a>
      </sup>
      and Mark Friedman.
      <sup class="reference" id="cite_ref-Book_5-0">
       <a href="#cite_note-Book-5">
        <span>
         [
        </span>
        5
        <span>
         ]
        </span>
       </a>
      </sup>
      In the second half of 2011, Google released the source code, terminated its server, and provided funding for the creation of The MIT Center for Mobile Learning, led by App Inventor creator Hal Abelson and fellow MIT professors Eric Klopfer and Mitchel Resnick.
      <sup class="reference" id="cite_ref-MIT_App_Inv_6-0">
       <a href="#cite_note-MIT_App_Inv-6">
        <span>
         [
        </span>
        6
        <span>
         ]
        </span>
       </a>
      </sup>
      <sup class="reference" id="cite_ref-cml_7-0">
       <a href="#cite_note-cml-7">
        <span>
         [
        </span>
        7
        <span>
         ]
        </span>
       </a>
      </sup>
      The MIT version was launched in March 2012.
      <sup class="reference" id="cite_ref-ai2_8-0">
       <a href="#cite_note-ai2-8">
        <span>
         [
        </span>
        8
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <p>
      On December 6, 2013 (the start of the
      <a class="mw-redirect" href="/wiki/Hour_of_Code" title="Hour of Code">
       Hour of Code
      </a>
      ),
      <sup class="reference" id="cite_ref-ai2_8-1">
       <a href="#cite_note-ai2-8">
        <span>
         [
        </span>
        8
        <span>
         ]
        </span>
       </a>
      </sup>
      MIT released App Inventor 2, renaming the original version "App Inventor Classic"
      <sup class="reference" id="cite_ref-classic_9-0">
       <a href="#cite_note-classic-9">
        <span>
         [
        </span>
        9
        <span>
         ]
        </span>
       </a>
      </sup>
      Major differences are:
     </p>
     <ul>
      <li>
       The
       <i>
        blocks editor
       </i>
       in the original version ran in a separate Java process, using the Open Blocks Java library for creating visual blocks programming languages and programming
      </li>
     </ul>
     <div class="thumb tright">
      <div class="thumbinner" style="width:302px;">
       <a class="image" href="/wiki/File:App_Inventor_Block_Editor.png">
        <img alt="" class="thumbimage" data-file-height="145" data-file-width="600" height="73" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ed/App_Inventor_Block_Editor.png/300px-App_Inventor_Block_Editor.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ed/App_Inventor_Block_Editor.png/450px-App_Inventor_Block_Editor.png 1.5x, //upload.wikimedia.org/wikipedia/commons/e/ed/App_Inventor_Block_Editor.png 2x" width="300"/>
       </a>
       <div class="thumbcaption">
        <div class="magnify">
         <a class="internal" href="/wiki/File:App_Inventor_Block_Editor.png" title="Enlarge">
         </a>
        </div>
        App Inventor Classic Blocks Editor
       </div>
      </div>
     </div>
     <p>
      Open Blocks is distributed by the
      <a href="/wiki/Massachusetts_Institute_of_Technology" title="Massachusetts Institute of Technology">
       Massachusetts Institute of Technology
      </a>
      's Scheller Teacher Education Program (STEP) and is derived from master's thesis research by Ricarose Roque. Professor Eric Klopfer and Daniel Wendel of the Scheller Program supported the distribution of Open Blocks under an
      <a href="/wiki/MIT_License" title="MIT License">
       MIT License
      </a>
      .
      <sup class="reference" id="cite_ref-Giants_2-1">
       <a href="#cite_note-Giants-2">
        <span>
         [
        </span>
        2
        <span>
         ]
        </span>
       </a>
      </sup>
      Open Blocks visual programming is closely related to
      <a class="mw-redirect" href="/wiki/StarLogo_TNG" title="StarLogo TNG">
       StarLogo TNG
      </a>
      , a project of STEP, and
      <a href="/wiki/Scratch_(programming_language)" title="Scratch (programming language)">
       Scratch
      </a>
      , a project of
      <a class="new" href="/w/index.php?title=MIT_Media_Laboratory%27s&amp;action=edit&amp;redlink=1" title="MIT Media Laboratory's (page does not exist)">
       MIT Media Laboratory's
      </a>
      <a href="/wiki/Mitchel_Resnick" title="Mitchel Resnick">
       Lifelong Kindergarten Group
      </a>
      . App Inventor 2
      <sup class="reference" id="cite_ref-classic_9-1">
       <a href="#cite_note-classic-9">
        <span>
         [
        </span>
        9
        <span>
         ]
        </span>
       </a>
      </sup>
      replaced Open Blocks with
      <a href="/wiki/Blockly" title="Blockly">
       Blockly
      </a>
      , a blocks editor that runs within the browser.
     </p>
     <ul>
      <li>
       The
       <a class="external text" href="https://play.google.com/store/apps/details?id=edu.mit.appinventor.aicompanion3" rel="nofollow">
        MIT AI2 Companion app
       </a>
       enables real-time debugging on connected devices via Wi-Fi, not just USB.
      </li>
     </ul>
     <p>
      As of May 2014, there were 87 thousand weekly active users of the service and 1.9 million registered users in 195 countries for a total of 4.7 million apps built.
