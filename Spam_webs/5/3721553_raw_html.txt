                    <div class="post-body entry-content" id="post-body-6010126531640568556" itemprop="articleBody">
                     There are never enough crochet appliques. And the only thing that is more fun than crocheting them, is designing them :) I was in the mood for an elaborate layered flower and this is the result. When finished, the flower is approximately 7cm / 3" in diameter.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://4.bp.blogspot.com/-aXxSs0DOW80/UxUshqle6gI/AAAAAAAAESY/EYhegtm8e10/s1600/funky_flower_crochet_pattern_blog.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                       <img alt="funky flower crochet pattern" border="0" src="http://4.bp.blogspot.com/-aXxSs0DOW80/UxUshqle6gI/AAAAAAAAESY/EYhegtm8e10/s1600/funky_flower_crochet_pattern_blog.jpg" title="funky flower crochet pattern"/>
                      </a>
                     </div>
                     <br/>
                     <div style="text-align: center;">
                      <span style="font-size: 16px;">
                       <a href="http://blog.loerinc.sk/crochetPDF/Funky-Flower-crochet-pattern.pdf" target="_blank">
                        <b>
                         <u>
                          FUNKY FLOWER FREE CROCHET PATTERN PDF
                         </u>
                        </b>
                       </a>
                      </span>
                     </div>
                     <div style="text-align: center;">
                      <br/>
                     </div>
                     <b>
                      <u>
                       MATERIALS
                      </u>
                     </b>
                     <br/>
                     <ul>
                      <li>
                       3.5 mm crochet hook (US: E)
                      </li>
                      <li>
                       Sport / 5 ply (12 wpi) yarn in 3 colors (I Catania Solids from Schachenmayr in dark yellow, light green and light purple)
                      </li>
                      <li>
                       Yarn needle
                      </li>
                     </ul>
                     <a name="more">
                     </a>
                     <br/>
                     <div>
                      <b>
                       <u>
                        ABBREVIATIONS (US)
                       </u>
                      </b>
                      <br/>
                      <ul>
                       <li>
                        mc - magic circle
                       </li>
                       <li>
                        ch - chain
                       </li>
                       <li>
                        st - stitch
                       </li>
                       <li>
                        sl st - slip stitch
                       </li>
                       <li>
                        sc - single crochet
                       </li>
                       <li>
                        hdc - half double crochet
                       </li>
                       <li>
                        dc - double crochet
                       </li>
                       <li>
                        tr - tremble crochet
                       </li>
                       <li>
                        { } - reapeat
                       </li>
                       <li>
                        [ ] - stitch count
                       </li>
                      </ul>
                      <ul>
                      </ul>
                      <br/>
                      <div style="text-align: center;">
                       <b>
                        <u>
                         FUNKY FLOWER CROCHET PATTERN
                        </u>
                       </b>
                      </div>
                      <div style="text-align: center;">
                       <i>
                        Note:
                       </i>
                       <i>
                        The flower is made of 3 separate parts that are continually connected. Each part is worked in rounds.
                       </i>
                      </div>
                      <i>
                       <br/>
                      </i>
                      <b>
                       <u>
                        PART1: MIDDLE CIRCLE (COLOR A - DARK YELLOW)
                       </u>
                      </b>
                      <br/>
                      <b>
                       <br/>
                      </b>
                      <b>
                       <u>
                        R1
                       </u>
                      </b>
                      <span style="background-color: white;">
                       : mc with 8sc, sl st in 1 st sc [8st]
                      </span>
                     </div>
                     <div>
                      <br/>
                      <i style="background-color: white;">
                       Note: Do not pull tight, leave a little hole in the middle)
                      </i>
                     </div>
                     <div>
                      <i>
                       <br/>
                      </i>
                      <b>
                       <u>
                        R2
                       </u>
                      </b>
                      <span style="background-color: white;">
                       : ch1, 2sc in each st around, sl st in 1st sc [16st]
                      </span>
                     </div>
                     <div>
                      <br/>
                      <b>
                       <u>
                        R3
                       </u>
                      </b>
                      <span style="background-color: white;">
                       : ch1, {sc, 2sc} in each st around, sl st in 1st sc [24st]
                      </span>
                     </div>
                     <div>
                      <br/>
                      <i style="background-color: white;">
                       Fasten off and weave in the end.
                      </i>
                     </div>
                     <div>
                      <i>
                       <br/>
                      </i>
                     </div>
                     <div>
                      <i>
                       <br/>
                      </i>
                     </div>
                     <div>
                      <b>
                       <u>
                        PART 2: INNER PETALS (COLOR B - LIGHT GREEN)
                       </u>
                      </b>
                      <br/>
                      <i style="background-color: white;">
                       Cut off 60cm (24) of the green yarn and put it aside (you will need it later). Working if the ball of yarn
                      </i>
                     </div>
                     <div>
                      <i>
                       <br/>
                      </i>
                      <b>
                       <u>
                        R1
                       </u>
                      </b>
                      <span style="background-color: white;">
                       : mc with 8sc, sl st in 1st sc [8st]
                      </span>
                     </div>
                     <div>
                      <br/>
                      <i style="background-color: white;">
                       Note: Do not pull tight, leave a little hole in the middle
                      </i>
                     </div>
                     <div>
                      <i>
                       <br/>
                      </i>
                      <b>
                       <u>
                        R2
                       </u>
                      </b>
                      <span style="background-color: white;">
                       : ch1, 2sc in each st around, sl st in 1st sc [16st]
                      </span>
                     </div>
                     <div>
                      <b>
                       <br/>
                      </b>
                     </div>
                     <div>
                      <span style="background-color: white;">
                      </span>
                      <b>
                       <u>
                        R3
                       </u>
                      </b>
                      <span style="background-color: white;">
                       : ch1, {sc, 2sc} *7, 2sc in next 2st, sl st in 1st sc [25st]
                      </span>
                     </div>
                     <div>
                      <br/>
                      <b>
                       <u>
                        R4
                       </u>
                      </b>
                      <span style="background-color: white;">
                       : {ch3, 2tr in next st, ch3, sl st in next 4st} *4, ch3, 2tr in next st, ch3, sl st in next 3 st
                      </span>
                     </div>
                     <div>
                      <br/>
                      <i style="background-color: white;">
                       DO NOT FASTEN OFF!!
                      </i>
                      <br/>
                      <ul>
                      </ul>

                                        var url = config.feed + '?alt=json&amp;v=2&amp;orderby=published&amp;reverse=false&amp;max-results=50';
                                        if (cursor) {
                                          url += '&amp;published-min=' + new Date(cursor).toISOString();
                                        }
                                        window.bloggercomments = function(data) {
                                          var parsed = parse(data);
                                          cursor = parsed.length &lt; 50 ? null
                                          : parseInt(parsed[parsed.length - 1].timestamp) + 1
                                          callback(parsed);
                                          window.bloggercomments = null;
                                        }
                                        url += '&amp;callback=bloggercomments';
                                        var script = document.createElement('script');
                                        script.type = 'text/javascript';
                                        script.src = url;
                                        document.getElementsByTagName('head')[0].appendChild(script);
                                      }
                                    };
                                    var hasMore = function() {
                                      return !!cursor;
                                    };
                                    var getMeta = function(key, comment) {
                                      if ('iswriter' == key) {
                                        var matches = !!comment.author
                                        &amp;&amp; comment.author.name == config.authorName
                                        &amp;&amp; comment.author.profileUrl == config.authorUrl;
                                        return matches ? 'true' : '';
                                      } else if ('deletelink' == key) {
                                        return config.baseUri + '/delete-comment.g?blogID='
                                        + config.blogId + '&amp;postID=' + comment.id;
                                      } else if ('deleteclass' == key) {
                                        return comment.deleteclass;
                                      }
                                      return '';
                                    };
                                    var replybox = null;
                                    var replyUrlParts = null;
                                    var replyParent = undefined;
                                    var onReply = function(commentId, domId) {
                                      if (replybox == null) {
                                        // lazily cache replybox, and adjust to suit this style:
                                        replybox = document.getElementById('comment-editor');
                                        if (replybox != null) {
                                          replybox.height = '250px';
                                          replybox.style.display = 'block';
                                          replyUrlParts = replybox.src.split('#');
                                        }
                                      }
                                      if (replybox &amp;&amp; (commentId !== replyParent)) {
                                        document.getElementById(domId).insertBefore(replybox, null);
                                        replybox.src = replyUrlParts[0]
                                        + (commentId ? '&amp;parentID=' + commentId : '')
                                        + '#' + replyUrlParts[1];
                                        replyParent = commentId;
                                      }
                                    };
                                    var hash = (window.location.hash || '#').substring(1);
                                    var startThread, targetComment;
                                    if (/^comment-form_/.test(hash)) {
                                      startThread = hash.substring('comment-form_'.length);
                                    } else if (/^c[0-9]+$/.test(hash)) {
                                      targetComment = hash.substring(1);
                                    }
                                    // Configure commenting API:
                                    var configJso = {
                                      'maxDepth': config.maxThreadDepth
                                    };
                                    var provider = {
                                      'id': config.postId,
                                      'data': items,
                                      'loadNext': paginator,
                                      'hasMore': hasMore,
                                      'getMeta': getMeta,
                                      'onReply': onReply,
                                      'rendered': true,
                                      'initComment': targetComment,
                                      'initReplyThread': startThread,
                                      'config': configJso,
                                      'messages': msgs
                                    };
                                    var render = function() {
                                      if (window.goog &amp;&amp; window.goog.comments) {
                                        var holder = document.getElementById('comment-holder');
                                        window.goog.comments.render(holder, provider);
                                      }
                                    };
                                    // render now, or queue to render when library loads:
                                    if (window.goog &amp;&amp; window.goog.comments) {
                                      render();
                                    } else {
                                      window.goog = window.goog || {};
                                      window.goog.comments = window.goog.comments || {};
                                      window.goog.comments.loadQueue = window.goog.comments.loadQueue || [];
                                      window.goog.comments.loadQueue.push(render);
                                    }
                                  })();
                                  // ]]&gt;
                     </script>
