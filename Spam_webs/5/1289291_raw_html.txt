        <h1>
         Installing Your Pool Table Cloth
        </h1>
        <!-- Go to www.addthis.com/dashboard to customize your tools -->
       </div>
       <div class="addthis_native_toolbox">

        <strong>
         Posted by

        <strong>
         <a href="http://plus.google.com/105534709656735273197/posts">

        <p>
         Over the years one of the most requested articles we've been asked for is some sort of help with installing pool table felt. As a result, we have gathered some information about our most popular table cloth,
         <a href="http://www.pooldawg.com/brand/simonis">
          Simonis
         </a>
         , and how to install it.
         <br/>
         <br/>
         <strong>
          Disclaimer: The following information is for reference only. PoolDawg.com only recommends that trained billiard table professionals install cloth as untrained installers often have trouble properly laying table cloth.
         </strong>
         <br/>
         <br/>
         That being said, here's some basic information about Simonis cloth. According to Simonis, when installing their cloth, put the smooth side up. You'll also notice that the
         <a href="http://www.pooldawg.com/brand/simonis">
          <img align="left" alt="Return to Simonis Homepage" border="0" height="91" src="http://www.pooldawg.com/uploads/images/simonislogo.gif" width="250"/>
         </a>
         smooth side has printing on the edge that will tell you the grade of the cloth (
         <a href="http://www.pooldawg.com/product/simonis-760-pool-table-cloth">
          Simonis 760
         </a>
         ,
         <a href="http://www.pooldawg.com/product/simonis-860-pool-table-cloth">
          Simonis 860
         </a>
         , etc).
         <br/>
         <br/>
         If you have any questions about installing cloth, please feel free to call us toll free at 866-THE-DAWG. In addition, we have included a link to the installation reference guide from Simonis. Again, please note that this is for reference only and any installation should only be done by a trained professional.
         <br/>
         <br/>
         <a href="http://www.simoniscloth.com/installation">
          Download Simonis Installation Diagrams (PDF Format)
         </a>
        </p>
