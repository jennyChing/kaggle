                    <h3 class="post-title entry-title" itemprop="name">
                     My little garden
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-3226496176442940713" itemprop="description articleBody">
                     <br/>
                     <span style="font-family: Verdana,sans-serif;">
                      <br/>
                      <a href="http://www.bloglovin.com/blog/10657371/?claim=h29rv7mbb9d">
                       Follow my blog with Bloglovin
                      </a>
                     </span>
                     <br/>
                     <span style="font-family: Verdana,sans-serif;">
                     </span>
                     <br/>
                     <span style="font-family: Verdana,sans-serif;">
                      Hey,
                     </span>
                     <br/>
                     <span style="font-family: Verdana,sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Verdana,sans-serif;">
                      today I want to tell you a little bit about my garden.
                     </span>
                     <br/>
                     <span style="font-family: Verdana,sans-serif;">
                      For the first time we planted some vegetables this year. I could hardly decide what to grow. At the end I had plenty of herbs, zucchini, corn, squash, cauliflower, broccoli.
                     </span>
                     <br/>
                     <span style="font-family: Verdana,sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Verdana,sans-serif;">
                      I wanted to go for tomatoes as well. But as the summer last year was extremely wet I was worried if my plants would survive. So I didn't go for it.
                     </span>
                     <br/>
                     <span style="font-family: Verdana,sans-serif;">
                      Turned out that we had a fantastic summer this year with less rain and lots of sun. But well, then I try it next year.
                     </span>
                     <br/>
                     <span style="font-family: Verdana,sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Verdana,sans-serif;">
                      The Zucchini plants are growing amazing. I cannot tell you how many we've got from it so far. And it won't stop. New Zucchinis keep on coming. My dad told me I would only need 1 plant for the two of us. Of course I would not believe him. He was perfectly right. My whole freezer is full of slices Zucchini and there is no end (just spottet 7 more today :-) )
                     </span>
                     <br/>
                     <span style="font-family: Verdana,sans-serif;">
                      <br/>
                     </span>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <span style="font-family: Verdana,sans-serif;">
                       <a href="http://2.bp.blogspot.com/-WdFwuKqxLAQ/UjXqSUEu7DI/AAAAAAAAADg/mehZsbkwvOg/s1600/Unbenannt.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                        <img border="0" height="175" src="http://2.bp.blogspot.com/-WdFwuKqxLAQ/UjXqSUEu7DI/AAAAAAAAADg/mehZsbkwvOg/s400/Unbenannt.png" width="400"/>
                       </a>
                      </span>
                     </div>
                     <span style="font-family: Verdana,sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Verdana,sans-serif;">
                      The cauliflower and broccoli did not survive this year. Also the squash plants got eaten by slugs.
                     </span>
                     <br/>
                     <span style="font-family: Verdana,sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Verdana,sans-serif;">
                      Before we could get started this year we had to do a lot of work outside. Even if it is only a little piece we were busy quite a lot.
                     </span>
                     <br/>
                     <span style="font-family: Verdana,sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Verdana,sans-serif;">
                      I have already many of Ideas for the next year.
                     </span>
                     <br/>
                     <span style="font-family: Verdana,sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Verdana,sans-serif;">
                      Have a nice sunday evening,
                     </span>
                     <br/>
                     <span style="font-family: Verdana,sans-serif;">
                      Susann
                     </span>
