                </a>
               </h2>
               <p class="blog-date">
                <span class="date-text">
                 8/21/2014
                </span>

               <p class="blog-comments">

                <div class="wsite-image wsite-image-border-none" style="padding-top:0;padding-bottom:0;margin-left:0;margin-right:0;text-align:center">
                 <a>
                  <img alt="Picture" src="/uploads/6/1/0/9/6109222/1408591789.jpg" style="width:auto;max-width:100%"/>
                 </a>
                 <div style="display:block;font-size:90%">
                 </div>
                </div>
               </div>
               <div class="paragraph">
                It is time to announce the completion of our top secretTitronic's project.We have incorporated for the first time EVER the ability to mark up to 20 constants on your scans that you can place individually that stay with each individual patient. So you can quickly check for pattern using the Duff analysis as taught by Dr. Duff.
                <br/>
                <span>
                </span>
                <br/>
                <span>
                </span>
                Yes, that'sright no more drawing stuff out by hand!
                <br/>
                <span>
                </span>
                <br/>
                <span>
                </span>
                We have completed beta testing and the update is now ready for use. Titronic's will release the update in the next few days. If you have the Titronics software 2.0 and above it will prompt you for the update. Otherwise you can go into set up and click check for updates. If this does not work then call Titronic's directly. To check to see which version of the software you have, simply go to set up which should be in your top left corner of the Titronic's page home screen.
                <br/>
                <span>
                </span>
                <br/>
                <span>
                </span>
                <strong>
                 <u>
                  Here is how it works:
                 </u>
                </strong>
                <br/>
                <span>
                </span>
                <br/>
                <span>
                </span>
                <strong>
                 Step 1
                </strong>
                - Pull up your Titronic's software andhit "
                <strong>
                 F5
                </strong>
                " select your best 3-5 initial scans for pattern (make sure the scans are all the same length!!), then hit "
                <strong>
                 overlay selected
                </strong>
                " at the top of the menu bar.
                <br/>
                <strong>
                 Step 2
                </strong>
                - To mark your constants hit "
                <strong>
                 L
                </strong>
                ", then mark the constants you want by clicking laterally to the lines.
                <br/>
                <strong>
                 Step 3
                </strong>
                - Hit "
                <strong>
                 L
                </strong>
                " again to when done lock themin.(Don't worry if you make an error justhit "
                <strong>
                 L
                </strong>
                " twice to erase all lines and begin again.)
                <br/>
                <strong>
                 Step 4
                </strong>
                -If you want to remove the constants from a particular patient just pull them up open a scan and hit "
                <strong>
                 L
                </strong>
                " two times to erase the constants that you marked.
                <br/>
                <span>
                </span>
                <br/>
                <span>
                </span>
                <strong>
                 P.S.
                </strong>
                - You can toggle the lines on and off the patient account by hitting "
                <strong>
                 H
                </strong>
                " so that you may also show just the scans.
                <br/>
                <span>
                </span>
                <br/>
                <span>
                </span>
                <strong>
                 P.S.S.
                </strong>
                - The lines will also stay if you wish to print scans as well!
                <br/>
                <span>

I would also like to thank Chris at Titronic's for his help in making this possible, and last but not least Dr. Mike Anderson for sharing his know how to help the rest of us grow.
