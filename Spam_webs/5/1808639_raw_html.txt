          <h1>
           What is BrewPi?
          </h1>
         </header>
         <section class="entry fix">
          <div class="threecol-two">
           <p>
            BrewPi is a fermentation temperature controller for brewing beer or wine. It runs on Raspberry Pi and an Arduino. Compared to other temperature controllers, BrewPi offers much better temperature control and a lot more features like data logging, a web interface and graphs.
           </p>
           <p>
            In our shop you can buy the BrewPi Arduino shield, temperature sensors, SSRs, a casing and more to convert your fridge or freezer into an awesome fermentation chamber.
           </p>
           <p>
            BrewPi is open source and has an active community. We are working hard to support mash control (BIAB, HERMS or RIMS) in the near future
           </p>
          </div>
          <div class="threecol-one last">
           <p style="text-align: center;">
            <a href="http://www.brewpi.com/wp-content/uploads/2014/01/BrewPi-Case-2.0-with-sensors-and-actuators.jpg" rel="lightbox[890]" title="BrewPi Case 2.0 with sensors and actuators">
             <img alt="BrewPi Case 2.0 with sensors and actuators" class="size-medium wp-image-1132 aligncenter" height="200" src="http://www.brewpi.com/wp-content/uploads/2014/01/BrewPi-Case-2.0-with-sensors-and-actuators-300x200.jpg" style="background: none repeat scroll 0% 0% #ffffff; box-shadow: 0px 1px 4px rgba(0, 0, 0, 0.3), 0px 0px 40px rgba(0, 0, 0, 0.1) inset;" title="BrewPi Case 2.0 with sensors and actuators" width="300"/>
            </a>
           </p>
           <p>
           </p>

        <div class="feature last">
         <h3 class="feature-title">
          Raspberry Pi + Arduino
         </h3>
         <div class="feature-content">
          <div class="threecol-one">
           <img alt="raspberry-arduino-200px" class="alignnone size-full wp-image-1065" height="259" src="http://www.brewpi.com/wp-content/uploads/2014/01/raspberry-arduino-200px.png" width="200"/>
          </div>
          <div class="threecol-two last">
           BrewPi runs on two very popular boards: a
           <a href="http://www.raspberrypi.org/faqs" target="_blank" title="Raspberry Pi">
            Raspberry Pi
           </a>
           and an
           <a href="http://arduino.cc/" target="_blank" title="Arduino">
            Arduino
           </a>
           . The Raspberry Pi is a credit card sized computer that runs Linux. In BrewPi it runs the web server and a python script to log data and manage settings. The Arduino is a microcontroller board that reads the temperature sensors and activates the cooler, heater and other actuators. It also has an LCD and knob to display and change settings.
           <br/>
           <br/>
           <h4>
            Stability and flexibility
           </h4>
           The Arduino runs the temperature control algorithm autonomously. If you don't care about data logging, it can control your beer without the Rasberry Pi. This makes this combo very stable: if the Raspberry Pi crashes, the Arduino will still keep your beer temperature perfectly under control. And being a simple embedded device, the Arduino just does not crash.

If you want to run BrewPi on something else than a Raspberry Pi, you can! Any platform that has USB and can run a web server and Python will be able to run BrewPi.
           <span style="line-height: 1.5em;">
           </span>
          </div>
          <div class="clear">
          </div>

        <div class="feature last">
         <h3 class="feature-title">
          Web interface
         </h3>
         <div class="feature-content">
          BrewPi runs a local web server that provides a great interface to change settings, view brewing data and to configure your controller.

The web interface shows interactive graphs of your current brew and previous brews. You can easily start a new brew, view what the control algorithm is doing, check the logs, load/save temperature profiles, edit the active profile, etc.

        <div class="feature last">
         <h3 class="feature-title">
          Advanced temperature control algorithm
         </h3>
         <div class="feature-content">
          <div class="threecol-one">
           <a class="cboxElement" href="http://www.brewpi.com/wp-content/uploads/2014/01/control-algorithm.png" rel="lightbox[1037]" title="You can view the calculations of the control algorithm in the web interface. The values for Kp, Ki and Kd are configurable.">
            <img alt="You can view the calculations of the control algorithm in the web interface. The values for Kp, Ki and Kd are configurable." class="size-medium wp-image-1097" height="268" src="http://www.brewpi.com/wp-content/uploads/2014/01/control-algorithm-300x268.png" width="300"/>
           </a>
          </div>
          <div class="threecol-two last">
           BrewPi has an advanced temperature control algorithm that can keep your beer temperature stable within 0.1 &amp;degC.
           <br/>
           <br/>
           Instead of just heating when the beer is too cold and cooling when the beer is too warm, BrewPi calculates what the fridge temperature should be to keep your beer temperature in check. It then controls the fridge temperature. This prevents fluctuations in the beer temperature. More details about the temperature control algorithm in BrewPi can be found here: (coming soon).
          </div>
          <div class="clear">
          </div>
         </div>
        </div>
        <div class="fix">
        </div>
        <div class="feature last">
         <h3 class="feature-title">
          Temperature profiles
         </h3>
         <div class="feature-content">
          <div class="threecol-two">
           With BrewPi, you have much more control than just keeping a constant beer temperature: you can define a temperature profile over time.
           <br/>
           <br/>
           An example of what you can do with a temperature profile:
           <br/>
           <br/>
           <ul>
            <li>
             You start at 20C for 36 hours, right in the middle of the temperature range for your yeast to make it easy for them to start up and multiply.
            </li>
            <li>
             After 36 hours, when your beer is bubbling well, you want drop the temperature to reduce ester formation. You set the profile to be at 17C at 3 days.
            </li>
            <li>
             You decide to keep the beer at 17C for 3 days, but fermentation is slowing down.
            </li>
            <li>
             Yeast is more active at higher temperatures, so you slowly raise to 22C over 5 days: as the yeast consumes more and more complex sugars, it can do this at a higher temperature.
            </li>
            <li>
             After reaching 22C, you keep it at that temp for 2 days (diacetyl rest), before cold crashing at 3C.
            </li>
           </ul>
           Using temperature profiles, you can do gradual changes in temperature. Yeast is sensitive to sudden temperature changes: a sudden drop in temperature can cause them to drop out of suspension because they need time to adapt to a new set point.
          </div>

           </a>
