     <p>
      <b>
       The Eight Principles of
       <i>
        Yong
       </i>
      </b>
      (
      <a href="/wiki/Chinese_language" title="Chinese language">
       Chinese
      </a>
      :
      <span lang="zh" xml:lang="zh">
       
      </span>
      ;
      <a href="/wiki/Pinyin" title="Pinyin">
       pinyin
      </a>
      :
      <i>
       <span lang="zh-Latn-pinyin" xml:lang="zh-Latn-pinyin">
        Yngz B F
       </span>
      </i>
      ;
      <a href="/wiki/Japanese_language" title="Japanese language">
       Japanese
      </a>
      :
      <span lang="ja" xml:lang="ja">
       /
      </span>
      ,
      <i>
       eiji happ
      </i>
      ;
      <a href="/wiki/Korean_language" title="Korean language">
       Korean
      </a>
      :
      <span lang="ko" xml:lang="ko">
       
      </span>
      ,
      <i>
       Yeongjapalbeop
      </i>
      ;
      <a href="/wiki/Vietnamese_language" title="Vietnamese language">
       Vietnamese
      </a>
      :
      <span lang="vi" xml:lang="vi">
       <i>
        Vnh T Bt Php/Tm Phng Php vit Ch Vnh
       </i>
      </span>
      ) explain how to write eight common
      <a class="mw-redirect" href="/wiki/Stroke_(Chinese_character)" title="Stroke (Chinese character)">
       strokes
      </a>
      in
      <a href="/wiki/Regular_script" title="Regular script">
       regular script
      </a>
      which are found all in the one character,
      <span lang="zh" xml:lang="zh">
       <a class="extiw" href="//en.wiktionary.org/wiki/%E6%B0%B8" title="wikt:">
        
       </a>
      </span>
      (
      <a href="/wiki/Pinyin" title="Pinyin">
       pinyin
      </a>
      :
      <i>
       <span lang="zh-Latn-pinyin" xml:lang="zh-Latn-pinyin">
        yng
       </span>
      </i>
      , "forever", "permanence"). It was traditionally believed that the frequent practice of these principles as a beginning
      <a href="/wiki/Chinese_calligraphy" title="Chinese calligraphy">
       calligrapher
      </a>
      could ensure
      <a href="/wiki/Beauty" title="Beauty">
       beauty
      </a>
      in one's writing.
     </p>
     <p>
      The Eight Principles are influenced by the earlier Seven Powers (
      <span lang="zh" xml:lang="zh">
       
      </span>
      ) by Lady
      <a href="/wiki/Wei_Shuo" title="Wei Shuo">
       Wei Shuo
      </a>
      (
      <span lang="zh" xml:lang="zh">
       
      </span>
      ) of
      <a class="mw-redirect" href="/wiki/Eastern_Jin" title="Eastern Jin">
       Eastern Jin
      </a>
      . Publications on the Principles include:
     </p>
     <ul>
      <li>
       <i>
        The Praise to the Eight Principles of "Yong"
       </i>
       (
       <span lang="zh" xml:lang="zh">
        
       </span>
       ) by
       <a href="/wiki/Liu_Zongyuan" title="Liu Zongyuan">
        Liu Zongyuan
       </a>
       (
       <span lang="zh" xml:lang="zh">
        
       </span>
       ) of the
       <a class="mw-redirect" href="/wiki/Tang_Dynasty" title="Tang Dynasty">
        Tang Dynasty
       </a>
       .
      </li>
      <li>
       <i>
        Explanations to the Eight Principles of "Yong"
       </i>
       (
       <span lang="zh" xml:lang="zh">
        
       </span>
       ) by Li Puguang (
       <span lang="zh" xml:lang="zh">
        
       </span>
       ) of the
       <a class="mw-redirect" href="/wiki/Yuan_Dynasty" title="Yuan Dynasty">
        Yuan Dynasty
       </a>
       . L provided two-character
       <a href="/wiki/Metaphor" title="Metaphor">
        metaphorical
       </a>
       names.
      </li>
     </ul>
