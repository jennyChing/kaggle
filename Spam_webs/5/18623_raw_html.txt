     <h2>
      <span class="mw-headline" id="Window_HotKeys">
       Window HotKeys
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Blender_3D:_HotKeys/All&amp;action=edit&amp;section=1" title="Edit section: Window HotKeys">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      Certain window managers also use the following hotkeys. So
      <b>
       ALT-CTRL
      </b>
      can be substituted for
      <b>
       CTRL
      </b>
      to perform the functions described below if a conflict arises.
     </p>
     <dl>
      <dd>
       <b>
        CTRL-LEFTARROW
       </b>
       . Go to the previous Screen.
      </dd>
      <dd>
       <b>
        CTRL-RIGHTARROW
       </b>
       . Go to the next Screen.
      </dd>
      <dd>
       <b>
        CTRL-UPARROW
       </b>
       or
       <b>
        CTRL-DOWNARROW
       </b>
       . Maximize the window or return to the previous window display size.
      </dd>
      <dd>
       <b>
        SHIFT-F4
       </b>
       . Change the window to a Data View
      </dd>
      <dd>
       <b>
        SHIFT-F5
       </b>
       . Change the window to a 3D Window
      </dd>
      <dd>
       <b>
        SHIFT-F6
       </b>
       . Change the window to an IPO Window
      </dd>
      <dd>
       <b>
        SHIFT-F7
       </b>
       . Change the window to a Buttons Window
      </dd>
      <dd>
       <b>
        SHIFT-F8
       </b>
       . Change the window to a Sequence Window
      </dd>
      <dd>
       <b>
        SHIFT-F9
       </b>
       . Change the window to an Outliner Window
      </dd>
      <dd>
       <b>
        SHIFT-F10
       </b>
       . Change the window to an Image Window
      </dd>
      <dd>
       <b>
        SHIFT-F11
       </b>
       . Change the window to a Text Window
      </dd>
      <dd>
       <b>
        SHIFT-F12
       </b>
       . Change the window to an Action Window
      </dd>
     </dl>
     <h2>
      <span class="mw-headline" id="Universal_HotKeys">
       Universal HotKeys
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Blender_3D:_HotKeys/All&amp;action=edit&amp;section=2" title="Edit section: Universal HotKeys">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      The following HotKeys work uniformly in all Blender Windows, if the Context allows:
     </p>
     <dl>
      <dd>
       <b>
        CTRL-LMB
       </b>
       . Lasso select: drag the mouse to form a freehand selection area.
      </dd>
     </dl>
     <ul>
      <li>
       <ul>
        <li>
         <b>
          ESC.
         </b>
        </li>
        <li>
         This key always cancels Blender functions without changes.
        </li>
        <li>
         <i>
          or
         </i>
         : FileWindow, DataView and ImageSelect: back to the previous window type.
        </li>
        <li>
         <i>
          or
         </i>
         : the RenderWindow is pushed to the background (or closed, that depends on the operating system).
        </li>
       </ul>
      </li>
     </ul>
     <dl>
      <dd>
       <b>
        SPACE.
       </b>
       Open the Toolbox.
      </dd>
      <dd>
       <b>
        CTRL-SPACE.
       </b>
       Opens menu with manipulator type choices. (Press 2, 3, 4, or 5 to change the widget type)
       <font color="red">
        *
       </font>
      </dd>
      <dd>
       <b>
        ALT-SPACE.
       </b>
       Opens menu with manipulator orientation choices.
       <font color="red">
        *
       </font>
      </dd>
      <dd>
       <b>
        TAB.
       </b>
       Start or quit EditMode. Toggles between horizontal and verticle splitting when the user is splitting the window.
      </dd>
      <dd>
       <b>
        F1.
       </b>
       Loads a Blender file. Changes the window to a FileWindow.
      </dd>
      <dd>
       <b>
        SHIFT-F1.
       </b>
       Appends parts from other files, or loads as Library-data. Changes the window to a FileWindow, making Blender files accessible as a directory.
      </dd>
      <dd>
       <b>
        F2.
       </b>
       Writes a Blender file. Change the window to a FileWindow.
      </dd>
      <dd>
       <b>
        SHIFT-F2.
       </b>
       Exports the scene as a DXF file
      </dd>
      <dd>
       <b>
        CTRL-F2.
       </b>
       Exports the scene as a VRML1 file
      </dd>
      <dd>
       <b>
        F3.
       </b>
       Writes a picture (if a picture has been rendered). The fileformat is as indicated in the DisplayButtons. The window becomes a File Select Window.
      </dd>
      <dd>
       <b>
        CTRL-F3 (ALT-CTRL-F3 on MacOSX)
       </b>
       . Saves a screendump of the active window. The fileformat is as indicated in the DisplayButtons. The window becomes a FileWindow.
      </dd>
      <dd>
       <b>
        SHIFT-CTRL-F3.
       </b>
       Saves a screendump of the whole Blender screen. The fileformat is as indicated in the DisplayButtons. The window becomes a FileWindow.
      </dd>
      <dd>
       <b>
        F4.
       </b>
       Displays the Logic Context (if a ButtonsWindow is available).
      </dd>
      <dd>
       <b>
        F5.
       </b>
       Displays the Shading Context (if a Buttons Window is available), Light, Material or World Sub-contextes depends on active object.
      </dd>
      <dd>
       <b>
        F6.
       </b>
       Displays the Shading Context and Texture Sub-context (if a ButtonsWindow is available).
      </dd>
      <dd>
       <b>
        F7.
       </b>
       Displays the Object Context (if a ButtonsWindow is available).
      </dd>
      <dd>
       <b>
        F8.
       </b>
       Displays the Shading Context and World Sub-context (if a ButtonsWindow is available).
      </dd>
      <dd>
       <b>
        F9.
       </b>
       Displays the Editing Context (if a ButtonsWindow is available).
      </dd>
      <dd>
       <b>
        F10.
       </b>

      <dd>
       <b>
        TAB
       </b>
       or
       <b>
        ALT-E.
       </b>
       This button starts and stops Edit Mode.
      </dd>
      <dd>
       <b>
        CTRL-TAB.
       </b>

      </dd>

       </b>

      </dd>

       </b>
       or
       <b>

       </b>
       .
      </dd>
      <dd>
       <b>

       </b>

      </dd>
      <dd>
       <b>
        NKEY.
       </b>
       Number Panel. Simpler than the Object Mode one, in Edit Mode works for Mesh, Curve, Surface: The location of the active vertex is displayed.
      </dd>

       </b>
       Switch in/out of Proportional Editing.
      </dd>
      <dd>
       <b>

       </b>
       Toggles between Smooth and Sharp Proportional Editing.
      </dd>
      <dd>
       <b>
        PKEY.
       </b>
       SeParate. You can choose to make a new object with all selected vertices, edges, faces and curves or create a new object from each separate group of interconnected vertices from a popup. Note that for curves you cannot separate connected control vertices. This operation is the opposite of Join (
       <b>
        CTRL-J
       </b>
       ).
      </dd>
      <dd>
       <b>
        CTRL-P.
       </b>
       Make Vertex Parent. If one object (or more than one) is/are selected and the active Object is in Edit Mode with 1 or 3 vertices selected then the Object in Edit Mode becomes the Vertex Parent of the selected Object(s). If only 1 vertex is selected, only the location of this vertex determines the Parent transformation; the rotation and dimensions of the Parent do not play a role here. If three vertices are selected, it is a `normal' Parent relationship in which the 3 vertices determine the rotation and location of the Child together. This method produces interesting effects with Vertex Keys. In EditMode, other Objects can be selected with CTRL-RMB.
      </dd>

       </b>

       </b>

      </dd>
      <dd>

       </b>

The mid-line of the circle is determined by the horizontal dimensions of the selected vertices. When you start, everything is already bent 90 degrees. Moving the mouse up or down increases or decreases the extent to which warping is done. By zooming in/out of the 3Dwindow, you can specify the maximum degree of warping. The CTRL limiter increments warping in steps of 5 degrees.
      </dd>

       Select Linked Selected. Starting with all selected vertices, all vertices connected to them are selected too.
      </dd>
      <dd>
       <b>
        MKEY.
       </b>
       Mirror. Opens a popup asking for the axis to mirror. 3 possible axis group are available, each of which contains three axes, for a total of nine choices. Axes can be Global (Blender Global Reference); Local (Current Object Local Reference) or View (Current View reference). Remember that mirroring, like scaling, happens with respect to the current pivot point.
      </dd>
      <dd>
       <b>
        ALT-M.
       </b>
       Merges selected vertices at barycentrum or at cursor depending on selection made on pop-up.
      </dd>
      <dd>
       <b>
        SHIFT+CTRL+ALT+MKEY
       </b>
       Selects unclosed edges / holes in the mesh (Edge select mode only).
      </dd>
      <dd>
       <b>
        CTRL-N.
       </b>
       Calculate Normals Outside. All normals from selected faces are recalculated and consistently set in the same direction. An attempt is made to direct all normals `outward'.
      </dd>
      <dd>
       <b>
        SHIFT-CTRL-N.
       </b>
       Calculate Normals Inside. All normals from selected faces are recalculated and consistently set in the same direction. An attempt is made to direct all normals `inward'.
      </dd>
      <dd>
       <b>
        ALT-S.
       </b>
       Whereas
       <b>
        SHIFT-S
       </b>
       scales in Edit Mode as it does in Object Mode, for Edit Mode a further option exists,
       <b>
        ALT-S
       </b>
       moves each vertex in the direction of its local normal, hence effectively shrinking/fattening the mesh.
      </dd>
      <dd>
       <b>
        CTRL-T.
       </b>
       Make Triangles. All selected faces are converted to triangles.
      </dd>
      <dd>
       <b>
        UKEY.
       </b>
       Undo. When starting Edit Mode, the original ObData block is saved and all subsequent changes are saved on a stack. This option enables you to restore the previous situation, one after the other.
      </dd>
      <dd>
       <b>
        SHIFT-U.
       </b>
       Redo. This let you re-apply any undone changes up to the moment in which Edit Mode was entered
      </dd>
      <dd>
       <b>
        ALT-U.
       </b>
       Undo Menu. This let you choose the exact point to which you want to undo changes.
      </dd>
     </dl>
     <ul>
      <li>
       <ul>
        <li>
         <b>
          WKEY.
         </b>
         Special Menu. A PopupMenu offers the following options:
         <ul>
          <li>
           <i>
            Subdivide:
           </i>
           all selected edges are split in two.
          </li>
          <li>
           <i>
            Subdivide Fractal:
           </i>
           all selected edges are split in two and middle vertex displaced randomly.
          </li>
          <li>
           <i>
            Subdivide Smooth:
           </i>
           all selected edges are split in two and middle vertex displaced along the normal.
          </li>
          <li>
           <i>
            Merge:
           </i>
           as
           <b>
            ALT-M.
           </b>
          </li>
          <li>
           <i>
            Remove Doubles:
           </i>
           All selected vertices closer to each other than a given threshold (See EditMode Button Window) are merged
           <b>
            ALT-M.
           </b>
          </li>
          <li>
           <i>
            Hide:
           </i>
           as
           <b>
            HKEY.
           </b>
          </li>
          <li>
           <i>
            Reveal:
           </i>
           as
           <b>
            ALT-H.
           </b>
          </li>
          <li>
           <i>
            Select Swap:
           </i>
           Selected vertices become unselected and vice versa.
          </li>
          <li>
           <i>
            Flip Normals:
           </i>
           Normals of selected faces are flipped.
          </li>
          <li>
           <i>
            Smooth:
           </i>
           Vertices are moved closer one to each other, getting a smoother object.
          </li>
          <li>
           <i>
            Bevel:
           </i>
           Faces are reduced in size and the space between edges is filled with a smoothly curving bevel of the desired order.
          </li>
         </ul>
        </li>
       </ul>
      </li>
     </ul>
     <ul>
      <li>
       <ul>
        <li>
         <b>
          XKEY.
         </b>
         Erase Selected. A PopupMenu offers the following options:
         <ul>
          <li>
           <i>
            Vertices:
           </i>
           all vertices are deleted. This includes the edges and faces they form.
          </li>
          <li>
           <i>
            Edges:
           </i>
           all edges with both vertices selected are deleted. If this `releases' certain vertices, they are deleted as well. Faces that can no longer exist as a result of this action are also deleted.
          </li>
          <li>
           <i>
            Faces:
           </i>
           all faces with all their vertices selected are deleted. If any vertices are `released' as a result of this action, they are deleted.
          </li>
          <li>
           <i>
            All:
           </i>
           everything is deleted.
          </li>
          <li>
           <i>
            Edges and Faces:
           </i>
           all selected edges and faces are deleted, but the vertices remain.
          </li>
          <li>
           <i>
            Only Faces:
           </i>
           all selected faces are deleted, but the edges and vertices remain.
          </li>
         </ul>
        </li>
       </ul>
      </li>
     </ul>
     <dl>
      <dd>
       <b>
        YKEY.
       </b>
       Split. This command splits the selected part of a Mesh without deleting faces. The split parts are no longer bound by edges. Use this command to control smoothing. Since the split parts have vertices at the same position, selection with
       <b>
        LKEY
       </b>
       is recommended.
      </dd>
     </dl>

      </dd>
      <dd>
       <b>
        LKEY.
       </b>
       Select Linked. If you start with an non-selected vertex near the mouse cursor, this vertex is selected together with all the vertices of the same curve.
      </dd>
      <dd>
       <b>
        SHIFT-L.
       </b>
       Deselect Linked. If you start with a selected vertex, it is deselected together with all the vertices of the same curve.
      </dd>
      <dd>
       <b>
        MKEY.
       </b>
       Mirror. Mirror selected control points exactly as for vertices in a Mesh.
      </dd>
      <dd>
       <b>
        TKEY.
       </b>
       Tilt mode. Specify an extra axis rotation, i.e. the tilt, for each vertex in a 3D curve.
      </dd>
      <dd>
       <b>
        ALT-T.
       </b>
       Clear Tilt. Set all axis rotations of the selected vertices to zero.
      </dd>
      <dd>
       <b>
        VKEY.
       </b>
       Vector Handle. The selected Bezier handles are converted to vector type.
      </dd>
     </dl>
     <ul>
      <li>
       <ul>
        <li>
         <b>
          WKEY.
         </b>
         The special menu for curves appears:
         <ul>
          <li>
           Subdivide. Subdivide the selected vertices.
          </li>
          <li>
           Switch direction. The direction of the selected curves is reversed. This is mainly for Curves that are used as paths!
          </li>
         </ul>
        </li>
       </ul>
      </li>
     </ul>

            Selected:
           </i>
           all selected vertices are deleted.
          </li>
          <li>
           <i>
            Segment:
           </i>
           a curve segment is deleted. This only works for single segments. Curves can be split in two using this option. Or use this option to specify the cyclic position within a cyclic curve.
          </li>
          <li>
           <i>
            All:
           </i>
           delete everything.
          </li>
         </ul>
        </li>
       </ul>
      </li>

           </i>

          </li>
