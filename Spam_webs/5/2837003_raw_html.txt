            <article id="article-body" itemscope="" itemtype="http://schema.org/Article" section="articleBody">
             <meta content="http://cnet3.cbsistatic.com/hub/i/r/2014/04/08/93bdfd60-4f84-4203-a30e-b139ce1ba5a0/thumbnail/770x433/7a6ca7e8da7c999d76a88d5d5317b684/windows-8-1.png" itemprop="thumbnailUrl"/>

             <div class="article-main-body row" data-component="lazyloadImages" data-use-autolinker="true" itemprop="articleBody">
              <p>
               <figure class="image image-full-width pull-none" section="shortcodeImage">
                <span class="imageContainer">
                 <img alt="windows-8-1.png" class="" height="578" itemprop="image" src="http://cnet1.cbsistatic.com/hub/i/r/2014/04/08/93bdfd60-4f84-4203-a30e-b139ce1ba5a0/resize/770x578/02737909d7ab1dfab7692306a82f1b5c/windows-8-1.png" width="770"/>
                </span>
                <figcaption>
                 <span class="credit">
                  Microsoft
                 </span>
                </figcaption>
               </figure>
              </p>
              <p>
               Windows 8.1 is due for another update this month, but don't get your hopes up for any radical changes.
              </p>
              <p>
               In a blog posted Tuesday, Microsoft Marketing Communications Manager Brandon LeBlanc cautioned users
               <a data-component="externalLink" href="http://blogs.windows.com/bloggingwindows/2014/08/05/august-updates-for-windows-8-1-and-windows-server-2012-r2/" target="_blank">
                not to expect the same level of key enhancements
               </a>
               that graced the
               <a href="http://www.cnet.com/news/the-windows-8-1-update-everything-you-need-to-know/">
                Windows 8.1 Update
               </a>
               launched in early April. Instead, this month's update, due August 12, will do little to change the way you work and play with
               <a data-is-autolink="true" href="/products/microsoft-windows-8-1/">
                Windows 8.1
               </a>
               .
              </p>
              <p>
               In an effort to appease disgruntled desktop users, Microsoft updated
               <a data-is-autolink="true" href="/products/windows-8-pro-upgrade/">
                Windows 8
               </a>
               and then 8.1 with features designed to make the OS more PC friendly. Windows 8.1 added the ability to boot directly to the desktop, while the April 8.1 update carried with it a dedicated Start screen Power button, the ability to pin Windows 8 apps to the taskbar, and other improvements for keyboard and mouse users. So why isn't Microsoft following the same course with the August update? The company is saving its big guns for next year.
              </p>
              <p>
               Initially, Microsoft had planned to reintroduce the Start menu in this month's 8.1 update along with other likely enhancements. But at some point, the company reversed course and is now
               <a href="http://www.cnet.com/news/windows-8-start-menu-from-microsoft-not-till-2015/">
                planning to bring back the Start menu in April of 2015
               </a>
               in a version of Windows currently known as Threshold and potentially dubbed
               <a data-is-autolink="true" href="/products/microsoft-windows-9/">
                Windows 9
               </a>
               when it debuts.
              </p>
              <p>
               In the meantime, Microsoft's current strategy for Windows 8.1 is to release more frequent but less breathtaking updates as it prepares for next year's major release. Just what does the August update promise for 8.1 users? Aside from the usual security patches, we can expect the following, according to LeBlanc:
              </p>
              <div class="ad-inpage-video-top" data-ad="inpage-video-top" id="inpage-video-top-55a421aa03cf9">
               <script>
                window.console &amp;&amp; console.log &amp;&amp; console.log("ADS: queuing inpage-video-top-55a421aa03cf9 for display");
        var cbsiGptDivIds = cbsiGptDivIds || [];
        cbsiGptDivIds.push("inpage-video-top-55a421aa03cf9");
               </script>
              </div>
              <ul>
               <li>
                <strong>
                 <a data-component="externalLink" href="http://windows.microsoft.com/en-us/windows-8/touchpad" target="_blank">
                  Precision touchpad
                 </a>
                 improvements
                </strong>
                - three new end-user settings have been added: leave touch pad on when a mouse is connected; allow right-clicks on the touchpad; double-tap and drag.
               </li>
               <li>
                <strong>
                 Miracast Receive
                </strong>
                - exposes a set of Wi-Fi direct APIs for Independent Hardware Vendor (IHV) drivers or OEM drivers to develop Windows 32-bit applications that run on all supported x86-based or x64-based versions of Windows 8.1, enabling the computer as a
                <a href="http://www.cnet.com/news/miracast-a-near-perfect-wireless-streaming-solution/">
                 Miracast
                </a>
                receiver.
               </li>
               <li>
                <strong>
                 Minimizing login prompts for SharePoint Online
                </strong>
                - reduces the number of prompts with federated use in accessing
                <a data-component="externalLink" href="http://office.microsoft.com/en-us/sharepoint/microsoft-sharepoint-online-collaboration-and-management-software-FX103789366.aspx" target="_blank">
                 SharePoint Online
                </a>
                sites. If you select the "Keep me signed in" check box when you log on for the first time, you will not see prompts for successive access to that SharePoint Online site.
               </li>
              </ul>
              <p>
               There may be more in store since LeBlanc calls these "some of the new features and improvement." But aside from the touchpad improvements, there's not much here to excite the average Windows 8.1 user. The August update will also roll out to Windows Server 2012 R2, which should be of interest to IT professionals.
              </p>
              <p>
               For current users of Windows 8.1 who have automatic updates turned on, the rollout will be gradual, meaning it won't automatically install for everyone starting August 12. But you can still install the update manually by selecting it from the Details pane in the Windows Update screen in 8.1 and clicking the Install button.
              </p>
              <p>
               Since the update is considered minor, it won't be dubbed "Windows 8.1 Update 2." LeBlanc didn't reveal an actual name in his blog. So perhaps calling it the Windows 8.1 August Update is probably the safest bet at this point.
              </p>
