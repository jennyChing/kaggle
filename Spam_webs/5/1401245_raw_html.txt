    <h1>
     Open Source Software projects at DRK
    </h1>
    <p>
     This is our list of software projects for Windows and other operating systems:
    </p>
    <ol class="project_list">
     <li>
      <a href="/spider.php">
       DRKSpider (GNU GPL)
      </a>
      - This website crawler will navigate a web site from a given URL, checking internal (and external) links, style sheets, images and other files. Broken links will be reported and the sitemap displayed in a tree control. You will be able to export the sitemap to a plain text file for submitting to Google, Yahoo and other search engines. DRKSpider is a free open source link checker and web site crawler.
     </li>
     <li>
      <a href="/daphne.php">
       Daphne (GNU GPL)
      </a>
      - Is a small (system tray) application for killing, controlling and debugging Windows' processes. It was born to kill a windows process and became almost a task manager repalcement. You can kill a process by dragging the mouse over the windows, by right-clicking the process in the main process list, or by typing its name with the "Kill all by name" command. You can set a any window to be always on top, to be transparent, to be enable, et cetera.
     </li>
     <li>
      <a href="/builder.php">
       TCP/IP Builder (GNU GPL)
      </a>
      - Is a Windows Socket testing tool. You will be able to create a socket manually, connect it (or listen for connections), send and receive data through it. You may setup local IP and Port, TCP/UDP protocol, keep alive, out-of-band data, DNS Resolver, System info, on-line help, RAW Packet Detail.
     </li>
     <li>
      <a href="/findmacs-discover-mac-addresses.php">
       findMACs (GNU GPL)
      </a>
      - Is a small Linux tool for querying every IP in a given range and print received MAC address to standard output.
     </li>
     <li>
      <a href="/drkap-webcam-usb-capture-windows.php">
       DRKap (GNU GPL)
      </a>
      - Is a minimalistic Windows tool for saving a frame in JPEG format, from a connected webcam.
     </li>
     <li>
      <a href="/coins.php">
       DRKCoins catalog
      </a>
      - Is web based application for maintaining personal coins collections.
     </li>
     <li>
      <a href="/watchman.php">
       WatchMan (GNU GPL)
      </a>
      - Is a process killer daemon. Is intented for killing process which may become a problem due to several reasons, like taking too much time in user or kernel mode, using too much CPU or memory, etc.
     </li>
    </ol>
