        <h2>
         <a href="http://www.howtogeek.com/howto/windows-vista/fix-for-outlook-2007-constantly-asking-for-password-on-vista/" rel="bookmark" title="Fix For Outlook 2007 Constantly Asking for Password on Vista">
          Fix For Outlook 2007 Constantly Asking for Password on Vista
         </a>
        </h2>
        <div class="thecontent">
         <!-- google_ad_section_start -->
         <p>
          If you are encountering an infuriating problem where Outlook constantly asks you for your password even though you check the Remember my password box every single time, then you are in luck because reader Malcolm wrote in with this tip on how to fix the problem.
         </p>
         <p>
          This fix should theoretically help with Windows Mail as well, but I havent been able to verify that yet.
         </p>
         <p>
          <img alt="image" height="336" src="http://cdn3.howtogeek.com/wp-content/uploads/2007/11/image93.png" width="372"/>
         </p>
         <p>
          Before you do anything else, close Outlook. You should also note that well be modifying system generated files here, so proceed with caution or not at all.
         </p>
         <p>
          Open up windows explorer window and then paste the following text into the address bar, which should bring you to a folder with a single folder inside with a really long name.
         </p>
         <blockquote>
          <p>
           %userprofile%\AppData\Roaming\Microsoft\Protect
          </p>
         </blockquote>
         <p>
          <img alt="image" height="280" src="http://cdn3.howtogeek.com/wp-content/uploads/2007/11/image94.png" width="556"/>
         </p>
         <p>
          What youll need to do is just rename that folder to something else (I suggest appending -old onto the end of it so that you could easily rename it back if things go wrong)
         </p>
         <p>
          <img alt="image" height="119" src="http://cdn3.howtogeek.com/wp-content/uploads/2007/11/image95.png" width="405"/>
         </p>
         <p>
          Now re-open Outlook, and then type in your password hopefully for the last time, making sure to check the box to remember it. You should now see that the original folder was re-created again.
         </p>
         <p>
          <img alt="image" height="260" src="http://cdn3.howtogeek.com/wp-content/uploads/2007/11/image96.png" width="515"/>
         </p>
         <p>
          At this point Outlook should remember your passwords, but you can close Outlook and reopen it to make sure. If you encounter problems with this, you can remove the new folder and then rename the -old folder back to the original name to put everything back the way it was.
         </p>
         <p>
          Note: Im hearing from some readers that this doesnt always solve the problem for them, so your mileage may vary.
         </p>

              The pesky and frequent dialogue box which wants you to check it is coming from a server timeout.
             </p>
             <p>
              Now set your send/receive times to at least 4 minutes and youll never see that pest again.
             </p>
             <p>
              James, MCSE/MCSA
