                    <meta content="http://3.bp.blogspot.com/-UTH7q0rgy58/U2-iPJvNwvI/AAAAAAAACkk/mWjh6EJRUz4/s1600/IMG_0710.jpg" itemprop="image_url"/>

                    <h3 class="post-title entry-title" itemprop="name">
                     I just got my BeagleBone Black, now what?
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-168808017293889208" itemprop="description articleBody">
                     <div class="MsoNormal">
                      When I received my BeagleBone, I wanted to start build my
Robot right away but I quickly realized that I needed to start off by setting
up my BeagleBone and configuring the Linux Distribution.
                      <span style="mso-spacerun: yes;">
                      </span>
                      You can install several different
distributions on your BeagleBone but I would recommend starting with
                      <a href="http://beagleboard.org/latest-images">
                       standardAngstrom distribution
                      </a>
                      .
                      <span style="mso-spacerun: yes;">
                      </span>
                      Once you are more familiar with your
BeagleBone then you can try other distributions.
                      <span style="mso-spacerun: yes;">
                      </span>
                      When you first login into the standard
Angstrom distribution, you will login as the root user with no password.
                      <o:p>
                      </o:p>
                     </div>
                     <div class="MsoNormal">
                      <br/>
                     </div>
                     <div class="MsoNormal">
                      A great place to start is
                      <a href="http://beagleboard.org/Getting+Started">
                       BeagleBones Getting started page
                      </a>
                      .
                      <span style="mso-spacerun: yes;">
                      </span>
                      This page walks you though the initial setup, installation of the
drivers on your local computer and flashing the latest software onto your
BeagleBone.
                      <span style="mso-spacerun: yes;">
                      </span>
                      This page documents these
steps very well so I will not rehash them here but I did have one problem when
I went though this page.
                      <span style="mso-spacerun: yes;">
                      </span>
                      The problem
was I have an Ubuntu laptop and a MacBook Pro but no Windows machine to write
the image to the SD card as documented on the getting started page.
                      <span style="mso-spacerun: yes;">
                      </span>
                      After a brief Google search, I found
                      <a href="https://learn.adafruit.com/beaglebone-black-installing-operating-systems/mac-os-x">
                       Adafruitspage
                      </a>
                      that walked me though using PiFiller to write the image to a SD card.
                      <span style="mso-spacerun: yes;">
                      </span>
                      The total time for this setup was
approximately two and a half hours for me so patience is needed.
                      <span style="mso-spacerun: yes;">
                      </span>
                      <o:p>
                      </o:p>
                     </div>
                     <div class="MsoNormal">
                      <br/>
                     </div>
                     <div class="MsoNormal">
                      I connected my BeagleBone black to an old monitor I had
lying around the house by using a micro HDMI to VGA converter.
                      <span style="mso-spacerun: yes;">
                      </span>
                      I also bought a mini wireless keyboard with a
trackpad so I was able to program my BeagleBone like a desktop computer.
                      <span style="mso-spacerun: yes;">
                      </span>
                      While this is not necessary, it makes
learning the BeagleBone a lot easer.
                      <span style="mso-spacerun: yes;">
                      </span>
                      I
used
                      <a href="http://www.amazon.com/gp/product/B00879EZJI/ref=as_li_tl?ie=UTF8&amp;camp=1789&amp;creative=9325&amp;creativeASIN=B00879EZJI&amp;linkCode=as2&amp;tag=myadveinrobo-20&amp;linkId=36WUY6TKDXH62RKH">
                       Cable Matters Active Micro HDMI to VGA Male to Female Adapter with 3 Feet USB Power Cable in Black
                      </a>
