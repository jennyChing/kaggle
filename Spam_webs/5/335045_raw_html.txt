            How to set password in any file by using winrar at Windows 7 &amp; Windows 8
           </h1>

               <img alt="" class="alignnone size-medium wp-image-1096" height="154" src="http://www.problemsolutions24.com/wp-content/uploads/2013/12/How-to-lock-files-and-folder-with-password2-300x225.jpg" width="328"/>

             <p>
              Here is given the way to set password in any file at Windows 7 &amp; Windows 8. To do these follow the steps:
             </p>
             <p>
              Step-1: Click right button by keeping cursor on file or files which you want to set password.
             </p>
             <p>
              <a href="http://www.problemsolutions24.com/how-to-set-password-in-any-file-by-using-winrar-at-windows-7-windows-8/screenshot1/" rel="attachment wp-att-1087">
               <img alt="" class="alignnone size-medium wp-image-1087" height="254" src="http://www.problemsolutions24.com/wp-content/uploads/2013/12/Screenshot1-300x254.png" width="300"/>
              </a>
             </p>
             <p>
              Step 2: Click on Add to archive.
             </p>
             <p>
              Step 3: Select Rar option from format and then click on set password option.
             </p>
             <p>
              <a href="http://www.problemsolutions24.com/how-to-set-password-in-any-file-by-using-winrar-at-windows-7-windows-8/screenshot2/" rel="attachment wp-att-1088">
               <img alt="" class="alignnone size-medium wp-image-1088" height="300" src="http://www.problemsolutions24.com/wp-content/uploads/2013/12/Screenshot2-298x300.png" width="298"/>
              </a>
             </p>
             <p>
              Step 4: Give password for two times to set password.
             </p>
             <p>
              <a href="http://www.problemsolutions24.com/how-to-set-password-in-any-file-by-using-winrar-at-windows-7-windows-8/screenshot3/" rel="attachment wp-att-1090">
               <img alt="" class="alignnone size-medium wp-image-1090" height="197" src="http://www.problemsolutions24.com/wp-content/uploads/2013/12/Screenshot3-300x197.png" width="300"/>
              </a>
             </p>
             <p>
              Step 5: Press Ok
             </p>
             <p>
              Your password is set. If you want to open file then it will require password. After providing correct password your file will open.
             </p>
             <p>
              <a href="http://www.problemsolutions24.com/how-to-set-password-in-any-file-by-using-winrar-at-windows-7-windows-8/screenshot4/" rel="attachment wp-att-1089">
