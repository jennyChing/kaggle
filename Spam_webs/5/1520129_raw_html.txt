     <div class="body">
      Looking for a creative way to cook? All the burners on your stove taken up with preparations for tomorrow's gigantic dinner party? You can use your dishwasher to cook dinner. Dishwasher catfish to the rescue!
      <br/>
      <br/>
      <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
       <table align="center" border="0" class="image center" width="500">
        <tbody>
         <tr>
          <td>
           <a href="http://blogs.ocweekly.com/stickaforkinit/catfish-after.jpg" onclick="window.open('http://blogs.ocweekly.com/stickaforkinit/catfish-after.jpg','popup','width=1000,height=750,scrollbars=no,resizable=no,toolbar=no,directories=no,location=no,menubar=no,status=no,left=0,top=0'); return false">
            <img alt="catfish-after.jpg" height="375" src="http://blogs.ocweekly.com/stickaforkinit/assets_c/2010/03/catfish-after-thumb-500x375.jpg" width="500"/>
           </a>
          </td>
         </tr>
         <tr>
          <td class="credit">
           Dave Lieberman
          </td>
         </tr>
        </tbody>
       </table>
       
      </span>
      <a name="more">
      </a>
      <br/>
      You'll need the following:
      <br/>
      <br/>
      1 pound of catfish fillet or catfish "nuggets", thawed if frozen
      <br/>
      1 lemon (try it with a Meyer lemon while they're in season!)
      <br/>
      1 lime
      <br/>
      3 cloves garlic
      <br/>
      1 inch ginger
      <br/>
      1 chile (for less heat, use a jalapeo; for more, use a Thai bird chile)
      <br/>
      1 Tbsp. soy sauce
      <br/>
      1/2 bunch fresh cilantro
      <br/>
      Salt and pepper
      <br/>
      Heavy duty aluminum foil
      <br/>
      <br/>
      1. Cut the catfish into four pieces (if you're using frozen catfish nuggets, use four), then salt and pepper both sides.
      <br/>
      <br/>
      2. Slice the lemon, lime, garlic and ginger thinly.
      <br/>
      <br/>
      3. Cut the chile in half (carefully).
      <br/>
      <br/>
      4. Lay out a big piece of foil, double it over itself, then flute the edges to create a small rim.
      <br/>
      <br/>
      5. In the middle of the foil, lay out half the lemon slices, half
the lime slices, half the garlic, half the ginger, half the cilantro
(with the stems pointing in both directions) and one chile half.
      <br/>
      <br/>
      6. Lay the catfish in one layer on top of the produce.
      <br/>
      <br/>
      7. Layer the remaining produce on, then pour the soy sauce over the whole thing.
      <br/>
      <br/>
      8.Wrap the foil packet by folding the top down, then folding the
sides in. Make sure there are no leaks (turn it over and over and make
sure no soy sauce drips out).
      <br/>
      <br/>
      9. Put the packet in the rack closest to the heat source, close the
dishwasher, and run it on its hottest wash and the heated dry. You want
the heated dry to run for at least 45 minutes; an hour would be better.
      <br/>
      <br/>
      10. The fish should flake when done, but still be moist.
      <br/>
      <br/>
      If you just can't hack the idea of using your dishwasher to cook, you
can do this in the oven set to 250F. Also, it shouldn't be necessary
to say this, but you do clean your dishwasher, right? I mean, it
doesn't just get clean by running with dirty dishes in it.
      <a href="http://www.channel4.com/4homes/on-tv/how-clean-is-your-house/">
       Kim and
Aggie
      </a>
      would be so ashamed.
