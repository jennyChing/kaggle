     <p>
      The
      <b>
       KDE Education Project
      </b>
      (or KDE-Edu project) develops
      <a href="/wiki/Free_software" title="Free software">
       free
      </a>
      <a href="/wiki/Educational_software" title="Educational software">
       educational software
      </a>
      based on the
      <a class="mw-redirect" href="/wiki/KDE_Platform" title="KDE Platform">
       KDE technologies
      </a>
      for students and parents. These educational software is translated into more than 65 languages, so that users can access them without any problems. The KDE-Edu project also provides free software educational to support and facilitate teachers in planning lessons.
     </p>
     <p>
      The KDE-Edu project is available for Unix, BSD and Linux, currently can be used in Microsoft Windows, but this is something that is in beta.
     </p>

     <h2>
      <span class="mw-headline" id="History">
       History
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=KDE_Education_Project&amp;action=edit&amp;section=1" title="Edit section: History">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      The KDE-Edu project was started in July 2001. The goal of the project is to develop Free Educational Software (
      <a href="/wiki/GNU_General_Public_License" title="GNU General Public License">
       GNU General Public License
      </a>
      ) within the KDE environment. This software is mainly aimed to schools, to students and to parents at home as well as to adults willing to extend their knowledge.
      <sup class="reference" id="cite_ref-2">
       <a href="#cite_note-2">
        <span>
         [
        </span>
        2
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>

      <span class="mw-headline" id="List_of_software">

     <ul>
      <li>
       <a href="/wiki/Kalzium" title="Kalzium">
        Kalzium
       </a>
       - Visual representation of the
       <a href="/wiki/Periodic_table" title="Periodic table">
        periodic table of elements
       </a>
       and includes basic information about all common elements.
       <sup class="reference" id="cite_ref-19">
        <a href="#cite_note-19">
         <span>
          [
         </span>
         19
         <span>
          ]
         </span>
        </a>
       </sup>
      </li>
      <li>
       <a href="/wiki/KStars" title="KStars">
        KStars
       </a>
       - Desktop
       <a href="/wiki/Planetarium" title="Planetarium">
        planetarium
       </a>
       program for
       <a href="/wiki/KDE" title="KDE">
        KDE
       </a>
       providing an accurate graphical simulation of the night sky, from any location on Earth, at any date and time.
       <sup class="reference" id="cite_ref-20">
        <a href="#cite_note-20">
         <span>
          [
         </span>
         20
         <span>
          ]
         </span>
        </a>
       </sup>
      </li>
      <li>
       <a class="mw-redirect" href="/wiki/Marble_(KDE)" title="Marble (KDE)">
        Marble
       </a>
       - Virtual globe and world atlas that you can use to learn more about the Earth.
       <sup class="reference" id="cite_ref-21">
        <a href="#cite_note-21">
         <span>
          [
         </span>
         21
         <span>
          ]
         </span>
        </a>
       </sup>
      </li>
      <li>
       <a class="mw-redirect" href="/wiki/Step_(KDE)" title="Step (KDE)">
        Step
       </a>
       - An interactive physics simulator, allowing you to explore the physical world through simulations.
       <sup class="reference" id="cite_ref-22">
        <a href="#cite_note-22">
         <span>
          [
         </span>
         22
         <span>
          ]
         </span>
        </a>
       </sup>
      </li>
      <li>
       <a class="mw-redirect" href="/wiki/Rocs" title="Rocs">
        Rocs
       </a>
       - Graph Theory IDE for designing and analyzing graphs and algorithms with an easy to use visual data structure editor and a powerful scripting engine to execute algorithms.
       <sup class="reference" id="cite_ref-23">
        <a href="#cite_note-23">
         <span>

         </span>
         23
         <span>
