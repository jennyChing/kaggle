       </a>
      </header>
      <p>
       Userstyle created to turn my Firefox theme FT DeepDark's unselected browser tabs + tab bar background (when tabs are on top) translucent. It also turns the selected tab font color into a blue one to improve highlight towards translucent non selected tabs (this color can be changed by editing the style in the Firefox add-ons manager).  This style should be used with Firefox on Windows 7 or Vista with aero style enabled. It is also recommanded to use the latest version of FT DeepDark if you want it to look like on the screenshot (by the way the window frame color has to be set in Windows appearance options, this userstyle has no influence on it).
