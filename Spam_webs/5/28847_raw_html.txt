       <h1 class="entry-title">
        See How Easily You Can Increase Page Load Time 10 Fold With jQuery
       </h1>
       <div class="entry-content">
        <div style="float:right;margin:10px 0 10px 10px;">
         <script type="text/javascript">

        <h3>
         Introduction
        </h3>
        <p>
         Its no new news to us that images hurt a pages load time worst then anything out there. There are a lot of things you can do to try and alleviate some of the problem. You can use smaler images, you can reduce the quality on images, etc.
        </p>
        <p>
         But consider this. You know that your website is going to a long one. Meaning that the user is going to have to scroll. This means that they dont see the entire page when they first load. Wouldnt it be nice to only load what they need? You could load only the images that are in view. This will give the user time to read the content about the fold on your page, then as he scrolls down, you can gradually load the rest of the images.
        </p>
        <p>
         This is referred to as Lazy Loading. Its exactly the opposite of preloading images. We preload images for a lot of reasons. The classic example is in those old school JavaScript (and even CSS) rollovers, where we preload the rollover image so that there is no delay when they are needed.
        </p>
        <h3>
         Enter Lazy Load Plugin for jQuery
        </h3>
        <p>
         <img align="left" data-recalc-dims="1" src="http://i1.wp.com/www.appelsiini.net/images/jquery.gif"/>
         Lazyloader is inspired by
         <a href="http://developer.yahoo.com/yui/imageloader/">
          <span class="caps">
           YUI
          </span>
          ImageLoader
         </a>
         Utility by Matt Mlinac. It simply delays the loading of images on the web page until they are within view. This gives the page quicker initial load time.
        </p>
        <p>
         This works out great if you have a page with a lot of heavy images lower down and a lot of navigation links at the top. If the user is trying to get to a specific page using this plugin would be a great help.
        </p>
        <p>
         <span id="more-218">
         </span>
        </p>
        <h3>
         What You Need For Lazy Loading
        </h3>
        <ul>
         <li>
          The fabulous jQuery of course.
         </li>
         <li>
          The jQuery
          <a href="http://brandonaaron.net/docs/dimensions/">
           Dimensions
          </a>
          plugin.
         </li>
         <li>
          The
          <a href="http://www.appelsiini.net/projects/lazyload">
           Lazy Load
          </a>
          Plugin
         </li>
        </ul>
        <h3>
         Using Lazy Load
        </h3>
        <p>
         First thing, as with any jQuery script is to load the libraries in the
         <em>
          head
         </em>
         of your HTML file.
        </p>
        <pre>
         &lt;script src="jquery.js" type="text/javascript"&gt;&lt;/script&gt;
        </pre>
        <pre>
         &lt;script src="jquery.dimensions.js" type="text/javascript"&gt;&lt;/script&gt;
        </pre>
        <pre>
         &lt;script src="jquery.lazyload.js" type="text/javascript"&gt;&lt;/script&gt;
        </pre>
        <p>
         The we make one simple call to the initialize the plugin:
        </p>
        <pre>
         <code>
          $("img").lazyload();
         </code>
        </pre>
        <p>
         We can even go one step further and add a little bit of padding to when the image loads:
        </p>
        <pre>
         <code>
          $("img").lazyload({ threshold : 200 });
         </code>
        </pre>
        <p>
         This says that we load images that are 200 pixels from the page load. With this setting the images are still loaded before the user gets to them.
        </p>
        <p>
         Check out the
         <a href="http://www.appelsiini.net/projects/lazyload">
          Lazy Load
         </a>
         plugin, its great work. Check out the
         <a href="http://www.appelsiini.net/projects/lazyload/enabled.html">
          Demo
         </a>
         also.
        </p>
