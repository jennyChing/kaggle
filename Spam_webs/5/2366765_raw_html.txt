       CropPro to supervise crop productivity
      </h1>
      <div class="entry-meta">
       Posted about

        10 months ago
       </abbr>
       <span class="entry-sep">

      <div class="entry entry-content">
       <p class="qowt-stl0" id="E71" style="color: #000000;">
        <strong>
         <span id="E72">
          Grains Research and Development Corporation (GRDC) has brought out an internet-based tool called CropPro to aid agronomists in handling cropping restraints.
         </span>
        </strong>
       </p>
       <p class="qowt-stl0" id="E74" style="color: #000000;">
        CropPro delivers diagnostic and economic tools for efficient identification and management of constraints to crop productivity, saidVictorian Department of Environment and Primary Industries (DEPI) Project Leader Chris Pittock.
       </p>
       <p class="qowt-stl0" id="E81" style="color: #000000;">
        <span id="E82">
         The three core functions of CropPro are to diagnose the cause of crop problems, support risk analysis and to provide evidence-based information for management of crop constraints.
        </span>
       </p>
       <div class="wp-caption alignright" id="attachment_4697" style="width: 310px">
        <a href="http://australianresources.com.au/wp-content/uploads/2014/09/croppro.jpg">
         <img alt="Image source: croppro.com.au" class="wp-image-4697 size-medium" height="198" src="http://australianresources.com.au/wp-content/uploads/2014/09/croppro-300x198.jpg" width="300"/>
        </a>
        <p class="wp-caption-text">
         Image source: croppro.com.au
        </p>
       </div>
       <p class="qowt-stl0" id="E84" style="color: #000000;">
        <span id="E85">
         <a href="http://www.croppro.com.au/">
          CropPro
         </a>
         tool provides several components such as explanatory videos elaborating concepts toward dealing with future dangers regarding cost hazards and economic circumstances.
        </span>
       </p>
       <p class="qowt-stl0" id="E87" style="color: #000000;">
        <span id="E88">
         Each constraint has an information page that provides short-form summary information and links to key resources on identification, cause and management of the constraint. he added.
        </span>
       </p>
       <p class="qowt-stl0" style="color: #000000;">
        Queensland Country Life has
        <a href="http://www.queenslandcountrylife.com.au/news/agriculture/cropping/general-news/crop-diagnosis-a-click-away/2710524.aspx?src=rss">
         enumerated
        </a>
        several CropPro functions.
       </p>
       <p class="qowt-stl0" id="E90" style="color: #000000;">
        <span id="E91">
         The explore functionoffers Crop Disease Manual and a bundle of review articles. It is extended with a search option to present recent, evidence-based credible data.
        </span>
       </p>
       <p class="qowt-stl0" id="E93" style="color: #000000;">
        <span id="E94">
         Through the options section, economic considerations and constraint management can be viewed.
        </span>
       </p>
       <p class="qowt-stl0" id="E96" style="color: #000000;">
        <span id="E97">
         On the other hand, there is a do nothing option available for any management actions tested by users.
        </span>
       </p>
       <p class="qowt-stl0" id="E99" style="color: #000000;">
        <span id="E100">
         CropPro was launched last month to the southern cropping region and is now available at
         <a href="http://www.croppro.com.au/">
          www.
         </a>
        </span>
        <a href="http://www.croppro.com.au/">
         <span id="E101">
          croppro.com.au.
         </span>
        </a>
       </p>

                if (i.toString() == parts[0]){

                    index = parts[0];

                    msg = parts[1];

                } else {

                    index = -1;

                    msg = resp.msg;

                }

            }

        } catch(e){

            index = -1;

            msg = resp.msg;

        }

        try{

            if (index== -1){

                $('#mce-'+resp.result+'-response').show();

                $('#mce-'+resp.result+'-response').html(msg);            

            } else {

                err_id = 'mce_tmp_error_msg';

                html = '&lt;div id="'+err_id+'" style="'+err_style+'"&gt; '+msg+'&lt;/div&gt;';

                

                var input_id = '#mc_embed_signup';

                var f = $(input_id);

                if (ftypes[index]=='address'){

                    input_id = '#mce-'+fnames[index]+'-addr1';

                    f = $(input_id).parent().parent().get(0);

                } else if (ftypes[index]=='date'){

                    input_id = '#mce-'+fnames[index]+'-month';

                    f = $(input_id).parent().parent().get(0);

                } else {

                    input_id = '#mce-'+fnames[index];

                    f = $().parent(input_id).get(0);

                }

                if (f){

                    $(f).append(html);

                    $(input_id).focus();

                } else {

                    $('#mce-'+resp.result+'-response').show();

                    $('#mce-'+resp.result+'-response').html(msg);

                }

            }

        } catch(e){

            $('#mce-'+resp.result+'-response').show();

            $('#mce-'+resp.result+'-response').html(msg);

        }

    }
