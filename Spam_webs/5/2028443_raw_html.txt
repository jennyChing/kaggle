       <h1 class="post-title entry-title">
        How to Convert Movie Scenes into Animated GIFs
       </h1>
       <p class="post-excerpt">
        This video tutorial describes how you can easily convert videos into animated GIF images using free tools.

        <p>
         Animated GIF images may remind you of those cluttered and distracting homepages of the
         <a href="http://www.labnol.org/internet/geocities-website-screenshot/18163/">
          Geocities era
         </a>
         but take a peek at these
         <a href="http://www.labnol.org/tech/gif-cinemagraphs/19203/">
          cinemagraphs
         </a>
         and it might forever change the way you think of GIFs. These are animated GIFs created using frames of classic movie scenes that play in an endless loop.
        </p>

        <h2>
         How to Create GIF Animations?
        </h2>
        <p>
         There are free tools available that let you easily convert video clips into animated GIFs that you can later host on sites like Flickr or Tumblr  not all image hosting sites support animated GIFs but the one mentioned here do.
        </p>
        <p>
         This is a two-step process  first convert the video into a sequence of image frames and then assemble these frames into a looping GIF animation with some custom delay.
        </p>
        <p>
         <strong>
          Step 1:
         </strong>
         Download
         <a href="http://www.labnol.org/internet/useful-ffmpeg-commands/28490/">
          FFmpeg
         </a>
         and extract the ffmpeg.exe file to, say, the c:\ffmpeg\ folder. The .7z files can be unzipped using the 7-zip utility.
        </p>
        <p>
         <strong>
          Step 2:
         </strong>
         Now well extract the frames from our video file. The command* is simple.
        </p>
        <pre>
         c:\ffmpeg\ffmpeg.exe -i movie.avi img-%03d.gif
        </pre>
        <p>
         If your video file is in another folder, you need to specify the full path of that file. Also, you may use MP4, MOV or any video format as well instead of AVI.
        </p>
        <p>
         <strong>
          Step 3:
         </strong>
         Grab this portable utility called
         <a href="http://www.whitsoftdev.com/unfreez/" target="_blank">
          UnFreez
         </a>
         and drag-n-drop all the image frames created in the previous step from Windows Explorer into the UnFreez window.
        </p>
        <p>
         Enable loop animation and set the frame delay to say 20 cs (1 cs = 0.01 second). Click Make animated GIF and preview your image is in any browser.
        </p>
        <p>
         Tutorial:
         <a href="http://www.labnol.org/software/cinemagraph-tutorial/19453/">
          Make Cinemagraphs with Photoshop
         </a>
        </p>
        <p>
         <strong>
          Screencast: Converting videos into GIF movies
         </strong>
        </p>
        <p>
        </p>
        <div class="youtube" data-id="_QC1qYComNg" id="videoPlayer">
         <a href="https://youtu.be/_QC1qYComNg" onclick='playVideo("_QC1qYComNg"); return false;' target="_blank">

        [*] If you have a fairly long video file and would only like to convert a portion of it into an image, you need to slightly modify the ffmpeg parameters as shown below:
        <pre>
         c:\ffmpeg\ffmpeg.exe -i movie.avi   ss 5   t 3 img-%03d.gif
        </pre>
        <p>
         The parameter ss refers to the start position in the video while t is the duration for which you want to
         <a href="http://www.labnol.org/software/extract-image-frames-from-video/18137/">
          extract the frames
         </a>
         . So in the above example, image frames for the file movie.avi will be at created from the 5 second mark for the next 3 seconds on the video.
        </p>
        <p>
         If you would like to
         <a href="http://labnol.blogspot.com/2007/08/how-to-stop-gif-image-animations-from.html" target="_blank">
          stop GIFs from animating
         </a>
         in the browser, just hit the Escape key.
        </p>
