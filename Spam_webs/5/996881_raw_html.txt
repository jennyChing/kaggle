             <h3 class="post-title entry-title" itemprop="name">
              Crosman Phantom 500 Disassembly guide
             </h3>
             <div class="post-header">
              <div class="post-header-line-1">
              </div>
             </div>
             <div class="post-body entry-content" id="post-body-7366651290243609605" itemprop="description articleBody">
              <div align="center">
               <a href="http://4.bp.blogspot.com/_pMuK8ZC6R5w/SB1BizM5GUI/AAAAAAAAAAM/_FIEX_fX0vU/s1600-h/IMG_0001.jpg">
                <img alt="" border="0" id="BLOGGER_PHOTO_ID_5196381611171060034" src="http://4.bp.blogspot.com/_pMuK8ZC6R5w/SB1BizM5GUI/AAAAAAAAAAM/_FIEX_fX0vU/s320/IMG_0001.jpg" style="DISPLAY: block; MARGIN: 0px auto 10px; CURSOR: hand; TEXT-ALIGN: center"/>

              <div align="left">
               Hello
               <br/>
               Id like to start by stating that this was my first attempt at tuning an air rifle. I used a combination of guides found online from various sources including Charlie da tuna and information from several air gun discussion groups. I decided to make a guide
               <em>
                specifically
               </em>
               for the Crosman Phantom 500 .22 cal air rifle since there were none to be found at this time.
              </div>
              <div align="center">
               <br/>
               <br/>
               <br/>
               <br/>
              </div>
              <div align="left">
               <em>
                Disclaimer: I am not responsible for any damage done to you or your air rifle thru the use of this document. It is strictly a guide. Your results will vary depending on your own abilities.
                <br/>
                That said if you take it slow and work methodically you should be able to achieve successful results.
               </em>
              </div>
              <div align="center">
               <br/>
               <br/>
               <br/>
               <br/>
              </div>
              <div align="left">
               First things first: Make sure the rifle is uncocked! The safety should be on as well.
               <br/>
               Now onto the tear down.
               <br/>
               1: Remove the 2 forward stock screws and washers.
               <br/>
               2: Remove the 1 screw behind the trigger.
               <br/>
               3: Remove the receiver from the stock.
               <br/>
              </div>
              <div align="center">
               <br/>
               <br/>
               <br/>
              </div>
              <div align="left">
               <a href="http://1.bp.blogspot.com/_pMuK8ZC6R5w/SDoMlqFGmnI/AAAAAAAAADE/ugTe50CXdrQ/s1600-h/IMG_0007.JPG">
                <img alt="" border="0" id="BLOGGER_PHOTO_ID_5204486160470415986" src="http://1.bp.blogspot.com/_pMuK8ZC6R5w/SDoMlqFGmnI/AAAAAAAAADE/ugTe50CXdrQ/s400/IMG_0007.JPG" style="CURSOR: hand"/>
               </a>
               <br/>
               <a href="http://3.bp.blogspot.com/_pMuK8ZC6R5w/SB1C2jM5GVI/AAAAAAAAAAU/tlEbGJkpxo4/s1600-h/img_0007.jpg">
               </a>
               <br/>
               <br/>
               4: Break the barrel free but do not cock it! Remove the barrel block pivot bolt to separate it from the receiver forks.
              </div>
              <div align="center">
               <br/>
               <br/>
               <br/>
              </div>
              <p align="left">
               <a href="http://1.bp.blogspot.com/_pMuK8ZC6R5w/SDoN6qFGmpI/AAAAAAAAADU/weixqAYGONc/s1600-h/IMG_0013.JPG">
                <img alt="" border="0" id="BLOGGER_PHOTO_ID_5204487620759296658" src="http://1.bp.blogspot.com/_pMuK8ZC6R5w/SDoN6qFGmpI/AAAAAAAAADU/weixqAYGONc/s400/IMG_0013.JPG" style="CURSOR: hand"/>
               </a>
              </p>
              <br/>
              <br/>
              <div align="center">
              </div>
              <br/>
              <br/>
              <div align="left">
               5:Separate the sliding block from away from the receiver.
               <br/>
               <a href="http://4.bp.blogspot.com/_pMuK8ZC6R5w/SDoO9aFGmrI/AAAAAAAAADk/CHfyUG-5QCo/s1600-h/IMG_0018.JPG">
                <img alt="" border="0" id="BLOGGER_PHOTO_ID_5204488767515564722" src="http://4.bp.blogspot.com/_pMuK8ZC6R5w/SDoO9aFGmrI/AAAAAAAAADk/CHfyUG-5QCo/s400/IMG_0018.JPG" style="FLOAT: left; MARGIN: 0px 10px 10px 0px; CURSOR: hand"/>
               </a>
               <br/>
               <br/>
               <br/>
              </div>
              <br/>
              <br/>
              <div align="left">
              </div>
              <br/>
              <br/>
              <div align="center">
               <br/>
               <br/>
               <br/>
               <br/>
              </div>
              <br/>
              <br/>
              <div align="left">
              </div>
              <br/>
              <br/>
              <div align="center">
               <br/>
               <br/>
               <br/>
               <br/>
              </div>
              <br/>
              <br/>
              <div align="left">
               <br/>
              </div>
              <br/>
              <br/>
              <div align="left">
               <br/>
               6: Next you have to unhook the sliding block spring from the trigger block. Unhook it at the front leaving it hooked at the back on the trigger block. You can now remove the e clip as well.
              </div>
              <br/>
              <br/>
              <div align="center">
               <br/>
               <a href="http://1.bp.blogspot.com/_pMuK8ZC6R5w/SDoPWqFGmsI/AAAAAAAAADs/hjsA_A3tr_I/s1600-h/triggerblock.jpg">
                <img alt="" border="0" id="BLOGGER_PHOTO_ID_5204489201307261634" src="http://1.bp.blogspot.com/_pMuK8ZC6R5w/SDoPWqFGmsI/AAAAAAAAADs/hjsA_A3tr_I/s400/triggerblock.jpg" style="FLOAT: left; MARGIN: 0px 10px 10px 0px; CURSOR: hand"/>
               </a>
               <br/>
               <br/>
               <br/>
              </div>
              <br/>
              <br/>
              <div align="left">
              </div>
              <br/>
              <br/>
              <div align="center">
               <br/>
               <br/>
               <br/>
               <br/>
              </div>
              <br/>
              <br/>
              <div align="left">
              </div>
              <br/>
              <br/>
              <div align="center">
               <br/>
               <br/>
               <br/>
               <br/>
              </div>
              <div align="left">
               7: Now remove the trigger block retention screw, I used a pair of vice grips.
               <br/>
              </div>
              <br/>
              <br/>
              <div align="left">
               <br/>
              </div>
              <a href="http://2.bp.blogspot.com/_pMuK8ZC6R5w/SDoQG6FGmtI/AAAAAAAAAD0/W2m-Y1PO3vM/s1600-h/IMG_0021.JPG">
               <img alt="" border="0" id="BLOGGER_PHOTO_ID_5204490030235949778" src="http://2.bp.blogspot.com/_pMuK8ZC6R5w/SDoQG6FGmtI/AAAAAAAAAD0/W2m-Y1PO3vM/s400/IMG_0021.JPG" style="FLOAT: left; MARGIN: 0px 10px 10px 0px; CURSOR: hand"/>
              </a>
              <br/>
              <br/>
              <div align="left">
               <a href="http://2.bp.blogspot.com/_pMuK8ZC6R5w/SB1EuTM5GZI/AAAAAAAAAA0/OHFPrSnSbYM/s1600-h/img_0021.jpg">
               </a>
              </div>
              <br/>
              <br/>
              <div align="center">
               <br/>
               <br/>
               <br/>
               <br/>
              </div>
              <br/>
              <br/>
              <div align="left">
              </div>
              <br/>
              <br/>
              <div align="center">
               <br/>
               <br/>
               <br/>
               <br/>
              </div>
              <br/>
              <br/>
              <div align="left">
              </div>
              <div align="left">
               8: Next comes the removal of the lock pin. When this is removed the spring will be released. This could pose a risk of injury. It is recommended that you use a spring compressor prior to completely punching out the lock pin. I did a really fast spring compressor for this job. Alternately for the disassembly part of the gun you can do this step without a compressor by butting the flat part of the end cap against a bench while pushing the receiver against it with force.
               <br/>
               Then punch out the lock pin with a wooden dowel or a screw driver and a mallet. Once the pin is out while still retaining pressure on the receiver, pull out the dowel or screwdriver now in the hole. Slowly back off the pressure as the spring pushes out the end cap. Voila!
               <br/>
              </div>
              <br/>
              <br/>
              <div align="left">
               <br/>
               <a href="http://1.bp.blogspot.com/_pMuK8ZC6R5w/SB1F3DM5GaI/AAAAAAAAAA8/ZuItFeLqPy4/s1600-h/img_0022.jpg">
                <img alt="" border="0" height="202" id="BLOGGER_PHOTO_ID_5196386357109922210" src="http://1.bp.blogspot.com/_pMuK8ZC6R5w/SB1F3DM5GaI/AAAAAAAAAA8/ZuItFeLqPy4/s320/img_0022.jpg" style="WIDTH: 394px; CURSOR: hand; HEIGHT: 258px" width="350"/>
               </a>
              </div>
              <br/>
              <br/>
              <div align="left">
               <br/>
               Similarly when putting it all back together you can try to do it the same way without a compressor just in reverse order. However putting it back is a little more difficult as you have to line it up right and put the lock pin back in. If you decide to put in a new stronger spring it will prove even more difficult without a compressor. It is your choice how you proceed.
               <br/>
               9:Now you can remove the rear spring guide followed by the spring.
              </div>
              <br/>
              <br/>
              <div align="left">
               10:Remove the trigger block from the receiver.
              </div>
              <br/>
              <div align="left">
               <br/>
              </div>
              <p align="left">
               <a href="http://2.bp.blogspot.com/_pMuK8ZC6R5w/SB1GSTM5GbI/AAAAAAAAABE/ZQbC8X7Uneo/s1600-h/img_0025.jpg">
                <img alt="" border="0" height="194" id="BLOGGER_PHOTO_ID_5196386825261357490" src="http://2.bp.blogspot.com/_pMuK8ZC6R5w/SB1GSTM5GbI/AAAAAAAAABE/ZQbC8X7Uneo/s320/img_0025.jpg" style="WIDTH: 314px; CURSOR: hand; HEIGHT: 194px" width="410"/>
               </a>
               <a href="http://3.bp.blogspot.com/_pMuK8ZC6R5w/SEuyz2gG2yI/AAAAAAAAAEo/LCMTpT8LZ1Q/s1600-h/phantom+internals.jpg">
                <img alt="" border="0" id="BLOGGER_PHOTO_ID_5209453997857364770" src="http://3.bp.blogspot.com/_pMuK8ZC6R5w/SEuyz2gG2yI/AAAAAAAAAEo/LCMTpT8LZ1Q/s400/phantom+internals.jpg" style="FLOAT: left; MARGIN: 0px 10px 10px 0px; CURSOR: hand"/>
               </a>
              </p>
              <p align="left">
              </p>
              <br/>
              <p align="left">
              </p>
              <p align="left">
              </p>
              <p align="left">
              </p>
              <p align="left">
              </p>
              <p align="left">
              </p>
              <p align="left">
              </p>
              <p align="left">
              </p>
              <p align="left">
               11:Take the tophat off then pull the piston out using a screwdriver thru the slot of the receiver. My tophat did not fall out on its own. Seems there was some corrosion keeping it stuck in the piston. I sanded her down smooth before reassembly.
               <em>
                (the above diagram is taken from a web source)
               </em>
              </p>
              <br/>
              <div align="left">
               Congratulations ! You have now disassembled your Crosman Phantom 500 ! To reassemble your air rifle is basically the reverse of what you have done. Take care when you put back your piston. Make sure it is lubed lightly with silicone oil or dielectric grease on the seal and moly grease on the outside of the piston. (Not the seal) Align the long slot of the piston with the long slot of the receiver. You can use a dowel to push the piston all the way into the receiver.
              </div>
              <br/>
              <br/>
              <div align="left">
               Now that you have successfully taken apart your air rifle you can do a tune. Which may include upgrades such as springs or piston change thats up to you. Tuning is another guide which you should be able to find online. A basic lube tune involves getting rid of the factory lubes some sanding etc. In my tune I replaced the 500 piston with a Phantom 1000 piston. I kept the original spring. I cleaned all the parts, sanded the spring ends and any rough spots I found on the receiver slots etc. Lubed it lightly with moly grease. My phantom is now a smooth firing powerhouse, and no factory twannnnggg!!
              </div>
              <br/>
              <div align="left">
              </div>
              <br/>
              <div align="left">
               I hope this guide has helped you Phantom owners.
               <br/>
               Happy airgunning!

               <dd class="comment-body" id="Blog1_cmt-4303975403831994791">
                <p>
                 If by better you mean velocity, the .177 will shoot faster. So if that's what you are asking the .177
                 <br/>
                 is the higher velocity gun. The .22 however will give you a higher energy impact due to the weight of the pellet.
                 <br/>
                 For example:
                 <br/>
                 a .22 cal pellet weighing 14.3g
                 <br/>
                 going at 700fps will produce 15.6 foot pounds energy or 21.10 joules.
                 <br/>
                 whereas a .177 cal pellet weighing
                 <br/>
                 7.9g going at 900fps will produce
                 <br/>
                 14.21 foot pounds energy or 19.27 joules.
                 <br/>
                 So as you can see in the above scenario even though the .22 is travelling at a slower speed of 700fps it produces more fpe than the .177 pellet travelling 900 fps.
                 <br/>
                 So which is better, it's all up to you to decide.
                 <br/>
                 I used to shoot strictly .177 but since I tried the .22 I really enjoy shooting that caliber better. There is nothing wrong with the .177 cal it is strictly my preference.
                 <br/>
                 Hope this answers your question.
                </p>
