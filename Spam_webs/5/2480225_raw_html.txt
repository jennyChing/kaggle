        <h2 id="xen-with-graphical-user-interface-on-a-fedora-desktop-">
         Xen With Graphical User Interface On A Fedora 7 Desktop
        </h2>
        <p>
         Version 1.0
         <br/>
         Author: Oliver Meyer &lt;o [dot] meyer [at] projektfarm [dot] de&gt;
         <br/>
         Last edited 09/17/2007
        </p>
        <p>
         This document describes how to set up Xen on Fedora 7. Xen enables the paravirtualization of your hardware for its virtual machines if you have a CPU with Vanderpool (Intel) or Pacifica (AMD) technology. The paravirtualization provides high performance to your virtual machines. Fedora's virt-manager provides an easy to use GUI for setting up and managing your virtual machines. It does not have the extensive features like VMware Server, but the basics are in place.
        </p>
        <div style="width:336px;float:left; margin: 10px 15px 10px 0px;background-color:white;">
         <script type="text/javascript">
          GA_googleFillSlot('howtoforge_com_article_rectangle_a_300x250');
         </script>
        </div>
        <p>
         This howto is meant as a practical guide; it does not cover the theoretical backgrounds. They are treated in a lot of other documents in the web.
        </p>
        <p>
         This document comes without warranty of any kind! I want to say that this is
not the only way of setting up such a system. There are many ways of achieving this goal but this is the way I take. I do
not issue any guarantee that this will work for you!
        </p>
        <p>
        </p>
        <h3 id="-install-xen">
         1 Install Xen
        </h3>
        <h4 id="-fresh-installation">
         1.1 Fresh Installation
        </h4>
        <p>
         If you are going to set up a fresh system, you can select the needed packages during the installation. Choose
         <span class="system">
          Customize now
         </span>
         at the corresponding step to customize the software selection during the installation:
        </p>
        <p>
         <a class="fancybox" href="https://www.howtoforge.com/images/f7_xen/big/pic1.jpg" id="img-pic1">
          <img border="0" mce_real_src="https://www.howtoforge.com/images/f7_xen/pic1.jpg" src="https://www.howtoforge.com/images/f7_xen/pic1.jpg"/>
         </a>
        </p>
        <p>
         In the window for the software selection click
         <span class="system">
          Base System
         </span>
         on the left side and afterwards mark the checkbox
         <span class="system">
          Virtualization
         </span>
         on the right side:
        </p>
        <p>
         <a class="fancybox" href="https://www.howtoforge.com/images/f7_xen/big/pic2.jpg" id="img-pic2">
          <img border="0" mce_real_src="https://www.howtoforge.com/images/f7_xen/pic2.jpg" src="https://www.howtoforge.com/images/f7_xen/pic2.jpg"/>
         </a>
        </p>
        <p>
        </p>
        <h4 id="-existing-system">
         1.2 Existing System
        </h4>
        <p>
         If you want to use an existing Fedora 7 system, install the needed packages with yum:
        </p>
        <p class="command">
         yum install kernel-xen xen virt-manager
        </p>
        <p>
         Afterwards reboot your system and choose the xen-kernel from the grub menu.
        </p>
        <p>
        </p>
        <h3 id="-using-the-gui">
         2 Using The Gui
        </h3>
        <p>
         After the Xen installation, you'll have a new entry in the menu - the
         <span class="system">
          Virtual Machine Manager
         </span>
         :
        </p>
        <div style="width:336px;float:left; margin: 10px 15px 10px 0px;background-color:white;">
         <script type="text/javascript">
          GA_googleFillSlot('howtoforge_com_article_rectangle_b_300x250');
         </script>
        </div>
        <p>
         <a class="fancybox" href="https://www.howtoforge.com/images/f7_xen/big/pic3.jpg" id="img-pic3">
          <img border="0" mce_real_src="https://www.howtoforge.com/images/f7_xen/pic3.jpg" src="https://www.howtoforge.com/images/f7_xen/pic3.jpg"/>
         </a>
        </p>
        <p>
         The Virtual Machine Manager provides the basic features for setting up and managing virtual machines. The available help in the menu at the top is short but useful - cast an eye at it. Don't wonder about the running machine
         <span class="system">
          Domain 0
         </span>
         in the list - it's your own system.
        </p>
        <p>
         <a class="fancybox" href="https://www.howtoforge.com/images/f7_xen/big/pic4.jpg" id="img-pic4">
          <img border="0" mce_real_src="https://www.howtoforge.com/images/f7_xen/pic4.jpg" src="https://www.howtoforge.com/images/f7_xen/pic4.jpg"/>
         </a>
        </p>
        <p>
         Time to set up your first virtual machine with xen. Select
         <span class="system">
          New machine...
         </span>
         from the menu:
        </p>
        <p>
         <a class="fancybox" href="https://www.howtoforge.com/images/f7_xen/big/pic5.jpg" id="img-pic5">
          <img border="0" mce_real_src="https://www.howtoforge.com/images/f7_xen/pic5.jpg" src="https://www.howtoforge.com/images/f7_xen/pic5.jpg"/>
         </a>
        </p>
        <p>
         The first window in the new popup gives you a short introduction about the following steps:
        </p>
        <p>
         <a class="fancybox" href="https://www.howtoforge.com/images/f7_xen/big/pic6.jpg" id="img-pic6">
          <img border="0" mce_real_src="https://www.howtoforge.com/images/f7_xen/pic6.jpg" src="https://www.howtoforge.com/images/f7_xen/pic6.jpg"/>
         </a>
        </p>
        <p>
         Choose a name for your new virtual machine in the next window:
        </p>
        <p>
         <a class="fancybox" href="https://www.howtoforge.com/images/f7_xen/big/pic7.jpg" id="img-pic7">
          <img border="0" mce_real_src="https://www.howtoforge.com/images/f7_xen/pic7.jpg" src="https://www.howtoforge.com/images/f7_xen/pic7.jpg"/>
         </a>
        </p>
        <p>
         Next step is to choose between performance and compatibility - choose
         <span class="system">
          Paravirtualized
         </span>
         to get the maximum performance:
        </p>
        <p>
         <a class="fancybox" href="https://www.howtoforge.com/images/f7_xen/big/pic8.jpg" id="img-pic8">
          <img border="0" mce_real_src="https://www.howtoforge.com/images/f7_xen/pic8.jpg" src="https://www.howtoforge.com/images/f7_xen/pic8.jpg"/>
         </a>
        </p>
