     <p>
      The aim of this
      <b>
       list of programming languages
      </b>
      is to include all notable
      <a href="/wiki/Programming_language" title="Programming language">
       programming languages
      </a>
      in existence, both those in current use and historical ones, in alphabetical order, except for dialects of
      <a href="/wiki/BASIC" title="BASIC">
       BASIC
      </a>
      and
      <a class="mw-redirect" href="/wiki/Esoteric_programming_languages" title="Esoteric programming languages">
       esoteric programming languages
      </a>
      .
     </p>
     <dl>
      <dd>
       <i>
        <b>
         Note
        </b>
        : Dialects of
        <a href="/wiki/BASIC" title="BASIC">
         BASIC
        </a>
        have been moved to the separate
        <a href="/wiki/List_of_BASIC_dialects" title="List of BASIC dialects">
         List of BASIC dialects
        </a>
       </i>
       .
      </dd>
      <dd>
       <i>
        <b>
         Note
        </b>
        : This page does not list
        <a href="/wiki/Esoteric_programming_language" title="Esoteric programming language">
         esoteric programming languages
        </a>
        .
       </i>
      </dd>
     </dl>
