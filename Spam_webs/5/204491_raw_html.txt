      <p>
       Curtail is a lightweight jQuery plugin that truncates text to a specified number of characters (250 by default) and generates a 'Read More' link allowing the visitor to reveal the truncated text.
      </p>

      <h2>
       Basic Usage:
      </h2>
      <p>
       1. Include jQuery library and the jQuery curtail plugin at the bottom of your Html.
      </p>
      <pre class="brush:xml;">
       &lt;script src="//ajax.googleapis.com/ajax/libs/jquery/1.11.2/jquery.min.js"&gt;&lt;/script&gt;
&lt;script src="jQuery.curtail.js"&gt;&lt;/script&gt;
      </pre>
      <p>
       2. Wrap your text into a container element and create a 'read more' link at the end of the text.
      </p>
      <pre class="brush:xml;">
       &lt;div class="example"&gt;
  &lt;p&gt;Your Long Text Here&lt;/p&gt;
  &lt;a href="#" class="btn"&gt;Read More&lt;/a&gt; 
&lt;/div&gt;
      </pre>
      <p>
       3. Call the plugin on the text wrapper and done.
      </p>
      <pre class="brush:js;">
       $('.example').curtail();
      </pre>
      <p>
       4. Customize the plugin.
      </p>
      <pre class="brush:js;">
       $('.example').curtail({

// the number of character to be displayed
limit: 250,

// custom the ellipsis
ellipsis: '...'

});
      </pre>
