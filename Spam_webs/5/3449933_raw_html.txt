      The World Of Skyrim - Reaching New Heights
     </div>
     <div class="subtitle">
      by Corey Dockendorf [ Wednesday, 9th of November 2011 - 05:47 PM ]
     </div>
     <div>
     </div>
     <div style="margin-top:4px;">
      <iframe allowtransparency="true" frameborder="0" scrolling="no" src="http://www.facebook.com/plugins/like.php?href=http%3A%2F%2Faggrogamer.com%2Farticle%2F2533%2FThe-World-Of-Skyrim--Reaching-New-Heights" style="border:none; overflow:hidden; width:450px;height:28px;">

     <div class="articlebody">
      <img align="left" alt="The World Of Skyrim - Reaching New Heights" border="0" src="/assets/1320780311-1320780309-13125772541312577252skyrim300x300.jpg" style="margin-right:5px;" width="200"/>
      <p>
       The latest developer's diary is up at the
       <strong>
        Elder Scrolls
       </strong>
       Community page, and this time we are taken into the world of Cyrodiil and we see this world brought to life. Full of character and lush landscapes, this massive world is a time sync TES fans know and love.
      </p>
      <p>
       If you haven't checked it out yet I also recommend for the more technical reader to check out
       <a href="http://www.elderscrolls.com/community/brief-history-cyrodiil/" target="_blank" title="The Brief History Of Cyrodiil">
        the brief history of Cyrodiil
       </a>
       . This way you can see how a game world takes shape from a more technical aspect.
      </p>
      <center>
       <a href="http://aggrogamer.com/assets/1320780262-f60ab740a8cf91b8d2240052760b1ca3.jpg" rel="lightbox" title="The Elder Scrolls V: Skyrim">
        <img alt="The Elder Scrolls V: Skyrim" border="0" src="http://aggrogamer.com/assets/1320780262-f60ab740a8cf91b8d2240052760b1ca3.jpg" title="The Elder Scrolls V: Skyrim" width="350px"/>
       </a>
      </center>
      <p>
       It's amazing the detail that this team has brought into the world. From the simplicity of leaves to the detail of a "very furry cow" it's incredible to see a video game with such scope. Although what else would you expect from Bethesda? I recommend listening to the podcasts for these developer series, for a wealth of information and a very informative view on what makes this game tick.
      </p>
      <p>
       Check out the brief world video below.
      </p>
      <center>
       <object height="360" width="640">
        <param name="movie" value="http://www.youtube-nocookie.com/v/Dl48qyYQSPg?version=3&amp;hl=en_US&amp;hd=1">
        </param>
        <param name="allowFullScreen" value="true">
        </param>
        <param name="allowscriptaccess" value="always">
        </param>
        <embed allowfullscreen="true" allowscriptaccess="always" height="360" src="http://www.youtube-nocookie.com/v/Dl48qyYQSPg?version=3&amp;hl=en_US&amp;hd=1" type="application/x-shockwave-flash" width="640">
        </embed>
       </object>
      </center>
      <p>
       Each one of these videos makes me want this game even more. I'm ready Bethesda, take my money and give me dragons! Remember that all these videos and podcasts can be found over at the
       <strong>
        Elder Scrolls
       </strong>
       Community site
       <a href="http://www.elderscrolls.com/community/" target="_blank" title="The Elder Scrolls V: Skyrim">
        here
       </a>
       .
      </p>

   <script type="text/javascript">
    $(document).ready(function() 
{
   // Notice the use of the each method to gain access to each element individually
   $('.featuredhover a').each(function()
   {
      // Create image content using websnapr thumbnail service
       var content = '&lt;div align="left"&gt;';
       content += $(this).attr('rel');
       content += "&lt;/div&gt;";
     
      // Setup the tooltip with the content
      $(this).qtip(
      {
         content: content,
         position: {
                  corner: {
                     tooltip: "leftMiddle", // Use the corner...
                     target: "rightMiddle" // ...and opposite corner
                  }
               },

         style: {
			border: {
        	width: 0,
            	radius: 4
            },
            padding: 10, 
            textAlign: 'center',
            tip: true,
            name: 'dark'
        }
      });
   });
});
   </script>
