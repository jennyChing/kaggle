          <p>
           A strong candidate for layouts creation appears to claim his spot among the best plugins for wordpress this year. Grid Layout Shock is a plugin to create responsive grid layouts. Thanks to its core software, the masonry JS plugin, the tool can arrange posts in a very similar-to-pinterest mode.
           <span id="more-22272">
           </span>
          </p>
          <p>
           In this
           <a class="external" href="http://www.gridlayoutshock.com/" onclick="javascript:_gaq.push(['_trackEvent','outbound-article','http://www.gridlayoutshock.com']);" rel="nofollow" target="_blank">
            page
           </a>
           , capabilities of the tool are demonstrated: there is a responsivity menu to check how would the grid look in mobile devices, a control panel that renders your settings directly on the page to see the tool in action, also a complete documentation page. If you prefer to keep reading about its features, here are some of them:
          </p>

          <p>
           <a href="http://www.bestdesigntuts.com/wp-content/uploads/2013/03/tdemo-2.jpg" rel="lightbox[22272]" title="Grid Layout Shock: An Advanced Plugin for New Layouts">
            <img alt="" class="alignleft size-full wp-image-22273" height="335" src="http://www.bestdesigntuts.com/wp-content/uploads/2013/03/tdemo-2.jpg" width="630"/>
           </a>

          <p>
           You can select the skin for the box, you can control the order of the posts to either ascending or descending; the plugin offers the option to divide the grid in several pages and determine the number of posts per page. There are other options to change the font type and to select what is going to be shown on the grid box (title, picture, excerpt, read more button, etc.) Also, the option to display RSS feeds is very useful to display content from outer sources.
          </p>
          <p>
           Plugin is also capable of filtering posts to be shown using the available taxonomy options in the site, and more than one grid can be placed in a text area since each grid is produced using an independent shortcode.
          </p>
