         How to share files and folders in Ubuntu

        <p>
         This brief tutorial will outline how to enable file sharing in Ubuntu.
         <span id="more-256">
         </span>
        </p>
        <ol>
         <li>
          Sharing files and folders across your network from your Ubuntu PC is every bit as easy as sharing files in Windows (arguably, its easier). Start by right-clicking the folder you want to share, and select
          <strong>
           Share Folder
          </strong>
          . If the services required to share files/folders arent installed, youll be prompted to install them.
         </li>
         <p>
          <img border="0" src="http://www.simplehelp.net/images/sharedubuntu/fileshare01a.jpg"/>
         </p>
         <li>
          After clicking
          <strong>
           Install services
          </strong>
          they will automatically start to download and install.
         </li>
         <p>
          <img border="0" src="http://www.simplehelp.net/images/sharedubuntu/fileshare02a.jpg"/>
         </p>
         <li>
          Once that has completed youll be presented with an options window. From here you can select the type of sharing (SMB or NFS), give the shared folder a name and decide if you want read-only access to the folder. Click
          <strong>
           OK
          </strong>
          when youre done.
         </li>
         <p>
          <img border="0" src="http://www.simplehelp.net/images/sharedubuntu/fileshare03.png"/>
         </p>
         <li>
          If you right-click on your newly shared folder and select
          <strong>
           Properties
          </strong>
          and then the
          <strong>
           Emblems
          </strong>
          tab, you can give the folder a unique icon so youll know its shared.
         </li>
         <p>
          <a href="http://www.simplehelp.net/images/sharedubuntu/fileshare04.png">
           <img border="0" src="http://www.simplehelp.net/images/sharedubuntu/fileshare04a.jpg"/>
           <br/>
           click to enlarge
          </a>
         </p>
         <li>
          You can further modify the permissions of your shared folder by selecting the
          <strong>
           Permissions
          </strong>
          tab.
         </li>
         <p>
          <a href="http://www.simplehelp.net/images/sharedubuntu/fileshare08.png">
           <img border="0" src="http://www.simplehelp.net/images/sharedubuntu/fileshare08a.jpg"/>
           <br/>
           click to enlarge
          </a>
         </p>
         <li>
          Now open a terminal in Ubuntu and enter the command:
          <br/>
          <blockquote>
           <p>
            sudo smbpasswd -a username
           </p>
          </blockquote>
          <p>
           This will add a SMB user.
          </p>
         </li>
         <li>
          And now other computers can connect to your shared folder!
         </li>
         <p>
          <img border="0" src="http://www.simplehelp.net/images/sharedubuntu/fileshare05a.jpg"/>
         </p>
        </ol>

           <div class="dsq-comment-message" id="dsq-comment-message-11566">
            <p>
             Wanting to share a folder and having problems? Try this:
            </p>
            <p>
             1.
             <br/>
             sudo cp /etc/samba/smb.conf /etc/samba/smb.conf.orig
             <br/>
             (backs up the smb.comf samba file)
            </p>
            <p>
             2.
             <br/>
             gksu gedit /etc/samba/smb.conf
             <br/>
             (opens the smb.conf file)
            </p>
            <p>
             3.
             <br/>
             Add this line below the usershare allow guests = yes line and save it
             <br/>
             usershare owner only = False
            </p>
            <p>
             4. (might not have to do this step)
