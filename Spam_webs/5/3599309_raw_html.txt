            <p>
             The Marvel Universe that has been concurrent since 1961 will be officially ending with the upcoming
             <em>
              Secret Wars
             </em>
             series in May. Marvel Editor-in-Chief Axel Alonso and Senior Vice President of Publishing and Executive Editor Tom Brevoort announced the surprising news during a
             <i>
              Secret Wars
             </i>
             Kick-Off
             <i>
              
             </i>
             press event lateTuesday.
            </p>
            <p>
             As well as the primaryMarvel Universe coming to an end, the Marvel Ultimate Universewill be ending as well. As Brevoort explained, the idea behind
             <em>
              Secret Wars
             </em>
             and the universes being destroyed is that there has been an incursion, where the two parallel universesintersect  and it just so happens that the intersectionpoint is Earth. If someone in either universe was able to remove the Earth, the universes would be able to pass through each other and nothing would happen (except that one universe would be Earth-less). But, being that nothing can be done, the two universes are to clash togetherand destroy both of them in the process.
            </p>
            <p>
             The events of
             <em>
              Secret Wars
             </em>
             and their implications have been known by all of Marvels creative teams dating back to the beginningof Marvel NOW. All of the more recent big events, from
             <em>
              Original Sin
             </em>
             to
             <em>
              AXIS
             </em>
             have been done with
             <em>
              Secret Wars
             </em>
             in mind. As well, the writers and teams have been told that May is their deadline, and that all important plotlines should be dealt with by then.
            </p>
            <p>
             The last well see of our current Marvel Universe will be in
             <em>
              Avengers #44
             </em>
             and
             <em>
              New Avengers #33,
             </em>
             the last of both series respectively. During these issues, the final incursion will happen, which will lead directly into
             <em>
              Secret Wars #1.
             </em>
            </p>
            <p>
             <a href="http://moargeek.com/wp-content/uploads/2015/01/Secret-Wars.jpg">
              <img alt="" class="aligncenter wp-image-9722 size-full" height="562" src="http://moargeek.com/wp-content/uploads/2015/01/Secret-Wars.jpg" width="1000"/>
             </a>
            </p>
            <p>
             Once that happens, all that will be left of Marvel is the Battleworld  the staging area where the
             <em>
              Secret War
             </em>
             storyline is set to take place. Once we hit
             <em>
              Secret Wars #1,
             </em>
             there is no Marvel Universe, Ultimate Universe, or any other, Brevoort said during the presser, Its all Battleworld. This new Battleworld is the left over parts of both universes that have meldedthem together into a melting pot that the new Marvel Universe will be created from. Several new serieswith familiar characters will be released during
             <em>
              Secret Wars
             </em>
             as well as after it, all of which will be in this new Battleworld-based universe.
            </p>
            <p>
             All of this of course spoils the first issue of
             <em>
              Secret Wars,
             </em>
             which will actually detail all the events of the clash and set the events of the new Marvel Universe into motion. During this yearsfree comic book day, which is also in May, a special
             <em>
              Secret Wars #0
             </em>
             issue will be given away for free, so fans who havent kept up on the events can have a quick primer as the new series kicks off. Also included in
             <em>
              Secret Wars #0
             </em>
             will be the Marvel/
             <em>
              Attack on Titan
             </em>
             <a href="http://moargeek.com/2014/11/03/marvel-attack-titan-crossover-coming/" title="A Marvel And Attack On Titan Crossover Is Coming">
              crossover
             </a>
             that to this point has been exclusive to Japan.
            </p>
            <p>
             During a quick trailer teaser the colliding universes, an All-New Marvel logo is briefly teased, which is likely what this new Universe will be called.
            </p>
            <p>
             If youre strictly a fan of the Marvel movies, you have nothing to be worried about. Not only is the Marvel Cinematic Universe completely separate from the comic book equivalent, but Marvel has stated in the past that they have no desire to try and connect the two in any way.
            </p>
            <p>
             <em>
              Secret Wars
             </em>
             is being written by Jonathan Hickman (
             <em>
              Avengers, New Avengers, Infinity, Fantastic Four
             </em>
             )with art coming from Esad Ribic. Cover art is beingdone by the legendary Alex Ross, who has also produced some great promotional art for the series already.
            </p>

	function resizeStickyMenu() {
		//see if compact versions of menus should be shown
		if(jQuery('#section-menu-full').length &gt; 0) {
			var megaWidth = jQuery('.mega-menu').width();
			if(jQuery('.mega-menu').length == 0) {
				var megaWidth = jQuery('.non-mega-menu').width();
			}
			var standardWidth = jQuery('.secondary-menu-full').width();
			var compactWidth = jQuery('.secondary-menu-compact').width();
			var menusWidth = megaWidth + standardWidth;
			var compactMenusWidth = megaWidth + compactWidth;
			var logoWidth = jQuery('#sticky-bar .logo').width();
			//var logoLeft = jQuery('#sticky-bar .logo').offset();
			//logoWidth = logoWidth + logoLeft.left;
			var newWidth = jQuery('#new-articles').width();
			var stickyWidth = jQuery('#sticky-menu-selector').width();
			var randomWidth = jQuery('#random-article').width();
			var controlsWidth = jQuery('#sticky-controls').width();
			var barWidth = jQuery('#sticky-bar &gt; .row &gt; .col-md-12 &gt; .container').width();
			if(barWidth === null) barWidth = jQuery('#sticky-bar &gt; .row &gt; .col-xs-12 &gt; .container').width();
			var extraWidth = logoWidth + newWidth + stickyWidth + randomWidth + controlsWidth;
			var limitWidth = barWidth - extraWidth;
			//alert('megaWidth=' + megaWidth + '\nstandardWidth=' + standardWidth + '\nmenusWidth=' + menusWidth + '\nlogoWidth=' + logoWidth + '\nnewWidth=' + newWidth + '\nstickyWidth=' + stickyWidth + '\nrandomWidth=' + randomWidth + '\ncontrolsWidth=' + controlsWidth + '\nbarWidth=' + barWidth + '\nextraWidth=' + extraWidth + '\nlimitWidth=' + limitWidth);
			//mega menu alone passes limit
			if(megaWidth &gt; limitWidth) {
				jQuery('#section-menu-full').hide();
				jQuery('#section-menu-compact').show();
			}
			//standard menu alone passes limit
			if(standardWidth &gt; limitWidth) {
				jQuery('.secondary-menu-full').hide();
				jQuery('.secondary-menu-compact').show();
			}
			//both menus together pass limit
			if(menusWidth &gt; limitWidth) {
				//first reduce standard menu
				jQuery('.secondary-menu-full').hide();
				jQuery('.secondary-menu-compact').show();
				//compact standard plus mega menu pass limit
				if(compactMenusWidth &gt; limitWidth) {
					jQuery('#section-menu-full').hide();
					jQuery('#section-menu-compact').show();
				}
			}
		}
	}

	/**
	* Check a href for an anchor. If exists, and in document, scroll to it.
	* If href argument ommited, assumes context (this) is HTML Element,
	* which will be the case when invoked by jQuery after an event
	*/
	function scroll_if_anchor(href) {
	    href = typeof(href) == "string" ? href : jQuery(this).attr("href");
		
		//do not interfere with bootstrap carousels
		if(jQuery(href).length &gt; 0 &amp;&amp; !jQuery(this).hasClass('no-scroll')) {			
							var fromTop = 118;	
						
			//subtract contents menu height (and margin)
			if(jQuery('.contents-menu-wrapper').hasClass('vertical')) {
				fromTop = fromTop - 51;
			}
		
			// If our Href points to a valid, non-empty anchor, and is on the same page (e.g. #foo)
			// Legacy jQuery and IE7 may have issues: http://stackoverflow.com/q/1593174
			if(href.indexOf("#") == 0) {
				var $target = jQuery(href);
		
				// Older browser without pushState might flicker here, as they momentarily
				// jump to the wrong position (IE &lt; 10)
				if($target.length) {
					jQuery('html, body').animate({ scrollTop: $target.offset().top - fromTop });
					if(history &amp;&amp; "pushState" in history) {
						history.pushState({}, document.title, window.location.pathname + href);
						return false;
					}
				}
			}
		}
	}
