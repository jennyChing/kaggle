    <div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="es">
     <p>
      <b>
       JTAG
      </b>
      , un acrnimo para
      <b>
       Joint Test Action Group
      </b>
      , es el nombre comn utilizado para la norma
      <a href="/wiki/Institute_of_Electrical_and_Electronics_Engineers" title="Institute of Electrical and Electronics Engineers">
       IEEE
      </a>
      1149.1 titulada
      <b>
       Standard Test Access Port and Boundary-Scan Architecture
      </b>
      para
      <a class="new" href="/w/index.php?title=Test_access_port&amp;action=edit&amp;redlink=1" title="Test access port (an no redactado)">
       test access ports
      </a>
      utilizada para testear
      <a href="/wiki/Circuito_impreso" title="Circuito impreso">
       PCBs
      </a>
      utilizando escaneo de lmites.
     </p>
     <p>
      JTAG se estandariz en
      <a href="/wiki/1990" title="1990">
       1990
      </a>
      como la norma IEEE 1149.1-1990. En
      <a href="/wiki/1994" title="1994">
       1994
      </a>
      se agreg un suplemento que contiene una descripcin del
      <a class="new" href="/w/index.php?title=Boundary_scan_description_language&amp;action=edit&amp;redlink=1" title="Boundary scan description language (an no redactado)">
       boundary scan description language
      </a>
      (BSDL). Desde entonces, esta norma fue adoptada por las compaas electrnicas de todo el mundo. Actualmente, Boundary-scan y JTAG son sinnimos.
     </p>
     <p>
      Diseado originalmente para circuitos impresos, actualmente es utilizado para la prueba de submdulos de circuitos integrados, y es muy til tambin como mecanismo para
      <a class="mw-redirect" href="/wiki/Depuraci%C3%B3n_de_aplicaciones" title="Depuracin de aplicaciones">
       depuracin de aplicaciones
      </a>
      empotradas, puesto que provee una puerta trasera hacia dentro del sistema. Cuando se utiliza como herramienta de depuracin, un emulador en circuito que usa JTAG como mecanismo de transporte permite al programador acceder al mdulo de depuracin que se encuentra integrado dentro de la CPU. El mdulo de depuracin permite al programador corregir sus errores de cdigo y lgica de sus sistemas.
     </p>
     <p>
     </p>

     <h2>
      <span class="mw-headline" id="Caracter.C3.ADsticas_El.C3.A9ctricas">
       Caractersticas Elctricas
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=JTAG&amp;action=edit&amp;section=1" title="Editar seccin: Caractersticas Elctricas">
        editar
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      Una interfaz JTAG es una interfaz especial de cuatro o cinco pines agregadas a un chip, diseada de tal manera que varios chips en una tarjeta puedan tener sus lneas JTAG conectadas en
      <a href="/wiki/Daisy_chain" title="Daisy chain">
       daisy chain
      </a>
      , de manera tal que una sonda de testeo JTAG necesita conectarse a un solo "puerto JTAG" para acceder a todos los chips en un
      <a href="/wiki/Circuito_impreso" title="Circuito impreso">
       circuito impreso
      </a>
      . Los pines del conector son
     </p>
     <ol>
      <li>
       TDI (Entrada de Datos de Testeo)
      </li>
      <li>
       TDO (Salida de Datos de Testeo)
      </li>
      <li>
       TCK (Reloj de Testeo)
      </li>
      <li>
       TMS (Selector de Modo de Testeo)
      </li>
      <li>
       TRST (Reset de Testeo) es opcional.
      </li>
     </ol>
     <div class="center">
      <div class="floatnone">
       <a class="image" href="/wiki/Archivo:Jtag_chain.svg" title="Example of JTAG chain">
        <img alt="Example of JTAG chain" data-file-height="249" data-file-width="654" height="249" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/c9/Jtag_chain.svg/654px-Jtag_chain.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/c9/Jtag_chain.svg/981px-Jtag_chain.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/c9/Jtag_chain.svg/1308px-Jtag_chain.svg.png 2x" width="654"/>
       </a>
      </div>
     </div>
     <p>
      Ya que posee una sola lnea de datos, el protocolo es necesariamente serial, como el
      <a href="/wiki/Serial_Peripheral_Interface" title="Serial Peripheral Interface">
       Serial Peripheral Interface
      </a>
      . La entrada de la seal de reloj es por el pin TCK. La configuracin del dispositivo se realiza manipulando una mquina de estados de un bit empleando el pin TMS. Un bit de datos es cargado en TDI y otro sacado en TDO por cada pulso de reloj de la seal TCK. Se pueden cargar diferentes modo de instruccin como leer el ID del chip, muestrear el valor de pines de entrada/salida, manejar pines de salida, manipular funciones del chip, o funciones de bypass que unen el pin TDI con TDO para lgicamente unir cadenas de varios chips (chips en cascada). La frecuencia de trabajo de la seal de reloj del pin TCK vara en funcin de cada chip, pero tpicamente est en el rango de 10-100 MHz (10-100ns/bit).
     </p>
     <p>
      Cuando se hace la operacin de boundary scan en circuitos integrados, las seales manipuladas estn entre diferentes bloques funcionales del chip, ms que entre diferentes chips.
     </p>
     <p>
      El pin TRST es una seal opcional bajo-activa para reseteo o reinicio de la prueba lgica (por lo general asncrona, pero que a veces est sincronizada con el reloj, dependiendo del chip). Si no se dispone de dicho pin, la prueba lgica puede reiniciarse mediante una instruccin reset.
     </p>
     <p>
      Existen productos de consumo que tienen un puerto JTAG integrado, por lo que las conexiones estn a menudo disponibles en la
      <a href="/wiki/Circuito_impreso" title="Circuito impreso">
       PCB
      </a>
      como parte de la fase de prototipado del producto. Estas conexiones pueden proporcionar una sencilla forma de realizar
      <a href="/wiki/Ingenier%C3%ADa_inversa" title="Ingeniera inversa">
       ingeniera inversa
      </a>
      .
     </p>
