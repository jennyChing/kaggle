             <h2>
              Using Extensions in Firefox
             </h2>
             by
             <a href="/pub/au/944">
              Wei-Meng Lee
             </a>
             , author of
             <a href="http://www.oreilly.com/catalog/winxpunwired">
              Windows XP Unwired
             </a>
             <br/>
             09/21/2004
            </td>
           </tr>
          </table>
          <p>
           The 1.0 Preview Release of Mozilla's next-generation browser, Firefox, was rolled out last week, and like earlier versions, it packs many features that improve on Internet Explorer, such as faster load time and support for tabbed browsing.
          </p>
          <!--ONJava MPU Ad -->
          <p>
           One key Firefox feature that I like is its extensibility--you can download additional plug-ins, known as
           <em>
            extensions
           </em>
           , to enhance it. In this article, I will take a look at some of the more useful extensions that you can download for Firefox.
          </p>
          <p>
           Keep in mind that while not all extensions in the preview release currently work, it's expected that all will in time. For example, the Googlebar extension wasn't working initially, but within a few days it was updated and now works fine.
          </p>
          <h3>
           Where to Find Extensions for Firefox
          </h3>
          <p>
           Locating extensions for Firefox is a simple task. Simply go to the menu Tools -&gt; Extensions, and the Extensions window will appear (see Figure 1). The window lists the extensions currently installed on your browser. You can select an extension and uninstall it or update it. To find additional extensions, click on Get More Extensions.
          </p>
          <p>
           <img alt="Figure 1" height="364" src="/windows/2004/09/21/graphics/firefox001.gif" width="500"/>
           <br/>
           <em>
            Figure 1. Viewing the list of extensions installed on your browser
           </em>
          </p>
          <p>
           You can also go directly to the Firefox
           <a href="http://update.mozilla.org/extensions/">
            Extensions page
           </a>
           . In either case, you should see the following page, shown in Figure 2:
          </p>
          <p>
           <img alt="Figure 2" height="447" src="/windows/2004/09/21/graphics/firefox002.gif" width="500"/>
           <br/>
           <em>
            Figure 2. The Extensions page containing the list of extensions available
           </em>
          </p>
          <p>
           Installing an extension is easy. Click on the extension you're interested in, and you'll be sent to a page with a description of the extension, as well as an Install Now icon (see Figure 3).
          </p>
          <p>
           <img alt="Figure 3" height="71" src="/windows/2004/09/21/graphics/firefox003.gif" width="241"/>
           <br/>
           <em>
            Figure 3. Installing an extension
           </em>
          </p>
          <p>
           Click on Install Now, and you will see the Software Installation window (see Figure 4). Click on the Install Now button to proceed with the installation.
          </p>
          <p>
           <img alt="Figure 4" height="337" src="/windows/2004/09/21/graphics/firefox004.gif" width="346"/>
           <br/>
           <em>
            Figure 4. Installing an extension
           </em>
          </p>
          <h3>
           Adbar--Ads from Google
          </h3>
          <p>
           One extension that I find useful is Adbar. The Adbar extension displays Google ads related to pages you view. The extension sends your page request to Google whenever you visit a site. Relevant ads are then displayed in the toolbar, as shown in Figure 5.
          </p>
          <p>
           <img alt="Figure 5" height="163" src="/windows/2004/09/21/graphics/firefox005.gif" width="500"/>
           <br/>
           <em>
            Figure 5. Displaying Google's ads using the Adbar extension
           </em>
          </p>
          You can get the Adbar extension
          <a href="http://update.mozilla.org/extensions/moreinfo.php?id=217">
           here
          </a>
          .
