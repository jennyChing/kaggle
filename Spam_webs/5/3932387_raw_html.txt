      <h1>
       <a href="http://cybernetnews.com/dark-gmail-theme-available-for-stylish-users/" rel="bookmark" title="Link to Dark Gmail Theme Available for Stylish Users">
        Dark Gmail Theme Available for Stylish Users
       </a>
      </h1>
     </div>
     <div class="entry KonaBody">
      <p align="center">
       <a href="http://cybernetnews.com/wp-content/uploads/2008/04/gmail-theme.png">
        <img alt="gmail theme" border="0" height="305" src="http://cdn2.cybernetnews.com/wp-content/uploads/2008/04/gmail-theme-thumb.png" width="500"/>
       </a>
       <br/>
       <em>
        (Click to Enlarge)
       </em>
      </p>
      <p>
      </p>
      <center>
       <script async="" src="http://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js">
       </script>
       <br/>
       <!-- cybernet-full content center -->
       <br/>
       <ins class="adsbygoogle" data-ad-client="ca-pub-5444720511869362" data-ad-slot="7215171456" style="display:inline-block;width:336px;height:280px">
       </ins>
       <br/>
       <script>
        (adsbygoogle = window.adsbygoogle || []).push({});
       </script>
      </center>
      <p>
       I know that some of you like to live on the dark side, and
       <a href="http://userstyles.org/styles/5867">
        this Gmail theme
       </a>
       will absolutely fill that burning desire. Youll need to be using Firefox and have
       <a href="https://addons.mozilla.org/en-US/firefox/addon/2108">
        Stylish
       </a>
       installed in order for it to work, and I have to admit that this is one of the most complete Gmail themes that Ive ever seen.
      </p>
      <p>
       As you can see in the screenshot nearly everything has been skinned. Even the compose window includes a Google Docs-like toolbar that is pretty slick. Oh, and the Loading screen has even been changed so that its not so dull. Needless to say there has been a lot of work spent putting this theme together, and in all there are over 6,000 lines of code that make this as fluid as it is!
      </p>
      <p>
       I used the theme for a few hours today before determining that the overall darkness was a little too much for me to handle. What Im hoping, however, is that other developers will be able to take the work done on this theme to create more like it. Actually, Google should just have a skinning capabilities in Gmail much like they do in
       <a href="http://www.google.com/ig">
        iGoogle
       </a>
       .
      </p>
      <p>
       <a href="http://userstyles.org/styles/5867">
        Gmail Redesigned Theme

        Google Blogoscoped

      </p>
