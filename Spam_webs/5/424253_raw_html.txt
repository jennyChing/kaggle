              <p>
               The upcoming Project Spartan Web browser for Windows 10  now has an official name: Microsoft Edge.
              </p>
              <p>
               At
               <a href="http://live.cnet.com/Event/Microsoft_Build_2015_live_blog">
                Microsoft Build
               </a>
               , the company announced new features and details for the Web browser previously known as
               <a href="http://www.cnet.com/products/microsoft-project-spartan/">
                Project Spartan
               </a>
               . The most notable is the browser's official name -- Spartan will now be known as Microsoft Edge, as it's on the cutting edge of what consumers and creators want to do on the Web.
              </p>
              <p>
              </p>
              <div class="shortcode related-links float_right" section="shortcodeRelatedLinks">
               <h6>
                <span>
                 Microsoft Build 2015
                </span>
               </h6>
               <ul>
                <li>
                 <a href="http://www.cnet.com/news/at-build-2015-microsoft-windows-10-faces-most-critical-challenge-yet/">
                  Microsoft's top task for Windows 10: Proving it's worth starting up
                 </a>
                </li>
                <li>
                 <a href="http://www.cnet.com/news/microsoft-aims-for-1-billion-devices-running-windows-10/">
                  Microsoft sets goal of 1 billion devices running Windows 10
                 </a>
                </li>
               </ul>
              </div>
              <p>
               One of the newest updates to the renamed browser is a new page layout for new tabs, which creates what looks and feels like a customized home page complete with links and thumbnails of the websites you visit frequently, Web-based apps you use and personalized information pulled from Cortana.
              </p>
              <p>
               Popular extensions for the Chrome and
               <a data-is-autolink="true" href="/tags/firefox/">
                Firefox
               </a>
               browsers will also turn up on Edge, as these can now be repurposed for Edge with what Microsoft says are "just a few changes."
              </p>
              <div class="ad-inpage-video-top" data-ad="inpage-video-top" id="inpage-video-top-55a4248674d63">

              <p>
               Currently, if you want to try the browser formerly known by the code-name Spartan, you need to be running the latest build of the
               <span class="link" section="shortcodeLink">
                <a href="/products/microsoft-windows-10/">
                 Windows 10 Technical Preview
                </a>
               </span>
               , where its prerelease state is still a work in progress. In our
               <span class="link" section="shortcodeLink">
                <a href="/products/microsoft-project-spartan/">
                 hands-on preview of Project Spartan
                </a>
               </span>
               , we said, "a new rendering engine powering all of Spartan's features, and it feels appreciably zippy," but at the same time, "you'll click a link, and nothing will happen. So you'll click it again, and again, and moments later, when you've just about given up hope, several identical tabs spring to life."
              </p>
              <p>
               The most notable features previously announced for Spartan/Edge are the inclusion of Cortana, Microsoft's virtual assistant (and like the name Spartan, a reference to the Microsoft-made Halo games), and on-screen notation via Web Note, for drawing annotations directly on Web pages and sharing them.
              </p>
