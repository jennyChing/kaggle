       <h3>
        Read how to get best out of inbuilt Magento eCommerce SEO features.
       </h3>
       <p>
        <a href="http://kgntechnologies.com/wp-content/uploads/2013/10/magento_seo.png">
         <img alt="Magento SEO Tips" class="size-full wp-image-597 alignleft" height="213" src="http://kgntechnologies.com/wp-content/uploads/2013/10/magento_seo.png" title="Magento eCommerce SEO Tips" width="292"/>
        </a>
       </p>
       <p>
        We all already know by now that magento ecommerce is the most popular eCommerce platform which possesses solid built-in structure that enables entrepreneurs to build feature-rich shopping websites. Magento ecommerce healthy built-in structure also includes powerful SEO set up which can help to achieve high ranks in search engines. Atleast onsite SEO burden will be largely taken care if you use Magento eCommerce and most of the further efforts required will be more of online promotions, marketing and off page optimization.
       </p>
       <p>
        Most of the stores would target top rankings in Google as it is the largest search engine that owns around 70 percent of search market share. Getting Magento ecommerce website in top in Google will not only give you maximum revenue but increase your brand status as well. It is hard to secure top rankings and further it is equally hard to retain that position and for that a good SEO practice is required.
       </p>
       <p>
        For any eCommerce website regardless if its built on Magento eCommerce, ultimate goal is to absorb maximum revenues. In order to achieve that, an effective SEO campaign is required. For better optimized eCommerce website, the profit margin always stands on top. Therefore, we have brought few effective SEO tips which can help you further enhance search results of Magento eCommerce stores by using available SEO features.
       </p>
       <p>
        <b>
         Search Engine Friendly URL:
        </b>
       </p>
       <p>
        In order to make your product pages gets index fast, you need to correct pages permalinks to make them search engine friendly.
       </p>
       <ul>
        <li>
         Magento ecommerce built-in structure supports simplify the URLs. Go to system -&gt; configuration -&gt; web -&gt; search engine optimization. Correct the URLs of product pages.
        </li>
        <li>
         Move to URL Options menu and then assign NO to Add Store Code to URLs, this simply help in avoiding store codes to be added to pages URLs.
        </li>
        <li>
         Go to Search Engine Optimization menu and set Yes to Server URL Rewriters. The purpose of doing this is to remove the index.php from URLs.
        </li>
        <li>
         In order to create auto-create keyword rich URLs, go to web option from Configuration menu, then select Yes to let the magento decide the best URL for each page automatically.
        </li>
        <li>
         Check the unsecure/secure option in Configuration menu to decide whether you want your base URL to be the www or no-www version. Well, these both versions are acceptable in SEO methodology.
        </li>
       </ul>
       <p>
        <b>
         Allow search engines to index your website
        </b>
       </p>
       <p>
        In order to configure this, go to system -&gt; configuration -&gt; design -&gt; HTML head, here you can change the Default Robots setting to INDEX, FOLLOW. By doing this you enable search engines to index your site.
       </p>
       <p>
        <b>
         Capitalize metadata
        </b>
       </p>
       <p>
        Title and description, these both are the most important aspects in SEO mathematics. If you have properly written, keyword rich title and description, search robots love to read. Thus, it is recommended to change the default metadatas information in your magento store.
       </p>
       <p>
        Change the default title and description, go to admin interface -&gt; system -&gt; configuration -&gt; design -&gt; HTML heads. Then go individual pages and categories in the catalogue section of admin panel to set the unique title and description for each page.
       </p>
       <p>
        Label your images properly. Use relevant name for your image instead of image_123 etc.
       </p>
       <p>
        <b>
         Utilize canonical feature
        </b>
       </p>
       <p>
        To avoid the duplication in page and categories, the same content appears more than one in different categories then set YES to both Canonical Link Meta Tag fields.
       </p>
       <p>
        Site with duplicate content generally appears rear in search result page. Magento ecommerce built-in structure supports you to avoid the duplication of content. For that go to system -&gt; configuration -&gt; search engine optimization and set NO for Use Categories Path for Product URLs.
       </p>
       <p>
        <b>
         Submit fresh sitemap to Google
        </b>
       </p>
       <p>
        Generate Google sitemap to help Google index fast your eCommerce store. Magento eCommerce built-in structure supports generate XML sitemap to help you create and submit each time fresh XML sitemap to Google.
       </p>
       <p>
        To select your own priority, go to system -&gt; configuration -&gt; Google sitemap and go to Generation Settings to automate XML sitemap generate.
       </p>
       <p>
        Configure the sitemap settings on basis of adding new products, categories so that Google gets new sitemap.
       </p>
       <p>
        The purpose of doing this, your website latest products and categories can be indexed fast and gets maximum chances to appear fast in search results.
       </p>
       <p>
        <b>
         Enhance website speed
        </b>
       </p>
       <p>
        While optimizing Magento eCommerce website for search engines easy discovery and enhancing ranking, poor speed is one another important aspect which needs to be worked on as it can drag your website back in search result pages. Since Google has considered the website loading speed into ranking factors, it is really important to have a website which is light, quick and opens fast.
       </p>
       <p>
        Since slow loading website can ruin your rankings, it is ideal to optimize your Magento eCommerce store for fast loading by enabling caching, compressing script files, reducing image sizes and serving content using CDN and approaching dedicated host service.
       </p>
       <p>
        Read more about
        <a href="http://custommagentodevelopmentindia.wordpress.com/2013/06/24/how-to-enhance-conversion-of-your-magento-store/" rel="nofollow" target="_blank" title="Increase eCommerce Website Conversions">
         HOW TO ENHANCE CONVERSION OF YOUR MAGENTO STORE
        </a>
       </p>
       <h4>
        If you have any betters suggestions of Magento eCommerce features which can assist in better SEO strategy, feel free to let us know through comments.
