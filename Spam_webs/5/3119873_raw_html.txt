         Format acum 800,000 de ani in urma prin retragerea oceanului, Mono Lake este unic in lume pentru micro-organizmele care s-au dezvoltat aici.
        </p>
        <p>
         <img alt="" height="375" src="http://assets.acasatv.ro/assets/fiicurios/2012/06/27/image_galleries/291/top-9-enigme-ale-naturii-pentru-care-nu-exista-un-raspuns_6.jpg" width="500"/>
         <br/>
         <br/>
         <strong>
          Ochiul Africii, Mauritania
         </strong>
         <br/>
         Ochiul Africii este o enigma pentru oamenii de stiinta. Inca nu se poate explica cum s-a format roca in forma actuala in zona respectiva din Mauritania . Explicatia plauzibila este aceea ca vanturile au modelat rocile in felul acesta, iar explicatia mai putin plauzibila este aceea ca in trecut aici a existat o "pista de aterizare" pentru extraterestrii :).
        </p>

         <br/>
         <img alt="" src="http://assets.acasatv.ro/assets/fiicurios/2012/06/27/image_galleries/291/top-9-enigme-ale-naturii-pentru-care-nu-exista-un-raspuns_2.jpg"/>
         <br/>
         <br/>
         <strong>
          Valea Mortii, SUA
         </strong>
         <br/>

         <br/>

         <br/>
         <br/>
         Niciodata nu a fost filmat acest fenomen, dar pietrele isi schimba pozitia o data la cativa ani, atunci cand se "plimba" de mai multe ori prin valea arida.
        </p>
        <p>
         <img alt="" src="http://assets.acasatv.ro/assets/fiicurios/2012/06/27/image_galleries/291/top-9-enigme-ale-naturii-pentru-care-nu-exista-un-raspuns_1.jpg"/>
         <br/>

          Rosin Split Apple Rock, Noua Zeelanda
