            <h4>
             Review: Zero DS Electric Motorcycle Long Distance Test - Day 6
            </h4>
           </td>

              <td>
               This is part of a multi-part article
               <br/>
               reviewing the Zero electric motorcycle
               <br/>
               in a rural environment.
               <br/>
               <br/>
               <a href="http://news.motorbiker.org/blogs.nsf/dx/review-zero-ds-electric-motorcycle-on-a-railroad-run---day-5.htm">
                Click here to read the previous article
               </a>
               <br/>
               <a href="http://news.motorbiker.org/blogs.nsf/dx/review-zero-ds-electric-motorcycle-at-a-concert---day-7.htm">

            One of the reasons electric motorcycles aren't selling like warm croissants is range. Many bikers, quite wrongly, see that if the motorcycle can't do 1000 miles, if of no use to them. But according to studies, most motorcycle riders only ride on average 30 miles in a day. Of course there are many bikers who do more than that, but we're talking average. Many motorcycle riders commute, or do short trips, usually in the city.
            <br/>
            <br/>
            But the fact remains that the range of an electric motorcycle is not very far. granted, many petrol-powered motorcycles don't have a very long range either, but with these you just pull up to a gas station and fill up the tank. With electric motorcycles, you need to plan ahead if you are going to ride long distances. You will need to know where you can plug in your bike.
            <br/>
            <br/>
            To get a feel about the real range of the
            <a href="http://www.zeromotorcycles.com/" onclick="javascript: pageTracker._trackPageview('/outgoing/Zeromotorcycles');">
             Zero
            </a>
            <a href="http://www.zeromotorcycles.com/" onclick="javascript: pageTracker._trackPageview('/outgoing/Zeromotorcycles');" target="_blank">
             <img alt="Open link in a new window" border="0" src="New-window.gif/$file/New-window.gif" title="Open link in a new window"/>
            </a>
            , and to see how range anxiety was going to be with me, I decided to ride the Zero to the closest big city to me, the harbor city of
            <a href="http://visitnormandy.org/Normandy.nsf/Visit/Normandy/Upper-Normandy/Le%20Havre.htm" onclick="javascript: pageTracker._trackPageview('/outgoing/Visitnormandy');">
             Le Havre
            </a>
            <a href="http://visitnormandy.org/Normandy.nsf/Visit/Normandy/Upper-Normandy/Le%20Havre.htm" onclick="javascript: pageTracker._trackPageview('/outgoing/Visitnormandy');" target="_blank">
             <img alt="Open Le Havre link in a new window" border="0" src="New-window.gif/$file/New-window.gif" title="Open Le Havre link in a new window"/>
            </a>
            . There is a motorway going into the city, but I tend to avoid motorways, so instead I chose a nice ride via the coast.
            <br/>
            <br/>
            <a href="http://photos.imageevent.com/motorbiker/newspics4/Zero-test-Home-Le-havre.jpg" rel="thumbnail" title="">
            </a>
            <table align="center" border="0" cellpadding="5" cellspacing="0" width="560">
             <tr>
              <td>
               <div class="Pict">
                <img alt="Click for bigger version of Zero test Home to Le havre" height="557" solid="" src="http://photos.imageevent.com/motorbiker/newspics4/Zero-test-Home-Le-havre.jpg" title="Click for bigger version of Zero test Home to Le havre" white="" width="550"/>

            <br/>
            A one-way trip to the city is about
            <strong>
             70 kms,
            </strong>
            plus riding in the city would bring a single trip to 80 kilometers. A return trip would therefore be 150 to
            <strong>
             160 kms
            </strong>
            , something well inside the quoted range of the Zero. Unfortunately, there would be several long stretches of straight road ahead, but also some curves and hills. Accompanied with only a power lead, I set out for my adventure.
            <br/>
            <br/>
            I stuck religiously to the speed limits, and never opened the throttle fully open (
            <em>
             the tourque was set to 80%; top speed 110 kph, 100% regeneration for engine and brakes
            </em>
            );
            <br/>
            <br/>
            So I took a leisurely ride to Le Havre. I arrived after 1 hour and 20 minutes ride (
            <em>
             my bum hurt from the granite hard saddle, but that's what you are used to, soft or not
            </em>
            ).
            <br/>
            <br/>
            <a href="http://photos.imageevent.com/motorbiker/newspics4/Zero-6-1.jpg" rel="thumbnail" title="Zero motorcycle at the Le Havre harbor">
            </a>
            <table align="center" border="0" cellpadding="5" cellspacing="0" width="560">

            <br/>
            Le Havre is a harbor city, with massive container ships, oil ships, cruise boats and ferries.
            <br/>
            <br/>
            I arrived at some friends who run Le Havre's biggest motorcycle dealership. As you can see from the photo below, the number of kilometers I had done was 71.6 and there was about 50% battery left.
            <br/>
            <br/>
            <a href="http://photos.imageevent.com/motorbiker/newspics4/Zero-6-2.jpg" rel="thumbnail" title="Zero motorcycle odometer and battery level">
            </a>
            <table align="center" border="0" cellpadding="5" cellspacing="0" width="560">

            <br/>
            Since my friend was going to have a ride on the Zero, we plugged it into the power outlet in the garage for 30 minutes, hopefully enough for him to test ride it for 10 kilometers or so.
            <br/>
            <br/>
            When he came back, he had a very big smile on his face. He loved the ride.
            <br/>
            <br/>
            I then went back home, riding gently and a bit anxious about the range. The battery was showing 50%, so would I have to push it the last few kilometers?
            <br/>
            <br/>
            On the way I stopped for a sandwich and to admire the view of Le Havre beach:
            <br/>
            <br/>
            <a href="http://photos.imageevent.com/motorbiker/newspics4/Zero-6-3.jpg" rel="thumbnail" title="Zero motorcycle at the Le Havre beach">
            </a>

            <br/>
            Arriving closer and closer to home, I was relieved. At the last mile the battery indicator went to the last bar (
            <em>
             it started flashing when there were only 2 bars left
            </em>
            ).
            <br/>
            <br/>
            Having arrived I took the photo of the dash:
            <br/>
            <br/>
            <a href="http://photos.imageevent.com/motorbiker/newspics4/Zero-6-4.jpg" rel="thumbnail" title="Zero motorcycle odometer and battery level">
            </a>
            <table align="center" border="0" cellpadding="5" cellspacing="0" width="560">

            <br/>
            As you can see I went
            <strong>
             145.6
            </strong>
            kilometers on almost the full battery.
            <strong>
             <br/>
             <br/>
             Summary:
            </strong>
            <br/>
            I have no idea how you can reach 190 kilometers range unless you ride like a moped. During the whole ride, I never opened the throttle and never exceeded 90 kph. But there was about 20 kilometers of a straight road limited at 90 kph. So maybe the bike uses too much power without an chance of regenerating the battery.
            <br/>
            <br/>
            But still, I enjoyed the ride, and know that I can make it and back, with little to spare.
