      <td>
       <p>
        BES has re-engineered the attachments to fit both the 3/16" and the 5/32" rods. They will still be compatible with all existing parts in the field, and the cost remains the same
       </p>
       <ul>
        <li class="text">
         <span class="text-links">
          FishHook
         </span>
         - Use to retrieve or snag wire and cable
        </li>
        <li class="text">
         <span class="text-links">
          Direction Tool
         </span>
         - Offset bend helps guide the Fiberfish around obstructions in walls
        </li>
        <li class="text">
         <span class="text-links">
          Fish Finder
         </span>
         - Locates Fiberfish inside walls
        </li>
        <li class="text">
         <span class="text-links">
          Magnet Head
         </span>
         - Use with Fish Finder or to catch the ball chain inside wall cavities
        </li>
        <li class="text">
         <span class="text-links">
          Pull Ring
         </span>
         - Used to attach multiple wires for larger pulls
        </li>
        <li class="text">
         <span class="text-links">
          Ball Chain
         </span>
         - Attach to Fiberfish for interior wall pulls, can be retrieved through small access hole in wall with magnet or Flex-Fish
        </li>
        <li class="text">
         <span class="text-links">
          Wisp Head
         </span>
         - Use over suspended ceilings, won't catch on T-bars
        </li>
        <li>
         <span class="text-links">
          Screw Tip
         </span>
         <span class="text">
          - Screw into the end of small communication wires for faster and easier pulls.
         </span>
        </li>
       </ul>
