           <div class="post-text" itemprop="text">
            <p>
             When I try to scrap
             <a href="http://www.oddsportal.com/baseball/usa/mlb/results/page/">
              this site
             </a>
             with Phantomjs, by default, Phantomjs send the following headers to server:
            </p>
            <pre>
             <code>
              "name":"User-Agent",
"value":"Mozilla/5.0 (Unknown; Linux i686) AppleWebKit/534.34 (KHTML, like Gecko) PhantomJS/1.9.1 Safari/534.34"}
             </code>
            </pre>
            <p>
             And I get an
             <code>
              status 405 "Not Allowed"
             </code>
             response.
            </p>
            <p>
             I read in the
             <strong>
              Phantomjs API Reference
             </strong>
             that in order to imitate a request coming from some other browser, I should change my User-Agent value. On Wikipedia I found the value I should use for pretending to be
             <strong>
              Firefox
             </strong>
             under
             <strong>
              Ubuntu
             </strong>
             :
            </p>
            <pre>
             <code>
              'name': 'User-Agent',
'value': 'Mozilla/5.0 (X11; Ubuntu; Linux i686; rv:16.0) Gecko/20120815 Firefox/16.0'
             </code>
            </pre>
            <p>
             In what part of Phantomjs should I put this properties? Where should I insert them - inside
             <strong>
              page.open
             </strong>
             , or inside
             <strong>
              page.evaluate
             </strong>
             , or at the top of it?
            </p>
           </div>

            <p>
             Actually, is on
             <code>
              page.settings
             </code>
             . Do it before the
             <code>
              open
             </code>
             .
            </p>
            <p>
             Here is an example using it against that page you linked:
            </p>
            <pre>
             <code>
              var page = require('webpage').create();
page.settings.userAgent = 'Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/28.0.1500.71 Safari/537.36';
page.open('http://www.oddsportal.com/baseball/usa/mlb/results/page/', function() {
    window.setTimeout(function() {
        var output = page.evaluate(function() {
            return document.getElementById('tournamentTable')
           .getElementsByClassName('deactivate')[0]
           .getElementsByTagName('a')[0]
           .textContent;
        });
        console.log(output);
    }, 1000);
});
             </code>
            </pre>
            <p>
             This example will scrape the match name in first row on the table. (Which, in this precise moment is "
             <code>
              San Francisco Giants - Boston Red Sox
             </code>
             ")
            </p>
            <hr/>
            <p>
             About your comment, actually you can use jquery under phantomjs! Check this example:
            </p>
            <pre>
             <code>
              var page = require('webpage').create();
page.settings.userAgent = 'Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/28.0.1500.71 Safari/537.36';
page.open('http://www.oddsportal.com/baseball/usa/mlb/results/page/', function() {
    window.setTimeout(function() {
        page.includeJs("http://ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js", function() {
            var output = page.evaluate(function () {
                return jQuery('#tournamentTable .deactivate:first a:first').text();
            });
            console.log(output);
        });
    }, 1000);
});
             </code>
            </pre>
            <p>
             By the way, for waiting, instead of the
             <code>
              window.setTimeout
             </code>
             I used on this examples, I recommend you to use
             <a href="https://github.com/ariya/phantomjs/blob/master/examples/waitfor.js">
              waitfor.js
             </a>
             instead.
            </p>
