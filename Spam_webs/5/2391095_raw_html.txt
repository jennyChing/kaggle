       <div class="txt step-body">
        <br/>
        <strong>
         .22 caliber rifle
        </strong>
        (pictured Mossberg 802 Plinkster Bolt Action) semi-auto best but bolt action usuable (to anyone who thinks the 22lr is not powerful enough watch this http://www.youtube.com/watch?v=UyU1l5KgqA4&amp;feature=channel_video_title
        <br/>
        <strong>
         9mm, .40 caliber, or .45 caliber Pistol
        </strong>
        (example 4th generation Glock 19 or Colt 1911)
        <br/>
        <strong>
         500 rounds or more of .22 caliber ammunition
        </strong>
        (best because you can carry many more rounds than larger ammo, cheap to get, and easy to find)
        <br/>
        <strong>
         100 rounds or more of your pistol ammunition
        </strong>
        <br/>
        <strong>
         Crowbar
        </strong>
        (best bludgeon weapon as the curved side can be used to crack zombies head, pointed to stab through temple to brain)
        <br/>
        <strong>
         Slingshot
        </strong>
        (ball bearings and lead shot .54 caliber best, silent also)
        <br/>
        <strong>
         Aluminum Baseball Bat
        </strong>
        (aluminum best as it won't crack or break like a wooden one)
        <br/>
        <strong>
         Trench Knife
        </strong>
        (perfect for close quarter combat)
        <br/>
        <strong>
         Machete
        </strong>
        (multiple use weapon effective for close quarters, creating shelter, etc.)
        <br/>
        <strong>
         Canned food, dry food such as rice, oats, rye, flour, etc
         <br/>
         Magnesium fire starter, disposable lighter
        </strong>
        (in case of fire needed to be created fast)
        <br/>
        <strong>
         Multi-tool
         <br/>
         Heavy boots
        </strong>
        (resistant to zombie bites)
        <br/>
        <strong>
         Earmuffs
        </strong>
        (used to tune out the zombie moans)
        <br/>
        <strong>
         Mountain Bike
        </strong>
        (with bicycle pump and extra innertube)
        <br/>
        <strong>
         Hammer
        </strong>
        (useful for barricade construction, digging, and a last ditch zombie killing weapon)
        <br/>
        <strong>
         Camelback
         <br/>
         Fishing Rod
         <br/>
         Backpack
        </strong>
        (ones like in this link are ideal http://grizzlymike.com/military/PACKS.html)
        <br/>
        <strong>
         Poncho
        </strong>
        <br/>
        <strong>
         Gun Cleaning Kit
         <br/>
         Survival Radio
        </strong>
        (rechargeable from solar power, human power)
