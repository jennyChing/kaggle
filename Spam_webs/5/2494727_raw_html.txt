      <p>
       <b>
        osFileManager
       </b>
       is an open source file management script written in PHP.
		Includes User CP, Admin CP, and many basic file creation/modifying tools. File Functions include: List, Open, View, Edit, Create, Upload, Rename and Move. User Functions include: Change password, and Change color scheme. Admin Functions include: New user, Edit user, Delete user.
       <br/>
       <br/>
       osFileManager requires PHP 4 or greater, and MySQL 3 or greater. If your webhost does not support either of those, consider
       <a href="http://www.linode.com/?r=2fcc9c05d3c2a544b99566504db8f86c215fdb0b" target="_blank">
        signing up for one that does
       </a>
       . osFileManager is released under the GNU GPL license.
      </p>
