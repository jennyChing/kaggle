       <h1 class="title">
        <a href="http://zipso.net/how-to-fix-list-item-indentation-in-wordpress/" rel="bookmark" title="Permalink to How to fix List Item Indentation in WordPress">
         How to fix List Item Indentation in WordPress
        </a>
       </h1>
       <div class="top-meta clearfix">
        <span>
         February 26, 2014 at 7:34 pm
        </span>
       </div>
       <div class="entry">
        <!--
						&lt;div style="float:right; display:block;width:180px; height:150px; border:1px solid #000;"&gt;
						&lt;/div&gt;
-->
        <h3 class="nomargin">
         <img alt="list-items" class="alignleft size-large wp-image-322" height="325" src="http://static1.zipso.net/wp-content/uploads/2014/02/list-items-1024x513.jpg" width="650"/>
        </h3>
        <h3 class="nomargin">
         Making your blog or website look attractive is often a case of fixing a large number of minor cosmetic issues.
        </h3>
        <h3 class="nomargin">
         The default styling in some WordPress themes can be also be a hindrance.
        </h3>
        <h3 class="nomargin">
         Here we show you how to change the position of bullet points in list items to work better with multi-line text.
        </h3>
        <p>
         Lets say that you have a list of items on your page. The problem happens when the text of one of the items wraps around onto a new line  the text becomes aligned with the bullet point instead of the line above it. It could look something like this:
        </p>
        <p style="text-align: center;">
         <img alt="list-inside" class="size-full wp-image-290 aligncenter bigborder" height="172" src="http://static1.zipso.net/wp-content/uploads/2014/02/list-inside.png" style="margin-top: 25px; margin-bottom: 25px;" width="520"/>
        </p>
        <div style="float:right;margin:8px 0px 8px 8px;">
         <div style="margin-right:9px">
          <!-- zipso post para2 -->
          <ins class="adsbygoogle" data-ad-client="ca-pub-8959601970688211" data-ad-slot="1606386652" style="display:inline-block;width:240px;height:160px">
          </ins>
          <script>
           (adsbygoogle = window.adsbygoogle || []).push({});
          </script>
         </div>
        </div>
        <p>
         Whats happening is that
         <strong>
          list-style-position: inside
         </strong>
         is being set somewhere in your themes stylesheet.
        </p>
        <p>
         In our case, it was being set as part of the 
         <strong>
          .entry
         </strong>
          CSS class.
        </p>
        <p>
         Anything in the entry class controls how the content of a post or a page is displayed. It styles everything within the post below the title, but it doesnt affect the rest of the page such as the sidebar and heading etc.
        </p>
        <p>
         Changing list-style-position to
         <strong>
          outside
         </strong>
         will now shift the bullet points left, so that the text on each line is aligned.
        </p>
        <div class="codebox">
         .entry li { list-style-position: outside; }
        </div>
        <p>
         Now the list looks like this:
        </p>
        <p style="text-align: center;">
         <img alt="list-outside" class="size-full wp-image-294 aligncenter bigborder" height="163" src="http://static1.zipso.net/wp-content/uploads/2014/02/list-outside.png" style="margin-top: 25px; margin-bottom: 25px;" width="520"/>
        </p>
        <p>
         Note that this change will actually shift your list to the left, so you may need to compensate for this by increasing the padding value. (In some cases, the bullet points may actually appear cut-off due to being moved too far left.)
        </p>
        <p>
         This could be done with something like the following:
        </p>
        <div class="codebox">
         .entry ul { padding-left: 7px; }
        </div>
        <h3>
         Editing the stylesheet
        </h3>
        <p>
         To add your own custom CSS, you can either
         <a href="http://codex.wordpress.org/Child_Themes" target="_blank">
          create a child theme
         </a>
         , or use a
         <a href="http://wordpress.org/plugins/simple-custom-css/" target="_blank">
          Custom CSS WordPress plugin
         </a>
         .
        </p>
        <p>
         Some themes may have a file such as custom.css for custom changes. You can usually directly edit this, and it wont be overwritten if you update the theme.
        </p>
        <p>
         However, be careful about directly editing the style.css file, since you may lose changes made here after updates.
        </p>
        <p>
         Not using WordPress? You can still go ahead and use these modifications in the stylesheet of your website or blog.
        </p>
        <h3>
         Bonus style tip!
        </h3>
        <p>
         The standard HTML bullet points are a little ugly, so you may think of replacing them with something else. You can use an image instead, and theres a good guide
         <a href="http://www.techforluddites.com/2014/01/replacing-list-bullets-with-images-using-css.html" target="_blank">
          here
         </a>
         on how to do this.
        </p>
        <p>
         However, with more devices using Retina style displays, this can be a problem, since the text on the page may be rendered at a different resolution to your images. This could make your bullet-point image look a little blurry. Fortunately, its possible to use special characters as bullet points too.
        </p>
        <p>
         For example, this is using a double-chevron as the bullet:
        </p>
        <p style="text-align: center;">
         <img alt="list-character" class="size-full wp-image-304 aligncenter bigborder" height="158" src="http://static1.zipso.net/wp-content/uploads/2014/02/list-character.png" style="margin-top: 25px; margin-bottom: 25px;" width="520"/>
        </p>
        <p>
         To use this, first disable the normal list styling (adapt for your stylesheet):
        </p>
        <div class="codebox">
         ul { padding-left: 30px; list-style:none; }
        </div>
        <p>
         Now add in the chevron with something like this:
        </p>
        <div class="codebox">
         li:before { content: \00BB \0020; font-weight:bold; float:left; margin-left:-1em;}
        </div>
        <p>
         
         <strong>
          :before
         </strong>
          is a special selector which can be used to insert content before each list item.
        </p>
        <p>
         The 
         <strong>
          \00BB
         </strong>
          is the hex code for the chevron character, and 
         <strong>
          \0020
         </strong>
          is simply a space. The float and margin attributes ensure that the lists are indented correctly.
        </p>
        <p>
         You can also change the colour and font size of the bullet point here too, just like any normal text.
        </p>
        <p>
         With this setup, thelist-style-position modifier is no longer used, since were no longer using the built-in list styles. Instead, you can tweak margin and padding settings in the code above if you need different indentation.
        </p>
        <p>
        </p>
        <p>
         Do you have any tips for styling lists in WordPress?
        </p>
        <p>
         Let us know in the comments below.
        </p>
