       <td>
        <p>
         <a class="external text" href="http://dlang.org/index.html" rel="nofollow">
          The D Programming Language
         </a>
         has been said to be "what C++ wanted to be," which is a better C. D is developed with system level programming in mind, but brings to the table modern language design with a simple C-like syntax. For these reasons D makes for a good language choice for both performance code and application development.
        </p>
        <p>
         D is rapidly reaching a stable
         <a class="external text" href="http://dlang.org/spec.html" rel="nofollow">
          specification
         </a>
         and
         <a class="external text" href="http://dlang.org/download.html" rel="nofollow">
          implementation
         </a>
         .
        </p>
        <p>
         <a class="external text" href="http://www.amazon.com/exec/obidos/ASIN/0321635361/classicempire" rel="nofollow">
          "The D Programming Language"
         </a>
         by Andrei Alexandrescu is available on Amazon and other locations.
        </p>
       </td>
