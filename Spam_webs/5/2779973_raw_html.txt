            <div class="article-content-inner">
             <h2>
              10 Perfect Summer Drinks
             </h2>
             <p class="author">
              As prepared by
              <span class="name">
               S Caron
               <span class="author">
                ,
               </span>
              </span>
              <span class="date" content="2011-06-17" itemprop="datePublished">
               June 17, 2011
              </span>
             </p>
             <div class="generated-content" itemprop="articleBody">
              We all love homemade sun-tea and lemonade. But when Friday rolls around, what you really want is something a little more  adult. These 10 picks are perfect for summertime (and #10 is just for the kids).
              <br/>
              <h3>
               1.
               <a href="http://www.tablespoon.com/recipes/coconut-blue-hawaiian-recipe/1/" title="Coconut Blue Hawaiian Recipe">
                Coconut Blue Hawaiian
               </a>
              </h3>
              <br/>
              These frosty blue cocktails are so perfect for those sweltering summer days. A mix of coconuts, blue curacao and rum  doesnt it just make you swoon?
              <br/>
              <br/>
              <a href="http://www.tablespoon.com/recipes/coconut-blue-hawaiian-recipe/1/" title="Coconut Blue Hawaiian Recipe">
               <img alt="Coconut Blue Hawaiian Summer Drink Recipe" class="alignnone size-full wp-image-11086" height="200" src="/-/media/Images/Articles/PostImages/2011/06/week3/2011-06-17-1-summer-drinks-blue-hawaiian.jpg" width="275"/>
              </a>
              <br/>
              <h3>
               2.
               <a href="http://www.tablespoon.com/recipes/banana-daiquiri-slush-recipe/1/" title="Banana Daiquiri Slush Recipe">
                Banana Daiquiri Slush
               </a>
              </h3>
              <br/>
              With just four ingredients, this easy icy drink is a fun twist on a summer classic.
              <br/>
              <br/>
              <a href="http://www.tablespoon.com/recipes/banana-daiquiri-slush-recipe/1/" title="Banana Daiquiri Slush Recipe">
               <img alt="Banana Daiquiri Slush Summer Drink Recipe" class="alignnone size-full wp-image-11087" height="200" src="/-/media/Images/Articles/PostImages/2011/06/week3/2011-06-17-2-summer-drinks-banana-daiquiri.jpg" width="275"/>
              </a>
              <br/>
              <h3>
               3.
               <a href="http://www.tablespoon.com/recipes/sangria-recipe/2/" title="Sangria Recipe">
                Sangria
               </a>
              </h3>
              <br/>
              Sangria is totally the drink of summertime. This easy version is tinted with citrus in a perfect adults-only punch.
              <br/>
              <br/>
              <a href="http://www.tablespoon.com/recipes/sangria-recipe/2/" title="Sangria Recipe">
               <img alt="Sangria Summer Drink Recipe" class="alignnone size-full wp-image-11102" height="200" src="/-/media/Images/Articles/PostImages/2011/06/week3/2011-06-17-3-summer-drinks-sangria.jpg" width="275"/>
              </a>
              <br/>
              <h3>
               4.
               <a href="http://www.tablespoon.com/recipes/bathtub-gin-fizz-recipe/1/" title="Bathtub Gin Fizz Recipe">
                Bathtub Gin Fizz
               </a>
              </h3>
              <br/>
              Classic cocktails like the Gin Fizz are so fun to break out and enjoy on hot nights. Double the recipe to imbibe a crowd.
              <br/>
              <br/>
              <a href="http://www.tablespoon.com/recipes/bathtub-gin-fizz-recipe/1/" title="Bathtub Gin Fizz Recipe">
               <img alt="Bathtub Gin Fizz Summer Drink Recipe" class="alignnone size-full wp-image-11089" height="200" src="/-/media/Images/Articles/PostImages/2011/06/week3/2011-06-17-4-summer-drinks-bathtub-gun-fizz.jpg" width="275"/>
              </a>
              <br/>
              <h3>
               5.
               <a href="http://www.tablespoon.com/recipes/mexican-margaritas-recipe/1/" title="Mexican Margaritas Recipe">
                Mexican Margaritas
               </a>
              </h3>
              <br/>
              This south-of-the-border drink is so refreshing  as long as you can handle the tequila.
              <br/>
              <br/>
              <a href="http://www.tablespoon.com/recipes/mexican-margaritas-recipe/1/" title="Mexican Margaritas Recipe">
               <img alt="Mexican Margarita Summer Drink Recipe" class="alignnone size-full wp-image-11090" height="200" src="/-/media/Images/Articles/PostImages/2011/06/week3/2011-06-17-5-summer-drinks-mexican-margarita.jpg" width="275"/>
              </a>
              <br/>
              <h3>
               6.
               <a href="http://www.tablespoon.com/recipes/white-wine-sangria-recipe/1/" title="White Wine Sangria Recipe">
                White Wine Sangria
               </a>
              </h3>
              <br/>
              White wine is my favorite for summertime, and this version of sangria (with peaches!) is a perfect way to feature it.
              <br/>
              <br/>
              <a href="http://www.tablespoon.com/recipes/white-wine-sangria-recipe/1/" title="White Wine Sangria Recipe">
               <img alt="White Wine Sangria Summer Drink Recipe" class="alignnone size-full wp-image-11091" height="200" src="/-/media/Images/Articles/PostImages/2011/06/week3/2011-06-17-6-summer-drinks-white-wine-sangria.jpg" width="275"/>
              </a>
              <br/>
              <h3>
               7.
               <a href="http://www.tablespoon.com/recipes/blueberry-hard-lemonade-recipe/1/" title="Blueberry Hard Lemonade Recipe">
                Blueberry Hard Lemonade
               </a>
              </h3>
              <br/>
              Lemonade is a summer classic  and hard lemonade is the mouthwatering adult version. This fizzy version uses fresh blueberries and mint -- making it perfect for July when blueberries are in season.
              <br/>
              <br/>
              <a href="http://www.tablespoon.com/recipes/blueberry-hard-lemonade-recipe/1/" title="Blueberry Hard Lemonade Summer Drink Recipe">
               <img alt="Blueberry Hard Lemonade Summer Drink Recipe" class="alignnone size-full wp-image-11092" height="200" src="/-/media/Images/Articles/PostImages/2011/06/week3/2011-06-17-7-summer-drinks-blueberry-hard-lemonade.jpg" width="275"/>
              </a>
              <br/>
              <h3>
               8.
               <a href="http://www.tablespoon.com/recipes/frozen-strawberry-margaritas-recipe/2/" title="Frozen Strawberry Margarita Recipe">
                Frozen Strawberry Margarita
               </a>
              </h3>
              <br/>
              Frozen? Strawberry? Margarita? SOLD!
              <br/>
              <br/>
              <a href="http://www.tablespoon.com/recipes/frozen-strawberry-margaritas-recipe/2/" title="Frozen Strawberry Margaritas Summer Drink Recipe">
               <img alt="Frozen Strawberry Margarita Summer Drink Recipe" class="alignnone size-full wp-image-11093" height="200" src="/-/media/Images/Articles/PostImages/2011/06/week3/2011-06-17-8-summer-drinks-frozen-strawberry-margarita.jpg" width="275"/>
              </a>
              <br/>
              <h3>
               9.
               <a href="http://www.tablespoon.com/recipes/kiwi-lime-rum-slush-recipe/1/" title="Kiwi Lime Run Slush Recipe">
                Kiwi-Lime-Rum Slush
               </a>
              </h3>
              <br/>
              Have I mentioned how much I love slushy drinks in summertime? This spiked one even has delicious kiwifruit in it.
              <br/>
              <br/>
              <a href="http://www.tablespoon.com/recipes/kiwi-lime-rum-slush-recipe/1/" title="Kiwi Lime Rum Slush Recipe">
               <img alt="Kiwi Lime Rum Slush Summer Drink Recipe" class="alignnone size-full wp-image-11094" height="200" src="/-/media/Images/Articles/PostImages/2011/06/week3/2011-06-17-9-kiwi-lime-rum-slush.jpg" width="275"/>
              </a>
              <br/>
              <h3>
               10.
               <a href="http://www.tablespoon.com/recipes/watermelon-lemonade-with-kiwifruit-splash-recipe/1/" title="Watermelon Lemonade with Kiwifruit Splash Recipe">
                Watermelon Lemonade with Kiwifruit Splash
               </a>
              </h3>
              <br/>
              This one is for the kids. Its a fun and totally virgin lemonade with watermelon and kiwi. Perfect.
              <br/>
              <br/>
              <a href="http://www.tablespoon.com/recipes/watermelon-lemonade-with-kiwifruit-splash-recipe/1/" title="Lemonade with Kiwifruit Splash Recipe">
               <img alt="Watermelon Lemonade Summer Drink Recipe" class="alignnone size-full wp-image-11095" height="200" src="/-/media/Images/Articles/PostImages/2011/06/week3/2011-06-17-10-summer-drinks-watermelon-lemonade.jpg" width="275"/>
              </a>
