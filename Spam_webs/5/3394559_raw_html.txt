     <h1>
      How a Pure Sine Wave Power Inverter Can Help Field Researchers and Workers
     </h1>
     <p class="text-muted">
      Author: Jeffrey R.  
    Date Posted:4 July 2014
     </p>
     <hr/>
     <!--  &lt;img src="/assets/pixel.gif" class="pull-right"/&gt;
    &lt;img src="/assets/pixel.gif"  class="pull-right"/&gt;-->
     <p class="ccontent">
     </p>
     <p class="ccontent2">
      Thanks to a
      <a href="http://www.elinz.com.au/shop/inverters/227">
       <strong>
        pure sine wave power inverter
       </strong>
      </a>
      , working onsite is no longer as complicated as before. This is one of the primary reasons why many field researchers and those who are often tasked to do field work love this nifty piece of equipment. If you often find yourself being sent to different places all over the country and being asked to work onsite for an extended period of time, you should consider
      <a href="http://www.elinz.com.au/buy/pure-sine-wave-power-inverter-1500w-3000w-24v-240v/INTP24V1500">
       <strong>
        investing in a good power inverter
       </strong>
      </a>
      .
      <br/>
      <br/>
      As you may know, it isn't easy to do field work. You have to spend long periods of time being out in the open and with no access to necessities that can make your job and your life much easier. Being a field researcher is like being sent to an extended camping trip without having a clear idea when you will be returning to civilisation. Fortunately, a
      <a href="http://www.elinz.com.au/buy/pure-sine-wave-power-inverter-2500w-5000w-24v-240v/INTP24V2500">
       <strong>
        pure sine wave inverter
       </strong>
      </a>
      can offer you reprieve from the challenges that you face on daily basis.
      <br/>
      <br/>
      While they fundamentally serve the same purpose, which is to provide you with a mobile power source, there are things a pure sine wave power inverter can do that a conventional power generator simply cannot. Being a researcher or a field worker, you probably have a lot of equipment, gadgets, and gizmos that you need to lug around, and carrying the extra weight of a big, hulking power generator won't certainly help. Finding storage for it when you're done could be a problem, too.
      <br/>
      <br/>
      Good thing this is something that you don't have to deal with if you have a power inverter instead. It is portable and easy to carry so transporting it to different places won't be a headache. Storage is easy, too. Because it is small and lightweight, you can keep it in the boot of your car or other similar places.
      <br/>
      <br/>
      Another reason why a pure sine wave power inverter is a huge help to field researchers and workers is that it is easy to set up. If you need to quickly connect your laptop or a piece of electronic equipment to a power source because it ran out of power, you can simply connect it to the inverter, then connect the latter to your car's power source, and you're all set. There's no need to pull out that heavy power generator from where it is hidden and decide how much petrol you need to put in it.
      <br/>
      <br/>
      A pure sine waver power inverter definitely makes things much easier and more convenient for those who are constantly working on the field. If you want to make working onsite less stressful, now is a great time to invest in such a reliable and portable mobile power source.

		var recent_sales_offset = 0;
		var recent_sales_fifo = [];
		 var recent_sales_show = (function(){
			if (recent_sales_fifo.length==0) {
				if (++recent_sales_count &gt;15) {
					// console.log('ran out of recents, hit limit, no more');
					clearInterval(recent_sales_interval);
					$('#recent_sales_ticker').addClass('hidden');
				} else {
					console.log('ran out of recents, getting more, since',recent_sales_last_ts);
					var obj = {};
					if (recent_sales_last_ts) {
						obj['ts'] = recent_sales_last_ts;
					}
					$.do_ajax(
						'recent_sales',
						obj,
						true,
						{
							'SUCCESS': {
								'def' : { 'list':[] },
								'fn' : function (code, data) {
	//								console.log(code, data);
									if (data['list'].length &gt; 0) {
										recent_sales_fifo = data['list'];
										recent_sales_last_ts = data['ts'];
										recent_sales_offset = ((new Date())-(new Date(data['now']*1000)))/1000;
										
										recent_sales_show();
									}
								}
							},
							'FAIL': {
								'def' : { 'msg': '' },
								'fn' : function (code, data) {
								}
							},
							'ERROR': {
								'def' : { 'msg': '' },
								'fn' : function (code, data) {
								}
							}
						}
					);
				}
			} else {
				var d = recent_sales_fifo.shift();
				
				var ddf = Math.round(((new Date())-(new Date(d['ts']*1000))-recent_sales_offset)/1000);			
				var ddf1;
				var ago_text = 'recently';
				
				if (ddf&lt;60) 
					{ 
					ago_text = ddf + ' second'+(ddf==1?'':'s')+' ago'; 
					
					}
						else if (ddf&lt;3600) 
						{ 
						ddf1 = Math.round(ddf/60); 
							ago_text = ddf1 + ' minute'+(ddf==1?'':'s')+' ago';

						}
								else if (ddf&gt;3600 &amp;&amp; ddf&lt;=7200)
									{ 
									ddf1 = Math.round(ddf/3600); 
									ago_text = ddf1 + ' hour'+(ddf==1?'':'s')+' ago'; 
									
									}
									
									
									
			if(ddf&lt;=7200)
				{
			
					$('#recent_sales_content_img').html('&lt;a href="'+d['url']+'"&gt;&lt;img src="'+d['img']+'"&gt;&lt;/a&gt;');
					$('#recent_sales_content_description').html('Someone in &lt;span style="text-transform: capitalize"&gt;'+d['city'].toLowerCase()+'&lt;/span&gt; bought a &lt;a href="'+d['url']+'"&gt;'+d['description']+'&lt;/a&gt; &lt;span class="ago"&gt;'+ago_text+'&lt;/span&gt;');
					$('#recent_sales_ticker').removeClass('fadeOutLeft hidden');
					$("#recent_sales_ticker").removeClass('not-show');
					setInterval(function() { 
						$("#recent_sales_ticker").hide(); 
						setTimeout(function() {     
						   $("#recent_sales_ticker").show();     
						},7000);
					},7000); 
					
				}
				else
				{ 
										
					$("#recent_sales_ticker").addClass('not-show');
				
				
				}
				
			
				}  
				});
			$(function(){

				$('body').append($('&lt;div class="arrow-cnt"&gt;&gt;&lt;/div&gt;&lt;div id="recent_sales_ticker" class="fadeOutLeft animated fadeInLeft hidden"&gt;&lt;div id="recent_sales_close" class="btn pull-right"&gt;&times;&lt;/div&gt;&lt;div id="recent_sales_content"&gt;&lt;table&gt;&lt;tr&gt;&lt;td id="recent_sales_content_img"&gt;&lt;/td&gt;&lt;td id="recent_sales_content_description"&gt;&lt;/td&gt;&lt;/tr&gt;&lt;/table&gt;&lt;/div&gt;&lt;/div&gt;'));
				$('#recent_sales_close').click(function(){
					clearInterval(recent_sales_interval);
					
					$('#recent_sales_ticker').hide(5000);
					$('.arrow-cnt').show();
				});
				$('.arrow-cnt').click(function(){
					$('#recent_sales_ticker').show(5000);
					$('.arrow-cnt').hide();
				});
				 setTimeout(function () {

					recent_sales_show();
					recent_sales_interval = setInterval(recent_sales_show, 10000);
				},7000);   
			

		});
	})();
	}
