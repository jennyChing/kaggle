                    <h3 class="post-title entry-title">
                     Creating Your ultimate Ubuntu 7.10 Desktop
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                      <span class="post-comment-link">
                      </span>
                      <span class="post-labels">

                       <a href="http://linuxondesktop.blogspot.com/search/label/Tutorials" rel="tag">

                       <a href="http://linuxondesktop.blogspot.com/search/label/Ubuntu" rel="tag">

                       <a href="http://linuxondesktop.blogspot.com/search/label/Ubuntu%207.10%20Tutorials" rel="tag">

                    <div class="post-body entry-content" id="post-body-7117278886377615119">
                     <div style="text-align: center;">
                      <span style="color: rgb(0, 0, 102);font-size:130%;">
                       Creating Your ultimate Ubuntu 7.10 Desktop
                      </span>
                      <br/>
                     </div>
                     <br/>
                     Ubuntu  7.10 Codenamed Gusty Gibbon was released few weeks back with much fanfare ,now Ubuntu for past couple of years has been one of the most popular Desktop Linux distributions . Ubuntu 7.10 includes a number of features , applets and Wizards to simplify desktop Linux experience however because of many licensing restrictions it does not include out of the box support for popular audio/video codecs and many commercial but good applications . Also because of the limitation of trying to accommodate  all kind of applications on  single CD Ubuntu comes with a limited set of applications  .
                     <br/>
                     <br/>
                     Now , i often have arguments with my friends who are new to computers that windows is better , mac is better and blah blah blah and they often use this argument that it's difficult for newbies
                     <adsense>
                      to   install applications  , codecs etc on Linux  as a way to justify their arguments . So well i decided being such a Linux fanatic i have to do something about it so well i thought of writing a guide sort of step by step tutorial that would allow anyone to have a kicking Linux desktop with most of the codecs enabled and descent set of applications  in few steps .
                      <br/>
                      <br/>
                      Steps Written in RED are of paramount importance and other steps may not work correctly if these steps are not followed .
                      <br/>
                      <br/>
                      <span style="color: rgb(255, 0, 0);">
                       Step 1 : - Enabling Additional Repositories
                      </span>
                      <br/>
                      <br/>
                      Now many applications need additional repositories to be installed or some to be enabled in Synaptic package manager so before trying out steps given below ensure that repositories in order.
                      <br/>
                      <br/>
                      Launch Synaptic Package Manager (
                      <span style="color: rgb(0, 0, 102);">
                       System -&gt; Administration -&gt; Synaptic Package Manager
                      </span>
                      ) , then in Synaptic package manager go to (
                      <span style="color: rgb(0, 0, 102);">
                       Settings -&gt; Repositories
                      </span>
                      <span style="font-weight: bold; color: rgb(0, 0, 102);">
                      </span>
                      ) you will find window like this . Ensure that all the check boxes are marked leaving source code(if you want to you can enable this also but you are not going to need this unless you are software developer) the dialog box should look like this .
                      <br/>
                     </adsense>
                     <div style="text-align: center;">
                      <a href="http://bp1.blogger.com/_crimgO_xQv0/R1Rie-4cbZI/AAAAAAAAAjQ/UYVVDrLfX-M/s1600-R/synaptic-software-sources.png" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                       <img alt="" border="0" id="BLOGGER_PHOTO_ID_5139841359151918482" src="http://bp1.blogger.com/_crimgO_xQv0/R1Rie-4cbZI/AAAAAAAAAjQ/6nrLZX32TFs/s320/synaptic-software-sources.png" style="margin: 0px auto 10px; display: block; text-align: center; cursor: pointer;"/>
                      </a>
                      <span style="color: rgb(204, 0, 0);font-size:78%;">
                       Dialog Box showing the repositories that should be enabled
                       <br/>
                       <br/>
                      </span>
                      <div style="text-align: left;">
                       <span style="color: rgb(0, 0, 0);">
                        After completing above step you will find a dialog box like this
                        <br/>
                        <span style="font-weight: bold;">
                         <br/>
                        </span>
                       </span>
                      </div>
                     </div>
                     <a href="http://bp2.blogger.com/_crimgO_xQv0/R1Ri_O4cbaI/AAAAAAAAAjY/VEY1dg3yMYo/s1600-R/synaptic-update.png" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5139841913202699682" src="http://bp2.blogger.com/_crimgO_xQv0/R1Ri_O4cbaI/AAAAAAAAAjY/acl1ohxpCPA/s320/synaptic-update.png" style="margin: 0px auto 10px; display: block; text-align: center; cursor: pointer;"/>
                     </a>
                     <br/>
                     Besides these you may also like to add medibuntu repositories if you want to install applications like -------  , to add mediubuntu repositories follow the following steps
                     <br/>
                     <br/>
                     <blockquote>
                      echo "deb http://packages.medibuntu.org/ gutsy free non-free" | sudo tee -a /etc/apt/sources.list
                      <br/>
                     </blockquote>
                     <br/>
                     <div style="text-align: center;">
                      <span style="font-style: italic; font-weight: bold;">
                       and
                      </span>
                      <br/>
                     </div>
                     <br/>
                     <blockquote>
                      wget -q http://packages.medibuntu.org/medibuntu-key.gpg -O- | sudo apt-key add - &amp;&amp; sudo apt-get update
                     </blockquote>
                     <br/>
                     <br/>
                     <br/>
                     <span style="font-weight: bold;">
                      2. Enabling Cool Graphics Effects
                     </span>
                     <br/>
                     <br/>
                     Unlike previous releases of Ubuntu where one had to manually download and install compiz,beryl etc  (Compiz Fusion is composite window manager that provides best features of Compiz and Beryl) from Repositories if one wanted to have glitzy visual effects .  Ubuntu 7.10 comes with Compiz Fusion pre-installed and on supported hardware offers a wide array of Visual Effect . Now depending on graphic hardware of computer one could chose from three level of Visual Effect (From
                     <span style="font-weight: bold;">
                      System -&gt; Preferences -&gt; Appearance
                     </span>
                     )
                     <br/>
                     <br/>
                     None : - This mode causes Ubuntu to use Metacity instead of Compiz Fusion , with no visual effect
                     <br/>
                     <br/>
                     Basic : -  Has only simple visual effects like shadows , fading windows-menus etc
                     <br/>
                     <br/>
                     Advanced : - Recommended for PC with descent graphic hardware , enables effects like wobbly windows, transparency , animated workspace switching etc
                     <br/>
                     <div style="text-align: center;">
                      <a href="http://bp0.blogger.com/_crimgO_xQv0/RzRW91_XuqI/AAAAAAAAAhA/n8mJPZjRIYs/s1600-h/desktop-compiz.png" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                       <img alt="" border="0" id="BLOGGER_PHOTO_ID_5130821495946721954" src="http://bp0.blogger.com/_crimgO_xQv0/RzRW91_XuqI/AAAAAAAAAhA/n8mJPZjRIYs/s320/desktop-compiz.png" style="margin: 0px auto 10px; display: block; text-align: center; cursor: pointer;"/>
                      </a>
                      <span style="color: rgb(255, 0, 0);font-size:78%;">
                       Visual Effect Dialog Box
                      </span>
                      <br/>
                     </div>
                     Now , the biggest advantage of Compiz Fusion is even on really slow hardware one can have descent set of visual effects even on my slow laptop i could use Extra graphics effects though it was painful while using application like OpenOffice because of my slow hardware.
                     <br/>
                     <br/>
                     However , compiz-fusion is capable of much more and you can enable more desktop effects /customize compiz by typing the following command in the terminal window : -
                     <br/>
                     <br/>
                     <blockquote>
                      sudo apt-get install compizconfig-settings-manager
                     </blockquote>
                     After completing above step , you can customize compiz by going to
                     <b>
                      System &gt; Preferences &gt; Advanced Desktop Effects Settings .
                     </b>
                     <br/>
                     <br/>
                     <span style="font-weight: bold;">
                      3. Installing Audio/Video Codecs
                     </span>
                     <br/>
                     <br/>
                     The first thing anyone who wants to use  Ubuntu as a replacement to their windows operating system would like to do is install support for all kind of audio/video multimedia codecs . Ubuntu does not come installed with support for major audio/video formats because of licensing issue and other issues . To install support for audio/video codecs type in the following command in the terminal window : -
                     <br/>
                     <br/>
                     <blockquote>
                      <span style="font-size:100%;">
                       sudo apt-get install gstreamer0.10-plugins-bad gstreamer0.10-ffmpeg  gstreamer0.10-plugins-ugly
                       <br/>
                       <span class="fullpost">
                       </span>
                      </span>
                     </blockquote>
                     And if you want to install mp3 support in nautilus you may like to install mpg123 which is a nice command line based utility for playing mp3 files and it works well even on modest hardware .
                     <br/>
                     <br/>
                     <blockquote>
                      sudo apt-get install mpg123
                     </blockquote>
                     <br/>
                     This would enable most of the popularly used codecs in the Totem Movie player which uses the GStreamer framework to play audio/video files.
                     <br/>
                     <br/>
                     If you want to use a different media player and framework you might try installing MPlayer or VLC (depending on which you prefer ) instructions on installing them are listed below .
                     <br/>
                     <br/>
                     <span style="font-weight: bold;">
                      Installing VLC Player : -
                     </span>
                     <br/>
                     <br/>
                     <span class="fullpost" style="font-size:100%;">
                      LC Player is another of popular video player available under Linux . It is released under GNU General Public license and is available for different platform including Windows,Linux,BeOS,Mac OS X etc. VLC player is based upon free open source libraries like libdvdcss,FFMpeg for decoding various video formats.
                     </span>
                     <span class="fullpost" style="font-size:100%;">
                      One Important feature of VLC Player is it's ability to play files over NetWork Protocols. The Frontend of VLC player is created using wxWidgets toolkit and it's appearance can be changed by using different skins. One Popular feature of VLC Player is it's ability to play files that are incomplete/broken or partially downloaded , making it useful for previewing file while downloading on file-sharing networks.
                      <br/>
                      <br/>
                      To install VLC Player type the following command in the terminal window : -
                      <br/>
                      <br/>
                     </span>
                     <div style="text-align: center;">
                      <span class="fullpost" style="font-size:100%;">
                       <blockquote>
                        sudo apt-get install vlc
                       </blockquote>
                      </span>
                     </div>
                     <br/>
                     After completing above step launch vlc player from (
                     <span style="font-weight: bold;">
                      Applications -&gt; Sound and Video -&gt; VLC media player
                     </span>
                     )
                     <br/>
                     <br/>
                     <br/>
                     <span style="font-weight: bold;">
                      3. Installing Linux DC++ Client
                     </span>
                     <br/>
                     <br/>
                     DC++ is a popuar tool used for p2p file sharing and is especially popular in college campuses , to install dc++ for linux follow the following instructions .
                     <br/>
                     <br/>
                     Type the following command in the terminal window: -
                     <br/>
                     <br/>
                     <blockquote>
                      sudo apt-get install linuxdcpp
                      <br/>
                     </blockquote>
                     And after  completing above step launch dcpp from (
                     <span style="font-weight: bold;">
                      Applications -&gt; Internet -&gt; DC++
                     </span>
                     )
                     <br/>
                     <div style="text-align: center;">
                      <a href="http://bp0.blogger.com/_crimgO_xQv0/R1Rowu4cbbI/AAAAAAAAAjg/trq35HrVbTQ/s1600-R/linuxdcpp.png" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                       <img alt="" border="0" id="BLOGGER_PHOTO_ID_5139848261164363186" src="http://bp0.blogger.com/_crimgO_xQv0/R1Rowu4cbbI/AAAAAAAAAjg/eRd33_ijv6M/s320/linuxdcpp.png" style="margin: 0px auto 10px; display: block; text-align: center; cursor: pointer;"/>
                      </a>
                      <span style="color: rgb(204, 0, 0);font-size:78%;">
                       Linux Client of popular DC++ Application
                       <br/>
                       <br/>
                      </span>
                      <div style="text-align: left;">
                       <br/>
                       <span style="font-weight: bold;font-size:130%;">
                        aMule
                       </span>

                         Application -&gt; Sound &amp; Video -&gt; MPlayer Movie Player
                        </span>
                        )

                         <span style="font-weight: bold;">
                          Quod Libet : -
                         </span>
                        </span>
                        <br/>
                        <br/>
                        Quod Libet is an extremely versatile music player based on GTK+ that is extendible via plugins , has a rich feature set and has really unique feature of building playlist by searching for particular kind of music files based on search terms or regular expression . It has been programmed in Python programming language and hence it's plugin are also created in Python. A number of plugins are available including ones that help in copying songs to digital audio players , last.fm plugin , advanced editing features and many more .
                        <br/>
                        <br/>
                        Quod Libet uses GStreamer framework of GNOME to play audio files and hence integrates quite well the the GNOME desktop environment , also audio format supported would be same as the one supported by GStreamer so you may want to install mp3 support since by default many linux distributions these days dont come with mp3 support installed.
                        <br/>
                        <br/>
                        To install Quod Libet type the following command in the Terminal Window : -
                        <br/>
                        <br/>
                        <blockquote>
                         sudo apt-get install quodlibet
                         <br/>
                        </blockquote>
                        and after completing above step launch quod-libet from (
                        <span style="font-weight: bold;">
                         Application -&gt; Sound &amp; Video -&gt; Quod Libet
                        </span>
                        )
