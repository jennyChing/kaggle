     <h1 align="center">
      NAND Gate
     </h1>
     <p>
      The output is high when either of inputs A or B is high, or if neither is high. In other words, it is normally high, going low only if both A and B are high.
     </p>
     <table>
      <tr>
       <td>
        <img src="ietron/nand.gif"/>
       </td>
       <td>
        <p>
         The NAND gate and the
         <a href="nor.html#c1">
          NOR gate
         </a>
         can be said to be universal gates since
         <a href="#c4">
          combinations
         </a>
         of them can be used to accomplish any of the
         <a href="diglog.html#c1">
          basic operations
         </a>
         and can thus produce an
         <a href="buffer.html#c3">
          inverter
         </a>
         , an
         <a href="or.html#c1">
          OR gate
         </a>
         or an
         <a href="and.html#c1">
          AND gate
         </a>
         . The non-inverting gates do not have this versatility since they can't produce an invert.
        </p>
       </td>
      </tr>

     <h1 align="center">
      NAND Gate Application
     </h1>
     <p>
      Suppose you want a high output when either A or B is high but C is low. The boolean expression and straightforward gate version of this are:
     </p>
     <center>
      <img src="ietron/nand6.gif"/>
     </center>

      But the same task can be accomplished with
      <a href="#c4">
       NAND gates only
      </a>
      since NAND's are universal gates. Integrated circuits such as the
      <a href="#c3">
       7400

      make this practical.
