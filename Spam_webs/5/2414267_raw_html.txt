     <h2>
      What is scrtchpd?
     </h2>
     <p>
      scrtchpd is a simple tool for quick note taking.
      <strong>
       Just start typing.
      </strong>
     </p>
     <ul>
      <li>
       scrtchpd saves your notes in your browser, locally, so you don't need to worry about it.
      </li>
      <li>
       scrtchpd works best when paired with the
       <a href="https://chrome.google.com/webstore/detail/new-tab-redirect/icpgjfneehieebagbmdbhnlpiopdcmna">
        New Tab Redirect Chrome extension
       </a>
       . Just set up the extension to direct new tabs to scrtchpd.com and you'll have your notes available every time you open a new tab.
      </li>
      <li>
       scrtchpd now supports live Markdown previews, to help organize your thoughts a bit easier. Check out the
       <a href="http://daringfireball.net/projects/markdown/">
        Markdown Documentation
       </a>
       for details. A few quick Examples:
      </li>
     </ul>
