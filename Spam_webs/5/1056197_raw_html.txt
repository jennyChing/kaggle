            <div class="content">
             <p>
              For some users,
              <a href="http://www.microsoft.com/windows/default.aspx" target="_blank">
               Microsoft Windows 7
              </a>
              can never go fast enough. There is always a tweak or Registry edit that can squeeze more speed and performance out of the operating system. This Quick Tip shows you how to speed up the display of menu items in
              <a href="http://www.techrepublic.com/blog/window-on-windows/" target="_blank">
               Windows 7
              </a>
              with a simple Registry edit.
             </p>
             <strong>
              Note:
             </strong>
             Editing the Windows Registry file, if not done correctly, could make a PC unusable. Please, back up the Registry file before you attempt this tip.
             <span style="color: #993300;">
              <em>
               This blog post is also available in PDF format in a
               <a href="http://www.techrepublic.com/downloads/abstract.aspx?docid=2421849" target="_blank">
                TechRepublic download
               </a>
               .
              </em>
             </span>
             <blockquote>
              <span style="color: #3366ff;">
               <em>
                Stay on top of the latest Microsoft Windows tips and tricks with TechRepublic's Windows Desktop newsletter, delivered every Monday and Thursday.
                <a href="/newsletters/">
                 Automatically sign up today!
                </a>
               </em>
              </span>
             </blockquote>
             <h2>
              Speedy menus
             </h2>
             Follow these steps to speed up menu display in Windows 7. In the Start Menu's Search box, type the word "regedit" and click the regedit.exe entry (
             <strong>
              Figure A
             </strong>
             ).
             <h4>
              Figure A
             </h4>
             <h5>
              <img alt="" class="alignnone" height="555" src="http://tr1.cbsistatic.com/hub/i/2011/02/01/2c5f18aa-c3ac-11e2-bc00-02911874f8c8/6190661.png" title="Start the registry editor" width="455"/>
             </h5>
             <h6>
              Start the Registry editor.
             </h6>
             <p>
              Under the HKEY_CURRENT_USER hive, navigate to this key:
             </p>
             <p>
              <em>
               Computer\HKEY_CURRENT_USER\Control Panel\Desktop
              </em>
             </p>
             In the right-hand pane, look for the MenuShowDelay item (
             <strong>
              Figure B
             </strong>
             ).
             <h4>
              Figure B
             </h4>
             <h5>
              <img alt="" class="alignnone" height="319" src="http://tr1.cbsistatic.com/hub/i/2011/02/01/2c754649-c3ac-11e2-bc00-02911874f8c8/6190662.png" title="MenuShowDelay item" width="452"/>
             </h5>
             <h6>
              Locate the MenuShowDelay item.
             </h6>
             Right-click on the MenuShowDelay item and select the Modify option in the context menu (
             <strong>
              Figure C
             </strong>
             ).
             <h4>
              Figure C
             </h4>
             <h5>
              <img alt="" class="alignnone" height="325" src="http://tr1.cbsistatic.com/hub/i/2011/02/01/2c8b89e9-c3ac-11e2-bc00-02911874f8c8/6190663.png" title="Click Modify" width="459"/>
             </h5>
             <h6>
              Click Modify.
             </h6>
             You can now change the value in the Edit String dialog box to a number lower than the default 400 (
             <strong>
              Figure D
             </strong>
             ).
             <p>
              Warning, don't go crazy and set the value at zero or otherwise extraordinarily low because it could make navigating Windows impossible. A better value would be around 100.
             </p>
             <h4>
              Figure D
             </h4>
             <h5>
              <img alt="" class="alignnone" height="322" src="http://tr1.cbsistatic.com/hub/i/2011/02/01/2ca1b8c1-c3ac-11e2-bc00-02911874f8c8/6190664.png" title="Change default value" width="456"/>
             </h5>
             <h6>
              Change the default value.
             </h6>
