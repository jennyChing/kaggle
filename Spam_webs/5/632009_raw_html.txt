        <p>
         I wrote simple Ajax function which you can use to put result of ajax request to any named div on page.
         <br/>
         You just include script
         <a href="http://www.kokeshnet.com/wordpress/wp-content/ajax.js" title="ajax.js">
          ajax.js
         </a>
         into head and than you can use function javascript getdata() wherever you want in document, all you need is to have named
         <b>
          &lt;div&gt;
         </b>
         areas, for example like this:
         <br/>
         <b>
          &lt;a href=# onclick=
          <font color="blue">
           getdata
          </font>
          (
          <font color="green">
           page_to_load
          </font>
          ,
          <font color="green">
           where_to_put_it
          </font>
          );&gt;Click here&lt;/a&gt;
         </b>
        </p>
        <p>
         <script type="text/javascript">
          &lt;!--
google_ad_client = "pub-0459251002113156";
/* 468x60, created 8/31/08 */
google_ad_slot = "8630887235";
google_ad_width = 468;
google_ad_height = 60;
//--&gt;
         </script>
         <script src="http://pagead2.googlesyndication.com/pagead/show_ads.js" type="text/javascript">
         </script>
        </p>
        <p>
         Here is the simple exmaple of usage (youll need file
         <font color="magenta">
          text.php
         </font>
         for this, it should output just anything, which will be put into target DIVs):
         <br/>
         <font style="font-size:11px;">
          <br/>
          &lt;head&gt;
         </font>
        </p>
        <p>
         &lt;title&gt;Ajax Demo&lt;/title&gt;
        </p>
        <p>
         &lt;script type=text/javascript src=ajax.js&gt;&lt;/script&gt;
         <br/>
         &lt;/head&gt;
        </p>
