      <p>
       Mythbuntu 10.04 like the other members of the Ubuntu family is quickly approaching its final release.  Mythbuntu also known as Ubuntu with MythTV Media center is a community supported operating system that focuses on setting up a standalone MythTV based PVR (personal video recorder) system. The development cycle of Mythbuntu closely follows that of Ubuntu, releasing every six months along side Ubuntu releases.
       <br/>
       <span id="more-4856">
       </span>
       <br/>
       Mythbuntu 10.04 release candidate is jammed with new stuff, while its compatible only with MythTV 0.23 systems (previous release can be upgraded), its got better integration with services like
       <a href="http://upstart.ubuntu.com/" rel="nofollow">
        upstart
       </a>
       and
       <a href="http://www.freedesktop.org/wiki/Software/ConsoleKit" rel="nofollow">
        consolekit
       </a>
       . New plug-ins have been added to Mythbuntu Control Centre, and MythTv has a new theme, ArcLight.
      </p>
      <p>
       Mythbuntu 10.04 Lucid Lynx release candidate is built on top of Xfce 4 Desktop Environment version 4.6.1, it includes Mozilla Firefox 3.6.3, Thunar File Manager  1.0.1, and VLC media player 1.0.5 Goldeneye.
      </p>

      <p>
       To test it on your Ubuntu, download and install
       <a href="http://www.mythbuntu.org/auto-builds">
        mythbuntu-repos
       </a>
        it installs and configures the Mythbuntu repos for you. It includes weekly builds (fixes or trunk depending on user input), and testing (depending on user choice during install). Update and install MythTV with
       <em>
        sudo apt-get install mythtv
       </em>
        both MythTV Frontend (Applications -&gt; Sound &amp; Video -&gt; MythTV Frontend) and Backend (System -&gt; Administration -&gt; MythTV Backend Setup) are installed together with the new ArcLight theme, mythtv-database, several other themes and dependencies.
      </p>
      <p>
       You can also run Mythbuntu Frontend from a
       <a href="http://www.mythbuntu.org/10.04/rc">
        Live CD
       </a>
       if you already have a MythTV backend available on your network.
       <br/>
        MythTV Frontend opens up in fullscreen use ALT+TAB to switch between applications.
       <br/>
        MythTVs  Arclight theme background, with those pixels might not be an error, but an intended mosaic effect.
      </p>
