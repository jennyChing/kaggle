       <header class="cluster-header">
        <h1>
         <span class="dh2 head1">
          Oneget
         </span>
         <span class="implied">
          :
         </span>
         <span class="dh1 head5">
          Paketmanager von Microsoft fr Windows 10
         </span>
        </h1>
        <figure id="gheadimg">
         <img alt="Der Paketmanager Oneget wurde in Windows 10 integriert." src="http://www.golem.de/1410/110139-88941-i_rc.jpg"/>
         <figcaption>
          Der Paketmanager Oneget wurde in Windows 10 integriert. (Bild: Microsoft)
         </figcaption>
        </figure>
        <style type="text/css">
         #gheadimg { display:none }
        </style>
        <p>
         <a class="golem-internal-url golem-url-specials" href="http://www.golem.de/specials/microsoft/" target="_blank">
          Microsoft
         </a>
         hat einen Linux-hnlichen Paketmanager in
         <a class="golem-internal-url golem-url-specials" href="http://www.golem.de/specials/windows10/" target="_blank">
          Windows 10
         </a>
         integriert. Mit Oneget soll sich knftig Software wie VLC oder Chrome per Kommandozeile installieren und aktualisieren lassen.
        </p>
       </header>
       <!-- BANNER -->
       <!-- begin ad tag (tile=4) -->
       <div style="margin:15px 0; min-height:103px;">
        <div id="iqadtile4" style="text-align:center;">

       <div class="formatted">
        <!--content-->
        <p>
         Mit Oneget hat Microsoft einen Paketmanager fr die Kommandozeile in Windows 10 eingefgt. Darber soll sich knftig Software wie der Mediaplayer VLC oder der Browser Chrome ber die Powershell installieren lassen. Oneget funktioniert hnlich wie die unter Linux gebruchlichen Paketmanager Apt, Yum oder Zypper. Sie ermglichen das schnelle Einspielen von Aktualisierungen.
        </p>
        <figure id="gvideo_14295">
        </figure>
        <p>
         Bislang ist
         <a class="golem-external-url" href="http://blogs.technet.com/b/windowsserver/archive/2014/04/03/windows-management-framework-v5-preview.aspx" target="_blank">
          Oneget ein Kommandozeilenwerkzeug
         </a>
         , das in Powershell unter Windows 10 integriert wurde. Fr Windows 8 lsst sich Oneget samt Powershell mit dem Windows Management Framework 5.0 Preview installieren und nutzen.
         <a class="golem-external-url" href="http://blogs.msdn.com/b/garretts/archive/2014/04/01/my-little-secret-windows-powershell-oneget.aspx" target="_blank">
          Bereits Anfang April 2014
         </a>
         hatte Microsoft-Entwickler Garrett Serack Oneget auf der Entwicklerkonferenz Build vorgestellt. Seitdem wird der
         <a class="golem-external-url" href="https://github.com/OneGet/oneget" target="_blank">
          Quellcode auf Github
         </a>
         gepflegt.
        </p>
        <h3>
         Einfache Updates freier Software
        </h3>
        <p>
         Die Installation erfolgt aktuell ber die Kommandozeile. Mit dem Befehl
         <em>
          Install-Package -Name VLC
         </em>
         lsst sich beispielsweise der Mediaplayer VLC installieren. Oneget lsst sich auch nutzen, um Software wieder zu deinstallieren. Auerdem knnen Softwarequellen - sogenannte Repositories - mit Oneget verwaltet werden. Besonders Updates lassen sich ber Oneget schnell installieren.
        </p>
        <p>
         Wegen der Softwarequellen drfte Microsoft die grten Bedenken haben. Denn Nutzer knnen diese Quellen selbst einrichten, darunter wohl auch solche, die nicht vertrauenswrdig sind. Serack hat eine enge Zusammenarbeit mit der Community versprochen, auch um die Auswahl der Software zu verwalten. Entwickler Rob Reynolds
         <a class="golem-external-url" href="https://twitter.com/ferventcoder/status/526583152220078082" target="_blank">
          hatte per Twitter
         </a>
         eine Kooperation Microsofts mit seinem Projekt Chocolately bekanntgegeben.
         <a class="golem-external-url" href="https://chocolatey.org/" target="_blank">
          Chocolately
         </a>
         bietet ebenfalls einen Kommandozeilen-Paketmanager fr Windows und stellt zudem ein Repository zur Verfgung, ber das freie Software zur Installation ber den Paketmanager bereitsteht. Eine weitere Hrde drfte die zustzliche Verwaltung von Software-Bibliotheken sein, die oftmals in verschiedenen Versionen fr Anwendungen bentigt werden. Bei Chocolately erhlt jedes Softwarepaket bislang eine einfache XML-Beschreibungsdatei, in der lediglich der Paketname und die Softwarequelle angegeben werden.
        </p>
        <h3>
         Chocolately als Softwarequelle
        </h3>
        <p>
         In einer ersten Version von Oneget war das Repository von Chocolately integriert. Allerdings wurde es in spteren Versionen wieder entfernt. Aktuell lasse sich zwar die Chocolately-Softwarequelle ber Oneget wieder einrichten, Software lasse sich jedoch von dort nicht installieren, schreibt Entwickler Chris Hoffman
         <a class="golem-external-url" href="http://www.howtogeek.com/200334/windows-10-includes-a-linux-style-package-manager-named-oneget/?PageSpeed=noscript" target="_blank">
          auf der Webseite Howtogeek
         </a>
         .
        </p>
        <p>
         <a class="golem-external-url" href="https://github.com/chocolatey/chocolatey/wiki" target="_blank">
          Chocolatelys Installationsclient
         </a>
         muss auch mit der Rechteverwaltung unter Windows klarkommen. Die Software installiert sich beispielsweise im Ordner C:\ProgamData\Chocolately, damit smtliche Nutzer auch mit normalen Rechten die Software nutzen knnen und gleichzeitig die Angriffsflche auf die Software selbst reduziert wird. Auerdem nutzt Chocolately die Registry nicht, deshalb fllt auch der Ordner Programme als Installationsziel weg.
        </p>
        <h3>
         Bislang nur portable Software
        </h3>
        <p>
         ber Chocolately lassen sich bislang nur Portable Applikationen installieren, da der Paketmanager keinen Zugriff auf die Registry hat. Deshalb taucht dort installierte Software auch nicht in der Softwareverwaltung der Systemsteuerung von Windows auf. Anwendungen lassen sich daher nur ber Chocolately wieder deinstallieren. In dessen Softwarequellen befinden sich aktuell
         <a class="golem-external-url" href="https://chocolatey.org/packages" target="_blank">
          etwa 2.300 Pakete
         </a>
         . Derzeit versucht das Projekt
         <a class="golem-external-url" href="https://www.kickstarter.com/projects/ferventcoder/chocolatey-the-alternative-windows-store-like-yum" target="_blank">
          ber Kickstarter
         </a>
         Geld zu sammeln, auch um die Entwicklung einer grafischen Benutzeroberflche zu finanzieren.
        </p>
        <p>
         Ob Microsofts Oneget sich auch auf die Installation von portablen Anwendungen beschrnken oder sogar einen Zugriff auf den Windows Store erlauben wird, ist noch nicht bekannt. Zumindest sollen die APIs Paketmanagern wie Chocolately Zugriff auf diverse Windows-Funktionen ermglichen. Mglicherweise wird das API auch die Nutzung grafischer Installationswerkzeuge erlauben.
        </p>
