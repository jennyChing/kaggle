     <p>
      <b>
       Johanna Franziska Frmyot von Chantal
      </b>
      (Jeanne Franoise Frmyot de Chantal) (*
      <a href="/wiki/23._Januar" title="23. Januar">
       23. Januar
      </a>
      <sup class="reference" id="cite_ref-unglaublichlangerref_1-1">
       <a href="#cite_note-unglaublichlangerref-1">
        [1]
       </a>
      </sup>
      <a href="/wiki/1572" title="1572">
       1572
      </a>
      in
      <a href="/wiki/Dijon" title="Dijon">
       Dijon
      </a>
      ; 
      <a href="/wiki/13._Dezember" title="13. Dezember">
       13.Dezember
      </a>
      <a href="/wiki/1641" title="1641">
       1641
      </a>
      in
      <a href="/wiki/Moulins_(Allier)" title="Moulins (Allier)">
       Moulins
      </a>
      ) war die Ehefrau von Christophe Rabutin de Chantal, Mutter von sechs Kindern, geistliche Freundin des heiligen
      <a href="/wiki/Franz_von_Sales" title="Franz von Sales">
       Franz von Sales
      </a>
      und Mitbegrnderin der
      <a class="mw-redirect" href="/wiki/Salesianerinnen" title="Salesianerinnen">
       Schwestern von der Heimsuchung Mariens (Salesianerinnen)
      </a>
      . Sie ist eine
      <a href="/wiki/Heiliger" title="Heiliger">
       Heilige
      </a>
      der katholischen Kirche, die Gromutter der Schriftstellerin
      <a href="/wiki/Marie_de_Rabutin-Chantal,_Marquise_de_S%C3%A9vign%C3%A9" title="Marie de Rabutin-Chantal, Marquise de Svign">
       Marie de Rabutin-Chantal, Marquise de Svign
      </a>
      und Schwester von Andr Frmyot, des Erzbischofs von
      <a href="/wiki/Erzbistum_Bourges" title="Erzbistum Bourges">
       Bourges
      </a>
      .
     </p>
