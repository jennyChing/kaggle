       <h1 class="title chunkfive_roman">
        Primeros pasos con un web hosting profesional
       </h1>
       <div class="postmeta">
        <a class="color" href="feed/rss" rel="bookmark" target="_blank" title="RSS">
         RSS
        </a>
        |evahernan26|Lunes, 9 de junio de 2014|
        <a href="http://blogs.diariosur.es/techvirtual/2014/06/09/primeros-pasos-con-un-web-hosting-profesional/#respond" title="Comentarios en Primeros pasos con un web hosting profesional">
         Sin comentarios
        </a>
        &amp;nbsp|&amp;nbsp&amp;nbsp|&amp;nbsp
       </div>
       <div class="entry">
        <p>
         Cuando hemos contratado un web
         <a href="http://www.evidaliahost.com" rel="external nofollow" title="hosting">
          hosting
         </a>
         <a href="http://blogs.diariosur.es/techvirtual/files/2014/06/hosting.jpg">
          <img alt="" class="alignleft size-full wp-image-7" height="183" src="http://blogs.diariosur.es/techvirtual/files/2014/06/hosting.jpg" title="hosting" width="276"/>
         </a>
         profesional para instalar nuestra pgina web, al principio es lgico que nos encontremos un poco perdidos si no lo hemos manejado nunca. No todas las interfaces son sencillas y en ocasiones la multitud de funcionalidades y opciones hacen que el usuario se pierda entre tantas posibilidades. Sin embargo, para empezar a trabajar slo hay que seguir ciertas pautas y al cabo de un tiempo procurar entrar en nuestro hosting slo cuando sea muy necesario.
        </p>
        <p>
        </p>
        <p>
         Qu primeros pasos deberamos dar cuando hemos alquilado un hosting compartido? Qu son las tareas imprescindibles que deberamos hacer para llevar a cabo la configuracin y tenerlo todo a punto para poner en marcha una pgina web? Vamos a verlo en algunos puntos muy sencillos.
        </p>
        <p>
        </p>
        <p>
         <strong>
          Lo primero que debes hacer con un web hosting profesional
         </strong>
        </p>
        <p>
         <strong>
         </strong>
        </p>
        <p>
         En el momento en el que te registras en tu web hosting, despus de haber investigado cules son los planes de hosting, haber elegido uno y tener ya tu usuario y contrasea, es empezar algunas decisiones.
        </p>
        <p>
        </p>
        <ol>
         <li>
          <strong>
           Elegir un dominio.
          </strong>
          La mayora de los hosting os incluyen un dominio gratuito en el paquete de alojamiento web. As que lo importante al empezar ser tener ya claro cul es el dominio que queremos registrar.
         </li>
        </ol>
        <p>
         <strong>
         </strong>
        </p>
        <ol>
         <li>
          <strong>
           Crear un usuario de FTP.
          </strong>
          Tambin ser importante que busquemos la seccin para crear usuarios de FTP. Esto nos permitir gestionar nuestros archivos desde un programa instalado en nuestro ordenador, como por ejemplo FilezillaClient. Es mucho ms cmodo y sencillo que entrar en el hosting para hacer cualquier cambio.
         </li>
        </ol>
        <p>
         <strong>
         </strong>
        </p>
        <ol>
         <li>
          <strong>
           Crear una base de datos.
          </strong>
          Cualquiera que sea el sistema de gestin que vayas a instalar (WordPress, Joomla, Prestashop), necesitars una base de datos. Incluso si piensas crear una web desde cero, nadie hoy da hace una pgina web sin tener una base de datos.
         </li>
        </ol>
        <p>
         <strong>
         </strong>
        </p>
        <ol>
         <li>
          <strong>
           Instalar el sistema de gestin de contenidos.
          </strong>
          Muchos web hosting vienen ya con algn sistema de gestin instalado, como WordPress, Prestashop, etc. En caso de que no lo llevara y tengas que instalarlo t, dirgete a la pgina web del programa, descrgalo y sigue los pasos para instalarlo en tu hosting a travs del FTP.
         </li>
        </ol>
        <p>
         <strong>
         </strong>
        </p>
        <ol>
         <li>
          <strong>
           Comprobar opciones de seguridad y backup.
          </strong>
          Otro punto importante que deberemos tener en cuenta al alquilar nuestro hosting es el sistema de proteccin de antivirus. Est activado por defecto? Tiene un sistema antispam para el correo? Hay backups automatizados? Debemos estudiar estos puntos y, por ejemplo, determinar cada cuanto tiempo se harn las copias de seguridad y dnde se guardarn.
         </li>
        </ol>
        <p>
         <strong>
         </strong>
        </p>
        <p>
         Con estos puntos, espero haberte ayudado a saber qu tienes que hacer la primera vez que alquilas un web hosting y cules son los primeros pasos que debes dar. Por supuesto, hay muchas ms cosas que puedes ir trasteando con el tiempo. Qu ms sugieres o consideras importante?
        </p>
        <p>
        </p>
