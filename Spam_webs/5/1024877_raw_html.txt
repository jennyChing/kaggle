              (adsbygoogle=window.adsbygoogle||[]).push({});&lt;/script&gt;&lt;/div&gt;&lt;p&gt;&lt;strong&gt;How to install Java on Ubuntu ?&lt;/strong&gt; Installing Java on your system is one of a first &lt;a title="Things to do after Install Ubuntu 14.10 Utopic Unicorn" href="http://opnux.com/things-to-do-after-install-ubuntu-14-10-utopic-unicorn" target="_blank"&gt;things to do after you install your Ubuntu Linux&lt;/a&gt;. Many people have problems when installing Java, in fact it is very easy to do so. In this post I will show you &lt;strong&gt;how to install java on your Ubuntu Linux&lt;/strong&gt;. This post will cover the installation for 32-bit and 64-bit version of &lt;strong&gt;Oracle Java.&lt;/strong&gt; Java version used in this tutorial is &lt;strong&gt;1.7.0_45&lt;/strong&gt;, but don&#8217;t worry because almost of all java version use the same installation method. These instructions also work on Linux Mint, Elementary OS, and Debian. So &lt;strong&gt;how do we install our Oracle Java ?&lt;/strong&gt; Just follow my instructions below.&lt;/p&gt;&lt;h2&gt;OpenJDK, Oracle JDK, and JRE&lt;/h2&gt;&lt;p&gt;First thing before we going to the installation progress, we should know what is Open JDK, Oracle JDK, and JRE ? Let&#8217;s see the difference.&lt;/p&gt;&lt;ul&gt;&lt;li&gt;OpenJDK is a free and Open Source version of Oracle Java Development Kit as the result of an effor Sun Microsystems in 2006.&lt;/li&gt;&lt;li&gt;JDK stands for Java Development Kit, is a official version of JDK from Oracle while the OpenJDK is the open source version and this is what we need to install in this tutorial. JDK is needed when you have to do some development (programming).&lt;/li&gt;&lt;li&gt;JRE stands for Java Runtime Environment is what you need when you want to run Java based application.&lt;/li&gt;&lt;/ul&gt;&lt;p&gt;Now, we know the difference, let&#8217;s begin the installation.&lt;/p&gt;&lt;h2&gt;Step 1&lt;/h2&gt;&lt;h3&gt;Check if you have Java installed&lt;/h3&gt;&lt;p&gt;To check your Java, you have to run the command from your terminal, so open your terminal and enter the following command :&lt;/p&gt;&lt;pre&gt;$ java -version&lt;/pre&gt;&lt;p&gt;If you see output :&lt;/p&gt;&lt;pre&gt;java version "1.7.0_15"
&lt;strong&gt;OpenJDK&lt;/strong&gt; Runtime Environment (IcedTea6 1.10pre) (7b15~pre1-0lucid1)
&lt;strong&gt;OpenJDK&lt;/strong&gt; 64-Bit Server VM (build 19.0-b09, mixed mode)&lt;/pre&gt;&lt;p&gt;Then you have wrong version of Java, you have to uninstall OpenJDK before continue the installation.&lt;/p&gt;&lt;h2&gt;Step 2&lt;/h2&gt;&lt;h3&gt;Uninstall OpenJDK (Skip this step if OpenJDK isn&#8217;t installed)&lt;/h3&gt;&lt;p&gt;If you not completely remove the OpenJDK/JRE from your system, this will make a conflict between the different vendor of java and make your system confused. To uninstall OpenJDK, open your terminal and enter the following command :&lt;/p&gt;&lt;pre&gt;$ sudo apt-get purge openjdk-\*&lt;/pre&gt;&lt;h2&gt;Step 3&lt;/h2&gt;&lt;h3&gt;Download Oracle JDK/JRE for Linux&lt;/h3&gt;&lt;p&gt;Go to Java Official site or just click this &lt;a title="Download Java" href="http://www.oracle.com/technetwork/java/javase/downloads/index.html" target="_blank"&gt;Link&lt;/a&gt;.&lt;/p&gt; &lt;figure id="attachment_346" class="thumbnail wp-caption aligncenter" style="width: 598px"&gt;&lt;a href="http://opnux.com/wp-content/uploads/2014/12/How-to-Install-Java-on-Ubuntu-1.png"&gt;&lt;img class="size-full wp-image-346" src="http://opnux.com/wp-content/uploads/2014/12/How-to-Install-Java-on-Ubuntu-1.png" alt="How to Install Java on Ubuntu - 1" width="588" height="200"/&gt;&lt;/a&gt;&lt;figcaption class="caption wp-caption-text"&gt;How to Install Java on Ubuntu &#8211; 1&lt;/figcaption&gt;&lt;/figure&gt;&lt;p&gt;Then scroll down until you found &lt;strong&gt;License Agreement for Java SE,&lt;/strong&gt; Accept the License Agreement.&lt;/p&gt;&lt;p&gt;Now select the download link for your Linux Version, don&#8217;t get confuse just download the &lt;strong&gt;tar.gz &lt;/strong&gt;for &lt;strong&gt;Linux x64 or x86 depends on your Architecture&lt;/strong&gt; (RPM is for RedHat Linux).&lt;/p&gt; &lt;figure id="attachment_347" class="thumbnail wp-caption aligncenter" style="width: 572px"&gt;&lt;a href="http://opnux.com/wp-content/uploads/2014/12/How-to-Install-Java-on-Ubuntu-2.png"&gt;&lt;img class="size-full wp-image-347" src="http://opnux.com/wp-content/uploads/2014/12/How-to-Install-Java-on-Ubuntu-2.png" alt="How to Install Java on Ubuntu - 2" width="562" height="423"/&gt;&lt;/a&gt;&lt;figcaption class="caption wp-caption-text"&gt;How to Install Java on Ubuntu &#8211; 2&lt;/figcaption&gt;&lt;/figure&gt;&lt;p&gt;Now wait till the download is completed.&lt;/p&gt;&lt;h2&gt;Step 4&lt;/h2&gt;&lt;h3&gt;Copy Oracle Java binaries&lt;/h3&gt;&lt;p&gt;Before we copy our java binaries, we must create &lt;strong&gt;/usr/local/java &lt;/strong&gt;directory, to do this, enter the following command :&lt;/p&gt;&lt;pre&gt;$ sudo mkdir -p /usr/local/java&lt;/pre&gt;&lt;p&gt;Then after creating the directory run this command to copy your java binaries :&lt;/p&gt;&lt;pre&gt;&lt;strong&gt;For 32-bit and 64-bit&lt;/strong&gt;

$ sudo cp jdk-7u45-linux-x64.tar.gz /usr/local/java/
$ cd /usr/local/java
$ ls&lt;/pre&gt;&lt;figure id="attachment_348" class="thumbnail wp-caption aligncenter" style="width: 732px"&gt;&lt;a href="http://opnux.com/wp-content/uploads/2014/12/How-to-Install-Java-on-Ubuntu-3.png"&gt;&lt;img class="size-full wp-image-348" src="http://opnux.com/wp-content/uploads/2014/12/How-to-Install-Java-on-Ubuntu-3.png" alt="How to Install Java on Ubuntu - 3" width="722" height="192"/&gt;&lt;/a&gt;&lt;figcaption class="caption wp-caption-text"&gt;How to Install Java on Ubuntu &#8211; 3&lt;/figcaption&gt;&lt;/figure&gt;&lt;h2&gt;Step 5&lt;/h2&gt;&lt;h3&gt;Unpack Compressed Java binaries&lt;/h3&gt;&lt;p&gt;Unpack your Compressed binaries to your &lt;strong&gt;/usr/local/java&lt;/strong&gt;, use this following command to uncompressed :&lt;/p&gt;&lt;pre&gt;&lt;strong&gt;For 32-bit and 64-bit&lt;/strong&gt;

$ sudo tar -xvfz jdk-7u45-linux-x64.tar.gz
$ ls&lt;/pre&gt;&lt;figure id="attachment_352" class="thumbnail wp-caption aligncenter" style="width: 732px"&gt;&lt;a href="http://opnux.com/wp-content/uploads/2014/12/How-to-Install-Java-on-Ubuntu-4.png"&gt;&lt;img class="size-full wp-image-352" src="http://opnux.com/wp-content/uploads/2014/12/How-to-Install-Java-on-Ubuntu-4.png" alt="How to Install Java on Ubuntu - 4" width="722" height="264"/&gt;&lt;/a&gt;&lt;figcaption class="caption wp-caption-text"&gt;How to Install Java on Ubuntu &#8211; 4&lt;/figcaption&gt;&lt;/figure&gt;&lt;h2&gt;Step 6&lt;/h2&gt;&lt;h3&gt;Edit system PATH file&lt;/h3&gt;&lt;p&gt;Edit your system PATH file in &lt;strong&gt;/etc/profile&lt;/strong&gt; and add some system variables to your system path, To edit your system path use any text editor, and open &lt;strong&gt;/etc/profile &lt;/strong&gt;as &lt;strong&gt;root&lt;/strong&gt;&lt;/p&gt;&lt;pre&gt;$ sudo vim &lt;strong&gt;/etc/profile&lt;/strong&gt;&lt;/pre&gt;&lt;p&gt;After open up system path file, scroll down to the end of the file and add the following lines below to your system :&lt;/p&gt;&lt;pre&gt;&lt;strong&gt;For JDK add this lines&lt;/strong&gt;&lt;/pre&gt;&lt;pre&gt;JAVA_HOME=/usr/local/java/&lt;b&gt;jdk1.7.0_45&lt;/b&gt;
JRE_HOME=$JAVA_HOME/jre
PATH=$PATH:$JAVA_HOME/bin:$JRE_HOME/bin
export JAVA_HOME
export JRE_HOME
export PATH&lt;/pre&gt;&lt;pre&gt;&lt;strong&gt;For JRE add this lines&lt;/strong&gt;&lt;/pre&gt;&lt;pre&gt;JRE_HOME=/usr/local/java/&lt;b&gt;jre1.7.0_45&lt;/b&gt;
PATH=$PATH:$JRE_HOME/bin
export JRE_HOME
export PATH&lt;/pre&gt;&lt;p&gt;Save your new system path and exit.&lt;/p&gt; &lt;figure id="attachment_353" class="thumbnail wp-caption aligncenter" style="width: 732px"&gt;&lt;a href="http://opnux.com/wp-content/uploads/2014/12/How-to-Install-Java-on-Ubuntu-5.png"&gt;&lt;img class="size-full wp-image-353" src="http://opnux.com/wp-content/uploads/2014/12/How-to-Install-Java-on-Ubuntu-5.png" alt="How to Install Java on Ubuntu - 5" width="722" height="372"/&gt;&lt;/a&gt;&lt;figcaption class="caption wp-caption-text"&gt;How to Install Java on Ubuntu &#8211; 5&lt;/figcaption&gt;&lt;/figure&gt;&lt;h2&gt;Step 7&lt;/h2&gt;&lt;h3&gt;Tell your Ubuntu system where is your Oracle JDK/JRE is Located&lt;/h3&gt;&lt;p&gt;Before using your java you should tell your system where is your Java is available for use, enter this following command to notifies your system that Java is available in your system and ready to use.&lt;/p&gt;&lt;pre&gt;&lt;strong&gt;For JDK&lt;/strong&gt;&lt;/pre&gt;&lt;pre&gt;$ sudo update-alternatives --install "/usr/bin/java" "java" "/usr/local/java/jdk1.7.0_45/jre/bin/java" 1
$ sudo update-alternatives --install "/usr/bin/javac" "javac" "/usr/local/java/jdk1.7.0_45/bin/javac" 1&lt;/pre&gt;&lt;pre&gt;&lt;strong&gt;For JRE Only&lt;/strong&gt;&lt;/pre&gt;&lt;pre&gt;$ sudo update-alternatives --install "/usr/bin/java" "java" "/usr/local/java/jre1.7.0_45/bin/java" 1&lt;/pre&gt;&lt;pre&gt;&lt;strong&gt;For Oracle Java Web start (javaws)
&lt;/strong&gt;&lt;/pre&gt;&lt;pre&gt;$ sudo update-alternatives --install "/usr/bin/javaws" "javaws" "/usr/local/java/jdk1.7.0_45/bin/javaws" 1&lt;/pre&gt;&lt;pre&gt;&lt;strong&gt;or if you installing jre only&lt;/strong&gt;, &lt;strong&gt;use the command below :&lt;/strong&gt;

$ sudo update-alternatives --install "/usr/bin/javaws" "javaws" "/usr/local/java/jre1.7.0_45/bin/javaws" 1&lt;/pre&gt;&lt;p&gt;After use those command, reload your system wide PATH by typing the following command&lt;/p&gt;&lt;pre&gt;$ . /etc/profile&lt;/pre&gt;&lt;figure id="attachment_354" class="thumbnail wp-caption aligncenter" style="width: 730px"&gt;&lt;a href="http://opnux.com/wp-content/uploads/2014/12/How-to-Install-Java-on-Ubuntu-6.png"&gt;&lt;img class="size-full wp-image-354" src="http://opnux.com/wp-content/uploads/2014/12/How-to-Install-Java-on-Ubuntu-6.png" alt="How to Install Java on Ubuntu - 6" width="720" height="158"/&gt;&lt;/a&gt;&lt;figcaption class="caption wp-caption-text"&gt;How to Install Java on Ubuntu &#8211; 6&lt;/figcaption&gt;&lt;/figure&gt;&lt;h2&gt;Last Step&lt;/h2&gt;&lt;h3&gt;Now Test your Oracle Java&lt;/h3&gt;&lt;p&gt;Test your java to see if Java was installed correctly on your system, to test just run the following commands below, this command also work both for 32-bit and 64-bit system&lt;/p&gt;&lt;pre&gt;&lt;strong&gt;Test your Java&lt;/strong&gt;&lt;/pre&gt;&lt;pre&gt;$ java -version&lt;/pre&gt;&lt;pre&gt;&lt;strong&gt;Test your Java Compiler&lt;/strong&gt;&lt;/pre&gt;&lt;pre&gt;$ javac -version&lt;/pre&gt;&lt;p&gt;If you see output like the screenshot below, then congratulations your are successful installed &lt;strong&gt;Oracle Java&lt;/strong&gt;on your Linux System.&lt;/p&gt; &lt;figure id="attachment_355" class="thumbnail wp-caption aligncenter" style="width: 732px"&gt;&lt;a href="http://opnux.com/wp-content/uploads/2014/12/How-to-Install-Java-on-Ubuntu-7.png"&gt;&lt;img class="size-full wp-image-355" src="http://opnux.com/wp-content/uploads/2014/12/How-to-Install-Java-on-Ubuntu-7.png" alt="How to Install Java on Ubuntu - 7" width="722" height="264"/&gt;&lt;/a&gt;&lt;figcaption class="caption wp-caption-text"&gt;How to Install Java on Ubuntu &#8211; 7&lt;/figcaption&gt;&lt;/figure&gt;&lt;p&gt;Now before using your system and using Java, it&#8217;s recommended to restart your system, so your system will be fully configured.&lt;/p&gt;&lt;p&gt;So I hope this post will help you to &lt;strong&gt;&#8220;How to Install Java on Ubuntu&#8221;, &lt;/strong&gt;as I said before this tutorial is not only for Ubuntu, you can use this tutorial for Linux Mint, Elementary OS, and other Distributions that based on Ubuntu.
