         <p>
          <strong>
           The Cross Bow 2 Mod
          </strong>
          for Minecraft v1.3.2 is a custom Mod. This mod grants you a bow which has several features and you can use it with your requirement. Cross Bow 2 is the upgraded version of Cross Bow which was earlier released and now the more modified version is here. Some new features are added which are mentioned below in the post.
         </p>

         </p>

         </p>

         </p>
         <p style="text-align: left;">
          <span style="font-size: 20px;">
           <strong>
            Futures :
           </strong>
          </span>
         </p>
         <ul>
          <li>
           Added brand new Crossbow Crafting Bench!
          </li>
          <li>
           Added brand new 148 customization combinations  Create the crossbow of your dreams!
          </li>
          <li>
           Added brand new Update checker  Always have the newest features!
          </li>
          <li>
           Added brand new Add scopes and attachments to each base crossbow!
          </li>
          <li>
           Added brand new Auto-firing crossbows!
          </li>
          <li>
           Added brand new Fully configurable item IDs!Added brand new Achievements!
          </li>
         </ul>
         <p>
          <span style="font-size: 20px;">
           <strong>
            How to install :
           </strong>
          </span>
         </p>
         <p>
         </p>
         <ol>
          <li>
           Download
           <a href="http://www.modsdownloads.com/minecraft-forge-2/">
            Minecraft Forge
           </a>
           and install it
          </li>
          <li>
           Download this mod
          </li>
          <li>
           rag and drop the downloaded jar file into the C:UsersYOURUSERNAMEAppDataRoaming.minecraftmods folder. If one does not exist you can create one
          </li>
          <li>
           Done!!!
          </li>
         </ol>
