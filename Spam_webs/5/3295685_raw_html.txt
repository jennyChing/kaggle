&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;p style="text-align: justify;"&gt;Welcome tothe first postof my 4 part blog series &#8211; all aboutGarage Organization. Many of you have shared stories about your lack of&#8220;garage&#8221; organization and stated that you needed major help. I hope today I can inspire you to get out there and finally get it organized. (I&#8217;m talking to you too husbands!) I&#8217;m going to start off the series bytalking about &#8220;STORAGE&#8221;. Everything needs a place or there will be garage chaos &amp; setting up a &#8220;system of organization&#8221; is the key! I will be highlighting the 4 most important pointsthat will help you to finally get your garage in order.   But before I begin, I would like to thank &lt;a href="http://www.containerstore.com/welcome.htm" target="_blank"&gt;The Container Store&lt;/a&gt; for sponsoring this project. I&#8217;m grateful to be able to share it with my readers.&lt;/p&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;p style="text-align: left;"&gt;&lt;strong&gt;4 STEPS TO STORAGE&lt;/strong&gt;&lt;/p&gt;
&lt;ol&gt;
&lt;li style="text-align: left;"&gt;Shelving&lt;/li&gt;
&lt;li style="text-align: left;"&gt;Drawers&lt;/li&gt;
&lt;li style="text-align: left;"&gt;Labeling&lt;/li&gt;
&lt;li style="text-align: left;"&gt;Containers&lt;br /&gt;
&nbsp;&lt;br /&gt;
&lt;a href="http://www.abowlfulloflemons.net/wp-content/uploads/2014/10/Garage-Organization-via-A-Bowl-Full-of-Lemons-1.jpg"&gt;&lt;img class="aligncenter size-full wp-image-105837" src="http://www.abowlfulloflemons.net/wp-content/uploads/2014/10/Garage-Organization-via-A-Bowl-Full-of-Lemons-1.jpg" alt="Garage Organization via A Bowl Full of Lemons " width="1000" height="666" /&gt;&lt;/a&gt;&lt;/li&gt;
&lt;/ol&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;p style="text-align: center;"&gt;&lt;strong&gt;SHELVING&lt;/strong&gt;&lt;/p&gt;
&lt;p style="text-align: justify;"&gt;The first challenge that you need to address is figuring out where you are going to put everything? Do you have a designated shelf? In my garage, I try to keep everything up off of the floor. My husband and I agreed that quality was very important to the longevity of our organization, so we went with &lt;a href="%20http://www.containerstore.com/elfa/whatiselfa/index.html" target="_blank"&gt;Elfa Shelving&lt;/a&gt; from The Container Store. They&#8217;re super sturdy, high quality and worth every single penny. FYI, these shelves are on sale for 25% off right now!! When choosing your shelving, make sure to measure. You want your bins to fit!&lt;/p&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;p style="text-align: center;"&gt;&lt;a href="http://www.abowlfulloflemons.net/wp-content/uploads/2014/10/Garage-Organization-via-A-Bowl-Full-of-Lemons-12.jpg"&gt;&lt;img class="aligncenter size-full wp-image-105848" src="http://www.abowlfulloflemons.net/wp-content/uploads/2014/10/Garage-Organization-via-A-Bowl-Full-of-Lemons-12.jpg" alt="Garage Organization via A Bowl Full of Lemons " width="550" height="826" /&gt;&lt;/a&gt;&lt;/p&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;p style="text-align: center;"&gt;&lt;a href="http://www.abowlfulloflemons.net/wp-content/uploads/2014/10/Garage-Organization-via-A-Bowl-Full-of-Lemons-7.jpg"&gt;&lt;img class="aligncenter size-full wp-image-105843" src="http://www.abowlfulloflemons.net/wp-content/uploads/2014/10/Garage-Organization-via-A-Bowl-Full-of-Lemons-7.jpg" alt="Garage Organization via A Bowl Full of Lemons " width="550" height="826" /&gt;&lt;/a&gt;&lt;/p&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;p style="text-align: center;"&gt;&lt;strong&gt;DRAWERS&lt;/strong&gt;&lt;/p&gt;
&lt;p style="text-align: justify;"&gt;The next challengeyou need to address is &#8211; where are you going toput your most often used items? Things like batteries, flash lights, trash bags, and outdoor play toys need to be easily accessible. The perfect solution for this problem is to usedrawers. I chose to usethe &lt;a href="http://www.containerstore.com/shop/elfa/components/elfaVentilatedShelving?productId=10022922&amp;N=80346" target="_blank"&gt;Elfa Mesh Drawer System&lt;/a&gt;. Love, love, love it! It&#8217;s my favorite part of the garage organization. Everything is so convenient now. Again, this system is on sale for one more week &#8211;25% off! (I love a good sale). You can also use something like &lt;a href="http://www.containerstore.com/shop?productId=10015091&amp;N=&amp;Ntt=drawer" target="_blank"&gt;this&lt;/a&gt; or &lt;a href="http://www.containerstore.com/shop?productId=10009305&amp;N=&amp;Ntt=drawer" target="_blank"&gt;this&lt;/a&gt;. Really any type of drawer system will work well. &lt;img src="http://www.abowlfulloflemons.net/wp-includes/images/smilies/icon_smile.gif" alt=":)" class="wp-smiley" /&gt;&lt;/p&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;p style="text-align: center;"&gt;&lt;a href="http://www.abowlfulloflemons.net/wp-content/uploads/2014/10/Garage-Organization-via-A-Bowl-Full-of-Lemons-3.jpg"&gt;&lt;img class="aligncenter size-full wp-image-105839" src="http://www.abowlfulloflemons.net/wp-content/uploads/2014/10/Garage-Organization-via-A-Bowl-Full-of-Lemons-3.jpg" alt="Garage Organization via A Bowl Full of Lemons " width="1000" height="666" /&gt;&lt;/a&gt;&lt;/p&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;p style="text-align: center;"&gt;&lt;strong&gt;COVERING MESH DRAWERS&lt;/strong&gt;&lt;/p&gt;
&lt;p style="text-align: justify;"&gt;I didn&#8217;t like that you could see everything that was inside of my drawers, so I came up with a creative solution. I covered them with black gingham fabric (from JoAnns). I made a template out of cardboard then cut 18pieces to fit the drawers. I adhered the fabric to the drawers withspray&lt;a href="http://www.amazon.com/gp/product/B003ZFW68G?ie=UTF8&amp;creativeASIN=B003ZFW68G&amp;linkCode=xm2&amp;tag=abfolshop-20" target="_blank"&gt;adhesive&lt;/a&gt;. I sprayed the front of the fabric (not the drawer). Finally, I stuck the fabric (front side forward) on the drawer &amp; smoothed it out. Worked like a charm.&lt;/p&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;

&lt;ul&gt;
&lt;li style="text-align: justify;"&gt;For the drawers, I used the &lt;a href="http://www.containerstore.com/shop?productId=10028114&amp;N=&amp;Nao=60&amp;Ntt=elfa+drawer" target="_blank"&gt;Elfa Label Holders.&lt;/a&gt; These holders slide right down onto the drawer so they should work withany drawer system if&lt;span style="text-decoration: underline;"&gt;it&#8217;s thin&lt;/span&gt;. I created the tags on my computer and printed them out on brown kraft card stock paper.&lt;/li&gt;
&lt;li style="text-align: justify;"&gt;For the large white containers,I used a&lt;a href="http://www.amazon.com/gp/product/B0018N7VDC?ie=UTF8&amp;creativeASIN=B0018N7VDC&amp;linkCode=xm2&amp;tag=abfolshop-20" target="_blank"&gt;Cricut Machine&lt;/a&gt;, but you can also use a&lt;a href="http://www.amazon.com/gp/product/B0047T7JMW/ref=as_li_tl?ie=UTF8&amp;camp=1789&amp;creative=9325&amp;creativeASIN=B0047T7JMW&amp;linkCode=as2&amp;tag=abfolshop-20&amp;linkId=SVLQUQXP7ISJNJJL" target="_blank"&gt;label maker&lt;/a&gt;, blank stickersor even an index card taped to the box.&lt;/li&gt;
&lt;/ul&gt;

If you cannot afford an Elfa system, you can find many other systems to choose from &#8211; for all budgets. Don&#8217;t have any money to spend? Use cardboard boxes and cover them with contact paper. Then label them with index cards. Or you can repurposean old dresser thatyou&#8217;re no longer usingand paint it to give it new life. Then label each drawer. Use your imagination. The sky isthe limit with organizing.&lt;/p&gt;

  function textValidation(input) {
    if(input.id == 'signup_email') return;
    var val = input.value;
    if(val == '') {
      textFieldError(input);
      isValid = false;
      return;
    } else {
      removeTextFieldError(input)
    }
  }

  function dropdownValidation(field, input) {
    var val = input.value;
    if(val == '') {
      if(field.className.indexOf('invalid') == -1) field.className += ' invalid';
      onSelectCallback(input);
      isValid = false;
      return;
    } else {
      field.className = field.className.replace(/ invalid/g, '');
    }
  }

  function htmlEmbedDropdownValidation(field) {
    var dropdowns = field.querySelectorAll('.mimi_html_dropdown');
    for(var i = 0; i &lt; dropdowns.length; ++i) {
      var dropdown = dropdowns[i],
          val      = dropdown.value;
      if(val == '') {
        if(field.className.indexOf('invalid') == -1) field.className += ' invalid';
        isValid = false;
        dropdown.onchange = validate;
        return;
      } else {
        field.className = field.className.replace(/ invalid/g, '');
      }
    }
  }

  function textFieldError(input) {
    input.className   = 'required invalid';
    input.placeholder = input.getAttribute('data-required-field');
  }

  function removeTextFieldError(input) {
    input.className   = 'required';
    input.placeholder = '';
  }

  function onSelectCallback(input) {
    if(typeof Widget != 'undefined' &amp;&amp; Widget.BasicDropdown != undefined) {
      var dropdownEl = input.parentNode,
          instances  = Widget.BasicDropdown.instances;
      for(var i = 0; i &lt; instances.length; ++i) {
        var instance = instances[i];
        if(instance.wrapperEl == dropdownEl) {
          instance.onSelect = validate;
        }
      }
    }
  }

  function updateFormAfterValidation() {
    form.className   = isValid ? '' : 'mimi_invalid';
    submit.value     = submitButtonText();
    submit.disabled  = !isValid;
    submit.className = isValid ? 'submit' : 'disabled';
  }

  function submitButtonText() {
    var invalidFields = document.querySelectorAll('.invalid'),
        text;
    if(isValid || invalidFields == undefined) {
      text = submit.getAttribute('data-default-text');
    } else {
      if(invalidFields.length &gt; 1 || invalidFields[0].className.indexOf('checkgroup') == -1) {
        text = submit.getAttribute('data-invalid-text');
      } else {
        text = submit.getAttribute('data-choose-list');
      }
    }
    return text;
  }

  function revalidateOnChange() {
    var fields = form.querySelectorAll(".mimi_field.required");
    for(var i = 0; i &lt; fields.length; ++i) {
      var inputs = fields[i].getElementsByTagName('input');
      for(var j = 0; j &lt; inputs.length; ++j) {
        inputs[j].onchange = validate;
      }
    }
  }
