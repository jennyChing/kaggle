                There is no ready report available in Navision for Fund flow statement before releasing of new version of navision i.e NAV 2013
               </p>
               <p>
                In Navision 2013 you can have look at the cash flow statement functionality which will give you the desired result .For Setting up you can us this link http://msdn.microsoft.com/en-us/library/hh168290(v=nav.71).aspx
               </p>
               <p>
                If you are using Navision 2009 classic version then you need to create the statement using Account Schedules or creating a new report
