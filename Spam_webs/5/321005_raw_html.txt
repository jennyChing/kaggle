      <h2>
       LinuxCommand.org for Off-Line Viewing
      </h2>
      <p>
       This is a downloadable version of LinuxCommand.org you can view
without an Internet connection.  Simply download
       <a href="http://downloads.sourceforge.net/linuxcommand/linuxcommand.org-20080510.tar.gz?modtime=1210441966&amp;big_mirror=0">
        this file
       </a>
       and expand it with the following command:
      </p>
      <tt>
       tar xzvf linuxcommand.org-20080510.tar.gz
      </tt>
      <p>
       and point your browser to the
       <tt>
        index.html
       </tt>
       file contained within the
       <tt>
        linuxcommand.org
       </tt>
       directory.
      </p>
      <p>
       For those of you without access to the tar and gzip programs, the site is also
available in
       <a href="http://downloads.sourceforge.net/linuxcommand/linuxcommand.org-20080510.zip?modtime=1210442025&amp;big_mirror=0">
        zip file format
       </a>
       .
      </p>

      <h2>
       Screen Background Images
      </h2>
      <p class="fineprint">
       by William Shotts, Jr.
      </p>
      <p>
       Yeah, I know it's not a script, but I have been dismayed by
the quality of many of the screen background images that
are supplied with popular Linux distributions.
      </p>
      <p>
       To help make things better, I have designed
some custom background images.  They must be good,
I use them myself!
      </p>
      <p>
       You can download them from my
       <a href="http://www.williamshottsphotography.com/Wallpapers">
        photography site
       </a>
       .
      </p>

      <h2>
       <a name="rh-errata">
        rh-errata
       </a>
      </h2>
      <p class="fineprint">
       by William Shotts, Jr.
      </p>
      <p>
       <em>
        rh-errata
       </em>
       is a tool for maintaining the
integrity of a Red Hat Linux system. It builds a
respository of RPM files from a designated mirror
of the Red Hat updates site. It only downloads RPM
files that have not already been downloaded.
Optionally, it can produce a report that shows
which RPM files in your respository need to be
updated and which ones have already been
installed.
      </p>
      <p>
       <a href="html_text/rh-errata.README.html">
        View
README
       </a>
      </p>
      <p>
       <a href="html_text/rh-errata.html">
        View
Script
       </a>
      </p>
      <p>
       <a href="http://prdownloads.sourceforge.net/linuxcommand/rh-errata-1.2.4.tar.gz">
        Download the rh-errata package
