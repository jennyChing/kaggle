          <h1>
           BitTorrent macht P2P-Browser Project Maelstrom als Beta verfgbar
          </h1>
          <span class="byline">
           von
           <a class="url" href="/autor/florian-kalenda/">
            Florian Kalenda
           </a>
           am
           <a href="http://www.zdnet.de/date/2015/04/13/" rel="bookmark" title="20:02">
            <time datetime="2015-04-13" pubdate="">
             13. April 2015
            </time>
           </a>
           , 20:02 Uhr
          </span>
         </header>
        </div>
        <div id="articleContent">
         <p>
          BitTorrent hat seinen Peer-to-Peer-Browser Project Maelstrom als Beta
          <a href="http://blog.bittorrent.com/2015/04/10/project-maelstrom-enters-beta/" target="_blank">
           fr Windows verfgbar
          </a>
          gemacht.
          <a href="http://blog.bittorrent.com/2014/12/10/project-maelstrom-the-internet-we-build-next/" target="_blank">
           Seit Dezember
          </a>
          war er als Alpha-Version vorgelegen. Project Maelstrom basiert auf Chromium, kombiniert mit der Mglichkeit, Inhalte im Peer-to-Peer-Verfahren (P2P) zwischen Usern zu verteilen  ohne Server.
         </p>
         <script type="text/javascript">
          (function($){var device="desktop";var screenWidth=$(window).width();if((screenWidth&gt;750&amp;&amp;device=="desktop")||(screenWidth&lt;=750&amp;&amp;device=="mobile")){var creative_id="div-gpt-ad-DESKTOP_IN_ARTICLE-0";document.write('&lt;div id="'+creative_id+'" class="dfp_ad" height="0"&gt;');var word="";if(word!=""){document.write('&lt;div class="dfp_word"&gt;'+word+'&lt;/div&gt;');}googletag.cmd.push(function(){googletag.display("div-gpt-ad-DESKTOP_IN_ARTICLE-0");});document.write('&lt;/div&gt;');}})(jQuery);
         </script>
         <p>
          Der Browser setzt die gleichen Protokolle wie Torrent-Streams ein, um Inhalte auch von anderen Nutzern beziehen zu knnen. Dies knnte sich als gnstige Strategie fr Angebote wie das Torrent-Portal The Pirate Bay erweisen, dessen Server
          <a href="http://www.zdnet.de/88213589/pirate-bay-nach-razzia-schweden-offline/" target="_blank">
           in Schweden vergangenes Jahr vom Netz genommen
          </a>
          wurde. Sollte sich so etwas wiederholen, bliebe das Angebot ber andere User weiter verfgbar.
         </p>
         <span class="wp-caption aligncenter size-lar" id="attachment_88231410">
          <a class="zoomImage" href="http://www.zdnet.de/wp-content/uploads/2015/04/bittorrent-browser-maelstrom.png">
           <img alt="Browser Project Maelstrom (Bild: BitTorrent)" class="size-large wp-image-88231410" height="362" src="http://www.zdnet.de/wp-content/uploads/2015/04/bittorrent-browser-maelstrom-580x362.png" width="580"/>
          </a>
          <span class="wp-caption-text">
           P2P-Browser Project Maelstrom (Bild: BitTorrent)
          </span>
         </span>
         <p>
          Da Project Maelstrom auf der Open-Source-Version von
          <span class="wp_keywordlink">
           <a href="http://www.zdnet.de/themen/chrome/" title="Themenschwerpunkt: Google Chrome">
            Chrome
           </a>
          </span>
          basiert, kann er auch ganz normal per HTTP und HTTPS auf Sites zugreifen. BitTorrent feiert ihn als Mglichkeit, das Web offener zu gestalten und es der Kontrolle einiger seiner Trsteher zu entziehen. Auch sei P2P-Traffic eine Entlastung der Infrastruktur.
         </p>

         <p>
          <a href="http://blog.bittorrent.com/2014/12/10/project-maelstrom-the-internet-we-build-next/" target="_blank">
           In einem Blogbeitrag
          </a>
          schreibt BitTorrent: Wir glauben, dass dieses Projekt das Potenzial hat, einige der rgerlichsten Probleme des heutigen Internets zu lsen. Wie knnen wir das Internet offen halten? Wie knnen wir den Zugang zum Internet neutral gestalten? Wie knnen wir besser sicherstellen, dass unsere privaten Daten nicht von groen Firmen missbraucht werden? Wie knnen wir helfen, um das Internet effizient an die Zunahme der Inhalte anzupassen?
         </p>
         <p>
          Seit der Alpha-Verffentlichung des Browsers haben sich demnach 10.000 Entwickler und 3500 Verleger in das Projekt eingebracht. Mit Verfgbarkeit der Beta gibt es nun auch neue
          <a href="https://github.com/bittorrent/torrent-web-tools" target="_blank">
           Entwicklerwerkzeuge auf GitHub
          </a>
          . Damit lassen sich die Torrent-Dateien erstellen, die fr eine dynamische Verteilung statischer Webseiten sorgen knnen. Frs Blogsystem WordPress liegt sogar ein Plug-in vor, das Torrent-Dateien automatisch generiert.
         </p>
         <p>
          Das Unternehmen
          <a href="http://blog.bittorrent.com/2015/04/10/project-maelstrom-developer-tools/" target="_blank">
           schreibt
          </a>
          : Das Entwicklerwerkzeug ermglicht Ihnen, ganz einfach Dateien fr Project Maelstrom zu generieren, sogar von der Kommandozeile aus. So lassen sich ohne Umwege Inhalte erstellen und publizieren, auf die Nutzer mit Project Maelstrom zugreifen knnen. Unter anderem lassen sich Dateien einer Website so als Torrent verfgbar machen. Dann sind sie ber
          <a href="http://de.wikipedia.org/wiki/Liste_von_BitTorrent-Clients" target="_blank">
           BitTorrent-Clients
          </a>
          verfgbar, whrend sie in einem normalen Browser ber einen Magnet-Link erreicht werden knnen.
         </p>
         <p>
          Derzeit steht Project Maelstrom nur fr Windows zur Verfgung. Eine Mac-OS-Version ist in Vorbereitung. Auch eine Final hofft BitTorrent noch dieses Jahr anbieten zu knnen.
         </p>
