                     Makeup Wars: SPF Favorites!!!

                     <div class="post-header-line-1">

                       It is time for another
                       <b>
                        Makeup War
                       </b>
                       !!! This time the Makeup Wars bloggers are discussing their favorite sunscreens! SPF is not only our best anti-aging defense but also a total lifesaver. It is absolutely a product that I use everyday, without fail, on all exposed body parts: face, arms, and hands, etc.
                       <br/>
                       <br/>
                       Lets talk SPF! Summertime is the perfect time to look at your skincare routine to see what types of sun protection you need to add. I am a bit of an admitted SPF-junkie. My skin is super-duper fair. So, not only do I seek out the anti-aging and cancer protection benefits of sunscreen, but also I don't want to get burned! Plus, Skin Cancer runs in my family (My Dad) and I have already lost a very dear friend to Melanoma. So, I honestly do not mess around when it comes to SPF. I inherited my Dad's complexion. So, I know I have an extremely high risk for Skin Cancer because of his Cancer history.
                       <br/>
                       <br/>
                       <b>
                        What types of SPF?
                       </b>
                       <br/>
                       My SPF needs change depending upon what "mood" my temperamental combination skin is in, what time of year it is, and what activities I have planned for the day. Spending the day outdoors? Then I choose a high SPF. At the Pool? I apply something water resistant. Is my skin acting a bit oily? I go with a mattifying sunscreen. You get the idea! :) So, with that said, I have quite an assortment of sunscreens at my disposal and I choose whatever my skin requires.

                      <div style="text-align: left;">
                       Want to know more details? Keep reading for the scoop on my favorite sunscreens for the face, body, and the baby!
                       <br/>
                       <br/>
                       <a name="more">
                       </a>
                       My Current Arsenal of Sunscreens for the Face, Body, and for Baby!
                       <br/>
                       <br/>
                       My facial sunscreen needs are in constant fluctuation depending on how my combination skin is acting and the weather. In the colder months, my skin leans combo-dry. When it gets warmer, my skin gets quite oily in the t-zone. So, I have included some of the sunscreens that I like for my face for different times of the year.
                       <br/>
                       <br/>
                       1.
                       <a href="http://go.redirectingat.com/?id=43479X1133779&amp;xs=1&amp;url=http%3A%2F%2Fwww.target.com%2Fp%2Fneutrogena-pure-free-baby-sunblock-lotion-spf-60-3-oz%2F-%2FA-11133516">
                        Neutrogena Pure &amp; Free Baby Sunblock Lotion
                       </a>
                       <b>
                        :
                       </b>
                       This is my favorite baby sunscreen! I trust it to protect my baby from burns and the damage of the sun. I have complete confidence when I apply this product to her sweet baby skin! I love that the ingredients are pure. Who wants to put chemicals on a baby? Not me!!!
                       <br/>
                       <br/>
                       2.
                       <a href="http://go.redirectingat.com/?id=43479X1133779&amp;xs=1&amp;url=http%3A%2F%2Fwww.drugstore.com%2Faveeno-active-naturals-protect-hydrate-spf-30-lotion%2Fqxp461259%3Fcatid%3D184131">
                        Aveeno Protect &amp; Hydrate Lotion Sunscreen
                       </a>
                       : This new sunscreen line from Aveeno aims to actually leave your skin looking and feeling healthier than before your sun exposure! It provides broad-spectrum sun protection while also hydrating your skin. The formula smells soooo good! There is something so fresh about the scent! This product is designed for the body. However, it is oil-free. So, you can use it on your face, too. It is water and sweat resistant (for 80 minutes after application). It comes in SPF 30, 50, and 70 and retails for about $9.99.
                       <br/>
                       <br/>
                       3.
                       <a href="http://go.redirectingat.com/?id=43479X1133779&amp;xs=1&amp;url=http%3A%2F%2Fwww.rocskincare.com%2Fmulti-correxion%2F4-zone-daily-moisturizer-broad-spectrum-spf-30">
                        RoC's Multi Correxion 4 Zone Daily Moisturizer with SPF 30
                       </a>
                       : This is an SPF that combines anti-aging retinol skincare with an effective broad spectrum SPF. I feel that I get an "all-in-one" effect when I use this as you are giving your skin anti-aging benefits while also providing protection and moisturization. This is my daily SPF for my neck and decolletage. I rotate using this on my face with my other facial sunscreens. I find that I reach for this product for use on my face when I am in a hurry and I don't have time to layer sunscreen over an anti-aging serum. The 4-Zone name refers to it helping to protect the "4 Zones" of your face. I totally recommend this one! A time-saver for sure! :)
                       <br/>
                       <br/>
                       4.
                       <a href="http://go.redirectingat.com/?id=43479X1133779&amp;xs=1&amp;url=http%3A%2F%2Fshop.nordstrom.com%2Fs%2Fkate-somerville-body-glow-sunscreen-broad-spectrum-spf-20%2F3365861%3Fcm_cat%3Ddatafeed%26cm_ite%3Dkate_somerville(r)_%27body_glow%27_sunscreen_broad_spectrum_spf_20%3A598588%26cm_pla%3Dskin%2Fbody_treatment%3Awomen%3Asunscreen%26cm_ven%3DGoogle_Product_Ads%26mr%3AreferralID%3D4eb0e377-9dde-11e2-9dea-001b2166becc">
                        Kate Somerville Body Glow Sunscreen
                       </a>
                       : This Sunscreen is such a luxurious treat! Not only does it smell divine, but also it leaves your skin looking lovely! The scent has a hint of tropical citrus combined with the classic sunscreen smell. The formula contains light reflective pearls that gives you a beautiful and subtle shimmer while perfecting your skin's tone. Mango Butter makes this sunscreen super moisturizing! The active sunscreens are a combo of Avobenzone, Homosalate, Octisalate, Octocrylene, and Oxybenzone. This sunscreen retails for $45 for 5 fl oz. If money were no object, I would exclusively use this SPF for the body as I really love how it gives your skin a subtle shimmer. However, it is expensive. So, I use this when I particularly want my skin looking good!
                       <br/>
                       <br/>
                       5.
                       <a href="http://go.redirectingat.com/?id=43479X1133779&amp;xs=1&amp;url=http%3A%2F%2Fwww.ulta.com%2Fulta%2Fbrowse%2FproductDetail.jsp%3FproductId%3DxlsImpprod3060013">
                        Neutrogena Wet Skin Broad Spectrum SPF
                       </a>
                       : I think that spray sunscreens are seriously the best invention since sliced bread. This one in particular tips the scales of invention: A sunscreen that you can apply to wet skin! Genius!!! I find that I often need sunscreen the most when I am wet. You know, you climb out of the pool and even after you towel off, you are still a bit wet and much of your previous sunscreen has washed off. Just spray on this bad boy and you are protected again! This sunscreen magically cuts through the water and adheres to your skin to keep you protected. I honestly think that I will always make sure to keep a can of this amazing sunscreen in my pool/beach bag! I am very impressed!!! Plus, it is waterproof and sweat proof (and yes, you can apply it to dry skin as well).
                       <br/>
                       <br/>
                       6.
                       <a href="http://go.redirectingat.com/?id=43479X1133779&amp;xs=1&amp;url=http%3A%2F%2Fwww.skinstore.com%2Fp-16844-b-kamins-matte-moisturizer-spf-15.aspx">
                        B. Kamins Matte Moisturizer SPF 15
                       </a>
                       : This sunscreen is a great option for oily and combination to oily breakout prone skin! My skin has been very hormonal lately (ack!). I like using this sunscreen when my skin gets temperamental. It is a lightweight moisturizer that hydrates without leaving your skin feeling oily. It is free of fragrance, lanolin, and oil. I love the sleek glass jar packaging. It retails for about $70.
                       <br/>
                       <br/>
                       7.
                       <a href="http://go.redirectingat.com/?id=43479X1133779&amp;xs=1&amp;url=http%3A%2F%2Fwww.drugstore.com%2Faveeno-active-naturals-protect-hydrate-spf-30-lotion%2Fqxp461259%3Fcatid%3D184131">
                        B. Kamins Day Cream SPF 15
                       </a>
                       : When my skin is in combo-dry mode, I reach for richer sunscreens and moisturizers. This B. Kamins Day cream is a great option for dry skin! It is super thick and rich yet it is also non-greasy and has a lightweight feel. It comes with a "scooper" to dig the product out of the jar. It leaves your skin looking and feeling soft, smooth, and moisturized. It is a great primer for makeup, too! The formula contains Avobenzone, Octinoxate, and Oxybenzone. This cream retails for $71 for a 1.7 oz glass jar.
                       <br/>
                       <br/>
                       8.
                       <a href="http://go.redirectingat.com/?id=43479X1133779&amp;xs=1&amp;url=http%3A%2F%2Fwww.sephora.com%2Fregenerative-anti-aging-moisturizer-spf-20-P282937">
                        Algenist Regenerative Anti-Aging Moisturizer with SPF 20
                       </a>
                       : This is my go-to facial moisturizer with sunscreen for Fall and Winter. It is super duper moisturizing. So, I like to use it during the months when my skin is more on the dry side of combo. I adore the smell and texture of this luxurious lotion! Also, the SPF is a bit low at 20. So, that works for Fall and Winter, too (as I get less sun exposure when it is cold out). If you have never tried Algenist (one of my fav skincare brands!), this daily moisturizer is a great introduction into the line as it contains the line's key ingredient: Alguronic Acid. To find out more about this line and read my review of their amazing eye renewal balm, please see one of my previous posts:
                       <a href="http://www.pammyblogsbeauty.com/2012/04/newest-thing-in-skincare-alguronic-acid.html">
                        The Newest Thing in Skincare: Alguronic Acid
                       </a>
                       .

(function($){

	/* hoverIntent by Brian Cherne */

	$.fn.hoverIntent = function(f,g) {

		// default configuration options

		var cfg = {

			sensitivity: 7,

			interval: 100,

			timeout: 0

		};

		// override configuration options with user supplied object

		cfg = $.extend(cfg, g ? { over: f, out: g } : f );

 

		// instantiate variables

		// cX, cY = current X and Y position of mouse, updated by mousemove event

		// pX, pY = previous X and Y position of mouse, set by mouseover and polling interval

		var cX, cY, pX, pY;

 

		// A private function for getting mouse position

		var track = function(ev) {

			cX = ev.pageX;

			cY = ev.pageY;

		};

 

		// A private function for comparing current and previous mouse position

		var compare = function(ev,ob) {

			ob.hoverIntent_t = clearTimeout(ob.hoverIntent_t);

			// compare mouse positions to see if they've crossed the threshold

			if ( ( Math.abs(pX-cX) + Math.abs(pY-cY) ) &lt; cfg.sensitivity ) {

				$(ob).unbind("mousemove",track);

				// set hoverIntent state to true (so mouseOut can be called)

				ob.hoverIntent_s = 1;

				return cfg.over.apply(ob,[ev]);

			} else {

				// set previous coordinates for next time

				pX = cX; pY = cY;

				// use self-calling timeout, guarantees intervals are spaced out properly (avoids JavaScript timer bugs)

				ob.hoverIntent_t = setTimeout( function(){compare(ev, ob);} , cfg.interval );

			}

		};

 

		// A private function for delaying the mouseOut function

		var delay = function(ev,ob) {

			ob.hoverIntent_t = clearTimeout(ob.hoverIntent_t);

			ob.hoverIntent_s = 0;

			return cfg.out.apply(ob,[ev]);

		};

 

		// A private function for handling mouse 'hovering'

		var handleHover = function(e) {

			// next three lines copied from jQuery.hover, ignore children onMouseOver/onMouseOut

			var p = (e.type == "mouseover" ? e.fromElement : e.toElement) || e.relatedTarget;

			while ( p &amp;&amp; p != this ) { try { p = p.parentNode; } catch(e) { p = this; } }

			if ( p == this ) { return false; }

 

			// copy objects to be passed into t (required for event object to be passed in IE)

			var ev = jQuery.extend({},e);

			var ob = this;

 

			// cancel hoverIntent timer if it exists

			if (ob.hoverIntent_t) { ob.hoverIntent_t = clearTimeout(ob.hoverIntent_t); }

 

			// else e.type == "onmouseover"

			if (e.type == "mouseover") {

				// set "previous" X and Y position based on initial entry point

				pX = ev.pageX; pY = ev.pageY;

				// update "current" X and Y position based on mousemove

				$(ob).bind("mousemove",track);

				// start polling interval (self-calling timeout) to compare mouse coordinates over time

				if (ob.hoverIntent_s != 1) { ob.hoverIntent_t = setTimeout( function(){compare(ev,ob);} , cfg.interval );}

 

			// else e.type == "onmouseout"

			} else {

				// unbind expensive mousemove event

				$(ob).unbind("mousemove",track);

				// if hoverIntent state is true, then call the mouseOut function after the specified delay

				if (ob.hoverIntent_s == 1) { ob.hoverIntent_t = setTimeout( function(){delay(ev,ob);} , cfg.timeout );}

			}

		};

 

		// bind the function to the two event listeners

		return this.mouseover(handleHover).mouseout(handleHover);

	};

 

})(jQuery);
