        Frosted Grapes Recipe:  Elegant for Appetizer or Simple Treat for Kids!
       </h1>

        <div class="shareaholic-canvas" data-app="share_buttons" data-app-id="17931409" data-link="http://www.suburbia-unwrapped.com/2014/08/frosted-grapes-recipe.html" data-summary="" data-title="Frosted Grapes Recipe: Elegant for Appetizer or Simple Treat for Kids!">
        </div>
        <p>
         Grapes are one of my familys all time favorite fruits. If I put out a bowl of grapes I can pretty much be guaranteed that they will be gone by the end of the day. I have never actually done much COOKING with grapes, though, so when our Food Bloggers group decided to spotlight grapes as our weekly food challenge I was a little bit stumped. Honestly, I dont think I have any recipes in my repertoire that call for grapes! Usually we just pop them in our mouth until the bowl is empty! My first attempt at a grape recipe was a total flopI tried making grape jam and the resulting bowl of lumpy liquid was SO not anything that resembled grape jam! Oh well, off to get more grapes and find a new recipe, right? When I came across the idea of frosted grapes I was intrigued!
        </p>
        <p>
         <a href="http://www.suburbia-unwrapped.com/wp-content/uploads/2014/08/frosted-grapes_edited-1.jpg">
          <img alt="How to make Frosted Grapes" class="aligncenter size-full wp-image-5490" height="358" src="http://www.suburbia-unwrapped.com/wp-content/uploads/2014/08/frosted-grapes_edited-1.jpg" width="540"/>
         </a>
        </p>
        <p>
         So, why make frosted grapes, you ask? Arent they sweet enough as it is? Yes, I must admit that plain old grapes are plenty sweet! But, frosted grapes make a cute addition to an appetizer platter surrounded by crackers and cheese! They are also appealing to kids who may be looking for a little something sweet but you dont want them eating candy. I came across several recipes and suggestions, many of which used Jello brand jello powder. I was at Sprouts Market and came across a (&gt;&gt;&gt;
         <em>
          affiliated link
         </em>
         )
         <a href="http://www.amazon.com/s/?_encoding=UTF8&amp;camp=1789&amp;creative=390957&amp;field-keywords=simply%20delish%20jello&amp;linkCode=ur2&amp;sprefix=simply%20delish%20%2Caps%2C577&amp;tag=turtheclobac-20&amp;url=search-alias%3Dgrocery&amp;linkId=WQI2ZIQIB5AV7FCD" onclick="_gaq.push(['_trackEvent', 'outbound-article', 'http://www.amazon.com/s/?_encoding=UTF8&amp;camp=1789&amp;creative=390957&amp;field-keywords=simply%20delish%20jello&amp;linkCode=ur2&amp;sprefix=simply%20delish%20%2Caps%2C577&amp;tag=turtheclobac-20&amp;url=search-alias%3Dgrocery&amp;linkId=WQI2ZIQIB5AV7FCD', 'natural gelatin dessert']);" rel="nofollow" target="_blank">
          natural gelatin dessert
         </a>
         <img alt="" border="0" height="1" src="https://ir-na.amazon-adsystem.com/e/ir?t=turtheclobac-20&amp;l=ur2&amp;o=1" style="border: none !important; margin: 0px !important;" width="1"/>
         that worked just great. No artificial colors and flavors is always a good thing in my mind!
        </p>
        <p>
         To make frosted grapes some people recommend brushing the grapes with beaten egg whites and then sprinkling them with sugar or flavored gelatin. I was a little hesitant to have raw egg whites on my grapes and honestly, water worked just fine! Just make sure you pat the grapes dry a bit so they are not really WETjust a tad bit damp!
        </p>

          this.textAndDropdownValidation(field, type);
        }
      }
    },

    fieldType: function(field) {
      var type = field.querySelectorAll('.field_type');

      if (type.length) {
        return type[0].getAttribute('data-field-type');
      } else if (field.className.indexOf('checkgroup') &gt;= 0) {
        return 'checkboxes';
      } else {
        return 'text_field';
      }
    },

    checkboxAndRadioValidation: function(field) {
      var inputs   = field.getElementsByTagName('input'),
          selected = false;

      for (var i = 0; i &lt; inputs.length; ++i) {
        var input = inputs[i];
        if((input.type === 'checkbox' || input.type === 'radio') &amp;&amp; input.checked) {
          selected = true;
        }
      }

      if (selected) {
        field.className = field.className.replace(/ invalid/g, '');
      } else {
        if (field.className.indexOf('invalid') === -1) {
          field.className += ' invalid';
        }

        this.isValid = false;
      }
    },

    textAndDropdownValidation: function(field, type) {
      var inputs = field.getElementsByTagName('input');

      for (var i = 0; i &lt; inputs.length; ++i) {
        var input = inputs[i];
        if (input.name.indexOf('signup') &gt;= 0) {
          if (type === 'text_field') {
            this.textValidation(input);
          } else {
            this.dropdownValidation(field, input);
          }
        }
      }
      this.htmlEmbedDropdownValidation(field);
    },

    textValidation: function(input) {
      if (input.id === 'signup_email') return;

      if (input.value) {
        this.removeTextFieldError(input);
      } else {
        this.textFieldError(input);
        this.isValid = false;
      }
    },

    dropdownValidation: function(field, input) {
      if (input.value) {
        field.className = field.className.replace(/ invalid/g, '');
      } else {
        if (field.className.indexOf('invalid') === -1) field.className += ' invalid';
        this.onSelectCallback(input);
        this.isValid = false;
      }
    },

    htmlEmbedDropdownValidation: function(field) {
      var dropdowns = field.querySelectorAll('.mimi_html_dropdown');
      var _this = this;

      for (var i = 0; i &lt; dropdowns.length; ++i) {
        var dropdown = dropdowns[i];

        if (dropdown.value) {
          field.className = field.className.replace(/ invalid/g, '');
        } else {
          if (field.className.indexOf('invalid') === -1) field.className += ' invalid';
          this.isValid = false;
          dropdown.onchange = (function(){ _this.validate(); });
        }
      }
    },

      input.className   = 'required invalid';

      }

          return name += ' mimi_invalid';
        } else {
          return name;
        }
      }
    },

        text = this.submit.getAttribute('data-default-text');
      } else {
        if (invalidFields.length || invalidFields[0].className.indexOf('checkgroup') === -1) {
          text = this.submit.getAttribute('data-invalid-text');
        } else {
          text = this.submit.getAttribute('data-choose-list');
        }

      return text;

    submitForm: function() {
      this.formSubmitting();

      var _this = this;
      window[this.callbackName] = function(response) {
        delete window[this.callbackName];
        document.body.removeChild(script);

      document.body.appendChild(script);

      var action  = this.form.action;
      if (format === 'json') action += '.json';
      return action + '?callback=' + this.callbackName + '&amp;' + serialize(this.form);
    },

      this.form.className  += ' mimi_submitting';

      this.submit.disabled  = true;
      this.submit.className = 'disabled';

      } else {
        top.location.href = this.formUrl('html');
      }
    },

    onSubmitSuccess: function(result) {
      if (result.has_redirect) {
        top.location.href = result.redirect;
      } else if(result.single_opt_in || !result.confirmation_html) {
        this.disableForm();
        this.updateSubmitButtonText(this.submit.getAttribute('data-thanks'));
      } else {
        this.showConfirmationText(result.confirmation_html);
      }

    },

      var elements = this.form.elements;
      for (var i = 0; i &lt; elements.length; ++i) {
        elements[i].disabled = true;
      }
    },

      this.submit.value = text;
    },

          _this = this;

              }
            }
          } else {
            inputs[j].onchange = function(){ _this.validate() };
          }
        }
      }
    }
