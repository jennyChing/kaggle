        <div class="entry-content">
         <p>
          Sometimes a person needs to hide an important file somewhere in order to protect it from unauthorized access. Making it only hidden is not the best way to hide an important file since it can be accessed by using powerful 3rd party search tools. Why not hide the file inside a calculator?
         </p>
         <p>
          <a href="http://krilome.com/aprod/index-cse.html">
           Safe Calculator
          </a>
          does exactly that. It is a portable tool that is both a calculator and a secret safe. You can save only one file inside it, if you have multiple files, you can zip them and then save it inside the calculator. This is how it works, when you enter the secret PIN, and hit MS key, it will automatically turn into a safe where you can hide/extract/view the file.
         </p>
         <div id="single-paragraph-add">
         </div>
         <p>
          <strong>
           Update:
          </strong>
          This post is just for fun, it wont really hide large files. The best way is to use encryption software such as Windows 7s
          <a href="http://www.addictivetips.com/windows-tips/protect-your-drives-with-bitlocker-drive-encryption-in-windows-7/">
           Bitlocker
          </a>
          or
          <a href="http://www.addictivetips.com/windows-tips/usb-safeguard-usb-flash-drive-encryption-software/">
           USB Safeguard
          </a>
          .
         </p>
         <p>
          <span id="more-5361">
          </span>
         </p>
         <blockquote>
          <p>
           <strong>
            Note:
           </strong>
           The default PIN is 123, but you can change it later by entering a new PIN.
          </p>

          <img alt="Calc-En-v510" height="260" src="http://cloud.addictivetips.com/wp-content/uploads/2009/05/calcenv510.gif" style="display: block; float: none; margin-left: auto; margin-right: auto;" title="Calc-En-v510" width="260"/>
         </p>
         <p>
          You do not need to install the tool, just extract the content in the zip file and run it. When you add a file, the calculator will vanish for a few milliseconds which is normal and the file will be moved from its original location to inside the calculator.
         </p>
         <p>
          <img alt="safe calculator adding file" border="0" height="352" src="http://cloud.addictivetips.com/wp-content/uploads/2009/05/safecalculatoraddingfile.jpg" style="display: block; float: none; margin-left: auto; margin-right: auto; border: 0px;" title="safe calculator adding file" width="584"/>
         </p>
         <p>
          When you extract a file, it will be extracted in the same folder where the calculator is located. If you are saving a text file, it will automatically get encrypted. To get the full list of commands have a look at the Help file inside the zip archive. It works on all versions of Windows. Enjoy!
         </p>
