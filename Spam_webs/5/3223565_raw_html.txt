       <h2 class="step-title">
        Step 5: Ribbing
       </h2>
       <div class="photoset" data-entry-id="SAP4V1PGGC2U06H" data-entry-url="/id/Easy-Knit-Hat-Pattern/step5/Ribbing/" id="photoset-SAP4V1PGGC2U06H">
        <div class="row" style="width:600px;">
         <a class="photoset-link" data-fancybox-group="photoset-gallery-SAP4V1PGGC2U06H" data-fancybox-href="http://cdn.instructables.com/F4M/MDH5/GGMMIUR9/F4MMDH5GGMMIUR9.LARGE.jpg" data-href="/file/F4MMDH5GGMMIUR9" data-notes="" href="/file/F4MMDH5GGMMIUR9" id="photoset-link-FPNXKX2GGPESZHU-0" style="">
          <div class="photo-container" style="width:600px;">
           <img alt="Picture of Ribbing" class="photoset-photo id_F4MMDH5GGMMIUR9 " data-notes="" src="http://cdn.instructables.com/F4M/MDH5/GGMMIUR9/F4MMDH5GGMMIUR9.MEDIUM.jpg" style="width:600px;"/>
          </div>
         </a>
        </div>
       </div>
       <div class="txt step-body">
        <br/>
        This step is optional, and depends on if you want the brim of the hat to be ribbed or rolled.
        <br/>
        <br/>
        For ribbing, work the hat in a Knit 2, Purl 2 for about an inch. I like to use a stitch marker to indicate where I started ribbing so I can stop at the same point, but it's really not necessary.
        <br/>
        <br/>
        For the brim to roll up on itself, just start in continuous knitting.
