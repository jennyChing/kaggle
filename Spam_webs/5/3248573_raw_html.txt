                      </span>

                     <div class="separator" style="clear: both; text-align: center;">
                      <br/>
                     </div>
                     <div class="separator" style="clear: both; text-align: center;">

                       <br/>
                      </span>
                      <span style="background-color: #444444; color: white; font-family: Helvetica Neue, Arial, Helvetica, sans-serif; font-size: large;">
                       Supplies and Tools:
                      </span>
                      <span style="background-color: white;">
                       <o:p>
                       </o:p>
                      </span>
                     </div>
                     <div class="MsoNormal">
                      <span style="font-family: Trebuchet MS, sans-serif;">
                       Unused XL shirt from your husband/SO/boyfriend or thrift store, especially that has nice detail on the side
                       <o:p>
                       </o:p>
                      </span>
                     </div>
                     <div class="MsoNormal">
                      <span style="font-family: Trebuchet MS, sans-serif;">
                       Shorts for pattern
                      </span>
                     </div>
                     <div class="MsoNormal">
                      <span style="font-family: Trebuchet MS, sans-serif;">
                       Elastic band (I use 5mm)
                       <o:p>
                       </o:p>
                      </span>
                     </div>
                     <div class="MsoNormal">
                      <span style="font-family: Trebuchet MS, sans-serif;">
                       Elastic thread
                       <o:p>
                       </o:p>

                      </span>
                     </div>
                     <div class="MsoNormal">

                       Bias tape maker (if you dont have it, no worries)

                      </span>

                     <div class="MsoNormal">
                      <span style="background-color: #444444; color: white; font-family: Trebuchet MS, sans-serif; font-size: large;">
                       How to:
                      </span>
                     </div>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://2.bp.blogspot.com/-cnZ-HTO0I7c/UCO27_EqOkI/AAAAAAAAAYk/lDBpm7LHRLk/s1600/step1.jpg" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;">
                       <img border="0" height="235" src="http://2.bp.blogspot.com/-cnZ-HTO0I7c/UCO27_EqOkI/AAAAAAAAAYk/lDBpm7LHRLk/s400/step1.jpg" width="400"/>
                      </a>
                     </div>
                     <span style="font-family: 'Trebuchet MS', sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: 'Trebuchet MS', sans-serif;">
                      Cutsleeves, collar and the inside yoke.
                     </span>
                     <br/>
                     <span style="font-family: 'Trebuchet MS', sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: 'Trebuchet MS', sans-serif;">
                      Measure length from your shoulder to slightly under your waist and then cut the bodice.
                     </span>
                     <br/>
                     <span style="font-family: 'Trebuchet MS', sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: 'Trebuchet MS', sans-serif;">
                      The below bodice will be use as the bottom part (shorts)
                     </span>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://1.bp.blogspot.com/-ZjTCsFREOPw/UCO2-TIqziI/AAAAAAAAAY4/BC-Q8h32x58/s1600/step2.jpg" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;">
                       <img border="0" height="400" src="http://1.bp.blogspot.com/-ZjTCsFREOPw/UCO2-TIqziI/AAAAAAAAAY4/BC-Q8h32x58/s400/step2.jpg" width="345"/>
                      </a>
                     </div>
                     <span style="font-family: 'Trebuchet MS', sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: 'Trebuchet MS', sans-serif;">
                      Open wide the below bodice, cut in half right in the middle.
                     </span>
                     <br/>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      (I did this since I want to preserve the side detail and also it less sew for me in making the shorts)
                     </span>
                     <br/>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      Salvage one of the sleeves to add to the sides.
                     </span>
                     <br/>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      <br/>
                     </span>
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://1.bp.blogspot.com/-5wazDtM1-7k/UCO3AbgQlLI/AAAAAAAAAZA/dzRM-TocG1w/s1600/step3.jpg" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;">
                       <img border="0" height="400" src="http://1.bp.blogspot.com/-5wazDtM1-7k/UCO3AbgQlLI/AAAAAAAAAZA/dzRM-TocG1w/s400/step3.jpg" width="351"/>
                      </a>
                     </div>
                     <br/>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      Trace pattern from the short you have and arrange it as in the picture.
                     </span>
                     <br/>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      The bottom part has no front opening, so make sure the width of the top opening is larger than your hip measurement
                     </span>
                     <br/>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      With right side facing each other, pin and sew the sides.
                     </span>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://4.bp.blogspot.com/-U7YRralSiJ8/UCO3CGtQZWI/AAAAAAAAAZI/1x6ivXJfimY/s1600/step4.jpg" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;">
                       <img border="0" height="228" src="http://4.bp.blogspot.com/-U7YRralSiJ8/UCO3CGtQZWI/AAAAAAAAAZI/1x6ivXJfimY/s320/step4.jpg" width="320"/>
                      </a>
                     </div>
                     <br/>
                     <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                      To sew the crotch, put the fabric like normal short, wrong side up.
                     </span>
                     <br/>
                     <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                      Pin and sew the crotch.
                     </span>
                     <br/>
                     <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                      Hem all the bottom edge of the shorts.
                     </span>
                     <br/>
                     <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                      You finished the bottom part of the romper
                     </span>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <span style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em; text-align: justify;">
                       <img border="0" height="244" src="http://2.bp.blogspot.com/-sadkfyitZHU/UCO3DP_IxlI/AAAAAAAAAZQ/eMkhh6ABM98/s640/step5.jpg" width="640"/>
                       <span style="font-family: Trebuchet MS, sans-serif;">
                        Next, create a racer back on the back side of the top.
                       </span>
                      </span>
                      <span style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em; text-align: justify;">
                       <span style="font-family: Trebuchet MS, sans-serif;">
                        Cut in curve or C-shaped
                       </span>
                      </span>
                     </div>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://4.bp.blogspot.com/-Y_DdN83D-l0/UCO3EMmA2cI/AAAAAAAAAZY/k_uAnZ2MeMo/s1600/step6.jpg" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;">
                       <img border="0" height="403" src="http://4.bp.blogspot.com/-Y_DdN83D-l0/UCO3EMmA2cI/AAAAAAAAAZY/k_uAnZ2MeMo/s640/step6.jpg" width="640"/>
                      </a>
                     </div>
                     <div style="text-align: left;">
                      <span style="font-family: Trebuchet MS, sans-serif;">
                       Before attaching the top and bottom part, make sure you have the same opening size. If its larger, pinch as your measurement from the sleeves opening.
                      </span>
                     </div>
                     <div style="text-align: left;">
                      <span style="font-family: Trebuchet MS, sans-serif;">
                       <br/>
                      </span>
                     </div>
                     <div style="text-align: left;">
                      <span style="font-family: Trebuchet MS, sans-serif;">
                       Attach the top part and the bottom part by facing the right side together.
                      </span>
                     </div>
                     <div style="text-align: left;">
                      <span style="font-family: Trebuchet MS, sans-serif;">
                       <br/>
                      </span>
                     </div>
                     <div style="text-align: left;">
                      <span style="font-family: 'Trebuchet MS', sans-serif;">
                       Pin and sew the parts by giving enough seam allowance for the elastic band (i.e if your elastic band 1 cm size, sew 2 cm minimum from the side)
                      </span>
                     </div>
                     <div style="text-align: left;">
                      <span style="font-family: 'Trebuchet MS', sans-serif;">
                       <br/>
                      </span>
                     </div>
                     <div style="text-align: left;">
                      <span style="background-color: #444444; color: white; font-family: Helvetica Neue, Arial, Helvetica, sans-serif; font-size: large;">
                       TIP:
                      </span>
                      <span style="background-color: white; font-family: Helvetica Neue, Arial, Helvetica, sans-serif; font-size: large;">
                      </span>
                      <span style="background-color: white; font-family: Trebuchet MS, sans-serif;">
                       you can add length by salvaging fabric from the sleeves if you want to lengthen the bottom/top part
                      </span>
                     </div>
                     <div style="text-align: left;">
                      <span style="font-family: 'Trebuchet MS', sans-serif;">
                       <br/>
                      </span>
                     </div>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://4.bp.blogspot.com/-TwUnQiASgEo/UCO3FpyMixI/AAAAAAAAAZg/hVxdOmg1lWY/s1600/step7.jpg" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;">
                       <img border="0" height="640" src="http://4.bp.blogspot.com/-TwUnQiASgEo/UCO3FpyMixI/AAAAAAAAAZg/hVxdOmg1lWY/s640/step7.jpg" width="457"/>
                      </a>
                     </div>
                     <br/>
                     <span style="font-family: 'Trebuchet MS', sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: 'Trebuchet MS', sans-serif;">
                      At this stage, you almost done but the shape of your new romper is unflattering.
                     </span>
                     <br/>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      To make it more fitted, smock or shirr the back side using elastic thread.
                     </span>
                     <br/>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      Technique on how to shirred you can read from
                      <a href="http://pretty-ditty.blogspot.nl/2009/04/smockingshirring-tutorial.html" target="_blank">
                       here
                      </a>
                      or ifyou're like me who use a Brother machine, use
                      <a href="http://www.designedbydawnnicole.com/2011/12/shirring-on-brother-sewing-machine-with.html" target="_blank">
                       this technique
                      </a>
                      instead.
                     </span>
                     <br/>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      <br/>
                     </span>
                     <span style="font-family: Trebuchet MS, sans-serif;">
                      I created 9 rows of 20 cm shirred on the back, with 1 cm space in between.
                     </span>
                     <br/>
                     <div class="" style="clear: both; text-align: left;">
                      <br/>
                     </div>
                     <div class="" style="clear: both; text-align: left;">
                      <a href="http://4.bp.blogspot.com/-IRXPsptyRvQ/UCO3HGwwSzI/AAAAAAAAAZo/40r1B6QEf-A/s1600/step8.jpg" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;">
                       <img border="0" height="640" src="http://4.bp.blogspot.com/-IRXPsptyRvQ/UCO3HGwwSzI/AAAAAAAAAZo/40r1B6QEf-A/s640/step8.jpg" width="377"/>
                      </a>
                      <span style="font-family: 'Trebuchet MS', sans-serif;">
                      </span>
                      <br/>
                      <span style="font-family: 'Trebuchet MS', sans-serif;">
                       <br/>
                      </span>
                      Using the seam allowance, sew around the waist but leave 2 cm opening for the elastic band.
                      <br/>
                      <span style="font-family: Trebuchet MS, sans-serif;">
                       <br/>
                      </span>
                      <span style="font-family: Trebuchet MS, sans-serif;">
                       Use safety pin to ease your way inserting elastic band.
                      </span>
                      <br/>
                      <span style="font-family: 'Trebuchet MS', sans-serif;">
                       <br/>
                      </span>
                      <span style="font-family: 'Trebuchet MS', sans-serif;">
                       Tie the elastic band per your measurement, and shut closed the opening gap.
                      </span>
                      <br/>
                      <br/>

                     </div>

                      <br/>

                     <div class="" style="clear: both; text-align: left;">

                       <img border="0" height="249" src="http://2.bp.blogspot.com/-PbowC0z2QB8/UCO3IWIZBdI/AAAAAAAAAZw/2rQbIgCGYzc/s640/step9.jpg" width="640"/>

                       To finished the neckline and the arm hole, make bias tape from the inside yoke.
