            var docViewTop = $(window).scrollTop(); //num of pixels hidden above current screen
            var docViewBottom = docViewTop + $(window).height();

       15 Really Useful Web-based HTML Editors
      </h1>

       <p>
        Web-based HTML Editor, WYSIWYG editor, or rich text editor, is a web component that let users enter rich text input within the browser. Most of the time, these html editors are used within content management systems where administrators or authors can easily create posts and contents from the backend system.
       </p>
       <p>
        Without talking much, lets see these powerful Web-based HTML Editors.
       </p>
       <h3>
        1.
        <a href="http://www.fckeditor.net/" target="_blank" title="FCKeditor">
         FCKeditor
        </a>
       </h3>
       <p>
        FCKeditor is one of the most famous online HTML editor. It brings the power and features of desktop editors to the web. It offers a complete integration pack for ASP, ASP.NET, PHP, Java, Perl, Phyton and etc. It supports skins, spell checker, lightweight, fast and has a lot of cool features.
        <br/>
        <img alt="FCKeditor" class="alignnone size-full wp-image-968" height="161" src="http://www.webdesignbooth.com/wp-content/uploads/2009/07/fckeditor.jpg" title="FCKeditor" width="500"/>
       </p>
       <h3>
        2.
        <a href="http://nicedit.com/" target="_blank" title="NicEdit">
         NicEdit
        </a>
       </h3>
       <p>
        NicEdit is a Javascript that integrates into any web page easily. It can turns any element/div to become editable or convert standard textareas to rich text editing.
        <br/>
        <img alt="NicEdit" class="alignnone size-full wp-image-969" height="104" src="http://www.webdesignbooth.com/wp-content/uploads/2009/07/nicedit.jpg" title="NicEdit" width="500"/>
       </p>
       <div class="center">

       <h3>
        3.
        <a href="http://tinymce.moxiecode.com/" target="_blank" title="TinyMCE">
         TinyMCE
        </a>
       </h3>
       <p>
        TinyMCE is an online wysiwyg html editor released under LGPL license. Like other wysiwyg html editors, it is able to convert textarea or other html elements to editor instances.
        <br/>
        <img alt="TinyMCE" class="alignnone size-full wp-image-970" height="181" src="http://www.webdesignbooth.com/wp-content/uploads/2009/07/tinymce.jpg" title="TinyMCE" width="500"/>
       </p>
       <h3>
        4.
        <a href="http://code.google.com/p/jwysiwyg/" target="_blank" title="jwysiwyg">
         jwysiwyg
        </a>
       </h3>
       <p>
        jwysiwyg is a jQuery WYSIWYG plugin with a tiny size (7kb packed). It is very simple and without much features.
        <br/>
        <img alt="jwysiwyg - jQuery WYSIWYG plugin" class="alignnone size-full wp-image-971" height="198" src="http://www.webdesignbooth.com/wp-content/uploads/2009/07/jwysiwyg.jpg" title="jwysiwyg - jQuery WYSIWYG plugin" width="500"/>
       </p>
       <h3>
        5.
        <a href="http://developer.yahoo.com/yui/editor/" target="_blank" title="Yahoo! UI Library: Rich Text Editor">
         Yahoo! UI Library: Rich Text Editor
        </a>
       </h3>
       <p>
        The Rich Text Editor is part of the Yahoo YUI library and users can easily extend the features via creating plugin for it. It is very suitable for those who already familiar with other controls from YUI library.
        <br/>
        <img alt="Yahoo! UI Library: Rich Text Editor" class="alignnone size-full wp-image-972" height="331" src="http://www.webdesignbooth.com/wp-content/uploads/2009/07/yui-rich-text-editor.jpg" title="Yahoo! UI Library: Rich Text Editor" width="500"/>
       </p>
       <h3>
        6.
        <a href="http://xinha.webfactional.com/" target="_blank" title="Xinha">
         Xinha
        </a>
       </h3>
       <p>
        Xinha is a powerful WYSIWYG HTML editor component that works in all current browsers. It is built by the open source community, and everyone are welcome to contribute to the project.
        <br/>
        <img alt="xinha" class="alignnone size-full wp-image-973" height="330" src="http://www.webdesignbooth.com/wp-content/uploads/2009/07/xinha.jpg" title="xinha" width="500"/>
       </p>
       <h3>
        7.
        <a href="http://www.openwebware.com/" target="_blank" title="Openwysiwyg">
         Openwysiwyg
        </a>
       </h3>
       <p>
        Openwysiwyg is yet another powerful open source cross browser WYSIWYG editor that contains a lot of features. It enhances table creation and you can easily create tables and customize them with built-in properties.
        <br/>
        <img alt="openwysiwyg: Free cross browser wysiwyg editor" class="alignnone size-full wp-image-974" height="246" src="http://www.webdesignbooth.com/wp-content/uploads/2009/07/openwysiwyg.jpg" title="openwysiwyg: Free cross browser wysiwyg editor" width="500"/>
       </p>
       <h3>
        8.
        <a href="http://freerichtexteditor.com/" target="_blank" title="Free Rich Text Editor">
         Free Rich Text Editor
        </a>
       </h3>
       <p>
        Free Rich Text Editor is an extremely easy to use FREE javascript based HTML WYSIWYG editor for your website. It is easy to implement and manage to output XHTML compliant code.
        <br/>
        <img alt="Free Rich Text Editor" class="alignnone size-full wp-image-976" height="382" src="http://www.webdesignbooth.com/wp-content/uploads/2009/07/free-rich-text-editor.jpg" title="Free Rich Text Editor" width="500"/>
       </p>
       <h3>
        9.
        <a href="http://wmd-editor.com/" target="_blank" title="WMD: The Wysiwym Markdown Editor">
         WMD: The Wysiwym Markdown Editor
        </a>
       </h3>
       <p>
        WMD is a simple and lightweight HTML editor which can be used in blog comment section or forum posts.
        <br/>
        <img alt="WMD: The Wysiwym Markdown Editor" class="alignnone size-full wp-image-977" height="286" src="http://www.webdesignbooth.com/wp-content/uploads/2009/07/wmd.jpg" title="WMD: The Wysiwym Markdown Editor" width="500"/>
       </p>
       <h3>
        10.
        <a href="http://koivi.com/WYSIWYG-Editor/" target="_blank" title="TTW HTML Editor">
         TTW HTML Editor
        </a>
       </h3>
       <p>
        TTW HTML Editor is a very simple wysiwyg html editor that powered by javascript. Its spell checking feature is powered by SpellerPages, which can easily being implemented by adding parameter to the editor.
        <br/>
        <img alt="TTW HTML Editor" class="alignnone size-full wp-image-978" height="313" src="http://www.webdesignbooth.com/wp-content/uploads/2009/07/ttw-html-editor.jpg" title="TTW HTML Editor" width="500"/>
       </p>
       <h3>
        11.
        <a href="http://freetextbox.com/" target="_blank" title="Free Text Box">
         Free Text Box
        </a>
       </h3>
       <p>
        FreeTextBox is the most-used HTML editor for ASP.NET. It is compatible with IE on the PC, and Mozilla and Firefox on all platforms.
        <br/>
        <img alt="Free Text Box" class="alignnone size-full wp-image-979" height="378" src="http://www.webdesignbooth.com/wp-content/uploads/2009/07/free-text-box.jpg" title="Free Text Box" width="500"/>
       </p>
       <h3>
        12.
        <a href="http://www.wymeditor.org/" target="_blank" title="WYMeditor">
         WYMeditor
        </a>
       </h3>
       <p>
        WYMeditor is a web based XHTML editor. WYMeditor has been created to generate perfectly structured XHTML strict code, to conform to the W3C XHTML specifications and to facilitate further processing by modern applications.
        <br/>
        <img alt="WYMeditor" class="alignnone size-full wp-image-980" height="203" src="http://www.webdesignbooth.com/wp-content/uploads/2009/07/wymeditor.jpg" title="WYMeditor" width="500"/>
       </p>
       <h3>
        13.
        <a href="http://www.blueshoes.org/en/javascript/editor/" target="_blank" title="BlueShoes Wysiwyg Editor">
         BlueShoes Wysiwyg Editor
        </a>
       </h3>
       <p>
        It is a DHTML and Javascript based editor which has a lot of cool features. For example, user can easily resize the editor, select special characters, and pick color using the color picker tool.
        <br/>
        <img alt="BlueShoes Wysiwyg Editor" class="alignnone size-full wp-image-981" height="350" src="http://www.webdesignbooth.com/wp-content/uploads/2009/07/blueshoes-wysiwys-editor.jpg" title="BlueShoes Wysiwyg Editor" width="500"/>
       </p>
       <h3>
        14.
        <a href="http://markitup.jaysalvat.com/home/" target="_blank" title="markItUp">
         markItUp
        </a>
       </h3>
       <p>
        markItUp! is a JavaScript plugin built on the jQuery library. It is very lightweight and very customizable. You can even define your favorite keyboard shortcuts, which adds extra user friendly feature.
        <br/>
        <img alt="markItUp! Universal markup editor" class="alignnone size-full wp-image-982" height="341" src="http://www.webdesignbooth.com/wp-content/uploads/2009/07/mark-it-up.jpg" title="markItUp! Universal markup editor" width="500"/>
       </p>
       <h3>
        15.
        <a href="http://spaweditor.com/en/disp.php/en_products/en_spaw/en_spaw_intro" target="_blank" title="SPAW Editor">
         SPAW Editor
        </a>
       </h3>
       <p>
        It is a web based WYSIWYG editor which had tabbed multi-document interface, floating/shared toolbars and a very cool interface. It is available in PHP and .NET versions.
        <br/>
        <img alt="spaw-editor" class="alignnone size-full wp-image-983" height="418" src="http://www.webdesignbooth.com/wp-content/uploads/2009/07/spaw-editor.jpg" title="spaw-editor" width="500"/>
       </p>
