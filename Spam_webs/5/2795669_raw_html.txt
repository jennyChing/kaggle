  <p>
   <strong>
    EULAlyzer
   </strong>
   <br/>
   Wonder what youre agreeing to when you click
   <strong>
    I Accept
   </strong>
   ? Dont worry; just drag
   <a href="http://www.javacoolsoftware.com">
    EULAlyzer
   </a>
   s target icon onto the EULA for a quick report on any troublesome language.
  </p>
  <p>
   <strong>
    Hotspot Shield
   </strong>
   <br/>
   Create an instant virtual private network (VPN) tunnel between your laptop and the router at any Wi-Fi hot spot to protect your data from snoops on the public airwaves, even if you are using extra encryption. If youre on a network that limits the use of certain applications, such as Skype, the VPN could give you unfettered access. It even works with your wired connection for an extra layer of security.
   <br/>
   Web Site:
   <a href="http://www.anchorfree.com">
    www.anchorfree.com
   </a>
  </p>
  <p>
   <strong>
    Kruptos 2
   </strong>
   <br/>
   Kruptos makes a file accessible only to someone who knows the password; it includes a file shredder.
   <br/>
   Web Site:
   <a href="http://www.kruptos2.co.uk">
    www.kruptos2.co.uk
   </a>
  </p>
  <p>
   <strong>
    PC Flank
   </strong>
   <br/>
   While ShieldsUP! tests your firewalls ability to stealth all ports, PC Flank performs surgical strikes that emulate specific malware attacks to verify that your firewall blocks them.
   <br/>
   WebSite:
   <a href="http://www.pcflank.com">
    www.pcflank.com
   </a>
  </p>
  <p>
   <strong>
    SendShield
   </strong>
   <br/>
   This Outlook add-on strips out extra info and tracked changes in Word documents attached to messages.
   <br/>
   Website:
   <a href="http://www.sendshield.com">
    www.sendshield.com
   </a>
