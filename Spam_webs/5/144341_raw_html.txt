       <h1 class="headline" itemprop="name">
        Bash History: Display Date And Time For Each Command
       </h1>
       <div class="byline small">
        <span class="post_author_intro">
         by
        </span>
        <span class="post_author" itemprop="author">
         Vivek Gite
        </span>
        <meta content="2010-03-11" itemprop="datePublished"/>
        <meta content="2010-03-11" itemprop="dateModified"/>
        <span class="post_date_intro">
         on
        </span>
        <span class="post_date" title="2010-03-11">
         March 11, 2010
        </span>
       </div>
      </div>
      <div class="post_content" itemprop="articleBody">
       <div style="float:right;margin-top:0px;margin-left:5px;">
        <a href="http://www.cyberciti.biz/faq/category/bash-shell/" title="See all Bash/Shell scripting related FAQ">
         <img border="0" src="http://c.cyberciti.biz/cbzcache/3rdparty/terminal.png"/>
        </a>
       </div>
       <p>
        <span class="drop_cap">
         H
        </span>
        ow do I display shell command history with date and time under UNIX or Linux operating systems?
        <br/>
        <span id="more-6759">
        </span>
        <br/>
        If  the HISTTIMEFORMAT is set, the time stamp information associated with each history entry is written to the history file, marked with the history comment character.  Defining the environment variable as follows:
        <br/>
        <code>
         $ HISTTIMEFORMAT="%d/%m/%y %T "
        </code>
        <br/>
        OR
        <br/>
        <code>
         $ echo 'export HISTTIMEFORMAT="%d/%m/%y %T "' &gt;&gt; ~/.bash_profile
        </code>
        <br/>
        Where,
        <br/>
        %d - Day
        <br/>
        %m - Month
        <br/>
        %y - Year
        <br/>
        %T - Time
        <br/>
        To see history type
        <br/>
        <code>
         $ history
        </code>
        <br/>
        Sample outputs:
       </p>
       <pre>
        ....
..
  986  11/03/10 04:31:36 memcached-tool  10.10.28.22:11211 stats
  987  11/03/10 04:31:36 w
  988  11/03/10 04:31:37 iostat
  989  11/03/10 04:31:37 top
  990  11/03/10 04:31:37 at
  991  11/03/10 04:31:38 atop
  992  11/03/10 04:31:40 collectl
  993  11/03/10 04:31:41 grep CPU /proc/cpuinfo
  994  11/03/10 04:31:45 vmstat 3 100
  995  11/03/10 04:31:55 sar -W -f /var/log/sa/sa12
....
..
       </pre>
       <h4>
        References:
       </h4>
       <p>
        For more info type the following commands:
        <br/>
        <code>
         man bash
         <br/>
         help history
         <br/>
         man 3 strftime
        </code>
       </p>

            Just for anyone who might still find this thread (like I did today):
           </p>
           <p>
            export HISTTIMEFORMAT=%F %T : $(echo $SSH_CONNECTION | cut -d\  -f1) : 
           </p>

           </p>
