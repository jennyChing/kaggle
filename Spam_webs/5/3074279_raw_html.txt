          Privilege Escalation Vulnerability affects Billions of Android Devices
         </div>
         <div itemprop="articleBody">
          <p style="text-align: justify;">
           Newly found
           <strong>
            Privilege Escalation Vulnerability
           </strong>
           affects billions of
           <strong>
            android mobile phones
           </strong>
           and
           <strong>
            tablets
           </strong>
           , except the devices that are running latest Android 5.0 Lollipop. Privilege Escalation is the act of exploiting a bug, design flaw or configuration in an operating system to gain administrator or root access.  Wikipedia. Security Researcher named
           <strong>
            Jann Horn
           </strong>
           initially discovered this android security vulnerability that could allow any attacker to bypass Address Space Layout Randomization (ASLR) defense and execute arbitrary code on the target device.
          </p>
          <p style="text-align: justify;">
           <img alt="android-privilege-escalation" class="aligncenter wp-image-27766" height="448" src="http://www.dailytut.com/wp-content/uploads/2014/11/android-privilege-escalation.png" width="800"/>
          </p>
          <p style="text-align: justify;">
           Any android device running below
           <strong>
            android 5.0 lollipop
           </strong>
           operating system is vulnerable to this attack. Security bug found in java.io.ObjectInputStream is the main cause of this vulnerability, that fails to check whether an Object that is being deserialized is actually a serializable Object.
          </p>
          <p style="text-align: justify;">
           The Researcher also notified Google about this vulnerability earlier this year and the android team has fixed this vulnerability on Android 5.0 Lollipop.
          </p>
          <p style="text-align: justify;">
           Also Read,
           <a href="http://www.dailytut.com/apple/mspy-review-iphone-android-blackberry.html">
            mSpy Review and 15% mSpy Coupon Code Discount
           </a>
           , Best Spy App for Android and iPhone.
          </p>
          <h2 itemprop="alternativeHeadline" style="text-align: justify;">
           <span style="color: #ff6600;">
            Android Privilege Escalation Vulnerability
           </span>
          </h2>
          <p style="text-align: justify;">
           By default, android apps communicate with system_service, which runs under admin privileges. They transfer data as arraymaps which contain serialized data, where any android app can take control of the system_service by injecting a malicious code. Researcher Jann Horn has wrote about this vulnerability in
           <a class="external" href="http://seclists.org/fulldisclosure/2014/Nov/51" rel="nofollow" target="_blank">
            seclists
           </a>
           which gives complete detail about how this privilege escalation attack works on devices running earlier versions than android 5.0 lollipop.
          </p>
          <p style="text-align: justify;">
           Google released Lollipop source recently and mobile manufacturers already started cooking an update for existing flagship devices and for few more specifically in each brand. This naturally takes more than usual time to get updates to those devices.
           <a href="http://www.dailytut.com/answers/topic/htc-m8-running-android-lollipop-htc-sense-6-leaked-photos" target="_blank">
            HTC marked 90 days time for releasing android 5.0 lollipop update
           </a>
           and they also announced a list of devices that are eligible for the 90th day update. Developers are working day and night to release android lollipop update for devices and fixing bugs in all corners.
          </p>
          <p style="text-align: justify;">
           There are other compatibility problems currently faced by
           <strong>
            android 5.0 lollipop
           </strong>
           where users who have received upgrades on Motorola Moto series phones advising others not to upgrade immediately as they are experiencing applications being broken and crashed frequently. There are also reports that the new lollipop update slowdown users mobile devices and Google should work on fix in the next update to address all these stability and performance related issues.
          </p>
