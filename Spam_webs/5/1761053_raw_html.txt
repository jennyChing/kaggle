         <p>
          <span style="line-height: 1.6em;">
           Procurorul ef al DNA, Codrua Kovesi, a anunat, joi, n cadrul edinei de prezentare a bilanului pe 2014 al Poliiei Romne, c numrul poliitilor cu funcii de conducere cercetai de DNA a crescut anul trecut,ca urmare a relaiilor acestora cu mediul politic i cel de afaceri, informeaz
           <a href="http://www.mediafax.ro" target="_blank">
            Mediafax
           </a>
           .
          </span>
         </p>
         <p>
          <span style="line-height: 20.7999992370605px;">
           "Pot s spun c i DNA a avut procurori care au fost arestai pentru fapte de corupie, poliiti arestai pentru fapte de corupie, dar este este foarte important c aceste fapte au fost identificate i cred c se impune s analizm cu mare atenie cauzele care au dus la apariia acestui fenomen, dar i vulnerabilitile", a mai spus efa DNA.
          </span>
         </p>
         <p>
          Pe de alt parte, Kovesi a apreciat sprijinul pe care angajaii Poliiei Romne l-au acordat procurorilor DNA n dosarele n care au fost investigai poliiti suspectai de acte de corupie.
         </p>
         <p>
          "Sunt un reprezentat al unei instituii n care lucreaz190 poliiti. Aceti poliiti au un rol esenial n activitatea DNA ntruct lucreazdirect cu procurorii i n mod nemijlocit. n2014 am nregistrat peste 4.900 de dosare penale i azi dimineaaveam n lucru peste 5.100 de dosare penale care sunt repartizate unui numr de 86 de procurori. Avem n jur de 110 procurori n DNA, dar 25 dintre ei participn edinele de judecat. Deci vedei o ncrcturfoarte mare de dosare repartizate unui numr mic de procurori. De aceea, rolul pe care l au poliitii judiciari n cadrul DNA este extrem de important", a subliniat Kovesi.
         </p>
         <p>
          Codrua Kovesi a menionat, cu acelai prilej, c "una dintre vulnerabilitile identificate" a fost apropierea ofierilor care fac obiectul dosarelor penale cu diferite persoane din mediul politic, dar i din rndul oamenilor de afaceri investigai de poliiti.
         </p>
         <p>
          Procurorul ef al DNA a explicat c meseria de poliist nu presupune numai drepturi, ci i restricii, iar una dintre aceste restricii este pstrarea distanei fa de mediul politic, n aa fel nct poliistul "s fac ceea ce este corect i n interesul instituiei".
         </p>
         <p>
          Oficiali din DNA au precizat, pentru MEDIAFAX, c peste 80 de poliiti sunt cercetai n dosare deschise anul trecut de procurorii anticorupie.
         </p>

    },
    init : function() {
        var fn = this;
        
        $('.widget-article-footer-votes .star')        
        .on('mouseover',function() {
            fn.currVoteGrade = $(this).attr('data-vote-grade');
            
            $('[data-vote-grade]').each(function(){
                voteGrade = $(this).attr('data-vote-grade');
               
                if( voteGrade &lt;= fn.currVoteGrade ) {
                    $(this).find('a').addClass('red');
                    $(this).find('a').removeClass('gray');
                    
                }
            }) 
        })
        .on('mouseout', function(){
            $('[data-vote-grade]').find('a').removeClass('red');
            $('[data-vote-grade]').find('a').addClass('gray');
        })
        .on('click', function(){
            fn.vote($(this).attr('data-vote-grade'));
            fn.computeNewVoteGradeAndVoteNr($(this).attr('data-vote-grade'));
            
            $('.widget-article-footer-votes .star').unbind('mouseout');
            $('.widget-article-footer-votes .star').unbind('mouseover');
            $('.widget-article-footer-votes .star').unbind('click');
        })
    }
    
}

    var formData = $('#commentForm').serialize() + '&amp;validateAjax='+true;

    $.ajax({
        type: "POST",
        url: "/adauga-comentariu",
        data: formData,
        beforeSend: function( e ) {
        	$('#wrapperContainerCommentErrors').hide();
        }
    })
    .done(function( response ) {
     
        var data = $.parseJSON(response);
        
        if( data.errors ) {

                errMsg += '&lt;p&gt;'+data.errorsArr[k][0]+'&lt;/p&gt;';

            $('#wrapperContainerCommentErrors').html(errMsg);
            $('html, body').animate({
                scrollTop: $("#wrapperContainerCommentErrors").offset().top - 50
            }, 800);
        }
        else if( data.success ) {
            $('#commentForm').submit();
        }
            
    });

    return false;
