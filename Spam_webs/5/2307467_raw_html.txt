         Tutorial: Monogrammed Man Bags
        </h1>

        <p>
         <span style="font-size: 18pt;">
          <img alt="Sewing Tutorial: Monogrammed Man Bags on believeninspire.com" class="alignnone wp-image-4836 size-full" height="650" src="http://believeninspire.com/wp-content/uploads/2014/10/cg1.jpg" width="650"/>
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Cut out the following pieces of fabrics and mark them accordingly with a post-it note. (Refer to
          <a href="http://believeninspire.com/2014/08/cut-fabric-rotary-cutter.html" title="How to Cut Fabric WITH a Rotary Cutter">
           this post
          </a>
          to ensure how to cut straight fabrics using rotary cutter and
          <a href="http://believeninspire.com/2014/07/cut-fabrics-straight-without-rotary-cutter.html" title="How to Cut Fabrics Straight WITHOUT a Rotary Cutter">
           this post
          </a>
          for without rotary cutter).
         </span>
         <span id="more-4835">
         </span>
        </p>
        <ul>
         <li>
          <span style="font-size: 18pt;">
           A  12x15 body (Cut two from lining, cut two from exterior fabric)
          </span>
         </li>
         <li>
          <span style="font-size: 18pt;">
           B  11x15 flap (Cut one from lining and one from exterior)
          </span>
         </li>
         <li>
          <span style="font-size: 18pt;">
           C  10x15 pocket (Cut one from exterior)
          </span>
         </li>
         <li>
          <span style="font-size: 18pt;">
           D  4x37 gusset (Cut one from exterior and one from lining)
          </span>
         </li>
         <li>
          <span style="font-size: 18pt;">
           E  4x34 strap (Cut one from exterior)
          </span>
         </li>
         <li>
          <span style="font-size: 18pt;">
           F  2.5x2.5 strap extender (Cut 4 from exterior)
          </span>
         </li>
        </ul>
        <p>
         <span style="font-size: 18pt;">
          Cut pieces from hard fusible interfacing for A and D exterior.
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Cut pieces from soft fusible interfacing for A (lining), B D (lining) and E
         </span>
        </p>
        <div class="code-block code-block-1" style="text-align:left;margin:8px 0px;">
         <script type="text/javascript">
          &lt;!--

//--&gt;
         </script>
         <script src="//pagead2.googlesyndication.com/pagead/show_ads.js" type="text/javascript">
         </script>
        </div>
        <p>
         <span style="font-size: 18pt;">
          Print and cut out the pieces for applique (i.e. the monogram and the circle). In
          <a href="http://believeninspire.com/wp-content/uploads/2014/11/man-bag.docx" target="_blank">
           this word document
          </a>
          you will find the circle for right size. You will have to just add your letter to it, and then you can print it.
         </span>
        </p>
        <div class="code-block code-block-5" style="margin:8px 0px;">
         <script id="mNCC" language="javascript">
          medianet_width='600';  medianet_height= '250';  medianet_crid='510823735';
         </script>
         <script id="mNSC" language="javascript" src="http://contextual.media.net/nmedianet.js?cid=8CUBT8YR1">
         </script>
        </div>
        <p>
         <span style="font-size: 18pt;">
          <img alt="DSC_1117" class="alignnone size-full wp-image-4838" height="433" src="http://believeninspire.com/wp-content/uploads/2014/10/DSC_1117.jpg" width="650"/>
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Fuse them with double sided soft interfacing. Next, fuse the letter onto the circle. Using a zig zag stitch sew along the edges of the letter. Fuse the circle onto the exterior flap. Using a zig zag stitch sew along the edge of the circle.
         </span>
        </p>
        <div class="code-block code-block-3" style="text-align:center;margin-left:auto;margin-right:auto;margin-top:8px;margin-bottom:8px;">
         <script type="text/javascript">
          &lt;!--

/* Leaderboard */

//--&gt;
         </script>
         <script src="//pagead2.googlesyndication.com/pagead/show_ads.js" type="text/javascript">
         </script>
         <a href="http://www.shareasale.com/r.cfm?b=458153&amp;u=753312&amp;m=29190&amp;urllink=&amp;afftrack=" target="_blank">
          <img alt="Craftsy " border="0" src="http://static.shareasale.com/image/29190/BL_Valentine_300x250_FINAL.jpg"/>
         </a>
        </div>
        <p>
         <span style="font-size: 18pt;">
          <img alt="DSC_1118" class="alignnone size-full wp-image-4839" height="433" src="http://believeninspire.com/wp-content/uploads/2014/10/DSC_1118.jpg" width="650"/>
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Take one of your flap piece fold it into half so that the edges meet each other and cut a diagonal line as shown.
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          <img alt="DSC_1211" class="alignnone size-full wp-image-4845" height="650" src="http://believeninspire.com/wp-content/uploads/2014/10/DSC_1211.jpg" width="453"/>
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Open the fold and place the flap on top of the other flap and using the cut portion as a guide cut out the other flap too.
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          <img alt="DSC_1212" class="alignnone size-full wp-image-4846" height="522" src="http://believeninspire.com/wp-content/uploads/2014/10/DSC_1212.jpg" width="650"/>
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Take the lining flap and pin the Velcro in the middle and sew from all sides.
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          <img alt="DSC_1213" class="alignnone size-full wp-image-4847" height="433" src="http://believeninspire.com/wp-content/uploads/2014/10/DSC_1213.jpg" width="650"/>
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Take the two finished flaps; place them together so that the two right sides are facing each other. Sew from all sides except the top. Clip the corners and turn. Press and do a edge stitch followed by top stitch (read more about these
          <a href="http://believeninspire.com/2014/08/how-to-top-stitch-sewing-tips-for-beginners.html" title="How to Top stitch (Sewing Tips for Beginners)">
           here)
          </a>
          .
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          <img alt="DSC_1218" class="alignnone size-full wp-image-4850" height="433" src="http://believeninspire.com/wp-content/uploads/2014/10/DSC_1218.jpg" width="650"/>
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Take your organizer piece and fold it around 1 towards the inside, and sew along the folded edge.
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Now fold the frayed edge and sew.
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          <img alt="DSC_1119" class="alignnone size-full wp-image-4840" height="450" src="http://believeninspire.com/wp-content/uploads/2014/10/DSC_1119.jpg" width="650"/>
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Place the piece onto the b piece and sew lines as shown.
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          <img alt="DSC_1216" class="alignnone size-full wp-image-4849" height="476" src="http://believeninspire.com/wp-content/uploads/2014/10/DSC_1216.jpg" width="650"/>
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          For the flap, fuse the interfacing first then, fold it 1/2 from sides as shown and press.
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          <img alt="DSC_1124" class="alignnone size-full wp-image-4843" height="433" src="http://believeninspire.com/wp-content/uploads/2014/10/DSC_1124.jpg" width="650"/>
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Then fold the strap again from the center and press. Sew along the two longer edges making sure to keep the top open for the next step.
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          <img alt="DSC_1126" class="alignnone size-full wp-image-4844" height="433" src="http://believeninspire.com/wp-content/uploads/2014/10/DSC_1126.jpg" width="650"/>
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Take your x pieces, place the edges together and sew. Turn right side out. Press it so that the crease is in the middle as shown.
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          <img alt="DSC_1219" class="alignnone size-full wp-image-4851" height="423" src="http://believeninspire.com/wp-content/uploads/2014/10/DSC_12191.jpg" width="650"/>
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Insert the ring inside the piece fold it back onto itself and place it inside the strap as shown. Sew 2-3 lines to close the opening.
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          <img alt="DSC_1220" class="alignnone size-full wp-image-4852" height="437" src="http://believeninspire.com/wp-content/uploads/2014/10/DSC_1220.jpg" width="650"/>
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Next take the other x piece, insert it into the ring and attach it to the inner side of the bag using a pin and sew along the dotted line.
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          <img alt="DSC_1225" class="alignnone size-full wp-image-4853" height="579" src="http://believeninspire.com/wp-content/uploads/2014/10/DSC_1225.jpg" width="650"/>
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Set your completed exterior in front of you. Placethe flap against the back side of the bag.The Exterior Flap and Exterior Main Panel should beright sides facing with raw edges aligned, and the Flapshould be centered. Pin, then baste the flap in place.
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          <img alt="DSC_1227" class="alignnone size-full wp-image-4854" height="433" src="http://believeninspire.com/wp-content/uploads/2014/10/DSC_1227.jpg" width="650"/>
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Place the completed lining over the exterior sothat they are right sides together. Make surethat the Flap and Strap is tucked safely inside.Align the side seams, then pin in place. Sew along the
         </span>
         <br/>
         <span style="font-size: 18pt;">
          entire top edge of the bag.
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          Turn the bag right side out through the hole in the lining. Press. Topstitch or slipstitch the hole in the liningclosed.
         </span>
         <br/>
         <span style="font-size: 18pt;">
          Topstitch around the entire top opening that you sewed in previous step and youre DONE!
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          <img alt="Sewing Tutorial: Monogrammed Man Bags on believeninspire.com" class="alignnone wp-image-4855 size-full" height="931" src="http://believeninspire.com/wp-content/uploads/2014/10/DSC_1316.jpg" width="650"/>
         </span>
        </p>
        <p>
         <span style="font-size: 18pt;">
          This is a very basic version of a bag, though I loved it in the end but now when I look at the pictures I think I couldve done better by adding more details and design element. Hopefully my next man bag will be much cooler and satisfactory for me. XD
         </span>
        </p>

  function validate() {
    var isValid = true;
    
    for(var i = 0; i &lt; form.elements.length; ++i) {
      var input = form.elements[i],
          allDivs = input.parentNode.getElementsByTagName("div");

      if(input.className.indexOf("required") &gt;= 0) {
        if(input.id == "signup_email") {
          if(!email.test(input.value)) {
            emailErrorMessage(input, allDivs);
            isValid = false;
          } else {
            removeErrorMessage(input, allDivs);
          }
        } else {
          if((input.type == "checkbox" &amp;&amp; !input.checked) || input.value == "" || input.value == "-1") {
            fieldErrorMessage(input, allDivs);
            isValid = false;
          } else {
            removeErrorMessage(input, allDivs);
          }
        }
      }
    }
    
    form.className = isValid ? "" : "mimi_invalid";
    submit.value = isValid ? submit.getAttribute("data-default-text") : submit.getAttribute("data-invalid-text");
		submit.disabled = !isValid;
		submit.className = isValid ? "submit" : "disabled";
    
    return isValid;
  }

  function emailErrorMessage(input, allDivs) {
    if(input.getAttribute("data-webform-type") == "iframe") {
      input.className = "required invalid";
      input.placeholder = input.getAttribute("data-required-message") || "This field is required";
    } else {
      allDivs[0].innerHTML = input.getAttribute("data-invalid-message") || "This field is invalid";
    }
  }

  function fieldErrorMessage(input, allDivs) {
    if(input.getAttribute("data-webform-type") == "iframe") {
      input.className = "required invalid";
      input.placeholder = input.getAttribute("data-required-message") || "This field is required";
    } else {
      for(var i = 0; i &lt; allDivs.length; ++i) {
        var element = allDivs[i];
        if(element.className.indexOf("mimi_field_feedback") &gt;= 0) {
          return element.innerHTML = input.getAttribute("data-required-message") || "This field is required";
        }
      }
    }
  }

  function removeErrorMessage(input, allDivs) {
    if(input.getAttribute("data-webform-type") == "iframe") {
      input.className = "required";
    }

    for(var i = 0; i &lt; allDivs.length; ++i) {
      var element = allDivs[i];
      if(element.className.indexOf("mimi_field_feedback") &gt;= 0) {
        return element.innerHTML = "";
      }
    }
  }
})();
