            <h3 class="post-name">
             How to setup Windows 8.1 Mail App without using a Microsoft Account
            </h3>
            <div class="post-rating">
             <span class="ui-rate" data-configuration="ReadOnly=true" data-contentid="df2e6c74-1f92-4111-a672-b53c8ab3e70f" data-contenttypeid="f7d226ab-d59f-475c-9d22-4a79e3f0ec07" data-initialcount="0" data-initialvalue="0" data-readonly="true">
             </span>
            </div>
            <div class="post-author">
             <span class="avatar" style="height:32px;">
              <img alt="Paul Jones (MSFT)" border="0" src="http://i1.social.s-msft.com/profile/u/avatar.jpg?displayname=paul+jones+(msft)&amp;size=large" style="width: 32px;height:0;padding:32px 0 0 0;overflow:hidden;background:url('http://i1.social.s-msft.com/profile/u/avatar.jpg?displayname=paul+jones+(msft)&amp;size=large') center center no-repeat;background-size:cover;"/>

            <div class="post-content user-defined-markup">
             <p>
              With the release of Windows 8.1, it is no longer required to have a Microsoft Account (aka Live ID) to run Windows Mail Client. I will walk you through the steps (with Screenshots)on how to enable this new feature.
             </p>
             <p>
              When you first launch the Mail Client, it will prompt you to "Switch to a Microsoft account on this PC" if you did not sign in with a Microsoft Account - see screenshot below. Customers that use domain accounts and do not use Microsoft Accounts, are still able to use Windows Mail Client with Windows 8.1.
             </p>
             <p>
             </p>
             <p>
              <a href="http://blogs.technet.com/cfs-file.ashx/__key/communityserver-blogs-components-weblogfiles/00-00-00-72-08/5657.Mail_2D00_1.JPG">
               <img alt="" border="0" src="http://blogs.technet.com/resized-image.ashx/__size/550x0/__key/communityserver-blogs-components-weblogfiles/00-00-00-72-08/5657.Mail_2D00_1.JPG"/>
              </a>
             </p>
             <p>
             </p>
             <p>
              To Turn this Feature Off, you have to
              <strong>
               Enable
              </strong>
              the followingGroup Policy:
             </p>
             <p>
              Computer Configuration -&gt; Administrative Templates -&gt; Windows Components -&gt; App runtime -&gt; Allow Microsoft accounts to be optional
             </p>
             <p>
              This does not onlyapply to Windows Mail Client, but other Windows Store Apps thatsupport the feature and requireda Microsoft Account.
             </p>
             <p>
             </p>
             <p>
              <a href="http://blogs.technet.com/cfs-file.ashx/__key/communityserver-blogs-components-weblogfiles/00-00-00-72-08/2248.Mail_2D00_2.JPG">
               <img alt="" border="0" src="http://blogs.technet.com/resized-image.ashx/__size/550x0/__key/communityserver-blogs-components-weblogfiles/00-00-00-72-08/2248.Mail_2D00_2.JPG"/>
              </a>
             </p>
             <p>
             </p>
             <p>
              Once that policy is enabled, you will now be prompted and able to use an Enterprise Account instead - see screenshot below:
             </p>
             <p>
              <a href="http://blogs.technet.com/cfs-file.ashx/__key/communityserver-blogs-components-weblogfiles/00-00-00-72-08/6330.MAil_2D00_3.JPG">
               <img alt="" border="0" src="http://blogs.technet.com/resized-image.ashx/__size/550x0/__key/communityserver-blogs-components-weblogfiles/00-00-00-72-08/6330.MAil_2D00_3.JPG"/>
              </a>
             </p>
             <p>
             </p>
             <p>
              Ifyou do not publish your Email Server settings,Click on Show more details, then you are now able to enter your: Email Address, Server Address, Domain, Username and Password as show below.
             </p>
             <p>
             </p>
             <p>
              <a href="http://blogs.technet.com/cfs-file.ashx/__key/communityserver-blogs-components-weblogfiles/00-00-00-72-08/3225.Mail_2D00_4.JPG">
               <img alt="" border="0" src="http://blogs.technet.com/resized-image.ashx/__size/550x0/__key/communityserver-blogs-components-weblogfiles/00-00-00-72-08/3225.Mail_2D00_4.JPG"/>
              </a>
             </p>
             <p>
             </p>
             <p>
              I hope you find this Blog Post helpful when trying to setup the Windows Mail App without using a Microsoft Account.
             </p>
