      <h2 class="post-title">
       Supprimez les shellbags sous Windows pour plus de confidentialit
      </h2>
      <div class="post-thumbnail">
       <p class="ribbon">
        <a href="http://korben.info/shellbags.html#disqus_thread">
        </a>
       </p>
       <p class="image">
        <img alt="Supprimez les shellbags sous Windows pour plus de confidentialit" class="presentation size-full aligncenter" height="200" src="http://korben.info/wp-content/uploads/2014/06/shellbags.jpg" title="Supprimez les shellbags sous Windows pour plus de confidentialit" width="650"/>
       </p>
      </div>
      <div class="post-content">
       <div id="greet_block">
       </div>
       <p>
        Saviez vous qu'en jetant simplement un oeil dans votre base de registre
        <a href="http://korben.info/univers/windows">
         Windows
        </a>
        , n'importe qui peut y trouver la liste des dossiers et des fichiers rcemment accds ?
       </p>
       <p>
        Ces lments sont les
        <strong>
         ShellBags
        </strong>
        et ils sont utiles au systme puisque c'est ce qui permet d'enregistrer les paramtres d'affichage dossier par dossier. Mais c'est aussi un lment qui peut "trahir" vos activits sur votre PC. En effet, ces "
        <strong>
         shellbags
        </strong>
        " sont conservs avec la date de cration et la date d'accs aux dossiers et cela mme si le dossier a t supprim.
       </p>
       <p>
        Alors que faire ? Et bien il y a la mthode manuelle qui consiste  se rendre dans la base de registres puis  supprimer les cls suivantes :
       </p>
       <ul>
        <li>
         HKEY_CURRENT_USER\Software\Microsoft\Windows\Shell\Bags
        </li>
        <li>
         HKEY_CURRENT_USER\Software\Microsoft\Windows\Shell\BagMRU
        </li>
        <li>
         HKEY_CURRENT_USER\Software\Microsoft\Windows\ShellNoRoam\Bags
        </li>
        <li>
         HKEY_CURRENT_USER\Software\Microsoft\Windows\ShellNoRoam\BagMRU
        </li>
        <li>
         HKEY_CURRENT_USER\Software\Classes\Local Settings\Software\Microsoft\Windows\Shell\BagMRU
        </li>
        <li>
         HKEY_CURRENT_USER\Software\Classes\Local Settings\Software\Microsoft\Windows\Shell\Bags
        </li>
       </ul>
       <p>
        Et pour les Windows 64-bit, supprimez aussi :
       </p>
       <ul>
        <li>
         HKEY_CURRENT_USER\Software\Classes\Wow6432Node\Local Settings\Software\Microsoft\Windows\Shell\Bags
        </li>
        <li>
         HKEY_CURRENT_USER\Software\Classes\Wow6432Node\Local Settings\Software\Microsoft\Windows\Shell\BagMRU
        </li>
       </ul>
       <p>
        Puis  recrer manuellement les cls suivantes :
       </p>
       <ul>
        <li>
         HKEY_CURRENT_USER\Software\Classes\Local Settings\Software\Microsoft\Windows\Shell\BagMRU
        </li>
        <li>
         HKEY_CURRENT_USER\Software\Classes\Local Settings\Software\Microsoft\Windows\Shell\Bags
        </li>
       </ul>
       <p>
        Et pour les Windows 64-bit, crez aussi les cls :
       </p>
       <ul>
        <li>
         HKEY_CURRENT_USER\Software\Classes\Wow6432Node\Local Settings\Software\Microsoft\Windows\Shell\Bags
        </li>
        <li>
         HKEY_CURRENT_USER\Software\Classes\Wow6432Node\Local Settings\Software\Microsoft\Windows\Shell\BagMRU
        </li>
       </ul>
       <p>
        Voil, le mnage sera fait comme a.
       </p>
       <p>
        Mais si toucher  votre base de registre vous fait peur, sachez qu'il existe un excellent petit soft baptis "
        <a href="http://privazer.com/download-shellbag-analyzer-shellbag-cleaner.php">
         Shellbag Analyzer &amp; Cleaner
        </a>
        " qui va vous permettre de faire le mnage plus finement et de manire plus visuelle.
       </p>
       <p style="text-align: center;">
        <a href="http://korben.info/wp-content/uploads/2014/06/shellbag.png">
         <img alt="shellbag" class="aligncenter size-new-size wp-image-52836" height="577" src="http://korben.info/wp-content/uploads/2014/06/shellbag-650x577.png" title="" width="650"/>
        </a>
       </p>
       <p style="text-align: center;">
        <a href="http://korben.info/wp-content/uploads/2014/06/shellbag2.png" title="">
         <img alt="shellbag2" class="aligncenter size-new-size wp-image-52837" height="501" src="http://korben.info/wp-content/uploads/2014/06/shellbag2-650x501.png" title="" width="650"/>
        </a>
       </p>
       <p>
        Et pour ceux qui ne jurent que par la ligne de commande, je vous renvoie vers
        <a href="https://www.tzworks.net/prototype_page.php?proto_id=14">
         cet outil
        </a>
        pour la console qui permet de faire la mme chose.
