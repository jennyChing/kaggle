                 <h1>
                  Sysinternals Networking Utilities
                 </h1>
                 <p>
                  <a href="https://technet.microsoft.com/en-us/sysinternals/bb963907">
                   AD Explorer
                  </a>
                  <br/>
                  Active Directory Explorer is an advanced Active Directory (AD) viewer and editor.
                 </p>
                 <p>
                  <a href="https://technet.microsoft.com/en-us/sysinternals/bb897539">
                   AD Insight
                  </a>
                  <br/>
                  AD Insight is an LDAP (Light-weight Directory Access Protocol) real-time monitoring tool aimed at troubleshooting Active Directory client applications.
                 </p>
                 <p>
                  <a href="https://technet.microsoft.com/en-us/sysinternals/bb963906">
                   AdRestore
                  </a>
                  <br/>
                  Undelete Server 2003 Active Directory objects.
                 </p>
                 <p>
                  <a href="https://technet.microsoft.com/en-us/sysinternals/dd581625">
                   PipeList
                  </a>
                  <br/>
                  Displays the named pipes on your system, including the number of maximum instances and active instances for each pipe.
                 </p>
                 <p>
                  <a href="https://technet.microsoft.com/en-us/sysinternals/bb897552">
                   PsFile
                  </a>
                  <br/>
                  See what files are opened remotely.
                 </p>
                 <p>
                  <a href="https://technet.microsoft.com/en-us/sysinternals/jj729731">
                   PsPing
                  </a>
                  <br/>
                  Measures network performance.
                 </p>
                 <p>
                  <a href="https://technet.microsoft.com/en-us/sysinternals/bb896649">
                   PsTools
                  </a>
                  <br/>
                  The PsTools suite includes command-line utilities for listing the processes running on local or remote computers, running processes remotely, rebooting computers, dumping event logs, and more.
                 </p>
                 <p>
                  <a href="https://technet.microsoft.com/en-us/sysinternals/bb897442">
                   ShareEnum
                  </a>
                  <br/>
                  Scan file shares on your network and view their security settings to close security holes.
                 </p>
                 <p>
                  <a href="https://technet.microsoft.com/en-us/sysinternals/bb897437">
                   TCPView
                  </a>
                  <br/>
                  Active socket command-line viewer.
                 </p>
                 <p>
                  <a href="https://technet.microsoft.com/en-us/sysinternals/bb897435">
                   Whois
                  </a>
                  <br/>
                  See who owns an Internet address.
                  <br/>
                 </p>
