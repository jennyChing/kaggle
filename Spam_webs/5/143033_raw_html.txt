       <b>
        EASY PUNCH RECIPES - SLUSHY
       </b>
      </font>
      <br/>
      <br/>
      <b>
       <u>
        LEMON SLUSH PUNCH
       </u>
      </b>
      <br/>
      <br/>
      <ul>
       <li>
        Country Time lemonade mix
        <br/>
       </li>
       <li>
        Chilled liter of Sprite
        <br/>
       </li>
       <li>
        Snack-size Ziploc bags
        <br/>
       </li>
      </ul>
      <br/>
      To create slush, make lemonade according to package directions, pour into snack-sized Ziploc bags, and freeze. Break slush into pieces, put in punch bowl, and add Sprite.
      <br/>
      <br/>
      <b>
       <u>
        SLUSHY PARTY PUNCH
       </u>
      </b>
      <br/>
      <br/>
      <ul>
       <li>
        46-oz. chilled pineapple juice
        <br/>
       </li>
       <li>
        6-oz frozen orange juice concentrate, thawed
        <br/>
       </li>
       <li>
        Chilled liter of Sprite
        <br/>
       </li>
      </ul>
      <br/>
      Combine pineapple juice and orange juice concentrate and mix well. Gradually add Sprite. Cover and freeze overnight. Thaw for 2 hours before serving slushy punch.
      <br/>
      <br/>
      <br/>
      <font size="+1">
       <b>
        BIRTHDAY PARTY PUNCH DRINKS
       </b>
      </font>
      <br/>
      <br/>
      <b>
       <u>
        EASY PUNCH RECIPES FOR BIRTHDAY PARTIES
       </u>
      </b>
      <br/>
      <br/>
      <ul>
       <li>
        2 qts. Sunny Delight
        <br/>
       </li>
       <li>
        2 liters of Sprite
        <br/>
       </li>
       <li>
        2 1/3 oz. pkg. powdered Kool-Aid Tropical Punch mix
        <br/>
       </li>
      </ul>
      <br/>
      Empty Kool-Aid powder into large punch bowl. Pour in sunny Delight and Sprite. Stir until mixed. Chill. Serve over ice cubes or crushed ice.
      <br/>
      <br/>
      <b>
       <u>
        KOOL AID PUNCH
       </u>
      </b>
      <br/>
      <br/>
      <ul>
       <li>
        1 lrg. chilled pineapple juice
        <br/>
       </li>
       <li>
        1 pkg. strawberry Kool-Aid
        <br/>
       </li>
       <li>
        1 c sugar
        <br/>
       </li>
      </ul>
      <br/>
      Combine all ingredients in punch bowl. Place in freezer. Let ice crystals form. Serve.
      <br/>
      <br/>
      <b>
       <u>
        FRUIT PUNCH
       </u>
      </b>
      <br/>
      <br/>
      <ul>
       <li>
        48-oz. chilled orange juice
        <br/>
       </li>
       <li>
        48-oz. chilled pineapple juice
        <br/>
       </li>
       <li>
        3 - 12-oz. cans frozen lemonade
        <br/>
       </li>
       <li>
        4 qts. chilled water
        <br/>
       </li>
       <li>
        2 liters chilled ginger ale
        <br/>
       </li>
      </ul>
      <br/>
      Mix all juices and water in punch bowl. Add ginger ale and ice cubes when ready to serve.
      <br/>
      <br/>
      <b>
       <u>
        PARTY PUNCH
       </u>
      </b>
      <br/>
      <br/>
      <ul>
       <li>
        2 cups sugar
        <br/>
       </li>
       <li>
        2.5 cups boiling water
        <br/>
       </li>
       <li>
        48-oz. chilled orange juice
        <br/>
       </li>
       <li>
        48-oz. chilled pineapple juice
        <br/>
       </li>
       <li>
        48-oz. chilled grapefruit juice
        <br/>
       </li>
       <li>
        1 cup lemon juice
        <br/>
       </li>
       <li>
        2 qts. chilled Sprite
        <br/>
       </li>
      </ul>
      <br/>
      In punch bowl, dissolve sugar in water. Add juices. Refrigerate 2-4 hours. Add Sprite when ready to serve.
      <br/>
      <br/>
      <br/>
      <font size="+1">
       <b>
        EASY PUNCH RECIPES - FRUIT
       </b>
      </font>
      <br/>
      <br/>
      <b>
       <u>
        FRESH FRUIT PUNCH
       </u>
      </b>
      <br/>
      <br/>
      <ul>
       <li>
        64-oz chilled pineapple juice
        <br/>
       </li>
       <li>
        64-oz chilled orange juice
        <br/>
       </li>
       <li>
        64-oz chilled cranberry juice
        <br/>
       </li>
       <li>
        64-oz chilled lemonade
        <br/>
       </li>
       <li>
        Fresh pineapple cut-up or canned chunks
        <br/>
       </li>
       <li>
        Fresh strawberries with tops removed.
        <br/>
       </li>
      </ul>
      <br/>
      Put fruit in punch bowl. Add all juices. Serve.
      <br/>
      <br/>
      <b>
       <u>
        CRANBERRY LEMONADE PUNCH
       </u>
      </b>
      <br/>
      <br/>
      <ul>
       <li>
        32-oz. cranberry juice
        <br/>
       </li>
       <li>
        6-oz. frozen lemonade concentrate, thawed
        <br/>
       </li>
       <li>
        1 liter chilled club soda
        <br/>
       </li>
       <li>
        Fresh orange slices
        <br/>
       </li>
      </ul>
      <br/>
      Mix juice and lemonade in punch bowl. Add club soda, ice cubes, and orange slices when ready to serve.
      <br/>
      <br/>
      <b>
       <u>
        STRAWBERRY PUNCH
       </u>
      </b>
      <br/>
      <br/>
      <ul>
       <li>
        12-oz. can frozen strawberry juice concentrate
        <br/>
       </li>
       <li>
        46-oz. can pineapple juice
        <br/>
       </li>
       <li>
        1 liter ginger ale
        <br/>
       </li>
       <li>
        3 cups water
        <br/>
       </li>
      </ul>
      <br/>
      Combine the strawberry juice, pineapple jice, and water into a punch bowl. Chill. Just before serving, add the ginger ale. Serve over ice.
      <br/>
      <br/>
      <b>
       <u>
        HOLIDAY PUNCH
       </u>
      </b>
      <br/>
      <br/>
      <ul>
       <li>
        4 cups chilled cranberry juice
        <br/>
       </li>
       <li>
        1 bottle chilled sparkling cider
        <br/>
       </li>
       <li>
        1 liter chilled gingerale
        <br/>
       </li>
       <li>
        1/4 cup fresh lemon juice, about 2 lemons
        <br/>
       </li>
       <li>
        1 orange, sliced
        <br/>
       </li>
      </ul>
      <br/>
      Mix together all the liquid ingredients in a large punch bowl. Add ice, garnish with sliced oranges and serve.
      <br/>
      <br/>
      <br/>
      <font size="+1">
       <b>
        EASY PUNCH RECIPES - SHERBET
       </b>
      </font>
      <br/>
      <br/>
      <b>
       <u>
        LIME SHERBERT PUNCH
       </u>
      </b>
      <br/>
      <br/>
      <ul>
       <li>
        1 qt. lime sherbet, softened
        <br/>
       </li>
       <li>
        1/2 cup frozen limeade concentrate
        <br/>
       </li>
       <li>
        2 tbsp. sugar
        <br/>
       </li>
       <li>
        2 - 12 oz. cans chilled Lemon-Lime soda
        <br/>
       </li>
      </ul>
      <br/>
      Blend sherbet, limeade, and sugar in a punch bowl. Stir in soda and serve over ice.
      <br/>
      <br/>
      <b>
       <u>
        ORANGE SHERBET PUNCH
       </u>
      </b>
      <br/>
      <br/>
      <ul>
       <li>
        1 qt. apple juice
        <br/>
       </li>
       <li>
        1 qt. orange juice
        <br/>
       </li>
       <li>
        1 qt. pineapple juice
        <br/>
       </li>
       <li>
        2 qts. ginger ale
        <br/>
       </li>
       <li>
        Orange sherbet
        <br/>
       </li>
      </ul>
      <br/>
      Mix juices in punch bowl. Add ginger ale and sherbet when ready to serve.
      <br/>
      <br/>
      <b>
       <u>
        PINEAPPLE SHERBET PUNCH
       </u>
      </b>
      <br/>
      <br/>
      <ul>
       <li>
        46-oz. orange juice
        <br/>
       </li>
       <li>
        46-oz. pineapple juice
        <br/>
       </li>
       <li>
        36-oz. apricot nectar
        <br/>
       </li>
       <li>
        Pineapple sherbet
        <br/>
       </li>
       <li>
        3 qts. chilled Sprite
        <br/>
       </li>
      </ul>
      <br/>
      Mix juices in punch bowl. Add Sprite and sherbet when ready to serve.
      <br/>
      <br/>
      <b>
       <u>
        ORANGE SHERBET PUNCH
       </u>
      </b>
      <br/>
      <br/>
      <ul>
       <li>
        1 qt. fruit punch
        <br/>
       </li>
       <li>
        2.5 gal. chilled Sprite
        <br/>
       </li>
       <li>
        2.5 gal. orange sherbet
        <br/>
       </li>
      </ul>
      <br/>
      Scoop sherbet into punch bowl. Add Sprite. Add fruit punch. Serve immediately.
      <br/>
      <br/>
      <b>
       <u>
        RAINBOW SHERBET PUNCH
       </u>
      </b>
      <br/>
      <br/>
      <ul>
       <li>
        64-oz. fruit juice
        <br/>
       </li>
       <li>
        8-oz. orange juice concentrate, thawed
        <br/>
       </li>
       <li>
        8-oz. lemonade concentrate, thawed
        <br/>
       </li>
       <li>
        2 liters chilled Sprite
        <br/>
       </li>
       <li>
        1 pt. rainbow sherbet
        <br/>
       </li>
       <li>
        1 jar Maraschino cherries
        <br/>
       </li>
      </ul>
      <br/>
      Mix fruit juice, orange juice concentrate, and lemonade in punch bowl. Add Sprite, sherbet, and cherries when ready to serve.
