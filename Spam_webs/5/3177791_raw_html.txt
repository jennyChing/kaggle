       <div class="card-subtitle">
        Behind the scenes
       </div>
       Wysp runs on handcrafted artisan code of the finest quality, and it is brought to your screen using only locally-sourced pixels. It has been running without interruption since the initial writing of the current back-end in July 2013. Wysp was designed to be completely autonomous, and as such requires no administration or maintenance.
       <br/>
       <br/>
       Moderator interventions have been exceedingly rare, thanks to the excellent behavior of Wyspers. Since Wysp was launched, a total of two people have been banned.
       <br/>
       <br/>
       <div class="card-subtitle">
        History of Wysp
       </div>
       Wysp was originally launched on November 26, 2011 as a website challenging people to make one sketch a day as a way to push their skills to the next level. It provided a daily drawing prompt, and enabled members to share their sketches with the community and keep track of their progress.
       <br/>
       <br/>
       After attracting quite a bit of popular interest, it was entirely rewritten in March-April 2012 as a social network (Wysp 2). The Wysp learning engine was released a few weeks later, and has been widly successful since, recording over one million pageviews throughout 2012.
       <br/>
       <br/>
       During 2012 and early 2013, the learning engine was the main attraction of Wysp. Then, in July 2013, the server code was rewritten from scratch and new features were added, making the website much faster and more modern, as well as giving it social stickiness: Wysp 3. A few days after the relaunch, the new social network went viral on Tumblr and added over nine thousand users in a matter of days.
       <br/>
       <br/>
       Since then, Wysp has been a lively art community with thousands of regulars. In July 2014, it underwent a complete redesign to become Wysp 4, which you are now using.

		function setNotifBubble(bubble, notifCount) {
			if (notifCount &gt; 0){
				bubble.html(notifCount);
				st = notifCount + '';
				bubble.css('width', (10+st.length*6) + 'px');
				bubble.show();
			}
			else {

			}

		/* Search bars */
		$('.search-btn').on('click', function(){
			searchInput = $(this).prev();
			q = searchInput.val();
			document.location.href = '/search/?q=' + encodeURIComponent(q);
		});

			
		/* Follow */
		$('.followLink').live('click', function(){
			var button = $(this);
			if (button.attr('busy') == 'true') return;
			button.attr('busy', 'true');
			button.html('');
			if (button.attr('status') == "not_following"){
				$.post('/user/follow/?uid=' + button.attr('utarget'), function(data){
					button.html('Following!');
					button.removeClass('btn-primary');
					button.removeClass('btn-danger');
					button.removeClass('btn-default');
					button.addClass('btn-success');
					button.attr('status', 'following');
					button.attr('busy', 'false');
				}).error(function(){
					alert('Unable to contact server, please try again later.');
				});
			}
			else {
				$.post('/user/follow/?uid=' + button.attr('utarget'), function(data){
					button.html('Unfollowed!');
					button.removeClass('btn-danger');
					button.removeClass('btn-success');
					button.removeClass('btn-primary');
					button.addClass('btn-default');
					button.attr('status', 'not_following');
					button.attr('busy', 'false');
				}).error(function(){
					alert('Unable to contact server, please try again later.');
				});
			}
		});

		/* Theater */
		function launchTheater(newLocation){
			/*
			 * Launches the theater. After launch, you must fill the div #theaterFront with your content.
			 */
			$("#backToTop").hide();
			var BG = document.getElementById("theaterBG");
			
			$('#theaterBG').attr('pastLocation', window.location.pathname);
			if (newLocation != ''){
				window.history.pushState("url update", newLocation, newLocation);
			}
			
			$(BG).attr('locked', 0);
			
			document.body.style.overflow="hidden";
			$(BG).css("overflow-y", "scroll");
			$(BG).css("overflow-x", "hidden");
			$('#theaterBG').html('&lt;div id="exitTheater" onclick="exitTheater();"&gt;&lt;/div&gt;&lt;div id="theaterFront"&gt;&lt;/div&gt;');
			$("#theaterFront").css("display", "none");
			$("#theaterBG").fadeIn(100);
			$("#theaterFront").fadeIn(100);
		}
		
		function exitTheater(){
			window.history.pushState("Past location url update", "Wysp", $('#theaterBG').attr('pastLocation'));
			$("#theaterBG").fadeOut(100, function(){$("#theaterFront").remove();});
			document.body.style.overflow="visible";
			var btt = $("#backToTop");
			if (btt.attr('state') == 'on') btt.fadeIn();
		}
		
		$('#theaterBG').live('click', function(event) {
			  BG = document.getElementById("theaterBG");
			  if ($(BG).attr('locked') == 1) return;
			  if (event.target == BG) {
			    exitTheater();
			  }
		});
		
		$(document).keyup(function(e){
            if(e.keyCode == 27){
                exitTheater();
            }
        });
		
		function lockTheater(){
			$('#theaterBG').attr('locked', 1);
		}
		function unlockTheater(){
			$('#theaterBG').attr('locked', 0);
		}
		function centerTheater(thH){
			theater = $("#theaterFront");
			theater.css('position', 'absolute');
			theater.offset({top:$(window).scrollTop() + 20, left:($(window).width()-theater.width())*0.5});
		}
