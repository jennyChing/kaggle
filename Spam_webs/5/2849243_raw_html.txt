        &lt;p&gt;&lt;font face="Arial, Helvetica, sans-serif" size="-1"&gt;&lt;br&gt;
          Create a white letter on a blank background in Photoshop, using your 
          project's resolution (720 x 480 pixels). Make the background black and 
          save the picture as a working copy. Duplicate the text layer, turn off 
          the upper text layer, select the background layer and choose &lt;b&gt;Layer 
          / Merge Visible&lt;/b&gt;. Turn off the background layer and turn on the text 
          layer. Now you have the mask in the lower layer and the text in the 
          upper layer. Add layer effects to the text such as shadow and outer 
          glow.&lt;/font&gt;&lt;/p&gt;
        &lt;p&gt;&lt;font face="Arial, Helvetica, sans-serif" size="-1"&gt;&lt;i&gt;See the result 
          of the first steps.&nbsp;&lt;/i&gt;&lt;/font&gt;&lt;/p&gt;
        &lt;p&gt;&lt;font face="Arial, Helvetica, sans-serif" size="-1"&gt;&lt;i&gt;* You`ll find 
          the pictures in the &lt;b&gt;Workshops&lt;/b&gt; folder on the &lt;b&gt;CGM DVE &lt;/b&gt; CD. 
          If you don`t have the CD, all the pictures used in the workshops can 
          be found in the download area of our website (&lt;a href="http://www.cgm-online.de"&gt;www.cgm-online.com&lt;/a&gt;), 
          contained in a single file &lt;b&gt;&quot;Example Graphics (used in the workshops)&quot;&lt;/b&gt;.&lt;/i&gt;&lt;/font&gt;&lt;/p&gt;

      &lt;td&gt; 
        &lt;p&gt;&lt;font face="Arial, Helvetica, sans-serif" size="-1"&gt;&lt;br&gt;
          Select the text layer. From the pull-down menu choose &lt;b&gt;Layer / Effects 
          / Create Layer&lt;/b&gt;, select one of the new layers and make: &lt;b&gt;Layer 
          / Merge Visible&lt;/b&gt;.&lt;/font&gt;&lt;/p&gt;
        &lt;p&gt;&lt;font face="Arial, Helvetica, sans-serif" size="-1"&gt;&lt;i&gt;See the new 
          layers and the text layer.&lt;/i&gt;&lt;/font&gt;&lt;/p&gt;
      &lt;/td&gt;
      &lt;td&gt;&lt;img SRC="bild_3_e.gif" NOSAVE height=362 width=435&gt;&lt;/td&gt;
    &lt;/tr&gt;
    &lt;tr VALIGN=TOP&gt; 
      &lt;td&gt; 
        &lt;p&gt;&lt;font face="Arial, Helvetica, sans-serif" size="-1"&gt;&lt;br&gt;
          Create a selection by clicking the white letter (therefore use the &lt;b&gt;Magic 
          Wand Tool&lt;/b&gt;, Tolerance: 2, Anti-Aliased: ON, Contiguous: ON, Use All 
          Layers: OFF). Turn off the lower layer, select the upper layer and delete 
          the white color by using the &lt;b&gt;Eraser Tool&lt;/b&gt;. Save the results to 
          disk. Load the previously saved working-copy and modify the letter as 
          described without shifting it vertically. Repeat the entire process 
          for each new letter.&lt;/font&gt;&lt;/p&gt;
        &lt;p&gt;&lt;font face="Arial, Helvetica, sans-serif" size="-1"&gt;&lt;i&gt;Tip: Use copy and paste to recreate layer effects (shadow, glow etc).&nbsp;&lt;/i&gt;&lt;/font&gt;&lt;/p&gt;
      &lt;/td&gt;
      &lt;td&gt;&lt;img SRC="bild_4_e.gif" NOSAVE height=362 width=435&gt;&lt;/td&gt;
    &lt;/tr&gt;
    &lt;tr VALIGN=TOP&gt; 
      &lt;td&gt;
        &lt;p&gt;&lt;font face="Arial, Helvetica, sans-serif" size="-1"&gt;Drag the first 
          letter into the timeline and double-click. Lengthen both scenes up to 
          4 seconds. Select V1 and add a new track. Place the video scene you 
          wish as the pattern into the new V2 and modify its composite mode: &lt;b&gt;Modify 
          / Composite Mode / Travel Matte-Luma&lt;/b&gt;.&lt;/font&gt;&lt;/p&gt;
        &lt;p&gt;&lt;font face="Arial, Helvetica, sans-serif" size="-1"&gt;&lt;i&gt;Do not forget 
          to turn on the V1.&lt;/i&gt;&lt;/font&gt;&lt;/p&gt;
      &lt;/td&gt;
      &lt;td&gt;&lt;img SRC="bild_5.gif" NOSAVE height=166 width=435&gt;&lt;/td&gt;
    &lt;/tr&gt;
    &lt;tr VALIGN=TOP&gt; 
      &lt;td&gt;
        &lt;p&gt;&lt;font face="Arial, Helvetica, sans-serif" size="-1"&gt;Drag the rest of your letters onto the timeline in new tracks and repeat 
          these changes for each letter separately.&lt;/font&gt;&lt;/p&gt;
        &lt;p&gt;&lt;font face="Arial, Helvetica, sans-serif" size="-1"&gt;To set the first 
          letter's movement open its viewer using the context menu (keyboard ctrl-key 
          &amp; click). Define two keyframes on the &lt;b&gt;Center&lt;/b&gt; button: 720 
          for the first frame and -720 at the end. Now the letter moves from right 
          to left across the video. Copy the scene. Hold down the ctrl-key while 
          you select the next letter, activate &lt;b&gt;Paste Attributes&lt;/b&gt; &amp; cross 
          the box for &lt;b&gt;Basic Motion&lt;/b&gt; only. Repeat this for all the letters. 
          You can modify the distance between the letters by moving them in the 
          timeline.&lt;/font&gt;&lt;/p&gt;
        &lt;p&gt;&lt;font face="Arial, Helvetica, sans-serif" size="-1"&gt;&lt;i&gt;Notice: Since 
          version 3 Final Cut Pro has a new behavior in using alpha channels. 
          This is the reason for the dark edges outside the outline on the letters. 
          If you put all layers directly into the main timeline the problem is 
          gone.&lt;/i&gt;&lt;/font&gt;&lt;/p&gt;
