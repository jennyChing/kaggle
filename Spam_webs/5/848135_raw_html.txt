   <h1 class="firstHeading" id="firstHeading" lang="en">
    Breeder (cellular automaton)
   </h1>
   <div class="mw-body-content" id="bodyContent">
    <div id="siteSub">
     From Wikipedia, the free encyclopedia
    </div>
    <div id="contentSub">
     <span class="mw-redirectedfrom">

     </span>

       <a class="image" href="/wiki/File:Conways_game_of_life_breeder_animation.gif">
        <img alt="" class="thumbimage" data-file-height="192" data-file-width="379" height="192" src="//upload.wikimedia.org/wikipedia/commons/e/e6/Conways_game_of_life_breeder_animation.gif" width="379"/>

       <div class="thumbcaption">

        Evolution of an MSM breeder  a
        <a href="/wiki/Puffer_train" title="Puffer train">
         puffer
        </a>
        that produces
        <a class="mw-redirect" href="/wiki/Gosper_gun" title="Gosper gun">
         Gosper guns
        </a>

        <a href="/wiki/Glider_(Conway%27s_Life)" title="Glider (Conway's Life)">

     </div>
     <p>
      In
      <a href="/wiki/Cellular_automaton" title="Cellular automaton">
       cellular automata
      </a>
      such as
      <a href="/wiki/Conway%27s_Game_of_Life" title="Conway's Game of Life">
       Conway's Game of Life
      </a>
      , a
      <b>
       breeder
      </b>
      is a pattern that exhibits
      <a href="/wiki/Quadratic_growth" title="Quadratic growth">
       quadratic growth
      </a>
      , by generating multiple copies of a secondary pattern, each of which then generates multiple copies of a tertiary pattern.
      <sup class="reference" id="cite_ref-weisstein_1-0">
       <a href="#cite_note-weisstein-1">
        <span>
         [
        </span>
        1
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <h2>
      <span class="mw-headline" id="Classification">
       Classification
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Breeder_(cellular_automaton)&amp;action=edit&amp;section=1" title="Edit section: Classification">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      Breeders can be classed by the relative motion of the patterns. The classes are denoted by three-letter codes, which denote whether the primary, secondary and tertiary elements respectively are moving (M) or stationary (S). The four basic types are:
      <sup class="reference" id="cite_ref-weisstein_1-1">
       <a href="#cite_note-weisstein-1">
        <span>
         [
        </span>
        1
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <ul>
      <li>
       SMM  A
       <a href="/wiki/Gun_(cellular_automaton)" title="Gun (cellular automaton)">
        gun
       </a>
       that fires out
       <a href="/wiki/Rake_(cellular_automaton)" title="Rake (cellular automaton)">
        rakes
       </a>
       .
      </li>
      <li>
       MSM  A
       <a href="/wiki/Puffer_train" title="Puffer train">
        puffer
       </a>
       that leaves
       <a href="/wiki/Gun_(cellular_automaton)" title="Gun (cellular automaton)">
        guns
       </a>
       in its wake.
      </li>
      <li>
       MMS  A
       <a href="/wiki/Rake_(cellular_automaton)" title="Rake (cellular automaton)">
        rake
       </a>
       that fires out
       <a class="mw-redirect" href="/wiki/Puffer_train_(cellular_automaton)" title="Puffer train (cellular automaton)">
        puffers
       </a>
       .
      </li>
      <li>
       MMM  A
       <a href="/wiki/Rake_(cellular_automaton)" title="Rake (cellular automaton)">
        rake
       </a>
       that fires out more
       <a href="/wiki/Rake_(cellular_automaton)" title="Rake (cellular automaton)">
        rakes
       </a>
       , such that there are no stationary elements.
      </li>
     </ul>
     <p>
      A
      <a href="/wiki/Spacefiller" title="Spacefiller">
       spacefiller
      </a>
      (which also undergoes quadratic growth) may be thought of as a fifth class of breeder. However it differs from a true breeder in that it expands a single island of cells, rather than creating independent objects.
     </p>
