            <h2>
             Introduction
            </h2>
            <p>
             In my search for a program to stop and start Web sites programmatically, I saw examples of handling IIS in VB Script, but not in .NET. I also found examples of WMI in C# and VB.NET, but not for working with IIS. I decided to put the two ideas together and came up with this example. I did borrow the WMI ideas in this code from another article I read. I wish I could remember which one, so I could reference it.
            </p>
            <p>
             This is the second incarnation of this article. Since the first, I have cleaned up the code and added some comments. I have also enhanced the code to support monitoring and controlling remote systems (depending on access rights).
            </p>
            <h2>
             Using the Code
            </h2>
            <p>
             I will elaborate on two sections of the code. For complete details, please refer to the source code provided.
            </p>
            <p>
             Below is an excerpt from the method which returns a list of the Web sites which exist on the selected server. The
             <code>
              getDirectoryEntry
             </code>
             method uses an alternate user id and password if provided. Using ActiveDirectory, we enumerate the children of IIS, collecting the Web site names (assumed to be stored in the "
             <code>
              ServerComment
             </code>
             " property) for display in the Web sites combo box.
            </p>
            <pre lang="cs">
             List&lt;
             <span class="code-keyword">
              string
             </span>
             &gt; siteNames =
             <span class="code-keyword">
              new
             </span>
             List&lt;
             <span class="code-keyword">
              string
             </span>
             &gt;();

DirectoryEntry root = getDirectoryEntry(
             <span class="code-string">
              "
             </span>
             <span class="code-string">
              IIS://"
             </span>
             + txtServer.Text +
             <span class="code-string">
              "
             </span>
             <span class="code-string">
              /W3SVC"
             </span>
             );
             <span class="code-keyword">
              foreach
             </span>
             (DirectoryEntry e
             <span class="code-keyword">
              in
             </span>
             root.Children)
{
             <span class="code-keyword">
              if
             </span>
             (e.SchemaClassName ==
             <span class="code-string">
              "
             </span>

                         I suppose it depends how you define shared hosting. Certainly if you have multiple virtual directories under one website then I agree, all of the virtuals would stop with the website. But if that's the case then you should set up individual application pools, one for each virtual directory. Then stop and start the app pool instead. That code is very similar. Something like this:
                         <br/>
                         <br/>
                         <pre>
                          DirectoryEntry pool = null;
                          <span class="code-comment">
                           //
                          </span>
                          <span class="code-comment">
                           Get the Application Pools DirectoryEntry
                          </span>
                          DirectoryEntry de =
                          <span class="code-keyword">
                           new
                          </span>
                          DirectoryEntry(
                          <span class="code-string">
                           "
                          </span>
                          <span class="code-string">
                           IIS://localhost/W3SVC/AppPools"
                          </span>
                          );
                          <span class="code-comment">
                           //
                          </span>
                          <span class="code-comment">
                           Find the one you want by name
                          </span>
                          foreach (DirectoryEntry p
                          <span class="code-keyword">
                           in
                          </span>
                          de.Children)
{
                          <span class="code-keyword">
                           if
                          </span>
                          (pool.Name ==
                          <span class="code-string">
                           "
                          </span>
                          <span class="code-string">
                           YourApplicationPoolName"
                          </span>
                          )
	{
		pool = p;
                          <span class="code-keyword">
                           break
                          </span>
                          ;
	}
}
                          <span class="code-keyword">
                           if
                          </span>
                          (pool != null)
{
                          <span class="code-comment">
                           //
                          </span>
                          <span class="code-comment">
                           Set the pool command
                          </span>
                          PropertyCollection appPoolProps = pool.Properties;
	appPoolProps[
                          <span class="code-string">
                           "
                          </span>
                          <span class="code-string">
                           AppPoolCommand"
                          </span>
                          ].Value =
                          <span class="code-digit">
                           1
                          </span>
                          ;
                          <span class="code-comment">
                           //
                          </span>
                          <span class="code-comment">
                           1=start, 2=stop, 3=recycle
                          </span>
                          pool.CommitChanges();
}
                         </pre>
                         <br/>
