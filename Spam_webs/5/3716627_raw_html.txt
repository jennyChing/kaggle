       The Invisible Web Revealed
      </h2>

       <h2>
        <strong>
         Is there an invisible web?
        </strong>
       </h2>
       <p class="MsoNormal">
        Did you ever think how much time and effort was required for a web page to be the first when you perform a certain search? When you need information and you seek for it with search engines such as
        <a href="http://www.google.com" title="Google">
         Google
        </a>
        ,
        <a href="http://www.yahoo.com" title="Yahoo">
         Yahoo
        </a>
        ,
        <a href="http://www.msn.com" title="MSN">
         MSN
        </a>
        or whatever other, the results are brought to you through an automatic process of matching your search phrase or keyword with sites containing it. But in order for this to happen, those sites need to be visited by robots and indexed beforehand. In order to help robots correctly indexing a site, I, the webmaster, need to put some info there especially for those programs, so called robots. If I dont to it properly, you wont reach my site, regardless how useful it might be to you in that situation. Or if all my relevant info is published as photos, the search engines wont find my site for you, because robots cant read photographs, they know only how to handle text information. Nor robots know how to read pdf files. Nor databases. All info you cannot reach by classic searches is the so called invisible web.
       </p>
       <p class="MsoNormal">
        <strong>
         <br/>
        </strong>
       </p>
       <h2>
        <strong>
         How to see the invisible web?
        </strong>
       </h2>
       <p class="MsoNormal">
        You might want to ask now: so, if there is an invisible web, how can we reach it, though? Besides
        <a href="http://library.rider.edu/scholarly/rlackie/Invisible/Inv_Web.html" title="Invisible web">
         portals and directories, invisible or deep web, and specialized search engines
        </a>
        ,
        <a href="http://en.wikipedia.org/wiki/Social_bookmarking" title="Social bookmarking">
         social bookmarking
        </a>
        sites must be mentioned here.
       </p>
       <p class="MsoNormal">
        <strong>
         Invisible web resources
        </strong>
       </p>
       <p class="MsoNormal">
        There are many resources available for deep invisible web exploration. For the curious ones, here there are a few of the best ones:
       </p>
       <p class="MsoNormal">
        In the article
        <a href="http://www.lifehacker.com/software/search-engines/special-seek-and-ye-shall-find-128317.php" title="How to search the invisible web">
         How to Search the Invisible Web
        </a>
        , Lifehacker teaches us the basics of a better targeted deep web search.
       </p>
       <p class="MsoNormal">
        If you want to know more on specific topics, you can follow the
        <a href="http://www.free-ed.net/free-ed/" title="free ed online courses">
         online courses
        </a>
        from free-ed. All courses are free and very easy to subscribe to.
       </p>
       <p class="MsoNormal">
        The Berkeley University has a
        <a href="http://www.lib.berkeley.edu/TeachingLib/Guides/Internet/InvisibleWeb.html" title="invisible web guide">
         comprehensive guide of invisible web
        </a>
        : what it is, how to find it and its inherent ambiguity.
       </p>
       <p class="MsoNormal">
        At the Web Lens you can find a lot of
        <a href="http://www.weblens.org/invisible.html" title="invisible web research tools">
         invisible web research tools
        </a>
        which students and researchers might find useful.
       </p>
       <p class="MsoNormal">
        <a href="http://www.ericdigests.org/2002-2/hidden.htm" title="Eric Digests">
         Eric Digests
        </a>
        present some interesting facts and references for further reading. The article is rather old, but still good.
       </p>
       <p class="MsoNormal">
        A thorough, illustrated
        <a href="http://www.press.umich.edu/jep/07-01/bergman.html" title="web search process illustrated">
         description of the web search process
        </a>
        can be found at JEP (The Journal of Electronic Publishing published by the University of Michigan Press).
       </p>
       <p class="MsoNormal">
        This being said, remember to go for the deep web whenever you get frustrated by the surface web search results.
       </p>
