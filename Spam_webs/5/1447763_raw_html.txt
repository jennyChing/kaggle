            <p class="first-text" data-textannotation-id="0d22c1a4ddc55d4fb1cabd70f9e065df">
             <br/>
             <br/>
             <span class="customObject framed item_0">
              <a class="noHrefOverride" href="index.php?op=showcustomobject&amp;postId=285489&amp;item=0" rel="lyteframe" rev="width: 75px; height: 102px;" target="_blank">
               Click to view
              </a>
             </span>
             Despite the fact that BitTorrent has been around for a good 6 years now, the lightning fast
             <a class="autolink" href="http://lifehacker.com/tag/file-sharing/" title="Click here to read more posts tagged FILE SHARING">
              file sharing
             </a>
             protocol hasn't completely taken off in the mainstream. Since we post
             <a href="http://lifehacker.com/software/bittorrent/">
              a decent amount about BitTorrent
             </a>
             around here, we figured it was just time we put out a beginner's guide to BitTorrent. This is the guide you can send to your friend next time he gets that glassy look in his eyes when you mention BitTorrent and how quick and easy it makes downloading
             <span style="text-decoration: line-through;">
              albums
             </span>
             educational, public domain videos and other large files.
            </p>
            <p data-textannotation-id="026c85e7d9ab3f137f7d13a05b69437c">
             Without going into too much detail, here's a crash course in the file sharing protocol that is BitTorrent (feel free to skip to the
             <a href="#1" target="_blank">
              How to find and download a file with BitTorrent
             </a>
             section if you're not all that interested in the details).
            </p>
            <h3 data-textannotation-id="5760b67f2c9e58f44bcc514ce68f4599">
             <a id="b2" name="b2" target="_blank">
             </a>
             What is it
            </h3>
            <p data-textannotation-id="8916d82f3161306ce4701f7e23fc6922">
             BitTorrent is not a program.
             <a href="#2" target="_blank">
              [1]
             </a>
             It's a method of downloading files using a distributed peer-to-peer file sharing system. The programs that you use to download files via the BitTorrent protocol are called BitTorrent clients.
            </p>
            <div class="ad-unit js_ad-unit hide-contents ad-mobile js_ad-mobile">
             <div class="ad-mobile-inner">
              <p class="ad-label proxima">
               <small class=" proxima">
                Advertisement
               </small>
              </p>
              <div class="ad-container js_ad-container" data-zone-type="MOBILE" id="ad-container-53508420">
              </div>
              <p class="ad-label-bottom">
              </p>
             </div>
            </div>
            <p data-textannotation-id="a8e26a023506bfee167bd5029b37cc48">
             BitTorrent is not like Limewire/Kazaa/Napster/other P2P programs you've used in the past. This is often the biggest source of confusion for people new to BitTorrent. It's not difficult to use, it's just different. As soon as you forget about your old file-sharing program (and you
             <em>
              will
             </em>
             once you start using BT), the easier it will be to start using BitTorrent.
            </p>
            <h3 data-textannotation-id="d441a7db0258f96404911c9762b2f2a7">
             How does it work
            </h3>
            <p class="has-media media-300 has-image" data-textannotation-id="0463c0bfce0937090f5f275d318728de">
             <span class="lightBoxWrapper">
              <span class="img-border">
               <img alt="A beginner's guide to BitTorrent" data-asset-url="http://i.kinja-img.com/gawker-media/image/upload/s--XD5BpD_R--/18fcgocti31v6png.png" data-chomp-id="18fcgocti31v6png" data-format="png" height="202" src="http://i.kinja-img.com/gawker-media/image/upload/s--XD5BpD_R--/18fcgocti31v6png.png" width="215"/>
              </span>
              <span class="magnifier lightBox hide">
               <span class="icon icon-white icon-search-plus">
               </span>
              </span>
             </span>
            </p>
            <p data-textannotation-id="144aa84576d52023b1f53161211aa058">
             What makes the BitTorrent protocol unique is that it distributes the sharing of files across all users who have downloaded or are
             <em>
              in the process
             </em>
             of downloading a file. Because BitTorrent breaks up and distributes files in hundreds of small chunks, you don't even need to have downloaded the whole file before you start sharing. As soon as you have even a piece of the file, you can start sharing that piece with other users. That's what makes BitTorrent so fast; your BitTorrent client starts sharing as soon as it downloads one chunk of the file (instead of waiting until the entire download has been completed).
            </p>
            <p data-textannotation-id="debefc36483e7b60c22269c4f180b110">
             In order to download a file like the educational public domain video we mentioned above, you have to find and download a torrent file (which uses the .torrent file extension) and then open it with your BitTorrent client. The torrent file
             <strong>
              does not contain your files
             </strong>
             . Instead, it contains information which tells your BitTorrent client where it can find peers who are also sharing and downloading the file.
            </p>
            <h3 data-textannotation-id="aca4529a8ad733f4b633430bd535f6db">
             <a name="1" target="_blank">
             </a>
             <a class="autolink" href="http://lifehacker.com/tag/how-to/" title="Click here to read more posts tagged HOW TO">
              How to
             </a>
             find and download a file with BitTorrent
            </h3>
            <p data-textannotation-id="a1b732df39c85a922fade36dea49c8b7">
             Now that you've got a better idea of the terminology and process behind BitTorrent, let's jump right into using BitTorrent.
            </p>
            <p data-textannotation-id="35f0c65a34701f1510f7e9629731548c">
             <strong>
              First you need to download a BitTorrent client
             </strong>
             (the program that manages your BitTorrent downloads). I'd recommend:
            </p>
            <ul>
             <li data-textannotation-id="299bc40b36cdf363b9fe9fa191f4600d">
              <a href="http://utorrent.com/" target="_blank">
               uTorrent
              </a>
              for Windows/Mac
             </li>
             <li data-textannotation-id="74cfd7915872467f313f7abf6a181c8e">
              <a href="http://transmission.m0k.org/download.php" target="_blank">
               Transmission
              </a>
              for Mac
             </li>
             <li data-textannotation-id="17fd7c7faccfb0478f3db6e446ac7754">
              <a href="http://www.vuze.com/" target="_blank">
               Vuze
              </a>
              or
              <a href="http://ktorrent.org/" target="_blank">
               KTorrent
              </a>
              for Linux (Actually, Vuze is cross platform, meaning it will work on Windows and Mac, but on those platforms we still prefer the alternatives listed above.)
             </li>
            </ul>
            <p data-textannotation-id="84480f1740d735abd2cb00b008ad7bac">
             <br/>
             <br/>
             <br/>
             <strong>
              Search for a good torrent.
             </strong>
             There are a handful of really good web sites for downloading torrents (that's right, you search for torrents on the internet). The sites I'd recommend (in no particular order) are:
            </p>

               <img alt="A beginner's guide to BitTorrent" data-asset-url="http://i.kinja-img.com/gawker-media/image/upload/s--eBCYlaw5--/18fcgocthiuswpng.png" data-chomp-id="18fcgocthiuswpng" data-format="png" height="78" src="http://i.kinja-img.com/gawker-media/image/upload/s--eBCYlaw5--/18fcgocthiuswpng.png" width="232"/>

            <p data-textannotation-id="f286a3475a1a4cb4bb5c7a7c49db96b1">
             Try out whichever one you like. One might fit your tastes better than another, but I've had good experiences with all of these. From this point, search the site using their search box like you're using Googlejust type in the name of what you're looking for. You'll likely get several results, but you want to choose the torrent with the highest number of seeders (indicated in most BitTorrent search results under a field labeled 'S'). Seeders are people who have already downloaded and are sharing the entire file. The more seeders, the faster your download will be. Some sites also provide you with a health meter, which is generally a measure of seeders vs. active downloaders.
            </p>
            <p class="has-media media-300 has-image" data-textannotation-id="c871ac9f8d250071a1e6fd1e35c316a7">

              Download the torrent.
             </strong>
             Once you've found a good and healthy torrent, find the download link and download the torrent. Your browser will ask you what you want to do with the file, so be sure to tell it to open the torrent in the BitTorrent client you downloaded above.
            </p>
            <p class="has-media media-470 has-image" data-textannotation-id="f187978f9304a0f26f4544815cf16706">

              </span>
              <span class="magnifier lightBox hide">
               <span class="icon icon-white icon-search-plus">
               </span>

            <p data-textannotation-id="73665b90f13030b7e766041fd62d83e3">
             Your BitTorrent client will open and (possibly) ask you where you want to save the file(s). Pick your save location, hit OK, and that's it; your file will begin downloading. If you're not impressed with the speed at first, be patient. It can sometimes take a minute or two before the download ramps up to full speed. If you're still not happy, try searching for another torrent with more seeders.
            </p>
            <h3 data-textannotation-id="b27e3a022a2a3476018f09d1892b656a">
             That's it?
            </h3>
            <p data-textannotation-id="2cc5a65060aabd615e48bcdc979c8e9e">
             Yep, that's it. That, in a nutshell, is how to download files using BitTorrent. There
             <em>
              can
             </em>
             be more to it, of course, if you want to dive in a bit deeper. For example, you can run through the Speed Guide in uTorrent to improve your download speeds (the guide is fairly self explanatoryjust go to Options -&gt; Speed Guide to get started), download select files from the torrent rather than every file, throttle your bandwidth, and so on, but this basic guide should get you started.
            </p>
            <p class="has-media media-300 has-image" data-textannotation-id="8d81b8de4d21b1abe3bef011fd32c4b7">
             <span class="lightBoxWrapper">
              <span class="img-border">
               <img alt="A beginner's guide to BitTorrent" data-asset-url="http://i.kinja-img.com/gawker-media/image/upload/s--QL7CK0rq--/18fcgoctc8v3mpng.png" data-chomp-id="18fcgoctc8v3mpng" data-format="png" height="89" src="http://i.kinja-img.com/gawker-media/image/upload/s--QL7CK0rq--/18fcgoctc8v3mpng.png" width="123"/>
              </span>
              <span class="magnifier lightBox hide">
               <span class="icon icon-white icon-search-plus">
               </span>
              </span>
             </span>
            </p>
            <p data-textannotation-id="97a0a145d9bb3ecc263aa4bc79d18426">
             Also, to ensure you stay in good standing in the BitTorrent community (and aren't labeled a leecher), you should always try to upload as much as you download. Most BitTorrent clients keep track of your upload/download ratio, and you should generally continue sharing a file until your ratio reaches 1, after which you can feel free to remove it from your client (the file will remain on your computeryou just stop sharing it).
            </p>
