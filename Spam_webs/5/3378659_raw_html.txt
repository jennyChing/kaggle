        <h2 class="title">
         How to Save and Print the Free Patterns
        </h2>
        <div class="widget-content">
         <span class="Apple-style-span" style="  white-space: pre-wrap; font-family:monospace;font-size:13px;">
          <span class="Apple-style-span" style="  color: rgb(51, 51, 51); font-family:Verdana, Arial, sans-serif;font-size:13px;">
           <div class="widget-content">
            <ul>
             <span class="Apple-style-span" style="white-space: pre-wrap;   font-family:-webkit-monospace;font-size:13px;">
              This method is for Internet Explorer.  Other browsers may use different terms but the procedure is basically the same.
             </span>
             <center>
              <b>
               Always Save Pattern on Your Computer Never Print From Screen
              </b>
             </center>
             <li>
              Left Click on the Pattern you wish to save. The pattern will open in a new page.
             </li>
             <li>
              Right click on that image, then from the drop down menu select "Save image as.." (note that this pattern image may appear much too large to print on a single sheet of paper. That is okay, that only means the pattern is of a higher resolution than that of your monitor, it will still print correctly.)
             </li>
             <li>
              In the new "Save As" dialog window select the location on you computer where you would like to save the image, then click "Save".
             </li>
             <li>
              When you are ready to print the pattern, open the image file in your graphic or print program and set print options to print at 100%. The pattern will print at the intended size when using this procedure. Note, that the pattern will almost never print correctly if you attempt to print directly from the blog page as these images are nearly always reduced in size and are often at reduced resolution.
             </li>
            </ul>
           </div>
