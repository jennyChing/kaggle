                      <a class="pr-review-faceoff-readcomplete" data-pr-event="faceoff-negative-link" href="#" onclick="POWERREVIEWS.display.displayFaceoffModal('RIVETINGKO','pr-review-faceoff-negative-RIVETINGKO'); return false">

                     <p class="pr-review-faceoff-review-full">
                      I started out with a cute bolero pattern that was designed for this yarn. Since this yarn is 100% cotton, the pattern stated to expect about 12% shrinkage after washing and drying (which was taken into account in the pattern). Though the pattern was 'stinkin' cute, and my daughter would have loved it), I found it to be too complex for me to endure to the end, so I ripped it out.
                      <br/>
                      <br/>
                      However, I was still intrigued with the yarn (being made from recycled jeans) so I decided to make a simple lacy scarf, using large size 13 needles on two strands of Riveting. It turned out cute and my daughter really likes it, but the shrinkage in lenght was about 18%, not 12%. At this point, I still have 3 skeins of yarn left, so I will try making the same scarf with just one strand and a bit smaller needle so as to be lighter in weight and a slightly tighter knit.
                      <br/>
                      <br/>
                      I think possibly the difference in the needle size affects the amount of shrinkage. I was expecting to be able to make some things to sell, but with an unstable shrinkage factor, it would not be profitable. For personal use, if you like a good challenge to create something with this unique yarn, I say, 'go for it.'

Since this yarn is 100% cotton, the pattern stated to expect about 12% shrinkage after washing and drying (which was taken into account in the pattern). Though the pattern was 'stinkin' cute, and my daughter would have loved it), I found it to be too complex for me to endure to the end, so I ripped it out.
                    <br/>
                    <br/>
                    However, I was still intrigued with the yarn (being made from recycled jeans) so I decided to make a simple lacy scarf, using large size 13 needles on two strands of Riveting. It turned out cute and my daughter really likes it, but the shrinkage in lenght was about 18%, not 12%. At this point, I still have 3 skeins of yarn left, so I will try making the same scarf with just one strand and a bit smaller needle so as to be lighter in weight and a slightly tighter knit.
                    <br/>
                    <br/>
                    I think possibly the difference in the needle size affects the amount of shrinkage. I was expecting to be able to make some things to sell, but with an unstable shrinkage factor, it would not be profitable. For personal use, if you like a good challenge to create something with this unique yarn, I say, 'go for it.'
