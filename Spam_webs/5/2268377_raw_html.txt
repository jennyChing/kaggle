&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: justify;"&gt;In less than 3 weeks, we will be moving into our new home. For most people, moving is a huge pain and a stressful time in their lives. It is for us as well. But being organized and having a plan has helped us tremendously, especially since we dont have any family to help us here. I have been researching on the web, ways to organize our move. I have found a couple fantastic ideas to make things easy during the transition.&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: justify;"&gt;&lt;span class="Apple-style-span" style="font-size: x-large;"&gt;1. Create a moving binder.&lt;/span&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: justify;"&gt;A binder specifically designated for &#8220;the move&#8221; will save you time, energy, and possibly money (if you lose receipts, etc). I organized my binder into categories.&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;1. &lt;strong&gt;Checklists-Find an online printable that you can use as a guide to what needs to be done before, during, and after the move.&lt;/strong&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;2. &lt;strong&gt;Utilities- To keep track of appointments: who, where, what, and when, this is a good idea.&lt;/strong&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;3. &lt;strong&gt;Loan Documents- For easy access &amp; reference, I keep these here until after the move.&lt;/strong&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;4. &lt;strong&gt;Receipts- This section is for any purchases that will be made in reference to the move or the house. I purchased our kitchen faucet today and printed out the receipt to put here. If something doesnt fit, or you need to return or exchange for any reason, there is no searching for the receipt.&lt;/strong&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;5. &lt;strong&gt;Movers- This section is for your moving company or truck rental contract &amp; contact numbers.&lt;/strong&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;6. &lt;strong&gt;Builder- Any contact numbers for your builder, any warranties, paperwork, etc. will go here.&lt;/strong&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;7. &lt;strong&gt;Inventory List- You can find these online for free. Just print one out and fill it out.&lt;/strong&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;8. &lt;strong&gt;Schedules- This section is for appointments (utilities), moving company schedule, etc&#8230; I have lots of appointments during our move time so I used a word document calendar, filled it in, and printed it out and placed it in the binder. Mine has our house closing appointment, walk through, movers times, utility hook up times, etc.. Its a good section to have.&lt;/strong&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;*Most of the paperwork in the binder will later be filed. What is not needed will be sent to the recycling bin.&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;a style="margin-left: 1em; margin-right: 1em;" href="http://3.bp.blogspot.com/-7EqfV-txdxQ/T-Sb8JT00NI/AAAAAAAAFIc/iJMyEwH8QRs/s1600/2w.jpg"&gt;&lt;img src="http://3.bp.blogspot.com/-7EqfV-txdxQ/T-Sb8JT00NI/AAAAAAAAFIc/iJMyEwH8QRs/s1600/2w.jpg" alt="" border="0" /&gt;&lt;/a&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div style="text-align: center;"&gt;&lt;/div&gt;
&lt;div style="text-align: center;"&gt;&lt;/div&gt;
&lt;div style="text-align: justify;"&gt;The binder &amp; tabs were purchased at Staples. I used my labeler for the categories, so I can later remove them and reuse the tabs for a different binder.&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div style="text-align: center;"&gt;&lt;/div&gt;
&lt;div style="text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;a style="margin-left: 1em; margin-right: 1em;" href="http://1.bp.blogspot.com/-8EVsdOpDVxU/T-Sb7vcluDI/AAAAAAAAFIU/ybBxZ_D_iVM/s1600/12w.jpg"&gt;&lt;img src="http://1.bp.blogspot.com/-8EVsdOpDVxU/T-Sb7vcluDI/AAAAAAAAFIU/ybBxZ_D_iVM/s1600/12w.jpg" alt="" border="0" /&gt;&lt;/a&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: justify;"&gt;&lt;strong&gt;&lt;span class="Apple-style-span" style="font-size: x-large;"&gt;2. Create a color coded moving legend.&lt;/span&gt;&lt;/strong&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div style="text-align: justify;"&gt;This tip was inspired by&lt;a href="http://alejandra.tv/"&gt;Alejandra&lt;/a&gt; (my organizing friend). She moved recently and used this same idea for her move. I created a moving legend, using circle labels to color code each designated area in the house where each moving box will go. I added to Alejandra&#8217;s idea with the use of color coded cardstock to place on each door, as a map to show the movers where they need to put the boxes. This helps alleviate any confusion as to where the boxes go. (I need to add a couple more colors for the bathrooms, bonus room, etc.).&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div style="text-align: justify;"&gt;&lt;span class="Apple-style-span" style="color: #ffd966; font-size: x-large;"&gt;*&lt;/span&gt;*&lt;strong&gt;TIP&#8211; PHOTO COPY your moving legend so it can be placed through out the house and on the front, back, and garage doors. You do &lt;span style="text-decoration: underline;"&gt;not&lt;/span&gt; need to make more than one legend. Utilize your copier/printer!!&lt;br /&gt;
&lt;/strong&gt;&lt;/p&gt;
&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;a style="margin-left: 1em; margin-right: 1em;" href="http://4.bp.blogspot.com/-0-Yf0G5llYw/T-Sb-HIgAYI/AAAAAAAAFI0/XAHapSp5RZY/s1600/6w.jpg"&gt;&lt;img src="http://4.bp.blogspot.com/-0-Yf0G5llYw/T-Sb-HIgAYI/AAAAAAAAFI0/XAHapSp5RZY/s1600/6w.jpg" alt="" border="0" /&gt;&lt;/a&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: justify;"&gt;The color coded cardstock will be hung on the doors of each room.&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;a style="margin-left: 1em; margin-right: 1em;" href="http://4.bp.blogspot.com/-59XPxNke4Gw/T-Sb8i9bfaI/AAAAAAAAFIg/4VVOq_m8aIE/s1600/4w.jpg"&gt;&lt;img src="http://4.bp.blogspot.com/-59XPxNke4Gw/T-Sb8i9bfaI/AAAAAAAAFIg/4VVOq_m8aIE/s1600/4w.jpg" alt="" border="0" /&gt;&lt;/a&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;a style="margin-left: 1em; margin-right: 1em;" href="http://4.bp.blogspot.com/-eQHUkro4Bug/T-Sb9JgCEWI/AAAAAAAAFIs/k3pVngnG4_o/s1600/5w.jpg"&gt;&lt;img src="http://4.bp.blogspot.com/-eQHUkro4Bug/T-Sb9JgCEWI/AAAAAAAAFIs/k3pVngnG4_o/s1600/5w.jpg" alt="" border="0" /&gt;&lt;/a&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;a style="margin-left: 1em; margin-right: 1em;" href="http://2.bp.blogspot.com/-OhOwLDIx0ZQ/T-Sb-dAYYcI/AAAAAAAAFI8/_vgcE6JCyP8/s1600/7w.jpg"&gt;&lt;img src="http://2.bp.blogspot.com/-OhOwLDIx0ZQ/T-Sb-dAYYcI/AAAAAAAAFI8/_vgcE6JCyP8/s1600/7w.jpg" alt="" border="0" /&gt;&lt;/a&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: justify;"&gt;The color coded stickers are placed on the label of each box, as well as one at the top of each box.&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;a style="margin-left: 1em; margin-right: 1em;" href="http://4.bp.blogspot.com/-T1AMjMPQsIc/T-Sb_SMRYsI/AAAAAAAAFJM/u7eR43UdgOw/s1600/9w.jpg"&gt;&lt;img src="http://4.bp.blogspot.com/-T1AMjMPQsIc/T-Sb_SMRYsI/AAAAAAAAFJM/u7eR43UdgOw/s1600/9w.jpg" alt="" border="0" /&gt;&lt;/a&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: justify;"&gt;&lt;strong&gt;&lt;span class="Apple-style-span" style="font-size: x-large;"&gt;3. Get like sized boxes!&lt;/span&gt;&lt;/strong&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: justify;"&gt;The cheapest place I found boxes (other than free ones) were at &lt;a href="http://www.lowes.com/Search=moving+boxes?storeId=10151&amp;N=0&amp;langId=-1&amp;catalogId=10051&amp;rpp=16"&gt;Lowes&lt;/a&gt;. They are made well and stack well (which is important when loading them onto the truck). We have boxes galore! They range from .70 cents for the small box, $1.12 for the medium box, and $1.36 for the largest box. When stacking these boxes, there are only 3 sizes, so you can stack the small boxes together, medium together, and finally the large ones together. This will efficiently help you fit things into your moving truck better. You can purchase them at your local store, or order them online &lt;a href="http://www.lowes.com/Search=moving+boxes?storeId=10151&amp;N=0&amp;langId=-1&amp;catalogId=10051&amp;rpp=16"&gt;here&lt;/a&gt;.&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;a style="margin-left: 1em; margin-right: 1em;" href="http://4.bp.blogspot.com/-YWGUBhi7y70/T-Sb-ygSZ1I/AAAAAAAAFJE/pywud3FjT28/s1600/8w.jpg"&gt;&lt;img src="http://4.bp.blogspot.com/-YWGUBhi7y70/T-Sb-ygSZ1I/AAAAAAAAFJE/pywud3FjT28/s1600/8w.jpg" alt="" border="0" /&gt;&lt;/a&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div style="text-align: justify;"&gt;&lt;strong&gt;&lt;span class="Apple-style-span" style="font-size: x-large;"&gt;4. Create a &#8220;MOVING BASKET&#8221;&lt;/span&gt;&lt;/strong&gt;&lt;/div&gt;
&lt;div style="text-align: center;"&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: justify;"&gt;Another good idea is to create a &#8220;moving basket&#8221; for all of the necessary labels, tape, sharpies, scissors, etc. When you move from room to room, just grab the basket. No more searching for lost items.&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;a style="margin-left: 1em; margin-right: 1em;" href="http://3.bp.blogspot.com/-itjcZT_Au5c/T-Sb4h1pAAI/AAAAAAAAFH8/2LSdexA7zKY/s1600/10w.jpg"&gt;&lt;img src="http://3.bp.blogspot.com/-itjcZT_Au5c/T-Sb4h1pAAI/AAAAAAAAFH8/2LSdexA7zKY/s1600/10w.jpg" alt="" border="0" /&gt;&lt;/a&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div style="text-align: justify;"&gt;&lt;span class="Apple-style-span" style="font-size: x-large;"&gt;&lt;strong&gt;5. Use ziploc freezer bags&lt;/strong&gt;&lt;/span&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div style="text-align: center;"&gt;&lt;/div&gt;
&lt;div style="text-align: justify;"&gt;Ziploc bags are my favorite packing &amp; organizing item. I use them for everything. They are cheap and effective. I purchased a 4 pack of freezer bags at Sams. Freezer bags are thicker and will hold things better. My 5 year old separated all of her little toys into these bags and then we packed them into a larger box. When its time to unpack, everything is already organized! I also use them for bathroom accessories, pens, pencils, etc&#8230; Really anything that can fit into them and needs separated, I use them for.&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div style="text-align: center;"&gt;&lt;/div&gt;
&lt;div style="text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;a style="margin-left: 1em; margin-right: 1em;" href="http://2.bp.blogspot.com/-zHsccfpUydc/T-Sb6f672mI/AAAAAAAAFIE/yP_3xwBBXhI/s1600/11w.jpg"&gt;&lt;img src="http://2.bp.blogspot.com/-zHsccfpUydc/T-Sb6f672mI/AAAAAAAAFIE/yP_3xwBBXhI/s1600/11w.jpg" alt="" border="0" /&gt;&lt;/a&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: justify;"&gt;&lt;strong&gt;&lt;span class="Apple-style-span" style="font-size: x-large;"&gt;6. Pack with bubble wrap&lt;/span&gt;&lt;/strong&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: justify;"&gt;Finally, bubble wrap is needed for breakables. This stuff is reusable too!!&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;a style="margin-left: 1em; margin-right: 1em;" href="http://3.bp.blogspot.com/-B12xkn-K4lA/T-Sb623ARXI/AAAAAAAAFIM/UMjeHfUs2OA/s1600/12q.jpg"&gt;&lt;img src="http://3.bp.blogspot.com/-B12xkn-K4lA/T-Sb623ARXI/AAAAAAAAFIM/UMjeHfUs2OA/s1600/12q.jpg" alt="" border="0" /&gt;&lt;/a&gt;&lt;/div&gt;
&lt;p&gt;&nbsp;&lt;/p&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: justify;"&gt;There are many other tips and tricks online to help get organized for a move. &#8220;&lt;a href="http://www.upack.com/moving-resources/checklist.asp"&gt;This site&lt;/a&gt;&#8221; has a great list of tips organized by a weekly countdown. Good luck moving.I hope your move goes as smoothly as ours has (so far). See you soon!&lt;/div&gt;

        (function() {
  var form = document.getElementById('mad_mimi_signup_form'),
      submit = document.getElementById('webform_submit_button'),
      validEmail = /.+@.+\..+/,
      isValid;
      
  form.onsubmit = function(event) {
    validate();
    if(!isValid) {
      revalidateOnChange();
      return false;
    }
  };
  
  function validate() {
    isValid = true;
    emailValidation();
    fieldAndListValidation();
    updateFormAfterValidation();
  }

  function emailValidation() {
    var email = document.getElementById('signup_email');
    if(!validEmail.test(email.value)) {
      textFieldError(email);
      isValid = false;
    } else {
      removeTextFieldError(email);
    }
  }

  function fieldAndListValidation() {
    var fields = form.querySelectorAll('.mimi_field.required');
    for(var i = 0; i &lt; fields.length; ++i) {
      var field = fields[i],
          type  = fieldType(field);
      if(type == 'checkboxes' || type == 'radio_buttons') {
        checkboxAndRadioValidation(field);
      } else {
        textAndDropdownValidation(field, type);
      }
    }
  }

  function fieldType(field) {
    var type = field.querySelectorAll('.field_type');
    if(type.length &gt; 0) {
      return type[0].getAttribute('data-field-type');
    } else if(field.className.indexOf('checkgroup') &gt;= 0) {
      return 'checkboxes';
    } else {
      return 'text_field';
    }
  }

  function checkboxAndRadioValidation(field) {
    var inputs   = field.getElementsByTagName('input'),
        selected = false;
    for(var i = 0; i &lt; inputs.length; ++i) {
      var input = inputs[i];
      if((input.type == 'checkbox' || input.type == 'radio') &amp;&amp; input.checked) selected = true;
    }
    if(selected) {
      field.className = field.className.replace(/ invalid/g, '');
    } else {
      if(field.className.indexOf('invalid') == -1) field.className += ' invalid';
      isValid = false;
    }
  }

  function textAndDropdownValidation(field, type) {
    var inputs = field.getElementsByTagName('input');
    for(var i = 0; i &lt; inputs.length; ++i) {
      var input = inputs[i];
      if(input.name.indexOf('signup') &gt;= 0) {
        if(type == 'text_field') {
          textValidation(input);
        } else {
          dropdownValidation(field, input);
        }
      }
    }
    htmlEmbedDropdownValidation(field);
  }

  function textValidation(input) {
    if(input.id == 'signup_email') return;
    var val = input.value;
    if(val == '') {
      textFieldError(input);
      isValid = false;
      return;
    } else {
      removeTextFieldError(input)
    }
  }

  function dropdownValidation(field, input) {
    var val = input.value;
    if(val == '') {
      if(field.className.indexOf('invalid') == -1) field.className += ' invalid';
      onSelectCallback(input);
      isValid = false;
      return;
    } else {
      field.className = field.className.replace(/ invalid/g, '');
    }
  }

  function htmlEmbedDropdownValidation(field) {
    var dropdowns = field.querySelectorAll('.mimi_html_dropdown');
    for(var i = 0; i &lt; dropdowns.length; ++i) {
      var dropdown = dropdowns[i],
          val      = dropdown.value;
      if(val == '') {
        if(field.className.indexOf('invalid') == -1) field.className += ' invalid';
        isValid = false;
        dropdown.onchange = validate;
        return;
      } else {
        field.className = field.className.replace(/ invalid/g, '');
      }
    }
  }

  function textFieldError(input) {
    input.className   = 'required invalid';
    input.placeholder = input.getAttribute('data-required-field');
  }

  function removeTextFieldError(input) {
    input.className   = 'required';
    input.placeholder = '';
  }

  function onSelectCallback(input) {
    if(typeof Widget != 'undefined' &amp;&amp; Widget.BasicDropdown != undefined) {
      var dropdownEl = input.parentNode,
          instances  = Widget.BasicDropdown.instances;
      for(var i = 0; i &lt; instances.length; ++i) {
        var instance = instances[i];
        if(instance.wrapperEl == dropdownEl) {
          instance.onSelect = validate;
        }
      }
    }
  }

  function updateFormAfterValidation() {
    form.className   = isValid ? '' : 'mimi_invalid';
    submit.value     = submitButtonText();
    submit.disabled  = !isValid;
    submit.className = isValid ? 'submit' : 'disabled';
  }

  function submitButtonText() {
    var invalidFields = document.querySelectorAll('.invalid'),
        text;
    if(isValid || invalidFields == undefined) {
      text = submit.getAttribute('data-default-text');
    } else {
      if(invalidFields.length &gt; 1 || invalidFields[0].className.indexOf('checkgroup') == -1) {
        text = submit.getAttribute('data-invalid-text');
      } else {
        text = submit.getAttribute('data-choose-list');
      }
    }
    return text;
  }

  function revalidateOnChange() {
    var fields = form.querySelectorAll(".mimi_field.required");
    for(var i = 0; i &lt; fields.length; ++i) {
      var inputs = fields[i].getElementsByTagName('input');
      for(var j = 0; j &lt; inputs.length; ++j) {
        inputs[j].onchange = validate;
      }
    }
  }
})();
