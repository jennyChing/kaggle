       <p>
        Web browser is the important software in our computer. Everybody knows without web browser we cant do anything. We spend most of the time in web browser while we chat,email,browse,shopping,reading news, online banking, etc,
        <br/>
        <a href="http://www.2daygeek.com/wp-content/uploads/2014/11/google-chrome-39-release-installation-steps.jpg">
         <img alt="google-chrome-39-release-installation-steps" class="aligncenter size-full wp-image-4238" data-lazy-src="http://www.2daygeek.com/wp-content/uploads/2014/11/google-chrome-39-release-installation-steps.jpg" height="300" src="http://www.2daygeek.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" width="610"/>

          <img alt="google-chrome-39-release-installation-steps" class="aligncenter size-full wp-image-4238" height="300" src="http://www.2daygeek.com/wp-content/uploads/2014/11/google-chrome-39-release-installation-steps.jpg" width="610"/>
         </noscript>
        </a>
        <br/>
        Google Chrome is the best browser ever, because they use V8 engine. V8 is Googles open source high performance JavaScript engine. Google chrome is a light weight browser. Every tab uses different process so its very fast compared to other browsers. Google Inc has proudly announced the release of new version of Google
        <strong>
         Chrome 39
        </strong>
        on
        <strong>
         November 18, 2014 (3 days ago)
        </strong>
        which supports Windows, Mac &amp; Linux.
       </p>
       <ul>
        <li>
         Follow this link for
         <a href="http://www.2daygeek.com/google-chrome-37-installation-in-linux-centos-7-rhel-7-fedora-20/" target="_blank" title="google chrome 38 installation">
          Fresh installation of google chrome 39

         to CentOS / RHEL / Fedora
        </li>
        <li>
         Follow this link for
         <a href="http://www.2daygeek.com/google-chrome-37-installation-in-linux-mint-17-ubuntu-14-04-debian-7-6/" target="_blank" title="google chrome 38 installation">
          Fresh installation of google chrome 39

         to Ubuntu / Debian / Linux Mint
        </li>
        <li>
         Follow this link for
         <a href="http://www.2daygeek.com/install-google-chrome38-in-linux/" target="_blank" title="google chrome 38 installation">
          Google Chrome 38 upgrade steps
         </a>
         to Ubuntu / Debian / Linux Mint
        </li>
       </ul>
       <h4>
        <span style="color:#FA720E">
         Whats new in Google Chrome 39
        </span>
       </h4>
       <p>
       </p>
       <ul>
        <li>
         Enable Chromecast support for media files stored on Google Drive
        </li>
        <li>
         Update Network Error page design
        </li>
        <li>
         Display apps installed in Google Drive in the Chrome OS Launcher
        </li>
        <li>
         64-bit support for Mac
        </li>
        <li>
         A number of new apps/extension APIs
        </li>
        <li>
         Lot of under the hood changes for stability and performance
        </li>
        <li>
         This update includes 42 security fixes.
        </li>
        <li>
         For complete
         <a href="http://googlechromereleases.blogspot.in/2014/11/stable-channel-update_18.html" target="_blank" title="google chrome 39 bug fixes">
          bug fix
         </a>
        </li>
       </ul>
       <h4>
        <span style="color:#a20000">
         1) Upgrading google chrome to CentOS, RHEL, Fedora from version 38 to 39
        </span>
       </h4>
       <p>
        Use the below commands to upgrade your google chrome version to latest release in CentOS, RHEL, Fedora, Oracle Linux,etc,
       </p>

       <h4>
        <span style="color:#a20000">
         1a) Upgrading google chrome to Ubuntu, Debian, Linux Mint from version 38 to 39
        </span>
       </h4>
       <p>
        Use the below commands to upgrade your google chrome version to latest release in Ubuntu, Debian, Linux Mint,etc,
       </p>
       <pre>
        <span style="color:#848484">
         # Check Google chrome upgrade is available Ubuntu, Debian, Linux Mint #
        </span>
        magesh@magesh-desktop:~$
        <span style="color:#fff000">
         apt-cache policy google-chrome-stable
        </span>
        google-chrome-stable:
  Installed: 38.0.2125.111-1
  Candidate: 38.0.2125.111-1
  Version table:
 *** 38.0.2125.111-1 0
        500 http://dl.google.com/linux/chrome/deb/ stable/main amd64 Packages
        100 /var/lib/dpkg/status
        <span style="color:#848484">
         # Update your system #
        </span>
        magesh@magesh-desktop:~$
        <span style="color:#fff000">
         sudo apt-get update
        </span>
        <span style="color:#848484">
         # Re-Check Google chrome upgrade is available Ubuntu, Debian, Linux Mint #
        </span>
        magesh@magesh-desktop:~$
        <span style="color:#fff000">
         apt-cache policy google-chrome-stable
        </span>
        google-chrome-stable:
  Installed: 38.0.2125.111-1
  Candidate: 39.0.2171.65-1
  Version table:
     39.0.2171.65-1 0
        500 http://dl.google.com/linux/chrome/deb/ stable/main amd64 Packages
 *** 38.0.2125.111-1 0
        100 /var/lib/dpkg/status
        <span style="color:#848484">
         # Google chrome upgrade in Ubuntu/Debian/Linux Mint #
        </span>
        magesh@magesh-desktop:~$
        <span style="color:#fff000">
         sudo apt-get install --only-upgrade google-chrome-stable
        </span>
       </pre>
