                   <div class="entry_content">
                    <p>
                     There are
                     <del datetime="2011-08-13T11:21:35+00:00">
                      two
                     </del>
                     three methods to check if a user is fan of a certain page:
                    </p>
                    <a class="sectionindex" id="using_the_graph_api" name="using_the_graph_api">
                    </a>
                    <h2>
                     Using the Graph API
                     <code>
                      /USER_ID/likes/PAGE_ID
                     </code>
                     method
                    </h2>
                    <p>
                     Facebook has just
                     <a href="https://developers.facebook.com/blog/post/540/" target="_blank">
                      released
                     </a>
                     another Graph API way for you to check for this information:
                     <br/>
                     <code>
                      https://graph.facebook.com/me/likes/PAGE_ID
                      <br/>
                      &amp;access_token=ACCESS_TOKEN
                     </code>
                     <br/>
                     This would return either an empty
                     <code>
                      data
                     </code>
                     array:
                    </p>
                    <pre>
                     Array
(
    [data] =&gt; Array
        (
        )

)
                    </pre>
                    <p>
                     Or Something like:
                    </p>
                    <pre>
                     Array
(
    [data] =&gt; Array
        (
            [0] =&gt; Array
                (
                    [name] =&gt; Real Madrid C.F.
                    [category] =&gt; Professional sports team
                    [id] =&gt; 19034719952
                    [created_time] =&gt; 2011-05-03T20:53:26+0000
                )

        )

)
                    </pre>
                    <p>
                     So this is how we check using the PHP-SDK:
                    </p>
                    <pre class="brush: php; title: ; notranslate" title="">
                     &lt;?php
require '../src/facebook.php';

// Create our Application instance (replace this with your appId and secret).
$facebook = new Facebook(array(
  'appId' =&gt; 'APP_ID',
  'secret' =&gt; 'APP_SECRET',
));

$user = $facebook-&gt;getUser();

if ($user) {
  try {
    $likes = $facebook-&gt;api("/me/likes/PAGE_ID");
	if( !empty($likes['data']) )
		echo "I like!";
	else
		echo "not a fan!";
  } catch (FacebookApiException $e) {
    error_log($e);
    $user = null;
  }
}

if ($user) {
  $logoutUrl = $facebook-&gt;getLogoutUrl();
} else {
  $loginUrl = $facebook-&gt;getLoginUrl(array(
	'scope' =&gt; 'user_likes'
  ));
}

// rest of code here
?&gt;
                    </pre>
                    <p>
                     Similar request in Javascript:
                    </p>
                    <pre class="brush: jscript; title: ; notranslate" title="">
                     FB.api('/me/likes/PAGE_ID',function(response) {
	if( response.data ) {
		if( !isEmpty(response.data) )
			alert('You are a fan!');
		else 
			alert('Not a fan!');
	} else {
		alert('ERROR!');
	}
});

// function to check for an empty object
function isEmpty(obj) {
	for(var prop in obj) {
		if(obj.hasOwnProperty(prop))
			return false;
	}

	return true;
}
                    </pre>
                    <p>
                     More about the
                     <code>
                      isEmpty
                     </code>
                     function
                     <a href="http://stackoverflow.com/questions/679915/how-do-i-test-for-an-empty-javascript-object-from-json" target="_blank">
                      here
                     </a>
                     .
                    </p>
                    <a class="back_to_top" href="#page-main">
                     Back to top
                    </a>
                    <hr/>
                    <a class="sectionindex" id="using_the_rest_api" name="using_the_rest_api">
                    </a>
                    <h2>
                     Using the REST API pages.isFan method
                    </h2>
                    <p>
                     While Facebook is in the process of deprecating the REST API,
                     <a href="http://developers.facebook.com/docs/reference/rest/pages.isfan">
                      <code>
                       pages.isFan
                      </code>
                     </a>
                     is still working and can be called from the new PHP-SDK:
                    </p>
                    <pre class="brush: php; title: ; notranslate" title="">
                     $isFan = $facebook-&gt;api(array(
	"method"	=&gt; "pages.isFan",
	"page_id"	=&gt; $page_id,
	"uid"		=&gt; $user_id
));
if($isFan === TRUE)
	echo "I'm a fan!";
                    </pre>
                    <p>
                     Here are two notes to consider:
                    </p>
                    <ul>
                     <li>
                      Using
                      <code>
                       if($isFan)
                      </code>
                      is not enough because:
                      <br/>
                      <blockquote>
                       <p>
                        If the users pages are set to less than everyone privacy, you must ask the user for the
                        <code>
                         user_likes
                        </code>
                        extended permission and include a valid user access token in the API call.
                       </p>
                      </blockquote>
                      <p>
                       So basically you may get an ugly Permissions error message and itll actually validate!
                      </p>
                     </li>
                     <li>
                      If the user you are checking against has a valid session then you dont need the
                      <code>
                       uid
                      </code>
                      parameter
                     </li>
                    </ul>
                    <p>
                     Same API call but in Javascript:
                    </p>
                    <pre class="brush: jscript; title: ; notranslate" title="">
                     FB.api({ method: 'pages.isFan', page_id: '184484190795', uid: 'user_id' }, function(resp) {
	if (resp == true) {
	  Log.info('user_id likes the Application.');
	} else if(resp.error_code) {
	  Log.error(resp.error_msg);
	} else {
	  Log.error("user_id doesn't like the Application.");
	}
});
                    </pre>
                    <p>
                     Once again, you need to check the response of the call if true or notusing
                     <code>
                      if(resp)
                     </code>
                     is
                     <strong>
                      NOT
                     </strong>
                     enough!
                     <br/>
                     The example in the
                     <a href="http://developers.facebook.com/tools/console">
                      Facebook JavaScript Test Console
                     </a>
                     is a bit misleading (the does-like example), since if you dont grant the
                     <code>
                      user_likes
                     </code>
                     permission and just try it with
                     <code>
                      uid: '579187141'
                     </code>
                     it would result that the user likes the application which is not true!
                    </p>
                    <p>
                     <em>
                      P.S: I just decremented my user id to get the above user which (it seems) he has a strict privacy for pages he likes!
                     </em>
                    </p>
                    <a class="back_to_top" href="#page-main">
                     Back to top
                    </a>
                    <hr/>
                    <a class="sectionindex" id="using_the_fql_page_fan" name="using_the_fql_page_fan">
                    </a>
                    <h2>
                     Using the FQL page_fan table
                    </h2>
                    <p>
                     We can query the
                     <a href="http://developers.facebook.com/docs/reference/fql/page_fan/">
                      <code>
                       page_fan
                      </code>
                     </a>
                     table to check if the user is a fan of a page or not like so:
                    </p>
                    <pre class="brush: php; title: ; notranslate" title="">
                     $result = $facebook-&gt;api(array(
	"method"	=&gt; "fql.query",
	"query"		=&gt; "SELECT uid FROM page_fan WHERE uid=$user_id AND page_id=$page_id"
));
if(count($result))
	echo "$user_id is a fan!";
                    </pre>
                    <p>
                     Following the same concept as the previous method, if you have a user valid session you can use
                     <code>
                      me()
                     </code>
                     instead of
                     <code>
                      $user_id
                     </code>
                     .
                    </p>
                    <p>
                     The FQL in Javascript:
                    </p>
                    <pre class="brush: jscript; title: ; notranslate" title="">
                     FB.api({
		method:		'fql.query', 
		query:	'SELECT uid FROM page_fan WHERE uid=user_id AND page_id=page_id'
	}, function(resp) {
		if (resp.length) {
			alert('A fan!')
		} else {
			alert('Not a fan!');
		}
	}
);
                    </pre>
                    <a class="back_to_top" href="#page-main">
                     Back to top
                    </a>
                    <hr/>
                    <a class="sectionindex" id="notes" name="notes">
                    </a>
                    <h2>
                     Notes
                    </h2>
                    <ul>
                     <li>
                      Always ask for the
                      <code>
                       user_likes
                      </code>
                      <a href="http://developers.facebook.com/docs/authentication/permissions">
                       permission
                      </a>
                      before calling any of the above to make sure you get a correct result
                     </li>
                     <li>
                      Remember when having a valid session, you can use the
                      <code>
                       pages.isFan
                      </code>
                      without the
                      <code>
                       uid
                      </code>
                      parameter and you can use
                      <code>
                       me()
                      </code>
                      instead of the user id when using FQL.
                     </li>
                     <li>
                      For FQL, you can always use
                      <code>
                       https://api.facebook.com/method/fql.query?query=QUERY
                      </code>
                      as mentioned in the
                      <a href="http://developers.facebook.com/docs/reference/fql/">
                       documentation
                      </a>
                     </li>
                    </ul>
