A lot of WordPress themes are built without the support of Gravatars. So, here is some help for you to add Gravatars in your theme.  Open up your comments.php file from the theme folder. Find this piece of code :-
        </p>
        <p>
         <code markup="none">
         </code>
        </p>
        <p>
         Replace the above code with the following code :-
        </p>
        <p>
         <code markup="none">
          <div class="gravs">
           =2.5)
           <br/>
           echo get_avatar( $comment-&gt;comment_author_email, $size = '50', $comment-&gt;comment_author_link);?&gt;

          </div>
          <p>
           <br clear="all"/>
          </p>
         </code>
        </p>
        <p>

        </p>

          .gravs {margin-top:20px;}

        <h3>
         Image Gallery in WordPress
        </h3>
        <p>
         All the versions of WordPress 2.5+ have inbuilt Image Gallery function where you can upload your images in a set and then insert the gallery either into your post or a new page.
        </p>
        <p>
         <a href="http://www..com/wp-content/pop/2008/08/gallery.jpg" rel="nofollow" target="_blank">
          <img alt="" class="alignnone size-full wp-image-726" height="180" src="http://www.blogohblog.com/wp-content/pop/2008/08/gallery.jpg" title="gallery" width="268"/>
         </a>
        </p>
        <p>
         But almost all the old themes(before 2.5) and many new ones do not have the integrated functionality. In order to add this feature, here is what you have to do :-
        </p>
        <p>
         In your existing theme, open single.php and save it as image.php in your theme folder. Now open this image.php file in an editor and find the line that displays the post content. It should be somewhat in the following form. It can differ a bit but the function is called by
         <strong>
          the_content
         </strong>
         like this :-
        </p>
        <p>
         <code markup="none">
         </code>
        </p>
        <p>
         Now insert the following code above the aforementioned code (the_content) :-
        </p>
        <p>
         <code markup="none">
          <p class="attachment">
           <a _blank="" href="&lt;?php echo wp_get_attachment_url($post- target=" rel="nofollow">
            ID); ?&gt;"&gt;ID, 'medium' ); ?&gt;
           </a>
          </p>
          <div class="caption">
           post_excerpt) ) the_excerpt(); // this is the "caption" ?&gt;
          </div>
          <p>
          </p>
         </code>
        </p>
        <p>
         Insert the following code below the aforementioned code (the_content) :-
        </p>
        <p>
         <code markup="none">
          <div class="imgnav">
           <div class="imgleft">
           </div>
           <div class="imgright">
           </div>
          </div>
          <p>
           <br clear="all"/>
          </p>
         </code>
        </p>
        <p>
         Now, add this CSS to your themes style.css file :-
        </p>
        <p>
         <code markup="none">
          /****************Image Gallery *********************/
          <br/>
          .gallery {text-align:center;}
          <br/>
          .gallery img {padding:2px; height:100px; width:100px;}
          <br/>
          .gallery a:hover {background-color:#ffffff;}
          <br/>
          .attachment {text-align:center;}
          <br/>
          .attachment img { padding:2px; border:1px solid #999999;}
          <br/>
          .attachment a:hover {background-color:#FFFFFF;}
          <br/>
          .imgnav {text-align:center;}
          <br/>
          .imgleft {float:left;}
          <br/>
          .imgleft a:hover {background-color:#FFFFFF;}
          <br/>
          .imgleft img{ padding:2px; border:1px solid #999999; height:100px; width:100px;}
          <br/>
          .imgright {float:right;}
          <br/>
          .imgright a:hover {background-color:#FFFFFF;}
          <br/>
          .imgright img{ padding:2px; border:1px solid #999999; height:100px; width:100px;}
         </code>
        </p>
        <p>
         The above CSS code fixes the default gallery of WordPress which doesnt look so good. So, now when you go and upload your images in a post or a page, go to the gallery option (after you have finished uploading all your images) and insert gallery into your post/page. Thats it!
        </p>
        <h3>
         Adding a Subscribe to feed message after every post
        </h3>
        <p>
         <img alt="" class="alignnone size-full wp-image-727" height="112" src="http://www.blogohblog.com/wp-content/pop/2008/08/subscribe.jpg" title="subscribe" width="262"/>
        </p>
        <p>
         Many people use this kind of message to remind the readers to subscribe to their blogs and place them at the end of every post. This can be accomplished by using a simple plugin or you can do the following. If you want the message to show under all your posts on the homepage, open up index.php and just where your content finishes (the_content), add this line :-
        </p>
        <p>
         <code markup="none">
          <div style="padding:5px; border:1px solid #999999; margin-top:10px; background-color:#FFF8AF;">
           If you enjoyed this post, make sure you subscribe to my RSS Feed
          </div>
          <p>
          </p>
         </code>
        </p>
        <p>
         You can do the same for your single.php file in the theme folder which is used to display your individual posts.
        </p>

        </p>

         In a multi-user blog, it can be beneficial to show the Authors Bio under every single post for the users to read about the post author. This can be done using the
         <strong>
          Biographical Info
         </strong>
         in the
         <strong>
          Your profile
         </strong>
         setting under
         <strong>
          Users
         </strong>
         section of WP-Admin.
        </p>
        <p>
         To integrate this bio, open up your single.php file and under
         <strong>
          the_content
         </strong>
         line (as before), add this code :-
        </p>
        <p>
         <code markup="none">
          <div class="author">
          </div>
          <p>
          </p>
         </code>
        </p>
        <p>
         Now, add the following CSS code to your style.css in the theme folder:-
        </p>
        <p>
         <code markup="none">
          .author{
          <br/>
          color: #222222;
          <br/>
          font-family: Arial;
          <br/>
          font-size: 12px;
          <br/>
          border:1px solid #CCCCCC;
          <br/>
          width: 500px;
          <br/>
          padding: 5px;
          <br/>
          margin-top:10px;
          <br/>
          margin-bottom:10px;
          <br/>
          }
         </code>
        </p>
        <p>
         Now if you refresh your individual post pages, you will see the authors info under the post content.
        </p>

        <p>
         To add a good looking drop down that will list all your existing categories, insert the following code in your blog template. You can do it either in your sidebar.php file or anywhere in the index.php. This is the code :-
        </p>
        <p>
         <code markup="none">
          <br/>
          <form action="&lt;?php bloginfo('url'); ?&gt;/" method="get">
           <br/>
           $select = wp_dropdown_categories('show_option_none=Select category&amp;show_count=1&amp;orderby=name&amp;echo=0');
           <br/>
           $select = preg_replace("#
           <select>
            ]*)&gt;#", "
            <select onchange="return this.form.submit()">
             ", $select);
             <br/>
             echo $select;
             <br/>
             ?&gt;
             <br/>
             <noscript>
              <input type="submit" value="View"/>
             </noscript>
             <br/>
            </select>
           </select>
          </form>
          <p>
          </p>
         </code>
        </p>
        <h3>
         Archives drop down
        </h3>
        <p>
         Just like in categories, you can have your monthly archives listed in a drop down. Add the following code to your template files :-
        </p>
        <p>
         <code markup="none">
          <select name='\"archive-dropdown\"' onchange="document.location.href=this.options[this.selectedIndex].value;">
           <option value='\"\"'>
           </option>
          </select>
         </code>
        </p>
        <h3>
         Adding 125125 Ads to your sidebar
        </h3>
        <p>
         Many people do not know how to integrate 125125 banner ads into their theme sidebars. I will explain it to you. Save the following image to your desktop (this is a sample 125125 banner that we will use) :-
        </p>
        <p>
         <img alt="" class="alignnone size-full wp-image-731" height="125" src="http://www.blogohblog.com/wp-content/pop/2008/08/125.gif" title="125" width="125"/>
        </p>
        <p>
         Make a folder called ads in your theme folder (the theme in which you want to add these banners) and place this image (125.gif) in that new folder. Now, add the following code to your sidebar where you want the banners to appear :-
        </p>
        <p>
         <code markup="none">
          <div class="bannerads">
           <div class="ad_125x125">
            <a href="#" rel="nofollow" target="_blank">
             <img alt="Advertising" border="0" src="&lt;?php bloginfo('template_directory'); ?&gt;/ads/125x125.gif"/>
            </a>
           </div>
           <div class="ad_125x125">
            <a href="#" rel="nofollow" target="_blank">
             <img alt="Advertising" border="0" src="&lt;?php bloginfo('template_directory'); ?&gt;/ads/125x125.gif"/>
            </a>
           </div>
          </div>
          <p>
           <!-- bannerads -->
           <br/>
           <br clear="all"/>
          </p>
         </code>
        </p>
        <p>
         Finally, add the following CSS code to your style.css file :-
        </p>
        <p>
         <code markup="none">
          .bannerads {width:270px; margin:10px auto;}
          <br/>
          .ad_125x125 {float:left; margin:0px 5px 10px 5px; width:125px; height:125px;}
         </code>
        </p>
        <p>
         When you refresh your theme, you will see two adjacent 125125 banner ads in your sidebar. You can then change the images as you like and also the hyperlinks.
        </p>
        <h3>
         Displaying most commented (popular) posts
        </h3>
        <p>
         <img alt="" class="alignnone size-full wp-image-732" height="90" src="http://www.blogohblog.com/wp-content/pop/2008/08/popular-posts.jpg" title="popular-posts" width="48"/>
        </p>
        <p>
         There is always an urge in bloggers to showcase their most popular posts or the posts with the most comments. This is how you can do it:-
        </p>
        <p>
         Place the following code at the end of your header.php file in the theme folder. You can change the number of popular post titles that will be pulled from the database by altering the $no_posts variable in the code :-
        </p>
        <p>
         <code markup="none">
          ', $after = '
          <p>
           ', $show_pass_post = false, $duration='') {
           <br/>
           global $wpdb;
           <br/>
           $request = "SELECT ID, post_title, COUNT($wpdb-&gt;comments.comment_post_ID) AS 'comment_count' FROM $wpdb-&gt;posts, $wpdb-&gt;comments";
           <br/>
           $request .= " WHERE comment_approved = '1' AND $wpdb-&gt;posts.ID=$wpdb-&gt;comments.comment_post_ID AND post_status = 'publish'";
           <br/>
           if(!$show_pass_post) $request .= " AND post_password =''";
           <br/>
           if($duration !="") { $request .= " AND DATE_SUB(CURDATE(),INTERVAL ".$duration." DAY) 
}
           <br/>
           $request .= " GROUP BY $wpdb-&gt;comments.comment_post_ID ORDER BY comment_count DESC LIMIT $no_posts";
           <br/>
           $posts = $wpdb-&gt;get_results($request);
           <br/>
           $output = '';
           <br/>
           if ($posts) {
           <br/>
           foreach ($posts as $post) {
           <br/>
           $post_title = stripslashes($post-&gt;post_title);
           <br/>
           $comment_count = $post-&gt;comment_count;
           <br/>
           $permalink = get_permalink($post-&gt;ID);
           <br/>
           $output .= $before . '
           <a href="' . $permalink . '" rel="nofollow" target="_blank" title="' . $post_title.'">
            ' . $post_title . '
           </a>
           (' . $comment_count.')' . $after;
           <br/>
           }
           <br/>
           } else {
           <br/>
           $output .= $before . "None found" . $after;
           <br/>
           }
           <br/>
           echo $output;
           <br/>
           } ?&gt;
          </p>
         </code>
        </p>
        <p>
         Now, wherever you want to display the popular posts list, place the following code. You can do it in the sidebar or the index.php file.
        </p>
        <p>

         </code>
        </p>

              Next I used your Twitter calling code.  Easy enough, I put my latest Twitter message right beside where I already had a Twitter chiclet.
             </p>
             <p>
              Luckily Twitter warned me to put the script call in my WordPress Footer.  That way, if Twitter happens to be down for the day, my page will still load.
             </p>
             <p>
              On your Print tip, I think the thing thats missing is a printable stylesheet.  A lot of times the regular webpage, with ads and colors and borders, is too much to print.  I use Gamerz WP-Print plugin for this.
             </p>
