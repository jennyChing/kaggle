       <p>
        Weve covered tons of
        <a href="http://hacknmod.com/?s=batman">
         batman projects
        </a>
        on Hack N Mod previously and heres another superb build. The final result looks fantastic, allowing you to project any image youd like using
        <br/>
        an Arduino and RGB LEDs. Full
        <a href="http://www.thingiverse.com/thing:1329">
         project build plans
        </a>

       <p>
        <strong>
         Features:
        </strong>
       </p>
       <ul>
        <li>
         <a href="http://hacknmod.com/hack/top-40-arduino-projects-of-the-web/">
          Arduino controlled
         </a>
         so it can blink in different speeds, color fade, etc. and you can reprogram it to project whatever youd like.
        </li>
        <li>
         Three potentiometers to set the red, green and blue LEDs.
        </li>
        <li>
         Interchangable stencils
        </li>
        <li>
         A rotating axis to project onto a wall or the ceiling.
        </li>
       </ul>
