          <div class="postTextContent">
           I own both the mouse (G700s) and the wireless headset. The mouse is great for gaming since the buttons add extra control without ever getting in the way. Most of the time I keep it plugged in since while recharging it works as a wired mouse but when I don't the battery seems to last a while (after I made sure to switch performance on wireless) and works about the same as wired. Only complaint I've had is that it feels a little light, but it problem won't bother most people.
           <br/>
           <br/>
           For the headset I've been using it everyday since I bought it back in December and I've got to say that I really enjoy using it. It drops connection about once everyday but its about a 3 second delay in which it cuts out and then automatically reconnects. Never had the battery die on me as the life of it seems to be about 7 hours. Sound quality is good, not audiophile quality but nobody would expect that with a wireless headset at this price. One complaint I have is that unlike the mouse which works as wired while recharging is this only works as wireless, would've been a nice added bonus but not a deal breaker for me. Another thing I've noticed since I've recently started using a PS4 controller via bluetooth is that the headset disconnects when too close to the controller, someone who uses a lot of bluetooth devices while probably notice a lot more disconnects than I did.
