            <div class="wp-caption aligncenter" id="attachment_511858" style="width: 800px">
             <img alt="Egyptians are poking fun of the ISIS &quot;national anthem&quot;  by uploading videos to YouTube featuring dancers swaying to the song's words " class="size-large wp-image-511858" height="455" src="http://globalvoicesonline.org/wp-content/uploads/2015/03/Screen-Shot-2015-03-02-at-10.54.09-PM-800x455.png" width="800"/>
             <p class="wp-caption-text">
              Egyptians are poking fun of the ISIS national anthem by uploading videos to YouTube featuring dancers swaying to the song's words
             </p>
            </div>
            <p>
             The militant group, ISIS (or Daesh as it is called in Arabic) has its own flag, a self-declared head of state and even, what seems to be: its very own national anthem.
            </p>
            <p>
             Whenever ISIS releases a highly produced and stylized videos of them beheading, bombing or burning someone, they always have their Saleel Al-Sawarem song playing in the background. Saleel Al-Sawarem, in Arabic, literally translates to the clanking of the sharp swords. The lyrics of the song is meant to instill fear in listeners. The words are:
            </p>
            <blockquote>
             <p>
                 
              <br/>
                 
              <br/>
                 
              <br/>
                 
             </p>
            </blockquote>
            <blockquote class="translation">
             <p>
              Clinging of the sharp swords is the anthem for the proud ones.
              <br/>
              Fighting is the way for the life to be lived.
              <br/>
              Between storming and annihilation of tyrants,
              <br/>
              And the beautiful voice of a silenced gun.
             </p>
            </blockquote>
            <p>
             The song is
             <a href="http://www.onmasr.com/videos/86144" target="_blank">
              part of four other releases
             </a>
             , none of which are any less violent. However, recently, the song was turned into a joke among social media users in Egypt.
            </p>
            <p>
             The song is being remixed and made as a background for dancing clips, as a way of mocking the ISIS, the Al-Qaeda affiliate, which has come to control large swathes of Iraq and Syria. In other words, Egyptians are stating that they do not fear them.
            </p>
            <p>
             Karem Farok made the following remix under the name, Saleel Al-Sawarem  the pop version:
            </p>
            <p>
             <iframe allowfullscreen="" frameborder="0" height="366" src="http://www.youtube.com/embed/GZCT6013Skg?feature=oembed" width="650">
             </iframe>
            </p>
            <p>
             Others
             <a href="https://www.youtube.com/watch?v=nrW8IWQjlVE" target="_blank">
              belly-danced
             </a>
             , or made videos of belly-dancers, with the song playing as a background:
            </p>
            <p>
             <iframe allowfullscreen="" frameborder="0" height="366" src="http://www.youtube.com/embed/nrW8IWQjlVE?feature=oembed" width="650">
             </iframe>
            </p>
            <p>
             And some used dance scenes from Egyptian movies with the song in the background:
            </p>
            <p>
             <iframe allowfullscreen="" frameborder="0" height="366" src="http://www.youtube.com/embed/357cbgFdp1c?feature=oembed" width="650">
             </iframe>
            </p>
            <p>
             More videos are available on
             <a href="http://albedaiah.com/news/2015/02/28/83944" target="_blank">
              this link
             </a>
             and
             <a href="https://twitter.com/search?q=%D8%B5%D9%84%D9%8A%D9%84%20%D8%A7%D9%84%D8%B5%D9%88%D8%A7%D8%B1%D9%85&amp;src=typd" target="_blank">
              this Twitter hashtag
             </a>
             .
            </p>
            <p>
             Mocking ISIS and those who side with them is not a new thing. One common rhetoric is that oppression and poverty is the main reason for people in the Arab world to join such groups. Sometimes people go further and use oppression as an excuse for them to do what they do, like with this Twitter user, who goes by the alias Abu Anas:
            </p>
            <blockquote class="twitter-tweet" lang="en">
             <p>
                          ..     !!
              <a href="https://twitter.com/hashtag/%D8%A7%D9%84%D8%AF%D9%88%D9%84%D8%A9_%D8%A7%D9%84%D8%A5%D8%B3%D9%84%D8%A7%D9%85%D9%8A%D8%A9?src=hash">
               #_
              </a>
               
              <a href="https://twitter.com/hashtag/%D8%AF%D8%A7%D8%B9%D8%B4?src=hash">
               #
              </a>
              .
             </p>
             <p>
                (@abuanas572)
              <a href="https://twitter.com/abuanas572/status/572422327885017089">
               March 2, 2015
              </a>
             </p>
            </blockquote>
            <blockquote class="translation">
             <p>
              How come is it that when the oppressed try to fight to lift the unjustise they live in, they are considered as dangerous terrorists. ISIS is the state of pride.
             </p>
            </blockquote>
            <p>
             A few weeks ago, another hashtag 
             <a href="https://twitter.com/hashtag/%D9%87%D8%A7%D9%86%D8%B6%D9%85_%D9%84%D8%AF%D8%A7%D8%B9%D8%B4?src=hash" target="_blank">
              I am going to join Daesh
             </a>
              went viral, where
             <a href="https://twitter.com/youmnanabil2/status/568892314451636225" target="_blank">
              people posted tweets like
             </a>
             , My brother took my charger and my phone's battery is dead now. I am going to join Daesh, in order to ridicule this argument.
            </p>
            <p>
             The mockery is not limited only to Egypt. In the United States, Saturday Night Live had

             </a>
             .
            </p>
            <p>

             </a>

            </p>

		/**
		 * Handle JSON response returned by Form AJAX Request.
		 *
		 * Should be called in gvEsfAjaxSend() as success of ajax request.
		 */
		function gvEsfAjaxSendResponseHandler(jsonResponse) {

			var output = '';

			// Make sure we recieved a response
			if (!jsonResponse) {
				output += '&lt;p class="error"&gt;ERROR: NO AJAX RESPONSE&lt;/p&gt;';
			}

			// Parse the response into an object
			// NOTE: new jQuery version means we don't need to parse JSON any more
			// var response = JSON.parse(jsonResponse);
			var response = jsonResponse;

			// Ghettoize error
			if (response.ghettoize) {
				gvAefGhettoize();
			}

			// If counter is too high then also ghettoize
			if (3 &lt;= attemptCounter) {
				gvAefGhettoize();
			}
			attemptCounter++;

			// Email form error
			if (response.email_form_error) {
				output += '&lt;p class="error"&gt;ERROR! Please fill out your email address, name and message, then resubmit&lt;/p&gt;' ;
			}

			// Captcha Error
			if (response.captcha_error) {
				output += '&lt;p class="error"&gt;ERROR! Incorrect response to test question. Please copy the letters and numbers exactly.&lt;/p&gt;' ;
			}

			// Email Sent message
			if (response.email_sent) {

				// Clear any errors
				$('#email-share-form-messages').empty();
				// Clear the captcha field
				$('#email_address').val('');
				// Clear the recipient email field for next sending
				$('#gv_esf_to_3Rt8').val('');
				// Reset the counter in case they want to send another message for some reason.
				attemptCounter = 0;

				// Add success message to output
				output += '&lt;p class="success"&gt;Email successfully sent&lt;/p&gt;' ;
			}

			// Append any output to the messages div
			$('#email-share-form-messages').append(output);

			// Hide the loading icon
			$('#esf-sending-icon').hide();
		}
		/**
		 * Ghettoize this visitor (send them to a dead end error page)
		 *
		 *  Used to avoid bots retrying the form indefinitely.
		 *
		 * Depends on gv_email_ghetto() php function to serve ghetto
		 */
		function gvAefGhettoize() {
			var message = $('#gv_esf_message').val();
			var messageEscaped = escape(message);
			window.location = '?gv_email_ghetto=1\&amp;email_text=' + messageEscaped;
		}
	});

  <script type="text/javascript">
   jQuery(document).ready(function($) {
		/**
		 * Fixed position sharing tools 
		 * 
		 * Makes sharing tools fixed position only when the viewport is mid-post
		 * 
		 * Loosely based on http://jqueryfordesigners.com/fixed-floating-elements/
		 */
		
		/**
		 * Get the jQuery objects to operate on
		 * 
		 * Run this ONLY ONCE to avoid re-quiering DOM on every scroll+resize event. 
		 */
		// Parent of fixed element, should be absolute positioned.
		var fixedParent = $('.post-header-sharing');
		// Fixed element, should have top:100px; to match fourth argument of update_fixed_element below
		var fixedElement = $('.sharing-tools-floating');
		// Container to match bottom to, must be position:relative
		var container = $('#single-post');

		/**
		 * Update the fixed element for the initial pageload
		 */
		update_fixed_element(fixedParent, fixedElement, container, 100);
		
		/**
		 * When the window is resized, updated the fixed status
		 */
		$(window).resize(function () {			
			update_fixed_element(fixedParent, fixedElement, container, 100);
		});

		// performance-enhancing timer http://ejohn.org/blog/learning-from-twitter/#comment-391945
//		var fixedElementScrollTimer = 0;

		/**
		 * When window is scrolled update the fixed status
		 */
		$(window).scroll(function () {
		
			// Use a buffer so we don't call myCallback too often.
//			if (fixedElementScrollTimer) {
//				clearTimeout(fixedElementScrollTimer);
//			}
//			fixedElementScrollTimer = setTimeout(myCallback, 100);

			update_fixed_element(fixedParent, fixedElement, container, 100);
		});

		/**
		 * Update the position:fixed status of fixedElement inside fixedParent
		 * 
		 * Checks the position of the parent and element in relation to the viewport and
		 * makes fixedElement position:fixed; if it would otherwise have scrolled out
		 * of view.
		 * 
		 * Also disables the fixed position if the element is at the bottom of container, instead
		 * making fixedParent absolute positioned at the bottom of container. 
		 * 
		 * @param object fixedParent jQuery object with results of search for the container div for fixedElement (must be absolute positioned)
		 * @param object fixedElement jQuery object with results of search for the element that will sometimes be position:fixed
		 * @param integer topSpacing Vertical height in pixes of "padding" above fixed element when position:fixed;
		 */
		function update_fixed_element(fixedParent, fixedElement, container, topSpacing) {
			
			// get the distance the viewport has scrolled down the page 
			var scrollPosition = $(window).scrollTop();
	
			// get the distance from the top of screen to the top of the parent element
			fixedParentTop = fixedParent.offset().top;
			
			// Get the bottom position fixed element itself
			fixedElementBottom = fixedElement.offset().top + fixedElement.height();
			
			// Get the bottom position of the container
			containerBottom = container.offset().top + container.height();
						
			// Get the minimum scrollPosition at which we'll make fixedElement fixed
			// i.e. its natural position minus our padding/spacing so that we start early
			fixedScrollPosition = container.offset().top - topSpacing;
			
			// Get the minimum scrollPosition at which we'll stick fixedParent to the bottom of container
			// i.e. the point when fixedElement would be outside container
			bottomScrollPosition = containerBottom - fixedElement.height() - topSpacing;
			
//			console.log('fixedElementBottom:' + fixedElementBottom + ' containerBottom:' + containerBottom + ' fixedScrollPosition:' + fixedScrollPosition + ' scrollPosition:' + scrollPosition + '');
//			console.log('fixedElement.height:' + fixedElement.height());
//			console.log('fixedParent.height:' + fixedParent.height());
//			console.log('fixedParentTop:' + fixedParentTop);
//			console.log('fixedScrollPosition:' + fixedScrollPosition);
//			console.log('bottomScrollPosition:' + bottomScrollPosition);
//			console.log('fixedElementBottom:' + fixedElementBottom);
//			console.log('containerBottom:' + containerBottom);
			
//			fixedParent.css('background', 'purple');
//			fixedElement.css('background', 'pink');
//			container.css('background', 'orange');
						
			/**
			 * If the window scroll is less than the fixed starting position, set it to default positioning
			 */
			if (scrollPosition &lt; fixedScrollPosition) {
				// Make it static (default)
				fixedElement.css('position', 'static');
				// Reset the parent's bottom in case it was set to zero
				fixedParent.css('bottom', 'auto');
				
				//console.log ('scrollPosition (' + scrollPosition + ') &lt; fixedScrollPosition (' + fixedScrollPosition + ') -- make it top');
			
			/**
			 * If we have scrolled passed the bottom position (where fixedElement hits bottom of container) stick fixedParent to bottom
			 */
			}  else if (scrollPosition &gt;= bottomScrollPosition) {
				// Set the element itself to static
				fixedElement.css('position', 'static');
				// Set the absolute-positioned parent to be at the bottom
				fixedParent.css('bottom', '0');

				//console.log ('fixedElementBottom (' + fixedElementBottom + ') &gt;= containerBottom (' + containerBottom + ') -- make it bottom');
			
			/**
			 * Otherwise make fixedElement position:fixed so it can float up and down inside container
			 */
			} else {
				fixedElement.css('position', 'fixed');
				// Reset the absolute-positioned parent's bottom case it was set to 0
				fixedParent.css('bottom', 'auto');

				//console.log ('fixedElementBottom (' + fixedElementBottom + ') &lt; containerBottom (' + containerBottom + ') -- make it fixed');
			};
		}
	}); // end document.ready

		function gv_clone_menu(source_container, target_container, target_css_class, toggle_selector, menu_container_selector, menu_content_selector) {

				// Wrap the entire contents of the container in div.toggle-menu-content
				$(menu_container_selector).contents().wrapAll("&lt;div class='toggle-menu-content'&gt;&lt;/div&gt;");
			}
			return false;
		}

		/**
		 * Toggle the up/down font icon arrow inside parent_selector
		 * 
		 * @param string parent_selector jQuery selector for element that contains the span.icon 
		 */
		function gv_toggle_font_icon_arrow(parent_selector) {
			// Define the font icon classes to add to span
			icon_up_class = 'icon-caret-up';
			icon_down_class = 'icon-caret-down';
			
			// Get the .icon and make sure it exists
			arrow_icon = $(parent_selector).find('.icon');
			if (!arrow_icon.length) {
				return;
			}
				
			// Swap the icon classes
			if (arrow_icon.hasClass(icon_up_class)) {
				arrow_icon.removeClass(icon_up_class);
				arrow_icon.addClass(icon_down_class);
			} else if (arrow_icon.hasClass(icon_down_class)) {
				arrow_icon.removeClass(icon_down_class);
				arrow_icon.addClass(icon_up_class);
			}
		}

	function gv_data_close() {
		var cookies = $.cookie();

		// Scan all data-close elements
		$('[data-close]').each(function(){
			var id = $(this).attr('data-close');

			// Hide the element if gv_hide_$id cookie is present
			if (typeof $.cookie('gv_hide_' + id) !== 'undefined') {
				$('#' + id).hide();
			// Explicitly show the element if the cookie doesn't exist
			} else if (typeof $.cookie('gv_hide_' + id) === 'undefined') {
				$('#' + id).show();
			}
		});

		// Add on click events to all data-close elements
		$('[data-close]').on('click', function() {
			var id = $(this).attr('data-close');

			$.cookie('gv_hide_' + id, true, { path: '/', expires: 30 });
			$('#' + id).hide();
		});
	}
	gv_data_close();

	function gv_wpp_popup_js_fixes() {
		var cookies = $.cookie();
		var wpp_popup = $('[id^=wpp_html_theme-auto_popup-]');
		var wpp_popup_active = false;
		var wpp_popup_cookie_name = null;

		// Checking the length of an element is used to determined if it exists
		if (wpp_popup.length) {
			var match = /^wpp_html_theme-auto_popup-(.+)/im.exec(wpp_popup.attr('id'));

			// Extra validation to make sure we found a match.
			if (null !== match) {
				wpp_popup_cookie_name = 'wpp_popup_' + match[1] + '_closed'
				wpp_popup_active = true;
			}
		}

		// Scan all cookies for WP Popup cookie to see if it is closed. Update flag if found.
		for (var name in cookies) {
			if (wpp_popup_cookie_name === name) {
				wpp_popup_active = false;
				break;
			}
		}

		// Don't hide anything if popup isn't active.
		if (wpp_popup_active === false) {
			return;
		}

		// Scan all data-close elements and close the ID they refer to
		$('[data-close]').each(function(){
			var id = $(this).attr('data-close');
			$('#' + id).hide();
		});
	}
