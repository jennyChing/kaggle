      <h2 class="itemTitle withLeftToolbar">
       How-To: Install A RAMDisk &amp; Accelerate Games, Photoshop, Premiere
      </h2>
      <div class="clr">
      </div>
      <div class="itemHeaderLeft">
       <div class="itemHeader">
        <!-- Item Author -->
        <span class="itemAuthor">
         <span>
          By
         </span>
         <a href="/blog/michael-kerns" rel="author">
          Michael Kerns
         </a>
        </span>
        <!-- Date created -->
        <span class="itemDateCreated">
         <span>
          Published
         </span>
         May 16, 2014at4:18 pm
        </span>
        <div class="clr">
        </div>
        <!-- Plugins: AfterDisplayTitle -->
        <!-- K2 Plugins: K2AfterDisplayTitle -->
       </div>
      </div>
      <div class="left small-16 medium-8 large-8 itemToolbar hide-for-small">
       <ul>
        <!-- Anchor link to comments below - if enabled -->
        <li>
         <!-- K2 Plugins: K2CommentsCounter -->
        </li>
       </ul>
       <div class="clr">
       </div>
      </div>
      <div class="clr">
      </div>
      <div class="itemImageBlockWrapper">
       <div class="itemImageBlock">
        <span class="itemImage">
         <a class="modal" href="http://media.gamersnexus.net/images/media/2013/california/kingston-slider.jpg" rel="{handler: 'image'}" title="Click to preview image">
          <img alt="How-To: Install A RAMDisk &amp; Accelerate Games, Photoshop, Premiere" class="img-responsive" src="http://media.gamersnexus.net/images/media/2013/california/kingston-slider.jpg"/>
         </a>
        </span>
        <div class="clr">
        </div>
       </div>
       <!-- Item category -->
       <div class="itemCategory ">
        <a href="/features/swguides">
         Software Guides
        </a>
       </div>
      </div>
      <div class="itemBodyWrapper">
       <div class="itemBodyInnerWrapper withLeftToolbar">
        <div class="itemBody">
         <!-- Plugins: BeforeDisplayContent -->
         <!-- K2 Plugins: K2BeforeDisplayContent -->
         <!-- Item introtext -->
         <div class="itemIntroText">
          <div class="destopview item" style="float:left;margin-right:5px">
           <!-- /8133643/gn-left-square -->
           <div id="div-gpt-ad-1428686104988-5">
            <script type="text/javascript">
             googletag.cmd.push(function() { googletag.display('div-gpt-ad-1428686104988-5'); });
            </script>
           </div>
          </div>
          <p>
           <span style="background-color: transparent;">
            To most users, RAM is simply RAM; we put it in the computer and go about our business, for the most part. Rendering loves as much as it can get, gaming needs a couple gigabytes, mainstream use needs about 2-4GBs, and so forth. RAM is fairly abundant in gaming PC builds --
           </span>
           <a href="/pcbuilds" style="background-color: transparent;">
            most of our guides
           </a>
           <span style="background-color: transparent;">
            suggest
           </span>
           <a href="http://www.amazon.com/Kingston-1600MHz-240-pin-KHX1600C10D3B1-8G/dp/B007TTEHUY/" style="background-color: transparent;">
            8GB kits
           </a>
           <span style="background-color: transparent;">
            -- but that still leaves a lot of RAM left over for other purposes. As for what users can do with extra RAM, weve got a few options that can put it to work -- the one were talking about today is in the form of a RAMDisk.
           </span>
          </p>
          <p>
           <a class="jcepopup" href="http://media.gamersnexus.net/images/media/2013/california/kingston-slider.jpg" target="_blank">
            <span style="background-color: transparent;">
            </span>
           </a>
          </p>
          <p dir="ltr">
           In this software guide, well answer what a RAMDisk is, talk about the advantages of a RAMDisk, how to install/create a RAMDisk, and alternatives to popular options - like AMDs RAMDisk. The next article in this installment will compare / contrast multiple RAMDisks with one another, including AMDs, which has
           <a href="/news/1412-amd-never-settle-forever-refresh">
            recently been bundled with video cards
           </a>
           .
          </p>
          <p dir="ltr">
          </p>
         </div>
         <!-- Item fulltext -->
         <div class="itemFullText lamvt_contents">
          <h3 dir="ltr">
           What is a RAMDisk?
          </h3>
          <p dir="ltr">
           A RAMDisk is a virtual drive that uses RAM to store information. This can be thought of as the RAM pretending to be an incredibly fast hard drive so you can install non-volatile games/programs on it or make it a scratch disk.
          </p>
          <p dir="ltr">
           In the simplest of terms, a RAMDisk is RAM in a hard drive costume. The RAM is still volatile, but builds and maintains an index throughout user-defined intervals in use; it will write to an index on non-volatile storage to keep data permanently.
          </p>
          <p dir="ltr">
           To put things into perspective: 1866MHz RAM has a peak data rate of approximately 15GB/s, so even matched against the worlds fastest consumer-class SSD (or enterprise-class, for that matter), RAM will far-and-away outperform it.
          </p>
          <p dir="ltr">
           <a class="jcepopup" href="http://media.gamersnexus.net/images/media/2013/california/amd-ramdisk-1.jpg" target="_blank">
            <img alt="amd-ramdisk-1" height="383" src="http://media.gamersnexus.net/images/media/2013/california/amd-ramdisk-1.jpg" style="display: block; margin-left: auto; margin-right: auto;" width="689"/>
           </a>
          </p>
          <h3 dir="ltr">
           Advantages/disadvantages of a RAMDisk
          </h3>
          <p dir="ltr">
           A RAMDisk offers quite a few unique advantages and disadvantages. The largest advantage to a RAMDisk is the 100-150x transfer speed boost over HDDs and even ~25x over SSDs (goodbye loading screens!). Because RAM does not wear noticeably from I/O like regular drives do, it can be helpful to people who want to boost performance by using it in a read/write intensive situation, like as a scratch disk for Photoshop or Premiere.
          </p>
          <p dir="ltr">
           But RAMDisks are not without their fair share of disadvantages. The most immediate is decreasing the amount of available RAM to the host system; further, in the event of power loss, data on the RAMDisk that has not been backed-up will be lost in shut-down. RAMDisks can also drastically increase shutdown time if you are saving to an image on an HDD or SSD for permanent storage; if youre loading that image on startup, it can drastically increase boot time.
          </p>
          <p dir="ltr">
           These advantages and disadvantages should be taken into consideration before creating a RAMDisk. Some disadvantages can be mitigated through creative implementation and careful indexing, but theyre still present.
          </p>
          <h3 dir="ltr">
           Unique Software Implementations
          </h3>
          <p dir="ltr">
           Different RAMDisk software takes different creative approaches toward the disadvantages of RAMDisks to try mitigate them as much as possible. One of these techniques is saving to an image so the user can retain data stored on the RAMDisk at shutdown. This is a fairly simple concept and some software even allows for auto-saving every X minutes (X is set by the user). But some software, like AMDs RAMDisk, takes this a step further and keeps a running tab (record) of what is on the image; AMDs RAMDisk then analyzes what, if anything, has changed and will only rewrite changes while leaving static data untouched. This leads to less wear on host drives and shutdowns taking much less time.
          </p>
          <p dir="ltr">
           Another approach is featured by
           <a href="http://www.ltr-data.se/opencode.html/">
            Imdisk
           </a>
           , yet another RAMDisk solution. Imdisk uses dynamic RAMDisk scaling; in dynamic RAMDisk scaling, the user sets the max capacity limit for the RAMDisk, then as the user puts more or less data onto the RAMDisk, itll change the amount of RAM reserved. More RAM is accessible if your RAMDisk is empty, less if it is filling up.
          </p>
          <p dir="ltr">
           <a href="http://lse.sourceforge.net/numa/faq/">
            NUMA
           </a>
           can be useful for large server arrays, but for the majority of RAMDisk users, it will be completely irrelevant.
          </p>
          <h3 dir="ltr">
           Who can best use RAMDisks?
          </h3>
          <p dir="ltr">
           RAMDisks are not usually for the average user. They appeal more to power users who want them for specific uses. Gamers with lots of RAM can benefit from the much lower loading times, users doing I/O-intensive tasks can also benefit from using a RAMDisk for quick retrieval of data or a scratch-disk, but outside of this, most users dont need or want a RAMDisk.
          </p>
          <p dir="ltr">
           Weve got a series of RAMDisk benchmarks coming up in our next article on this topic, but as a bit of a spoiler, weve seen load times decrease as much as 2x-3x in games like PlanetSide 2 when using a RAMDisk.
          </p>
          <p dir="ltr">
           Even if a user doesnt have a purpose for a RAMDisk, I still suggest playing around with them just to learn about the process.
          </p>
          <h3 dir="ltr">
           How to Create &amp; Install a RAMDisk
          </h3>
          <p dir="ltr">
           You don't need to understand any complicated programming or scripting to create a RAMDisk, with thanks to programs like
           <a href="http://www.amd.com/en-us/products/memory/ramdisk">
            AMD RAMDisk
           </a>
           ,
           <a href="http://www.softperfect.com/products/ramdisk/">
            SoftPerfect RAM Disk
           </a>
           ,
           <a href="http://www.ltr-data.se/opencode.html/">
            ImDisk
           </a>
           , and many others. Creating a RAMDisk has become something quite simple and user-friendly.
          </p>
          <p dir="ltr">
           To create a RAMDisk, you need to first download and install RAMDisk software. In this walkthrough, we will be using
           <a href="http://www.softperfect.com/products/ramdisk/">
            SoftPerfect RAM Disk
           </a>
           since its free, fairly easy to use, and is my personal favorite. In this tutorial we will show how to make a RAMDisk that starts on boot-up and saves to an image.
          </p>
          <p dir="ltr">
           First, in
           <a href="http://www.softperfect.com/products/ramdisk/">
            SoftPerfect RAM Disk
           </a>
           , go to image at the top and then select Create Image.
          </p>
          <p dir="ltr">
           <a class="jcepopup" href="http://media.gamersnexus.net/images/media/2014/guides/ramdisk-1.jpg" target="_blank">
            <img alt="ramdisk-1" height="230" src="http://media.gamersnexus.net/images/media/2014/guides/ramdisk-1.jpg" style="display: block; margin-left: auto; margin-right: auto;" width="361"/>
           </a>
          </p>
          <p dir="ltr">
           Type your image name, define where it will be stored, and set its size in MB. I generally dont change the image type or file system.
          </p>
          <p>
           <span style="background-color: transparent;">
            Click Disk at the top after creating the image. Select Add Boot Disk and in this menu choose the image you just made (your RAMDisk will be the size of the image automatically), the drive letter, and make sure to check Save Contents to Image so you dont lose any data you place on your RAMDisk.
           </span>
          </p>
          <p dir="ltr">
           <a class="jcepopup" href="http://media.gamersnexus.net/images/media/2014/guides/ramdisk-2.jpg" target="_blank">
            <img alt="ramdisk-2" height="460" src="http://media.gamersnexus.net/images/media/2014/guides/ramdisk-2.jpg" style="display: block; margin-left: auto; margin-right: auto;" width="412"/>
           </a>
          </p>
          <p dir="ltr">
           Clicking advanced brings up a number of options. Most users need not worry about many of them except the option to save the contents of the RAMDisk to an image every x minutes. This will ensure there is no data loss in the event of a power surge.
          </p>
          <p dir="ltr">
           For servers,
           <a href="http://lse.sourceforge.net/numa/faq/">
            NUMA
           </a>
           may be be useful, but for the majority of users it wont be and can be ignored. Now simply click OK out of those two windows and your RAMDisk is made! Navigate to Tools and then Settings to check that SoftPerfect RAMDisk is set to launch automatically with Windows so itll automatically load on boot. You can now install programs on your RAMDisk and use it however you see fit. I personally find that file-intensive games like Planetside 2 benefit greatly from being on a RAMDisk.
          </p>
          <p dir="ltr">
           Our next article on RAMDisks will include benchmarks for RAMDisks and how they affect performance.
          </p>
          <p dir="ltr">
           - Michael The Bear Kerns.
          </p>
