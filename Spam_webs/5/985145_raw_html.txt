    <div class="yui3-u" id="grub-rail-leftRailSpanner" itemscope="" itemtype="http://schema.org/Recipe">
     <h1 class="grub-topOfColumn" itemprop="name" style="font-size: 27px;">
      Frosty Mexican Bulldog Margarita
     </h1>
     <img height="795" src="http://cookingstoned.tv/wp-content/uploads/2013/01/Frosty-Mexican-Bulldog-Margarita-790x795.jpg" width="790"/>
     <div class="yui3-g">
      <div class="yui3-u" id="grub-rail-well">
       <div class="grub-byline" style="margin-top: 1em;">
        Recipe by
        <a href="https://plus.google.com/113274240511284571445" rel="author">
         Jerry James Stone
        </a>
         January 26, 2013
       </div>
       <div class="grub-mainPostContent" itemprop="recipeInstructions" not-style="font-size: 16px; line-height: 23px;">
        <p>
         <a class="grub-popupPhotoLink" grub-popupphotoheight="1007" grub-popupphotopath="/recipe/frosty-mexican-bulldog-margarita/frosty-mexican-bulldog-margarita/" grub-popupphotowidth="1000" href="#photo/frosty-mexican-bulldog-margarita" rel="attachment wp-att-1302">
         </a>
        </p>
        <p>
         One of the best parts about traveling is the exposure to new and exciting things. And it doesnt always require busting out your passport. This little gem comes from one of my many trips to Texas. Austin, Texas to be specific. I am a SXSW regular, you see, and this little puppy has kicked off many a tequila-filled nights while there. I hope you enjoy it as much as I do.
        </p>
        <p>
        </p>
        <div id="grub_inlinePlayer_4RSe_WEHV5s" style="width: 600px; height: 338px;">
        </div>
        <div class="yui3-g" id="grub-recipeSummary">
         <div class="yui3-u" id="grub-recipeSummary-left">
          <h2>
           Ingredients
          </h2>
          <ul id="grub-ingredients">
           <li itemprop="ingredients">
            5 Cups Ice
           </li>
           <li itemprop="ingredients">
            1 Cup Tequila
           </li>
           <li itemprop="ingredients">
             Cup Grand Marnier
           </li>
           <li itemprop="ingredients">
            5 Limes, more for garnish
           </li>
           <li itemprop="ingredients">
            1 Orange
           </li>
           <li itemprop="ingredients">
            1 Cup
            <a href="http://cookingstoned.tv/recipe/grapefruit-basil-vodka-popsicles/">
             simple syrup
            </a>
           </li>
           <li itemprop="ingredients">
            6 Beer, Corona
           </li>
           <li itemprop="ingredients">
             Cup coarse Salt
           </li>
          </ul>
         </div>
         <div class="yui3-u" id="grub-recipeSummary-right">
          <h2>
           <a class="grub-button" href="http://cookingstoned.tv/recipe/frosty-mexican-bulldog-margarita/printable/" target="_blank">
            <span class="grub-printIcon">
            </span>
            Printable View
           </a>
          </h2>
          <h2>
           Yield:
           <span itemprop="recipeYield">
            6 servings
           </span>
          </h2>
          <meta content="PT10M" itemprop="totalTime"/>
          <h2>
           Total Time
           <div style="float: right;">
            10 minutes
           </div>
          </h2>
          <meta content="PT10M" itemprop="prepTime"/>
          <div style="margin-bottom: 3px;">
           Prep time
           <div style="float: right">
            10 minutes
           </div>
          </div>
          <div class="grub-recipeTaxonomyList">
           <h2>
            Main Ingredients:
           </h2>
           Beer, Tequila
          </div>
          <div class="grub-recipeTaxonomyList">
           <h2>
            Recipe Type:
           </h2>
           <a href="http://cookingstoned.tv/recipe-type/cocktail/">
            Cocktail
           </a>
          </div>
          <div class="grub-recipeTaxonomyList">
           <h2>
            Diet:
           </h2>
           <a href="http://cookingstoned.tv/diet/vegan/">
            Vegan
           </a>
          </div>
         </div>
        </div>
        <p>
         1. Place 5 limes and 1 orangewhich should be at room temperatureon the counter and roll them with your hand, back and forth, while applying pressure. Be careful, you dont want to break them open, do it semi-gently. Theyll feel like a sem-filled water balloon when ready. This way the citrus will yield more liquid when you juice them.
        </p>
        <p>
         <a class="grub-popupPhotoLink" grub-popupphotoheight="666" grub-popupphotopath="/recipe/frosty-mexican-bulldog-margarita/blender-for-margarita/" grub-popupphotowidth="1000" href="#photo/blender-for-margarita" rel="attachment wp-att-1299">
          <img alt="Blender for Margarita" class="alignnone size-grub-wellWidth wp-image-1299" height="399" src="http://cookingstoned.tv/wp-content/uploads/2013/01/Blender-for-Margarita-600x399.jpg" title="Blender for Margarita" width="600"/>
         </a>
        </p>
        <p>
         2. Juice the limes and the orange in a large measuring cup, then add in 1 cup of tequila, 1/2 cup of grand marnier, and simple syrup. Mix together.
        </p>
        <p>
         <a class="grub-popupPhotoLink" grub-popupphotoheight="666" grub-popupphotopath="/recipe/frosty-mexican-bulldog-margarita/making-a-frozen-margarita/" grub-popupphotowidth="1000" href="#photo/making-a-frozen-margarita" rel="attachment wp-att-1303">
          <img alt="Making a Frozen Margarita" class="alignnone size-grub-wellWidth wp-image-1303" height="399" src="http://cookingstoned.tv/wp-content/uploads/2013/01/Making-a-Frozen-Margarita-600x399.jpg" title="Making a Frozen Margarita" width="600"/>
         </a>
        </p>
        <p>
         3. Pour 1/3 cup of coarse salt into a shallow bowl. Using one of the juiced limes, swipe the outer rim of one of your glasses. Swivel the glass upside down in the salt until the rim is covered. Repeat for each glass.
        </p>
        <p>
         <a class="grub-popupPhotoLink" grub-popupphotoheight="666" grub-popupphotopath="/recipe/frosty-mexican-bulldog-margarita/salted-glass-for-margarita/" grub-popupphotowidth="1000" href="#photo/salted-glass-for-margarita" rel="attachment wp-att-1305">
          <img alt="Salted Glass for Margarita" class="alignnone size-grub-wellWidth wp-image-1305" height="399" src="http://cookingstoned.tv/wp-content/uploads/2013/01/Salted-Glass-for-Margarita-600x399.jpg" title="Salted Glass for Margarita" width="600"/>
         </a>
        </p>
        <p>
         4. Add a heaping cup of ice to the blender and just enough of the margarita mixture that the ice is not covered. Blend, adding more ice or margarita mix until the desired thickness is achieved.
        </p>
        <p>
         4. Pour the frozen margarita into a salt-rimmed glass, pop open a Corona, and place it upside down in the glass, submerging it into the margarita. Repeat for each glass.
        </p>
        <p>
         <a class="grub-popupPhotoLink" grub-popupphotoheight="666" grub-popupphotopath="/recipe/frosty-mexican-bulldog-margarita/top/" grub-popupphotowidth="1000" href="#photo/top" rel="attachment wp-att-1306">
          <img alt="Top" class="alignnone size-grub-wellWidth wp-image-1306" height="399" src="http://cookingstoned.tv/wp-content/uploads/2013/01/Top-600x399.jpg" title="Top" width="600"/>
         </a>
        </p>
        <p>
         Now get that party started and enjoy!
        </p>

         </a>

function _grub_renderInlinePlayer(videoID, width, height) {
    var playerState = 'none', playbackStage = 'loading',
        playbackPollInterval = null, tsIntro, tsContentHalfWay, tsOuttro;

    var onStateChange = function(evt) {
        // We take a somewhat simplified view, ignoring some states (buffering, paused) and only
        // tracking that the video has started or finished

        if (evt.data == YT.PlayerState.PLAYING) {
            if (playerState == 'none') {
                _gaq.push(['_trackEvent', 'Video', 'play', 'inline']);

                playbackPollInterval = setInterval(playbackPollIntervalTick, 500);
            }

            playerState = 'playing';
        } else if (evt.data == YT.PlayerState.ENDED) {
            if (playerState == 'playing') {
                _gaq.push(['_trackEvent', 'Video', 'finish', 'inline']);
                
                if (playbackPollInterval) {
                    clearInterval(playbackPollInterval);
                    playbackPollInterval = null;
                }
            }

            playerState = 'ended';
        }
    };

    var player = new YT.Player('grub_inlinePlayer_' + videoID, {
        width: width,
        height: height,
        videoId: videoID,
        events: {
            onStateChange: onStateChange
        }
    });

    var playbackPollIntervalTick = function() {
        if (playbackStage == 'loading') {
            var duration = player.getDuration();

            if (duration == 0) {
                return;
            } else {
                playbackStage = 'beginning';
                tsIntro = 6;
                tsContentHalfWay = 6 + ((duration - 28) / 2);
                tsOuttro = duration - 22.5;
            }
        }

        var newPlaybackStage, currentTime = player.getCurrentTime();

        if (currentTime &gt; tsOuttro) {
            newPlaybackStage = 'Outtro';
        } else if (currentTime &gt; tsContentHalfWay) {
            newPlaybackStage = 'HalfWay';
        } else if (currentTime &gt; tsIntro) {
            newPlaybackStage = 'Content';
        } else {
            newPlaybackStage = 'beginning';
        }

        if (playbackStage != newPlaybackStage) {
            playbackStage = newPlaybackStage;

            _gaq.push(['_trackEvent', 'Video', 'hit' + newPlaybackStage, 'inline']);
        }
    };
}
