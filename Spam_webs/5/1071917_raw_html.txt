   <h1 class="firstHeading" id="firstHeading" lang="en">
    Unconventional computing
   </h1>

     <p>
      <b>
       Unconventional computing
      </b>
      is
      <a href="/wiki/Computing" title="Computing">
       computing
      </a>
      by a wide range of new or unusual methods. It is also known as
      <b>
       alternative computing
      </b>
      . The different methods of unconventional computing include
      <a href="/wiki/Optical_computing" title="Optical computing">
       optical computing
      </a>
      ,
      <a href="/wiki/Quantum_computing" title="Quantum computing">
       quantum computing
      </a>
      ,
      <a href="/wiki/Chemical_computer" title="Chemical computer">
       chemical computing
      </a>
      ,
      <a class="mw-redirect" href="/wiki/Natural_computation" title="Natural computation">
       natural computing
      </a>
      ,
      <a class="mw-redirect" href="/wiki/Biologically-inspired_computing" title="Biologically-inspired computing">
       biologically-inspired computing
      </a>
      ,
      <a href="/wiki/Wetware_computer" title="Wetware computer">
       wetware computing
      </a>
      ,
      <a class="mw-redirect" href="/wiki/DNA_computer" title="DNA computer">
       DNA computing
      </a>
      ,
      <a class="mw-redirect" href="/wiki/Molecular_computer" title="Molecular computer">
       molecular computing
      </a>
      ,
      <a href="/wiki/Amorphous_computing" title="Amorphous computing">
       amorphous computing
      </a>
      ,
      <a href="/wiki/Nanocomputer" title="Nanocomputer">
       nanocomputing
      </a>
      ,
      <a href="/wiki/Reversible_computing" title="Reversible computing">
       reversible computing
      </a>
      ,
      <a href="/wiki/Ternary_computer" title="Ternary computer">
       ternary computing
      </a>
      ,
      <a href="/wiki/Fluidics" title="Fluidics">
       fluidics
      </a>
      ,
      <a class="mw-redirect" href="/wiki/Analogue_computer" title="Analogue computer">
       analogue computing
      </a>
      ,
      <a href="/wiki/Human_computer" title="Human computer">
       human
      </a>
      and
      <a href="/wiki/Domino_computer" title="Domino computer">
       domino computation
      </a>
      .
     </p>
     <p>
      Historically,
      <a href="/wiki/Mechanical_computer" title="Mechanical computer">
       mechanical computers
      </a>
      were used in industry before the advent of the
      <a href="/wiki/Transistor" title="Transistor">
       transistor
      </a>
      . Mechanical computers retain some interest today both in research and as analogue computers. Some mechanical computers have a theoretical or didactic relevance, such as
      <a href="/wiki/Billiard-ball_computer" title="Billiard-ball computer">
       billiard-ball computers
      </a>
      or hydraulic ones,.
      <sup class="reference" id="cite_ref-pen-empnew_1-0">
       <a href="#cite_note-pen-empnew-1">
        <span>
         [
        </span>
        1
        <span>
         ]
        </span>
       </a>
      </sup>
      While some are actually simulated, others are not; no attempt is made to build a functioning computer through the mechanical collisions of billiard balls. The
      <a href="/wiki/Domino_computer" title="Domino computer">
       domino computer
      </a>
      is another theoretically interesting mechanical computing scheme.
     </p>
     <p>
      Unconventional computing is, according to a recent conference description,
      <sup class="reference" id="cite_ref-2">
       <a href="#cite_note-2">
        <span>
         [
        </span>
        2
        <span>
         ]
        </span>
       </a>
      </sup>
      "an interdisciplinary research area with the main goal to enrich or go beyond the standard models, such as the
      <a href="/wiki/Von_Neumann_architecture" title="Von Neumann architecture">
       Von Neumann computer architecture
      </a>
      and the
      <a href="/wiki/Turing_machine" title="Turing machine">
       Turing machine
      </a>
      , which have dominated computer science for more than half a century". These methods model their computational operations based on non-standard paradigms, and are currently mostly in the research and development stage. This computing behavior can be "simulated" using the classical silicon-based micro-transistors or
      <a class="mw-redirect" href="/wiki/Solid_state_(electronics)" title="Solid state (electronics)">
       solid state
      </a>
      computing technologies, but aim to achieve a new kind of computing engineering inspired in nature.
     </p>
     <h2>
      <span class="mw-headline" id="Implementations">
       Implementations
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Unconventional_computing&amp;action=edit&amp;section=1" title="Edit section: Implementations">

       </span>
      </span>
     </h2>
     <ul>
      <li>
       <a href="/wiki/Billiard_ball" title="Billiard ball">
        Billiard balls
       </a>
       (
       <a class="mw-redirect" href="/wiki/Billiard_ball_computer" title="Billiard ball computer">
        billiard ball computer
       </a>
       )
      </li>
     </ul>
     <dl>
      <dd>
       This is an unintuitive and pedagogical example that a computer can be made out of almost anything.
      </dd>
     </dl>
     <ul>
      <li>
       <a href="/wiki/Light" title="Light">
        Light
       </a>
       (
       <a href="/wiki/Optical_computing" title="Optical computing">
        optical computing
       </a>
       )
      </li>
     </ul>
     <dl>
      <dd>
       Computers can manipulate information as light (rather than electricity or billiard balls).
      </dd>
     </dl>
     <ul>
      <li>
       <a class="mw-redirect" href="/wiki/Molecules" title="Molecules">
        Molecules
       </a>
       (
       <a href="/wiki/DNA_computing" title="DNA computing">
        DNA computing
       </a>
       ,
       <a class="mw-redirect" href="/wiki/Chemical_computing" title="Chemical computing">
        chemical computing
       </a>
       )
      </li>
      <li>
       Gears, levels, dials, etc. (
       <a href="/wiki/Mechanical_computer" title="Mechanical computer">
        mechanical computer
       </a>
       )
      </li>
      <li>
       Continuous-valued electronics or fluidics, etc. (e.g.
       <a href="/wiki/Analog_computer" title="Analog computer">
        analog computer
       </a>
       although these may also be implemented mechanically)
      </li>
      <li>
       <a class="mw-redirect" href="/wiki/Neurons" title="Neurons">
        Neurons
       </a>
       (
       <a href="/wiki/Wetware_computer" title="Wetware computer">
        wetware computer
       </a>
       )
      </li>
      <li>
       <a href="/wiki/Fluid" title="Fluid">
        Fluid
       </a>
       (
       <a href="/wiki/Fluidics" title="Fluidics">
        fluidics
       </a>
       )
      </li>
      <li>
       Software agents acting under a special set of rules (e.g.
       <a class="mw-redirect" href="/wiki/Cellular_automata" title="Cellular automata">
        cellular automata
       </a>
       )
      </li>
      <li>
       <a href="/wiki/Quantum_mechanics" title="Quantum mechanics">
        Quantum mechanics
       </a>
       (
       <a href="/wiki/Quantum_computing" title="Quantum computing">
        quantum computing
       </a>
       )
      </li>
      <li>
       <a href="/wiki/Human_computer" title="Human computer">
        Human computer
       </a>
      </li>
      <li>
       <a href="/wiki/Chaos_computing" title="Chaos computing">
        Chaos computing
       </a>
      </li>
      <li>
       <a href="/wiki/Stochastic_computing" title="Stochastic computing">
        Stochastic computing
       </a>
      </li>
     </ul>
