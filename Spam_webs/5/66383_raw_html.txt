             <meta content="http://3.bp.blogspot.com/_hQy6EE5XJg8/SKcJejvv9xI/AAAAAAAABrk/6vn1_iiNBpY/s320/DSC06502.JPG" itemprop="image_url"/>

             <h3 class="post-title entry-title" itemprop="name">
              Acrylic Paint Transfer
             </h3>
             <div class="post-header">
              <div class="post-header-line-1">
              </div>
             </div>
             <div class="post-body entry-content" id="post-body-8659077451717128031" itemprop="description articleBody">
              <span style="font-family:arial;">
               <strong>
                Supplies needed: Acrylic dabbers, photocopy of an image, water spritzer bottle, paintbrush, card stock, craft sheet and heat tool (optional)
               </strong>
              </span>
              <br/>
              <span style="font-family:arial;">
               <strong>
               </strong>
              </span>
              <br/>
              <a href="http://3.bp.blogspot.com/_hQy6EE5XJg8/SKcJejvv9xI/AAAAAAAABrk/6vn1_iiNBpY/s1600-h/DSC06502.JPG">
               <img alt="" border="0" id="BLOGGER_PHOTO_ID_5235163512437733138" src="http://3.bp.blogspot.com/_hQy6EE5XJg8/SKcJejvv9xI/AAAAAAAABrk/6vn1_iiNBpY/s320/DSC06502.JPG" style="CURSOR: hand"/>
              </a>
              <br/>
              <br/>
              <strong>
               <span style="font-family:arial;">
                Take the lid off the dabbers and brush the paint onto the card stock, ensure you get a good coverage
                <br/>
               </span>
              </strong>
              <a href="http://1.bp.blogspot.com/_hQy6EE5XJg8/SKcJe1joPTI/AAAAAAAABrs/0QQPVDdyphU/s1600-h/DSC06506.JPG">
               <img alt="" border="0" id="BLOGGER_PHOTO_ID_5235163517218733362" src="http://1.bp.blogspot.com/_hQy6EE5XJg8/SKcJe1joPTI/AAAAAAAABrs/0QQPVDdyphU/s320/DSC06506.JPG" style="CURSOR: hand"/>
              </a>
              <br/>
              <br/>
              <strong>
               <span style="font-family:arial;">
                Take your photocopied image ( remember that you will get a reverse of the image, so don't use bold words), flip it over and place it in the acrylic. Lightly press it down making sure it's smooth and not wrinkled
               </span>
              </strong>
              <br/>
              <a href="http://4.bp.blogspot.com/_hQy6EE5XJg8/SKcJfIGGIeI/AAAAAAAABr0/QhyvVm3GJ74/s1600-h/DSC06507.JPG">
               <img alt="" border="0" id="BLOGGER_PHOTO_ID_5235163522195136994" src="http://4.bp.blogspot.com/_hQy6EE5XJg8/SKcJfIGGIeI/AAAAAAAABr0/QhyvVm3GJ74/s320/DSC06507.JPG" style="CURSOR: hand"/>
              </a>
              <br/>
              <br/>
              <strong>
               <span style="font-family:arial;">
                Leave to air dry for at least 15 minutes and then if you wish give it a blast with the heat tool
               </span>
              </strong>
              <br/>
              <a href="http://1.bp.blogspot.com/_hQy6EE5XJg8/SKcJfWladGI/AAAAAAAABr8/6xVQMv_8kiE/s1600-h/DSC06510.JPG">
               <img alt="" border="0" id="BLOGGER_PHOTO_ID_5235163526084588642" src="http://1.bp.blogspot.com/_hQy6EE5XJg8/SKcJfWladGI/AAAAAAAABr8/6xVQMv_8kiE/s320/DSC06510.JPG" style="CURSOR: hand"/>
              </a>
              <br/>
              <br/>
              <strong>
               <span style="font-family:arial;">
                Only move onto this step when you are sure your paint is completely dry
               </span>
              </strong>
              <br/>
              <strong>
               <span style="font-family:Arial;">
               </span>
              </strong>
              <br/>
              <strong>
               <span style="font-family:Arial;">
                Take your water filled spritzer bottle, spray the back of the paper no more than two squirts, you don't want it too wet
               </span>
              </strong>
              <br/>
              <a href="http://1.bp.blogspot.com/_hQy6EE5XJg8/SKcIgbgG2YI/AAAAAAAABrE/zcd_dyurbRk/s1600-h/DSC06514.JPG">
               <img alt="" border="0" id="BLOGGER_PHOTO_ID_5235162445072750978" src="http://1.bp.blogspot.com/_hQy6EE5XJg8/SKcIgbgG2YI/AAAAAAAABrE/zcd_dyurbRk/s320/DSC06514.JPG" style="CURSOR: hand"/>
              </a>
              <br/>
              <br/>
              <strong>
               <span style="font-family:arial;">
                Next start to rub the paper very gently with your finger
               </span>
              </strong>
              <br/>
              <a href="http://4.bp.blogspot.com/_hQy6EE5XJg8/SKcIg_g3vlI/AAAAAAAABrM/tfC9PV6u4TY/s1600-h/DSC06518.JPG">
               <img alt="" border="0" id="BLOGGER_PHOTO_ID_5235162454739631698" src="http://4.bp.blogspot.com/_hQy6EE5XJg8/SKcIg_g3vlI/AAAAAAAABrM/tfC9PV6u4TY/s320/DSC06518.JPG" style="CURSOR: hand"/>
              </a>
              <br/>
              <br/>
              <strong>
               <span style="font-family:arial;">
                Keep rubbing and extra spritzing if you need too
               </span>
              </strong>
              <br/>
              <a href="http://2.bp.blogspot.com/_hQy6EE5XJg8/SKcIhOUsxpI/AAAAAAAABrU/hgizFUDoXLM/s1600-h/DSC06519.JPG">
               <img alt="" border="0" id="BLOGGER_PHOTO_ID_5235162458715113106" src="http://2.bp.blogspot.com/_hQy6EE5XJg8/SKcIhOUsxpI/AAAAAAAABrU/hgizFUDoXLM/s320/DSC06519.JPG" style="CURSOR: hand"/>
              </a>
              <br/>
              <br/>
              <strong>
               <span style="font-family:arial;">
                Eventually you will get rid of all the paper, but it does take a bit of patience as you have to be careful not to wet it too much and rub the image away
               </span>
              </strong>
              <br/>
              <a href="http://3.bp.blogspot.com/_hQy6EE5XJg8/SKcIhVpg6kI/AAAAAAAABrc/-dsDu92wmlU/s1600-h/DSC06522.JPG">
               <img alt="" border="0" id="BLOGGER_PHOTO_ID_5235162460681464386" src="http://3.bp.blogspot.com/_hQy6EE5XJg8/SKcIhVpg6kI/AAAAAAAABrc/-dsDu92wmlU/s320/DSC06522.JPG" style="CURSOR: hand"/>
              </a>
              <br/>
              <br/>
              <strong>
               <span style="font-family:arial;">
                You can do exactly the same on a canvas
               </span>
              </strong>
              <br/>
