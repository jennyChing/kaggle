          <h1 class="h1black" style="margin-bottom:12px;">
           Japan Loves Mama Chari
          </h1>
          <div id="messagebox">
          </div>
          <span class="h5grey">
           posted by
           <a href="http://www.japan-talk.com/jt/new/johnSpacey" rel="author">
            John Spacey
           </a>
           , June 23, 2012
          </span>
          <div class="readability">
           <span class="blogy">
            <br/>
            One of the great things about Japan (and Asia in general) is how practical everyone is.
            <br/>
            <br/>
            Case in point: bicycles.   Japan loves Mama Chari.   Mama Chari are sturdy, reliable unisex bicycles with a basket.
            <br/>
            <br/>
            <img alt="" src="http://www.japan-talk.com/images/jt/thumbnails/girl-on-bike-japan-734.jpg"/>
            <br/>
            <br/>
            Mama Chari is a Japanese term that somehow sprang from the English "Mother's Chariot".
            <br/>
            <br/>
            <img alt="" src="http://www.japan-talk.com/images/jt/thumbnails/bicycle-moms-731.jpg"/>
            <br/>
            <br/>
            In Japan, everyone rides mama chari.   Check any bicycle parking lot: at least 95% mama chari.
            <br/>
            <br/>
            <img alt="" src="http://www.japan-talk.com/images/jt/thumbnails/massive-bicycle-parks-734.jpg"/>
            <br/>
            <br/>
            <img alt="" src="http://www.japan-talk.com/images/jt/thumbnails/train-crossing-734.jpg"/>
            <br/>
            <br/>
            Office ladies ride them.
            <br/>
            <br/>
            <img alt="" src="http://www.japan-talk.com/images/jt/thumbnails/office-lady-on-bicycle-734.jpg"/>
            <br/>
            <br/>
            Oji-san ride them.
            <br/>
            <br/>
            <img alt="" src="http://www.japan-talk.com/images/jt/thumbnails/salaryman-bicycle-734.jpg"/>
            <br/>
            <br/>
            Salary men ride them.
            <br/>
            <br/>
            <img alt="" src="http://www.japan-talk.com/images/jt/thumbnails/bike-coworkers-734.jpg"/>
            <br/>
            <br/>
            School children ride them.
            <br/>
            <br/>
            <img alt="" src="http://www.japan-talk.com/images/jt/thumbnails/school-girls-on-bicycles-734.jpg"/>
            <br/>
            <br/>
            Kids learn to ride on them.
            <br/>
            <br/>
            <img alt="" src="http://www.japan-talk.com/images/jt/thumbnails/learning-to-fly-734.jpg"/>
            <br/>
            <br/>
            Even the police ride them.
            <br/>
            <br/>
            <img alt="" src="http://www.japan-talk.com/images/jt/thumbnails/japanese-culture-734.jpg"/>
            <br/>
            <br/>
            They are great for carrying groceries (here a security guard helps a woman with her bicycle as she loads groceries).
            <br/>
            <br/>
            <img alt="" src="http://www.japan-talk.com/images/jt/thumbnails/worlds-first-useful-security-guard-734.jpg"/>
            <br/>
           </span>
