   <h3>
    Instructions
   </h3>
   <ul>
    <li>
     To view any changes you have made, click "update display", "add column" or "remove column".
    </li>
    <li>
     To change the order the blocks are displayed in, enter a new block number in the top input field below the relevant column number.
     <em>
      The original block number will automatically be moved to the column the new block number previously occupied.
     </em>
    </li>
    <li>
     To change the width of a column, enter a new number in the bottom input field below the relevant column number.
    </li>
    <li>
     To change the type of unit the column widths are specified in, select a new type from the "unit" drop-down menu.
     <em>
      If you have switched to ems or pixels, you will be asked to set a default width for the columns. If you have switched to percent, the column width inputs will be automatically updated to add up to 100% as equally as possible with any excess amount due to rounding being added to the lowest numbered columns.
     </em>
    </li>
    <li>
     To add a column, click "add column".
     <em>
      The new column will be added to the right of the pre-existing columns and if the type of unit is percent, the column widths will be adjusted proprtionately - this action will be performed in addition to any other changes you may have made.
     </em>
    </li>
    <li>
     To remove a column, click "remove column".
     <em>
      The div with the highest number (4) will be removed and if the type of unit is percent, the column widths will be adjusted proprtionately - this action will be performed in addition to any other changes you may have made.
     </em>
    </li>
