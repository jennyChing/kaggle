           <h2 style="text-align: justify;">
            BWL Knowledge Base Manager
           </h2>
           <p>
            BWL knowledge base one of the most comprehensive wiki knowledge base plugins available currently.It offers you all the important functionality required in a high quality knowledge base plugin.
           </p>
           <p>
            This is a newbie friendly plugin which doesnt need any complicated settings, you can get started almost instantly. It offers you great looking responsive grid style to show your content along with related knowledge base questions.
           </p>
           <p>
            You can classify your content based on your categories and show them in beautifulhierarchy. You can also show beautiful animated tans for popular and recent questions.
           </p>
           <p>
            Overall, this is a great plugin and probably the best in this list of wiki knowledge base plugins
           </p>

           <h2>
            KnB WordPress Knowledge Base Plugin
           </h2>
           <p style="text-align: justify;">
            Its a fantastic knowledge base WordPress plugin. It covers a lot of features like shortcode generator which you can show on every category by title or date, ratings and control views and layouts.
           </p>
           <p style="text-align: justify;">
            Youve a widget to show latest, most popular (by votes) otherwise you can display with alphabetic order on the sidebars.

           <h2>
            Very Simple Knowledge Base Plugin
           </h2>
           <p>
            Very simple knowledge plugin is a responsive knowledge base plugin to create FAQ, Wiki or bulletins. All you require is shortcode for showing categories and posts in columns as it make use of default WordPress built-in system.
           </p>
           <p>
            This plugin used the default WordPress category and post system to produce the content which you can place withshortcode [knowledgebase-three] or [knowledgebase] to display your categories and posts in 3 or 4 columns on a page. In mobile screens 2 columns.
           </p>
           <a class="buttons btn_red left" href="https://wordpress.org/plugins/very-simple-knowledge-base/" target="_self">

            Wiki
           </h3>
           <p>

            UserPress  Wiki Plugin For WordPress

            This plugin works perfectly with any WordPress theme, so you downt have to change your current design for using this plugin.

            Knowledge Base  Wiki WordPress Plugin

           <p>
