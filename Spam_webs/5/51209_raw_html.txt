      <h1>
       Use jQuery to make external links open in a new window
      </h1>
      <time datetime="2010-08-12T00:00Z">
       <span class="day">
        11
       </span>
       <span class="month">
        Jun
       </span>
       <span class="year">
        2008
       </span>
      </time>
     </header>
     <p>
      I was just reading the comments of Raymond Camden's post about the
      <a href="http://www.coldfusionjedi.com/index.cfm/2008/5/28/Ask-a-Jedi-Best-way-to-trim-text">
       best way to trim text
      </a>
      and saw that someone was asking how to add target="_blank" to all external links. Now this can be done in ColdFusion using the CFMatch tag (a great addition to CF8!), but I'd let jQuery do the hard work for me. I posted my solution on Raymond's blog but thought I'd put it here as well.
     </p>
     <p>
      Here's how...
     </p>
     <pre>
      <span style="color:#000000">
       <br/>
       $('a[href^="http://"]')
       <br/>
       .attr("target", "_blank");
       <br/>
      </span>
     </pre>
     <p>
      Or if you want to get fancy:
     </p>
     <pre>
      <span style="color:#000000">
       <br/>
       $('a[href^="http://"]')
       <br/>
       .attr(
       <span style="color:#000099">
        {
       </span>
       <br/>
       target: "_blank",
       <br/>
       title: "Opens in a
       <span style="color:#990000">
        new
       </span>
       window"
       <br/>
       <span style="color:#000099">
        }
       </span>
       );
       <br/>
      </span>
     </pre>
     <p>
      You can even add an image or text to your hyperlink with a little bit of chaining. This example adds [^] to the end of your link text.
     </p>
     <pre>
      <span style="color:#000000">
       <br/>
       $('a[href^="http://"]')
       <br/>
       .attr(
       <span style="color:#000099">
        {
       </span>
       <br/>
       target: "_blank",
       <br/>
       title: "Opens in a
       <span style="color:#990000">
        new
       </span>
       window"
       <br/>
       <span style="color:#000099">
        }
       </span>
       )
       <br/>
       .append(' [^]');
       <br/>
      </span>
     </pre>
     <p>
      Pretty neat huh! I should point out that these examples assume that all your external links start with http:// but the selectors in jQuery are powerful so you can filter by domain names etc.
     </p>
