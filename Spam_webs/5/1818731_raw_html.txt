                     CodeIgniter | Remove index.php from URL
                    </h1>
                    <div class="post-header">

                     <div dir="ltr" style="text-align: left;" trbidi="on">
                      <p>
                       Hi, well see
                       <strong>
                        how to remove index.php from codeigniter url
                       </strong>
                       .  Even though codeigniter uses search engine friendly urls, nobody would have missed the awkward presence of
                       <strong>
                        'index.php'
                       </strong>
                       in between those codeigniter's urls.  Have you ever wondered how to get rid of it to get more clean urls in your
                       <a href="http://www.kodingmadesimple.com/2014/08/how-to-create-login-form-codeigniter-mysql-twitter-bootstrap.html">
                        codeigniter application
                       </a>
                       ? Say your app has an url something like
                       <code>
                        http://www.example.com/index.php/contact
                       </code>
                       and you want it to look like
                       <code>
                        http://www.example.com/contact
                       </code>
                       .  Yes, you can do it.  In codeigniter removing index.php from url should be done by rewriting urls in htaccess file.
                      </p>
                      <div class="separator" style="clear: both; text-align: center;">
                       <a href="http://1.bp.blogspot.com/-tPiLdUAtjWc/VJCfYXGalhI/AAAAAAAAAY8/cPoTTbxlC5c/s1600/codeigniter-remove-index.php-from-url.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                        <img alt="codeigniter-remove-index.php-from-url" border="0" src="http://1.bp.blogspot.com/-tPiLdUAtjWc/VJCfYXGalhI/AAAAAAAAAY8/cPoTTbxlC5c/s1600/codeigniter-remove-index.php-from-url.png" title="codeigniter-remove-index.php-from-url"/>
                       </a>
                      </div>
                      <a name="more">
                      </a>
                      <h2>
                       Step-by-Step Instructions to Remove index.php in CodeIgniter URL
                      </h2>
                      <p class="recmnd-block">
                       Recommended Read:
                       <a href="http://www.kodingmadesimple.com/2014/08/how-to-create-login-form-codeigniter-mysql-twitter-bootstrap.html">
                        How to Create Login Form in CodeIgniter, MySQL and Twitter Bootstrap
                       </a>
                      </p>
                      <p class="recmnd-block">
                       Recommended Read:
                       <a href="http://www.kodingmadesimple.com/2014/07/codeigniter-mysql-tutorial-read-display-data-bootstrap.html">
                        CodeIgniter/MySQL Tutorial - Read and Display data with Twitter Bootstrap
                       </a>
                      </p>
                      <h2>
                       Step 1: Enable Mod Rewrite Option in your APACHE Server
                      </h2>
                      <p>
                       To rewrite urls in the htaccess file, the mod_rewrite option should be enabled in the apache server.  Goto apache's "httpd.conf" file and search for the line,
                      </p>
                      <code>
                       LoadModule rewrite_module modules/mod_rewrite.so
                      </code>
                      <p>
                       If the above line is preceded with # (in which case, the module is disabled), then remove the # (hash) symbol to enable url rewriting.
                      </p>
                      <h2>
                       Step 2: Create '.htaccess' File
                      </h2>
                      <p>
                       Next create the
                       <strong>
                        .htaccess
                       </strong>
                       file at the root directory.  To create the htaccess file, open your favourite text editor and create a new text file.  Save it as ".htaccess" (make sure you type the filename within quotes to avoid the file to be saved as text file) in the root directory.
                      </p>
                      <div class="separator" style="clear: both; text-align: center;">
                       <a href="http://3.bp.blogspot.com/-jRTaZ0i-8rA/VJCfYWcSxCI/AAAAAAAAAZA/1id666bRMeQ/s1600/htaccess-file-codeigniter.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                        <img alt="htaccess-file-codeigniter" border="0" src="http://3.bp.blogspot.com/-jRTaZ0i-8rA/VJCfYWcSxCI/AAAAAAAAAZA/1id666bRMeQ/s1600/htaccess-file-codeigniter.png" title="htaccess-file-codeigniter"/>
                       </a>
                      </div>
                      <p>
                       Now copy paste this code to the htaccess file and save it.
                      </p>
                      <pre class="codebk">
                       &lt;IfModule mod_rewrite.c&gt;
RewriteEngine On
RewriteBase /
RewriteCond %{THE_REQUEST} ^GET.*index\.php [NC]
RewriteCond %{REQUEST_URI} !/system/.* [NC]
RewriteRule (.*?)index\.php/*(.*) /$1$2 [R=301,NE,L]
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ /index.php/$1 [L]
&lt;/IfModule&gt;
                      </pre>
                      <p>
                       If you run your
                       <a href="http://www.kodingmadesimple.com/2014/07/codeigniter-mysql-tutorial-read-display-data-bootstrap.html">
                        codeigniter
                       </a>
                       app as a subdirectory instead of root domain like,
                       <code>
                        http://example.com/cisite/
                       </code>
                       instead of
                       <code>
                        http://example.com/
                       </code>
                       then the above code won't work.  You have to tweak it little to tell that your site runs as a subdirectory.  Add this code to the htaccess file instead of the above one.
                      </p>
                      <pre class="codebk">
                       &lt;IfModule mod_rewrite.c&gt;
RewriteEngine On
RewriteBase /
RewriteCond %{THE_REQUEST} ^GET.*index\.php [NC]
RewriteCond %{REQUEST_URI} !/system/.* [NC]
RewriteRule (.*?)index\.php/*(.*) /$1$2 [R=301,NE,L]
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ index.php/$1 [L]
&lt;/IfModule&gt;
                      </pre>
                      <h2>
                       Step 3: Modify CodeIgniter Config Settings
                      </h2>
                      <p>
                       Open
                       <strong>
                        config.php
                       </strong>
                       file under application &gt;&gt; config folder and search for the line,
                      </p>
                      <code>
                       $config['index_page'] = 'index.php';
                      </code>
                      <p>
                       Remove index.php from it to look like this,
                      </p>
                      <code>
                       $config['index_page'] = '';
                      </code>
                      <p>
                       Now restart apache and check the url.  eg.,
                       <code>
                        http://www.example.com/contact
                       </code>
                       instead of
                       <code>
                        http://www.example.com/index.php/contact
                       </code>
                      </p>
                      <p class="recmnd-block">
                       Recommended Read:
                       <a href="http://www.kodingmadesimple.com/2014/11/how-to-upload-files-in-php.html">
                        How to Upload Files in PHP Securely
                       </a>
                      </p>
                      <p class="recmnd-block">
                       Recommended Read:
                       <a href="http://www.kodingmadesimple.com/2014/12/how-to-insert-json-data-into-mysql-php.html">
                        How to Insert JSON into MySQL using PHP
                       </a>
                      </p>
                      <p>
                       If it doesn't work, you may be having problem with uri protocol.  In that case find
                       <code>
                        $config['uri_protocol'] ='AUTO'
                       </code>
                       setting in config file and replace with
                       <code>
                        $config['uri_protocol'] = 'REQUEST_URI'
                       </code>
                       .
                      </p>
                      <p>
                       <em>
                        Restart apache server again and you can check the codeigniter site to see that index.php has been removed from codeigniter url.
                       </em>
                      </p>

            comment.timestamp = Date.parse(entry.published.$t) + '';
            if (entry.author &amp;&amp; entry.author.constructor === Array) {
              var auth = entry.author[0];
              if (auth) {
                comment.author = {
                  name: (auth.name ? auth.name.$t : undefined),
                  profileUrl: (auth.uri ? auth.uri.$t : undefined),
                  avatarUrl: (auth.gd$image ? auth.gd$image.src : undefined)
                };
              }
            }
            if (entry.link) {
              if (entry.link[2]) {
                comment.link = comment.permalink = entry.link[2].href;
              }
              if (entry.link[3]) {
                var pid = /.*comments\/default\/(\d+)\?.*/.exec(entry.link[3].href);
                if (pid &amp;&amp; pid[1]) {
                  comment.parentId = pid[1];
                }
              }
            }
            comment.deleteclass = 'item-control blog-admin';
            if (entry.gd$extendedProperty) {
              for (var k in entry.gd$extendedProperty) {
                if (entry.gd$extendedProperty[k].name == 'blogger.itemClass') {
                  comment.deleteclass += ' ' + entry.gd$extendedProperty[k].value;
                } else if (entry.gd$extendedProperty[k].name == 'blogger.displayTime') {
                  comment.displayTime = entry.gd$extendedProperty[k].value;
                }
              }
            }
            comments.push(comment);
          }
        }
        return comments;
      };

      var paginator = function(callback) {
        if (hasMore()) {
          var url = config.feed + '?alt=json&amp;v=2&amp;orderby=published&amp;reverse=false&amp;max-results=50';
          if (cursor) {
            url += '&amp;published-min=' + new Date(cursor).toISOString();
          }
          window.bloggercomments = function(data) {
            var parsed = parse(data);
            cursor = parsed.length &lt; 50 ? null
                : parseInt(parsed[parsed.length - 1].timestamp) + 1
            callback(parsed);
            window.bloggercomments = null;
          }
          url += '&amp;callback=bloggercomments';
          var script = document.createElement('script');
          script.type = 'text/javascript';
          script.src = url;
          document.getElementsByTagName('head')[0].appendChild(script);
        }
      };
      var hasMore = function() {
        return !!cursor;
      };
      var getMeta = function(key, comment) {
        if ('iswriter' == key) {
          var matches = !!comment.author
              &amp;&amp; comment.author.name == config.authorName
              &amp;&amp; comment.author.profileUrl == config.authorUrl;
          return matches ? 'true' : '';
        } else if ('deletelink' == key) {
          return config.baseUri + '/delete-comment.g?blogID='
               + config.blogId + '&amp;postID=' + comment.id;
        } else if ('deleteclass' == key) {
          return comment.deleteclass;
        }
        return '';
      };

      var replybox = null;
      var replyUrlParts = null;
      var replyParent = undefined;

      var onReply = function(commentId, domId) {
        if (replybox == null) {
          // lazily cache replybox, and adjust to suit this style:
          replybox = document.getElementById('comment-editor');
          if (replybox != null) {
            replybox.height = '250px';
            replybox.style.display = 'block';
            replyUrlParts = replybox.src.split('#');
          }
        }
        if (replybox &amp;&amp; (commentId !== replyParent)) {
          document.getElementById(domId).insertBefore(replybox, null);
          replybox.src = replyUrlParts[0]
              + (commentId ? '&amp;parentID=' + commentId : '')
              + '#' + replyUrlParts[1];
          replyParent = commentId;
        }
      };

      var hash = (window.location.hash || '#').substring(1);
      var startThread, targetComment;
      if (/^comment-form_/.test(hash)) {
        startThread = hash.substring('comment-form_'.length);
      } else if (/^c[0-9]+$/.test(hash)) {
        targetComment = hash.substring(1);
      }

      // Configure commenting API:
      var configJso = {
        'maxDepth': config.maxThreadDepth
      };
      var provider = {
        'id': config.postId,
        'data': items,
        'loadNext': paginator,
        'hasMore': hasMore,
        'getMeta': getMeta,
        'onReply': onReply,
        'rendered': true,
        'initComment': targetComment,
        'initReplyThread': startThread,
        'config': configJso,
        'messages': msgs
      };

      var render = function() {
        if (window.goog &amp;&amp; window.goog.comments) {
          var holder = document.getElementById('comment-holder');
          window.goog.comments.render(holder, provider);
        }
      };

      // render now, or queue to render when library loads:
      if (window.goog &amp;&amp; window.goog.comments) {
        render();
      } else {
        window.goog = window.goog || {};
        window.goog.comments = window.goog.comments || {};
        window.goog.comments.loadQueue = window.goog.comments.loadQueue || [];
        window.goog.comments.loadQueue.push(render);
      }
    })();
// ]]&gt;
                     </script>
