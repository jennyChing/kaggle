       <h1 class="entry-title">
        Easy Cranberry Margaritas
       </h1>
       <div style="margin-bottom:45px;">
       </div>
       <div class="post-info">
       </div>
       <div class="entry-content">
        <input data-frizzlyhovercontainer="" data-frizzlypostcontainer="" data-frizzlyposttitle="Easy Cranberry Margaritas" data-frizzlyposturl="http://domesticsuperhero.com/2013/12/15/easy-cranberry-margaritas/" type="hidden" value=""/>
        <p style="text-align: center;">
         <a href="http://domesticsuperhero.com/wp-content/uploads/2013/12/cranberry-margs.jpg">
          <img alt="Easy Cranberry Margaritas" border="0" class="aligncenter" height="900" src="http://domesticsuperhero.com/wp-content/uploads/2013/12/cranberry-margs_thumb.jpg" style="border: 0px;" title="Easy Cranberry Margaritas" width="600"/>
         </a>
        </p>
        <p style="text-align: center;">
         Its no secret that I love margaritas, but I really love taking a couple shortcuts while making them. No one ever notices anything different when I make margaritas, so its really a win win. Making the perfect margarita doesnt have to be hard, and even with a simple substitution, you dont sacrifice any flavor.
        </p>
        <p style="text-align: center;">
         I love the idea of Cranberry Margaritas around the holidays. Even though margaritas have a reputation of being a summer drink, by adding the more tart cranberries, it brings them down to a less tropical-fruity level. Even though the cranberries are tart, the other ingredients pick up the sweetness, so this isnt a bitter drink. Its really perfect in my opinion. Serve these at your holiday party or just hanging out. I am going to mix up another batch for New Years Eve!
        </p>
        <p>
         <a href="http://domesticsuperhero.com/wp-content/uploads/2013/12/cranberry-margs-4.jpg">
          <img alt="Easy Cranberry Margaritas" border="0" height="900" src="http://domesticsuperhero.com/wp-content/uploads/2013/12/cranberry-margs-4_thumb.jpg" style="background-image: none; padding-top: 0px; padding-left: 0px; margin-left: auto; display: block; padding-right: 0px; margin-right: auto; border: 0px;" title="Easy Cranberry Margaritas" width="600"/>
         </a>
        </p>
        <div class="blog-yumprint-recipe blog-yumprint-standard " itemscope="" itemtype="http://schema.org/Recipe" yumprintrecipe="d-1d">
         <div class="blog-yumprint-recipe-title" itemprop="name">
          Easy Cranberry Margaritas
         </div>
         <div class="blog-yumprint-recipe-published" itemprop="datePublished">
          2013-12-15 15:38:18
         </div>
         <div class="blog-yumprint-header">
          <div class="blog-yumprint-save blog-yumprint-action">
           <a href="http://yumprint.com/app/object/d-1d">
            Save Recipe
           </a>
          </div>
          <div class="blog-yumprint-print blog-yumprint-action">
           Print
          </div>
         </div>
         <div class="blog-yumprint-spacer">
         </div>
         <div class="blog-yumprint-recipe-contents">
          <div class="blog-yumprint-ingredient-section" yumprintsection="0">
           <div class="blog-yumprint-subheader">
            Ingredients
           </div>
           <ol class="blog-yumprint-ingredients">
            <li class="blog-yumprint-ingredient-item" itemprop="ingredients" yumprintitem="0">
             1 can (12 ounces) frozen limeade concentrate (+2.5 cans of water)
            </li>
            <li class="blog-yumprint-ingredient-item" itemprop="ingredients" yumprintitem="1">
             1 cup cranberries, rinsed plus some for garnish
            </li>
            <li class="blog-yumprint-ingredient-item" itemprop="ingredients" yumprintitem="2">
             1/4 cup sugar
            </li>
            <li class="blog-yumprint-ingredient-item" itemprop="ingredients" yumprintitem="3">
             1 cup cranberry juice
            </li>
            <li class="blog-yumprint-ingredient-item" itemprop="ingredients" yumprintitem="4">
             2 to 2 1/2 cups tequila (gold or silver)
            </li>
            <li class="blog-yumprint-ingredient-item" itemprop="ingredients" yumprintitem="5">
             lime for garnish
            </li>
            <li class="blog-yumprint-ingredient-item" itemprop="ingredients" yumprintitem="6">
             sugar or salt for glass rim
            </li>
           </ol>
          </div>
          <div class="blog-yumprint-method-section" yumprintsection="1">
           <div class="blog-yumprint-subheader">
            Instructions
           </div>
           <ol class="blog-yumprint-methods" itemprop="recipeInstructions">
            <li class="blog-yumprint-method-item" yumprintitem="0">
             1. Empty the limeade concentrate into a pitcher. Add 2 1/2 cans of water. Add 16-20 ounces (your preference) of tequila.  Add cranberry juice, 1/2 cup cranberries, and sugar.  Mix thoroughly.
            </li>
            <li class="blog-yumprint-method-item" yumprintitem="1">
             2.Place the pitcher, covered, in the fridge for at least 2 hours. Serve in sugar/salt rimmed glasses garnished with lime and cranberries if desired.
            </li>
           </ol>
          </div>
          <div class="blog-yumprint-note-section" yumprintsection="2">
           <div class="blog-yumprint-subheader">
            Notes
           </div>
           <ol class="blog-yumprint-notes">
            <li class="blog-yumprint-note-item" yumprintitem="0">
             *If you aren't a fan of tequila, you could also make these with your favorite vodka.
            </li>
           </ol>

String.prototype.trim = function() {
  return this.replace(/^\s+|\s+$/g,"");
}

function createCookie(name,value,days) {
  if (days) {
    var date = new Date();
    date.setTime(date.getTime()+(days*24*60*60*1000));
    var expires = "; expires="+date.toGMTString();
  } else {
    var expires = "";
  }
  document.cookie = name+"="+value+expires+"; path=/";
}

function readCookie(name) {
  var nameEQ = name + "=";
  var ca = document.cookie.split(';');
  for(var i=0;i &lt; ca.length;i++) {
    var c = ca[i];
    while (c.charAt(0)==' ') {
      c = c.substring(1,c.length);
    }
    if (c.indexOf(nameEQ) == 0) {
      return c.substring(nameEQ.length,c.length);
    }
  }
  return null;
}

function eraseCookie(name) {
  createCookie(name,"",-1);
}

    collapseArch(this, expandSym, collapseSym);

  });
}

    jQuery(symbol).children('.sym').html(collapseSym);
  createCookie(theDiv.attr('id'), 1, 7);
