             <p>
              Its all around now; CSS3 has made an impact among the web designers. They are trying this technique to create many attractive design projects. The most amazing part to use CSS3 is  it gives your flexibility to use with any web design technology. You will feel the same lookout of Flash using CSS3 styling in your web projects. Playing with typography is very old fashion for the designers, using graphic application it is normal but with CSS3 it is experts fold.
             </p>
             <p>
             </p>
             <p>
             </p>
             <center>
              <a href="http://bloggerspath.com/wp-content/uploads/2013/03/3-Awesome-CSS3-Typography-Techniques-to-Learn.jpg" rel="wp-prettyPhoto[g8851]">
               <img alt="3 Awesome CSS3 Typography Techniques to Learn" class="aligncenter size-full wp-image-8852" id="spl" src="/wp-content/uploads/2013/03/3-Awesome-CSS3-Typography-Techniques-to-Learn.jpg"/>
              </a>
             </center>
             <p>
             </p>
             <p>
              Using CSS3 you can spice up the normal typography and can turn those texts into attractive text effects. Following 3 CSS3 typography techniques are easier to learn and implement into web project. These will give your project stunning look and good visitor appeal. You have to use web-kit enabled browsers to see the actual effects of these CSS3 typography. Now, its old news to use Flash for designing rich text graphic, you can get the same result with right combination of color, font with CSS3 and HTML5.
             </p>
             <p>
             </p>
             <hr/>
             <h2>
              1. Textured Shadow Effect
             </h2>
             <p>
              This is eye pleasing effect for your site visitors, just apply the following code to the text of your site get the response from your visitors.
              <br/>
             </p>
             <center>
              <a href="http://bloggerspath.com/wp-content/uploads/2013/03/Textured-Shadow-Effect.jpg" rel="wp-prettyPhoto[g8851]">
               <img alt="Textured Shadow Effect" class="aligncenter size-full wp-image-8854" id="spl" src="/wp-content/uploads/2013/03/Textured-Shadow-Effect.jpg"/>
              </a>
             </center>
             <br/>
             <strong>
              CSS Code:
             </strong>
             <pre class="brush: css; title: ; notranslate" title="">
              body {
    background: #441369;   
}
 
h1#gradient {
    text-align:center;
}
h1#gradient span {
    position:relative;
    display: inline-block;
    color: #FF0052; /* Fallback Color */
    text-transform: uppercase;
    font-weight: bold;
    font-family: helvetica;
    text-align:center;
    font-size: 70px;
    letter-spacing: -4px;
    text-shadow: 4px 4px 0px #441369, 8px 8px 0px rgba(255,255,255,0.1);  /* Fallback Shadow */
}
@media  screen and (-webkit-min-device-pixel-ratio:0) {
    h1#gradient span{
      background: -webkit-gradient(linear,left top,left bottom,from(#FF0052),to(#802F7B));
      -webkit-background-clip: text;
      -webkit-text-fill-color: transparent;
      text-shadow:none !important;
    }
    h1#gradient span:after {
        content: attr(data-text);
        left: 8px;
        position: absolute;
        background: url(http://i.imgur.com/RkDRMcJ.png); /* image source for your texture */
        -webkit-background-clip: text;
        -webkit-text-fill-color: transparent;
        text-shadow: -4px -4px 0px #441369, -1px 0px 0px rgba(255, 255, 255, 0.1);
        top: 8px;
        z-index: -1;
        width: 100%;
    }
}

             <h2>
              2. Gradient on Text
             </h2>
             <p>
              If you want to create gradient background, then use the following CSS3 code to get the effect.
              <br/>
             </p>
             <center>
              <a href="http://bloggerspath.com/wp-content/uploads/2013/03/Gradient-on-Text.jpg" rel="wp-prettyPhoto[g8851]">
               <img alt="Gradient on Text" class="aligncenter size-full wp-image-8853" id="spl" src="/wp-content/uploads/2013/03/Gradient-on-Text.jpg"/>
              </a>

             <br/>
             <strong>
              CSS Code:
             </strong>
             <pre class="brush: css; title: ; notranslate" title="">
              h1#gradient {

             <h2>
              3. The Relief Shade Effect
             </h2>
             <p>
              You can extend gradient effect by using the Relief Shade effect to give some unique touch to your font. just try this piece of code to achieve this 
              <br/>
             </p>
             <center>
              <a href="http://bloggerspath.com/wp-content/uploads/2013/03/The-Relief-Shade-Effect.jpg" rel="wp-prettyPhoto[g8851]">
               <img alt="The Relief Shade Effect" class="aligncenter size-full wp-image-8855" id="spl" src="/wp-content/uploads/2013/03/The-Relief-Shade-Effect.jpg"/>
              </a>
             </center>
             <br/>
             <strong>
              CSS Code:
             </strong>
             <pre class="brush: css; title: ; notranslate" title="">
              body {
    background: #441369;   
}
 
h1#gradient {
    text-align:center;
}
h1#gradient span {
    position:relative;
    display: inline-block;
    color: #FF0052; /* Fallback Color */
    text-transform: uppercase;
    font-weight: bold;
    font-family: helvetica;
    text-align:center;
    font-size: 70px;
    letter-spacing: -4px;
    text-shadow: 4px 4px 0px #441369, 8px 8px 0px rgba(255,255,255,0.1);  /* Fallback Shadow */
}
@media  screen and (-webkit-min-device-pixel-ratio:0) {
    h1#gradient span{
        background: -webkit-gradient(linear,left top,left bottom,from(#FF0052),to(#802F7B));
        -webkit-background-clip: text;
        -webkit-text-fill-color: transparent;
        text-shadow:none !important;
    }
    h1#gradient span:after {
        background: none;
        content: attr(data-text);
        left: 0;
        position: absolute;
        text-shadow: 4px 4px 0px #441369, 8px 8px 0px rgba(255,255,255,0.1); //relief shade effect 
        top: 0;
        z-index: -1;
    }
}
