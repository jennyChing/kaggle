             <li data-textannotation-id="76db693a4a06cbecc0ce9ea43ed53304">
              <a href="http://lifehacker.com/5814024/firefox-5-now-available" x-inset="1">
               Firefox 5 Now Available with 1,000 Small Improvements, Better Tab Closing
              </a>
              (Windows/Mac/Linux/Android)
              <br/>
              <em>
               Mozilla has released Firefox 5, the first update for its new rapid-release three-month development cycle. Firefox 5's new features include support for CSS animations, do-not-track on Android, and faster canvas graphics. Perhaps our favorite Firefox 5 feature: You won't have to jump around so much with your mouse next time you start closing tabs. Firefox has adopted Chrome-like tab closing, in which tabs stay the same size while you're closing them.
              </em>
             </li>
             <li data-textannotation-id="f4580bfb7d276cd747671ef4fec7e070">
              <a href="http://lifehacker.com/5813895/the-best-photo-management-app-for-windows" x-inset="1">
               The Best Photo Management App for Windows
              </a>
              (Windows)
              <br/>
              <em>
               Windows has its fair share of photo management programs, but we've chosen Picasa as the best for its impressive feature set and relative ease of use.
              </em>
             </li>
             <li data-textannotation-id="2cfa3a1dfe000c740ad75da5323554c5">
              <a href="http://lifehacker.com/5813662/new-version-of-chrome-frame-doesnt-require-admin-rights-to-install" x-inset="1">
               Get Chrome's Features in Your IT-Mandated Internet Explorer, No Admin Rights Necessary
              </a>
              (Windows)
              <br/>
              <em>
               If you're stuck on an office computer and forced to use Internet Explorer because your system is locked down, Google has your back. The new version of Chrome Frame brings the features of Google Chrome to IE and doesn't need administrative rights to your PC to install.
              </em>
             </li>
             <li data-textannotation-id="015d495108a0bbea1e2080a6d2e5ea4b">
              <a href="http://lifehacker.com/5814905/utorrent-adds-media-streaming-drag+and+drop-file-sharing-to-our-favorite-bittorrent-client" x-inset="1">
               uTorrent 3.0 Adds Instant Media Streaming, Drag-and-Drop File Sharing to Our Favorite BitTorrent Client
              </a>
              (Windows)
              <br/>
              <em>
               The newest version of our favorite BitTorrent client, uTorrent, brings a host of awesome features to file sharing on Windows, including streaming media as you download, a new remote control interface, and drag-and-drop file sharing.
              </em>
             </li>
             <li data-textannotation-id="4ab9ac09fb60f95dd2f80c5aebceb9c5">
              <a href="http://lifehacker.com/5813720/imuscle-teaches-you-to-stretch-and-exercise-by-literally-showing-you-the-muscles-in-action" x-inset="1">
               iMuscle Teaches You to Stretch and Exercise by Literally Showing You the Muscles in Action
              </a>
              (iOS)
              <br/>
              <em>
               Most fitness apps for iOS will teach you to stretch and exercise through specific demonstrations, but iMuscle takes it a step further by actually showing you the muscles in action. It works by providing a ton of animated diagrams that are accessible with just a touch.
              </em>
             </li>
             <li data-textannotation-id="a45952bb2f387941ebfa870f403b350d">
              <a href="http://lifehacker.com/5813514/the-best-photo-management-app-for-mac-os-x" x-inset="1">
               The Best Photo Management App for Mac OS X
              </a>
              (Mac)
              <br/>
              <em>
               There aren't many Mac photo management apps that aren't somewhat specialized, but among the few we think iPhoto is the best for it's simplicity, rich feature set, and good integration with popular online services.
              </em>
             </li>
             <li data-textannotation-id="4b5547d5bd7d3944edd1bf5ddc015264">
              <a href="http://lifehacker.com/5813246/usb-alert-makes-sure-you-never-leave-your-usb-drive-behind-again" x-inset="1">
               USB Alert Makes Sure You Never Leave Your USB Drive Behind Again
              </a>
              (Windows)
              <br/>
              <em>
               Everyone who's ever used a USB drive knows the horror of looking for it when you need it only to realize that you left it behind plugged into the last computer you used. With USB Alert, you'll get a warning when you shut down to unplug your USB drive before you go.
              </em>
             </li>
             <li data-textannotation-id="826df54972861430ac6ed33f371e2f54">
              <a href="http://lifehacker.com/5813496/overlap-wallpaper-displays-a-second-image-on-top-of-your-wallpaper-keeps-you-organized" x-inset="1">
               Overlap Wallpaper Displays a Second Image on Top of Your Wallpaper, Keeps You Organized
              </a>
              (Windows)
              <br/>
              <em>
               Changing up your desktop wallpaper can be really fun, but if you want a certain image on your desktop at all times, Overlap Wallpaper takes any image on your computer and displays it over your desktop.
              </em>
             </li>
             <li data-textannotation-id="625a63142950241a85f402044a988cdf">
              <a href="http://lifehacker.com/5815396/zetro-brings-windows-8s-metro+style-interface-to-windows-7" x-inset="1">
               Zetro Brings Windows 8's Metro-Style Interface to Windows 7
              </a>
              (Windows)
              <br/>
              <em>
               Microsoft's Metro UI, which graces both Windows 8 and Windows Phone, is pretty snazzy, and if you're sick of the Aero look, you can bring the Metro UI to Windows 7 with the Zetro theme.
              </em>
             </li>
             <li data-textannotation-id="c52239b5c96de9ac691081f632842a7c">
              <a href="http://lifehacker.com/5813609/youtube-options-for-google-chrome-disables-ads-hides-comments" x-inset="1">
               YouTube Options for Google Chrome Disables Ads, Hides Comments
              </a>
              (Chrome)
              <br/>
              <em>
               YouTube is great, but some of its features are annoying. With the YouTube Options for Google Chrome extension, you can suppress in-video ads, remove comments, disable annotations, and more. The extension allows you to completely change the look and feel of YouTube.
              </em>
             </li>
            </ul>
