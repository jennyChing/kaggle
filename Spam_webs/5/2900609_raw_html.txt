         Justice League
        </h2>
        <div class="imgright" style="width: 184px;">
         <div class="imgmain">
          <a href="/img/2kh6l4n0a1za9qu7/Justice-League">
           <img alt="Justice League " height="76" src="http://img.wikinut.com/img/2kh6l4n0a1za9qu7/jpeg/180x300/Justice-League.jpeg" title="Justice League " width="180"/>

        <p>
         <b>
          Superman
         </b>
         <br/>
         Blue costume, red cape
         <br/>
         Flying over Metropolis
         <br/>
         The Greatest Superhero
         <br/>
         <br/>
         <b>
          Batman
         </b>
         <br/>
         The Caped Crusader
         <br/>
         Hes the World's Greatest Detective
         <br/>
         The Icon of Gotham
         <br/>
         <br/>
         <b>
          Flash
         </b>
         <br/>
         The Fastest super-hero alive
         <br/>
         Flirty, confident, arrogant
         <br/>
         Reddest crime fighter
         <br/>
         <br/>
         <b>
          Green Lantern
         </b>
         <br/>
         Magical green ring
         <br/>
         Has Extraordinary powers
         <br/>
         Protects the Universe
         <br/>
         <br/>
         <b>
          Wonder Woman
         </b>
         <br/>
         The Warrior princess
         <br/>
         Indestructible bracelets
         <br/>
         Sexy Woman in boots
         <br/>
         <br/>
         <b>
          Aquaman
         </b>
         <br/>
         King of Atlantis
         <br/>
         The Undersea explorer
         <br/>
         Marine life lover
         <br/>
         <br/>
         <b>
          Martian Manhunter
         </b>
         <br/>
         The green-skinned superhero
         <br/>
         extraterrestrial man
         <br/>
         Favorite martian
        </p>
