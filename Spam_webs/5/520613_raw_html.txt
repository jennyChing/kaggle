          <h2 class="posttitle">
           <a href="http://es.autoblog.com/2012/11/08/direcciones-mecatronicas-lo-bueno-y-lo-malo/" rel="bookmark">
            <span id="ppt20373975">
             Direcciones mecatrnicas: lo bueno y lo malo
            </span>
           </a>
          </h2>
          <p class="byline">
           por
           <strong>
            <a href="/bloggers/guillermo-alfonsin/">
             Guillermo Alfonsin
            </a>
           </strong>
           de 08 de Nov del 2012
          </p>
          <div class="postbody">
           <div style="text-align: left; ">
           </div>
           <a href="http://es.autoblog.com/2012/11/08/direcciones-mecatronicas-lo-bueno-y-lo-malo/">
            <img alt="" border="0" height="387" src="http://www.blogcdn.com/es.autoblog.com/media/2012/10/img4258.jpg" width="580"/>
           </a>
           <br/>
           <em>
            Qu tiene de bueno una direccin que no est conectada mecnicamente a las ruedas?
           </em>
           <br/>
           <br/>
           <a href="http://es.autoblog.com/2012/10/17/los-futuros-infiniti-tendran-una-direccion-de-tipo-steer-by-wire/">
            Hace unos das Dani nos traa buenas nuevas
           </a>
           sobre las intenciones de
           <a href="http://es.autoblog.com/category/Infiniti/">
            Infiniti
           </a>
           de incorporar un sistema de direccin mecatrnica a sus coches a medio plazo. Muchos preguntsteis el por qu resultaba interesante hacerlo, o el por qu no se haban implementado antes.
           <br/>
           <br/>
           Hoy os ofrezco una visin privilegiada de este tema. Curiosamente, en mi trabajo de final de carrera, ampliado despus en un proyecto fsico y real, desarroll un sistema de direccin mecatrnica, y por ello acab descubriendo y experimentando sus puntos fuertes y dbiles, as como la problemtica legal existente.
           <br/>
           <br/>
           Empezar primero concretando a qu nos referimos con direccin mecatrnica. Se trata de un sistema de direccin "por cable" (by-wire, en ingls), en el que el volante no tiene un acoplamiento mecnico directo con la cremallera de direccin y las ruedas. El volante acta sobre un sensor electrnico, ste enva esa seal a una centralita, que calcula cunto giro hay que aplicar a las ruedas, y mediante una serie de actuadores aplica el giro deseado a las ruedas.
           <br/>
           <br/>
           <p class="attention" style="margin-top: 2px; width: 240px; margin-bottom: 2px; float: left; margin-right: 8px">
            <em>
             <strong>
              <span style="color: #ff0000">
               <big>
                Disear la geometra de direccin y suspensin de un coche es puro compromiso, compromiso que se aligera con un sistema de direccin mecatrnico
               </big>
              </span>
             </strong>
            </em>
           </p>
           Las ventajas de un sistema de este tipo son tan enormes que ms pronto que tarde se acabarn imponiendo en cualquier tipo de automvil. De hecho, si fuera legal su uso en competicin, y el estado de la tcnica fuera el adecuado, se implementaran maana mismo en todos los coches de carreras.
           <br/>
           <br/>
           Para empezar, la primera ventaja es geomtrica. Como bien sabes, las ruedas de un coche pivotan en relacin a sus anclajes en la suspensin de una determinada manera. Las ruedas delanteras se colocan con un ngulo de "avance". Este ngulo de avance lo que genera es estabilidad lineal. Cuanto mayor es el ngulo de avance, las ruedas ofrecen ms resistencia a dejar de ir en lnea recta.
           <br/>
           <br/>
           El ngulo de avance, adems, genera un par de "autocentrado" para que la direccin vuelva a su posicin de "recto". Uno de los muchos, muchsimos compromisos que afronta un ingeniero en el desarrollo de un coche es saber elegir cunto ngulo de avance coloca a su coche, en funcin de la cantidad de fuerza que haya que hacer sobre el volante para girar las ruedas, y en funcin tambin de cunta informacin quiera que las ruedas transfieran al volante.
           <br/>
           <br/>
           <img alt="" border="0" height="290" hspace="4" id="vimage_5420647" src="http://www.blogcdn.com/es.autoblog.com/media/2012/11/iaalenkradverstellung.jpg" vspace="4" width="580"/>
           <br/>
           <em>
            Especialistas como Brose ya ofrecen columnas de direccin pensadas para estos sistemas
           </em>
           <br/>
           <br/>
           Con la llegada de las direcciones asistidas, cada vez ms elaboradas, se pudo pasar a usar ngulos de avance diferentes a los convencionales, ya que aunque hiciera falta mucha ms fuerza a la hora de girar las ruedas, se poda contar con la ayuda de la asistencia.
           <br/>
           <br/>
           Otra de las claves geomtricas de las ruedas de un coche es su convergencia. Las ruedas de un coche no suelen estar orientadas en lnea recta, sino que siempre tienden a mirar hacia adentro o hacia afuera de la lnea longitudinal del coche. A esto lo denominamos convergencia, y sirve para hacer que un coche sea ms o menos gil en la entrada de curva, o ms estable en recta.
           <br/>
           <br/>
           Estos dos parmetros son fijos en la prctica totalidad de los coches (la convergencia puede variar en el tren trasero en coches con sistemas de suspensin multibrazo, con cojinetes de deformacin programada, o con direcciones a las cuatro ruedas). El problema para los ingenieros automovilsticos es que nos encontramos siempre en la calle del compromiso.
           <br/>
           <br/>
           En coche dirigidos al gran pblico resulta extremadamente complicado encontrar un compromiso que permita mantener un coche gil, pero que al mismo tiempo sea seguro, neutro y fcil de conducir.
           <br/>
           <br/>
           A la postre, ante la necesidad de crear coches fciles y estables para todas las personas en todas las circunstancias de uso (cargado a tope, lloviendo, cuesta arriba en un puerto de montaa de noche), se acaban optando por geometras que podramos llamar conservadoras.
           <br/>
           <br/>
           <img alt="" border="0" hspace="4" id="vimage_5420639" src="http://www.blogcdn.com/es.autoblog.com/media/2012/11/direccionesmecatronicas.jpg" vspace="4"/>
           <br/>
           <em>
            Sistema con cuatro actuadores independientes, uno por rueda
           </em>
           <br/>
           <br/>
           <p class="attention" style="margin-top: 2px; width: 240px; margin-bottom: 2px; float: left; margin-right: 8px">
            <em>
             <strong>
              <span style="color: #ff0000">
               <big>
                Un coche con este tipo de sistema puede ser al mismo tiempo ms estable en lnea recta y mucho ms gil en curva lenta y rpida
               </big>
              </span>
             </strong>
            </em>
           </p>
           Pues bien, la ventaja de un sistema mecatrnico apunta justo a esta problemtica. Se pueden optar por muchos tipos de sistemas mecatrnicos, conservando una cremallera de direccin convencional, como es el caso de Infiniti, por un tema legal que luego explicaremos, u optando por un sistema de cuatro ruedas directrices controladas por electrnica.
           <br/>
           <br/>
           En mi proyecto personal, que se qued en prototipo para un gran fabricante automovilstico, de cuyo nombre no puedo acordarme por temas de confidencialidad, se propona un sistema de cuatro ruedas directrices, con cuatro actuadores independientes, uno para cada una de las cuatro ruedas.
           <br/>
           <br/>
           Una veloz centralita electrnica calculaba la cantidad ideal de giro de cada rueda en cada milsima de segundo, en funcin de la guiada (giro sobre s mismo) del coche, el giro del volante, la velocidad, el deslizamiento, y una serie extra de parmetros, para conseguir un coche neutro en todo momento.
           <br/>
           <br/>
           Se poda prescindir de reglajes en esttico de convergencia para las cuatro ruedas, y al mismo tiempo se poda contar con un avance de prcticamente cero grados en las ruedas anteriores. Gracias a ello se poda conseguir un coche mucho ms gil, que gastaba mucho mejor los neumticos, y que poda interactuar con el sistema de control de estabilidad para salvar cualquier imprevisto dinmico (derrapes antes de que el conductor los perciba, por ejemplo), al tiempo que era estable en lnea recta.
           <br/>
           <br/>
           <img alt="" border="0" hspace="4" id="vimage_5420644" src="http://www.blogcdn.com/es.autoblog.com/media/2012/11/giromaspar.jpg" vspace="4"/>
           <br/>
           <em>
            Sistema en funcionamiento, con giro independiente de cada rueda
           </em>
           <br/>
           <br/>
           El concepto era equivalente al diseo de los aviones caza de combate modernos, que estn desarrollados sobre el principio de la inestabilidad. Un coche con este tipo de sistema de direccin mecatrnica es inestable sin el sistema electrnico activo. Aprovechar esta inestabilidad a favor de la agilidad del vehculo es la clave de la ventaja de este tipo de direcciones mecatrnicas.
           <br/>
           <br/>
           Los resultados eran claramente positivos y palpables. Podrs pensar "y si la electrnica falla?", bien, estos sistemas se hacen redundantes, con dos o tres juegos de sensores por toma, y todo por duplicado para evitar problemas. As funcionan en aviones comerciales y aviones militares, sin tasa sensible de fallo, por lo que replicarlo para la industria automovilstica no debera resultar complicado.
           <br/>
           <br/>
           Otro aspecto de desarrollo complejo es ofrecer un feedback adecuado a travs del volante, una informacin til para que el conductor pueda saber la cantidad de agarre disponible en las ruedas. Para ello, tal y como se usa en los volantes de los videojuegos, el propio volante del vehculo contaba con un sistema motorizado para generar fuerzas de autocentrado variables en funcin del agarre disponible, calculado en funcin del esfuerzo requerido en cada uno de los actuadores de las ruedas, creando una comunicacin importante para el conductor, y que poda ser adems regulada en su intensidad.
           <br/>
           <br/>
           La complejidad, y las pegas de este tipo de sistemas vienen de otro lado.
           <br/>
           <br/>
           El primer problema es la potencia de clculo de los sistemas electrnicos disponibles, el coste de los actuadores, y las enormes diferencias con los coches ya existentes en el mercado. Un sistema de este tipo ha de contar con una electrnica de clculo muy potente, que en el ao en el que yo la trabaj (2005) no era precisamente accesible para el mercado en masa. Adems, por la falta de potencia de clculo se generaban retrasos de dcimas de segundo, que aunque en teora no representaban problemas reales, s que se dejaban notar a la hora de conducir.
           <br/>
           <br/>
           Las cosas cambian muy rpido en este campo, y a da de hoy este ya no es un factor determinante, por lo que ahora s es viable industrialmente hablando.
           <br/>
           <br/>
           El otro problema, en cambio, es bastante ms grave. Y es que por cuestiones legales, no se puede homologar un coche con un volante completamente desacoplado de las ruedas. En el caso de Infiniti, por ejemplo, se va a usar un actuador sobre una cremallera de direccin convencional, en lugar de un actuador para cada rueda de manera independiente. Esto se va a hacer as para que, en caso de emergencia, un embrague acople el volante con la columna de direccin para evitar quedarnos sin direccin si el sistema falla.
           <br/>
           <br/>
           <img alt="" border="0" hspace="4" id="vimage_5420645" src="http://www.blogcdn.com/es.autoblog.com/media/2012/11/protoconsismtea.jpg" vspace="4"/>
           <br/>
           <em>
            Uno de los prototipos conceptuales creados para el proyecto, que no pas de estado de chasis mvil
           </em>
           <br/>
           <br/>
           Una problemtica similar tambin ocurra con los frenos "sensotrnicos" desarrollados por Mercedes, estrenados en los Clase E y SLR, y abandonados poco tiempo despus. Tenan que tener acoplado un sistema "backup" por si estos fallaban, para poder frenar el coche sin la ayuda electrnica.
           <br/>
           <br/>
           Esta problemtica legal homologativa es realmente un enorme escollo. Sera como si la agencia que regula las licencias de vuelo de nuevos aviones obligara a mantener un vnculo de unin por sirgas con la palanca de vuelo de los aviones. No tiene sentido.
           <br/>
           <br/>
           Las normativas europeas deberan adaptarse al potencial tecnolgico de estos sistemas y reescribirse. El acoplamiento mecnico es redundante siempre que los sistemas electrnicos estn bien pensados y estn duplicados o triplicados para un funcionamiento redundante (si falla un circuito, se usa un segundo subsistema de emergencia, o un tercer subsistema). De la misma manera que el sistema electrnico, an as, podra fallar, tambin se puede romper una junta cardnica en una columna de direccin.
           <br/>
           <br/>
           <p class="attention" style="margin-top: 2px; width: 240px; margin-bottom: 2px; float: left; margin-right: 8px">
            <em>
             <strong>
              <span style="color: #ff0000">
               <big>
                Las limitaciones de potencia de clculo electrnico se estn evaporando cada ao que pasa, pero las de homologacin siguen ah por el momento
               </big>
              </span>
             </strong>
            </em>
           </p>
           Desacoplar el giro direccional de las ruedas de una conexin mecnica con el volante cambiara el universo automovilstico de una manera inimaginable, ya que permitira configuraciones ms abiertas para el habitculo, versiones con volante a izquierda y derecha ms econmicas de desarrollar, coches ms rpidos, ms seguros, ms cmodos, e incluso coches autnomos de conduccin automatizada ms fciles de conceptualizar y desarrollar.
           <br/>
           <br/>
           Llegar, llegar... Pero en un mundo tan "controlado" por normas que velan por nuestra seguridad, pero que se exceden en su rigidez ante el avance de la tcnica, no estara de ms un poco de apertura al mundo tcnico para permitirnos evolucionar al ritmo que se haca a principios del siglo XX, cuando prcticamente cualquier coche poda montar cualquier innovacin sin tener que pasar por un frreo organismo regulador.
           <br/>
           <br/>
           Un aplauso desde aqu por la iniciativa de Infiniti, en todo caso.

function gcpComments() {
var options = {
		countId : '.aol-comments-',
		$countElem : $('.gcp-comment-count'),
		url : 'http://c.aol.com/read/get_topics',
		comment : ' Comentario',
		comments : ' Comentarios'
	};
        var topicIds = new Array();
        options.$countElem.each(function(i,e){
            topicIds.push(e.id.substring(13))
	    $(this).removeClass('gcp-comment-count')
        });
        if (topicIds.length &gt; 0) {
            var params = {
                channel_id : '248',
                topic_id: topicIds,
                version : 1
            }
            $.ajax({
                url : options.url,
                dataType: 'jsonp',
                data : params,
                traditional:true,
                cache: false,
                success: function(resp) {
                    var topics = resp.topicList;
                    var topicVsCount = {};
                    for (var i=0; i&lt;topics.length; i++) {
                        var t = topics[i],
                        	totalComments = t.totalComments;
                        if(!(window.location.href.split('/')[4]=="videos")){
                        	totalComments = ((totalComments &gt; 1 || totalComments == 0) ? totalComments + options.comments : totalComments + options.comment )
			}                         
                        $(options.countId + t.topicId).text(totalComments)
                        topicVsCount[t.topicId] = t.totalComments;
                        if ($(options.countId + t.topicId).is(':hidden')) {
                        	$(options.countId + t.topicId).show();
	                    }
                    }
                    for (i=0; i&lt;topicIds.length; i++) {
                        if (!topicVsCount[topicIds[i]] || topicVsCount[topicIds[i]] === null || topicVsCount[topicIds[i]] === 0) {
                            if ($(options.countId + topicIds[i]).is(':hidden')) $(options.countId + topicIds[i]).show();
                        }
                    }
                }
            });
        }
}

function GCPMostCommented(siteconfig) {
			siteconfig.sort = "commented";
			siteconfig.version = 1;
			$.ajax({
				url : 'http://c.aol.com/read/get_topics',
				dataType : 'jsonp',
				data : {channel_id : siteconfig.ChannelID, sort : siteconfig.sort, version : siteconfig.version, count : siteconfig.Count},
				traditional:true,
				cache: false,
				success: function(resp) {
					var topics = resp.topicList;
					var commentList = '&lt;div class="gcpTopicsListOuter"&gt;&lt;ul class="gcpTopicsListInner"&gt;';
					if (topics){
						for (var i=0; i &lt; topics.length; i++) {
							var t = topics[i];
							if (t.totalComments &gt; 0){
								commentList += '&lt;li class="gcpTopicsListChild"&gt;';
								if(siteconfig.numberedList){
									commentList += '&lt;span class="gcpPostNumber"&gt;' + (i+1) + '&lt;/span&gt;';
								}
								//Swallow dev, staging, sandbox etc. MONEY - aarghhh!
								if (t.permaLink.match(/money\.aol\./gi)) {
									t.permaLink = t.permaLink.replace(/(staging\.|stage\.|dev\.|[a-z0-9\-\_]+\.sandbox\.)/gi, '')
								}
								commentList += '&lt;a href="' + t.permaLink + '" title="' + t.title + '"&gt;' + t.title + '&lt;/a&gt;.. (' + t.totalComments + ')&lt;/li&gt;';
							}
						}
					} else {
						commentList += "&lt;li&gt;No posts with comments yet!&lt;/li&gt;";
					}
					commentList += "&lt;/ul&gt;&lt;/div&gt;";
					$(siteconfig.Container).html(commentList);
				}
			});
		}
