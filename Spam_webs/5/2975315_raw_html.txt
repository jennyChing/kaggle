      Random Lines Made With Mesh
     </h1>
     <p class="article_byline p-author h-card">
      <a class="u-url" href="#author">

    <div class="article_main e-content">
     <p class="lede">
      I know that Adobe Illustrator can be a bit daunting for people who arent really advanced users of the program, but you would be amazed by how easy you can create cool effects or backgrounds. In this short tutorial I show you how to create a cool looking background only in 5 steps.
     </p>
     <h2>
      Step 1  Create Lines
     </h2>
     <img alt="Step 1 - create lines" height="180" src="/examples/random-lines-made-with-mesh/mesh01.gif" width="326"/>
     <p>
      Create lines using random widths and harmonious suitable colors. If you get stuck on finding the right colors, check out
      <a href="http://kuler.adobe.com/">
       Adobes Kuler
      </a>
      and start experimenting.
     </p>
     <h2>
      Step 2  Convert Strokes to Fills
     </h2>
     <img alt="Step 2 - convert stroked to fills" height="180" src="/examples/random-lines-made-with-mesh/mesh02.gif" width="326"/>
     <p>
      Select all lines and convert them to fills. Go to the Object menu, select Path &gt; Outline Stroke. Select the Rectangle tool and draw 1 big rectangle on top the lines. Give the rectangle a suitable color. With the rectangle still selected, go to the Object menu, select Arrange &gt; Send to Back.
     </p>
     <h2>
      Step 3  Convert to Mesh
     </h2>
     <img alt="Step 3 - convert to mesh" height="180" src="/examples/random-lines-made-with-mesh/mesh03.gif" width="380"/>
     <p>
      Select all objects by pressing the command key (for Mac users), control key (for Windows users) + the a key. Go to the Object menu and select the Envelope Distort &gt; Make with Mesh option. Enter 2 rows and 2 columns. Check the preview box to see what happens and click the OK button.
     </p>
     <h2>
      Step 4  Play Around with The Mesh Points
     </h2>
     <img alt="step 4" height="258" src="/examples/random-lines-made-with-mesh/mesh04.gif" width="489"/>
     <p>
      Play around with the points of the mesh using the Direct Selection tool (the white arrow in the Toolbox). Click on the top right point of the mesh. Once youre starting to drag hold down the shift key and move the point upwards.
     </p>
     <img alt="step 5" height="400" src="/examples/random-lines-made-with-mesh/mesh05.gif" width="383"/>
     <p>
      Now start dragging the bezier handles on the mesh to achieve the effect as shown in the above picture. Of course you can try out all kind of different effects here.
     </p>
     <h2>
      The Final Result
     </h2>
     <img alt="final result - example" height="174" src="/examples/random-lines-made-with-mesh/mesh06.gif" width="550"/>
     <p>
      This is an example of how the final result can look. You can try out all kinds of different shapes dragging the handles of the mesh points. This is just one of the many results you can get. So next time you havent got inspiration for a background of a header, a banner or whatever, just experiment with a few basic shapes such as lines and try out the Envelope Distort options in Illustrator or the Make with Mesh option and experiment, youll be amazed by the unexpected creative results.
     </p>
