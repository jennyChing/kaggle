          <p class="graf--p graf--last" id="53d5" name="53d5">
           I thought Id share my workflow and give a few tips for how to make your designs a bit more future-proof and vector based in digital design. This article is about a workflow that is using vectors. It also gives you a few tricks about drawing techniques with Illustrator and handling vector based icons in Photoshop. Vector shapes are core parts of the next generation web but without taking the time to optimise them for screen usage. It almost doesnt matter if theyre not SVGs or high resolution images. Hopefully it will make your design process a lot better.

            </em>

           Create a new document and select the
           <em class="markup--em markup--p-em">

          </h4>
          <p class="graf--p" id="36a0" name="36a0">
           When designing an icon, most of the times you know where it is going to be, so you design for a certain size and purpose. Sometimes, I paste a screenshot of the interface into the Illustrator file to see what size would fit into the area Im designing for. What Id like to say with this, is that if you know what size you design for, you have a rough idea about the physics of your icon.
          </p>
          <p class="graf--p" id="bd58" name="bd58">
           When you need half size of an icon which has a 1px outline it is a lot more easier to scale and re-size it in Illustrator. At this point you its crucial to have the original editable version because scaling it in Photoshop would make the line-weight half size, so 0.5px.
          </p>
          <p class="graf--p" id="4f00" name="4f00">
           This doesnt mean you cant apply strokes in Photoshop but it only works on simple objects, like buttons or basic shapes. Its just a lot harder to create real assets for use if you do it that way. If youre designing for real web use, you need to create real assets, so in our case vector icons and illustrations.
          </p>
          <p class="graf--p graf--last" id="b4a3" name="b4a3">
           So here scaling means to multiply the drawings by 2x, 3x, 4x etc. because we are designing for different resolutions and shapes that are flexible and they use screens physics so its in the pixel space.
