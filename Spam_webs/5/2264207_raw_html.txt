    <h1>
     Readers Pick: 12 Excellent Websites to Follow if Youre into Web Design
    </h1>
    <p class="post-author">
     By Jacob Gube
    </p>
    <div class="wrap-bsap_1243576">
     <span class="ad-notice-bsap_1243576">
      Advertisement
      <div class="bsarocks bsap_6989dd4b5220d0b14530453de7387991" id="bsap_1243576">
      </div>
     </span>
    </div>
    <p>
     The  easiest way to keep up with the latest news and resources is by subscribing to RSS feeds of your favorite  sites. When I wrote an article called "
     <a href="http://sixrevisions.com/resources/websites_for_web_development/" title="http://sixrevisions.com/resources/websites_for_web_development/">
      30  Websites to follow if youre into Web Development
     </a>
     ", I knew I couldnt  include everyones favorites, so I asked readers to suggest other websites worth following.
    </p>
    <p>
     This is a collection of
     <em>
      just some
     </em>
     of your suggestions. Each entry  has a link directly to the sites RSS feed, but if you want to include all of  them into your feed reader, heres an
     <a href="http://sixrevisions.com/download/12_web_design_websites.zip">
      <strong>
       OPML file for all of them
      </strong>
     </a>
     .
    </p>
    <h3>
     1.
     <a href="http://css-tricks.com/" title="CSS-Tricks - Tips, Tricks, and Techniques on using Cascading Style Sheets.">
      CSS-Tricks
     </a>
    </h3>
    <p>
     <a href="http://css-tricks.com/">
      <img alt="CSS-Tricks - Screen shot" height="200" src="http://images.sixrevisions.com/2008/06/14-01_css_tricks.jpg" width="450"/>
     </a>
     <span class="figure-caption">
      Suggested by: Matthew Adams of
      <a href="http://www.101poundwebsite.com/" title="Home - 101PoundWebsite.com - 101 Pound Website - Low Cost Web Design">
       101PoundWebsite.com
      </a>
      ,  David Lano of
      <a href="http://www.davidlano.com/blog/" title="David Lano's Blog">
       David  Lanos Blog
      </a>
      , Jake of
      <a href="http://designerfied.com/" title="DesignerFied.com">
       DesignerFied.com
      </a>
      , and Nodster
     </span>
    </p>
    <p>
     <span id="more-166">
     </span>
    </p>
    <p>
     <em>
      CSS-Tricks
     </em>
     is a site by Chris Coyier that shares information  about CSS tips, tricks, techniques, and news. CSS-Tricks also has a fairly  active forums for those who want to talk about CSS and web design, as well as a
     <a href="http://css-tricks.com/videos/" title="CSS-Tricks | Video Screencasts">
      Video  section
     </a>
     containing screencasts on web development and design topics.
    </p>
    <ul>
     <li>
      <a href="http://feeds.feedburner.com/CssTricks" title="CSS-Tricks - powered by FeedBurner">
       CSS-Tricks articles feed
      </a>
     </li>
     <li>
      <a href="http://feeds.feedburner.com/CSS-Tricks-Screencasts" title="CSS-Tricks Screencasts - powered by FeedBurner">
       CSS-Tricks video feed
      </a>
     </li>
    </ul>
    <h3>
     2.
     <a href="http://devkick.com/" title="DevKick | Web Development for Designers">
      DevKick
     </a>
    </h3>
    <p>
     <a href="http://devkick.com/">
      <img alt="DevKick - Screen shot" height="200" src="http://images.sixrevisions.com/2008/06/14-02_devkick.jpg" width="450"/>
     </a>
     <span class="figure-caption">
      Suggested by: David of
      <a href="http://devkick.com/" title="DevKick | Web Development for Designers">
       DevKick
      </a>
     </span>
    </p>
    <p>
     <em>
      DevKick
     </em>
     focuses on web development articles geared towards  web designers. It covers topics such as
     <a href="http://devkick.com/blog/parsing-strings-with-jquery/" title="Parsing Strings With jQuery : DevKick Blog">
      Parsing strings with jQuery
     </a>
     ,
     <a href="http://devkick.com/blog/milestone-01-70-high-end-components-for-web-designers-and-developers/" title="Milestone 01 - 70+ High-End Components for Web Designers and Developers : DevKick Blog">
      web  components/widgets for web designers/developers
     </a>
     and inspirational posts  such as
     <a href="http://devkick.com/blog/design-inspiration-european-graphic-design-from-1950-1970/" title="Design Inspiration: European Graphic Design from 1950-1970 : DevKick Blog">
      Graphic  Design from 1950-1970
     </a>
     . They also have a News feed that shares resources  from around the web.
    </p>
    <ul>
     <li>
      <a href="http://feeds.feedburner.com/devkick/blog" title="DevKick Blog - powered by FeedBurner">
       DevKick Blog feed
      </a>
     </li>
     <li>
      <a href="http://feeds.feedburner.com/devkick/news" title="del.icio.us/devkick - powered by FeedBurner">
       DevKick News feed
      </a>
     </li>
    </ul>

     3.
     <a href="http://designbump.com/" title=" Design Bump">
      Design  Bump
     </a>
    </h3>
    <p>
     <a href="http://designbump.com/">
      <img alt="Design Bump - Screen shot" height="200" src="http://images.sixrevisions.com/2008/06/14-04_design_bump.jpg" width="450"/>
     </a>
     <span class="figure-caption">
      Suggested by: John Campbell of
      <a href="http://designbump.com/" title="Popular links | Design Bump">
       Design Bump
      </a>

      Design Bump
     </em>
     is a community-driven social media site  dedicated to design  much like
     <a href="http://www.designfloat.com/" title="Design Float / Popular Entries">
      Design Float
     </a>
     and
     <a href="http://www.dzone.com/links/index.html" title="DZone - fresh links for developers">
      DZone
     </a>
     (for developers). You can  subscribe to particular categories such as
     <em>
      Web Design
     </em>
     ,
     <em>
      Galleries and Showcases
     </em>
     ,
     <em>
      Graphics Design
     </em>
     ,
     <em>
      Programming
     </em>
     , and
     <em>
      Resources
     </em>
     .
    </p>
    <ul>
     <li>
      <a href="http://feeds.feedburner.com/DesignBump-All" title="Design Bump - Popular news - powered by FeedBurner">
       Design Bump Popular  feed
      </a>
     </li>
    </ul>
