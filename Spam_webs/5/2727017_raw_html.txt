                    <h3 class="post-title entry-title">
                     <a href="http://www.cookshideout.com/2014/11/vegetarian-meatloaf-with-rice-lentils.html">
                      Vegetarian Meatloaf (with Rice &amp; Lentils) with Checca Suace
                     </a>
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content">
                     <div dir="ltr" style="text-align: left;" trbidi="on">
                      <b>
                       <i>
                        Blogging Marthon# 46: Week 1/ Day 2
                       </i>
                      </b>
                      <br/>
                      <b>
                       <i>
                        Theme: Festival Special -- Thanksgiving
                       </i>
                      </b>
                      <br/>
                      <b>
                       <i>
                        Dish: Vegetarian Meatloaf (with Rice &amp; Lentils) with Checca Sauce
                       </i>
                      </b>
                      <br/>
                      <div style="text-align: justify;">
                       <a href="http://well.blogs.nytimes.com/2010/11/22/a-vegetarian-thanksgiving-even-for-carnivores/">
                        Well's Vegetarian Thanksgiving
                       </a>
                       from New York Times is a great resource for vegetarian and vegan dishes to make for Thanksgiving. The New York Times editors curate a huge list of non-traditional no-turkey/ meat dishes on their
                       <a href="http://cooking.nytimes.com/68861692-nyt-cooking/445045-a-wells-vegetarian-thanksgiving">
                        Well's blog
                       </a>
                       . I think they have been doing this every year since 2010. They have an impressive line up of dishes from famous food writers and chefs utilizing the fall harvest and delicious vegetables.
                      </div>
                      <div class="separator" style="clear: both; text-align: center;">
                       <a href="http://1.bp.blogspot.com/-CJ48RaZ75sQ/VFfXdUI4G6I/AAAAAAAASeg/r3AQj_G5eEM/s1600/Veggie%2BMeatloaf3.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                        <img alt="Vegetarian Meatloaf (with Rice &amp; Lentils) with Checca Suace" border="0" src="http://1.bp.blogspot.com/-CJ48RaZ75sQ/VFfXdUI4G6I/AAAAAAAASeg/r3AQj_G5eEM/s1600/Veggie%2BMeatloaf3.jpg" title="Vegetarian Meatloaf (with Rice &amp; Lentils) with Checca Suace"/>
                       </a>
                      </div>
                      <div style="text-align: justify;">
                       The idea to make a Vegetarian meatloaf came from one of the Well's Thanksgiving menus.
                       <a href="http://en.wikipedia.org/wiki/Meatloaf">
                        Meatloaf
                       </a>
                       is exactly what it sounds like --- meat made into a loaf. Most vegetarian meatloaves are made of lentils, mushrooms or TVP (soy granules). The recipe I used today uses both lentils and rice for the structure. I remember watching Giada making it on Food Network and wanted to try the recipe.
                      </div>
                      <a name="more">
                      </a>
                      <br/>
                      <div class="separator" style="clear: both; text-align: center;">
                       <a href="http://2.bp.blogspot.com/-7FQjV95zz4Y/VFfXbxnk9tI/AAAAAAAAShg/Cc8gm47GVgU/s1600/Veggie%2BMeatloaf.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                        <img alt="Vegetarian Meatloaf (with Rice &amp; Lentils) with Checca Suace" border="0" src="http://2.bp.blogspot.com/-7FQjV95zz4Y/VFfXbxnk9tI/AAAAAAAAShg/Cc8gm47GVgU/s1600/Veggie%2BMeatloaf.jpg" title="Vegetarian Meatloaf (with Rice &amp; Lentils) with Checca Suace"/>
                       </a>
                      </div>
                      <div style="text-align: justify;">
                       To speed up the process, I cooked the rice and lentils in a pressure cooker instead of in a open pot. I used a 8"x4" loaf pan instead of the strange 10"x4" pan recommended in the original recipe. I added an egg just to make sure that the loaf holds its shape, But I think that egg can easily be substituted with either ground flax meal or egg replacer. It was a little difficult to get the first piece out of the pan, but cooling the loaf did the trick for me.
                      </div>
                      <div class="separator" style="clear: both; text-align: center;">
                       <a href="http://1.bp.blogspot.com/-DQ6CPMAXF1g/VFfXd3auhQI/AAAAAAAASes/UPLlNi8pvC8/s1600/Veggie%2BMeatloaf4.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                        <img alt="Vegetarian Meatloaf (with Rice &amp; Lentils) with Checca Suace" border="0" src="http://1.bp.blogspot.com/-DQ6CPMAXF1g/VFfXd3auhQI/AAAAAAAASes/UPLlNi8pvC8/s1600/Veggie%2BMeatloaf4.jpg" title="Vegetarian Meatloaf (with Rice &amp; Lentils) with Checca Suace"/>
                       </a>
                      </div>
                      <div style="text-align: justify;">
                       This is a very flavorful and filling meatloaf. Checca sauce is a raw tomato sauce that is added to the loaf itself and is also served on the side for added flavor. My favorite part is the melted cheese and the tomatoes on the top of the loaf. Also I halved the recipe since I was cooking just for myself. I froze the leftovers for later meals.
                       <span style="text-align: left;">
                       </span>
                       <span style="text-align: left;">
                       </span>
                      </div>
                      <div class="separator" style="clear: both; text-align: center;">
                       <a href="http://2.bp.blogspot.com/-0IYD6u0WrUw/VFfXe059y6I/AAAAAAAAShM/l3YKBvQ4F5A/s1600/Veggie%2BMeatloaf6.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                        <img alt="Vegetarian Meatloaf (with Rice &amp; Lentils) with Checca Suace" border="0" src="http://2.bp.blogspot.com/-0IYD6u0WrUw/VFfXe059y6I/AAAAAAAAShM/l3YKBvQ4F5A/s1600/Veggie%2BMeatloaf6.jpg" title="Vegetarian Meatloaf (with Rice &amp; Lentils) with Checca Suace"/>
                       </a>
                      </div>
                      <br/>
                      <div class="separator" style="clear: both; text-align: center;">
                       <a href="http://2.bp.blogspot.com/-74W3C9k5Hgw/VFfXgNVaBQI/AAAAAAAASfc/ICiJO7R9tqM/s1600/Veggie%2BMeatloaf8.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                        <img alt="Vegetarian Meatloaf (with Rice &amp; Lentils) with Checca Suace" border="0" src="http://2.bp.blogspot.com/-74W3C9k5Hgw/VFfXgNVaBQI/AAAAAAAASfc/ICiJO7R9tqM/s1600/Veggie%2BMeatloaf8.jpg" title="Vegetarian Meatloaf (with Rice &amp; Lentils) with Checca Suace"/>
                       </a>
                      </div>
                      <i>
                       Recipe adapted from
                       <a href="http://www.foodnetwork.com/recipes/giada-de-laurentiis/veronicas-veggie-meatloaf-with-checca-sauce-recipe.html#sni-reviews">
                        here
                       </a>
                      </i>
                      .
                      <br/>
                      <div style="text-align: justify;">
                       <b>
                        Ingredients
                       </b>
                       :
                       <i>
                        Serves 3~4
                       </i>
                       <br/>
                       <b>
                        <i>
                         For the Meatloaf
                        </i>
                       </b>
                       :
                       <br/>
                       Lentils - cup, soaked for 30 minutes (I used Brown lentils, but green lentils can also be used)
                       <br/>
                       Rice - cup (I used Jasmine rice, original recipe used brown rice)
                       <br/>
                       Carrot - 1 small, diced
                       <br/>
                       Onion - 1 small, finely chopped
                       <br/>
                       Frozen Corn - 3tbsp
                       <br/>
                       Frozen Spinach - cup (after thawing) or use 2 cups of fresh spinach that is cooked and squeezed dry
                       <br/>
                       Butter - 1tbsp
                       <br/>
                       Egg - 1, lightly beaten (can also use vegan egg substitutes like egg replacer or flax seeds)
                       <br/>
                       Mozzarella Cheese - cup, cubed, divided (I used grated mozzarella)
                       <br/>
                       Parmesan cheese - 2tbsp, grated
                       <br/>
                       Cilantro - cup, chopped
                       <br/>
                       Tomato - 1 medium, sliced
                       <br/>
                       Salt &amp; Pepper - to taste
                       <br/>
                       <br/>
                       <i style="font-weight: bold;">
                        For the Checca Sauce
                       </i>
                       :
                       <br/>
                       Ripe Tomatoes - 2 medium, chopped
                       <br/>
                       Onion - 2tbsp
                       <br/>
                       Garlic - 1 clove, minced
                       <br/>
                       Extra virgin Olive oil - 1tbsp
                       <br/>
                       Cilantro - 2tbsp, finely chopped
                       <br/>
                       Salt &amp; Pepper - to taste
                       <br/>
                       <br/>
                       <b>
                        Method
                       </b>
                       :
                       <br/>
                       <ul>
                        <li>
                         Preheat the oven to 350F. Grease a 8"x4" loaf pan with 1tsp butter.
                        </li>
                        <li>
                         <i style="font-weight: bold;">
                          Make the Checca Sauce
                         </i>
                         : Grind all the ingredients for the checca sauce until tomatoes are coarsely chopped. Make sure that the tomatoes are not pureed,
                        </li>
                        <li>
                         <b>
                          <i>
                           For the Lentil Loaf
                          </i>
                         </b>
                         : Heat 2tsp oil in a pressure cooker; add the onions, carrots and cook for 2~3 minutes or until the veggies are slightly tender.
                        </li>
                        <li>
                         Drain the lentils and add to the veggies. Next add the rice and 1cups of water. Season with salt and pepper and cook till rice and lentils are tender. I cooked for 2~3 whistles in my cooker.
                        </li>
                        <li>
                         Let the pressure of the cooker release naturally, open the lid and add the frozen corn, set aside for 5 minutes. Gently fluff the rice and cool for 5 minutes.
                        </li>
                        <li>
                         In a mixing bowl, combine the rice &amp; lentil mixture, half of the Checca sauce, thawed frozen spinach, beaten egg, half of the mozzarella cheese, 1tbsp parmesan cheese, chopped cilantro and season with salt &amp; pepper. Mix gently.
                        </li>
                        <li>
                         Transfer the mixture into the prepared loaf pan. Place the tomatoes in 1 layer, sprinkle the remaining mozzarella cheese and parmesan cheese. Dot the top with the remaining 2tsp butter.
                        </li>
                        <li>
                         Bake for 30~35 minutes or until the lentil mixture is heated through and the top is browned and bubbly.
                        </li>
                        <li>
                         Set aside for 15~20 minutes before slicing. Serve with checca sauce on the side.
                        </li>
                       </ul>
                      </div>

                : parseInt(parsed[parsed.length - 1].timestamp) + 1
            callback(parsed);
            window.bloggercomments = null;
          }
          url += '&amp;callback=bloggercomments';
          var script = document.createElement('script');
          script.type = 'text/javascript';
          script.src = url;
          document.getElementsByTagName('head')[0].appendChild(script);
        }
      };
      var hasMore = function() {
        return !!cursor;
      };
      var getMeta = function(key, comment) {
        if ('iswriter' == key) {
          var matches = !!comment.author
              &amp;&amp; comment.author.name == config.authorName
              &amp;&amp; comment.author.profileUrl == config.authorUrl;
          return matches ? 'true' : '';
        } else if ('deletelink' == key) {
          return config.baseUri + '/delete-comment.g?blogID='
               + config.blogId + '&amp;postID=' + comment.id;
        } else if ('deleteclass' == key) {
          return comment.deleteclass;
        }
        return '';
      };

      var replybox = null;
      var replyUrlParts = null;
      var replyParent = undefined;

      var onReply = function(commentId, domId) {
        if (replybox == null) {
          // lazily cache replybox, and adjust to suit this style:
          replybox = document.getElementById('comment-editor');
          if (replybox != null) {
            replybox.height = '250px';
            replybox.style.display = 'block';
            replyUrlParts = replybox.src.split('#');
          }
        }
        if (replybox &amp;&amp; (commentId !== replyParent)) {
          document.getElementById(domId).insertBefore(replybox, null);
          replybox.src = replyUrlParts[0]
              + (commentId ? '&amp;parentID=' + commentId : '')
              + '#' + replyUrlParts[1];
          replyParent = commentId;
        }
      };

      var hash = (window.location.hash || '#').substring(1);
      var startThread, targetComment;
      if (/^comment-form_/.test(hash)) {
        startThread = hash.substring('comment-form_'.length);
      } else if (/^c[0-9]+$/.test(hash)) {
        targetComment = hash.substring(1);
      }

      // Configure commenting API:
      var configJso = {
        'maxDepth': config.maxThreadDepth
      };
      var provider = {
        'id': config.postId,
        'data': items,
        'loadNext': paginator,
        'hasMore': hasMore,
        'getMeta': getMeta,
        'onReply': onReply,
        'rendered': true,
        'initComment': targetComment,
        'initReplyThread': startThread,
        'config': configJso,
        'messages': msgs
      };

      var render = function() {
        if (window.goog &amp;&amp; window.goog.comments) {
          var holder = document.getElementById('comment-holder');
          window.goog.comments.render(holder, provider);
        }
      };

      // render now, or queue to render when library loads:
      if (window.goog &amp;&amp; window.goog.comments) {
        render();
      } else {
        window.goog = window.goog || {};
        window.goog.comments = window.goog.comments || {};
        window.goog.comments.loadQueue = window.goog.comments.loadQueue || [];
        window.goog.comments.loadQueue.push(render);
