       Features
      </h4>
     </header>
     <article>
      <ul>
       <li>
        <strong>
         Smooth scrolling
        </strong>
        by disabling pointer events
       </li>
       <li>
        <strong>
         Lightweight
        </strong>
        at around 1KB minified
       </li>
       <li>
        <strong>
         Refreshes
        </strong>
        on window resize and DOM mutation
       </li>
       <li>
        <strong>
         Compatible
        </strong>
        with the latest version of your favorite browser
       </li>
      </ul>
