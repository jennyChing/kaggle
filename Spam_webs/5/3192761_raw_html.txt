   <h1>
    Using Japanese language on your computer
   </h1>
   <div class="lead">
    <p>
     You can display
     <img alt="Nihongo" height="13" src="nihongo.gif" width="49"/>
     , or Japanese, on your computer in some ways. With appropriate software, you can type and edit Japanese as well (i.e. no specific hardware required). This page provides some (relatively historical) information to help in handling Japanese.
    </p>
   </div>
   <section>
    <h2 id="jp-helper">
     Softwares that help Japanese handling
    </h2>
    <p>
     Note that the latest OSs such as Windows 98/2000/XP and MacOS 8/9/X should have the
     <em>
      built-in functionality
     </em>
     to provide world wide language support (e.g., install all Japanese supports with English version of Windows 2000, and set 'default language' to Japanese). Please consult your manual or ask a computer shop.
    </p>
    <section>
     <h3 id="dos">
      DOS/Windows and Japanese
     </h3>
     <ul>
      <li class="foaf.Document">
       <a class="about" href="http://www.microsoft.com/windows/ie/ie6/downloads/recommended/ime/default.mspx">
        Microsoft Global IME
       </a>
       - by launching any language version of Word 2000, Internet Explorer 5.01 or greater, or Outlook Express then selecting Global IME 5.02 from the taskbar,
       <span class="description">
        you can write the language you choose. And you can input Chinese, Japanese, and Korean text into supported applications.
       </span>
      </li>
      <li class="foaf.Organization">
       <a class="ifp" href="http://www.unionway.com/">
        UnionWay
       </a>
       <span class="description">
        provides
        <em>
         AsianSuite
        </em>
        , a suite of multilingual Windows applications which run on any of Microsoft Windows products and supports for Chinese, Japanese, and Korean language on Windows.
       </span>
       It also has a multilingual
       <abbr title="Peer to peer">
        P2P
       </abbr>
       software.
      </li>
      <li class="foaf.Organization">
       <a class="ifp" href="http://www.njstar.com/">
        NJSTAR
       </a>
       from Australia
       <span class="description">
        provides CJK Viewer and Word Processors
       </span>
       -
       <a href="http://www.njstar.com/njwin/">
        NJWIN
       </a>
       is a "plugin" software designed to enable normal windows programs to display and print CJK characters under the standard English and Western Windows environment. Also,
       <a href="http://www.njstar.com/njstar/japanese/">
        Japanese Word Processor
       </a>
       for Windows 95/98/ME/NT/2000/XP.
       <!-- DOS version of the program also available.-->
      </li>
      <li class="foaf.Organization">
       <a class="ifp" href="http://www.twinbridge.com/">
        TWINBRIDGE
       </a>
       -
       <span class="description">
        its
        <em>
         CJK Partner
        </em>
        allows Enable you to display, edit and print Chinese, Japanese or Korean documents
       </span>
       , supporting most internal codes including Unicode for CJK characters. Japanese font download also available.
      </li>
      <li class="foaf.Document">
       Basis Technology's
       <a class="about" href="http://www.basistech.com/rj/">
        RJ (Read Japanese!)
       </a>
       is
       <span class="description">
        a Windows program that helps you read and understand Japanese. Simply click on characters and words to instantaneously view their pronunciation, meaning, and much more information.
       </span>
      </li>
      <li class="foaf.Document">
       <a class="about" href="http://ftp.cc.monash.edu.au/pub/nihongo/00INDEX.html">
        The Monash Nihongo ftp Archive
       </a>
       <span class="description">
        provides very rich resources to use Japanese language, though some technical knowledge may required.
       </span>
      </li>
      <li class="foaf.Document">
       <a class="about" href="http://www.jw-stumpel.nl/kdplus.html">
        KDPlus
       </a>
       by Jan Stumpel is an improved version of the KD kanji driver program by Izumi Ohzawa.
       <span class="description">
        KDPLUS can be used to obtain Japanese character display, and allow Japanese character input, in many text-mode programs for MS-DOS (historical).
       </span>
      </li>
      <!--ftp://ftp.ora.com/pub/examples/nutshell/ujip/dos/-->
     </ul>
     <ul class="archive">
      <li>
       NeocorTech's
       <a href="http://www.coyer.com/neocor/tsunami/">
        Tsunami MT Version 6.0
       </a>
       English to Japanese translator provides both a Japanese Kanji and Kana text editor/FEP, a machine translator, and cross-platform compatibility (NeocorTech was acquired by speech and language technology leader Lernout &amp; Hauspie, which went bankrupt in 2001. So who knows about Tsunami today ?).
      </li>
     </ul>

    </section>

   <section>
    <h2 id="general">
     General Information on Japanese processing
    </h2>
    <ul>
     <li class="foaf.Document">
      <a class="about" href="http://www.csse.monash.edu.au/~jwb/japanese.html">
       <strong>
        Jim Breen's Japanese Page
       </strong>
      </a>
      <span class="description">
       provides a lot of interesting and useful things related to Japanese processing.
      </span>
      Quite active.
     </li>
     <li class="foaf.Organization">
      <a class="ifp" href="http://www.tjp.washington.edu/">
       The Technical Japanese Program
      </a>
      at University of Washingtons is
      <span class="description">
       a unique inter-engineering program that combines study in advanced technical Japanese with graduate-level engineering courses.
      </span>
     </li>
     <li class="foaf.Document">
      <a class="about" href="http://www.lfw.org/text/jp.html">
       Japanese text encoding
      </a>
      is
      <span class="description">
       a useful document describing some technical aspects of Japanese on computer.
      </span>
     </li>
     <li class="foaf.Document">
      <a class="about" href="http://fll.stcloudstate.edu/hi/japanese/japaneseindex.html">
       The Japanese Homepage
      </a>
      by Foreign Languages and Literature, St. Cloud State University, is
      <span class="description">
       a source of Japan and Japanese language.
      </span>
     </li>
     <li class="foaf.Document">
      <a class="about" href="http://www.sla.purdue.edu/academic/fll/JapanProj/Je-mail/">
       A Guide to Japanese E-mail
      </a>
      , by Atsushi Fukada &amp; Kazumi Hatasa at Purdue University, is
      <span class="description">
       a good documentation on  Japanese and methods to use it in e-mail
      </span>
     </li>
    </ul>
    <ul class="archive">
     <li>
      <a href="http://lundestudio.com/">
       Ken Lunde
      </a>
      , the author of the magnificent book on Japanese,
      <em>
       <a href="http://lundestudio.com/ujip_info.html">
        Understanding Japanese Information Processing
       </a>
      </em>
      provided some Japanese code manipulation tools. Archives of his
      <a href="http://web.archive.org/web/20061205022408/http://praxagora.com/lunde/cjk_inf.html" title="CJK.INF as of 2006-12-05">
       <strong>
        CJK.INF
       </strong>
      </a>
      file
is detailed online document explaining Chinese, Japanese and Korean language character sets and endcoding.
     </li>
     <li>
      (All Douglas Horn's pages seem disappeared 2003...)
      <a href="http://web.archive.org/web/20030803034724/http://www.halcyon.com/horn/default.htm">
       Douglas Horn
      </a>
      wrote many articles on Japanese language processing, including
      <a href="http://web.archive.org/web/20030605150246/http://www.halcyon.com/horn/pages/mj_excu.htm">
       It's time to teach your computer Japanese
      </a>
      ,
      <a href="http://web.archive.org/web/20030817114447/http://www.halcyon.com/horn/pages/mj_qa.htm">
       Q &amp; A
      </a>
      and
      <a href="http://web.archive.org/web/20030817110036/http://www.halcyon.com/horn/pages/mlc_jint.htm">
       Japanese Internet Primer
      </a>
      .
     </li>
    </ul>

    <h2 id="netscape">
     To display Japanese with:
    </h2>
    <section>
     <h3>
      Netscape Navigator and Internet Explorer
     </h3>
     <p>
      <img alt="* Menu names may be different according to the browser version" class="right" height="129" src="ns-setting.gif" width="228"/>
      If you have Japanese system installed on your computer, things are qute simple.
     </p>
     <ol>
      <li>
       Pull down "Options" menu from the menubar
(or "
       <code class="attn">
        View
       </code>
       " menu if you use version 4)
      </li>
      <li>
       Select
       <strong>
        Japanese (Auto-Detect)
       </strong>
       from submenu of Document Encoding (or
       <code class="attn">
        Character Set
       </code>
       or
       <code class="attn">
        Font
       </code>
       )
      </li>
      <li>
       If necessary, save your options (or "Set default")
      </li>
     </ol>

    <section>
     <h3>
      or if you prefer Lynx
     </h3>
     <ul>
      <li>
       Obtain your copy of Lynx from, for example,
       <a href="http://lynx.browser.org/" itemprop="about" title="Lynx Information">
        http://lynx.browser.org/
       </a>
       . Now Win95 version is available.
      </li>
      <li>
       Open configuration file (i.e. Lynx.cfg)
      </li>
      <li>
       Find CHRACTER_SET option and set it to Japanese (SJIS)
      </li>
      <li>
       Run Lynx. If still has a problem, check Option and set CJK mode ON
      </li>
     </ul>
