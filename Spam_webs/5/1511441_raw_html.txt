     <section class="primary island c">
      <h2>
       About this Add-on
      </h2>
      <div class="prose">
       <div class="prose" id="addon-description">
        <a href="http://outgoing.mozilla.org/v1/4cf4c19a1a8e5c79cec4fcf39ff317efb059aaf6/http%3A//Bookmax.net" rel="nofollow">
         Bookmax.net
        </a>
        is an online bookmark manager to store bookmarks, lists, contacts, memos, notes and more online.
        <br/>
        <br/>
        Features:
        <br/>
        Easy drag and drop handling
        <br/>
        Hierarchical Folders
        <br/>
        Search function for all your entries
        <br/>
        Favorites list for fast access
        <br/>
        3 kind of bookmarks views with thumbnails or favicons
        <br/>
        Statistics of Last Used, Most Used and Last Added bookmarks.
        <br/>
        Import (Premium) and Export Tools
        <br/>
        <br/>
        This Add-on lets you add bookmarks with a button in the navigation bar and view your list with a menu in the Extras menu and a button in the add-on bar.
        <br/>
        <br/>
        <br/>
        Create all kinds of lists, like To Do Lists, Friends Birthdays, Collections, Invoices, etc.
        <br/>
        Lists can have Texts (short and long), Numbers (with sum), Dates, Pulldowns or Checkmarks.
        <br/>
        <br/>
        Save short Memos and long Notes.
        <br/>
        Inlcude your favourite RSS Feeds as News.
        <br/>
        <br/>
        Choose a Template or create your own.
       </div>
      </div>
