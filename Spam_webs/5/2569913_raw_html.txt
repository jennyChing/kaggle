         <p>
          The canvas element is part of HTML5 and allows for dynamic, scriptable rendering of 2D shapes and bitmap images.
          <br/>
          Canvas consists of a drawable region defined in HTML code with height and width attributes.
          <br/>
          JavaScript code may access the area through a full set of drawing functions similar to those of other common 2D APIs, thus allowing for dynamically generated graphics.
         </p>
         <p>
          In this example Im going to show you
          <strong>
           how to animate a single bouncing ball
          </strong>
          on a web page using the canvas tag.
         </p>
         <p>
          <a href="http://www.xappsoftware.com/wordpress/wp-content/uploads/2015/01/h5canvas_img.jpg">
           <img alt="h5canvas_img" class="aligncenter size-full wp-image-7352" height="240" src="http://www.xappsoftware.com/wordpress/wp-content/uploads/2015/01/h5canvas_img.jpg" width="223"/>
          </a>
          <br/>
          The html page is very simple, it will look like the following one:
         </p>
         <p>
          <code>
           &lt;html&gt;
          </code>
          <br/>
          <code>
           &lt;body onLoad="init();"&gt;
          </code>
          <br/>
          <code>
           &lt;canvas id="ballCanvas" width="300" height="300" &gt;
          </code>
          <br/>
          <code>
           &lt;/canvas&gt;
          </code>
          <br/>
          <code>
           &lt;/body&gt;
          </code>
          <br/>
          <code>
           &lt;script src="js/bBall.js"&gt; &lt;/script&gt;
          </code>
          <br/>
          <code>
           &lt;/html&gt;
          </code>
         </p>
         <p>
          <strong>
           Lets start coding
          </strong>
          <br/>
          As you can see we will start our animation calling the init function when the html body has been loaded, the init function will retrieve the DOM node for the canvas, then it will get the drawing context using
          <strong>
           getContext
          </strong>
          () method.
         </p>
         <p>
          <code>
           var theCanvas;
          </code>
          <br/>
          <code>
           var theContext;
          </code>
          <br/>
          <code>
           init()
          </code>
          <br/>
          <code>
           {
          </code>
          <br/>
          <code>
           theCanvas = document.getElementById('theCanvas');
          </code>
          <br/>
          <code>
           theContext = theCanvas.getContext('2d');
          </code>
          <br/>
          <code>
           }
          </code>
         </p>
         <p>
         </p>
         <p>
          Lets start drawing the ball:
          <br/>
          we will need at least the following info for the ball:
          <br/>
           the position, expressed in x and y
          <br/>
           the starting position for the ball (initially sets to the center of the canvas 150,150)
          <br/>
           the direction, expressed in deltax and deltay
          <br/>
           the direction for the ball sets to 5,-10
         </p>
         <p>
          <code>
           var x=150;
          </code>
          <br/>
          <code>
           var y=150;
          </code>
          <br/>
          <code>
           var deltax=5;
          </code>
          <br/>
          <code>
           var deltay=-10;
          </code>
         </p>
         <p>
          and now the draw function:
         </p>
         <p>
          <code>
           function draw()
          </code>
          <br/>
         </p>
         <div style="padding:7px; float: left; padding-left: 0px; margin: 3px;">
          <script type="text/javascript">
           google_ad_client = "pub-4434139616132571"; 
    	google_ad_channel ="";

         <code>
          {
         </code>
         <br/>
         first of all we will clear the context, so each time well draw the ball on a white canvas.
         <br/>
         <code>
          context.clearRect(0,0, 300,300);
         </code>
         <br/>
         <code>
          then we'll draw the ball using the arc method
         </code>
         <br/>
         <code>
          theContext.fillStyle="#0000ff";
         </code>
         <br/>
         <code>
          theContext.arc(x,y,20,0,Math.PI*2,true);
         </code>
         <br/>
         <code>
          theContext.closePath();
         </code>
         <br/>
         <code>
          theContext.fill();
         </code>
         <br/>
         last we have to compute next position (if the new position go out of the canvas area we will change the direction)
         <br/>
         <code>
          if( x&lt;0 || x&gt;300)
         </code>
         <br/>
         <code>
          dx=-dx;
         </code>
         <br/>
         <code>
          if( y&lt;0 || y&gt;300)
         </code>
         <br/>
         <code>
          dy=-dy;
         </code>
         <br/>
         <code>
          x+=dx;
         </code>
         <br/>
         <code>
          y+=dy;
         </code>
         <br/>
         <code>
          }
         </code>
         <p>
          Finally we have to call the draw function, well call the function every 20 milliseconds from the init function using the setInterval(function)
          <br/>
          <code>
           init()
          </code>
          <br/>
          <code>
           {
          </code>
          <br/>
          <code>
           theCanvas = document.getElementById('theCanvas');
          </code>
          <br/>
          <code>
           theContext = theCanvas.getContext('2d');
          </code>
          <br/>
          <code>
           setInterval(draw,20);
          </code>
          <br/>
          <code>
           }
          </code>
         </p>
