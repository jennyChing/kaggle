   <h1 class="firstHeading" id="firstHeading" lang="en">
    Reverse vending machine
   </h1>
   <div class="mw-body-content" id="bodyContent">
    <div id="siteSub">

       <div class="thumbcaption">

     <p>
      A
      <b>
       reverse vending machine
      </b>
      is a device that accepts used (empty)
      <a class="mw-redirect" href="/wiki/Beverage_containers" title="Beverage containers">
       beverage containers
      </a>
      and returns money to the user. The machines are popular in places that have mandatory
      <a href="/wiki/Recycling" title="Recycling">
       recycling
      </a>
      laws or
      <a href="/wiki/Container_deposit_legislation" title="Container deposit legislation">
       container deposit legislation
      </a>
      . In some places, bottlers paid funds into a centralized pool to be disbursed to people who recycled the containers. Any excess funds were to be used for general environmental cleanup.
      <sup class="reference" id="cite_ref-1">
       <a href="#cite_note-1">
        <span>
         [
        </span>
        1
        <span>
         ]
        </span>
       </a>
      </sup>
      In other places, such as
      <a href="/wiki/Norway" title="Norway">
       Norway
      </a>
      , the state mandated that a vendor pay for recycled bottles, but left the system in the hands of private industry.
     </p>
     <p>
      The main vendors of reverse vending machines are
      <a href="/wiki/Tomra" title="Tomra">
       Tomra
      </a>
      of
      <a href="/wiki/Norway" title="Norway">
       Norway
      </a>
      and
      <a href="/wiki/Wincor_Nixdorf" title="Wincor Nixdorf">
       Wincor Nixdorf
      </a>
      of
      <a href="/wiki/Germany" title="Germany">
       Germany
      </a>
      , and
      <a href="/wiki/Envipco" title="Envipco">
       Envipco
      </a>
      of USA, Envirobank of Australia and Reverse Vending Corporation.
     </p>
     <p>
     </p>

     <h2>
      <span class="mw-headline" id="Operation">
       Operation
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Reverse_vending_machine&amp;action=edit&amp;section=1" title="Edit section: Operation">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      The recycler places the empty bottle/can into the receiving aperture; the horizontal in-feed system allows the user to insert containers one at a time. (An alternative system, found in many older machines, is one in which the user opens a door by hand and places the empty container in a pan. When the door is released and closed, the process continues.) The bottle/can is then automatically rotated; the bottle/can is then scanned by an Omnidirectional
      <a href="/wiki/Universal_Product_Code" title="Universal Product Code">
       UPC
      </a>
      Scanner, which scans the beverage container's
      <a href="/wiki/Universal_Product_Code" title="Universal Product Code">
       UPC
      </a>
      .
     </p>
     <p>
      Once a container is scanned, identified (matched to database) and determined to be a participating container, it is processed and typically crushed (for one-time-use Containers) to reduce its size, to avoid spillages of liquid and to increase storage capacity. Re-fillable containers are collected and sorted by hand to be brought back to the bottling company. The machines can use
      <a class="new" href="/w/index.php?title=Material_recognition&amp;action=edit&amp;redlink=1" title="Material recognition (page does not exist)">
       material recognition
      </a>
      instead of/as well as a
      <a class="mw-redirect" href="/wiki/Bar_code_scanner" title="Bar code scanner">
       bar code scanner
      </a>
      when needed.
      <sup class="reference" id="cite_ref-2">
       <a href="#cite_note-2">
        <span>
         [
        </span>
        2
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <h2>
      <span class="mw-headline" id="History">
       History
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Reverse_vending_machine&amp;action=edit&amp;section=2" title="Edit section: History">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      The first patent for a Bottle Return and Handling Machine was filed in America in 1920.
      <sup class="reference" id="cite_ref-ReverseVending_About_3-0">
       <a href="#cite_note-ReverseVending_About-3">
        <span>
         [
        </span>
        3
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <p>
      The first working Bottle Return Machine was invented and manufactured by Wicanders from Sweden used in the late 1950s.
      <sup class="reference" id="cite_ref-ReverseVending_About_3-1">
       <a href="#cite_note-ReverseVending_About-3">
        <span>
         [
        </span>
        3
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <p>
      In 1962 an advanced Automatic Bottle Return Machine was designed by Aage Tveitan and manufactured in Norway by his company Arthur Tveitan ASA.
      <sup class="reference" id="cite_ref-ReverseVending_About_3-2">
       <a href="#cite_note-ReverseVending_About-3">
        <span>
         [
        </span>
        3
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
