             Install Atom a Hackable Text Editor for Ubuntu, Linux Mint, Elementary OS

            <p>
             <a href="http://opnux.com/wp-content/uploads/2014/10/Atom-a-Hackable-Text-Editor-for-Ubuntu-Linux-Mint-Elementary-OS.png">
              <img alt="Atom-a-Hackable-Text-Editor-for-Ubuntu-Linux-Mint-Elementary-OS" class="aligncenter size-full wp-image-271" height="60" src="http://opnux.com/wp-content/uploads/2014/10/Atom-a-Hackable-Text-Editor-for-Ubuntu-Linux-Mint-Elementary-OS.png" width="290"/>

            <figure class="thumbnail wp-caption aligncenter" id="attachment_272" style="width: 710px">
             <a href="http://opnux.com/wp-content/uploads/2014/10/Atom-a-Hackable-Text-Editor-for-Ubuntu-Linux-Mint-Elementary-OS1.jpeg">

            <p>
             <a href="http://opnux.com/atom-text-edito" target="_blank" title="Atom Text Editor">
              Atom Text Editor
             </a>
             is modern, approachable,
             <a href="https://github.com/atom/atom" target="_blank" title="Atom Text Editor GitHub">
              hackable
             </a>
             , andopen source text editor from GitHub, It has a lot of features and its currently on beta, if you want to try it, we shall try this tutorial for
             <a href="http://opnux.com/atom-text-editor" target="_blank" title="Atom Text Editor">
              Install Atom a Hackable Text Editor for Ubuntu, Linux Mint, Elementary OS
             </a>
             .
            </p>
            <p>
             The look of atom text editor and even features is lot a like
             <strong>
              Sublime Text Editor,
             </strong>
             a cross platform text editor but closed source. Like other desktop apps, Atom has its own icon , file browser, and native menus and dialogs. I think this is the best features of
             <a href="http://opnux.com/atom-text-edito" target="_blank" title="Atom Text Editor">
              Atom Text Editor
             </a>
             , you can take control of your editor easily and tweak the look of your Atoms interface with CSS, or adding some features with HTML and JavaScript.
            </p>
            <p>
             <a href="http://opnux.com/atom-text-edito" target="_blank" title="Atom Text Editor">
              Atom Text Editor
             </a>
             has support Node.js so it makes it trivial to access file system or start servers directly from your editor. You can choose from over 50 thousand library in Nodes package repository. With Seamless integration its allows you to freely mix the usage of node and browser APIs.
             <a href="http://opnux.com/atom-text-edito" target="_blank" title="Atom Text Editor">
              Atom Text Editor
             </a>
             is composed of over 50 open-source packages. Here are some features of
             <a href="http://opnux.com/atom-text-edito" target="_blank" title="Atom Text Editor">
              Atom Text Editor
             </a>
             :
            </p>
            <ul>
             <li>
              File system browser
             </li>
             <li>
              Fuzzy finder for quickly opening files
             </li>
             <li>
              Fast project-wide search and replace
             </li>
             <li>
              Multiple cursors and selections
             </li>
             <li>
              Multiple panes
             </li>
             <li>
              Snippets
             </li>
             <li>
              Code folding
             </li>
             <li>
              A clean preferences UI
             </li>
             <li>
              Import TextMate grammars and themes
             </li>
            </ul>
            <h2>
             Install AtomText Editor
            </h2>
            <p>
             For now since Atom is still in beta, there is no binaries for Linux, But dont worry, you dont have to compile it on your own because
             <a href="https://launchpad.net/~nilarimogard/+archive/ubuntu/webupd8" target="_blank">
              Webupd8 team
             </a>
             have created Atom PPA that lest you to easily install Atom Editor in both 32 and 64 bit systems. Thanks for
             <a href="https://launchpad.net/~nilarimogard/+archive/ubuntu/webupd8" target="_blank">
              Webupd8 team
             </a>
             effort !.
            </p>
            <p>
             To install Atom via PPA, Open a terminal and use the following commands:
            </p>
            <pre>
             sudo add-apt-repository ppa:webupd8team/atom
sudo apt-get update
sudo apt-get install atom
            </pre>
            <p>
             After Installation you can run you
             <a href="http://opnux.com/atom-text-edito" target="_blank" title="Atom Text Editor">
              Atom Text Editor
             </a>
             via Applications Menu.
            </p>
            <h3>
             Uninstall Atom Text Editor
            </h3>
            <p>
             There could be plenty of reasons why you remove Atom from your system. Maybe
             <a href="http://opnux.com/atom-text-edito" target="_blank" title="Atom Text Editor">
              Atom Text Editor
             </a>
             isnt as what you expected, being unstable or there other major reasons ?, here is how to uninstall
             <a href="http://opnux.com/atom-text-edito" target="_blank" title="Atom Text Editor">
              Atom Text Editor
             </a>
             Completely from your system :
            </p>
            <pre>
