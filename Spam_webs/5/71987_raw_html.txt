              <div class="storyBody" data-component="lazyloadImages" data-lazyload-images-options='{"threshold":500}' itemprop="articleBody">
               <p>
                I've been preaching the gospel of Linux security for decades now, but it's always nice to see proof-positive from an independent organization that Linux is indeed the most secure operating system around.
               </p>
               <p>
                The
                <a href="http://www.cesg.gov.uk/Pages/homepage.aspx">
                 Communications-Electronics Security Group (CESG)
                </a>
                , the group within the
                <a href="http://www.gchq.gov.uk/Pages/homepage.aspx">
                 UK Government Communications Headquarters (GCHQ)
                </a>
                that assesses operating systems and software for security issues, has found that while no end-user operating system is as secure as they'd like it to be,
                <a href="https://www.gov.uk/government/collections/end-user-devices-security-guidance%E2%80%942">
                 Ubuntu 12.04 is the best of the lot
                </a>
                .
               </p>
               <div class="ad-inpage-video-top" data-ad="inpage-video-top" id="inpage-video-top-55a1b17d45ee5">
                <script>
                 window.console &amp;&amp; console.log &amp;&amp; console.log("ADS: queuing inpage-video-top-55a1b17d45ee5 for display");

        cbsiGptDivIds.push("inpage-video-top-55a1b17d45ee5");
                </script>
               </div>
               <figure class="image alignRight image-thumbnail shortcode-image">
                <span class="img ">
                 <img alt="tux-guard" class="" height="170" src="http://zdnet1.cbsistatic.com/hub/i/r/2014/10/04/5bc69e6a-4bf1-11e4-b6a0-d4ae52e95e57/resize/170x170/49a7ad140e1e6dbe6af26bd5da1298d7/tux-guard.png" width="170"/>
                </span>
               </figure>
               <p>
                In late 2013, the
                <a href="http://insights.ubuntu.com/wp-content/uploads/UK-Gov-Report-Summary.pdf">
                 CESG looked at the security of the most popular end-user operating systems for desktops, smartphones, and tablets [PDF Link].
                </a>
                This included: Android 4.2, Android 4.2 on Samsung devices; iOS 6, Blackberry 10.1, Google's Chrome OS 26, Ubuntu 12.04, Windows 7 and 8; Windows 8 RT, and Windows Phone 8. These were judged for their security suitability for OFFICIAL level use according to the
                <a href="https://www.gov.uk/government/uploads/system/uploads/attachment_data/file/251480/Government-Security-Classifications-April-2014.pdf">
                 UK Government Security Classifications
                </a>
                (PDF Link). This is the UK's government lowest security level.
               </p>
               <p>
                <strong>
                 <span class="link" section="shortcodeLink">
                  <a href="/pictures/ubuntu-13-10-a-desktop-tour-slideshow/">
                   Ubuntu 13.10: A desktop tour (Slideshow)
                  </a>
                 </span>
                </strong>
               </p>
               <p>
                According to
                <a href="http://www.canonical.com">
                 Canonical
                </a>
                ,
                <a href="http://www.ubuntu.com">
                 Ubuntu
                </a>
                's parent company, "No currently available operating system can meet all of these requirements. Ubuntu however, scores the highest in a direct comparison. " Ubuntu 12.04 is Ubuntu's latest Long Term Support (LTS) version, and it's recommended for use by businesses.
               </p>
               <p>
                The CESG examined each operating system security on the following grounds:
               </p>
               <p>
                 Virtual Private Network (VPN)
                <br/>
                 Disk Encryption
                <br/>
                 Authentication
                <br/>
                 Secure Boot
                <br/>
                 Platform Integrity and Application Sandboxing
                <br/>
                 Application Whitelisting
                <br/>
                 Malicious Code Detection and Prevention
                <br/>
                 Security Policy Enforcement
                <br/>
                 External Interface Protection
                <br/>
                 Device Update Policy
                <br/>
                 Event Collection for Enterprise Analysis
                <br/>
                 Incident Response
               </p>
               <p>
                Ubuntu has three problem areas that kept it from a perfect score. Others had more. Windows Phone 8 has the most "Significant Risk" items with two and Blackberry 10.1 Corporate has the most "Some Risk" areas with six. Where Ubuntu could stand improvement is in VPN, Disk Encryption and Secure Boot.
               </p>

               <p>
                For VPNs, the CESG found that Ubuntu's "built-in VPN has not been independently assured to Foundation Grade." That means, technically Ubuntu's VPN is good enough, but it hasn't been shown to meet the security requirement by an independent third party. Canonical hopes to gets its VPN approved by April's
                <span class="link" section="shortcodeLink">
                 <a href="/article/whats-coming-in-ubuntu-14-04-trusty-tahr/">
                  Ubuntu 14.04 release
                 </a>
                </span>
                .
               </p>

                Ubuntu faces the same problem with its built-in disk encryption tools:
                <a href="https://code.google.com/p/cryptsetup/">

                </a>
                and
                <a href="https://code.google.com/p/cryptsetup/wiki/DMCrypt">
                 dm-crypt
                </a>
                Canonical is seeking a company or organization to put its disk encryption software through the assessment process.
               </p>
               <p>
                Like all Linux distributions, Ubuntu does support Microsoft's version of
                <a href="http://www.itworld.com/hardware/222791/goodbye-bios-hello-uefi">
                 Unified Extensible Firmware Interface (UEFI)

                  Secure Boot
                 </a>
                </span>
                . They're just not happy about it.
               </p>
               <p>
                Canonical's current position, from Ubuntu 12.10 onwards, is "to adopt
                <a href="https://help.ubuntu.com/community/Grub2">
                 Grub2
                </a>
                as the default bootloader, with support for Secure Boot, but with an ability to turn off secure boot to modify the OS, if required. We believe this gives users and enterprises the best compromise between security and ability to customize after sale."
               </p>
               <p>
                Problems aside, the simple truth is that if security is what you want most from desktop, smartphone, or tablet operating systems than Ubuntu is what you should be using.
               </p>
               <p>
                True, security is always a moving target, but year-in and year out, Linux-based operating systems are more secure then their competition. As
                <span class="link" section="shortcodeLink">
                 <a href="/article/windows-xp-end-of-support-in-april-three-more-questions-answered/">
                  Windows XP's support clock ticks to its end of supported life

                  Ubuntu One, is still a work in progress
                 </a>
