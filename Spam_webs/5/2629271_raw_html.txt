            <span style="font-weight:bold">
             CorneliOS 5.5r24 released

            <br/>
            May 24, 2015
            <br/>
            <br/>
            The CorneliOS Team is pleased to announce the release of CorneliOS 5.5r24.
            <br/>
            <br/>
            CorneliOS is a web based multi-user OS that runs in your web browser. All applications and files are stored on the CorneliOS server and are accessible all the time and anywhere. CorneliOS is platform and browser independent and offers a web application development framework.
            <br/>
            <br/>
            CorneliOS 5.5r24 offers a greatly improved greatly improved CIOS LMS tools, greatly improved CorneliOS and CIOS SQL APIs offering better ODBC compatibility, new CorneliOS DBMS export SQL and execute SQL features, an improved CorneliOS DBMS table viewer, a CIOS Web API thats no longer limited to the school web environment, improved CorneliOS and CIOS mobile compatibility, a new CorneliOS Parser ExtHTML device command that allows for easier mobile website creation, CorneliOS and CIOS code cleanups, a CIOS Media Manager now supporting ViewerJS, a CIOS Publisher API now supporting ViewerJS as well as updated system images, language kits and style sheets.
            <br/>
            <br/>
            Of course the release also includes the usual bug fixes, code enhancements, minor feature changes and documentation updates.
            <br/>
            <br/>
            Special thanks to all of those who help us by downloading, testing and giving us feedback - end users, testers as well as contributors.
