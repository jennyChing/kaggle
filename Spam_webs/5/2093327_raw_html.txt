         <h1 class="entry-title">
          How to use colored Emoji using Internet Explorer on Windows 8.1
         </h1>
         <!-- &lt;div class="entry-meta"&gt;&lt;/div&gt; .entry-meta -->
        </div>
        <!-- .entry-header -->
        <!-- Ad -->
        <div class="entry-content">
         <!-- Ad -->
         <div class="ad" id="single-ad-2">
          <strong>
           RECOMMENDED:
           <a href="http://link.tweakbit.com/tds/1avD" rel="nofollow" target="_blank">
            Click here to fix Windows errors and optimize system performance
           </a>
          </strong>
         </div>
         <p>
          A few days back, we checked out
          <a href="http://www.thewindowsclub.com/windows-8-touch-keyboard" target="_blank" title="Touch Keyboard in Windows 8">
           Windows 8 Touch Keyboard
          </a>
          settings and some tips to help you get started. Today we will go a step further and see how you can use Emoji using the Touch Keyboard on Windows 8.1.
         </p>
         <p>
          <strong>
           Emoji
          </strong>
          is a type of emoticon used on
          <strong>
           Windows 8
          </strong>
          , iPhone, iPad, Android and Mac OS. What is an Emoji? It is a small digital image used to express emotions. The word Emoji means Picture letter in Japanese.
         </p>
         <h2>
          How to use Emoji on Windows 8
         </h2>
         <p>
          The touch keyboard icon appears and sits in your taskbar near the notification area. Tap it to make the touch keyboard appear.
         </p>
         <p>
          <img alt="" class="aligncenter" height="203" src="http://thewindowsclub.thewindowsclubco.netdna-cdn.com/wp-content/uploads/2014/08/touch-keyboard-in-windows-8.png" width="400"/>
          <br/>
          When the default layout of the touch keyboard opens, tab the
          <strong>
           Smiley
          </strong>
          <span class="gmw_">
           . You will see all the Emojis here. Click on the Emojis in the bottom-most row, to see all the categories of
           <span class="gm_ gm_ea751743-e94a-4b4f-a439-1bb9a32c89f4 gm-spell">
            emojis
           </span>
           <span class="gmw_">
            . Also make use of the Left and Right arrows to see all the
            <span class="gm_ gm_70daecdd-3a0d-b215-079c-70482f58c4e8 gm-spell">
             emojis
            </span>
            within a particular category.
           </span>
          </span>
         </p>
         <!-- Quick Adsense WordPress Plugin: http://quicksense.net/ -->
         <div style="float:none;margin:10px 0 10px 0;text-align:center;">
          <script async="" src="//pagead2.googlesyndication.com/pagead/js/adsbygoogle.js">
          </script>
          <!-- Async 336x280 Center of post TWC/N -->
          <ins class="adsbygoogle" data-ad-client="ca-pub-5950002284129980" data-ad-slot="6708462760" style="display:inline-block;width:336px;height:280px">
          </ins>
          <script>
           (adsbygoogle = window.adsbygoogle || []).push({});
          </script>
         </div>
         <p>
          <img alt="emoji on windows 8" class="aligncenter size-full wp-image-90248" height="170" src="http://thewindowsclub.thewindowsclubco.netdna-cdn.com/wp-content/uploads/2014/08/emoji-windows-8.png" width="600"/>
         </p>
         <p>
          Emojis are supported on Windows 7 or above, including Windows RT, Windows 8.1. Windows Vista and below does not support Emoji.
         </p>
         <p>
          To be able to see Emoji in color, Windows 8.1 is required  and you will be able to see them only in Internet Explorer. Chrome does not support native Emoji on any version of Windows. On Firefox too, you wont be ale to see colored Emojis.
         </p>
         <p>
          <strong>
           Emoji Keys
          </strong>
          <span class="gmw_">
           is an app available in the Windows Store that allows you to convert iPhone
           <span class="gm_ gm_4c786905-ef53-13fd-f2ca-413500e53780 gm-spell">
            emojis
           </span>
           <span class="gmw_">
            into icons, so that you can understand them. It also lets you send your own back to them using the full
            <span class="gm_ gm_56dde3fe-a45e-d140-b81c-6ff7762ec16b gm-spell">
             emoji
            </span>
            keyboard. It is not a free app, however.
           </span>
          </span>
         </p>
         <p>
          <em>
           If you wish to use an Emoji without using the Touch keyboard, just visit
           <a href="http://getemoji.com/" onclick="__gaTracker('send', 'event', 'outbound-article', 'http://getemoji.com/', 'GetEmoji.com');" rel="nofollow" target="_blank">
            GetEmoji.com
           </a>
           in your browser, select the Emoji and copy-paste it. You will be able to use these Emojis anywhere  even in your Twitter tweets. You will see the colored Emojis only when you use Internet Explorer on Windows 8.
          </em>
         </p>
