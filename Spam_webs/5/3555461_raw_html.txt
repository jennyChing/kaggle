         <span style="line-height: 1.714285714; font-size: 1rem;">

         </span>
        </p>
        <p dir="ltr">
         From within the spreadsheet (or presentation, etc.) select
         <em>
          Publish to the Web 
         </em>
         from the
         <em>
          File
         </em>

         <em>
          All Cells
         </em>
         .
        </p>

          File &gt; Embed
