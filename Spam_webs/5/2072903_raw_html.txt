           </div>

         <br/>
         <p>
          <strong>
           Size: 18 x 36 inches
          </strong>
         </p>
         <p>
          Materials Required:
          <br/>
          <strong>
           AMERICAN THREAD "AUNT LYDIA'S"
          </strong>
          <br/>
          <strong>
           HEAVY RUG YARN
          </strong>
          <br/>
          370 yd. skeins each White,  Jade and Black
          <br/>
          Wooden Crochet Hook Size Q
         </p>
         <p>
          <strong>
           GAUGE:
          </strong>
          7 sts = 6 inches
         </p>
         <p>
          Work with 3 strands  throughout. Work up a swatch to check gauge. If you get less sts than called  for; work looser; if you get more sts than called for; work tighter.
         </p>
         <p>
          With 1 strand of each color ch 22, 1 s c in 2nd  st from hook and in each remaining st of ch, ch 1 to turn all rows.
          <strong>
           2nd ROW:
          </strong>
          Working in back loops of sts  only, work 1 sc in each sc.
          <strong>
           NEXT 6 ROWS:
          </strong>
          Repeat last row.
          <strong>
           NEXT 33 ROWS:
          </strong>
          Working in both loops 1 s c in each sc.
          <strong>
           NEXT  ROW:
          </strong>
          Working in back loop of sts only, 1 s c in each sc.
          <strong>
           NEXT 7 ROWS:
          </strong>
          Repeat last row, cut yarn.
         </p>
