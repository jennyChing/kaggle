         <p>
          <img align="right" alt="onehoursoftware | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" height="76" src="http://www.inspiredgeek.com/wp-content/uploads/2008/03/onehoursoftware.jpg" style="margin: 5px 0px 0px" title="onehoursoftware | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" width="244"/>
          <a href="http://www.donationcoder.com/index.html">
           DonationCoder
          </a>
          is one of my favorite sites to keep in-touch on the latest software related news and ideas. They have a dedicated section for some small but very useful windows utilities based on the Auto Hot Key(AHK) scripts called
          <em>
           1 HOUR SOFTWARE by Skrommel
          </em>
          (based on name of the author). Some of the listed utilities are really very helpful , so I decided to pick some of the most useful from these and present them to my readers with their intended use and some usual comments. Note that you can download all of these as either .
          <em>
           ahk
          </em>
          scripts and compile them using the
          <a href="http://www.autohotkey.com/">
           Auto Hot Key
          </a>
          scripting software or you can avoid the hassle by downloading the pre-compiled executable files. I will recommend the later option for the most users unless you want to play around with the code and make some of your own customizations.
         </p>
         <h4>
          1.
          <a href="http://www.donationcoder.com/Software/Skrommel/index.html#CloseMany">
           CloseMany
          </a>
         </h4>
         <p>
          This utility allows to close/kill multiple process in one go. Just select all the processes in the list-box that you want to close and press the Close or the Kill button. A user can also restart a process if desired. Process related controls are available in the first tab. You can also control the
          <em>
           system services
          </em>
          installed in the Windows system from the second tab. All usual windows services related options are available(start, stop, restart, auto, manual, Demand). Advantage is the same that you can select multiple services and perform the desired operation rather than to select each service one by one and enable/disable it(as you do in the services applet in the
          <em>
           Windows Control Panel
          </em>
          ).
         </p>
         <p>
          <img alt="closemany | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" border="0" height="159" src="http://www.inspiredgeek.com/wp-content/uploads/2008/03/closemany.jpg" style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" title="closemany | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" width="324"/>
         </p>
         <h4>
          2.
          <a href="http://www.donationcoder.com/Software/Skrommel/index.html#GoneIn60s">
           GoneIn60s
          </a>
         </h4>
         <p>
          Allows to recover the closed programs when running in the background. To recover right-click the tray icon and choose an application to recover that, double-click the tray icon to recover all the closed applications. Very useful if you unintentionally closed a running application.
         </p>
         <p>
          <img alt="gonein60sec | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" border="0" height="156" src="http://www.inspiredgeek.com/wp-content/uploads/2008/03/gonein60sec.jpg" style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" title="gonein60sec | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" width="240"/>
         </p>
         <h4>
          3.
          <a href="http://www.donationcoder.com/Software/Skrommel/index.html#IdleRun">
           IdleRun
          </a>
         </h4>
         <p>
          Run a program or show a message when the CPU is in idle-state. You can set the
          <em>
           Idle Duration
          </em>
          and
          <em>
           CPU Threshold
          </em>
          . This utility may be useful if you want to run some
          <em>
           housekeeping tasks
          </em>
          but they are processor intensive for example to run a virus scanner to scan the local hard-drives, system de-fragmentation software, content indexing or some other computation intensive tasks when there is no CPU activity.
         </p>
         <h4>
          4.
          <a href="http://www.donationcoder.com/Software/Skrommel/index.html#Desktop">
           Desktop!
          </a>
         </h4>
         <p>
          Give the desktop its own taskbar button! You can Click it, Alt-Tab to it, or drag files to it to show or hide the desktop. I am sure it will be useful if we get used to it.
         </p>
         <p>
          <img alt="desktop | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" border="0" height="199" src="http://www.inspiredgeek.com/wp-content/uploads/2008/03/desktop.jpg" style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" title="desktop | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" width="204"/>
         </p>
         <h4>
          5.
          <a href="http://www.donationcoder.com/Software/Skrommel/index.html#RunSaver">
           RunSaver
          </a>
         </h4>
         <p>
          This is a special screensaver that will run a command when run. If you can figure-out its use?
         </p>
         <p>
          <img alt="runsaver | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" border="0" height="207" src="http://www.inspiredgeek.com/wp-content/uploads/2008/03/runsaver.jpg" style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" title="runsaver | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" width="206"/>
         </p>
         <p>
          <span id="more-418">
          </span>
         </p>
         <h4>
          6.
          <a href="http://www.donationcoder.com/Software/Skrommel/index.html#ShortCutter">
           ShortCutter
          </a>
         </h4>
         <p>
          It makes auto-updating collection of shortcuts of chosen file types from a folder. It is very configurable and you have to define some rules before it properly works. Rules can be defined in the Rules tab of the interface. In each rule you define you will specify the following:
         </p>
         <p>
          - Source  Where files are located for which you want to create shortcuts to
         </p>
         <p>
          - Target  Where auto-updating shortcuts will be placed
         </p>
         <p>
          - Files  Which files types you want to create shortcuts for (wild-cards supported)
         </p>
         <p>
          - Ignore  What sub-strings (parts of the filename) to ignore
         </p>
         <p>
          - Recurse  If it should recurse subdirectories?
         </p>
         <p>
          <img alt="shortcutter | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" border="0" height="226" src="http://www.inspiredgeek.com/wp-content/uploads/2008/03/shortcutter.jpg" style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" title="shortcutter | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" width="324"/>
         </p>
         <p>
          <img alt="shortcutter options | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" border="0" height="220" src="http://www.inspiredgeek.com/wp-content/uploads/2008/03/shortcutter-options.jpg" style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" title="shortcutter options | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" width="324"/>
         </p>
         <p>
          From the Rules tab each defined rule can be enabled or disabled. A timer option is also available to specify the amount of time after it will check for new eligible files in the source folder to have a shortcuts in the target folder. ShortCutter can be globally enabled/disabled from the traybar menu as well.
         </p>
         <h4>
          7.
          <a href="http://www.donationcoder.com/Software/Skrommel/index.html#MoveOut">
           MoveOut
          </a>
         </h4>
         <p>
          MoveOut can be used to move files from one folder on your hard-drive to another automatically based on the rules you define as in the case of ShortCutter. You can define new rules in the Rules tab of the Settings. For each rule defined you will specify the following:
         </p>
         <p>
          - Source  Source where the files are located now
         </p>
         <p>
          - Target  Destination folder where files will be moved
         </p>
         <p>
          - Files  Which files types you want to move(wild-cards supported)
         </p>
         <p>
          - Ignore  What sub-strings (parts of the filename) to ignore
         </p>
         <p>
          - Replace  What to do if a file already exists at the destination?
         </p>
         <p>
          All the created rules can be disabled/enabled on individual basis. An interval can be specified in the settings which governs that how often the files will checked to be moved from the source to the destination. Moving feature can also be enabled/disabled on global basis from the traybar menu.
         </p>
         <p>
          <img alt="moveout | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" border="0" height="226" src="http://www.inspiredgeek.com/wp-content/uploads/2008/03/moveout.jpg" style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" title="moveout | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" width="324"/>
         </p>
         <p>
          <img alt="moveout rules | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" border="0" height="220" src="http://www.inspiredgeek.com/wp-content/uploads/2008/03/moveout-rules.jpg" style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" title="moveout rules | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" width="324"/>
         </p>
         <p>
          This utility can be useful in couple of scenarios I think for example, consider you are downloading some files from the Internet on to your desktop. By using this utility you can define rules to copy all video files to your video collection and all the software to the software collection directory.
         </p>
         <h4>
          8.
          <a href="http://www.donationcoder.com/Software/Skrommel/index.html#BlockInput">
           BlockInput
          </a>
         </h4>
         <p>
          BlockInput will block all the input devices (keyboard, mouse etc.) when you press a defined hotkey(default is Ctr+Q) for a specified amount of time. Time interval can be specified in the settings and after that time-period has elapsed, all the input devices will be unlocked.
         </p>
         <p>
          <img alt="block input | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" border="0" height="152" src="http://www.inspiredgeek.com/wp-content/uploads/2008/03/block-input.jpg" style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" title="block input | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" width="324"/>
         </p>
         <h4>
          9.
          <a href="http://www.donationcoder.com/Software/Skrommel/index.html#UrlHistory">
           URLHistory
          </a>
         </h4>
         <p>
          Watches the clipboard for web addresses and saves them with a comment(optional). Whenever a URL will be copied to the clipboard a dialog box will appear that will allow to enter a comment for the current URL to be saved with it. These URLs along with the comments are saved in the URLHistory.htm file. You can also access the URLs copied so far from the traybar menu. A URL can also be manually entered from the traybar menu.
         </p>
         <p>
          This one may come handy when you need to copy a lot of URL addresses but from different websites that are not at one place and as a bonus will you will be able to enter descriptive comments.
         </p>
         <p>
          <img alt="urlhistoryscreen | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" border="0" height="204" src="http://www.inspiredgeek.com/wp-content/uploads/2008/03/urlhistoryscreen.jpg" style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" title="urlhistoryscreen | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" width="203"/>
         </p>
         <h4>
          10.
          <a href="http://www.donationcoder.com/Software/Skrommel/BatteryRun/BatteryRun.exe">
           ReRun
          </a>
         </h4>
         <p>
          Allows to run a startup program that is not running now. This is very useful in some situations where you have a lot of programs specified in the windows startup but you want to play a resource/memory intensive game. Then you can simply close those
          <em>
           heavy-duty
          </em>
          programs from the traybar(to free up system resources) and play your game. After you are finished simply execute the ReRun to restart all the startup programs that you had closed. I personally use it sometimes for a similar purpose.
          <br/>
         </p>
         <h4>
          11.
          <a href="http://www.donationcoder.com/Software/Skrommel/index.html#WinWarden">
           WinWarden
          </a>
         </h4>
         <p>
          Automatically control how to display a window. Can control the following attributes of a window:
          <br/>
          - Move, maximize, minimize, restore, enable, disable, hide, show, ontop, bottom, alwaysontop, clip, transparent, transparent color, move relative to another window, stick to the edge of the screen, center, run a program.
          <br/>
          - Double-click trayicon to center the active window.
         </p>
         <p>
          <img alt="winwardenscreen | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" border="0" height="203" src="http://www.inspiredgeek.com/wp-content/uploads/2008/03/winwardenscreen.jpg" style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" title="winwardenscreen | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" width="202"/>
          <br/>
         </p>
         <h4>
          12.
          <a href="http://www.donationcoder.com/Software/Skrommel/index.html#DeskLock">
           DeskLock
          </a>
         </h4>
         <p>
          Prevents the moving of desktop icons. Double-Click the tray icon to Disable/Enable. Also has an option to disable left drag and right mouse button.
         </p>
         <p>
          <img alt="desklock | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" border="0" height="246" src="http://www.inspiredgeek.com/wp-content/uploads/2008/03/desklock.jpg" style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" title="desklock | 15 Most Useful Auto Hot Key Generated System Utilities for Windows Users" width="239"/>
         </p>
         <h4>
          13.
          <a href="http://www.donationcoder.com/Software/Skrommel/index.html#NoClose">
           NoClose
          </a>
         </h4>
         <p>
          Disable the Close button (X) of selected windows. Maybe useful to prevent an important application from an accidental close.
         </p>
         <p>
