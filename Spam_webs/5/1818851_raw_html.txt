        <h2 id="enabling-compiz-fusion-on-an-ubuntu-desktop-nvidia-geforce-fx-">
         Enabling Compiz Fusion On An Ubuntu 8.10 Desktop (NVIDIA GeForce FX 5200)
        </h2>
        <p>
         Version 1.0
         <br/>
         Author: Falko Timme
         <br/>
         Last edited 11/03/2008
        </p>
        <p>
         This tutorial shows how you can enable Compiz Fusion on an Ubuntu 8.10 (Intrepid Ibex) desktop (the system must have a 3D-capable graphics card - I'm using an NVIDIA GeForce FX 5200 here). With Compiz Fusion you can use beautiful 3D effects like wobbly windows or a desktop cube on your desktop.
        </p>
        <div style="width:336px;float:left; margin: 10px 15px 10px 0px;background-color:white;">
         <script type="text/javascript">
          GA_googleFillSlot('howtoforge_com_article_rectangle_a_300x250');
         </script>
        </div>
        <p>
         This document comes without warranty of any kind! I do not issue any guarantee that this will work for you!
        </p>
        <p>
        </p>
        <h3 id="-preliminary-note">
         1 Preliminary Note
        </h3>
        <p>
         I have tried this on a desktop computer with an NVIDIA GeForce FX 5200. It should work the same way with all other NVIDIA graphics cards.
        </p>
        <p>
        </p>
        <h3 id="-enabling-the-nvidia-driver-">
         2 Enabling The NVIDIA Driver
        </h3>
        <p>
         To use 3D effects on an NVIDIA graphics card, we need the proprietary NVIDIA driver which we can install from the
         <span class="system">
          Hardware Drivers
         </span>
         tool. The
         <span class="system">
          Hardware Drivers
         </span>
         tool can be started from
         <span class="system">
          System &gt; Administration &gt; Hardware Drivers
         </span>
         :
        </p>
        <p>
         <a class="fancybox" href="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/big/1.jpg" id="img-1">
          <img height="440" mce_real_src="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/1.jpg" src="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/1.jpg" width="550"/>
         </a>
        </p>
        <p>
         The
         <span class="system">
          Hardware Drivers
         </span>
         tool should tell you that there's an NVIDIA graphics driver available for your system. To install it, click on
         <span class="system">
          Activate
         </span>
         :
        </p>
        <p>
         <a class="fancybox" href="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/big/2.jpg" id="img-2">
          <img height="440" mce_real_src="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/2.jpg" src="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/2.jpg" width="550"/>
         </a>
        </p>
        <p>
         Type in your password:
        </p>
        <p>
         <a class="fancybox" href="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/big/3.jpg" id="img-3">
          <img height="440" mce_real_src="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/3.jpg" src="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/3.jpg" width="550"/>
         </a>
        </p>
        <p>
         Afterwards the NVIDIA driver is being downloaded and installed:
        </p>
        <p>
         <a class="fancybox" href="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/big/4.jpg" id="img-4">
          <img height="440" mce_real_src="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/4.jpg" src="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/4.jpg" width="550"/>
         </a>
        </p>
        <div style="width:336px;float:left; margin: 10px 15px 10px 0px;background-color:white;">
         <script type="text/javascript">
          GA_googleFillSlot('howtoforge_com_article_rectangle_b_300x250');
         </script>
        </div>
        <p>
         The
         <span class="system">
          Hardware Drivers
         </span>
         tool should now tell you that a system restart is required to activate the driver. Click on
         <span class="system">
          Close
         </span>
         to leave the
         <span class="system">
          Hardware Drivers
         </span>
         tool...
        </p>
        <p>
         <a class="fancybox" href="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/big/5.jpg" id="img-5">
          <img height="440" mce_real_src="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/5.jpg" src="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/5.jpg" width="550"/>
         </a>
        </p>
        <p>
         ... and then reboot the system:
        </p>
        <p>
         <a class="fancybox" href="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/big/6.jpg" id="img-6">
          <img height="440" mce_real_src="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/6.jpg" src="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/6.jpg" width="550"/>
         </a>
        </p>
        <p>
         After the reboot the system uses the proprietary NVIDIA graphics driver.
        </p>
        <p>
        </p>
        <h3 id="-installing-simple-compizconfig-settings-manager-simpleccsm-">
         3 Installing Simple Compizconfig Settings Manager (simple-ccsm)
        </h3>
        <p>
         Open the
         <span class="system">
          Synaptic Package Manager
         </span>
         (
         <span class="system">
          System &gt; Administration &gt; Synaptic Package Manager
         </span>
         ):
        </p>
        <p>
         <a class="fancybox" href="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/big/7.jpg" id="img-7">
          <img height="440" mce_real_src="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/7.jpg" src="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/7.jpg" width="550"/>
         </a>
        </p>
        <p>
         Select the package
         <span class="system">
          simple-ccsm
         </span>
         (simple Compizconfig settings manager) and install it:
        </p>
        <p>
         <a class="fancybox" href="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/big/7a.png" id="img-7a">
          <img height="440" mce_real_src="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/7a.png" src="https://www.howtoforge.com/images/compiz_fusion_ubuntu_8.10_nvidia/7a.png" width="550"/>
         </a>
        </p>
