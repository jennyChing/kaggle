function resizeForIE()
{
  var ua = window.navigator.userAgent;
  var msie = ua.indexOf("MSIE ");
  
  if(msie === -1)
  {
    return;
  }

  // for ie7/8 as don t support media queries
  var windowWidth = $(window).innerWidth();
  
  if(windowWidth &lt;= 475)
  {
    $("body").addClass("super-compact-view");
    $("body").removeClass("compact-view");      
  }
  else if(windowWidth &lt;= 946)
  {
    $("body").addClass("compact-view"); 
    $("body").removeClass("super-compact-view");      
  }
  else
  {

        <h3>
         Features &amp; Benefits
        </h3>
        <ul class="feature-list">
         <li class="li-keyboard-o">
          Easily create
          <strong>
           quick keyboard shortcuts to URLs
          </strong>
          you need access to often.
         </li>
         <li class="li-desktop">
          The visually appealing
          <strong>
           <em>
            launcher
           </em>
           speed-dial panel
          </strong>
          means no need to actually memorize the shortcut
        keys, simply remember the one hotkey and your good to gothus giving you the time-saving benefit of keyboard
        shortcuts without the hassle of having to remember them all!
         </li>
         <li class="li-rocket">
          Alternatively, if you have a good memory; skip the
          <em>
           launcher
          </em>
          panel and use visually transparent shortcuts by
        pressing the
          <strong>
           Direct-to-Site
                  hotkey
          </strong>
          combined with a site key that you have memorized.
         </li>
         <li class="li-eyedropper">
          <strong>
           Highly customizable appearance
          </strong>
          - change your
          <em>
           launcher
          </em>
          's layout style; number of columns,
        foreground and background colors, padding, font sizes, icon sizes, show only icons without titles, hide site keys,
        add background transparency, gradients, icon effects and more.
         </li>
         <li class="li-list-alt">
          The
          <strong>
           Shortcut Manager
          </strong>
          makes it super-quick to add, edit and remove shortcuts. Access the Shortcut Manager
        when the
          <em>
           launcher
          </em>
          is showing by right clicking or hitting Tab.
         </li>
         <li class="li-folder-o">
          <strong>
           Arrange related shortcuts into labeled groups
          </strong>
          to have them appear together in a visually helpful way.
         </li>
         <li class="li-home">
          Optionally, set the
          <strong>
           <em>
            launcher
           </em>
           to appear on Firefox's homepage/startpage
          </strong>
          .
         </li>
         <li class="li-toggle-right">
          Set SiteLauncher to
          <strong>
           open sites in the current or a new tab
          </strong>
          .
         </li>
         <li class="li-photo">
          Ability to
          <strong>
           assign custom icons to site shortcuts
          </strong>
          using your own images (otherwise, site's own icon
        is automatically fetched and used).
         </li>
         <li class="li-bookmark">
          Quickly shortcut any link you see on the web, by right clicking and pressing "Shortcut this link". Also, quickly shortcut
        the current webpage by pressing the
          <strong>
           <em>
            Shortcut This Page
           </em>
           hotkey
          </strong>
          , Ctrl + Shift + S by default
         </li>
         <li class="li-cogs">
          Advanced feature
          <strong>
           create shortcuts to JavaScript bookmarklets
          </strong>
          (not the same as bookmarks). Perform
        quick on-page actions using your favorite bookmarklets taken from the web. The shortcuts will automatically include
        icons too; taken from the sites they reference. Simply right click a bookmarklet and click "Shortcut this link" to
        add it to SiteLauncher.
         </li>
        </ul>

       <div class="section" id="hotkey-section">
        <h3>
         SiteLauncher's Keyboard Hotkeys Explained
        </h3>
        <p>
         SiteLauncher is designed to be intuitive and doesn't have a long manualthe only thing you really need remember is
         <strong>
          Ctrl +
                Space
         </strong>
         which is the
         <em>
          launcher
         </em>
         hotkey. However, to get the fastest and most useful experience
      out of SiteLauncher you might wish to memorize all SiteLauncher's hotkeys; as listed below:
        </p>
        <ul class="hotkey-list">
         <li class="li-keyboard-o">
          <div>
           <strong>
            Launcher Hotkey
           </strong>
          </div>
          <div>
           <em>
            Ctrl+Space
           </em>
          </div>
          <div>
           This is the main hotkey, if you only remember one hotkey, make it this one. Press the Launcher hotkey to bring forward the
           <em>
            launcher
           </em>
           . Then, simply press the key indicated for the site you wish to open (or open the shortcut by
          mouse click or touch).
          </div>
         </li>
         <li class="li-keyboard-o">
          <div>
           <strong>
            Direct-to-site Hotkey
           </strong>
          </div>
          <div>
           <em>
            Shift+Alt on Windows or Linux
           </em>
           <br/>
           <em>
            Shift+Ctrl on Mac
           </em>
          </div>
          <div>
           An alternative hotkey which doesn't provide the visual feedback of the
           <em>
            launcher
           </em>
           panel. If you have memorized
          a key for a site, then you can open that site by pressing this hotkey combined with the key for the site. If you
          prefer the visual feedback of the
           <em>
            launcher
           </em>
           feel free to disregard this hotkey.
          </div>
         </li>
         <li class="li-keyboard-o">
          <div>
           <strong>
            Shortcut-page Hotkey
           </strong>
          </div>
          <div>
           <em>
            Ctrl+Shift+S
           </em>
          </div>
          <div>
           Quickly creates a new shortcut to the current page you are on at the time. It will also bring up the Shortcut Manager with
          the new shortcut's edit box in focus so you can make revisions to the newly created shortcut should you wish.
          </div>
         </li>
         <p>
          Note: all these hotkeys are the out-of-box defaults and can be changed if needed from within SiteLauncher's Options
        panel.
         </p>
        </ul>
