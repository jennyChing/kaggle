           <h1 class="h1">
            <span class="headline">
             Childish Gambino Writes Poem About Being Black In America, And Refuses To Apologize For It
            </span>
           </h1>
           <div class="subhead">
            He also takes Twitter activism to task.
           </div>
          </div>
          <section class="entry-content clearfix">
           <div class="hr">
           </div>
           <span class="byline">
            by
            <span class="author">
             <a href="http://www.mtv.com/news/author/fleischa/" rel="author" title="Posts by Adam Fleischer">
              Adam Fleischer
             </a>
            </span>
            <span class="date">
             8/14/2014
            </span>
           </span>
           <div class="sm4ShareWidget sm4Widget" data-contenturi="id::mgid:wordpress:post:articles.mtv.com:1899822::og::http://www.mtv.com/news/1899822/childish-gambino-poem-black-america/?date_last_modified=1435002641" data-widget="postShare">
           </div>
           <p>
            Never one to shy away from discussions of race, race relations, or racial identity politics (listen to
            <a href="https://www.youtube.com/watch?v=xv_d2RUDMag" target="_blank">
             Hold You Down
            </a>
            for a brief primer),
            <a href="http://www.mtv.com/artists/childish-gambino/" target="_blank">
             Childish Gambino
            </a>
            again tackled some weighty issues early Thursday morning (August 14). This time, his method was poetry, and his medium was
            <a href="https://twitter.com/donaldglover" target="_blank">
             Twitter
            </a>
            .
           </p>
           <p>
            Donald posted a lengthy string of tweets, touching on an array of subjects, from white privilege to the music industry to the publics perception of young black males. He tweeted that he wrote the poem after hearing a friend sobbing and after talking with my mom.
           </p>
           <p>
            And though he didnt explicitly say so, the piece seems to be inspired, at least in some way, by the recent
            <a href="http://www.mtv.com/news/1894905/michael-brown-shooting-police-brutality-teenager/" target="_blank">
             death of 18-year-old Mike Brown
            </a>
            , who was shot and killed by a police officer in Ferguson, Missouri over the weekend, as well as
            <a href="http://www.mtv.com/news/1897468/michael-brown-shooting-timeline/" target="_blank">
             the subsequent reactions
            </a>
             of the public, police, and media  to the tragic incident. He also is critical of twitter activism, potentially a response to hashtags, such as
            <a href="http://www.mtv.com/news/1897755/iftheygunnedmedown-hashtag-mike-brown-media/" target="_blank">
             #IfTheyGunnedMeDown
            </a>
            , which have become popular in the aftermath of the shooting.
           </p>
           <p>
            Read the poem below:
           </p>
           <blockquote>
            <p>
             childish gambino is a white rapper
             <br/>
             i wanna be a white rapper.
             <br/>
             i wanna be so white im the biggest rapper of all time.
             <br/>
             i wanna be so white i can have a number one song with cursing and parents are fine with it.
             <br/>
             i wanna be so white and so big i get eat dinner with the koch brothers.
             <br/>
             i hope Im so big and white i can go to clippers games and it not be a statement.
             <br/>
             i hope Im so white they let my friend out of jail sooner.
             <br/>
             i hope Im so big and white my cousin wasnt shot and stabbed twice in the neck twice last month.
             <br/>
             i wanna be so big.  so white.
             <br/>
             i wanna be so big and and so white that white dads feel comfortable sending their daughters,
             <br/>
             who are home for the summer from Stanford, to my show.
             <br/>
             and after the performance they come hang with me on my bus and we smoke and then we fk to young dro
             <br/>
             and she holds my face in her hands and her eyes roll back in her head.
             <br/>
             then she goes home and her dad says how was the show? and she says it was fun. they had lasers.
             <br/>
             i hope i become so big and and so white that G-Eazy  will say damn, this na is white and everyone will agree and nod.
             <br/>
             i want to be so big and white that people are scared.  what if this spreads?
             <br/>
             what if everyone starts to get big and white?  what if this works for everyone and everyone can experience this whiteness and this bigness?
             <br/>
             i hope i become too big and too white.
             <br/>
             but i am just a black male.
             <br/>
             i am a na.
            </p>
           </blockquote>
           <blockquote class="twitter-tweet" width="500">
            <p>
             twitter activism is wack.  

marches dont work anymore.  

police should be forced to wear recording devices.
            </p>
            <p>
              childish gambino (@donaldglover)
             <a href="https://twitter.com/donaldglover/statuses/499804677093855232">
              August 14, 2014
             </a>
            </p>
           </blockquote>
           <p>
            The Georgia native hopped back on his account on Thursday afternoon to clarify that his word werent written hastily, and again attempt to hammer home his points.
           </p>
           <blockquote class="twitter-tweet" width="500">
            <p>
             i want people to understand i wasnt under any influence last night.
            </p>
            <p>
              childish gambino (@donaldglover)
             <a href="https://twitter.com/donaldglover/statuses/499984705824251904">
              August 14, 2014
             </a>
            </p>
           </blockquote>
           <blockquote class="twitter-tweet" width="500">
            <p>
             after hearing a friend sobbing and after talking with my mom, i wrote that poem.
            </p>
            <p>
              childish gambino (@donaldglover)
             <a href="https://twitter.com/donaldglover/statuses/499984818638446595">
              August 14, 2014
             </a>
            </p>
           </blockquote>
           <blockquote class="twitter-tweet" width="500">
            <p>
             I've been threatened by police twice this year, hand on holster.  and i'm childish gambino".
            </p>
            <p>
              childish gambino (@donaldglover)
             <a href="https://twitter.com/donaldglover/statuses/499984857519632384">
              August 14, 2014
             </a>
            </p>
           </blockquote>
           <blockquote class="twitter-tweet" width="500">
            <p>
             so when someone calls me a white rapper, i wanna make sure they understand i dont have all the benefits.
            </p>
            <p>
              childish gambino (@donaldglover)
             <a href="https://twitter.com/donaldglover/statuses/499984966852554752">
              August 14, 2014
             </a>
            </p>
           </blockquote>
           <blockquote class="twitter-tweet" width="500">
            <p>
             twitter isnt wack.  activism isnt wack.  twitter activism is wack to me.  its only half of activism.
            </p>
            <p>
              childish gambino (@donaldglover)
             <a href="https://twitter.com/donaldglover/statuses/499985028106190849">
              August 14, 2014
             </a>
            </p>
           </blockquote>
           <blockquote class="twitter-tweet" width="500">
            <p>
             this is not an apology.
            </p>
            <p>
              childish gambino (@donaldglover)
             <a href="https://twitter.com/donaldglover/statuses/499985101410013184">
              August 14, 2014
             </a>

function postClickCallback (evt, target)
{
	//it's a category click
	if ($(target).closest(".category").length &gt; 0)
	{
		return "NewsFeed:"+$(target).text();
	}
	else
	{
		//finding the main post container.
		var $post  = $(target).closest("article").parent();
		var $media = $post.find(".item");

		var position;
		//we calculate the relative position of the post, within its group.
		var relative_position = parseInt($post.attr("class").match(/post-\d+/)[0].replace('post-', ''), 10);

		//if it was loaded dynamically by ajax, we calculate how many post were before and add the relative position.
		var ajax_classes = $post.parent().attr("class").match(/mtvn-ajax-placeholder-\d+/);
		if (ajax_classes !== null)
		{
			var post_per_page = $post.parent().parent().children(".post").length;
			var ajax_count = parseInt(ajax_classes[0].replace('mtvn-ajax-placeholder-', ''), 10);
			position = (ajax_count - 1) * post_per_page + relative_position;
		}
		else
		{
			position = relative_position;
		}

		//selectiong the size
		var size = getSize($media);

		if ($("body").hasClass("home"))
		{
			section = "NewsFeed:";
		}
		else
		{
			section = "CategoryFeed:";
		}

		return section+position+":"+size;
	}
}

function getSize($media)
{
	var size;

	if ($media.hasClass("item-md"))
	{
		size = "medium";
	}
	else if ($media.hasClass("item-sm"))
	{
		size = "small";
	}
	else if ($media.hasClass("item-lg"))
	{
		size = "large";
	}
	else if ($media.hasClass("postcard-left") || $media.hasClass("postcard-right"))
	{
		size = "small";
	}

	return size;
}
