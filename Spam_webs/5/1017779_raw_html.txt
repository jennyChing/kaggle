                    <div class="post-header">
                     <div class="post-header-line-1">

                    <div class="post-body entry-content" id="post-body-93150810226952064" itemprop="description articleBody">
                     One of the next things we are preparing for is the exterior paint.  I want to get this out of the way first for more then one reason.  We will more then likely be covering some of the windows.  In order to cover them I need to know where everything is going to be on the inside.  But I can't start framing the inside until the metal is in place on the outside so we will just have to figure out where everything goes.  Then we will put up the metal and then we will paint.  Today was just some minimal prep work that had to be done sooner or later so I chose sooner.
                     <br/>
                     <br/>
                     First, I went around and removed all the existing mirrors on the bus.  Then I removed the arm mechanism that swings out when the bus stops for pick ups and drop offs.  It left a few holes in the bumper but nothing I am worried about.
                     <br/>
                     <br/>
                     <a href="http://2.bp.blogspot.com/_ul0vmo0wTLM/SgeYzRTTX2I/AAAAAAAAAGY/f-Itv04w63A/s1600-h/Day+5+-+Arm.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5334400290228756322" src="http://2.bp.blogspot.com/_ul0vmo0wTLM/SgeYzRTTX2I/AAAAAAAAAGY/f-Itv04w63A/s320/Day+5+-+Arm.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     I also removed the stop sign.  There are two nice size holes where the stop sign was.  I will have to patch those before I paint.
                     <br/>
                     <br/>
                     <a href="http://2.bp.blogspot.com/_ul0vmo0wTLM/SgeZArZWpMI/AAAAAAAAAGg/VTrr1OkBhZE/s1600-h/Day+5+-+Stop.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5334400520571757762" src="http://2.bp.blogspot.com/_ul0vmo0wTLM/SgeZArZWpMI/AAAAAAAAAGg/VTrr1OkBhZE/s320/Day+5+-+Stop.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     Then I started washing the bus.  I used a brush on a pole and the Clorox All Natural Green Works Cleaner.  I washed all the way around the bus.  Man is that thing big.
                     <br/>
                     <br/>
                     It was dirty and definitely looks better now.  My main goal was to get all the major dirt and grim off the paint to begin preparing it for paint.
                     <br/>
                     <br/>
                     <a href="http://2.bp.blogspot.com/_ul0vmo0wTLM/SgeaC-GJpnI/AAAAAAAAAGo/Fe2iQ9R80iY/s1600-h/Day+5+-+Right+Washed.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5334401659462854258" src="http://2.bp.blogspot.com/_ul0vmo0wTLM/SgeaC-GJpnI/AAAAAAAAAGo/Fe2iQ9R80iY/s320/Day+5+-+Right+Washed.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     <a href="http://4.bp.blogspot.com/_ul0vmo0wTLM/SgeaI3Ujp4I/AAAAAAAAAGw/lnN9bCuunoU/s1600-h/Day+5+-+Left+Washed.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5334401760723445634" src="http://4.bp.blogspot.com/_ul0vmo0wTLM/SgeaI3Ujp4I/AAAAAAAAAGw/lnN9bCuunoU/s320/Day+5+-+Left+Washed.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     Then I had to tackle the roof and it was the dirtiest.
                     <br/>
                     <br/>
                     I am about half way done here.
                     <br/>
                     <br/>
                     <a href="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgeacNMHtKI/AAAAAAAAAG4/emsYo57yfM0/s1600-h/Day+5+-+Roof.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5334402093011154082" src="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgeacNMHtKI/AAAAAAAAAG4/emsYo57yfM0/s320/Day+5+-+Roof.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     Here is a before and after of the roof.
                     <br/>
                     <br/>
                     <a href="http://2.bp.blogspot.com/_ul0vmo0wTLM/SgeaqMbEN8I/AAAAAAAAAHA/iZBi9CheRG0/s1600-h/Day+5+-+Before+%26+After.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5334402333323573186" src="http://2.bp.blogspot.com/_ul0vmo0wTLM/SgeaqMbEN8I/AAAAAAAAAHA/iZBi9CheRG0/s320/Day+5+-+Before+%26+After.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     I got a nice sunburn today and a good work out that's for sure.
                     <br/>
                     <br/>
                     In the back top of the bus it looks like the previous owner had backed into something and bent the metal in right where the warning lights are.   I will be covering these holes so it isn't a big deal but I need the surface to be flat and it isn't.  I was able to pull the metal back out a little.   I will probably work on this more later.
                     <br/>
                     <br/>
                     <a href="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgebZ14I81I/AAAAAAAAAHI/qvkxCpyiedk/s1600-h/Day+5+-+Rear+Damage.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5334403151905223506" src="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgebZ14I81I/AAAAAAAAAHI/qvkxCpyiedk/s320/Day+5+-+Rear+Damage.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     To rap the day up I went in the bus and removed all the "Emergency Exit" stickers that seemed to be on about every other window (Slight exaggeration...only slight.)
                     <br/>
                     <br/>
                     <a href="http://3.bp.blogspot.com/_ul0vmo0wTLM/Sgecgzna7nI/AAAAAAAAAHQ/kDb3pzNZO0g/s1600-h/Day+5+-+Sticker+Removal.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5334404371068939890" src="http://3.bp.blogspot.com/_ul0vmo0wTLM/Sgecgzna7nI/AAAAAAAAAHQ/kDb3pzNZO0g/s320/Day+5+-+Sticker+Removal.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>

                    <h3 class="post-title entry-title" itemprop="name">
                     <a href="http://www.goodnewsbus.com/2009/05/prep-work-foam-plywood.html">
                      Day 4 - Prep Work, Foam &amp; Plywood
                     </a>
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-2380857122040171473" itemprop="description articleBody">
                     Today we actually drove the bus down to the local lumberyard to get the necessary materials that we would need.  This was the first time money was actually spent on this project (not counting the the bus purchase of course).  I think the total amount spent today was around $250.00.  We purchased 8 sheets of 1/2" Plywood, 8 sheets of 1/2" Construction Foam, 105 Phillips Oval Head Stainless Steel Screws sized at: 12 x 1-1/2" (for screwing the plywood down), 1 can of Rust Stop Primer (probably could have used 2), 1 roll of Window Flashing (the sticky black tar stuff) and 2 tubes silicone caulk.  I think that was everything.
                     <br/>
                     <br/>
                     Next we made sure the bus was dry from the previous day of cleaning.
                     <br/>
                     <br/>
                     Then we re-swept everything and made sure it was good and clean.
                     <br/>
                     <br/>
                     I decided to go around and use the rust stop on all major spots of rust.
                     <br/>
                     <br/>
                     <a href="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJEieRpYHI/AAAAAAAAADs/6JqxFKZRSWY/s1600-h/Day+4+-+Rust+Stop+Can.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332900267793408114" src="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJEieRpYHI/AAAAAAAAADs/6JqxFKZRSWY/s320/Day+4+-+Rust+Stop+Can.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     <a href="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgJEpMnY-NI/AAAAAAAAAD0/Q1swIVWbQ5E/s1600-h/Day+4+-+Rust+Stop.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332900383311853778" src="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgJEpMnY-NI/AAAAAAAAAD0/Q1swIVWbQ5E/s320/Day+4+-+Rust+Stop.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     This was just to insure we would not have future problem with rust.
                     <br/>
                     <br/>
                     Then we used stick down rubber window flashing (hope that is what it is called) to cover all the bus seat bolt holes.
                     <br/>
                     <br/>
                     <a href="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgJGhSFDq4I/AAAAAAAAAEc/3iaRkJ7Ypj4/s1600-h/Day+4+-+Holes+to+fill.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332902446362766210" src="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgJGhSFDq4I/AAAAAAAAAEc/3iaRkJ7Ypj4/s320/Day+4+-+Holes+to+fill.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     <a href="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJFbGeOBGI/AAAAAAAAAD8/iAl2rregAUo/s1600-h/Day+4+-+Window+Seal.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332901240656233570" src="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJFbGeOBGI/AAAAAAAAAD8/iAl2rregAUo/s320/Day+4+-+Window+Seal.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     <a href="http://4.bp.blogspot.com/_ul0vmo0wTLM/SgJFifGGIEI/AAAAAAAAAEE/ZriGYicROFs/s1600-h/Day+4+-+Window+Seal+Done.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332901367525023810" src="http://4.bp.blogspot.com/_ul0vmo0wTLM/SgJFifGGIEI/AAAAAAAAAEE/ZriGYicROFs/s320/Day+4+-+Window+Seal+Done.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     After that I went around with Silicone and sealed every crack and small holes left over.
                     <br/>
                     <br/>
                     <a href="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJF06ovkqI/AAAAAAAAAEM/YQMVaimHtV0/s1600-h/Day+4+-+Silicone.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332901684155749026" src="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJF06ovkqI/AAAAAAAAAEM/YQMVaimHtV0/s320/Day+4+-+Silicone.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     <a href="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJF6PhKv0I/AAAAAAAAAEU/SOEv4sY7T-Y/s1600-h/Day+4+-+Silicone+holes.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332901775660465986" src="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJF6PhKv0I/AAAAAAAAAEU/SOEv4sY7T-Y/s320/Day+4+-+Silicone+holes.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     We decided to lay construction foam for a base.  This will provide a vapor barrier as well as insulation and help deaden any unwanted noise.  I haven't seen anyone else do it this way so I hope it all works out.
                     <br/>
                     <br/>
                     The back of the bus is rounded and I needed a way to get a perfect cut.  The roll of flashing worked perfect as a guide.
                     <br/>
                     <br/>
                     <a href="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgJGtYyfvkI/AAAAAAAAAEk/I-YXG-vNmxE/s1600-h/Day+4+-+Foam+Cutting.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332902654322392642" src="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgJGtYyfvkI/AAAAAAAAAEk/I-YXG-vNmxE/s320/Day+4+-+Foam+Cutting.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     <a href="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJG-Xm9aUI/AAAAAAAAAEs/X1Q7SZmQKiI/s1600-h/Day+4+-+Foam+Cut+Corner.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332902946063345986" src="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJG-Xm9aUI/AAAAAAAAAEs/X1Q7SZmQKiI/s320/Day+4+-+Foam+Cut+Corner.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     Looks pretty good!  I will have to use the same pattern for the plywood.
                     <br/>
                     <br/>
                     <a href="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJHOpoARcI/AAAAAAAAAE0/Kt-CNCA8M9s/s1600-h/Day+4+-+Plywood+%26+Foam+Stage+0.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332903225777472962" src="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJHOpoARcI/AAAAAAAAAE0/Kt-CNCA8M9s/s320/Day+4+-+Plywood+%26+Foam+Stage+0.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     Turned out great!
                     <br/>
                     <br/>
                     We laid the foam vertical and the plywood horizontal.  We would lay a few pieces of foam and then plywood and then more foam.  We did it this way and worked our way to the front.
                     <br/>
                     <br/>
                     We pre-drilled the screw holes first.
                     <br/>
                     <br/>
                     I was not very impressed with the stainless steel sheet metal screws that we purchased.  Many of the heads broke off and some of the threads stripped.  I ended up having to drive them in very slow then they worked ok.  I guess the metal on these buses are really ridged.
                     <br/>
                     <br/>
                     <a href="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJJi22HXfI/AAAAAAAAAFk/bRW-pmoFenQ/s1600-h/Day+4+-+Screwing+Plywood.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332905771946958322" src="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJJi22HXfI/AAAAAAAAAFk/bRW-pmoFenQ/s320/Day+4+-+Screwing+Plywood.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     As we went we made sure to sweep and used the shop vac to make sure no debris got under the plywood and foam.
                     <br/>
                     <br/>
                     <a href="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJIJZttdzI/AAAAAAAAAE8/xwieXKJAdSU/s1600-h/Day+4+-+Plywood+%26+Foam+Stage+1.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332904235118720818" src="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJIJZttdzI/AAAAAAAAAE8/xwieXKJAdSU/s320/Day+4+-+Plywood+%26+Foam+Stage+1.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     As you can see we used the window flashing to also seal all the cracks in the foam.
                     <br/>
                     <br/>
                     <a href="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJIQoRQZJI/AAAAAAAAAFE/Sn8KwYbnHdY/s1600-h/Day+4+-+Plywood+%26+Foam+Stage+2.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332904359284991122" src="http://3.bp.blogspot.com/_ul0vmo0wTLM/SgJIQoRQZJI/AAAAAAAAAFE/Sn8KwYbnHdY/s320/Day+4+-+Plywood+%26+Foam+Stage+2.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     As we moved forward we moved all the things that were now in our way to the back of the bus.  This saved moving it over and over again.
                     <br/>
                     <br/>
                     <a href="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgJIcj52QfI/AAAAAAAAAFM/karjHOQNqy0/s1600-h/Day+4+-+Plywood+%26+Foam+Stage+3.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332904564271497714" src="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgJIcj52QfI/AAAAAAAAAFM/karjHOQNqy0/s320/Day+4+-+Plywood+%26+Foam+Stage+3.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     By now it was getting dark and we were tired.  We have just a couple peices of plywood to go and then we are done with that stage.
                     <br/>
                     <br/>
                     <a href="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgJI5bHfMqI/AAAAAAAAAFc/SwyHABd_CAY/s1600-h/Day+4+-+Plywood+%26+Foam+Stage+4.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332905060128993954" src="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgJI5bHfMqI/AAAAAAAAAFc/SwyHABd_CAY/s320/Day+4+-+Plywood+%26+Foam+Stage+4.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     We will probably be back on the bus on friday.  See ya all then!

                    <h3 class="post-title entry-title" itemprop="name">
                     <a href="http://www.goodnewsbus.com/2009/05/day-3-clean.html">
                      Day 3 - CLEAN!
                     </a>
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-888082964458274812" itemprop="description articleBody">
                     Today was pretty strait forward.  Clean, clean, clean.  As you may have noticed by some of the previous pictures in this blog, this bus was dirty and had that old bus smell.  We removed any remaining screws and obstacles.  We swept the bus clean.  Then we brought in the hose, soaked the bus down and mopped, scrubbed, and washed everything in sight.  From the floor to the ceiling, nothing was left untouched.
                     <br/>
                     <br/>
                     <a href="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgER2hPLToI/AAAAAAAAADU/VMNZIwEPW88/s1600-h/Wash+Mop.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332563062116011650" src="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgER2hPLToI/AAAAAAAAADU/VMNZIwEPW88/s320/Wash+Mop.jpg" style="cursor: pointer; width: 240px; height: 320px;"/>
                     </a>
                     <br/>
                     <br/>
                     This step was important because there was a lot of moisture that had been trapped under the plywood and the rubber mat so mildew was present.   Plus this gave the bus a lot better smell.  We used the new Clorox All Natural Green Works Cleaner with the lemon scent.  This made the work day like a big aromatherapy session.
                     <br/>
                     <br/>
                     <a href="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgESL3vqZHI/AAAAAAAAADc/zsnLzlYYpa4/s1600-h/Wash+Floor.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332563428935099506" src="http://1.bp.blogspot.com/_ul0vmo0wTLM/SgESL3vqZHI/AAAAAAAAADc/zsnLzlYYpa4/s320/Wash+Floor.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     The following pic is a before and after of one of the back walls in the bus (not really easy to see).   I had thought that we would end up painting the interior but everything cleaned up really nice so I think we will keep the original paint.
                     <br/>
                     <br/>
                     <a href="http://4.bp.blogspot.com/_ul0vmo0wTLM/SgESiLN864I/AAAAAAAAADk/5Kdlc4Jv8RI/s1600-h/Wash+Before+%26+After.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332563812119538562" src="http://4.bp.blogspot.com/_ul0vmo0wTLM/SgESiLN864I/AAAAAAAAADk/5Kdlc4Jv8RI/s320/Wash+Before+%26+After.jpg" style="cursor: pointer; width: 300px; height: 225px;"/>
                     </a>
                     <br/>
                     <br/>
                     Besides a few other odds and ends this day just consisted of a lot of mops, rags, soapy water and elbow grease (Oh, and don't forget that Fresh Scent).  I am glad this step is over.  Tomorrow we should be about ready to lay the new plywood as long as the floors are completely dry.  I will have a fan blowing on the floors all night.

                 <h2 class="date-header">
                  <span>
                   Monday, May 4, 2009
                  </span>
                 </h2>
                 <div class="date-posts">
                  <div class="post-outer">
                   <div class="post hentry" itemprop="blogPost" itemscope="itemscope" itemtype="http://schema.org/BlogPosting">
                    <meta content="http://2.bp.blogspot.com/_ul0vmo0wTLM/Sf-dBFpsUgI/AAAAAAAAABs/05Rte4znPF8/s320/Heater+Attached.jpg" itemprop="image_url"/>
                    <meta content="9066499431115544049" itemprop="blogId"/>
                    <meta content="6307704222514485113" itemprop="postId"/>
                    <a name="6307704222514485113">
                    </a>
                    <h3 class="post-title entry-title" itemprop="name">
                     <a href="http://www.goodnewsbus.com/2009/05/plywood-removal-more.html">
                      Day 2 - Plywood Removal &amp; More!
                     </a>
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-6307704222514485113" itemprop="description articleBody">
                     Today was long.  Even though I got rained out for part of the day I think overall it went well and was productive.
                     <br/>
                     <br/>
                     First thing that had to be done before the plywood could be removed was the removal of the rear heater.
                     <br/>
                     <br/>
                     <a href="http://2.bp.blogspot.com/_ul0vmo0wTLM/Sf-dBFpsUgI/AAAAAAAAABs/05Rte4znPF8/s1600-h/Heater+Attached.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332153125852434946" src="http://2.bp.blogspot.com/_ul0vmo0wTLM/Sf-dBFpsUgI/AAAAAAAAABs/05Rte4znPF8/s320/Heater+Attached.jpg" style="cursor: pointer; height: 225px; width: 300px;"/>
                     </a>
                     <br/>
                     <br/>
                     The heaters in these buses rely on the heated water/antifreeze from the engine.  It is transported from the front all the way to the back through 2 hoses and attaches into the heater.  One has water coming from the engine the other has water going to the engine.
                     <br/>
                     <br/>
                     <a href="http://1.bp.blogspot.com/_ul0vmo0wTLM/Sf-dKIGHgpI/AAAAAAAAAB0/8_3TZfTQVPA/s1600-h/Heater+w+hoses.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332153281127350930" src="http://1.bp.blogspot.com/_ul0vmo0wTLM/Sf-dKIGHgpI/AAAAAAAAAB0/8_3TZfTQVPA/s320/Heater+w+hoses.jpg" style="cursor: pointer; height: 225px; width: 300px;"/>
                     </a>
                     <br/>
                     <br/>
                     There is water/anti freeze in these lines.  I detached the heater from the floor by removing a few screws and then moved the heater to the side emergency exit where I detached the hoses over a bucket to catch as much of the fluid as possible.
                     <br/>
                     <br/>
                     <a href="http://4.bp.blogspot.com/_ul0vmo0wTLM/Sf-eA3F7_TI/AAAAAAAAAB8/0l9cmzaXSkc/s1600-h/Hanging+Hoses.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332154221455998258" src="http://4.bp.blogspot.com/_ul0vmo0wTLM/Sf-eA3F7_TI/AAAAAAAAAB8/0l9cmzaXSkc/s320/Hanging+Hoses.jpg" style="cursor: pointer; height: 225px; width: 300px;"/>
                     </a>
                     <br/>
                     <br/>
                     After this I put the heater on the trailer.  It's ready for the dump!
                     <br/>
                     <br/>
                     <a href="http://3.bp.blogspot.com/_ul0vmo0wTLM/Sf-eQH3x2cI/AAAAAAAAACE/lfRZdVbSaC0/s1600-h/Heater.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332154483658054082" src="http://3.bp.blogspot.com/_ul0vmo0wTLM/Sf-eQH3x2cI/AAAAAAAAACE/lfRZdVbSaC0/s320/Heater.jpg" style="cursor: pointer; height: 225px; width: 300px;"/>
                     </a>
                     <br/>
                     <br/>
                     The next part was a little tricky.  I had to follow the hose all the way into the engine compartment to figure out how I was going to cap it off.
                     <br/>
                     This is where the hose went from the inside of the bus down under it into the engine compartment.
                     <br/>
                     <br/>
                     <a href="http://2.bp.blogspot.com/_ul0vmo0wTLM/Sf-eyN2QAmI/AAAAAAAAACM/3jH9xffD-LA/s1600-h/Hoses+Up+front.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332155069377806946" src="http://2.bp.blogspot.com/_ul0vmo0wTLM/Sf-eyN2QAmI/AAAAAAAAACM/3jH9xffD-LA/s320/Hoses+Up+front.jpg" style="cursor: pointer; height: 225px; width: 300px;"/>
                     </a>
                     <br/>
                     <br/>
                     This is where it comes down under the bus.
                     <br/>
                     <br/>
                     <a href="http://1.bp.blogspot.com/_ul0vmo0wTLM/Sf-fGXoeDKI/AAAAAAAAACU/DzxySexuN7I/s1600-h/Hose+before.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332155415601745058" src="http://1.bp.blogspot.com/_ul0vmo0wTLM/Sf-fGXoeDKI/AAAAAAAAACU/DzxySexuN7I/s320/Hose+before.jpg" style="cursor: pointer; height: 225px; width: 300px;"/>
                     </a>
                     <br/>
                     <br/>
                     I am no expert but as far as I know there is no harm in just looping the fluid.   I took a piece of hose and made a loop.
                     <br/>
                     <br/>
                     <a href="http://4.bp.blogspot.com/_ul0vmo0wTLM/Sf-fX4_vicI/AAAAAAAAACc/zZva9mOA16M/s1600-h/Manufactuered+Hose.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332155716615506370" src="http://4.bp.blogspot.com/_ul0vmo0wTLM/Sf-fX4_vicI/AAAAAAAAACc/zZva9mOA16M/s320/Manufactuered+Hose.jpg" style="cursor: pointer; height: 225px; width: 300px;"/>
                     </a>
                     <br/>
                     <br/>
                     As I was sitting under the bus working on this it started to rain.  So I was getting wet quick.  Not only that but when I cut into one of the hoses I basically took a bath in anti freeze.  We all know how good that stuff is for you.  So this part was not to fun.  After working on it some time it was done.  The hose is a little kinked but that won't matter anyway because it is just a loop.
                     <br/>
                     <br/>
                     <a href="http://4.bp.blogspot.com/_ul0vmo0wTLM/Sf-gS4uvz-I/AAAAAAAAACk/ObapmKZH2oo/s1600-h/Installed+Hose.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332156730156503010" src="http://4.bp.blogspot.com/_ul0vmo0wTLM/Sf-gS4uvz-I/AAAAAAAAACk/ObapmKZH2oo/s320/Installed+Hose.jpg" style="cursor: pointer; height: 225px; width: 300px;"/>
                     </a>
                     <br/>
                     <br/>
                     UPDATE: I later disconnected the two hoses at the front of the bus where they orginated.
                     <br/>
                     <br/>
                     After the rain let up I decided to start on the plywood removal.
                     <br/>
                     <br/>
                     At first I planned on unscrewing all the stripping with my drill.
                     <br/>
                     <br/>
                     <a href="http://1.bp.blogspot.com/_ul0vmo0wTLM/Sf-glQzKo9I/AAAAAAAAACs/wjnReoNf4Qw/s1600-h/Stripping+Removal.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332157045855134674" src="http://1.bp.blogspot.com/_ul0vmo0wTLM/Sf-glQzKo9I/AAAAAAAAACs/wjnReoNf4Qw/s320/Stripping+Removal.jpg" style="cursor: pointer; height: 225px; width: 300px;"/>
                     </a>
                     <br/>
                     <br/>
                     This didn't really work to well because old rusty screws just don't like to come up so I just decided to rip up the plywood and the stripping all at once.  Probably the fastest way to do this is with a large pry bar.  A really large pry bar.
                     <br/>
                     <br/>
                     <a href="http://1.bp.blogspot.com/_ul0vmo0wTLM/Sf-hGMvTIJI/AAAAAAAAAC0/Anjp7kbNJqA/s1600-h/Steel+Bar+-+Ply.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332157611700854930" src="http://1.bp.blogspot.com/_ul0vmo0wTLM/Sf-hGMvTIJI/AAAAAAAAAC0/Anjp7kbNJqA/s320/Steel+Bar+-+Ply.jpg" style="cursor: pointer; height: 225px; width: 300px;"/>
                     </a>
                     <br/>
                     <br/>
                     Give it a good shove under the wood and lift up and the screws give.  Luckly there were not a ton of screws in this wood.  I think the manufacturer relyed on the bolts from the seats to hold the wood down.
                     <br/>
                     <br/>
                     <a href="http://3.bp.blogspot.com/_ul0vmo0wTLM/Sf-hf0LmdXI/AAAAAAAAAC8/1lTqdkWKAHg/s1600-h/Plywood+removal.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332158051785274738" src="http://3.bp.blogspot.com/_ul0vmo0wTLM/Sf-hf0LmdXI/AAAAAAAAAC8/1lTqdkWKAHg/s320/Plywood+removal.jpg" style="cursor: pointer; height: 225px; width: 300px;"/>
                     </a>
                     <br/>
                     <br/>
                     After doing this for a few hours I was basically done and the plywood for the most part was all out and loaded on the trailer ready for the dump.  What a day!
                     <br/>
                     <br/>
                     <a href="http://4.bp.blogspot.com/_ul0vmo0wTLM/Sf-iIsp43sI/AAAAAAAAADE/ZtvAmLs8UJE/s1600-h/Front+Seats+gone.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332158754139463362" src="http://4.bp.blogspot.com/_ul0vmo0wTLM/Sf-iIsp43sI/AAAAAAAAADE/ZtvAmLs8UJE/s320/Front+Seats+gone.jpg" style="cursor: pointer; height: 225px; width: 300px;"/>
                     </a>
                     <br/>
                     <br/>
                     <a href="http://1.bp.blogspot.com/_ul0vmo0wTLM/Sf-iPW4C6eI/AAAAAAAAADM/cPPMMDgxMVw/s1600-h/Back+Seats+gone.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                      <img alt="" border="0" id="BLOGGER_PHOTO_ID_5332158868552346082" src="http://1.bp.blogspot.com/_ul0vmo0wTLM/Sf-iPW4C6eI/AAAAAAAAADM/cPPMMDgxMVw/s320/Back+Seats+gone.jpg" style="cursor: pointer; height: 225px; width: 300px;"/>
