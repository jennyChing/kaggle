        <p>
         A pie-like time clock in the flat style, built on top of SVG, CSS, CSS3 transforms anda little bit Javascript. Can be used as a timer or a loader to indicate the loading progress.
        </p>
        <h2>
         Getting started:
        </h2>
        <p>
         Draw a time clock using
         <code>
          SVG
         </code>
         element.
        </p>
        <pre class="brush:xml">
         &lt;svg class="rotate" viewbox="0 0 250 250"&gt;
    &lt;path id="loader" transform="translate(125, 125)"/&gt;
&lt;/svg&gt;
        </pre>
        <p>
         Create dots for the time clock.
        </p>
        <pre class="brush:xml">
         &lt;div class="dots"&gt;
    &lt;span class="time deg0"&gt;&lt;/span&gt;
    &lt;span class="time deg45"&gt;&lt;/span&gt;
    &lt;span class="time deg90"&gt;&lt;/span&gt;
    &lt;span class="time deg135"&gt;&lt;/span&gt;
&lt;/div&gt;
        </pre>
        <p>
         Wrap them into a container element with CSS class of timer.
        </p>
        <pre class="brush:xml">
         &lt;div class="timer"&gt;
  &lt;svg class="rotate" viewbox="0 0 250 250"&gt;
    &lt;path id="loader" transform="translate(125, 125)"/&gt;
  &lt;/svg&gt;
  &lt;div class="dots"&gt;
    &lt;span class="time deg0"&gt;&lt;/span&gt;
    &lt;span class="time deg45"&gt;&lt;/span&gt;
    &lt;span class="time deg90"&gt;&lt;/span&gt;
    &lt;span class="time deg135"&gt;&lt;/span&gt;
  &lt;/div&gt;
&lt;/div&gt;
        </pre>
        <p>
         The CSS rules.
        </p>
        <pre class="brush:css">
         .timer {
  width: 300px;
  height: 300px;
  overflow: hidden;
  margin: 0 auto;
  position: relative;
}
.timer .rotate {
  width: 100%;
  height: 100%;
  display: block;
  position: relative;
  z-index: 10;
}
.timer .rotate #loader {
  fill: #ff6d69;
}
.timer .dots {
  width: 100%;
  height: 100%;
  position: absolute;
  z-index: 15;
  top: 0;
  left: 0;
}
.timer .dots .time {
  height: 100%;
  width: 6%;
  display: block;
  background-image: -moz-linear-gradient(#ffc8c6, #ffc8c6 16%, rgba(0, 0, 0, 0) 16%, rgba(0, 0, 0, 0) 84%, #ffc8c6 84%, #ffc8c6);
  background-image: -webkit-linear-gradient(#ffc8c6, #ffc8c6 16%, rgba(0, 0, 0, 0) 16%, rgba(0, 0, 0, 0) 84%, #ffc8c6 84%, #ffc8c6);
  background-image: linear-gradient(#ffc8c6, #ffc8c6 16%, rgba(0, 0, 0, 0) 16%, rgba(0, 0, 0, 0) 84%, #ffc8c6 84%, #ffc8c6);
  margin-left: -3%;
  left: 50%;
  position: absolute;
}
.timer .dots .time.deg45 {
  -moz-transform: rotate(45deg);
  -ms-transform: rotate(45deg);
  -webkit-transform: rotate(45deg);
  transform: rotate(45deg);
}
.timer .dots .time.deg90 {
  -moz-transform: rotate(90deg);
  -ms-transform: rotate(90deg);
  -webkit-transform: rotate(90deg);
  transform: rotate(90deg);
}
.timer .dots .time.deg135 {
  -moz-transform: rotate(135deg);
  -ms-transform: rotate(135deg);
  -webkit-transform: rotate(135deg);
  transform: rotate(135deg);
}
        </pre>
        <p>
         The javascript to enable the time clock.
        </p>
        <pre class="brush:javascript">
         var seconds = 10;
var doPlay = true;
var loader = document.getElementById('loader')
  ,  = 0
  ,  = Math.PI
  , t = (seconds/360 * 1000);

(function draw() {
  ++;
   %= 360;
  var r = (  *  / 180 )
    , x = Math.sin( r ) * 125
    , y = Math.cos( r ) * - 125
    , mid = (  &gt; 180 ) ? 1 : 0
    , anim = 'M 0 0 v -125 A 125 125 1 ' 
           + mid + ' 1 ' 
           +  x  + ' ' 
           +  y  + ' z';
  //[x,y].forEach(function( d ){
  //  d = Math.round( d * 1e3 ) / 1e3;
  //});

  loader.setAttribute( 'd', anim );

  if(doPlay){
    setTimeout(draw, t); // Redraw
  }
})();
        </pre>
