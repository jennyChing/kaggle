      Learn how to make Windows Vista automatically shutdown at a certain time.

     <p>
      <strong>
       <a href="http://maximumpcguides.com/windows-vista/set-vistas-start-menu-power-button-to-turn-off-computer-instead-of-hibernating/#comment-609">
        Electro Geeza

      asked: 
      <em>
       Right now, I am actually looking for some instructions on how to automatically shutdown the computer by using only the built-in components of Windows Vista, without any additional shutdown computer applications (oh boy, there are thousands or even billions of those). I am thinking about using Task scheduler in Windows to set up a computer shutdown command in some way. Any other ideas?
      </em>
      
     </p>
     <p>
      Im 100% with you Electro Geeza.  I try to always use built-in functions in Windows to do a task before I use other software.  Using little applications to perform tasks opens the door to bugs and crashes.  Most of the times there are built in functions to perform most tasks.  Most people just dont know about or how to use them.
     </p>
     <h2>
      How-to Shutdown Windows Vista at a Specified Time
     </h2>
     <p>
      Im going to use the shutdown command included in Windows Vista to shutdown the computer and the task scheduler to shut it down at a specific time.
     </p>
     <ol>
      <li>
       Open Task Scheduler by going to
       <strong>
        Start orb
       </strong>
       ,
       <strong>
        All Programs
       </strong>
       ,
       <strong>
        Accessories
       </strong>
       ,
       <strong>
        System Tools
       </strong>
       , and then open
       <strong>
        Task Scheduler
       </strong>
       .
      </li>
      <li>
       Click
       <strong>
        Create basic task
       </strong>
       on the right side of the Task Scheduler.
       <p>
        <a href="http://maximumpcguides.com/windows-vista/wp-content/uploads/2008/10/shutdown-1.png">
         <img alt="" class="alignnone size-medium wp-image-583" height="333" src="http://cdn.maximumpcguides.com/windows-vista/wp-content/uploads/2008/10/shutdown-1-468x333.png" title="shutdown-1" width="468"/>
        </a>
       </p>
      </li>
      <li>
       Give the task a name and then click
       <strong>
        Next
       </strong>
       .
       <p>
        <a href="http://maximumpcguides.com/windows-vista/wp-content/uploads/2008/10/shutdown-2.png">
         <img alt="" class="alignnone size-medium wp-image-584" height="324" src="http://cdn.maximumpcguides.com/windows-vista/wp-content/uploads/2008/10/shutdown-2-468x324.png" title="shutdown-2" width="468"/>
        </a>
       </p>
      </li>
      <li>
       Select when you want to shutdown the computer and then click
       <strong>
        Next
       </strong>
       .  The next screen will differ depending on when you want the computer to shutdown.  In this example I want the computer to shutdown every Monday  Friday at 6pm.
       <p>
        <a href="http://maximumpcguides.com/windows-vista/wp-content/uploads/2008/10/shutdown-3.png">
         <img alt="" class="alignnone size-medium wp-image-585" height="324" src="http://cdn.maximumpcguides.com/windows-vista/wp-content/uploads/2008/10/shutdown-3-468x324.png" title="shutdown-3" width="468"/>
        </a>
       </p>
      </li>
      <li>
       Next, choose the rest of the details about when you want to shutdown the computer and then click
       <strong>
        Next
       </strong>
       .
       <p>
        <a href="http://maximumpcguides.com/windows-vista/wp-content/uploads/2008/10/shutdown-4.png">
         <img alt="" class="alignnone size-medium wp-image-586" height="324" src="http://cdn.maximumpcguides.com/windows-vista/wp-content/uploads/2008/10/shutdown-4-468x324.png" title="shutdown-4" width="468"/>
        </a>
       </p>
      </li>
      <li>
       Choose to
       <strong>
        Start a program
       </strong>
       and then click
       <strong>
        Next
       </strong>
       .
       <p>
        <a href="http://maximumpcguides.com/windows-vista/wp-content/uploads/2008/10/shutdown-5.png">
         <img alt="" class="alignnone size-medium wp-image-587" height="324" src="http://cdn.maximumpcguides.com/windows-vista/wp-content/uploads/2008/10/shutdown-5-468x324.png" title="shutdown-5" width="468"/>
        </a>
       </p>
      </li>
      <li>
       Type
       <strong>
        shutdown
       </strong>
       under Program/script and type
       <strong>
        s f t 0
       </strong>
       in the Add arguments box and then click
       <strong>
        Next
       </strong>
       .
       <p>
        <a href="http://maximumpcguides.com/windows-vista/wp-content/uploads/2008/10/shutdown-6.png">
         <img alt="" class="alignnone size-medium wp-image-588" height="324" src="http://cdn.maximumpcguides.com/windows-vista/wp-content/uploads/2008/10/shutdown-6-468x324.png" title="shutdown-6" width="468"/>
        </a>
       </p>
       <p>
        Learn more about the
        <a href="http://maximumpcguides.com/windows-vista/use-the-remote-shutdown-tool-to-shutdown-restart-or-logoff-a-local-or-networked-computer/">
         shutdown command
        </a>
        .
       </p>
      </li>
      <li>
       Click
       <strong>
        Finish
       </strong>
       on the next screen to save this scheduled task.
       <p>
        <a href="http://maximumpcguides.com/windows-vista/wp-content/uploads/2008/10/shutdown-7.png">
         <img alt="" class="alignnone size-medium wp-image-589" height="324" src="http://cdn.maximumpcguides.com/windows-vista/wp-content/uploads/2008/10/shutdown-7-468x324.png" title="shutdown-7" width="468"/>
        </a>
       </p>
      </li>
     </ol>

          <p>
           If your computer is hibernating it has already shut down.
          </p>
          <p>
           If its sleeping you can set the scheduled task to wake the computer and run the command by checking off that option in the Conditions tab of the shutdown task you created.
          </p>
          <p>
           <strong>
            Example:
           </strong>
           After youve created the shutdown task, edit the task by opening Task Scheduler, find and open the shutdown task youve created, select the Conditions tab in the shutdown task, and then check off Wake the computer to run this task.
          </p>
