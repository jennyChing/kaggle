&lt;h3 class='post-title entry-title' itemprop='name'&gt;
Reading Around the World: Books About Afghanistan
&lt;/h3&gt;
&lt;div class='post-header'&gt;
&lt;div class='post-header-line-1'&gt;&lt;/div&gt;
&lt;div class='shareaholic-canvas' data-app='share_buttons' data-app-id='7718105'&gt;&lt;/div&gt;
&lt;/div&gt;
&lt;div class='post-body entry-content' id='post-body-1689157021195429939' itemprop='articleBody'&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;
&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;
&lt;a href="http://1.bp.blogspot.com/-SrGrrKiExio/VMkIACLN-yI/AAAAAAAAMRw/3xZra1FET7I/s1600/Afghanistanbooks1creativefamilyfun.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"&gt;&lt;img border="0" src="http://1.bp.blogspot.com/-SrGrrKiExio/VMkIACLN-yI/AAAAAAAAMRw/3xZra1FET7I/s1600/Afghanistanbooks1creativefamilyfun.png" /&gt;&lt;/a&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: center;"&gt;
&lt;br /&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;As part of &lt;a href="http://www.creativefamilyfun.net/2015/01/project-around-world-exploring-world.html" target="_blank"&gt;Project Around the World&lt;/a&gt;, we're trying to read books about each of the countries we virtually visit. I'm learning, though, that this may be easier said than done. Ideally, I'd love to find a combination of both fiction and nonfiction books for all ages for each country. Realistically, we'll do the best we can and with some countries we may have to depend entirely on the Internet.&nbsp;&lt;/span&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&lt;br /&gt;&lt;/span&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;Here are my recommendations for books about Afghanistan.&lt;/span&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&lt;br /&gt;&lt;/span&gt;&lt;/div&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;

Please see my &lt;/i&gt;&lt;a href="http://www.creativefamilyfun.net/p/disclosure-copyright.html" style="font-style: italic;" target="_blank"&gt;disclosure&lt;/a&gt;&lt;i&gt; for more details.&lt;/i&gt;&lt;/span&gt;&lt;/div&gt;
&lt;a name='more'&gt;&lt;/a&gt;&lt;b&gt;&lt;u&gt;&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&lt;br /&gt;&lt;/span&gt;&lt;/u&gt;&lt;/b&gt;
&lt;b&gt;&lt;u&gt;&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;For Elementary-Aged Kids&lt;/span&gt;&lt;/u&gt;&lt;/b&gt;&lt;br /&gt;
&lt;div class="separator" style="clear: both; text-align: left;"&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&lt;br /&gt;&lt;/span&gt;&lt;/div&gt;
&lt;div style="text-align: center;"&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&lt;a href="http://www.amazon.com/gp/product/1575058804/ref=as_li_tl?ie=UTF8&amp;camp=1789&amp;creative=390957&amp;creativeASIN=1575058804&amp;linkCode=as2&amp;tag=crefamfun-20&amp;linkId=KODAOHZGABYIAG4V"&gt;&lt;img border="0" src="http://ws-na.amazon-adsystem.com/widgets/q?_encoding=UTF8&amp;ASIN=1575058804&amp;Format=_SL160_&amp;ID=AsinImage&amp;MarketPlace=US&amp;ServiceVersion=20070822&amp;WS=1&amp;tag=crefamfun-20" /&gt;&lt;/a&gt;&lt;img alt="" border="0" src="http://ir-na.amazon-adsystem.com/e/ir?t=crefamfun-20&amp;l=as2&amp;o=1&amp;a=1575058804" height="1" style="border: none !important; margin: 0px !important;" width="1" /&gt;&nbsp;&lt;/span&gt;&lt;/div&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&lt;a href="http://www.amazon.com/gp/product/0822560496/ref=as_li_tl?ie=UTF8&amp;camp=1789&amp;creative=390957&amp;creativeASIN=0822560496&amp;linkCode=as2&amp;tag=crefamfun-20&amp;linkId=LDJLZVTW2T35W3VP" rel="nofollow" target="_blank"&gt;Count Your Way Through Afghanistan by Jim Haskins &amp; Kathleen Benson&lt;/a&gt;&lt;img alt="" border="0" src="http://ir-na.amazon-adsystem.com/e/ir?t=crefamfun-20&amp;l=as2&amp;o=1&amp;a=0822560496" height="1" style="border: none !important; margin: 0px !important;" width="1" /&gt;&nbsp;- This book will give your kids a good intro about Afghan culture and many of Afghanistan's unique features. It was simple and kept both of my kid's attention. We also loved learning to count to 10 in Pashto, one of the major languages of Afghanistan. This is a great introductory book.&lt;/span&gt;&lt;br /&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&lt;br /&gt;&lt;/span&gt;

&lt;div style="text-align: center;"&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&lt;a href="http://www.amazon.com/gp/product/0981872085/ref=as_li_tl?ie=UTF8&amp;camp=1789&amp;creative=390957&amp;creativeASIN=0981872085&amp;linkCode=as2&amp;tag=crefamfun-20&amp;linkId=SCIA43P3A6BPU5CI"&gt;&lt;img border="0" src="http://ws-na.amazon-adsystem.com/widgets/q?_encoding=UTF8&amp;ASIN=0981872085&amp;Format=_SL160_&amp;ID=AsinImage&amp;MarketPlace=US&amp;ServiceVersion=20070822&amp;WS=1&amp;tag=crefamfun-20" /&gt;&lt;/a&gt;&lt;img alt="" border="0" src="http://ir-na.amazon-adsystem.com/e/ir?t=crefamfun-20&amp;l=as2&amp;o=1&amp;a=0981872085" height="1" style="border: none !important; margin: 0px !important;" width="1" /&gt;&nbsp;&lt;/span&gt;&lt;/div&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&lt;a href="http://www.amazon.com/gp/product/0981872085/ref=as_li_tl?ie=UTF8&amp;camp=1789&amp;creative=390957&amp;creativeASIN=0981872085&amp;linkCode=as2&amp;tag=crefamfun-20&amp;linkId=7MGAIC33SGDECC3W" rel="nofollow" target="_blank"&gt;I See the Sun in Afghanistan by Dodie King&lt;/a&gt;&lt;img alt="" border="0" src="http://ir-na.amazon-adsystem.com/e/ir?t=crefamfun-20&amp;l=as2&amp;o=1&amp;a=0981872085" height="1" style="border: none !important; margin: 0px !important;" width="1" /&gt;&nbsp;- I wish I would have know about this book when we did our project. It's a perfect choice because it tells about the culture through the eyes of a child. I may now have read this book, but my friend Becky at &lt;a href="http://kidworldcitizen.org/2015/01/26/i-see-the-sun-multicultural-books/#more-7133" target="_blank"&gt;Kid World Citizen&lt;/a&gt; recently reviewed it. &lt;a href="http://kidworldcitizen.org/2015/01/26/i-see-the-sun-multicultural-books/#more-7133" target="_blank"&gt;Find her review here&lt;/a&gt;.&lt;/span&gt;&lt;br /&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&lt;br /&gt;&lt;/span&gt;
&lt;br /&gt;
&lt;div style="text-align: center;"&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&lt;a href="http://www.amazon.com/gp/product/0516278150/ref=as_li_tl?ie=UTF8&amp;camp=1789&amp;creative=390957&amp;creativeASIN=0516278150&amp;linkCode=as2&amp;tag=crefamfun-20&amp;linkId=LTIB4A4TVZYA4MQV"&gt;&lt;img border="0" src="http://ws-na.amazon-adsystem.com/widgets/q?_encoding=UTF8&amp;ASIN=0516278150&amp;Format=_SL160_&amp;ID=AsinImage&amp;MarketPlace=US&amp;ServiceVersion=20070822&amp;WS=1&amp;tag=crefamfun-20" /&gt;&lt;/a&gt;&lt;img alt="" border="0" src="http://ir-na.amazon-adsystem.com/e/ir?t=crefamfun-20&amp;l=as2&amp;o=1&amp;a=0516278150" height="1" style="border: none !important; margin: 0px !important;" width="1" /&gt;&nbsp;&lt;/span&gt;&lt;/div&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&nbsp;&lt;a href="http://www.amazon.com/gp/product/0516278150/ref=as_li_tl?ie=UTF8&amp;camp=1789&amp;creative=390957&amp;creativeASIN=0516278150&amp;linkCode=as2&amp;tag=crefamfun-20&amp;linkId=LTIB4A4TVZYA4MQV" rel="nofollow" target="_blank"&gt;A True Book: Afghanistan by Ann Heinrichs&lt;/a&gt;&lt;img alt="" border="0" src="http://ir-na.amazon-adsystem.com/e/ir?t=crefamfun-20&amp;l=as2&amp;o=1&amp;a=0516278150" height="1" style="border: none !important; margin: 0px !important;" width="1" /&gt;&nbsp;- I love this series of books for the basic, age-appropriate introduction it gives about each country of the world. It may be a bit long for a younger child, but it's easy to summarize and it has a lot of great photos. &lt;/span&gt;&lt;br /&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&lt;br /&gt;&lt;/span&gt;
&lt;b&gt;&lt;u&gt;&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;Tweens and Teens&lt;/span&gt;&lt;/u&gt;&lt;/b&gt;&lt;br /&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&lt;br /&gt;&lt;/span&gt;
&lt;br /&gt;
&lt;div style="text-align: center;"&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&lt;a href="http://www.amazon.com/gp/product/1416949313/ref=as_li_tl?ie=UTF8&amp;camp=1789&amp;creative=390957&amp;creativeASIN=1416949313&amp;linkCode=as2&amp;tag=crefamfun-20&amp;linkId=DZO23UMNJ3FP7BIC"&gt;&lt;img border="0" src="http://ws-na.amazon-adsystem.com/widgets/q?_encoding=UTF8&amp;ASIN=1416949313&amp;Format=_SL160_&amp;ID=AsinImage&amp;MarketPlace=US&amp;ServiceVersion=20070822&amp;WS=1&amp;tag=crefamfun-20" /&gt;&lt;/a&gt;&lt;img alt="" border="0" src="http://ir-na.amazon-adsystem.com/e/ir?t=crefamfun-20&amp;l=as2&amp;o=1&amp;a=1416949313" height="1" style="border: none !important; margin: 0px !important;" width="1" /&gt;&nbsp;&lt;/span&gt;&lt;/div&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&nbsp;&lt;a href="http://www.amazon.com/gp/product/1416949313/ref=as_li_tl?ie=UTF8&amp;camp=1789&amp;creative=390957&amp;creativeASIN=1416949313&amp;linkCode=as2&amp;tag=crefamfun-20&amp;linkId=KT7K44WYIVBFMHUB" rel="nofollow" target="_blank"&gt;Extra Credit by Andrew Clements&lt;/a&gt;&lt;img alt="" border="0" src="http://ir-na.amazon-adsystem.com/e/ir?t=crefamfun-20&amp;l=as2&amp;o=1&amp;a=1416949313" height="1" style="border: none !important; margin: 0px !important;" width="1" /&gt;&nbsp;(appropriate for grades 3 through 7) - Sixth grader, Abby Carson, hates doing homework. But, when she finds out she's in danger of being held back, she agrees to do an extra credit project, writing to a pen pal in another country. The letters flow back and forth between Abby and her pen pal, Sadeed and his sister Amira, from Afghanistan. This is a fantastic book that shows that despite cultural differences, we're all the same. Read this one together if you're looking for a book that will spark a lot of great conversations.&lt;/span&gt;&lt;img alt="" border="0" src="http://ir-na.amazon-adsystem.com/e/ir?t=crefamfun-20&amp;l=as2&amp;o=1&amp;a=0888994168" height="1" style="border: none !important; margin: 0px !important;" width="1" /&gt;&lt;span style="font-family: Arial, Helvetica, sans-serif; text-align: center;"&gt;&nbsp;&lt;/span&gt;&lt;br /&gt;
&lt;img alt="" border="0" src="http://ir-na.amazon-adsystem.com/e/ir?t=crefamfun-20&amp;l=as2&amp;o=1&amp;a=B00IN8OTAI" height="1" style="border: none !important; margin: 0px !important;" width="1" /&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&nbsp;&lt;/span&gt;&lt;br /&gt;
&lt;div style="text-align: center;"&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&lt;a href="http://www.amazon.com/gp/product/B00IN8OTAI/ref=as_li_tl?ie=UTF8&amp;camp=1789&amp;creative=390957&amp;creativeASIN=B00IN8OTAI&amp;linkCode=as2&amp;tag=crefamfun-20&amp;linkId=JJWSUJYYJVAOXA4Q"&gt;&lt;img border="0" src="http://ws-na.amazon-adsystem.com/widgets/q?_encoding=UTF8&amp;ASIN=B00IN8OTAI&amp;Format=_SL160_&amp;ID=AsinImage&amp;MarketPlace=US&amp;ServiceVersion=20070822&amp;WS=1&amp;tag=crefamfun-20" /&gt;&lt;/a&gt;&lt;img alt="" border="0" src="http://ir-na.amazon-adsystem.com/e/ir?t=crefamfun-20&amp;l=as2&amp;o=1&amp;a=B00IN8OTAI" height="1" style="border: none !important; margin: 0px !important;" width="1" /&gt;&nbsp;&lt;/span&gt;&lt;/div&gt;
&lt;span style="font-family: Arial, Helvetica, sans-serif;"&gt;&lt;a href="http://www.amazon.com/gp/product/0888994168/ref=as_li_tl?ie=UTF8&amp;camp=1789&amp;creative=390957&amp;creativeASIN=0888994168&amp;linkCode=as2&amp;tag=crefamfun-20&amp;linkId=P4PUHLLD6OWR6I6K" rel="nofollow" target="_blank"&gt;The Breadwinner by Deborah Ellis&lt;/a&gt;&lt;img alt="" border="0" src="http://ir-na.amazon-adsystem.com/e/ir?t=crefamfun-20&amp;l=as2&amp;o=1&amp;a=0888994168" height="1" style="border: none !important; margin: 0px !important;" width="1" /&gt;&nbsp;(appropriate for grades 5 and up) - This book was good friends, really good. It's the story of 11-year-old Parvana, who is living under the Taliban regime in Kabul, Afghanistan. After her father is arrested, Parvana is the only person in the family that is able to work and provide money. She's only able to do this because she can disguise herself as a boy. The story is engaging and eye-opening. It's a must read. Note: This is the first book in a trilogy. I have not yet read the other two, &lt;a href="http://www.amazon.com/gp/product/0888995199/ref=as_li_tl?ie=UTF8&amp;camp=1789&amp;creative=390957&amp;creativeASIN=0888995199&amp;linkCode=as2&amp;tag=crefamfun-20&amp;linkId=UQFNZGNCAHY5HJHU" rel="nofollow" target="_blank"&gt;Parvana's Journey&lt;/a&gt;&lt;img alt="" border="0" src="http://ir-na.amazon-adsystem.com/e/ir?t=crefamfun-20&amp;l=as2&amp;o=1&amp;a=0888995199" height="1" style="border: none !important; margin: 0px !important;" width="1" /&gt;&nbsp;and &lt;a href="http://www.amazon.com/gp/product/0888995423/ref=as_li_tl?ie=UTF8&amp;camp=1789&amp;creative=390957&amp;creativeASIN=0888995423&amp;linkCode=as2&amp;tag=crefamfun-20&amp;linkId=6OEJ7T3PLVSIXFSB" rel="nofollow" target="_blank"&gt;Mud City&lt;/a&gt;&lt;img alt="" border="0" src="http://ir-na.amazon-adsystem.com/e/ir?t=crefamfun-20&amp;l=as2&amp;o=1&amp;a=0888995423" height="1" style="border: none !important; margin: 0px !important;" width="1" /&gt;, but they are on my must-read list.&lt;/span&gt;&lt;br /&gt;

function mce_success_cb(resp){
   $('#mce-success-response').hide();
   $('#mce-error-response').hide();
   if (resp.result=="success"){
       $('#mce-'+resp.result+'-response').show();
       $('#mce-'+resp.result+'-response').html(resp.msg);
       $('#mc-embedded-subscribe-form').each(function(){
           this.reset();
   	});
   } else {
       var index = -1;
       var msg;
       try {
           var parts = resp.msg.split(' - ',2);
           if (parts[1]==undefined){
               msg = resp.msg;
           } else {
               i = parseInt(parts[0]);
               if (i.toString() == parts[0]){
                   index = parts[0];
                   msg = parts[1];
               } else {
                   index = -1;
                   msg = resp.msg;
               }
           }
       } catch(e){
           index = -1;
           msg = resp.msg;
       }
       try{
           if (index== -1){
               $('#mce-'+resp.result+'-response').show();
               $('#mce-'+resp.result+'-response').html(msg);           
           } else {
               err_id = 'mce_tmp_error_msg';
               html = '&lt;div id="'+err_id+'" style="'+err_style+'"&gt; '+msg+'&lt;/div&gt;';
              
               var input_id = '#mc_embed_signup';
               var f = $(input_id);
               if (ftypes[index]=='address'){
                   input_id = '#mce-'+fnames[index]+'-addr1';
                   f = $(input_id).parent().parent().get(0);
               } else if (ftypes[index]=='date'){
                   input_id = '#mce-'+fnames[index]+'-month';
                   f = $(input_id).parent().parent().get(0);
               } else {
                   input_id = '#mce-'+fnames[index];
                   f = $().parent(input_id).get(0);
               }
               if (f){
                   $(f).append(html);
                   $(input_id).focus();
               } else {
                   $('#mce-'+resp.result+'-response').show();
                   $('#mce-'+resp.result+'-response').html(msg);
               }
           }
       } catch(e){
           $('#mce-'+resp.result+'-response').show();
           $('#mce-'+resp.result+'-response').html(msg);
       }
   }
}
