      <h1>
       Easy Ways to Sort Arrays in PHP
      </h1>
      <div style="margin-bottom: 20px; overflow: hidden;">

      <p id="articleSummary">
       Learn how to sort arrays quickly and simply in PHP. This tutorial shows you how to sort indexed, associative and multidimensional arrays using some handy PHP array functions.
      </p>
      <img alt="Using foreach() to Loop Through PHP Arrays" class="newstuff" src="/res/Image/newstuff/generic-php.jpg"/>
      <p>
       As I mentioned in my earlier article,
       <a href="/articles/creating-php-arrays/">
        Creating Arrays in PHP
       </a>
       , arrays are very useful if you need to store and manipulate a large amount of data in one go. One useful way you can manipulate this data is to sort it in a particular order.
      </p>
      <p>
       PHP gives you a large number of sorting functions that can sort arrays using many different criteria. For example, you can sort in ascending or descending order, by key, or by value. These functions make it really easy to sort arrays without getting bogged down in the details of sort algorithms.
      </p>
      <p>
       We won't try to cover every single PHP array-sorting function in this tutorial. Instead, you'll learn about the functions that you're likely to find the most useful in your everyday coding.
      </p>
      <p>
       Let's get started!
      </p>
      <h2>
       Sorting indexed arrays:
       <code>
        sort()
       </code>
       and
       <code>
        rsort()
       </code>
      </h2>
      <p>
       The
       <code>
        sort()
       </code>
       function sorts the values in an indexed array in ascending order. This means that A comes before Z, and 0 comes before 9. Uppercase letters come before lowercase letters, and all letters come before numbers. For example:
      </p>
      <pre class="code">
       <code>
        $myArray = array( 1, 2, 'a', 'b', 'A', 'B' );
sort( $myArray );

// Displays "A B a b 1 2"
foreach ( $myArray as $val ) echo "$val ";
       </code>
      </pre>
      <p>
       <code>
        rsort()
       </code>
       works like
       <code>
        sort()
       </code>
       , except it sorts in descending order:
      </p>
      <pre class="code">
       <code>
        $myArray = array( 1, 2, 'a', 'b', 'A', 'B' );
rsort( $myArray );

// Displays "2 1 b a B A"
foreach ( $myArray as $val ) echo "$val ";
       </code>
      </pre>
      <p class="infoBox">
       All the sorting functions in this tutorial return
       <code>
        true
       </code>
       on success and
       <code>
        false
       </code>
       on failure.
      </p>
      <h2>
       Sorting associative arrays:
       <code>
        asort()
       </code>
       and
       <code>
        arsort()
       </code>
      </h2>
      <p>
       <code>
        sort()
       </code>
       and
       <code>
        rsort()
       </code>
       are fine for indexed arrays, where you don't usually care about the relationship between keys and values. However, they can cause problems with associative arrays. Take the following example:
      </p>
      <pre class="code">
       <code>
        $movie = array( "title" =&gt; "Rear Window",
                "director" =&gt; "Alfred Hitchcock",
                "year" =&gt; 1954,
                "minutes" =&gt; 112 );

sort( $movie );

// Displays "Array ( [0] =&gt; Alfred Hitchcock [1] =&gt; Rear Window [2] =&gt; 112 [3] =&gt; 1954 )"
print_r( $movie );
       </code>
      </pre>
      <p>
       As you can see, the
       <code>
        sort()
       </code>
       function has reindexed the array with numeric indices, destroying the original string indices of
       <code>
        "title"
       </code>
       ,
       <code>
        "director"
       </code>
       ,
       <code>
        "year"
       </code>
       and
       <code>
        "minutes"
       </code>
       .
      </p>
      <p>
       If you want to sort the values in an associative array while preserving keys, use
       <code>
        asort()
       </code>
       and
       <code>
        arsort()
       </code>
       instead. These functions preserve not only the keys, but also the relationships between the keys and their values.
      </p>
      <p>
       To sort the values in ascending order, use
       <code>
        asort()
       </code>
       :
      </p>
      <pre class="code">
       <code>
        $movie = array( "title" =&gt; "Rear Window",
                "director" =&gt; "Alfred Hitchcock",
                "year" =&gt; 1954,
                "minutes" =&gt; 112 );

asort( $movie );

// Displays "Array ( [director] =&gt; Alfred Hitchcock [title] =&gt; Rear Window [minutes] =&gt; 112 [year] =&gt; 1954 )"
print_r( $movie );
       </code>
      </pre>
      <p>
       <code>
        arsort()
       </code>
       sorts the values in descending order  again while preserving the keys:
      </p>
      <pre class="code">
       <code>
        $movie = array( "title" =&gt; "Rear Window",
                "director" =&gt; "Alfred Hitchcock",
                "year" =&gt; 1954,
                "minutes" =&gt; 112 );

arsort( $movie );

// Displays "Array ( [year] =&gt; 1954 [minutes] =&gt; 112 [title] =&gt; Rear Window [director] =&gt; Alfred Hitchcock )"
print_r( $movie );
       </code>
      </pre>
      <h2>
       Sorting associative arrays by key:
       <code>
        ksort()
       </code>
       and
       <code>
        krsort()
       </code>
      </h2>
      <p>
       As well as sorting associative arrays by value, you can also sort them by key.
       <code>
        ksort()
       </code>
       sorts the elements in ascending key order, while
       <code>
        krsort()
       </code>
       sorts in descending key order. Like
       <code>
        asort()
       </code>
       and
       <code>
        arsort()
       </code>
       , these functions preserve the relationships between keys and values.
      </p>
      <p>
       Here are some examples:
      </p>
      <pre class="code">
       <code>
        $movie = array( "title" =&gt; "Rear Window",
                "director" =&gt; "Alfred Hitchcock",
                "year" =&gt; 1954,
                "minutes" =&gt; 112 );

// Displays "Array ( [director] =&gt; Alfred Hitchcock [minutes] =&gt; 112 [title] =&gt; Rear Window [year] =&gt; 1954 )"
ksort( $movie );
print_r( $movie );

// Displays "Array ( [year] =&gt; 1954 [title] =&gt; Rear Window [minutes] =&gt; 112 [director] =&gt; Alfred Hitchcock )"
krsort( $movie );
print_r( $movie );
       </code>
      </pre>
      <div class="articleImageWrap">
       <img alt="Numbers" class="articleImage" src="/res/Image/articles/development/php/easy-ways-to-sort-arrays-in-php/numbers.jpg" style="max-width: 450px; border: 1px solid gray;"/>
      </div>
      <h2>
       Sorting multiple arrays and multidimensional arrays with
       <code>
        array_multisort()
       </code>
      </h2>
      <p>
       The powerful
       <code>
        array_multisort()
       </code>
       function can sort multiple related arrays in one go, preserving the relationships between the arrays. It can also sort multidimensional arrays, so it's a pretty flexible function! Let's take a look at sorting multiple arrays first, then we'll explore sorting multidimensional arrays.
      </p>
      <h3>
       Sorting multiple arrays
      </h3>
      <p>
       To sort multiple arrays using
       <code>
        array_multisort()
       </code>
       , simply pass in all the arrays you want to sort. Here's an example:
      </p>
      <pre class="code">
       <code>
        $directors = array( "Stanley Kubrick", "Alfred Hitchcock", "Martin Scorsese" );
$titles = array( "Full Metal Jacket", "Rear Window", "Mean Streets" );
$years = array( 1987, 1954, 1973 );

array_multisort( $directors, $titles, $years );

print_r( $directors );
echo "&lt;br /&gt;";
print_r( $titles );
echo "&lt;br /&gt;";
print_r( $years );
echo "&lt;br /&gt;";
       </code>
      </pre>
      <p>
       The above code displays the following:
      </p>
      <pre class="samp">
       <samp>
        Array ( [0] =&gt; Alfred Hitchcock [1] =&gt; Martin Scorsese [2] =&gt; Stanley Kubrick )
Array ( [0] =&gt; Rear Window [1] =&gt; Mean Streets [2] =&gt; Full Metal Jacket )
Array ( [0] =&gt; 1954 [1] =&gt; 1973 [2] =&gt; 1987 )
       </samp>
      </pre>
      <p>
       Notice how
       <code>
        array_multisort()
       </code>
       has sorted the values in
       <code>
        $directors
       </code>
       in ascending order, then sorted the other 2 arrays so that the element order matches that of the now-sorted
       <code>
        $directors
       </code>
       array.
      </p>
      <p>
       What if you want to sort by, say, title instead? Just put
       <code>
        $titles
       </code>
       first in the list:
      </p>
      <pre class="code">
       <code>
        array_multisort( $titles, $directors, $years );
       </code>
      </pre>
      <h3>
       Sort by... then by...
      </h3>
      <p>
       If the first array contains some identical values then
       <code>
        array_multisort()
       </code>
       sorts by the values in the first array, then by the values in the second array, and so on. For example:
      </p>
      <pre class="code">
       <code>
        $directors = array( "Stanley Kubrick", "Alfred Hitchcock", "Martin Scorsese", "Stanley Kubrick" );
$titles = array( "Full Metal Jacket", "Rear Window", "Mean Streets", "A Clockwork Orange" );
$years = array( 1987, 1954, 1973, 1971 );

array_multisort( $directors, $titles, $years );

print_r( $directors );
echo "&lt;br /&gt;";
print_r( $titles );
echo "&lt;br /&gt;";
print_r( $years );
echo "&lt;br /&gt;";
       </code>
      </pre>
      <p>
       The above code outputs the following  notice that
       <code>
        "A Clockwork Orange"
       </code>
       appears before
       <code>
        "Full Metal Jacket"
       </code>
       :
      </p>
      <pre class="samp">
       <samp>
        Array ( [0] =&gt; Alfred Hitchcock [1] =&gt; Martin Scorsese [2] =&gt; Stanley Kubrick [3] =&gt; Stanley Kubrick )
Array ( [0] =&gt; Rear Window [1] =&gt; Mean Streets [2] =&gt; A Clockwork Orange [3] =&gt; Full Metal Jacket )
Array ( [0] =&gt; 1954 [1] =&gt; 1973 [2] =&gt; 1971 [3] =&gt; 1987 )
       </samp>
      </pre>
      <h3>
       Changing sort order
      </h3>
      <p>
       You can pass an optional flag argument after an array argument to control the sort order for that particular array:
      </p>
      <dl>
       <dt>
        <code>
         SORT_ASC
        </code>
       </dt>
       <dd>
        Sort in ascending order (the default)
       </dd>
       <dt>
        <code>
         SORT_DESC
        </code>
       </dt>
       <dd>
        Sort in descending order
       </dd>
      </dl>
      <p class="infoBox">
       There are other flags that you can pass to
       <code>
        array_multisort()
       </code>
        and other sorting functions, for that matter  to control the way that arrays are sorted. See the
       <a href="http://www.php.net/manual/en/function.sort.php">
        <code>
         sort()
        </code>
        entry in the PHP manual
       </a>
       for details.
      </p>
      <p>
       The following example sorts the
       <code>
        $directors
       </code>
       array in ascending order, then the
       <code>
        $titles
       </code>
       array in descending order:
      </p>
      <pre class="code">
       <code>
        $directors = array( "Stanley Kubrick", "Alfred Hitchcock", "Martin Scorsese", "Stanley Kubrick" );
$titles = array( "Full Metal Jacket", "Rear Window", "Mean Streets", "A Clockwork Orange" );
$years = array( 1987, 1954, 1973, 1971 );

array_multisort( $directors, SORT_ASC, $titles, SORT_DESC, $years );

print_r( $directors );
echo "&lt;br /&gt;";
print_r( $titles );
echo "&lt;br /&gt;";
print_r( $years );
echo "&lt;br /&gt;";
       </code>
      </pre>
      <p>
       Here's the output  notice that
       <code>
        "Full Metal Jacket"
       </code>
       now comes
       <em>
        before
       </em>
       <code>
        "A Clockwork Orange"
       </code>
       :
      </p>
      <pre class="code">
       <code>
        Array ( [0] =&gt; Alfred Hitchcock [1] =&gt; Martin Scorsese [2] =&gt; Stanley Kubrick [3] =&gt; Stanley Kubrick )
Array ( [0] =&gt; Rear Window [1] =&gt; Mean Streets [2] =&gt; Full Metal Jacket [3] =&gt; A Clockwork Orange )
Array ( [0] =&gt; 1954 [1] =&gt; 1973 [2] =&gt; 1987 [3] =&gt; 1971 )
       </code>
      </pre>
      <h3>
       Sorting multidimensional arrays
      </h3>
      <p>
       If you pass a multidimensional array to
       <code>
        array_multisort()
       </code>
       then it sorts the array by looking at the first element of each nested array. If 2 such elements have the same value then it sorts by the second element, and so on.
      </p>
      <p>
       Here's an example that shows how this works. It uses
       <code>
        array_multisort()
       </code>
       to sort a multidimensional array by director, then by title, then by year:
      </p>
      <pre class="code">
       <code>
        $movies = array(
  array(
    "director" =&gt; "Alfred Hitchcock",
    "title" =&gt; "Rear Window",
    "year" =&gt; 1954
  ),
  array(
    "director" =&gt; "Stanley Kubrick",
    "title" =&gt; "Full Metal Jacket",
    "year" =&gt; 1987
  ),
  array(
    "director" =&gt; "Martin Scorsese",
    "title" =&gt; "Mean Streets",
    "year" =&gt; 1973
  ),
  array(
    "director" =&gt; "Stanley Kubrick",
    "title" =&gt; "A Clockwork Orange",
    "year" =&gt; 1971
  )
);

array_multisort( $movies );
echo "&lt;pre&gt;";
print_r( $movies );
echo "&lt;/pre&gt;";
       </code>
      </pre>
      <p>
       The above code produces the following:
      </p>
      <pre class="code">
       <code>
        Array
(
    [0] =&gt; Array
        (
            [director] =&gt; Alfred Hitchcock
            [title] =&gt; Rear Window
            [year] =&gt; 1954
        )

    [1] =&gt; Array
        (
            [director] =&gt; Martin Scorsese
            [title] =&gt; Mean Streets
            [year] =&gt; 1973
        )

    [2] =&gt; Array
        (
            [director] =&gt; Stanley Kubrick
            [title] =&gt; A Clockwork Orange
            [year] =&gt; 1971
        )

    [3] =&gt; Array
        (
            [director] =&gt; Stanley Kubrick
            [title] =&gt; Full Metal Jacket
            [year] =&gt; 1987
        )

)
       </code>
      </pre>
      <p>
       You can see that
       <code>
        array_multisort()
       </code>
       has sorted the array by director. Where the director is the same (
       <code>
        "Stanley Kubrick"
       </code>
       ), it's then sorted by title.
      </p>
      <p class="infoBox">
       To sort the array in descending order instead, pass the
       <code>
        SORT_DESC
       </code>
       flag as the second argument to
       <code>
        array_multisort()
       </code>
       . Easy!
      </p>
      <h2>
       Summary
      </h2>
      <p>
       In this tutorial you've looked at a few common PHP functions for sorting arrays:
      </p>
      <ul>
       <li>
        <code>
         sort()
        </code>
        and
        <code>
         rsort()
        </code>
        for sorting indexed arrays
       </li>
       <li>
        <code>
         asort()
        </code>
        and
        <code>
         arsort()
        </code>
        for sorting associative arrays
       </li>
       <li>
        <code>
         ksort()
        </code>
        and
        <code>
         krsort()
        </code>
        for sorting associative arrays by key
       </li>
       <li>
        <code>
         array_multisort()
        </code>
        for sorting multiple related arrays, as well as multidimensional arrays
       </li>
      </ul>
      <p>
       PHP's array sorting functions are very powerful and flexible, and let you easily and quickly sort arrays any way you like. In fact, at the time of writing there are 13 different PHP functions for sorting arrays! To find out more, check out
       <a href="http://www.php.net/manual/en/array.sorting.php">
        this list
       </a>
       in the PHP manual.
      </p>
      <p>
       Happy coding!
      </p>
