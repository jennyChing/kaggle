   <h1>
    Windows Key Shortcuts
   </h1>
   <p>
    These first thirteen (13) shortcuts utilize the
    <strong>
     Windows Key
    </strong>
    <img alt="" height="13" src="http://www.seoconsultants.com/windows/tips/images/windows-key.gif" width="15"/>
    which is located at the lower left hand corner and, the lower right hand corner of your keyboard between the
    <strong>
     Ctrl
    </strong>
    and
    <strong>
     Alt
    </strong>
    keys. If it is not there, then you do not have a Windows keyboard.
   </p>
   <p class="i20">
    <kbd>
     Ctrl
    </kbd>
    <kbd>
     <img alt="" height="13" src="http://www.seoconsultants.com/windows/tips/images/windows-key.gif" width="15"/>
     Windows Key
    </kbd>
    <kbd>
     Alt
    </kbd>
   </p>
   <p>
    The keyboard image effects that you see above and below are being controlled through an external .css (Cascading Style Sheet) file. These are not actual images, but a carefully thought out list of CSS attributes that gives the appearance of a key on a Windows keyboard.
   </p>
