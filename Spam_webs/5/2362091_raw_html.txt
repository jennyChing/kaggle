            I thought it might be useful to collate all the information about the new 'Smart' Location Bar as implemented in Firefox 3.
            <br/>
            This guide will detail some methods to customise the bar, and its Functioning.
            <br/>
            <br/>
            FIRST ...
            <span style="font-weight: bold">
             <span style="color: #FF0000">
              There is currently no way to make the Location bar behave
              <span style="font-style: italic">
               <span style="text-decoration: underline">
                exactly
               </span>
              </span>
              like Firefox 2
             </span>
            </span>
            but I hope this guide helps...
            <br/>
            <br/>
            NEXT .... Make sure you actually know how the bar works before you rush to customise, or criticise it...
            <br/>
            Here is a good guide to how the new Location bar works...
            <!-- m -->
            <a class="postlink" href="http://www.dria.org/wordpress/archives/2008/04/17/628/">
             http://www.dria.org/wordpress/archives/2008/04/17/628/
            </a>
            <!-- m -->
            <br/>
            <br/>
            Much discussion on the Forums has been had about this, already. (see the end of section 3)
            <br/>
            Firefox 3.1 will have more ways to control the Location Bar when it is released ( I will include details as I get them... currently collecting them here
            <!-- l -->
            <a class="postlink-local" href="http://forums.mozillazine.org/viewtopic.php?f=23&amp;t=827405">
             viewtopic.php?f=23&amp;t=827405
            </a>
            <!-- l -->
            )
            <br/>
            <br/>
            I am dividing this Guide into four sections...
            <br/>
            <br/>
            1. Appearance
            <br/>
            2. Extensions to Alter the Location Bar
            <br/>
            3. Altering what is searched for and the Search Algorithm (The way the Search Works)
            <br/>
            4. MINI TUTORIAL - Adding code snippets to userChrome.css and userContent file
            <br/>
            _________________________________________________________________________________________________________
            <br/>
            <br/>
            <span style="font-size: 125%; line-height: 116%;">
             <span style="font-weight: bold">
              <span style="text-decoration: underline">
               1. Appearance
              </span>
             </span>
            </span>
            <br/>
            <br/>
            <span style="font-weight: bold">
             a) Appearance and Number of Results
            </span>
            <br/>
            <br/>
            i. To increase the number of results, Install this extension
            <!-- m -->
            <a class="postlink" href="https://addons.mozilla.org/en-US/firefox/addon/7674">
             https://addons.mozilla.org/en-US/firefox/addon/7674
            </a>
            <!-- m -->
            <br/>
            (This is an experimental Addon, so you will need to register at the addons site to Install)
            <br/>
            <br/>
            ii. To get the results to appear on one line, like Firefox 2, install the Old Bar Extension
            <!-- m -->
            <a class="postlink" href="https://addons.mozilla.org/en-US/firefox/addon/6227">
             https://addons.mozilla.org/en-US/firefox/addon/6227
            </a>
            <!-- m -->
            <br/>
            (Note, this only affects the appearance, NOT the function)
            <br/>
            <br/>
            iii. To change the colours and font of the results, see this article
            <!-- m -->
            <a class="postlink" href="http://mozillalinks.org/wp/2008/03/tweak-the-location-bar-autocomplete-menu/">
             http://mozillalinks.org/wp/2008/03/twea ... lete-menu/
            </a>
            <!-- m -->
            <br/>
            <br/>
            <span style="font-weight: bold">
             b) Location Bar Appearance
            </span>
            <br/>
            <br/>
            While some extensions exist to modify the appearance, many fixes can be accomplished via CSS placed in the userchrome.css file
            <br/>
            For information on this, see the Guide in Section 4 of this article.
            <br/>
            <br/>
            <span style="font-weight: bold">
             i. The Star
            </span>
            <br/>
            By default, the star should always visible for bookmarkable pages.
            <br/>
            <br/>
            This code  will hide the star only if the page is bookmarked
            <br/>
            <dl class="codebox">
             <dt>
              Code:
              <a href="#" onclick="selectCode(this); return false;">
               Select all
              </a>
             </dt>
             <dd>
              <code>
               #urlbar-icons &gt; #star-button[starred="true"]
               <br/>
               {
               <br/>
               display: none !important;
               <br/>
               }
              </code>
             </dd>
            </dl>
            <br/>
            <br/>
            This code will hide the star only if the page is not bookmarked
            <br/>
            <dl class="codebox">
             <dt>
              Code:
              <a href="#" onclick="selectCode(this); return false;">
               Select all
              </a>
             </dt>
             <dd>
              <code>
               #urlbar-icons &gt; #star-button:not([starred="true"])
               <br/>
               {
               <br/>
               display: none !important;
               <br/>
               }
              </code>
             </dd>
            </dl>
            <br/>
            <br/>
            If you want to always hide the star, no matter what, use this code
            <br/>
            <dl class="codebox">
             <dt>
              Code:
              <a href="#" onclick="selectCode(this); return false;">
               Select all
              </a>
             </dt>
             <dd>
              <code>
               #urlbar-icons &gt; #star-button
               <br/>
               {
               <br/>
               display: none !important;
               <br/>
               }
              </code>
             </dd>
            </dl>
            <br/>
            <br/>
            To keep Always Visible use this code
            <br/>
            <dl class="codebox">
             <dt>
              Code:
              <a href="#" onclick="selectCode(this); return false;">
               Select all
              </a>
             </dt>
             <dd>
              <code>
               #urlbar[pageproxystate="invalid"] &gt; #urlbar-icons &gt; #star-button
               <br/>
               {
               <br/>
               visibility: visible !important;
               <br/>
               }
              </code>
             </dd>
            </dl>
            <br/>
            <br/>
            <br/>
            <span style="font-weight: bold">

            <br/>
            <span style="font-size: 125%; line-height: 116%;">
             <span style="text-decoration: underline">
              <span style="font-weight: bold">
               2. Extensions to Alter the Location Bar
              </span>
             </span>
            </span>
            <br/>
            <br/>
            The following Extensions set some of the preferences listed below, and may be easier to use.
            <br/>
            <ul>
             <li>
              Hide Unvisited 3
              <!-- m -->
              <a class="postlink" href="https://addons.mozilla.org/en-US/firefox/addon/7429">
               https://addons.mozilla.org/en-US/firefox/addon/7429
              </a>
              <!-- m -->
              <br/>
              This shows only URLs in your History
             </li>
             <li>
              Awesome Options Extension
              <!-- m -->
              <a class="postlink" href="https://addons.mozilla.org/en-US/firefox/addon/8310">
               https://addons.mozilla.org/en-US/firefox/addon/8310
              </a>
              <!-- m -->
              <br/>
              This gives you many options to fine tune the Location Bar.
             </li>
             <li>
              Old Location Bar
              <!-- m -->
              <a class="postlink" href="https://addons.mozilla.org/en-US/firefox/addon/7637">
               https://addons.mozilla.org/en-US/firefox/addon/7637
              </a>
              <!-- m -->
              <br/>
              This returns the Location Bar to an
              <span style="font-style: italic">
               almost
              </span>
              Firefox 2 Behaviour...
             </li>
            </ul>
            Note, some of these extensions are experimental, you will need to register at AMO to install them.
            <br/>
            <br/>
            <span style="font-size: 125%; line-height: 116%;">
             <span style="font-weight: bold">
              <span style="text-decoration: underline">
               3.Altering what is searched for and the Search Algorithm (The way the Search Works)
              </span>
             </span>
            </span>
            <br/>
            <br/>
            Type
            <span style="font-weight: bold">
             about:config
            </span>
            in the Location Bar and look for these Preferences..
            <br/>
            <br/>
            <span style="font-weight: bold">
             a) Exclude All Results, or change the number of results
            </span>
            <br/>
            <br/>
            <span style="font-weight: bold">
             browser.urlbar.maxRichResults
            </span>
            <br/>
            This setting is only really useful if you either want to remove the autocomplete function entirely, or limit the list to a shorter length on your screen real estate, now that titles are a part of the result. Setting to 0 (zero) will just stop all autocomplete results.
            <br/>
            <br/>
            <span style="font-weight: bold">
             b) Exclude Bookmarks you have not visited
            </span>
            <br/>
            <br/>
            <span style="font-weight: bold">
             places.frecency.unvisitedBookmarkBonus
            </span>
            This preference set to 0 (zero) will remove the bookmarks you haven't visited from your results.
            <br/>
            <br/>
            <span style="font-weight: bold">
             c) Exclude Bookmarks you have visited
            </span>
            <br/>
            <br/>
            <span style="font-weight: bold">
             places.frecency.bookmarkVisitBonus
            </span>
            <br/>
            This preference set to 0 (zero) will remove the bookmarks you have visited from your results.
            <br/>
            <br/>
            <span style="font-weight: bold">
             d) Exclude URLS you have not visited
            </span>
            <br/>
            <br/>
            <span style="font-weight: bold">
             places.frecency.unvisitedTypedBonus
            </span>
            <br/>
            This setting will have the greatest impact on how your results are found.. A setting of 0 (zero) should give you a result of finding your URLs based on what you type being anywhere in the URL or page title. A setting of 1 should result in placing of what you type anywhere in the title or URL being first, followed by the URL or title beginning with what you type. A setting of 2 should result in only showing results based on what you type being at the beginning of the URL or page title.
            <br/>
            <br/>
            <span style="font-weight: bold">
             e) Include only URLS you have typed
            </span>
            <br/>
            <br/>
            <span style="font-weight: bold">
             browser.urlbar.matchOnlyTyped
            </span>
            Set to true
            <br/>
            <br/>
            <span style="font-weight: bold">
             f) Changing order of Search
            </span>
            <br/>
            <br/>
            <span style="font-weight: bold">
             browser.urlbar.matchBehavior
            </span>
            <br/>
            This setting will have the greatest impact on how your results are found, regardless of what sort of results you remove from the ones available (which will be limited in following entries). I've not seen results as I'd expect from this setting yet, but I'm not done working with it. A setting of 0 (zero) should give you a result of finding your URLs based on what you type being anywhere in the URL or page title. A setting of 1 should result in placing of what you type anywhere in the title or URL being first, followed by the URL or title beginning with what you type. A setting of 2 should result in only showing results based on what you type being at the beginning of the URL or page title.
            <br/>
            <br/>
            <span style="font-weight: bold">
             g) Enabling Inline Auto Complete
            </span>
            <br/>
            <br/>
            <span style="font-weight: bold">
             browser.urlbar.autoFill
            </span>
            <br/>
            Set this to TRUE to enable IE style Autocomplete in the Locationbar.
            <br/>
            <br/>
            As I said, above, there is currently no way to make the Location bar behave
            <span style="font-style: italic">
             exactly
            </span>
            like Firefox 2 but I hope this guide helps...

            <span style="font-size: 125%; line-height: 116%;">
             <span style="font-weight: bold">
              <span style="text-decoration: underline">
               4.MINI TUTORIAL
              </span>
              - Adding code snippets to userChrome.css and userContent files
             </span>
            </span>
            by Frank Lion
            <br/>
            <br/>
            We can alter how Firefox looks or behaves by adding some code lines to files in your Firefox
            <a class="postlink" href="http://kb.mozillazine.org/Profile_folder" target="_blank">
             profile
            </a>
            called
            <a class="postlink" href="http://kb.mozillazine.org/UserChrome.css" target="_blank">
             userChrome.css.
            </a>
            or
            <a class="postlink" href="http://kb.mozillazine.org/UserContent.css" target="_blank">
             userContent.css
            </a>
            <br/>
            <br/>
            In the following example, we will be adding a code snippet to userChrome.css
            <br/>
            <br/>
            <span style="font-weight: bold">
             1.
            </span>
            Using the information in the Knowledge Base links above, locate the chrome folder within your profile. If you have not edited your userChrome.css before, then you will see a file called userChrome-example.css in the chrome folder, together with userContent-example.css
            <br/>
            <br/>
            <span style="font-weight: bold">
             2.
            </span>
            Now, we can edit this file using any old editor, even Notepad or Wordpad. But, I suggest you use this program,
            <a class="postlink" href="http://www.pnotepad.org/" target="_blank">
             Programmers Notepad.
            </a>
            This will show the important parts in colour for you and if you go to Options/General/Default, will allow you to have line numbers, as well. This makes it easier for everybody to help you, if you ever get stuck.
            <br/>
            <br/>
            <span style="font-weight: bold">
             3.
            </span>
            Open the userChrome-example.css file with your editor of choice and unless otherwise directed, copy and paste your code snippet - just as it is written,
            <span style="font-weight: bold">
             below
            </span>
            this line -
            <br/>
            <br/>
            <span style="font-weight: bold">
             @namespace url("http://www.mozilla.org/keymaster/gatekeeper/there.is.only.xul");
            </span>
            <br/>
            <br/>
            This is called the namespace line and you must not alter it.
            <br/>
            <br/>
            <span style="font-weight: bold">
             4.
            </span>
            Use
            <span style="font-weight: bold">
             'Save As'
            </span>
            to save your file as
            <span style="font-weight: bold">
             userChrome.css
            </span>
            into the same chrome folder as the example one.
            <br/>
            <br/>
            <span style="font-weight: bold">
             5.
            </span>
            Finally, restart Firefox to see the result of all your work. Contrary to myth, Firefox does not need to be closed as you make these changes, but you must always restart to actually implement the changes.
            <br/>
            <br/>
            <br/>
            <br/>
            <span style="font-weight: bold">
             Notes :
            </span>
            <br/>
            <br/>
            <span style="font-weight: bold">
             a.)
            </span>
            If you later wish to disable a particular code snippet then just move the */ onto the end of the final line, like this :
            <br/>
            <br/>
            <span style="font-weight: bold">
             OFF:
             <br/>
            </span>
            <br/>
            /*....Frank's Urlbar Yellow on FF3 https sites...
            <br/>
            #urlbar[level] .autocomplete-textbox-container {
            <br/>
            background-color: #FFFFB7 !important;
            <br/>
            }*/
            <br/>
            <br/>
            <br/>
            <span style="font-weight: bold">
             ON :
            </span>
            <br/>
            <br/>
            /*....Frank's Urlbar Yellow on FF3 https sites...*/
            <br/>
            #urlbar[level] .autocomplete-textbox-container {
            <br/>
            background-color: #FFFFB7 !important;
            <br/>
            }
            <br/>
            <br/>
            <br/>
            By moving that, */ ... we are telling the program just to ignore it. The ON stuff will be in colour in the P.Notepad and in black when it's OFF.
            <br/>
            <br/>
            <br/>
            <span style="font-weight: bold">
             b.)
            </span>
            If you think you have messed up anything, then just EXIT from your Editor - do not 'save'. It will all be as it was before.
