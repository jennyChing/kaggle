                    <h3 class="post-title entry-title" itemprop="name">
                     Raw Almond Butter
                    </h3>

                     <div class="post-header-line-1">

                       <br/>
                       <br/>
                       I feel so guilty knowing that I ever bought nut butter from the store. I never knew how much cheaper it is to make it at home... quite honestly, I didn't know it was possible until a few months ago. Plus, it's super easy, incredibly convenient, and oh so fresh and healthy!
                       <br/>
                       <br/>
                       So let's calculate how much
                       <i>
                        cha-ching!
                       </i>
                       we save by making almond butter at home:
                       <br/>
                       <br/>
                       One 3lb bag of raw almonds costed me $10.99 at Costco (YAY for food in bulk!), with the capability of making me ~5.5 cups of raw almond butter. That's only
                       <b>
                        $1.99 per cup
                       </b>
                       !
                       <br/>
                       <br/>
                       Compare that to
                       <a href="http://www.artisanafoods.com/products/almond" target="_blank">
                        Artisana's
                       </a>
                       raw almond butter (the brand I used to buy ~1x a month), which costs $11.99 for a 16oz jar, or
                       <b>
                        $5.99 per cup
                       </b>
                       !
                       <br/>
                       <br/>
                       <br/>
                       <div class="separator" style="clear: both; text-align: center;">
                        <a href="http://2.bp.blogspot.com/-RfFxzM4Dylw/T--UZHmYviI/AAAAAAAAEoY/UaO76h0MoXg/s1600/2.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                         <img border="0" height="488" src="http://2.bp.blogspot.com/-RfFxzM4Dylw/T--UZHmYviI/AAAAAAAAEoY/UaO76h0MoXg/s640/2.jpg" width="640"/>
                        </a>
                       </div>
                       <br/>
                       So after making this almond butter and trying to count how many jars of almond butter I've gone through(and extra dollars I've spent)over the years, I want to slap myself.
                       <i>
                        Hard
                       </i>
                       .
                       <br/>
                       <br/>
                       But you don't have to slap yourself for wasting any money, because now you can make almond butter at home too :)
                       <br/>
                       <div>
                        <br/>
                       </div>
                       <br/>
                       <b>
                        <u>
                         Raw Almond Butter:
                        </u>
                       </b>
                       <br/>
                       <i>
                        (makes about 1.5 cups)
                       </i>
                       <br/>
                       3 cups Raw, unsalted Almonds
                       <br/>
                       <br/>
                       <u>
                        Directions:
                       </u>
                       <br/>
                       1. Add the almonds to a food processor and pulse to chop.
                       <br/>
                       2. Blend the chopped almonds for about 10-15 minutes, scraping the sides when necessary.
                       <br/>
                       3. Add salt or
                       <a href="http://astore.amazon.com/chockohlawtay-20/detail/B001E5E3JY" target="_blank">
                        stevia
                       </a>
                       /sweetener to taste (I didn't, but it all depends on taste) and scoop/pour into a pretty sealable jar. Store tightly sealed in the fridge.
                       <br/>
                       <br/>
                       <u>
                        Note
                       </u>
                       : I used a 4-cup processor (yes it's small and anal retentive, but it works!) that recommends running for a max of 60 seconds, so it took me a little longer than it would a person with a normal sized processor. However, the process is actually kinda fun so I didn't mind the extra 5 minutes or so. While you process, note that a ball of nut butter will form, which I hear is totally normal so don't worry if that happens to you, just split it up while you scrape down the sides of the processor. After processing a few more times the ball will disappear. When it looks nice and buttery, just run it a couple minutes longer to make sure it'll stay like that after refrigeration.
                       <br/>
                       <br/>
                       <b>
                        <u>

                        </u>
