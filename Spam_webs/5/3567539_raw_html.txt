       <article class="article-body">
        <p data-diply="0">
         You know when you name a celebrity based off of something that just reminds you of them? Sometimes it's a character they played so well that they immediately and immortally become THAT person, or naming them something after a distinct feature they have.
        </p>
        <p data-diply="1">
         Kind of like the way we call Lisa Kudrow "Phoebe" or Patrick Dempsey "McDreamy" or Beyonc "Queen Bey" or even calling Jason Momoa "My future husband" (...wait, that's just me, isn't it?)
        </p>
        <p data-diply="2">
         Well, lo and behold, whatever "weird" nicknames you had for them aren't so weird after all!
        </p>
        <p data-diply="3">
         Turns out, according to
         <a href="http://edition.cnn.com/2015/02/03/entertainment/china-katy-perry-celebrity-nicknames/index.html" target="_blank">
          CNN
         </a>
         , China has a slew of weird nicknames for celebrities.
        </p>
        <p data-diply="3">
         But obviously, since these are culturally specific, those of us who aren't Chinese won't fully grasp the full translation or understand the meaning.
        </p>
        <p data-diply="3">
         Regardless, check them out, they'll guarantee a laugh:
        </p>
        <h3 data-diply="4">
         1.
         <a href="http://edition.cnn.com/2015/02/03/entertainment/china-katy-perry-celebrity-nicknames/index.html" target="_blank">
          Nicki Minaj
         </a>
         , also known as Numbing-Spicy Chicken:
        </h3>
        <div data-diply="5">
         <p data-diply="6">
          <span style="font-size: 1em; line-height: 1.6em;">
           I'm not entirely sure why they've named Nicki "Numbing-Spicy Chicken," but it pretty much implies that she's so incredibly spicy hot that she'll numb your senses.
          </span>
         </p>

         <h3>
          2.
          <a href="http://edition.cnn.com/2015/02/03/entertainment/china-katy-perry-celebrity-nicknames/index.html" target="_blank">

          </a>
          ,or in Chinese layman's terms, Pikachu:
         </h3>
         <p>
          This happened out of a sheer fluke incident when, in 2011, a Taiwanese news anchor was struggling to pronounce DiCaprio's surname and ended up blurting out the word "Pikachu."

    function ToggleLikeButtonOnPostDetails(response, appId) {
        var ngScope = $('#pd-scope').scope();
        if (ngScope.fblike.show) {
            if (response.status === 'connected') {
                if (typeof FB != 'undefined') {
                    FB.api('/me/likes/' + appId, function (mydata) {
                        if (mydata.data.length &gt; 0) {
                            ngScope.CloseFacebookLike();
                            $('.fbPageAlert').hide();
                        } else {
                            ngScope.ShowFacebookLike();
                        }
                    });
                }
            } else if (response.status === 'not_authorized') {
                // the user is logged in to Facebook but has not authenticated your app
                $('.fbPageAlert').hide();
            } else {
                // the user isn't logged in to Facebook.
                $('.fbPageAlert').hide();
            }
        } else {
            $('.fbPageAlert').hide();
        }
    }

    function getSocialSum() {

        if ($('.social-count').attr('data-facebookShares') != undefined
        &amp;&amp; $('.social-count').attr('data-twitterShares') != undefined
        &amp;&amp; $('.social-count').attr('data-pinshares') != undefined
        &amp;&amp; $('.social-count').attr('data-googShares') != undefined) {
            var fbShares = parseInt($('.social-count').attr('data-facebookShares'));
            var twitShares = parseInt($('.social-count').attr('data-twitterShares'));
            var pinShares = parseInt($('.social-count').attr('data-pinshares'));
            var googShares = parseInt($('.social-count').attr('data-googshares'));
            var totalShares = fbShares + twitShares + pinShares + googShares;
            $('.share-count').text(totalShares);
            if (totalShares &gt;= 500)
                $('.social-count').fadeIn();
        } else if (socialCountRetries &gt; 0) {
            //Try again
            setTimeout(function () {
                getSocialSum();
            }, 500);
            socialCountRetries--;
        }
    }
