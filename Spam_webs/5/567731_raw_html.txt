      <b>
       Step 3.Make a Script file and define Effects for Login and Popup Box
      </b>
     </p>
     <p class="explain_para">
      In this we define effects for our Popup Box using jQuery and save it with a name
      <b>
       login_effect.js
      </b>

You can dowload jQuery from this
      <a href="http://jquery.com/">

     <div class="code_div">
      <pre>
       $(document).ready(function(){

   $("#show_login").click(function(){
    showpopup();
   });
   $("#close_login").click(function(){
    hidepopup();
   });

});


function showpopup()
{
   $("#loginform").fadeIn();
   $("#loginform").css({"visibility":"visible","display":"block"});
}

function hidepopup()
{
   $("#loginform").fadeOut();
   $("#loginform").css({"visibility":"hidden","display":"none"});
}
      </pre>
     </div>
