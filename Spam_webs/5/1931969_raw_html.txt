        <h1 class="entry-title">
         How to disable Metro edge swipes and touch charms bar gestures for trackpads (touchpads) in Windows 8.1 and Windows 8
        </h1>
       </header>
       <!-- .entry-header -->
       <div class="entry-content">
        <p>
         One of the most annoying features of Windows 8.1 and Windows 8 are the touch gestures for indirect touch devices such as trackpads (touchpads). These gestures invoke various aspects of the Modern UI such as Charms, App Switcher, App bar etc. On the Desktop, these gestures aren't of much use and they often get triggered accidentally when you use your touchpad. Even if you disabled on-screen mouse pointer gestures for the Modern UI, which are triggered from hot corners, these annoying gestures remain enabled and popup at the most inopportune times. Let us see how to disable them.
        </p>
        <p>
         <span id="more-2428">
         </span>
        </p>
        <p>
         The on-screen hot corners which open various aspects of the Metro UI can easily be disabled using apps such as
         <a href="http://winaero.com/comment.php?comment.news.219" target="_blank" title="Winaero Charms Bar Killer allows you to get rid of Charms Bar in Windows 8.1/Windows 8 completely">
          Winaero Charms Bar Killer
         </a>
         or
         <!--noindex-->
         <a href="http://www.classicshell.net/" rel="nofollow" target="_blank">
          Classic Shell
         </a>
         <!--/noindex-->
         . Even though Windows 8.1 comes with built-in options to disable the top left and top right hot corners, setting the options on the Taskbar Properties Navigation tab disables these features globally, even inside Metro apps. Instead if you use apps like Classic Shell to disable hot corners, they get disabled only on the Desktop but remain operational inside Modern apps and the Start screen, where they can be useful.
        </p>
        <p>
         As for indirect touch gestures, Microsoft worked with various touchpad vendors to explicitly enable these in the drivers for these touchpads. There are multiple edge swipe gestures:
        </p>
        <ul>
         <li>
          Swiping in from the left edge of the touchpad towards the center shows the app switcher or switches to the most recently used Metro app
         </li>
         <li>
          Swiping in from the right edge of the touchpad towards the center shows the Charms
         </li>
         <li>
          Swiping in from the top edge of the touchpad towards the center shows the App Bar (if you are in a Metro app)
          <br/>
          ....and others
         </li>
        </ul>
        <p>
         These third party touchpad drivers often integrate with your Mouse Control Panel, so that is where you must go to disable them. Open Control Panel (
         <a href="http://winaero.com/blog/do-you-know-all-these-ways-to-open-control-panel-in-windows-8-1/">
          see how
         </a>
         ) and then open Mouse settings. There are multiple touchpad vendors and each vendor has its own settings UI to control these. Let us see how to disable these one by one:
        </p>
        <p>
         <strong style="font-size: medium;">
          <span style="font-size: medium;">
           Synaptics touchpad
          </span>
          <br/>
         </strong>
        </p>
        <ol>
         <li>
          Click the Device Settings tab in the Mouse Control Panel.
         </li>
         <li>
          Click on the Settings button.
         </li>
         <li>
          Uncheck the option 'Enable Edge Swipes' and click OK.
          <div class="wp-caption aligncenter" id="attachment_2436" style="width: 478px">
           <a data-lightbox="true" href="http://winaero.com/blog/wp-content/uploads/2014/01/Synaptics1.png">
            <img alt="Synaptics touchpad settings" class="size-full wp-image-2436" height="467" src="http://winaero.com/blog/wp-content/uploads/2014/01/Synaptics1.png" width="468"/>
           </a>
           <p class="wp-caption-text">
            Synaptics touchpad settings
           </p>
          </div>
          <p>
          </p>
          <div class="wp-caption aligncenter" id="attachment_2437" style="width: 632px">
           <a data-lightbox="true" href="http://winaero.com/blog/wp-content/uploads/2014/01/Synaptics2.png">
            <img alt="Synaptics touchpad settings" class="size-full wp-image-2437" height="464" src="http://winaero.com/blog/wp-content/uploads/2014/01/Synaptics2.png" width="622"/>
           </a>
           <p class="wp-caption-text">
            Synaptics touchpad settings
           </p>
          </div>
         </li>
        </ol>
        <p>
         <strong>
          <span style="font-size: medium;">
           Lenovo UltraNav touchpad
          </span>
          <br/>
         </strong>
        </p>
        <ol>
         <li>
          Click the UltraNav tab in the Mouse Control Panel.
         </li>
         <li>
          Under the TouchPad section, click the Settings button.
         </li>
         <li>
          Expand the Application Gestures section and click on Edge Swipes.
         </li>
         <li>
          Uncheck 'Enable Edge Swipes'.
          <div class="wp-caption aligncenter" id="attachment_2438" style="width: 499px">
           <a data-lightbox="true" href="http://winaero.com/blog/wp-content/uploads/2014/01/Ultranav1.png">
            <img alt="UltraNav touchpad settings" class="size-full wp-image-2438" height="536" src="http://winaero.com/blog/wp-content/uploads/2014/01/Ultranav1.png" width="489"/>
           </a>
           <p class="wp-caption-text">
            UltraNav touchpad settings
           </p>
          </div>
          <p>
          </p>
          <div class="wp-caption aligncenter" id="attachment_2439" style="width: 632px">
           <a data-lightbox="true" href="http://winaero.com/blog/wp-content/uploads/2014/01/Ultranav2.png">
            <img alt="UltraNav touchpad settings" class="size-full wp-image-2439" height="464" src="http://winaero.com/blog/wp-content/uploads/2014/01/Ultranav2.png" width="622"/>
           </a>
           <p class="wp-caption-text">
            UltraNav touchpad settings
           </p>
          </div>
         </li>
        </ol>
        <p>
         <span style="font-size: medium;">
          <strong>
           Alps touchpad
          </strong>
         </span>
        </p>
        <ol>
         <li>
          Click the EdgeAction (TM) tab in the Mouse Control Panel.
         </li>
         <li>
          Uncheck the gestures you don't want (Left/Right/Top/Top Left EdgeAction), and then click OK.
          <p>
          </p>
          <div class="wp-caption aligncenter" id="attachment_2431" style="width: 490px">
           <a data-lightbox="true" href="http://winaero.com/blog/wp-content/uploads/2014/01/Alps.png">
            <img alt="Alps touchpad settings" class="size-full wp-image-2431" height="500" src="http://winaero.com/blog/wp-content/uploads/2014/01/Alps.png" width="480"/>
           </a>
           <p class="wp-caption-text">
            Alps touchpad settings
           </p>
          </div>
         </li>
        </ol>
        <p>
         <span style="font-size: medium;">
          <strong>
           Elan touchpad
          </strong>
         </span>
        </p>
        <ol>
         <li>
          Click the ELAN tab in the Mouse Control Panel.
         </li>
         <li>
          Click the Enable Device button if that button is disabled, then click Options.
         </li>
         <li>
          Click Edge Swipe and disable them.
          <div class="wp-caption aligncenter" id="attachment_2433" style="width: 457px">
           <a data-lightbox="true" href="http://winaero.com/blog/wp-content/uploads/2014/01/Elan1.png">
            <img alt="Elan touchpad settings" class="size-full wp-image-2433" height="463" src="http://winaero.com/blog/wp-content/uploads/2014/01/Elan1.png" width="447"/>
           </a>
           <p class="wp-caption-text">
            Elan touchpad settings
           </p>
          </div>
          <p>
          </p>
          <div class="wp-caption aligncenter" id="attachment_2434" style="width: 702px">
           <a data-lightbox="true" href="http://winaero.com/blog/wp-content/uploads/2014/01/Elan2.png">
            <img alt="Elan touchpad settings" class="size-full wp-image-2434" height="509" src="http://winaero.com/blog/wp-content/uploads/2014/01/Elan2.png" width="692"/>
           </a>
           <p class="wp-caption-text">
            Elan touchpad settings
           </p>
          </div>
         </li>
        </ol>
        <p>
         <span style="font-size: medium;">
          <strong>
           Logitech touchpad
          </strong>
         </span>
        </p>
        <ol>
         <li>
          Open Logitech SetPoint Settings.
         </li>
         <li>
          Click the section called Navigate Windows (with the icon of the black pointing hand)
         </li>
         <li>
          Uncheck any options you want. Only Switch applications, Show Charms and Show Windows 8 App Bar are 1-finger gestures, the rest are multi-touch so you are unlikely to trigger them accidentally. Click OK.
          <p>
          </p>
          <div class="wp-caption aligncenter" id="attachment_2435" style="width: 826px">
           <a data-lightbox="true" href="http://winaero.com/blog/wp-content/uploads/2014/01/Logitech.png">
            <img alt="Logitech touchpad settings" class="size-full wp-image-2435" height="619" src="http://winaero.com/blog/wp-content/uploads/2014/01/Logitech.png" width="816"/>
           </a>
           <p class="wp-caption-text">
            Logitech touchpad settings
           </p>
          </div>
         </li>
        </ol>
        <p>
         <strong>
          <span style="font-size: medium;">
           Dell Touchpad
          </span>
         </strong>
        </p>
        <ol>
         <li>
          Open the Mouse Control Panel and you will see the Dell Touchpad tab.
         </li>
         <li>
          Click the link called 'Click to change Dell Touchpad settings'.
         </li>
         <li>
          Another window will open. Click the Gestures section there.
         </li>
         <li>
          You can turn off the gestures you don't want individually or turn off gestures completely. Then click the Save button and close the window.
          <br/>
          <a data-lightbox="true" href="http://winaero.com/blog/wp-content/uploads/2014/01/Dell-Touchpad.png">
           <img alt="Dell Touchpad" class="size-full wp-image-3521 alignleft" height="383" src="http://winaero.com/blog/wp-content/uploads/2014/01/Dell-Touchpad.png" width="624"/>
          </a>
         </li>
        </ol>
        <p>
         <strong>
          <span style="font-size: medium;">
           <br/>
           ASUS Smart Gesture touchpad
          </span>
         </strong>
        </p>
        <ol>
         <li>
          In the notification area (system tray), click the icon for ASUS Smart Gesture touchpad. The icon may be hidden in the overflow area of the tray, in that case, click the tiny arrow and then click the icon.
         </li>
         <li>
          Click the Edge Gesture tab.
         </li>
         <li>
          Uncheck any options you don't want: Toggle Charm Bar (right edge), Toggle Menu Bar (top edge) and Switch Running Applications (left edge). Then click OK.
          <p>
          </p>
          <div class="wp-caption aligncenter" id="attachment_2432" style="width: 941px">
           <a data-lightbox="true" href="http://winaero.com/blog/wp-content/uploads/2014/01/Asus-Smart-Gesture.png">
            <img alt="Asus Smart Gesture touchpad settings" class="size-full wp-image-2432" height="691" src="http://winaero.com/blog/wp-content/uploads/2014/01/Asus-Smart-Gesture.png" width="931"/>
           </a>
           <p class="wp-caption-text">
            Asus Smart Gesture touchpad settings
           </p>
          </div>
         </li>
        </ol>
        <p>
         Often various hardware OEMs (Lenovo, Samsung, HP) rebrand these touchpad driver settings in the Mouse Control Panel so the exact name of the tab may be different. For example, if you have OEM drivers for the touchpad installed, then the tab may be named differently. But the settings UI should be more or less the same and it should be easy enough for you to figure where to disable these annoying swipe gestures. The touchpad settings will in a majority of cases be in either the Mouse Control Panel or in the taskbar notification (tray) area.
        </p>
        <p>
         Finally, if you have a relatively new PC with a modern Precision Touchpad designed to work with Windows 8.1, then you can disable these gestures right from PC Settings.
        </p>
        <ol>
         <li>
          Press the Win + I keys together on your keyboard and click PC Settings. Go to PC and devices -&gt; Mouse and touchpad.Tip: You can create a shortcut to directly open Mouse and Touchpad settings.
          <a href="http://winaero.com/blog/how-to-create-a-shortcut-to-the-mouse-and-touchpad-settings-in-windows-8-1/">
           See how
          </a>
          .
         </li>
         <li>
          If you have a precision touchpad, then there will be options to disable those edges swipes/gestures there.
         </li>
         <li>
          Turn off the option "Enable swiping in from the left or right edge".
          <div class="wp-caption aligncenter" id="attachment_2447" style="width: 494px">
           <a data-lightbox="true" href="http://winaero.com/blog/wp-content/uploads/2014/01/89e64fca-53f1-4c39-b740-d1b6d2ac7699_171.png">
            <img alt="Windows 8.1 Touchpad settings" class="size-full wp-image-2447" height="316" src="http://winaero.com/blog/wp-content/uploads/2014/01/89e64fca-53f1-4c39-b740-d1b6d2ac7699_171.png" width="484"/>
           </a>
           <p class="wp-caption-text">
            Windows 8.1 Touchpad settings
           </p>
          </div>
          <p>
           That's it. Now those touch gestures won't bother you any more. Works like a charm, oops, pardon the pun...actually it works quite unlike a charm! :)
          </p>
         </li>
        </ol>
       </div>

              <br/>
              1. Open the Registry editor
              <br/>
              2. Go to the following registry key:
             </p>
             <pre>
              HKEY_CURRENT_USER\Software\Elantech\SmartPadDisplay
             </pre>
             <p>
              3. Create here a new dword value named EdgeSwipe_Display and set it to 1
              <br/>
              4. Create here yet another value named EdgeSwipeWin8_Display and set it to 1
              <br/>
              5. Reboot your PC and check if that required option will appear in Elan touchpad settings.

           <p>
            Hi,
           </p>
           <p>
            Is the method to disable edge swiping from PC Settings driver or Windows dependent?
           </p>
           <p>
            I have a Synaptics v7.5 touchpad on my laptop with latest OEM driver v18.1.3.0 and Windows 8.1 with Update 1 ( integrated clean install) and in the Synaptics driver settings theres no option to enable/disable Edge Swiping.
           </p>
           <p>
            In Windows PC Settings &gt; Corner and Edges, I only have the option to turn on/off When I swipe from the left edge, switch between my recent apps.
           </p>
           <p>
            What worked for me to disable Edge Swiping with my Synaptics touchpad and Windows 8.1 with Update 1 was changing these values via Registry Editor:
           </p>
           <p>
            [HKEY_CURRENT_USER\Software\Synaptics\SynTPEnh\ZoneConfig\TouchPadPS2\Left Slider Zone]
            <br/>
            ActionType=dword:00000000
            <br/>
            *This value disables left edge app switching slider, which can be done via Windows PC Settings on my laptop*
            <br/>
            [HKEY_CURRENT_USER\Software\Synaptics\SynTPEnh\ZoneConfig\TouchPadPS2\Right Edge Pull Extended Zone]
            <br/>
            ActionType=dword:00000000
            <br/>
            *This value disables right edge charms bar in which CANNOT be done via Windows PC Settings on my laptop, default value is 2*
            <br/>
            [HKEY_CURRENT_USER\Software\Synaptics\SynTPEnh\ZoneConfig\TouchPadPS2\Top Edge Pull]
            <br/>
            ActionType=dword:00000000
            <br/>
            *This value disables top edge swipe which was disabled by default with my drivers*
            <br/>
            [HKEY_CURRENT_USER\Software\Synaptics\SynTPEnh\ZoneConfig\TouchPadPS2\Bottom Edge Pull]
            <br/>
            ActionType=dword:00000000
            <br/>
            *This value disabled bottom edge swipe which has no functions with touchpad, default value is 2*
           </p>
