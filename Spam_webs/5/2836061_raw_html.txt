        <h2>
         1. Task Manager for Websites
        </h2>
        <p>
         While you are inside Google Chrome, press Shift+Escape and it will open up a task manager with a list of all websites currently open inside Chrome.
        </p>
        <p>
         <img alt="chrom task manager" border="0" height="242" labnol="http://img.labnol.org/di/taskmanager.png" src="" title="chrom task manager" width="511"/>
        </p>
        <noscript>
         <img alt="chrom task manager" border="0" height="242" src="http://img.labnol.org/di/taskmanager.png" title="chrom task manager" width="511"/>
        </noscript>
        <p>
         This will give you an idea about how much physical memory is consumed by different websites and if any particular page is causing your system to crawl, you can do an "end process" to close that tab directly from Task Manager.
        </p>
        <h2>
         2. Visual Browser History
        </h2>
        <p>
         This is a feature Chrome borrowed from Google Desktop / Google Web History. Type Ctrl+H to open your browser history and search for a term.
        </p>
        <p>
         <img alt="visual-history" border="0" height="247" labnol="http://img.labnol.org/di/visualhistory.png" src="" title="visual-history" width="504"/>
        </p>
        <noscript>
         <img alt="visual-history" border="0" height="247" src="http://img.labnol.org/di/visualhistory.png" title="visual-history" width="504"/>
        </noscript>
        <p>
         In addition to matching pages, Google Chrome will also show you a small thumbnail image of that web pages in the history results. No need for
         <a href="http://labnol.blogspot.com/2007/06/view-your-web-browser-history-in-visual.html">
          any extensions
         </a>
         .
        </p>
        <h2>
         <img align="right" alt="chrome contextual menu" border="0" height="124" labnol="http://img.labnol.org/di/chromemenu.png" src="" title="chrome contextual menu" width="172"/>
         <noscript>
          <img align="right" alt="chrome contextual menu" border="0" height="124" src="http://img.labnol.org/di/chromemenu.png" title="chrome contextual menu" width="172"/>
         </noscript>
         3. Super Clean Contextual Menus
        </h2>
        <p>
         I simply love this. You right click an hyperlink on a web page and you get only five relevant options to deal with that link.
        </p>
        <p>
         The number comes down to four when you right click any image in Chrome. Theres way too much
         <a href="http://labnol.blogspot.com/2007/06/remove-unwanted-clutter-from-firefox.html">
          clutter in Firefox menus
         </a>
         .
        </p>
        <h2>
         4. Search Your Favorite Websites from the Address Bar
        </h2>
        <p>
         This is brilliant. If you visit a website that includes site search (for instance:
         <a href="http://search.labnol.org">
          search.labnol.org
         </a>
         )  Chrome will automatically recognize and add that search engine for you so the next time you can perform a search on that site via the Chrome address bar itself.
        </p>
        <p>
         <img alt="chrome search" border="0" height="39" labnol="http://img.labnol.org/di/searchinchrome.png" src="" title="chrome search" width="531"/>
        </p>
        <noscript>
         <img alt="chrome search" border="0" height="39" src="http://img.labnol.org/di/searchinchrome.png" title="chrome search" width="531"/>
        </noscript>
        <p>
         To see this in action, open cnn.com or search.labnol.org in your Chrome browser and then type cnn.com followed by the tab key to search CNN. Simple.
        </p>
        <h2>
         5. See Memory Used by Different Browsers
        </h2>
        <p>
         Open a new tab inside Chrome browser and type "about:memory" (without quotes)  somewhere at the top, youll see a list of browser processes that are currently running on your system and the amount of memory they are using.
        </p>
        <p>
         <img alt="browser memory" border="0" height="151" labnol="http://img.labnol.org/di/browsermemory.png" src="" title="browser memory" width="466"/>
        </p>
        <noscript>
         <img alt="browser memory" border="0" height="151" src="http://img.labnol.org/di/browsermemory.png" title="browser memory" width="466"/>
        </noscript>
        <h2>
         6. Reopen Website tabs that you closed by mistake
        </h2>
        <p>
         Firefox 3 has this "Undo Closed Tab" option in the menu while you can open closed tabs in Opera via the Ctrl+Z shortcut.
        </p>
        <p>
         <img alt="closed-tabs" border="0" height="222" labnol="http://img.labnol.org/di/closedtabs.png" src="" title="closed-tabs" width="500"/>
        </p>
        <noscript>
         <img alt="closed-tabs" border="0" height="222" src="http://img.labnol.org/di/closedtabs.png" title="closed-tabs" width="500"/>
        </noscript>
        <p>
         To re-open a closed tab in Google Chrome, just hit Ctrl+T and youll see an option that says "Recently closed tabs"  click the one you closed by accident.
        </p>
        <h2>
         7. Launch Websites from the Start Menu / Quick Launch Bar
        </h2>
        <p>
         <a href="http://www.labnol.org/internet/office/quickly-create-new-google-docs-documents-desktop-shortcuts/3668/">
          Desktop shortcuts
         </a>
         for web pages are possible with other browsers as well but Google Chrome make the whole flow very easy. Open any site and choose "Create application shortcut" from the File menu.
        </p>
        <p>
         <img alt="web-shortcuts" border="0" height="234" labnol="http://img.labnol.org/di/webshortcuts.png" src="" title="web-shortcuts" width="500"/>
        </p>
        <noscript>
         <img alt="web-shortcuts" border="0" height="234" src="http://img.labnol.org/di/webshortcuts.png" title="web-shortcuts" width="500"/>
        </noscript>
        <p>
         This will essentially create a desktop shortcut that looks something like this:
        </p>
        <pre>
         C:\Users\labnol\AppData\Local\Google\Chrome\Application\chrome.exe 
    --app=http://www.labnol.org/
        </pre>
        <p>
         Clicking the shortcut will launch that web page in a new instance of Chrome that may not contain have any more new tabs.
        </p>
        <p>
         Related Tutorial:
         <a href="http://www.labnol.org/software/create-family-profiles-in-google-chrome/4394/">
          <strong>
           How to Create User Profiles in Google Chrome
          </strong>
         </a>
        </p>
        <p>
         Chrome lets you move tabs around different windows but I didnt find this feature very useful. I like the Firefox approach better where you can drag tabs onto the desktop and turn them into web shortcuts.
        </p>
        <p>
         Overall, I am pretty impressed with Chrome but did miss support for extensions especially
         <a href="http://www.labnol.org/software/browsers/make-google-better-customize-google-firefox/2500/">
          Customize Google
         </a>
         . But Chrome may be a good thing if you are web publisher as it has
         <a href="http://labnol.blogspot.com/2007/01/google-recommends-firefox-mozilla.html">
          no adblock plus
         </a>
          good for Google as well.
        </p>
        <p>
         <img alt="google chrome error" border="0" height="251" labnol="http://img.labnol.org/di/googlechromeerror.png" src="" title="google chrome error" width="501"/>
        </p>
        <noscript>
         <img alt="google chrome error" border="0" height="251" src="http://img.labnol.org/di/googlechromeerror.png" title="google chrome error" width="501"/>
