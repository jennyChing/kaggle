                <div id="mainPostContent">
                 <div dir="ltr" style="text-align: left;" trbidi="on">
                  <a href="http://3.bp.blogspot.com/-BJKbj9MnJ8E/UOLrWfA_4TI/AAAAAAAAGZg/j3c-QIML9cY/s1600/truecaller.png" imageanchor="1" style="clear: right; float: right; margin-bottom: 1em; margin-left: 1em;">
                   <span style="font-family: Arial, Helvetica, sans-serif;">
                    <img border="0" height="88" src="http://3.bp.blogspot.com/-BJKbj9MnJ8E/UOLrWfA_4TI/AAAAAAAAGZg/j3c-QIML9cY/s320/truecaller.png" width="320"/>
                   </span>
                  </a>
                  <span style="font-family: Arial, Helvetica, sans-serif;">
                   Few days back, I wrote a nice post on
                   <a href="http://www.usethistip.com/2013/01/how-to-trace-phone-number-and-find.html" rel="nofollow" target="_blank">
                    tracing a phone number and search the owner
                   </a>
                   . That post was about a nice app Trucaller. Truecaller is mobile app that is available in all popular mobile platform. This app contains a large database of phone numbers and their owners. And allows users to search a phone number and know about the owner.
                  </span>
                  <br/>
                  <a name="more">
                  </a>
                  <br/>
                  <ul style="text-align: left;">
                   <li>
                    <span style="font-family: Arial, Helvetica, sans-serif;">
                     What if I do not want to be traced?
                    </span>
                   </li>
                   <li>
                    <span style="font-family: Arial, Helvetica, sans-serif;">
                     How to keep out information private and remove my number from Truecaller if it has my number?
                    </span>
                   </li>
                  </ul>
                  <span style="font-family: Arial, Helvetica, sans-serif;">
                   These are the two questions which I was askedmany times after writing the post about Truecaller.
                  </span>
                  <br/>
                  <span style="font-family: Arial, Helvetica, sans-serif;">
                   <br/>
                  </span>
                  <span style="font-family: Arial, Helvetica, sans-serif;">
                   Before knowing this, you should know how Truecaller works.
                  </span>
                  <br/>
                  <h3>
                   <span style="font-family: Arial, Helvetica, sans-serif;">
                    How True Caller Works?
                   </span>
                  </h3>
                  <span style="font-family: Arial, Helvetica, sans-serif;">
                   TrueCaller is a mobile app and is available on all popular mobile platforms. When people installs Truecaller app in their mobile phones, it grabs all contacts from the phone and put it in Truecaller database. All contacts in TrueCaller database are available publicly. Any one can search number from Truecaller database.
                  </span>
                  <br/>
                  <h3>
                   <span style="font-family: Arial, Helvetica, sans-serif;">
                    How to remove number from TrueCaller Database?
                   </span>
                  </h3>
                  <span style="font-family: Arial, Helvetica, sans-serif;">
                   If you want to remove your number from TrueCaller, you can request from the company directly from a form. Just
                   <a href="http://www.truecaller.com/unlist" rel="nofollow" target="_blank">
                    visit this form
                   </a>
                   and enter your mobile number.
                  </span>
                  <br/>
                  <div class="separator" style="clear: both; text-align: center;">
                   <a href="http://3.bp.blogspot.com/-_L7Jb9bnx9U/UP_tTYWE67I/AAAAAAAAHEg/vgpSU7JcM1w/s1600/trucaller-unlist.bmp" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                    <span style="font-family: Arial, Helvetica, sans-serif;">
                     <img border="0" height="165" src="http://3.bp.blogspot.com/-_L7Jb9bnx9U/UP_tTYWE67I/AAAAAAAAHEg/vgpSU7JcM1w/s320/trucaller-unlist.bmp" width="320"/>
                    </span>
                   </a>
                  </div>
                  <span style="font-family: Arial, Helvetica, sans-serif;">
                   <br/>
                  </span>
                  <span style="font-family: Arial, Helvetica, sans-serif;">
                   After submitting this form, your number will be removed from TrueCaller database within few days.
                  </span>
                 </div>
