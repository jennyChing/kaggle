       A detailed description of the feature flags is in the
       <a href="https://www.freebsd.org/cgi/man.cgi?query=zpool-features&amp;apropos=0&amp;sektion=7&amp;manpath=FreeBSD+10.1-stable&amp;arch=default&amp;format=html" onclick="_gaq.push(['_trackPageview', '/extlink/www.freebsd.org/cgi/man.cgi?query=zpool-features&amp;apropos=0&amp;sektion=7&amp;manpath=FreeBSD+10.1-stable&amp;arch=default&amp;format=html']);">
        <b>
         zpool-features(7)
        </b>
       </a>
       manual page.
       <br/>
       FreeBSD releases up to 8.3 and including 9.0 and 9.1 do not support feature flags.
       <br/>
       <br/>
       <b>
        UPDATE 16.01.2015:
       </b>
       fixed unavailable embedded_data on FreeBSD 9.3, thanks to Dmitry Kononov
       <br/>
       <b>
        UPDATE 12.04.2015:
       </b>
       updated zfsonlinux from 0.6.3 to 0.6.4, OpenZFS OSX from 1.3.0 to 1.3.1, OmniOS from r151012 to r151014
       <br/>
       <br/>
       If you plan to access your ZFS pools from earlier operating system versions (or other operating systems) you have to consider that if you enable any unsupported features the pool may become read-only or unreadable, depending on the specific feature. Features supported by all systems above are async_destroy, empty_bpobj and lz4_compress.
       <br/>
       <br/>
       To create a ZFS pool that is compatible with all operating system versions in the matrix above, use the
       <a href="https://www.freebsd.org/cgi/man.cgi?query=zpool&amp;apropos=0&amp;sektion=8&amp;manpath=FreeBSD+10.1-stable&amp;arch=default&amp;format=html" onclick="_gaq.push(['_trackPageview', '/extlink/www.freebsd.org/cgi/man.cgi?query=zpool&amp;apropos=0&amp;sektion=8&amp;manpath=FreeBSD+10.1-stable&amp;arch=default&amp;format=html']);">
        <b>
         zpool(8)
        </b>
       </a>
       command with following syntax:
       <br/>
       <b>
        zpool create -d -o feature@async_destroy=enabled -o feature@empty_bpobj=enabled -o feature@lz4_compress=enabled [device(s)]
       </b>
       <br/>
       <br/>
       More information:
       <br/>
       <a href="https://www.freebsd.org/cgi/man.cgi?query=zpool-features&amp;apropos=0&amp;sektion=7&amp;manpath=FreeBSD+10.1-stable&amp;arch=default&amp;format=html" onclick="_gaq.push(['_trackPageview', '/extlink/www.freebsd.org/cgi/man.cgi?query=zpool-features&amp;apropos=0&amp;sektion=7&amp;manpath=FreeBSD+10.1-stable&amp;arch=default&amp;format=html']);">
        zpool-features(7)
       </a>
       manual page with detailed description of all supported features.
