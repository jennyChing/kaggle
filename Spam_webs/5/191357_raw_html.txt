     <h2>
      <span class="mw-headline" id="Goal_.26_description">
       Goal &amp; description
      </span>
     </h2>
     <p>
      This wiki is a translation of the Dutch original. All content applies to the Netherlands and may differ from other countries. 
The goal of this
      <a class="external text" href="http://mijndev.openstreetmap.nl/~ligfietser/fiets/index.html" rel="nofollow">
       web map
      </a>
      with  bicycle tags is rendering bicycle related tags that are not (properly) rendered on OSMs  'Mapnik'  map or on the cyclemap version.
     </p>
     <p>
      The map contains the following layers
     </p>
     <dl>
      <dt>
       Cycleway tags
      </dt>
      <dd>
       Most commonly used bicycle related properties of (cycle) ways
      </dd>
      <dt>
       Surface tags
      </dt>
      <dd>
       A description of the highway surface
      </dd>
      <dt>
       Route tags
      </dt>
      <dd>
       Overview of cycle route relations and (non) cycleable way (subjective)
      </dd>
      <dt>
       Bugs in tags
      </dt>
      <dd>
       Possible bugs on the map.
      </dd>
     </dl>
     <p>
      Please see the  OSM forum
      <a class="external text" href="http://forum.openstreetmap.org/viewtopic.php?id=18951&amp;p=1" rel="nofollow">
       topic
      </a>
      about  this map (Dutch)
     </p>

     <h2>
      <span class="mw-headline" id="Popup_window">
       Popup window
      </span>
     </h2>
     <p>
      By clicking on the map more detailed info about the highway is retrieved from OSM.
      <br/>
      There are also shortcuts to OSM.org, Google Maps, Bing Aerial as well as Openstreetmap Notes (bug report) and Keepright for error checks.
      <br/>
      The area can also be loaded in your favourite OSM editor JOSM, Potlatch or IDeditor.
     </p>
     <h2>
      <span class="mw-headline" id="Browser_problems">
       Browser problems
      </span>
     </h2>
     <p>
      The map works fine using Firefox or Chrome, but Internet Explorer (IE) can have issues.
     </p>
     <p>
      Possible solution:
     </p>
     <ul>
      <li>
       change IE security level. Add the domain mijndev.openstreetmap.nl to your trusted sites, set security to low.
      </li>
     </ul>
