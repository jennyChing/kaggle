         <h1 class="title entry-title">
          Sleeping your way to Better Sex
         </h1>
        </header>
        <div class="post-meta">
         <span class="small">
          By
         </span>
         <span class="author vcard">
          <span class="fn">
           <a href="http://cikipedia.com/author/ccadmin" rel="author" title="Posts by ciki">
            ciki
           </a>
          </span>
         </span>
         <span class="small">
          on
         </span>
         <abbr class="date time published updated" title="2015-03-19T10:43:27+0800">
          March 19, 2015
         </abbr>
         <span class="small">
          in
         </span>
         <span class="categories">
          <a href="http://cikipedia.com/category/dailys" title="View all items in Dailys">
           Dailys
          </a>
          ,
          <a href="http://cikipedia.com/category/general-health" title="View all items in General Health">
           General Health
          </a>
         </span>
        </div>
        <section class="entry">
         <p>
          Word is out.
         </p>
         <p>
          The more you sleep, the better the sex.
         </p>
         <p>
          <a href="https://www.flickr.com/photos/cumidanciki/16858459571" title="IMG_20150220_081102-001 by @agentcikay, on Flickr">
           <img alt="IMG_20150220_081102-001" class=" aligncenter" height="640" src="https://farm8.staticflickr.com/7606/16858459571_898a5e109b_z.jpg" width="407"/>
          </a>
         </p>
         <p>
          And this is not an arbitrary statement.. a study called the *The Impact of Sleep on Female Sexual Response and Behavior; Kalmbach et.al. published in the JournalThe Journal of Sexual Medicine (16 MAR 2015) documentstheir findings.
         </p>
         <p>
          This study looked at therole of sleep in 171 women, and how it correlated to sexual desire.
         </p>
         <p>
          Research so far has shown that short sleep duration and poor sleep quality lead to poor female sexual response. However, prior research consists of cross-sectional studies, and the influence of sleep on sexual functioning and behavior has not been prospectively examined.
         </p>
         <p>
          This study examined the influence of nightly sleep duration, sleep quality, and sleep onset latency on daily female sexual response and activity.
         </p>
         <p>
          So,171 women free of antidepressants were tested, and the findings were amazing!
         </p>
         <h4>
          <em>
           In women, with every 1 hour extra you sleep = 14% increase in sexual drive!
          </em>
         </h4>
         <p>
          So as a huge advocate of sleep (and there are other diseases which sleep can cure), trust me when I say, thatsleep is key and we really should get more of it. And now, great sex is yet another excuse to sleep more!
         </p>
         <p>
          Heres a post on
          <a href="http://cikipedia.com/best-foods-for-deep-sleep" target="_blank">
           foods that aid in good sleep
          </a>
          and also
          <a href="http://cikipedia.com/sleep-helps-reduce-obesity" target="_blank">
           how sleep can reduce obesity
          </a>
          .
         </p>
         <p>
          Results of the study for those who are interested in statistics and calculations : Thelonger the sleep duration was related to greater next-day sexual desire (b=0.32, P=0.02), and that a 1-hour increase in sleep length corresponded to a 14% increase in odds of engaging in partnered sexual activity (odds ratio=1.14, P&lt;0.05). In contrast, sleeping longer predicted poorer next-day genital arousal (b=0.19, P&lt;0.01). However, results showed that women with longer average sleep duration reported better genital arousal than women with shorter average sleep length (b=0.54, P=0.03).
         </p>
         <p>
          <em>
           *The Impact of Sleep on Female Sexual Response and Behavior: A Pilot Study
          </em>
          <br/>
          <em>
           David A. Kalmbach PhD1,*, J. Todd Arnedt PhD1, Vivek Pillai PhD2 andJeffrey A. Ciesla PhD3
          </em>
         </p>

function mce_success_cb(resp){
    $('#mce-success-response').hide();
    $('#mce-error-response').hide();
    if (resp.result=="success"){
        $('#mce-'+resp.result+'-response').show();
        $('#mce-'+resp.result+'-response').html(resp.msg);
        $('#mc-embedded-subscribe-form').each(function(){
            this.reset();
    	});
    } else {
        var index = -1;
        var msg;
        try {
            var parts = resp.msg.split(' - ',2);
            if (parts[1]==undefined){
                msg = resp.msg;
            } else {
                i = parseInt(parts[0]);
                if (i.toString() == parts[0]){
                    index = parts[0];
                    msg = parts[1];
                } else {
                    index = -1;
                    msg = resp.msg;
                }
            }
        } catch(e){
            index = -1;
            msg = resp.msg;
        }
        try{
            if (index== -1){
                $('#mce-'+resp.result+'-response').show();
                $('#mce-'+resp.result+'-response').html(msg);            
            } else {
                err_id = 'mce_tmp_error_msg';
                html = '&lt;div id="'+err_id+'" style="'+err_style+'"&gt; '+msg+'&lt;/div&gt;';
                
                var input_id = '#mc_embed_signup';

                    f = $(input_id).parent().parent().get(0);
                } else if (ftypes[index]=='date'){
                    input_id = '#mce-'+fnames[index]+'-month';
                    f = $(input_id).parent().parent().get(0);
                } else {
                    input_id = '#mce-'+fnames[index];
                    f = $().parent(input_id).get(0);
                }
                if (f){
                    $(f).append(html);
                    $(input_id).focus();
                } else {
                    $('#mce-'+resp.result+'-response').show();
                    $('#mce-'+resp.result+'-response').html(msg);
                }
            }
        } catch(e){
            $('#mce-'+resp.result+'-response').show();
            $('#mce-'+resp.result+'-response').html(msg);
        }
    }
}
