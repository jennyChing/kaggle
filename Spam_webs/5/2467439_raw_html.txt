                       <h1 class="articleTitle">
                        A Guide to Cryptography in PHP
                       </h1>
                       <h2 class="articleDek">
                        This comprehensive guide discusses where to get, how to install, and how to use the various cryptography packages available to enhance the security of your PHP applications.

                n an ideal world, words like cryptography and security wouldn't even exist, but the real world is far from perfect, so software developers have to spend a good deal of time building security into applications. Cryptography is just one piece of the security puzzle, along with SSL/TLS, certificates, digital signatures, and so on. This article explains how to use PHP to implement the most common cryptographic algorithms. In addition to describing PHP's default encryption functions, you'll see how to use a wide variety of cryptographic libraries and packages.
                <br/>
                <br/>
                The code examples in this article use the contents of a short text file,
                <span class="pf">
                 textfile.txt
                </span>
                , which contains the following plain-text content:
                <br/>
                <br/>
                <pre>
                 <p>
                  <code>
                   For every difficult and complicated 
   question there is an answer 
   that is simple, easily understood, 
   and wrong. H.L. Mencken
                  </code>
                 </p>
                </pre>
                <strong>
                 Default PHP Encryption Functions
                </strong>
                <br/>
                PHP ships with three built-in encryption functions:
                <span class="pf">
                 md5()
                </span>
                ,
                <span class="pf">
                 crypt()
                </span>
                , and
                <span class="pf">
                 sha1()
                </span>
                . The
                <span class="pf">
                 md5()
                </span>
                function prototype is:
                <br/>
                <br/>
                <pre>
                 <p>
                  <code>
                   string md5(string $str [, bool $raw_output ])
                  </code>
                 </p>
                </pre>
                The function calculates the MD5 hash of a supplied string using the MD5 Message-Digest algorithm. The
                <span class="pf">
                 $str
                </span>
                argument represents the string to be encrypted. If you pass
                <span class="pf">
                 FALSE
                </span>
                in the
                <span class="pf">
                 $raw_output
                </span>
                argument (the default), the function returns the hash as a 32-character hexadecimal number. If you pass
                <span class="pf">
                 TRUE
                </span>
                then the function returns a 16-byte raw binary value.
                <br/>
                <br/>
                The PHP
                <span class="pf">
                 crypt()
                </span>
                function is a one-way encryption function that lets you confirm that an entered password matches a stored encrypted onewithout having to decrypt anything. The
                <span class="pf">
                 crypt()
                </span>
                function prototype is:
                <br/>
                <br/>
                <pre>
                 <div align="center" style="margin-left:25px;">
                  <script language="javascript" src="http://www.devx.com/sl/assetlisting/?wsn=www.devx.com&amp;wpu=/webdev/Article/37821&amp;lurl=http://o1.qnsr.com/cgi/r?WT.qs_dlk=VaVeXArIZ2IAACK3fxkAAAAD;;;n=203;c=1395300;s=9514;x=7936;f=201308291431480;u=j;z=TIMESTAMP;k=http://www.devx.com/ebooks&amp;wpos=onelineassetlisting&amp;lyt=l6&amp;css=http://www.devx.com/css/onePaidAsset.css&amp;wi=19737010">
                  </script>
                 </div>
                 <br/>
                 <br/>
                 <!-- div style="clear:both;"&gt;&lt;/div -->
                 <p>
                  <code>
                   string crypt (string $str [, string $salt ])
                  </code>
                 </p>
                </pre>
                It returns an encrypted string using the standard Unix DES-based encryption algorithm (or alternative algorithms that may be available on the system). The
                <span class="pf">
                 $str
                </span>
                argument is the string to be encrypted and the optional
                <span class="pf">
                 $salt
                </span>
                argument is a string on which to base the encryption. If you don't provide the salt string, PHP will randomly generate one each time you call this function.
                <br/>
                <br/>
                The PHP
                <span class="pf">
                 sha1()
                </span>
                function  calculates the SHA-1 hash of a string. The
                <span class="pf">
                 sha1()
                </span>
                function prototype is:
                <br/>
                <br/>
                <pre>
                 <p>
                  <code>
                   string sha1 (string $str [, bool $raw_output ])
                  </code>
                 </p>
                </pre>
                <table align="right" border="0" cellpadding="5" cellspacing="0" width="175">
                 <tr>
                  <td valign="top">
                   <a href="JavaScript:showSupportItem('figure1');">
                    <img alt="" border="0" height="40" src="http://assets.devx.com/articlefigs/image002.jpg" width="175"/>
                   </a>
                  </td>
                  <td width="12">
                  </td>
                 </tr>
                 <tr>
                  <td class="smallfont">
                   <a href="JavaScript:showSupportItem('figure1');">
                    Figure 1
                   </a>
                   .
                   <strong>
                    Encrypted File
                   </strong>
                   : The encrypted.txt file contains a password encrypted with md5, crypt, and sha1 PHP default functions.
                  </td>
                 </tr>
                </table>
                The function returns the SHA-1 hash as a string. Again, the
                <span class="pf">
                 $str
                </span>
                argument represents the input string. If you set the optional
                <span class="pf">
                 $raw_output
                </span>
                argument to
                <span class="pf">
                 TRUE
                </span>
                , the function returns the sha1 hash in raw binary format with a length of 20 characters; if you set it to
                <span class="pf">
                 FALSE
                </span>
                , it returns a 40-character hexadecimal number.
                <br/>
                <br/>
                As an example, the following code shows how to use the PHP default encryption functions to encrypt the contents of
                <span class="pf">
                 texfile.txt
                </span>
                file and write the encrypted result in the file
                <span class="pf">
                 encrypted.txt
                </span>
                (see
                <a href="JavaScript:showSupportItem('figure1');">
                 Figure 1
                </a>
                ):
                <br/>
                <br/>
                <pre>
                 <p>
                  <code>
                   &lt;?php
    
   $file = 'textfile.txt';
   $initial_contents = file_get_contents($file);  
   
   if($initial_contents){
   
      $password = 'OctaviaAnghel';
   
      //Calculates the md5 hash 
      $md5_data = md5($password);
   
      //This function encrypts data
      $crypt = crypt($password);
   
      //Calculate the sha1 hash
      $sha1 = sha1($password);
   
      $encrypted_file = @fopen('encrypted.txt','w');
      $ok_encrypt = @fwrite($encrypted_file,'md5: '. $md5_data."\r\n".'crypt: 
          '.$crypt."\r\n".'sha1: '.$sha1);
   
      if($ok_encrypt){
         echo 'The encrypted code was succesfully created'.
         ' in encrypted_file.txt!!!'.'
                   <br/>
                   ';
      }
      else{
         echo ("The write of this file failed!");
      }
      
      @fclose($encrypted_file);
   } 
   ?&gt;
                  </code>
                 </p>
                </pre>
                In addition to the built-in functions, PHP supports encryption via external libraries and packages. Table 1 shows the libraries and packages described in the rest of this article.
                <br/>
                <br/>
                <div style="width:600px">
                 <span class="smallfont">
                  <strong>
                   Table 1
                  </strong>
                  .
                  <strong>
                   Cryptography in PHP
                  </strong>
                  : The table contains a list of packages and libraries described in this article that work with PHP to perform various types of encryption and decryption.
                 </span>
                </div>
                <table border="1" bordercolordark="white" bordercolorlight="black" cellspacing="0" width="615">
                 <tr>
                  <td align="left" bgcolor="#99CCFF" valign="top" width="168">
                   <strong>
                    Package/Library
                   </strong>

                   <strong>

                 </tr>
                 <tr>
                  <td align="left" bgcolor="white" valign="top" width="168">
                   MCrypt
                  </td>
                  <td align="left" bgcolor="white" valign="top" width="437">
                   Use MCrypt to encrypt large files or data streams using any of a wide range of encryption functions. You can find more information at
                   <a href="http://mcrypt.sourceforge.net/" target="_blank">
                    http://mcrypt.sourceforge.net/
                   </a>
                   .
                  </td>
                 </tr>
                 <tr>
                  <td align="left" bgcolor="white" valign="top" width="168">
                   MHash
                  </td>
                  <td align="left" bgcolor="white" valign="top" width="437">
                   Use the MHash library to obtain hashes. MHash supports the most popular algorithms and  implementations, such as SHA, MD5,  and CRC. You can use these algorithms to compute checksums, message digests, and create other signatures. MHash is often used to obtain password hashes for passwords entered into HTML password fields. You can find more information at
                   <a href="http://mhash.sourceforge.net/" target="_blank">
                    http://mhash.sourceforge.net/
                   </a>
                   .
                  </td>
                 </tr>
                 <tr>
                  <td align="left" bgcolor="white" valign="top" width="168">

                  </td>
                  <td align="left" bgcolor="white" valign="top" width="437">
                   Use Crypt_Blowfish for quick two-way encryption both with or without a secret key. You don't need the MCrypt PHP extension to use Crypt_Blowfish; however, the package can use MCrypt if it's installed. More details at:
                   <a href="http://pear.php.net/package/Crypt_Blowfish" target="_blank">
                    http://pear.php.net/package/Crypt_Blowfish
                   </a>
                   .
                  </td>
                 </tr>
                 <tr>
                  <td align="left" bgcolor="white" valign="top" width="168">

                  </td>
                  <td align="left" bgcolor="white" valign="top" width="437">
                   Crypt_RSA provides RSA-like key generation, encryption/decryption, signing and signature checking. More details here:
                   <a href="http://pear.php.net/package/Crypt_RSA" target="_blank">
                    http://pear.php.net/package/Crypt_RSA
                   </a>
                   .
                  </td>
                 </tr>
                 <tr>
                  <td align="left" bgcolor="white" valign="top" width="168">
                   Crypt_ HMAC
                  </td>
                  <td align="left" bgcolor="white" valign="top" width="437">
                   This class calculates RFC 2104-compliant hashes. You'll find complete information at:
                   <a href="http://pear.php.net/package/Crypt_HMAC" target="_blank">
                    http://pear.php.net/package/Crypt_HMAC
                   </a>
                   .
                  </td>
                 </tr>
                 <tr>
                  <td align="left" bgcolor="white" valign="top" width="168">
                   Crypt_DiffieHellman
                  </td>
                  <td align="left" bgcolor="white" valign="top" width="437">
                   This package is a PHP5 implementation of the Diffie-Hellman Key Exchange cryptographic protocol. You can find more information at
                   <a href="http://pear.php.net/package/Crypt_DiffieHellman" target="_blank">
                    http://pear.php.net/package/Crypt_DiffieHellman
                   </a>
                   .
