                &lt;section id="why-use-phpdocumentor" class="tab-pane active"&gt;
                    &lt;section&gt;
                        &lt;h2&gt;What is phpDocumentor?&lt;/h2&gt;
                        &lt;img
                            src="/bundles/phpdocumentorwebsite/images/icons/icon_logo_green.png"
                            alt="phpDocumentor"
                        /&gt;

                        &lt;p&gt;
                            phpDocumentor 2 is a tool that makes it possible to generate documentation directly from your PHP
                            source code. With this you can provide your consumers with more information regarding the
                            functionality embedded within your source and not just what is usable to them from your user
                            interface.
                        &lt;/p&gt;

                        &lt;p&gt;
                            Documentation generated by phpDocumentor 2 does not aim to be a replacement for conventional
                            documentation but is rather supplemental, or reference, documentation.This proves to be
                            useful in the following example situations:
                        &lt;/p&gt;
                        &lt;ul&gt;
                            &lt;li&gt;Sets of libraries or applications providing an API, such as phpDocumentor 2 itself&lt;/li&gt;
                            &lt;li&gt;Frameworks, such as Zend Framework or Symfony&lt;/li&gt;
                            &lt;li&gt;Pluggable architectures, such as WordPress or PyroCMS&lt;/li&gt;
                            &lt;li&gt;Long-running, complex projects, to help you find the right function or method for the job&lt;/li&gt;
                        &lt;/ul&gt;
                    &lt;/section&gt;
                    &lt;section&gt;
                        &lt;h2&gt;Templates&lt;/h2&gt;
                        &lt;img
                                src="/bundles/phpdocumentorwebsite/images/icons/icon_templates_green.png"
                                alt="Templates"
                                /&gt;

                        &lt;p&gt;
                            phpDocumentor 2 contains an incredibly flexible template system which enables you to alter
                            your output in any way imaginable. This can range from simply applying your own branding
                            for the HTML output by merely altering a couple of CSS files to determining which types of
                            output are generated and where they are made available.
                        &lt;/p&gt;
                    &lt;/section&gt;
                    &lt;section&gt;
                        &lt;h2&gt;Code Analysis&lt;/h2&gt;
                        &lt;img
                                src="/bundles/phpdocumentorwebsite/images/icons/icon_charts_green.png"
                                alt="Code Analysis"
                                /&gt;

                        &lt;p&gt;
                            phpDocumentor 2 is capable of extracting interesting information and presenting it in the
                            form of graphs and reports. The following are currently supported:
                        &lt;/p&gt;
                        &lt;ul&gt;
                            &lt;li&gt;An inheritance diagram showing all subclassing and implementing of interfaces.&lt;/li&gt;
                            &lt;li&gt;Reporting errors in your source code's inline documentation.&lt;/li&gt;
                            &lt;li&gt;Reporting which elements are marked as deprecated.&lt;/li&gt;
                            &lt;li&gt;Reporting where TODOs are placed in your code and what is to be done.&lt;/li&gt;
                        &lt;/ul&gt;
                        &lt;p&gt;
                            This list will only grow in the future to help you determine where to invest your resources
                            in order to make your code more stable and maintainable.
                        &lt;/p&gt;
                    &lt;/section&gt;

        &lt;section class="span4" id="quickstart"&gt;
            &lt;article id="installing"&gt;
                &lt;h2&gt;Installing&lt;/h2&gt;
                &lt;div class="accordion" id="installation"&gt;
                    &lt;div class="accordion-group"&gt;
                        &lt;div class="accordion-heading"&gt;
                            &lt;a class="accordion-toggle" data-toggle="collapse" data-parent="#installation" href="#install-pear"&gt;
                                &lt;i class="icon-chevron-right"&gt;&lt;/i&gt; Using PEAR
                            &lt;/a&gt;
                        &lt;/div&gt;
                        &lt;div id="install-pear" class="accordion-body collapse in"&gt;
                            &lt;div class="accordion-inner"&gt;
                                &lt;p&gt;With &lt;a href=""&gt;PEAR&lt;/a&gt; installed, please run the following commands:&lt;/p&gt;
                                &lt;pre&gt;&lt;code class="language-php"&gt;pear channel-discover pear.phpdoc.org
pear install phpdoc/phpDocumentor&lt;/code&gt;&lt;/pre&gt;
                                &lt;p&gt;
                                    Once the installation is complete a global command, called &lt;code&gt;phpdoc&lt;/code&gt;, will be
                                    available.
                                &lt;/p&gt;
                            &lt;/div&gt;
                        &lt;/div&gt;
                    &lt;/div&gt;
                    &lt;div class="accordion-group"&gt;
                        &lt;div class="accordion-heading"&gt;
                            &lt;a class="accordion-toggle" data-toggle="collapse" data-parent="#installation" href="#install-composer"&gt;
                                &lt;i class="icon-chevron-right"&gt;&lt;/i&gt; Using Composer
                            &lt;/a&gt;
                        &lt;/div&gt;
                        &lt;div id="install-composer" class="accordion-body collapse"&gt;
                            &lt;div class="accordion-inner"&gt;
                                &lt;p&gt;
                                    Add a dependency to
                                    &lt;a href="https://packagist.org/packages/phpdocumentor/phpdocumentor"&gt;phpdocumentor/phpdocumentor&lt;/a&gt;
                                    to the &lt;code&gt;require-dev&lt;/code&gt; section of your project's &lt;code&gt;composer.json&lt;/code&gt;
                                    configuration file, and update your application.
                                &lt;/p&gt;

                                &lt;p&gt;
                                    The following code is an example of a minimal &lt;code&gt;composer.json&lt;/code&gt; file:
                                &lt;/p&gt;
                                &lt;pre&gt;&lt;code class="language-javascript"&gt;{
    "require-dev": {
        "phpdocumentor/phpdocumentor": "2.*"
    }
}&lt;/code&gt;&lt;/pre&gt;
                                &lt;p&gt;
                                    Once the installation is complete there will be a command &lt;code&gt;phpdoc.php&lt;/code&gt; in
                                    the binary folder of your application (&lt;code&gt;/vendor/bin&lt;/code&gt; by default).
                                &lt;/p&gt;
                                &lt;p class="alert alert-info"&gt;
                                    &lt;strong&gt;Hint&lt;/strong&gt;: phpDocumentor consumes a fair amount of dependencies, if this is inconvenient
                                    then we suggest you choose one of the other installation options.
                                &lt;/p&gt;
                            &lt;/div&gt;
                        &lt;/div&gt;
                    &lt;/div&gt;
                    &lt;div class="accordion-group"&gt;
                        &lt;div class="accordion-heading"&gt;
                            &lt;a class="accordion-toggle" data-toggle="collapse" data-parent="#installation" href="#install-phar"&gt;
                                &lt;i class="icon-chevron-right"&gt;&lt;/i&gt; As a PHAR
                            &lt;/a&gt;
                        &lt;/div&gt;
                        &lt;div id="install-phar" class="accordion-body collapse"&gt;
                            &lt;div class="accordion-inner"&gt;
                                &lt;p&gt;
                                    &lt;a href="http://phpdoc.org/phpDocumentor.phar"&gt;
                                        All you need to do is download the phar binary.
                                    &lt;/a&gt;
                                &lt;/p&gt;
                                &lt;p&gt;That's it. Really.&lt;/p&gt;
                            &lt;/div&gt;
                        &lt;/div&gt;
                    &lt;/div&gt;
                    &lt;div class="accordion-group"&gt;
                        &lt;div class="accordion-heading"&gt;
                            &lt;a class="accordion-toggle" data-toggle="collapse" data-parent="#installation" href="#install-manual"&gt;
                                &lt;i class="icon-chevron-right"&gt;&lt;/i&gt; Manually
                            &lt;/a&gt;
                        &lt;/div&gt;
                        &lt;div id="install-manual" class="accordion-body collapse"&gt;
                            &lt;div class="accordion-inner"&gt;
                                &lt;p&gt;To do a manual installation, follow these steps&lt;/p&gt;
                                &lt;ol&gt;
                                    &lt;li&gt;
                                        Locate the latest release on the
                                        &lt;a href="https://github.com/phpDocumentor/phpDocumentor2/releases"&gt;releases page for phpDocumentor2 on GitHub&lt;/a&gt;
                                    &lt;/li&gt;
                                    &lt;li&gt;Download the attached tarball (not the source code!)&lt;/li&gt;
                                    &lt;li&gt;Extract the downloaded tarball into your destination directory.&lt;/li&gt;
                                &lt;/ol&gt;
                                &lt;p&gt;
                                    After you have done the above can the executable be found at
                                    &lt;code&gt;bin/phpdoc.php&lt;/code&gt; in the installation directory.
                                &lt;/p&gt;
                            &lt;/div&gt;
                        &lt;/div&gt;
