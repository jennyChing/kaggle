                    <h3 class="post-title entry-title" itemprop="name">
                     Chunky Willow Tree Slouchy Beanie Crochet Pattern
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-7589924615283565355" itemprop="articleBody">
                     Raise your hand if you know a 9 year-old who cried when her favourite climbing tree got chopped down? -raises hand-
                     <br/>
                     Now before you jump to any conclusions, I am in fact NOT the child mentioned above. That honour has been reserved for my lovable baby sister... As you can tell by my '
                     <a href="http://www.hopefulhoney.com/p/about-me.html" target="_blank">
                      About Me
                     </a>
                     ' page, she definitely takes after her sister in the tree climbing department!
                     <br/>
                     You see, while dividing up our property to sell, we unfortunately had to have the tree removed which invoked a few small sobs on her part.
                     <br/>
                     <br/>
                     However, once her short-lived friendship with Little Miss Tree was over, she made it her duty in life to convince us to plant a tree at our new house. And not just any tree... A willow tree!
                     <br/>
                     <br/>
                     We're currently still on the fence about the whole matter, but none the less I was feeling inspired and decided to create this beanie below!
                     <br/>
                     What was you favourite place to spend as a child?
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://3.bp.blogspot.com/-nZ3FO1qF4co/U7EM6x9JjDI/AAAAAAAALYg/CO8s-Id9jBE/s1600/Chunky+Willow+Tree+Slouchy+Beanie+Crochet+Pattern+6.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                       <img alt="" border="0" height="640" src="http://3.bp.blogspot.com/-nZ3FO1qF4co/U7EM6x9JjDI/AAAAAAAALYg/CO8s-Id9jBE/s1600/Chunky+Willow+Tree+Slouchy+Beanie+Crochet+Pattern+6.jpg" title="Chunky Willow Tree Slouchy Beanie Crochet Pattern" width="406"/>
                      </a>
                     </div>

                      <a data-pin-color="white" data-pin-config="beside" data-pin-do="buttonPin" href="https://www.pinterest.com/pin/create/button/?url=http%3A%2F%2Fwww.hopefulhoney.com%2F2014%2F07%2Fchunky-willow-tree-slouchy-beanie.html&amp;media=http%3A%2F%2F3.bp.blogspot.com%2F-nZ3FO1qF4co%2FU7EM6x9JjDI%2FAAAAAAAALYg%2FCO8s-Id9jBE%2Fs1600%2FChunky%2BWillow%2BTree%2BSlouchy%2BBeanie%2BCrochet%2BPattern%2B6.jpg&amp;description=Chunky%20Willow%20Tree%20Slouchy%20Beanie%20Crochet%20Pattern%20">

                     <div style="text-align: center;">
                      <br/>
                      <b>
                       What You'll Need
                      </b>
                     </div>
                     <div style="text-align: center;">
                      12.00mm Crochet Hook
                     </div>
                     <div style="text-align: center;">
                      <span class="userContent">

                      {Easily adjustable, as explained in pattern}
                      <br/>
                      <br/>
                      <div class="separator" style="clear: both; text-align: center;">
                       <a href="http://4.bp.blogspot.com/-6FDxgnRYagM/U7EXXhWj5ZI/AAAAAAAALYw/xA9Msrh9dZk/s1600/Chunky+Willow+Tree+Slouchy+Beanie+Crochet+Pattern+8.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">

                     </div>
                     <div class="separator" style="clear: both; text-align: center;">
                     </div>
                     <br/>
                     Taking your 12.00mm crochet hook and super bulky yarn, start of by making a
                     <a href="https://www.youtube.com/watch?v=H6jSPhWTWGo" target="_blank">
                      Slip Knot
                     </a>
                     , then crochet 23
                     <a href="https://www.youtube.com/watch?v=PV06HW97EfY" target="_blank">
                      Chains
                     </a>
                     .
                     <br/>
                     <b>
                      Row 1
                     </b>
                     ~
                     <a href="https://www.youtube.com/watch?v=PV06HW97EfY" target="_blank">
                      Chain
                     </a>
                     2. Crochet 1
                     <a href="https://www.youtube.com/watch?v=DhKyDgnDIdg" target="_blank">
                      Half Double Crochet
                     </a>
                     into the fourth chain from your hook. 
Then crochet 1
                     <a href="https://www.youtube.com/watch?v=DhKyDgnDIdg" target="_blank">
                      Half Double Crochet
                     </a>
                     into the next 17 chains. Crochet 1
                     <a href="https://www.youtube.com/watch?v=JPSM0bnmU8U" target="_blank">
                      Single Crochet
                     </a>
                     into the next 5 stitches. You should 
have 23 stitches now.
                     <br/>
                     <b>
                      Row 2
                     </b>
                     ~ Crochet 1
                     <a href="https://www.youtube.com/watch?v=XVURLgzfJsM" target="_blank">
                      Slip Stitch
                     </a>
                     into every stitch across.
                     <br/>
                     <b>
                      Row 3
                     </b>
                     ~
                     <a href="https://www.youtube.com/watch?v=PV06HW97EfY" target="_blank">
                      Chain
                     </a>
                     2. Crochet 1
                     <a href="https://www.youtube.com/watch?v=DhKyDgnDIdg" target="_blank">
                      Half Double Crochet
                     </a>
                     into the back loop of the next 17 stitches. Crochet 1
                     <a href="https://www.youtube.com/watch?v=JPSM0bnmU8U" target="_blank">
                      Single Crochet
                     </a>
                     into back loop of the next 5 stitches.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://2.bp.blogspot.com/-SID0-aUuTJE/U6-k-V_CeUI/AAAAAAAALXc/nGGRE4ltIjE/s1600/Chunky+Willow+Tree+Slouchy+Beanie+Crochet+Pattern.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                       <img alt="" border="0" height="426" src="http://2.bp.blogspot.com/-SID0-aUuTJE/U6-k-V_CeUI/AAAAAAAALXc/nGGRE4ltIjE/s1600/Chunky+Willow+Tree+Slouchy+Beanie+Crochet+Pattern.jpg" title="Chunky Willow Tree Slouchy Beanie Crochet Pattern" width="640"/>
                      </a>
                     </div>
                     <br/>
                     <b>
                      Row 4
                     </b>
                     ~ Crochet 1
                     <a href="https://www.youtube.com/watch?v=XVURLgzfJsM" target="_blank">
                      Slip Stitch
                     </a>
                     into every stitch across.
                     <br/>
                     <b>
                      Row 5 - 33
                     </b>
                     ~ Repeat rows 3 - 4. {
                     <i>
                      <b>
                       Note
                      </b>
                      : You can either add or take away a couple sets of rows 3 - 4, depending on your head's circumference.
                     </i>
                     }
                     <br/>
                     Fasten off, and weave in loose ends.
                     <br/>
                     <br/>
                     Now taking the yarn and tapestry needle, thread the yarn through the ends of the rows along the edge. {
                     <i>
                      <b>
                       Note:
                      </b>
                      Be sure to thread the yarn through the ends consisting of single crochets, not the ends with half double crochets.
                     </i>
                     }
                     <br/>
                     <br/>

                if (entry.gd$extendedProperty[k].name == 'blogger.itemClass') {
                  comment.deleteclass += ' ' + entry.gd$extendedProperty[k].value;
                } else if (entry.gd$extendedProperty[k].name == 'blogger.displayTime') {
                  comment.displayTime = entry.gd$extendedProperty[k].value;
                }
              }
            }
            comments.push(comment);
          }
        }
        return comments;
      };

      var paginator = function(callback) {
        if (hasMore()) {
          var url = config.feed + '?alt=json&amp;v=2&amp;orderby=published&amp;reverse=false&amp;max-results=50';
          if (cursor) {
            url += '&amp;published-min=' + new Date(cursor).toISOString();
          }
          window.bloggercomments = function(data) {
            var parsed = parse(data);
            cursor = parsed.length &lt; 50 ? null
                : parseInt(parsed[parsed.length - 1].timestamp) + 1
            callback(parsed);
            window.bloggercomments = null;
          }
          url += '&amp;callback=bloggercomments';
          var script = document.createElement('script');
          script.type = 'text/javascript';
          script.src = url;
          document.getElementsByTagName('head')[0].appendChild(script);
        }
      };
      var hasMore = function() {
        return !!cursor;
      };
      var getMeta = function(key, comment) {
        if ('iswriter' == key) {
          var matches = !!comment.author
              &amp;&amp; comment.author.name == config.authorName
              &amp;&amp; comment.author.profileUrl == config.authorUrl;
          return matches ? 'true' : '';
        } else if ('deletelink' == key) {
          return config.baseUri + '/delete-comment.g?blogID='
               + config.blogId + '&amp;postID=' + comment.id;
        } else if ('deleteclass' == key) {
          return comment.deleteclass;
        }
        return '';
      };

      var replybox = null;
      var replyUrlParts = null;
      var replyParent = undefined;

      var onReply = function(commentId, domId) {
        if (replybox == null) {
          // lazily cache replybox, and adjust to suit this style:
          replybox = document.getElementById('comment-editor');
          if (replybox != null) {
            replybox.height = '250px';
            replybox.style.display = 'block';
            replyUrlParts = replybox.src.split('#');
          }
        }
        if (replybox &amp;&amp; (commentId !== replyParent)) {
          document.getElementById(domId).insertBefore(replybox, null);
          replybox.src = replyUrlParts[0]
              + (commentId ? '&amp;parentID=' + commentId : '')
              + '#' + replyUrlParts[1];
          replyParent = commentId;
        }
      };

      var hash = (window.location.hash || '#').substring(1);
      var startThread, targetComment;
      if (/^comment-form_/.test(hash)) {
        startThread = hash.substring('comment-form_'.length);
      } else if (/^c[0-9]+$/.test(hash)) {
        targetComment = hash.substring(1);
      }

      // Configure commenting API:
      var configJso = {
        'maxDepth': config.maxThreadDepth
      };
      var provider = {
        'id': config.postId,
        'data': items,
        'loadNext': paginator,
        'hasMore': hasMore,
        'getMeta': getMeta,
        'onReply': onReply,
        'rendered': true,
        'initComment': targetComment,
        'initReplyThread': startThread,
        'config': configJso,
        'messages': msgs
      };

      var render = function() {
        if (window.goog &amp;&amp; window.goog.comments) {
          var holder = document.getElementById('comment-holder');
          window.goog.comments.render(holder, provider);
        }
      };

      // render now, or queue to render when library loads:
      if (window.goog &amp;&amp; window.goog.comments) {
        render();
      } else {
        window.goog = window.goog || {};
        window.goog.comments = window.goog.comments || {};
        window.goog.comments.loadQueue = window.goog.comments.loadQueue || [];
        window.goog.comments.loadQueue.push(render);
      }
    })();
// ]]&gt;
                     </script>
