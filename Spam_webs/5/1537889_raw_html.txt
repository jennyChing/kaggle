      <a class="headlinelink-large" href="../News-Politics/11_Most_Painfully_Obvious_Newspaper_Articles_Ever">
       11 Most Painfully Obvious Newspaper Articles Ever
       <br/>
      </a>
      <span class="author">
       written by Sam Greenspan
       <br/>
      </span>
      <br/>
      <div class="p402_premium">
       <span class="entry">
        Every time I do a list about newspapers doing hilariously dumb things (like
        <a href="http://www.11points.com/News-Politics/11_Hilarious_(and_Very_Necessary)_Newspaper_Corrections">
         11 Hilarious (and Very Necessary) Newspaper Corrections
        </a>
        or
        <a href="http://www.11points.com/News-Politics/11_Great_Moments_in_Police_Blotter_History">
         11 Great Moments in Police Blotter History
        </a>
        ), I lament the fact that newspapers are on a crash course with extinction. Without the immediacy of the Internet, the disregard for ethics and proper reporting of the blogs, or the classifieds -- and prostitutes -- of Craigslist, the print media simply isn't viable in the modern world.
        <br/>
        <br/>
        And that's sad for many reasons... but today, I'm going to call it sad because it means that, one day, there will be no more printed stories like the ones below.
        <br/>
        <br/>
        So let's all channel our inner Jay Leno for a moment and enjoy these painfully, and hilariously, obvious newspaper articles. (You can also make your voice high-pitched and say, "So I hear President Clinton's in the newths again. Have you seen this? Have you heard about this? Yeah... watch out interns!")
        <ol>
         <li>
          <span class="exagger">
           Two different strategies for keeping warm.
          </span>
          The first of these strategies is very, very intuitive...
          <br/>
          <br/>
          <img src="../images/painfullyobvious/keepwarm-1.jpg"/>
          <br/>
          <br/>
          ...but the second one, thanks to a typo, takes the same basic advice to an entirely different place. (It also seems like it isn't very tactful advice to give to "older adults.")
          <br/>
          <br/>
          <img src="../images/painfullyobvious/keepwarm-2.jpg"/>
          <br/>
          <br/>
         </li>
         <li>
          <span class="exagger">
           OK, follow-up: So what makes a billionaire?
          </span>
          My favorite part is that they had to go to the senior vice president of investments at AG Edwards and Sons to get the answer to this question. They couldn't think of a simpler way to find what they were looking for?
          <br/>
          <br/>
          <img src="../images/painfullyobvious/millionaire.jpg"/>
          <br/>
          <br/>
         </li>
         <li>
          <span class="exagger">
           Book 'em Lou: One count of being a bear, one count of being an accessory to a bear.
          </span>
          I'd actually be curious to see if they could come up with a third theory.
          <br/>
          <br/>
          <img src="../images/painfullyobvious/twotheories.jpg"/>
          <br/>
          <br/>
         </li>
         <li>
          <span class="exagger">
           The big difference between sinus infections and jellyfish stings.
          </span>
          Sometimes I think people are just
          <i>
           looking
          </i>
          for reasons to get doused with urine.
          <br/>
          <br/>
          <img src="../images/painfullyobvious/urineears.jpg"/>
          <br/>
          <br/>
         </li>
         <li>
          <span class="exagger">
           Talk to me about pregnancy for a second.
          </span>
          I have four different ones on pregnancy... it just seems to be supple, supple ground for Yahoo Answers-level discourse.
          <br/>
          <br/>
          <img src="../images/painfullyobvious/teenpregnancy.jpg"/>
          <br/>
          <br/>
          <img src="../images/painfullyobvious/frequentsex.jpg"/>
          <br/>
          <br/>
          <img src="../images/painfullyobvious/weightgain.jpg"/>
          <br/>
          <br/>
          <img src="../images/painfullyobvious/population.jpg"/>
          <br/>
          <br/>
         </li>
         <li>
          <span class="exagger">
           Hey, it's something to build on.
          </span>
          I've re-read this one over and over, and I'm still not exactly sure why they chose to print that sentence. I just can't figure out what purpose it ever could've served.
          <br/>
          <br/>
          <img src="../images/painfullyobvious/crashoccurred.jpg"/>
          <br/>
          <br/>
         </li>
         <li>
          <span class="exagger">
           Guns don't kill people...
          </span>
          ... dying kills people. Apparently.
          <br/>
          <br/>
          <img src="../images/painfullyobvious/deathkiller.jpg"/>
          <br/>
          <br/>
         </li>
         <li>
          <span class="exagger">
           Women really are quite responsible.
          </span>
          It's funny, because guys are always diagnosing other guys with strained and/or sandy vaginas. With just a little more preventative care those could probably be avoided.
          <br/>
          <br/>
          <img src="../images/painfullyobvious/womensdiseases.jpg"/>
          <br/>
          <br/>
         </li>
         <li>
          <span class="exagger">
           Stop snitchin', right?
          </span>
          Ya know, I learned that dead men tell no tales on the Pirates of the Caribbean ride at Disneyland a long time ago.
          <br/>
          <br/>
          <img src="../images/painfullyobvious/homicidevictims.jpg"/>
          <br/>
          <br/>
         </li>
         <li>
          <span class="exagger">
           One of these wishes is not like the others.
          </span>
          I guess it really is the most obvious thing to wish for (other than a billion wishes).
          <br/>
          <br/>
          <img src="../images/painfullyobvious/parentsalive.jpg"/>
          <br/>
          <br/>
         </li>
         <li>
          <span class="exagger">
           Feels like it's out of The Onion.
          </span>
          It's like forming a committee to analyze if you've got too much bureaucracy. Or going to a brothel to get a second opinion on whether you're a sex addict.
          <br/>
          <br/>
          <img src="../images/painfullyobvious/reportonreports.jpg"/>
          <br/>
          <br/>
         </li>
        </ol>
       </span>
      </div>
