Businesses are typically monitoring social media for negative comments or praise, but they can't check every discussion thread everywhere on the internet. Diffbot's new API aims to help them keep track of comments on any site, too.

The idea is that using the new API, developerscan build custom apps to monitor discussions on any site (be it a specific site or a wider search) for mentions of abusiness, product or keyword and pick up whenever it's mentioned, along with other information like the sentiment of the post (if it's negative or positive), any links that were added and more.



Diffbot provides a number of APIs that help structure information found on the Web for use in applications; it allows apps to see the web in the way that humans understand it looking at the page.

Diffbot's discussion extraction API is available from today for developers to start integrating into their apps. It's easy to imagine it being quickly adopted considering how easy it makes it to scrape discussions.

       <div class="l-postSingle-content-main-content postContent" itemprop="articleBody">
        <p>
         Diffbots
         <a href="https://www.diffbot.com/dev/docs/discussion/">
          new discussions API
         </a>
         , announced today, makes it easy for developers to monitor comment threads almost everywhere on the Web from a single place.
        </p>
        <p>
         Businesses are typically monitoring social media for negative comments or praise, but they cant check every discussion thread everywhere on the internet. Diffbots new API aims to help them keep track of comments on any site, too.
        </p>
        <p>
         The idea is that using the new API, developerscan build custom apps to monitor discussions on any site (be it a specific site or a wider search) for mentions of abusiness, product or keyword and pick up whenever its mentioned, along with other information like the sentiment of the post (if its negative or positive), any links that were added and more.
        </p>
        <p>
         <a href="http://cdn1.tnwcdn.com/wp-content/blogs.dir/1/files/2015/03/discussion1-large.gif">
          <div class="postContent-image">
           <div class="postContent-image-wrap">
            <img alt="discussion1-large" class="alignnone size-full wp-image-848637" height="754" src="http://cdn1.tnwcdn.com/wp-content/blogs.dir/1/files/2015/03/discussion1-large.gif" width="971"/>
           </div>
          </div>
         </a>
        </p>
        <p>
         Diffbot provides a number of APIs that help structure information found on the Web for use in applications; it allows apps to see the web in the way that humans understand it looking at the page.
        </p>
        <p>
         Diffbots discussion extraction API is available from today for
         <a href="https://www.diffbot.com/dev/docs/discussion/">
          developers to start integrating
         </a>
         into their apps. Its easy to imagine it being quickly adopted considering how easy it makes it to scrape discussions.
        </p>
        <p>
         You can take the new APIfor a visual test drive
         <a href="http://www.diffbot.com/testdrive/?api=discussion&amp;url=https%3A%2F%2Fnews.ycombinator.com%2Fitem%3Fid%3D9213804">
          here
         </a>
         to see what kind of data it pulls out from sites and presents to you.
        </p>
        <p class="p1">
         
         <a href="https://www.diffbot.com/dev/docs/discussion/">
          Diffbot Discussion API
         </a>
        </p>
