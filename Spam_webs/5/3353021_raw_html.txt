     <h1>
      <a href="http://www.milienzo.com/wordpress-plugins/i-love-social-bookmarking/" rel="bookmark" title="Permanent Link: I Love Social Bookmarking plugin">
       I Love Social Bookmarking plugin
      </a>
     </h1>
     <p>
      <img alt="I Love Social Bookmarking" border="0" class="centered" height="240" src="http://www.milienzo.com/wp-content/uploads/2007/09/social_bmarking.jpg" title="I Love Social Bookmarking" width="480"/>
     </p>
     <p>
      <em>
       I Love Social Bookmarking
      </em>
      is a simple WordPress plugin that allows your readers to submit your content to social media services via a clutter-free drop-down list of attractive icons.
     </p>
     <h3>
      What are social media services?
     </h3>
     <p>
      If youve not heard of social media sites such as Digg or del.icio.us you should get to know them. They allow Internet users to save and rank web pages that they find interesting and share them with their friends and other users. When your content gets shared with others this can potentially generate huge levels of traffic for your blog. Just what you want, right?
     </p>
     <h3>
      How is
      <em>
       ILSB
      </em>
      different?
     </h3>
     <p>
      There are other social media plugins that do a similar job.
      <em>
       ILSB
      </em>
      aims to keep things as simple as possible for both you and your readers, to run with minimal system resources, and to look good whilst shes at it. The plugin features attractive icons provided by
      <a href="http://www.fasticon.com" onclick="javascript:pageTracker._trackPageview('/outgoing/www.fasticon.com');" title="FastIcon.com">
       FastIcon.com
      </a>
      , is easily customisable to fit your blogs theme, is XHTML standards-compliant and is SEO optimised by using the nofollow tag.
     </p>
     <h3>
      Which social media services are supported?
     </h3>
     <p>
      Currently supported: Blinklist, Blogmarks, Digg, del.icio.us, Furl, Ma.gnolia, NewsVine, Reddit, StumbleUpon, Technorati. More to come soon!
     </p>
     <h2>
      Download
     </h2>
     <ul>
      <li>
       <a href="http://www.milienzo.com/downloads/ilsb.zip" onclick="javascript:pageTracker._trackPageview('/outgoing/www.milienzo.com/downloads/ilsb.zip');" title="I Love Social Bookmarking">
        Download Share the Love 0.3.2
       </a>
       (Released 7 March 2008)
      </li>
     </ul>
     <p>
      If upgrading from a previous version, you may need to hit the Reset Defaults button to initialise all the new features and options.
     </p>
     <h2>
      Installation
     </h2>
     <ol>
      <li>
       Unzip the download file and upload the i-love-social-bookmarking folder into your WordPress plugins directory. (eg /wp-content/plugins/i-love-social-bookmarking/ )
      </li>
      <li>
       From the Plugins page of your WordPress administration panel click activate for
       <em>
        I Love Social Bookmarking
       </em>
       .
      </li>
     </ol>
     <h2>
      Administrator options
     </h2>
     <p>
      From your WordPress administration panel (under Options &gt; I Love Social Bookmarking) the following options are available:
     </p>
     <ul>
      <li>
       <strong>
        Automatic or manual display
       </strong>
        select whether to have
       <em>
        I Love Social Bookmarking
       </em>
       automatically inserted at the end of each post or whether to manually insert it into your template files. If manual use the following code:
       <pre>
        &lt;?php if (function_exists('ilsb')){ilsb();} ?&gt;
       </pre>
      </li>
      <li>
       <strong>
        Select WordPress pages
       </strong>
        Select which WordPress page types on which the plugin will automatically be placed.
      </li>
      <li>
       <strong>
        Select social media services
       </strong>
        Select which social media services you would like to include
      </li>
      <li>
       <strong>
        List title
       </strong>
        A string of text that will be used alongside the Love icon
      </li>
      <li>
       <strong>
        List style
       </strong>
        Horizontal or vertical
      </li>
      <li>
       <strong>
        Background colour
       </strong>
        The hexadecimal colour of the background of the list
      </li>
      <li>
       <strong>
        Link anchor
       </strong>
        Whether or not to display the name of the social media service
      </li>
      <li>
       <strong>
        Font
       </strong>
        The font face to be used
      </li>
      <li>
       <strong>
        Font size
       </strong>
        The font size to be used
      </li>
      <li>
       <strong>
        Link target
       </strong>
        Option to force the links to open in a new window
      </li>
     </ul>

      Awesome plug in! Thank you! I think I can give something back too
     </p>
     <p>
      For people having the CSS problems with UL from your main style sheet overwriting it on either Kubrick theme or a theme derived from Kubrick (or maybe others that use :before).
     </p>
     <p>
      Find this entry in your theme style.css -
     </p>
     <p>
      .entry ul li:before, #sidebar ul ul li:before {
      <br/>
      content: 0BB 020;
      <br/>
      }
      <br/>
      delete this entry, itll remove those (frankly hideous) &gt;&gt; bullets and ta da no more &gt;&gt; in the lovely drop down ILSN menu.
     </p>
     <p>
      Then youll need to find any entries that say
     </p>
     <p>
      list-style-type: none;
     </p>
     <p>
      and update them to normal bullets as you see fit.
     </p>
     <p>
      The :before could be in other stylesheets too.
     </p>
