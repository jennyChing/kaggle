       <header class="entry-header">
        <h1 class="entry-title">
         Enabling SMS subscriptions to your MailChimp list using Plivo, Part 1
        </h1>
        <div class="entry-meta">
         <a class="author" href="http://vincedasta.com/p/author/vdasta/" rel="author">
          Vince
         </a>
         /
         <a class="entry-date" href="http://vincedasta.com/p/enabling-sms-subscriptions-pt1/">
          January 23, 2014
         </a>
        </div>
       </header>
       <div class="entry-content">
        <p>
         I use MailChimp to power the mailing list on this site. Its a great service has been my go-to for years. One of the greatest features about MailChimp is its API which lets you integrate your list into all kinds of services. For my site, given the theme, I wanted to integrate the mailing list with a phone application. Over the next few posts Im going to show how I created my MailChimp phone application that lets someone subscribe to my list by texting me their email address as well as a few other cool features that are less practical, but still neat.
        </p>
        <p>
         I can think of several useful applications of this script so I decided to release it to anyone that want it. Next time youre out and about and meet someone that wants to subscribe to your list you can just tell them text me your email instead of, go to my website, click on newsletter, fill out the form, and then check your email for the confirmation.
        </p>
        <p>
         Much cooler and way more natural.
        </p>
        <h2>
         Getting started with Plivo
        </h2>
        <p>
         If youre not familiar with Plivo, its a service which allows you to interact with the phone system using a really powerful API. Similar to Twilio, but
         <a href="http://vincedasta.com/p/why-i-went-with-plivo/" title="Why I went with Plivo, not Twilio">
          much better in my opinion
         </a>
         . If you dont have an account you can get a free developer account without a credit card at
         <a href="http://www.plivo.com">
          http://www.plivo.com
         </a>
         .
        </p>
        <h2>
         Getting a MailChimp API Key
        </h2>
        <p>
         Youll also need a MailChimp API key and the numeric List ID for the list you want to subscribe your callers to. You can create an API key in MailChimp under My Account -&gt; Extras
        </p>
        <p>
         <img alt="resource" class="aligncenter size-full wp-image-115" height="388" src="http://vincedasta.com/wp-content/uploads/2014/01/resource1.jpg" width="700"/>
         The List ID is a little tougher to find, but if you navigate to the list and click the Settings dropdown you will find the List ID.
        </p>
        <p style="text-align: center;">
         <img alt="resource" class="aligncenter wp-image-116" height="293" src="http://vincedasta.com/wp-content/uploads/2014/01/resource2.jpg" width="645"/>
        </p>
        <h2>
         The Plivo app
        </h2>
        <p>
         I built the Plivo message app in PHP because its widespread enough for anyone to figure out. This app is simple enough, feel free to use any language you prefer, Ill use the official Plivo helper library and a
         <a href="https://github.com/drewm/mailchimp-api/" target="_blank">
          really simple MailChimp API
         </a>
         wrapper from
         <a href="http://allinthehead.com/" target="_blank">
          Drew McLellan.
         </a>
        </p>
        <h3>
         Anatomy of a Plivo SMS message
        </h3>
        <p>
         When Plivo receives an SMS it makes a simple POST request to your Message url, which we will define a little later when we setup our Plivo application and add our number, it expects the script to return an XML document with a
         <code>
          &lt;Response&gt;
         </code>
         element. Plivo has a really straight forward PHP library that well use to create the XML, so all we need to do is capture the data from the POST and do something with it. The full description is at
         <a href="https://plivo.com/docs/xml/request/" target="_blank">
          https://plivo.com/docs/xml/request/
         </a>
         If you want to understand the whole process.
        </p>
        <p>
         Heres the first block of code which will load the Plivo and MailChimp libraries and assign the variables sent by Plivo.
        </p>
        <pre>
         &lt;?php
require('lib/plivo.php');
require('lib/mailchimp.php');

//Collect the details from the SMS
$to = $_REQUEST['To'];
$from = $_REQUEST['From'];
$text = $_REQUEST['Text'];
        </pre>
        <p>
         So now that we have our variables set, lets check to see if the body of the SMS is a valid email address and if so subscribe them to my MailChimp list.
        </p>
        <p>
         The contents of the SMS will be in the
         <code>
          $text
         </code>
         variable. For now Im just going to validate this variable using the php
         <code>
          filter_var
         </code>
         function. This will do a basic evaluation on the string to make sure it follows the general email address rules, it doesnt actually verify that the email is valid, but we just want to make sure they sent us something that looks like an email that well pass on the MailChimp.
        </p>
        <pre>
         if (filter_var($text, FILTER_VALIDATE_EMAIL)) { 
   ## Subscribe the caller
   $MailChimp = new MailChimp('API_KEY');
   $result = $MailChimp-&gt;call( 'lists/subscribe', 
				array('id' =&gt; 'LIST ID', 'email' =&gt; array('email'=&gt; $text)));
        </pre>
        <p>
         Next well make sure the MailChimp API Call worked and respond to the SMS with a confirmation.
        </p>
        <pre>
         if ($result){
                 $body = "Thanks! Check your email to confirm your subscription";
                 }
}else{ //Email didn't Validate
        $body = "Sorry, that does not look like a valid email";
        }
//Setup the Plivo Response      
$r = new Response();
$r-&gt;addMessage($body,$params);
header('Content-type: application/xml');
echo $r-&gt;toXML();
?&gt;
        </pre>
        <p>
         Thats really all there is to the basic script. Because my mailing list on MailChimp is setup to Double Opt-In the email address will get a confirmation link to click on. This makes sure that the email is valid because we really didnt do any validation except check the format. Now that weve got the script which will answer our incoming SMS we need to create a Plivo application and tell it to use this script.
        </p>
        <h3>
         Create the Plivo application
        </h3>
        <p>
         First, login to plivo.com. From the Applications tab, create a new application and give it a name. The only thing you need to change from the default is the Message url. Well put the URL to the script we just created and make sure the method is POST.
        </p>
        <p>
         <img alt="resource" class="aligncenter size-full wp-image-111" height="711" src="http://vincedasta.com/wp-content/uploads/2014/01/resource.jpg" width="644"/>
        </p>
        <p>
         Now that the application has been created well add a phone number. From the
         <strong>
          Numbers
         </strong>
         section, click on
         <strong>
          Buy Number
         </strong>
         and find a number you like, just make sure you check
         <strong>
          SMS Enabled
         </strong>
         . Unfortunately you can only pick the area code and a random number will be assigned, but you can always drop the number and try again if you get one you dont like.
        </p>
        <p>
         <img alt="resource 2" class="aligncenter size-full wp-image-119" height="497" src="http://vincedasta.com/wp-content/uploads/2014/01/resource-2.jpg" width="783"/>
        </p>
        <p>
         Once you select an area code, just make sure to assign the number to the application you just created.
        </p>
        <p>
         <img alt="resource" class="aligncenter size-full wp-image-120" height="277" src="http://vincedasta.com/wp-content/uploads/2014/01/resource3.jpg" width="595"/>
        </p>
        <p>
         Thats all there is to it. Now just tell people to text you their email and theyre subscribed to your MailChimp list!
        </p>
        <p>
         Want to see it in action? just text your email to
         <strong>
          (779) 201-1688
         </strong>
         and Ill notify you once the next part of this tutorial is posted.
        </p>
        <p>
         You can download the
         <a href="http://vincedasta.com/wp-content/uploads/2014/01/SMS-MailChimp-1.zip">
          complete code and libraries here if you want.
         </a>
        </p>
        <h2>
         Next Steps
        </h2>
        <p>
         Check back next week for part two. In part two well add some more interactivity, expand our application to handle more complex messages, and show you how to integrate more meta-data to the list like tracking the phone number and caller ID name in your MailChimp List. Ill also show you how to let your subscribers opt-in for SMS notifications when you send updates to your list.
        </p>
        <p>
         As always, if youre interested in learning how to build voice and SMS powered applications, check out
         <a href="http://vincedasta.com/books/building-phone-applications/" title="Building Phone Applications">
          Building Phone Applications
         </a>
         or of course you could just text me your email at
         <strong>
          779-201-1688
         </strong>
         and Ill send you some more applications.
        </p>
        <p>
