              Nintendos famous portable is 25 years old today. The handheld would see numerous successors, peripherals, and notable games during its 15 years of dominance before the Nintendo DS would take over as the new brand for the Japanese gaming giants portable endeavors.
             </p>
             <p>
              Below is a timeline that includes all of the Game Boys majormilestones. Note that with the exception ofthe Japanese launch, all of these dates correlate with U.S. releases. Enjoy your nostalgic trip through handheld history, and check out our
              <a href="http://venturebeat.com/2014/04/21/gamesbeat-community-write-about-game-boy-for-its-25th-birthday/" title="GamesBeat community: Write about Game Boy for its 25th birthday">
               community prompt
              </a>
              if you want to write about the Game Boy yourself!
             </p>
             <hr/>
             <div class="wp-caption alignright" id="attachment_1454020" style="max-width:191px;">
              <a href="http://venturebeat.com/2014/04/21/25-years-of-the-game-boy-a-timeline-of-the-systems-accessories-and-games/gameboy-6/" rel="attachment wp-att-1454020">
               <img alt="The original Game Boy." class="wp-image-1454020" data-src="http://venturebeat.com/wp-content/uploads/2014/04/gameboy.jpg?w=254" height="300" src="//vbstatic.co/assets/img/logo-placeholder.png" width="191"/>
               <noscript>
                <img alt="The original Game Boy." class="wp-image-1454020" data-recalc-dims="1" src="http://i1.wp.com/venturebeat.com/wp-content/uploads/2014/04/gameboy.jpg?resize=191%2C300"/>
               </noscript>
              </a>
              <p class="wp-caption-text">
               Above: The original Game Boy.
              </p>
              <div class="vb_image_source">
               <em>
                Image Credit: Wikipedia
               </em>
              </div>
             </div>
             <h2>
              <strong>
               April 21, 1989: Nintendo releases the Game Boy in Japan
              </strong>
             </h2>
             <p>
              The original Game Boy was the brainchild of Nintendos Gunpei Yokoi, who had previously produced the innovative Metroidaction-platformer for the Nintendo Entertainment System.The Game Boy was an 8-bit handheld console that supported interchangeable cartridges, although its screen could only display games in four colors of gray.
             </p>
             <p>
              Super Mario Land, a 2D sidescroller similar to Super Mario Bros. on the Nintendo Entertainment System, was the most notable launch game.
             </p>
             <h2>
              <strong>
               July 31, 1989:The Game Boy releases in the U.S. with Tetris
              </strong>
             </h2>
             <p>
              While the U.S. launch also got Super Mario Land, the puzzle game Tetris made the biggest impact. Nintendo bundled it with the system. With 35 million copies sold, it was the best-selling Game Boy game.
             </p>
             <h2>
              <strong>
               Aug. 1, 1992: Kirbys Dream Land
              </strong>
             </h2>
             <p>
              While most of Nintendos notable franchises, like Mario and Zelda, established themselves on home consoles and later showed up on the Game Boy, the Kirby series debuted on the handheld. The next game in the long-lived series comes out May 2: Kirby Triple Deluxe for the Nintendo 3DS handheld.
             </p>
             <h2>
              <strong>
               Jun 28, 1994: The Super Game Boy
              </strong>
             </h2>
             <p>
              The Super Game Boy was a cartridge for the Super Nintendo home console that had a slot for Game Boy games  with it, you couldplay Game Boy titles on a TV. Certain games, like the Game Boy version of Donkey Kong, would receive color and other enhancements when you played it on the Super Game Boy.
             </p>
             <h2>
              <strong>
               Aug. 14, 1995: The Virtual Boy
              </strong>
             </h2>
             <p>
              The Virtual Boy was a handheld system that Yokoi also created. Itsimulated a 3D view of games but could only display themin black and red. Unlike the Game Boy, it was a massive flop.
             </p>
             <div class="wp-caption alignright" id="attachment_1454027" style="max-width:187px;">
              <a href="http://venturebeat.com/2014/04/21/25-years-of-the-game-boy-a-timeline-of-the-systems-accessories-and-games/gbpocket/" rel="attachment wp-att-1454027">
               <img alt="The Game Boy Pocket." class="wp-image-1454027" data-src="http://venturebeat.com/wp-content/uploads/2014/04/gbpocket.png?w=249" height="300" src="//vbstatic.co/assets/img/logo-placeholder.png" width="187"/>
               <noscript>
                <img alt="The Game Boy Pocket." class="wp-image-1454027" data-recalc-dims="1" src="http://i0.wp.com/venturebeat.com/wp-content/uploads/2014/04/gbpocket.png?resize=187%2C300"/>
               </noscript>
              </a>
              <p class="wp-caption-text">
               Above: The Game Boy Pocket.
              </p>
              <div class="vb_image_source">
               <em>
                Image Credit: Nintendo Wikia
               </em>
              </div>
             </div>
             <h2>
              <strong>
               July 21, 1996: The Game Boy Pocket
              </strong>
             </h2>
             <p>
              The Game Boy Pocket was a slimmer version of the original Game Boy with the same hardware specs.
             </p>
             <h2>
              <strong>
               Oct. 4, 1997: Yokoi dies
              </strong>
             </h2>
             <p>
              A traffic accident kills the father of the Game Boy.
             </p>
             <h2>
              <strong>
               June 1, 1998: The Game Boy Camera and Game Boy Printer
              </strong>
             </h2>
             <p>
              The accessories enabled Game Boy owners to take and print pictures.
             </p>
             <h2>
              <strong>
               Sept. 28, 1998: Pokmon Red and Blue
              </strong>
             </h2>
             <p>
              Pokmon was a role-playing game that tasked players with capturing, training, and fighting 151 different pocket monsters. Itlaunched a media franchise that included a popular card game and anime.
             </p>
             <p>
              Today, its still one of Nintendos biggest properties, and at least one new generation ofPokmon games releases on every Nintendo handheld system.These games are always among the best-selling for their respective systems.
             </p>
             <div class="wp-caption alignright" id="attachment_1454026" style="max-width:177px;">
              <a href="http://venturebeat.com/2014/04/21/25-years-of-the-game-boy-a-timeline-of-the-systems-accessories-and-games/gbcolor/" rel="attachment wp-att-1454026">
               <img alt="The Game Boy Color." class="wp-image-1454026" data-src="http://venturebeat.com/wp-content/uploads/2014/04/gbcolor.png?w=236" height="300" src="//vbstatic.co/assets/img/logo-placeholder.png" width="177"/>
               <noscript>
                <img alt="The Game Boy Color." class="wp-image-1454026" data-recalc-dims="1" src="http://i0.wp.com/venturebeat.com/wp-content/uploads/2014/04/gbcolor.png?resize=177%2C300"/>
               </noscript>
              </a>
              <p class="wp-caption-text">
               Above: The Game Boy Color.
              </p>
              <div class="vb_image_source">
               <em>
                Image Credit: Wikipedia
               </em>
              </div>
             </div>
             <h2>
              <strong>
               Nov.18, 1998: The Game Boy Color
              </strong>
             </h2>
             <p>
              The Game Boy Color was the first true successor to the original Game Boy. As the name implies, it could support up to 56 colors.
             </p>
             <p>
              The Game Boy Color was backward compatible, meaning it could also play original Game Boy games.
             </p>
             <p>
              While the original Game Boy dominated the portable market for almost 10 years before getting a proper upgrade, Nintendo would replace the Game Boy Color in less than three years.
             </p>
             <h2>
              <strong>
               June 21, 2001: The Game Boy Advance
              </strong>
             </h2>
             <p>
              The Game Boy Advance was the follow-up to the Game Boy Color and featured graphics similar to the Super Nintendo.
             </p>
             <div class="wp-caption alignright" id="attachment_1454023" style="max-width:300px;">
              <a href="http://venturebeat.com/2014/04/21/25-years-of-the-game-boy-a-timeline-of-the-systems-accessories-and-games/gba/" rel="attachment wp-att-1454023">
               <img alt="The Game Boy Advance." class="size-medium wp-image-1454023" data-src="http://venturebeat.com/wp-content/uploads/2014/04/gba.png?w=300" height="179" src="//vbstatic.co/assets/img/logo-placeholder.png" width="300"/>
               <noscript>
                <img alt="The Game Boy Advance." class="size-medium wp-image-1454023" data-recalc-dims="1" src="http://i1.wp.com/venturebeat.com/wp-content/uploads/2014/04/gba.png?resize=300%2C179"/>
               </noscript>
              </a>
              <p class="wp-caption-text">
               Above: The Game Boy Advance.
              </p>
              <div class="vb_image_source">
               <em>
                Image Credit: Zelda Wikia
               </em>
              </div>
             </div>
             <p>
              It was backward compatible with Game Boy and Game Boy Color games.
             </p>
             <h2>
              <strong>
               Sept. 17, 2002: The e-Reader
              </strong>
             </h2>
             <p>
              The e-Reader was a peripheral for the Game Boy Advance that could read data from cards. This data turned intoadditional content for Game Boy Advance games and full versions of original Nintendo Entertainment System games.
             </p>
             <div class="wp-caption alignright" id="attachment_1454024" style="max-width:247px;">
              <a href="http://venturebeat.com/2014/04/21/25-years-of-the-game-boy-a-timeline-of-the-systems-accessories-and-games/sony-dsc-42/" rel="attachment wp-att-1454024">
               <img alt="The Game Boy Advance SP." class="wp-image-1454024" data-src="http://venturebeat.com/wp-content/uploads/2014/04/gbasp.jpg?w=300" height="200" src="//vbstatic.co/assets/img/logo-placeholder.png" width="247"/>
               <noscript>
                <img alt="The Game Boy Advance SP." class="wp-image-1454024" data-recalc-dims="1" src="http://i1.wp.com/venturebeat.com/wp-content/uploads/2014/04/gbasp.jpg?resize=247%2C200"/>
               </noscript>
              </a>
              <p class="wp-caption-text">
               Above: The Game Boy Advance SP.
              </p>
              <div class="vb_image_source">
               <em>
                Image Credit: Wikipedia
               </em>
              </div>
             </div>
             <h2>
              <strong>
               March 23, 2003: The Game Boy Advance SP
              </strong>
             </h2>
             <p>
              The SP was a new model of the Game Boy Advance that featured a back-lit screen. It also introduced a clamshell-like design that most Nintendo handhelds have adopted since.
             </p>
             <h2>
              <strong>
               June 24, 2003: The Game Boy Player
              </strong>
             </h2>
             <p>
              The Game Boy Player was an add-on that attached to the bottom of the GameCube console. It could play Game Boy, Game Boy Color, and Game Boy Advance games.
             </p>
             <div class="wp-caption alignright" id="attachment_1454022" style="max-width:265px;">
              <a href="http://venturebeat.com/2014/04/21/25-years-of-the-game-boy-a-timeline-of-the-systems-accessories-and-games/game-boy-micro/" rel="attachment wp-att-1454022">
               <img alt="The Game Boy Micro." class="wp-image-1454022" data-src="http://venturebeat.com/wp-content/uploads/2014/04/game-boy-micro.png?w=300" height="150" src="//vbstatic.co/assets/img/logo-placeholder.png" width="265"/>
               <noscript>
                <img alt="The Game Boy Micro." class="wp-image-1454022" data-recalc-dims="1" src="http://i1.wp.com/venturebeat.com/wp-content/uploads/2014/04/game-boy-micro.png?resize=265%2C150"/>
               </noscript>
              </a>
              <p class="wp-caption-text">
               Above: The Game Boy Micro.
              </p>
              <div class="vb_image_source">
               <em>
                Image Credit: Wikipedia
               </em>
              </div>
             </div>
             <h2>
              <strong>
               September 19, 2005: The Game Boy Micro
              </strong>
             </h2>
             <p>
              The Micro was a redesigned version of the Game Boy Advance that dropped a few features, notably backward compatibility with original Game Boy and Game Boy Color games, in order to be as small as possible. Released a year after the Nintendo DS, the successful portable system that replaced the Game Boy brand, this was the last iteration of the Game Boy Nintendo would release.
             </p>
             <h2>
              <strong>
               June 6, 2011: Game Boy games appear on the Nintendo 3DS Virtual Console
              </strong>
             </h2>
             <p>
              Nintendo originally created the Virtual Consolefor the Wii so people could buy, download, and play classic Nintendo games. The Nintendo 3DS portable includes a version of the digital store that sells Game Boy and Game Boy Color games. Game Boy Advance games would also become available, but only for those who bought the Nintendo 3DS at launch.
             </p>
             <h2>
              <strong>
               April 3, 2014: Game Boy Advance games appear on the Wii U Virtual Console
              </strong>
             </h2>
             <p>
              Similar to the Nintendo 3DS service, the Wii U Virtual Console offersclassic games. While the Wii U version had originally stuck to home consoles, Game Boy Advance titles eventually appeared on the digital store. Wii U Virtual Console games can display on a TV or on the systems tablet controller.
             </p>
