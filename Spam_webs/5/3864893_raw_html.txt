       <p>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_intro_image.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i0.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_intro_image.jpg?resize=629%2C600" style=""/>
        </span>
        <br/>
        <strong>
         By Danielle Thompson
        </strong>
        <br/>
        I really love the look of vintage needlepoint (especially needlepoint handbags!) from the 50s, 60s, and 70s, and wanted to try my hand at needlepoint myself. So I grabbed some plastic canvas from the craft store, got to stitching, and soon realized: needlepoint is pretty easy! Even the simplest stitches produce a beautiful effect. A lot of people associate ugly designs from the 80s when they think of plastic canvas, but its a great way to introduce yourself to the gorgeous world of needlepoint. The canvas is easy to use  it has a large weave so projects work up very fast, and its not finicky like fabric canvas can be. Ive put together a tutorial for making a stylish needlepoint purse with a design thats a modern twist on this favorite from the past.
       </p>
       <p>
        <span id="more-17358">
        </span>
       </p>
       <h3>
        Materials
       </h3>
       <p>
        <strong>
         (2) 13 1/2 x 10 1/2 plastic needlepoint canvas
         <br/>
         Scissors
         <br/>
         Plastic canvas needles
         <br/>
         Worsted-weight yarn in various colors
        </strong>
        (this is a great stash buster)
        <br/>
        <strong>
         1 button cover kit
        </strong>
        ; youll be making 3 buttons
        <br/>
        <strong>
         Wool felt: (2) 8 3/4 x 4 pieces; (2) 13 1/4 x 8 3/4 pieces; scraps for felt buttons
        </strong>
        your choice of colors
        <br/>
        <strong>
         Purse handles
        </strong>
        <br/>
        See the supplies/resources guide at end of this tutorial for tips to help you find these items.
       </p>
       <table>
        <tr>
         <td>
          <img alt="Download PDF" border="0" data-recalc-dims="1" src="http://i1.wp.com/cdn.makezine.com/uploads/2008/11/icon_pdf.gif?resize=18%2C23"/>
         </td>
         <td>
          <a href="http://cdn.makezine.com/uploads/2012/02/needlepointpurse.pdf">
           Download the Needlepoint design PDF
          </a>
          <br/>
          Right click to save the PDF to your desktop. Directions on
          <a href="http://blog.craftzine.com/archive/2009/09/ask_craft_downloading_our_pdfs.html">
           downloading PDFs.
          </a>
         </td>
        </tr>
       </table>
       <h3>
        Directions
       </h3>
       <p>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_step1a.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i0.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_step1a.jpg?resize=629%2C434" style=""/>
        </span>
        <br/>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_step1b.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i0.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_step1b.jpg?resize=629%2C419" style=""/>
        </span>
        <br/>
        <strong>
         Step 1:
        </strong>
        Trim your plastic needlepoint canvas down with scissors. For a 13 1/2 x 10 1/2 piece, cut off 12 horizontal plastic threads at the bottom and 1 vertical plastic thread off one side. Be careful to cut down the last nubs of plastic fairly close, while being careful not to cut into the next thread. When I made this purse, I cut down my canvas after I finished my needlepoint design  its up to you if you want to do it before or after.
        <br/>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_step2.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i0.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_step2.jpg?resize=629%2C429" style=""/>
        </span>
        <br/>
        <strong>
         Step 2:
        </strong>
        Get stitching! Only the front panel of the purse is created using needlepoint. You can download the needlepoint design PDF (above).
        <a href="http://needlepoint.about.com/od/stitchdiagrams/ss/Continental.htm">
         The Continental Tent Stitch
        </a>
        and
        <a href="http://books.google.com/books?id=KgE1OpKIjVwC&amp;pg=PA199&amp;lpg=PA199&amp;dq=crossed+corners+cushion+stitch+needlepoint&amp;source=bl&amp;ots=gEdGLaziTI&amp;sig=CpIx79yqHx5Jwyyt1j7Fhfeamn0&amp;hl=en&amp;sa=X&amp;ei=W7clT7P2DIectwfnq42iCw&amp;ved=0CFUQ6AEwAw#v=onepage&amp;q=crossed%20corners%20cushion%20stitch%20needlepoint&amp;f=false">
         Crossed Corners Cushion Stitch
        </a>
        are used. These are very simple stitches. The Crossed Corners Cushion Stitch is great because it covers a large area quickly. And the weave on plastic canvas is so large, the piece works up fairly fast. As you can see, the arrangement of four Crossed Corners Cushion Stitches together creates a pretty pinwheel effect.
        <br/>
        <a href="http://thompsonfamily.typepad.com/thompson_familylife/2012/02/vintage-style-needlepoint-purse-pattern-how-to-on-craft.html">
         Here are some diagrams
        </a>
        on ending and starting threads for needlepoint that directly relate to this purse project.
        <br/>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_step3a.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i1.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_step3a.jpg?resize=629%2C600" style=""/>
        </span>
        <br/>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_step3b.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i2.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_step3b.jpg?resize=629%2C419" style=""/>
        </span>
        <br/>
        <strong>
         Step 3:
        </strong>
        Add your felt covered buttons. If you dont have a button-covering kit, some large-sized vintage buttons would be a pretty addition. Make your buttons according to the instructions in the image above. To attach the buttons to your needlepoint, knot one end of a length of yarn, pull your needle through the middle of one set of four Crossed Corners Cushion Stitches, pull the yarn through the shank of your button, and then insert your needle back into the same hole through the back. Secure your thread and cut. Add three buttons down the front center of the purse.
        <br/>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_step4a.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i0.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_step4a.jpg?resize=629%2C446" style=""/>
        </span>
        <br/>
        <strong>
         Step 4:
        </strong>
        Cut out the remaining pieces of your purse according to the image above.
        <br/>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_step5a.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i0.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_step5a.jpg?resize=629%2C359" style=""/>
        </span>
        <br/>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_step5b.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i1.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_step5b.jpg?resize=629%2C600" style=""/>
        </span>
        <br/>
        <strong>
         Step 5:
        </strong>
        You can choose to back your needlepoint with felt or not, but I thought it was important to protect the back of the piece from whatever youre putting in your purse. Place your needlepoint backing on top of the wrong side of your needlepoint piece. Anchor the four corners of the backing to your needlepoint piece so that it stays in place while you stitch the two together. Stitch all the way around as shown above. Use a yarn color that blends in well with your plastic canvas. I used blue so that you could see the stitches well for this tutorial, but ended up using a dark brown to match my canvas.
        <br/>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_step6a.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i2.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_step6a.jpg?resize=629%2C486" style=""/>
        </span>
        <br/>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_step6b.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i0.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_step6b.jpg?resize=629%2C600" style=""/>
        </span>
        <br/>
        <strong>
         Step 6:
        </strong>
        Stitch all parts of the bag together as shown in the above image, in this order:
       </p>
       <ol>
        <li>
         Whipstitch A to B.
        </li>
        <li>
         Using a running stitch, stitch A to C, C to D, D to E, and then E to A. Leave approximately 1/4 of a seam allowance (keeping this to the inside of the bag). You could also use a sewing machine to stitch C to D and D to E to save time.
        </li>
        <li>
         Carefully flip the bag inside out. Whipstitch B to the bottoms of C, D, and E.
        </li>
       </ol>
       <p>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_step7a.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i2.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_step7a.jpg?resize=629%2C208" style=""/>
        </span>
        <br/>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_step7b.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i2.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_step7b.jpg?resize=629%2C318" style=""/>
        </span>
        <br/>
        <strong>
         Step 7:
        </strong>
        Attach the purse handles. For the front side of the purse, anchor a length of yarn (use a color that blends in with the color of the purse handle) to the top of the needlepoint panel, loop through one end of the purse handle opening, and then back through the needlepoint canvas. Continue doing this wide whipstitch to connect the two together until you get to the end of the purse handle opening. I wanted the look to be thick, so I doubled my stitches through the weave of the canvas, as shown above.
        <br/>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_step8.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i0.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_step8.jpg?resize=629%2C482" style=""/>
        </span>
        <br/>
        <strong>
         Step 8:
        </strong>
        Attach the other purse handle to the back panel. Since its the back, I didnt care too much how it looked so I didnt double up my stitches. Since the back panel is felt, I wanted to reinforce where the handle will be attached, so I stitched a small piece of plastic canvas to the top using a running stitch, and then attached the handle using a whipstitch (see the above image).
        <br/>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_step9a.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i2.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_step9a.jpg?resize=629%2C579" style=""/>
        </span>
        <br/>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_step9b.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i0.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_step9b.jpg?resize=629%2C482" style=""/>
        </span>
        <br/>
        <strong>
         Step 9:
        </strong>
        Pinch the tops of the side panels and secure with a small stitch. Youre done!
        <br/>
        <span class="mt-enclosure mt-enclosure-image" style="display: inline;">
         <img alt="needlepoint_purse_opener1.jpg" class="mt-image-none" data-recalc-dims="1" src="http://i0.wp.com/cdn.makezine.com/uploads/2012/02/needlepoint_purse_opener1.jpg?resize=629%2C628" style=""/>
        </span>

	// listen for play/pause, other states such as rewind and end could also be added
	// also report % played every second
	function onPlayerStateChange(e) {
	    e["data"] == YT.PlayerState.PLAYING &amp;&amp; setTimeout(onPlayerPercent, 1000, e["target"]);
	    var video_data = e.target["getVideoData"](),
	        label = video_data.title;
	    if (e["data"] == YT.PlayerState.PLAYING &amp;&amp; YT.gtmLastAction == "p") {
	        dataLayer.push({
	            event: "videos",
	            action: "play",
	            label: label
	        });
	        YT.gtmLastAction = "";
	    }
	    if (e["data"] == YT.PlayerState.PAUSED) {
	        dataLayer.push({
	            event: "videos",
	            action: "pause",
	            label: label
	        });
	        YT.gtmLastAction = "p";
	    }
	}

	// catch all to report errors through the GTM data layer
	// once the error is exposed to GTM, it can be tracked in UA as an event!
	function onPlayerError(e) {
	    dataLayer.push({
	        event: "error",
	        action: "GTM",
	        label: "videos:" + e["target"]["src"] + "-" + e["data"]
	    })
	}

	// report the % played if it matches 0%, 25%, 50%, 75% or completed
	function onPlayerPercent(e) {
	    if (e["getPlayerState"]() == YT.PlayerState.PLAYING) {
	        var t = e["getDuration"]() - e["getCurrentTime"]() &lt;= 1.5 ? 1 : (Math.floor(e["getCurrentTime"]() / e["getDuration"]() * 4) / 4).toFixed(2);        if (!e["lastP"] || t &gt; e["lastP"]) {
	            var video_data = e["getVideoData"](),
	                label = video_data.title;
	            e["lastP"] = t;
	            dataLayer.push({
	                event: "videos",
	                action: t * 100 + "%",
	                label: label
	            })
	        }
	        e["lastP"] != 1 &amp;&amp; setTimeout(onPlayerPercent, 1000, e);
	    }
	}
