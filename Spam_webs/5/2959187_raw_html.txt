   <section class="inner" id="main_content">
    <p>
     Linuxbrew is a fork of
     <a href="http://brew.sh">
      Homebrew
     </a>
     , the Mac OS package manager, for Linux.
    </p>
    <p>
     Features, usage and installation instructions are
     <a href="http://brew.sh/linuxbrew/">
      summarised on the homepage
     </a>
     .
    </p>
    <h2>
     <a aria-hidden="true" class="anchor" href="#install-linuxbrew-tldr" id="install-linuxbrew-tldr">
      <span class="octicon octicon-link">
      </span>
     </a>
     Install Linuxbrew (tl;dr)
    </h2>
    <p>
     Paste at a Terminal prompt:
    </p>
    <div class="highlight highlight-sh">
     <pre>
      ruby -e
      <span class="pl-s1">
       <span class="pl-pds">
        "
       </span>
       <span class="pl-s1">
        <span class="pl-pds">
         $(
        </span>
        curl -fsSL https://raw.githubusercontent.com/Homebrew/linuxbrew/go/install
        <span class="pl-pds">
         )
        </span>
       </span>
       <span class="pl-pds">
        "
       </span>
      </span>
     </pre>
    </div>
    <p>
     See
     <a href="#dependencies">
      Dependencies
     </a>
     and
     <a href="#installation">
      Installation
     </a>
     below for more details.
    </p>
    <h2>
     <a aria-hidden="true" class="anchor" href="#features" id="features">
      <span class="octicon octicon-link">
      </span>
     </a>
     Features
    </h2>
    <ul>
     <li>
      Can install software to a home directory and so does not require sudo
     </li>
     <li>
      Install software not packaged by the native distribution
     </li>
     <li>
      Install up-to-date versions of software when the native distribution is old
     </li>
     <li>
      Use the same package manager to manage both your Mac and Linux machines
     </li>
    </ul>
    <h2>
     <a aria-hidden="true" class="anchor" href="#dependencies" id="dependencies">
      <span class="octicon octicon-link">
      </span>
     </a>
     Dependencies
    </h2>
    <ul>
     <li>
      <strong>
       Ruby
      </strong>
      1.8.6 or newer
     </li>
     <li>
      <strong>
       GCC
      </strong>
      4.2 or newer
     </li>
    </ul>
    <p>
     Paste at a Terminal prompt:
    </p>
    <h3>
     <a aria-hidden="true" class="anchor" href="#debian-or-ubuntu" id="debian-or-ubuntu">
      <span class="octicon octicon-link">
      </span>
     </a>
     Debian or Ubuntu
    </h3>
    <div class="highlight highlight-sh">
     <pre>
      sudo apt-get install build-essential curl git m4 ruby texinfo libbz2-dev libcurl4-openssl-dev libexpat-dev libncurses-dev zlib1g-dev
     </pre>
    </div>
    <h3>
     <a aria-hidden="true" class="anchor" href="#fedora" id="fedora">
      <span class="octicon octicon-link">
      </span>
     </a>
     Fedora
    </h3>
    <div class="highlight highlight-sh">
     <pre>
      sudo yum groupinstall
      <span class="pl-s1">
       <span class="pl-pds">
        '
       </span>
       Development Tools
       <span class="pl-pds">
        '
       </span>
      </span>
      <span class="pl-k">
       &amp;&amp;
      </span>
      sudo yum install curl git m4 ruby texinfo bzip2-devel curl-devel expat-devel ncurses-devel zlib-devel
     </pre>
    </div>
    <h2>
     <a aria-hidden="true" class="anchor" href="#installation" id="installation">
      <span class="octicon octicon-link">
      </span>
     </a>
     Installation
    </h2>
    <p>
     Paste at a Terminal prompt:
    </p>
    <div class="highlight highlight-sh">
     <pre>
      ruby -e
      <span class="pl-s1">
       <span class="pl-pds">
        "
       </span>
       <span class="pl-s1">
        <span class="pl-pds">
         $(
        </span>
        curl -fsSL https://raw.githubusercontent.com/Homebrew/linuxbrew/go/install
        <span class="pl-pds">
         )
        </span>
       </span>
       <span class="pl-pds">
        "
       </span>
      </span>
     </pre>
    </div>
    <p>
     Or if you prefer:
    </p>
    <div class="highlight highlight-sh">
     <pre>
      git clone https://github.com/Homebrew/linuxbrew.git
      <span class="pl-k">
       ~
      </span>
      /.linuxbrew
     </pre>
    </div>
    <p>
     Add to your
     <code>
      .bashrc
     </code>
     or
     <code>
      .zshrc
     </code>
     :
    </p>
    <div class="highlight highlight-sh">
     <pre>
      <span class="pl-s">
       export
      </span>
      PATH=
      <span class="pl-s1">
       <span class="pl-pds">
        "
       </span>
       <span class="pl-vo">
        $HOME
       </span>
       /.linuxbrew/bin:
       <span class="pl-vo">
        $PATH
       </span>
       <span class="pl-pds">
        "
       </span>
      </span>
      <span class="pl-s">
       export
      </span>
      MANPATH=
      <span class="pl-s1">
       <span class="pl-pds">
        "
       </span>
       <span class="pl-vo">
        $HOME
       </span>
       /.linuxbrew/share/man:
       <span class="pl-vo">
        $MANPATH
       </span>
       <span class="pl-pds">
        "
       </span>
      </span>
      <span class="pl-s">
       export
      </span>
      INFOPATH=
      <span class="pl-s1">
       <span class="pl-pds">
        "
       </span>
       <span class="pl-vo">
        $HOME
       </span>
       /.linuxbrew/share/info:
       <span class="pl-vo">
        $INFOPATH
       </span>
       <span class="pl-pds">
        "
       </span>
      </span>
     </pre>
    </div>
    <p>
     You're done!
    </p>
    <div class="highlight highlight-sh">
     <pre>
      brew install
      <span class="pl-vo">
       $WHATEVER_YOU_WANT
      </span>
     </pre>
    </div>

      <a href="https://github.com/telemachus/homebrew-desc">
       <code>
        brew desc
       </code>
      </a>
      to browse packages from the command line.
     </li>
    </ol>

    <h2>
     <a aria-hidden="true" class="anchor" href="#troubleshooting" id="troubleshooting">
      <span class="octicon octicon-link">
      </span>
     </a>
     Troubleshooting
    </h2>
    <p>
     First, please run
     <code>
      brew update
     </code>
     and
     <code>
      brew doctor
     </code>
     .
    </p>
    <p>
     Second, read the
     <a href="https://github.com/Homebrew/linuxbrew/blob/master/share/doc/homebrew/Troubleshooting.md#troubleshooting">
      Troubleshooting Checklist
     </a>
     .
    </p>
    <p>
     <strong>
      If you don't read these it will take us far longer to help you with your problem.
     </strong>
    </p>
    <h2>
     <a aria-hidden="true" class="anchor" href="#something-broke" id="something-broke">
      <span class="octicon octicon-link">
      </span>
     </a>
     Something broke!
    </h2>
    <p>
     Many of the Homebrew formulae work on either Mac or Linux without changes, but some formulae will need to be adapted for Linux. If a formula doesn't work,
     <a href="https://github.com/Homebrew/linuxbrew/issues">
      open an issue on GitHub
     </a>
     or, even better, submit a pull request.
    </p>
