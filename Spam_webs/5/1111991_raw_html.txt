          <div class="post entry-content " itemprop="commentText">
           The Azoo Palm Filter is personally one of the coolest things on the aquarium market. Azoo took the idea of the filter and made it very very small. The only problem with this is that the "3-stage filtration" got hit where it hurts. Today I took a gander at what makes up the filter.
           <br/>
           <br/>
           <span rel="lightbox">
            <img alt="Posted Image" class="bbc_img" src="http://img78.imageshack.us/img78/6470/1001464mediumaw9.jpg"/>
           </span>
           <br/>
           <br/>
           The filter is made like any basic filter containing a body, impeller, intake tubing, body cover, and a flow spout. This gives the filter some
           <acronym class="bbc ipSeoAcronym" title="Do it yourself">
            DIY
           </acronym>
           space.
           <br/>
           <br/>
           <span rel="lightbox">
            <img alt="Posted Image" class="bbc_img" src="http://img62.imageshack.us/img62/7302/1001465mediumdu2.jpg"/>
           </span>
           <br/>
           <br/>
           Further examining the filter, I looked that the "filter media" that the Azoo Palm Filter gives you. It is rubbish! soooo we have to think here. In every good filter we have biological, mechanical, and chemical filtration. At the moment it fails.
           <br/>
           <br/>
           <span rel="lightbox">
            <img alt="Posted Image" class="bbc_img" src="http://img452.imageshack.us/img452/129/1001466mediumpk1.jpg"/>
           </span>
           <br/>
           <br/>
           So follow these steps to improve your filter. Take the black course sponge and set it aside. Take the packed filter floss and peel thin slices of it off as such:
           <br/>
           <br/>
           <span rel="lightbox">
            <img alt="Posted Image" class="bbc_img" src="http://img233.imageshack.us/img233/8877/1001467mediumsv9.jpg"/>
           </span>
           <br/>
           <br/>
           Take a common sowing needle and some fishing line. Do the usual knots when you are mending your clothing.
           <br/>
           <br/>
           <span rel="lightbox">
            <img alt="Posted Image" class="bbc_img" src="http://img233.imageshack.us/img233/9431/1001468mediumur2.jpg"/>
           </span>
           <br/>
           <br/>
           Start sowing at the top corner and sow all the way down one of the long sides.
           <br/>
           <br/>
           <span rel="lightbox">
            <img alt="Posted Image" class="bbc_img" src="http://img171.imageshack.us/img171/9889/1001469mediummi4.jpg"/>
           </span>
           <br/>
           <br/>
           Go down the bottom side, go around another corner and stop half way up the other side.
           <br/>
           <br/>
           <span rel="lightbox">
            <img alt="Posted Image" class="bbc_img" src="http://img171.imageshack.us/img171/8909/1001470mediumsz6.jpg"/>
           </span>
           <br/>
           <br/>
           Obtain a thin plasic spoon and your favorite brand of carbon.
           <br/>
           <br/>
           <span rel="lightbox">
            <img alt="Posted Image" class="bbc_img" src="http://img526.imageshack.us/img526/8316/1001471mediumfw2.jpg"/>
           </span>
           <br/>
           <br/>
           Stretch and pull at the filter floss a bit to make a pocket and that allows more carbon into the going-to-be newly formed media pocket.
           <br/>
           <br/>
           <span rel="lightbox">
            <img alt="Posted Image" class="bbc_img" src="http://img253.imageshack.us/img253/3224/1001472mediumss1.jpg"/>
           </span>
           <br/>
           <br/>
           Sow the packet to the top and fill the pocket all the way full. Continue to sow the pocket closed and tie it off.
           <br/>
           <br/>
           <span rel="lightbox">
            <img alt="Posted Image" class="bbc_img" src="http://img388.imageshack.us/img388/899/1001473mediumfd3.jpg"/>
           </span>
           <br/>
           <br/>
           Insert the carbon pocket in front of the black sponge from before and set it in the filter. Continue to cover the filter with the hood.
           <br/>
           <br/>
           <span rel="lightbox">
            <img alt="Posted Image" class="bbc_img" src="http://img248.imageshack.us/img248/8689/1001476mediumad0.jpg"/>
           </span>
           <br/>
           <br/>
           This is an overview of the filter on a 2 1/2 gallon tank.
           <br/>
           <br/>
           <span rel="lightbox">
            <img alt="Posted Image" class="bbc_img" src="http://img403.imageshack.us/img403/4706/1001478mediumlg5.jpg"/>
           </span>
           <br/>
           <br/>
           This is a size comparison to show how small it is. That is a 20 gallon long in the background. Sorry about the mess
           <span rel="lightbox">
            <img alt=":crazy:" class="bbc_img" src="http://www.fishforums.net/public/style_emoticons/default/crazy.gif"/>
           </span>
           <br/>
           <br/>
           <span rel="lightbox">
            <img alt="Posted Image" class="bbc_img" src="http://img170.imageshack.us/img170/309/1001479mediummu3.jpg"/>
           </span>
           <br/>
           <br/>
           This is a picture of the filter running. The filter, for its size, it quite impressive once modified.
           <br/>
           <br/>
           <span rel="lightbox">
            <img alt="Posted Image" class="bbc_img" src="http://img185.imageshack.us/img185/4782/1001480mediumhy8.jpg"/>
           </span>
           <br/>
           <br/>
           And finally this is a possible tank scenario that I threw together in 3 or so minutes:
           <br/>
           <br/>
           <span rel="lightbox">
            <img alt="Posted Image" class="bbc_img" src="http://img62.imageshack.us/img62/9514/1001481mediumyk8.jpg"/>
           </span>
           <br/>
           <br/>
           <br/>
           The possibilites are endless. You could even make a marine-pico tank! If you have this filter and have any experiances with it please share your stories!
           <br/>
           <br/>
           Maybe Ill set my other 2 1/2 gallon tank with two of these puppies later on today
