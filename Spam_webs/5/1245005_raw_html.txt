             <h3 class="post-title">
              <a href="http://wskills.blogspot.com/2006/12/16-hidden-windows-xp-goodies.html">
               16 Hidden Windows XP Goodies
              </a>
             </h3>
             <div class="post-header-line-1">
             </div>
             <div class="post-body">
              <p>
               <span style="font-weight: bold;">
                1. Installer music:
               </span>
               Start &gt; Run &gt; C:\Windows\system32\oobe\images\title.wma &gt; Ok
               <br/>
               <br/>
               <span style="font-weight: bold;">
                2. Hibernate:
               </span>
               Start &gt; Turn Off Computer &gt; press Shift key to change the Stand By button to Hibernate
               <br/>
               <br/>
               <span style="font-weight: bold;">
                3. Hidden Devices:
               </span>
               Control Panel &gt; System &gt; Hardware &gt; Device Manager &gt; select View and Show hidden devices
               <br/>
               <br/>
               <span style="font-weight: bold;">
                4. Character Map:
               </span>
               Start &gt; Run &gt; charmap.exe &gt; Ok
               <br/>
               <br/>
               <span style="font-weight: bold;">
                5. Clipboard Viewer:
               </span>
               Start &gt; Run &gt; clipbrd.exe &gt; Ok
               <br/>
               <br/>
               <span style="font-weight: bold;">
                6. Dr Watson:
               </span>
               Start &gt; Run &gt; drwtsn32.exe &gt; Ok
               <br/>
               <br/>
               <span style="font-weight: bold;">
                7. IExpress Wizard:
               </span>
               Start &gt; Run &gt; iexpress.exe &gt; Ok
               <br/>
               <br/>
               <span style="font-weight: bold;">
                8. Old Windows Media Player 5.1:
               </span>
               Start &gt; Run &gt; mplay32.exe &gt; Ok
               <br/>
               <br/>
               <span style="font-weight: bold;">
                9. ODBC Data Source Administrator:
               </span>
               Start &gt; Run &gt; odbcad32.exe &gt; Ok
               <br/>
               <br/>
               <span style="font-weight: bold;">
                10. Object Packager:
               </span>
               Start &gt; Run &gt; packager.exe &gt; Ok
               <br/>
               <br/>
               <span style="font-weight: bold;">
                11. System Monitor:
               </span>
               Start &gt; Run &gt; perfmon.exe &gt; Ok
               <br/>
               <br/>
               <span style="font-weight: bold;">
                12. Network shared folder wizard:
               </span>
               Start &gt; Run &gt; shrpubw.exe &gt; Ok
               <br/>
               <br/>
               <span style="font-weight: bold;">
                13. File siganture verification tool:
               </span>
               Start &gt; Run &gt; sigverif.exe &gt; Ok
               <br/>
               <br/>
               <span style="font-weight: bold;">
                14. System Configuration Editor:
               </span>
               Start &gt; Run &gt; sysedit.exe &gt; Ok
               <br/>
               <br/>
               <span style="font-weight: bold;">
                15. Driver Verifier Manager:
               </span>
               Start &gt; Run &gt; verifier.exe &gt; Ok
               <br/>
               <br/>
               <span style="font-weight: bold;">
                16. Windows for Workgroups Chat:
               </span>
               Start &gt; Run &gt; winchat.exe &gt; Ok
               <br/>
