   function getPos(elem, myroot, pos) {
	var myroot = myroot || "HTML";
	if (elem == null) return false;
		var rootT=elem.offsetTop 
	    	var rootL=elem.offsetLeft 
	        var rootB=elem.offsetHeight
	    	var rootR=elem.offsetWidth;
	    	var elemP = elem.parentNode;
	    	var mynode = elemP.id;
	    	
	    	while (mynode != myroot) {
	        	rootT += elemP.offsetTop;
	        	rootL += elemP.offsetLeft;
	        	
	        	elemP = elemP.parentNode;
	        	mynode = elemP.nodeName;
	    	}
	    		
	    	rootB += rootT;
	    	rootR += rootL;
	    		
	    	switch(pos)
		{
			case "top":
  				return rootT;
  				break;
			case "left":
  				return rootL;
  				break;
  			case "bottom":
  				return rootB;
  				break;
  			case "right":
  				return rootR;
  				break;
			default:
				return("error");
		}
	}

             Illinois state Attorney General Lisa Madigan launched a motion with the Illinois state Supreme Court Friday morning in a bid to have
             <a href="http://www.chicagobreakingnews.com/2008/12/attorney-gen-lisa-madigan-supreme-court-blagojevich-unable-to-serve.html">
              Gov. Rod Blagojevich declared unfit for office
             </a>

             <br/>
             <br/>
             Madigan is invoking
             <a href="http://www.suntimes.com/news/metro/blagojevich/1328298,madigan-blagojevich-121208.article">
              Rule 382
             </a>
             , which asks the justices of the states highest court to vote on a governors ability to govern. Use of the bylaw is unprecedented in Illinois history, which has seen four governors come under indictment in recent history.
             <br/>
             <br/>
             Soon after Blagojevich was arrested by the FBI onDec. 11on bribery solicitation and wire and mail fraud charges in connection with an alleged attempt to sell Obamas Senate seat, Madigan indicated she would use Rule 382 to get the governor out of office should he not resign or the state legislature not move with impeachment proceedings.
             <br/>
             <br/>
             Some state officials have mentioned the possibility of impeachment proceedings to remove Blagojevich from office. If successful, Madigans move would render impeachment unnecessary.
             <br/>
             <br/>
             Blagojevich's
             <a href="http://www.chicagobreakingnews.com/2008/12/blagojevich-chief-of-staff-harris-resigns.html">
              chief of staff John Harris resigned Friday
             </a>
             , after being arrested by the FBI on Tuesday.
             <br/>
             <br/>
             Madigan,a fellow Chicago North Side Democrat and once an ally of Blagojevichs, has come out as a
             <a href="http://www.findingdulcinea.com/news/politics/September-October-08/Illinois-Governor--Legislature-in-Tug-of-War-over-Campaign-Ethics.html">
              staunch political adversary of Blagojevich
             </a>
             over the past few years. Both Madigan and U.S. Attorney Patrick Fitzgerald had launched investigations into allegedplay-for-play politics in the Blagojevich administrations.In mid-2006 Madigan, with approval from Fitzgerald, released a letter to the public outlining the possible misconduct. Fitzgerald then had Madigan step aside from the case so as not to impede the federal investigation.

             <br/>
