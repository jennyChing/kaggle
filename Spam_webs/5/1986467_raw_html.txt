           <h1 class="page-title">
            Prostate Procrastination: 6 Foods to Eat Today
           </h1>
           <div class="box-byblock c">
            <div class="byline attribution">

             <br/>

                <li>
                 <div class="box-slide c" id="page_1">
                  <h2 class="slide-title hl-secondary-heading hl-secondary-heading-override-slideshow">
                   Foods to Boost Prostate Health
                  </h2>
                  <div class="storyArea">
                   <h2 class="story-title hl-secondary-heading hl-secondary-heading-override-slideshow">
                    Foods to Boost Prostate Health
                   </h2>
                   <div class="subhead hl-content-copy hl-content-copy-slideshow-override">
                    <p class="Normal1">
                     Its easy to put off getting your prostate
checked by your doctor. Its also easy to add some healthy prostate-friendly
foods to your diet. According to the
                     <a href="http://www.cancer.org/cancer/prostatecancer/detailedguide/prostate-cancer-key-statistics" target="_blank">
                      American Cancer Society
                     </a>
                     , prostate cancer is the most common cancer among
men. It afflicts 1 in 7 men in the United States.Its believed that the
high-fat, high-sugar Western diet may contribute to increased rates of prostate
cancer. So while
                     <a href="http://www.healthline.com/health/prostate-cancer/what-you-need-to-know#Overview1" target="_blank">
                      youll need to see a doctor eventually
                     </a>
                     , you can start boosting your prostate health by
trying these six foods.
                    </p>
                    <p>
                     <a href="https://trialreach.com/prescreening/s/226/?var=1&amp;utm_source=healthline_com_com&amp;utm_medium=referraldriver&amp;utm_campaign=TM226">
                      Looking For New Treatment Options? Learn If You Qualify For This New Trial 
                     </a>
                    </p>
                   </div>
                  </div>
                  <!-- storyArea -->
                  <div class="imageArea">
                   <div class="imageAreaInnerBox">
                    <img alt="" src="http://www.healthline.com/hlcmsresource/images/slideshow/285X285-ss-prostate-check.jpg"/>
                    <div>
                    </div>
                   </div>
                  </div>
                  <!-- imageArea -->
                 </div>
                 <!-- page_1 -->
                </li>
                <li>
                 <div class="box-slide c" id="page_2">
                  <h2 class="slide-title hl-secondary-heading hl-secondary-heading-override-slideshow">
                   Tomatoes
                  </h2>
                  <div class="storyArea">
                   <h2 class="story-title hl-secondary-heading hl-secondary-heading-override-slideshow">
                    Tomatoes
                   </h2>
                   <div class="subhead hl-content-copy hl-content-copy-slideshow-override">
                    <p class="Normal1">
                     Lycopene is a powerful antioxidant found in
tomatoes. It may help prevent prostate cancer as well as reduce tumor growth
among men with prostate cancer. In a review of 11 studies published in the
                     <a href="https://www.jstage.jst.go.jp/article/jnsv/59/3/59_213/_article" target="_blank">
                      Journal of Nutritional Science and
Vitaminology
                     </a>
                     , researchers found
that men who ate more tomatoes and tomato-based products, both raw and cooked,
were less likely to develop prostate cancer.
                    </p>
                    <p class="Normal1">
                     But because lycopene is tightly bound to cell
walls, our bodies have a difficult time extracting it from raw tomatoes.Cooked
or pureed tomato products may be better options. Look to products like these:
                    </p>
                    <ul>
                     <li>
                      tomato paste
                     </li>
                     <li>
                      spaghetti sauce
                     </li>
                     <li>
                      sun-dried tomatoes
                     </li>
                     <li>
                      tomato juice
                     </li>
                     <li>
                      ketchup
                     </li>
                    </ul>
                   </div>
                  </div>
                  <!-- storyArea -->
                  <div class="imageArea">
                   <div class="imageAreaInnerBox">
                    <img alt="" src="http://www.healthline.com/hlcmsresource/images/slideshow/285X285-ss-tomato.jpg"/>
                    <div>
                    </div>
                   </div>
                  </div>
                  <!-- imageArea -->
                 </div>
                 <!-- page_2 -->
                </li>
                <li>
                 <div class="box-slide c" id="page_3">
                  <h2 class="slide-title hl-secondary-heading hl-secondary-heading-override-slideshow">
                   Broccoli
                  </h2>
                  <div class="storyArea">
                   <h2 class="story-title hl-secondary-heading hl-secondary-heading-override-slideshow">
                    Broccoli
                   </h2>
                   <div class="subhead hl-content-copy hl-content-copy-slideshow-override">
                    <p class="Normal1">
                     Broccoli is a vegetable that contains many
complex compounds that may help protect some people from cancer. According to
the
                     <a href="http://www.cancer.org/treatment/treatmentsandsideeffects/complementaryandalternativemedicine/dietandnutrition/broccoli" target="_blank">
                      American Cancer Society
                     </a>
                     , some studies suggest theres a link between the
amount of cruciferous vegetables you eat and your prostate cancer risk.The
reasons why are still unclear. But researchers propose that one of the
phytochemicals found in these vegetables, called sulforaphane, selectively
targets and kills cancer cells while leaving normal prostate cells healthy and
unaffected.
                    </p>
                   </div>
                  </div>
                  <!-- storyArea -->
                  <div class="imageArea">
                  </div>
                  <!-- imageArea -->
                 </div>
                 <!-- page_3 -->
                </li>
                <li>
                 <div class="box-slide c" id="page_4">
                  <h2 class="slide-title hl-secondary-heading hl-secondary-heading-override-slideshow">
                   Green Tea
                  </h2>
                  <div class="storyArea">
                   <h2 class="story-title hl-secondary-heading hl-secondary-heading-override-slideshow">
                    Green Tea
                   </h2>
                   <div class="subhead hl-content-copy hl-content-copy-slideshow-override">
                    <p class="Normal1">
                     Green tea is a large component of the Asian
diet. Its been consumed for thousands of years.Its not clear if green
tea is the reason why prostate cancer rates in Asia are so much lower than in the
United States.However, components of green tea are being studied for
their effects on health.These include:
                    </p>
                    <ul>
                     <li>
                      catechin
                     </li>
                     <li>
                      xanthine derivatives
                     </li>
                     <li>
                      epigallocatechin-3-gallate (EGCG)
                     </li>
                     <li>
                      epicatechin
                      <span style="line-height: 20.7999992370605px;">
                      </span>
                     </li>
                    </ul>
                    <p class="Normal1">
                     Theres now some evidence to support that
these compounds found in green tea may prevent the development of prostate
cancer. In a review published by
                     <a href="http://www.ncbi.nlm.nih.gov/pmc/articles/PMC2577676/" target="_blank">
                      Chinese Medicine
                     </a>
                     , researchers found a decreased risk of prostate cancer among men who
consumed more than 5 cups of green tea per day.
                    </p>
                   </div>
                  </div>
                  <!-- storyArea -->
                  <div class="imageArea">
                  </div>
                  <!-- imageArea -->
                 </div>
                 <!-- page_4 -->
                </li>
                <li>
                 <div class="box-slide c" id="page_5">
                  <h2 class="slide-title hl-secondary-heading hl-secondary-heading-override-slideshow">
                   Legumes and Soybeans
                  </h2>
                  <div class="storyArea">
                   <h2 class="story-title hl-secondary-heading hl-secondary-heading-override-slideshow">
                    Legumes and Soybeans
                   </h2>
                   <div class="subhead hl-content-copy hl-content-copy-slideshow-override">
                    <p class="Normal1">
                     Legumes such as beans, peanuts, and lentils
all contain biologically active plant compounds known as
phytoestrogens.Isoflavones are one such phytoestrogen. They may contain
cancer-fighting properties. This could suppress tumor growth in prostate cancer
cells.
                    </p>
                    <p class="Normal1">
                     While theres still a need for more
conclusive research, the preliminary research reported by the
                     <a href="http://www.cancer.gov/cancertopics/pdq/cam/prostatesupplements/healthprofessional/page8" target="_blank">
                      National Cancer Institute
                     </a>
                     shows a link between the consumption of soy and
reduced levels of prostate-specific antigen (PSA). PSA is a protein produced by
your prostate; the PSA test measures the level of PSA in your blood and is used
as a screening test for prostate cancer. This research also seemed to indicate
soy was more effective when it was eaten in combination with other cancer-fighting
foods.
                    </p>
                   </div>
                  </div>
                  <!-- storyArea -->
                  <div class="imageArea">
                  </div>
                  <!-- imageArea -->
                 </div>
                 <!-- page_5 -->
                </li>
                <li>
                 <div class="box-slide c" id="page_6">
                  <h2 class="slide-title hl-secondary-heading hl-secondary-heading-override-slideshow">
                   Pomegranate Juice
                  </h2>
                  <div class="storyArea">
                   <h2 class="story-title hl-secondary-heading hl-secondary-heading-override-slideshow">
                    Pomegranate Juice
                   </h2>
                   <div class="subhead hl-content-copy hl-content-copy-slideshow-override">
                    <p class="Normal1">
                     Much like red wine or green tea, pomegranate
is a rich source of antioxidants. Its been touted as a miracle fruit in
preventing chronic diseases related to oxidative stress.Scientists think the
antioxidant found in pomegranate works in a seek and destroy method, exclusively
targeting the prostate cancer cells and not the healthy cells.
                     <a href="http://www.cancer.gov/cancertopics/pdq/cam/prostatesupplements/healthprofessional/page6" target="_blank">
                      Studies
                     </a>
                     have found that pomegranate juice and extract, hinder the production of
different prostate cancer cells.
                    </p>
                   </div>
                  </div>
                  <!-- storyArea -->
                  <div class="imageArea">
                  </div>
                  <!-- imageArea -->
                 </div>
                 <!-- page_6 -->
                </li>
                <li>
                 <div class="box-slide c" id="page_7">
                  <h2 class="slide-title hl-secondary-heading hl-secondary-heading-override-slideshow">
                   Fish
                  </h2>
                  <div class="storyArea">
                   <h2 class="story-title hl-secondary-heading hl-secondary-heading-override-slideshow">
                    Fish
                   </h2>
                   <div class="subhead hl-content-copy hl-content-copy-slideshow-override">
                    <p class="Normal1">
                     Polyunsaturated fats, like omega-3s and
omega-6s, are essential fatty acids found exclusively in the diet. Theyre not
synthesized by the body.The traditional Western diet has a lot of omega-6
fatty acids but not many omega-3s.Having a healthier balance of omega-3
and omega-6 fatty acids may help prevent the development and progression of
prostate cancer.
                    </p>
                    <p class="Normal1">
                     In a review published in the
                     <a href="http://www.ajcn.org/cgi/pmidlookup?view=long&amp;pmid=20844069" target="_blank">
                      American Journal of Clinical
Nutrition
                     </a>
                     ,
                     <a href="http://www.ajcn.org/cgi/pmidlookup?view=long&amp;pmid=20844069" target="_blank">
                      researchers found a link between eating fish and a reduced mortality
rate.
                     </a>
                     Try eating fatty fish found in cold
waters to increase your omega-3 intake. These include:
                    </p>
                    <ul>
                     <li>
                      salmon
                     </li>
                     <li>
                      herring
                     </li>
                     <li>
                      mackerel
                     </li>
                     <li>
                      sardines
                     </li>
                     <li>
                      trout
                     </li>
                    </ul>
                   </div>
                  </div>
                  <!-- storyArea -->
                  <div class="imageArea">
                  </div>
                  <!-- imageArea -->
                 </div>
                 <!-- page_7 -->
                </li>
                <li>
                 <div class="box-slide c" id="page_8">
                  <h2 class="slide-title hl-secondary-heading hl-secondary-heading-override-slideshow">
                   Talk to Your Doctor
                  </h2>
                  <div class="storyArea">
                   <h2 class="story-title hl-secondary-heading hl-secondary-heading-override-slideshow">
                    Talk to Your Doctor
                   </h2>
                   <div class="subhead hl-content-copy hl-content-copy-slideshow-override">
                    <p class="Normal1">
                     Check with your doctor when making dietary
changes, especially if youre receiving treatment for prostate cancer. Some
foods may interact with certain drugs and therapies.
                    </p>
                   </div>

   var fireOnce = false;
  require(['base/async'], function( Async ) {

    var waitForSignal = false;

    function GPTRefreshAllAds() {
      // KIMPS
      for( var i = 0; i &lt; adsArrayForKIMP.length; i++) {
        if( window.HL_ga_sendTrackEvent &amp;&amp; adsArrayForKIMP[i] ) {
          try {
            HL_ga_sendTrackEvent({
              gaEventCategory : 'Any Page',
              gaEventAction : 'Key Impression',
              gaEventLabel: adsArrayForKIMP[i] + ": hn.us.hl.sld.x.x.x",
              gaEventNonInteraction: true
            });
          } catch (e) {
            // if an undefined value in array exists, do no log in GA
            // this does not affect ad refresh
          }
        }
      }
      // THIS IS THE ONE CALL FOR ENTIRE SITE
      googletag.cmd.push( function() {
        googletag.pubads().refresh( adsArrayToRefresh );
      });
    }

    Async( arrayPromiseFunctions, function( PromiseFunction, callback) {
      PromiseFunction().then( function( dataObj ){
        if( dataObj ){
          adsArrayToRefresh.push( dataObj.slotObj );
          adsArrayForKIMP.push( dataObj.slotName );
        } else {
          waitForSignal = true;
        }
        callback();
      }, function() {
        callback();
      });
    }, function(){
      if ( waitForSignal ) {
        $( document ).on( 'wslAdNotShow', function() {
          if( !fireOnce ) {
            GPTRefreshAllAds();
            fireOnce = true;
          }
        });
      } else {
        GPTRefreshAllAds();
      }
    });
  });
