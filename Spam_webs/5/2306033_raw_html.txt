             <h3 class="post-title entry-title">
              <a href="http://www.brikasia.com/2014/08/v-neck-t-shirt.html">
               Let's Appreciate V-Neck T-Shirts
              </a>
             </h3>
             <script>
              var ultimaFecha = '8.28.2014';
             </script>
             <h2 class="date-header">
              posted on: 8.28.2014
             </h2>
             <div class="post-header-line-1">
             </div>
             <div class="post-body entry-content">
              <div class="separator" style="clear: both; text-align: center;">
               <a href="http://4.bp.blogspot.com/-8LJcXFP19g8/U_SdMk6yD1I/AAAAAAAAAKM/JF2Kw311m0U/s1600/081414-DisneyPost.png" imageanchor="1" style="margin-left: 0em; margin-right: 1em;">
                <img alt="v neck t shirt, best v neck shirt, madewell slub vneck shirt" border="0" height="640" src="http://4.bp.blogspot.com/-8LJcXFP19g8/U_SdMk6yD1I/AAAAAAAAAKM/JF2Kw311m0U/s1600/081414-DisneyPost.png" title="Madewell Slub V Neck Shirt" width="640"/>
               </a>
              </div>
              Can we just take a moment to appreciate the glorious simplicity of the understated-yet-always-stylish v-neck t-shirt? Like, forrrreal you guys. When I was younger and was a dancer, I remember seeing some older dancers wearing a v-neck shirt on top of their sports bra and dance shorts and thinking to myself, "They are SO cool." Ever since that moment, v-neck shirts have been a staple in my wardrobe. I am a supremely lazy dresser, and knowing I can just slap a v-neck on and have it pair with just about anything in my closet makes me a very happy girl.
              <br/>
              <br/>
              But not all v-neck shirts are made equal. If I had a nickel for every time I blindly ordered a v-neck online and had it fit terribly, I'd have like, six nickels. For me, I'm all about trying to conceal my midsection since I have a very short torso and all the weight I gain tends to go straight to my gut, so I tend to favor oversized and slouchy v-necks. My absolute favorite v-neck is the
              <a href="http://api.shopstyle.com/action/apiVisitRetailer?id=432914267&amp;pid=uid3361-8910231-96" rel="nofollow" target="_blank">
               Madewell Slub V-Neck
              </a>
              , which is featured in the photo above from
              <a href="http://www.instagram.com/brikasiabeauty" rel="nofollow" target="_blank">
               my Instagram
              </a>
              . It's really soft, comfortable, and flattering and doesn't reveal to much cleavage.
              <br/>
              <br/>
              And here's a bonus: one of my favorite digital shorts from Saturday Night Live about dudes wearing v-necks. Slightly inappropriate for you youngin's reading!

               </iframe>

              <br/>
              How do you feel about v-necks? Do you have any other wardrobe essentials?

              &amp;&amp; comment.author.name == config.authorName

          }

          replybox.src = replyUrlParts[0]
              + (commentId ? '&amp;parentID=' + commentId : '')
              + '#' + replyUrlParts[1];
          replyParent = commentId;
        }
      };

      var startThread, targetComment;
      if (/^comment-form_/.test(hash)) {
        startThread = hash.substring('comment-form_'.length);
      } else if (/^c[0-9]+$/.test(hash)) {
        targetComment = hash.substring(1);
      }

      // Configure commenting API:
      var configJso = {
        'maxDepth': config.maxThreadDepth
      };
      var provider = {
        'id': config.postId,
        'data': items,
        'loadNext': paginator,
        'hasMore': hasMore,
        'getMeta': getMeta,
        'onReply': onReply,
        'rendered': true,
        'initComment': targetComment,
        'initReplyThread': startThread,
        'config': configJso,
        'messages': msgs
      };

      var render = function() {
        if (window.goog &amp;&amp; window.goog.comments) {
          var holder = document.getElementById('comment-holder');
          window.goog.comments.render(holder, provider);
        }
      };

      // render now, or queue to render when library loads:
      if (window.goog &amp;&amp; window.goog.comments) {
        render();
      } else {
        window.goog = window.goog || {};
        window.goog.comments = window.goog.comments || {};
        window.goog.comments.loadQueue = window.goog.comments.loadQueue || [];
        window.goog.comments.loadQueue.push(render);
