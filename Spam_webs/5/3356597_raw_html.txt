                    <h3 class="post-title entry-title" itemprop="name">
                     Rules To Live By  Or At Least To TRY To Live By
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-1317182173015601357" itemprop="description articleBody">
                     <span xmlns="">
                      I don't know if any of you watch NCIS. But I do, and it's one of my favorite shows. Anyway, Gibbs has these rules that he tries to live by. It's a long story about how the Rules started, but that's neither here nor there.
                      <br/>
                      <br/>
                      Gibbs's rules gave me an idea for my own rules. Most of these I borrowed from other people. I think some might even be from the show. Most of the time I'll just hear somebody say something, and I think to myself,
                      <i>
                       That's a good rule
                      </i>
                      . And I add it to the list. The numbers don't really mean anything, because what I think is the most important one is number eighteen. (But I still break it all the timeand immediately regret it.)
                      <br/>
                      <br/>
                      So read and enjoy  oh, and feel free to take some and make them your own.
                      <br/>
                     </span>
                     <br/>
                     <ol>
                      <li>
                       <span xmlns="">
                        Read.
                        <br/>
                       </span>
                      </li>
                      <span xmlns="">
                       <li>
                        Never EVER settle for second best.
                        <br/>
                       </li>
                       <li>
                        Never take anything for granted.
                        <br/>
                       </li>
                       <li>
                        Imagination and knowledge are equally important.
                        <br/>
                       </li>
                       <li>
                        Never date an artist.
                        <br/>
                       </li>
                       <li>
                        Thou shalt not panic.
                        <br/>
                       </li>
                       <li>
                        Go somewhere new every chance you get.
                        <br/>
                       </li>
                       <li>
                        Comfort zones are for pussies.
                        <br/>
                       </li>
                       <li>
                        Never nap while day-drinking.
                        <br/>
                       </li>
                       <li>
                        Never doubt yourself.
                        <br/>
                       </li>
                       <li>
                        Always designate a safe word.
                        <br/>
                       </li>
                       <li>
                        Driver picks the music.
                        <br/>
                       </li>
                       <li>
                        Cry only after everyone has gone.
                        <br/>
                       </li>
                       <li>
                        Dress codes are for people who have no style.
                        <br/>
                       </li>
                       <li>
                        Share your knowledge.
                        <br/>
                       </li>
                       <li>
                        Experiment in the kitchen and the bedroom.
                        <br/>
                       </li>
                       <li>
                        Being alone is better than a lot of other things.
                        <br/>
                       </li>
                       <li>
                        Never pass up an opportunity to pee.
                        <br/>
                       </li>
                       <li>
                        Always have a project.
                        <br/>
                       </li>
                       <li>
                        Be generous.
                        <br/>
                       </li>
                       <li>
                        Go to parties.
                        <br/>
                       </li>
                       <li>
                        Fail with pride.
                        <br/>
                       </li>
                       <li>
                        Read newspapers only for the gossip and the comics.
                        <br/>
                       </li>
                       <li>
                        Always keep a dictionary close to hand.
                        <br/>
                       </li>
                       <li>
                        I come first.
                        <br/>
                       </li>
                       <li>
                        Be creative at every opportunity.
                        <br/>
                       </li>
                       <li>
                        Be grateful for your friends. They put up with a lot of your bullshit.
                        <br/>
                       </li>
                       <li>
                        Be early.
                        <br/>
                       </li>
                       <li>
                        Don't worry, it only seems kinky the first time.
                        <br/>
                       </li>
                       <li>
                        Be nice to waiters.
                        <br/>
                       </li>
                       <li>
                        Do your best or do nothing.
                        <br/>
                       </li>
                       <li>
                        Laugh at your own jokes.  Somebody has to.
                        <br/>
                       </li>
                       <li>
                        Mistakes are okay.  Regrets are not.
                        <br/>
                       </li>
                       <li>
                        Smile.  It makes you seem friendly.
                       </li>
                      </span>
