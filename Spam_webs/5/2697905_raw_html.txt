       <h1 id="page_title">
        Pope falls victim to Catalan 'shitter' tradition
       </h1>
       <div class="clr">
       </div>
       <p class="small">
        Published: 16 Nov 2013 10:30 GMT+01:00
       </p>
       <div>
        <a href="#" onclick="popupShareWindow('https://www.facebook.com/sharer/sharer.php?u=http://www.thelocal.it/page/view/pope-francis-latest-victim-of-catalan-caganer-tradition', 520, 340); return false;">
         <img alt="Facebook" src="/assets/common/image/article-social/facebook.png" style="border: 0; height: 20px; width: 60px; padding: 0; margin: 0 8px 0 0;"/>
        </a>
        <a href="#" onclick="popupShareWindow('https://twitter.com/intent/tweet?text=Pope+falls+victim+to+Catalan+%27shitter%27+tradition&amp;via=TheLocalItaly&amp;url=http://www.thelocal.it/20131116/pope-francis-latest-victim-of-catalan-caganer-tradition', 520, 440); return false;">
         <img alt="Twitter" src="/assets/common/image/article-social/twitter.png" style="border: 0; height: 20px; width: 60px; padding: 0; margin: 0 8px 0 0;"/>
        </a>
        <a href="#" onclick="popupShareWindow('https://plus.google.com/share?url=http://www.thelocal.it/page/view/pope-francis-latest-victim-of-catalan-caganer-tradition', 520, 490); return false;">
         <img alt="Google+" src="/assets/common/image/article-social/google.png" style="border: 0; height: 20px; width: 60px; padding: 0; margin: 0 0 0 0;"/>
        </a>
        <a href="http://reddit.com/submit?url=http://www.thelocal.it/20131116/pope-francis-latest-victim-of-catalan-caganer-tradition&amp;title=">
         <img alt="reddit" src="/assets/common/image/article-social/reddit.png" style="border: 0; height: 20px; width: 60px; padding: 0; margin: 0 0 0 8px;"/>
        </a>
       </div>
       <p class="articleTeaser">
        <strong>
         <a href="/page/view/tag/Pope Francis">
          Pope Francis
         </a>
         was caught with his chasuble rolled up at the waist in the latest batch of Spain's "caganers", holiday season figurines which depict bare-bottom celebrities defecating.
        </strong>
       </p>
       <ul style="list-style:disc; padding-left:20px; line-height:1.3em;">
       </ul>
       <div style="text-align: center;">
        <!-- it.under.ra -->
        <div id="337026">
        </div>
        <script>
         if (!window.OX_ads) { OX_ads = []; }
  OX_ads.push({ "slot_id":"337026", "auid":"337026" });
        </script>
       </div>
       <div class="articleContent">
        <p>
         <span style="font-family: arial, sans-serif; font-size: 13px;">
          The ceramic statuettes have been sold in the northeastern Catalonia region
         </span>
         <span style="font-family: arial, sans-serif; font-size: 13px;">
          since the 18th century.
         </span>
        </p>
        <p>
         <span style="font-family: arial, sans-serif; font-size: 13px;">
          Catalonians hide caganers ("shitters" in
          <a href="/page/view/tag/Catalan">
           Catalan
          </a>
          ) in Christmas Nativity
         </span>
         <span style="font-family: arial, sans-serif; font-size: 13px;">
          scenes and invite friends to find them. The figures symbolise fertilisation,
         </span>
         <span style="font-family: arial, sans-serif; font-size: 13px;">
          hope and prosperity for the coming year.
         </span>
        </p>
        <p>
         <span style="font-family: arial, sans-serif; font-size: 13px;">
          Among the new caganers expected to fly off the shelves are La Liga's star
         </span>
         <span style="font-family: arial, sans-serif; font-size: 13px;">
          rookies: Real Madrid's new Welsh winger Gareth Bale and Barcelona's Brazilian
         </span>
         <span style="font-family: arial, sans-serif; font-size: 13px;">
          prodigy Neymar.
         </span>
        </p>
        <p>
         <span style="font-family: arial, sans-serif; font-size: 13px;">
          "There's more sport and less politics, which once in a while is not
         </span>
         <span style="font-family: arial, sans-serif; font-size: 13px;">
          unpleasant," Marc Alos, whose family is the leading "shitter" producer, told
         </span>
         <span style="font-family: arial, sans-serif; font-size: 13px;">
          AFP.
         </span>
        </p>
        <p>
         <span style="font-family: arial, sans-serif; font-size: 13px;">
          <a href="/page/view/tag/Pope Francis">
           Pope Francis
          </a>
          was also joined by his predecessor Benedict XVI, South African
         </span>
         <span style="font-family: arial, sans-serif; font-size: 13px;">
          anti-apartheid icon Nelson Mandela and reggae legend Bob Marley in this year's
         </span>
         <span style="font-family: arial, sans-serif; font-size: 13px;">
          new releases.
         </span>
        </p>
       </div>
       <div class="author">
        <p>

          news@thelocal.it
         </a>
         )
        </p>
