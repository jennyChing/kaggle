          <div id="details">
           <div itemprop="description">
            <p>
             This course is designed to introduce Java Developers to the development and testing of enterprise applications that comply with Java 2 Enterprise Platform standards (JEE Specifications). The course also covers Enterprise Java Beans Component Architecture
 including how to implement Session Beans and Entity Beans and how to connect Entity Beans to the database also known as Session Faade. Students gain hands-on experience through the labs that build end-to-end, distributed application which contains reusable
 components to be deployed in the JEE platform runtime environment.
            </p>
            <p>
             This course also covers best practices for building JEE Web applications, including the use of Model-View-Controller (MVC) framework to separate concerns among the Java Beans, Java Server Pages and Servlets in an application. The course also describes deployment
 issues pertinent to the developer.
            </p>
            <p>
            </p>
            <p>
             <b>
              After completing this course, students should be able to:
             </b>
            </p>
            <p>
             Describe the JEE component model and its use in building server-side applications.
            </p>
            <p>
             Understand and apply component model of the J2EE platform
            </p>
            <p>
             Understand and apply the core JEE technologies - Servlets, JSPs, EJBs
            </p>
            <p>
             Use JavaBeans, JSPs, and Servlets in accordance with the Model-View-Controller (MVC) programming model
            </p>
            <p>
             Develop web applications using Servlets and JavaServer Pages (JSPs)
            </p>
            <p>
             Develop business components using Enterprise JavaBeans (EJB)
            </p>
            <p>
              Identify the best practices needed to design and build Web applications
            </p>
            <p>
              Access database using JDBC
            </p>
