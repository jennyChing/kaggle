                    <h3 class="post-title entry-title" itemprop="name">
                     Homemade Self-Hardening Clay
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-7412101624044920110" itemprop="articleBody">
                     <a data-pin-config="beside" data-pin-do="buttonPin" href="//pinterest.com/pin/create/button/?url=http%3A%2F%2Fwww.redflycreations.com%2F2013%2F07%2Fhomemade-self-hardening-clay.html&amp;media=http%3A%2F%2Fmedia-cache-ak3.pinimg.com%2F736x%2F5c%2Fb2%2F95%2F5cb295734fa39b2d084bf60a64db636f.jpg&amp;description=Homemade%20Self-Hardening%20Modeling%20Clay%20from%20redflycreations.com.%20%20Just%20a%205%20minute%20recipe!%3F">

                     </div>
                     <div class="" style="clear: both; text-align: left;">
                      Kids love to play with clay. You can create as much as the imagination allows. Self-hardening clay is extra fun because your creations can be preserved forever. Plus, you get to further personalize by painting with bold bright colors!
                     </div>
                     <br/>
                     <div style="text-align: center;">
                      <span style="font-size: x-large;">
                       <b>
                        Recipe for Self-Hardening Clay
                       </b>
                      </span>
                      <br/>
                      <span style="font-size: x-large;">
                       <b>
                        <br/>
                       </b>
                      </span>
                     </div>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://www.redflycreations.com/" style="clear: right; float: right; margin-bottom: 1em; margin-left: 1em; text-align: center;">
                       <img border="0" height="112" src="http://4.bp.blogspot.com/-rUdg71rnI_4/UbeXb6JE5ZI/AAAAAAAAJmQ/ZyDKlSH7ISU/s200/Summer+Fun+Main2.jpg" width="200"/>
                      </a>
                     </div>
                     <br/>
                     4 c flour
                     <br/>
                     1 tsp alum (found in the baking isle)
                     <br/>
                     1 1/2 c salt
                     <br/>
                     1 1/2 c water
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     Mix all the ingredients together. Divide into sections. If you are not going to use it right away, you can store it in a sealed zip lock bag in the fridge.
                     <br/>
                     <div>
                      <br/>
                     </div>
                     <table align="center" cellpadding="0" cellspacing="0" class="tr-caption-container" style="margin-left: auto; margin-right: auto; text-align: center;">

                          <img border="0" height="400" src="http://1.bp.blogspot.com/-PhGJLnBt-dw/UdWuGahrt2I/AAAAAAAAKOU/jy3vSa9h734/s400/Homemade+Molding+Clay.jpg" width="265"/>
                         </a>
