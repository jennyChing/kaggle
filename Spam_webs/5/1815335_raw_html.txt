         Create Web 2.0 Stickers

        <p>
         <img alt="Web 2.0 Stickers Thumbnail" class="alignleft" src="http://vectips.com/wp-content/uploads/2008/02/stky_th.jpg"/>
        </p>
        <p class="intro">
         I wonder how long the Web 2.0 style will be around.  All those shiny graphics are great to look at and create, but it cant last.  In some respects, its good to capitalize on the trend, but it the same respect its good to explore new techniques and styles. Regardless, there is enough demand with the Web 2.0 style to justify a tutorial on some nifty Web 2.0 Stickers!
        </p>
        <p>
         <span id="more-42">
         </span>
        </p>
        <h4>
         Notes
        </h4>
        <p>
         <small class="note">
          This tutorial was created in Adobe Illustrator CS3.  Most, if not all, functions should be available in Illustrator 10 and up.  Functions specific to Illustrator CS3 will be noted.
         </small>
        </p>
        <p>
         <small class="note">
          Keyboard shortcuts are displayed in orange.  is displayed for the Command key (mac), with the Ctrl key being the Windows equivalent (not displayed).
         </small>
        </p>
        <h3 class="step">
         Step 1: Create The Sticker and Fold Shapes
        </h3>
        <p>
         Create a box with rounded corners with the
         <strong>
          Rounded Rectangle Tool.
         </strong>
         Next, create another box that is a little smaller.  Grab a corner of the second box with the
         <strong>
          Selection Tool
          <span class="orange">
           (v)
          </span>
         </strong>
         , hold
         <strong>
          Shift,
         </strong>
         and rotate the box 45 degrees.  Place the second box on the bottom right corner of the first box where you would like the fold of the sticker.
         <strong>
          Copy
          <span class="orange">
           (c)
          </span>
         </strong>
         both boxes and
         <strong>
          Paste In Front
          <span class="orange">
           (f)
          </span>
         </strong>
         .  Select one of the original boxes and one of the rotated boxes and
         <strong>
          Intersect
         </strong>
         and
         <strong>
          Expand
         </strong>
         the shapes, creating the fold. Take the second copy of each box and
         <strong>
          Subtract
         </strong>
         and
         <strong>
          Expand
         </strong>
         them, creating the sticker. Now, select the fold, grab a corner, hold
         <strong>
          Shift
         </strong>
         and rotate 180 degrees.
        </p>
        <p>
         <img alt="Create The Sticker and Fold Shape" class="centered" src="http://vectips.com/wp-content/uploads/2008/02/stky_1.jpg"/>
        </p>
        <h3 class="step">
         Step 2: Create The Shimmer Shape
        </h3>
        <p>
         Use the
         <strong>
          Ellipse Tool
          <span class="orange">
           (l)
          </span>
         </strong>
         and draw a circle half way down the sticker overlapping the left, top, and right corners of the sticker shape generously.
         <strong>
          Copy
          <span class="orange">
           (c)
          </span>
         </strong>
         the sticker and
         <strong>
          Paste In Front
          <span class="orange">
           (f)
          </span>
         </strong>
         . Take one sticker shape and the circle and
         <strong>
          Intersect
         </strong>
         and
         <strong>
          Expand
         </strong>
         , creating the shimmer shape.
        </p>
        <p>
         <img alt="Create The Shimmer Shape" class="centered" src="http://vectips.com/wp-content/uploads/2008/02/stky_2.jpg"/>
        </p>
        <h3 class="step">
         Step 3: Create The Shiny Effect
        </h3>
        <p>
         In this step I am briefly going to explain using gradients to create the shiny effect.  If you need a more in depth explanation, refer to
         <strong>
          Step 3
         </strong>
         in my previous post
         <a href="http://vectips.com/tutorials/shiny-buttons-without-blends-or-transparencies-in-illustrator/" target="_blank">
          Shiny Buttons Without Blends or Transparencies
         </a>
         .
        </p>
        <p>
         Use the follow steps for the shiny effect.
        </p>
        <ol>
         <li>
          Select the shimmer shape and create a
          <strong>
           Linear
          </strong>
          gradient with the first swatch in the gradient 20% Cyan and second swatch in the gradient 100% cyan.
         </li>
         <li>
          Adjust the gradient with the
          <strong>
           Gradient Tool
           <span class="orange">
            (g)
           </span>
          </strong>
          .
         </li>
         <li>
          Remove the borders.
         </li>
         <li>
          Select the sticker shape and create a
          <strong>
           Linear
          </strong>
          gradient with the first swatch 100% Cyan and the second swatch 100% Cyan and 50% Magenta.
         </li>
         <li>
          Adjust the gradient with the
          <strong>
           Gradient Tool
           <span class="orange">
            (g)
           </span>
          </strong>
          .
         </li>
         <li>
          Remove the borders.
         </li>
        </ol>
        <p>
         Again, if that was too brief, refer to
         <strong>
          Step 3
         </strong>
         in the
         <a href="http://vectips.com/tutorials/shiny-buttons-without-blends-or-transparencies-in-illustrator/" target="_blank">
          Shiny Button post
         </a>
         .
        </p>
        <p>
         <img alt="Creating the Shinny Effect" class="centered" src="http://vectips.com/wp-content/uploads/2008/02/stky_3.jpg"/>
        </p>
        <h3 class="step">
         Step 4: Adding Detail to the Fold
        </h3>
        <p>
         Now it is time to give the fold some depth.  Select the fold shape and create a
         <strong>
          Linear
         </strong>
         gradient. Keep white as your first swatch, select 20% black as the second swatch and remove the border.  When you adjust this gradient, start in the middle of the fold shape, hold
         <strong>
          Shift
         </strong>
         , and drag out to the bottom right of the fold at a 45 degree angle.
        </p>
        <p>
         Now the fold is looking pretty good, but to give it a little more depth we can add a shadow to the fold.  Select the fold shape, hold down the
         <strong>
          Alt/Option
         </strong>
         button and drag the shape slightly down and to the left.
         <strong>
          Copy
          <span class="orange">
           (c)
          </span>
         </strong>
         the the sticker shape and
         <strong>
          Paste In Front
          <span class="orange">
           (f)
          </span>
         </strong>
         .  Select the newly copied sticker shape and the copied fold shape and
         <strong>
          Intersect
         </strong>
         and
         <strong>
          Expand
         </strong>
         . Change the color of the fold shadow to 100% Cyan and 50% Magenta. Select the original fold shape and
         <strong>
          Bring  It To The Front
          <span class="orange">
           ( Shift ]).
          </span>
         </strong>
         Now you have a fancy Web 2.0 sticker!
        </p>
        <p>
         <img alt="Adding Detail to the Fold" class="centered" src="http://vectips.com/wp-content/uploads/2008/02/stky_4.jpg"/>
        </p>
        <h3 class="step">
         Step 5: Expand
        </h3>
        <p>
         These stickers look great with icons or text on them, just experiment!  Try adding different colors, shapes, or whatever.  If you are to lazy or need some sticky icons quick, go
         <a href="http://www.istockphoto.com/file_closeup.php?id=5480613" target="_blank">
          check out
         </a>
         some I created on iStock!
        </p>
        <p>
         <img alt="Expand" class="centered" src="http://vectips.com/wp-content/uploads/2008/02/stky_5.jpg"/>
        </p>
