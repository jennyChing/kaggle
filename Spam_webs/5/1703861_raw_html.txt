       <div class="best-section">

       <p>
        The Vizio M55-C2 is the top choice for an affordable 4k TV. It has great contrast and uniformity, little input lag, the motion interpolation feature, and good 1080p upscaling capabilities. It also has full-array local dimming backlighting, though the local dimming doesnt work as well as it should.
       </p>
       <p>
        It has a few other downsides. The firmware is pretty buggy, which causes the interpolation feature to turn itself on at random. It doesnt upscale lower resolutions as well as the JU7100 does, so youll need to increase sharpness to make that kind of content look good. More importantly, fast-moving objects displayed on the M have a lengthy trail, so this isnt a great TV for gamers or sports fans.
       </p>
       <p>
        For movies and regular TV, though, this is a great deal.
       </p>

       <p>
        The Sony XBR55X850C is very nearly as good as the Samsung JU7100. It has good contrast, good uniformity, and is able to handle fast movement with no issues. Its main failings are that its input lag is a bit higher than is ideal for gamers, and also that its screen is only semi-gloss, vs the glossy screen you get with higher-end Samsung TVs.

        <span class="value score score-good score-medium">
         8.3
        </span>
        Samsung UN55JU7100
       </div>
       <p>
        The Samsung UN55JU7100 has good overall picture quality, with little motion blur, and it also has low input lag (26.1 ms) and the option to interpolate motion. Because it doesn't have the same color capabilities, it's not quite as good as the JS8500, but it's a better choice for gamers looking to get the lowest input lag they can.
        <a href="/reviews/tv/lcd-led/samsung/ju7100" style="line-height: 20.7999992370605px;">
         See our review
        </a>

        <span class="value score score-good score-medium">
         8.2
        </span>
        Samsung UN55JU6500
       </div>
       <p>
        The Samsung UN55JU6500 has similar picture quality to the JU7100, but with more motion blur. Its only a 60 hz TV, and it doesnt have a 24p playback mode. That means that Blu-ray content and other 24p sources will have judder. It also doesnt have the smart remote. If you dont mind those downsides, consider this cheaper alternative.

   if (e.style.zIndex == "-1") {

function expandSideMenu(e) {
   var list = e.nextElementSibling;
   if (list.style.display == 'block')
      list.style.display = 'none';
   else
      list.style.display = 'block';
   var icon = e.firstElementChild;
   if (icon.className == "topmenutreeclose") {
      icon.className = "topmenutreeopen";
   }
   else if (icon.className == "topmenutreeopen") {
      icon.className = "topmenutreeclose";
   }
   
}
