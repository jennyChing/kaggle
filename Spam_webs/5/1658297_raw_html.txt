      <div class="entrytext">
       <p>
        Recently, a lot of my favorite knitbloggers have been sucked into the world of weaving, and have produced piece after
        <a href="http://www.marniemaclean.com/words/2009/03/keeping_it_simp.html" rel="nofollow">
         gorgeous
        </a>
        ,
        <a href="http://www.knitandtonic.net/knitandtonic/2009/03/so-i-showed-this-to-hwwv-yesterday-and-he-said-didnt-you-take-up-a-new-craft-right-around-the-last-time-you-wrote-a-book-t.html" rel="nofollow">
         drool-worthy
        </a>
        , jealousy-inducing piece.  Whats a broke college student to do?
        <br/>
        <br/>
       </p>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003f0h1g/g39" rel="nofollow">
         <img alt="My First Weaving" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003f0h1g/s320x240" width="180"/>
        </a>
        <br/>
        <strong>
         My First Weaving
        </strong>
       </center>
       <br/>
       <br/>
       This project is all about making something on the cheap and upcycling,  two very hot topics in the current crafting universe.  I was inspired by
       <a href="http://www.threadbanger.com/episode/THR_20090102" rel="nofollow">
        this Threadbanger video
       </a>
       , which got me thinking about weaving without a painfully expensive loom.  I did things a touch differently, however, and so have written up a tutorial.
       <br/>
       <br/>
       <a name="cutid1">
       </a>
       <br/>
       <br/>
       This project has three parts: the yarn, the loom, and the weaving.
       <br/>
       <br/>
       <center>
        <b>
         The Yarn:
        </b>
       </center>
       <br/>
       <br/>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003dcfts/g39" rel="nofollow">
         <img alt="Thrift Store Raid" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003dcfts/s320x240" width="320"/>
        </a>
        <br/>
        <strong>
         Thrift Store Raid
        </strong>
       </center>
       <br/>
       <br/>
       I began by raiding the thrift store one warm Saturday morning.  To the great dismay of XXL thrift-store patron men, I absconded with just about every green XXL mens t-shirt the thrift store had, plus one large white shirt.  Then I turned these shirts into yarn.
       <br/>
       <br/>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003ddes5/g39" rel="nofollow">
         <img alt="Yarn!" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003ddes5/s320x240" width="180"/>
        </a>
        <br/>
        <strong>
         Yarn!
        </strong>
       </center>
       <br/>
       <br/>
       To turn a t-shirt into yarn:
       <br/>
       <br/>
       1) Unpick the bottom hem of the t-shirt (if youre looking to make the most of your fabric), or simply cut it off (if you dont mind losing an inch or so).
       <br/>
       <br/>
       2) Cut the shirt into a single long, narrow strip (I did mine about a quarter of an inch wide) by spiraling from the bottom up.  Stop at the armpits, where the shirt is no longer continuous.
       <br/>
       <br/>
       3) Before you wind it up into a ball stretch your strip lightly between two hands; the knit fabric will then curl in on itself and create your t-shirt yarn.
       <br/>
       <br/>
       <center>
        <b>
         The Loom
        </b>
       </center>
       <br/>
       <br/>
       For this one, I raided my boyfriends pile o scrap wood.  Should you lack a pile of scrap wood, you can also ask your local lumber yard if they have any damaged or warped wood scraps theyre throwing out.  Barring that, you need little enough wood (and quality doesnt matter) that it should come cheap. You will also need a hammer and a box of cheap nails (I used old wire brads).  Finally, youll need a big workspace that you dont mind sticking tape on- your desk, your kitchen table, your kitchen counter, the floor, or in my case a folding table used specifically for crafty ventures.  Youll also need a roll of strong tape (duct would probably work best, but all I had was packing).
       <br/>
       <br/>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003eap6y/g39" rel="nofollow">
         <img alt="Marked" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003eap6y/s320x240" width="180"/>
        </a>
        <br/>
        <strong>
         Marked
        </strong>
       </center>
       <br/>
       <br/>
       1) Take two chunks of wood that are as long (or perhaps a little longer) as the widest item you plan to weave, and draw a line longitudinally down their centers.
       <br/>
       <br/>
       2)Mark every half an inch on this line.
       <br/>
       <br/>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003eb0sy/g39" rel="nofollow">
         <img alt="Loom Parts" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003eb0sy/s320x240" width="320"/>
        </a>
        <br/>
        <strong>
         Loom Parts
        </strong>
       </center>
       <br/>
       <br/>
       3) Hammer a nail in every half an inch on both pieces of wood, until youve got enough nails for how wide you want your scarf.  Said scarf will likely end up a bit narrower than the nails, depending on how tight you pull it, so keep that in mind.  It helps if you hammer them in at an angle, so that when you line both pieces of wood up, the nails lean away from each other.
       <br/>
       <br/>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003ecfr2/g39" rel="nofollow">
         <img alt="Plus 10 inches" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003ecfr2/s320x240" width="180"/>
        </a>
        <br/>
        <strong>
         Plus 10 inches
        </strong>
       </center>
       <br/>
       <br/>
       4)  Line your pieces of wood up on your workspace with the nails leaning away from each other as far apart as you want your scarf to be long, add about ten inches (trust me on this one- I didn't, and it sucked), then tape them down with a vengeance.  Voila!  A loom.  And hey, its length adjustable if you have enough tape!
       <br/>
       <br/>
       <center>
        <b>
         Weaving Your Scarf:
        </b>
       </center>
       <br/>
       <br/>
       Now to put it all together.  Some weaving lingo, courtesy of
       <a href="http://en.wikipedia.org/wiki/Weaving" rel="nofollow">
        Wikipedia
       </a>
       : the
       <i>
        warp
       </i>
       threads are the threads that run from nail to nail on your loom.  The
       <i>
        weft
       </i>
       threads are the filling- the ones you weave through the warp.  In this case, to make it obvious what I was doing, I used the large white t-shirt for my warp and green shirts for my weft.  This scarf will be done in a plain weave.
       <br/>
       <br/>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003edps4/g39" rel="nofollow">
         <img alt="Tie on the warp" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003edps4/s320x240" width="180"/>
        </a>
        <br/>
        <strong>
         Tie on the warp
        </strong>
       </center>
       <br/>
       <br/>
       1) Take your ball of weft yarn, and tie its end to the nail furthest from you on the left (or on the right, or closest on the left or right- tie it to one of the corners).
       <br/>
       <br/>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003ee400/g39" rel="nofollow">
         <img alt="Stringing the loom" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003ee400/s320x240" width="180"/>
        </a>
        <br/>
        <strong>
         Stringing the loom
        </strong>
       </center>
       <br/>
       <br/>
       2)  Wrap the yarn with enough tension that it doesnt sag in the middle around the nail directly opposite of the first, then come back and wrap it around the second nail next to the one with the knot, then back to the second on the other side, then back and forth until youve got as many nails wrapped as youd like.
       <br/>
       <br/>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003ef8ey/g39" rel="nofollow">
         <img alt="Tie off the warp" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003ef8ey/s320x240" width="320"/>
        </a>
        <br/>
        <strong>
         Tie off the warp
        </strong>
       </center>
       <br/>
       <br/>
       3)Tie another knot to secure your yarn on the last nail.
       <br/>
       <br/>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003eh9gr/g39" rel="nofollow">
         <img alt="Plain Weave" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003eh9gr/s320x240" width="180"/>
        </a>
        <br/>
        <strong>
         Plain Weave
        </strong>
       </center>
       <br/>
       <br/>
       4) Thread your first piece of yarn through where you want your scarf to start, leaving a bit of space -say, five inches worth, and trust me more is better- to tie off the end of the scarf when you're done (and form the fringe, on this scarf).  When you thread your weft through the warp (that so does not sound like English), make sure to go over one warp thread and than under the next.
       <br/>
       <br/>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003ekp42/g39" rel="nofollow">
         <img alt="Every Other Thread" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003ekp42/s320x240" width="320"/>
        </a>
        <br/>
        <strong>
         Every Other Thread
        </strong>
       </center>
       <br/>
       <br/>
       5)  Bring your weft yarn back through your warp threads again, this time going under the warp threads you went over before and over those you went under.  The easiest way to do this is to pick up every other thread with one hand (the threads you want the weft to go under), then put the whole ball of yarn through the space.
       <br/>
       <br/>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003epdcc/g39" rel="nofollow">
         <img alt="Pack it down" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003epdcc/s320x240" width="180"/>
        </a>
        <br/>
        <strong>
         Pack it down
        </strong>
       </center>
       <br/>
       <br/>
       6) Use the flat of your hand, still between the warp threads that went under and over your last bit of weft, to push the weft neatly up against the rest of the weaving.
       <br/>
       <br/>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003eg82x/g39" rel="nofollow">
         <img alt="Helper" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003eg82x/s320x240" width="180"/>
        </a>
        <br/>
        <strong>
         Helper
        </strong>
       </center>
       <br/>
       <br/>
       7) Repeat steps 4 and 5 until your scarf is as long as you want it.  It goes surprisingly fast for someone used to knitting, even with a loom as basic and bare-bones as this.
       <br/>
       <br/>
       8)  If you want to start any new colors, you can knot or sew your new color to the tail of your previous color and continue that way.  Or maybe you can just weave the end in with the rest of the weft.  I don't know; I haven't tried yet.
       <br/>
       <br/>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003eryt9/g39" rel="nofollow">
         <img alt="Near the End" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003eryt9/s320x240" width="180"/>
        </a>
        <br/>
        <strong>
         Near the End
        </strong>
       </center>
       <br/>
       <br/>
       9) As you get towards the end you're going to hit a point where you can't fit your yarn ball between the threads anymore.  That's okay- you added ten inches to your goal length anyway, right?  And you want enough warp left to tie things off.
       <br/>
       <br/>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003esfwh/g39" rel="nofollow">
         <img alt="Cut it off" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003esfwh/s320x240" width="180"/>
        </a>
        <br/>
        <strong>
         Cut it off
        </strong>
        <br/>
        <br/>
        <a href="http://pics.livejournal.com/corvustristis/pic/003etysw/g39" rel="nofollow">
         <img alt="One over, one under" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003etysw/s320x240" width="180"/>
        </a>
        <br/>
        <strong>
         One over, one under
        </strong>
        <br/>
        <br/>
        <a href="http://pics.livejournal.com/corvustristis/pic/003ew0pa/g39" rel="nofollow">
         <img alt="Knot" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003ew0pa/s320x240" width="320"/>
        </a>
        <br/>
        <strong>
         Knot
        </strong>
       </center>
       <br/>
       <br/>
       10) To finish the thing, leave a long tail on your weft yarn and cut the warp yarns off of the loom, tying knots in the warp yarn as you go to secure the weaving and create a fringe.  Make sure each knot has a piece of the warp that went OVER the last weft and a piece that went under.
       <br/>
       <br/>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003exk9y/g39" rel="nofollow">
         <img alt="Tie the weft in" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003exk9y/s320x240" width="320"/>
        </a>
        <br/>
        <strong>
         Tie the weft in
        </strong>
       </center>
       <br/>
       <br/>
       11) Tie the tail of the weft into the nearest bit of fringe to secure it, or weave it in as you would a new piece of yarn, or sew it to itself.  You may be able to sew or weave in the weft ends too if you dont want a fringe- I havent tried it myself.  If you do, let me know how it goes!
       <br/>
       <br/>
       <center>
        <a href="http://pics.livejournal.com/corvustristis/pic/003eyp9f/g39" rel="nofollow">
         <img alt="Ta da!" border="0" height="240" src="http://pics.livejournal.com/corvustristis/pic/003eyp9f/s320x240" width="180"/>
        </a>
        <br/>
        <strong>
         Ta da!
        </strong>
       </center>
       <a name="cutid1-end">
       </a>
       <br/>
       <br/>
       It took me less than two balls of XXL t-shirt yarn for this project, so I've got plenty more to play with from the thrift store raid.
       <br/>
       <br/>
       Now that I've got a grounding with the basic idea, I have lots and lots of variations I'd like to try.  For example:
       <br/>
       <blockquote>
        - Vary the size of the warp and weft yarns, both generally and in relation to each other
        <br/>
        - Different weaving patterns.  Perhaps Herringbone?
        <br/>
        - Different fabrics (for less "old-school-nylon-loop-potholder look" and more "funky modern fashion" look)
        <br/>
        - Don't stretch the t-shirt yarn
        <br/>
        - Make the distance between weft threads larger
        <br/>
        - Weave looser
       </blockquote>
       <br/>
       <br/>
       If you like this tutorial, please share it with your friends!  Also, Id love to see what you make!
