         <div id="description-text">
          #Dare to #dazzle with this #radiant #Titanium    #Barbell from #Beltane Moon
          <br/>
          <br/>
          Bar~
          <br/>
          Material: titanium
          <br/>
          Gauge: 14 (1.6mm)
          <br/>
          Length: 5/8"
          <br/>
          Threading: internal (threads inside bar - see photo#3)
          <br/>
          Color: electric blue (blue with purple haze overlay) with gold titanium colored stripes
          <br/>
          <br/>
          Balls~
          <br/>
          Material: titanium
          <br/>
          Size: approx 8mm
          <br/>
          Threading: internal (threads inside bar - see photo#3)
          <br/>
          Color: electric blue (blue with purple haze overlay)
          <br/>
          <br/>
          Titanium jewelry is lightweight (around 60% the weight of stainless steel given the same volume), it is highly corrosion resistant and less likely to react with body fluids, is not magnetic, it can be anodized to create a layer of colored oxide on the surface.
          <br/>
          <br/>
          Based on your pH, titanium coloration may loose it's brilliance over time. Smoking, alcohol and certain acidic foods may contribute to fading of color. Loss of color will not affect your piercing. Jewelers may be able to change the color for you through a process called anodizing.
          <br/>
          <br/>
          The internal threading is another outstanding feature to quality body jewelry.
          <br/>
          <br/>
          Ensure a secure fit on top and bottom ball by gently tightening or securing the threading daily.
          <br/>
          <br/>
          The threading is of the highest caliber and machined from one piece - not press fitted or soldered on, making the quality above the rest.
          <br/>
          <br/>
          Check secure threading by tightening both balls daily. Turn right to tighten,. left to loosen. Do not over-tighten as this may cause threading to strip.
          <br/>
          <br/>
          Clean jewelry often: use a soft toothbrush to remove plaque buildup from the jewelry daily. Rinse with  Biotene mouth wash and dry with disposable paper towel.
          <br/>
          <br/>
          Color on monitor may also vary according to settings. Questions should be addressed prior to placing order.
          <br/>
          <br/>
          Please consider putting a towel in the cleaning area should a ball drop while cleaning. It is easier to find in a towel than down a drain or rolling across the floor.
          <br/>
          <br/>
          vv
          <br/>
          <br/>
          AUTOCLAVE STERILIZATION
          <br/>
          <br/>
          () Q. What is an autoclave?
          <br/>
          <br/>
          (--) A. An autoclave is an instrument used to sterilize equipment and supplies by subjecting them to high pressure saturated steam at 121 C for around 1520 minutes
          <br/>
          <br/>
          () Q. You have your own autoclave?
          <br/>
          <br/>
          (--) A. Yes.
          <br/>
          <br/>
          Read more about this process on my blog:
          <br/>
          http://beltanemoon.blogspot.com/2012/03/autoclave-sterilization.html
          <br/>
          <br/>
          All jewelry is inspected prior to autoclaving (sterilization).
          <br/>
          <br/>
          Sterilized jewelry is sealed in a professional autoclave pouch with a processing indicator printed on the outside. No extra charge for this one-time processing service.
          <br/>
          <br/>
          Jewelry is shipped first class mail w/ tracking number for locating your item during transit. If a different method of shipping is preferred, please contact Beltane Moon before ordering.
          <br/>
          <br/>
          In the unlikely event your item is damaged in transit, please contact Beltane Moon w/in 72 hours of delivery.
          <br/>
          <br/>
          Due to monitor color settings, the actual color may slightly vary. Please address questions regarding color and/or fit prior to purchase.
