                    <h3 class="post-title entry-title">
                     ResQWalk Makes It Easy to Help Shelter Animals While You Walk
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                     <div style="float:left;padding:4px;">
                      <a name="fb_share" rel="nofollow" share_url="http://www.catladyland.net/2014/06/resqwalk-makes-it-easy-to-help-shelter.html" type="button_count">

                     <br/>
                     Now that the weather is nice here in Minnesota, I enjoy walking most of the days during the week. I just learned about a way I can help shelter animals
                     <i>
                      while
                     </i>
                     I walk. What a deal, right? Heck, it will probably even encourage me to get out and walk more often!
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                     </div>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://3.bp.blogspot.com/-_k9cEv2ktN8/U7BDs39uCJI/AAAAAAAALZU/oPq_NvwFSwQ/s1600/Cat+ResQwalk.png" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;">
                       <img border="0" height="450" src="http://3.bp.blogspot.com/-_k9cEv2ktN8/U7BDs39uCJI/AAAAAAAALZU/oPq_NvwFSwQ/s1600/Cat+ResQwalk.png" width="450"/>
                      </a>
                     </div>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <a href="http://www.resqwalk.com/" target="_blank">
                      ResQWalk
                     </a>
                     is a brand new iPhone application (Android not available just yet) that tracks your walking distance and then
                     <i>
                      pays
                     </i>
                     your chosen shelter/s. Here's how it works:
                     <br/>
                     <br/>
                     1. Download the
                     <a href="https://itunes.apple.com/us/app/resqwalk/id889050235?mt=8" target="_blank">
                      FREE app
                     </a>
                     .
                     <br/>
                     <br/>
                     2. When you're ready to go for a walk, log in with either your Facebook or Twitter sign-on.
                     <br/>
                     <br/>
                     3. See weekly donation pool, and choose the shelter/s you'd like to support.
                     <br/>
                     <br/>
                     4. Begin walk.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://1.bp.blogspot.com/-f_zvR9G-lWE/U7BEBW0_J9I/AAAAAAAALZc/ukOMDCVvWR4/s1600/RQW_Begin+Walk_text.png" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;">
                       <img border="0" height="450" src="http://1.bp.blogspot.com/-f_zvR9G-lWE/U7BEBW0_J9I/AAAAAAAALZc/ukOMDCVvWR4/s1600/RQW_Begin+Walk_text.png" width="450"/>
                      </a>
                     </div>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     5. GPS tracks your route.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://2.bp.blogspot.com/-5R4wCGnib8w/U7BEXAEYe8I/AAAAAAAALZk/Y2hojgEeDQ4/s1600/RQW_Tracking_text.png" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;">
                       <img border="0" height="450" src="http://2.bp.blogspot.com/-5R4wCGnib8w/U7BEXAEYe8I/AAAAAAAALZk/Y2hojgEeDQ4/s1600/RQW_Tracking_text.png" width="450"/>

                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     6. Pause walk as needed, and then indicate when your walk is completed.
                     <br/>
                     <br/>
                     7. At the end of the walk, you get to see your results, and receive an award.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://4.bp.blogspot.com/-8RE0M1HNNIw/U7BEkD5JAzI/AAAAAAAALZs/La4iK2vMDJw/s1600/RQW_End+of+Walk_text.png" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;">
                       <img border="0" height="450" src="http://4.bp.blogspot.com/-8RE0M1HNNIw/U7BEkD5JAzI/AAAAAAAALZs/La4iK2vMDJw/s1600/RQW_End+of+Walk_text.png" width="450"/>
                      </a>
                     </div>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     8. At the end of the week, donations are paid out to the various rescues proportionally to the total distance walked. ResQWalk tracks both miles and kilometers.
                     <br/>
                     <br/>
                     Here's a short overview of the app:
                     <br/>
                     <br/>
                     <br/>
                     <object height="360" width="480">

                      </embed>

                     <br/>
                     At this time there are 450 shelters available to support via the app, although any shelter is welcome to sign up on the site to participate at
                     <i>

                     </i>
