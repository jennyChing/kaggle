     <p>
      <b>
       Zotero
      </b>
      <span class="nowrap">
       <span class="IPA nopopups">
        <a href="/wiki/Help:IPA_for_English" title="Help:IPA for English">
         /
         <span style="border-bottom:1px dotted">
          <span title="'z' in 'Zion'">
           z
          </span>
          <span title="/o/ long 'o' in 'code'">
           o
          </span>
          <span title="// primary stress follows">
           
          </span>
          <span title="'t' in 'tie'">
           t
          </span>
          <span title="/r/ 'err' in 'merry'">
           r
          </span>
          <span title="/o/ long 'o' in 'code'">
           o
          </span>
         </span>
         /
        </a>
       </span>
      </span>
      is
      <a href="/wiki/Free_and_open-source_software" title="Free and open-source software">
       free and open-source
      </a>
      <a href="/wiki/Reference_management_software" title="Reference management software">
       reference management software
      </a>
      to manage bibliographic data and related research materials (such as
      <a href="/wiki/Portable_Document_Format" title="Portable Document Format">
       PDF
      </a>
      files). Notable features include web browser integration, online syncing, generation of in-text citations, footnotes and bibliographies, as well as integration with the word processors
      <a href="/wiki/Microsoft_Word" title="Microsoft Word">
       Microsoft Word
      </a>
      ,
      <a href="/wiki/LibreOffice" title="LibreOffice">
       LibreOffice
      </a>
      ,
      <a class="mw-redirect" href="/wiki/OpenOffice.org_Writer" title="OpenOffice.org Writer">
       OpenOffice.org Writer
      </a>
      and
      <a href="/wiki/NeoOffice" title="NeoOffice">
       NeoOffice
      </a>
      . It is produced by the
      <a class="mw-redirect" href="/wiki/Center_for_History_and_New_Media" title="Center for History and New Media">
       Center for History and New Media
      </a>
      of
      <a href="/wiki/George_Mason_University" title="George Mason University">
       George Mason University
      </a>
      (GMU).

     <h2>
      <span class="mw-headline" id="Etymology">
       Etymology
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Zotero&amp;action=edit&amp;section=1" title="Edit section: Etymology">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      The name 'Zotero' is loosely derived from an
      <a href="/wiki/Albanian_language" title="Albanian language">
       Albanian
      </a>
      verb meaning "to master".
      <sup class="reference" id="cite_ref-5">
       <a href="#cite_note-5">
        <span>
         [
        </span>
        5
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <h2>
      <span class="mw-headline" id="Features">
       Features
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Zotero&amp;action=edit&amp;section=2" title="Edit section: Features">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      Zotero shows an icon when a resource (book, article, thesis) is being viewed on many websites (such as library catalogs,
      <a href="/wiki/PubMed" title="PubMed">
       PubMed
      </a>
      ,
      <a href="/wiki/Google_Scholar" title="Google Scholar">
       Google Scholar
      </a>
      ,
      <a href="/wiki/Google_Books" title="Google Books">
       Google Books
      </a>
      ,
      <a href="/wiki/Amazon.com" title="Amazon.com">
       Amazon.com
      </a>
      ,
      <a href="/wiki/Wikipedia" title="Wikipedia">
       Wikipedia
      </a>
      , and publisher's websites). Clicking this icon saves the full reference information to the Zotero library. Zotero can also save a copy of the webpage, or, in the case of academic articles, a copy of the full text PDF. Users can then add notes, tags, attachments, and their own
      <a href="/wiki/Metadata" title="Metadata">
       metadata
      </a>
      .
     </p>
     <p>
      Items are organized through a drag-and-drop
      <a href="/wiki/ITunes" title="ITunes">
       iTunes
      </a>
      -like interface, and can be searched.
     </p>
     <p>
      Selections of the local reference library data can later be exported as formatted bibliographies. Furthermore, all entries including bibliographic information and user-created rich-text memos of the selected articles can be summarized into an HTML report.
      <sup class="reference" id="cite_ref-6">
       <a href="#cite_note-6">
        <span>
         [
        </span>
        6
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <h3>
      <span class="mw-headline" id="Citation_formatting">
       Citation formatting
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Zotero&amp;action=edit&amp;section=3" title="Edit section: Citation formatting">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h3>
     <p>
      Zotero users can generate citations and bibliographies through word processor plugins, or directly in Zotero, using
      <a href="/wiki/Citation_Style_Language" title="Citation Style Language">
       Citation Style Language
      </a>
      styles. The house styles of most
      <a class="mw-redirect" href="/wiki/Academic_journals" title="Academic journals">
       academic journals
      </a>
      are available in Zotero, and the bibliography can be reformatted with a few clicks. Zotero also allows users to create their own customized citation styles.
     </p>
     <p>
      Zotero can import and export citations from/to many formats, including
      <a href="/wiki/Wikipedia:Citation_templates" title="Wikipedia:Citation templates">
       Wikipedia Citation Templates
      </a>
      ,
      <a href="/wiki/BibTeX" title="BibTeX">
       BibTeX
      </a>
      ,
      <a href="/wiki/BibTeX#Reimplementations" title="BibTeX">
       BibLateX
      </a>
      ,
      <a href="/wiki/RefWorks" title="RefWorks">
       RefWorks
      </a>
      ,
      <a href="/wiki/Metadata_Object_Description_Schema" title="Metadata Object Description Schema">
       MODS
      </a>
      ,
      <a href="/wiki/COinS" title="COinS">
       COinS
      </a>
      ,
      <a href="/wiki/Citation_Style_Language" title="Citation Style Language">
       Citation Style Language
      </a>
      /
      <a href="/wiki/JSON" title="JSON">
       JSON
      </a>
      ,
      <a href="/wiki/Refer_(software)" title="Refer (software)">
       refer
      </a>
      /
      <a class="new" href="/w/index.php?title=BibIX&amp;action=edit&amp;redlink=1" title="BibIX (page does not exist)">
       BibIX
      </a>
      ,
      <a href="/wiki/RIS_(file_format)" title="RIS (file format)">
       RIS
      </a>
      ,
      <a href="/wiki/Text_Encoding_Initiative" title="Text Encoding Initiative">
       TEI
      </a>
      , several flavours of
      <a href="/wiki/RDF_query_language" title="RDF query language">
       RDF
      </a>
      ,
      <a href="/wiki/Evernote" title="Evernote">
       Evernote
      </a>
      ,
      <a href="/wiki/EndNote" title="EndNote">
       EndNote
      </a>
      .
     </p>
     <h3>
      <span class="mw-headline" id="Annotation_and_mobile_devices">
       Annotation and mobile devices
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Zotero&amp;action=edit&amp;section=4" title="Edit section: Annotation and mobile devices">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h3>
     <p>
      Zotero can associate notes with bibliographic items. It can annotate PDFs and sync them with a mobile PDF reader through the
      <a class="new" href="/w/index.php?title=Zotfile&amp;action=edit&amp;redlink=1" title="Zotfile (page does not exist)">
       Zotfile
      </a>
      plugin.
      <sup class="reference" id="cite_ref-7">
       <a href="#cite_note-7">
        <span>
         [
        </span>
        7
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <h3>
      <span class="mw-headline" id="Language_support">
       Language support
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Zotero&amp;action=edit&amp;section=5" title="Edit section: Language support">
        edit
       </a>
       <span class="mw-editsection-bracket">

     </p>
     <h3>
      <span class="mw-headline" id="Product_support">
       Product support
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [

       </a>
       <span class="mw-editsection-bracket">

       </span>

     <p>

Questions and issues raised in the forums are answered quickly, with users and developers suggesting solutions.
      <sup class="reference" id="cite_ref-8">
       <a href="#cite_note-8">
        <span>
         [
        </span>
        8
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <p>
      Many academic institutions provide Zotero tutorials to their members.
     </p>

      <span class="mw-editsection">

     <h3>
      <span class="mw-headline" id="Zotero_1.0">
       Zotero 1.0
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Zotero&amp;action=edit&amp;section=9" title="Edit section: Zotero 1.0">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h3>
     <p>
      The first release of Zotero, 1.0.0b2.r1, was made available in October 2006 as an
      <a href="/wiki/Add-on_(Mozilla)" title="Add-on (Mozilla)">
       add-on
      </a>
      for the
      <a href="/wiki/Firefox" title="Firefox">
       Firefox
      </a>
      web browser. Development of Zotero 1.0.x continued until May 2009, when Zotero 1.0.10 was released.
     </p>
     <h3>
      <span class="mw-headline" id="EndNote_lawsuit">
       EndNote lawsuit
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Zotero&amp;action=edit&amp;section=10" title="Edit section: EndNote lawsuit">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h3>
     <p>
      In 2008,
      <a href="/wiki/Thomson_Reuters" title="Thomson Reuters">
       Thomson Reuters
      </a>
      sued the Commonwealth of
      <a href="/wiki/Virginia" title="Virginia">
       Virginia
      </a>
      and
      <a href="/wiki/George_Mason_University" title="George Mason University">
       George Mason University
      </a>
      , based on the claim that Zotero's developers had, in violation of the EndNote
      <a class="mw-redirect" href="/wiki/Software_license_agreement" title="Software license agreement">
       EULA
      </a>
      , reverse-engineered
      <a href="/wiki/EndNote" title="EndNote">
       EndNote
      </a>
      and provided Zotero with the ability to convert
      <a href="/wiki/EndNote" title="EndNote">
       EndNote
      </a>
      's proprietary .ens styles into
      <a href="/wiki/Citation_Style_Language" title="Citation Style Language">
       Citation Style Language
      </a>
      styles.
      <sup class="reference" id="cite_ref-10">
       <a href="#cite_note-10">
        <span>
         [
        </span>
        10
        <span>
         ]
        </span>
       </a>
      </sup>
      George Mason University responded that they would not renew their site license for EndNote and that "anything created by users of Zotero belongs to those users, and that it should be as easy as possible for Zotero users to move to and from the software as they wish, without friction".
      <sup class="reference" id="cite_ref-11">
       <a href="#cite_note-11">
        <span>
         [
        </span>
        11
        <span>
         ]
        </span>
       </a>
      </sup>
      The journal
      <i>
       <a href="/wiki/Nature_(journal)" title="Nature (journal)">
        Nature
       </a>
      </i>
      editorialized that "the virtues of interoperability and easy data-sharing among researchers are worth restating. Imagine if
      <a href="/wiki/Microsoft_Word" title="Microsoft Word">
       Microsoft Word
      </a>
      or
      <a href="/wiki/Microsoft_Excel" title="Microsoft Excel">
       Excel
      </a>
      files could be opened and saved only in these proprietary formats, for example. It would be impossible for
      <a href="/wiki/OpenOffice.org" title="OpenOffice.org">
       OpenOffice
      </a>
      and other such software to read and save these files using open standardsas they can legally do."
      <sup class="reference" id="cite_ref-12">
       <a href="#cite_note-12">
        <span>
         [
        </span>
        12
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <p>
      The case was dismissed on June 4, 2009 due to a lack of jurisdiction.
      <sup class="reference" id="cite_ref-13">
       <a href="#cite_note-13">
        <span>
         [
        </span>
        13
        <span>
         ]
        </span>
       </a>
      </sup>
      Although the
      <a href="/wiki/Supreme_Court_of_Virginia" title="Supreme Court of Virginia">
       Virginia Supreme Court
      </a>
      granted an appeal to Thomson Reuters in this case on December 18, 2009, the appeal was withdrawn on January 11, 2011.
     </p>
     <h3>
      <span class="mw-headline" id="Zotero_2.0">
       Zotero 2.0
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Zotero&amp;action=edit&amp;section=11" title="Edit section: Zotero 2.0">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h3>
     <p>
      Zotero 2.0, released in February 2010, added online features like metadata and file syncing and group libraries, and included a license change from the
      <a href="/wiki/Educational_Community_License" title="Educational Community License">
       Educational Community License
      </a>
      to
      <a href="/wiki/GNU_General_Public_License" title="GNU General Public License">
       GPLv3
      </a>
      .
      <sup class="reference" id="cite_ref-14">
       <a href="#cite_note-14">
        <span>
         [
        </span>
        14
        <span>
         ]
        </span>
       </a>
      </sup>
      Development of Zotero 2.0.x continued until October 2010, when Zotero 2.0.9 was released.
     </p>
     <h3>
      <span class="mw-headline" id="Zotero_2.1">
       Zotero 2.1
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Zotero&amp;action=edit&amp;section=12" title="Edit section: Zotero 2.1">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h3>
     <p>
      Zotero 2.1, released in March 2011, adds
      <a href="/wiki/Citation_Style_Language" title="Citation Style Language">
       CSL
      </a>
      1.0 support, Firefox 4 compatibility (the minimum is Firefox 3.6), and Zotero Commons, through which materials can be uploaded to the
      <a href="/wiki/Internet_Archive" title="Internet Archive">
       Internet Archive
      </a>
      .
     </p>
     <h3>
      <span class="mw-headline" id="Zotero_Standalone">
       Zotero Standalone
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Zotero&amp;action=edit&amp;section=13" title="Edit section: Zotero Standalone">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h3>
     <p>
      Zotero Standalone, first released in January 2011, allows Zotero to be run as an independent program outside Firefox. Using
      <a href="/wiki/XULRunner" title="XULRunner">
       XULRunner
      </a>
      , Zotero Standalone is available for Windows,
      <a class="mw-redirect" href="/wiki/GNU/Linux" title="GNU/Linux">
       GNU/Linux
      </a>
      and
      <span class="nowrap">
       <a class="mw-redirect" href="/wiki/Mac_OS_X" title="Mac OS X">
        Mac OS X
       </a>
      </span>
      . Connectors are available to integrate Zotero Standalone into the web browsers Safari and Chrome.
     </p>
     <h3>
      <span class="mw-headline" id="Zotero_3.0">
       Zotero 3.0
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Zotero&amp;action=edit&amp;section=14" title="Edit section: Zotero 3.0">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h3>
     <p>
      Zotero 3.0, released in January 2012, includes the stable release of Zotero Standalone as well as several new major features, including overhauled
      <a href="/wiki/Microsoft_Word" title="Microsoft Word">
       Word
      </a>
      and
      <a href="/wiki/LibreOffice" title="LibreOffice">
       LibreOffice
      </a>
      integration and duplicate detection.
     </p>
