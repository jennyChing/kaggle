                    <h3 class="post-title entry-title">
                     <a href="http://www.tasteasyougo.com/2009/08/milk-steamed-corn-on-cob-with-lemon.html">
                      Milk-Steamed Corn on the Cob with Lemon Butter and Parmesan
                     </a>
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content">
                     <div style="text-align: center;">
                      <div style="text-align: center;">
                       <div style="text-align: center;">
                        <table align="center" cellpadding="0" cellspacing="0" class="tr-caption-container" style="margin-left: auto; margin-right: auto; text-align: center;">
                         <tbody>
                          <tr>
                           <td style="text-align: center;">
                            <img alt="Milk-Steamed-Corn-on-Cob-Lemon-Butter-Parmesan-tasteasyougo.com" src="http://lh5.ggpht.com/_cgp7H29nAUs/SlaxFdo1wGI/AAAAAAAAQOE/MlBDvZiQICE/s400/IMG_1897.JPG" style="margin-left: auto; margin-right: auto;" title="Milk-Steamed Corn on the Cob with Lemon Butter and Parmesan | Taste As You Go"/>
                           </td>
                          </tr>
                          <tr>
                           <td class="tr-caption" style="text-align: center;">
                            Milk-Steamed Corn on the Cob with Lemon Butter and Parmesan
                           </td>
                          </tr>
                         </tbody>
                        </table>
                       </div>
                      </div>
                     </div>
                     <br/>
                     It's been nearly a week since I've posted anything.  If you've been
                     <a href="http://www.twitter.com/tasteasyougo" target="_blank">
                      following me on Twitter
                     </a>
                     , then you know I've been a
                     <a href="http://www.tasteasyougo.com/2009/03/sweet-friday-wishes.html" target="_blank">
                      little
                     </a>
                     <a href="http://www.tasteasyougo.com/2009/04/simple-mixed-berry-smoothie.html" target="_blank">
                      preoccupied
                     </a>
                     <a href="http://www.tasteasyougo.com/2009/05/fell-little-short-with-oikos-greek.html" target="_blank">
                      with
                     </a>
                     <a href="http://www.tasteasyougo.com/2009/05/food2-launch-party-at-food-network.html" target="_blank">
                      running
                     </a>
                     lately.  Within the last week, I ran my second 5K race (in the pouring rain, over the Brooklyn Bridge)
                     <span style="font-style: italic;">
                      and
                     </span>
                     signed up to run the
                     <a href="http://www.philadelphiamarathon.com/page/half-marathon" target="_blank">
                      Philadelphia Half Marathon
                     </a>
                     in November.  I also wrote a
                     <a href="http://megansmunchies.com/guest-post-taste-as-you-gos-montana-whoppers/" target="_blank">
                      guest post
                     </a>
                     for
                     <a href="http://www.megansmunchies.com/" target="_blank">
                      Megan's Munchies
                     </a>
                     .  But now, it's time to turn my attention back to my own blog, especially since
                     <a href="http://www.tasteasyougo.com/" target="_blank">
                      Taste As You Go
                     </a>
                     is
                     <span style="font-weight: bold;">
                      turning one
                     </span>
                     in just over two weeks!
                     <br/>
                     <br/>
                     <a name="more">
                     </a>
                     <div style="text-align: center;">
                      <table align="center" cellpadding="0" cellspacing="0" class="tr-caption-container" style="margin-left: auto; margin-right: auto; text-align: center;">
                       <tbody>
                        <tr>
                         <td style="text-align: center;">
                          <img alt="Fresh-Corn-tasteasyougo.com" as="" by="" corn="" fresh="" go="" photo="" src="http://lh5.ggpht.com/_cgp7H29nAUs/SlaxC5l3EpI/AAAAAAAAQNs/p4Mya1jhd68/s400/IMG_1880.JPG%20alt=" style="margin-left: auto; margin-right: auto;" taste="" title="Fresh Corn | Taste As You Go" you=""/>
                         </td>
                        </tr>
                        <tr>
                         <td class="tr-caption" style="text-align: center;">
                          Fresh Corn
                         </td>
                        </tr>
                       </tbody>
                      </table>
                     </div>
                     <br/>
                     One of my favorite vegetables to eat in the summer is corn on the cob, especially when it has been grilled and placed on a plate next to a big piece of barbecue chicken.  Unfortunately, living in New York, I don't have access to a grill and grilling corn on a grill pan inside just isn't the same.  I thought about roasting the two ears of corn I had, but I didn't want to add more heat to my sweltering apartment, so I scrapped that idea.  After thinking about it for a few minutes, I decided to steam the corn.
                     <br/>
                     <br/>
                     Rather than steaming the corn in water, I used the rest of the milk I had in the refrigerator to help bring out the natural sweetness of the corn.  Had I planned ahead and purchased the appropriate ingredients, after steaming the corn, I would have used the milk as the base of some
                     <a href="http://www.tasteasyougo.com/2008/11/potato-corn-chowder.html" target="_blank">
                      Potato Corn Chowder
                     </a>
                     .  I took a little taste of the milk after removing the corn from the pan -- such flavor!
                     <br/>
                     <br/>
                     <div style="text-align: center;">
                      <div style="text-align: center;">
                       <table align="center" cellpadding="0" cellspacing="0" class="tr-caption-container" style="margin-left: auto; margin-right: auto; text-align: center;">
                        <tbody>
                         <tr>
                          <td style="text-align: center;">
                           <img alt="Milk-Steamed-Corn-on-Cob-Lemon-Butter-Parmesan-tasteasyougo.com" src="http://lh3.ggpht.com/_cgp7H29nAUs/SlaxE69lwJI/AAAAAAAAQOA/o5cu6jGDTIc/s400/IMG_1895.JPG" style="margin-left: auto; margin-right: auto;" title="Milk-Steamed Corn on the Cob with Lemon Butter and Parmesan | Taste As You Go"/>
                          </td>
                         </tr>
                         <tr>
                          <td class="tr-caption" style="text-align: center;">
                           Milk-Steamed Corn on the Cob with Lemon Butter and Parmesan
                          </td>
                         </tr>
                        </tbody>
                       </table>
                       <br/>
                      </div>
                     </div>
                     The corn fit perfectly in a rectangular
                     <a href="http://www.marxfoods.com/products/Eco-Friendly-Plates" target="_blank">
                      Palm Leaf Plate
                     </a>
                     that I received from
                     <a href="http://marxfood.com/" target="_blank">
                      Marx Foods
                     </a>
                     .  I received an assortment of these environmentally-friendly biodegradable and compostable plates to try a while ago, and I've been very satisfied with how durable they are.  I
                     <span style="font-style: italic;">
                      was
                     </span>
                     concerned that the plate would start to get soggy after holding the hot corn, but it definitely withstood the heat.  Eating the corn off the Palm Leaf Plate was way more festive than eating it off one of my regular dinner plates, and it made it feel more like a cookout for me.
                     <br/>
                     <br/>
                     To finish the corn, I spread some lemon butter onto the hot ears and then grated some fresh Parmesan cheese on top in lieu of sprinkling salt.  The dish just came together beautifully.  I
                     <span style="font-style: italic;">
                      love
                     </span>
                     it when that happens!
                     <br/>
                     <br/>
                     <span style="font-weight: bold;">
                      What do you like to serve at your cookouts?
                     </span>
                     <br/>
                     <br/>
                     <div class="hrecipe">
                      <span class="item">
                      </span>
                      <br/>
                      <div class="fn">
                       <span class="item" style="font-size: large;">
                        <b>
                         Milk-Steamed Corn on the Cob with Lemon Butter and Parmesan
                        </b>
                       </span>
                      </div>
                      <span class="item">
                      </span>
                      <span style="font-style: italic;">
                       <a href="http://sites.google.com/site/tasteasyougorecipes/milk-steamed-corn-on-the-cob-with-lemon-butter-and-parmesan" target="_blank">
                        Printable Recipe
                       </a>
                      </span>
                      <br/>
                      <br/>
                      <b>
                       INGREDIENTS
                      </b>
                      <br/>
                      <ul>
                       <li class="ingredient">
                        <span class="amount">
                         2 ears
                        </span>
                        <span class="name">
                         fresh corn
                        </span>
                       </li>
                       <li class="ingredient">
                        <span class="name">
                         1% milk
                        </span>
                       </li>
                       <li class="ingredient">
                        <span class="amount">
                         2 tablespoons
                        </span>
                        <span class="name">
                         butter, at room temperature
                        </span>
                       </li>
                       <li class="ingredient">
                        <span class="amount">
                         1 teaspoon
                        </span>
                        <span class="name">
                         lemon zest
                        </span>
                       </li>
                       <li class="ingredient">
                        <span class="name">
                         Sea salt, to taste
                        </span>
                       </li>
                       <li class="ingredient">
                        <span class="name">
                         Freshly grated Parmesan cheese
                        </span>
                       </li>
                      </ul>
                      <br/>
                      <b>
                       INSTRUCTIONS
                      </b>
                      <br/>
                      <ol class="instructions">
                       <li class="instruction">
                        Place corn in a shallow pan. Pour enough milk into the pan so the corn is half-covered. Bring the milk to a boil.
                       </li>
                       <br/>
                       <li class="instruction">
                        Lower the heat to medium-low and cover. Cook until just hot, about 10 minutes.
                       </li>
                       <br/>
                       <li class="instruction">
                        While the corn is steaming, combine butter and lemon zest. Add sea salt, to taste.
                       </li>
                       <br/>
                       <li class="instruction">
                        When the corn has finished cooking, remove it from the milk. Spread with lemon butter and sprinkle with freshly grated Parmesan. Serve while hot/warm.
                       </li>
                      </ol>
                     </div>

        var elem = this.getElementsByTagName('div');
        for (var i = 0; i &lt; elem.length; i++) 
        {
            var classes = elem[i].className;
            if (myclass.test(classes)) 
            {
                var container = elem[i];
                for (var b = 0; b &lt; container.childNodes.length; b++)
                {
                    var item = container.childNodes[b].className;
                    if (myTitleContainer.test(item))
                    {
                        var link = container.childNodes[b].getElementsByTagName('a');
                        if (typeof(link[0]) != 'undefined')
                        {
                            var url = link[0].href;
                            var title = link[0].innerHTML;
                        }
                        else
                        {
                            var url = document.url;
                            var title =  container.childNodes[b].innerHTML;
                        }
                        if (typeof(url) == 'undefined'|| url == 'undefined' ){
                            url = window.location.href;
                        }
                    var singleq = new RegExp("'", 'g');
                    var doubleq = new RegExp('"', 'g');                                                                                                                                                                                                                          
                    title = title.replace(singleq, '&#39;', 'gi');
                    title = title.replace(doubleq, '&#34;', 'gi');
                    
                    }
                    if (myPostContent.test(item))
                    {
                        var footer = container.childNodes[b];
                    }
                }
                var n = document.createElement('div');
                var at = "&lt;div class='addthis_toolbox addthis_default_style ' addthis:title='"+title+"' addthis:url='"+encodeURI(url)+"'   &gt; &lt;a class='addthis_button_facebook_like' fb:like:layout='button_count'&gt;&lt;/a&gt; &lt;a class='addthis_button_tweet'&gt;&lt;/a&gt; &lt;a class='addthis_button_google_plusone' g:plusone:size='medium'&gt;&lt;/a&gt; &lt;a class='addthis_counter addthis_pill_style'&gt;&lt;/a&gt; &lt;/div&gt; ";

            }
        }
