     <h1>
      <a id="teradata-to-oracle-migration" name="teradata-to-oracle-migration">
       Teradata to Oracle Migration
      </a>
     </h1>
     <div class="level1">
      <p>
       SQLines provides tools and services to help you transfer data, convert database schema (DDL), views, stored procedures, functions, triggers, macros, queries, BTEQ and
       <acronym title="Structured Query Language">
        SQL
       </acronym>
       scripts from Teradata to Oracle.
      </p>
      <p>
       We also help convert embedded
       <acronym title="Structured Query Language">
        SQL
       </acronym>
       statements in C/C++ (ODBC, ESQL/C), C#, Java, PowerBuilder,
       <acronym title="Visual Basic">
        VB
       </acronym>
       /
       <acronym title="Visual Basic">
        VB
       </acronym>
       .NET,
       <acronym title="Active Server Pages">
        ASP
       </acronym>
       /
       <acronym title="Active Server Pages">
        ASP
       </acronym>
       .NET,
       <acronym title="Practical Extraction and Report Language">
        Perl
       </acronym>
       /
       <acronym title="Hypertext Preprocessor">
        PHP
       </acronym>
       and SAS applications.
      </p>
      <ul>
       <li class="level1">
        <div class="li">
         <strong>
          Teradata
         </strong>
         14, 13.x, 12, V2R6 and V2R5
        </div>
       </li>
       <li class="level1">
        <div class="li">
         <strong>
          Oracle
         </strong>
         12c, 11g and 10g
        </div>
       </li>
      </ul>
     </div>
