        <div class="entry-content post-content">
         <div class="pd-art-con">
          <img alt="Adolescent boy playing on digital tablet" data-caption="Adolescent boy playing on digital tablet" data-credit="Getty Images/Cultura RF" data-mep="31633" src="http://o.aolcdn.com/dims-shared/dims3/GLOB/crop/5010x3369+70+0/resize/620x413!/format/jpg/quality/85/http://o.aolcdn.com/hss/storage/adam/aaf554d956ee761aceb4181334e03c99/175816646.jpg"/>
         </div>
         <div class="pd-cont-con">
          The British Medical Journal recently published a decade-long study that reveals the truth about kids and gaming. The results found that
          <a href="http://adc.bmj.com/content/early/2013/02/21/archdischild-2011-301508.full.pdf+html" target="_blank">
           gaming has no negative impact on children's behaviour
          </a>
          , despite the misbelief that video games are bad.
          <br/>
          <br/>
          Over 11,000 children in the UK were included in the study, which
          <a href="http://ca.ign.com/articles/2013/11/18/games-definitely-dont-harm-kids-says-huge-study" target="_blank">
           sought to discover the connection between screen time and a child's psychosocial development
          </a>
          . The key finding was that "playing electronic games at 5 years was not associated with increased risk of problems." The study found no negative effect on children's mood, behaviour, or attention issues later in life. Neither television nor video games were found to cause emotional problems and the results were the same for both genders.
          <br/>
          <br/>
          However, what's surprising is the fact that
          <a href="http://www.forbes.com/sites/jordanshapiro/2013/11/15/study-says-gaming-has-no-negative-effect-on-kids-behavior/" target="_blank">
           television was found to be far worse than video games
          </a>
          the more frequently it was watched. The study revealed that children at the age of five who watched more than three hours of TV a day had "a small increase in conduct problems" between the ages of five and seven, compared to those who only watched for an hour.
          <br/>
          <br/>
          <div class="fad">
          </div>
          <div class="slideshow">
           <div class="knot-slideshow" data-slideshow-id="89801">
            <p class="slide-title">
             TV and your toddler: the latest reserach
            </p>

           <div class="slide">
            <p class="cont">
            </p>
            <div class="add" id="add">
             <span class="addlbl">
              Advertisement
             </span>
            </div>
           </div>
          </div>
          <ul class="knot-slideshow-data" style="display:none;">
           <li>
            <a href="http://www.blogcdn.com/slideshows/images/slides/107/102/3/S1071023/slug/l/shutterstock-128667257-1.jpg">
             Less success in school?
            </a>
            <p>
            </p>
            <p>
             A
             <a href="http://life.nationalpost.com/2013/08/09/toddlers-watching-three-hours-of-tv-a-day-more-likely-to-struggle-with-learning-bullies-study/" target="_blank">
              study published in August from researchers in Montreal
             </a>
             found that for every hour increased in TV watching for a tot of 29 months, success in kindergarten went down on a number of fronts.
            </p>
            <p>
             Heavy TV watchers suffered later on with worse vocabulary and math skills, classroom engagement, social interactions with classmates and physical skills. Researchers followed about 2,000 Quebec children to get their results.
            </p>
           </li>
           <li>
            <a href="http://www.blogcdn.com/slideshows/images/slides/107/102/2/S1071022/slug/l/shutterstock-79158400-1.jpg">
             232 minutes a day
            </a>
            <p>
             A
             <a href="http://www.education.com/magazine/article/background-tv/" target="_blank">
              2012 study
             </a>
             based on telephone interviews with 1,500 US parents found that kids between eight months and eight years old spend an average of 232 minutes a day with the TV on in the background. (Along with that, most kids watch an average of 80 minutes a day.)
            </p>
           </li>
           <li>
            <a href="http://www.blogcdn.com/slideshows/images/slides/107/102/0/S1071020/slug/l/shutterstock-25104988-1.jpg">
             Can't concentrate
            </a>
            <p>
             Also, a report from the journal Child Development found that toddlers cannot focus and learn while background TV is on. In a controlled environment, researchers observed 50 kids aged 1 to 3 while a game show was on and compared their behaviour to screen-free actions. They found the children had trouble concentrating on their toys during screen-time.
            </p>
           </li>
           <li>
            <a href="http://www.blogcdn.com/slideshows/images/slides/107/102/1/S1071021/slug/l/shutterstock-45939937-1.jpg">
             The good news
            </a>
            <p>
             While there are not numerous reports finding a positive outcome for young kids who watch TV, a review study from the journal
             <a href="http://pediatrics.aappublications.org/" target="_blank">
              Pediatrics
             </a>
             found that educational shows in particular broaden kids' knowledge, impact their racial attitudes and help their imaginative play.
            </p>
           </li>
          </ul>
          <br/>
          <br/>
          This UK study is the first to separate its findings on video games and television. It also backs up previously found evidence that video games are good for kids. For instance, earlier this year, a paper was published in
          <em>
           Current Biology
          </em>
          from the University of Padua stating that
          <a href="http://ca.ign.com/articles/2013/05/29/can-video-games-make-dyslexic-children-read-better" target="_blank">
           video games make dyslexic children read better
          </a>
          . The study explored how children between the ages of 7 and 13 could read faster after playing 80 minutes of mini-games from Rayman Raving Rabbids.
          <br/>
          <br/>
          Also, last month, a study by Max Planck Institute for Human Development and Charit University Medicine St. Hedwig-Krankenhaus discussed the
          <a href="http://www.mpg.de/7588840/video-games-brain" target="_blank">
           benefits of video games on the brain
          </a>
          . There, researchers found that playing video games was beneficial for the brain's "
          <a href="http://ca.ign.com/articles/2013/11/04/the-positive-effects-games-have-on-your-brain" target="_blank">
           spatial orientation, memory formation and strategic planning as well as fine motor skills
          </a>
          ."
          <br/>
          <br/>
          With studies like these, the negative stigma placed on video games is slowly breaking down.

  <script>
   jQuery(document).ready(function() {
  floater = {
    floatContainer : null,
    floatWrapper : null,
    floatingLike : null,
    isLikeSafe : null,
    likeURL : null,
    isWide : null,
    firstLikeView : true,
    stickyStopElement : jQuery('#aol-comments'),
    init : function(id) {
      if (jQuery('.re_wide_format')[0]) {
        this.isWide = true;
      }
      this.floatContainer = jQuery(id);
      this.floatContainer.wrap('&lt;div id="float_tracker" class="bN" /&gt;');
      this.floatWrapper = jQuery("#float_tracker");
      this.trackFloater();
    },
    trackFloater : function() {
      jQuery(window).scroll(function() {
        if (!floater.floatContainer.hasClass('floatingShare')) {
          floater.floatWrapper.height(floater.floatContainer.innerHeight()).width(floater.floatContainer.innerWidth());
        }
        if (floater.floatWrapper.height() == 0) {
          floater.floatWrapper.height(floater.floatContainer.innerHeight());
        }
        var stickyStop = floater.stickyStopElement.offset().top - floater.floatContainer.height();
        var stickyStart = floater.floatWrapper.offset().top + floater.floatWrapper.height() + 70 ;
        var topper = jQuery(window).scrollTop();
        if (stickyStart &lt; topper) {
          if (stickyStop &lt; topper) {
            floater.belowScreen();
          } else {
            floater.offScreen();
          }
        } else {
          floater.onScreen();
        }
      });
    },
    onScreen : function() {
      if (this.floatContainer.hasClass('floatingShare')) {
        this.floatContainer.removeClass('floatingShare');
      }
    },
    belowScreen : function() {
      if (this.floatContainer.hasClass('floatingShare')) {
        this.floatContainer.fadeOut();
      }
    },
    offScreen : function() {
      if (!this.floatContainer.hasClass('floatingShare')) {
        this.floatContainer.hide().fadeIn().addClass('floatingShare');
        this.setOffset();
      } else {
        this.floatContainer.fadeIn();
      }
      jQuery(window).resize(function() {
        floater.setOffset();
      });
    },
    setOffset : function() {
      if (this.isWide == true) {
        var offset = this.wrapperOffset = jQuery('#main').offset().left - 138;
      } else {
        var offset = this.wrapperOffset = jQuery('#main').offset().left - 128;
      }
      if (offset &lt; 0) {
        if (this.floatContainer.hasClass('floatingShare')) {
          this.floatContainer.removeClass('floatingShare');
        }
      } else {
        this.floatContainer.css({
          left : offset
        });
      }
    }
  }
  floater.init("#fltShrBar");
});
  </script>
