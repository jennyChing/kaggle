     <p>
      <b>
       Windows logo key
      </b>
      (also known as
      <b>
       Windows key
      </b>
      ,
      <b>
       start key
      </b>
      ,
      <b>
       logo key
      </b>
      ,
      <b>
       flag key
      </b>
      ,
      <b>
       super key
      </b>
      , or
      <b>
       flag
      </b>
      ) is a
      <a href="/wiki/Computer_keyboard" title="Computer keyboard">
       keyboard
      </a>
      key which was originally introduced on the
      <a href="/wiki/Microsoft_Natural_keyboard" title="Microsoft Natural keyboard">
       Microsoft Natural keyboard
      </a>
      before the release of
      <a href="/wiki/Windows_95" title="Windows 95">
       Windows 95
      </a>
      . This key became a standard key on PC keyboards. Tapping this key invokes the operating system's
      <a href="/wiki/Start_menu" title="Start menu">
       start menu
      </a>
      , if it has one. In Windows,
      <kbd class="keyboard-key nowrap" style="border: 1px solid #aaa; -moz-border-radius: 0.2em; -webkit-border-radius: 0.2em; border-radius: 0.2em; -moz-box-shadow: 0.1em 0.2em 0.2em #ddd; -webkit-box-shadow: 0.1em 0.2em 0.2em #ddd; box-shadow: 0.1em 0.2em 0.2em #ddd; background-color: #f9f9f9; background-image: -moz-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -o-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -webkit-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: linear-gradient(to bottom, #eee, #f9f9f9, #eee); padding: 0.1em 0.3em; font-family: inherit; font-size: 0.85em;">
       Ctrl
      </kbd>
      +
      <kbd class="keyboard-key nowrap" style="border: 1px solid #aaa; -moz-border-radius: 0.2em; -webkit-border-radius: 0.2em; border-radius: 0.2em; -moz-box-shadow: 0.1em 0.2em 0.2em #ddd; -webkit-box-shadow: 0.1em 0.2em 0.2em #ddd; box-shadow: 0.1em 0.2em 0.2em #ddd; background-color: #f9f9f9; background-image: -moz-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -o-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -webkit-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: linear-gradient(to bottom, #eee, #f9f9f9, #eee); padding: 0.1em 0.3em; font-family: inherit; font-size: 0.85em;">
       Esc
      </kbd>
      performs the same function, in case the keyboard lacks this key.
     </p>
     <p>
      Historically, the addition of two Windows keys and a
      <a href="/wiki/Menu_key" title="Menu key">
       menu key
      </a>
      marked the change from the 101/102-key to 104/105-key layout for
      <a href="/wiki/IBM_PC_keyboard" title="IBM PC keyboard">
       PC keyboards
      </a>
      :
      <sup class="reference" id="cite_ref-1">
       <a href="#cite_note-1">
        <span>
         [
        </span>
        1
        <span>
         ]
        </span>
       </a>
      </sup>
      compared to the former layout, a Windows key was placed between the left
      <kbd class="keyboard-key nowrap" style="border: 1px solid #aaa; -moz-border-radius: 0.2em; -webkit-border-radius: 0.2em; border-radius: 0.2em; -moz-box-shadow: 0.1em 0.2em 0.2em #ddd; -webkit-box-shadow: 0.1em 0.2em 0.2em #ddd; box-shadow: 0.1em 0.2em 0.2em #ddd; background-color: #f9f9f9; background-image: -moz-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -o-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -webkit-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: linear-gradient(to bottom, #eee, #f9f9f9, #eee); padding: 0.1em 0.3em; font-family: inherit; font-size: 0.85em;">
       Ctrl
      </kbd>
      and the left
      <kbd class="keyboard-key nowrap" style="border: 1px solid #aaa; -moz-border-radius: 0.2em; -webkit-border-radius: 0.2em; border-radius: 0.2em; -moz-box-shadow: 0.1em 0.2em 0.2em #ddd; -webkit-box-shadow: 0.1em 0.2em 0.2em #ddd; box-shadow: 0.1em 0.2em 0.2em #ddd; background-color: #f9f9f9; background-image: -moz-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -o-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -webkit-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: linear-gradient(to bottom, #eee, #f9f9f9, #eee); padding: 0.1em 0.3em; font-family: inherit; font-size: 0.85em;">
       Alt
      </kbd>
      ; another Windows key andimmediately to its righta menu key were placed between the
      <kbd class="keyboard-key nowrap" style="border: 1px solid #aaa; -moz-border-radius: 0.2em; -webkit-border-radius: 0.2em; border-radius: 0.2em; -moz-box-shadow: 0.1em 0.2em 0.2em #ddd; -webkit-box-shadow: 0.1em 0.2em 0.2em #ddd; box-shadow: 0.1em 0.2em 0.2em #ddd; background-color: #f9f9f9; background-image: -moz-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -o-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -webkit-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: linear-gradient(to bottom, #eee, #f9f9f9, #eee); padding: 0.1em 0.3em; font-family: inherit; font-size: 0.85em;">
       <a href="/wiki/AltGr_key" title="AltGr key">
        AltGr
       </a>
      </kbd>
      (or right
      <kbd class="keyboard-key nowrap" style="border: 1px solid #aaa; -moz-border-radius: 0.2em; -webkit-border-radius: 0.2em; border-radius: 0.2em; -moz-box-shadow: 0.1em 0.2em 0.2em #ddd; -webkit-box-shadow: 0.1em 0.2em 0.2em #ddd; box-shadow: 0.1em 0.2em 0.2em #ddd; background-color: #f9f9f9; background-image: -moz-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -o-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -webkit-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: linear-gradient(to bottom, #eee, #f9f9f9, #eee); padding: 0.1em 0.3em; font-family: inherit; font-size: 0.85em;">
       Alt
      </kbd>
      key on keyboards that lack
      <kbd class="keyboard-key nowrap" style="border: 1px solid #aaa; -moz-border-radius: 0.2em; -webkit-border-radius: 0.2em; border-radius: 0.2em; -moz-box-shadow: 0.1em 0.2em 0.2em #ddd; -webkit-box-shadow: 0.1em 0.2em 0.2em #ddd; box-shadow: 0.1em 0.2em 0.2em #ddd; background-color: #f9f9f9; background-image: -moz-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -o-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -webkit-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: linear-gradient(to bottom, #eee, #f9f9f9, #eee); padding: 0.1em 0.3em; font-family: inherit; font-size: 0.85em;">
       AltGr
      </kbd>
      ) and the right control key. In laptop and other compact keyboards it is common to have just one Windows key (usually on the left). Also, on Microsoft's Entertainment Desktop sets (designed for
      <a href="/wiki/Windows_Vista" title="Windows Vista">
       Windows Vista
      </a>
      ), the Windows key is in the middle of the keyboard, below all other keys (where the user's thumbs rest).
     </p>
     <p>
      On
      <a href="/wiki/Windows_8" title="Windows 8">
       Windows 8
      </a>
      <a href="/wiki/Tablet_computer" title="Tablet computer">
       tablet computers
      </a>
      , hardware certification requirements initially mandated that the Windows key is centered on the bezel below the screen, except on a convertible laptop, where the button is allowed to be off-center in a tablet configuration. This requirement was relaxed in
      <a href="/wiki/Windows_8.1" title="Windows 8.1">
       Windows 8.1
      </a>
      , allowing the Windows key to be placed on any bezel or edge of the unit, though a centered location along the bottom bezel is still preferred.
      <sup class="reference" id="cite_ref-ms-whcr2_2-0">
       <a href="#cite_note-ms-whcr2-2">
        <span>
         [
        </span>
        2
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>

     <h2>
      <span class="mw-headline" id="Licensing">
       Licensing
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Windows_key&amp;action=edit&amp;section=1" title="Edit section: Licensing">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      Microsoft regulates the appearance of the Windows key logo picture with a specially crafted license for keyboard manufacturers ("Microsoft Windows Logo Key Logo License Agreement for Keyboard Manufacturers"). With the introduction of a new Microsoft Windows logo, first used with Windows XP, the agreement was updated to require that the new design be adopted for all keyboards manufactured after September 1, 2003.
      <sup class="reference" id="cite_ref-3">
       <a href="#cite_note-3">
        <span>
         [
        </span>
        3
        <span>
         ]
        </span>
       </a>
      </sup>
      However, with the release of Windows Vista, Microsoft published guidelines for a new Windows Logo key that incorporates the Windows logo recessed in a
      <a href="/wiki/Chamfer" title="Chamfer">
       chamfered
      </a>
      lowered circle with a contrast ratio of at least 3:1 with respect to background that the key is applied to.
      <sup class="reference" id="cite_ref-4">
       <a href="#cite_note-4">
        <span>
         [
        </span>
        4
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <p>
      In
      <a href="/wiki/Common_Building_Block" title="Common Building Block">
       Common Building Block
      </a>
      Keyboard Specification, all CBB compliant keyboards were to comply with the Windows Vista Hardware Start Button specification beginning in 2007-06-01.
      <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
       [
       <i>
        <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
         <span title="This claim needs references to reliable sources. (September 2010)">
          citation needed
         </span>
        </a>
       </i>
       ]
      </sup>
     </p>
     <h2>
      <span class="mw-headline" id="Use_with_Microsoft_Windows">
       Use with Microsoft Windows
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Windows_key&amp;action=edit&amp;section=2" title="Edit section: Use with Microsoft Windows">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      On
      <a href="/wiki/Windows_9x" title="Windows 9x">
       Windows 9x
      </a>
      and
      <a href="/wiki/Windows_NT" title="Windows NT">
       Windows NT
      </a>
      families of Windows operating system, tapping the Windows key by itself traditionally revealed
      <a href="/wiki/Taskbar#Microsoft_Windows" title="Taskbar">
       Windows Taskbar
      </a>

     <h2>
      <span class="mw-headline" id="Use_with_non-Microsoft_operating_systems">
       Use with non-Microsoft operating systems
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Windows_key&amp;action=edit&amp;section=12" title="Edit section: Use with non-Microsoft operating systems">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      The Windows key can also be used on other operating systems.
     </p>
     <p>
      On
      <a href="/wiki/Unix" title="Unix">
       Unix
      </a>
      and
      <a href="/wiki/Unix-like" title="Unix-like">
       Unix-like
      </a>
      operating systems, it is sometimes called "
      <a href="/wiki/Meta_key" title="Meta key">
       Meta
      </a>
      " or "
      <a href="/wiki/Super_key_(keyboard_button)" title="Super key (keyboard button)">
       Super
      </a>
      ". The
      <a class="mw-redirect" href="/wiki/X_window_system" title="X window system">
       X window system
      </a>
      usually treats this key as modifier
      <b>
       MOD4
      </b>
      .
      <a href="/wiki/KDE" title="KDE">
       KDE
      </a>
      and
      <a href="/wiki/GNOME" title="GNOME">
       GNOME
      </a>
      , which are commonly the default desktop environments of
      <a href="/wiki/Linux_distribution" title="Linux distribution">
       Linux distributions
      </a>
      usually support the key, though it may be necessary to configure its functionality after installation.
      <a href="/wiki/GNOME_Shell" title="GNOME Shell">
       GNOME Shell
      </a>
      uses the Super key as its default keyboard shortcut for bringing up the Activities Overview. Similarly,
      <a href="/wiki/Cinnamon_(software)" title="Cinnamon (software)">
       Cinnamon
      </a>
      uses the Super key to pop-up its main system menu (normally docked at lower-left of screen). In the
      <a href="/wiki/Compiz" title="Compiz">
       Compiz
      </a>
      window manager, the Super key can by default be used in conjunction with the scroll wheel to zoom in or out of any part of the desktop.
      <a href="/wiki/OS_X" title="OS X">
       OS X
      </a>
      uses the Windows key as a replacement for the
      <a href="/wiki/Command_key" title="Command key">
       Command key
      </a>
      if the keyboard does not include the latter. This sometimes leads to placement issues for users used to Apple keyboards however, as the Command key is usually placed where the Alt key is on most keyboards (next to the Space bar).
     </p>
     <p>
      On
      <a href="/wiki/Xbox_360" title="Xbox 360">
       Xbox 360
      </a>
      , pressing the Windows key performs the same action as the Guide button on
      <a href="/wiki/Xbox_360_Controller" title="Xbox 360 Controller">
       Xbox 360 Controller
      </a>
      or remote controls, opening the Xbox Guide. Holding down the Windows key and pressing M opens a pop up conversation window over gameplay, if an
      <a class="mw-redirect" href="/wiki/Instant_message" title="Instant message">
       instant message
      </a>
      conversation is in progress. On a
      <a href="/wiki/PlayStation_3" title="PlayStation 3">
       PlayStation 3
      </a>
      console, pressing the Windows key performs the same action as the PS Button on the
      <a href="/wiki/Sixaxis" title="Sixaxis">
       Sixaxis Controller
      </a>
      , opening the
      <a href="/wiki/XrossMediaBar" title="XrossMediaBar">
       XrossMediaBar
      </a>
      .
     </p>
