1.e4 d6 2.d4 Nf6 3.Nc3 { regular pirc lines } 3...g6 { g6 is red on rybka opening book. but even the main line is over 30 moves deep and its just slightly better for white in the end. } 4.h3 { Bg7 regular move } 4...Bg7 { h3 is playable but horrible and as 0 games in the book } 5.g4 { g4 optimistic guy this Yudin! } 5...c5 { c5 a fast couterattack to get the white king exposed in the center of the board } 6.g5 { g5 looks incredibly bad for me but rybka recommends it here } 6...Nfd7 7.d5 { rybka also says d5 is best for whie =(0.13) the seond best move Be3 =0.06 I would still go with Nf3 here } 7...Na6 { I don't like this move a N on the rim is dimm but it develops pieces so its fine :) } 8.h4 { h4 is extremely optimistic better would be Bg2 or even Nge2 } 8...Qa5 { Now the tactics fest begin } 9.Bd2 { Rybka recommends Ne2 or BxN } 9...Qb4 { very strong move and the best in the postion, now everything is under attack. } 10.Rh3 { Rybka likes Bg2 here but white is already slightly worst. } 10...O-O { natural and correct move } 11.Rb1 { this move is were things start to go wrong fo white, better would be h5 starting their own attack. } 11...Bxc3 { black decides to simplify the position with hopes to mine even more whites king position } 12.Bxc3 Qxe4+ 13.Re3 Qh1 { nice square for the queen I really love to see it there now the N is hanging. on Ne2 white as some problems with the bishop } 14.Nf3 Nb6 { now black finally brings the white B into the attack } 15.Rxe7 { white is hopeless so just grabs another pawn } 15...Bg4 { and white resigns. This was a really bad defeat expecially for a GM! }
