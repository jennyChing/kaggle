    <div class="yui3-u" id="grub-rail-leftRailSpanner" itemscope="" itemtype="http://schema.org/Recipe">
     <h1 class="grub-topOfColumn" itemprop="name" style="font-size: 27px;">
      Mini Peach &amp; Basil Pizzas
     </h1>
     <img height="498" src="http://cookingstoned.tv/wp-content/uploads/2013/07/Mini-Peach-Basil-Pizza-790x498.jpg" width="790"/>
     <div class="yui3-g">
      <div class="yui3-u" id="grub-rail-well">
       <div class="grub-byline" style="margin-top: 1em;">
        Recipe by
        <a href="https://plus.google.com/113274240511284571445" rel="author">
         Jerry James Stone
        </a>
         July 30, 2013
       </div>
       <div class="grub-mainPostContent" itemprop="recipeInstructions" not-style="font-size: 16px; line-height: 23px;">
        <p>
         <a class="grub-popupPhotoLink" grub-popupphotoheight="630" grub-popupphotopath="/recipe/mini-peach-basil-pizzas/mini-peach-basil-pizza/" grub-popupphotowidth="1000" href="#photo/mini-peach-basil-pizza" rel="attachment wp-att-3752">
         </a>
        </p>
        <p>
         Fresh, fuzzy and oh so sweet. What is not to love about peaches? Now that you know
         <a href="http://cookingstoned.tv/how-to/all-about-peaches/" title="All About Peaches">
          how to buy a peach
         </a>
         and
         <a href="http://cookingstoned.tv/how-to/all-about-peaches/" title="All About Peaches">
          how to store a peach
         </a>
         , lets get to eating them!
        </p>
        <p>
        </p>
        <div id="grub_inlinePlayer_tLY7drMn2N8" style="width: 600px; height: 338px;">
        </div>
        <div class="yui3-g" id="grub-recipeSummary">
         <div class="yui3-u" id="grub-recipeSummary-left">
          <h2>
           Ingredients
          </h2>
          <ul id="grub-ingredients">
           <li itemprop="ingredients">
            <a href="http://cookingstoned.tv/recipe/chewy-homemade-pizza-dough/">
             Chewy Homemade Pizza Dough
            </a>
           </li>
           <li itemprop="ingredients">
            1 Pound Peachs (about 3)
           </li>
           <li itemprop="ingredients">
            16 Ounces Mozzarella
           </li>
           <li itemprop="ingredients">
            6 fresh basil leaves
           </li>
           <li itemprop="ingredients">
             Cup Olive Oil
           </li>
          </ul>
         </div>
         <div class="yui3-u" id="grub-recipeSummary-right">
          <h2>
           <a class="grub-button" href="http://cookingstoned.tv/recipe/mini-peach-basil-pizzas/printable/" target="_blank">
            <span class="grub-printIcon">
            </span>
            Printable View
           </a>
          </h2>
          <h2>
           Yield:
           <span itemprop="recipeYield">
            16 pizzettas
           </span>
          </h2>
          <meta content="PT17M" itemprop="totalTime"/>
          <h2>
           Total Time
           <div style="float: right;">
            17 minutes
           </div>
          </h2>
          <meta content="PT5M" itemprop="prepTime"/>
          <div style="margin-bottom: 3px;">
           Prep time
           <div style="float: right">
            5 minutes
           </div>
          </div>
          <meta content="PT12M" itemprop="cookTime"/>
          <div style="margin-bottom: 3px;">
           Cook time
           <div style="float: right">
            12 minutes
           </div>
          </div>
          <h2>
           Oven Temperature:
           <span>
            500 F
           </span>
          </h2>
          <div class="grub-recipeTaxonomyList">
           <h2>
            Recipe Type:
           </h2>
           Appetizer
          </div>
          <div class="grub-recipeTaxonomyList">
           <h2>
            Main Ingredients:
           </h2>
           Basil, Peach
          </div>
         </div>
        </div>
        <p>
         1. For this recipe I used my
         <a href="http://cookingstoned.tv/recipe/chewy-homemade-pizza-dough/" title="Chewy Homemade Pizza Dough">
          Chewy Homemade Pizza Dough
         </a>
         . I made the recipe as follows but before l let it rise, I cut it into 16 individual pieces for the pizzettas.
        </p>
        <p>
         <a class="grub-popupPhotoLink" grub-popupphotoheight="677" grub-popupphotopath="/recipe/mini-peach-basil-pizzas/pizza-dough/" grub-popupphotowidth="1000" href="#photo/pizza-dough" rel="attachment wp-att-3733">
          <img alt="Pizza Dough" class="alignnone size-grub-wellWidth wp-image-3733" height="406" src="http://cookingstoned.tv/wp-content/uploads/2013/07/Pizza-Dough-600x406.jpg" width="600"/>
         </a>
        </p>
        <p>
         2. You want the pound of peaches to be just ripe, so they are full of flavor but not super soft. It will make grilling them a pain in the butt if they are overly ripe.
        </p>
        <p>
         <a class="grub-popupPhotoLink" grub-popupphotoheight="666" grub-popupphotopath="/how-to/all-about-peaches/about-peaches/" grub-popupphotowidth="1000" href="#photo/about-peaches" rel="attachment wp-att-3705">
          <img alt="About Peaches" class="alignnone size-grub-wellWidth wp-image-3705" height="399" src="http://cookingstoned.tv/wp-content/uploads/2013/07/About-Peaches-600x399.jpg" width="600"/>
         </a>
        </p>
        <p>
         3. Now cut them in half, following the seam of the peach. Give the halves a twist and pull them apart, then remove the pit. Cut the peaches into slices, about 1/4 thick.
        </p>
        <p>
         <a class="grub-popupPhotoLink" grub-popupphotoheight="670" grub-popupphotopath="/recipe/mini-peach-basil-pizzas/peaches/" grub-popupphotowidth="1000" href="#photo/peaches" rel="attachment wp-att-3735">
          <img alt="Peaches" class="alignnone size-grub-wellWidth wp-image-3735" height="402" src="http://cookingstoned.tv/wp-content/uploads/2013/07/Peaches-600x402.jpg" width="600"/>
         </a>
        </p>
        <p>
         <a class="grub-popupPhotoLink" grub-popupphotoheight="488" grub-popupphotopath="/recipe/mini-peach-basil-pizzas/sliced-peaches/" grub-popupphotowidth="1000" href="#photo/sliced-peaches" rel="attachment wp-att-3734">
          <img alt="Sliced Peaches" class="alignnone size-grub-wellWidth wp-image-3734" height="292" src="http://cookingstoned.tv/wp-content/uploads/2013/07/Sliced-Peaches-600x292.jpg" width="600"/>
         </a>
        </p>
        <p>
         4. Now lets grill these babies! Were just going to grill them on one side. For one, were gonna be baking them on a pizza. Grilling them and then cooking them will make them a bit more mushy then I would prefer. Grill them for about a minute or so, until they have those lovely marks.
        </p>
        <p>
         <a class="grub-popupPhotoLink" grub-popupphotoheight="659" grub-popupphotopath="/recipe/mini-peach-basil-pizzas/grilled-peaches/" grub-popupphotowidth="1000" href="#photo/grilled-peaches" rel="attachment wp-att-3732">
          <img alt="Grilled Peaches" class="alignnone size-grub-wellWidth wp-image-3732" height="395" src="http://cookingstoned.tv/wp-content/uploads/2013/07/Grilled-Peaches-600x395.jpg" width="600"/>
         </a>
        </p>
        <p>
         5. Now that mini pizza doughs have risen, go ahead and flatten them out and place them on a baking sheet lined with parchment paper. You can use a rolling pin but I find it is easiest to just use your fingers (thats what she said).
        </p>
        <p>
         <a class="grub-popupPhotoLink" grub-popupphotoheight="575" grub-popupphotopath="/recipe/mini-peach-basil-pizzas/mini-pizza-dough/" grub-popupphotowidth="1000" href="#photo/mini-pizza-dough" rel="attachment wp-att-3731">
          <img alt="Mini Pizza Dough" class="alignnone size-grub-wellWidth wp-image-3731" height="345" src="http://cookingstoned.tv/wp-content/uploads/2013/07/Mini-Pizza-Dough-600x345.jpg" width="600"/>
         </a>
        </p>
        <p>
         6. Brush each mini pizza with just enough olive oil to coat and then top them with mozzarella and a couple of grilled peach slices.
        </p>
        <p>
         <a class="grub-popupPhotoLink" grub-popupphotoheight="587" grub-popupphotopath="/recipe/mini-peach-basil-pizzas/grilled-peach-pizza/" grub-popupphotowidth="1000" href="#photo/grilled-peach-pizza" rel="attachment wp-att-3730">
          <img alt="Grilled Peach Pizza" class="alignnone size-grub-wellWidth wp-image-3730" height="352" src="http://cookingstoned.tv/wp-content/uploads/2013/07/Grilled-Peach-Pizza-600x352.jpg" width="600"/>
         </a>
        </p>
        <p>
         7. Bake them at 500 degrees F for about 10 minutes until the cheese is fully melted, beginning to brown and the crust is golden.
        </p>
        <p>
         8. Top with the chopped basil and serve!
        </p>
        <p>
         Enjoy.
        </p>
        <p>
         <a class="grub-popupPhotoLink" grub-popupphotoheight="1027" grub-popupphotopath="/recipe/mini-peach-basil-pizzas/mini-peach-basil-pizza-2/" grub-popupphotowidth="1000" href="#photo/mini-peach-basil-pizza-2" rel="attachment wp-att-3729">
          <img alt="Mini Peach &amp; Basil Pizza" class="alignnone size-grub-wellWidth wp-image-3729" height="616" itemprop="image" src="http://cookingstoned.tv/wp-content/uploads/2013/07/Mini-Peach-Basil-Pizza-2-600x616.jpg" width="600"/>
         </a>
        </p>
       </div>

function _grub_renderInlinePlayer(videoID, width, height) {
    var playerState = 'none', playbackStage = 'loading',
        playbackPollInterval = null, tsIntro, tsContentHalfWay, tsOuttro;

    var onStateChange = function(evt) {
        // We take a somewhat simplified view, ignoring some states (buffering, paused) and only
        // tracking that the video has started or finished

        if (evt.data == YT.PlayerState.PLAYING) {
            if (playerState == 'none') {
                _gaq.push(['_trackEvent', 'Video', 'play', 'inline']);

                playbackPollInterval = setInterval(playbackPollIntervalTick, 500);
            }

            playerState = 'playing';
        } else if (evt.data == YT.PlayerState.ENDED) {
            if (playerState == 'playing') {
                _gaq.push(['_trackEvent', 'Video', 'finish', 'inline']);
                
                if (playbackPollInterval) {
                    clearInterval(playbackPollInterval);
                    playbackPollInterval = null;
                }
            }

            playerState = 'ended';
        }
    };

    var player = new YT.Player('grub_inlinePlayer_' + videoID, {
        width: width,
        height: height,
        videoId: videoID,
        events: {
            onStateChange: onStateChange
        }
    });

    var playbackPollIntervalTick = function() {
        if (playbackStage == 'loading') {
            var duration = player.getDuration();

            if (duration == 0) {
                return;
            } else {
                playbackStage = 'beginning';
                tsIntro = 6;
                tsContentHalfWay = 6 + ((duration - 28) / 2);
                tsOuttro = duration - 22.5;
            }
        }

        var newPlaybackStage, currentTime = player.getCurrentTime();

        if (currentTime &gt; tsOuttro) {
            newPlaybackStage = 'Outtro';
        } else if (currentTime &gt; tsContentHalfWay) {
            newPlaybackStage = 'HalfWay';
        } else if (currentTime &gt; tsIntro) {
            newPlaybackStage = 'Content';
        } else {
            newPlaybackStage = 'beginning';
        }

        if (playbackStage != newPlaybackStage) {
            playbackStage = newPlaybackStage;

            _gaq.push(['_trackEvent', 'Video', 'hit' + newPlaybackStage, 'inline']);
        }
    };
}
