           <p>
            <script type="application/ld+json">

           <div>
            <div>
             <p class="graf--p" style="text-align: left;">
              Its thatinfamous time of year known as 4/20. In case you choose to indulge, here are the best tips for recovery from your festivities
             </p>
             <blockquote>
              <p>
               <a href="https://www.urbandictionary.com/define.php?term=Weed+Hangover" target="_blank">
                Weed hangover
               </a>
               : The morning after a huge smoke session where you wake up after sleeping for twelve hours and feel groggy and lazy.
              </p>
             </blockquote>
             <h2 style="text-align: left;">
              Top 5 Cures for Weed Hangover:
             </h2>
            </div>
            <div>
             <ol class="weed.hangover.cure">
              <li class="graf--li">
               <strong>
                Go for a run.
               </strong>
               Exercise increases blood flow and eliminates brain fog caused by weed hangovers. Youll feel like a new person after only 30 minutes of cardio. Plus, your lungs will benefit too!
               <p>
               </p>
               <div class="wp-caption aligncenter" id="attachment_4465" style="width: 510px">
                <img alt="Running toward sun through a field for mental clarity." class="wp-image-4465 size-full" height="334" src="https://www.getoptimind.com/wp-content/uploads/2015/04/weed-hangover-cure-run.jpg" width="500"/>
                <p class="wp-caption-text">
                 Exercise can relieve hangover symptoms and bring mental clarity.
                </p>
               </div>
              </li>
              <li class="graf--li">
               <strong>
                Take
                <a href="https://www.getoptimind.com/lp/v4?content=brain_fog&amp;utm_source=blog&amp;utm_medium=article&amp;utm_term=brain-health&amp;utm_content=weed-hangover&amp;utm_campaign=mlf" rel="nofollow" style="font-weight: bold;" target="_blank" title="Try OptiMind, a daily nootropic, for free.">
                 OptiMind
                </a>
                .
               </strong>
               Believe it or not, but theres actually a pill for this kind of ill. If you have work the morning following 4/20, try taking nootropics like
               <a class="markup--anchor markup--li-anchor" href="https://www.getoptimind.com/lp/v4?content=brain_fog&amp;utm_source=blog&amp;utm_medium=article&amp;utm_term=brain-health&amp;utm_content=weed-hangover&amp;utm_campaign=mlf" rel="nofollow" target="_blank">
                OptiMind
               </a>
               !OptiMind is formulated to clear mental fog, increase energy, and sharpen your focus within 1 hour. Itsthe best cure for a weed hangover, hands down!
              </li>
              <li class="graf--li">
               <strong>
                Sleep in.
               </strong>
               If youre one of the lucky few whosets theirown hours, then trysleeping in until you wake up sober and straight. Catch up on some ZZZs as your body heals from the night before. Just keep hitting that snooze on your alarm!
              </li>
              <li class="graf--li">
               <strong>
                Take a shower.
               </strong>
               Cant sleep in late to cure your
               <a href="http://www.sensibleseeds.com/blog/can-you-get-a-hangover-from-consuming-too-much-cannabis/" target="_blank">
                marijuana hangover
               </a>
               ? Start with cold water to jolt your system then turn the heat up. Hot water is great for curing a weed hangover because it clearsyour sinuses and soothes them. Youll feel completely refreshed after showering.
              </li>
              <li class="graf--li">
               <strong>
                Get out.
               </strong>
               Talking withyour friends is a great way to get your brain churning after youve slowed it down. Conversation uses several parts of your brain and will easily get your neural activity up!
              </li>
             </ol>
            </div>
            <div>
             <div class="wp-caption alignright" id="attachment_5008" style="width: 310px">
              <img alt="Weed hangover can be brought on by munchies and dehydration." class="size-medium wp-image-5008" height="220" src="https://www.getoptimind.com/wp-content/uploads/2015/04/weed_hangover_crave_case-300x220.jpg" width="300"/>
              <p class="wp-caption-text">
               Crave Case from White Castle may be the iconic stoner meal, but its not going to help your weed hangover.
              </p>
             </div>
             <p class="graf--p">
              If you want to
              <strong class="markup--strong markup--p-strong">
               prevent a marijuanahangover
              </strong>
              , make sure to
              <strong class="markup--strong markup--p-strong">
               stay hydrated and eat properly
              </strong>
              during your weedadventures. Avoid that Crave Case!
             </p>
             <p class="graf--p">
              However, next time you do find yourself waking up with a dry mouth and headache, try one of these methods to cure your weed hangover!For more advice, check out this
              <a href="https://www.getoptimind.com/6-incredibly-easy-ways-to-cure-brain-fog-naturally/?utm_source=Blog&amp;utm_medium=Article&amp;utm_term=weed-hangover&amp;utm_content=cure-weed-hangover&amp;utm_campaign=weed-hangover" rel="nofollow" target="_blank">
               article on clearing brain fog
              </a>
              !
             </p>
             <p class="graf--p">
              <strong>
               My final words
              </strong>
             </p>
             <div>
              <p class="graf--p">
               In case you havent heard of it, weed hangovers are actually quite the commonphenomenon. Symptoms of
               <a href="http://elitedaily.com/life/the-10-steps-to-recovering-from-a-weed-hangover/" target="_blank">
                weed hangover
               </a>
               includecottonmouth, headache, and brain fog. Hangovers from weed can even last up to 2 days with
               <strong>
                difficulty thinking, increased anxiety, and a headache
               </strong>
               .
              </p>
             </div>
             <div>
              <div class="wp-caption alignright" id="attachment_4464" style="width: 310px">
               <img alt="Indica marijuana strains cause weed hangovers, according to some sources." class="wp-image-4464 size-medium" height="245" src="https://www.getoptimind.com/wp-content/uploads/2015/04/marijuana-hangover-indica-300x245.jpg" width="300"/>
               <p class="wp-caption-text">
                Some sources blame indica strains of marijuana for hangover effects.
               </p>
              </div>
              <p class="graf--p">
               Surprisingly, theres some dispute over the existence of a weed hangover.
               <a class="markup--anchor markup--p-anchor" href="https://www.theweedblog.com/marijuana-mythbusters-the-marijuana-hangover/" target="_blank">
                TheWeedBlog
               </a>
               argues that its just waking up still high. That means youre still feeling the effects of marijuana from the night before. Well, scientists dont agree. A study from the
               <a class="markup--anchor markup--p-anchor" href="https://www.ncbi.nlm.nih.gov/pubmed/2992898" target="_blank">
                National Institute of Health
               </a>
               examined 13 men smoking marijuana and tested them for hangover symptoms. Theyuncovered
               <strong>
                hangover effects that were remarkablydifferent from active marijuanaeffects
               </strong>
               .
              </p>
              <p class="graf--p">
               Cannabis connoisseurs discuss the smoking experience and side effects on forums, like
               <a href="http://www.reddit.com/r/trees/" target="_blank">
                reddit.com/r/trees
               </a>
               . Some chuck the hangoverup to the
               <strong>
                strain of bud
               </strong>
               , pointing to Indica or Indica-dominant strains. Others argue that
               <strong>
                tolerance is to blame
               </strong>
               for a marijuanahangover. However, pot veterans report hangover symptoms.
              </p>
              <p class="graf--p">
               Regardless of the cause for a weed hangover,dont let 4/20 ruin your work week! Use these practicalcures for your weed hangover this April 21st!
              </p>

    hashToCookie(flatQueryParams());
  }

  function setFirstLastName() {
    var cookies = getCookies();
    if (cookies['user[name]'] &amp;&amp; !cookies['user[first_name]']) {
      var names = extractFirstLastName(cookies['user[name]']);
      setCookie('user[first_name]', names.first, 90, topLevelDomain());
      setCookie('user[last_name]', names.last, 90, topLevelDomain());
    }
  }

  // super_param cookies
  function setCookieGUID() {
    var cookies = getCookies();
    if (!cookies.cookie_guid) {
      setCookie('cookie_guid', guid(), 180, topLevelDomain());
    }
  }

  function setInitialURL() {
    var cookies = getCookies();
    if (!cookies.initial_url) {
      setCookie('initial_url', window.location.href, 180, topLevelDomain());
    }
  }

  function setInitialReferer() {
    var cookies = getCookies();
    if (!cookies.initial_referer) {
      setCookie('initial_referer', document.referrer, 180, topLevelDomain());
    }
  }

  function setLandingPageURL() {
    var cookies = getCookies();
    var url = window.location.href;
    var lp = window.location.href.match(/\/lp\//);
    if (!cookies.lp_url &amp;&amp; lp) {
      setCookie('lp_url', url, 180, topLevelDomain());
    }
  }

  function setLandingPageVersion(version) {
    var cookies = getCookies();
    if (!cookies.lp_version) {
      setCookie('lp_version', version, 180, topLevelDomain());
    }
  }

  // param cookies
  function setProduct(product) {
    setCookie('product_handle', product, 180, topLevelDomain());
  }

  function setLastPageType(type) {
    if(type) {
      setCookie('page_type', type, 180, topLevelDomain());
    }
  }

  function setStepNumber(step) {
    var cookies = getCookies();
    var previous_step = (cookies['step']) ? cookies['step'] : '0';
    if(step &gt; previous_step){
      setCookie('step', step, 180, topLevelDomain());
    }
  }

  function setReferrerUser(id)
  {
    var cookies = getCookies();
    if (!cookies['r'])
    {
      setCookie('r', id, 180, topLevelDomain());
    }
  }

  function cookiesEnabled() {
    if (navigator.cookieEnabled) {
      return true;
    } else {
      // document.getElementById('require_cookies_warning').style.display = 'block';
      return false;
    }
  }

  if(cookiesEnabled()) {
    setCookieGUID();
    setURLParamsCookie();
    setFirstLastName();
    setInitialURL();
    setInitialReferer();
    setLandingPageURL();
  }
  </script>
