       Nokia Lumia 2520 Windows RT 8.1 tablet goes on sale

      <div class="article-ad">

      <p>
       <img alt="lumiatablet1" class="feature-image-left" height="147" src="http://tg.mmxdev.com/wp-content/uploads/lumiatablet1.jpg" style="float: left;" width="168"/>
       Nokias first ever tablet  the Lumia 2520  will be available in stores from this week preloaded with Windows RT 8.1 and 4G/LTE and wi-fi connectivity.
      </p>
      <p>
       The Nokia Lumia 2520 has a 10.1-inch high definition display and has 2GB RAM and 32GB internal memory which is expandable with a microSD card slot.
      </p>
      <p>
       The device will be available in Telstra stores from this week before being offered in more stores nationally from May 13.
      </p>
      <p>
       <span id="more-56844">
       </span>
      </p>
      <p>
       Onboard is a vivid display with enough brightness and clarity to be easily read inside and out and a battery that can power through for up to 11 hours.
      </p>
      <p>
       It also supports fast charging with the ability to get to 80 per cent battery after just one hour of charging.
      </p>
      <p>
       <img alt="The Nokia Lumia 2520 tablet is available now" height="494" src="http://tg.mmxdev.com/wp-content/uploads/lumiatablet3.jpg" style="vertical-align: middle;" width="640"/>
      </p>
      <p>
       Theres also a 6.7-megapixel camera complete with Zeiss optics  the first time these optics have been used on a tablet  and combine with proprietary Nokia imaging algorithm to help you take superb pictures.
      </p>
      <p>
       Users can pair the Lumia 2520 with the Nokia Power Keyboard which is not only a fully functional keyboard  it also two USB ports and gesture trackpad.
      </p>
      <p>
       The Nokia Lumia 2520 can be purchased on the Telstra $35 Tablet Plan plus $25 per month device repayment for 24 hours which includes 4GB of data.
      </p>
      <p>
       <img alt="The Nokia Lumia 2520 10.1-inch tablet is avaiable in a number of colours" height="406" src="http://tg.mmxdev.com/wp-content/uploads/lumiatablet2.jpg" style="vertical-align: middle;" width="640"/>
      </p>
      <p>
       Customers can also buy the Nokia Lumia 2520 in black outright for $840. The Nokia Power Keyboard is priced at $240.
      </p>

(function($) {
 
  "use strict";
 
  // current page host
  var baseURI = window.location.host;
 
  // click event on body
  $("body").on("click", function(e) {
 
    // abandon if link already aborted or analytics is not available
    if (e.isDefaultPrevented() || typeof ga !== "function") return;
 
    // abandon if no active link or link within domain
    var link = $(e.target).closest("a");
    if (link.length != 1 || baseURI == link[0].host) return;
 
    // cancel event and record outbound link
    e.preventDefault();
    var href = link[0].href;
    ga('send', {

      'eventCategory': 'outbound',
      'eventAction': 'link',
      'eventLabel': href,
      'hitCallback': loadPage
    });
 
    // redirect after one second if recording takes too long
    setTimeout(loadPage, 1000);
 
    // redirect to outbound page
    function loadPage() {
      document.location = href;
    }
 
  });
 
})(jQuery); // pass another library here if required
