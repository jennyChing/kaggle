                     The Real Meal Revolution - Eat Fat, Feel Better, and lose weight!

                    <div class="post-body entry-content" id="post-body-6530018930278242050" itemprop="articleBody">
                     <div class="separator" style="clear: both; text-align: center;">
                     </div>
                     <div class="separator" style="margin-left: 1em; margin-right: 1em;">
                      <br/>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       There are some people who are health conscious yet they are obese, there are some who are very athletic yet have high blood pressure, some look fit but they have diabetes, what's wrong? I myself am one of them. I amasthmatic! Could the book
                       <b>
                        THE REAL MEAL REVOLUTION by TimNoakes
                       </b>
                       be the answer?
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       <br/>
                      </span>
                     </div>
                     <div class="separator" style="clear: both; text-align: center;">
                      <img border="0" height="400" src="https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcT2Y5104mDip5ohV5dcL23NLEBM11rVQMMFYPIIzBF5R2ChYAaE" width="314"/>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       <br/>
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       A book so controversial, it's given South Africa a different perspective on how people should be eating and what they are putting into their mouths. One of the best sellers in South Africa,
                       <span style="font-size: large;">
                        <b>
                         THE REAL MEAL REVOLUTION
                        </b>
                       </span>
                       is going to hit the Philippines by storm and I want to be the first one in our country to start this meal revolution.
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       <br/>
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       This book includes all the research and different studies they made, the foods that you must and must not eat if you want to practice the Low Carb High Fat (LCHF) way of eating, and delicious and simple recipes for everyone to enjoy.
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       <br/>
                      </span>
                     </div>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://banting121.co.za/wp-content/uploads/2014/06/lchf-diet_538d80ba46464_w1500.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" height="400" src="http://banting121.co.za/wp-content/uploads/2014/06/lchf-diet_538d80ba46464_w1500.jpg" width="282"/>
                      </a>
                     </div>
                     <div class="separator" style="clear: both; text-align: center;">
                      Source: banting121.co.za
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       <br/>
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       This book promotes the
                      </span>
                      <span style="font-family: 'Helvetica Neue', Arial, Helvetica, sans-serif;">
                       Low Carb High Fat (LCHF)
                      </span>
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       way of eating.
                      </span>
                      <span style="font-family: 'Helvetica Neue', Arial, Helvetica, sans-serif;">
                       Inspired by Sir Frederick Grant Banting.
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       <br/>
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif; font-size: x-large;">
                       <b>
                        10 Golden Rules of Banting
                       </b>
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       (Source: Health24.com)
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       <br/>
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       1. Remember: this is not a high protein diet. It's a high fat, medium protein, low carb way of eating.
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       2. Choose real foods that look like what they are, and cook them from scratch
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       3. Fat is not the enemy. Enjoy it!
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       4. Eat only when you are hungry; eat until you are satisfied - then stop
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       5. Don't eat when you're not hungry. You won't die if you occasionally skip a meal you don't feel like eating.
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       6. Stop snacking. You won't need to - it's just a habit.
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       7. No sugar. It's an addiction, and it's probably best to go cold turkey. But if you need to make it a transition, substitute with Stevia, Xylitol or Erythritol - NOT artificial sweeteners.
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       8. No grains of any kind
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       9. No (or very, very little) fruit. Think of it as a sweet rather than a health snack.
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       10. Embrace eggs. They're healthy, satisfying and very good for you.
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       <br/>
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       <br/>
                      </span>
                     </div>
                     <div class="separator" style="clear: both; text-align: center;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       <img border="0" height="300" src="http://4.bp.blogspot.com/-0lNNRlmL3d0/U-Ni0eYLYVI/AAAAAAAAMjw/9d1QC9uT-ak/s1600/P7110427.JPG" width="400"/>
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       <br/>
                      </span>
                     </div>
                     <div class="separator" style="clear: both;">
                      <span style="font-family: Helvetica Neue, Arial, Helvetica, sans-serif;">
                       During my visit to the National Arts Festival in Grahamstown, South Africa, Tim Noake's who is one of the major contributors who started this meal revolution conducted a talk about this book.
                      </span>

              &amp;&amp; comment.author.profileUrl == config.authorUrl;
          return matches ? 'true' : '';
        } else if ('deletelink' == key) {
          return config.baseUri + '/delete-comment.g?blogID='
               + config.blogId + '&amp;postID=' + comment.id;
        } else if ('deleteclass' == key) {
          return comment.deleteclass;
        }
        return '';
      };

      var replybox = null;
      var replyUrlParts = null;
      var replyParent = undefined;

      var onReply = function(commentId, domId) {
        if (replybox == null) {
          // lazily cache replybox, and adjust to suit this style:
          replybox = document.getElementById('comment-editor');
          if (replybox != null) {
            replybox.height = '250px';
            replybox.style.display = 'block';
            replyUrlParts = replybox.src.split('#');
          }
        }
        if (replybox &amp;&amp; (commentId !== replyParent)) {
          document.getElementById(domId).insertBefore(replybox, null);
          replybox.src = replyUrlParts[0]
              + (commentId ? '&amp;parentID=' + commentId : '')
              + '#' + replyUrlParts[1];
          replyParent = commentId;
        }
      };

      var hash = (window.location.hash || '#').substring(1);
      var startThread, targetComment;
      if (/^comment-form_/.test(hash)) {
        startThread = hash.substring('comment-form_'.length);
      } else if (/^c[0-9]+$/.test(hash)) {
        targetComment = hash.substring(1);
      }

      // Configure commenting API:
      var configJso = {
        'maxDepth': config.maxThreadDepth
      };
      var provider = {
        'id': config.postId,
        'data': items,
        'loadNext': paginator,
        'hasMore': hasMore,
        'getMeta': getMeta,
        'onReply': onReply,
        'rendered': true,
        'initComment': targetComment,
        'initReplyThread': startThread,
        'config': configJso,
        'messages': msgs
      };

      var render = function() {
        if (window.goog &amp;&amp; window.goog.comments) {
          var holder = document.getElementById('comment-holder');
          window.goog.comments.render(holder, provider);
        }
      };

      // render now, or queue to render when library loads:
      if (window.goog &amp;&amp; window.goog.comments) {
        render();
      } else {
        window.goog = window.goog || {};
        window.goog.comments = window.goog.comments || {};
        window.goog.comments.loadQueue = window.goog.comments.loadQueue || [];
        window.goog.comments.loadQueue.push(render);
      }
    })();
// ]]&gt;
                     </script>
