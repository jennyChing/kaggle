        <h1>
         Live Chat
        </h1>
        <img alt="Automatic Screenshot Tracker" class="screen" src="/Content/images/inner/chat-screenshot.jpg"/>
        <p>
         <span style="">
          Talygen Live Chat Socialize module is a web chat that allows Users to communicate within in a Company. Users can create groups for group chat and can also chat with individual person. In addition, every User who is included in the group chat or single chat can view their history logs.
         </span>
        </p>
        <h3 style="padding:40px 0px 10px 0px;">
         Key Features
        </h3>
        <div class="section_row">
         <div class="section_right">
          <div class="short_features">
           <div class="short_icon">
            <img alt="" src="/Content/images/inner/tourShorticon/groupchat.png"/>
           </div>
           <div class="short_content">
            <h4>
             <b>
              <span style="font-size:12.0pt;">
               Group Chat
              </span>
             </b>
            </h4>
            <p>
             <span style="">
              Users can easily interact in group. In group chat mode, multiple Users can be added. Users can chat with other online Users or leave messages to offline Users
             </span>
             .
            </p>
           </div>
          </div>
          <div class="short_features">
           <div class="short_icon">
            <img alt="" src="/Content/images/inner/tourShorticon/fileSharing.png"/>
           </div>
           <div class="short_content">
            <h4>
             <b>
              <span style="font-size:12.0pt;">
               File sharing
              </span>
             </b>
            </h4>
            <p>
             <span style="">
              Users can share files easily with Talygens Live Chat module. You can share documents, images, screenshots and many more things.
             </span>
             .
            </p>
           </div>
          </div>
          <div class="short_features">
           <div class="short_icon">
            <img alt="" src="/Content/images/inner/tourShorticon/viewlog.png"/>
           </div>
           <div class="short_content">
            <h4>
             <b>
              <span style="font-size:12.0pt;">
               Log View
              </span>
             </b>
            </h4>
            <p>
             <span style="">
              The message gets automatically saved in the database and thus allows Users to view the communication history from within the chat window. The Users included in the group chat can access the logs.
             </span>
            </p>
            <p>
             .
            </p>
           </div>
          </div>
