      <h3>
       Excerpt
      </h3>
      <div class="compressed">
       <p>
        Treating the internet and all its sources as a database, YQL seeks to allow developers to explore government, social, api and all other external data in a standardized way.  Further allowing developers to manipulate this data and mash different sources together, YQL works to open up the web and all its sources.
       </p>
      </div>
      <h3>
       Description
      </h3>
      <p>
       The Yahoo! Query Language provides a rich and dynamic method for obtaining and manipulating data from any source or
       <span class="caps">
        API
       </span>
       on the internet  with
       <span class="caps">
        YQL
       </span>
       the internet becomes your database.  Using the simplified
       <span class="caps">
        SQL
       </span>
       syntax that
       <span class="caps">
        YQL
       </span>
       is based in,
       <span class="caps">
        YQL
       </span>
       seeks to open all data on the web into a standardized format.  Manipulating and mashing up sources as if they were tables,
       <span class="caps">
        YQL
       </span>
       becomes a repository for exploring government, event, social and
       <span class="caps">
        API
       </span>
       data on the web.
      </p>
      <p>
       This talk will cover the core techniques within
       <span class="caps">
        YQL
       </span>
       , including server-side JavaScript with native E4X support for manipulating data, key / value pair data storage and the process of creating your own
       <span class="caps">
        YQL
       </span>
       tables for accessing web based content.  Going further with the integration of the open authentication standards defined by OAuth, well delve into advanced authentication techniques using this standard within
       <span class="caps">
        YQL
       </span>
       .
      </p>
