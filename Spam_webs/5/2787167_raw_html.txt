                    <h3 class="post-title entry-title">
                     TheBrain helps keep your thoughts straight
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-3970338296686995184">
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://3.bp.blogspot.com/-53Aij5GyMRg/TtpfGWao40I/AAAAAAAAAtw/sCgva-qAp8I/s1600/thebrain.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" height="267" src="http://3.bp.blogspot.com/-53Aij5GyMRg/TtpfGWao40I/AAAAAAAAAtw/sCgva-qAp8I/s320/thebrain.jpg" width="320"/>
                      </a>
                     </div>
                     <span class="Apple-style-span" style="font-family: Arial, Helvetica, sans-serif;">
                      <br/>
                     </span>
                     <br/>
                     <span class="Apple-style-span" style="font-family: Arial, Helvetica, sans-serif;">
                      What if you were able to put your entire brain into one computer program? Every thought, work-related or personal, with links to
                      <a class="zem_slink" href="http://en.wikipedia.org/wiki/Web_page" rel="wikipedia" title="Web page">
                       Web pages
                      </a>
                      or files on your computer, and any additional notes you'd care to make. And what if you could then link those
                      <a class="zem_slink" href="http://en.wikipedia.org/wiki/Thought" rel="wikipedia" title="Thought">
                       thoughts
                      </a>
                      together, weaving them into free and complex associative patterns, much like an actual train of thought going through your head? That's what TheBrain tries to let you do.
                     </span>
                     <br/>
                     <span class="Apple-style-span" style="font-family: Arial, Helvetica, sans-serif;">
                      <br/>
                     </span>
                     <br/>
                     <span class="Apple-style-span" style="font-family: Arial, Helvetica, sans-serif;">
                      At its core,
                      <a href="http://www.thebrain.com/">
                       TheBrain 7.0
                      </a>
                      is a powerful and flexible mind-mapping program. If you're looking to create a simple
                      <a class="zem_slink" href="http://en.wikipedia.org/wiki/Mind_map" rel="wikipedia" title="Mind map">
                       mind map
                      </a>
                      for just one project, you could always go with a minimalistic free app like Blumind, or even with mind-mapper favorite
                      <a class="zem_slink" href="http://freemind.sourceforge.net/wiki/index.php/Main_Page" rel="homepage" title="FreeMind">
                       Freemind
                      </a>
                      . But if you'd like to create a vast mind map which covers a lot of ground, TheBrain might just be what you need (and its free version retains lots of functionality).
                     </span>
                     <br/>
                     <span class="Apple-style-span" style="font-family: Arial, Helvetica, sans-serif;">
                      <br/>
                     </span>
                     <br/>
                     <span class="Apple-style-span" style="font-family: Arial, Helvetica, sans-serif;">
                      The first thing you see when creating a new brain is just a single "thought" against a background called "the
                      <a class="zem_slink" href="http://www.plexapp.com/" rel="homepage" title="Plex (software)">
                       Plex
                      </a>
                      ." In TheBrain, a thought is just a name for a node in your mind map--much like an "idea" in online mind-mapper
                      <a class="zem_slink" href="http://www.mindmeister.com/" rel="homepage" title="MindMeister">
                       MindMeister
                      </a>
                      . Each thought can have multiple siblings, multiple children (sub-thoughts), and even multiple parents. That last one is not an obvious feature, and allows for creating very complex layouts. For example, in a film-related mind map, you could have actor
                      <a class="zem_slink" href="http://www.rottentomatoes.com/celebrity/keanu_reeves" rel="rottentomatoes" title="Keanu Reeves">
                       Keanu Reeves
                      </a>
                      both under "male heroes" and under "
                      <a class="zem_slink" href="http://www.rottentomatoes.com/m/matrix" rel="rottentomatoes" title="The Matrix">
                       Matrix
                      </a>
                      cast."
                     </span>
                     <br/>
                     <span class="Apple-style-span" style="font-family: Arial, Helvetica, sans-serif;">
                      <br/>
                     </span>
                     <br/>
                     <span class="Apple-style-span" style="font-family: Arial, Helvetica, sans-serif;">
                      The Plex itself hasn't changed much since I first used TheBrain (when it was still called
                      <a class="zem_slink" href="http://www.thebrain.com/" rel="homepage" title="PersonalBrain">
                       PersonalBrain
                      </a>
                      ), about five years ago. It is still visually impressive and fun to use. As you click a thought you'd like to focus on, it smoothly floats over to the center of the Plex, and the other thoughts get rearranged (or shown or hide), all with slick, futuristic animations.
                     </span>
                     <br/>
                     <span class="Apple-style-span" style="font-family: Arial, Helvetica, sans-serif;">
                      <br/>
                     </span>
                     <br/>
                     <span class="Apple-style-span" style="font-family: Arial, Helvetica, sans-serif;">
                      One major visual change for TheBrain 7 is that links between thoughts are now curved, rather than simple straight lines. That sounds minor, but when you're working on a huge mind map, it does make a visual impact.
                     </span>
                     <br/>
                     <span class="Apple-style-span" style="font-family: Arial, Helvetica, sans-serif;">
                      <br/>
                     </span>
                     <br/>
                     <span class="Apple-style-span" style="font-family: Arial, Helvetica, sans-serif;">
                      Official WebSite:
                      <a href="http://www.thebrain.com/">
                       TheBrain.com
                      </a>
                     </span>
                     <br/>
                     <span class="Apple-style-span" style="font-family: Arial, Helvetica, sans-serif;">
