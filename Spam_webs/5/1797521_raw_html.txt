                    <h3 class="post-title entry-title">
                     [Windows Phone, XBOX, iOS, Android, OUYA] Square Off #game #review
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-4361244979061688814">
                     Today in focus is
                     <a href="http://www.gnomicstudios.com/square-off/">
                      Square Off
                     </a>
                     . It's my first #game review with
                     <a href="https://www.ouya.tv/game/Square-Off/">
                      OUYA
                     </a>
                     version :)
                     <br/>
                     <br/>
                     First of all I like loaded trip-hop sound track, then oldschool graphics stylization in intro and best of all that there is a version for my Windows Phone :)
                     <a href="http://www.windowsphone.com/en-us/store/app/square-off-free/6a8eef97-9b36-4caa-8d4b-bea5732406de">
                      Free version
                     </a>
                     has permanent banner and completely playable besides this limitation.
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://www.windowsphone.com/en-us/store/app/square-off/3a3e5ed5-6570-e011-81d2-78e7d1fa76f8">
                       <img alt="http://www.windowsphone.com/en-us/store/app/square-off/3a3e5ed5-6570-e011-81d2-78e7d1fa76f8" border="0" height="240" src="http://2.bp.blogspot.com/-49arcrUfVjA/Utbt6znXNkI/AAAAAAAAESY/8dMwIO3VCcw/s1600/DSCN2701.JPG" width="320"/>
                      </a>
                     </div>
                     <br/>
                     In game music is still great, active and not too electronic. Controls are similar to
                     <a href="http://www.windowsphone.com/en-us/store/app/halo-sa-lite/cf3f117d-d5a6-4e81-9786-56dd337b9b02">
                      Halo: Spartan Assault
                     </a>
                     (next in review queue), now too easy to jump in for #PC #gamer and looks like the only solution for the #shooter on smartphone and natural for #console #gamers.
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://3.bp.blogspot.com/-HLSJKQKZDdY/UtbwxV1VW7I/AAAAAAAAESg/1BdjsuVBl1g/s1600/depth.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" height="192" src="http://3.bp.blogspot.com/-HLSJKQKZDdY/UtbwxV1VW7I/AAAAAAAAESg/1BdjsuVBl1g/s1600/depth.png" width="320"/>
                      </a>
                     </div>
                     <br/>
                     Cartoon graphics, perspective and depth for 2D view enhancement and proper #hardcore rush gameplay makes really good game. I've played several levels and going to play more and maybe buy the full version too. And just forgotten a bit too cute but still great designed characters and aliens for sure :P
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://4.bp.blogspot.com/-7adhWdfZ1v4/Utbw8v8p_eI/AAAAAAAAESo/EWjzknOfraU/s1600/characters.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" height="192" src="http://4.bp.blogspot.com/-7adhWdfZ1v4/Utbw8v8p_eI/AAAAAAAAESo/EWjzknOfraU/s1600/characters.png" width="320"/>
                      </a>
                     </div>
                     <br/>
                     Available on
                     <a href="http://marketplace.xbox.com/en-US/Product/Square-Off/66acd000-77fe-1000-9115-d802585503c0">
                      Xbox 360
                     </a>
                     ,
                     <a href="https://itunes.apple.com/au/app/square-off-free/id738256121?mt=8&amp;ign-mpt=uo%3D4">
                      iOS
                     </a>
                     ,
                     <a href="https://play.google.com/store/apps/details?id=squareoffgame.android">
                      Android
                     </a>
                     and
                     <a href="https://www.ouya.tv/game/Square-Off/">
                      OUYA
                     </a>
                     , 9/10 Recommended.
                     <br/>
                     <br/>
                     Have an amazing day, see You, work less, play more (c) ;)
