   <h1 class="firstHeading" id="firstHeading" lang="en">
    Jordan matrix
   </h1>
   <div class="mw-body-content" id="bodyContent">
    <div id="siteSub">
     From Wikipedia, the free encyclopedia
    </div>
    <div id="contentSub">
    </div>
    <div class="mw-jump" id="jump-to-nav">
     Jump to:
     <a href="#mw-head">
      navigation
     </a>
     ,
     <a href="#p-search">
      search
     </a>
    </div>
    <div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en">
     <p>
      In the
      <a href="/wiki/Mathematics" title="Mathematics">
       mathematical
      </a>
      discipline of
      <a href="/wiki/Matrix_(mathematics)" title="Matrix (mathematics)">
       matrix theory
      </a>
      , a
      <b>
       Jordan block
      </b>
      over a
      <a href="/wiki/Ring_(mathematics)" title="Ring (mathematics)">
       ring
      </a>
      <img alt="R" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/e/1/e/e1e1d3d40573127e9ee0480caf1283d6.png"/>
      (whose
      <a href="/wiki/Identity_element" title="Identity element">
       identities
      </a>
      are the
      <a href="/wiki/0_(number)" title="0 (number)">
       zero
      </a>
      0 and
      <a href="/wiki/1_(number)" title="1 (number)">
       one
      </a>
      1) is a
      <a href="/wiki/Matrix_(mathematics)" title="Matrix (mathematics)">
       matrix
      </a>
      composed of 0 elements everywhere except for the diagonal, which is filled with a fixed element
      <img alt="\lambda\in R" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/2/b/4/2b49203ff489ae15921308ceadaf398b.png"/>
      , and for the
      <a class="mw-redirect" href="/wiki/Superdiagonal" title="Superdiagonal">
       superdiagonal
      </a>
      , which is composed of ones. The concept is named after
      <a href="/wiki/Camille_Jordan" title="Camille Jordan">
       Camille Jordan
      </a>
      .
     </p>
     <dl>
      <dd>
       <img alt="\begin{pmatrix}
\lambda &amp; 1       &amp; 0       &amp; \cdots  &amp; 0 \\
0       &amp; \lambda &amp; 1       &amp; \cdots  &amp; 0 \\
\vdots  &amp; \vdots  &amp; \vdots&amp; \ddots  &amp; \vdots \\
0       &amp; 0       &amp; 0        &amp; \lambda &amp; 1       \\
0       &amp; 0       &amp; 0       &amp; 0       &amp; \lambda \end{pmatrix}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/4/0/8/408917f56324954e6db7ba1b1e138291.png"/>
      </dd>
     </dl>
     <p>
      Every Jordan block is thus specified by its dimension
      <i>
       n
      </i>
      and its
      <a class="mw-redirect" href="/wiki/Eigenvalue" title="Eigenvalue">
       eigenvalue
      </a>
      <img alt="\lambda" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/e/0/5/e05a30d96800384dd38b22851322a6b5.png"/>
      and is indicated as
      <img alt="J_{\lambda,n}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/a/b/1/ab17c35a0e06f5d38a1142c43ade8aed.png"/>
      . Any
      <a href="/wiki/Block_matrix" title="Block matrix">
       block diagonal matrix
      </a>
      whose blocks are Jordan blocks is called a
      <b>
       Jordan matrix
      </b>
      ; using either the
      <img alt="\oplus" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/b/7/1/b71edd70fcad670e99a9912ba5e55d77.png"/>
      or the 
      <img alt="\mathrm{diag}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/9/c/2/9c28434fbeb69849b6b233c8f8bb6eab.png"/>
       symbol, the
      <img alt="(l+m+n)\times (l+m+n)" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/8/a/2/8a2f399eda42a68fc69721cc7ab449bc.png"/>
      block diagonal square matrix whose first diagonal block is
      <img alt="J_{\alpha,l}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/4/2/6/4268785b12e951e72a0d26d78459cd83.png"/>
      , whose second diagonal block is
      <img alt="J_{\beta,m}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/3/7/5/375e5eb2dfb643e8722c9b29d171655c.png"/>
      and whose third diagonal block is
      <img alt="J_{\gamma,n}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/6/d/2/6d259f65044f503e803812af02d829b5.png"/>
      is compactly indicated as
      <img alt="J_{\alpha,l}\oplus J_{\beta,m}\oplus J_{\gamma,n}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/e/b/a/ebaff15d055e15b8815e41ce87d449b1.png"/>
      or
      <img alt="\mathrm{diag}\left(J_{\alpha,l}, J_{\beta,m}, J_{\gamma,n}\right)" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/3/8/d/38d135b7d6c65ee501bccc595c0f8e20.png"/>
      , respectively. For example the matrix
     </p>
     <dl>
      <dd>
       <img alt="
J=\left(\begin{matrix}
0 &amp; 1 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 \\
0 &amp; 0 &amp; 1 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 \\
0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 \\
0 &amp; 0 &amp; 0 &amp; i &amp; 1 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 \\
0 &amp; 0 &amp; 0 &amp; 0 &amp; i &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 \\
0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; i &amp; 1 &amp; 0 &amp; 0 &amp; 0 \\
0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; i &amp; 0 &amp; 0 &amp; 0 \\
0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 7 &amp; 1 &amp; 0 \\
0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 7 &amp; 1 \\
0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 0 &amp; 7 \end{matrix}\right)" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/9/7/2/972497838197c0d58393c9979b6378d6.png"/>
      </dd>
     </dl>
     <p>
      is a
      <img alt="10\times 10" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/c/c/7cc2dc5a0d351d6e6e0929b4a4f03dc5.png"/>
      Jordan matrix with a
      <img alt="3\times 3" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/b/a/0/ba0dffbf721ff09756465187f3aca372.png"/>
      block with
      <a class="mw-redirect" href="/wiki/Eigenvalue" title="Eigenvalue">
       eigenvalue
      </a>
      <img alt="0" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/c/f/c/cfcd208495d565ef66e7dff9f98764da.png"/>
      , two
      <img alt="2\times 2" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/8/6/0/8608c2f6f82c35cf210f370f7885f1d0.png"/>
      blocks with eigenvalue the
      <a href="/wiki/Imaginary_unit" title="Imaginary unit">
       imaginary unit
      </a>
      and a
      <img alt="3\times 3" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/b/a/0/ba0dffbf721ff09756465187f3aca372.png"/>
      block with eigenvalue7. Its Jordan-block structure can also be written as either
      <img alt="J_{0,3}\oplus J_{i,2}\oplus J_{i,2}\oplus J_{7,3}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/6/6/6/66606dd7a5ff96af2456f4615684e270.png"/>
      or
      <img alt="\mathrm{diag}\left(J_{0,3},J_{i,2},J_{i,2},J_{7,3}\right)" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/a/2/9/a29cd0451c020e67ed32f180e408301b.png"/>
      .
     </p>

     </p>

     <h2>
      <span class="mw-headline" id="Linear_algebra">
       Linear algebra
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Jordan_matrix&amp;action=edit&amp;section=1" title="Edit section: Linear algebra">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      Any
      <img alt="n\times n" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/6/0/7/607acaa73c762411b20745149a11e90b.png"/>
      square matrix
      <img alt="A" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/f/c/7fc56270e7a70fa81a5935b72eacbe29.png"/>
      whose elements are in an
      <a href="/wiki/Algebraically_closed_field" title="Algebraically closed field">
       algebraically closed field
      </a>
      <img alt="K" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/a/5/f/a5f3c6a11b03839d46af9fb43c97c188.png"/>
      is
      <a href="/wiki/Matrix_similarity" title="Matrix similarity">
       similar
      </a>
      to a Jordan matrix
      <img alt="J" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/f/f/4/ff44570aca8241914870afbc310cdb85.png"/>
      , also in
      <img alt="\mathbb{M}_n (K)" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/4/5/4/454341fe161331672fd6f08a1332b162.png"/>
      , which is unique up to a permutation of its diagonal blocks themselves.
      <img alt="J" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/f/f/4/ff44570aca8241914870afbc310cdb85.png"/>
      is called the
      <a href="/wiki/Jordan_normal_form" title="Jordan normal form">
       Jordan normal form
      </a>
      of
      <img alt="A" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/f/c/7fc56270e7a70fa81a5935b72eacbe29.png"/>
      and corresponds to a generalization of the diagonalization procedure.
      <sup class="reference" id="cite_ref-1">
       <a href="#cite_note-1">
        <span>
         [
        </span>
        1
        <span>
         ]
        </span>
       </a>
      </sup>
      <sup class="reference" id="cite_ref-2">
       <a href="#cite_note-2">
        <span>
         [
        </span>
        2
        <span>
         ]
        </span>
       </a>
      </sup>
      <sup class="reference" id="cite_ref-3">
       <a href="#cite_note-3">
        <span>
         [
        </span>
        3
        <span>
         ]
        </span>
       </a>
      </sup>
      A
      <a href="/wiki/Diagonalizable_matrix" title="Diagonalizable matrix">
       diagonalizable matrix
      </a>
      is similar, in fact, to a special case of Jordan matrix: the matrix whose blocks are all
      <img alt="1\times 1" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/b/9/9/b99a31f00d8eff828fb2b1657efe2f4b.png"/>
      .
      <sup class="reference" id="cite_ref-4">
       <a href="#cite_note-4">
        <span>
         [
        </span>
        4
        <span>
         ]
        </span>
       </a>
      </sup>
      <sup class="reference" id="cite_ref-5">
       <a href="#cite_note-5">
        <span>
         [
        </span>
        5
        <span>
         ]
        </span>
       </a>
      </sup>
      <sup class="reference" id="cite_ref-6">
       <a href="#cite_note-6">
        <span>
         [
        </span>
        6
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <p>
      More generally, given a Jordan matrix
      <img alt="J=J_{\lambda_1,m_1}\oplus J_{\lambda_2,m_2} \oplus\ldots\oplus J_{\lambda_N,m_N}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/5/9/a/59a3f7cbf71231fe86636161effbce24.png"/>
      , i.e. whose
      <img alt="k^\text{th}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/6/2/0/620f0ee72494a57ff9905ca09afd8369.png"/>
      diagonal block,
      <img alt="1\leq k\leq N" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/9/b/a/9ba466271b9d26fd13d680dbf369cd82.png"/>
      is the Jordan block
      <img alt="J_{\lambda_k,m_k}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/f/a/0/fa08a34aecaa0cf5e59f333ebcc0170d.png"/>
      and whose diagonal elements
      <img alt="\lambda_k" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/8/2/6/826d0ea0eb8713b67a505b4f78d61b40.png"/>
      may not all be distinct, the
      <a class="mw-redirect" href="/wiki/Geometric_multiplicity" title="Geometric multiplicity">
       geometric multiplicity
      </a>
      of
      <img alt="\lambda\in K" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/1/1/7/117c33d7e9e2dc477b8404376e5e676c.png"/>
      for the matrix
      <img alt="J" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/f/f/4/ff44570aca8241914870afbc310cdb85.png"/>
      , indicated as
      <img alt="\mathrm{gmul}_J \lambda\," class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/9/5/1/951604acb4c31a8e4796c0512e259145.png"/>
      , corresponds to the number of Jordan blocks whose eigenvalue is
      <img alt="\lambda" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/e/0/5/e05a30d96800384dd38b22851322a6b5.png"/>
      . Whereas the
      <b>
       index
      </b>
      of an eigenvalue
      <img alt="\lambda" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/e/0/5/e05a30d96800384dd38b22851322a6b5.png"/>
      for
      <img alt="J" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/f/f/4/ff44570aca8241914870afbc310cdb85.png"/>
      , indicated as
      <img alt="\mathrm{idx}_J \lambda\," class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/8/d/6/8d6bc7b01575a6675e2a40f19c6b3829.png"/>
      , is defined as the dimension of the largest Jordan block associated to that eigenvalue.
     </p>
     <p>
      The same goes for all the matrices
      <img alt="A" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/f/c/7fc56270e7a70fa81a5935b72eacbe29.png"/>
      similar to
      <img alt="J" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/f/f/4/ff44570aca8241914870afbc310cdb85.png"/>
      , so
      <img alt="\mathrm{idx}_A \lambda\," class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/4/3/d/43def7b2aae1fadc7577fe28b003e69e.png"/>
      can be defined accordingly with respect to the
      <a href="/wiki/Jordan_normal_form" title="Jordan normal form">
       Jordan normal form
      </a>
      of
      <img alt="A" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/f/c/7fc56270e7a70fa81a5935b72eacbe29.png"/>
      for any of its eigenvalues
      <img alt="\lambda \in\mathrm{spec}A" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/0/0/9/00934fbf0ad3dca621ea7a420fbe0d0d.png"/>
      . In this case one can check that the index of
      <img alt="\lambda" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/e/0/5/e05a30d96800384dd38b22851322a6b5.png"/>
      for
      <img alt="A" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/f/c/7fc56270e7a70fa81a5935b72eacbe29.png"/>
      is equal to its multiplicity as a
      <a href="/wiki/Root" title="Root">
       root
      </a>
      of the
      <a href="/wiki/Minimal_polynomial_(linear_algebra)" title="Minimal polynomial (linear algebra)">
       minimal polynomial
      </a>
      of
      <img alt="A" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/f/c/7fc56270e7a70fa81a5935b72eacbe29.png"/>
      (whereas, by definition, its
      <a class="mw-redirect" href="/wiki/Algebraic_multiplicity" title="Algebraic multiplicity">
       algebraic multiplicity
      </a>
      for
      <img alt="A" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/f/c/7fc56270e7a70fa81a5935b72eacbe29.png"/>
      ,
      <img alt="\mathrm{mul}_A \lambda\," class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/2/2/2/222c0f0f06db075a8dd373e4d99c5e1e.png"/>
      , is its multiplicity as a root of the
      <a href="/wiki/Characteristic_polynomial" title="Characteristic polynomial">
       characteristic polynomial
      </a>
      of
      <img alt="A" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/f/c/7fc56270e7a70fa81a5935b72eacbe29.png"/>
      , i.e.
      <img alt="\det(A-xI)\in K[x]" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/9/a/7/9a7d73a8d99760e4368a683856e0994f.png"/>
      ). An equivalent necessary and sufficient condition for
      <img alt="A" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/f/c/7fc56270e7a70fa81a5935b72eacbe29.png"/>
      to be diagonalizable in
      <img alt="K" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/a/5/f/a5f3c6a11b03839d46af9fb43c97c188.png"/>
      is that all of its eigenvalues have index equal to
      <img alt="1" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/c/4/c/c4ca4238a0b923820dcc509a6f75849b.png"/>
      , i.e. its minimal polynomial has only simple roots.
     </p>
     <p>
      Note that knowing a matrix's spectrum with all of its algebraic/geometric multiplicities and indexes does not always allow for the computation of its
      <a href="/wiki/Jordan_normal_form" title="Jordan normal form">
       Jordan normal form
      </a>
      (this may be a sufficient condition only for spectrally simple, usually low-dimensional matrices): the
      <a href="/wiki/Jordan%E2%80%93Chevalley_decomposition" title="JordanChevalley decomposition">
       Jordan decomposition
      </a>
      is, in general, a computationally challenging task. From the
      <a href="/wiki/Vector_space" title="Vector space">
       vector space
      </a>
      point of view, the
      <a href="/wiki/Jordan%E2%80%93Chevalley_decomposition" title="JordanChevalley decomposition">
       Jordan decomposition
      </a>
      is equivalent to finding an orthogonal decomposition (i.e. via
      <a class="mw-redirect" href="/wiki/Direct_sum_of_vector_spaces" title="Direct sum of vector spaces">
       direct sums
      </a>
      of eigenspaces represented by Jordan blocks) of the domain which the associated
      <a href="/wiki/Generalized_eigenvector" title="Generalized eigenvector">
       generalized eigenvectors
      </a>
      make a basis for.
     </p>
     <h2>
      <span class="mw-headline" id="Functions_of_matrices">
       Functions of matrices
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Jordan_matrix&amp;action=edit&amp;section=2" title="Edit section: Functions of matrices">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      Let
      <img alt="A\in\mathbb{M}_n (\mathbb{C})" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/c/a/e/caed018ca6c6931d2f184405e370ade9.png"/>
      (i.e. a
      <img alt="n\times n" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/6/0/7/607acaa73c762411b20745149a11e90b.png"/>
      complex matrix) and
      <img alt="C\in\mathrm{GL}_n (\mathbb{C})" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/c/e/0/ce0b0137a131ba559916af7835e2262b.png"/>
      be the
      <a href="/wiki/Change_of_basis" title="Change of basis">
       change of basis
      </a>
      matrix to the
      <a href="/wiki/Jordan_normal_form" title="Jordan normal form">
       Jordan normal form
      </a>
      of
      <img alt="A" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/f/c/7fc56270e7a70fa81a5935b72eacbe29.png"/>
      , i.e.
      <img alt="A=C^{-1}JC" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/5/7/2/57251987efffe414d035535a938a6798.png"/>
      . Now let
      <img alt="f(z)" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/b/2/3/b23d8bcdb490736c53d5b677455a8cd2.png"/>
      be a
      <a href="/wiki/Holomorphic_function" title="Holomorphic function">
       holomorphic function
      </a>
      on an open set
      <img alt="\mathit{\Omega}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/a/7/7a7c5c94a2a2504cac91558195ca31d8.png"/>
      such that
      <img alt="\mathrm{spec}A\subset \mathit{\Omega}\subseteq\mathbb{C}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/a/e/e/aee0f2033024291b04e9ef8f79651104.png"/>
      , i.e. the spectrum of the matrix is contained inside the
      <a href="/wiki/Domain_of_holomorphy" title="Domain of holomorphy">
       domain of holomorphy
      </a>
      of
      <img alt="f" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/8/f/a/8fa14cdd754f91cc6554c9e71929cce7.png"/>
      . Let
     </p>
     <dl>
      <dd>
       <img alt="f(z)=\sum_{h=0}^{\infty}a_h (z-z_0)^h" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/a/8/d/a8dea4bb3d6f62c0d2e01496d7fe8526.png"/>
      </dd>
     </dl>
     <p>
      be the
      <a href="/wiki/Power_series" title="Power series">
       power series
      </a>
      expansion of
      <img alt="f" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/8/f/a/8fa14cdd754f91cc6554c9e71929cce7.png"/>
      around
      <img alt="z_0\in\mathit{\Omega}\backslash\mathrm{spec}A" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/1/1/7/11732ed2de933959e2a9e57e8f875f0a.png"/>
      , which will be hereinafter supposed to be
      <a class="new" href="/w/index.php?title=0_(zero)&amp;action=edit&amp;redlink=1" title="0 (zero) (page does not exist)">
       0
      </a>
      for simplicity's sake. The matrix
      <img alt="f(A)" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/f/a/1/fa1e95b77f0e486fa5738b876146cfec.png"/>
      is then defined via the following
      <a href="/wiki/Formal_power_series" title="Formal power series">
       formal power series
      </a>
     </p>
     <dl>
      <dd>
       <img alt="f(A)=\sum_{h=0}^{\infty}a_h A^h" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/0/b/a/0ba6aeab62ccf86f310c4199863d30f2.png"/>
      </dd>
     </dl>
     <p>
      is
      <a class="mw-redirect" href="/wiki/Absolutely_convergent" title="Absolutely convergent">
       absolutely convergent
      </a>
      with respect to the
      <a class="mw-redirect" href="/wiki/Euclidean_norm" title="Euclidean norm">
       Euclidean norm
      </a>
      of
      <img alt="\mathbb{M}_n (\mathbb{C})" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/e/d/8/ed86486a5b930c865f1a51807cd96233.png"/>
      . To put it another way,
      <img alt="f(A)\," class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/c/b/9/cb982496772163d87dfae6b9531bfdc5.png"/>
      converges absolutely for every square matrix whose
      <a href="/wiki/Spectral_radius" title="Spectral radius">
       spectral radius
      </a>
      is less than the
      <a href="/wiki/Radius_of_convergence" title="Radius of convergence">
       radius of convergence
      </a>
      of
      <img alt="f" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/8/f/a/8fa14cdd754f91cc6554c9e71929cce7.png"/>
      around
      <img alt="0" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/c/f/c/cfcd208495d565ef66e7dff9f98764da.png"/>
      and is
      <a class="mw-redirect" href="/wiki/Uniformly_convergent" title="Uniformly convergent">
       uniformly convergent
      </a>
      on any compact subsets of
      <img alt="\mathbb{M}_n (\mathbb{C})" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/e/d/8/ed86486a5b930c865f1a51807cd96233.png"/>
      satisfying this property in the
      <a class="mw-redirect" href="/wiki/Matrix_Lie_group" title="Matrix Lie group">
       matrix Lie group
      </a>
      topology.
     </p>
     <p>
      The
      <a href="/wiki/Jordan_normal_form" title="Jordan normal form">
       Jordan normal form
      </a>
      allows the computation of functions of matrices without explicitly computing an
      <a class="mw-redirect" href="/wiki/Infinite_series" title="Infinite series">
       infinite series
      </a>
      , which is one of the main achievements of Jordan matrices. Using the facts that the
      <img alt="k^\mathrm{th}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/0/7/2/072d25e54869605cdfe97569697012b7.png"/>
      power (
      <img alt="k\in\mathbb{N}_0" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/c/c/d/ccd40460efe48baee6a3f851262f7330.png"/>
      ) of a diagonal
      <a href="/wiki/Block_matrix" title="Block matrix">
       block matrix
      </a>
      is the diagonal block matrix whose blocks are the
      <img alt="k^\mathrm{th}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/0/7/2/072d25e54869605cdfe97569697012b7.png"/>
      powers of the respective blocks, i.e.
      <img alt="\left(A_1 \oplus A_2 \oplus A_3 \oplus\ldots\right)^k=A^k_1 \oplus A_2^k \oplus A_3^k \oplus\ldots" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/3/d/6/3d67bd8eb3347b6cf5d6b6307904f48a.png"/>
      , and that
      <img alt="A^k=C^{-1}J^k C\," class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/2/6/a/26ac3c0ece9fcaf19709bee6787541d6.png"/>
      , the above matrix power series becomes
     </p>
     <dl>
      <dd>
       <img alt="f(A)=C^{-1}f(J)C=C^{-1}\left(\bigoplus_{k=1}^N f\left(J_{\lambda_k ,m_k}\right)\right)C" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/b/2/2/b220ff11a09f2d3898f5c87261c88da8.png"/>
      </dd>
     </dl>
     <p>
      where the last series must not be computed explicitly via power series of every Jordan block. In fact, if
      <img alt="\lambda\in\mathit{\Omega}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/5/4/7544a00621593663c17a080d0a870ad3.png"/>
      , any
      <a href="/wiki/Holomorphic_function" title="Holomorphic function">
       holomorphic function
      </a>
      of a Jordan block
      <img alt="f(J_{\lambda,n})\," class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/d/7/e/d7eb79be0812d7f83b0a9bf6729b007d.png"/>
      is the following upper
      <a href="/wiki/Triangular_matrix" title="Triangular matrix">
       triangular matrix
      </a>
      :
     </p>
     <dl>
      <dd>
       <img alt="f(J_{\lambda,n})=\left(\begin{matrix}
f(\lambda) &amp; f^\prime (\lambda) &amp; \frac{f^{\prime\prime}(\lambda)}{2} &amp; \cdots &amp; \frac{f^{(n-2)}(\lambda)}{(n-2)!} &amp; \frac{f^{(n-1)}(\lambda)}{(n-1)!} \\
0 &amp; f(\lambda) &amp; f^\prime (\lambda) &amp; \cdots &amp; \frac{f^{(n-3)}(\lambda)}{(n-3)!} &amp; \frac{f^{(n-2)}(\lambda)}{(n-2)!} \\
0 &amp; 0 &amp; f(\lambda) &amp; \cdots &amp; \frac{f^{(n-4)}(\lambda)}{(n-4)!} &amp; \frac{f^{(n-3)}(\lambda)}{(n-3)!} \\
\vdots &amp; \vdots &amp; \vdots &amp; \ddots &amp; \vdots &amp; \vdots \\
0 &amp; 0 &amp; 0 &amp; \cdots &amp; f(\lambda) &amp; f^\prime (\lambda) \\
0 &amp; 0 &amp; 0 &amp; \cdots &amp; 0 &amp; f(\lambda) \\
\end{matrix}\right)=\left(\begin{matrix}
a_0 &amp; a_1 &amp; a_2 &amp; \cdots &amp; a_{n-1} \\
0 &amp; a_0 &amp; a_1 &amp; \cdots &amp; a_{n-2} \\
0 &amp; 0 &amp; a_0 &amp; \cdots &amp; a_{n-3} \\
\vdots &amp; \vdots &amp; \vdots &amp; \ddots &amp; \vdots \\
0 &amp; 0 &amp; 0 &amp; \cdots &amp; a_1 \\
0 &amp; 0 &amp; 0 &amp; \cdots &amp; a_0
\end{matrix}\right)." class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/f/8/e/f8eda78babd3d8cbfabe80c04681c091.png"/>
      </dd>
     </dl>
     <p>
      As a consequence of this, the computation of any functions of a matrix is straightforward whenever its Jordan normal form and its change-of-basis matrix are known. Also,
      <img alt="\mathrm{spec}f(A)=f(\mathrm{spec}A)" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/3/8/0/38052f189363a30b98181ac19eed2918.png"/>
      , i.e. every eigenvalue
      <img alt="\lambda\in\mathrm{spec}A" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/0/0/9/00934fbf0ad3dca621ea7a420fbe0d0d.png"/>
      corresponds to the eigenvalue
      <img alt="f(\lambda)\in\mathrm{spec}f(A)" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/0/2/2/02294a102abd0ee19b5d1b5e523a84dc.png"/>
      , but it has, in general, different
      <a class="mw-redirect" href="/wiki/Algebraic_multiplicity" title="Algebraic multiplicity">
       algebraic multiplicity
      </a>
      , geometric multiplicity and index. However, the algebraic multiplicity may be computed as follows:
     </p>
     <dl>
      <dd>
       <img alt="\text{mul}_{f(A)}f(\lambda)=\sum_{\mu\in\text{spec}A\cap f^{-1}(f(\lambda))}~\text{mul}_A \mu.\," class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/8/5/0/8501d8cd595cc3b18664e4207ef257f5.png"/>
      </dd>
     </dl>
     <p>
      The function
      <img alt="f(T)" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/b/0/5/b053b8f225fcc290ba66114eadc9d163.png"/>
      of a
      <a class="mw-redirect" href="/wiki/Linear_transformation" title="Linear transformation">
       linear transformation
      </a>
      <img alt="T" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/b/9/e/b9ece18c950afbfa6b0fdbfa4ff731d3.png"/>
      between vector spaces can be defined in a similar way according to the
      <a href="/wiki/Holomorphic_functional_calculus" title="Holomorphic functional calculus">
       holomorphic functional calculus
      </a>
      , where
      <a href="/wiki/Banach_space" title="Banach space">
       Banach space
      </a>
      and
      <a href="/wiki/Riemann_surface" title="Riemann surface">
       Riemann surface
      </a>
      theories play a fundamental role. In the case of finite-dimensional spaces, both theories perfectly match.
     </p>
     <h2>
      <span class="mw-headline" id="Dynamical_systems">
       Dynamical systems
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Jordan_matrix&amp;action=edit&amp;section=3" title="Edit section: Dynamical systems">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      Now suppose a (complex)
      <a href="/wiki/Dynamical_system" title="Dynamical system">
       dynamical system
      </a>
      is simply defined by the equation
     </p>
     <dl>
      <dd>
       <img alt="\dot{\mathbf{z}}(t)=A(\mathbf{c})\mathbf{z}(t)," class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/0/1/e/01ee5fe86359d853e0443dc0f8f63624.png"/>
      </dd>
      <dd>
       <img alt="\mathbf{z}(0)=\mathbf{z}_0 \in\mathbb{C}^n," class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/3/3/a/33ac0b6cf3fe6238bdd303a6e0d17967.png"/>
      </dd>
     </dl>
     <p>
      where
      <img alt="\mathbf{z}:\mathbb{R_+}\rightarrow \mathcal{R}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/8/a/2/8a27d3609a6ca12c1c4120a8eee988b7.png"/>
      is the (
      <img alt="n" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/b/8/7b8b965ad4bca0e41ab51de7b31363a1.png"/>
      -dimensional) curve parametrization of an orbit on the
      <a href="/wiki/Riemann_surface" title="Riemann surface">
       Riemann surface
      </a>
      <img alt="\mathcal{R}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/1/3/1/131ce5f8ec4744bd9a86d87cefda1d11.png"/>
      of the dynamical system, whereas
      <img alt="A(\mathbf{c})" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/a/0/b/a0b7139898a44d79b9e77bbf281ab3f7.png"/>
      is an
      <img alt="n\times n" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/6/0/7/607acaa73c762411b20745149a11e90b.png"/>
      complex matrix whose elements are complex functions of a
      <img alt="d" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/8/2/7/8277e0910d750195b448797616e091ad.png"/>
      -dimensional parameter
      <img alt="\mathbf{c}\in\mathbb{C}^d" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/5/f/f/5ff47088cea7d7bb84bf46669219743a.png"/>
      . Even if
      <img alt="A\in\mathbb{M}_n \left(\mathrm{C}^0(\mathbb{C}^d)\right)" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/f/5/7f5975000576c05f8f80b016095668bf.png"/>
      (i.e.
      <img alt="A" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/f/c/7fc56270e7a70fa81a5935b72eacbe29.png"/>
      continuously depends on the parameter
      <img alt="\mathbf{c}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/8/9/1/891394a9962c76dc890bd4010f6b111e.png"/>
      ) the
      <a href="/wiki/Jordan_normal_form" title="Jordan normal form">
       Jordan normal form
      </a>
      of the matrix is continuously deformed
      <a href="/wiki/Almost_everywhere" title="Almost everywhere">
       almost everywhere
      </a>
      on
      <img alt="\mathbb{C}^d" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/0/9/3/0939dd0125c6daf75bcbcc11c5b6bd12.png"/>
      but, in general,
      <em>
       not
      </em>
      everywhere: there is some critical submanifold of
      <img alt="\mathbb{C}^d" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/0/9/3/0939dd0125c6daf75bcbcc11c5b6bd12.png"/>
      on which the Jordan form abruptly changes its structure whenever the parameter crosses or simply travels around it (
      <a href="/wiki/Monodromy" title="Monodromy">
       monodromy
      </a>
      ). Such changes mean that several Jordan blocks (either belonging to different eigenvalues or not) join together to a unique Jordan block, or vice versa (i.e. one Jordan block splits into two or more different ones). Many aspects of
      <a href="/wiki/Bifurcation_theory" title="Bifurcation theory">
       bifurcation theory
      </a>
      for both continuous and discrete dynamical systems can be interpreted with the analysis of functional Jordan matrices.
     </p>
     <p>
      From the
      <a href="/wiki/Tangent_space" title="Tangent space">
       tangent space
      </a>
      dynamics, this means that the orthogonal decomposition of the dynamical system's
      <a href="/wiki/Phase_space" title="Phase space">
       phase space
      </a>
      changes and, for example, different orbits gain periodicity, or lose it, or shift from a certain kind of periodicity to another (such as
      <i>
       period-doubling
      </i>
      , cfr.
      <a href="/wiki/Logistic_map" title="Logistic map">
       logistic map
      </a>
      ).
     </p>
     <p>
      In a sentence, the qualitative behaviour of such a dynamical system may substantially change as the
      <a class="new" href="/w/index.php?title=Versal_deformation&amp;action=edit&amp;redlink=1" title="Versal deformation (page does not exist)">
       versal deformation
      </a>
      of the Jordan normal form of
      <img alt="A(\mathbf{c})" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/a/0/b/a0b7139898a44d79b9e77bbf281ab3f7.png"/>
      .
     </p>
     <h2>
      <span class="mw-headline" id="Linear_ordinary_differential_equations">
       Linear ordinary differential equations
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Jordan_matrix&amp;action=edit&amp;section=4" title="Edit section: Linear ordinary differential equations">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      The simplest example of a
      <a href="/wiki/Dynamical_system" title="Dynamical system">
       dynamical system
      </a>
      is a system of linear, constant-coefficient, ordinary differential equations, i.e. let
      <img alt="A\in\mathbb{M}_n (\mathbb{C})" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/c/a/e/caed018ca6c6931d2f184405e370ade9.png"/>
      and
      <img alt="\mathbf{z}_0 \in\mathbb{C}^n" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/f/0/3/f0381f64219ed95b5d41f46106317372.png"/>
      :
     </p>
     <dl>
      <dd>
       <img alt="\dot{\mathbf{z}}(t)=A\mathbf{z}(t)," class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/c/9/0/c900fc1d343c921f9acb1c7bcf470fbf.png"/>
      </dd>
      <dd>
       <img alt="\mathbf{z}(0)=\mathbf{z}_0," class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/d/9/5/d95d76f7d2d2d9369a4b75452470f71c.png"/>
      </dd>
     </dl>
     <p>
      whose direct closed-form solution involves computation of the
      <a href="/wiki/Matrix_exponential" title="Matrix exponential">
       matrix exponential
      </a>
      :
     </p>
     <dl>
      <dd>
       <img alt="\mathbf{z}(t)=e^{tA}\mathbf{z}_0." class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/9/4/4/944756d7a123eeaad20c24788c21b1c1.png"/>
      </dd>
     </dl>
     <p>
      Another way, provided the solution is restricted to the local
      <a href="/wiki/Lp_space" title="Lp space">
       Lebesgue space
      </a>
      of
      <img alt="n" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/b/8/7b8b965ad4bca0e41ab51de7b31363a1.png"/>
      -dimensional vector fields
      <img alt="\mathbf{z}\in\mathrm{L}_{\mathrm{loc}}^1 (\mathbb{R}_+)^n" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/4/7/3/473c06fe89601c033a01aaa7a3d5f689.png"/>
      , is to use its
      <a href="/wiki/Laplace_transform" title="Laplace transform">
       Laplace transform
      </a>
      <img alt="\mathbf{Z}(s)=\mathcal{L}[\mathbf{z}](s)" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/f/7/5/f753b474f9a25059f1fce17be04152f7.png"/>
      . In this case
     </p>
     <dl>
      <dd>
       <img alt="\mathbf{Z}(s)=\left(sI-A\right)^{-1}\mathbf{z}_0." class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/b/3/7b33655153b68579b6f7e3b8aa32370d.png"/>
      </dd>
     </dl>
     <p>
      The matrix function
      <img alt="\left(A-sI\right)^{-1}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/e/6/7e682f31d9f3bf8fad24ac2339358bd5.png"/>
      is called the
      <a class="mw-redirect" href="/wiki/Resolvent_matrix" title="Resolvent matrix">
       resolvent matrix
      </a>
      of the
      <a href="/wiki/Differential_operator" title="Differential operator">
       differential operator
      </a>
      <img alt="\frac{\mathrm{d}}{\mathrm{d}t}-A" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/c/a/7/ca703f19b7875dfd64e36e1adef4bf1d.png"/>
      . It is
      <a class="mw-redirect" href="/wiki/Meromorphic" title="Meromorphic">
       meromorphic
      </a>
      with respect to the complex parameter
      <img alt="s\in\mathbb{C}" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/2/8/6/286e088ed9bf5fa971e9033f7f3d4fbd.png"/>
      since its matrix elements are rational functions whose denominator is equal for all to
      <img alt="\det (A-sI)" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/9/5/3/95390fd9fb25145e4795f3768b24ae3f.png"/>
      . Its polar singularities are the eigenvalues of
      <img alt="A" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/7/f/c/7fc56270e7a70fa81a5935b72eacbe29.png"/>
      , whose order equals their index for it, i.e.
      <img alt="\mathrm{ord}_{(A-sI)^{-1}}\lambda=\mathrm{idx}_A \lambda" class="mwe-math-fallback-image-inline tex" src="//upload.wikimedia.org/math/b/0/f/b0f552b7993dfdf433b936aaf6d5b80b.png"/>
      .
     </p>
