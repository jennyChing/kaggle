          <p>
           Done Setup Notes:
           <br/>
           Drone setup is still ongoing. Video shaky or shutters when gimbal is in full lock. Need to look at setting some gains into the gimbal so it doesnt reach full extension. Getting around 7 minutes of flight time. Using a new battery voltage tester on the phantom to sound an alarm once each batter cell falls under 3.8V.
          </p>
          <p>
           GPS and Mapping Data:
           <br/>
           https://goo.gl/maps/4pBqv
          </p>
          <p>
           Equipment Used for Aerial Video and Photography
          </p>
          <p>
            DJI Phantom
           <br/>
            Gimbal with tilt / pitch
           <br/>
            Filmed in 1980 x 1440 X 30fps
           <br/>
            Removed fish eye effect
           <br/>
            Added Protune video effect
           <br/>
            FPV by Boscam 5.8Ghz 200mw
           <br/>
            8 Inch Carbon Fibre Props
           <br/>
            GoPro Hero3
          </p>
          <p>
           DJI Gains via Naza Software = = Default Used
          </p>
          <p>
           Edited within GoPro Studio and Magic Movie Edit Pro 2014
           <br/>
           Music: Waves  Mr Probz
          </p>
