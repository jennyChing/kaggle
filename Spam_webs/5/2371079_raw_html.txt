     <h3>
      Potential Applications
     </h3>
     <blockquote>
      <p>
       <a href="../WeightExercises/DeltoidAnterior/BBShoulderPress.html">
        <img align="RIGHT" alt="Barbell Shoulder Press" border="0" height="300" naturalsizeflag="3" src="../Images/WeightTraining/BarbellShoulderPress.jpg" width="300"/>
       </a>
      </p>
     </blockquote>
     <ul>
      <li>
       See
       <a href="Research.html#anchor235318">
        summaries of studies
       </a>
      </li>
      <li>
       Ideal for intermediate, advanced trainee, or athletes
       <ul>
        <li>
         Implement after a series of strength plateaus have been encountered
         <ul>
          <li>
           due to
           <a href="../ExInfo/Overtraining.html">
            chronic overtraining
           </a>
           .
          </li>
          <li>
           or despite
           <a href="ChangeWtEx.html">
            changing exercises periodically
           </a>
           .
          </li>
         </ul>
        </li>
        <li>
         Can be integrated with
         <a href="Periodization.html#anchor22046">
          altering
        workload regimen
         </a>
         .
        </li>
        <li>
         Also see
         <a href="../Questions/MuscleMass.html">
          muscle mass
        tips
         </a>
        </li>
       </ul>
      </li>
      <li>
       Athletes
       <ul>
        <li>
         <a href="Weightlifting/AthleticProgram.html">
          Sample Athletic
        Strength Program
         </a>
        </li>
        <li>
         <a href="PowerliftingPrograms.html">
          Powerlifting Programs
         </a>
        </li>
        <li>
         <a href="Weightlifting.html">
          Olympic-style Weightlifting
        Programs
         </a>
        </li>
       </ul>
      </li>
      <li>
       Can also be adapted for older adults
      </li>
      <li>
       This sort of log can be adapted for
       <a href="Periodization.html#anchor22046">
        programs
      of varying percentages of predicted 1 rep max
       </a>
       <ul>
        <li>
         Replace rep ranges (i.e. 8-10, 5-7, etc.) with percentages
        (80%, 85%, etc.)
        </li>
       </ul>
      </li>
      <li>
       A light / heavy protocol can be implemented on exercises
      where one resistance is too light and the next resistance is
      too heavy.
       <ul>
        <li>
         Example:
         <ul>
          <li>
           Too light: 20 lbs x 12 reps
          </li>
          <li>
           Too heavy: 25 lbs x 4 reps
          </li>
         </ul>
        </li>
        <li>
         Dilemma with proportionally large resistance increments
         <ul>
          <li>
           If a too-heavy resistance is continually used, little muscular
          endurance can be achieved to progress in repetitions
          </li>
         </ul>
        </li>
        <li>
         If a too-light resistance is continually used, little muscular
        strength can be achieved to progress in resistance
        </li>
       </ul>
      </li>
     </ul>
     <h3>
      Example Workload Variations
     </h3>
     <blockquote>
      <p>
       <a href="../WeightExercises/RectusAbdominis/DBPushSitUp.html">
        <img align="RIGHT" alt="Dumbbell Push Sit-up" border="0" height="320" naturalsizeflag="3" src="../Images/WeightTraining/DumbbellSitup.jpg" width="240"/>
       </a>
      </p>
     </blockquote>
     <ul>
      <li>
       Power option
       <ul>
        <li>
         Heavy days: 2-3 reps, light days: 5-6 reps
        </li>
        <li>
         Suitable for
         <a href="../Lists/PowerExercises.html">
          Olympic-style
        weightlifts
         </a>
        </li>
       </ul>
      </li>
      <li>
       Strength program:
       <ul>
        <li>
         Heavy days: 6-8 reps, light days: 10-12 reps
        </li>
        <li>
         Also see advance powerlifting program,
         <a href="Powerlifting/Westside.html">
          Westside
        Barbell Program
         </a>
        </li>
       </ul>
      </li>
      <li>
       Endurance / strength program:
       <ul>
        <li>
         Heavy days: 8-10 reps, light days: 12-15 reps
        </li>
        <li>
         <i>
          or
         </i>
         Heavy days: 6-9 reps, light days: 10-15 reps
        </li>
       </ul>
      </li>
      <li>
       Endurance option:
       <ul>
        <li>
         Heavy days: 20-30 reps, light days: 40-50 reps
        </li>
        <li>
         <i>
          or
         </i>
         Heavy days: 15-25 reps, light days: 30-50 reps
        </li>
        <li>
         Suitable for abdominal training
        </li>
       </ul>
      </li>
      <li>
       See
       <a href="LowVolumeTraining.html#anchor236641">
        low volume
      method
       </a>
       and
       <a href="Periodization.html">
        Periodization
       </a>
      </li>
     </ul>
     <h3>
      <a name="anchor133121">
      </a>
      Sample Log
     </h3>
     <ul>
      <li>
       Initial Resistances
       <ul>
        <li>
         Record initial trial resistances and repetitions achieved
        in correct form
        </li>
        <li>
         Estimate or
         <a href="../Calculators/Workload.html">
          calculate
         </a>
         initial resistances for first workout with varied workload (ie:
        Medium Load)
        </li>
        <li>
         Record resistances for other workloads (Light &amp; Heavy
        Days)
        </li>
       </ul>
      </li>
      <li>
       Progressions
       <ul>
        <li>
         Increase weight on respective workout approximately 2.5%
        - 10% when the upper repetition range has been achieved
         <ul>
          <li>
           See
           <a href="Resistance.html#anchor2406581">
            Systematic Resistance
          Increases
           </a>
          </li>
         </ul>
        </li>
        <li>
         Examples
         <ul>
          <li>
           <font color="#ff00ff">
            Pink
           </font>
           "4" on exercise
          (5) below signifies minimum repetition range has not been achieved
          </li>
          <li>
           <font color="#ff0000">
            RED
           </font>
           repetitions mark reaching
          upper repetition range
          </li>
          <li>
           <font color="#0000ff">
            BLUE
           </font>
           numbers mark new resistances
          recorded for the next time that particular repetition range is
          performed
          </li>
         </ul>
        </li>
       </ul>
      </li>
     </ul>

     <h3>
      Short Hand Log Options
     </h3>
     <blockquote>
      <p>
       <img align="RIGHT" border="0" height="320" naturalsizeflag="3" src="../Images/WeightTraining/PowerSnatchBottom.jpg" width="240"/>
      </p>
     </blockquote>
     <ul>
      <li>
       Consider using one of three variations of a shorthand log
       <ul>
        <li>
         Multiple columns to right of exercise
         <ul>
          <li>
           Designated Light columns and heavy columns
          </li>
          <li>
           Only record new workout weight if upper rep range has been
          achieved for each varying load
          </li>
         </ul>
        </li>
        <li>
         Multiple lines under exercise name
         <ul>
          <li>
           Exercise name on first line
          </li>
          <li>
           Record weights for light day on second line under exercise
          name
          </li>
          <li>
           Record weights for heavy day on third line
          </li>
          <li>
           Subsequent exercises follow
           <ul>
            <li>
             Each exercise occupies multiple lines
            </li>
           </ul>
          </li>
         </ul>
        </li>
        <li>
         Record a single number representing base weight for each
        exercise
         <ul>
          <li>
           Base moderate, light, and heavy workloads on base weight
          </li>
          <li>
           See
           <a href="LogMedLightHeavy.html">
            Shorthand Log: Implementing
          Varying Workloads
           </a>
          </li>
         </ul>
        </li>
       </ul>
      </li>
