        <!-- Begin Entry Skype and other VoIP Apps on Mobile Phones -->
        <div class="entry-content entry">
         It seems new software comes out daily that enables you to run a VoIP application, such as Skype, on your mobile phone natively, through a bridged gateway, or using the Skype API. While Skype is certainly the most popular VoIP application, other software solutions exist that use the SIP standard. Still others provide SIP-to-Skype bridging capabilities. I thought it would be useful to have a rundown of the various solutions out there.
         <br/>
         <br/>
         <a href="http://www.soonrtalk.com">
          <img align="right" alt="SoonR Talk" src="http://blog.tmcnet.com/blog/tom-keating/images/soonr-talk.gif"/>
          SoonR Talk
         </a>
         - I broke the news about this first software application to
         <a href="http://blog.tmcnet.com/blog/tom-keating/skype/skype-on-iphone.asp">
          enable VoIP on the Apple iPhone
         </a>
         . It features a nice clean interface that emulates the Skype client look and feel, as seen in the screenshot to the right. The way it works is that the AJAX interface acts as a remote control for your computer where the SoonR service is installed along with Skype. There is nothing to install on your phone because SoonR Talk uses a browser that supports AJAX. You simply login into SoonR website.
         <br clear="all"/>
         <br/>
         <a href="http://www.shapeservices.com/en/products/details.php?product=skype&amp;platform=iphone">
          <img align="right" alt="IM+ for Skype" src="http://blog.tmcnet.com/blog/tom-keating/images/im-for-skype.jpg"/>
          Shape Services' IM+ for Skype
         </a>
         - lets you run Skype on the iPhone with their IM+ for Skype  application. IM+ for Skype,also works on the iPod touch. The beauty of IM+ for  Skype is that it doesn't even require Skype to be running on your PC - it  leverages the Skype API and servers run by the IM+ for Skype folks (Shape  Services). You can find out more info in my
         <a href="http://blog.tmcnet.com/blog/tom-keating/voip/skype-on-the-apple-ipod-touch.asp">
          Skype  on the Apple iPod touch
         </a>
         article.
         <br clear="all"/>
         <br/>
         <a href="http://blog.tmcnet.com/blog/tom-keating/skype/sip-to-skype-calls.asp">
          SIP to Skype calls
         </a>
         - Check out my article
         <a href="http://blog.tmcnet.com/blog/tom-keating/skype/sip-to-skype-calls.asp">
          here
         </a>
         for a how-to step-by-step guide that lets you deliver SIP calls directly to Skype.  Essentially you can have a SIP DID number ring your Skype client. The workaround  uses Net2Max.com's One Click Contact number (1CC number) to make this  possible.
         <br/>
         <br/>
         <a href="http://blog.tmcnet.com/blog/tom-keating/skype/sippyskype-siptoskype-gateway.asp">
          SippySkype
         </a>
         - Java software that allows you to make and receive
         <a href="http://www.tmcnet.com/tmcnet/snapshots/snapshots.aspx?Company=Skype">
          Skype
         </a>
         calls from your SIP/VoIP adapter or SIP softphone. Basically it's a Skype/SIP Bridge/Gateway/Proxy. You'll need a SIP client on your mobile phone for this to work. But assuming your mobile phone can run a SIP client, you can make SIP-to-Skype calls.
         <br/>
         <br/>
         <img align="right" alt="iPhoneGnome on iPhone" src="http://blog.tmcnet.com/blog/tom-keating/images/iphonegnome.png"/>
         <a href="http://www.phonegnome.com/blog/?p=121">
          iPhoneGnome
         </a>
         , enables  PhoneGnome calling from your iPhone. You can call other PhoneGnome numbers as  well as anyone using SIP-based services, FWD, MSN, Yahoo,
         <a href="http://www.tmcnet.com/tmcnet/snapshots/snapshots.aspx?Company=Google">
          Google
         </a>
         Talk, and any other numbers that you can call from your PhoneGnome account.
         <br clear="all"/>
         <br/>
         I should stress that there are non-Skype VoIP apps that run on mobile phones, many of which target the Apple iPhone, but not all. This next example, Jajah does support the iPhone, but they actually support
         <em>
          any
         </em>
         mobile phone.
         <br/>
         <br/>
         <img align="right" alt="Jajah iPhone" src="http://blog.tmcnet.com/blog/tom-keating/images/jajah-web-app-iphone.png"/>
         <a href="http://blog.tmcnet.com/blog/tom-keating/tag/jajah">
          Jajah
         </a>
         - They have several mobile phone-to-VoIP solutions available. There's the
         <a href="http://blog.tmcnet.com/blog/tom-keating/voip/jajah-apple-iphone-and-ipod-touch-app-launches.asp">
          JAJAH Apple iPhone and iPod Touch App.
         </a>
         There's
         <a href="http://blog.tmcnet.com/blog/tom-keating/voip/jajah-direct-launches.asp">
          Jajah Direct
         </a>
         , which allows you to simply dial a local access number to talk globally - no Internet or data access needed and you can skip having to dial from your PC using VoIP software (i.e.
         <a href="http://www.skype.com/">
          Skype
         </a>
         ). There's also the
         <a href="http://blog.tmcnet.com/blog/tom-keating/voip/jajah-native-sip-client-for-emone-phone.asp">
          Jajah native SIP client
         </a>
         . And of course there's Jajah Mobile Web (
         <a href="http://mobile.jajah.com/">
          http://mobile.jajah.com
         </a>
         ) which gives you one-click access to their free or low-cost global calling service, directly from the browser, which works with any mobile phone with web access since no application download is needed.
         <br clear="all"/>
         <br/>
         <img align="right" alt="truphone configure" src="http://blog.tmcnet.com/blog/tom-keating/images/truphone-configure.jpg"/>
         <a href="http://www.truphone.com">
          truphone
         </a>
         - truphone has native VoIP applications for several mobile phones. It currently works on many of the popular Nokia N-series (eg, N95) and E-series phones (eg, E90). truphone leverages the SIP standard for all of their softclients. Also, truphone
         <a href="http://blog.tmcnet.com/blog/tom-keating/voip/100-voip-on-the-iphone.asp">
          supports the Apple iPhone
         </a>
         and was the first to offer a true 100% native VoIP application on the iPhone. You have to
         <a href="http://blog.tmcnet.com/blog/tom-keating/voip/100-voip-on-the-iphone.asp">
          jailbreak your iPhone to install it
         </a>
         though.
         <br clear="all"/>
         <br/>
         <img align="right" alt="Hipsip" src="http://blog.tmcnet.com/blog/tom-keating/images/hipsip.jpg"/>
         <a href="http://www.sipcall.com">
          Sipcall
         </a>
         - This is an exciting new offering currently in beta. They offer a free Skype and SIP calling service called
         <a href="http://www.hipsip.com">
          Hipsip
         </a>
         that works from any mobile phone's web browser. You simply install their bridging software on your PC and then from your mobile phone you can access Hipsip and call any Skype
         <strong>
          or
         </strong>
         SIP user. The software will make a cell phone call to a local Hipsip number automatically. Hipsip converts the call to VoIP and bridges the call with the Skype or SIP user you dialed. (
         <em>
          Hat tip to
          <a href="http://www.goebel.net/technews/2008/03/call-skype-contacts-from-mobile-phones.html">
           Markus Gbel for discovering this software
          </a>
          and for the screenshot. Go check out his recent write-up!
         </em>
         )
         <br/>
         <br clear="all"/>
         <a href="http://blog.tmcnet.com/blog/tom-keating/voip/talkplus-voice-20-app-launches.asp">
          <img align="right" alt="Talkplus on Windows Mobile" src="http://blog.tmcnet.com/blog/tom-keating/images/talkplus-windows-mobile.jpg"/>
          Talkplus
         </a>
         - unique mobile phone service that leverages VoIP, identity management,  scheduling/contact rules, and other advanced business/personal processes that  make your mobile experience more productive while enabling privacy controls. TalkPlus gives cell phone users a 2nd virtual phone number with powerful  accessibility and privacy rules while leveraging the cost savings of VoIP  termination. TalkPlus's patented technology is network agnostic, and supports standard cell  phones that support the BREW and JAVA platforms. They also support native clients for Nokia phones, Windows Mobile, Blackberry, and Palm. TalkPlus is also WAP compliant  which virtually all mobile phones support. One interesting thing I talked about at
         <a href="http://www.tmcnet.com/voip/conference/">
          ITEXPO
         </a>
         with Talkplus CEO Jeff Black is that TalkPlus has  built their own
         <strong>
          Skype gateway
         </strong>
         . In fact, when pressed further,  Jeff mentioned they actually
         <em>
          reverse engineered
         </em>
         Skype's protocol.  Although the
         <a href="http://www.tmcnet.com/tmcnet/snapshots/snapshots.aspx?Company=Skype">
          Skype
         </a>
         gateway isn't part of TalkPlus's launch today, Jeff explained that they have  tested it in their labs and it's working very well.
         <br/>
         <br/>
         <img align="right" alt="iSkoot" src="http://blog.tmcnet.com/blog/tom-keating/images/iskoot.png"/>
         <a href="http://blog.tmcnet.com/blog/tom-keating/skype/skype-from-mobile-cell-phone.asp">
          iSkoot
         </a>
         - lets you call your buddy list from your mobile phone without needing your PC. iSkoot software is simply a software download to your mobile phone to enable you to make Skype calls. Supported phones include
         <a href="http://www.tmcnet.com/tmcnet/snapshots/snapshots.aspx?Company=Nokia">
          Nokia
         </a>
         Series 60 handsets: 6600, 6620, 6630, 6670, 6680, 6682, 7610, N70. They also support Motorola
         <br/>
         RAZR, Treo, and Nokia Series 40 handsets. Windows Mobile and Blackberry phones are also supported. iSkoot Mobile Network allows consumers to place and receive Internet calls with their mobile phones without the need for PCs or Wi-Fi hot spots.
         <br/>
         <br/>
         <a href="http://www.mobivox.com">
          <img align="right" alt="Mobivox VoxGirl" src="http://blog.tmcnet.com/blog/tom-keating/images/mobivox-voxgirl.jpg"/>
          Mobivox
         </a>
         - Mobivox allows you to make free calls to other Skype users as well as free calls to other Mobivox users. What's unique about Mobivox is they offer 3 ways of initiating a call using ANY mobile phone. First, you can call a local access number and VoxGirl asks you which number you want to dial. Using speech-recognition it will connect the call. Second, you can use web call back. Logon to your Mobivox account and click who you want to call and Mobivox will connect you both. Third, using SMS you can 'text' MOBIVOX the name of the person or number you wish to call. VoxGirl will call you back and connect you.
         <br/>
         <br/>
         <a href="http://blog.tmcnet.com/blog/tom-keating/microsoft/fring-coming-to-windows-mobile-phones.asp">
          <img align="right" alt="Fring" src="http://blog.tmcnet.com/blog/tom-keating/images/fring_sip.png"/>
          fring
         </a>
         - It's a thin client that allows you to talk &amp; IM via your handset's Internet  connection to other mobile phones and PC-based services such as Skype,
         <a href="http://www.tmcnet.com/tmcnet/snapshots/snapshots.aspx?Company=Google">
          Google
         </a>
         Talk, MSN Messenger, SIP and Twitter for free. It also unifies all of your  various contacts across all of these IM/VoIP networks so you can use a single  app to view presence and initiate chat sessions or voice calls. It works on Nokia and Windows Mobile phones. You can also share files using fring.
         <br/>
         <br/>
         <a href="http://blog.tmcnet.com/blog/tom-keating/skype/eqos-mobiletoskype-buddy-list-solution.asp">
          <img align="right" alt="eqo skype" height="527" src="http://blog.tmcnet.com/blog/tom-keating/images/eqo-exit-eqo.png" width="350"/>
          eqo mobile-to-Skype solution
         </a>
         - Essentially, eqo acts as the "glue" betwen online communities such as LinkedIn,  MySpaces, orkut, or
         <a href="http://www.tmcnet.com/tmcnet/snapshots/snapshots.aspx?Company=Skype">
          Skype
         </a>
         with mobile carriers such as T-Mobile, Cingular, Vodaphone, etc. Their  architecture is a patent pending core IP that uses proprietary call signaling,  that is P2P-based and cross device/network domain. It leverages mobile data  (SMS) to control the signalling and the mobile voice network for the media  stream to the mobile phone, so you don't need the Skype client running on your  cell phone. Essentially, their software client is a J2ME application that  resides on your cell phone and which keeps your buddy list synched with Skype.  eqo did a good job at keeping the application small since it's only a 121k JAR  file, most of which is the graphics for the user interface. Their latest client, which still
         <a href="http://blog.tmcnet.com/blog/tom-keating/mobile-phones/eqo-voip-im-client-now-supports-windows-mobile-and-blackberry-mobile-p.asp">
          runs  on Java compatible phones
         </a>
         , lets you make VoIP calls, as well as instant  message across networks (MSN, Yahoo, AIM, GoogleTalk, ICQ, Jabber).
         <br/>
         <br/>
         And there you have it. Several ways to VoIP on your mobile phone using both Skype and non-Skype VoIP solutions. What's your favorite mobile VoIP application? Post a comment...

        <div id="disqus_post_title" style="display:none;">
         Skype and other VoIP Apps on Mobile Phones
        </div>
        <div id="disqus_post_message" style="display:none;">
         It seems new software comes out daily that enables you to run a VoIP application, such as Skype, on your mobile phone natively, through a bridged gateway, or using the Skype API. While Skype is certainly the most popular VoIP application, other software solutions exist that use the SIP standard. Still others provide SIP-to-Skype bridging capabilities. I thought it would be useful to have a rundown of the various solutions out there.
         <br/>
         \n
         <br/>
         \n
         <a href='\"http://www.soonrtalk.com\"'>
          <img align='\"right\"' alt='\"SoonR' src='\"http://blog.tmcnet.com/blog/tom-keating/images/soonr-talk.gif\"' talk=""/>
          SoonR Talk
         </a>
         - I broke the news about this first software application to
         <a href='\"http://blog.tmcnet.com/blog/tom-keating/skype/skype-on-iphone.asp\"'>
          enable VoIP on the Apple iPhone
         </a>
         . It features a nice clean interface that emulates the Skype client look and feel, as seen in the screenshot to the right. The way it works is that the AJAX interface acts as a remote control for your computer where the SoonR service is installed along with Skype. There is nothing to install on your phone because SoonR Talk uses a browser that supports AJAX. You simply login into SoonR website.
         <br clear='\"all\"'/>
         \n
         <br/>
         \n
         <a href='\"http://www.shapeservices.com/en/products/details.php?product=skype&amp;platform=iphone\"'>
          <img align='\"right\"' alt='\"IM+' for="" skype="" src='\"http://blog.tmcnet.com/blog/tom-keating/images/im-for-skype.jpg\"'/>
          Shape Services' IM+ for Skype
         </a>
         - lets you run Skype on the iPhone with their IM+ for Skype  application. IM+ for Skype,also works on the iPod touch. The beauty of IM+ for  Skype is that it doesn't even require Skype to be running on your PC - it  leverages the Skype API and servers run by the IM+ for Skype folks (Shape  Services). You can find out more info in my
         <a href='\"http://blog.tmcnet.com/blog/tom-keating/voip/skype-on-the-apple-ipod-touch.asp\"'>
          Skype  on the Apple iPod touch
         </a>
         article.
         <br clear='\"all\"'/>
         \n
         <br/>
         \n
         <a href='\"http://blog.tmcnet.com/blog/tom-keating/skype/sip-to-skype-calls.asp\"'>
          SIP to Skype calls
         </a>
         - Check out my article
         <a href='\"http://blog.tmcnet.com/blog/tom-keating/skype/sip-to-skype-calls.asp\"'>
          here
         </a>
         for a how-to step-by-step guide that lets you deliver SIP calls directly to Skype.  Essentially you can have a SIP DID number ring your Skype client. The workaround  uses Net2Max.com's One Click Contact number (1CC number) to make this  possible.
         <br/>
         \n
         <br/>
         \n
         <a href='\"http://blog.tmcnet.com/blog/tom-keating/skype/sippyskype-siptoskype-gateway.asp\"'>
          SippySkype
         </a>
         - Java software that allows you to make and receive
         <a href='\"http://www.tmcnet.com/tmcnet/snapshots/snapshots.aspx?Company=Skype\"'>
          Skype
         </a>
         calls from your SIP/VoIP adapter or SIP softphone. Basically it's a Skype/SIP Bridge/Gateway/Proxy. You'll need a SIP client on your mobile phone for this to work. But assuming your mobile phone can run a SIP client, you can make SIP-to-Skype calls.
         <br/>
         \n
         <br/>
         \n
         <img align='\"right\"' alt='\"iPhoneGnome' iphone="" on="" src='\"http://blog.tmcnet.com/blog/tom-keating/images/iphonegnome.png\"'/>
         <a href='\"http://www.phonegnome.com/blog/?p=121\"'>
          iPhoneGnome
         </a>
         , enables  PhoneGnome calling from your iPhone. You can call other PhoneGnome numbers as  well as anyone using SIP-based services, FWD, MSN, Yahoo,
         <a href='\"http://www.tmcnet.com/tmcnet/snapshots/snapshots.aspx?Company=Google\"'>
          Google
         </a>
         Talk, and any other numbers that you can call from your PhoneGnome account.
         <br clear='\"all\"'/>
         \n
         <br/>
         \nI should stress that there are non-Skype VoIP apps that run on mobile phones, many of which target the Apple iPhone, but not all. This next example, Jajah does support the iPhone, but they actually support
         <em>
          any
         </em>
         mobile phone.
         <br/>
         \n
         <br/>
         \n
         <img align='\"right\"' alt='\"Jajah' iphone="" src='\"http://blog.tmcnet.com/blog/tom-keating/images/jajah-web-app-iphone.png\"'/>
         <a href='\"http://blog.tmcnet.com/blog/tom-keating/tag/jajah\"'>
          Jajah
         </a>
         - They have several mobile phone-to-VoIP solutions available. There's the
         <a href='\"http://blog.tmcnet.com/blog/tom-keating/voip/jajah-apple-iphone-and-ipod-touch-app-launches.asp\"'>
          JAJAH Apple iPhone and iPod Touch App.
         </a>
         There's
         <a href='\"http://blog.tmcnet.com/blog/tom-keating/voip/jajah-direct-launches.asp\"'>
          Jajah Direct
         </a>
         , which allows you to simply dial a local access number to talk globally - no Internet or data access needed and you can skip having to dial from your PC using VoIP software (i.e.
         <a href='\"http://www.skype.com/\"'>
          Skype
         </a>
         ). There's also the
         <a href='\"http://blog.tmcnet.com/blog/tom-keating/voip/jajah-native-sip-client-for-emone-phone.asp\"'>
          Jajah native SIP client
         </a>
         . And of course there's Jajah Mobile Web (
         <a href='\"http://mobile.jajah.com/\"'>
          http://mobile.jajah.com
         </a>
         ) which gives you one-click access to their free or low-cost global calling service, directly from the browser, which works with any mobile phone with web access since no application download is needed.
         <br clear='\"all\"'/>
         \n
         <br/>
         \n
         <img align='\"right\"' alt='\"truphone' configure="" src='\"http://blog.tmcnet.com/blog/tom-keating/images/truphone-configure.jpg\"'/>
         <a href='\"http://www.truphone.com\"'>
          truphone
         </a>
         - truphone has native VoIP applications for several mobile phones. It currently works on many of the popular Nokia N-series (eg, N95) and E-series phones (eg, E90). truphone leverages the SIP standard for all of their softclients. Also, truphone
         <a href='\"http://blog.tmcnet.com/blog/tom-keating/voip/100-voip-on-the-iphone.asp\"'>
          supports the Apple iPhone
         </a>
         and was the first to offer a true 100% native VoIP application on the iPhone. You have to
         <a href='\"http://blog.tmcnet.com/blog/tom-keating/voip/100-voip-on-the-iphone.asp\"'>
          jailbreak your iPhone to install it
         </a>
         though.
         <br clear='\"all\"'/>
         \n
         <br/>
         \n
         <img align='\"right\"' alt='\"Hipsip\"' src='\"http://blog.tmcnet.com/blog/tom-keating/images/hipsip.jpg\"'/>
         <a href='\"http://www.sipcall.com\"'>
          Sipcall
         </a>
         - This is an exciting new offering currently in beta. They offer a free Skype and SIP calling service called
         <a href='\"http://www.hipsip.com\"'>
          Hipsip
         </a>
         that works from any mobile phone's web browser. You simply install their bridging software on your PC and then from your mobile phone you can access Hipsip and call any Skype
         <strong>
          or
         </strong>
         SIP user. The software will make a cell phone call to a local Hipsip number automatically. Hipsip converts the call to VoIP and bridges the call with the Skype or SIP user you dialed. (
         <em>
          Hat tip to
          <a href='\"http://www.goebel.net/technews/2008/03/call-skype-contacts-from-mobile-phones.html\"'>
           Markus Gbel for discovering this software
          </a>
          and for the screenshot. Go check out his recent write-up!
         </em>
         )
         <br/>
         \n
         <br clear='\"all\"'/>
         \n
         <a href='\"http://blog.tmcnet.com/blog/tom-keating/voip/talkplus-voice-20-app-launches.asp\"'>
          <img align='\"right\"' alt='\"Talkplus' mobile="" on="" src='\"http://blog.tmcnet.com/blog/tom-keating/images/talkplus-windows-mobile.jpg\"' windows=""/>
          Talkplus
         </a>
         - unique mobile phone service that leverages VoIP, identity management,  scheduling/contact rules, and other advanced business/personal processes that  make your mobile experience more productive while enabling privacy controls. TalkPlus gives cell phone users a 2nd virtual phone number with powerful  accessibility and privacy rules while leveraging the cost savings of VoIP  termination. TalkPlus's patented technology is network agnostic, and supports standard cell  phones that support the BREW and JAVA platforms. They also support native clients for Nokia phones, Windows Mobile, Blackberry, and Palm. TalkPlus is also WAP compliant  which virtually all mobile phones support. One interesting thing I talked about at
         <a href='\"http://www.tmcnet.com/voip/conference/\"'>
          ITEXPO
         </a>
         with Talkplus CEO Jeff Black is that TalkPlus has  built their own
         <strong>
          Skype gateway
         </strong>
         . In fact, when pressed further,  Jeff mentioned they actually
         <em>
          reverse engineered
         </em>
         Skype's protocol.  Although the
         <a href='\"http://www.tmcnet.com/tmcnet/snapshots/snapshots.aspx?Company=Skype\"'>
          Skype
         </a>
         gateway isn't part of TalkPlus's launch today, Jeff explained that they have  tested it in their labs and it's working very well.
         <br/>
         \n
         <br/>
         \n
         <img align='\"right\"' alt='\"iSkoot\"' src='\"http://blog.tmcnet.com/blog/tom-keating/images/iskoot.png\"'/>
         <a href='\"http://blog.tmcnet.com/blog/tom-keating/skype/skype-from-mobile-cell-phone.asp\"'>
          iSkoot
         </a>
         - lets you call your buddy list from your mobile phone without needing your PC. iSkoot software is simply a software download to your mobile phone to enable you to make Skype calls. Supported phones include
         <a href='\"http://www.tmcnet.com/tmcnet/snapshots/snapshots.aspx?Company=Nokia\"'>
          Nokia
         </a>
         Series 60 handsets: 6600, 6620, 6630, 6670, 6680, 6682, 7610, N70. They also support Motorola
         <br/>
         \nRAZR, Treo, and Nokia Series 40 handsets. Windows Mobile and Blackberry phones are also supported. iSkoot Mobile Network allows consumers to place and receive Internet calls with their mobile phones without the need for PCs or Wi-Fi hot spots.
         <br/>
         \n
         <br/>
         \n
         <a href='\"http://www.mobivox.com\"'>
          <img align='\"right\"' alt='\"Mobivox' src='\"http://blog.tmcnet.com/blog/tom-keating/images/mobivox-voxgirl.jpg\"' voxgirl=""/>
          Mobivox
         </a>
         - Mobivox allows you to make free calls to other Skype users as well as free calls to other Mobivox users. What's unique about Mobivox is they offer 3 ways of initiating a call using ANY mobile phone. First, you can call a local access number and VoxGirl asks you which number you want to dial. Using speech-recognition it will connect the call. Second, you can use web call back. Logon to your Mobivox account and click who you want to call and Mobivox will connect you both. Third, using SMS you can 'text' MOBIVOX the name of the person or number you wish to call. VoxGirl will call you back and connect you.
         <br/>
         \n
         <br/>
         \n
         <a href='\"http://blog.tmcnet.com/blog/tom-keating/microsoft/fring-coming-to-windows-mobile-phones.asp\"'>
          <img align='\"right\"' alt='\"Fring\"' src='\"http://blog.tmcnet.com/blog/tom-keating/images/fring_sip.png\"'/>
          fring
         </a>
         - It's a thin client that allows you to talk &amp; IM via your handset's Internet  connection to other mobile phones and PC-based services such as Skype,
         <a href='\"http://www.tmcnet.com/tmcnet/snapshots/snapshots.aspx?Company=Google\"'>
          Google
         </a>
         Talk, MSN Messenger, SIP and Twitter for free. It also unifies all of your  various contacts across all of these IM/VoIP networks so you can use a single  app to view presence and initiate chat sessions or voice calls. It works on Nokia and Windows Mobile phones. You can also share files using fring.
         <br/>
         \n
         <br/>
         \n
         <a href='\"http://blog.tmcnet.com/blog/tom-keating/skype/eqos-mobiletoskype-buddy-list-solution.asp\"'>
          <img align='\"right\"' alt='\"eqo' height='\"527\"' skype="" src='\"http://blog.tmcnet.com/blog/tom-keating/images/eqo-exit-eqo.png\"' width='\"350\"'/>
          eqo mobile-to-Skype solution
         </a>
         - Essentially, eqo acts as the "glue" betwen online communities such as LinkedIn,  MySpaces, orkut, or
         <a href='\"http://www.tmcnet.com/tmcnet/snapshots/snapshots.aspx?Company=Skype\"'>
          Skype
         </a>
         with mobile carriers such as T-Mobile, Cingular, Vodaphone, etc. Their  architecture is a patent pending core IP that uses proprietary call signaling,  that is P2P-based and cross device/network domain. It leverages mobile data  (SMS) to control the signalling and the mobile voice network for the media  stream to the mobile phone, so you don't need the Skype client running on your  cell phone. Essentially, their software client is a J2ME application that  resides on your cell phone and which keeps your buddy list synched with Skype.  eqo did a good job at keeping the application small since it's only a 121k JAR  file, most of which is the graphics for the user interface. Their latest client, which still
         <a href='\"http://blog.tmcnet.com/blog/tom-keating/mobile-phones/eqo-voip-im-client-now-supports-windows-mobile-and-blackberry-mobile-p.asp\"'>
          runs  on Java compatible phones
         </a>
         , lets you make VoIP calls, as well as instant  message across networks (MSN, Yahoo, AIM, GoogleTalk, ICQ, Jabber).
         <br/>
