              <div class="post-body entry-content" id="post-body-8480238255010007462">
               <div id="summary8480238255010007462">
                <div dir="ltr" style="text-align: left;" trbidi="on">
                 <div style="text-align: justify;">
                  <a href="http://3.bp.blogspot.com/-LnAt1K803MQ/T-Vk7h_fRCI/AAAAAAAAEFM/WcSueBhgMFk/s1600/SNMP.PNG" imageanchor="1" style="clear: right; float: right; margin-bottom: 1em; margin-left: 1em;">
                   <img border="0" height="175" src="http://3.bp.blogspot.com/-LnAt1K803MQ/T-Vk7h_fRCI/AAAAAAAAEFM/WcSueBhgMFk/s200/SNMP.PNG" width="200"/>
                  </a>
                  <b>
                   Simple Network Management Protocol
                  </b>
                  (SNMP) is application protocol for managing devices on IP networks. Devices include routers, switches, servers, workstations, print servers and more. Though it helps managing network devices,SNMP never seems to get the attention These days. The reason for this is vulnerabilities that it has.
                  <br/>
                  <br/>
                  <div style="text-align: left;">
                   <div style="text-align: justify;">
                    <b>
                     SNMP
                    </b>
                    is based on Manager-Agent communications. Managers, called administrative computers, have the task of monitoring or managing a group of hosts or devices on a computer network. Each managed system executes, at all times, a software component called an agent which reports information via SNMP to the manager. The agents recognizes their managers using a string called community string(same as password),which is send on each request.
                   </div>
                   <br/>
                   <div style="text-align: justify;">
                    This tutorial doesnot provides detailed description on SNMP.This tutorial focuses on enumerating an SNMP Agent.For those whose are not familiar with SNMP,I suggest a quick read through
                   </div>
                   <br/>
                   http://net-snmp.sourceforge.net/
                   <br/>
                   ttp://docwiki.cisco.com/wiki/Internetworking_Technology_Handbook
                   <br/>
                   <br/>
                   <div style="text-align: justify;">
                    This tutorial uses 2 utilities
                    <b>
                     snmpwalk
                    </b>
                    and
                    <b>
                     snmputil
                    </b>
                    . Snmpwalk dumps a lot of information on a single run that includes open ports, services, users, ipaddresses, interfaces and a lot more. While snmputil can be used to retrieve a particular information such as system name.
                   </div>
                  </div>
                  <div style="text-align: left;">
                   <br/>
                   <a href="http://my-realhackings.googlecode.com/files/snmp.rar">
                    Download
                   </a>
                   SNMP now
                   <br/>
                   <br/>
                   <b>
                    Few basic terms
                   </b>
                   <br/>
                   <br/>
                   <b>
                    MIB
                   </b>
                   - (Management Information Base) - provides a standard representation of the SNMP agent's available information and where it is stored.
                   <br/>
                   <br/>
                   <b>
                    OID
                   </b>
                   (Object Identifier)-identifier for each object kept in MIB.
                   <br/>
                   <br/>
                   Eg- .1.3.6.1.2.1.1.1.0
                   <br/>
                   <br/>
                   <b>
                    Enumeration using snmpwalk and snmputil
                   </b>
                  </div>
                  <div style="text-align: left;">
                   <br/>
                   Step 1: start snmpwalk
                   <br/>
                   <br/>

                 <div style="text-align: justify;">

                 <div style="text-align: justify;">

                 <div style="text-align: justify;">
                  <br/>
                 </div>
                 <div style="text-align: justify;">
                  2. Extract (save) resources to file in: *.res format; as a binary; or as decompiled resource scripts or images. Icons, bitmaps, cursors, menus, dialogs, string tables, message tables, accelerators, Borland forms and version info resources can be fully decompiled into their respective formats, whether as image files or  *.rc text
                 </div>

                 <div style="text-align: justify;">

Image resources (icons, cursors and bitmaps) can be replaced with an image from a corresponding image file (*.ico, *.cur, *.bmp), a *.res file or even another *.exe file. Dialogs, menus, stringtables, accelerators and messagetable resource scripts (and also Borland forms) can be edited and recompiled using the internal resource script editor.Resources can also be replaced with resources from a *.res file as long as the replacement resource is of the same type and has the same
                 </div>
                 name.
                 <br/>
                 <br/>
                 4. Add new resources to executables.Enable a program to support multiple languages, or add a custom icon or bitmap (company logo etc) to a program's dialog.
                 <br/>
                 <br/>
                 5. Delete resources.
                 <br/>
                 Most compilers add resources into applications which are never used by the application. Removing these unused resources can reduce an application's size.

                 </div>
                 <div style="text-align: justify;">
                  Microsoft Research released a community game cum programming environment for the Xbox 360 called
                  <b>
                   Kodu.
                  </b>
                  Unlike most other video games, Kodu would let players create their own video games for the Xbox without any prior knowledge of programming.
                  <br/>
                  <u>
                   <b>
                    <br/>
                    What is Kodu?
                   </b>
                  </u>
                 </div>
                 <br/>
                 <div style="text-align: justify;">
                  <b>
                   Kodu
                  </b>
                  lets kids create games on the PC and XBox via a simple visual programming language. Kodu can be used to teach creativity, problem solving, storytelling, as well as programming. Anyone can use Kodu to make a game, young children as well as adults with no design or programming skills.
                  <br/>
                  <br/>
                  The initial version of Kodu required the Xbox 360 console but now Kodu is available as a free download for your Windows PC as well. This means you no longer need an Xbox to design games with Kodu and you can play them on just about any computer using a keyboard and mouse (or an Xbox game controller, if you have one).
                  <br/>
                  <br/>
                  The entire Kodu program is more like a video game than a programming environment. It includes several tutorials to help you quickly figure out how to create new games, or you can look at the sample games bundled with the program for new ideas.
                  <br/>
                  <br/>
                  You can easily edit any game by pressing escape, make the changes and then jump back into playing the game with your changes. Although you cannot edit every little thing inside the game, it still gives you a nice launch pad for creating simple games.
                  <br/>
                  <br/>
                  You can download Kodu for PC from
                  <a href="http://fuse.microsoft.com/kodu/" target="_blank">
                   fuse.microsoft.com/kodu
                  </a>
                  . Additionally, the graphics settings can be turned down to allow it to run on a computer with lower resources. Kodu requires .NET 3.5 and the XNA framework 3.1 to run - if you dont have them, the Kodu installer will automatically download and install them for you.
                  <br/>
                  <b>
                   <br/>
                   <u>
                    Kodu Teaches more than Programming
                   </u>
                  </b>
                  <br/>
                  <ul>
                   <li>
                    Kodu is a rich tool for narrative creation and storytelling
                   </li>
                   <li>
                    Kodu demonstrates that programming is a creative medium
                   </li>
                   <li>
                    Kodu helps children with critical thinking, breaking a complex goal into manageable steps, and iterate on the design process  an approach applicable to all academic subjects, business and personal relationships
                   </li>
                   <li>
                    Kodu introduces the logic and problem solving of programming
                   </li>
                   <li>
                    Kodu introduces conditions and sequences, which teaches cause and effect
                   </li>
                   <li>
                    Students learn about cooperation, logic and creativity in addition to programming
                   </li>
                  </ul>
                  <u>
                   <b>
                    System Requirements
                   </b>
                  </u>
                  <br/>
                  <ul>
                   <li>
                    Operating Systems:
                   </li>
                   <li>
                    Windows 7
                   </li>
                   <li>
                    Windows Vista
                   </li>
                   <li>
                    Windows XP (latest updates required)
                   </li>
                   <li>
                    Additional requirements:
                   </li>
                   <li>
                    A graphics card that supports DirectX 9.0c and Shader Model 2.0 or higher
                   </li>
                   <li>
                    .NET Framework 3.5 or higher
                   </li>
                   <li>
                    XNA Framework 3.1 Redistributable
                   </li>
                  </ul>

                   In
                   <b>
                    cryptography
                   </b>
                   ,
                   <b>
                    encryption
                   </b>
                   is the process of transforming information (referred to as
                   <b>
                    plaintext
                   </b>
                   ) using an algorithm (called a cipher) to make it unreadable to anyone except those possessing special knowledge, usually referred to as a
                   <b>
                    key
                   </b>
                   . The result of the process is encrypted information (in cryptography, referred to as
                   <b>
                    ciphertext
                   </b>
                   ). In many contexts, the word encryption also implicitly refers to the reverse process, decryption (e.g. software for encryption can typically also perform decryption), to make the encrypted information readable again (i.e. to make it unencrypted).
                   <br/>
                   <br/>
                   <b>
                    Encryption software
                   </b>
                   executes an algorithm that is designed to encrypt computer data in such a way that it cannot be recovered without access to the
                   <b>
                    key
                   </b>
                   . Software encryption is a fundamental part of all aspects of modern computer communication and file protection and may include features like file shredding.The purpose of encryption is to prevent third parties from recovering the original information. This is particularly important for sensitive data like credit card numbers. Similarly this technique can be used for
                   <b>
                    virus
                   </b>
                   files to make it
                   <b>
                    undectable
                   </b>
                   to the antivirus programs.
                   <br/>
                   <br/>
                   On other hand,
                   <b>
                    Binding Exe
                   </b>
                   files means combining 2 or more exe files so that running the main execution file will also run a hidden execution file in background without the knowledge of user who is running it. Normally binding technique is done to make some virus files (mostly keyloggers) attach to some genuine softwares so that when anyone install this software, The keylogger attached to it also gets installed without the knowledge or permission of the user who is installing it.We have already discussed some binding method with

This allows Gmail engineers to  obtain user input about new features to improve them and also to assess  their popularity and whether they merit developing into regular Gmail  features. All Labs features are experimental and are subject to  termination at any time.
