         <a href="http://www.howtogeek.com/117261/how-to-activate-vlcs-web-interface-control-vlc-from-a-browser-use-any-smartphone-as-a-remote/" rel="bookmark" title="How to Activate VLCs Web Interface, Control VLC From a Browser, &amp; Use Any Smartphone as a Remote">
          How to Activate VLCs Web Interface, Control VLC From a Browser, &amp; Use Any Smartphone as a Remote
         </a>
        </h2>
        <div class="thecontent">
         <!-- google_ad_section_start -->
         <p>
          <img alt="image" border="0" height="256" src="http://cdn3.howtogeek.com/wp-content/uploads/2012/06/650x256ximage314.png.pagespeed.ic.9TIO9T9Qon.png" style="background-image: none; margin: 0px; padding-left: 0px; padding-right: 0px; display: inline; padding-top: 0px; border: 0px;" title="image" width="650"/>
         </p>
         <p>
          VLC includes a web interface, which you can enable to access your VLC player from a web browser, controlling playback from another device  particularly useful for a media center PC. VLC also offers a mobile web interface for smartphones.
         </p>
         <p>
          The web interface is turned off and locked down by default  you have to edit the web servers .hosts file or VLC will disallow all incoming connections from other devices.
         </p>
         <div id="div-1342-1" style="margin-top:10px;padding-top:10px;border-top:1px solid #CCC;margin-bottom:10px;border-bottom:1px solid #CCC;">

         <h3>
          Activating the Web Interface
         </h3>
         <p>
          To activate the web interface, click the Tools menu in VLC and select Preferences.
         </p>
         <p>
          <img alt="image" border="0" height="367" src="http://cdn3.howtogeek.com/wp-content/uploads/2012/06/616x367ximage315.png.pagespeed.ic._OgnP-9ATW.png" style="background-image: none; margin: 0px; padding-left: 0px; padding-right: 0px; display: inline; padding-top: 0px; border: 0px;" title="image" width="616"/>
         </p>
         <p>
          Click the All option under
          <em>
           Show settings
          </em>
          to view VLCs advanced settings. Scroll down in the list of advanced settings and select
          <em>
           Main interfaces
          </em>
          under the Interface header.

         </p>
         <h3>
          Allowing Remote Access
         </h3>
         <p>
          By default, the web interface is completely locked down  its restricted to localhost, which means you can only access it from the machine VLC is running on. Youll see a 403 Forbidden error page if you try to access VLCs HTTP server from any other device.
         </p>
         <p>
          <img alt="image" border="0" height="252" onload="pagespeed.lazyLoadImages.loadIfVisibleAndMaybeBeacon(this);" pagespeed_lazy_src="http://cdn3.howtogeek.com/wp-content/uploads/2012/06/290x252ximage320.png.pagespeed.ic.KEaKtKiOPG.png" src="/pagespeed_static/1.JiBnMqyl6S.gif" style="background-image: none; padding-left: 0px; padding-right: 0px; display: block; float: none; margin-left: auto; margin-right: auto; padding-top: 0px; border: 0px;" title="image" width="290"/>
         </p>
         <p>
          To allow access from other computers, youll have to edit the web interfaces .hosts file. Youll find this file in different directories on different operating systems:
         </p>
         <ul>
          <li>
           <strong>
            Windows
           </strong>
            C:\Program Files (x86)\VideoLAN\VLC\lua\http (use Program Files instead of Program Files (x86) on 32-bit versions of Windows.)
          </li>
          <li>
           <strong>
            Mac OS X
           </strong>
            /Applications/VLC.app/Contents/MacOS/share/lua/http/.hosts
          </li>
          <li>
           <strong>
            Linux
           </strong>
            /usr/share/vlc/lua/http/.hosts
          </li>
         </ul>
         <p>
          <img alt="image" border="0" height="333" onload="pagespeed.lazyLoadImages.loadIfVisibleAndMaybeBeacon(this);" pagespeed_lazy_src="http://cdn3.howtogeek.com/wp-content/uploads/2012/06/650x333ximage321.png.pagespeed.ic.kDrH6o9Eq8.png" src="/pagespeed_static/1.JiBnMqyl6S.gif" style="background-image: none; margin: 0px; padding-left: 0px; padding-right: 0px; display: inline; padding-top: 0px; border: 0px;" title="image" width="650"/>
         </p>
         <p>
          To edit this file on Windows, youll have to open Notepad  or another text editor  as Administrator. Browse to the folder mentioned above and select All Files in Notepads open dialog to view the .hosts file.
         </p>
         <p>
          <img alt="image" border="0" height="513" onload="pagespeed.lazyLoadImages.loadIfVisibleAndMaybeBeacon(this);" pagespeed_lazy_src="http://cdn3.howtogeek.com/wp-content/uploads/2012/06/432x513ximage322.png.pagespeed.ic.RUXTCf7ouN.png" src="/pagespeed_static/1.JiBnMqyl6S.gif" style="background-image: none; padding-left: 0px; padding-right: 0px; display: inline; padding-top: 0px; border: 0px;" title="image" width="432"/>
         </p>
         <p>
          You can uncomment the last two lines (to uncomment a line, remove the # at the beginning of the line) to allow access from all IP addresses, but the file notes this isnt completely safe. You could also allow a range of IP addresses  or specify the IP address of each other device you want to allow here (add each IP address on a separate line).
         </p>
         <p>
          <img alt="image" border="0" height="481" onload="pagespeed.lazyLoadImages.loadIfVisibleAndMaybeBeacon(this);" pagespeed_lazy_src="http://cdn3.howtogeek.com/wp-content/uploads/2012/06/650x481ximage323.png.pagespeed.ic.vrzsrXfI-T.png" src="/pagespeed_static/1.JiBnMqyl6S.gif" style="background-image: none; padding-left: 0px; padding-right: 0px; display: inline; padding-top: 0px; border: 0px;" title="image" width="650"/>
         </p>
         <p>
          Save the file and restart VLC after making the change.
         </p>
         <h3>
          Using the Web Interface
         </h3>
         <p>
          Plug http://123.456.7.89:8080 into a web browser on an allowed computer or smartphone to view VLCs web interface. Replace the 123.456.7.89 in the address with the IP address of the computer running VLC.
         </p>
         <p>
          If you need to find your computers IP address, you can use the ipconfig command in a Command Prompt window. Look for the IPv4 Address row under the name of your connection.
         </p>
         <p>
          <img alt="image" border="0" height="287" onload="pagespeed.lazyLoadImages.loadIfVisibleAndMaybeBeacon(this);" pagespeed_lazy_src="http://cdn3.howtogeek.com/wp-content/uploads/2012/06/645x287ximage324.png.pagespeed.ic.zVB52RMFNT.png" src="/pagespeed_static/1.JiBnMqyl6S.gif" style="background-image: none; margin: 0px; padding-left: 0px; padding-right: 0px; display: inline; padding-top: 0px; border: 0px;" title="image" width="645"/>
         </p>
         <hr/>
         <p>
          If you also want to access VLCs web interface over the Internet instead of your local network, youll have to
          <a href="http://www.howtogeek.com/66214/how-to-forward-ports-on-your-router/">
           forward ports on your router
          </a>
          .
         </p>
