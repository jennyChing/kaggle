      <div class="well well2" style="margin-top: 10px;">
       I've wanted to be Tank Girl since I was a kid. So dressing like her seemed the next logical step. hehe
       <br/>
       <br/>
       So after years of talking about it, I finally hacked all my hair off. I rocked a foot long mohawk for awhile before cutting and bleaching it in the Tank Girl 'do I have now.
       <br/>
       <br/>
       My mom my costume, I helped where I could.
       <br/>
       <br/>
       <br/>
       From the ground up:
       <br/>
       <br/>
       The boots were my own
       <br/>
       <br/>
       I just threw on some dark tights
       <br/>
       <br/>
       My mom made the chaps from scratch
       <br/>
       <br/>
       The red shorts were red jeans that she cut the legs off of and I put some small jewels on
       <br/>
       <br/>
       My mom made the vest from scratch. She got all the same patches from the movie, and made the skull ones.
       <br/>
       <br/>
       Not sure where the pouch came from, she may have made it too.
       <br/>
       <br/>
       I got the shirt at Susie's Deals I believe. I screen printed the bullseye on it.
       <br/>
       <br/>
       The gloves were leather welding gloves that my mom dyed red. Then we painted on the circles with the stars in the middle.
       <br/>
       <br/>
       Mom got me the dog collar, and she glued jewels on it.
       <br/>
       <br/>
       The helmet was the liner of a army helmet so I could have the shape but not the weight. Then my mom covered it in faux fur and used fabric paint to darken it up.
       <br/>
       <br/>
       <br/>
       My sister Gingi and her fiance Jonathan dressed as Jet Girl and Richard and we threw together a Booga costume for my friend Ejay.
       <br/>
       <br/>
       Gingi made some calls and found the REAL tank from the movie for us to take pix with!

    if(document.activeElement.nodeName === "INPUT"
    || document.activeElement.nodeName === "TEXTAREA") {
        return; // abort if focusing input box
    }

    var elems = document.getElementsByTagName("a"),
        links = {};

    for(var i = 0; i &lt; elems.length; i++) { // filter link elements
        var elem = elems[i];
        if(elem.rel === "prev") { // add prev to links object
            links.prev = elem;
        } else if(elem.rel === "next") { // ad next to links object
            links.next = elem;
        }
    }

    if(e.keyCode === 37) { // left key
        location.href = links.prev.href;
    } else if(e.keyCode === 39) { // right key
        location.href = links.next.href;
    }
};
