      </h1>

     <div id="body">
      <p>
       <strong class="trailer">
        First Look
       </strong>
       Microsoft has released a preview of Windows 10 for Raspberry Pi 2, the 30 ARMv7 computer board produced by the Cambridge-based Raspberry Pi Foundation.
      </p>
      <p>
       The version of Windows 10 for the Pi (which is also available for the Intel Atom MinnowBoard Max) is called Windows IoT Core, one of three Windows 10 IoT editions. The other two are Windows 10 IoT for mobile devices (which is ARM only and similar to Windows 10 Mobile) and Windows 10 IoT for industry devices, which is Intel only and similar to Windows 10 Enterprise, tweaked to run a single locked-down application such as for a cash or vending machine.
      </p>
      <div class="article_side_content not_moved">
       <div id="article-mpu-container">
       </div>
      </div>
      <p>
       In other words, the company has hijacked the IoT (Internet of Things) buzzword and applied it to embedded Windows. That said, the old Windows CE apparently lives on for those who need it, since unlike other versions of Windows it is a real-time operating system (RTOS).
      </p>
      <p>
       Windows 10 IoT Core is an oddity in that while it does have a GUI stack, it is limited to Microsofts Universal App Platform (UAP), though note that this includes DirectX as well as XAML (Microsofts presentation language for UAP) and HTML.
      </p>
      <p>
       This means that there is no Windows desktop, nor even a command prompt. That said, it does support PowerShell remoting, which gets you a remote PowerShell terminal from which you can run familiar Windows commands.
      </p>
      <div class="CaptionedImage Center Float">
       <a href="https://regmedia.co.uk/2015/05/20/windows10iot.png">
        <img alt="Windows 10 IoT Editions" height="364" src="https://regmedia.co.uk/2015/05/20/windows10iot.png?x=648&amp;y=364&amp;infer_y=1" title="Windows 10 IoT Editions" width="648"/>
       </a>
       <p>
        Windows 10 IoT Editions
       </p>
      </div>
      <p>
       Although education and hobbyists seems to be the main target markets for Windows 10 IoT Core on the Pi, there are also features aimed at business users, though it is not clear whether these are in the preview.
      </p>
      <p>
       Devices running the OS will be manageable via InTune (a cloud service) or System Center, and you will be able to configure updates in one of three modes: all features, security patches only or none. Domain join is not possible, though.
      </p>
      <p>
       The price? Windows 10 will include a new IoT edition for small devices that is tuned to run Windows universal apps and drivers and is royalty free to makers and device builders, said Microsofts Don Box in
       <a href="http://blogs.windows.com/bloggingwindows/2015/03/18/developing-for-the-windows-10-device-platform/" target="_blank">
        this post
       </a>
       .
      </p>
      <p>
       Note that IoT Core is not limited to UAP apps. Native Win32 apps run, but you will not see any output other than in a remote session. You can create server apps, though, and one of the samples uses Node.js with a native extension to return memory status to a browser. There is no web server in IoT Core, but Node.js has one built-in. Node.js normally uses the Chrome JavaScript runtime, but in this case it uses Microsofts Chakra engine instead.
      </p>
      <p>
       The current preview has several limitations. Wi-Fi and Bluetooth are not supported, and there is a
       <a href="http://ms-iot.github.io/content/win10/ReleaseNotes.htm" target="_blank">
        long list
       </a>
       of other issues which make the preview suitable for experimentation only. In addition, not all UAP APIs are supported.
      </p>
