        <p>
         Whoopie pies are one of my favorite desserts, but instead of my normal chocolate whoopie pies, I decided to go pumpkin, using none other than
         <a href="http://app.socialspark.com/clicks?lid=34428&amp;oid=12328500" rel="nofollow">
          Libbys Pumpkin

         <br/>
        </p>
        <h4>
         <strong>
          Advantages of Using Pumpkin
         </strong>
        </h4>
        <p>
         Yes, its spring, but pumpkin is delicious all year long, not just in the fall.
         <img alt="Oatmeal Pumpkin Whoopie Pies with Libby's Pumpkin" class="aligncenter" src="http://adventuresofmel.com/wp-content/uploads/2014/04/Oatmeal-Pumpkin-Whoopie-Pies-with-Libbys-Pumpkin.jpg" style="width: 700px; height: 503px;"/>
         Plus, did you know that pumpkin can be used in place of ingredients like butter to help reduce calories? Mmhm, thats what Im talkin about. Pumpkin is a superfood; check it out.
        </p>
        <ul>
         <li>
          Pumpkin is a great source of both fiber and Vitamin A; we all know that Vitamin A promotes healthy eye development.
         </li>
         <li>
          Pumpkin can reduce fat in recipes. You can also use pumpkin as a substitute in recipes to reduce calories.
         </li>
         <li>
          Pumpkin can be mixed with jarred or prepared sauces to reduce the sodium per serving amount.
         </li>
        </ul>
        <p>
         Pumpkin can be used year round too, in all sorts of recipes, not just in those beautiful autumn dishes that we all love to make. Throw a spoonful in a smoothie, in macaroni and cheese, or one of my favorite things to do. add this superfood as a hidden nutrient in spaghetti sauce or pasta sauce. I love adding hidden veggies and nutrients to our pasta sauce. My kids are catching on to me, but they dont seem to mind! Im definitely using pumpkin in our next pasta dish. Have leftover pumpkin? Just store it in the freezer, and youll have it for when you next need it.
        </p>
        <h4>
         <strong>
          Pumpkin Whoopie Pies
         </strong>
        </h4>
        <p>
         I love baking, so it seemed only natural to create a sweet pumpkin treat.
         <img alt="Oatmeal Pumpkin Cookies" class="aligncenter" src="http://adventuresofmel.com/wp-content/uploads/2014/04/Oatmeal-Pumpkin-Cookies.jpg" style="width: 700px; height: 495px;"/>
         These cookies were inspired by a friends oatmeal cookie recipe, but the recipe has been totally revamped to make it my own. I hope you enjoy!
         <img alt="Oatmeal Pumpkin Whoopie Pie Cookies" class="aligncenter" src="http://adventuresofmel.com/wp-content/uploads/2014/04/Oatmeal-Pumpkin-Whoopie-Pie-Cookies.jpg" style="width: 700px; height: 521px;"/>
        </p>
        <div class="easyrecipe" itemscope="" itemtype="http://schema.org/Recipe">
         <link href="http://adventuresofmel.com/wp-content/uploads/2014/04/Oatmeal-Pumpkin-Whoopie-Pie-Recipe-from-MamaBuzz.png" itemprop="image"/>

          <ul>
           <li class="ingredient" itemprop="ingredients">
            Whoopie Pie Ingredients:
           </li>
           <li class="ingredient" itemprop="ingredients">
             cup pumpkin
           </li>
           <li class="ingredient" itemprop="ingredients">
            1 cup sugar
           </li>
           <li class="ingredient" itemprop="ingredients">
            1 cup brown sugar
           </li>
           <li class="ingredient" itemprop="ingredients">
            1 egg
           </li>
           <li class="ingredient" itemprop="ingredients">
            1 tsp. vanilla
           </li>
           <li class="ingredient" itemprop="ingredients">
            2 cups flour
           </li>
           <li class="ingredient" itemprop="ingredients">
            1 tsp. salt
           </li>
           <li class="ingredient" itemprop="ingredients">
            1 tsp. baking soda
           </li>
           <li class="ingredient" itemprop="ingredients">
            1 tsp. pumpkin pie spice
           </li>
           <li class="ingredient" itemprop="ingredients">
            3 cups quick oats
           </li>
           <li class="ingredient" itemprop="ingredients">
            Frosting Ingredients:
           </li>
           <li class="ingredient" itemprop="ingredients">
            1 tbsp. butter, melted
           </li>
           <li class="ingredient" itemprop="ingredients">
            1 tsp. vanilla
           </li>
           <li class="ingredient" itemprop="ingredients">
             tsp. almond extract
           </li>
           <li class="ingredient" itemprop="ingredients">
            2 cups powdered sugar
           </li>
           <li class="ingredient" itemprop="ingredients">
            4 tbsp. skim milk
           </li>
          </ul>
          <div class="ERSClear">
          </div>

         <div class="ERSInstructions">
          <div class="ERSInstructionsHeader ERSHeading">
           Instructions
          </div>
          <ol>
           <li class="instruction" itemprop="recipeInstructions">
            Pre-heat oven to 350. Grease a cookie sheet.
           </li>
           <li class="instruction" itemprop="recipeInstructions">
            Cream pumpkin and sugars. Add in egg and vanilla, mixing well. Fold in dry ingredients, adding quick oats last. Place by tablespoonful onto prepared cookie sheet, and press with fork. Bake at 350 for 8-12 minutes or until cookies begin to turn golden.
           </li>
           <li class="instruction" itemprop="recipeInstructions">
            While cookies are baking, mix all ingredients for frosting. After cookies have cooled, sandwich together with thin layer of frosting.
           </li>
           <li class="instruction" itemprop="recipeInstructions">
            Serve and enjoy!
           </li>

  extend(Mimi.Signups.EmbedValidation.prototype, {

      this.fieldAndListValidation();

    emailValidation: function() {

          validEmail = /.+@.+\..+/;

      if(!validEmail.test(email.value)) {

      for(var i = 0; i &lt; fields.length; ++i) {
        var field = fields[i],
            type  = this.fieldType(field);
        if(type == 'checkboxes' || type == 'radio_buttons') {
          this.checkboxAndRadioValidation(field);
        } else {
          this.textAndDropdownValidation(field, type);
        }
      }
    },

    fieldType: function(field) {
      var type = field.querySelectorAll('.field_type');

      if(type.length &gt; 0) {
        return type[0].getAttribute('data-field-type');
      } else if(field.className.indexOf('checkgroup') &gt;= 0) {
        return 'checkboxes';
      } else {
        return 'text_field';
      }
    },

    checkboxAndRadioValidation: function(field) {
      var inputs   = field.getElementsByTagName('input'),
          selected = false;

      for(var i = 0; i &lt; inputs.length; ++i) {
        var input = inputs[i];
        if((input.type == 'checkbox' || input.type == 'radio') &amp;&amp; input.checked) selected = true;
      }

      if(selected) {
        field.className = field.className.replace(/ invalid/g, '');
      } else {
        if(field.className.indexOf('invalid') == -1) field.className += ' invalid';
        this.isValid = false;
      }
    },

    textAndDropdownValidation: function(field, type) {
      var inputs = field.getElementsByTagName('input');

      for(var i = 0; i &lt; inputs.length; ++i) {
        var input = inputs[i];
        if(input.name.indexOf('signup') &gt;= 0) {
          if(type == 'text_field') {
            this.textValidation(input);
          } else {
            this.dropdownValidation(field, input);
          }
        }
      }
      this.htmlEmbedDropdownValidation(field);
    },

    textValidation: function(input) {
      if(input.id == 'signup_email') return;

      var val = input.value;
      if(val == '') {
        this.textFieldError(input);
        this.isValid = false;
      } else {
        this.removeTextFieldError(input)
      }
    },

    dropdownValidation: function(field, input) {
      var val = input.value;
      if(val == '') {
        if(field.className.indexOf('invalid') == -1) field.className += ' invalid';
        this.onSelectCallback(input);
        this.isValid = false;
      } else {
        field.className = field.className.replace(/ invalid/g, '');
      }
    },

    htmlEmbedDropdownValidation: function(field) {
      var dropdowns = field.querySelectorAll('.mimi_html_dropdown');

      for(var i = 0; i &lt; dropdowns.length; ++i) {
        var dropdown = dropdowns[i],
            val      = dropdown.value;
        if(val == '') {
          if(field.className.indexOf('invalid') == -1) field.className += ' invalid';
          this.isValid = false;
          dropdown.onchange = this.validate.bind(this);
        } else {
          field.className = field.className.replace(/ invalid/g, '');
        }
      }
    },

    textFieldError: function(input) {
      input.className   = 'required invalid';
      input.placeholder = input.getAttribute('data-required-field');
    },

    removeTextFieldError: function(input) {
      input.className   = 'required';
      input.placeholder = '';
    },

    onSelectCallback: function(input) {

        }
      }
    },

    updateFormAfterValidation: function() {

    },

    setFormClassName: function() {
      var name = this.form.className;

      if(this.isValid) {
        return name.replace(/\s?mimi_invalid/, '');
      } else {
        if(name.indexOf('mimi_invalid') == -1) {
          return name += ' mimi_invalid';
        } else {
          return name;
        }
      }
    },

          text = this.submit.getAttribute('data-invalid-text');
        } else {
          text = this.submit.getAttribute('data-choose-list');

      var _this = this;
      window[this.callbackName] = function(response) {

        _this.onSubmitCallback(response);

        }
      }

jQuery(document).ready(function($) {
$(function() {	
	var process_script_url = 'http://adventuresofmel.com/wp-content/plugins/wp-cart-for-digital-products' + '/ajax_process_cart_requests.php';
	var save_ret_cart_msg = $('div.eStore_save_retrieve_cart_message');
	
	$('.eStore_save_cart_section').on("click", function (){
		save_ret_cart_msg.html("").append("Processing Your Request...");		
		var dataString = 'eStore_cart_action=eStore_save_cart';		
		$.ajax({
			type: "POST",
			url: process_script_url,
			data: dataString,
			dataType: "json",
			success: function(data) {				
				//var dataStr = JSON.stringify(data); console.log(dataStr);
				if(data.status == "success"){
					save_ret_cart_msg.html("").append('Your cart has been saved! Shopping Cart ID : ' + data.ID);
				}
				else{
					save_ret_cart_msg.html("").append('&lt;span class="eStore_warning"&gt;' + data.details + '&lt;/span&gt;');
				}
			}
		});				
		return false;
	});

	$('.eStore_retrieve_cart_section').on("click", function (){
		var cart_id = prompt("Shopping Cart ID", "");	
		if(cart_id == null){
			return false;
		}
		else if(cart_id ==""){
			save_ret_cart_msg.html("").append('&lt;span class="eStore_warning"&gt;The shopping cart ID you entered does not appear to be correct!&lt;/span&gt;');
			return false;
		}
		cart_id = $.trim(cart_id);
		save_ret_cart_msg.html("").append("Processing Your Request...");
		var dataString = 'eStore_cart_action=eStore_retrieve_cart&amp;cart_id=' + cart_id;
		$.ajax({
			type: "POST",
			url: process_script_url,
			data: dataString,
			dataType: "json",
			success: function(data) {
				if(data.status == "success"){	
					save_ret_cart_msg.html("").append('Success');	
					target_url = window.location.href.split("#")[0];
					window.location.href = target_url;					
				}
				else{
					save_ret_cart_msg.html("").append('&lt;span class="eStore_warning"&gt;The shopping cart ID you entered does not appear to be correct!&lt;/span&gt;');	
				}	
			}
		});		
		return false;
	});
});
});
