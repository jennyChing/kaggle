           Hi Everyone!
           <br/>
           <br/>
           Members have come to me and asked how I HQ so I thought I would just make a guide for our Free Company to utilize. Hopefully with my guide you will be able to walk away with the knowledge to empower yourself and begin popping out those HQs like crazy!
           <br/>
           <br/>
           With my skills and 18 step rotation, I am able to HQ
           <strong>
            ALL
           </strong>
           2 Star items without the need of any HQ mats, RNG or FOOD.
           <br/>
           <br/>
           Keep in mind, I do have all DoH at 50 and understand that all of these abilities are not available to all, but it will give you an idea on the mechanics that might point you into the right direction.
           <br/>
           <br/>
           The DoH and levels that are required for this rotation is outlined below:
           <br/>
           <br/>
           50 Alchemy
           <br/>
           50 Weaver
           <br/>
           50 Carpenter
           <br/>
           37 Culinary
           <br/>
           15 Blacksmith
           <br/>
           15 Leatherworker
           <br/>
           <br/>
           I have all fully melded i55 DoH specific gear and the following stats are what is needed to do this 18 step HQ. (Edit: I have higher stats as time has went on since I wrote this, but rest assured the following are still valid for the rotation)
           <br/>
           <br/>
           Craftsmanship: 359
           <br/>
           Control: 351
           <br/>
           CP: 345 (Only 341 is needed for this rotation)
           <br/>
           <br/>
           A guide on how you would want to maximize the melding of your crafting gear can be found
           <a href="http://www.ffxivguild.com/ff14-arr-doh-crafting-gear-guide-endgame-l50-maximum-materia-melds/" target="_blank">
            here
           </a>
           <br/>
           <br/>
           With that said, here we go!!
           <br/>
           <br/>
           <strong>
            <span style="font-size:16px">
             Base skills you will need that is available for all DoH:
            </span>
           </strong>
           <br/>
           <br/>
           <img class="bbcode" src="http://xivdbimg.zamimg.com/images/icons/002000/002502.png"/>
           Basic Touch
           <br/>
           <strong>
            DoH:Lv5 -
           </strong>
           Increases quality.
           <br/>
           Efficiency: 100%
           <br/>
           Success Rate: 70%
           <br/>
           <br/>
           <img class="bbcode" src="http://xivdbimg.zamimg.com/images/icons/002000/002313.png"/>
           Steady Hand
           <br/>
           <strong>
            DoH:Lv9 -
           </strong>
           Improves action success rate by 20% for the next five steps.
           <br/>
           <br/>
           <img class="bbcode" src="http://xivdb.com/images/icons/002000/002603.png"/>
           Inner Quiet
           <br/>
           <strong>
            DoH:Lv11 -
           </strong>
           Grants a bonus to control with every increase in quality.
           <br/>
           <br/>
           <img class="bbcode" src="http://xivdb.com/images/icons/002000/002620.png"/>
           Great Strides
           <br/>
           <strong>
            DoH:Lv21 -
           </strong>
           Doubles efficiency of next "Touch" action. Effect active for three steps.
           <br/>
           <br/>
           <br/>
           <strong>
            <span style="font-size:16px">
             Cross class skills you will need:
            </span>
           </strong>
           <br/>
           <br/>
           <img class="bbcode" src="http://xivdb.com/images/icons/002000/002622.png"/>
           Comfort Zone
           <br/>
           <strong>
            ALC:Lv50 -
           </strong>
           Restores 8 CP after each step for the next ten steps.
           <br/>
           <br/>
           <img class="bbcode" src="http://xivdb.com/images/icons/002000/002522.png"/>
           Careful Synthesis II
           <br/>
           <strong>
            WVR:Lv50 -
           </strong>
           Guarantee 100% success at 120% efficiency.
           <br/>
           <br/>
           <img class="bbcode" src="http://xivdbimg.zamimg.com/images/icons/002000/002406.png"/>
           Waste Not
           <br/>
           <strong>
            LTW:Lv15 -
           </strong>
           Reduces loss of durability by 50% for the next four steps.
           <br/>
           <br/>
           <img class="bbcode" src="http://xivdb.com/images/icons/002000/002721.png"/>
           Steady Hand II
           <br/>
           <strong>
            CUL:Lv37 -
           </strong>
           Improves action success rate by 30% for the next five steps.
           <br/>
           <br/>
           <img class="bbcode" src="http://xivdbimg.zamimg.com/images/icons/002000/002106.png"/>
           Ingenuity
           <br/>
           <strong>
            BSM:Lv15 -
           </strong>
           Lowers recipe level to current level for the next five steps.
           <br/>
           <br/>
           <img class="bbcode" src="http://xivdb.com/images/icons/002000/002021.png"/>
           Byregot's Blessing
           <br/>
           <strong>
            CRP:Lv50 -
           </strong>
           Increases quality.
           <br/>
           <br/>
