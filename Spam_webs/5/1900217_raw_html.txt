        <p>
         NEW YORK (
         <a href="http://www.thestreet.com" target="_blank">
          TheStreet
         </a>
         ) -- As I read reaction to
         <a href="http://www.thestreet.com/story/11977003/1/a-hulu-deal-never-made-sense-anyway.html" target="_blank">
          the dead-again
          <b>
           Hulu
          </b>
          deal
         </a>
         , practically every point that gets made misses the larger, more meaningful one.
        </p>
        <p>
        </p>
        For instance, it strikes me kinda funny -- and should strike you the same -- when Peter Kafka at
        <a href="http://allthingsd.com/20130712/hulus-new-plan-compete-with-netflix-and-amazon-how-is-hulu-going-to-pay-for-that/" rel="nofollow" target="_blank">
         <i>
          All Things D
         </i>
        </a>
        contends that Hulu's "new plan" is to compete with
        <b>
         Netflix
        </b>
        <span class=" TICKERFLAT">
         (
         <a href="/quote/NFLX.html">
          NFLX
         </a>
         -
         <a href="http://secure2.thestreet.com/cap/prm.do?OID=028198&amp;ticker=NFLX">
          Get Report
         </a>
         <a class=" arrow" href="/quote/NFLX.html">
          <span class=" tickerChange" id="story_NFLX">
          </span>
         </a>
         )
        </span>
        and
        <b>
         Amazon.com
        </b>
        <span class=" TICKERFLAT">
         (
         <a href="/quote/AMZN.html">
          AMZN
         </a>
         -
         <a href="http://secure2.thestreet.com/cap/prm.do?OID=028198&amp;ticker=AMZN">
          Get Report
         </a>
         <a class=" arrow" href="/quote/AMZN.html">
          <span class=" tickerChange" id="story_AMZN">
          </span>
         </a>
         )
        </span>
        before asking, with a sort of blissful ignorance, how the company plans to pay for it. Kafka claims to have a source -- he defines it as "someone who knows" -- but this person absolutely cannot be high-level. If they are, he's only getting the basic information he reports, but, clearly, very little to inform his understanding of the situation.
        <p>
        </p>
        <div class=" butonTextPromoAd">
         <div class=" ym" id="ym_44444440">
         </div>
        </div>
        There's a major misconception about how big media television executives view Netflix. They do not view it as a threat. Because, if they did, they would turn Hulu into the ideal one-stop shop for all content -- nostalgia, catch-up programming and first-run/live television -- tomorrow. Such a move, even if only slightly less halfhearted than it already is, would put an almost-instant kibosh on Netflix.
        <p>
        </p>
        Recall who owns and controls Hulu: A consortium of
        <b>
         Twenty-First Century Fox
        </b>
        <span class=" TICKERFLAT">
         (
         <a href="/quote/FOX.html">
          FOX
         </a>
         -
         <a href="http://secure2.thestreet.com/cap/prm.do?OID=028198&amp;ticker=FOX">
          Get Report
         </a>
         <a class=" arrow" href="/quote/FOX.html">
          <span class=" tickerChange" id="story_FOX">
          </span>
         </a>
         )
        </span>
        ,
        <b>
         Disney
        </b>
        <span class=" TICKERFLAT">
         (
         <a href="/quote/DIS.html">
          DIS
         </a>
         -
         <a href="http://secure2.thestreet.com/cap/prm.do?OID=028198&amp;ticker=DIS">
          Get Report
         </a>
         <a class=" arrow" href="/quote/DIS.html">
          <span class=" tickerChange" id="story_DIS">
          </span>
         </a>
         )
        </span>
        and
        <b>
         NBC Universal
        </b>
        , a division of
        <b>
         Comcast
        </b>
        <span class=" TICKERFLAT">
         (
         <a href="/quote/CMCSA.html">
          CMCSA
         </a>
         -
         <a href="http://secure2.thestreet.com/cap/prm.do?OID=028198&amp;ticker=CMCSA">
          Get Report
         </a>
         <a class=" arrow" href="/quote/CMCSA.html">
          <span class=" tickerChange" id="story_CMCSA">
          </span>
         </a>
         )
        </span>
        .
        <p>
        </p>
        These guys could unleash an all-out assault that would put Netflix out of business in an instant. But that would be a kamikaze move that should not happen and does not have to happen anytime soon. There's no need, given the reality that Netflix has little, if any, material impact on the big media establishment, particularly the major content producers. These guys continue to deliver numbers -- ratings and revenue -- hand over fist.
        <p>
        </p>
        Tech guys like Kafka and media critics such as Bryan Stelter at
        <i>
         The New York Times
        </i>
        <a href="http://www.nytimes.com/2013/07/13/business/media/owners-of-hulu-call-off-sale-and-plan-to-invest-750-million.html" rel="nofollow" target="_blank">
         eat up
        </a>
        talking points such as Hulu will invest to compete with Netflix or Disney CEO Robert Iger's quotable:
        <i>
         The future of Hulu is bright, and if the future of Hulu is bright, then we should hold onto it
        </i>
        .
        <p>
        </p>
        We should stop calling folks who produce this type of story "reporters"; they're "regurgitators."
        <p>
        </p>
        There's only a shred of truth at the surface of the
        <i>
         Hulu will compete with Netflix
        </i>
        and
        <i>
         Hulu has a bright future
        </i>
        talking points.

function addToTargeting(key, value) {
    try {
        if (typeof value == "string") {
            value = value.toLowerCase();
            value = clean(value);
        }


        if (targetingValues.hasOwnProperty(key)) {
            targetingValues[key].push(value);
        }
        else {
            targetingValues[key] = new Array();
            targetingValues[key][0] = value;
        }
    } catch (e) {
    }
}

function clean(str) {
    //   s = s.toLowerCase();
    var s = str.toString();
    s = s.toLowerCase();
    s = s.replace(/ /g, "-");
    s = s.replace(/\|\.|\[|\]|\*|\!|\@|#|$|\%|\^|\&amp;|\(|\)|\:|\'|\"|\&lt;|\&gt;|\?|\\|\/|\~|\`/g, "");
    return s;
}

function animateSlider(data) {
    $(document).ready(function () {
        var w = window.innerWidth;
        var shiftLeft = w/2 - 150;
        try {
            $('body').append($('&lt;div/&gt;', {
                id:'swslider',
                style:'position: fixed;top: 200px;left:-400px'

            }));
            $("#swslider").html(data);
            setTimeout(function () {
                $("#swslider").animate({"left":shiftLeft}, 1900);
            }, 6000);

            setTimeout(function () {
                $("#swslider").hide();
            }, 26000);
        } catch (e) {
        }
    });

}

function bottomOverlay(data) {
    $(document).ready(function () {
        try {
            var now = new Date();
            var cookieName = "tstbottomoverlay_" + now.getMonth() + "" + now.getDate() + "" + now.getYear();
            if (document.cookie.indexOf(cookieName) == -1) {
                var expires = new Date();
                expires.setHours(23);
                expires.setMinutes(59);
                expires.setSeconds(59);
                expires = expires.toGMTString();
                document.cookie = cookieName + "=" + now + ";expires=" + expires + ";path=/;domain=.thestreet.com";
            }
            else {
                return false;
            }
            $('body').append($('&lt;div/&gt;', {
                id:'btmoverlay',
                style:'position: fixed;bottom:-120px;left:820px'
            }));
            $("#btmoverlay").html(data);
            setTimeout(function () {
                $("#btmoverlay").animate({"bottom":"0px"}, 1200);
            }, 15000);

            setTimeout(function () {
                $("#btmoverlay").hide();
            }, 30000);
        } catch (e) {
        }
    });

}
