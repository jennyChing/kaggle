         <h1 class="title">
          How-To Enable Aero Peek Preview With Firefox Tabs
         </h1>
        </header>
        <div class="post-meta">
         <span class="author vcard">
          <span class="fn">
           <a href="http://www.groovypost.com/howto/author/groovycrew/" rel="author" title="Posts by Austin Ruthruff">
            Austin Ruthruff
           </a>
          </span>
         </span>
         |
         <abbr class="date time published" title="2010-05-25T07:00:31-0700">
          May 25, 2010
         </abbr>
         <span class="small">
          in
         </span>
         <span class="categories">
          <a href="http://www.groovypost.com/firefox/">
           Firefox
          </a>
         </span>
        </div>
        <section class="entry">
         <p>
          <a href="http://www.groovypost.com/tags/firefox/" rel="nofollow" title="Groovy Firefox Tips, Tricks, How-to, and Downloads">
           <img align="left" alt="Groovy Firefox Logo" border="0" height="100" src="//www.groovypost.com/wp-content/uploads/2010/05/image61.png" style="margin: 0px 15px 15px 0px; border: 0px;" width="102"/>
          </a>
          Ever since the introduction of tabbed browsing, there hasnt been very many circumstances where having extra windows open is useful. This is especially true with the awesome new Aero Peek feature built into the Windows 7 Taskbar. Unfortunately, it wasnt until Firefox 3.6 that Firefox actually supported Aero Peek with its browser Tabs and fore some reason Firefox comes with this feature disabled by default??? No worries, I have a simple tweak to enable
          <strong>
           Aero Peek Preview with Firefox Tabs
          </strong>
          !
         </p>
         <p>
          <span style="text-decoration: underline;">
           <strong>
            How To Enable Aero Peek Preview with Firefox Tabs
           </strong>
          </span>
         </p>
         <p>
          <strong>
           1.
          </strong>
          In a Firefox window
          <em>
           Type
          </em>
          <strong>
           about:config
          </strong>
          in the address bar. When the page loads youll see a warning that you are about to enter the advanced settings of Firefox.
          <em>
           Click
          </em>
          the
          <strong>
           Ill be careful, I promise!
          </strong>
          button to enter.
         </p>
         <p>
          <img alt="about:config firefox warning" border="0" height="495" src="//www.groovypost.com/wp-content/uploads/2010/05/image_440.png" style="margin: 10px; border-width: 0px;" width="580"/>
         </p>
         <p>
          <strong>
           2.
          </strong>
          In the Filter box
          <em>
           Type
          </em>
          or
          <em>
           Paste
          </em>
          in the following:
         </p>
         <p>
          <strong>
           browser.taskbar.previews.enable
          </strong>
         </p>
         <p>
          The filter should now display the preference with a matching name. Once you see it
          <em>
           Right-Click
          </em>
          it and
          <strong>
           Select
          </strong>
          toggle. This will switch the Value from
          <strong>
           false
          </strong>
          to
          <strong>
           true
          </strong>
          . You can also
          <em>
           Double Click
          </em>
          the setting to set the value as
          <strong>
           True
          </strong>
          .
         </p>
         <p>
          <img alt="change the browser.taskbar.previews.enable preference" border="0" height="495" src="//www.groovypost.com/wp-content/uploads/2010/05/image_531.png" style="margin: 10px; border-width: 0px;" width="580"/>
         </p>
         <p>
          The setting should take affect immediately (
          <em>
           it did on my system
          </em>
          ) however you might need to restart Firefox for the setting to fully take effect. Once completed, just hover over the Firefox Icon to see all the Firefox tabs currently open in all your Firefox windows.
         </p>
         <p>
          <img alt="firefox with tabs shown as previews" border="0" height="111" src="//www.groovypost.com/wp-content/uploads/2010/05/image_630.png" style="margin: 10px; border-width: 0px;" width="650"/>
         </p>
         <p>
          Youll also notice that the Firefox taskbar icon now has tabbed edges, the icon will now change functionally to identify that there are additional tabs open within Firefox windows.
         </p>
         <p>
          <img alt="fanned or tabbed edges on firefox icon in taskbar" border="0" height="152" src="//www.groovypost.com/wp-content/uploads/2010/05/image_724.png" style="margin: 10px; border-width: 0px;" width="253"/>
         </p>
