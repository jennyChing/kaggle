     <p data-content-expand-id="description">
      Have you ever dreamed about activating your car accessories remotely from your mobile device?
      <br/>
      <br/>
      The Remote EX for NISSAN allows you to activate several accessories from your car remotely. You can turn on the head lights, sound the horn and lock/unlock doors without touching your car keys.
      <br/>
      * Turn Lights Flasher
      <br/>
      * Parking Locator (*)
      <br/>
      * Head Lights (*)
      <br/>
      * Low Beam (*)
      <br/>
      * High Beam (*)
      <br/>
      * Front Fog Lights (*)
      <br/>
      * Turn Lights (*)
      <br/>
      * Open Trunk (*)
      <br/>
      * Lock/Unlock Doors (*)
      <br/>
      * Horn (*)
      <br/>
      * Wipers (*)
      <br/>
      <br/>
      Commands marked with (*) are available through in-app purchases.
      <br/>
      <br/>
      <br/>
      <br/>
      *PLEASE NOTE* that other NISSAN models may be supported, but the plugin was tested only on the following models:
      <br/>
      <br/>
      * 350Z 3.5 (Z33)
      <br/>
      * 370Z 3.7 (Z34)
      <br/>
      * Altima 2.5/3.5 (L32)
      <br/>
      * Armada 5.6 (TA60)
      <br/>
      * Frontier 4.0 (D40)
      <br/>
      * Juke 1.6/3.8 (F15)
      <br/>
      * March/Micra 1.5 (K13)
      <br/>
      * Maxima 3.5 (A35)
      <br/>
      * Murano 3.5 (Z51)
      <br/>
      * Pathfinder 4.0 (R51)
      <br/>
      * Qashqai 1.6/2.0 (J10)
      <br/>
      * Tiida/Versa 1.8 (C11)
      <br/>
      * Titan 5.6 (A60)
      <br/>
      * Xterra 4.0 (N50)
      <br/>
      * X-Trail 2.0 (T31)
      <br/>
      <br/>
      <br/>
      <br/>
      Remote EX can also be used with the latest version of Torque Pro.
      <br/>
      <br/>
      <br/>
      <br/>
      <br/>
      Installation
      <br/>
      -------------------
      <br/>
      <br/>
      1) After purchasing the application on Google Play, make sure that you see it listed on your Android device installed applications list.
      <br/>
      <br/>
      2) Launch the app directly from your device or through Torque Pro by clicking on the app icon
      <br/>
      <br/>
      3) Select your car model on the MODEL tab and confirm
      <br/>
      <br/>
      4) The app will display the available functions for your car on the COMMANDS tab
      <br/>
      <br/>
      5) Commands displayed in GREEN are available to be used immediately while commands in GRAY are available through in-app purchases
      <br/>
      <br/>
      6) Scroll to the STORE tab to check all the available commands for sale
      <br/>
      <br/>
      7) Purchased commands are automatically added to the available command list and it is managed by Google Play, being available on all mobile devices linked to your Google account.
      <br/>
      <br/>
      If using the plugin from Torque, make sure that you also:
      <br/>
      <br/>
      1) Go to Torque Pro "Settings"
      <br/>
      <br/>
      2) Make sure that you can see the plugin listed also on Torque Pro by clicking "Settings" &gt; "Plugins"&gt; "Installed Plugins".
      <br/>
      <br/>
      3) Mark the "Allow plugins full access" box, since Remote EX needs access to the OBD adapter.
      <br/>
      <br/>
      More features/parameters will be added in further releases. If you have comments and/or suggestions please just let me know.
