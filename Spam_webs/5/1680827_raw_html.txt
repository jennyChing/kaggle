          <h2 class="post-title">
           <a href="http://www.tekgoblin.com/2010/10/26/walgreens-sells-android-tablet-for-99-but-is-it-real/" title="Permanent Link to Walgreens sells Android tablet for $99, but is it real">
            Walgreens sells Android tablet for $99, but is it real
           </a>
          </h2>
          <div style="clear: both;">
          </div>
          <p style="text-align: center;">
           <a href="http://cdn3.tekgoblinmedia.com/wp-content/uploads/walgreens-tablet.jpg?eebfd3">
            <img alt="" class="size-full wp-image-6559 aligncenter" height="299" src="http://cdn3.tekgoblinmedia.com/wp-content/uploads/walgreens-tablet.jpg?eebfd3" title="walgreens-tablet" width="443"/>
           </a>
          </p>
          <p>
           The drugstore Walgreens has an Android Tablet called the Maylong M-150 on their site for sale for only $99. They claim that the tablet sports a 7-inch LCD screen, and runs the Android operating system but does not state which version. The rest of the specifications are very nice if true for the price, they are as follows:
          </p>
          <ul>
           <li>
            Google Android Operating System
           </li>
           <li>
            LCD color touch-screen 7-inches
           </li>
           <li>
            Full Internet browsing capability
           </li>
           <li>
            Experience YouTube at your fingertips
           </li>
           <li>
            Easy access to emails
           </li>
           <li>
            Download and play games
           </li>
           <li>
            Included digital music, video player and digital picture viewer
           </li>
           <li>
            Digital picture frame
           </li>
           <li>
            e-Book reader
           </li>
           <li>
            Download and install custom Google Android Applications
           </li>
          </ul>
          <ul>
           <li>
            <strong>
             CPU
            </strong>
            : ARM9(VM8505+)
           </li>
           <li>
            <strong>
             Memory  RAM
            </strong>
            : 256MB DDR
           </li>
           <li>
            <strong>
             Memory  Flash
            </strong>
            : Built-in Flash
           </li>
           <li>
            <strong>
             Display  Touch Panel
            </strong>
            : 7-inch TFT LCD
           </li>
           <li>
            <strong>
             Display
            </strong>
            : Resolution 800 X 480 Pixels
           </li>
           <li>
            <strong>
             WIFI
            </strong>
            : 802.11b/g
           </li>
           <li>
            <strong>
             Input/Output  Touch Panel
            </strong>
            : Resistive type touch panel
           </li>
           <li>
            <strong>
             Input/Output  Speaker
            </strong>
            : Built-in loud speakers
           </li>
           <li>
            <strong>
             Input/Output  SD/MMC
            </strong>
            : T-Flash card slot
           </li>
           <li>
            <strong>
             Input/Output  Network/USB
            </strong>
            : Dongle for RJ-45 network and USB connection
           </li>
           <li>
            <strong>
             Buttons/Switches
            </strong>
            : Power On/Off, Volume adjustment
           </li>
           <li>
            <strong>
             Battery
            </strong>
            : Built-in Li-Polymer battery
           </li>
           <li>
            <strong>
             Charger  Input
            </strong>
            : AC 100-240V, 0.5A
           </li>
           <li>
            <strong>
             Charger  Output
            </strong>
            : 9V, 1500MA
           </li>
          </ul>
          <ul>
           <li>
            Tablet device with 7-inch color touch screen
           </li>
           <li>
            Stylus
           </li>
           <li>
            AC Adapter
           </li>
           <li>
            Network/USB dongle
           </li>
           <li>
            Users manual
           </li>
          </ul>
          <p>
           One of the main problems I have with this tablet is the picture, take a good look what do you see. It is a sideways picture of an iPad and not the actual tablet that they claim to sell. I doubt that Walgreens would sell a fake tablet but they need to actually use the correct picture.
          </p>
          <p>
           The specifications specify an ARM9 CPU but not the clock speed. Upon searching for the processor I found that it could be around 533 MHZ which isnt too bad and I even found a
           <a href="http://www.alibaba.com/product-gs/333991221/10_1_MID_PDA_laptop_with_VIA_VM8505_CPU.html">
            similar tablet
           </a>
           for the same price. The tablet I found is a TOLOVO which runs Android 1.6 which is quite possible the version of Android that the Walgreens tablet may be running.
          </p>
          <p style="text-align: center;">
           <a href="http://cdn3.tekgoblinmedia.com/wp-content/uploads/cheaptablet.png?eebfd3">
            <img alt="" class="size-full wp-image-6558 aligncenter" height="222" src="http://cdn3.tekgoblinmedia.com/wp-content/uploads/cheaptablet.png?eebfd3" title="cheaptablet" width="381"/>
           </a>
          </p>
          <p>
           Now I have found another tablet that has the exact same picture that is on the Walgreens site found
           <a href="http://www.alibaba.com/product-gs/333991221/10_1_MID_PDA_laptop_with.html?newId=335639882&amp;pn=2&amp;pt=10&amp;t=12&amp;cids=">
            here.
           </a>
           The tablet specs almost match the Walgreens specs and believe it or not it does look like an iPad. Check it out.
          </p>
          <p>
           <strong>
            Update
           </strong>
           : I have found the actual tablet
           <a href="http://www.maylonggroup.com/M-150_MID.htm">
            here
           </a>
           .
          </p>
