     <div class="entry">
      <p>
       pagescroll is a super lightweight jQuery one page scrolling plugin that utilizes CSS3
       <a href="http://www.jqueryscript.net/tags.php?/3D/">
        3D
       </a>
       transforms for page transition effects, with a a fallback to jQuery's
       <code>
        animate()
       </code>
       when CSS3 are not supported.
      </p>
      <h2>
       How to use it:
      </h2>
      <p>
       1. Split your one page website into several sections. With this plugin, you can horizontally or vertically navigate through these content sections with arrow keys.
      </p>
      <pre class="brush:xml;">
       &lt;div id="container"&gt;
  &lt;div class="section active" id="section0"&gt; &lt;/div&gt;
  &lt;div class="section" id="section1"&gt; &lt;/div&gt;
  &lt;div class="section" id="section2"&gt; &lt;/div&gt;
  &lt;div class="section" id="section3"&gt; &lt;/div&gt;
&lt;/div&gt;
      </pre>
      <p>
       2. Make the content sections 100% height and width.
      </p>
      <pre class="brush:css;">
       body,
html {
  padding: 0;
  margin: 0;
  height: 100%;
  overflow: hidden;
}

#container,
.section {
  height: 100%;
  position: relative;
}
      </pre>
      <p>
       3. Load jQuery library and jQuery pagescroll plugin at the end of the document..
      </p>
      <pre class="brush:xml;">
       &lt;script src="jquery.min.js"&gt;&lt;/script&gt; 
&lt;script src="onepagescroll.js"&gt;&lt;/script&gt;
      </pre>
      <p>
       4. Call the plugin on the wrapper element and set the position of page transitions.
      </p>
      <pre class="brush:js;">
       $('#container').onepagescroll({
  'direction': 'vertical'
})
      </pre>
      <p>
       5. Full plugin options.
      </p>
      <pre class="brush:js;">
       $('#container').onepagescroll({

// animation duration in ms
'duration': 500,

// horizontal or vertical
'direction': 'horizontal',

// show side pagination
'showpages': true

})
      </pre>
