     <div class="headline">
      <h1 class="heading-sm">
       Derivatives In Calculus
      </h1>
     </div>
     <div class="number_descp">
      <blockquote>
       <p>
        Differentiation is one of the basic branches of Calculus. It describes the real world rates of change and helps us describe the physical universe and natural phenomena in terms of Mathematics.
       </p>
       <p>
        Rate of change involves 2 or more variables:
       </p>
       <br/>
       <ul class="hand blackquote-ul-text-size">
        <li>
         <i class="fa fa-hand-o-right">
         </i>
         Dependent variables
        </li>
        <li>
         <i class="fa fa-hand-o-right">
         </i>
         Independent variables
        </li>
       </ul>
      </blockquote>
      <hr/>
      <h2 class="font-bold zero-bottom-margin font-size-20 heading-decoration">
       <span>
        Dependent and Independent Variables
       </span>
      </h2>
      <br/>
      <p>
       <strong>
        In differential Calculus
       </strong>
       , we mainly deal with the rate of change of a dependent variable with respect to one or more independent variables.
                We first explain the terms dependent and independent variables.
       <br/>
       <br/>
       We usually write `y = f(x)` where `f(x)` is the value of `f ` at ` x  D ` (the Domain of the function f). Let us consider the functional relation
                `y = f(x) = x^2 + 1`
       <br/>
       <br/>
       For different values of `x  D, f(x)` or the expression `x^2 + 1` assumes different values. For example; if `x = 1, 1.5, 2` etc., then
      </p>
      <br/>
      <div class="function">
       `f(1) =(1)^2 + 1 = 2,`
       <br/>
       <br/>
       `f(1.5) `=`(1.5)^2 + 1` = `2.25 + 1 `= `3.25,`
       <br/>
       <br/>
       `f(2) =(2)^2 + 1 = 5`
       <br/>
      </div>
      <br/>
      <p>
       We see that for the change `1.5  1 = 0.5` in the value of x, the corresponding change in the value of `y ` or ` f(x)` is given by
                `f(1.5)  f(1)` =` 3.25  2 `= `1.25`
       <br/>
       <br/>
       It is obvious that the change in the value of the expression `x^2 + 1 (or f(x)) `depends upon the change in the value of the variable `x`. As `x`
                behaves independently, so we call it the
       <b>
        independent
       </b>
       variable. But the behavior of `y or f(x)`
       <b>
        depends
       </b>
       on the variable `x`, so we
                call it the dependent variable.
       <br/>
       <br/>
       The change in the value of `x` (positive or negative) is called the increment of `x` and is denoted by the symbol `x` (read as delta x). The
                corresponding change in the value of dependent variable `y or f(x)` for the change `x` in the value of `x` is denoted by ` yor f = f(x + x)  f(x)`.
                Usually the small changes in the value of the variables are taken as increments of variables.
      </p>
      <hr/>
      <h2 class="font-bold zero-bottom-margin font-size-20 heading-decoration">
       <span>
        AVERAGE / INSTANTANEOUS RATE OF CHANGE
       </span>
      </h2>
      <br/>
      <p>
       Suppose a particle (or an object) is moving in a straight line and its positions from (from some fixed point) after times t and t1 are given by` s(t)`
                and `s(t1)`, respectively. Then the distance traveled in the time interval `t1  t` where `t1 &gt; t ` is ` s(t1)  s(t)`.
      </p>
      <br/>
      <p>
       The difference quotient`(s(t1)  s(t))/(t1  t)`represents the average rate of change of distance over the time interval `t1  t`.
      </p>
      <br/>
      <p>
       If `t1  t` is not small, then the average rate of change does not represent an accurate rate of change near `t`. If however `t1` approaches `t`, then
      </p>
      <br/>
      <div class="function text-center">
       `Limt1  t(s(t1)  s(t))/(t1  t)`
      </div>
      <br/>
      <p>
       Provided this limit exists, is called the
       <b>
        instantaneous
       </b>
       rate of change of s with respect to t at t and is written as s'(t).
      </p>
      <hr/>
      <h2 class="font-bold zero-bottom-margin font-size-20 heading-decoration">
       <span>
        Definition of Derivative
       </span>
      </h2>
      <br/>
      <div class="function">
       If for some  `x1 = x +x ` i.e.,` x1  x = ` ,  then the limit
      </div>
      <div class="text-center">
       `Limx 0(f(x+ x)  f(x))/x`
      </div>
     </div>
     <p>
      provided it exists is defined to be the
      <b>
       derivative of f.
      </b>
      A common notation used for derivative of a function `f(x)` is `f'(x)` (read as `  f-`
            prime of ` x ` ).
     </p>
     <hr/>
     <h2 class="font-bold zero-bottom-margin font-size-20 heading-decoration">
      <span>
       Definition of Differentiation
      </span>
     </h2>
     <br/>
     <p>
      Differentiation is the process of finding derivative. The two things are same, so do not be confused between the two.
     </p>
     <hr/>
     <h2 class="font-bold zero-bottom-margin font-size-20 heading-decoration">
      <span>
       Notation used for Derivative
      </span>
     </h2>
     <br/>
     <p>
      `dy/dx` is a widely used notation for the derivative of a dependant variable y w.r.t. an independent variable
      <br/>
      X. Also this derivative is represented by y'.We will be using all these notations in the future throughout this course content.
     </p>
     <hr/>
     <h2 class="font-bold zero-bottom-margin font-size-20 heading-decoration">
      <span>
       Techniques / Theorems to find Derivatives of Functions:
      </span>
     </h2>
     <br/>
     <p>
      Depending on different types of functions, we can have various methods to find their derivatives. The most basic approach used for finding derivatives of
            functions is as shown above, and is called
      <b>
       differentiation by definition.
      </b>
     </p>
     <hr/>
     <h2 class="font-bold zero-bottom-margin font-size-20 heading-decoration">
      <span>
       Differentiation By Definition / Ab-initio / First Principle Method:
      </span>
     </h2>
     <br/>
     <p>
      We present an example of differentiation that makes use of this method.
     </p>
     <br/>
     <h5 class="example">
      Example :
     </h5>
     <div class="panel-group acc-v1 learn-acc zero-bottom-margin" id="accordion1">
      <div class="panel panel-default">
       <div class="panel-heading">
        <h2 class="panel-title">
         <a class="accordion-toggle" data-parent="#accordion1" data-toggle="collapse" href="#collapse1_1">
          Find the derivative of the function by definition:
                            `f(x)  =  x^2`
          <i class="fa arrow-alignment fa-chevron-left">
          </i>
         </a>
        </h2>
       </div>
       <div class="panel-collapse collapse" id="collapse1_1">
        <div class="panel-body">
         <h5 class="margin-left-five">
          <i class="fa fa-check-square">
          </i>
          <span>
           ` 2x`
          </span>
         </h5>
         <h4 class="h4-heading">
          Explanation:
         </h4>
         <div class="funny-boxes funny-boxes-top-sea zero-bottom-margin explanation-padding">
          <p>
           From the basic definition of derivative, we have
          </p>
          <div class="function">
           `f(x)=Limx 0(f(x+ x)  f(x))/x`
           <br/>
           <br/>
           `f(x)=  x^2  ;  f(x+ x)  =   (x + x )^2`
           <br/>
           <br/>
           `f(x)=Limx 0(  ( x + x )^2     x^2)/x`
           <br/>
           <br/>
           =`Limx 0(x^2  + ( x )^2+ 2(x)(x)   x^2)/x`
           <br/>
           <br/>
           =`Limx 0(  ( x )^2+ 2(x)(x))/x=  Limx 0( 2x +x)`
          </div>
          <br/>
          <p>
           Putting the limit `x  0 `into the expression
          </p>
          <br/>
          <p>
           <i>
            `f(x)   =  2x`
           </i>
           <br/>
           <br/>
           that is the required derivative of the given function.
          </p>
         </div>
        </div>
       </div>
      </div>
     </div>
     <br/>
     <h4 class="definition-note">
      Note:
     </h4>
     <p>
      This is the most basic approach for finding the derivative of a function.
      <br/>
      Since the definition of  derivative itself is employed in this method, hence the name differentiation by definition.
     </p>
     <p>
      Next we will learn some easier methods / theorems that are employed effectively to find derivatives of different types of functions.
     </p>
     <hr/>
     <h2 class="font-bold zero-bottom-margin font-size-20 heading-decoration">
      <span>
       The Power Rule
      </span>
     </h2>
     <br/>
     <br/>
     <blockquote>
      <p>
       Differentiation of `x^n` where n is an integer
      </p>
      <br/>
      <div class="function">
       `(d [x^n])/dx=  n. x^(n-1)` where ` n Z`
      </div>
      <br/>
      <p>
       The above identity is called the
       <b>
        Power Rule,
       </b>
       and it can be proved very easily using the ab-initio method of differentiation.
      </p>
      <br/>
     </blockquote>
     <h2 class="learn-example-heading font-bold zero-bottom-margin font-size-20">
      Power Rule Examples
     </h2>
     <br/>
     <h5 class="example">
      Example 1:
     </h5>
     <div class="panel-group acc-v1 learn-acc zero-bottom-margin" id="accordion2">
      <div class="panel panel-default">
       <div class="panel-heading">
        <h2 class="panel-title">
         <a class="accordion-toggle" data-parent="#accordion2" data-toggle="collapse" href="#collapse2_1">
          Find the derivative of function using Power Rule:
                            `f(x)=x^2`
          <i class="fa arrow-alignment fa-chevron-left">
          </i>
         </a>
        </h2>
       </div>
       <div class="panel-collapse collapse" id="collapse2_1">
        <div class="panel-body">
         <h5 class="margin-left-five">
          <i class="fa fa-check-square">
          </i>
          <span>
           ` 2x`
          </span>
         </h5>
         <h4 class="h4-heading">
          Explanation:
         </h4>
         <div class="funny-boxes funny-boxes-top-sea zero-bottom-margin explanation-padding">
          <p>
           (Using the Power Rule)
          </p>
          <div class="function">
           ` (d [x^n])/dx=  n. x^(n  1)`
           <br/>
           <br/>
           ` =&gt;(d [x^2])/dx=2. x^(2  1 )  = 2x`
          </div>
          <br/>
          <p>
           that is the same result as obtained previously by ab-initio method. This shows that the Power Rule is consistent with the definition of derivative.
          </p>
         </div>
        </div>
       </div>
      </div>
     </div>
     <br/>
     <h5 class="example">
      Example 2:
     </h5>
     <div class="panel-group acc-v1 learn-acc zero-bottom-margin" id="accordion3">
      <div class="panel panel-default">
       <div class="panel-heading">
        <h2 class="panel-title">
         <a class="accordion-toggle" data-parent="#accordion3" data-toggle="collapse" href="#collapse3_1">
          Find the derivative of function using Power Rule:
                            `f(x)=1/  x^(4 )`
          <i class="fa arrow-alignment fa-chevron-left">
          </i>
         </a>
        </h2>
       </div>
       <div class="panel-collapse collapse" id="collapse3_1">
        <div class="panel-body">
         <h5 class="margin-left-five">
          <i class="fa fa-check-square">
          </i>
          <span>
           `-4x^( -5)`
          </span>
         </h5>
         <h4 class="h4-heading">
          Explanation:
         </h4>
         <div class="funny-boxes funny-boxes-top-sea zero-bottom-margin explanation-padding">
          <p>
           (According to the Power rule we studied above)
          </p>
          <div class="function">
           `(d [x^n])/dx=  n. x^(n  1)`
           <br/>
           <br/>
           We have
                                `f(x)= x^( -4 )`
           <br/>
           <br/>
           `i.e., ` we have ` n = 4`
                                ` =&gt;f^' (x)=(-4) x^( -4-1)`
           <br/>
           <br/>
           `  =&gt;f^' (x)=-4x^( -5)`
          </div>
          <br/>
          <p>
           that is the required derivative of the given function.
          </p>
         </div>
        </div>
       </div>
      </div>
     </div>
     <br/>
     <h5 class="example">
      Example 3:
     </h5>
     <div class="panel-group acc-v1 learn-acc zero-bottom-margin" id="accordion4">
      <div class="panel panel-default">
       <div class="panel-heading">
        <h2 class="panel-title">
         <a class="accordion-toggle" data-parent="#accordion4" data-toggle="collapse" href="#collapse4_1">
          Find the derivative of the function:
                            `f(x)= 2x^(4/5)`
          <i class="fa arrow-alignment fa-chevron-left">
          </i>
         </a>
        </h2>
       </div>
       <div class="panel-collapse collapse" id="collapse4_1">
        <div class="panel-body">
         <p class="margin-left-five">
          <i class="fa fa-check-square">
          </i>
          <span>
           `8/5x ^(1/5)`
          </span>
         </p>
         <h4 class="h4-heading">
          Explanation:
         </h4>
         <div class="funny-boxes funny-boxes-top-sea zero-bottom-margin explanation-padding">
          <p>
           According to the theorem
          </p>
          <div class="function">
           `(d/dx (kf(x)) = k d/dx f(x)  =  kf^' (x))`
           <br/>
           <br/>
           `f'(x)=d/dx(2x^(4/5)) =2d/dx( x^(4/5)) = 2. 4/5x^(4/5  1) = 8/5x ^(1/5)`
          </div>
         </div>
        </div>
       </div>
      </div>
     </div>
     <br/>
     <h4 class="definition-note">
      Note:
     </h4>
     <p>
      Please remember that the derivative of a constant function is zero. Mathematically it can be written as:
      <b>
       `d/dx(k) = 0`,where  `k Z.`
      </b>
     </p>
     <div class="function">
      Derivative of  y = `kf(x)w.r.t. x`
      <br/>
      <br/>
      `d/dx (kf(x)) = k d/dx f(x)  =  kf^' (x) ` where ` k Z.`
     </div>
     <hr/>
     <h2 class="font-bold zero-bottom-margin font-size-20 heading-decoration">
      <span>
       Derivative of the Sum or the Difference of two or more functions
      </span>
     </h2>
     <br/>
     <blockquote>
      <p>
       The derivative of the Sum / Difference of two functions is given by the Sum / Difference of their derivatives.
      </p>
     </blockquote>
     Mathematically, we can write this as follows:
     <br/>
     <div class="function">
      ` d/dx [u(x) + v(x)]  `= `d/dx [u(x)] + d/dx  [v(x)] `
     </div>
     <br/>
     <p>
      Same is true for the difference of two functions:
     </p>
     <br/>
     <div class="function">
      ` d/dx [u(x) - v(x)]`  = `d/dx [u(x)] -d/dx  [v(x)] `
     </div>
     <br/>
     <br/>
     <h4 class="definition-note">
      Note:
     </h4>
     <p>
      We can extend this rule to  find the derivative of the Sum and Difference of more than two functions.
     </p>
     <p>
      We next present a few examples of these two simple theorems.
     </p>
     <br/>
     <h2 class="learn-example-heading font-bold zero-bottom-margin font-size-20">
      Sum and Difference Theorem Examples
     </h2>
     <br/>
     <h5 class="example">
      Example :
     </h5>
     <div class="panel-group acc-v1 learn-acc zero-bottom-margin" id="accordion5">
      <div class="panel panel-default">
       <div class="panel-heading">
        <h2 class="panel-title">
         <a class="accordion-toggle" data-parent="#accordion5" data-toggle="collapse" href="#collapse5_1">
          Find the derivative of the function using Sum and Difference Theorem:
                            ` f(x)= (x+5)(x+6) `
          <i class="fa arrow-alignment fa-chevron-left">
          </i>
         </a>
        </h2>
       </div>
       <div class="panel-collapse collapse" id="collapse5_1">
        <div class="panel-body">
         <h5 class="margin-left-five">
          <i class="fa fa-check-square">
          </i>
          <span>
           `2x + 11 `
          </span>
         </h5>
         <h4 class="h4-heading">
          Explanation:
         </h4>
         <div class="funny-boxes funny-boxes-top-sea zero-bottom-margin explanation-padding">
          <p>
           We first multiply the two terms in order to express the function ` f(x) ` in terms of the sum of sub-functions as shown below.
          </p>
          <br/>
          <div class="function">
           ` f(x)= x^2 + 11x + 30 `
           <br/>
           <br/>
           ` f'(x)= d/dx[ x^2] +  d/dx[11x ] +  d/dx[30] `
           <br/>
           <br/>
           ` f'(x)=  2x + 11 `
          </div>
         </div>
        </div>
       </div>
      </div>
     </div>
     <hr/>
     <h2 class="font-bold zero-bottom-margin font-size-20 heading-decoration">
      <span>
       Derivative of the Product of Two Functions
      </span>
     </h2>
     <br/>
     <br/>
     <p>
      If two functions ` u(x) ` and ` v(x) ` can be differentiated at X, then their product ` u(x)v(x) ` can also be differentiated at X. And we can use the following rule to determine such a derivative.
     </p>
     <br/>
     <div class="function">
      ` d/dx [u(x) . v(x)] ` =` d/dx [u(x)] .v(x) + d/dx  [v(x)] . u(x) `
     </div>
     <br/>
     <br/>
     <p>
      Note: The rule can be shown to be true by using the ab-initio method.
      <br/>
     </p>
     <br/>
     <h2 class="learn-example-heading font-bold zero-bottom-margin font-size-20">
      Product of Functions Theorem Examples
     </h2>
     <br/>
     <h5 class="example">
      Example :
     </h5>
     <div class="panel-group acc-v1 learn-acc zero-bottom-margin" id="accordion6">
      <div class="panel panel-default">
       <div class="panel-heading">
        <h2 class="panel-title">
         <a class="accordion-toggle" data-parent="#accordion6" data-toggle="collapse" href="#collapse6_1">
          Find the derivative of the function using Product of Functions Theorem of Differentiation :  ` f(x)= (x+5)(x+6) `
          <i class="fa arrow-alignment fa-chevron-left">
          </i>
         </a>
        </h2>
       </div>
       <div class="panel-collapse collapse" id="collapse6_1">
        <div class="panel-body">
         <h5 class="margin-left-five">
          <i class="fa fa-check-square">
          </i>
          <span>
           `2x + 11 `
          </span>
         </h5>
         <h4 class="h4-heading">
          Explanation:
         </h4>
         <div class="funny-boxes funny-boxes-top-sea zero-bottom-margin explanation-padding">
          <p>
           We are given that
          </p>
          <br/>
          <div class="function">
           ` f(x)= (x+5)(x+6) `
          </div>
          <br/>
          <p>
           Using the product of functions formula:
          </p>
          <br/>
          <div class="function">
           ` d/dx [(x+5) . (x+6)]  =d/dx [(x+5)] . (x+6) + d/dx  [(x+6)] . (x+5) `
           <br/>
           <br/>
           ` f'(x)=  (1)(x+6) + (1)(x+5) `
           <br/>
           <br/>
           ` f'(x)=  x+6 + x+5 =  2x + 11 `
          </div>
         </div>
        </div>
       </div>
      </div>
     </div>
     <p>
      <br/>
      <strong>
       Note:
      </strong>
      <br/>
      <i class="fa fa-hand-o-right">
      </i>
      Results obtained in above two examples are same.
     </p>
     <hr/>
     <h2 class="font-bold zero-bottom-margin font-size-20 heading-decoration">
      <span>
       Derivative of the Quotient of Two Functions
      </span>
     </h2>
     <br/>
     <blockquote>
      <p>
       If two functions ` u(x) ` and ` v(x) ` can be differentiated at X, then their quotient ` (u(x))/(v(x)) `
                can also be differentiated at X (provided ` v(x)  0 ` at ` x ` ). We can use the following rule to determine such a derivative.
      </p>
      <br/>
      <div class="function">
       ` d/dx [(u(x))/(v(x))] ` = `(d/dx [u(x) ]  .v(x)- d/dx [v(x) ].u(x))/([v(x)]2) `
      </div>
     </blockquote>
     <h2 class="learn-example-heading font-bold zero-bottom-margin font-size-20">
      Quotient Rule Examples
     </h2>
     <br/>
     <h5 class="example">
      Example :
     </h5>
     <div class="panel-group acc-v1 learn-acc zero-bottom-margin" id="accordion7">
      <div class="panel panel-default">
       <div class="panel-heading">
        <h2 class="panel-title">
         <a class="accordion-toggle" data-parent="#accordion7" data-toggle="collapse" href="#collapse7_1">
          Find the derivative of the function :
                            ` ( 2x^(3 )- 3x^(2  )+ 2)/(  x^(2 )+ 1) `
          <i class="fa arrow-alignment fa-chevron-left">
          </i>
         </a>
        </h2>
       </div>
       <div class="panel-collapse collapse" id="collapse7_1">
        <div class="panel-body">
         <h5 class="margin-left-five">
          <i class="fa fa-check-square">
          </i>
          <span>
           `(   2x^4+ 6x^2- 10x  )/([( x^(2 )+ 1 )]2) `
          </span>
         </h5>
         <h4 class="h4-heading">
          Explanation:
         </h4>
         <div class="funny-boxes funny-boxes-top-sea zero-bottom-margin explanation-padding">
          <p>
           We make use of the
           <b>
            Quotient Rule
           </b>
           to find the derivative of this function.
          </p>
          <br/>
          <div class="function">
           ` d/dx [(u(x))/(v(x))]  = (d/dx [u(x) ]  .v(x)- d/dx [v(x) ].u(x))/([v(x)]2) `
           <br/>
           <br/>
           Let  ` u(x) =  2x^(3 )- 3x^(2  )+ 2 `
           <br/>
           <br/>
           ` v(x) =   x^(2 )+ 1 `
           <br/>
           <br/>
           ` =&gt;d/dx[( 2x^(3 )- 3x^(2  )+ 2)/(  x^(2 )+ 1)]=(d/dx [ 2x^(3 )- 3x^(2  )+ 2]  . (x^(2 )+ 1)- d/dx [( x^(2 )+ 1 )] ( 2x^(3 )- 3x^(2  )+ 2 ))/([( x^(2 )+ 1 )]2) `
           <br/>
           <br/>
           ` =([(3)(2x^2)  3(2x)]  . (x^(2 )+ 1) - [(2x)] ( 2x^(3 )- 3x^(2  )+ 2 ))/([( x^(2 )+ 1 )]2) `
           <br/>
           <br/>
           ` =(  (6x^(2 ) 6x)(x^(2 )+ 1)  (2x)( 2x^(3 ) 3x^(2  )+ 2 ))/([( x^(2 )+ 1 )]2) `
           <br/>
           <br/>
           ` =((6x^4 6x^3+6x^2-6x)   (4x^(4 ) 6x^(3 )+ 4x))/([( x^(2 )+ 1 )]2) `
           <br/>
           <br/>
           ` =&gt;d/dx[( 2x^(3 )- 3x^(2  )+ 2)/(  x^(2 )+ 1)]=(  6x^4- 6x^3+ 6x^2-6x  4x^(4 )+ 6x^(3 )- 4x)/([( x^(2 )+ 1 )]2) `
           <br/>
           <br/>
           ` =&gt;d/dx[( 2x^(3 )- 3x^(2  )+ 2)/(  x^(2 )+ 1)]=(   2x^4+ 6x^2- 10x  )/([( x^(2 )+ 1 )]2) `
           <br/>
          </div>
         </div>
        </div>
       </div>
      </div>
     </div>
    </div>
