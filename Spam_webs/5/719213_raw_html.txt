     <p>
      <b>
       Meta
      </b>
      (from the
      <a href="/wiki/Greek_language" title="Greek language">
       Greek
      </a>
      <a class="mw-redirect" href="/wiki/Preposition" title="Preposition">
       preposition
      </a>
      and
      <a href="/wiki/Prefix" title="Prefix">
       prefix
      </a>
      <i>
       meta-
      </i>
      (-) meaning "after", or "beyond") is a prefix used in
      <a href="/wiki/English_language" title="English language">
       English
      </a>
      to indicate a concept which is an
      <a href="/wiki/Abstraction" title="Abstraction">
       abstraction
      </a>
      from another concept, used to complete or add to the latter.
     </p>

     <h2>
      <span class="mw-headline" id="Original_Greek_meaning">
       Original Greek meaning
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Meta&amp;action=edit&amp;section=1" title="Edit section: Original Greek meaning">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      In
      <a class="mw-redirect" href="/wiki/Greek_(language)" title="Greek (language)">
       Greek
      </a>
      , the prefix
      <i>
       meta-
      </i>
      is generally less esoteric than in
      <a class="mw-redirect" href="/wiki/English_(language)" title="English (language)">
       English
      </a>
      ; Greek
      <i>
       meta-
      </i>
      is equivalent to the
      <a class="mw-redirect" href="/wiki/Latin_language" title="Latin language">
       Latin
      </a>
      words
      <i>
       post-
      </i>
      or
      <i>
       ad-
      </i>
      . The use of the prefix in this sense occurs occasionally in
      <a href="/wiki/English_language" title="English language">
       scientific English
      </a>
      terms derived from
      <a class="mw-redirect" href="/wiki/Greek_(language)" title="Greek (language)">
       Greek
      </a>
      . For example: the term
      <i>
       <a href="/wiki/Metatheria" title="Metatheria">
        Metatheria
       </a>
      </i>
      (the name for the
      <a href="/wiki/Clade" title="Clade">
       clade
      </a>
      of
      <a href="/wiki/Marsupial" title="Marsupial">
       marsupial
      </a>
      <a class="mw-redirect" href="/wiki/Mammals" title="Mammals">
       mammals
      </a>
      ) uses the prefix
      <i>
       meta-
      </i>
      merely in the sense that the
      <i>
       <a href="/wiki/Metatheria" title="Metatheria">
        Metatheria
       </a>
      </i>
      occur on the tree of life adjacent to the
      <i>
       <a href="/wiki/Theria" title="Theria">
        Theria
       </a>
      </i>
      (the
      <a class="mw-redirect" href="/wiki/Placental_mammals" title="Placental mammals">
       placental mammals
      </a>
      ).
     </p>
     <h2>
      <span class="mw-headline" id="About_.28its_own_category.29">
       About (its own category)
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Meta&amp;action=edit&amp;section=2" title="Edit section: About (its own category)">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      In
      <a href="/wiki/Epistemology" title="Epistemology">
       epistemology
      </a>
      , the prefix
      <i>
       meta-
      </i>
      is used to mean
      <i>
       about (its own category)
      </i>
      . For example,
      <a href="/wiki/Metadata" title="Metadata">
       metadata
      </a>
      are data about data (who has produced them, when, what format the data are in and so on). In database
      <a href="/wiki/Metadata" title="Metadata">
       metadata
      </a>
      are also data about data stored in a data dictionary and describes information (data) about database tables such as the table name, table owner, details about columns,  essentially describes the table. Also,
      <a href="/wiki/Metamemory" title="Metamemory">
       metamemory
      </a>
      in psychology means an individual's
      <a href="/wiki/Knowledge" title="Knowledge">
       knowledge
      </a>
      about whether or not they would remember something if they concentrated on recalling it. The modern sense of "an X about X" has given rise to concepts like "
      <a class="mw-redirect" href="/wiki/Meta-cognition" title="Meta-cognition">
       meta-cognition
      </a>
      " (i.e. cognition about cognition), "
      <a href="/wiki/Meta-emotion" title="Meta-emotion">
       meta-emotion
      </a>
      " (i.e. emotion about emotion), "
      <a href="/wiki/Meta-discussion" title="Meta-discussion">
       meta-discussion
      </a>
      " (i.e. discussion about discussion), "
      <a href="/wiki/Meta-joke" title="Meta-joke">
       meta-joke
      </a>
      " (i.e. joke about jokes), and "
      <a href="/wiki/Metaprogramming" title="Metaprogramming">
       metaprogramming
      </a>
      " (i.e. writing programs that manipulate programs).
      <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
       [
       <i>
        <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
         <span title="This claim needs references to reliable sources. (December 2012)">
          citation needed
         </span>
        </a>
       </i>
       ]
      </sup>
     </p>
     <p>
      In a
      <a href="/wiki/Rule-based_system" title="Rule-based system">
       rule-based system
      </a>
      , a
      <a class="extiw" href="//en.wiktionary.org/wiki/metarule" title="wiktionary:metarule">
       metarule
      </a>
      is a rule that governs the application of other rules.
      <sup class="reference" id="cite_ref-1">
       <a href="#cite_note-1">
        <span>
         [
        </span>
        1
        <span>
         ]
        </span>
       </a>
      </sup>
     </p>
     <h2>
      <span class="mw-headline" id="On_higher_level_of_abstraction">
       On higher level of abstraction
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Meta&amp;action=edit&amp;section=3" title="Edit section: On higher level of abstraction">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      Any subject can be said to have a
      <i>
       <a class="mw-redirect" href="/wiki/Meta-theory" title="Meta-theory">
        meta-theory
       </a>
      </i>
      , a theoretical consideration of its properties, such as its
      <a class="extiw" href="//en.wiktionary.org/wiki/foundation" title="wikt:foundation">
       foundations
      </a>
      ,
      <a class="extiw" href="//en.wiktionary.org/wiki/method" title="wikt:method">
       methods
      </a>
      ,
      <a class="extiw" href="//en.wiktionary.org/wiki/form" title="wikt:form">
       form
      </a>
      and
      <a href="/wiki/Utility" title="Utility">
       utility
      </a>
      , on a higher level of abstraction. In linguistics, a grammar is considered as being expressed in a
      <a href="/wiki/Metalanguage" title="Metalanguage">
       metalanguage
      </a>
      , language that operates on a higher level in order to describe properties of the
      <a href="/wiki/Plain_language" title="Plain language">
       plain language
      </a>
      (and not itself).
     </p>
     <p>
      <i>
       Meta
      </i>
      is also gaining currency as an adjective, as well as a prefix, as in the work of Douglas Hofstadter
      <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
       [
       <i>
        <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
         <span title="This claim needs references to reliable sources. (October 2014)">
          citation needed
         </span>
        </a>
       </i>
       ]
      </sup>
      .
     </p>
     <h2>
      <span class="mw-headline" id="Etymology">
       Etymology
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Meta&amp;action=edit&amp;section=4" title="Edit section: Etymology">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      The prefix comes from the
      <a href="/wiki/Greek_language" title="Greek language">
       Greek
      </a>
      <a class="mw-redirect" href="/wiki/Preposition" title="Preposition">
       preposition
      </a>
      and
      <a class="mw-redirect" href="/wiki/Prefix_(linguistics)" title="Prefix (linguistics)">
       prefix
      </a>
      <i>
       meta-
      </i>
      (-), from ,
      <sup class="reference" id="cite_ref-2">
       <a href="#cite_note-2">
        <span>
         [
        </span>
        2
        <span>
         ]
        </span>
       </a>
      </sup>
      which meant "after", "beside", "with", "among" (with respect to the preposition, some of these meanings were distinguished by
      <a class="mw-redirect" href="/wiki/Case_marking" title="Case marking">
       case marking
      </a>
      ). Other meanings include "beyond", "adjacent" and "self", and it is also commonly used in the form - as a prefix in Greek, with variants - before vowels and - "meth-" before
      <a class="mw-redirect" href="/wiki/Aspirated_vowel" title="Aspirated vowel">
       aspirated vowels
      </a>
      .
     </p>
     <p>
      The earliest attested form of the word "meta" is the
      <a href="/wiki/Mycenaean_Greek" title="Mycenaean Greek">
       Mycenaean Greek
      </a>
      <i>
       me-ta
      </i>
      , written in
      <a href="/wiki/Linear_B" title="Linear B">
       Linear B
      </a>
      syllabic script.
      <sup class="reference" id="cite_ref-3">
       <a href="#cite_note-3">
        <span>
         [
        </span>
        3
        <span>
         ]
        </span>
       </a>
      </sup>
      The Greek preposition is
      <a href="/wiki/Cognate" title="Cognate">
       cognate
      </a>
      with the Old English preposition
      <a class="extiw" href="//en.wiktionary.org/wiki/mid#Etymology_1" title="wikt:mid">
       <i>
        mid
       </i>
      </a>
      "with", still found as a prefix in
      <i>
       midwife
      </i>
      . Its use in English is the result of
      <a href="/wiki/Back-formation" title="Back-formation">
       back-formation
      </a>
      from the word "metaphysics". In origin
      <i>
       <a href="/wiki/Metaphysics_(Aristotle)" title="Metaphysics (Aristotle)">
        Metaphysics
       </a>
      </i>
      was just the title of one of the principal works of
      <a href="/wiki/Aristotle" title="Aristotle">
       Aristotle
      </a>
      ; it was so named (by
      <a href="/wiki/Andronicus_of_Rhodes" title="Andronicus of Rhodes">
       Andronicus of Rhodes
      </a>
      ) simply because in the customary ordering of the works of Aristotle it was the book following
      <i>
       <a href="/wiki/Physics_(Aristotle)" title="Physics (Aristotle)">
        Physics
       </a>
      </i>
      ; it thus meant nothing more than "[the book that comes] after [the book entitled]
      <i>
       Physics
      </i>
      ". However, even Latin writers misinterpreted this as entailing that metaphysics constituted "the science of what is beyond the physical".
      <sup class="reference" id="cite_ref-4">
       <a href="#cite_note-4">
        <span>
         [
        </span>
        4
        <span>
         ]
        </span>
       </a>
      </sup>
      Nonetheless, Aristotle's
      <i>
       Metaphysics
      </i>
      enunciates considerations of natures above physical realities, which can be examined through this particular part of philosophy, e.g., the existence of God. The use of the prefix was later extended to other contexts based on the understanding of metaphysics to mean "the science of what is beyond the physical".
     </p>
     <h2>
      <span class="mw-headline" id="Quine_and_Hofstadter">
       Quine and Hofstadter
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Meta&amp;action=edit&amp;section=5" title="Edit section: Quine and Hofstadter">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      The
      <a class="mw-redirect" href="/wiki/OED" title="OED">
       OED
      </a>
      cites uses of the meta- prefix as "beyond, about" (such as meta-economics and meta-philosophy) going back to 1917. However, these formations are directly parallel to the original "metaphysics" and "metaphysical", that is, as a prefix to general nouns (fields of study) or adjectives. Going by the OED citations, it began to be used with specific nouns in connection with mathematical logic sometime before 1929. (In 1920
      <a href="/wiki/David_Hilbert" title="David Hilbert">
       David Hilbert
      </a>
      proposed a research project in what was called "
      <a href="/wiki/Metamathematics" title="Metamathematics">
       metamathematics
      </a>
      .")
     </p>
     <p>
      A notable early citation is
      <a href="/wiki/Willard_Van_Orman_Quine" title="Willard Van Orman Quine">
       Quine
      </a>
      's 1937 use of the word "metatheorem",
      <sup class="reference" id="cite_ref-5">
       <a href="#cite_note-5">
        <span>
         [
        </span>
        5
        <span>
         ]
        </span>
       </a>
      </sup>
      where meta- clearly has the modern meaning of "an X about X". (Note that earlier uses of "meta-economics" and even "metaphysics" do not have this doubled conceptual structure  they are about or beyond X but they do not themselves constitute an X).
     </p>
     <p>
      <a href="/wiki/Douglas_Hofstadter" title="Douglas Hofstadter">
       Douglas Hofstadter
      </a>
      , in his 1979 book
      <i>
       <a href="/wiki/G%C3%B6del,_Escher,_Bach" title="Gdel, Escher, Bach">
        Gdel, Escher, Bach
       </a>
      </i>
      (and in the sequel,
      <i>
       <a href="/wiki/Metamagical_Themas" title="Metamagical Themas">
        Metamagical Themas
       </a>
      </i>
      ), popularized this meaning of the term. The book, which deals extensively with
      <a href="/wiki/Self-reference" title="Self-reference">
       self-reference
      </a>
      and
      <a href="/wiki/Strange_loop" title="Strange loop">
       strange loops
      </a>
      , and touches on Quine and his work, was influential in many computer-related subcultures and is probably largely responsible for the popularity of the prefix, for its use as a solo term, and for the many recent coinages which use it.
      <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
       [
       <i>
        <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
         <span title="This claim needs references to reliable sources. (May 2010)">
          citation needed
         </span>
        </a>
       </i>
       ]
      </sup>
      Hofstadter uses meta as a stand-alone word, both as an adjective and as a directional preposition ("going meta", a term he coins for the old rhetorical trick of taking a debate or analysis to another level of abstraction, as when somebody says "This debate isn't going anywhere"). This book is also probably responsible for the direct association of "meta" with strange loops, as opposed to just abstraction.
      <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
       [
       <i>
        <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
         <span title="This claim needs references to reliable sources. (May 2010)">
          citation needed
         </span>
        </a>
       </i>
       ]
      </sup>
      The sentence "This sentence contains thirty-six letters," and the sentence it is embedded in, are examples of "metasentences" that reference themselves in this way.
     </p>
