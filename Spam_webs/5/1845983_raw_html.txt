         <p>
          After portable devices and portable apps, there is one more similar concept that has the power of making your on the go digital life more exciting and handy.
          <span id="more-5313">
          </span>
         </p>
         <p>
          So, how about having a portable server for managing your
          <strong>
           WordPress blogs
          </strong>
          on the move? Do not worry; I will not tell you to buy such an innovative server device, as there is none out there in the market.
         </p>
         <div style="clear:both; width:100%; margin-bottom:10px;">
          <script type="text/javascript">
           &lt;!--

         <p>
          However; there exists a really creative way of obtaining a portable server. This is the way of making your flash drive act as a portable server. Isnt this interesting? Keep reading, as we will now see how to change a flash drive to a portable server for testing and running Web pages from any Windows machine.
         </p>
         <h3>
          <strong>
           Procedure Starts with XAMPP Lite
          </strong>
         </h3>
         <p>
          It is
          <strong>
           XAMPP Lite
          </strong>
          that allows you to store an entire Web server on your
          <strong>
           USB drive
          </strong>
          .
          <strong>
           XAMPP Lite for Windows
          </strong>
          refers to a preconfigured but yet a lightweight server that embeds the latest versions of
          <strong>
           PHP, MySQL, and Apache
          </strong>
          for running apps and sites.
         </p>
         <h3>
          <strong>
           Proceed with Download and Installation
          </strong>
         </h3>
         <p>
          You need to download its latest version from
          <strong>
           <a href="http://www.apachefriends.org/en/xampp-windows.html#646" target="_blank">
            http://www.apachefriends.org/en/xampp-windows.html#646
           </a>
          </strong>
          . Preferably,
          <strong>
           using the exe form
          </strong>
          is
          <strong>
           easier than the ZIP one
          </strong>
          . Now, just
          <strong>
           run the exe file and browse to the destination folder
          </strong>
          for installing your server.
         </p>
         <p style="text-align: center;">
          <a href="http://www.geekhowtos.com/wp-content/uploads/2011/10/xampp-lite-.jpg">
           <img alt="xampp lite" class="size-full wp-image-5378 aligncenter" height="149" src="http://www.geekhowtos.com/wp-content/uploads/2011/10/xampp-lite-.jpg" style="border-style: initial; border-color: initial;" title="xampp lite" width="450"/>
          </a>
         </p>
         <p>
          This has to be a folder in your
          <strong>
           flash drive
          </strong>
          and that the drive must have a minimum of
          <strong>
           250MB of free space
          </strong>
          . Now, just click
          <strong>
           Install
          </strong>
          and all the files will be stored in the
          <strong>
           xampplite
          </strong>
          folder during the installation.
         </p>
         <div style="clear:both; width:100%; margin-bottom:10px;">
          <script type="text/javascript">
           &lt;!--
google_ad_client = "ca-pub-9763352381900777";

//--&gt;
          </script>
          <script src="http://pagead2.googlesyndication.com/pagead/show_ads.js" type="text/javascript">
          </script>
         </div>
         <p style="text-align: center;">
          <a href="http://www.geekhowtos.com/wp-content/uploads/2011/10/installing-xampp-lite-for-windows.jpg">
           <img alt="installing xampp lite for windows" class="size-full wp-image-5379 aligncenter" height="342" src="http://www.geekhowtos.com/wp-content/uploads/2011/10/installing-xampp-lite-for-windows.jpg" title="installing xampp lite for windows" width="450"/>
          </a>
         </p>
         <p>
          Once this is done, a
          <strong>
           Command Prompt
          </strong>
          window appears for completing the installation. Firstly, it will prompt for adding shortcuts to the desktop and
          <strong>
           Start
          </strong>
          <strong>
           menu
          </strong>
          for which you should
          <strong>
           enter
          </strong>
          <strong>
           n
          </strong>
          , as they are not needed.
         </p>
         <p style="text-align: center;">
          <a href="http://www.geekhowtos.com/wp-content/uploads/2011/10/adding-shortcuts.jpg">
           <img alt="adding shortcuts to desktop" class="size-full wp-image-5380 aligncenter" height="236" src="http://www.geekhowtos.com/wp-content/uploads/2011/10/adding-shortcuts.jpg" title="adding shortcuts" width="450"/>
          </a>
         </p>
         <p>
          Now,
          <strong>
           enter y
          </strong>
          for configuring the folders.Lastly,
          <strong>
           enter y for making
          </strong>
          <strong>
           XAMPP portable
          </strong>
          after which it will
          <strong>
           setup servers
          </strong>
          so that they can execute without the need of any drive letter from any computer.
         </p>
         <p style="text-align: center;">
          <a href="http://www.geekhowtos.com/wp-content/uploads/2011/10/configure-XAMPPs-directories-.jpg">
           <img alt="configure XAMPPs directories" class="size-full wp-image-5381 aligncenter" height="236" src="http://www.geekhowtos.com/wp-content/uploads/2011/10/configure-XAMPPs-directories-.jpg" title="configure XAMPPs directories" width="450"/>
          </a>
         </p>
         <p>
          At the end, just press
          <strong>
           Enter
          </strong>
          after which you will see the command line form of the
          <strong>
           XAMPP server
          </strong>
          wherein you have to confirm your time zone. Thats it! Now, you can just
          <strong>
           enter
          </strong>
          <strong>
           1
          </strong>
          and
          <strong>
           run the control panel
          </strong>
          by using your
          <strong>
           flash drive
          </strong>
          on any other computer.
         </p>
         <p>
          <strong>
           Note
          </strong>
          : You can also
          <strong>
           install your
          </strong>
          <strong>
           portable browser
          </strong>
          before using the server.
         </p>
         <h3>
          <strong>
           Explore Your Portable Server
          </strong>
         </h3>
         <p>
          To use your portable server,
          <strong>
           double-click the
          </strong>
          <strong>
           xampp-control.exe
          </strong>
          in the
          <strong>
           xampplite
          </strong>
          folder located on your flash drive. You will now see the
          <strong>
           Control Panel
          </strong>
          whose
          <strong>
           Modules
          </strong>
          section allows you to
          <strong>
           Start
          </strong>
          your desired server.
         </p>
         <p style="text-align: center;">
          <a href="http://www.geekhowtos.com/wp-content/uploads/2011/10/xampp-control-.jpg">
           <img alt="running portable server" class="size-full wp-image-5382 aligncenter" height="377" src="http://www.geekhowtos.com/wp-content/uploads/2011/10/xampp-control-.jpg" title="xampp control" width="450"/>
          </a>
         </p>
         <p>
          Kindly note that the
          <strong>
           Svc
          </strong>
          box should not be checked, as that will start the
          <strong>
           server as a service for Windows
          </strong>
          . Further, Firewall might give you a prompt, but you must click
          <strong>
           Allow
          </strong>
          to let the server run.
         </p>
         <p style="text-align: center;">
          <a href="http://www.geekhowtos.com/wp-content/uploads/2011/10/allow-server-to-run.jpg">
           <img alt="firewall prompt" class="size-full wp-image-5383 aligncenter" height="328" src="http://www.geekhowtos.com/wp-content/uploads/2011/10/allow-server-to-run.jpg" title="allow server to run" width="450"/>
          </a>
         </p>
         <p>
          Finally, click
          <strong>
           Admin
          </strong>
          to view the default admin page 
          <strong>
           http://localhost/
          </strong>
          and
          <strong>
           click the Status
          </strong>
          link to ensure that everything is running well.
         </p>
         <p style="text-align: center;">
          <a href="http://www.geekhowtos.com/wp-content/uploads/2011/10/xampp-control-panel.jpg">
           <img alt="xampp control panel" class="size-full wp-image-5384 aligncenter" height="328" src="http://www.geekhowtos.com/wp-content/uploads/2011/10/xampp-control-panel.jpg" title="xampp control panel" width="450"/>
          </a>
         </p>
         <h3>
          <strong>
           Add Web Pages To Your Portable Server
          </strong>
         </h3>
         <p>
          You can now
          <strong>
           add your Web pages
          </strong>
          by storing files in
          <strong>
           \xampplight\htdocs\
          </strong>
          <strong>
           folder
          </strong>
          .
         </p>
         <p style="text-align: center;">
          <a href="http://www.geekhowtos.com/wp-content/uploads/2011/10/add-webpages-to-portable-server.jpg">
           <img alt="add webpages to portable server" class="size-full wp-image-5385 aligncenter" height="377" src="http://www.geekhowtos.com/wp-content/uploads/2011/10/add-webpages-to-portable-server.jpg" title="add webpages to portable server" width="450"/>
          </a>
         </p>
         <h3>
          <strong>
           Manage Web Pages With The WordPress CMS
          </strong>
         </h3>
         <p>
          To do so,
          <strong>
           first download WordPress
          </strong>
          from
          <strong>
           <a href="http://wordpress.org/download/ " target="_blank">
            http://wordpress.org/download/
           </a>
          </strong>
          and extract its files to the
          <strong>
           htdocs
          </strong>
          <strong>
           folder
          </strong>
          . Now, you will have to setup WordPress by opening the
          <strong>
           database admin page
          </strong>
          and creating a novel database for WordPress.
         </p>
         <p style="text-align: center;">
          <a href="http://www.geekhowtos.com/wp-content/uploads/2011/10/creating-wordpress-database.jpg">
           <img alt="creating wordpress database" class="size-full wp-image-5387 aligncenter" height="293" src="http://www.geekhowtos.com/wp-content/uploads/2011/10/creating-wordpress-database.jpg" title="creating wordpress database" width="450"/>
          </a>
         </p>
         <p>
          While creating, you need to enter the desired database name, select
          <strong>
           Add a new User
          </strong>
          by clicking the
          <strong>
           Privileges
          </strong>
          <strong>
           tab atop
          </strong>
          , and provide login credentials.
         </p>
         <p style="text-align: center;">
          <a href="http://www.geekhowtos.com/wp-content/uploads/2011/10/add-new-user-.jpg">
           <img alt="add new user" class="size-full wp-image-5386 aligncenter" height="269" src="http://www.geekhowtos.com/wp-content/uploads/2011/10/add-new-user-.jpg" title="add new user" width="450"/>
          </a>
         </p>
         <p>
          Now, go to
          <strong>
           http://localhost/wordpress/wp-admin/install.php
          </strong>
          , click
          <strong>
           Create a Configuration File
          </strong>
          and
          <strong>
           Lets Go!.
          </strong>
         </p>
         <p>
          <a href="http://www.geekhowtos.com/wp-content/uploads/2011/10/create-configuration-file.jpg">
           <img alt="create configuration file" class="size-full wp-image-5388 aligncenter" height="299" src="http://www.geekhowtos.com/wp-content/uploads/2011/10/create-configuration-file.jpg" style="border-style: initial; border-color: initial;" title="create configuration file" width="450"/>
          </a>
         </p>
         <p>
          <strong>
           Provide
          </strong>
          credentials and
          <strong>
           WordPress database name
          </strong>
          only, click
          <strong>
           Submit
          </strong>
          , and click
          <strong>
           Run the install
          </strong>
          for
          <strong>
           installing WordPress
          </strong>
          .
         </p>
         <p style="text-align: center;">
          <a href="http://www.geekhowtos.com/wp-content/uploads/2011/10/run-and-intall-wordpress.jpg">
           <img alt="run and intall wordpress" class="aligncenter size-full wp-image-5390" height="226" src="http://www.geekhowtos.com/wp-content/uploads/2011/10/run-and-intall-wordpress.jpg" title="run and intall wordpress" width="450"/>
          </a>
         </p>
         <p>
          Follow the instructions, log in correctly, and you can now
          <strong>
           add pages, themes, extensions, and posts for your new WordPress site
          </strong>
          . Finally, view your portable site at
          <strong>
           <a href="http://localhost/wordpress" target="_blank">
            http://localhost/wordpress
           </a>
           /
          </strong>
          .
         </p>
         <p style="text-align: center;">
          <a href="http://www.geekhowtos.com/wp-content/uploads/2011/10/xampp-stop-.jpg">
           <img alt="xampp stop" class="size-full wp-image-5391 aligncenter" height="377" src="http://www.geekhowtos.com/wp-content/uploads/2011/10/xampp-stop-.jpg" title="xampp stop" width="450"/>
          </a>
         </p>
         <h3>
          <strong>
           Shut Down The Server
          </strong>
         </h3>
         <p>
          To do so, just run the
          <strong>
           xampp_stop.exe
          </strong>
          from the
          <strong>
           xampplite
          </strong>
          <strong>
           folder.
          </strong>
         </p>
