       <div class="shareaholic-canvas" data-app="share_buttons" data-app-id="15186113" data-link="http://insights.dice.com/2011/09/21/windows-to-go-lets-you-run-windows-8-from-a-flash-drive/" data-summary="" data-title="Windows To Go Lets You Run Windows 8 From A Flash Drive">
       </div>
       <p>
        <a href="http://insights.dice.com/wp-content/uploads/2011/09/win8-mba2.jpg">
         <img alt="Windows 8 running on a MacBook Air" class="alignnone wp-image-24209" src="http://insights.dice.com/wp-content/uploads/2011/09/win8-mba2.jpg" title="Windows 8 running on a MacBook Air"/>
        </a>
       </p>
       <p>
        Yes, youre seeing Windows 8 running on a MacBook Air. And no, its not Boot Camp at work here. Instead, the operating system is running entirely from a high capacity USB flash drive, made possible by
        <a href="http://techie-buzz.com/microsoft/windows-8-usb-to-go-demo.html" target="_blank" title="[Videos] Microsoft Demonstrates Windows To Go (Run Windows 8 From USB On Any PC)">
         Windows to Go
        </a>
        :
       </p>
       <blockquote>
        <p>
         Windows To Go is a new feature in Windows 8 that enables enterprise administrators to create USB drives containing complete, managed Windows images that users can use to boot and run Windows on any Windows 7 or Windows 8 capable computer. Windows To Go makes it possible for employees to use a managed device whether they work from home, a client office or in a free seating environment.
        </p>
       </blockquote>
       <p>
        While its still unclear whether the feature will be available to all users or only enterprise customers, the description above seems to hint at the latter.
       </p>
       <p>
        According to Manan Kakkar, who was present when the feature was
        <a href="http://channel9.msdn.com/Events/BUILD/BUILD2011/HW-245T" target="_blank" title="Running Windows from an external USB drive with Windows To Go">
         demonstrated
        </a>
        at the BUILD conference last Thursday:
       </p>
       <ul>
        <li>
         Windows To Go supports both USB 2.0 and 3.0
        </li>
        <li>
         A Windows 8 machine will automatically boot to a Windows To Go drive if one is detected
        </li>
        <li>
         It wont be able to hibernate
        </li>
        <li>
         Itll work on legacy BIOS and UEFI firmware
        </li>
        <li>
         Crash and dump will be available for Windows To Go instances
        </li>
       </ul>
       <p>
        Also interesting: Windows 8 will resume working from the last point if the USB flash drive is plugged in again within 60 seconds after its removed. Users with kids or obnoxious co-workers should appreciate this.
       </p>
       <p>
        See it in action:
       </p>
