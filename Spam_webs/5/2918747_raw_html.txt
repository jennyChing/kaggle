                      Introduction:
                      <br/>
                      <br/>
                      <br/>
                     </font>
                    </b>
                    <br/>
                    The mission of PIHRO is to assist talented students in Pakistan for developing themselves as productive, advancing and fulfilled in their studies and to increase the quality, status and visibility in their carriers.
                    <br/>
                    <br/>
                    PIHRO Scholarships are awarded to university students, who are enrolled in the university in Undergraduate Program. Initially the project is covering the students of Punjab, Khyber PakhtunKhwa, Gilgit Baltistan and FATA. The scholarship awards support living and related expenses and research activities and initiatives. PIHRO will track down the student progress reports signed by head of the department for further extension in scholarship.
                    <br/>
                    <br/>
                    Eligibility Criteria:
                    <br/>
                    <br/>
                    Applicant / Student must be enrolled in any Undergraduate Program
                    <br/>
                    <br/>
                    Student studying in University/College must be registered with the Higher Education Commission
                    <br/>
                    <br/>
                    Student must have Computerized CNIC/Form B
                    <br/>
                    <br/>
                    Domicile: Punjab / Khyber PakhtunKhwa/ Gilgit Baltistan / FATA
                    <br/>
                    <br/>
                    <br/>
                    <br/>
                    Test Conduction:
                    <br/>
                    <br/>
                    Application form must be submitted to Open Testing Service (OTS), before 20/04/2014. OTS will conduct test for the said project.
                    <br/>
                    <br/>
                    Region Wise Award Division:
                    <br/>
                    <br/>
                    <br/>
                    <br/>
                    Punjab:	40%
                    <br/>
                    <br/>
                    Khyber PakhtunKhwa:	30%
                    <br/>
                    <br/>
                    FATA:	20%
                    <br/>
                    <br/>
                    Gilgit Baltistan:	10%
                    <br/>
                    <br/>
                    Financial Details:
                    <br/>
                    <br/>
                    Final awardees of the undergraduate scholarship program will be entitled for the following benefits.
                    <br/>
                    <br/>
                    <br/>
                    <br/>
                    PIHRO will provide financial assistance to the selected candidates maximum up to Rs 60000/- annually which includes:
                    <br/>
                    <br/>
                    1. Living Allowance: RS 4000 Per Month
                    <br/>
                    <br/>
                    2. Book Allowance: RS 4000 /Once
                    <br/>
                    <br/>
                    3. Traveling Allowance: RS 4000/ Once
                    <br/>
                    <br/>
                    4. Thesis Allowance: RS 4000/ Once
