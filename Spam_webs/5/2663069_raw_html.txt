        <p>
         Which are the
         <strong>
          most popular firefox addons
         </strong>
         and extensions? Here is an attempt to collect the 50 popular and
         <strong>
          best firefox extensions
         </strong>
         which make your browsing, downloading and navigation in Firefox as easy as possible, while harnessing the full power and features of Firefox.
        </p>

        <h3>
         Best Firefox Addons, Extensions, Plugins
        </h3>
        <p>
         <img alt="Firefox Extensions and Addons" src="http://farm4.static.flickr.com/3239/3092381868_db5665a4f1_d.jpg"/>
        </p>
        <p>
         These extensions should work on the latest versions of the Firefox. Use the extensions which
         <strong>
          work best for *your* needs
         </strong>
         . The fastest way to install these extensions from developer sites is to drag them to the url bar. Look at these
         <a href="http://www.quickonlinetips.com/archives/2006/01/top-extensions-to-manage-firefox-extensions/">
          top extensions to manage firefox extensions
         </a>
         if you decide to install lots of them. When you upgrade to a new Firefox version, some extensions might not work till the AddOn owners upgrade them, use the
         <a href="https://addons.mozilla.org/en-US/firefox/addon/6543">
          Nightly testers tools
         </a>
         to remove compatibility issues.
        </p>
        <h3>
         Control Firefox Tabs
        </h3>
        <ul>
         <li>
          <a href="http://pagesperso-orange.fr/marc.boullet/ext/extensions-en.html" rel="nofollow">
           All-in-One Gestures
          </a>
           merges the popular following extensions for management of mouse gestures, scrolling and power navigation. (Mix of Mouse Gestures, Rocker navigation, Tab scroller, History scroller, Link tooltip and Autoscrolling extensions)
         </li>
         <li>
          <a href="http://tmp.garyr.net/" rel="nofollow">
           Tab Mix Plus
          </a>
           More tweaks added to tabs. Ability to select and open muliple links in tabs, open link in a duplicated tab, merge tabs and close tabs from similar domain
         </li>
         <li>
          <a href="http://viamatic.com/firefox/" rel="nofollow">
           Viamatic foXpose
          </a>
           Click on the icon in the status bar to view all the browser windows with a single click.
         </li>
         <li>
          <a href="http://showcase.uworks.net/" rel="nofollow">
           Firefox Showcase
          </a>
           easily locate and select any open browser window in Firefox.
         </li>
         <li>
          <a href="http://code.google.com/p/uctb/" rel="nofollow">
           Undo Closed Tabs
          </a>
           allows you to undo closed tabs via a toolbar and/or tab bar button or the right-click context menu.
         </li>
        </ul>
        <h3>
         Control Website Links
        </h3>
        <ul>
         <li>
          <a href="http://gemal.dk/mozilla/linky.html" rel="nofollow">
           Linky
          </a>
           Lets you open or download all or selected links, image links and even web addresses found in the text in separate or different tabs or windows.
         </li>
         <li>
          <a href="http://yellow5.us/firefox/linkification/" rel="nofollow">
           Linkification
          </a>
           Allows Firefox to view plain-text URLs and e-mail addresses as actual links
         </li>
         <li>
          <a href="http://ietab.mozdev.org/" rel="nofollow">
           IE Tab
          </a>
           can open the current page or a selected link embedding Internet Explorer in tabs of Mozilla/Firefox. Very useful for those IE only pages.
         </li>
         <li>
          <a href="http://www.iosart.com/firefox/firefoxview/" rel="nofollow">
           FirefoxView
          </a>
           Open Firefox with the current page or a selected link displayed in Internet Explorer. Adds View in Firefox menu items to the content and link context menus.
         </li>
         <li>
          <a href="http://tecwizards.de/mozilla/" rel="nofollow">
           Paste and Go
          </a>
           lets you paste an URL from the clipboard into the address bar and load it as a single step, either via the adress bars context menu or by pressing Ctrl-Shift-V
         </li>
         <li>
          <a href="http://www.hyperwords.net/" rel="nofollow">
           Hyperwords
          </a>
           Interact with all the words on the web, not just links!
         </li>
        </ul>
        <h3>
         Control File Downloads
        </h3>
        <ul>
         <li>
          <a href="http://flashgot.net/" rel="nofollow">
           FlashGot
          </a>
           handles single and massive downloads with several external Download Managers.
         </li>
         <li>
          <a href="http://www.pdfdownload.org/" rel="nofollow">
           PDF Download
          </a>
           Every time you click on a link, checks if the target is a pdf file and in this case let you choose what you want to do (open pdf file inside a new tab, download it to the filesystem or view it as HTML).
         </li>
         <li>
          ScrapBook  helps you to save Web pages and easily manage collections.
         </li>
         <li>
          <a href="http://downthemall.mozdev.org/" rel="nofollow">
           DownThemAll!
          </a>
           adds new advanced downloading capabilities to your browser. It lets you download in just one click all the links or images contained in a webpage or refine your preferences using fully customizable filters.
         </li>
         <li>
          <a href="http://www.downloadhelper.net/" rel="nofollow">
           Video DownloadHelper
          </a>
           The easy way to download Web videos from hundreds of YouTube-like sites, audio and picture galleries.
         </li>
         <li>
          <a href="http://www.bolinfest.com/targetalert/" rel="nofollow">
           TargetAlert
          </a>
           provides visual cues for the destinations of hyperlinks in cases of pdf, doc, zip files etc.
         </li>
         <li>
          <a href="http://dmextension.mozdev.org/" rel="nofollow">
           Download Manager Tweak
          </a>
           modifies the default appearance of the firefox download manager and allows it to be opened in a separate window, the sidebar, or a tab.
         </li>
         <li>
          <a href="http://downloadstatusbar.mozdev.org/" rel="nofollow">
           Download Statusbar
          </a>
           is a browser extension that allows you to keep track of ongoing and completed downloads in a hide-away statusbar
         </li>
         <li>
          <a href="http://fireftp.mozdev.org/" rel="nofollow">
           FireFTP
          </a>
           is a free, secure, cross-platform FTP client for Mozilla Firefox which provides easy and intuitive access to FTP servers.
         </li>
         <li>
          <a href="http://www.openitonline.com/" rel="nofollow">
           Open IT Online
          </a>
           allows you to open your documents and images online without the need of any software to be installed.
         </li>
        </ul>
        <h3>
         Control Webpage Behaviour and Navigation
        </h3>
        <ul>
         <li>
          <a href="http://greasemonkey.mozdev.org/" rel="nofollow">
           GreaseMonkey
          </a>
           Install user scripts and change the behavior of any web page
         </li>
         <li>
          <a href="http://imagezoom.yellowgorilla.net/" rel="nofollow">
           Image Zoom
          </a>
           Right click on an image and select a zoom option from the popup menu, or, hold down the right mouse button in combination with the mouse wheel to zoom in or out on an image.
         </li>
         <li>
          <a href="http://fasterfox.mozdev.org/" rel="nofollow">
           Fasterfox
          </a>
           Speeds up Firefox by recycling idle bandwidth by silently prefetching and caching all of the links on the page you are browsing. (Some readers said it actually slowed down their firefox  verify yourself)
         </li>
         <li>
          <a href="http://spellbound.sourceforge.net/" rel="nofollow">
           SpellBound
          </a>
           enables spell checking in web forms such as html textarea / input elements (html input password elements are not checked by SpellBound) and rich text form elements. This allows you to spell check forms before submitting them.
         </li>
         <li>
          Taboo  lets you save a page for later (taking a screenshot, and using the Session Saver code to remember scroll location and form fields).
         </li>
         <li>
          <a href="http://autocopy.mozdev.org/" rel="nofollow">
           AutoCopy
          </a>
           Select text on any web page and it will be automatically copied to the clipboard. Middle click to Paste.
         </li>
         <li>
          <a href="http://adblockplus.org/en/" rel="nofollow">
           Adblock Plus
          </a>
           is an enhanced version of Adblock. Block ads, applets, flash, embedded-media etc.
         </li>
         <li>
          <a href="http://flashblock.mozdev.org/" rel="nofollow">
           Flashblock
          </a>
           blocks all Flash content from loading on a webpage.
         </li>
         <li>
          <a href="http://sage.blat.co.za/" rel="nofollow">
           Sage-Too
          </a>
           add a lightweight RSS and Atom feed aggregator which integrates with Firefoxs bookmark storage and Live Bookmarks.
         </li>
         <li>
          <a href="http://pearlcrescent.com/products/pagesaver/" rel="nofollow">
           Pearl Crescent Page Saver
          </a>
           capture screenshots and save full webpages as images easily.
         </li>
         <li>
          <a href="http://reloadevery.mozdev.org/" rel="nofollow">
           Reload Every
          </a>
           adds an option to the context menu to reload the web page you are viewing every so many seconds or minutes. Useful if you keep refreshing some pages often.
         </li>
         <li>
          <a href="http://informenter.mozdev.org/" rel="nofollow">
           InFormEnter
          </a>
           adds a small, clickable icon next to every input field in a web form, from where you can select the item to be inserted with your frequently used information such as name, email, address and whatever else you want to be available from the form menu.
         </li>
         <li>
          <a href="http://firefox.exxile.net/aios/index.php" rel="nofollow">
           All-in-One Sidebar
          </a>
           is a sidebar control, inspired by Opera that lets you quickly switch between sidebars, view dialog windows such as downloads, extensions, and more in the sidebar, or view source code or websites in the sidebar. Can be extensively customized.
         </li>
         <li>
          <a href="http://www.splintered.co.uk/experiments/70/" rel="nofollow">
           Text size toolbar
          </a>
           Adds buttons to increase or decrease text size or restore default size easily. Useful for those small unreadable font sizes.
         </li>
         <li>
          <a href="http://aluminum.sourmilk.net/reveal/" rel="nofollow">
           Reveal
          </a>
           allows you to see thumbnails of pages in your session history and quickly find the page you want.
         </li>
         <li>
          <a href="http://www.mystickies.com/" rel="nofollow">
           Mystickies
          </a>
           allows you to place sticky notes all over the web and organize them with tags.
         </li>
         <li>
          Xinha Here!  is a wrapper for the Xinha HTML editor that enables WYSIWYG editing in any textarea and text box on any website.
         </li>
         <li>
          <a href="http://www.yoono.com/" rel="nofollow">
           Yoono
          </a>
           instantly suggests alternate sites and people who share the same interests while you are surfing.
         </li>
         <li>
          <a href="http://www.adaptiveblue.com/" rel="nofollow">
           BlueOrganizer
          </a>
           It helps you personalize your web experience based on what you already like, helping you discover relevant new information and save time.
         </li>
         <li>
          <a href="http://www.piclens.com/" rel="nofollow">
           PicLens
          </a>
           transforms your browser into a visually stunning experience for finding and viewing online photos and videos.
         </li>
         <li>
          <a href="http://www.chrisfinke.com/addons/feedbar/" rel="nofollow">
           Feed Sidebar
          </a>
           displays the items from your Live Bookmarks in the sidebar.
         </li>
        </ul>
        <h3>
         Control Privacy / Security
        </h3>
        <ul>
         <li>
          <a href="http://foxyproxy.mozdev.org/" rel="nofollow">
           FoxyProxy
          </a>
          is an advanced proxy management tool that completely replaces Firefoxs limited proxying capabilities.
         </li>
         <li>
          <a href="http://noscript.net/" rel="nofollow">
           NoScript
          </a>
           allows JavaScript, Java (and other plugins) only for trusted domains of your choice. This whitelist based pre-emptive blocking approach prevents exploitation of security vulnerabilities with no loss of functionality
         </li>
         <li>
          <a href="http://cookieculler.mozdev.org" rel="nofollow">
           CookieCuller
          </a>
           Extended Cookie Manager to protect/unprotect selected cookies.
         </li>
         <li>
          <a href="http://passwordexporter.fligtar.com/" rel="nofollow">
           Password Exporter
          </a>
           allows you to export your saved passwords and disabled login hosts using XML or CSV files that can be imported in another browser or computer.
         </li>
        </ul>
        <h3>
         Control Web Searches
        </h3>
        <ul>
         <li>
          <a href="http://toolbar.google.com/firefox/index.html" rel="nofollow">
           Google Toolbar for Firefox
          </a>
           Lets you search google and all its services easily. Also powered by Google Suggest (Get query suggestions as you type in the search box), SpellCheck, AutoFill, Pagerank of webpage, access to gmail, WordTranslator etc.
         </li>
         <li>
          <a href="http://www.customizegoogle.com/" rel="nofollow">
           CustomizeGoogle
          </a>
           enhances Google search results by adding extra information (like links to Yahoo, Ask Jeeves, MSN etc) and removing unwanted information (like ads and spam).
         </li>
         <li>
          NextPlease!  allows you to assign keyboard shortcuts to jump to next and previous links on search results pages, like Google, Yahoo, Ebay, Amazon, and many other sites.
         </li>
         <li>
          <a href="https://addons.mozilla.org/en-US/firefox/addon/211">
           BetterSearch
          </a>
           enhances Google, MSN Search, Yahoo Search, A9, Answers.com, AllTheWeb, Dogpile.com, del.icio.us and Simpy.com bookmarks by adding previews (thumbnails) and Amazon product images and info etc.
         </li>
        </ul>
        <h3>
         Control Your Music / Video
        </h3>
        <ul>
         <li>
          <a href="http://www.foxytunes.com/firefox/" rel="nofollow">
           FoxyTunes
          </a>
           Listen to music while surfing the Web, and control your favorite media player without ever leaving the browser.
         </li>
         <li>
          <a href="http://membres.lycos.fr/sethnakht/" rel="nofollow">
           MediaPlayerConnectivity
          </a>
           Allow you to launch embed video of website in an external application with a simple click
         </li>
        </ul>
        <h3>
         Control Your Email
        </h3>
        <ul>
         <li>
          <a href="http://lifehacker.com/software/exclusive-lifehacker-download/better-gmail-2-firefox-extension-for-new-gmail-320618.php" rel="nofollow">
           Better Gmail 2
          </a>
           a compilation of Greasemonkey user scripts that adds advanced features to new version of Gmail.
         </li>
         <li>
          <a href="http://www.longfocus.com/firefox/gmanager/" rel="nofollow">
           Gmail Manager
          </a>
           Allows you to manage multiple Gmail accounts and receive new mail notifications. Displays your account details including unread messages, saved drafts, spam messages, labels with new mail, space used, and new mail snippets.
         </li>
         <li>
          <a href="http://webmailnotifier.mozdev.org/" rel="nofollow">
           WebMail Notifier
          </a>
           checks your webmail accounts and notifies the number of unread emails. Supports gmail, yahoo, hotmail etc.
         </li>
        </ul>
        <h3>
         Control Your Bookmarks
        </h3>
        <ul>
         <li>
          <a href="http://www.foxmarks.com/" rel="nofollow">
           Foxmarks Bookmark Synchronizer
          </a>
           Install Foxmarks on each computer, and it will work silently in the background to keep your bookmarks synchronized
         </li>
         <li>
          <a href="https://addons.mozilla.org/en-US/firefox/addon/138">
           StumbleUpon
          </a>
           is an intelligent browsing tool for sharing and discovering great websites.
         </li>
         <li>
          <a href="http://gemal.dk/mozilla/reliby.html" rel="nofollow">
           Reliby
          </a>
           provides you with Reload all Live Bookmarks functionality.
         </li>
         <li>
          <a href="http://shareaholic.com/" rel="nofollow">
           Shareaholic
          </a>
           Share, bookmark and e-mail web pages quickly without leaving your browser.
         </li>
        </ul>
        <h3>
         Control Other Popular Services
        </h3>
        <ul>
         <li>
          <a href="http://dictionarysearch.mozdev.org/" rel="nofollow">
           DictionarySearch
          </a>
