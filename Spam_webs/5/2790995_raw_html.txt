             </a>
            </h3>
            <h4 class="date-header">
             Wednesday, August 6, 2008
            </h4>
            <div class="legal">
            </div>
           </div>
           <div class="post-body entry-content">
            So my adorable hubby, who pays far more attention to this stuff than I do, tells me that yesterday nearly 5,000 of you were referred to Cake Wrecks from a
            <a href="http://fuglyhorseoftheday.blogspot.com/2008/08/nhr-but-really-isnt-everything-somewhat.html">
             blog about horses
            </a>
            . Which, you know, is a little odd. But hey (or should I say, "hay"?), I'm not complaining. Welcome, horse lovers!! Well, I mean, horse-
            <span style="font-style: italic;">
             riding
            </span>
            lovers, nothing sick or anything... Wait, horse-riding doesn't sound any better...Geez, uh...
            <br/>
            <br/>
            Here's a cake for ya'll, anyway:
            <br/>
            <br/>
            <a href="http://2.bp.blogspot.com/_wGr8njEWjtI/SJob2ryf-LI/AAAAAAAAANs/81Vd0UiA4SI/s1600-h/Shara+S.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">

            </a>
            <br/>
            In case you're wondering, it's a horse butt. Because, you know, all the
            <a href="http://cakewrecks.blogspot.com/2008/05/most-disturbing-cake-ever.html">
             cake baby butts
            </a>
            were getting SO last week.
            <br/>
            <br/>
            I am kind of curious about the "Good luck in life with Lacy &amp; Tina" thing, though. The most charitable route would be to assume Lacy &amp; Tina are themselves horses. If not, whoever commissioned this cake has a whole lotta 'splainin to do.

              <img alt="" border="0" id="BLOGGER_PHOTO_ID_5267115131177781186" src="http://4.bp.blogspot.com/_wGr8njEWjtI/SRiNTjDvx8I/AAAAAAAAA7w/FED91rh1DAI/s320/Debbie+M+2.jpg" style="margin: 0px auto 10px; display: block; text-align: center; cursor: pointer; width: 320px; height: 198px;"/>
             </a>
            </p>
            <p>
             Sometimes stupidity takes so much more effort.
            </p>
            <p>
             <br/>
            </p>
            <p>
             <a href="http://3.bp.blogspot.com/_wGr8njEWjtI/SRiNfPdcShI/AAAAAAAAA74/18MVlEY27jk/s1600-h/Emily+G.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
              <img alt="" border="0" id="BLOGGER_PHOTO_ID_5267115332075276818" src="http://3.bp.blogspot.com/_wGr8njEWjtI/SRiNfPdcShI/AAAAAAAAA74/18MVlEY27jk/s320/Emily+G.jpg" style="margin: 0px auto 10px; display: block; text-align: center; cursor: pointer; width: 320px; height: 240px;"/>
             </a>
            </p>
            <p>
             The instructions were "Write 'Happy Birthday Calvin' in green," as in green icing. I'm surprised the decorator didn't write it like this: "Gre - Happy Birthday Calvin - En".
            </p>
            <p>
             <br/>

             </a>
            </h3>
            <h4 class="date-header">
            </h4>
            <div class="legal">
            </div>
           </div>
           <div class="post-body entry-content">
            Ah, how sweet it is to be back posting here on Blogger, dear readers.
            <br/>
            <br/>
            As you probably know from the sidebar note I had posted, Blogger's anti-spam measures disabled my account last Friday afternoon, along with countless others. For the past five days I have been pleading on the Blogger Help Group boards for a live person to a) unlock my account, or failing that, to b) acknowledge my existence. I tried being polite. Then I tried being a little snippy. When those both failed, I resorted to the time-proven tactic of shameless begging.
            <br/>
            <br/>
            My post this morning read as follows:
            <br/>
            <br/>
            "
            <span id="thread_subject_site" style="font-style: italic;">
             Please
            </span>
            <span style="font-style: italic;">
             please please please please please please please please please please please please
            </span>
            <span style="font-style: italic;">
             please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please please  please please please please please please please please please please please pretty pretty pretty pretty please?!?"
             <br/>
             <br/>
            </span>
            About four hours later:
            <span style="font-style: italic;">
            </span>
            bingo! Cake Wrecks is unlocked!
            <span style="font-style: italic;">
             <br/>
             <br/>
            </span>
            See, it really
            <span style="font-style: italic;">
             does
            </span>
            work sometimes, guys. ;)
            <span style="font-style: italic;">
             <br/>
             <br/>
            </span>
            Thanks to all of you who checked in on me during my virtual absence: I'd take you guys over any other blog's readers, any day. Ya'll rock.
            <span style="font-style: italic;">
             <br/>
             <br/>
            </span>
            Now, back to the Wreckiness!

          <h2 class="date-header">
           Monday, August 4, 2008
          </h2>
          <div class="post hentry uncustomized-post-template">
           <a name="4035291142377816550">
           </a>
           <div class="post-header">
            <h3 class="post-title entry-title">
             <a href="http://cakewrecks.blogspot.com/2008/08/cake-wreckin-under-achievers-unite.html">
              Cake Wreckin' Under-Achievers, Unite!
             </a>
            </h3>
            <h4 class="date-header">
             Monday, August 4, 2008
            </h4>
            <div class="legal">
            </div>
           </div>
           <div class="post-body entry-content">
            You know how it is for some of these grocery-store bakery decorators: some days they're just not reaching their full cake-wrecking potential. On those days they give their
            <a href="http://cakewrecks.blogspot.com/2008/07/i-think-i-just-lost-my-appetite.html">
             airbrush
            </a>
            ,
            <a href="http://cakewrecks.blogspot.com/2008/05/this-made-me-laugh-til-i-cried.html">
             questionable design choices
            </a>
            , and
            <a href="http://cakewrecks.blogspot.com/2008/05/poor-bunjamioh.html">
             horrific color palettes
            </a>
            a rest, and instead produce a simpler, quieter kind of wreck. A wreck that says, with world-weary disdain, "Hey, I make $7 an hour. Deal with it."
            <br/>
            <br/>
            <br/>
            <br/>
            <a href="http://bp2.blogger.com/_wGr8njEWjtI/SJdZGrfvr_I/AAAAAAAAAMs/d8gc2GoBQVo/s1600-h/Lindsay+W+%26+Jessica+B.JPG" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
             <img alt="" border="0" id="BLOGGER_PHOTO_ID_5230747463504080882" src="http://bp2.blogger.com/_wGr8njEWjtI/SJdZGrfvr_I/AAAAAAAAAMs/d8gc2GoBQVo/s400/Lindsay+W+%26+Jessica+B.JPG" style="margin: 0px auto 10px; display: block; text-align: center; cursor: pointer;"/>
            </a>
            <br/>
            I guess the cracked icing counts as decoration.
            <br/>
            <br/>
            <br/>
            <br/>
            <br/>
            <a href="http://bp0.blogger.com/_wGr8njEWjtI/SJdZbJKPVCI/AAAAAAAAAM0/6ICOrQWNBE4/s1600-h/Phoebe+H.+cake.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
             <img alt="" border="0" id="BLOGGER_PHOTO_ID_5230747815064327202" src="http://bp0.blogger.com/_wGr8njEWjtI/SJdZbJKPVCI/AAAAAAAAAM0/6ICOrQWNBE4/s400/Phoebe+H.+cake.jpg" style="margin: 0px auto 10px; display: block; text-align: center; cursor: pointer;"/>
            </a>
            While I'm sure Tim appreciated the baby-pink roses for his big 4-0, I get the feeling that someone really enjoyed skewering the cake with those candles, dartboard style.
            <br/>
            <br/>
            <br/>
            <br/>
            <br/>
            <a href="http://bp1.blogger.com/_wGr8njEWjtI/SJdahC50vRI/AAAAAAAAANE/9gpnFMAaYsY/s1600-h/Fed+ex.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
             <img alt="" border="0" id="BLOGGER_PHOTO_ID_5230749015975705874" src="http://bp1.blogger.com/_wGr8njEWjtI/SJdahC50vRI/AAAAAAAAANE/9gpnFMAaYsY/s400/Fed+ex.jpg" style="margin: 0px auto 10px; display: block; text-align: center; cursor: pointer;"/>
            </a>
            <br/>
            This cake is Cake Wrecks approved! Way to knock yourselves out, there, decorators!
            <br/>
            (They get bonus points for the random capitalization.)
            <br/>
            <br/>

          <h2 class="date-header">
           Thursday, July 31, 2008
          </h2>
          <div class="post hentry uncustomized-post-template">
           <a name="1207346364237296707">
           </a>
           <div class="post-header">
            <h3 class="post-title entry-title">
             <a href="http://cakewrecks.blogspot.com/2008/07/ow.html">
              Ow.
             </a>
            </h3>
            <h4 class="date-header">
             Thursday, July 31, 2008
            </h4>
            <div class="legal">
            </div>
           </div>
           <div class="post-body entry-content">
            <a href="http://bp1.blogger.com/_wGr8njEWjtI/SJIa66ESP8I/AAAAAAAAAMk/sxFxAomAIeU/s1600-h/Jen+E.JPG" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
             <img alt="" border="0" id="BLOGGER_PHOTO_ID_5229271716652531650" src="http://bp1.blogger.com/_wGr8njEWjtI/SJIa66ESP8I/AAAAAAAAAMk/sxFxAomAIeU/s400/Jen+E.JPG" style="margin: 0px auto 10px; display: block; text-align: center; cursor: pointer;"/>
            </a>
            I'm pretty sure I just burst a blood vessel in my right eye, looking at this.
            <br/>
            <br/>
            [squinting] From what I can see, this appears to be a pimp-mobile on a suicide run through a radioactive river of blood - only it's a radioactive river of blood with jaunty blue outlining.
            <br/>
            <br/>
            I love the extra cupcake stuck on the side, too: what's that supposed to be? A pit stop?
            <br/>
            <br/>
            If you want to give yourself a migraine, try reading what it says in front of the car. Don't see any writing? Look closer. No, closer. Let's see...I think it says...
            <br/>
            <br/>
            Dang it!
            <br/>
            <br/>
            There went the other eye.
            <br/>
            <br/>
            <br/>
            <br/>
            Jen E., I think you owe me a bottle of Visine.

            <div class="legal">
            </div>
           </div>
           <div class="post-body entry-content">
            <a href="http://bp3.blogger.com/_wGr8njEWjtI/SJDJL432EUI/AAAAAAAAAMc/A_vIeSZFrYU/s1600-h/Drew9Bday002.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">

Now, Drew, remember when your father and I asked you what you wanted for your birthday, and you said you wanted a second Xbox 360 for your game room, and we said no? You do?           <h2 class="date-header">
           Tuesday, July 29, 2008
          </h2>
          <div class="post hentry uncustomized-post-template">
           <a name="4429358212180534521">
           </a>
           <div class="post-header">
            <h3 class="post-title entry-title">
             <a href="http://cakewrecks.blogspot.com/2008/07/get-me-holly-hobbies-head-on-platter.html">
              Get Me Holly Hobbie's Head on a Platter!
             </a>
            </h3>
            <h4 class="date-header">
             Tuesday, July 29, 2008
            </h4>
            <div class="legal">
            </div>
           </div>
           <div class="post-body entry-content">
            <a href="http://bp3.blogger.com/_wGr8njEWjtI/SI9Wm6OXjxI/AAAAAAAAAMM/f7KIeqs8VOs/s1600-h/Baby+Shower+Cake.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
             <img alt="" border="0" id="BLOGGER_PHOTO_ID_5228492918864973586" src="http://bp3.blogger.com/_wGr8njEWjtI/SI9Wm6OXjxI/AAAAAAAAAMM/f7KIeqs8VOs/s400/Baby+Shower+Cake.jpg" style="margin: 0px auto 10px; display: block; text-align: center; cursor: pointer;"/>
            </a>
            <br/>
            I guess its the camera angle, but every time I look at this cake I see a baby head wearing a giant bonnet. And yes, a body-less baby would be a little disquieting, but I think you could make the case that a Baby Burrito is just as bad. What's wrong with a Baby Burrito, you ask? Several things. Allow me to list them for you:
            <br/>
            <br/>
            1) Ok, first off, check out the baby's expression. It's no easy feat to achieve a look that's both dead-eyed and horrified at the same time, but this baker just has those kind of skills.
            <br/>
            2) Next, try to imagine the position the baby's body would have to be in to have his (it's supposed to be a boy - more on that in a minute) chin lying flat on the table and his body stretched out behind. Who needs that pesky spinal cord anyway, am I right?
            <br/>
            3) Baby is packing some serious junk in the trunk: check out that badonk-a-donk rump*!
            <br/>
            4) There are no arms. Which, come to think of it, might explain reason #1.
            <br/>
            <br/>
            <br/>
            Part of why I love all your e-mail submissions, folks, is that I have the option of grilling you for more details. In this case, I had to ask for a little more explanation regarding the, er, display.  Wrecks reader Katie explained that the blue &amp; white blanket was added to, and I quote, "boy it up", since the general consensus was that the cake was too feminine. And that black thing? That's a duck whistle. Yeah. It was also added to be, and again I quote "another signifier of the male gender".  Yeah - a duck whistle.
            <br/>
            <br/>
            Katie C, your family rocks.
            <br/>
            <br/>
            <br/>
            <br/>
            *My sincere apologies to all R&amp;B artists, individuals under 25, and non-honky people in general for attempting to use "hip" slang. I promise it won't happen again.

          <h2 class="date-header">
           Saturday, July 26, 2008
          </h2>
          <div class="post hentry uncustomized-post-template">
           <a name="3994550756767472045">
           </a>
           <div class="post-header">
            <h3 class="post-title entry-title">
             <a href="http://cakewrecks.blogspot.com/2008/07/bringing-whole-new-meaning-to-foot-in.html">
              Bringing a Whole New Meaning to "Foot in Mouth"
             </a>
            </h3>
            <h4 class="date-header">
             Saturday, July 26, 2008
            </h4>
            <div class="legal">
            </div>
           </div>
           <div class="post-body entry-content">
            <a href="http://bp3.blogger.com/_wGr8njEWjtI/SItc6HVC5II/AAAAAAAAALk/xfEJ7-9YgEY/s1600-h/foot+cake.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
             <img alt="" border="0" id="BLOGGER_PHOTO_ID_5227373945963078786" src="http://bp3.blogger.com/_wGr8njEWjtI/SItc6HVC5II/AAAAAAAAALk/xfEJ7-9YgEY/s400/foot+cake.jpg" style="margin: 0px auto 10px; display: block; text-align: center; cursor: pointer;"/>
            </a>
            <br/>
            Yes, it's supposed to be a foot: a "Chinese Lotus (Bound) Foot" to be precise. (It's what the Chinese used to do to their women's feet, back when the thought of fallen arches struck terror in the hearts of mere mortals.)
            <br/>
            <br/>
            Yes, it was for a podiatrist. Because when a doctor is forced to look at and/or handle other people's feet all day every day, I'm sure said doctor can think of nothing more appealing than eating a giant version of one of the most hideous foot deformities possible.  ("Alright! Please, tell me you got Pistachio
            <span class="blsp-spelling-error" id="SPELLING_ERROR_0">
             ice cream
            </span>
            to go with!?!")
            <br/>
            <br/>
            This is one instance where I actually pity the baker responsible; to make this, she had to reference an
            <span style="font-style: italic;">
             actual photo
            </span>
            . (All together now!
            <span style="font-style: italic;">
             <span class="blsp-spelling-error" id="SPELLING_ERROR_1">
              Ewwwwww
             </span>
            </span>
            .)
            <a href="http://flickr.com/photos/52702311@N00/113803423">
             Click here
            </a>
            to see her
            <span class="blsp-spelling-error" id="SPELLING_ERROR_2">
             Flickr
            </span>
            account, which has a link to the photo.
            <br/>
            <br/>
            B.F., I think the toenails have got to be the worst part.

         <h2 class="title">
          What's a Wreck?
         </h2>
         <div class="widget-content">
          <h5 class="highlight">
           What's a Wreck?
          </h5>
          <div class="box">
           <p>
            A Cake Wreck is any cake that is unintentionally sad, silly, creepy, inappropriate - you name it. A Wreck is not necessarily a poorly-made cake; it's simply one I find funny, for any of a number of reasons. Anyone who has ever smeared frosting on a baked good has made a Wreck at one time or another, so I'm not here to vilify decorators: Cake Wrecks is just about finding the funny in unexpected, sugar-filled places.
           </p>
           <p>
            Now, don't you have a photo you want to
            <a href="http://cakewrecks.blogspot.com/p/contact.html">
             send me
            </a>
            ? ;)
           </p>
           <div align="right">
            - Jen
