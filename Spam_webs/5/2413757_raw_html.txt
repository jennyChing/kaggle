   <h1 class="title">
    Pythonista
   </h1>
   <div id="banderole">
    <div id="hero">
    </div>
    Bring the Zen of Python to iOS.
   </div>
   <div id="main-text">
    <p>
     Pythonista is an integrated development environment for writing Python scripts on iOS. You can create interactive experiments and prototypes using multi-touch, animations, and sound  or just use the interactive prompt as a powerful calculator.
    </p>
    <a href="http://itunes.apple.com/us/app/pythonista/id528579881?ls=1&amp;mt=8">
     <div class="appstore-box">
      <!--&lt;img src="AppStoreBadge.png" style="float:left; margin-right: 20px; "/&gt;-->
      <div class="appstore-badge">
      </div>
      <div class="appstore-text">
       <span style="font-weight: 400;">
        $6.99
       </span>
        Universal App for iPad &amp; iPhone.
      </div>
     </div>
     <p>
      With extensive support for URL schemes, access to the iOS clipboard and photo library, and all the powerful libraries that come with Python, it is also possible to use Pythonista as a flexible automation tool for text or image processing.
     </p>
     <p>
      Pythonista is also an excellent companion for learning the Python language  You can get started easily with lots of ready-to-run examples, the interactive prompt helps you experiment, and you can read the entire documentation right within the app.
     </p>
     <p>
      The code editor has everything you'd expect: Syntax highlighting, code completion and an extended keyboard specifically designed for writing Python. You can even extend the editor with your own scripts.
     </p>
    </a>
   </div>
   <h2 class="sub-header">
    Features
   </h2>
   <div class="grid-container">
    <div class="row">
     <div class="column six">
      <div class="thumbnail">
       <a class="screenshot-retina" href="editor.png">
        <img height="100%" src="editor_thumb.png" width="100%"/>
       </a>
      </div>
      <strong>
       Full-featured Code Editor
      </strong>
      <br/>
      The code editor has syntax highlighting, basic code completion, and an extended keyboard with gesture controls.  And like any good programming editor, it is scriptable itself: Use the built-in
      <a href="http://omz-software.com/pythonista/docs/ios/editor.html">
       editor
      </a>
      module to extend it with your own commands. We've included some useful examples in the documentation to help you get started.
     </div>
     <div class="column six">
      <div class="thumbnail">
       <a class="screenshot-retina" href="examples.png">
        <img height="100%" src="examples_thumb.png" width="100%"/>
       </a>
      </div>
      <strong>
       Modules for Graphics and Touch
      </strong>
      <br/>
      Pythonista comes with easy-to-use libraries for 2D graphics, multitouch and sound that are inspired by the simplicity of
      <a href="http://processing.org/">
       Processing
      </a>
      . You can even use the accelerometer to create motion-controlled games or interactive experiments. We've included lots of ready-to-run examples, including three complete games and a multi-touch toy piano.
     </div>
    </div>
    <div class="row">
     <div class="column six">
      <div class="thumbnail">
       <a class="screenshot-retina" href="prompt.png">
        <img height="100%" src="prompt_thumb.png" width="100%"/>
       </a>
      </div>
      <strong>
       Powerful Interactive Prompt
      </strong>
      <br/>
      When you want to quickly try something out, the interactive prompt is always just a swipe away. It helps you with accurate code completion and a convenient command history.
     </div>
     <div class="column six">
      <div class="thumbnail">
       <a class="screenshot-retina" href="documentation.png">
        <img height="100%" src="documentation_thumb.png" width="100%"/>
       </a>
      </div>
      <strong>
       More Than a Toy
      </strong>
      <br/>
      Pythonista isn't just for games. The
      <a href="http://omz-software.com/pythonista/docs/library/index.html">
       Standard Library
      </a>
      has tons of modules for doing
      <a href="http://omz-software.com/pythonista/docs/library/numeric.html">
       math
      </a>
      ,
      <a href="http://omz-software.com/pythonista/docs/library/strings.html">
       processing text
      </a>
      , working with
      <a href="http://omz-software.com/pythonista/docs/library/internet.html">
       data from the web
      </a>
      , and much more. Additionally, you have access to the iOS
      <a href="http://omz-software.com/pythonista/docs/ios/clipboard.html">
       clipboard
      </a>
      , an in-app
      <a href="http://omz-software.com/pythonista/docs/library/webbrowser.html">
       web browser
      </a>
      and the powerful
      <a href="http://omz-software.com/pythonista/docs/ios/PIL.html">
       Python Imaging Library
      </a>
       including support for accessing photos in your
      <a href="http://omz-software.com/pythonista/docs/ios/photos.html">
       camera roll
      </a>
      .
     </div>
    </div>
    <div class="row">
     <div class="column six">
      <div class="thumbnail">
       <a class="screenshot-retina" href="mpl.png">
        <img height="100%" src="mpl_thumb.png" width="100%"/>
       </a>

       NumPy
      </a>
      and
      <a href="http://omz-software.com/pythonista/docs/matplotlib/index.html">
       matplotlib
      </a>
      packages for scientific computation and data visualization.
     </div>
     <div class="column six">
