            <h1 class="entry-title">
             How Centralization Makes Hosted Desktops More Secure
            </h1>
            <div class="entry-meta">
             <span class="meta-sep">
              by
             </span>
             <span class="author vcard">
              <a class="url fn n" href="http://www.onthenetoffice.com/blog/author/jason/" title="View all posts by Jason Pietryga">
               Jason Pietryga
              </a>
             </span>
             <span class="meta-prep meta-prep-author">
              on
             </span>
             <a href="http://www.onthenetoffice.com/blog/2013/09/desktop-hosting-services/" rel="bookmark" title="11:14 am">
              <span class="entry-date">
               September 6, 2013
              </span>
             </a>
            </div>
            <!-- .entry-meta -->
            <div class="entry-content">
             <a id="dd_start">
             </a>
             <p>
              <img alt="Hosted Desktops Secure OTNO Blog" class="alignnone size-full wp-image-2374" height="299" src="http://www.onthenetoffice.com/blog/wp-content/uploads/2013/09/Hosted-Desktops-Secure-OTNO-Blog.jpg" width="480"/>
             </p>
             <p>
              Every aspect of our
              <a href="http://www.onthenetoffice.com/">
               Hosted Desktop solutions
              </a>
              is continuously or regularly protected, monitored, and backed up. One of the keys to the effectiveness of our security infrastructure is the centralization of our Desktop Hosting solutions (our Hosted Desktops are stored on a small number of interconnected servers, between two and four Hosted Desktops per server).
             </p>
             <h2>
              The
              <strong>
               security benefits of centralization
              </strong>
              include:
             </h2>
             <ul>
              <ul>
               <li>
                <b>
                 Complete protection.
                </b>
               </li>
              </ul>
             </ul>
             <p>
              Though onthenetOffice Hosted Desktops can be accessed from anywhere and with any device, their files and applications remain on the servers in our datacenters. It might appear to the user that he or she is accessing a locally-installed Windows desktop operating system, but really the user is only sending keystrokes and mouse clicks to our servers, which handle all of the processing and storage of the Hosted Desktops. Since all of the files and applications of our Hosted Desktops remain on our servers at all times, this means that every bit of data on an onthenetOffice Virtual Desktop Infrastructure (VDI) is equally fully-protected.
             </p>
             <ul>
              <ul>
               <li>
                <b>
                 More efficient management.
                </b>
               </li>
              </ul>
             </ul>
             <p>
              We monitor all of our Hosted Desktops from a single, consolidated management console; this allows us to impose security measures efficiently and effectively and to respond quickly to any security incidents.
             </p>
             <ul>
              <ul>
               <li>
                <b>
                 Increased uniformity.
                </b>
               </li>
              </ul>
             </ul>
             <p>
              Decentralized IT infrastructures are made up of end-user devices that function independently and that may not have access to any centrally-hosted applications or databases. In this kind of IT environment, files and applications usually end up being stored on hard drives and on servers that are difficult or impossible to access. In addition, in a decentralized IT infrastructure the users are largely responsible for maintaining the security of their own devices, but they often lack the ability, time, or motivation to do so. With onthenetOffice Hosted Desktops, however, security measures and policies are uniformly strong throughout and dont rely on users to do anything.
             </p>
             <p>
              Of course, the centralization of our Hosted Desktops would be meaningless if the security measures and policies that it helped impose werent effective. onthenetOffices security infrastructure, however, is advanced, comprehensive, and thorough. Its features include:
             </p>
             <ul>
              <ul>
               <li>
                <a href="http://www.onthenetoffice.com/features/4-d-security/">
                 <b>
                  4-D Security
                 </b>
                </a>
               </li>
              </ul>
             </ul>
             <p>
              We protect our Hosted Desktops with a multidimensional security system, which provides physical, operational, logical, and system protections.
             </p>
             <ul>
              <ul>
               <li>
                <b>
                 24x7x365 monitoring.
                </b>
               </li>
              </ul>
             </ul>
             <p>
              Our technicians monitor businesses Desktop Hosting solutions around-the-clock for indications of hacking attempts or security breaches.
             </p>
             <ul>
              <ul>
               <li>
                <b>
                 Dedicated hosting.
                </b>
               </li>
              </ul>
             </ul>
             <p>
              Though we host multiple Virtual Desktops on a single physical server, we never host the Virtual Desktops of two different clients on the same physical server. Dedicated hosting eliminates the possibility of co-tenants of a server inappropriately accessing each others files.
             </p>
             <ul>
              <ul>
               <li>
                <b>
                 Compliant hosting.
                </b>
               </li>
              </ul>
             </ul>
             <p>
              Our Hosted Desktops are compliant with data security regulations such as PCI DSS, HIPAA, GLBA, SOX, and others.
             </p>
             <p>
              For a Hosted Desktop solution thats centralized and secure,
              <strong>
               sign up
              </strong>
              for Desktop Hosting,
              <a href="http://www.onthenetoffice.com/managed-hosting/compliance-hosting/">
               PCI Compliant Hosting
              </a>
              , or HIPAA Compliant Hosting from onthenetOffice today!
             </p>
