     <h2>
      Purpose
     </h2>
     <p>
      Merges bracketed photos and saves the merged image as Floating Point TIFF file.
      <!-- ready for Tone Mapping in Lightroom Develop module.-->
     </p>
     <p>
      This plugin does not render the Floating Point TIFF file it creates. If you want to create HDR photos from Lightroom with Photomatix HDR Tone Mapping and Exposure Fusion, please download instead the
      <a href="http://www.hdrsoft.com/download/lrplugin.html">
       Photomatix Plugin for Lightroom
      </a>
      , which is included for free in Photomatix Pro.
     </p>
     <h2>
      Difference with Lightroom CC Merge to HDR
     </h2>
     <p>
      Since Lightroom CC/6, Lightroom includes a feature to merge to HDR, and therefore the same feature as the Merge to 32-bit Plugin.
     </p>
     <p>
      However, the Merge to 32-bit Plugin doesn't use the same algorithms as Lightroom CC to merge, align and remove ghosts. Also, Lightroom sets the default rendering for the merged image differently when it is created by Lightroom's built-in Merge to HDR.  The results of the Merge to 32 Plugin look therefore different from the results of Lightroom's Merge to HDR.
     </p>
     <p>
      Additionally, Lightroom's Merge to HDR saves the merged image as a 16-bit Floating Point DNG file, while the Plugin saves it as 32-bit TIFF file by default (with an option to save as 16-bit Floating Point TIFF).
     </p>
     <h2>
      Requirements
     </h2>
     <p>
      This plugin requires Lightroom
      <b>
       4.1
      </b>
      or higher.

     <h2>
      Using the Plugin
     </h2>
     <ol class="list olist">
      <li>
       After installation, restart Lightroom if you are on Windows (or used the installer on Mac)
      </li>
      <li>
       Select bracketed photos from either the Filmstrip or the Grid.
      </li>
      <li>
       Right-click to invoke the contextual menu.
      </li>
      <li>
       Select
       <span class="bold">
        Export
       </span>
       and then
       <span class="bold">
        Merge to 32-bit HDR
       </span>
      </li>
      <li>
       Select the desired options and click on the Merge button
      </li>
      <li>
       In case the merged image does not show once the merge has completed, click above the filmstrip and check
       <span class="bold">
        All Photographs
       </span>
       .
      </li>
      <li>
       You can then adjust the merged image in Lightroom's
       <span class="bold">
        Develop
       </span>
       module.
      </li>
     </ol>
     <h2>
      Important Note
     </h2>
     <p>
      The Floating Point TIFF file that this plugin creates is meant to be rendered using Lightroom Develop module, and not Photomatix HDR processes.
     </p>
     <p>
      Images shown in the user gallery of the HDRsoft.com website were
      <span class="strong">
       not
      </span>
      created using the Merge to 32-bit Plug-in.
     </p>
     <h2>
      Merging to 32-bit without the Plugin
     </h2>
     <p>
      Photomatix Pro can already merge to HDR to create a 32-bit HDR file accepted by Lightroom as described below. The Merge to 32-bit Plug-in for Lightroom adds convenience, but is not necessary if you have Photomatix Pro.
     </p>
     <ol class="list olist">
      <li>
       Open Photomatix Pro and click on the
       <span class="bold">
        Load Bracketed Photos
       </span>
       button.
      </li>
      <li>
       On the dialog to load the photos, check the
       <span class="bold">
        Show intermediary 32-bit HDR image
       </span>
       box.
      </li>
      <li>
       Select the desired option on the Pre-Processing Options dialog, and click OK
      </li>
      <li>
       Once the merge is completed, choose
       <span class="bold">
        File &gt; Save As...
       </span>
       .
      </li>
      <li>
       Select the
       <span class="bold">
        Floating Point TIFF (.tif)
       </span>
       item on the File Format combo-box and click OK.
      </li>
      <li>
       Import the saved TIFF file into Lightroom.
      </li>
     </ol>
