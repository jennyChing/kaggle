      <h1 class="cx-blue">
       What to Do If Experiencing Laptop Battery Problems
      </h1>
      <!--Posted May 2nd, 2015 under &lt;a href="http://www.supportrix.com/knowledgebase/pc-tune-up-repair/" rel="tag"&gt;PC Tune-Up &amp; Repair&lt;/a&gt;&lt;br /&gt;&lt;br /&gt;-->
      <div class="entry p-gray">
       <div id="body_kb">
        <p class="kb_KBTxt">
         Have you ever been in a situation wherein your laptop shows that you still have some battery power left (say 45 percent), but then it suddenly died on you? Your laptops battery capacity will eventually diminish no matter how you take good care of it. And regardless if your laptops on top of the line during the time you bought it, theres still a possibility that its built-in power meter will give you incorrect readings as to how much battery life you have left. Factors such as age, heat, and prolonged usage also contributes to its decreased capacity.
        </p>
        <p class="kb_KBTxt">
         Because of this, manufacturers usually recommend calibrating the battery every two to three months to help keep your battery readings accurate. If your laptop dies on you without giving out any warning, you may need to do this sooner. Even though your computer has a pre-installed app that you can use to calibrate the battery, it is still helpful to know how to do this manually. Below are some quick and easy methods on how you can calibrate your laptop battery.
        </p>
        <p class="kb_Titles_OPTIONS">
         Top Up Your Battery
        </p>
        <p class="kb_KBTxt">
         First thing you need to do is to top up your battery by plugging it in and let the battery charge normally. When it reaches 100 percent, leave the laptop plugged in for a few hours so that the battery has a chance to breathe and cool down. The reason for this is that the heat can result to giving out inaccurate results.
        </p>
        <p class="kb_Titles_OPTIONS">
         Set Up the Battery
        </p>
        <blockquote>
         <p>
          <span class="kb_STEPno">
           STEP 1:
          </span>
          <span class="kb_KBTxt">
           Go to the Power Options in Windows either by searching for it in the search field in the Start menu, or by right-clicking on the charging icon on the taskbar.
          </span>
         </p>
         <p>
          <span class="kb_STEPno">
           STEP 2:
          </span>
          <span class="kb_KBTxt">
           Select the
           <strong>
            Balanced
           </strong>
           option.
          </span>
         </p>
         <p>
          <span class="kb_STEPno">
           STEP 3:
          </span>
          <span class="kb_KBTxt">
           Click on
           <strong>
            Edit Plan Settings
           </strong>
           , then take note what each of the boxes under
           <strong>
            On battery
           </strong>
           is set as.
          </span>
         </p>
         <blockquote>
          <p>
           <img alt="" class="alignnone size-full wp-image-9187" height="504" src="http://www.supportrix.com/main/wp-content/uploads/2015/04/laptop-batt-prob_01.png" width="607"/>
          </p>
         </blockquote>
         <p>
          <span class="kb_STEPno">
           STEP 4:
          </span>
          <span class="kb_KBTxt">
           Change them all to
           <strong>
            Never
           </strong>
           from the drop-down menus.
          </span>
         </p>
        </blockquote>
        <p class="kb_Titles_OPTIONS">
         Set Up Hibernation Point
        </p>
        <blockquote>
         <p>
          <span class="kb_STEPno">
           STEP 1:
          </span>
          <span class="kb_KBTxt">
           To make your battery work as long as it possibly can, find
           <strong>
            Battery
           </strong>
           under
           <strong>
            Power Options
           </strong>
           , then make sure it is set to
           <strong>
            Hibernate
           </strong>
           .
          </span>
         </p>
         <blockquote>
          <p>
           <img alt="" class="alignnone size-full wp-image-9188" height="521" src="http://www.supportrix.com/main/wp-content/uploads/2015/04/laptop-batt-prob_02.png" width="433"/>
          </p>
         </blockquote>
         <p>
          <span class="kb_STEPno">
           STEP 2:
          </span>
          <span class="kb_KBTxt">
           Next, check
           <strong>
            Low Battery Action
           </strong>
           and make sure that all options are set to
           <strong>
            Nothing
           </strong>
           .
          </span>
         </p>
         <p>
          <span class="kb_STEPno">
           STEP 3:
          </span>
          <span class="kb_KBTxt">
           Under
           <strong>
            Critical Battery Level
           </strong>
           , select
           <strong>
            5 percent
           </strong>
           from the drop-down list.
          </span>
         </p>
         <p>
          <span class="kb_STEPno">
           STEP 4:
          </span>
          <span class="kb_KBTxt">
           Click
           <strong>
            OK
           </strong>
           on all windows.
          </span>
         </p>
         <blockquote>
          <p>
           <img alt="" class="alignnone size-full wp-image-9189" height="482" src="http://www.supportrix.com/main/wp-content/uploads/2015/04/laptop-batt-prob_03.png" width="433"/>
          </p>
         </blockquote>
        </blockquote>
        <p class="kb_Titles_OPTIONS">
         Let the Battery Drain First Before Recharging
        </p>
        <blockquote>
         <p>
          <span class="kb_STEPno">
           STEP 1:
          </span>
          <span class="kb_KBTxt">
           Once you have clicked in Power Options, click on the
           <strong>
            Save changes
           </strong>
           button on the
           <strong>
            Plan Settings
           </strong>
           window.
          </span>
         </p>
         <p>
          <span class="kb_STEPno">
           STEP 2:
          </span>
          <span class="kb_KBTxt">
           Unplug your laptop to allow it to discharge
          </span>
          <span class="kb_KBTxt">
           the battery.
          </span>
         </p>
         <p>
          <span class="kb_STEPno">
           STEP 3:
          </span>
          <span class="kb_KBTxt">
           Use your laptop as you normally would, or leave it on to discharge.
          </span>
         </p>
         <blockquote>
          <p>
           <span class="kb_NOTE">
            NOTE:
           </span>
           <span class="kb_NOTEtxt">
            If you have set up everything correctly, your laptop will hibernate at the lowest level you have set your battery at. Your screen wont turn off, and it shouldnt enter sleep mode.
           </span>
          </p>
         </blockquote>
         <p>
          <span class="kb_STEPno">
           STEP 4:
          </span>
          <span class="kb_KBTxt">
           Now, wait before charging it back to 100 percent by using it as you normally would or you can leave your laptop on overnight.
          </span>
         </p>
        </blockquote>
        <p class="kb_Titles_OPTIONS">
         Revert Settings
        </p>
        <blockquote>
         <p>
          <span class="kb_STEPno">
           STEP 1:
          </span>
          <span class="kb_KBTxt">
           Once you have completed the previous step, go ahead and change your Power Option settings back to the way they were before.
          </span>
         </p>
         <p>
          <span class="kb_STEPno">
           STEP 2:
          </span>
          <span class="kb_KBTxt">
           Consider switcing off the screen sooner or even get your computer to sleep after ten minutes to help you conserve power.
          </span>
         </p>
        </blockquote>
       </div>

        $(this).removeClass('alert').removeClass('alert-danger');
        $(this).removeClass('alert').removeClass('alert-success');

      function validateContactForm(e){
        e.preventDefault();
        $(this).addClass('disabled');
        var name = $('#contact-us-name').val(),
            email = $('#contact-us-email').val(),
            phone = $('#contact-us-phone').val(),
            msg = $('#message').val();

        if(!name){
            handleResult({
            id : 'main-page-msg',
            success: false,
            msg: "Name is required",
            field: "contact-us-name"
          });
          $(this).removeClass('disabled');
          return false;
        }
        if(!email){
            handleResult({
            id : 'main-page-msg',
            success: false,
            msg: "Email is required",
            field: "contact-us-email"
          });
          $(this).removeClass('disabled');
          return false;
        }
        if(!ValidateEmail(email)){
            handleResult({
            id : 'main-page-msg',
            success: false,
            msg: "Please input valid email",
            field: "contact-us-email"
          });
          $(this).removeClass('disabled');
          return false;
        }
        if(!phone){
            handleResult({
            id : 'main-page-msg',
            success: false,
            msg: "Subject is required",
            field: "contact-us-phone"
          });
          $(this).removeClass('disabled');
          return false;
        }
        if(!msg){
            handleResult({
            id : 'main-page-msg',
            success: false,
            msg: "Message is required",
            field: "message"
          });
          $(this).removeClass('disabled');
          return false;
        }
        handleResult({
            id : 'main-page-msg',
            success: true,
            msg: "Email sent."
          });
        // $('#contact-us-name').addClass('alert alert-success');
        // $('#contact-us-email').addClass('alert alert-success');
        // $('#contact-us-phone').addClass('alert alert-success');
        // $('#message').addClass('alert alert-success');
        $('.contact-us-form').submit();
      }

    function ValidateEmail(email){
      var r = false;
      var emailReg = "^[\\w-_\.]*[\\w-_\.]\@[\\w]\.+[\\w]+[\\w]$";
      var regex = new RegExp(emailReg);
      var is_regex_true = regex.test(email);
      if(is_regex_true){
        var domain = email.substr(email.length - 4);
        if(domain == '.com')
          r = true;
      }
      return r;
    }

    function handleResult(data) {
      var el = $('#'+ data.id);
          $('.form-group').removeClass('has-error');
      if(data.field){
          // var container = $('#'+ data.field).parent().parent();
          var container = $('#' + data.field);
          container.addClass('alert alert-danger');
      }
      

      var addClass = (data.success) ? 'alert-success' : 'alert-danger';
      el.removeClass('alert-success').removeClass('alert-danger');
      el.addClass(addClass);
      el.find('p').text(data.msg);
      el.slideDown(400, function() {
          setTimeout(function() {
          el.slideUp();
          }, 5000);
      });
    }
