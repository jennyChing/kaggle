         <center>
          <span style="font-size: x-large;">
           Every prototype starts with a
           <a href=" http://dangerousprototypes.com/docs/Bus_Pirate">
            Bus Pirate
           </a>
          </span>
         </center>
         <br/>
         <center>
          <span style="font-size: medium;">
           One open source development tool. So many features. Only $30 bucks.
          </span>
         </center>
         <div id="bpbg1">
          <div style="position: absolute; left: 0px; top: 0px; width: 422px;">
           <strong>
            Develop easier
           </strong>
           <br/>
           The Bus Pirate is an open source hacker multi-tool that talks to electronic stuff. Its got a
           <a href="http://dangerousprototypes.com/docs/Features_overview">
            bunch of features
           </a>
           an intrepid hacker might need to prototype their next project.
           <ul>
            <li>
             $30, including world wide shipping.
            </li>
           </ul>
           <p>
            <a href="http://www.seeedstudio.com/depot/bus-pirate-v3-assembled-p-609.html?cPath=61_68">
             <img alt="" class="alignnone size-full wp-image-50739" height="70" src="http://dangerousprototypes.com/wp-content/media/2009/08/buynow.png" title="buynow" width="208"/>
            </a>
           </p>
          </div>
          <div style="position: absolute; left: 531px; top: 250px; width: 375px;">
           <strong>
            Talk to serial chips
           </strong>
           <br/>
           Connect a few wires an start talking to 1-Wire, I2C, SPI, serial, JTAG, MIDI, PC Keyboards, etc. through a simple command line interface. Even program many AVR and PIC microcontrollers, debug ARM chip, program CPLDs, and flash serial ROMs.
           <p>
            <strong>
             Tools you need
            </strong>
            <br/>
            Frequency measurement, frequency generation, servo drivers, logic analyzer, voltage measurement, and more. Tools you need to get your project done.
           </p>
           <p>
            <strong>
             Lots of extras
            </strong>
            <br/>
            Many protocols need pullup resistors, the Bus Pirate has them on-board. Want to control your project from a computer? Try the scripting mode.
           </p>
          </div>
