                   <br style="color: rgb(0, 0, 0); font-family: arial, verdana, helvetica, sans-serif;"/>
                   <br style="color: rgb(0, 0, 0); font-family: arial, verdana, helvetica, sans-serif;"/>
                   <strong>
                    Shrink Plastic Horse Necklaces - Materials:
                   </strong>
                   <br style="color: rgb(0, 0, 0); font-family: arial, verdana, helvetica, sans-serif;"/>
                   - Permanent Markers
                   <br style="color: rgb(0, 0, 0); font-family: arial, verdana, helvetica, sans-serif;"/>
                   - Shrink Plastic
                   <br style="color: rgb(0, 0, 0); font-family: arial, verdana, helvetica, sans-serif;"/>
                   - Scissors
                   <br style="color: rgb(0, 0, 0); font-family: arial, verdana, helvetica, sans-serif;"/>
                   - Hole Punch
                   <br style="color: rgb(0, 0, 0); font-family: arial, verdana, helvetica, sans-serif;"/>
                   -
                   <a href="https://drive.google.com/file/d/0B_wlvQ9TsQRSbS1CNXRJOUE0eTA/view?usp=sharing" target="_blank">
                    Horse Template
                   </a>
                   <br style="color: rgb(0, 0, 0); font-family: arial, verdana, helvetica, sans-serif;"/>
                   - Ribbon or elastic
                   <br style="color: rgb(0, 0, 0); font-family: arial, verdana, helvetica, sans-serif;"/>
                   - Horse Template
                   <br style="color: rgb(0, 0, 0); font-family: arial, verdana, helvetica, sans-serif;"/>
                   - Beads
                   <br style="color: rgb(0, 0, 0); font-family: arial, verdana, helvetica, sans-serif;"/>
                   - Oven Pan

                   <br/>

                   <br style="color: rgb(0, 0, 0); font-family: arial, verdana, helvetica, sans-serif;"/>
                   Place cookie sheet in the oven at this time to pre-heat the oven, too. Pre-heating the pan aids in keeping the shrink plastics from curling up on themselves.
                   <br style="color: rgb(0, 0, 0); font-family: arial, verdana, helvetica, sans-serif;"/>
                   <br style="color: rgb(0, 0, 0); font-family: arial, verdana, helvetica, sans-serif;"/>

                    horse template
                   </a>
                   . Place horse template under shrink plastic.
                   <br/>

                   <br/>

                  </span>
                 </span>

                 <span style="color:#696969;">

                 <br/>
                 <span style="font-family:verdana,geneva,sans-serif;">

                   </span>
                  </span>
                 </span>

                 <br/>

                  </span>
                 </span>

                 <br/>
                 <br/>
                 <span style="color: rgb(0, 0, 0); font-family: arial, verdana, helvetica, sans-serif;">

                  </span>
