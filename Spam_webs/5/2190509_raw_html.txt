      <font color="#000000" face="Geneva, Arial, Helvetica, sans-serif" size="+1">
       Fractal Zplot is an interactive program for generating fractal images. Originally named Zplot, this program has been in continuous development since 1989. Fractal Zplot currently supports the 
  
      Mandelbrot set, Julia sets, Newton's method, and Phoenix curves, with millions of 
  
      mapping variations. 3D fractal types include quaternion, hypercomplex, 
  
      orbital fractals and lsystems, in addition to the original 
  
      height field and landscape types.
      </font>
      <p>
       <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">
        <strong>
         Features of Fractal Zplot:
        </strong>
       </font>
      </p>
      <ul>
       <li>
        <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">
         Getting Started Tutorial
        </font>
       </li>
       <li>
        <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">
         208 Built in Formulas +
         <a href="FraSZle.html">
          FraSZle
         </a>
         Formula set
        </font>
       </li>
       <li>
        <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">
         8 Mandelbulb 3D formulas (trig and non-trig sine and cosine versions)
        </font>

       <li>
        <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">
         Detailed Parser information for creating your own custom formulas

         <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">
          Mesh simplification for smaller OBJ and POV export files

        </li>

         <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">
          Virtual reality [WRL]

         <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">
          AutoCad [DXF]

        <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">
         3D object exports:

         <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">
          Lsystems [OBJ, STL, POV, DXF]

        <li>
         <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">
          Orbital fractals [OBJ, DXF]
         </font>
        </li>

          Height fields [OBJ, POV, DXF]

       </li>
       <li>
        <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">
         Adjustable lighting model with Phong highlights
        </font>
       </li>
       <li>
        <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">
         Random image generators and random genetic formula generator
        </font>

        <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">
         Batch mode for unattended image creation

       </li>
       <li>
        <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">
         Integrated video routines allow easy morphing and rotating between key frames
        </font>
       </li>
       <li>

         <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">

       </li>
       <li>

        <font face="Geneva, Arial, Helvetica, sans-serif" size="+1">
