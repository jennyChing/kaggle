        <div class="post-headline">
         <h1>
          How to install Turbo C++ on Windows 7 64bit
         </h1>
        </div>
        <div class="post-bodycopy clearfix">
         <p>
          Few days ago we have posted an article about installing
          <strong>
           Turbo C++
          </strong>
          on
          <a href="http://blog.bestsoftware4download.com/2010/02/how-to-install-turbo-c-on-windows-7/" target="_blank" title="Read the article">
           <strong>
            32 bit Windows 7
           </strong>
           .
          </a>
          Now we are providing step-by-step procedure how to install Turbo C++ on
          <strong>
           64 bit Windows 7
          </strong>
          .
         </p>
         <p>
          <span id="more-2132">
          </span>
         </p>
         <table style="border: medium none; background-color: #ffffff; height: 1909px;" width="713">
          <tbody>
           <tr>
            <td width="675">
             1. Install the software
             <strong>
              DOSBox
             </strong>
             ver 0.73 :
             <a href="http://www.bestsoftware4download.com/software/t-free-dosbox-download-rqeabjvs.html" target="_blank" title="DOSBox ver 0.73">
              download here
             </a>
            </td>
           </tr>
           <tr>
            <td>
             2. Create a folder,for example  
             <strong>
              Turbo
             </strong>
                (c:\Turbo\)
            </td>
           </tr>
           <tr>
            <td>
             3. Download and extract
             <strong>
              TC
             </strong>
             into the Turbo folder  (c:\Turbo\):
             <a href="http://blog.bestsoftware4download.com/goto/http://s1.freedownloadme.info/turboc_installed.zip" rel="nofollow" target="_blank" title="ZIP file">
              download here
             </a>
            </td>
           </tr>
           <tr>
            <td>
             4. Run the DOSBox 0.73 from the icon located on the desktop:
            </td>
           </tr>
           <tr>
            <td>
             <a data-slb-active="1" data-slb-group="2132" data-slb-internal="0" href="http://blog.bestsoftware4download.com/wp-content/uploads/2010/02/step-1.jpg">
              <img alt="Run the DOSBox" class="alignleft size-full wp-image-2139" height="379" src="http://blog.bestsoftware4download.com/wp-content/uploads/2010/02/step-1.jpg" title="Run the DOSBox" width="603"/>
             </a>
            </td>
           </tr>
           <tr>
            <td>
            </td>
           </tr>
           <tr>
            <td>
             5.  Type the following commands at the command prompt [Z]:
             <strong>
              mount d c:\Turbo\
             </strong>
             [The folder
             <strong>
              TC
             </strong>
             is present inside the folder Turbo]
            </td>
           </tr>
           <tr>
            <td>
             Now you should get a message which says:
             <strong>
              Drive D is mounted as a local directory c:\Turbo\
             </strong>
            </td>
           </tr>
           <tr>
            <td>
             <a data-slb-active="1" data-slb-group="2132" data-slb-internal="0" href="http://blog.bestsoftware4download.com/wp-content/uploads/2010/02/step-2.jpg">
              <img alt="" class="alignleft size-full wp-image-2149" height="392" src="http://blog.bestsoftware4download.com/wp-content/uploads/2010/02/step-2.jpg" title="step 2" width="612"/>
             </a>
            </td>
           </tr>
           <tr>
            <td>
             6. Type
             <strong>
              d:
             </strong>
             to shift to d:
            </td>
           </tr>
           <tr>
            <td>
             <a data-slb-active="1" data-slb-group="2132" data-slb-internal="0" href="http://blog.bestsoftware4download.com/wp-content/uploads/2010/02/step-3.jpg">
              <img alt="" class="alignleft size-full wp-image-2150" height="380" src="http://blog.bestsoftware4download.com/wp-content/uploads/2010/02/step-3.jpg" title="step 3" width="606"/>
             </a>
            </td>
           </tr>
           <tr>
            <td>
             7. Next follow the commands below:
             <p>
              <strong>
               cd tc
              </strong>
             </p>
             <p>
              <strong>
               cd bin
              </strong>
             </p>
             <p>
              <strong>
               tc
              </strong>
              or
              <strong>
               tc.exe
              </strong>
              [
              <em>
               This starts you the Turbo C++ 3.0
              </em>
              ]
             </p>
            </td>
           </tr>
           <tr>
            <td>
             <a data-slb-active="1" data-slb-group="2132" data-slb-internal="0" href="http://blog.bestsoftware4download.com/wp-content/uploads/2010/02/step-4.jpg">
              <img alt="" class="alignleft size-full wp-image-2153" height="378" src="http://blog.bestsoftware4download.com/wp-content/uploads/2010/02/step-4.jpg" title="screenshot" width="603"/>
             </a>
            </td>
           </tr>
           <tr>
            <td>
             8. In the Turbo C++ goto Options&gt;Directories&gt; Change the source of TC to the source directory [D] ( i.e. virtual D: refers to original c:\Turbo\ . So make the path change to something like D:\TC\include and D:\TC\lib respectively )
            </td>
           </tr>
          </tbody>
         </table>
         <table style="height: 1022px;" width="713">
          <tbody>
           <tr>
            <td width="705">
             <h2>
              <strong>
               How to start TurboC++ in the DOSBox automatically:
              </strong>
             </h2>
             <p>
              You can save yourself some time by having DOSBox automatically mount your folders and start TurboC++:
             </p>
            </td>
           </tr>
           <tr>
            <td>
             For DOSBox versions older then 0.73 browse into program installation folder and open the dosbox.conf file in any text editor. For version 0.73 go to
             <strong>
              Start Menu
             </strong>
             and click on 
             <strong>
              Configuration
             </strong>
              and then 
             <strong>
              Edit Configuration
             </strong>
             . Then scroll down to the very end, and add the lines which you want to automatically execute when DOSBox starts.
            </td>
           </tr>
           <tr>
            <td>
             Automatically mount and start Turbo C++3.0 in DOSBox ver 0.73:
            </td>
           </tr>
           <tr>
            <td>
             <a data-slb-active="1" data-slb-group="2132" data-slb-internal="0" href="http://blog.bestsoftware4download.com/wp-content/uploads/2010/02/step-5.jpg">
              <img alt="" class="alignleft size-full wp-image-2157" height="376" src="http://blog.bestsoftware4download.com/wp-content/uploads/2010/02/step-5.jpg" title="screenshot" width="602"/>
             </a>
            </td>
           </tr>
           <tr>
            <td>
             Scroll down to the very end, and add the lines:
            </td>
           </tr>
           <tr>
            <td>
             <a data-slb-active="1" data-slb-group="2132" data-slb-internal="0" href="http://blog.bestsoftware4download.com/wp-content/uploads/2010/02/step-6.jpg">
              <img alt="" class="alignleft size-full wp-image-2158" height="379" src="http://blog.bestsoftware4download.com/wp-content/uploads/2010/02/step-6.jpg" title="screenshot" width="603"/>
             </a>
            </td>
           </tr>
           <tr>
            <td>
             Those commands will be executed automatically when DOSBox  starts!
            </td>
           </tr>
           <tr>
            <td>
             <strong>
              Please note:
             </strong>
             <p>
              Full screen:  Alt and Enter
             </p>
             <p>
              When you exit from the DosBox [precisely when u unmount the virtual drive where Turbo C++ 3.0 has been mounted] all the files you have saved or made changes in Turbo C++ 3.0 will be copied into the source directory(The directory which contains TC folder)
             </p>
             <p>
              Dont use shortcut keys to perform operations in TC because they might be a shortcut key for DOSBOX also . Eg : Ctrl+F9 will exit DOSBOX rather running the code .
             </p>
