         <div class="post-content">
          <!-- test &lt;img class="aligncenter size-full wp-image-865" src="http://www.flights.com/blog/wp-content/uploads/2014/12/Week33_5_McKenzie-Falls-Grampians-National-Park.jpg" alt="Week33_5_McKenzie Falls Grampians National Park" width="1450" height="965" /&gt; -->
          <p>
           Youre more extreme than a Mountain Dew commercial but cant seem to find a vacation spot that truly gets your blood flowing. Grab your Crocodile Dundee hat, daredevil, because Australia is the perfect place for you. Say gday to fivereasons that make this island nation the essential vacation locale for adrenaline junkies:
          </p>
          <h2>
           <strong>
            1. The Great Barrier Reef is right over there
           </strong>
          </h2>
          <p>
           <span style="position:relative; display:block;">
            <span style="position: absolute;top: 8px;right: 10px;">
             <a href="http://pinterest.com/pin/create/button/?url=http%3A%2F%2Fwww.flights.com%2Fblog%2F5-reasons-why-australia-is-a-playground-for-adrenaline-junkies%2F&amp;media=http://www.flights.com/blog/wp-content/uploads/2014/12/week33_1_Great-Barrier-Reef.jpg&amp;description=5+Reasons+Why+Australia+is+a+Playground+for+Adrenaline+Junkies">
             </a>
            </span>
            <a href="http://www.flights.com/blog/wp-content/uploads/2014/12/week33_1_Great-Barrier-Reef.jpg">
             <img alt="week33_1_Great Barrier Reef" class="aligncenter size-full wp-image-861" height="1199" src="http://www.flights.com/blog/wp-content/uploads/2014/12/week33_1_Great-Barrier-Reef.jpg" width="1800"/>
            </a>
           </span>
          </p>
          <p>
           You know that giant coral reef thats visible from space? The one where Nemo was finally found? Its basically Australias backyard. Perched just off the coast of Queensland, this oceanic wonderlandthe worlds largest reef at 133,000 square milesbeckons to adventurers of all stripes: divers, snorkelers, surfers, and even glass-bottomed boaters. Plus, if the Great Barrier Reef isnt exciting enough for you, there are roughly a trillion other spots to windsurf, kitesurf, and dive with sharks. Youre going to need a bigger boat.
          </p>
          <h2>
           <strong>
            2. Bushwalking is totally a thing
           </strong>
          </h2>
          <p>
           <span style="position:relative; display:block;">
            <span style="position: absolute;top: 8px;right: 10px;">
             <a href="http://pinterest.com/pin/create/button/?url=http%3A%2F%2Fwww.flights.com%2Fblog%2F5-reasons-why-australia-is-a-playground-for-adrenaline-junkies%2F&amp;media=http://www.flights.com/blog/wp-content/uploads/2014/12/Week33_2-_Young-Woman-Hiking.jpg&amp;description=5+Reasons+Why+Australia+is+a+Playground+for+Adrenaline+Junkies">
             </a>
            </span>
            <a href="http://www.flights.com/blog/wp-content/uploads/2014/12/Week33_2-_Young-Woman-Hiking.jpg">
             <img alt="Week33_2-_Young Woman Hiking" class="aligncenter size-full wp-image-862" height="1000" src="http://www.flights.com/blog/wp-content/uploads/2014/12/Week33_2-_Young-Woman-Hiking.jpg" width="1000"/>
            </a>
           </span>
          </p>
          <p>
           It may sound like something that belongs at the X-Games with Shaun White, but bushwalking is actually a common activity around here. Its essentially just a DIY hike with one major exception: The activity tends to involve rambling through the wilderness for days at a time. Tasmania is tops for these types of treksthe Overland Track is especially goodbut Western Australias Bibbulmun Track is among the more scenic. Fire up your Instagram app, pack some extra water, and prepare to make your friends very jealous.
          </p>
          <h2>
           <strong>
            3. Undara National Park has lava tubes
           </strong>
          </h2>
          <p>
           <span style="position:relative; display:block;">
            <span style="position: absolute;top: 8px;right: 10px;">
             <a href="http://pinterest.com/pin/create/button/?url=http%3A%2F%2Fwww.flights.com%2Fblog%2F5-reasons-why-australia-is-a-playground-for-adrenaline-junkies%2F&amp;media=http://www.flights.com/blog/wp-content/uploads/2014/12/Week33_3_Bats.jpg&amp;description=5+Reasons+Why+Australia+is+a+Playground+for+Adrenaline+Junkies">
             </a>
            </span>
            <a href="http://www.flights.com/blog/wp-content/uploads/2014/12/Week33_3_Bats.jpg">
             <img alt="Week33_3_Bats" class="aligncenter size-full wp-image-863" height="1125" src="http://www.flights.com/blog/wp-content/uploads/2014/12/Week33_3_Bats.jpg" width="1500"/>
            </a>
           </span>
          </p>
          <p>
           These things sound completely terrifying, but dont start piling on the SPF just yet. Queenslands tubes havent seen much action since a Cenozoic eruption formed them nearly 200,000 years ago, so youre mostly scoping out the cave-like byproducts. Still, its a fully subterranean experience that often includes a nose-to-nose look at microbats and creepy tree snakes. Our pulses are racing just thinking about it.
          </p>
          <h2>
           <strong>
            4. Theres a billion different ways to fly in the air
           </strong>
          </h2>
          <p>
           <span style="position:relative; display:block;">
            <span style="position: absolute;top: 8px;right: 10px;">
             <a href="http://pinterest.com/pin/create/button/?url=http%3A%2F%2Fwww.flights.com%2Fblog%2F5-reasons-why-australia-is-a-playground-for-adrenaline-junkies%2F&amp;media=http://www.flights.com/blog/wp-content/uploads/2014/12/Week33_4_Hot-air-balloon.jpg&amp;description=5+Reasons+Why+Australia+is+a+Playground+for+Adrenaline+Junkies">
             </a>
            </span>
            <a href="http://www.flights.com/blog/wp-content/uploads/2014/12/Week33_4_Hot-air-balloon.jpg">
             <img alt="Week33_4_Hot air balloon" class="aligncenter size-full wp-image-864" height="1200" src="http://www.flights.com/blog/wp-content/uploads/2014/12/Week33_4_Hot-air-balloon.jpg" width="1800"/>
            </a>
           </span>
          </p>
          <p>
           From skydiving in Perth and hang gliding in Sydney to bungee jumping in Queensland and hot air ballooning in Hunter Valley, the options are basically endless for thrill-seekers who love to get airborne. Plus, if youve got a need for speed, you can suit up like Maverick for a jet-fighter flight in Adelaide. Well give you bonus points if you listen to Danger Zone the entire time.
          </p>
          <h2>
           5.
           <strong>
            Good ol terra firma has plenty of thrills of its own
           </strong>
          </h2>
          <p>
           <span style="position:relative; display:block;">
            <span style="position: absolute;top: 8px;right: 10px;">
             <a href="http://pinterest.com/pin/create/button/?url=http%3A%2F%2Fwww.flights.com%2Fblog%2F5-reasons-why-australia-is-a-playground-for-adrenaline-junkies%2F&amp;media=http://www.flights.com/blog/wp-content/uploads/2014/12/Week33_5_McKenzie-Falls-Grampians-National-Park.jpg&amp;description=5+Reasons+Why+Australia+is+a+Playground+for+Adrenaline+Junkies">
             </a>
            </span>
            <a href="http://www.flights.com/blog/wp-content/uploads/2014/12/Week33_5_McKenzie-Falls-Grampians-National-Park.jpg">
             <img alt="Week33_5_McKenzie Falls Grampians National Park" class="aligncenter size-full wp-image-865" height="965" src="http://www.flights.com/blog/wp-content/uploads/2014/12/Week33_5_McKenzie-Falls-Grampians-National-Park.jpg" width="1450"/>
            </a>
           </span>
          </p>
          <p>
           Australia is home to more than 500 national parks, so theres very little you cant do while youre down under. Biking? Head to the trails in Sydneys Royal National Park. Climbing? Try the nerve-wracking rocks in Victorias Grampians National Park. You can even do a little off-roading in the sandy wilderness of Windy Harbours DEntrecasteaux. Leave the flying to the birds; this place is paradise for anyone who prefers solid ground. Enjoy your adventures in Australia, mate.
          </p>

        function play_mp3(flg, ids, mp3url, volume, loops)
        {
            //Check the file URL parameter value
            var pieces = mp3url.split("|");
            if (pieces.length &gt; 1) {//We have got an .ogg file too
                mp3file = pieces[0];
                oggfile = pieces[1];
                //set the file URL to be an array with the mp3 and ogg file
                mp3url = new Array(mp3file, oggfile);
            }

            soundManager.createSound({
                id: 'btnplay_' + ids,
                volume: volume,
                url: mp3url
            });

            if (flg == 'play') {
                    soundManager.play('btnplay_' + ids, {
                    onfinish: function() {
                        if (loops == 'true') {
                            loopSound('btnplay_' + ids);
                        }
                        else {
                            document.getElementById('btnplay_' + ids).style.display = 'inline';
                            document.getElementById('btnstop_' + ids).style.display = 'none';
                        }
                    }
                });
            }
            else if (flg == 'stop') {
    //soundManager.stop('btnplay_'+ids);
                soundManager.pause('btnplay_' + ids);
            }
        }
        function show_hide(flag, ids)
        {
            if (flag == 'play') {
                document.getElementById('btnplay_' + ids).style.display = 'none';
                document.getElementById('btnstop_' + ids).style.display = 'inline';
            }
            else if (flag == 'stop') {
                document.getElementById('btnplay_' + ids).style.display = 'inline';
                document.getElementById('btnstop_' + ids).style.display = 'none';
            }
        }
        function loopSound(soundID)
        {
            window.setTimeout(function() {
                soundManager.play(soundID, {onfinish: function() {
                        loopSound(soundID);
                    }});
            }, 1);
        }

                }
                if (inputs[i].id.indexOf("btnstop_") == 0) {
                    inputs[i].style.display = 'none';//Hide the stop button
                }
