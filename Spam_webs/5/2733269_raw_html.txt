     <h4 style="text-align:center">
      Chapter 4: Transcendental Knowledge
     </h4>
     <div class="Verse-Number-Heading">
      Bg 4.2
     </div>
     <div class="Textnum">
      TEXT 2
     </div>
     <div style="text-align:center; margin-bottom:3px;">
      <span class="Italic">
       evam parampara-praptam
       <span>
        <br/>
        <span class="Italic">
         imam rajarsayo viduh
         <span>
          <br/>
          <span class="Italic">
           sa kaleneha mahata
           <span>
            <br/>
            <span class="Italic">
             yogo nastah parantapa
             <span>
              <br/>
             </span>
            </span>
           </span>
          </span>
         </span>
        </span>
       </span>
      </span>
     </div>
     <div class="Synonyms-Section">
      SYNONYMS
     </div>
     <div class="Synonyms-SA">
      <span class="Italic">
       evam
      </span>
      thus;
      <span class="Italic">
       parampar
      </span>
      disciplic succession;
      <span class="Italic">
       prptam
      </span>
      received;
      <span class="Italic">
       imam
      </span>
      this science;
      <span class="Italic">
       rjaraya
      </span>
      the saintly kings;
      <span class="Italic">
       vidu
      </span>
      understood;
      <span class="Italic">
       sa
      </span>
      that knowledge;
      <span class="Italic">
       klena
      </span>
      in the course of time;
      <span class="Italic">
       iha
      </span>
      in this world;
      <span class="Italic">
       mahat
      </span>
      by great;
      <span class="Italic">
       yoga
      </span>
      the science of one's relationship with the Supreme;
      <span class="Italic">
       naa
      </span>
      scattered;
      <span class="Italic">
       parantapa
      </span>
      O Arjuna, subduer of the enemies.
     </div>
     <div class="Titles">
      TRANSLATION
     </div>
     <div class="Translation">
      This supreme science was thus received through the chain of disciplic succession, and the saintly kings understood it in that way. But in course of time the succession was broken, and therefore the science as it is appears to be lost.
     </div>
     <div class="Titles">
      PURPORT
     </div>
     <div class="Purport">
      It is clearly stated that the
      <span class="Italic">
       Gt
      </span>
      was especially meant for the saintly kings because they were to execute its purpose in ruling over the citizens. Certainly
      <span class="Italic">
       Bhagavad-gt
      </span>
      was never meant for the demonic persons, who would dissipate its value for no one's benefit and would devise all types of interpretations according to personal whims. As soon as the original purpose was scattered by the motives of the unscrupulous commentators, there arose the need to reestablish the disciplic succession. Five thousand years ago it was detected by the Lord Himself that the disciplic succession was broken, and therefore He declared that the purpose of the
      <span class="Italic">
       Gt
      </span>
      appeared to be lost. In the same way, at the present moment also there are so many editions of the
      <span class="Italic">
       Gt
      </span>
      (especially in English), but almost all of them are not according to authorized disciplic succession. There are innumerable interpretations rendered by different mundane scholars, but almost all of them do not accept the Supreme Personality of Godhead, Ka, although they make a good business on the words of r Ka. This spirit is demonic, because demons do not believe in God but simply enjoy the property of the Supreme. Since there is a great need of an edition of the
      <span class="Italic">
       Gt
      </span>
      in English, as it is received by the
      <span class="Italic">
       parampar
      </span>
      (disciplic succession) system, an attempt is made herewith to fulfill this great want.
      <span class="Italic">
       Bhagavad-gt-
      </span>
      accepted as it is-is a great boon to humanity; but if it is accepted as a treatise of philosophical speculations, it is simply a waste of time.
     </div>

  <script type="text/javascript">
   $(function(){
				$('.chapter').on('change',function(){
					var _this = $(this);
					var verse;
					var _this_val = _this.val();
					if( _this.val() != "" ){
						$.post(
							 'http://asitis.com/' +"verse.php",
							{
								verse:_this_val
							},
							function(data){
								$('.c-verse').html(data);
							}
						);
					}
				});
				
				$('.schapter').on('change',function(){
					var _this = $(this);
					var verse;
					var _this_val = _this.val();
					if( _this.val() != "" ){
						$.post(
							 'http://asitis.com/' +"verse.php",
							{
								verse:_this_val
							},
							function(data){
								$('.sc-verse').html(data);
							}
						);
					}
				});
				
				$('.btn-top-search').on('click',function(){ 
					var cid = $('.chapter').val();
					var tid = $('.c-verse .verse').val();										
					if( cid != ""  &amp;&amp; tid != ""){
						if(tid==0) {tid='';}
						var url = 'http://asitis.com/' + getPageURL(cid, tid);						
						window.location.href = url;					
					}
				});
				
				$('.sbtn-top-search').on('click',function(){					
					var cid = $('.schapter').val();
					var tid = $('.sc-verse .verse').val();										
					if( cid != ""  &amp;&amp; tid != ""){
						if(tid==0) {tid='';}
						var url = 'http://asitis.com/' + getPageURL(cid, tid);						
						window.location.href = url;					
					}
				});
			});
			
			
			var IS_PRODUCTION = ''; 
			function getPageURL(cid, tid){
				
				if(IS_PRODUCTION){
					return "?cid=" + cid + "&amp;tid=" + tid;	
				} else {
					var url = cid;
					if("" !=tid){
						url = url + "/" + tid + ".html";
					}
					return url;	
				}
			}
