          <h1 class="post-title text-center">
           50 Rules to Live Your Life by
          </h1>
          <div class="post-meta text-center">
           <ul class="list-inline">
            <li class="post-meta-author">
             by
             <a href="http://allswagga.com/blog/author/administrator/">
              allswagga
             </a>
            </li>
            <li class="post-meta-date">
             December 21, 2011
            </li>
            <li class="post-meta-category">
             in
             <a href="http://allswagga.com/blog/category/clever/" rel="category tag">
              Confident &amp; Manly
             </a>
            </li>
            <li>
             0
             <i class="icon icon-chat-messages-03">
             </i>
            </li>

          <!-- .post-meta -->
          <div class="entry">
           <div style="float: right; margin-left: 10px;">
            <a class="twitter-share-button" data-count="vertical" data-url="http://allswagga.com/blog/2011/12/21/50-rules-to-live-your-life-by/" data-via="all_swagga" href="https://twitter.com/share">
             Tweet
            </a>
           </div>
           <p>
            <em>
             Without ambition one starts nothing. Without work one finishes nothing. The prize will not be sent to you. You have to win it.  R. Emerson
            </em>
           </p>
           <p style="text-align: center;">
            <em>
             <a href="http://allswagga.com/blog/2011/12/21/50-rules-to-live-your-life-by/flickr-credit-to-zanthia/" rel="attachment wp-att-10180">
              <img alt="Photo Credit: Zanthia (flickr)" class="size-full wp-image-10180 aligncenter" height="394" src="http://allswagga.com/blog/wp-content/uploads/2011/12/Flickr-Credit-to-Zanthia.jpg" title="Photo Credit: Zanthia (flickr)" width="525"/>
             </a>
            </em>
           </p>
           <p>
            1. Focus on what really counts
            <br/>
            2. You cant please everyone
            <br/>
            3. Choose friends wisely
            <br/>
            4. Tell people what youre really thinking. People are not mind readers.
            <br/>
            5. Understand that you can only control yourself
            <br/>
            6. Learn a new skill this weekend. Repeat!
            <br/>
            7. Identify your preferred form of expression
            <br/>
            8. Self-motivation is not optional. Get some!
            <br/>
            9. Find great ideas then change the context
            <br/>
            10. Wake up claiming the best day ever
            <br/>
            11. Ditch the TV for more time
            <br/>
            12. Do whats in front of you
            <br/>
            13. Set alimit forhaving unfinished tasks
            <br/>
            14. Speak your truth
            <br/>
            15. Spend more time with the right people
            <br/>
            16. Set short deadlines for decisions
            <br/>
            17. Consider your appearance and proper grooming
            <br/>
            18. Reward yourself
            <br/>
            19. Take ownership dont blame others for your situation
            <br/>
            20. Let the little things go
            <br/>
            21. Learn how to negotiate a better deal on everything
            <br/>
            22. Know your top 5 values
            <br/>
            23. Simplify by eliminating anything thats not essential
            <br/>
            24. Start meaningful conversations
            <br/>
            25. Know what you want
            <br/>
            26. Limit what you borrow
            <br/>
            27. Get the important work done first
            <br/>
            28. Never be dissatisfied with what you have
            <br/>
            29. Lay the groundwork for some future fun
            <br/>
            30. Honor your commitments
            <br/>
            31. Dont settle for anything less than excellence
            <br/>
            32. Create a routine and stick to it
            <br/>
            33. Ignore the unimportant
            <br/>
            34. Make it manageable
            <br/>
            35. Be careful when you make comparisons
            <br/>
            36. Character counts!
            <br/>
            37. Never say, thats not my job
            <br/>
            38. Get organized
            <br/>
            39. Schedule small tasks for a given time of the day
            <br/>
            40. Dont underestimate the power of support
            <br/>
            41. Dont browse online as soon as you get home
            <br/>
            42. Get enough sleep
            <br/>
            43. Speak less. Listen more.
            <br/>
            44. Find gym and start going regularly
            <br/>
            45. Pay attention to what youre putting off for tomorrow
            <br/>
            46. Stop caring about mistakes, learn from them
            <br/>
            47. Get your financial house in order
            <br/>
            48. No Cheating!
            <br/>
            49. Do the right thing because its the right thing to do
            <br/>
            50. Know the difference between affirmation and information and be careful receiving too much of either
           </p>
           <p>
            <em>
             If you found this article interesting, please share it with someone on Twitter, Stumbleupon or leave a comment below. Thank You!
            </em>
