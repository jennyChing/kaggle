          <p class="post-content post-content-justify" itemprop="articleBody">
           Honestly, I did not know any of these and I broke one of them.
           <br/>
           I was on Facebook and my cousin posted this and I was like what does this have to say..
           <br/>
           So here is a quick summary:
           <br/>
           <br/>
           <a class="reuuux-url hashtag" href="/search/%23General" rel="nofollow" title="General">
            <em>
             General
            </em>
           </a>
           <a class="reuuux-url hashtag" href="/search/%23Rules" rel="nofollow" title="Rules">
            <em>
             Rules
            </em>
           </a>
           :
           <br/>
           -Chop sticks sticking straight up in a bowl is taboo
           <br/>
           <br/>
           <a class="reuuux-url hashtag" href="/search/%23Filipino" rel="nofollow" title="Filipino">
            <em>
             Filipino
            </em>
           </a>
           :
           <br/>
           -Don`t ask for chopsticks
           <br/>
           <br/>
           <a class="reuuux-url hashtag" href="/search/%23Thai" rel="nofollow" title="Thai">
            <em>
             Thai
            </em>
           </a>
           :
           <br/>
           -
           <a class="reuuux-url hashtag" href="/search/%23chopsticks" rel="nofollow" title="chopsticks">
            <em>
             chopsticks
            </em>
           </a>
           are for noodles
           <br/>
           <br/>
           <a class="reuuux-url hashtag" href="/search/%23Chinese" rel="nofollow" title="Chinese">
            <em>
             Chinese
            </em>
           </a>
           :
           <br/>
           -When eating dim sum, if someone pours you tea, always tap three fingers on the table as a sign of gratitude.
           <br/>
           -When eating a whole fish, don`t flip it over, as that symbolizes the capsizing of a boat.
           <br/>
           -When eating family-style and without a serving utensil, pick up food with the opposite end of your chopsticks; otherwise, you`re essentially double-dipping.
           <br/>
           <br/>
           <a class="reuuux-url hashtag" href="/search/%23Vietnamese" rel="nofollow" title="Vietnamese">
            <em>
             Vietnamese
            </em>
           </a>
           :
           <br/>
           -The minute food hits the table, you should start eating, rather than waiting for everyone`s food to arrive. It`s considered rude to let it get cold.
           <br/>
           -Show respect to the person who spent hours brewing the broth and don`t squirt
           <a class="reuuux-url hashtag" href="/search/%23Sriracha" rel="nofollow" title="Sriracha">
            <em>
             Sriracha
            </em>
           </a>
           or hoisin sauce into
           <a class="reuuux-url hashtag" href="/search/%23pho" rel="nofollow" title="pho">
            <em>
             pho
            </em>
           </a>
           . It`s like putting ketchup in chicken noodle soup. (BUT I LOVE SRIRACHA WITH MY PHO!)
           <br/>
           <br/>
           <a class="reuuux-url hashtag" href="/search/%23Korean" rel="nofollow" title="Korean">
            <em>
             Korean
            </em>
           </a>
           :
           <br/>
           -Don`t ever blow your nose during a meal, even if it`s running like crazy.
           <br/>
           -Don`t ever receive a dish or glass with one hand; always use two or put your left hand under the wrist of the right.
           <br/>
           <br/>
           <a class="reuuux-url hashtag" href="/search/%23Japanese" rel="nofollow" title="Japanese">
            <em>
             Japanese
            </em>
           </a>
           :
           <br/>
           -
           <a class="reuuux-url hashtag" href="/search/%23Ramen" rel="nofollow" title="Ramen">
            <em>
             Ramen
            </em>
           </a>
           has been specifically crafted by the chef as a complete item -- don`t customize it.
           <br/>
