                    <meta content="http://1.bp.blogspot.com/-3lU0aTW5vAI/U6BX3yl5YCI/AAAAAAAAGZQ/8jMi8iEwcPU/s72-c/how+to+ruffle+with+elastic+thread.png" itemprop="image_url"/>

                    <h3 class="post-title entry-title" itemprop="name">
                     Easiest Ruffling EVER with Elastic Thread
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-4854444256930447637" itemprop="articleBody">
                     There are so many different ways to ruffle fabric, but this is 1 easy way to ruffle fabric easily using elastic thread
                     <br/>
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://1.bp.blogspot.com/-3lU0aTW5vAI/U6BX3yl5YCI/AAAAAAAAGZQ/8jMi8iEwcPU/s1600/how+to+ruffle+with+elastic+thread.png" imageanchor="1" style="">
                       <img border="0" src="http://1.bp.blogspot.com/-3lU0aTW5vAI/U6BX3yl5YCI/AAAAAAAAGZQ/8jMi8iEwcPU/s1600/how+to+ruffle+with+elastic+thread.png"/>
                      </a>
                     </div>
                     <br/>
                     I love ruffles...maybe even more than the next person! I love them so 
much but I HATE ruffling fabric. It is so time consuming and I kinda 
like quick projects that I can fit in between being a mom to my 2 
girls...
                     <br/>
                     <br/>
                     Plus, I ruffle a TON for my online
                     <a href="http://www.avisiontoremember.etsy.com/" target="_blank">
                      Shop
                     </a>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="https://www.etsy.com/listing/177950559/custom-ruffled-crib-skirt-dust-ruffle?ref=shop_home_active_1&amp;ga_search_query=elephant%2Bcrib%2Bskirt" target="_blank">
                       <img alt="Ruffled Elephant Crib Skirt" border="0" height="185" src="http://3.bp.blogspot.com/-w8AdBQNi6Zk/U6BVUusyRlI/AAAAAAAAGYk/OgKq__iDzZY/s280/027+(5).JPG" title="Ruffled Elephant Crib Skirt" width="280"/>
                      </a>
                     </div>
                     Ruffle Crib Skirts
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="https://www.etsy.com/listing/111701486/custom-ruffled-crib-skirt-pink-and?ref=shop_home_active_9&amp;ga_search_query=ruffle%2Bcrib%2Bskirt" target="_blank">
                       <img alt="Ruffle Crib Skirt" border="0" height="210" src="http://3.bp.blogspot.com/-dabJn7uUGFE/U6BVfRxXmZI/AAAAAAAAGYs/H8usj7UYmwI/s280/IMG_4259.JPG" title="ruffle crib skirt" width="280"/>
                      </a>
                     </div>
                     <br/>
                     <a href="https://www.etsy.com/listing/158489831/rag-quilt-ruffled-flower-pink-yellow?ref=shop_home_active_22&amp;ga_search_query=ruffle%2Bflower%2Brag%2Bquilt" target="_blank">
                      Ruffled Flower Rag Quilt:
                     </a>
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="https://www.etsy.com/listing/158489831/rag-quilt-ruffled-flower-pink-yellow?ref=shop_home_active_22&amp;ga_search_query=ruffle%2Bflower%2Brag%2Bquilt" target="_blank">
                       <img alt="Ruffle Flower Rag Quilt" border="0" height="421" src="http://1.bp.blogspot.com/-ov8YXgspyUc/U6BVoNLQNGI/AAAAAAAAGY0/ayYj6NdtLnE/s280/062.JPG" width="280"/>
                      </a>
                     </div>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://2.bp.blogspot.com/-GFFHYUvd-go/U6BVuOZnXNI/AAAAAAAAGY8/4eOGeNyEmuU/s1600/010.JPG" imageanchor="1" style="">
                       <br/>
                      </a>
                     </div>
                     <br/>
                     <br/>
                     <br/>
                     <br/>
                     So here is how I have made ruffling as easy as a straight stitch:
                     <br/>
                     <br/>
                     What you will need:
                     <br/>
                     fabric to ruffle
                     <br/>
                     elastic thread
                     <br/>
                     <br/>
                     Hand wind your bobbin with the elastic thread. I don't pull it tight when I am winding it though.
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://2.bp.blogspot.com/-NMlxh2hzv4s/Tc2vZA0QcjI/AAAAAAAADF4/JW2ILK9gltA/s1600/DSC_1360.JPG" imageanchor="1" style="">
                       <img border="0" height="186" src="http://2.bp.blogspot.com/-NMlxh2hzv4s/Tc2vZA0QcjI/AAAAAAAADF4/JW2ILK9gltA/s280/DSC_1360.JPG" width="280"/>
                      </a>
                     </div>
                     <br/>
                     Now set your sewing machine to a lower tension number
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://3.bp.blogspot.com/-7uYJfJFt1EU/Tc2vbz5XVqI/AAAAAAAADGA/av9UzGNG7tY/s1600/DSC_1363.JPG" imageanchor="1" style="">
                       <img border="0" height="186" src="http://3.bp.blogspot.com/-7uYJfJFt1EU/Tc2vbz5XVqI/AAAAAAAADGA/av9UzGNG7tY/s280/DSC_1363.JPG" width="280"/>
                      </a>
                     </div>
                     And make your stitch as long as possible
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://1.bp.blogspot.com/-gRqcirgD8yg/Tc2vdLMgRMI/AAAAAAAADGE/DN2PBgaxREU/s1600/DSC_1364.JPG" imageanchor="1" style="">
                       <img border="0" height="186" src="http://1.bp.blogspot.com/-gRqcirgD8yg/Tc2vdLMgRMI/AAAAAAAADGE/DN2PBgaxREU/s280/DSC_1364.JPG" width="280"/>
                      </a>
                      <a href="http://2.bp.blogspot.com/-Ov_fSy_Byjg/Tc2vaeppbVI/AAAAAAAADF8/W8FbAXc5wds/s1600/DSC_1362.JPG" imageanchor="1" style="">
                       <img border="0" height="186" src="http://2.bp.blogspot.com/-Ov_fSy_Byjg/Tc2vaeppbVI/AAAAAAAADF8/W8FbAXc5wds/s280/DSC_1362.JPG" width="280"/>
                      </a>
                     </div>
                     Then sew....make sure to backstitch (or secure stitch) when you start and finish.
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://4.bp.blogspot.com/-Mj6pbOu8dR4/Tc2vey7OjOI/AAAAAAAADGI/3Pl8SefzMIY/s1600/DSC_1366.JPG" imageanchor="1" style="">
                       <img border="0" height="186" src="http://4.bp.blogspot.com/-Mj6pbOu8dR4/Tc2vey7OjOI/AAAAAAAADGI/3Pl8SefzMIY/s280/DSC_1366.JPG" width="280"/>
                      </a>
                     </div>
                     Then you will get some awesome ruffle fabric...
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://4.bp.blogspot.com/-d0zd4KLaY6Y/Tc2vgRIbyLI/AAAAAAAADGM/WGnaCN-xRfs/s1600/DSC_1368.JPG" imageanchor="1" style="">
                       <img border="0" height="186" src="http://4.bp.blogspot.com/-d0zd4KLaY6Y/Tc2vgRIbyLI/AAAAAAAADGM/WGnaCN-xRfs/s280/DSC_1368.JPG" width="280"/>
                      </a>
                     </div>
                     That fabric is going to be for some really adorable skirts for my girls similar to this:
                     <br/>
                     <a href="http://3.bp.blogspot.com/-MQOn7Dodrs8/U6BXyo0tf4I/AAAAAAAAGZI/bNdO4pktBeo/s1600/how+to+ruffle+with+elastic+thread+square.jpg" imageanchor="1" style="">
                      <img border="0" height="280" src="http://3.bp.blogspot.com/-MQOn7Dodrs8/U6BXyo0tf4I/AAAAAAAAGZI/bNdO4pktBeo/s280/how+to+ruffle+with+elastic+thread+square.jpg" width="280"/>
                     </a>
                     <a href="http://1.bp.blogspot.com/-GQHmqKnC8k8/Tc2yEdq1q6I/AAAAAAAADGQ/EVMxnvYi-Og/s1600/011.JPG" imageanchor="1" style="">
                     </a>
                     <br/>
                     <a href="http://www.mylivesignature.com/" target="_blank">
                      <img src="http://signatures.mylivesignature.com/54488/38/0DB02E7D6556925F4C5AC3372AE31CF5.png" style="background: transparent; border: 0 !important;"/>
                     </a>
