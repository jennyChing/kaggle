        <p>
         Privacy Manager is a Chrome extension which allows quickly tweak your privacy settings. In a small window you can quickly setup the basic, but also more advanced options. You can control the downloads, cleaning startup options, delete junk files, manage cookies and network traffic. All this in very cool manner  from a small pop-up window.
        </p>
        <p>
         Chrome settings are otherwise burried down in the settings page, in an advanced section. And when youre at the settings page, you usually have to scroll down quite a bit. This extension handles is quicker, because the options are present as small switches which needs to be turned On or Off.
        </p>
        <p>
         Also you can setup what to clean on startup, like Clear browser cookies, Clear websites appcaches, browser cache, browser download list and much more..
        </p>
        <p>
         <img alt="PrivacyManager - Chrome Extension" class="alignnone size-full wp-image-32977" data-lazy-src="http://www.vladan.fr/wp-content/uploads/images/PrivacyManager.jpg" height="483" src="data:image/gif;base64,R0lGODdhAQABAPAAAP///wAAACwAAAAAAQABAEACAkQBADs=" width="537"/>
        </p>
        <noscript>
         <img alt="PrivacyManager - Chrome Extension" class="alignnone size-full wp-image-32977" height="483" src="http://www.vladan.fr/wp-content/uploads/images/PrivacyManager.jpg" width="537"/>
        </noscript>
        <h3>
         You can control the following:
        </h3>
        <ul>
         <li>
          blocks third-party sites from setting cookies.
         </li>
         <li>
          automatically fill in forms settings management.
         </li>
         <li>
          Omnibox search requests management.
         </li>
         <li>
          Enable safe browsing in case if the option is disabled by another extension (if safe    browsing is enabled Chrome does its best to protect you from phishing and malware).
         </li>
         <li>
          Search suggestion management.
         </li>
         <li>
          Enable or disable correct spelling service.
         </li>
         <li>
          Translate service switch.
         </li>
         <li>
          Hyperlink audition management.
         </li>
         <li>
          Manage whether chrome must allow sending referer headers with requests.
         </li>
         <li>
          Network prediction management.
         </li>
         <li>
          Alternate error pages management.
         </li>
        </ul>
        <h3>
         Startup items clean-up control:
        </h3>
        <ul>
         <li>
          Clear browser cookies.
         </li>
         <li>
          Clear websites appcaches.
         </li>
         <li>
          Clear browser cache.
         </li>
         <li>
          Clear browser download list.
         </li>
         <li>
          Clear browser file system.
         </li>
         <li>
          Clear browser stored data.
         </li>
         <li>
          Clear browser history.
         </li>
         <li>
          Clear websites IndexedDB data.
         </li>
         <li>
          Clear websites local storage data.
         </li>
         <li>
          Clear plugins data.
         </li>
         <li>
          Clear stored passwords.
         </li>
         <li>
          Clear websites WebSQL data.
         </li>
        </ul>
        <p>
         I discovered this extension by seeking quicker way to setup basic security options in Chrome and found it quicker to access through this small pop-up window instead scrolling down the whole Chrome settings page. Neat.. -:)
        </p>
