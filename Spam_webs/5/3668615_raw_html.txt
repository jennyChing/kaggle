   <section class="ispatial" id="sect-tech-specs">
    <div class="section-inner clearfix">
     <h2>
      Technical Specifications
     </h2>
     <div class="column3">
      <strong>
       Front-end Services
      </strong>
      <br/>
      Javascript
      <ul>
       <li>
        Libraries that can be customized and built upon Message queue
       </li>
       <li>
        Integration to allow users to receive real time updates for session sharing and tracked data as well as communicate with other systems Navigation framework
       </li>
       <li>
        Utilized by permalinks, sharing, presentations, and zone alerts; Application state encoded, broadcase, and decoded and applied to the UI Event framework
       </li>
       <li>
        Classes can publish and subscribe with elements automatically baked into available RPC controllers
       </li>
      </ul>
     </div>
     <div class="column3">
      <strong>
       Back-end Services
      </strong>
      <br/>
      Django
      <ul>
       <li>
        Utilizes object-relational mapping to translate API calls to back-end operations WFS
       </li>
       <li>
        Supports open standard, generates complex queries, and outputs to GML, KML, GeoJSON, and PGDUMP Deferred tasks
       </li>
       <li>
        Utilizes the AMQP protocol to offload large tasks to dedicated workers Ingest service
       </li>
       <li>
        Geospatial Data Abstraction Library to provide 1-step ingestion workflow&lt;
       </li>
      </ul>
     </div>
     <div class="column3">
      <strong>
       Browser Requirements:
      </strong>
      <ul>
       <li>
        Free Google Earth plug-in to support 3D visualization
       </li>
       <li>
        Internet Explorer 8.0+*, Firefox 17+, Chrome 21+
       </li>
      </ul>
      <p>
       <strong>
        Server/Hosting Requirements**
       </strong>
      </p>
      <ul>
       <li>
        Google Earth Server or Google Maps/Maps Engine for Business
       </li>
       <li>
        Postgres/PostGIS 9.1.x or higher
       </li>
       <li>
        Supports Django-supported database types
       </li>
       <li>
        Python 2.7.3 (included)
       </li>
       <li>
        RedHat Enterprise Linux or CentOS Unix 6.4  (6.5 preferred)
       </li>
      </ul>
      <p>
       *Internet Explorer 7.0 supported, but with reduced capability
       <br/>
       **Cloud hosting available
      </p>
     </div>
