function updateCountdowns() {
	editbuttons = $('button.comment_edit:visible');
	if (editbuttons.length) {
		editbuttons.each(function() {
			timeremaining = $(this).data('editwindow-remaining');
			if (--timeremaining &lt; 0) {
				$(this).text('edit timer expired');
				$(this).fadeOut(); //no need for further time update; we only poll visibles
			} else {
				$(this).text('edit ('+timeremaining+' seconds remaining)');
				$(this).data('editwindow-remaining', timeremaining);
			}
		})
	} else {
		//no reason to keep this running when there's nothing to update
		clearInterval(countdownTimer);
	}
}

var countdownTimer = null;

$(document).bind('loginStatus', function(e,d){
	servertime = Number(d.unixtime);
	editing_window_seconds = Number(d.editing_window_seconds);
	if (d.logged_in) {
		userID = d.user_id;
		var handle = $("#user_login_control")
		.html('&lt;p&gt;You are currently logged in as: &lt;strong&gt;&lt;a href="http://reason.com/users/manage" class="handle"&gt;&lt;/a&gt;&lt;/strong&gt;. &lt;a href="http://reason.com/users/logout"&gt;&lt;span&gt;Sign Out&lt;/span&gt;&lt;/a&gt;&lt;/p&gt;')
		.find('.handle')
		handle.text(d.handle)
		handle.attr('href', handle.attr('href')+'/'+userID);
		//look for all the editable comments by this user
		$('div#commentcontainer li[data-user-id='+userID+']').each(function( index ) {
			commenttimestamp = Number($(this).data('comment-timestamp'));
			elapsedtime = servertime - commenttimestamp;
			if (elapsedtime &lt; editing_window_seconds) {
				secondsremaining = editing_window_seconds - elapsedtime;
				$(this).find('button.comment_edit').show().data('editwindow-remaining', secondsremaining);
			}
		});
		countdownTimer = setInterval(updateCountdowns, 1E3);
	} else {
		$("#user_login_control").html('&lt;a href="http://reason.com/users/login"&gt;&lt;span&gt;Login&lt;/span&gt;&lt;/a&gt; or &lt;a href="http://reason.com/users/register"&gt;&lt;span&gt;Sign-up&lt;/span&gt;&lt;/a&gt;');

          Massachusetts Sen. Elizabeth Warrena leading progressive populist, possible Democratic presidential candidate, self-proclaimed champion of the poor, and enemy of greedy corporations (not really, as you will see)supports the Export-Import Bank.
         </p>
         <p>
          That's right: The woman best known for demonizing big businesses nevertheless wants to maintain an outlandishly generous subsidy package for them.
         </p>
         <p>
          Warren's support for Ex-Im was revealed Friday after the Heritage Foundation reached out to her regarding a possible partnership on the issue. Libertarians and Tea Party conservatives are noted opponents of the subsidy, which they see as crony capitalism that helps politically-connected businesses cheat the free market. Liberty-inclined Republicans are working with the left on a host of issues, including NSA spying and marijuana legalizationwhy not make common cause with Warren on corporatism as well?
         </p>
         <p>
          Looks like the joke is on Heritage. From
          <a href="http://go.bloomberg.com/political-capital/2014-07-18/warren-backs-ex-im-bank-heritage-invites/">
           Bloomberg
          </a>
          :
         </p>
         <blockquote>
          <p>
           Senator Warren believes that the Export-Import Bank helps create American jobs and spur economic growth, but recognizes that there is room for improvement in the banks operations, Warren spokesman Lacey Rose tells us in an e-mail. She looks forward to reviewing re-authorization legislation if and when it is introduced.
          </p>
         </blockquote>
         <p>
          The irony of Warren rejecting an offer from the right to fight Ex-Im together was not lost on
          <em>
           <a href="http://washingtonexaminer.com/elizabeth-warren-loves-bank-subsidies/article/2551022?custom_click=rss&amp;utm_source=twitterfeed&amp;utm_medium=twitter">
            The Washington Examiner
           </a>
          </em>
          's Tim Carney:
         </p>
         <blockquote>
          <p>
           Democratic Sen.
           <a href="http://washingtonexaminer.com/section/elizabeth-warren">
            Elizabeth Warren
           </a>
           loves to shoot barbs at
           <a href="http://washingtonexaminer.com/section/wall-street">
            Wall Street
           </a>
           . She also enjoys forcing taxpayers to absorb Wall Street's risks while the banks pocket the profits.
          </p>
          <p>
           ... At Ex-Im's annual conference, one Wall Streeter described Ex-Im's loan guarantees to me as "free money." Is Elizabeth Warren really fine with free money to Wall Street?
          </p>
         </blockquote>

would be unthinkable in America today.

The Decade of Expansive Government, and the 2009 crisis just sort
