         <p>
          How long has it been since you last saw a new
          <strong>
           feature
          </strong>
          on Facebook? However lavish may the technological feature be; with the passage of time you get bored of it and start looking for something new and exciting. And, it is for this reason that the experts always remain on their toes to innovate something that can render you the inexhaustibly rich experience. Recently, Facebook has introduced a much talked about feature of
          <a href="https://www.facebook.com/thanks" onclick="__gaTracker('send', 'event', 'outbound-article', 'https://www.facebook.com/thanks', 'Say Thanks');">
           <strong>
            Say Thanks
           </strong>
           
          </a>
          . Now you can add some sweet and memorable flavor to your friendship or relationship by saying thanks in a very special and personalized way.
         </p>
         <p>
          <img alt="saythanks" class="wp-image-5089 aligncenter" height="258" src="http://socialchamps.com/wp-content/uploads/2014/12/saythanks.png" width="383"/>
         </p>
         <p>
          How does this
          <strong>
           say thanks
          </strong>
          feature function? Well, if you are a social media lover, or a regular user of the internet, it would not be a tough task to use this new feature. You can quite easily and conveniently create a personalized video by visiting the facebook.com/thanks. When you choose or click on a friend, you can see the video preview being generated by Facebook. Here you have the options of selecting different themes. Also, you can edit the photos that represent your relationship or friendship. Once you feel that the video is ready or satisfactory enough, you can share it by clicking share.
         </p>
         <p>
          What makes it still more fascinating is the feature of adding a personalized message. You can write whatever message you would like to share. Once you click share; your friend will be tagged and you can see the posted video on your timeline. Also, the same will be visible on the timeline of your friend. Whether you access the Facebook on your desktop or on mobile phone; the say thanks facility is available on both. At present, it is available in
          <strong>
           English, Indonesian, German, French, Turkish, Italian, Spanish and Portuguese languages.
          </strong>
         </p>
         <p>
         </p>
         <p>
          <iframe allowfullscreen="" frameborder="0" height="338" mozallowfullscreen="" src="//player.vimeo.com/video/111578650" title="Introducing Say Thanks" webkitallowfullscreen="" width="600">
          </iframe>
         </p>
         <p>
          The people accessing Facebook in an irregular manner may not be still aware of this new feature, and you have the opportunity to surprise them by sending a personalized video and message through
          <strong>
           <a href="https://www.facebook.com/thanks" onclick="__gaTracker('send', 'event', 'outbound-article', 'https://www.facebook.com/thanks', 'Say Thanks');">
            Say Thanks
           </a>
          </strong>
          . However, it is not for surprising anyone. It has been created with a view to help you in adding new flavor to your
          <strong>
           friendship or relationship
          </strong>
          , and you should surely make use of it as and when you feel it is suitable.
         </p>

function getga_has_universal_event_tracking()
{
	var scripts = document.getElementsByTagName('script');

	for (var i = scripts.length; i--;)
	{
	    if (scripts[i].src.indexOf(String.fromCharCode(103,111,111,103,108,101,45,97,110,97,108,121,116,105,99,115,46,99,111,109,47,97,110,97,108,121,116,105,99,115,46,106,115)) &gt; 0)
	    {
	    	return true;
	    }
	}

	return false;
}

function getga_add_event_tracking(has_universal_event_tracking, getga_selector, getga_event, getga_category, getga_action_label, getga_label)
{
	jQuery(function($){

			var NA = 'undefined';

			
			if(getga_label.indexOf('{ITEM_TITLE}') &gt; -1)
			{
				getga_label = getga_label.replace("{ITEM_TITLE}", ($(this).attr('title') !== undefined ? $(this).attr('title') : NA));
			}

			if(getga_label.indexOf('{PAGE_URL}') &gt; -1)
			{
				getga_label = getga_label.replace("{PAGE_URL}", location.href);
			}

			if(getga_label.indexOf('{PAGE_RELATIVE_URL}') &gt; -1)
			{
				getga_label = getga_label.replace("{PAGE_RELATIVE_URL}", location.pathname);
			}

			if(getga_label.indexOf('{LINK_URL}') &gt; -1)
			{
				getga_label = getga_label.replace("{LINK_URL}", ($(this).attr('href') !== undefined ? $(this).attr('href') : NA));
			}

			if(getga_label.indexOf('{LINK_RELATIVE_URL}') &gt; -1)
			{
				getga_label = getga_label.replace("{LINK_RELATIVE_URL}", ($(this).attr('href') !== undefined ? $(this).attr('href').replace(/^(?:\/\/|[^\/]+)*\//, '/') : NA));
			}

			{
				getga_label = getga_label.replace("{IMAGE_SRC}", ($(this).attr('src') !== undefined ? $(this).attr('src') : NA));
			}

			if(getga_label.indexOf('{IMAGE_ALT}') &gt; -1)
			{
				getga_label = getga_label.replace("{IMAGE_ALT}", ($(this).attr('alt') !== undefined ? $(this).attr('alt') : NA));
			}

			if(getga_label.indexOf('{TAG_HTML}') &gt; -1)
			{
				getga_label = getga_label.replace("{TAG_HTML}", $(this).html().replace(/(&lt;([^&gt;]+)&gt;)/ig,'').replace(/(\r\n|\n|\r)/gm,'').replace(/\s{2,}/g, ' '));
			}

			if(has_universal_event_tracking)

									}
				catch(err)

									}
			}

					_gaq.push(['_trackEvent', getga_category, getga_action_label, getga_label]);
									}
				catch(err)
				{
									}
			}
		});

	});
}
