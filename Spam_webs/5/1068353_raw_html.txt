          <!-- THE POST 102597 -->
          <div class="postcolor">
           This a is a very useful way to blend images into sigs or other pics.
           <br/>
           <br/>
           1) Open up your sig, i'm using this one.
           <br/>
           <br/>
           <img alt="user posted image" border="0" src="http://img.photobucket.com/albums/v380/alexreltonb/newabstractsigcopy.jpg"/>
           <br/>
           <br/>
           2) Then open up the picture you want to blend in and drag it onto your sig.
           <br/>
           <br/>
           <img alt="user posted image" border="0" src="http://img.photobucket.com/albums/v380/alexreltonb/tut1.jpg"/>
           <br/>
           <br/>
           3) As you can see it's far too big, so I am going to resize it by going to:
           <br/>
           Edit &gt; Transform &gt; Scale. (Remember to hold down shift when resizing
           <br/>
           so you don't distort the pic.)
           <br/>
           <br/>
           <img alt="user posted image" border="0" src="http://img.photobucket.com/albums/v380/alexreltonb/tut2.jpg"/>
           <br/>
           <br/>
           4) Grab your lasso tool, set the feather to 30px and make a rough outline of your picture.
           <br/>
           <br/>
           <img alt="user posted image" border="0" src="http://img.photobucket.com/albums/v380/alexreltonb/tut3.jpg"/>
           <br/>
           <br/>
           You'll notice that your selection will get smaller and rounder like the one above.
           <br/>
           <br/>
           5) Right, once you've done that, make a layer mask on the layer your picture's on.
           <br/>
           <br/>
           <img alt="user posted image" border="0" src="http://img.photobucket.com/albums/v380/alexreltonb/tut4.jpg"/>
           <br/>
           <br/>
           6) Your sig should look similar to this:
           <br/>
           <br/>
           <img alt="user posted image" border="0" src="http://img.photobucket.com/albums/v380/alexreltonb/tut5.jpg"/>
           <br/>
           <br/>
           7) In my case the colours clash rather horribly so I am going to set the picture layer to 'Luminosity'
           <br/>
           <br/>
           8) And there you have it! Add some text and a border and see what you get!
           <br/>
           <br/>
           <img alt="user posted image" border="0" src="http://img.photobucket.com/albums/v380/alexreltonb/tut6.jpg"/>
           <br/>

             <!--QuoteEBegin-->
             This a is a very useful way to blend images into sigs or other pics.
             <br/>
             <br/>
             1) Open up your sig, i'm using this one.
             <br/>
             <br/>
             <img alt="user posted image" border="0" src="http://img.photobucket.com/albums/v380/alexreltonb/newabstractsigcopy.jpg"/>
             <br/>
             <br/>
             2) Then open up the picture you want to blend in and drag it onto your sig.
             <br/>
             <br/>
             <img alt="user posted image" border="0" src="http://img.photobucket.com/albums/v380/alexreltonb/tut1.jpg"/>
             <br/>
             <br/>
             3) As you can see it's far too big, so I am going to resize it by going to:
             <br/>
             Edit &gt; Transform &gt; Scale. (Remember to hold down shift when resizing
             <br/>
             so you don't distort the pic.)
             <br/>
             <br/>
             <img alt="user posted image" border="0" src="http://img.photobucket.com/albums/v380/alexreltonb/tut2.jpg"/>
             <br/>
             <br/>
             4) Grab your lasso tool, set the feather to 30px and make a rough outline of your picture.
             <br/>
             <br/>
             <img alt="user posted image" border="0" src="http://img.photobucket.com/albums/v380/alexreltonb/tut3.jpg"/>
             <br/>
             <br/>
             You'll notice that your selection will get smaller and rounder like the one above.
             <br/>
             <br/>
             5) Right, once you've done that, make a layer mask on the layer your picture's on.
             <br/>
             <br/>
             <img alt="user posted image" border="0" src="http://img.photobucket.com/albums/v380/alexreltonb/tut4.jpg"/>
             <br/>
             <br/>
             6) Your sig should look similar to this:
             <br/>
             <br/>
             <img alt="user posted image" border="0" src="http://img.photobucket.com/albums/v380/alexreltonb/tut5.jpg"/>
             <br/>
             <br/>
             7) In my case the colours clash rather horribly so I am going to set the picture layer to 'Luminosity'
             <br/>
             <br/>
             8) And there you have it! Add some text and a border and see what you get!
             <br/>
             <br/>
             <img alt="user posted image" border="0" src="http://img.photobucket.com/albums/v380/alexreltonb/tut6.jpg"/>
             <br/>
