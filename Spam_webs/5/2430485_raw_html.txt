       <header class="entry-header">
        <h1 class="entry-title" itemprop="headline">
         Family Night Grilling Recipe: Smoked Bacon Cheddar Scalloped Potatoes on the Grill
        </h1>
        <p class="entry-meta">
         <time class="entry-time" datetime="2014-05-06T07:00:04+00:00" itemprop="datePublished">
          May 6, 2014
         </time>
         By:
         <span class="entry-author" itemprop="author" itemscope="itemscope" itemtype="http://schema.org/Person">
          <span class="entry-author-name" itemprop="name">
           Mel Lockcuff
          </span>
         </span>
         <span class="entry-comments-link">
          <a href="http://adventuresofmel.com/2014/05/family-night-grilling-recipe-smoked-bacon-cheddar-scalloped-potatoes-on-the-grill.html#comments">
           29 Comments
          </a>
         </span>
        </p>
       </header>
       <div class="entry-content" itemprop="text">
        <input class="jpibfi" type="hidden"/>
        <p>
         <img alt="Disclosure for Grilling Recipe #shop" class="aligncenter size-full wp-image-23041" height="45" src="http://adventuresofmel.com/wp-content/uploads/2014/05/Disclosure-for-Grilling-Recipe-shop.png" width="700"/>
         Ah, grilling season. how I love thee, let me count the ways! It would seem that in the last week here in Northwest Arkansas, weve gone from pretty chilly to completely hot in a nanosecond. But hey, Im not complaining; I love this time of year because the grill is uncovered, the table, chairs, and umbrella are all set up on the back deck, the pool is open, and were totally ready for family nights out on the back deck, grilling tools in hand.
         <img alt="Smoked Bacon Cheddar Scalloped Potatoes on the Grill - Recipe from MamaBuzz #shop" class="aligncenter size-full wp-image-23047" height="546" src="http://adventuresofmel.com/wp-content/uploads/2014/05/Smoked-Bacon-Cheddar-Scalloped-Potatoes-on-the-Grill-Recipe-from-MamaBuzz-shop.png" width="700"/>
        </p>
        <p>
         Now I would never have thought of grilling scalloped potatoes until a couple of nights ago when I was trying to come up with something different and unique for a family night grilling experience. But after making this recipe, I will never make scalloped potatoes in the oven again. Oh.my.word. I have never had scalloped potatoes with so much flavor and so much deliciousness. They were so creamy, you cant even imagine. I wish the photos could do the taste of these scalloped potatoes justice, but this is definitely a recipe you have to make for yourself to experience the flavors. So good!
        </p>
        <p>
         We started by shopping for
         <a href="http://www.johnsonville.com/home.html" rel="nofollow" target="_blank">
          Johnsonville
         </a>
         Butcher Shop Style Sausage at Walmart.
         <img alt="Shopping for Johnsonville Butcher Shop Style Sausages in Walmart #shop" class="aligncenter size-full wp-image-23044" height="525" src="http://adventuresofmel.com/wp-content/uploads/2014/05/Shopping-for-Johnsonville-Butcher-Shop-Style-Sausages-in-Walmart-shop.jpg" width="700"/>
         When wed found the two flavors we wanted, Bacon Cheddar and Smoked, we began to create our recipe in the kitchen and ended our recipe out on the deck.
         <img alt="Grilling Johnsonville Butcher Shop Style Sausages for Scalloped Potatoes #shop" class="aligncenter size-full wp-image-23043" height="555" src="http://adventuresofmel.com/wp-content/uploads/2014/05/Grilling-Johnsonville-Butcher-Shop-Style-Sausages-for-Scalloped-Potatoes-shop.jpg" width="700"/>
         The sausages sizzled and fizzled as they cooked on the grill.
         <img alt="Grilling Johnsonville Butcher Shop Style Sausages #shop" class="aligncenter size-full wp-image-23042" height="552" src="http://adventuresofmel.com/wp-content/uploads/2014/05/Grilling-Johnsonville-Butcher-Shop-Style-Sausages-shop.jpg" width="700"/>
         <br/>
         The scent wafting through the air was enough to make the neighbors a bit jealous.
         <img alt="Smoking the Scalloped Potatoes on the Grill #shop" class="aligncenter size-full wp-image-23048" height="532" src="http://adventuresofmel.com/wp-content/uploads/2014/05/Smoking-the-Scalloped-Potatoes-on-the-Grill-shop.jpg" width="700"/>
         While the recipe is a grilling recipe, I might venture to say this could also be a simple comfort food recipe. The potatoes, sausages, cream and Cheddar just all come together and make for a mesmerizing mix of flavors that will have the entire family asking for seconds.
         <img alt="Smoked and Grilled Scalloped Potatoes with Sausages #shop" class="aligncenter size-full wp-image-23045" height="542" src="http://adventuresofmel.com/wp-content/uploads/2014/05/Smoked-and-Grilled-Scalloped-Potatoes-with-Sausages-shop.jpg" width="700"/>
         Since the cooking ends on the deck, everyones outside enjoying the warm air, getting the table all set up, getting ready to enjoy some good eats.
        </p>
        <p>
         Below is a printable recipe. I truly hope you enjoy. Serve with a salad on the side, and youre all set. Its so easy, I ask myself why I never thought of this before. Enjoy!
         <img alt="Smoked Bacon Cheddar Scalloped Potatoes Grilling Recipe #shop" class="aligncenter size-full wp-image-23046" height="543" src="http://adventuresofmel.com/wp-content/uploads/2014/05/Smoked-Bacon-Cheddar-Scalloped-Potatoes-Grilling-Recipe-shop.jpg" width="700"/>
        </p>
        <div class="easyrecipe" itemscope="" itemtype="http://schema.org/Recipe">
         <link href="http://adventuresofmel.com/wp-content/uploads/2014/05/Disclosure-for-Grilling-Recipe-shop.png" itemprop="image"/>
         <div class="ERSSavePrint">
          <span class="ERSPrintBtnSpan">
           <a class="ERSPrintBtn" href="http://adventuresofmel.com/easyrecipe-print/23014-0/" rel="nofollow" target="_blank">
            Print
           </a>
          </span>
         </div>
         <div class="ERSName" itemprop="name">
          Family Night Grilling Recipe: Smoked Bacon Cheddar Scalloped Potatoes on the Grill
         </div>
         <div class="ERSClear">
         </div>
         <div class="ERSDetails">
          <div class="ERSClear">
          </div>
          <div class="ERSTimes">
          </div>
          <div class="ERSClear">
          </div>
          <div class="ERSClear">
          </div>
         </div>
         <div class="ERSIngredients">
          <div class="ERSIngredientsHeader ERSHeading">
           Ingredients
          </div>
          <ul>
           <li class="ingredient" itemprop="ingredients">
            4 lbs. red potatoes, peeled and sliced
           </li>
           <li class="ingredient" itemprop="ingredients">
            3 cups milk
           </li>
           <li class="ingredient" itemprop="ingredients">
            2 cups heavy whipping cream
           </li>
           <li class="ingredient" itemprop="ingredients">
            1 tsp. salt
           </li>
           <li class="ingredient" itemprop="ingredients">
             tsp. pepper
           </li>
           <li class="ingredient" itemprop="ingredients">
            1 lb. extra sharp white Cheddar
           </li>
           <li class="ingredient" itemprop="ingredients">
            1 pkg. Johnsonville Butcher Style Bacon Cheddar Sausages
           </li>
           <li class="ingredient" itemprop="ingredients">
            1 pkg. Johnsonville Butcher Style Smoked Sausages
           </li>
          </ul>
          <div class="ERSClear">
          </div>
         </div>
         <div class="ERSInstructions">
          <div class="ERSInstructionsHeader ERSHeading">
           Instructions
          </div>
          <ol>
           <li class="instruction" itemprop="recipeInstructions">
            Wash, peel, and slice potatoes, placing them into a Dutch oven pan. Add milk, whipping cream, salt, and pepper to potatoes, stirring well. Cook on medium heat until potatoes begin to soften, about 25-30 minutes, stirring or scraping bottom and sides of pan often to avoid scalding.
           </li>
           <li class="instruction" itemprop="recipeInstructions">
            Remove from heat.
           </li>
           <li class="instruction" itemprop="recipeInstructions">
            Grill sausages according to package directions. Remove from grill, and slice each sausage into multiple slices. Add to potatoes, mixing together.
           </li>
           <li class="instruction" itemprop="recipeInstructions">
            Fold half of Cheddar cheese into potato mixture. Add rest of cheese onto top of scalloped potatoes, and place the entire Dutch oven pan on the grill. We used cherry wood pieces in the bottom of our charcoal grill for added smoke flavoring. Do not cover your pan, but do place the cover on the grill. You'll need to let the potatoes grill/smoke for about 25 minutes, stirring often.
           </li>
           <li class="instruction" itemprop="recipeInstructions">
            Remove from grill. Serve and enjoy!
           </li>
          </ol>
          <div class="ERSClear">

          validEmail = /.+@.+\..+/;

        var field = fields[i],
            type  = this.fieldType(field);
        if(type == 'checkboxes' || type == 'radio_buttons') {
          this.checkboxAndRadioValidation(field);
        } else {
          this.textAndDropdownValidation(field, type);
        }
      }
    },

    fieldType: function(field) {
      var type = field.querySelectorAll('.field_type');

      if(type.length &gt; 0) {
        return type[0].getAttribute('data-field-type');
      } else if(field.className.indexOf('checkgroup') &gt;= 0) {
        return 'checkboxes';
      } else {
        return 'text_field';
      }
    },

    checkboxAndRadioValidation: function(field) {
      var inputs   = field.getElementsByTagName('input'),
          selected = false;

      for(var i = 0; i &lt; inputs.length; ++i) {
        var input = inputs[i];
        if((input.type == 'checkbox' || input.type == 'radio') &amp;&amp; input.checked) selected = true;
      }

      if(selected) {
        field.className = field.className.replace(/ invalid/g, '');
      } else {
        if(field.className.indexOf('invalid') == -1) field.className += ' invalid';
        this.isValid = false;
      }
    },

    textAndDropdownValidation: function(field, type) {
      var inputs = field.getElementsByTagName('input');

      for(var i = 0; i &lt; inputs.length; ++i) {
        var input = inputs[i];
        if(input.name.indexOf('signup') &gt;= 0) {
          if(type == 'text_field') {
            this.textValidation(input);
          } else {
            this.dropdownValidation(field, input);
          }
        }
      }
      this.htmlEmbedDropdownValidation(field);
    },

    textValidation: function(input) {
      if(input.id == 'signup_email') return;

      var val = input.value;
      if(val == '') {
        this.textFieldError(input);
        this.isValid = false;
      } else {
        this.removeTextFieldError(input)
      }
    },

    dropdownValidation: function(field, input) {
      var val = input.value;
      if(val == '') {
        if(field.className.indexOf('invalid') == -1) field.className += ' invalid';
        this.onSelectCallback(input);
        this.isValid = false;
      } else {
        field.className = field.className.replace(/ invalid/g, '');
      }
    },

    htmlEmbedDropdownValidation: function(field) {
      var dropdowns = field.querySelectorAll('.mimi_html_dropdown');

      for(var i = 0; i &lt; dropdowns.length; ++i) {
        var dropdown = dropdowns[i],
            val      = dropdown.value;
        if(val == '') {
          if(field.className.indexOf('invalid') == -1) field.className += ' invalid';
          this.isValid = false;
          dropdown.onchange = this.validate.bind(this);
        } else {
          field.className = field.className.replace(/ invalid/g, '');
        }
      }
    },

    textFieldError: function(input) {
      input.className   = 'required invalid';
      input.placeholder = input.getAttribute('data-required-field');
    },

    removeTextFieldError: function(input) {
      input.className   = 'required';
      input.placeholder = '';

    onSelectCallback: function(input) {

        }
      }
    },

    },

    setFormClassName: function() {
      var name = this.form.className;

      if(this.isValid) {
        return name.replace(/\s?mimi_invalid/, '');
      } else {
        if(name.indexOf('mimi_invalid') == -1) {
          return name += ' mimi_invalid';
        } else {
          return name;
        }
      }
    },

      var _this = this;
      window[this.callbackName] = function(response) {

	var save_ret_cart_msg = $('div.eStore_save_retrieve_cart_message');

		save_ret_cart_msg.html("").append("Processing Your Request...");		
		var dataString = 'eStore_cart_action=eStore_save_cart';		
		$.ajax({
			type: "POST",
			url: process_script_url,
			data: dataString,
			dataType: "json",
			success: function(data) {				
				//var dataStr = JSON.stringify(data); console.log(dataStr);
				if(data.status == "success"){
					save_ret_cart_msg.html("").append('Your cart has been saved! Shopping Cart ID : ' + data.ID);
				}
				else{
					save_ret_cart_msg.html("").append('&lt;span class="eStore_warning"&gt;' + data.details + '&lt;/span&gt;');
				}
			}
		});				
		return false;
	});

	$('.eStore_retrieve_cart_section').on("click", function (){
		var cart_id = prompt("Shopping Cart ID", "");	
		if(cart_id == null){
			return false;
		}
		else if(cart_id ==""){
			save_ret_cart_msg.html("").append('&lt;span class="eStore_warning"&gt;The shopping cart ID you entered does not appear to be correct!&lt;/span&gt;');
			return false;
		}
		cart_id = $.trim(cart_id);
		save_ret_cart_msg.html("").append("Processing Your Request...");
		var dataString = 'eStore_cart_action=eStore_retrieve_cart&amp;cart_id=' + cart_id;
		$.ajax({
			type: "POST",
			url: process_script_url,
			data: dataString,
			dataType: "json",
			success: function(data) {
				if(data.status == "success"){	
					save_ret_cart_msg.html("").append('Success');	
					target_url = window.location.href.split("#")[0];
					window.location.href = target_url;					
				}
				else{
					save_ret_cart_msg.html("").append('&lt;span class="eStore_warning"&gt;The shopping cart ID you entered does not appear to be correct!&lt;/span&gt;');	
				}	
			}
		});		
		return false;
	});
});
});
