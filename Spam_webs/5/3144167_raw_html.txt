       <p>
        The PowerPot is a one of the latest clever solutions to providing a small-scale electricity source, virtually anywhere. With a thermoelectric power source built into a cooking pot so that it generates electricity whilst cooking, the PowerPot is ideal for camping, or where traditional electricity sources are unavailable.
       </p>
       <p>
        The power provided by the PowerPot can be used to charge virtually anything in need of a good charge. It works by employing whats known as the Peltier, which converts heat into electricity. There wasnt much use for such a small amount of electricity until recently, when mobile phones and LEDs made five watts useful.
       </p>
       <p>
        <img alt="PowerPot" class="aligncenter size-full wp-image-12124" height="430" src="http://www.ecocitizenaustralia.com.au/wp-content/uploads/2013/04/PowerPot-main2.jpg" width="645"/>
        For camping and emergencies, the PowerPot is ideal. It takes up little room, has an indefinite shelf life and can be used indoors or outdoors, day or night, rain or shine, and on any heat source sufficient for cooking.
       </p>
       <p>
        But its not just great for people in the Western world. Many of todays mobile phone users around the world, especially in developing countries, dont have easy access to electricity, or the sources they do have are unreliable. For these people, the PowerPot offers a new, easy and affordable solution, especially as open wood fires are often the primary fuel source in this region.
       </p>
       <p>
        With no complicated moving parts or batteries, you can just add water and place the PowerPot on a fire and it will start generating electricity within seconds.Just attach the high temperature cable to the back of the pot and your USB devices charge simply and safely from the heat.
       </p>
       <p>
        <img alt="PowerPot" class="aligncenter size-full wp-image-12125" height="310" src="http://www.ecocitizenaustralia.com.au/wp-content/uploads/2013/04/PowerPot-main.jpg" width="645"/>
        The greater the temperature difference between the water in the pot and the bottom of the pot, the more electricity the PowerPot produces. For example, melting snow or ice in the device is a great way to generate electricity, because snow is so much colder than a flame. It wont overpower your device, because it has a built in regulator, which insures your USB devices will be safely charged.
       </p>
       <p>
        The PowerPot outputs five volts, which is the standard output required for a USB and the most any smartphone or MP3 player on the market can handle. This means when youre charging your USB device, it wont take any longer to charge than it would at home.
       </p>

function mce_success_cb(resp){
    $('#mce-success-response').hide();
    $('#mce-error-response').hide();
    if (resp.result=="success"){
        $('#mce-'+resp.result+'-response').show();
        $('#mce-'+resp.result+'-response').html(resp.msg);
        $('#mc-embedded-subscribe-form').each(function(){
            this.reset();
    	});
    } else {
        var index = -1;
        var msg;
        try {
            var parts = resp.msg.split(' - ',2);
            if (parts[1]==undefined){
                msg = resp.msg;
            } else {
                i = parseInt(parts[0]);
                if (i.toString() == parts[0]){
                    index = parts[0];
                    msg = parts[1];
                } else {
                    index = -1;
                    msg = resp.msg;
                }
            }
        } catch(e){
            index = -1;
            msg = resp.msg;
        }
        try{
            if (index== -1){
                $('#mce-'+resp.result+'-response').show();
                $('#mce-'+resp.result+'-response').html(msg);            
            } else {
                err_id = 'mce_tmp_error_msg';
                html = '&lt;div id="'+err_id+'" style="'+err_style+'"&gt; '+msg+'&lt;/div&gt;';
                
                var input_id = '#mc_embed_signup';
                var f = $(input_id);
                if (ftypes[index]=='address'){
                    input_id = '#mce-'+fnames[index]+'-addr1';
                    f = $(input_id).parent().parent().get(0);
                } else if (ftypes[index]=='date'){
                    input_id = '#mce-'+fnames[index]+'-month';
                    f = $(input_id).parent().parent().get(0);
                } else {
                    input_id = '#mce-'+fnames[index];
                    f = $().parent(input_id).get(0);
                }
                if (f){
                    $(f).append(html);
                    $(input_id).focus();
                } else {
                    $('#mce-'+resp.result+'-response').show();
                    $('#mce-'+resp.result+'-response').html(msg);
                }
            }
        } catch(e){
            $('#mce-'+resp.result+'-response').show();
            $('#mce-'+resp.result+'-response').html(msg);
        }
    }
}
