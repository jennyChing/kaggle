           Game Features:
           <br/>
           - Choose the size of the word search puzzle grid! smallest grid has 4 rows and 4 columns, while the harder grids have up to 14!
           <br/>
           - Choose the category of the words that you want to find!
           <br/>
           - Level Generator: No word soup will ever repeat!
           <br/>
           - Best times: Try to improve the time you solve the puzzle
           <br/>
           - The styles and colors of the lines you draw can be customized!
           <br/>
           - The words that you have to find are shown at the bottom of the grid
           <br/>
           - A puzzle solver function that solves the game for you in the case you get stuck.
           <br/>
           - Design is compatible with small mobile screens and big tablet screens
           <br/>
           <br/>
           Available word search categories:

           <br/>
           - Common words: The most frequently appearing words of the English language
           <br/>
           - Happy words: A collection of "positive" words
           <br/>
           - Cities: Some of the biggest cities of the world
           <br/>
           - Sports: various types of sports
           <br/>
           - Cars: Current and historic car manufacturers
           <br/>
           - Jobs: Common jobs like mechanic or lawyer, but also more "exotic" jobs like clown, pirate etc.
           <br/>
           - Engineering: A dictionary that contains technical words like: angle, pneumatic etc.
           <br/>

           <br/>

           <br/>

           <br/>
           - Mixed: all previously mentioned word categories together

       <div id="description_inner" itemprop="description" style="overflow: hidden">
        The word search puzzle with a twist! Twisty Word Search Puzzle Free is like the classic word search games, but words are not only straight lines, they can even be in zick-zack-lines. That makes the whole game a lot more fun! Moreover the puzzle grids are generated! This means unlimited levels - No puzzle will ever repeat!!
        <br/>
        <br/>
        Game Features:
        <br/>
        - Choose the size of the word search puzzle grid! smallest grid has 4 rows and 4 columns, while the harder grids have up to 14!
        <br/>
        - Choose the category of the words that you want to find!
        <br/>
        - Level Generator: No word soup will ever repeat!
        <br/>
        - Best times: Try to improve the time you solve the puzzle
        <br/>
        - The styles and colors of the lines you draw can be customized!
        <br/>
        - The words that you have to find are shown at the bottom of the grid
        <br/>
        - A puzzle solver function that solves the game for you in the case you get stuck.
        <br/>
        - Design is compatible with small mobile screens and big tablet screens
        <br/>
        <br/>
        Available word search categories:
        <br/>
        - Popular girl names: From "Amelia" till "Zoe"
        <br/>
        - Popular boy names: From "Adrian" till "Zachary"
        <br/>
        - Country names: From "Afghanistan" till "Zimbabwe"
        <br/>
        - Common words: The most frequently appearing words of the English language
        <br/>
        - Happy words: A collection of "positive" words
        <br/>
        - Cities: Some of the biggest cities of the world
        <br/>
        - Sports: various types of sports
        <br/>
        - Cars: Current and historic car manufacturers
        <br/>
        - Jobs: Common jobs like mechanic or lawyer, but also more "exotic" jobs like clown, pirate etc.
        <br/>
        - Engineering: A dictionary that contains technical words like: angle, pneumatic etc.
        <br/>
        - Computers: Words related to programming and computer science
        <br/>
        - Food: Vegetables, Fruits and other alimentation
        <br/>
        - Home: All kind of things that you can find inside &amp; around your home
        <br/>
        - Mixed: all previously mentioned word categories together
        <br/>
        <br/>
        This app is tested with:
        <br/>
        - Most popular mobile phones: Samsung Galaxy Nexus, Galaxy SL, Galaxy S2, Galaxy S3 and more.
        <br/>
        - small screen devices (smallest screen 2,7)
        <br/>
        - 7 tablets and 10 tablets
        <br/>
        - Android 2.3.3 "Gingerbread" up to 4.4.2 "KitKat"
