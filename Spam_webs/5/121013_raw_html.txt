          <h3 class="post-title entry-title">
           <a href="http://www.cooliphoneipadapps.com/2014/05/piano-companion-chords-scales-stave.html">
            Piano Companion: chords, scales, stave, circle of fifths, chord progression
           </a>
          </h3>
          <div class="post-header-line-1">
          </div>
          <div style="float:right;padding:6px;">
           <a addthis:title="Piano Companion: chords, scales, stave, circle of fifths, chord progression" addthis:url="http://www.cooliphoneipadapps.com/2014/05/piano-companion-chords-scales-stave.html" class="addthis_button">
            <img alt="Bookmark and Share" height="16" src="http://s7.addthis.com/static/btn/v2/lg-bookmark-en.gif" style="border:8" width="125"/>

           </script>
          </div>
          <br/>
          <div class="post-body entry-content">
           <br/>
           <br/>
           <div style="text-align: center;">
            <b>
             Developer Quote:
            </b>
            "
            <b>
             Piano Companion
            </b>
            is a flexible chord/scale dictionary with user libraries and reverse mode.
            <b>
             Piano Companion
            </b>
            allows you to find a chord or scale by its name."
            <br/>
            <br/>
            <div class="separator" style="clear: both; text-align: center;">
             <a href="http://3.bp.blogspot.com/-XFJ3jT7i1ew/U2KRTbfVB0I/AAAAAAAAVBw/HgAO-lN3p4U/s1600/iconpiano.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
              <img border="0" height="125" src="http://3.bp.blogspot.com/-XFJ3jT7i1ew/U2KRTbfVB0I/AAAAAAAAVBw/HgAO-lN3p4U/s320/iconpiano.png" width="125"/>
             </a>
            </div>
            <br/>
            <span style="color: #990000;">
             <b>
              - COST:   Free  - APP STORE STARS:   4-1/2
             </b>
            </span>
            <br/>
            <b>
             - Categories:
            </b>
            Education, Entertainment, Lifestyle, Music, Universal
            <br/>
            <div class="separator" style="clear: both; text-align: center;">
             <a href="http://3.bp.blogspot.com/-rEM62yR6uWY/U2KReZ1dcgI/AAAAAAAAVB4/5PaWmPAfK9Q/s1600/screen568x568.jpeg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
              <img border="0" src="http://3.bp.blogspot.com/-rEM62yR6uWY/U2KReZ1dcgI/AAAAAAAAVB4/5PaWmPAfK9Q/s320/screen568x568.jpeg"/>
             </a>
            </div>
           </div>
           <br/>
           <div style="text-align: justify;">
            <span style="color: #0b5394;">
             Piano Companion shows chord/scale notes, degrees, fingering for both hands in major and minor scales. You can listen how a scale or chord sounds. You can see the list of compatible chords in the selected scale. The great advantage of Piano Companion is the fact that it's a growing application, so we will be glad to consider any new features which helps you to increase usability. Take a look at the many features of this
             <a href="http://www.cooliphoneipadapps.com/2014/05/piano-companion-chords-scales-stave.html">
              music education iPhone, iPad app
             </a>
             :
             <br/>
             <br/>
             - 1500+ piano chords, up to 6th inversions
             <br/>
             - 10000+ scales
             <br/>
             - 12 root keys
             <br/>
             - Chord Progression (chord sequencer)
             <br/>
             - Circle of Fifths
             <br/>
             - MIDI keyboard (with iRig) support for reverse mode in iOS6+
             <br/>
             - List of available chords in Circle of Fifths
             <br/>
             - Key notations: English, Italian, Japanese, Russian, etc.
             <br/>
             - Reverse chord &amp; scale lookup
             <br/>
             - Ability to display chord on the staff
             <br/>
             - Plays chords
             <br/>
             <br/>
            </span>
            <br/>
            <div class="separator" style="clear: both; text-align: center;">
             <span style="color: #0b5394;">
              <a href="http://1.bp.blogspot.com/-r6seUxuG5rc/U2KWft_FoWI/AAAAAAAAVCI/DyxfV2apKeo/s1600/screen568x568+(1).jpeg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
               <img border="0" src="http://1.bp.blogspot.com/-r6seUxuG5rc/U2KWft_FoWI/AAAAAAAAVCI/DyxfV2apKeo/s320/screen568x568+(1).jpeg"/>
              </a>
             </span>
            </div>
            <span style="color: #0b5394;">
             - Plays scales
             <br/>
             - Plays arpeggiated chords
             <br/>
             - Audio support
             <br/>
             - Relative scale suggestions
             <br/>
             - Compatible scale chords
             <br/>
             - Scale lookup with major &amp; minor scale fingering
             <br/>
             - Flexible chord &amp; scale search using keyboard
             <br/>
             - User libraries
             <br/>
             - Ability to create custom chords for a user library
             <br/>
             - Ability to create favorite scales
             <br/>
             <br/>
            </span>

             <strong>
              --Date/Version:

             Updated 4/8/14  -  Version 4.7.404
