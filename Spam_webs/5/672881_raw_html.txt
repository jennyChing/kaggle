         <h1>
          the most useful firefox addons for webmasters
         </h1>
         <p class="poast_meta">
          January 16th, 2008                by
                aravind
         </p>
        </header>
        <div class="post-thumbnail">
         <img alt="Free Hosting Directory" class="attachment-main-thumb wp-post-image" height="105" src="http://thedesignsuperhero.com/wp-content/uploads/2008/01/freehost.jpg" width="501"/>
        </div>
        <div class="entry">
         <p>
          You know, the number one browser  Firefox has cool add-ons which make the job of website designers and developers much easier. Here I have a list of excellent Firefox add-ons that every web developer and designer should know.
         </p>
         <p>
          <a href="http://thedesignsuperhero.com/2008/01/the-most-usefull-firefox-addons-for-webmasters/free-hosting-directory/" rel="attachment wp-att-20" title="fff.png">
           <img alt="Firefox" src="http://thedesignsuperhero.com/blog/wp-content/uploads/2008/01/fff.png" style="float: none"/>
          </a>
          <span id="more-39">
          </span>
         </p>
         <h4>
          1.
          <a href="https://addons.mozilla.org/en-US/firefox/addon/60" target="_blank">
           Web Developer
          </a>
         </h4>
         <p>
          <a href="https://addons.mozilla.org/en-US/firefox/addon/60">
           <img align="right" alt="" height="150" src="https://addons.mozilla.org/en-US/firefox/images/addon_preview/60/1" style="float: none" width="200"/>
          </a>
         </p>
         <p>
          A developer essential, Web Developer adds an array of custom tools and menus to Firefox. Use it in tandem with Firebug for maximum functionality and control over your code.
         </p>
         <h4>
          2.
          <a href="https://addons.mozilla.org/en-US/firefox/addon/1843" target="_blank">
           Firebug
          </a>
         </h4>
         <p>
          <a href="https://addons.mozilla.org/en-US/firefox/addon/1843" target="_blank">
           <img alt="" height="157" src="https://addons.mozilla.org/en-US/firefox/images/preview/1843/1" style="float: none" width="209"/>
          </a>
         </p>
         <p>
          Firebug integrates with Firefox to put a wealth of development tools at your fingertips while you browse. You can edit, debug, and monitor CSS, HTML, and JavaScript live in any web page
         </p>
         <p>
          Firebug integrates with Firefox to put a wealth of development tools at your fingertips while you browse. You can edit, debug, and monitor CSS, HTML, and JavaScript live in any web page.
         </p>
         <h4>
          3.
          <a href="https://addons.mozilla.org/en-US/firefox/addon/1843" target="_blank">
           FireFTP
          </a>
         </h4>
         <p>
          <a href="https://addons.mozilla.org/en-US/firefox/addon/1843" target="_blank">
           <img alt="" height="128" src="https://addons.mozilla.org/en-US/firefox/images/addon_preview/684/1" style="float: none" width="128"/>
          </a>
          FireFTP is a free, secure, cross-platform FTP client for Mozilla Firefox which provides easy and intuitive access to FTP servers.  Along with transferring your files quickly and efficiently, FireFTP also includes more advanced features such as: directory comparison, syncing directories while navigating, SSL encryption, search/filtering, integrity checks, remote editing, drag &amp; drop, file hashing, and much more!
         </p>
         <h4>
          4.
          <a href="https://addons.mozilla.org/en-US/firefox/addon/271" target="_blank">
           ColorZilla
          </a>
         </h4>
         <p>
          <a href="https://addons.mozilla.org/en-US/firefox/addon/271" target="_blank">
           <img alt="" height="103" src="https://addons.mozilla.org/en-US/firefox/images/preview/271/1" width="200"/>
          </a>
          Advanced Eyedropper, ColorPicker, Page Zoomer and other colorful goodies.  With ColorZilla you can get a color reading from any point in your browser, quickly adjust this color and paste it into another program. You can Zoom the page you are viewing and measure distances between any two points on the page. The built-in palette browser allows choosing colors from pre-defined color sets and saving the most used colors in custom palettes. DOM spying features allow getting various information about DOM elements quickly and easily. And theres more
         </p>
         <h4>
          5.
          <a href="https://addons.mozilla.org/en-US/firefox/addon/748" target="_blank">
           Greasemonkey
          </a>
         </h4>
         <p>
          <a href="https://addons.mozilla.org/en-US/firefox/addon/748" target="_blank">
           <img alt="Greasemonkey" height="133" src="https://addons.mozilla.org/en-US/firefox/images/preview/748/1" title="Greasemonkey" width="176"/>
          </a>
          Allows you to customize the way a webpage displays using small bits of JavaScript.   Allows you to customize the way a webpage displays using small bits of JavaScript.  Hundreds of scripts, for a wide variety of popular sites, are already available at http://userscripts.org.  You can write your own scripts, too. Mark Pilgrims definitive Greasemonkey guide, diveintogreasemonkey.org will show you how.
         </p>
         <h4>
          6.
          <a href="https://addons.mozilla.org/en-US/firefox/addon/655" target="_blank">
           View Source Chart
          </a>
         </h4>
         <p>
          <a href="https://addons.mozilla.org/en-US/firefox/addon/655" target="_blank">
           <img alt="View Source Chart" height="150" src="https://addons.mozilla.org/en-US/firefox/images/preview/655/1" title="View Source Chart" width="143"/>
          </a>
          *   Draws A Color-Coded Chart of a Web Pages Source Code  Source Charting  * Graphically Displays HTML Tag Boundaries  * Graphically Defines Tag Nesting Order, Structure and Hierarchy  * Adds A Simple but Powerful User Interface *TO* Source Code  WHAT SOURCE CHARTING DOES THAT INDENTATION-ONLY FORMATTING DOES NOT:  Source Charting Allows the User To  * quickly scan and recognize a documents tags without reading a single tag  * see how deeply nested an element is just by looking to its left (no scrolling/finding/reading tags)  * identify an elements containing tags without having to scroll, find and read each tag
         </p>
         <h4>
          7.
          <a href="https://addons.mozilla.org/en-US/firefox/addon/1146" target="_blank">
           Screengrab!
          </a>
         </h4>
         <p>
          <a href="https://addons.mozilla.org/en-US/firefox/addon/1146" target="_blank">
           <img alt="Screen Grab" height="229" src="https://addons.mozilla.org/en-US/firefox/images/preview/1146/1" title="Screen Grab" width="305"/>
          </a>
          Screengrab saves entire webpages as images.  It will save what you can see in the window, the entire page, just a selection, a particular frame basically it saves webpages as images.
         </p>
         <h4>
          8.
          <a href="https://addons.mozilla.org/en-US/firefox/addon/4287" target="_blank">
           Split Browser
          </a>
         </h4>
         <p>
          <a href="https://addons.mozilla.org/en-US/firefox/addon/4287" target="_blank">
           <img alt="Split Browser" height="150" src="https://addons.mozilla.org/en-US/firefox/images/preview/4287/1" title="Split Browser" width="192"/>
          </a>
          This splits the content area of the browser window as you like. It will help you in various cases. For example, to compare multiple webpages, to show a calendar always, and so on.  There is some ways to split window.  * Choose the Split Browser to menu in the context menu.  * Choose the Load in Split Browser menu in the context menu on link.  * Click popup-button on top/bottom/left/right edges of the content area.  * Drop links, bookmarks, etc. to popup-button on top/bottom/left/right edges of the content area while dragging.  You can split browser infinitely, as many as your RAM and your screen allow. Enjoy split and split to your hearts content!
         </p>
         <h4>
          9.
          <a href="https://addons.mozilla.org/en-US/firefox/addon/3818" target="_blank">
           Resizeable Textarea
          </a>
         </h4>
         <p>
          A lot of forums and boards provide very small textareas for new postings. If you type a longer posting, you have to scroll up and down, to read what you have written and to proceed. This situation is annoying.  Now here is my solution:
          <a href="https://addons.mozilla.org/en-US/firefox/addon/3818" target="_blank">
           Resizeable Textarea extension
          </a>
          . It lets you resize most of those text areas.
         </p>
         <h4>

          <a href="http://www.mozilla.org/products/thunderbird/" target="_blank">
           Thunderbird
          </a>
          ,    but been frustrated that the text formatting (bold, italic, font, etc.) were copied with it?  This extension     copies the text without all the formatting.
